Backup Service

Backup Service allows other users to back up data from their Synology NAS devices via the package Backup & Restore to your Synology NAS.

Enabling Synology Backup Server

To enable backup service:

You can transform your Synology NAS into the data/LUN backup destination for other remote Synology NAS.

  1. Go to Control Panel > Backup Service.
  2. Click Enable backup destination for remote Synology servers.
  3. Click Apply to save the setting.
  4. Go to Control Panel > Shared Folder, select a shared folder, click Edit > Permissions, and then grant the Read/Write privilege to the desired users. Repeat this step to include more shared folders that can be used as backup destinations.

Note:

  • Remote Synology NAS must use the package Backup & Restore to back up their data/LUN to your Synology NAS.
  • To allow LUN backup from remote Synology NAS, go to Control Panel > File Services > rsync, and select Enable rsync service on the destination side.

To assign backup privileges:

Assign proper backup privileges to users so that these users can back up data from a source Synology NAS to the destination Synology NAS.

  1. Go to Control Panel > Privileges, click Synology Backup > Edit.
  2. Assign Allow privileges to the desired users, or further specify the allowed IP addresses for specific users.
