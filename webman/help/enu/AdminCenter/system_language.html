Language

You can select your preferred language for DSM user interface, Google Chrome desktop notifications, email, SMS, instant messaging notifications, and application codepage.

Display language

You can choose your preferred display language for DSM and Google Chrome desktop notifications, or have it the same as your default browser setting.

Notification language

You can set your preferred language for email, SMS, Windows Live Messenger, and Skype notifications. It will also be applied to the notifications received on Windows Phone.

However, the notification language on an iOS or Android device depends on the language setting of your mobile device.

Codepage

For applications that convert non-Unicode strings to Unicode, such as the FTP service, UPnP support, music metadata indexing, you will need to choose the appropriate codepage for them from the Codepage drop-down menu.
