Time

You can check the current time, manually set the server's date and time, or have them set automatically using a network time server.

You can check the current time and set the time zone.

To set the time zone:

Choose an option from the Time zone drop-down menu.

To set the date and time manually:

Choose Manually and select the date and time.

To sync the date and time with NTP server:

  1. Choose Synchronize with a NTP server, and enter the address of your network time server or choose a server from the drop-down menu.
  2. Click Update Now to make sure your DiskStation can successfully synchronize time with the server.
  3. Click Apply.

Note: To sync the date and time with a network time server, please make sure a valid DNS server IP has been entered at Main Menu > Control Panel > Network and that your DiskStation is connected to the Internet.

About Network Time Protocol (NTP) Server

The main purpose of Network Time Protocol (NTP) is for synchronizing clocks of computer system over data network.

Below are a few recommended NTP servers:

  • time.apple.com
  • time.asia.apple.com
  • time.euro.apple.com
  • time.nist.gov

For more information about NTP Server, please visit http://www.eecis.udel.edu/~mills/ntp/servers.html
