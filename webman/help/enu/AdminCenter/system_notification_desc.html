Notification

You can set your DiskStation to send notifications when specific events or errors occur, notifying you via email, SMS, mobile devices, or web browsers. For example, the system can automatically send a notification when the network connection is lost, cooling fan behavior changes, power failure occurs, storage volume is running out of space, or backup and restoration tasks fail to complete.
