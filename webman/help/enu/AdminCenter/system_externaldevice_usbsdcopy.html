USBCopy

You can select a destination folder for the USBCopy function by going to Control Panel > External Devices > USBCopy/SDCopy.

To enable USBCopy:

  1. Tick Enable USBCopy.
  2. Choose a destination shared folder from the drop-down menu.

When you attach a USB device (such as a digital camera) to the USB port on the front panel of Synology product, you can press the Copy button on the front panel to copy data on the USB device to the destination shared folder assigned here. If the USBCopy destination folder is deleted, the system will replace it with the public shared folder automatically.

Limitations

  1. Only USB devices in FAT, NTFS, ext3 and ext4 formats are supported.
  2. USB devices must be plugged into the USB port on the front panel of DiskStation device.
  3. Only the admin or users in the administrators group have proper access privileges to conduct backup tasks to USB devices.
  4. The available space of the destination shared folder must be larger than the used space of the USB device.
