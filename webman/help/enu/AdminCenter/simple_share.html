Shared Folder

Shared folders are the basic directories where you can store files and folders on your DiskStation. Before storing any files on your DiskStation, you'll need to create at least one shared folder in DSM. Data stored in shared folders can be kept private or shared with specific users or user groups according to customizable access permissions.

Below is a list of default shared folders that are created automatically when certain applications, services, or packages are installed or enabled.

Name Description
public The public folder is created automatically after your DiskStation is first set up.*
web The web folder is the default storage location for the contents of a website hosted on the DiskStation. It is created automatically when Web Station is enabled.
photo The photo folder is used to store photos and videos you want to share with Photo Station. It is created automatically when Photo Station or DLNA/UPnP Media Server is enabled.
music The music folder is used to store music you want to listen to with Audio Station. It is created automatically when Audio Station or DLNA/UPnP Media Serveris enabled.
video The video folder is used to store video files you want to browse through DLNA/UPnP devices. It is created automatically when Video Station or DLNA/UPnP Media Serveris enabled.
surveillance The surveillance folder is used to store Surveillance Station recordings. It is created automatically when Surveillance Station is enabled. It is read-only and can only be accessed by the system administrator.
home The home folders provide private spaces for each user to store data that only that user can access. It is created automatically when the User Home feature is enabled.
homes The homes folder contains the home folders of all users. It is created automatically when the User Home feature is enabled. Only the system administrator can access and manage all users' home folders.
NetBackup The NetBackup folder is created automatically when Network Backup is enabled.
usbshare The usbshare[number] folder is created automatically when you connect an USB external drive device to the USB port of the DiskStation.*
satashare The satashare folder is created automatically when you connect an eSATA external drive device to the eSATA port of the DiskStation.*

Note:

  • You can only edit users' shared folder permissions in the simple mode. If you want to create user groups or manage groups' access privileges to shared folders, please go to Control Panel (Advanced Mode) > Group.

To create a shared folder:

  1. Go to Control Panel (Simple Mode) > Shared Folder and click the + icon on the left panel.
  2. Specify a name and select a volume on which to create the shared folder.
  3. Tick Enable Recycle Bin if necessary. If this option is enabled, when files in the shared folder are deleted, they will be moved to a folder named #recycle. Access to the recycle bin can be limited to users belonging to the administrators group.
  4. Check or uncheck the appropriate boxes for each user to customize their access permissions for the shared folder:
    • Read/Write: The user can access and make changes to the files and subfolders in the shared folder.
    • Read only: The user can access the files and subfolders in the shared folder.
    • No access: The user cannot access the files or subfolders in the shared folder.
  5. Click Apply to save settings.

Note:

  • Shared folder names are case insensitive and can include 1-32 characters. Names cannot include the following characters:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • The following names are reserved for system use:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • The first character of a shared folder name cannot be a minus sign or a space, and the last character cannot be a space character.
  • When you encounter permissions conflicts, the permissions priority is as follows: No access > Read/Write > Read only.
  • When creating a new shared folder, if the permissions for the users belonging to the administrators group are set to No access, these users will only be able to see the shared folders at Control Panel > Shared Folder.
  • You can configure privilege settings of the Recycle Bin folder in File Station by right-clicking the #recycle folder. Choose Properties and click on the Permissions tab.

To edit a shared folder:

  1. Go to Control Panel (Simple Mode) > Shared Folder.
  2. Select the desired shared folder on the left panel.
  3. Edit the general information of the shared folder, such as name, location, and user permissions.
  4. If Enable Recycle Bin has been ticked already, you can click Empty Recycle Bin to remove all deleted files in the recycle bin permanently.
  5. Click Apply to save settings.

To delete a shared folder:

  1. Go to Control Panel (Simple Mode) > Shared Folder.
  2. Select the desired shared folder on the left panel.
  3. Click the Trash icon on the left panel to delete the selected shared folder.
