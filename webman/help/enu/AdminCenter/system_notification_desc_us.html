Notification

You can set up DiskStation to send you notification messages via e-mail, mobile devices, Windows Live Messenger, or Skype, when status changes or errors occur to the system, for example: connection loss, fan behavior change, power failure, storage volume running out of space, or back and restoration failure, etc.
