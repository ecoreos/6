Printers

DiskStation can become a print server, allowing client computers or mobile devices to access printers connected to DiskStation. In addition, Apple Wireless Printing and Google Cloud Print support allows you to remotely print using iOS devices or Google products and services.

Visit www.synology.com for compatible printers.

Setting up Printers

See the instructions below to set up and connect a USB or network printer with your DiskStation.

To set up a USB printer:

  1. Power on and connect the printer to a USB port on the DiskStation.
  2. Go to Control Panel > External Devices > Printer.
  3. Select the printer you wish to set up. Please note that DiskStation cannot recognize printers connected via USB hubs.
  4. Click Printer Manager and select Set Up Printer.
  5. On the dialog box, select one of the following modes:
    • Network MFP: Allows clients within the local network to use print, scan, and fax functions of your multifunction printer. Scanning and fax functions are available for Windows computers only.
    • Network Printer: Choose this mode for printers that only support printing or if you want to allow clients to use the printing function only.
  6. Configure Apple Wireless Printing or Google Cloud Print settings (optional).
  7. Click Save.

Note:

A network MFP can only be accessed by one client concurrently.

To set up a network printer:

  1. Power on and connect the network printer to the same local network as your DiskStation.
  2. Go to Control Panel > External Devices > Printer.
  3. Click Add Network Printer and follow the instructions of the setup wizard.

Note:

  • Printer Name: A printer name must be entered during setup and can include up to 32 characters, including letters, numbers, and underscores.
  • Protocol: In general, HP network printers use Socket; Canon network printers use BJNP; and others mostly use LPR.
  • Queue Name: Entering a queue name is required for LPR and IPP network printers, but not applicable for Socket or BJNP printers.
  • No Network MFP: Network printers connected to DiskStation cannot use Network MFP mode at present.
  • No USB Printers: USB printers shared by a Synology product cannot be set as network printers.

Managing Printers

The printer tab includes the following options to manage USB or network printers connected to the DiskStation.

  • Get Info: Displays information related to the selected printer, such as device name, manufacturer, status, etc.
  • Remove: Removes and disconnects a printer from the DiskStation.
  • Printer Manager: Includes the following options:
    • Set up Printer: Edit printer settings, such as Apple Wireless Printing or Google Cloud Print (see below for instructions).
    • Clean up spool space: Cancels all printing tasks on all printers.
    • Print Test Page: Prints a test page to make sure configurations are correct.
    • Release Printer: Disconnects the printer and client (for Network MFP mode only). When a client connects to a multifunctional printer to receive faxes or to scan files, the task might take a long time. If necessary, the connection can be stopped in order to allow the next client in the line to access the printer.

To enable Apple Wireless Printing:

  1. Go to Control Panel > External Devices > Printer.
  2. Select a printer from the device list.
  3. Click Printer Manager > Set Up Printer.
  4. Check the box Enable Apple Wireless Printing.
  5. Select your printer's brand and driver.
  6. Click Save.

After setup is complete, you can print a test page by clicking Printer Manager > Print Test Page.

Note:

  • Enabling Apple Wireless Printing also enables Bonjour Printer Broadcast for DSM.
  • Apple Wireless Printing is available for iOS 4.2 or later.

To enable Google Cloud Print:

  1. Go to Control Panel > External Devices > Printer.
  2. Select a printer from the device list.
  3. Click Printer Manager > Set Up Printer.
  4. Check the box Enable Google Cloud Print.
  5. Enter the following details:
    • Your Google account information.
    • Your printer's brand and driver.
  6. Click Save.

After setup is complete, you can print a test page by clicking Printer Manager > Print Test Page.

To manage Google Cloud Print tasks, click the Google Cloud Print link.

Accessing Printers

See the section below to access printers using a PC or Mac.

To access printers using a Windows PC:

The best way to access DiskStation device's printing service is to set up the connection to the printer with Synology Assistant. Refer to its Help for detailed instructions.

If Synology Assistant is not installed on your computer, see below for other ways of accessing DiskStation device's printing service.

Type 1. Shared Network Printer

  1. On you PC, open Network in File Explorer and double click on the DiskStation.
  2. Double click on usbprinter or network printer name; the system prompts you to install the correct driver, and then click Yes.
  3. Please get the printer driver from the CD or website of the manufacturer. Select the manufacturer and model of the printer, or click Have Disk to select the location of the driver. Then click OK.
  4. The system will start to set up the network printer. After the setup is completed, you may start using the printer.

Type 2. LPR Port (Recommended)

  1. On your PC, click Start and go to Devices and Printers.
  2. Click Add Printer.
  3. Choose Add a network, wireless or Bluetooth printer.
  4. Choose The printer that I want isn't listed.
  5. Select Add a printer using TCP/IP address or hostname. Click Next to continue.
  6. Choose the Device type to be TCP/IP Port Device.
  7. Enter the IP address of the DiskStation. Click Next to continue.
  8. Select Device Type: Custom > Settings > â€¦. Click Next to continue.
  9. Select LPR for protocol; Enter usbprinter or network printer name for Queue Name, and check the LPR Byte Counting Enabled checkbox. Click OK to continue.
  10. Please get the printer driver from the CD or website of the manufacturer. Select the manufacturer and model of the printer, or click Have Disk to select the location of the driver. Then click OK.
  11. The system will start to set up the network printer. After the setup is complete, you may start using the printer.

To access printers using a Mac:

Mac users can access the DiskStation device's printing service by enabling DSM's Bonjour printer broadcast and adding the printer in System Preferences.

Please make sure your DiskStation is using Postscript printer before proceeding.

Step 1. Enable DSM's Bonjour printer broadcast

  1. Log into DSM as admin or a user belonging to the administrators group.
  2. Go to Control Panel > File Services > Win/Mac/NFS.
  3. Tick Enable Bonjour Printer Broadcast.

Step 2. Access printer using Mac

  1. On your Mac, go to System Preferences > Print & Scan, and then click +.
  2. Choose usbprinter[number] or network printer name from the drop-down menu. Your Mac will set up the printer automatically.
