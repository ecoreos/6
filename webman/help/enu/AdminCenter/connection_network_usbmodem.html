Mobile*

When equipped with a mobile broadband dongle, the DiskStation can connect to the Internet via a wireless, 3G/LTE network. The settings at Control Panel > Network > Network Interface > Mobile will help you establish and modify a wireless Internet connection via your mobile broadband dongle.

Note:

For a complete list of mobile broadband dongles supported by Synology products, please visit www.synology.com.

To set up a mobile network:

When a mobile broadband dongle is connected to one of the USB ports on your DiskStation, the DiskStation will try to connect automatically. If a connection cannot be established automatically or you would like to modify connection settings, please see the following:

  • SIM Unlock/Settings: If the SIM card is PIN protected, it must be unlocked to allow the DiskStation to connect to the network. Once unlocked, this section provides options to modify PIN settings.
  • Edit: You can manually configure the connection or choose according to location and carrier. Manually setting up the connection requires entering your APN, phone number, username, and password. If you do not know this information, please contact your mobile carrier for support.
  • Connect/Disconnect: These buttons allow you to connect or disconnect the DiskStation from the 3G/LTE network.
  • Options: This section includes an option to set the mobile broadband dongle interface as the default gateway for your DiskStation. When you configure the mobile network as the default gateway, Internet traffic from your DiskStation will go through the mobile network instead of the wired network.

Note:

  1. When connected to the Internet, some mobile broadband dongles will not display the signal strength if they do not support this feature.
  2. For some mobile broadband dongles, you cannot change the SIM settings when they are connected to the Internet.
