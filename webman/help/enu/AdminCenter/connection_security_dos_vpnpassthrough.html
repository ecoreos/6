Protection

Control Panel > Security > Protection provides options to enhance network security for each network interface.

DoS Protection

Denial-of-service (DoS) protection helps prevent malicious attacks over the Internet.

To enable DoS protection:

  1. Check the Enable DoS protection box.
  2. Click Apply.

VPN Pass-through*

When the DiskStation acts as a router, VPN pass-through allows clients to pass through the DiskStation NAT and establish outbound or inbound VPN connections. If your internal setup includes a VPN server, please go to Control Panel > External Access > Router Configuration to set up port forwarding rules and allow the VPN server to establish VPN connections.

To edit VPN pass-through settings:

  1. The following settings can be enabled or disabled here:
    • PPTP Pass-through: Allows PPTP (Point-to-Point Tunneling Protocol) traffic to pass through and establish connections.
    • IPSec Pass-through: Allows IPSec (IP Security Protocol) traffic to pass through and establish connections.
    • L2TP Pass-through: Allows L2TP (Layer 2 Tunneling Protocol) traffic to pass through and establish connections.
  2. After modifying settings, click Apply.

Note:

If your DiskStation device connects to the Internet via PPPoE, the related DoS protection or VPN pass-through rules must be configured on the corresponding PPPoE interface.

____
*Only available on certain models.
