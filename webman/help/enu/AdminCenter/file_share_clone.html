Clone a Shared Folder*

You can clone a shared folder on Btrfs volumes to create near-instantaneous copies of an existing shared folder. Cloning a shared folder will consume only a fraction of storage at the time of creation.

To clone a shared folder:

  1. Go to Control Panel > Shared Folder.
  2. Select the desired shared folder in Btrfs file system.
  3. Click Create > Clone.
  4. Specify a name and description for the new shared folder, and modify other options if necessary. (For detailed information about these options, please refer to Create a Shared Folder.)

Note:

  • Only shared folders on Btrfs volumes can be cloned.
  • The cloned shared folder will be created on the same volume as the original shared folder.
  • The settings of advanced data integrity protection will be applied to the cloned shared folder and cannot be changed.

____
*Only available on certain models.
