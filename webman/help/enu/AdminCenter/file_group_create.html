Create a Group

  1. Go to Control Panel > Group.
  2. Click Create. This action launches the User Creation Wizard.
  3. Enter the group name and description. (See the section below for more information). Click Next.
  4. Follow the remaining steps of the wizard to finish creating the group.
  5. Click Apply to finish.

Naming conventions:

  • The group name is case sensitive and should be from 1 to 32 Unicode characters, excluding the following symbols:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • The first character cannot be a minus sign or a space, and the last character cannot be a space.
  • The group description is case sensitive and limited up to 64 displayable Unicode characters.
