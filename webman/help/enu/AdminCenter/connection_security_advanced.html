Advanced

You can modify the following advanced settings at Control Panel > Security > Advanced for DSM browsing sessions:

HTTP Compression

You can enable HTTP Compression for DiskStation. By lowering the bandwidth of network traffic, it can increase the speed of loading webpage.

To Enable / Disable HTTP Compressionï¼š

  • Check / uncheck the HTTP Compress at Control Panel > Security > Advanced.

TLS / SSL Profile

You can choose the security level of HTTPS encrypted connections. It provides 3 level as following:

  • Modern compatibility - Only compatible with latest browser and mobile devices.
  • Intermediate compatibility - Between top and lowest security level. It's system default value.
  • Old backward compatibility - Not secure, but support most of the browser and mobile devices.
