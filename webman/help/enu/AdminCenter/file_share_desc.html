Shared Folder

Shared folders are the basic directories where you can store files and folders on your DiskStation. Before storing any files on your DiskStation, you need to create at least one shared folder. Data stored in shared folders can be kept private or shared with specific users or user groups according to customizable access permissions. In addition, shared folders can be encrypted during creation, adding an additional layer of protection.

Default Shared Folders

Below is a list of shared folders that are created automatically when certain applications, services, or packages are installed or enabled.

Name Description
public The public folder is created automatically after your DiskStation is first set up.*
web The web folder is the default storage location for the contents of a website hosted on the DiskStation. It is created automatically when Web Station is enabled.
photo The photo folder is used to store photos and videos you want to share with Photo Station. It is created automatically when Photo Station or DLNA/UPnP Media Server is enabled.
music The music folder is used to store music you want to listen to with Audio Station. It is created automatically when Audio Station is enabled.
video The video folder is used to store video files you want to browse through DLNA/UPnP devices. It is created automatically when Media Server is enabled.
surveillance The surveillance folder is used to store Surveillance Station recordings. It is created automatically when Surveillance Station is enabled. It is read-only and can only be accessed by the system administrator.
home The home folders provide private spaces for each user to store data that only that user can access. It is created automatically when the User Home feature is enabled.
homes The homes folder contains the home folders of all users. It is created automatically when the User Home feature is enabled. Only the system administrator can access and manage all users' home folders.
NetBackup The NetBackup folder is created automatically when Network Backup is enabled.
usbshare The usbshare[number] folder is created automatically when you connect an USB external drive device to the USB port of the DiskStation.*
satashare The satashare folder is created automatically when you connect an eSATA external drive device to the eSATA port of the DiskStation.*

____
* Available on certain models only.
