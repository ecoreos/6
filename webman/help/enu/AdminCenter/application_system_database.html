System Database

DiskStation uses a system database to keep track of package data and system log entries stored on the shared folders created on your external storage devices, such as usbshare1.

Important: If you want to access package data stored on the external storage devices attached to your DiskStation, or if you want to see DSM's log records at Main Menu > Log Center, the system database has to be enabled first. Otherwise, all related services will be inaccessible.

To enable the system database:

  1. Tick Enable system database.
  2. Do any of the following:
    • Choose Automatic configuration if you want to have DSM automatically select a shared folder for storing the system database. If the external storage device which contains the shared folder is removed, DSM will automatically select any other shared folder for storing the database.
    • Choose Move the current database to the selected shared folder, and then choose from the drop-down menu to move the current database to another shared folder created on your external device.
    • Choose Choose one of the existing databases, and then choose from the drop-down menu to select a database in existence if there is more than one external device connected to your DiskStation.
  3. Click Apply.

Warning: To prevent the database from being corrupted, make sure you have ejected the storage device at Main Menu > Control Panel > External Devices > External Devices or by clicking the external devices icon at the top-right corner before removing it from DiskStation.
