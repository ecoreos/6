User

With flexible user management options, you can create user accounts for individual members of your family or business and define shared folder access permissions for each person.

To create a user:

  1. Go to Control Panel (Simple Mode) > User and click the + icon on the left panel.
  2. Specify a name and password for the user.
  3. Specify the groups to which the new user shall belong.
  4. Assign appropriate shared folder permissions to the new user. When you encounter permissions conflicts, the permission priority is as follows: No access > Read/Write > Read only.
    • Preview: This column displays the user's actual shared folder access privileges, according to current settings and groups to which the user belongs.
    • Read/Write: The user can access and make changes to the files and subfolders in the corresponding shared folder.
    • Read only: The user can access the files and subfolders in the corresponding shared folder.
    • No access: The user cannot access the files or subfolders in the corresponding shared folder.
  5. Click Apply to save settings.

    6. Username limitation:

    • The username is not case sensitive and can include 1 to 64 Unicode characters, but cannot contain the following symbols:
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • The first character cannot be a minus sign or a space, and the last character cannot be a space.
    • The following user names are reserved for system use:
      mailer-daemon, postmaster

    Password limitation:

    • The password is case sensitive and is limited up to 127 displayable characters, including letters, numbers, signs, and space.

To edit a user:

  1. Go to Control Panel (Simple Mode) > User.
  2. Select the desired user on the left panel.
  3. Edit the general information of the user, such as name, password, group, and shared folder permissions.
  4. Tick Disable this account if you want to make the account stop functioning.
  5. Click Apply to save settings.

To remove a user:

  1. Go to Control Panel (Simple Mode) > User.
  2. Select the desired user on the left panel.
  3. Click the Trash icon on the left panel to delete the selected user.
