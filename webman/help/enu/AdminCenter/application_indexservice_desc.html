Indexing Service

Media Indexing can automatically scan multimedia files, such as photos, music, and videos stored on your DiskStation, and compile them into a multimedia library to be used by multimedia applications. You can also set up thumbnail quality for photos and videos, manage conversion settings for videos, and create indexed folders to be accssed by DMA devices and DSM's multimedia applications.

File indexing allows you to index folders and the contained files on your DiskStation by filename, and you can further index the file content/metadata by file type. This detailed indexing function will quicken your search speed for more accurate matches.
