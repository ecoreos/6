General

Generic network options such as default gateway, DNS server, proxy and IPv6 tunneling are available at Control Panel > Network > General.

Renaming DiskStation Device

To rename your DiskStation device, enter the new server name and click Apply.

The server name is case insensitive and can be in the range of 1 to 15 characters, consisting of letters, numbers, underscores and minus signs. The first character of the name must be a letter.

Specifying Default Gateway

Simply enter the IP address for the default gateway and click Apply to save the setting.

Specifying DNS Server IP Address

You can choose to manually configure the IP address of your DNS server. In this case, the IP address of your preferred DNS server must be entered. An alternate DNS server IP address can also be entered, providing backup if the preferred DNS server experiences problems. Once the DNS server IP addresses have been configured as desired, please click Apply to save your settings.

Note:

The DNS server and Default gateway fields will be set automatically and cannot be changed when all your network interfaces are configured automatically or when you are using PPPoE connection.

Connecting via Proxy Server

To allow your DiskStation device to connect via a proxy server, tick Connect via a proxy server, enter the Address and Port, and then click Apply.

Note:

  • Click Advanced Settings to configure HTTP proxy, HTTPS proxy and authentication information.
  • By default, the HTTPS proxy settings are the same as HTTP proxy.
  • If Connect via a proxy server is ticked, DSM Update and Package Center along with some other packages will always connect to the Internet through proxy.
  • If Bypass proxy server for local addresses is ticked, please make sure the DNS server used by the DiskStation can resolve queries for devices in the local network.

Enabling IPv6 Tunneling

Until IPv6 completely supplants IPv4, IPv6 tunneling is needed to enable hosts with IPv6 addresses to reach IPv4 services. This technique provides a way to carry IPv6 traffic within existing IPv4 networks.
If you are not sure how to configure IPv6 or face any connection problems, please contact your Internet Service Provider.

To enable IPv6 Tunnel:

  1. Tick the Enable tunnel checkbox to enable tunnel support.
  2. In Server address, enter the hostname or the IP address of the tunnel service provider, for example: anonymous.freenet6.net.
  3. Select if you want to Connect anonymously or Connect with an existing account.

Note:

Your external IPv6 address may be automatically updated by the service provider. However, DiskStation device will not remove the old IPv6 address immediately.
