Win/Mac

Windows, Mac and Linux users can access your DiskStation from the network once you have enabled options at Control Panel > File Services > Win/Mac.

The following file sharing protocols are supported:

  • For Windows: SMB/CIFS (My Network Places), FTP, WebDAV
  • For Mac: SMB, FTP, AFP, WebDAV
  • For Linux: SMB, FTP, WebDAV
