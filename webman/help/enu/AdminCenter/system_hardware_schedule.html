Power Schedule*

DC Output*

At Control Panel > Hardware & Power > Power Schedule, you can set the system to start up or shut down automatically at a specified time.

At Control Panel > Hardware & Power > DC Output, you can set the system to turn on or off the DC output automatically at a specified time.

To add a new startup/shutdown schedule:

To add a new output on/output off schedule:

  1. Click Create button to open the Add Schedule dialog.
  2. Choose Startup or Shutdown.
  3. Set the Date and Time for startup or shutdown.
  4. Click OK.
  5. Click Save.
  1. Click Create button to open the Add Schedule dialog.
  2. Choose Output on or Output off.
  3. Set the Date and Time for the output-on/output-off schedule.
  4. Click OK.
  5. Click Save.

The new schedule will be added to the schedule list.

To view schedule summary:

Click Summary to browse the weekly schedule for startup/shutdown.

Click Summary to browse the weekly schedule for the output-on/output-off schedule.

Note:

  • You can specify up to 100 schedules for one DiskStation.
  • In the following cases, the system may fail to shut down automatically:
    • Storage Manager is performing tasks.
    • The system is encrypting share folders.
    • The system is performing Secure Erase on hard drives.
    • DSM is being upgraded.
    • The system is repairing RAID volumes.
    • Package Center is performing tasks.
    • The system is upgrading databases.
    • The system is performing data backup tasks.
  • To set up a startup schedule, please ensure that DiskStation will be supplied with continuous power. If DiskStation experiences any temporary interruption of power supply, the power schedule will become invalid.
  • You can specify up to 100 schedules for one DiskStation.
  • In the following cases, the system may fail to turn off the DC output automatically:
    • Storage Manager is performing tasks.
    • The system is encrypting share folders.
    • The system is performing Secure Erase on hard drives.
    • DSM is being upgraded.
    • Package Center is performing tasks.
    • The system is upgrading databases.
    • The system is performing data backup tasks.
