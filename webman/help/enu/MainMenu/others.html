Others

You can customize other personal account options at Personal > Others.

To customize other personal options:

  1. Choose any of the following:
    • Do not show confirmation dialog when leaving DSM: Skips the confirmation dialog when closing the browser window.
    • Resume DSM to my previous logout status when logging in: Remembers open applications or packages and restores them the next time you login or refresh the browser window.
    • Enable instant preview on taskbar: Displays a quick preview of an open application when you hover the mouse cursor over the application icon on the taskbar. Internet Explorer may not fully support this feature if it is not of the latest version.
    • Do not automatically launch DSM help at every login: The DSM help page will not be automatically opened when you log in to DSM.
    • Enable desktop keyboard shortcuts: You can use keyboard shortcuts to perform multiple types of desktop and DSM operations (e.g. switching between active windows, selecting/deselecting options, and so on). To view all the keyboard shortcuts, press Shift + ?.
    • Disable accessibility (no auditory cues available): Suspends the accessibility feature on DSM so that you will not hear auditory cues when the screen reader is on.
    • Enable browser desktop notifications: Shows notifications from your web browser on the DSM desktop. Browser desktop notifications are supported by Google Chrome and Safari only. Depending on the version of your web browser, the notification settings page may be different. Please go to the settings page of your browser to manage notification settings.
    • Eanble the Pilot View button on taskbar: Displays the Pilot View button to help you view all the open applications in the preview mode.
  2. Click OK.
