Notifications

You can enable rules to automatically notify you when specific, log-related events occur. For example, notifications can be triggered according to severity, the number logs per second, or when logs contain pre-defined keywords.

Note:

Before enabling this feature, please go to Control Panel > Notification > Email and set up email notifications. Log Center notifications are sent to the address specified there.

To enable a notification rules:

  1. Go to Log Center > Notifications.
  2. Tick any of the boxes next to the rule you would like to enable.
  3. Specify the rule criteria.
  4. Click Apply to save settings.
