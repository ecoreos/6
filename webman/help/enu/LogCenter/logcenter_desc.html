Log Center

Log Center is a centralized log management application. It includes comprehensive tools to accomplish the following tasks easily and efficiently. Please note that to get the advanced functions, you have to go to Package Center and install the Log Center package.

Basic Functions

  • Search for and filter local logs
  • Monitor log volume according to time
  • Send notifications to administrator when specified events occur

Advanced Functions

  • Send and receive logs from other network devices via syslog logging standard
  • Monitor log volume according to device
  • Specify log archival destination and create rules to automatically trigger log archival
  • Search for and filter logs received from other network devices
