Sending Files as Email Attachments

You can directly send files as email attachments in File Station by right-clicking selected files and then selecting Send as email attachments. An email window will appear with a few useful features:

  • To: If you have synced any contacts for your email account, a contacts list will appear, displaying synced contacts associated with your email account.
  • From: You can select the default email account or another added email account. The contacts list will automatically switch to the selected email account.
  • Attachments: The selected attachments appear at the bottom of the email. You can add more attachments using the attachments button.

Note:

  • Only files can be added as attachments.
  • You can attach up to 10 files in a single email.
  • The maximum total file size of all attachments in a single email will depend on your email service. If you are not sure, we recommend limiting the total file size of all attachments to 25MB.
