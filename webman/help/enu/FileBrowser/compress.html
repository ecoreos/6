Compress Files or Folders

  1. Choose files or folders from DiskStation in the main area.
  2. Choose Compress to “(file name)”.zip from the Action drop-down menu to start compressing.
  3. If you can't see the zip file when the compression is finished, choose Refresh from the Action drop-down menu to see the file.
  4. If you want to rename, encrypt the archive, or archive it using different compression levels or modes, choose Add to Archive... instead.
    Note:
  1. You cannot compress files or folders on your local computer using File Station.
  2. Supported archive formats: .zip, .7z.
  3. When you choose .zip as the Archive format at Add to Archive..., you can select a desired codepage for the .zip file.
