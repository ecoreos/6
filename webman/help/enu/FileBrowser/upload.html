Upload Files or Folders

Use any of the following methods to upload files or folders from your computer to DiskStation.

To have full access to the uploading functions shown below, install the latest Java Runtime Environment (JRE) and Flash Player first.

Method 1:

  1. From the upper part of the left panel, select a destination folder on DiskStation.
  2. Click Upload, choose Upload - Skip or Upload - Overwrite and select files or folders from your local computer.

Method 2:

  1. Go to Settings, tick Browse files from local computer with File Station and refresh the web page to view folders of your local computer on the lower part of the left panel.

    2. Note:

    The option Browse files from local computer with File Station takes effect only after you installed and enabled JRE on your computer.

  2. From the lower part of the left panel, select a folder on your local computer. The main section now displays the contents of the selected folder.
  3. Select the files or folders you want in the main section, and then click Upload.
  4. Select a destination folder on DiskStation in the window that appears.
  5. At the top-right corner of the window, select Overwrite if you want to overwrite files or folders with the same name in the destination folder. Otherwise, select Skip.
  6. Click OK to start uploading.

Method 3:

  1. Go to Settings, tick Browse files from local computer with File Station and refresh the web page to view folders of your local computer on the lower part of the left panel.
  2. On the left panel, drag a folder from your computer (the lower part) to any folder on the DiskStation (the upper part).
  3. In the pop-up menu that appears, choose whether you want to overwrite or skip files/folders with the same name in the destination folder.

Method 4:

  1. From the upper part of the left panel, select a destination folder on DiskStation.
  2. Select the files on your computer's desktop or file explorer (for example, Windows Explorer or Mac Finder), and then drag and drop them into File Station in the web browser.
  3. In the dialog that appears, choose whether you want to overwrite or skip files with the same name in the destination folder.

    4. Note:

    • Method 4 is supported on specific browsers only: Chrome, Firefox, Safari 5.1 or onwards. However, it is not supported for Mac when Browse files from local computer with File Station is ticked.
    • You can also drag and drop folders or empty files, but only when Browse files from local computer with File Station is ticked for browsers listed above. Otherwise, you cannot upload folders and empty files using Method 4.
    • If you are using Chrome for Linux and logged in to DSM via the HTTPS protocol, you need to import a trusted certificate (at Control Panel > Security > Certificate) to be able to upload using Method 4.

Note:

  1. You can add up to 1000 files to your upload queue.
  2. You can upload files with size over 4GB if you are using Google Chrome, Firefox, Safari 5.1 or later, or when Browse files from local computer with File Station is ticked. Otherwise, you can only upload up to 2GB of files at a time.
  3. In a non-encrypted shared folder, file/folder name should be within 255 characters (up to about 80 characters for Non-Latin languages); in an encrypted shared folder, it should be within 143 characters (up to about 47 characters for Non-Latin languages).
  4. File/folder name should exclude the following characters: \ / : * ? " > |.
  5. File/folder name cannot start with “._”.
  6. You can upload files that are 0KB in size only when you have installed JRE.

Troubleshooting

If you cannot browse local items stored on your Mac using Safari properly, please do the following:

  1. On your Mac, open Safari and choose Preferences from the Safari menu.
  2. Click Manage Website Settings in the Security tab.
  3. Under Java, choose your DSM "website."
  4. Select Run in Unsafe Mode.
