Rename a File or Folder

Use any of the following methods to rename files or sub-folders on DiskStation.

Method 1:

  1. Select a file or a folder you want to rename.
  2. Choose Rename from the Action menu.
  3. Enter a new name in the window that appears and click OK.

Method 2:

  1. Right-click a file or a folder and choose Rename.
  2. Enter a new name in the window that appears and click OK.
