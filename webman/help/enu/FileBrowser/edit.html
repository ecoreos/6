Edit Photos

At File Station, you can edit photos with Aviary Editor, a web-based external photo editor.

To edit a photo with Aviary Editor:

  1. Browse to the folder where the photo is located.
  2. Right-click the photo and choose Edit in Aviary.
  3. Aviary Editor will be opened in a new browser window. You can view your photo and edit it with the provided tools.
  4. After you finish photo-editing, click the save button on the top toolbar to save all the changes to your photo.

Note:

  • With Aviary Editor you can edit photos of the following formats: .jpg, .jpeg, .jpe, .png
  • The maximal filesize allowed by Aviary Editor is 5 MB.
  • Aviary Editor requires the network settings of your DiskStation to be properly configurated. Check the settings below for troubleshooting:
    • Make sure your DiskStation can be accessed via QuickConnect (at Control Panel > QuickConnect) or can retrieve an external IP address (at Control Panel > External Access > DDNS, or at Control Panel > Network > Network Interface > PPPoE).
    • If your DiskStation is located behind a router, make sure proper port forwarding rules are specified (at Control Panel > External Access > Router Configuration).
    • If you still have trouble opening the Aviary Editor, you may need to specify the router port for your DiskStation (at Control Panel > External Access > Advanced).
