Advanced

You can view and change advanced settings for Security Advisor.

Security Baseline

You can choose what security rules Security Advisor will check by selecting a Security Baseline. The number and types of rules will vary for each Security Baseline. Make your choice depending on your needs. Are you using your DiskStation for work and business or home and personal use? You can choose from the following options:

  • For home and personal use: Security Advisor will perform essential checks to protect your DiskStation against common cyber threats.
  • For work and business: Security Advisor will perform comprehensive checks to meet security compliance requirements for organizations.
  • Custom: Select which security rules you want Security Advisor to check.

Scan Schedule

You can define a regular scanning schedule by clicking Enable regular scan schedule and choosing what time you want Security Advisor to perform a scan. Security Advisor will check rules according to the Security Baseline you have chosen.
