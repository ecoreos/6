Results

You can view detailed information of your completed scan on a report-by-report basis. This page is laid out as follows:

  • Severity: The overall risk level that Security Advisor assesses for each security rule. There are four levels: Normal, Warning, Alarm, and Critical.
  • Description: Settings that Security Advisor looks for in a particular rule that may lead to vulnerabilities for your DiskStation if no action is taken.
  • Last update: The last time Security Advisor checked this security rule.
  • Status: The current status of each security rule. Security Advisor reports the result for each security rule as Passed or Failed. At times, statuses of Error, Unchecked, Running, and Skipped may also be shown. You can filter the results at the top of the page.

To check the report details of any security rule:

Select a rule and click View. In addition to Severity and Description, each report contains the following details:

  • Details: A brief explanation of any weaknesses, what Security Advisor recommends to handle them, and benefits you may gain from addressing the weakness.
  • Recommended Action: Detailed instructions on how to address the reported security weakness.
  • Link: For any failed security rules, clicking on Link will take you to the location in DSM to perform any recommended actions.
  • Fix: When malware is detected, clicking on Fix will instruct Security Advisor to remove the malware.

To scan a specific rule again:

Select the items you want to scan again, and click Scan.
