Network

This page contains the diagram and status table of the network interfaces.

Interface Diagram

Displays the selected item in the table, and different colors are used to present 1Gbps and 10Gbps network interfaces.

Interface Status Table

Displays the interface status on both servers, and whether the interface is monitored. This table shows the following information:

  • Network Interface: Displays the physical network interfaces on the servers.
  • Activer Server: Displays the interface status and connection speed on the active server.
  • Passive Server/Unknown Server: Displays the interface status and connection speed on the passive server. This column will show the name corresponding to the server's status.
  • Bond: Displays the link aggregation settings.
  • Status: Displays the network status if the interface is monitored. It may show the following statuses:
    • Warning: Connection fails on the passive server, or the interface has different connection speeds on the two servers;
    • Error: Connection fails on the active server;
    • Normal: No any failure.
  • Monitor: Displays if the interface is monitored. Go to Monitor > Network Interface to tick/untick Monitor checkbox.
