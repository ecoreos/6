Create a Disk Group

This article explains how to create a disk group on your DiskStation.

Before You Start:

  • Make sure there is no important data on the hard drives. All existing data will be deleted when creating a new disk group.
  • Go to the HDD/SSD page and make sure the status of each hard drive is Normal or Not initialized.

To create a disk group:

  1. Open Storage Manager.
  2. Go to Disk Group and click Create.
  3. When the disk group creation wizard appears, choose which hard drives shall be used to create the disk group. Then follow the steps of the wizard to finish.
