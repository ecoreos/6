Create a Volume

This article explains how to create a volume on your DiskStation. When creating a new volume, you can choose one of the following methods:

  • Quick: This is the best option for beginners. Choose this method if you want to automatically optimize storage capacity and performance, without worrying about technical aspects, such as RAID levels.
  • Custom: For advanced users who want to control the finer aspects of volume creation, such as RAID types and capacity allocation. This option supports different RAID types, allows you to create single or multiple volumes, and provides options to allocate specific amounts of capacity for each volume (only when creating multiple volumes on RAID).

Single or Multiple Volumes on RAID?

When creating a volume with custom mode, you can choose one of the following options:

  • Single Volume on RAID
    • Creates a single volume using all of the available capacity of the selected hard drives.
    • Provides better speed and performance.
    • Supports different RAID types.
  • Multiple Volumes on RAID
    • Allows you to create multiple volumes on a disk group.
    • Supports allocating specific amounts of capacity to each volume.
    • Provides greater storage management flexibility.
    • Supports different RAID types.

Before You Start:

  • Before starting, make sure there is no important data on hard drives. All existing data will be deleted during the creation process.
  • Go to the HDD/SSD page and make sure the status of each hard drive is Normal or Not initialized.

Note:

  • The minimum size for a Btrfs volume is 10GB.
  • Btrfs volumes will reserve 4% of space for metadata.

To create a volume:

  1. Open Storage Manager.
  2. Go to Volume and click Create.
  3. You'll see the volume creation wizard. Select Quick or Custom depending on your needs. Then follow the steps of the wizard to finish creating the volume.
