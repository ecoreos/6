Accessibility

With the built-in accessibility feature, DiskStation Manager provides auditory cues to guide users with visual impairments in navigating and managing the DSM services/settings.

DSM Accessibility Setup

To maximize accessibility, we suggest you set up the operation environment as below:

  • Install the NVDA screen reader (available only to the Windows environment) on your local computer.
  • Install the latest Mozilla Firefox web browser.
  • Use headsets or speakers and adjust the volume to suit your needs.
  • Use NVDA in the focus mode (Press the NVDA key + Space for mode switch). The default NVDA key is Insert.
  • Navigate DSM with the following hotkeys:
    Hotkey Description
    Tab Switch to other user interface sections
    Arrow keys Switch to items within the same user interface section
    Enter Open applications/windows
    Esc Close applications/windows

The accessibility feature is now at work in the following areas of DSM:

  • DSM login page, desktop, and taskbar
  • Main Menu
  • Control Panel
  • File Station

Note:

For details about NVDA operation, please refer to its user guide.

DSM keyboard shortcut guides

Along with the accessibility feature, DSM offers a rich set of keyboard shortcuts to implement smooth, streamlined management. Press Shift + ? to find the shortcuts:

  • Desktop Keyboard Shortcuts: Hotkeys for window operations and more
  • Component Keyboard Shortcuts: Hotkeys for controling user interface components (e.g. lists, buttons, and search fields)

To suspend desktop-related shortcuts, cancel the Enable desktop keyboard shortcuts option.
