Backup & Restore

You will learn how to use Backup & Restore in the following ways:

  • Backing up data to the local/remote Synology NAS devices
  • Backing up data to a remote rsync server
  • Backing up data to public clouds (e.g. Amazon S3 and Microsoft Azure)
  • Backing up iSCSI LUN* to external storage devices and to another Synology NAS device
  • Restoring backup files
