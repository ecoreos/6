Dosyaları veya Klasörleri Sıkıştır

  1. Ana alanda DiskStation sunucusundan dosya veya klasörler seçin.
  2. Sıkıştırmayı başlatmak için İşlem açılan menüsünden Sıkıştır “(dosya adı)”.zip seçimini yapın.
  3. Sıkıştırma tamamlandığında zip dosyasını göremezseniz, dosyayı görmek için İşlem açılır menüsünden Yenile'yi seçin.
  4. Arşivi yeniden adlandırmak, şifrelemek veya farklı sıkıştırma seviyeleri veya modları kullanarak arşivlemek istiyorsanız bunun yerine Arşive ekle... seçimini yapın.
    Not:
  1. Yerel bilgisayarınız üzerindeki dosya veya klasörleri File Station'ı kullanarak sıkıştıramazsınız.
  2. Desteklenen arşiv biçimleri: .zip, .7z.
  3. Arşive Ekle... öğesinde Arşiv biçimi olarak .zip seçtiğinizde, .zip dosyası için istenilen kod sayfasını seçebilirsiniz.
