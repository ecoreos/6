Dosyaları veya Klasörleri Karşıya Yükleyin

Bilgisayarınızdan DiskStation sunucusuna dosya veya klasör yüklemek için aşağıdaki yöntemlerden birini kullanın.

Aşağıda gösterilen karşıya yükleme işlevlerine tam erişime sahip olmak için, öncelikle en son Java Runtime Environment (JRE) ve Flash Player sürümlerini yükleyin.

Yöntem 1:

  1. Sol panelin üst kısmından DiskStation üzerinde bir hedef klasör seçin.
  2. Karşıya Yükle'ye tıklayın, Karşıya Yükle - Atla ya da Karşıya Yükle - Üzerine yaz'ı, ardından da yerel bilgisayarınızdan dosyaları veya klasörleri seçin.

Yöntem 2:

  1. Yerel bilgisayarınızda sol panelin alt kısmındaki dosyaları görmek içinAyarlar'a gidin,File Station ile yerel bilgisayar dosyalarına Gözat'ı tıklayın ve web sayfasını yenileyin.

    2. Not:

    File Station ile yerel bilgisayar dosyalarına gözat seçeneği yalnızca bilgisayarınızda JRE yüklenip etkinleştirildikten sonra etkili olacaktır.

  2. Sol panelin alt kısmından yerel bilgisayarınız üzerinde bir klasör seçin. Şimdi ana bölüm seçilen klasörün içeriğini gösterir.
  3. Ana bölümde istediğiniz dosya veya klasörleri seçin ve ardından Karşıya yükle seçimini yapın.
  4. Beliren pencerede DiskStation üzerindeki bir hedef klasörü seçin.
  5. Hedef klasördeki aynı isimli dosya veya klasörlerin üzerine yazılmasını istiyorsanız pencerenin sağ üst köşesinde Üzerine yaz seçimini yapın. Aksi halde Atla'yı seçin.
  6. Yüklemeyi başlatmak için Tamam üzerine tıklayın.

Yöntem 3:

  1. Yerel bilgisayarınızda sol panelin alt kısmındaki dosyaları görmek içinAyarlar'a gidin,File Station ile yerel bilgisayar dosyalarına Gözat'ı tıklayın ve web sayfasını yenileyin.
  2. Sol panelde, bilgisayarınızdan bir klasörü (alt kısım) DiskStation (üst kısım) üzerindeki bir klasöre sürükleyin.
  3. Açılan menüde, hedef klasörde aynı ada sahip dosyaların/klasörlerin üzerine yazmayı mı yoksa bu dosyaları atlamayı mı istediğinizi seçin.

Yöntem 4:

  1. Sol panelin üst kısmından DiskStation üzerinde bir hedef klasör seçin.
  2. Bilgisayarınızın masaüstü veya dosya gezgini (örneğin Windows Explorer veya Mac Finder) üzerinde dosyaları seçin ve ardından bunları sürükleyip web tarayıcıda File Station'a bırakın.
  3. Açılan iletişim kutusunda, hedef klasörde aynı ada sahip dosyaların üzerine yazmayı mı yoksa bu dosyaları atlamayı mı istediğinizi seçin.

    4. Not:

    • Yöntem 4 yalnız belirli tarayıcılarda desteklenir: Chrome, Firefox, Safari 5.1 veya üstü. Ancak, File Station ile yerel bilgisayar dosyalarına gözat seçeneği işaretliyken Mac için desteklenmez.
    • Klasörleri ya da boş dosyaları da yalnızca yukarıda listelenen tarayıcılar için File Station ile yerel bilgisayar dosyalarına gözat seçeneği işaretliyken sürükleyip bırakabilirsiniz. Aksi halde, klasörleri ve boş dosyaları Yöntem 4'ü kullanarak karşıya yükleyemezsiniz.
    • Linux için Chrome kullanıyorsanız ve DSM'de HTTPS protokolü aracılığıyla oturum açmışsanız, Yöntem 4 kullanarak yükleme yapabilmek için güvenilir sertifika (Denetim Masası > Güvenlik > Sertifika öğesinde) almanız gerekir.

Not:

  1. Karşıya yükleme kuyruğunuza 1000 dosyaya kadar ekleyebilirsiniz.
  2. Boyutu 4 GB üzerinde olan dosyaları, Google Chrome, Firefox, Safari 5.1 ya da üstünü kullanıyorsanız veya File Station ile yerel bilgisayar dosyalarına gözat seçeneği işaretli olduğunda karşıya yükleyebilirsiniz. Aksi halde, bir seferde yalnızca 2 GB boyutuna kadar dosya yükleyebilirsiniz.
  3. Şifrelenmemiş bir paylaşımlı klasörde, dosya/klasör adı 255 karakteri (Latin alfabesini kullanmayan diller için yaklaşık 80 karaktere kadar); şifrelenmiş paylaşımlı klasörde ise en fazla 143 karakter (Latin alfabesini kullanmayan diller için yaklaşık 47 karaktere kadar) geçmemelidir.
  4. Dosya/klasör adı aşağıdaki karakterleri içermemelidir: \ / : * ? " > |.
  5. Dosya/klasör adı _ ile başlayamaz.
  6. Boyutu 0 KB olan dosyaları yalnızca JRE yüklediğinizde karşıya yükleyebilirsiniz.

Sorun Giderme

Safari kullanarak Mac'inizdeki yerel öğelere göz atamıyorsanız, lütfen aşağıdakileri yapın:

  1. Mac'inizde, Safari'yi açın ve Safari menüsünden Tercihler öğesini seçin.
  2. Güvenlik sekmesinde Web Sitesi Ayarlarını Yönet üzerine tıklayın.
  3. Java altında, DSM "web sitenizi" seçin.
  4. Güvensiz Modda Çalıştır öğesini seçin.
