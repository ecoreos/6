File Station

File Station kullanımı kolay bir dosya yönetim aracıdır ve İnternet üzerindeki DSM kullanıcılarının DiskStation DiskStation klasörlerine web tarayıcıları ile erişmelerine izin verir. İki veya daha fazla File Station başlatarak, tüm DiskStation verilerinizi farklı File Station'lar arasında sürükleyip bırakarak yönetebilirsiniz.
