File Station Gereksinimleri

Tüm özelliklere sahip File Station kullanımı için en yeni web tarayıcısı, Java ve Flash Player gerekecektir.

  • Tarayıcı: Internet Explorer 8 veya üstü, Firefox 3.6 veya üstü, Safari 5.0 veya üstü, Chrome; AJAX ve JavaScript etkinleştirilmiş olarak
  • Java (yerel bilgisayar üzerinde klasörlere gözatmak için): Java Runtime Environment (JRE) 5 veya üstü
  • Flash: Flash Player 9.0.28 veya üstü
