Sadece SSL/TLS bağlantıya izin ver

Bu işlev etkinleştirildiğinde, sunucu FTP istemcilerden sadece SSL veya TLS bağlantı taleplerini kabul edecektir.

IP başına bağlantıyı sınırla

Bu işlev etkinleştirildiğinde, sunucu IP başına maksimum bağlantıyı sınırlandırır. Etkin bant genişliği yönetimi sağlamak amacıyla bu işlev ve FTP bant genişliği kısıtlaması etkinleştirilmelidir.

FTP bant genişliği kısıtlamasını etkinleştir
