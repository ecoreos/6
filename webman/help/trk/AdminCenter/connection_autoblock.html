Otomatik Engelleme

Otomatik engelleme yetkisiz oturum açmayı önlemenize olanak sağlar. Hizmet etkinleştirildikten sonra, çok sayıda başarısız oturum açma girişiminde bulunan bir IP adresi engellenir. SSH, Telnet, rsync, Network Backup, Shared Folder Sync, FTP, WebDAV, Synology mobile apps, File Station veya DSM aracılığıyla yapılan başarısız oturum açma girişimlerinin tümü eklenecektir. Hiçbir zaman engellenmeyecek IP adreslerini içeren bir İzin Listesine de sahip olabilirsiniz.

Otomatik engellemeyi kurmak için:

  1. Genel sekmesine gidip Otomatik engellemeyi etkinleştir'i işaretleyin.
  2. Belirlenen dakika sayısı içinde yapılan bir dizi başarısız oturum açma girişimlerinden sonra IP adresini engellemek için Oturum açma girişimleri sayısı ile Süre içinde (dakika) girişlerini yapın.
  3. Engelleme süre dolumunu etkinleştir'i işaretleyin ve belirlenen gün sayısı sonunda engellenen IP adresini kaldırmak için bir sayı girin. Bu seçenek seçilmez veya Engellemeyi sonra kaldır (gün) 0 olarak ayarlanmazsa engellenen IP adresleri listeden manuel olarak çıkarılmadıkça sonsuza dek engelleme listesinde kalır.
  4. Uygula'ya tıklayın.

Engellenen IP adreslerini eklemek için:

Engelleme Listesi sekmesine tıklayın ve Oluştur açılan menüsünden aşağıdakilerden birini seçin:

  • IP adresi ekle

    • Bir IP adresi girin ve sona erme zamanını ayarlayın.

  • IP adres listesini al

    • Bir kerede birtakım IP adreslerini eklemek ve sona erme zamanı ayarlamak için, IP adreslerinin listelendiği bir metin dosyasını içe aktarın. Var olan IP adreslerinin üzerine yazmak isterseniz uygun onay kutusunu işaretleyin. Seçenek işaretlenmezse, aynı IP adresleri atlanacaktır.

    Bir dosyayı içe aktarmak için, dosyanın aşağıdaki kriterlere uyması gerekir:

    • Dosya düz metin dosyası olmalıdır.
    • Dosyanın her bir satırı yalnızca bir IP adresi içermelidir.
    • Yorum satırları # (diyez işareti) ile başlar.

      • Seçtiğiniz dosya doğru biçime sahipse, önizleme alanında tüm IP adreslerini göreceksiniz. İçe aktarmak için Tamam'a tıklayın.

Engellenen IP adreslerini kaldırmak için:

  1. Engellenmiş IP adreslerinin listesini görmek için Engelleme Listesi sekmesine tıklayın.
  2. Listeden kaldırmak istediğiniz IP adreslerini seçip Kaldır'a tıklayın.
