Gelişmiş

DSM gözatma oturumları için aşağıdaki gelişmiş ayarları Denetim Masası > Güvenlik > Gelişmiş öğesinde değiştirebilirsiniz:

HTTP Sıkıştırması

DiskStation için HTTP Sıkıştırması'nı etkinleştirebilirsiniz. Ağ trafiğinin bant genişliğini düşürerek, web sayfasının yüklenme hızını artırabilir.

HTTP Sıkıştırmasını Etkinleştirmek / Devre Dışı Bırakmak için：

  • HTTP Sıkıştır seçeneğini Denetim Masası > Güvenlik > Gelişmiş öğesinde işaretleyin / işaretini kaldırın.

TLS / SSL Profili

HTTPS şifreli bağlantılarının güvenlik seviyesini seçebilirsiniz. 3 seviyeyi şu şekilde sunar:

  • Modern uyumluluk - Sadece en yeni tarayıcı ve mobil cihazlarla uyumludur.
  • Ara uyumluluk - En üst ve en alt güvenlik seviyesi arasında. Bu sistem varsayılan değeridir.
  • Eski geriye dönük uyumluluk - Güvenli değildir, ama tarayıcı ve mobil cihazların çoğunu destekler.
