Güvenlik

DSM gözatma oturumları için aşağıdaki güvenlik ayarlarını Denetim Masası > Güvenlik > Güvenlik öğesinde değiştirebilirsiniz:

  • Oturum kapatma zamanlayıcı (dakika): Kullanıcılar burada belirtilen zaman periyodu süresince etkin olmadıklarında DSM oturumları otomatik olarak kapatılır. 1 ila 65535 arasında bir değer girin.
  • IP kontrolünü atlayarak tarayıcı uyumluluğunu artır: DiskStation sunucusuna bir HTTP proxy'si üzerinden erişiyor ve rastgele oturum kapatma işlemleriyle karşılaşıyorsanız IP kontrolünü atlamak üzere bu seçeneği etkinleştirebilirsiniz.
  • Çapraz site talebi hırsızlık saldırılarına karşı korumayı arttır: Bu seçenek sistemin siteler arası komut dosyası saldırılarına karşı korunaklılığını arttırır. Bu seçenek DSM'de bir sonraki oturum açışınızda etkin hale gelir.
  • HTTP İçerik Güvenliği Politikası (CSP) başlığı ile güvenliği artırın: Bu seçenek sadece güvenilir kaynaklardan gelen verilere izin vererek ve hat içi komut dosyası çalışmasını kısıtlayarak siteler arası komut dosyası (XSS) saldırılarına karşı sistemin güvenliğini artırır.
  • DSM'nin iFrame ile bütünleşmesine engel ol: Diğer web sitelerinin iFrame ile DSM'yi diğer web sayfalarına gömmesine engel olmak için bu seçeneği etkinleştirebilir ve böylece zararlı web sitelerinden belirli saldırı tiplerini önleyebilirsiniz. Belirli web sitelerine İzin verilen web siteleri düğmesine tıklayarak iFrame ile DSM'yi gömmelerine izin verilebilir.
