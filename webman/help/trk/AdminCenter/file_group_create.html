Grup Oluştur

  1. Denetim Masası > Grup öğesine gidin.
  2. Oluştur'a tıklayın. Bu işlem Kullanıcı Oluşturma Sihirbazı'nı başlatır.
  3. Grup adını ve açıklamasını girin. (Daha fazla bilgi için aşağıdaki bölüme bakın). İleri'ye tıklayın.
  4. Grubun oluşturulmasını tamamlamak için sihirbazın kalan adımlarını izleyin.
  5. Bitirmek için Uygula'ya tıklayın.

Adlandırma kuralları:

  • Grup ismi büyük/küçük harfe duyarlıdır ve aşağıdaki semboller dışında 1 ila 32 Unicode karakter olmalıdır:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • İlk karakter eksi işareti veya boşluk olamaz ve en son karakter de boşluk olamaz.
  • Grup açıklaması büyük/küçük harfe duyarlıdır ve en fazla 64 tane görüntülenebilir Unicode karakterle sınırlıdır.
