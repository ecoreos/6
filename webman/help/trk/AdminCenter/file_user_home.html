Kullanıcı Ana Sayfa

Her kullanıcıya kişisel bir ana sayfa klasörü oluşturmak için kullanıcı ana sayfa etkinleştirilmelidir, guest bunun dışındadır. Tüm kullanıcılar kendi ana sayfa klasörlerine CIFS, AFP, FTP veya File Station yoluyla erişebilirler.

Admin homes varsayılan paylaşımlı klasöründeki tüm kişisel klasörlere erişebilir. Home klasörünün adı kullanıcı hesabı ile aynıdır.

Kullanıcı ana sayfa hizmetini etkinleştirmek için:

  1. Kullanıcı Ana Sayfa üzerine tıklayın.
  2. Kullanıcı ana sayfa hizmetini etkinleştir kutusunu işaretleyin.
  3. Synology DiskStation üzerinde birden fazla birim varsa, homes klasörünün depolanacağı konumu seçin.
  4. TAMAM üzerine tıklayın.

Not: Yerel kullanıcı ana sayfa hizmeti devre dışı bırakıldığında, aynı zamanda etki alanı kullanıcı ana sayfa hizmeti de devre dışı bırakılacaktır.

Kullanıcı ana sayfa hizmetini devre dışı bırakmak için:

  1. Kullanıcı Ana Sayfa üzerine tıklayın.
  2. Kullanıcı ana sayfa hizmetini etkinleştir'in işaretini kaldırın.

Kullanıcı ana sayfa servisi engellendikten sonra, homes paylaşımlı klasörü admin erişimi için ayrılacaktır. Kullanıcı ana sayfa servisi yeniden etkinleştirildiğinde kullanıcılar home klasörlerine tekrar erişebileceklerdir.

    Not:
  1. homes paylaşımlı klasörünü silmek için, önce kullanıcı ana sayfa servisi devre dışı bırakılmalıdır.
  2. Yerel kullanıcı ana sayfa hizmeti devre dışı bırakılırsa etki alanı kullanıcı ana sayfa hizmeti devre dışı olacaktır.
