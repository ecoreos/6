HTTP Hizmeti

Varsayılan bağlantı noktası numarası 80'e ilaveten Photo Station ve Web Station kullanımı için başka bir bağlantı noktası ekleyebilirsiniz.

HTTP erişimi için ilave port numarası eklemek için:

  1. İlave HTTP bağlantı noktası ekle'yi işaretleyin.
  2. Alana bağlantı noktası numarasını girin. HTTP erişimi için 1'den 65535'e kadar ilave bağlantı noktası numarası ekleyebilirsiniz.

Aşağıdaki bağlantı noktaları hariç tutulmalıdır:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule varsayılan bağlantı noktaları: 4662 (TCP), 4672(UDP) ve FTP varsayılan bağlantı noktası aralığı: gerçek aralık farklı modellere bağlı olarak değişiklik gösterebilir.

    Not:
  1. HTTP erişimi için port numaraları FTP/BT servisi için olan port numarası ile aynı olamaz.
  2. HTTP portu ve HTTPS portu birbiri ile karışamaz.
