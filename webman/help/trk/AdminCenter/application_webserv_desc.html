Web Hizmetleri

DiskStation ürününüz, zengin web barındırma özellikleri ile gelir. Web Station ile VHost desteğini ve ilave HTTP/HTTPS ayarlarını kullanarak kendi web sitenizi kolayca barındırabilir ve yayınlayabilirsiniz. Alternatif olarak yerleşik PHP desteğini uygulayabilir ve kişisel kullanımınız ya da işletmeniz için dinamik, veritabanına dayalı web siteleri oluşturabilirsiniz.
