Win/Mac/NFS

Windows, Mac ve Linux kullanıcıları, Denetim Masası > Dosya Hizmetleri > Win/Mac/NFS bölümündeki seçenekleri etkinleştirdikten sonra DiskStation cihazınıza ağ üzerinden erişebilirler.

Aşağıdaki dosya paylaşım protokolleri desteklenmektedir:

  • Windows için: SMB/CIFS (Ağ Bağlantılarım), FTP, WebDAV
  • Mac için: SMB, FTP, AFP, WebDAV
  • Linux için: SMB, FTP, NFS, WebDAV
