Adatok biztonsági mentése PC/Mac számítógépen a Cloud Station alkalmazás használatával

Áttekintés

A Cloud Station Server nem csupán egy összetett, több platformon is használható szinkronizáló szoftver, a felhasználók akár személyi számítógépükön tárolt adataikról is biztonsági mentést készíthetnek vele. A Cloud Station Backup egy valós idejű biztonsági mentést kínáló megoldás, mely sávszélesség-takarékos, magasabb fokú védelmet nyújt, és egy fájlról akár 32 előzményverziót is képes megőrizni.

A Cloud Station Server beállítása a Synology NAS eszközén

  1. Nyissa meg a Csomagkezelési központot, keresse meg a Cloud Station Server alkalmazást, majd kattintson a Telepítés parancsra.
  2. Indítsa el a Cloud Station Server alkalmazást.
  3. Lépjen az Áttekintés menüpontra, majd kattintson a Cloud Station Backup szakaszban található Letöltés most lehetőségre. (A Cloud Station Backup a Synology Letöltőközpontjából is letölthető.)
  4. Indítsa el a telepítőt a számítógépén, majd kövesse az utasításokat a Cloud Station Backup telepítéséhez.
  5. Telepítés után futtassa a Cloud Station Backup alkalmazást a számítógépén. Kattintson az Indítás most gombra.
  6. Adja meg a Cloud Station Server alkalmazást futtató Synology NAS címét (vagy QuickConnect ID azonosítóját), felhasználónevét és jelszavát. (A jobb oldalon található keresés ikonra is rákattinthat, ebben az esetben a Cloud Station Backup automatikusan megkeresi a helyi hálózaton elérhető többi Synology NAS szervert.) Kattintson a Tovább gombra.
  7. Válassza ki a biztonsági mentés forrását, majd törölje az almappákat, melyeket nem kíván szinkronizálni.
  8. A Kiválasztás gombra kattintva válasszon ki egy mappát a Synology NAS eszközről. A Cloud Station Backup létrehoz egy biztonsági mentési mappát a megadott célhelyen az Ön számítógépe nevével.
  9. Ha szinkronizálási szűrőket vagy szabályokat szeretne meghatározni, kattintson a Biztonsági mentési szabályok elemre.
  10. A biztonsági mentési beállítások áttekintéséhez kattintson a Tovább gombra, majd kattintson a Kész gombra.
  11. A biztonsági mentés előrehaladását a főalkalmazásban tekintheti meg.

Szinkronizált fájl korábbi verziójának letöltése vagy visszaállítása a Cloud Station Server alkalmazásból

  1. Lépjen a Cloud Station Server > Verzió előzmények menüpontra, majd keresse meg a visszaállítani kívánt fájlt vagy mappát.
  2. Válassza ki a lekérni kívánt fájlt, majd kattintson a Tevékenység gombra. Kattintson a Korábbi verziók tallózása elemre. A számítógépről véletlenül letörölt vagy eltávolított fájlokat a Visszaállítás elemre kattintva állíthatja vissza.
  3. Válassza ki a fájl letölteni kívánt verzióját, majd kattintson a Letöltés gombra. Amennyiben biztos benne, hogy a kiválasztott verziót kívánja használni, kattintson a Visszaállítás gombra, mellyel felülírja az aktuális verziót.

5. Adatok további védelme

Ha kíváncsi az adatok védelmére és biztonsági mentésére használható tovább módszerekre, kattintson ide és tekintse meg a DSM rendszerben elérhető többi biztonsági mentési szolgáltatás ismertetését.
