Csoport létrehozása

  1. Válassza a Vezérlőpult > Csoport opciót.
  2. Kattintson a Létrehozás gombra. Ez a művelet elindítja a Felhasználó létrehozás varázsló alkalmazást.
  3. Írja be a csoportnevet és leírást. (További információt lásd az alábbi részben). Kattintson a Tovább gombra.
  4. A csoport létrehozásának befejezéshez kövesse a varázsló további utasításait.
  5. A befejezéshez kattintson az Alkalmaz gombra.

Névadási szabályok:

  • A csoportnév nagybetűérzékeny, valamint a következő szimbólumok kizárásával, 1 és 32 közötti számú Unicode karakterből kell állnia:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Az első karakter nem lehet mínuszjel vagy szóköz, és az utolsó karakter nem lehet szóköz.
  • A csoport leírása nagybetűérzékeny, és maximum 64 megjeleníthető Unicode karakterből áll.
