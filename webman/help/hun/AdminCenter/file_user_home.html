Felhasználói kezdőlap

Ha minden egyes felhasználó számára személyes home mappát szeretne létrehozni, engedélyezze a felhasználói kezdőlapot, kivéve a guest fiókot. Minden felhasználó hozzáférhet saját home mappájához CIFS, AFP, FTP, vagy File Station segítségével.

Az admin az alapértelmezett homes megosztott mappában található összes személyes mappához hozzáférhet. A kezdőlap mappa neve megegyezik a felhasználói fiók nevével.

A felhasználói kezdőlap szolgáltatás engedélyezése:

  1. Kattintson a Felhasználói kezdőlap gombra.
  2. Jelölje be a következőt: Felhasználói kezdőlap szolgáltatás engedélyezése.
  3. Ha egynél több kötet van a Synology DiskStation-ön, válassza ki, hol szeretné tárolni a homes mappát.
  4. Kattintson az OK gombra.

Megjegyzés: Ha a helyi felhasználói kezdőlap szolgáltatás le van tiltva, a domain felhasználói kezdőlap szolgáltatás is párhuzamosan letiltásra kerül.

A felhasználói kezdőlap szolgáltatás tiltása:

  1. Kattintson a Felhasználói kezdőlap gombra.
  2. Törölje a következő kijelölését: Felhasználói kezdőlap szolgáltatás engedélyezése.

A user home szolgáltatás letiltása után a homes megosztott mappához csak az admin férhet hozzá. A felhasználók újra hozzáférhetnek kezdőlap mappáikhoz, amikor ismét engedélyezi a felhasználói kezdőlap szolgáltatást.

    Megjegyzés:
  1. A homes megosztott mappa törléséhez először le kell tiltani a felhasználói kezdőlap szolgáltatást.
  2. A domain felhasználói kezdőlap szolgáltatás letiltásra kerül, ha a helyi felhasználói kezdőlap szolgáltatást letiltják.
