PHP beállítások

Ha engedélyezte a Web Stationt, az alább leírt PHP beállításokat a Vezérlőpult > Webszolgáltatások > PHP beállítások menüpontra lépve konfigurálhatja.

PHP cache engedélyezése

Jelölje be a PHP cache engedélyezése lehetőséget, ha javítani szeretné a PHP teljesítményt. Bizonyos körülmények között azonban lehet, hogy ennek nem lesz hatása.

PHP hibaüzenet megjelenítése a weboldalon

Ha PHP hibaüzeneteket szeretne megjeleníteni a weboldalon, akkor jelölje be a Display_errors engedélyezése PHP hibaüzenet megjelenítéséhez lehetőséget.

PHP open_basedir testreszabása

  1. Jelölje be a PHP open_basedir testreszabása lehetőséget, ha szeretné testreszabni saját open_basedir opcióját.
  2. A szövegmezőbe írja be a tesreszabott elérési utat, majd kattintson az Alkalmaz gombra. Ha az alapértelmezett rendszerbeállítást szeretné használni, törölje a PHP open_basedir tesreszabása bejelöltségét, majd kattintson az Alkalmaz gombra.

Megjegyzés:

  • Használjon kettőspontot (:) az egyes útvonalak elválasztására. Például: /etc:/tmp:/usr/bin/php.
  • A homes megosztott mappa elérési útvonalai (pl. /volume1/homes) biztonsági okokból nem adhatók hozzá az open_basedir mappához.

PHP kiterjesztés kiválasztása

  1. A saját PHP kiterjesztés testreszabásához kattintson a PHP kiterjesztés kiválasztása lehetőségre.
  2. A felbukkanó PHP kiterjesztés lista párbeszédablakban válassza ki az engedélyezni kívánt kiterjesztést.
  3. Rákattinthat a párbeszédablak tetején található Alapállapot lehetőségre is, ha vissza szeretne térni az alapértelmezett beállításokhoz.
  4. A módosítások elmentéséhez kattintson az OK gombra.
