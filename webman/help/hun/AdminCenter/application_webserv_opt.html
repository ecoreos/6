HTTP-szolgáltatás

Az alapértelmezett 80-as portszámhoz másik portot is adhat a Photo Station és a Web Station használatához.

További port szám hozzáadása HTTP hozzáféréshez:

  1. Jelölje be a További HTTP port hozzáadása lehetőséget.
  2. A mezőben adja meg a portszámot. HTTP hozzáféréshez további port számokat adhat hozzá 1 - 65535 között.

A következő portok nem adhatók hozzá:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule alapértelmezett portok: 4662 (TCP), 4672 (UDP) és az FTP alapértelmezett porttartománya: a tartomány modellenként más és más lehet.

    Megjegyzés:
  1. A HTTP hozzáférés port számai nem lehetnek ugyanazok, mint az FTP/BT szolgáltatás port száma.
  2. A HTTP port és a HTTPS port nem ütközhetnek egymással.
