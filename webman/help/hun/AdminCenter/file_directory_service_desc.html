Könyvtárszolgáltatás

DiskStation eszköze Windows tartományi vagy LDAP-kliensként csatlakozhat egy meglévő könyvtárszolgáltatáshoz. Kezelheti a tartományi vagy LDAP-felhasználók megosztott mappákra és DSM-alkalmazásokra vonatkozó hozzáférési jogosultságait, illetve engedélyezheti számukra a home szolgáltatást. Könyvtárszolgáltatáshoz való csatlakozáshoz a(z) DiskStation eszközzel lépjen a Vezérlőpult > Könyvtárszolgáltatás menüpontra.
