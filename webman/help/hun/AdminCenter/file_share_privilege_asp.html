Speciális megosztott mappa engedélyek kezelése

A Vezérlőpult > Megosztott mappa > Szerkesztés > Speciális oldal az alábbi lehetőségeket tartalmazza egy megosztott mappa hozzáférési engedélyeinek további finomhangolásához.

Speciális megosztási engedélyek

A speciális megosztási engedélyek további ellenőrzési szintet biztosítanak a megosztott mappák hozzáférési engedélyeinek kezeléséhez. Ha engedélyezve van, a felhasználók és csoportok csak akkor tekinthetik meg vagy módosíthatják egy megosztott mappa tartalmát, ha a felhasználó vagy csoport rendelkezik mind speciális megosztási engedélyekkel, mind pedig Windows ACL engedélyekkel (lásd: Megosztott mappa > Szerkesztés > Engedélyek).

Speciális megosztási engedélyek engedélyezése:

  1. Jelölje be a Speciális megosztási engedélyek engedélyezése négyzetet.
  2. A megosztott mappa speciális megosztási engedélyeinek módosításához kattintson a Speciális megosztási engedélyek lehetőségre.

Megjegyzés:

  • A speciális megosztási engedélyek akkor kerülnek alkalmazásra, amikor egy felhasználó a következő fájlszolgáltatások segítségével fér hozzá egy megosztott mappához: Windows File Sharing, Apple File Sharing, File Station, FTP, és WebDAV.

Speciális beállítások

Szükség esetén ön további korlátozásokat léptethet életbe a felhasználók számára, amikor azok File Station, FTP, vagy WebDAV segítségével férnek hozzá egy megosztott mappához.

  • Könyvtárböngészés tiltása: Ennek a lehetőségnek az engedélyezése korlátozza a felhasználókat a megosztott mappa tartalmának megtekintésében.
  • Létező fájlok módosításának tiltása: Ennek a lehetőségnek az engedélyezése korlátozza a felhasználókat a megosztott mappában található fájlok áthelyezésében, törlésében vagy módosításában. Ne feledje, hogy a felhasználók továbbra is képesek a megosztott mappa tartalmának megtekintésére, letöltésére/feltöltésére, másolására, illetve kicsomagolására.
  • Fájlletöltés tiltása: Ennek a lehetőségnek az engedélyezése korlátozza a felhasználókat a megosztott mappa tartalmának letöltésében.

Megjegyzés:

A fenti korlátozások nem vonatkoznak az administrators csoporthoz tartozó felhasználókra.
