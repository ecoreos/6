USBCopy

Kiválaszthat és feállíthat egy célhelyet a USBCopy funkció számára.

Az USBCopy engedélyezése:

  1. Jelölje be az USBCopy engedélyezése lehetőséget.
  2. A legördülő menüből válasszon ki egy megosztott célmappát.
  3. Kattintson az OK gombra.

Ha USB eszközt (például egy digitális kamerát) csatlakoztatott a Synology DiskStation elülső paneljének USB portjára, akkor adatok USB eszközre másolásához az itt hozzárendelt megosztott célmappába megnyomhatja az elülső panelen található Copy gombot. Ha a USBCopy célmappája törlésre kerül, a program automatikusan helyettesíti azt a nyílvános megosztott mappával.

    Korlátozások:
  1. Csak FAT vagy NTFS formátumú USB eszközök támogatottak.
  2. Az USB eszközöket a Synology DiskStation előlapján található USB csatlakozóhoz kell csatlakoztatni.
  3. A megosztott célmappa rendelkezésre álló helyének nagyobbnak kell lennie az USB eszköz felhasznált helyénél.
