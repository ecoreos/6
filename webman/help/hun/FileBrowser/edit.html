Fényképek szerkesztése

A File Station alkalmazásban az Aviary Editor (egy webes alapú külső fényképszerkesztő program) segítségével szerkesztheti is fényképeit.

Fényképek szerkesztése az Aviary Editor alkalmazással:

  1. Nyissa meg a mappát, melyben a kívánt kép található.
  2. Kattintson jobb gombbal a fényképre, majd válassza a Szerkesztés az Aviary alkalmazásban lehetőséget.
  3. Az Aviary Editor megnyílik egy új böngészőablakban. Itt megtekintheti a fényképet, valamint a programban elérhető eszközök segítségével szerkesztheti is azt.
  4. A fényképszerkesztés befejezését követően kattintson a felső eszköztárban található mentés gombra a fényképen végzett módosítások mentéséhez.

Megjegyzés:

  • Az Aviary Editor az alábbi formátumú fényképek szerkesztését teszi lehetővé: .jpg, .jpeg, .jpe, .png
  • Az Aviary Editor legfeljebb 5 MB méretű képeket képes kezelni.
  • Az Aviary Editor használatához a(z) DiskStation eszköz hálózati beállításainak megfelelő konfigurálása szükséges. A hibaelhárításhoz tekintse át az alábbi beállításokat:
    • Ellenőrizze, hogy a(z) DiskStation eszköz elérhető-e a QuickConnect funkción keresztül (Vezérlőpult > QuickConnect), vagy képes-e lekérni egy külső IP-címet (Vezérlőpult > Külső hozzáférés > DDNS vagy Vezérlőpult > Hálózat > Hálózati interfész > PPPoE).
    • Ha DiskStation eszköze router mögött található, ellenőrizze, hogy megfelelően állította-e be a porttovábbítási szabályokat (Vezérlőpult > Külső hozzáférés > Router beállítása).
    • Ha továbbra sem tudja megnyitni az Aviary Editor alkalmazást, elképzelhető, hogy meg kell adnia DiskStation eszköze routerportját (Vezérlőpult > Külső hozzáférés > Speciális).
