File Station

A File Station egy könnyen használható fájlkezelő eszköze, ami lehetővé teszi a DSM felhasználók számára, hogy az Interneten keresztül hozzáférjenek a(z) DiskStation mappákhoz a webböngészőjük segítségével. Kettő vagy több File Station elindításával az összes DiskStation-adatot kezelheti úgy, hogy ide-oda húzza azokat a különböző File Stationök között.
