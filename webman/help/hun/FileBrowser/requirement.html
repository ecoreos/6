File Station-követelmények

A teljes kiépítettségű File Station használatához a legújabb webböngésző, Java és Flash Player szükséges.

  • Böngésző: Internet Explorer 8 vagy újabb, Firefox 3.6 vagy újabb, Safari 5.0 vagy újabb, Chrome; az AJAX-ot és a JavaScriptet engedélyezni kell
  • Java (mappák böngészéséhez a helyi számítógépen): Java Runtime Environment (JRE) 5 vagy újabb
  • Flash: Flash Player 9.0.28 vagy újabb
