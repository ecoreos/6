Fájlok vagy mappák másolása és áthelyezése

A File Station különböző módszereket biztosít fájlok/mappák átvitelére két DiskStation között, egy DiskStation eszközön belül, vagy egy helyi számítógépre.

Fájlok/mappák áthúzása egyik böngészőből a másikba

A fájlátviteli protokollok bonyolult beállítása nélkül közvetlenül feltölthet fájlokat/mappákat a File Station alkalmazáson keresztül az egyik DiskStation eszközről a másikra.

Áthúzás engedélyezése egyik böngészőből a másikba:

  1. Lépjen a Beállítások > Általános > File Station oldalra.
  2. Jelölje be a Húzás és elengedés engedélyezése böngészők között lehetőséget.
  3. Kattintson az OK gombra.
  4. Nyisson meg két ugyanolyan böngészőt (pl. Firefox), és csatlakoztassa őket a forrás, illetve a cél DiskStation eszközhöz.
  5. Indítsa el a File Station állomást mindkét DiskStation eszközön. Mostantól közvetlenül áthúzhat fájlokat/mappákat közöttük a File Station alkalmazáson keresztül.

Megjegyzés:

  • Áthúzhat fájlokat/mappákat ugyanazon böngésző két lapja között is.
  • Az Internet Explorer nem támogatja a böngészőlapok közötti húzást.
  • Ha egyszerre több fájlt/mappát választ ki áthúzásra, a rendszer egy fájlba tömöríti a fájlokat/mappákat a feltöltéshez.
  • Az áthúzás fájlméretmaximuma böngészőnként változik: Safari (1 GB).

Fájlok vagy mappák másolása vagy áthelyezése DiskStation eszközön belül vagy egy helyi számítógépre

Fájlok vagy mappák DiskStation eszközön belüli vagy egy helyi számítógépre való áthelyezéséhez vagy másolásához kövesse az alábbi lépéseket.

1. módszer:

  1. Válassza ki az áthelyezni, illetve másolni kívánt fájlokat vagy mappákat. (Többszörös kijelölés közben tartsa lenyomva a Shift vagy Ctrl billentyűt.)
  2. Válassza ki a Kivágás vagy Másolás lehetőséget a Tevékenység legördülő menüből.
  3. Lépjen be a(z) DiskStation készüléken vagy a számítógépen található célmappába.
  4. Attól függően, hogy a célhelyen található, azonos nevű fájlokat és mappákat felül kívánja-e írni, válassza a Beillesztés - Felülírás vagy Beillesztés - Kihagyás lehetőséget a Tevékenység legördülő menüből.

2. módszer:

  1. Válassza ki az áthelyezni, illetve másolni kívánt fájlokat vagy mappákat. (Többszörös kijelölés közben tartsa lenyomva a Shift vagy Ctrl billentyűt.)
  2. Húzza a mappákat vagy fájlokat a főrészből egy másik mappába a bal oldali panelen.
  3. Attól függően, hogy a célhelyen található, azonos nevű fájlokat és mappákat felül kívánja-e írni vagy ki akarja hagyni, a megjelenő előugró menüben válassza a Másolás - Felülírás, Másolás - Kihagyás, Áthelyezés - Felülírás, illetve Áthelyezés - Kihagyás lehetőséget.

3. módszer

  1. Válassza ki az áthelyezni, illetve másolni kívánt fájlokat vagy mappákat. (Többszörös kijelölés közben tartsa lenyomva a Shift vagy Ctrl billentyűt.)
  2. A fájlok/mappák kivágásához nyomja meg a Ctrl + X, a másolásához pedig a Ctrl + C billentyűkombinációt. Mac felhasználók nyomják meg a Command + X, illetve a Command + C billentyűkombinációkat.
  3. Lépjen be a(z) DiskStation készüléken vagy a számítógépen található célmappába.
  4. A fájlok/mappák beillesztéséhez nyomja meg a Ctrl + V billentyűkombinációt. (Command + V Mac esetén.)
  5. A felbukkanó párbeszédpanelen válassza ki, hogy a célmappában ugyanolyan néven található fájlokat felülírni vagy kihagyni szeretné-e.

Billentyűparancsok:

A következő billentyűparancsok hasznosak lehetnek, amikor a File Station alkalmazásban fájlokat és mappákat helyez át/másol:

  • F2: Átnevezés
  • Del: Törlés
  • Ctrl + A: Összes kiválasztása
  • Crtl + X: Kivágás
  • Crtl + C: Másolás
  • Crtl + V: Beillesztés

Megjegyzés:

  • Ha FAT fájlrendszerű könyvtárba másol vagy helyez át fájlokat/mappákat, a fájlnévben szereplő \ *?">| karakterek helyett – jelenik meg, a Mac fájlok Metadata adatai pedig törlődnek.
  • Ha a helyi számítógépről vág ki fájlokat, majd a(z) DiskStation egyik mappájába illeszti be őket, a fájlok megmaradnak a helyi számítógépen is.
  • A fájlok vagy mappák áthelyezésekor a „tulajdonos” ugyanaz marad. Egy fájl vagy mappa másolásakor azonban a „tulajdonos” a másolást végző felhasználó lesz.
