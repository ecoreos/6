패키지 관리

온라인으로 또는 수동으로 패키지를 설치한 후 패키지 센터에서 이를 최신 상태로 유지하고 원활하게 실행되도록 관리할 수 있습니다. 아래 지침은 패키지를 실행, 업데이트, 설치 제거, 실행 또는 중지하고 패키지 문제를 해결하는 방법에 대해 설명합니다.

패키지를 업데이트하려면 다음을 수행합니다:

패키지를 항상 최신 상태로 유지하는 것이 DiskStation을(를) 잘 관리하는 최선의 방법입니다.

  • 왼쪽 패널에서 업데이트를 클릭하여 사용 가능한 업데이트를 확인합니다. 업데이트 단추 또는 모두 업데이트를 클릭하여 패키지를 업데이트합니다.
  • 수동 설치를 클릭하고 .spk 파일을 업로드하여 패키지를 설치합니다(Synology의 다운로드 센터 또는 타사 웹사이트에서 구할 수 있음).
  • 패키지 업데이트는 설정 > 자동 업데이트에 있는 옵션에 따라 표시됩니다. 자세한 정보는 여기를 참조하십시오.

참고: 업데이트가 사용 가능한 경우 이메일 또는 바탕화면 알림을 수신하려면 설정 > 일반 > 알림을 클릭하고 선택 상자를 선택한 다음 확인을 클릭합니다.

패키지를 수리하려면 다음 중에서 수행합니다:

DiskStation의 패키지에 에러가 발생하면 이를 수리해서 다시 제대로 작동되도록 해야 합니다. 때때로 이러한 문제는 패키지가 너무 오래되어 최신 버전의 DSM과 호환되지 않을 때 발생하거나 패키지의 프로그램 파일이 누락되었거나 손상되었을 때 일어납니다. 수리가 필요한 패키지는 설치한 항목 범주 아래에 나열됩니다.

  • 패키지를 수리하려면 복구를 클릭합니다.
  • 수동 설치를 클릭하고 .spk 파일을 업로드하여 패키지를 다시 설치하고 수리합니다(Synology의 다운로드 센터 또는 타사 웹사이트에서 구할 수 있음).

참고: 복구를 클릭하여 실제로 이전 패키지를 수리하지 않고 최신 버전의 이 패키지를 설치합니다. 패키지 설치에 여전히 문제가 있는 경우, 이 패키지의 게시자나 개발자에게 문의하십시오.

기존 패키지 제거하기:

설치된 패키지를 클릭하고 작업 드롭다운 메뉴에서 설치 제거를 선택하여 패키지를 제거합니다.

패키지 서비스 실행/중지하기:

설치된 패키지를 클릭하고 작업 드롭다운 메뉴에서 실행 또는 중지를 선택하여 패키지를 실행하거나 중지합니다.

패키지 문제 해결하기:

패키지에 문제가 있으면 다음과 같이 시도해 보십시오:

  • 패키지를 최신 버전으로 업데이트합니다.
  • 패키지와 해당 데이터베이스(있을 경우)를 제거하고 다시 설치합니다.
  • 패키지의 게시자나 개발자에게 연락하여 도움과 지원을 받습니다.

패키지 게시자의 연락처 정보 찾기:

Synology 패키지 센터의 일부 패키지는 타사 개발자가 만들어서 게시한 것입니다. 따라서 Synology는 개별 응용 프로그램에 대한 기술 지원을 제공하지 않습니다. 기능 요청, 질문 및 기타 피드백에 대해서는 각 응용 프로그램의 게시자 또는 개발자에게 문의하십시오. 일부 패키지의 경우 게시자 외에 개발자도 있습니다. 게시자는 패키지를 만들고 Synology 제품에 사용할 수 있도록 하는 반면, 패키지의 필수 기능은 대개 개발자의 허가를 받아 제공됩니다. 게시자가 제공된 경우 지원을 위해 게시자를 찾으십시오.

  1. 설치된 패키지의 상세 정보 페이지로 이동합니다.
  2. 상세 정보 페이지의 왼쪽에서 해당 게시자 또는 개발자의 연락처 정보를 찾을 수 있습니다.
  3.  Synology가 개발한 패키지에 대한 기능 요청, 질문 또는 피드백이 있으면 Synology 웹사이트를 방문하여 직접 문의하십시오.

응용 프로그램 실행

패키지를 설치하고 실행한 후, 활성화된 응용 프로그램을 여러 가지 방법으로 시작할 수 있습니다.

설치되고 활성화된 응용 프로그램을 시작하려면 다음을 수행합니다:

  • 메인 메뉴 > [응용 프로그램 이름]으로 이동합니다.
  • 메인 메뉴를 열고 응용 프로그램의 섬네일을 DSM 바탕화면으로 끈 다음 바탕화면에서 [응용 프로그램 이름]을 클릭합니다.
  • 메인 메뉴 > [응용 프로그램 이름]으로 이동합니다. 작업 표시줄에 있는 응용 프로그램 아이콘을 마우스 오른쪽 단추로 클릭하고 작업 표시줄에 고정을 선택합니다. 응용 프로그램을 닫으면 해당 아이콘이 작업 표시줄에 고정된 상태로 있으며, 이를 클릭하여 응용 프로그램을 다시 시작할 수 있습니다.
