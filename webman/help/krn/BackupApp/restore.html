복원

백업 및 복원으로 이동하여 왼쪽 구석 아래에서 복원을 클릭하여 백업 데이터를 복원합니다.

데이터/LUN* 백업 복원하기:

  1. 왼쪽 아래 구석에서 복원을 클릭한 다음 데이터 또는 LUN*을 선택하고 복원할 백업 작업을 선택합니다.
  2. 시스템 구성, 서로 다른 버전의 백업 데이터 등을 선택하라는 메시지가 표시될 수 있습니다. 이 메시지는 복원하려는 백업 작업 유형에 따라 달라집니다.
  3. 백업 작업이 암호화된 경우 성공적인 복원을 위해 패스워드/암호화 키가 필요합니다.
  4. 마법사를 따라 복원을 완료합니다.

참고:

  • 제거된 패키지를 복원하는 경우 시스템에서는 이 응용 프로그램의 동일 버전 또는 최신 버전을 자동으로 설치합니다. 응용 프로그램이 설치되어 있지만 백업된 버전보다 오래된 경우 시스템이 자동으로 원래 백업된 버전으로 업그레이드합니다.
  • DSM 5.1 이상에서 백업한 응용 프로그램은 DSM 5.0 이하 버전에서 복원되지 않습니다.

참고:

  • 복원이 완료되면 블록 수준 및 파일 수준 LUN이 파일 수준 LUN으로 복원됩니다.*
  • 패키지를 복원할 때, 해당 패키지가 설치되어 있지 않으면 시스템이 같은 버전이거나 최신 버전인 응용 프로그램을 자동으로 설치합니다. 응용 프로그램이 설치되어 있지만 백업된 버전보다 오래된 경우 시스템이 자동으로 원래 백업된 버전으로 업그레이드합니다.
  • DSM 5.1 이상에서 백업한 응용 프로그램은 DSM 5.0 이하 버전에서 복원되지 않습니다.

복수 버전 백업 작업에 대해 단일 파일 또는 폴더 복원하기:

  1. 왼쪽 패널에서 원하는 백업 작업을 선택하고 돋보기 아이콘을 클릭하여 백업 탐색기를 실행합니다. 백업 작업이 암호화되어 있으면 패스워드/암호화 키가 필요합니다.
  2. 시각표 표시줄을 사용하거나 달력 아이콘을 통해 원하는 백업을 찾습니다.
  3. 포함된 파일 또는 폴더를 선택하고 복원, 복사 대상... 또는 다운로드를 클릭하여 선택한 파일/폴더를 검색합니다. 파일/폴더를 마우스 오른쪽 단추로 클릭해도 이러한 세 가지 옵션을 사용할 수 있습니다.

특정 버전 검색하기:

파일 검색 속도를 높이려면 키워드를 사용해 원하는 파일/폴더를 필터링합니다. 일치하지 않는 파일/폴더는 표시되지 않습니다.

  1. 백업 탐색기를 실행하고 오른쪽 위 구석에서 깔대기 아이콘을 클릭합니다.
  2. 드롭다운 메뉴에서 아래 기준에 따라 파일/폴더를 필터링할 수 있습니다.
    • 키워드: 파일 또는 폴더 이름을 입력합니다. 키워드에 와일드카드[*](1개 또는 그 이상의 문자를 나타냄) 또는 [?](1개의 임의 문자를 나타냄)를 사용할 수 있습니다. 예를 들어, [a*n]은 [action], [attention] 또는 [an]을 나타낼 수 있습니다.
    • 파일 유형: 임의, 임의 폴더 또는 임의 파일 중에서 선택하여 검색을 세밀하게 조정할 수 있습니다.
    • 크기(MB): 임의를 선택하거나 같음, 이상, 미만을 선택한 다음 값을 입력합니다.
    • 수정된 날짜: 오늘, 어제, 지난 주, 지난 달 또는 사용자 지정 중에서 선택한 다음 시작 및 종료 날짜를 입력하여 지정한 범위 내에서 버전을 찾습니다.
  3. 필터을 클릭하여 필터를 적용합니다.
  4. 다른 백업 버전에 동일한 필터를 적용하려면 시각표 표시줄을 사용하여 원하는 버전으로 전환하기만 하면 됩니다.
  5. 필터링된 검색을 재설정하려면, 재설정 단추를 클릭합니다.

참고:

  • 백업 탐색기는 복수 버전 백업 작업에만 사용할 수 있습니다.
  • 백업 대상이 DSM 5.1 이하에서 설정된 경우 백업 탐색기의 기능이 제한됩니다.
  • 시스템 구성 및 응용 프로그램은 복원을 통해서만 복원할 수 있습니다.
  • 필터 기능은 암호화되지 않은 백업 작업에만 사용할 수 있습니다.
  • 복원 대상... 또는 다운로드를 사용하여 검색한 파일/폴더는 원래 권한 설정이 손실됩니다.

다른 백업 대상에서 백업 작업 복원하기:

  1. 왼쪽 아래 구석에서 복원을 클릭한 다음 데이터 또는 LUN*을 선택합니다.
  2. 다음과 같이 백업 작업의 전체 유형을 선택하여 복원할 수 있습니다.
    • 백업 데이터 복원하기: 데이터 > 기존 리포지토리에서 복원을 클릭한 다음 데이터를 복원하려는 백업 대상을 선택합니다.
    • LUN 백업 복원하기: LUN*을 클릭합니다.
  3. 마법사의 지침을 따릅니다. 복원할 작업이 암호화되어 있으면 패스워드/암호화 키가 필요합니다.
  4. 적용을 클릭해서 복원을 종료합니다.

참고:

  • Embedded DataStation은 공유 폴더를 ext3/ext4 외부 장치에만 복원할 수 있습니다.
  • Embedded DataStation은 ext3/ext4 외부 장치가 없으면 응용 프로그램 복원을 지원하지 않습니다.

* 일부 모델에서만 가능.
