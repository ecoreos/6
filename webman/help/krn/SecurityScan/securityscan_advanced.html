Security Advisor의 고급 설정을 보고 변경할 수 있습니다.

보안 기준

보안 기준을 선택하여 Security Advisor가 검사할 규칙을 선택할 수 있습니다. 규칙의 수와 유형은 각 보안 기준에 따라 달라집니다. 필요에 맞추어 선택하십시오. 업무 및 비즈니스 용과 가정 및 개인용 중 어떤 용도로 DiskStation을(를) 사용 중입니까? 다음 옵션 중에서 선택할 수 있습니다:

  • 가정 및 개인적인 용도: Security Advisor는 일반 사이버 위협으로부터 DiskStation을(를) 보호하기 위한 필수 검사를 수행합니다.
  • 업무 및 비즈니스 용도: Security Advisor는 조직의 보안 준수 요건을 충족하기 위한 종합 검사를 수행합니다.
  • 사용자 지정: Security Advisor가 검사하게 할 보안 규칙을 선택합니다.

스캔 일정

정기 스캔 일정 활성화를 클릭하고 Security Advisor가 스캔을 수행할 시간을 선택하여 정기 스캔 일정을 정의할 수 있습니다. Security Advisor는 선택한 보안 기준에 따라 규칙을 검사합니다.
