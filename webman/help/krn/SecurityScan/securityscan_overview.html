개요

DSM의 현재 상태를 확인하고, 시스템을 스캔하고, 스캔 진행과 결과 요약을 확인할 수 있습니다. Security Advisor는 몇 가지 보안 규칙을 확인함으로써 DSM 및 DiskStation을(를) 스캔합니다. 규칙의 수와 유형은 보안 기준을 설정하는 방법에 따라 달라집니다. 보안 기준은 사용자의 필요에 의해 결정됩니다. 업무 및 비즈니스 용과 가정 및 개인용 중 어떤 용도로 DiskStation을(를) 사용 중입니까? 스캔을 처음 시작할 때 보안 기준을 선택하라는 요구를 받습니다. 고급 페이지에서 나중에 이를 변경할 수 있습니다.

시스템 상태

Security Advisor는 총 일곱 가지 상태를 가집니다. 스캔을 마치고 나면, 5가지 범주로 결과 요약이 나타납니다. 다양한 상태는 다음과 같습니다:

  • 스캔 준비 완료: DiskStation을(를) 아직 스캔하지 않았을 때 이 상태가 표시됩니다. 스캔을 클릭하여 스캔을 시작합니다.
  • 스캔 중: 스캔이 진행 중입니다. 스캔 진행률을 볼 수 있습니다. 스캔을 중지하려면 중지를 클릭합니다.
  • 좋음: 현재 스캔을 마쳤습니다. Security Advisor가 취약한 설정 또는 보안 위험을 발견하지 못했습니다. DiskStation이(가) 현재 안전합니다.
  • 경고: 현재 스캔을 마쳤습니다. 일부 취약한 보안 설정으로 인해 Security Advisor가 몇 가지 낮은/보통 보안 위험을 보고했습니다. DiskStation을(를) 더 안전하게 만들기 위한 권장 조치를 수행하는 것이 좋습니다.
  • 위험성: 현재 스캔을 마쳤습니다. 일부 취약한 보안 설정으로 인해 Security Advisor가 몇 가지 높은 보안 위험을 보고했습니다. DiskStation을(를) 더 안전하게 만들기 위한 권장 조치를 반드시 수행하는 것이 좋습니다.
  • 위험: 현재 스캔을 마쳤습니다. Security Advisor기 DiskStation에서 맬웨어를 발견했습니다. DiskStation에 원치 않는 결과가 생기는 것을 막기 위해 권장 조치를 수행해야 합니다.
  • 오래됨: DSM 버전이 최신 상태입니다. 경험을 향상시키고 DiskStation을(를) 안전하게 지키기 위해 DSM을 업데이트하여 최신 기능, 향상 사항 및 보안 수정을 받는 것이 좋습니다.

Security Advisor는 확인을 5가지 범주로 나눕니다:

  • 맬웨어: DiskStation에 맬웨어가 있는지 및 DiskStation이(가) 공격을 받고 있는지 감지합니다
  • 시스템: 시스템 관련 보안 설정을 감지합니다.
  • 계정: 계정 및 패스워드 설정을 감지합니다.
  • 네트워크: 네트워크 관련 보안 설정을 감지합니다.
  • 업데이트: DSM 및 설치된 다른 패키지가 최신 상태인 여부를 감지하고 업데이트 관련 설정을 확인합니다.

스캔을 마치고 나면, 결과 보기를 클릭하여 보고서별로 상세 결과를 볼 수 있습니다.
