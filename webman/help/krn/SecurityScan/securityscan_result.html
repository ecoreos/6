결과

보고서별로 완료된 스캔에 대한 상세 정보를 볼 수 있습니다. 이 페이지에는 다음 정보가 있습니다:

  • 심각도: Security Advisor가 각 보안 규칙에 대해 평가하는 전반적인 위험 수준. 4가지 수준이 있습니다: 보통, 경고, 알람, 중대.
  • 설명: 조치를 취하지 않을 경우 Security Advisor가 DiskStation에 문제가 될 수 있는 특정 규칙을 찾아보는 설정.
  • 마지막 업데이트: Security Advisor가 이 보안 규칙을 마지막으로 검사한 시간.
  • 상태: 각 보안 규칙의 현재 상태입니다. Security Advisor는 각 보안 규칙의 결과를 통과함 또는 실패로 보고합니다. 때로는 오류, 확인 안함, 실행 중 및 건너뜀 상태가 표시될 수 있습니다. 페이지 상단에서 결과를 필터링할 수 있습니다.

보안 규칙의 보고서 세부 사항 확인하기:

규칙을 선택하고 보기를 클릭합니다. 심각도 및 설명 이외에 각 보고서에는 다음 세부 사항이 포함됩니다:

  • 세부 사항: 약점에 대한 간단한 설명, Security Advisor가 약점을 처리할 때 권장하는 내용 및 약점을 처리해서 얻을 수 있는 이점.
  • 권장되는 작업: 보고된 보안 약점을 처리하는 방법에 대한 자세한 지침.
  • 링크: 실패한 보안 규칙의 경우, 링크를 클릭하면 권장 조치를 수행하기 위한 DSM의 위치로 이동됩니다.
  • 수정: 맬웨어가 감지될 때 수정을 누르면 Security Advisor가 이를 제거하도록 지시합니다.

특정 규칙 다시 스캔하기:

다시 스캔할 항목을 선택하고 스캔을 클릭합니다.
