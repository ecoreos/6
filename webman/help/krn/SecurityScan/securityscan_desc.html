보안 어드바이저

Security Advisor는 DSM 설정 및 DiskStation을(를) 스캔하는 DSM 보안 응용 프로그램입니다. Security Advisor는 설정을 검사하고 DiskStation의 안전을 유지하는 데 도움이 되는 변경 사항을 추천해 줍니다. 아래는 제공되는 기능의 목록입니다.

  • Security Advisor는 보안 위험이나 의심스런 활동의 징조일 수 있는 DSM 설정을 감지하기 위해 여러 가지 검사를 수행합니다.
  • Security Advisor는 상세한 결과를 제공하고, 취해야 할 권장 조치 및 그로 인한 혜택을 설명하는 보고서를 제공합니다.
  • 내부 보안 정의 데이터베이스는 자동으로 업데이트되어 변화하는 사이버 위협에 최신 상태로 대처합니다.

Security Advisor를 실행하려면 다음을 수행하십시오:

  • 메인 메뉴 > Security Advisor로 이동합니다.
