아카이브 압축 풀기

  1. 압축을 풀 아카이브를 선택합니다.
  2. 작업 메뉴에서 풀기를 선택합니다. 파일 목록에서 해당 아카이브의 내용을 확인할 수 있습니다.
  3. 아카이브가 패스워드로 보호된 경우 옵션 탭을 클릭하여 패스워드를 입력합니다.
  4. 맨 아래에 있는 대상을 클릭하고 나타나는 창에서 DiskStation의 대상 폴더를 선택합니다.
  5. 다음 중 하나를 수행하십시오.
    • 아카이브의 모든 파일과 폴더의 압축을 풀려면 모두 압축 풀기를 클릭합니다.
    • 아카이브의 일부 파일 또는 폴더의 압축을 풀려면 파일 목록에서 원하는 항목을 선택하고 선택 항목 압축 풀기를 클릭합니다.

참고: 지원되는 압축 파일 형식: .zip, .tar, .gz, .tgz, .rar, .7z, .iso (ISO 9660 + joliet).
