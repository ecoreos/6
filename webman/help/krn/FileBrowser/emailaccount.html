파일을 이메일 첨부 파일로 보내기

File Station에서 선택한 파일을 마우스 오른쪽 단추로 클릭한 다음 이메일 첨부 파일로 보내기를 선택하여 파일을 이메일 첨부 파일로 직접 보낼 수 있습니다. 이메일 창이 몇 가지 유용한 기능과 함께 표시됩니다:

  • 받는 사람: 이메일 계정의 연락처를 동기화한 경우, 사용자의 이메일 계정과 관련된 연락처 동기화가 표시된 연락처 목록이 나타납니다.
  • 원래 그룹: 기본 이메일 계정이나 다른 추가된 이메일 계정을 선택할 수 있습니다. 연락처 목록이 자동으로 선택한 이메일 계정으로 변경됩니다.
  • 첨부 파일: 이메일 하단에 선택한 첨부 파일이 표시됩니다. 첨부 단추를 사용하여 첨부 파일을 더 추가할 수 있습니다.

참고:

  • 파일만 첨부 파일로 추가할 수 있습니다.
  • 이메일을 한 번 보낼 때 최대 10개 파일을 첨부할 수 있습니다.
  • 이메일을 한 번 보낼 때 모든 첨부 파일의 최대 총 파일 크기는 사용자의 이메일 서비스에 따라 다릅니다. 잘 모를 경우 모든 첨부 파일의 총 파일 크기를 25MB로 제한하는 것이 좋습니다.
