파일 다운로드

아래 단계를 따라 파일 또는 폴더를 DiskStation에서 컴퓨터로 다운로드합니다.

로컬 컴퓨터로 파일 다운로드하기:

  1. 다운로드할 파일 또는 폴더를 선택합니다. (여러 항목을 선택하려면 Shift 또는 Ctrl 키를 누른 상태에서 선택하십시오.)
  2. 작업 > 다운로드를 클릭하고 웹 브라우저에 따라 원하는 다운로드 모드를 선택합니다.
    • Google Chrome: 다운로드를 클릭하여 개별 파일(압축되지 않은)을 다운로드합니다. .zip으로 다운로드를 클릭하여 여러 개의 파일을 .zip 파일로 다운로드합니다(설정 > 일반에서 지정한 File Station 코드페이지 사용).
    • 기타 브라우저: 다운로드를 클릭하여 하나 또는 여러 개의 파일을 .zip 파일로 다운로드합니다(설정 > 일반에서 지정한 File Station 코드페이지 사용).
    • 대기열에 추가 - 덮어쓰기 또는 대기열에 추가 - 건너뛰기: 대상에서 동일한 이름을 가진 파일 또는 폴더를 덮어쓸 것인지 여부에 따라, 옵션 중 하나를 선택하여 파일을 다운로드 대기열에 추가합니다(오른쪽 위 구석에서 아래 방향 화살표 클릭).
    참고:
  1. Windows 환경에서 파일 이름에 포함된 \ * ? " < > | : 문자들은 다운로드 후에 -으로 변경됩니다.
  2. Chrome에서 여러 개의 파일이 자동으로 다운로드되는 것을 중지하려면, 설정으로 가서 개인 정보 보호 섹션에서 내용 설정을 변경합니다.
  3. incognito 모드의 Chrome은 개별 파일을 압축되지 않은 파일로 다운로드하는 것을 지원하지 않습니다.

파일/폴더의 다운로드 링크 가져오기

파일 또는 폴더로 다운로드 링크를 가져와서 다른 사용자와 링크를 공유할 수 있습니다.

파일 또는 폴더의 다운로드 링크 가져오기:

  1. 다운로드 링크를 가져올 파일 또는 폴더를 선택합니다.
  2. 다음 중 하나를 수행하여 속성 창을 엽니다.
    • 작업 메뉴에서 속성을 선택합니다.
    • 파일 또는 폴더를 마우스 오른쪽 단추로 클릭하고 속성을 선택합니다.
  3. 다운로드 링크를 마우스 오른쪽 단추로 클릭하고 바로가기 메뉴에서 적절한 기능을 선택하여 다운로드 링크를 복사합니다. 기능의 이름은 사용 중인 웹 브라우저에 따라 다를 수 있습니다.
  4. 이메일 또는 인스턴트 메시지를 통해 다른 사용자에게 링크를 전송합니다. 그러면 사용자가 해당 사용자 이름과 패스워드를 입력해서 연결된 파일 또는 폴더를 다운로드할 수 있습니다.

참고: 다운로드 링크를 통해 파일 또는 폴더를 다운로드하려면 유효한 사용자명과 패스워드가 필요합니다.
