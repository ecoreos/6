파일 링크 공유

DiskStation에 저장된 파일이나 폴더는 빠르고 쉽게 공유할 수 있습니다. 간단히 파일이나 폴더를 마우스 오른쪽 단추로 클릭하고 파일 링크 공유를 선택하면 됩니다. 그러면 링크와 QR 코드가 생성되어 친구나 다른 사용자에게 보낼 수 있습니다. 받은 사람은 DSM 계정을 소유하고 있는지에 관계없이 링크를 통해 선택된 파일 또는 폴더를 다운로드할 수 있습니다.

파일 링크 공유하기:

  1. 공유할 파일 또는 폴더를 선택합니다.
  2. 선택된 파일 또는 폴더 중 하나를 마우스 오른쪽 단추로 클릭하거나 작업 메뉴를 클릭합니다. 그런 다음 파일 링크 공유를 선택합니다.
  3. 나타난 창에 공유 링크와 QR 코드가 표시됩니다.
  4. 또한 다음 옵션을 수정할 수 있습니다:
    • 유효 기간: 공유 링크에 대해 유효한 액세스 기간과 액세스 횟수를 사용자 지정할 수 있습니다.
      • 유효 기간 사용자 지정: 공유 링크가 생성 후 얼마 동안이나 활성 상태로 남아 있을지 결정할 수 있습니다. 링크는 지정된 기간이 지나면 무효로 바뀝니다.
      • 유효한 액세스 횟수 사용자 지정: 공유된 링크에 대해 액세스될 수 있는 횟수를 결정할 수 있습니다. 링크는 지정된 횟수가 지나면 무효로 바뀝니다.
    • DSM 사용자와 공유: 드롭다운 목록에서 DSM 사용자 또는 그룹을 선택하여 링크를 공유할 수 있습니다. guest 계정 및 사용자 그룹과는 공유할 수 없음에 유의하십시오.
    • 다른 사람과 공유: 패스워드로 보호된 링크는 비-DSM 사용자를 비롯한 다른 사람들과 공유할 수 있습니다.

gofile.me 링크 활성화하기:

  1. 제어판 > QuickConnect로 가서 QuickConnect 활성화가 선택되어 있는지 확인합니다. QuickConnect를 설정하지 않은 경우 이 페이지에서 설정할 수 있습니다.
  2. 다음, 고급을 클릭하고 파일 공유가 선택되어 있는지 확인합니다.
  3. 공유 파일은 DSM 계정이 없어도 gofile.me 공유 링크에 액세스하여 다운로드할 수 있습니다.

공유된 링크 페이지 사용자 지정

File Station은 공유된 링크 페이지에 스타일을 적용하기 위한 다양한 옵션을 제공합니다.

  1. File Station > 설정 > 공유된 링크 > 사용자 지정으로 갑니다.
  2. 공유된 링크 페이지 사용자 지정을 선택하고 사용자 지정 설정을 클릭합니다.
  3. 아래와 같은 방법으로 페이지를 사용자 지정할 수 있습니다:
    • 템플릿: 기본 템플릿 중 하나를 선택합니다.
    • 배경: 섬네일 또는 +를 클릭하고 기본 배경 무늬, DiskStation의 이미지 또는 이전에 적용된 이미지를 선택합니다.
    • 로고: 섬네일 또는 +를 클릭하고 업로드된 이미지(1MB 미만), DiskStation의 이미지 또는 이전에 적용된 이미지를 선택합니다. 로고는 네 모퉁이 중 아무 곳에나 위치할 수 있습니다.
  4. 미리보기를 클릭하여 조정이 필요한지 확인합니다.
  5. 확인을 클릭하여 설정을 적용합니다.

참고:

  1. 사용자 지정된 설정은 생성된 시간에 관계없이 모든 공유된 링크에 적용됩니다.
  2. 다음과 같은 형식의 이미지만 배경 무늬/로고로 적용할 수 있습니다: .jpe, jpeg, .jpg, .bmp, .gif, .png

공유 링크 관리

기존의 공유 링크를 관리하려면 도구 > 공유 링크 관리자로 갑니다. 나타난 창에는 기존 공유 링크를 편집, 공유, 제거할 수 있는 옵션이 있습니다.

참고:

  1. 최대 1000개의 공유 링크를 생성할 수 있습니다.
  2. 각 공유 링크의 상태 섹션에는 다음 중 하나가 표시됩니다.
    • 유효: 공유 링크가 제대로 작동합니다.
    • 만료됨: 공유 링크의 유효 기간이 만료되었습니다.
    • 잘못됨: 공유된 파일 또는 폴더가 이 더 이상 존재하지 않거나, 파일 또는 폴더에 접근하기 위한 적절한 권한이 없습니다.
    • 아직 효력 없음: 공유 링크가 아직 활성화되지 않았습니다.
