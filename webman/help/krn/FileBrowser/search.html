파일 또는 폴더 검색

File Station의 필터는 기준에 맞는 파일이나 폴더를 검색하여 표시하는 옵션을 제공합니다. 현재 폴더의 파일 또는 폴더를 필터링하거나, 모든 하위 폴더를 대상으로 고급 검색을 수행하여 좀더 세분화된 검색 결과를 가져올 수 있습니다.

현재 폴더의 파일 또는 폴더 검색하기:

  1. File Station을 사용하여 검색하려는 내용이 들어 있는 폴더로 이동합니다.
  2. 오른쪽 위 구석에 있는 필터 필드에 키워드를 입력한 후, 창의 메인 영역에 필터링된 결과를 확인합니다.

고급 검색 수행하기:

  1. File Station을 사용하여 검색하려는 내용이 들어 있는 폴더로 이동합니다.
  2. 오른쪽 위 구석에 있는 필터 필드 옆의 돋보기를 클릭하여 고급 검색 메뉴를 표시합니다.
  3. 아래에 검색 기준을 지정하거나 드롭다운 메뉴에서 선택합니다:
    • 키워드: 이름이 일치하는 파일 또는 폴더를 검색하려면 키워드를 입력합니다.

      참고:

      • 키워드에 와일드카드[*](0개 또는 그 이상의 문자를 나타냄) 또는 [?](1개의 임의 문자를 나타냄)를 사용할 수 있습니다. 예를 들어, [a*n]은 [action], [attention] 또는 [an]을 나타낼 수 있습니다.
      • 공백을 사용하여 키워드를 구분할 수 있습니다. 예를 들어, 키워드 [File Station]을 사용하면 [File], [Station] 또는 이 둘다를 포함하는 결과가 검색됩니다.
      • 키워드 시작 부분과 끝 부분에 큰 따옴표 ["]를 추가하면 대소문자까지 정확히 일치하는 항목을 검색할 수 있습니다. 예를 들어 ["Synology Inc."]는 [Synology Inc.]를 나타내지만 [Synology Incorporation]을 나타내지는 않습니다.
      • 좀더 세밀한 검색 결과를 위해 아래의 연산자를 사용할 수 있습니다:
        연산자 예제 사용
        && [X] && [Y]
      • [X] 및 [Y] 둘다를 포함하는 일치 항목을 나열합니다
        		•
        || [X] || [Y]
      • [X], [Y] 또는 둘다를 포함하는 일치 항목을 나열합니다
      • [X] 및 [Y] 둘다를 포함하는 일치 항목은 가장 위에 나열됩니다
        		•
        + +[X]
      • [X]를 포함하는 일치 항목을 나열합니다
        		•
        - -[X]
      • [X]를 포함하지 않는 일치 항목을 나열합니다
        		•
        ! ![X]
      • [X]를 포함하지 않는 일치 항목을 나열합니다
        		•
        \ \[SpecialCharacter]
      • 특수 문자들:   +   -   &&   ||    !    (    )    {    }    [    ]    ^    "    ~    *    ?    :   \
      • 위에 지정된 특수 문자를 포함하는 일치 항목을 나열합니다
        		•
    • 파일 내용 검색 활성화: 키워드와 일치하는 파일 내용 또는 메타데이터가 있는 색인된 파일을 검색합니다.
    • 위치: 드롭다운 메뉴에서 선택하여 고급 검색의 범위와 정도를 설정합니다. 또한 나열되어 있는 기타 다른 공유 폴더를 선택하여 검색 범위를 사용자 지정할 수 있습니다.
    • 파일 유형: 검색 범위를 특정 파일 형식으로 제한하려면 확장자를 선택하고, 폴더만 검색하려면 임의 폴더를 선택합니다.
    • 날짜: 수정된 날짜, 생성된 날짜 또는 마지막으로 액세스한 날짜 중에서 선택한 다음, 달력에서 날짜를 선택해서 날짜가 일치하는 파일을 검색합니다.
    • 크기(MB): 같음, 이상, 미만 중에서 선택한 다음, 크기를 입력해서 해당 크기와 일치하는 파일을 검색합니다. (같음을 선택할 때 검색 결과는 지정한 것과 거의 동일한 크기의 파일을 포함합니다.)
    • 소유자/그룹: 소유자 또는 그룹을 선택한 다음, 드롭다운 메뉴 중에서 선택해서 선택한 소유자 또는 그룹에 속하는 파일/폴더를 검색합니다.

      참고:

      • 고급 검색을 수행하려면, 위치와 더불어 최소한 하나의 다른 기준을 지정해야 합니다.
      • ISO 폴더 또는 원격 폴더가 인덱스된 공유 폴더에 마운트된 경우, 고급 검색을 통해 내부의 파일을 찾을 수 없습니다.
  4. 검색을 시작하려면 검색을 클릭하고, 검색 기준을 다시 지정하려면 재설정을 클릭합니다. 검색이 완료되면 File Station 의 메인 영역에 일치하는 파일이 나열됩니다.
