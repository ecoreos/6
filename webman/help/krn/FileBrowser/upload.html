파일 또는 폴더 업로드

컴퓨터에서 DiskStation(으)로 파일 또는 폴더를 업로드하려면 다음 방법 중 하나를 사용합니다.

아래 표시된 업로드 기능을 모두 이용하려면 먼저 최신 Java Runtime Environment(JRE) 및 Flash Player를 설치합니다.

방법 1:

  1. 왼쪽 패널의 상단에서 DiskStation의 대상 폴더를 선택합니다.
  2. 업로드를 클릭하고 업로드 - 건너뛰기 또는 업로드 - 덮어쓰기를 선택하고, 로컬 컴퓨터에서 파일 또는 폴더를 선택합니다.

방법 2:

  1. 설정으로 이동하여, File Station으로 로컬 컴퓨터에서 파일 검색을 선택하고, 웹 페이지를 새로 고쳐 왼쪽 패널 아래쪽에서 로컬 컴퓨터의 폴더를 확인합니다.

    2. 참고:

    File Station으로 로컬 컴퓨터에서 파일 검색 옵션은 컴퓨터에 JRE를 설치하고 활성화한 후에만 효과가 있습니다.

  2. 왼쪽 패널의 하단에서 로컬 컴퓨터의 폴더를 선택합니다. 메인 섹션에 선택한 폴더의 내용이 표시됩니다.
  3. 메인 섹션에서 원하는 파일 또는 폴더를 선택한 다음, 업로드를 클릭합니다.
  4. 나타나는 창에서 DiskStation의 대상 폴더를 선택합니다.
  5. 대상 폴더에 같은 이름을 가진 파일 또는 폴더를 덮어쓰려면 창의 상단 오른쪽 모서리에 있는 덮어쓰기를 선택합니다. 그렇지 않으면 건너뛰기를 선택합니다.
  6. 업로드를 시작하려면 확인을 클릭합니다.

방법 3:

  1. 설정으로 이동하여, File Station으로 로컬 컴퓨터에서 파일 검색을 선택하고, 웹 페이지를 새로 고쳐 왼쪽 패널 아래쪽에서 로컬 컴퓨터의 폴더를 확인합니다.
  2. 왼쪽 패널에서 컴퓨터의 폴더(하단)를 DiskStation의 아무 폴더(상단)로 끌어서 이동합니다.
  3. 나타나는 팝업 메뉴에서 대상 폴더에 같은 이름을 가진 파일/폴더를 덮어쓸지 또는 건너뛸지 선택합니다.

방법 4:

  1. 왼쪽 패널의 상단에서 DiskStation의 대상 폴더를 선택합니다.
  2. 컴퓨터의 바탕화면이나 파일 탐색기(예: Windows 탐색기 또는 Mac Finder)에서 파일을 선택한 다음, 웹 브라우저에서 File Station으로 끌어 놓습니다.
  3. 나타나는 대화 상자에서 대상 폴더에 같은 이름을 가진 파일을 덮어쓰거나 건너뛸지 여부를 선택합니다.

    4. 참고:

    • 방법 4는 특정 브라우저에서만 지원됩니다: Chrome, Firefox, Safari 5.1 이상. 그러나 File Station으로 로컬 컴퓨터에서 파일 검색이 선택되어 있으면 Mac에서 지원되지 않습니다.
    • 또한 위에 나열된 브라우저에 대해 File Station으로 로컬 컴퓨터에서 파일 검색이 선택되어 있을 때만 폴더 또는 빈 파일을 끌어다 놓을 수 있습니다. 그렇지 않으면 방법 4를 사용하여 폴더 또는 빈 파일을 업로드할 수 없습니다.
    • Linux용 Chrome을 사용하고 HTTPS 프로토콜을 통해 DSM에 로그인한 경우, 신뢰할 수 있는 인증서를 가져와야(제어판 > 보안 > 인증서) 방법 4를 사용하여 업로드할 수 있습니다.

참고:

  1. 최대 1000개의 파일을 업로드 대기열에 추가할 수 있습니다.
  2. Google Chrome, Firefox, Safari 5.1 이상을 사용하거나 File Station으로 로컬 컴퓨터에서 파일 검색이 선택되어 있으면 4GB 이상되는 파일을 업로드할 수 있습니다. 그렇지 않으면 한 번에 2GB까지만 업로드할 수 있습니다.
  3. 암호화되지 않은 공유 폴더에서 파일/폴더 이름은 255자 이내(비 라틴어의 경우 최대 80자)여야 합니다. 암호화된 폴더 내에서는 143자 이내(비 라틴어의 경우 최대 47자)여야 합니다.
  4. 파일/폴더 이름에는 다음 문자가 제외되어야 합니다: \ / : * ? " > |.
  5. 파일/폴더 이름은 "._"으로 시작될 수 없습니다.
  6. JRE를 설치했을 때만 크기가 0KB인 파일을 업로드할 수 있습니다.

문제 해결

Safari를 사용하여 Mac에 저장된 로컬 항목을 적절하게 탐색할 수 없으면 다음과 같이 하십시오:

  1. Mac에서 Safari를 열고 Safari 메뉴에서 기본 설정을 선택합니다.
  2. 보안 탭에서 웹사이트 설정 관리를 클릭합니다.
  3. Java 아래에서 DSM "웹사이트"를 선택합니다.
  4. 안전하지 않은 모드에서 실행를 선택합니다.
