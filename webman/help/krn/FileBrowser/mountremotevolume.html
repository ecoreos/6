원격 폴더 마운트

원격 서버(예: 네트워크 컴퓨터나 다른 Synology NAS 장치)에서 원격 폴더를 마운트하여 File Station을 통해 마운트된 폴더를 탐색 및 관리할 수 있습니다.

원격 CIFS 공유 폴더* 마운트하기:

  1. 도구 > 원격 폴더 마운트 > CIFS 공유 폴더를 클릭합니다.
  2. 폴더 필드에서, \\Remote_Server\Remote_Folder 형식(예: \\192.168.0.2\share) 또는 \\chad\cheeseburger\pictures) 형식으로 원격 폴더 경로를 지정합니다.
  3. 원격 서버에서 원격 폴더에 대한 액세스 권한을 필요로 하면 계정 이름과 패스워드 필드에 원격 서버의 사용자 자격증명을 입력합니다.
  4. 원격 폴더를 마운트하기 위해 DiskStation에서 빈 대상 폴더를 선택하거나 만들려면 찾아보기를 클릭하십시오.
  5. 시스템을 시작하거나 재부팅할 때마다 DiskStation에서 이 원격 폴더를 마운트하도록 하려면 시작 시 자동 마운트를 선택하십시오.
  6. 마운트를 클릭하여 대상 폴더에 원격 폴더가 마운트되도록 합니다. 이제 DiskStation에서 원격 폴더를 탐색 및 관리할 수 있습니다.

원격 NFS 공유 폴더 마운트하기:

  1. 도구 > 원격 폴더 마운트 > NFS 공유 폴더를 클릭합니다.
  2. 폴더 필드에서 Remote_Server:/Remote_Folder_Path 형식으로(예: 192.168.0.2:/volume1/share) 원격 폴더 경로를 지정합니다.
  3. 원하는 NFS 버전과 전송 프로토콜을 선택합니다.
  4. 원격 폴더를 마운트하기 위해 DiskStation에서 빈 대상 폴더를 선택하거나 만들려면 찾아보기를 클릭하십시오.
  5. 시스템을 시작하거나 재부팅할 때마다 DiskStation에서 이 원격 폴더를 마운트하도록 하려면 시작 시 자동 마운트를 선택하십시오.
  6. 마운트를 클릭하여 대상 폴더에 원격 폴더가 마운트되도록 합니다. 이제 DiskStation에서 원격 폴더를 탐색 및 관리할 수 있습니다.

참고:

NFSv4만 TCP 프로토콜을 지원합니다.

원격 폴더의 마운트를 해제하려면 다음과 같이 하십시오:

  1. 마운트 해제할 원격 폴더를 선택한 다음 도구 > 마운트 해제를 클릭합니다.
  2. 또는, 단순히 원격 폴더를 마우스 오른쪽 단추로 클릭하고 마운트 해제를 선택합니다.

참고:

  • 동시에 사용 가능한 원격 폴더의 최대 수는 100입니다.
  • CIFS 프로토콜 또는 NFS 프로토콜을 지원하는 원격 서버의 원격 폴더만 마운트할 수 있습니다.
  • 원결 폴더는 공유 폴더 아래의 하위 폴더에만 마운트할 수 있습니다.
  • 원격 폴더 마운트는 높은 권한 모드에서만 지원됩니다. 이 기능을 사용하려면, 호스트 DSM의 Docker 패키지로 가서, 암호화된 공유 폴더를 활성화할 Docker DSM을 선택하고, 편집을 클릭하고 높은 권한을 사용하여 컨테이너 실행 옆의 상자를 선택합니다.

____
* 일부 모델에서만 가능.
