File Station

File Station은 DSM 사용자가 자신의 웹 브라우저를 사용하여 인터넷에서 DiskStation 폴더에 접근할 수 있게 해주는 간편한 파일 관리 도구입니다. 두 개 이상의 File Station을 실행하고 File Station 간에 데이터를 끌어서 이동하는 방식으로 모든 DiskStation 데이터를 관리할 수 있습니다.
