가상 드라이브 마운트

가상 드라이브를 마운트하여 File Station으로 디스크 이미지의 내용에 액세스할 수 있습니다.

가상 드라이브를 마운트하려면 다음과 같이 하십시오:

  1. 디스크 이미지를 선택하고 도구 > 가상 드라이브 마운트를 클릭합니다.
  2. 가상 드라이브를 마운트하기 위해 DiskStation에서 빈 대상 폴더를 선택하거나 만들려면 찾아보기를 클릭하십시오.
  3. 시스템을 시작하거나 재부팅할 때마다 DiskStation에서 가상 드라이브를 마운트하도록 하려면 시작 시 자동 마운트를 선택하십시오. 마운트를 클릭해서 계속 진행합니다. 이제 가상 드라이브가 마운트된 대상 폴더를 탐색할 수 있고 디스크 이미지 내용에 액세스할 수 있습니다.

참고:

가상 드라이브 및 .iso 파일은 공유 폴더의 하위 폴더에만 마운트할 수 있습니다.

가상 드라이브의 마운트를 해제하려면 다음과 같이 하십시오:

  1. 가상 드라이브가 마운트된 대상 폴더를 마우스 오른쪽 단추로 클릭하고 마운트 해제를 선택합니다.
    참고:
  1. 지원되는 디스크 이미지 형식: ".iso" 파일(ISO 9660 및 UDF 파일 시스템 사용).
  2. 동시에 사용 가능한 가상 드라이브 최대 수는 16입니다.
  3. 가상 드라이브 마운트는 높은 권한 모드에서만 지원됩니다. 이 기능을 사용하려면, 호스트 DSM의 Docker 패키지로 가서, 암호화된 공유 폴더를 활성화할 Docker DSM을 선택하고, 편집을 클릭하고 높은 권한을 사용하여 컨테이너 실행 옆의 상자를 선택합니다.
