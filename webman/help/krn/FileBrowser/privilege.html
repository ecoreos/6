파일/폴더 권한 보기 또는 변경

File Station 을 사용하면 파일 및 하위 폴더에 대한 접근 권한을 설정할 수 있습니다. 이 기능은 다른 사용자 그룹이 공유 폴더에서 서로 다른 하위 폴더에 액세스할 수 있도록 설정할 때 특히 유용합니다.

사용자 및 그룹의 파일/하위 폴더 접근 허용하기:

  1. 접근 권한을 설정할 파일과 폴더를 선택합니다.
  2. 파일 또는 폴더 중 하나를 마우스 오른쪽 단추로 클릭하거나 작업 메뉴를 선택한 다음 속성을 선택합니다.
  3. 소유자 섹션의 소유자 드롭다운 메뉴에서 선택한 파일 및 폴더의 소유자를 선택합니다.
  4. 권한 탭을 클릭하고 사용자 또는 그룹을 선택합니다. 편집을 클릭하여 권한 검사자를 엽니다.
  5. 권한 편집기 창에서 다음을 수행하여 파일 또는 폴더의 ACL 권한을 관리합니다:
    1. 사용자 또는 그룹: 사용자 지정할 권한을 가진 사용자 또는 그룹을 지정합니다.
    2. 상속 대상: 보기만 할 수 있습니다. 여기에서 정보를 보고 권한이 상속되었는지(부모 폴더에서) 또는 명시적인지(없음으로 표시) 여부를 살펴봅니다.
    3. 종류: 허용 또는 거부를 선택하여 사용자 또는 그룹에 대한 권한을 부여하거나 거부합니다.
    4. 적용 대상: 폴더에 대한 권한 항목을 만드는 경우, 이 폴더, 이 폴더에 있는 폴더(또는 자식 폴더)나 파일(또는 자식 파일) 및 이 폴더에 포함된 모든 파일과 폴더(또는 모든 자손)에 항목을 적용하도록 선택 상자를 누릅니다.
    5. 관리: 읽기 권한, 권한 변경 또는 소유권 가져오기를 눌러 해당 항목에 대한 사용자 또는 그룹의 액세스 권한을 지정합니다.
    6. 읽기 또는 쓰기: 이러한 섹션에서 선택 상자를 눌러 파일 또는 폴더에 대한 사용자 및 그룹의 권한 설정을 수정합니다.
  6. 확인을 클릭합니다.

참고: 여기서는 photo 공유 폴더에 있는 파일과 하위 폴더에 대한 접근 권한을 편집할 수 없습니다.
