파일 또는 폴더 압축

  1. 메인 영역에서 DiskStation의 파일 또는 폴더를 선택합니다.
  2. 작업 드롭다운 메뉴에서 "(파일 이름)".zip으로 압축을 선택하여 압축을 시작합니다.
  3. 압축이 완료된 후 zip 파일을 볼 수 없는 경우 작업 드롭다운 메뉴에서 새로 고침을 클릭하면 해당 파일이 표시됩니다.
  4. 아카이브 이름을 변경하고 암호화하거나 다른 압축 수준 또는 모드를 사용하여 보관하려면 대신 아카이브에 추가...를 선택하십시오.
    참고:
  1. File Station을 사용하여 로컬 컴퓨터의 파일 또는 폴더를 압축할 수 없습니다.
  2. 지원되는 아카이브 형식: .zip, .7z.
  3. 아카이브에 추가...에서 아카이브 형식으로 .zip을 선택할 때, .zip 파일에 대해 원하는 코드페이지를 선택할 수 있습니다.
