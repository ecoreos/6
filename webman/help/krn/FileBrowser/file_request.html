파일 요청 생성

파일 요청은 DSM 사용자가 아닌 사용자에게 보낸 파일 업로드 초대입니다. 파일 요청 링크를 사용하면 DSM 사용자가 아닌 사용자가 해당 사용자 계정 및 파일 업로드 권한을 소유할 필요 없이 DiskStation에서 여러 파일을 File Station으로 직접 업로드할 수 있습니다.

파일 요청 링크 생성하기:

  1. File Station으로 이동하여 폴더를 업로드 대상으로 선택합니다.
  2. 작업을 클릭하거나 대상 폴더를 마우스 오른쪽 단추로 클릭한 다음 파일 요청 생성을 선택합니다.
  3. 팝업 창에서 아래 정보를 찾고 수정할 수 있습니다.
    • 파일 경로: 대상 디렉토리 경로를 나열합니다.
    • 링크: 대상 사용자와 공유할 파일 요청 링크를 나열합니다.
    • 귀하의 이름: 요청자 ID를 나타내는 이름을 입력합니다. 기본값은 사용자 이름입니다.
    • 메시지: 대상 사용자에 대한 메시지를 사용자 지정합니다.
    • 패스워드 보호 활성화: 이 옵션을 선택하면 패스워드가 있는 사용자만 생성된 링크를 통해 파일을 업로드할 수 있습니다. 아래 상자에서 패스워드를 지정합니다.
    • 유효 기간: 이 단추를 클릭하면 파일 요청 링크의 사용 기간을 사용자 지정할 수 있습니다.
      • 중지 시간 설정: 이 옵션을 선택하면 파일 업로드를 위해 링크 실행 중지 시점을 날짜 및 시간별로 결정할 수 있습니다.
      • 시작 시간 설정: 이 옵션을 선택하면 파일 업로드를 위해 링크 실행 시작 시점을 날짜 및 시간별로 결정할 수 있습니다.
      • 허용된 액세스 수: 이 옵션을 선택하면 링크에 액세스할 수 있는 횟수를 결정할 수 있습니다.
  4. 저장을 클릭하여 설정을 완료합니다.

파일 요청 링크 공유하기:

파일 요청 URL을 찾으려면 File Station > 도구 > 공유 링크 관리자로 이동하여 원하는 링크를 두 번 클릭합니다. 아래 두 가지 방법 중 하나로 링크를 공유할 수 있습니다.

  • URL을 대상 사용자로 복사합니다.
  • 기본 이메일(DSM 이메일 계정) 또는 메일(타사 이메일 응용 프로그램)을 통해 링크를 보냅니다. 최초 사용자의 경우 마법사의 지침에 따라 사용할 이메일 계정을 설정합니다.

파일 업로드하기:

링크를 통해 파일 업로드 요청을 받으면 다음과 같이 하십시오.

  1. 로컬 컴퓨터 또는 모바일 기기(Android 또는 iOS)에서 지정된 URL을 엽니다. 필요한 경우 패스워드를 입력합니다.
  2. 업로드 페이지에서 ID를 나타내는 이름을 입력합니다.
  3. 업로드할 파일을 끌어다 놓거나 파일 추가를 클릭하여 파일을 선택합니다.
  4. 업로드를 클릭하여 요청을 종료합니다.

참고:

  • 폴더는 파일 요청 링크를 통해 업로드할 수 없습니다.
  • "web" 공유 폴더 내의 폴더는 대상 폴더로 지정할 수 없습니다.
  • 업로드 기능은 웹 브라우저에 따라 달라집니다.
    • Internet Explorer 8 및 Internet Explorer 9에서는 끌어 놓기를 통한 파일 업로드를 지원하지 않습니다.
    • iOS 장치에 설치된 Chrome은 이미지 업로드만 지원합니다.
    • iOS 장치에 설치된 Safari는 이미지 및 비디오 업로드만 지원합니다.
    • Android 장치에 설치된 웹 브라우저는 모든 파일 유형 제한의 영향을 받지 않습니다.

업로드한 파일 찾기:

동일한 사용자가 업로드한 파일은 모두 대상 내의 동일한 폴더에 나타납니다. 기본적으로, 폴더 이름은 Uploader_Name(Requestor_Name)입니다.

예를 들어, Watson이 Holmes에게 파일을 업로드한 경우 Watson (Holmes)이라는 폴더에서 파일을 찾을 수 있습니다.

파일 요청 링크 관리

파일 요청 링크 수정하기:

파일 요청 링크를 생성한 후 File Station > 도구 > 공유 링크 관리자에서 설정을 수정할 수 있습니다.

  1. 원하는 링크를 두 번 클릭하거나 링크를 선택하고 편집을 클릭합니다.
  2. 팝업 창에서 관련 설정을 수정합니다.
  3. 저장을 클릭하여 새로운 설정을 적용합니다.

파일 요청 링크 삭제하기:

  1. File Station > 도구 > 공유 링크 관리자로 이동합니다.
  2. 원하는 링크를 선택하고 삭제를 클릭합니다. 해당 파일 요청 URL이 즉시 실행을 중지합니다.

파일 요청 권한 할당하기:

관리자이거나 administrators 그룹에 속한 사용자는 사용자에게 파일 요청 링크를 생성할 수 있는 권한을 할당할 수 있습니다.

  1. File Station > 설정 > 공유된 링크 > 파일 요청 링크로 이동합니다.
  2. 파일 요청 권한을 허용하려는 사용자 유형을 선택합니다.
  3. 확인을 클릭하여 설정을 저장합니다.
