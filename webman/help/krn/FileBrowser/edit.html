사진 편집

File Station에서 웹 기반 외부 사진 편집기인 Aviary 편집기를 사용해 사진을 편집할 수 있습니다.

Aviary 편집기로 사진 편집하기:

  1. 사진이 위치한 폴더를 찾습니다.
  2. 사진을 마우스 오른쪽 단추로 클릭하고 Aviary에서 편집을 선택합니다.
  3. Aviary 편집기가 새 브라우저 창에 열립니다. 제공된 도구를 사용하여 사진을 보고 편집할 수 있습니다.
  4. 사진 편집을 마치면 상단 도구 표시줄에서 저장 단추를 클릭하여 사진에 대한 모든 변경 사항을 저장합니다.

참고:

  • Aviary 편집기를 사용해 다음 형식의 사진을 편집할 수 있습니다: .jpg, .jpeg, .jpe, .png
  • Aviary 편집기에서 허용되는 최대 파일 크기는 5MB입니다.
  • Aviary 편집기에서는 DiskStation의 네트워크 설정을 제대로 구성해야 합니다. 문제 해결을 위해서는 아래의 설정을 확인하십시오:
    • 제어판 > QuickConnect에서 QuickConnect를 통해 DiskStation에 액세스할 수 있는지 확인하거나 제어판 > 외부 액세스 > DDNS 또는 제어판 > 네트워크 > 네트워크 인터페이스 > PPPoE에서 외부 IP 주소를 검색할 수 있는지 확인합니다.
    • DiskStation이(가) 라우터 안에 있을 경우 제어판 > 외부 액세스 > 라우터 구성에서적절한 포트 전달 규칙을 지정했는지 확인합니다.
    • 그래도 Aviary 편집기를 여는 데 문제가 있을 경우 제어판 > 외부 액세스 > 고급에서 DiskStation의 라우터 포트를 지정해야 할 수도 있습니다.
