디스크 상태

이 페이지에는 high-availability 클러스터 내 디스크 상태를 표시하는 탭이 포함됩니다. 각 서버의 개별 탭에는 다음 정보가 표시됩니다.

  • 개수: 각 디스크에서 사용하는 슬롯이 무엇인지 나타냅니다. 해당 슬롯을 사용하는 활성 서버와 수동 서버의 디스크 용량은 같아야 합니다.
  • 디스크 크기: 각 디스크의 용량을 나타냅니다.
  • 온도: 디스크 온도를 나타냅니다.
  • S.M.A.R.T. 상태: 각 디스크의 S.M.A.R.T. 상태를 나타냅니다.
  • 상태: 디스크 정보와 오류 알림을 표시합니다.

오작동하는 디스크 교체하기:

  1. 디스크 번호를 확인합니다.
  2. 오작동하는 디스크를 새 것으로 교체합니다. 하드 디스크 설치 단계에 대한 자세한 정보는 Synology 제품의 빠른 설치 가이드를 참조하십시오.
  3. 저장소 상태로 이동합니다.
  4. 손상된 볼륨을 선택합니다.
  5. 볼륨을 복구하려면 복구를 클릭합니다.

참고:

교체 하드 디스크의 용량은 오작동하는 하드 디스크와 같거나 커야 합니다.

high-availability 클러스터에 새 디스크 추가하기:

하드 디스크를 high-availability 클러스터에 추가하여 용량을 확장할 수 있습니다. 하드 디스크를 추가하기 전에 아래를 참조하십시오.

  • 활성 서버와 수동 서버가 온라인 상태일 때 디스크를 추가하는 것이 좋습니다.
  • 수동 서버의 하드 디스크 용량은 활성 서버의 디스크와 같거나 커야 합니다.

하드 디스크 추가하기:

  1. 활성 서버와 수동 서버의 해당 슬롯에 하드 디스크를 삽입합니다. 예를 들어 활성 서버 중 하나의 슬롯에 한 개의 하드 디스크를 삽입하면 수동 서버 중 하나의 슬롯에도 해당 하드 디스크를 삽입해야 합니다.
  2. 저장소 관리자 > 볼륨(또는 상황과 모델에 따라 디스크 그룹/iSCSI LUN/RAID Group)으로 갑니다. 확장할 볼륨을 선택하고 관리를 클릭하고, 지침을 따라 하드 디스크를 추가합니다.
  3. 하드 디스크 추가에 성공한 후, High Availability 관리자 > 저장소 상태로 가서 상태가 정상인지 확인합니다.
