Cloud Station을 사용하여 PC/Mac의 데이터 백업

개요

Cloud Station Server는 정교한 크로스 플랫폼 동기화 소프트웨어일 뿐 아니라, 사용자들은 개인 컴퓨터를 더 스마트하게 백업하는 수단으로서 이것을 사용할 수 있습니다. 실시간 백업 솔루션인 Cloud Station Backup은 즉각적이고 대역폭을 덜 소모하며 단일 파일의 기록 버전을 최대 32개까지 보다 효율적으로 보호합니다.

Synology NAS에서 Cloud Station Server 설정

  1. 패키지 센터로 이동하여 Cloud Station Server를 찾은 다음 설치를 클릭합니다.
  2. Cloud Station Server를 실행합니다.
  3. 개요로 이동하여 Cloud Station Backup 아래에 있는 지금 다운로드를 클릭합니다. (Synology 다운로드 센터에서도 Cloud Station Backup을 다운로드할 수 있습니다.)
  4. 컴퓨터에서 설치 관리자를 실행하고 지침을 따라 Cloud Station Backup을 설치합니다.
  5. 설치를 마친 후 컴퓨터에서 Cloud Station Backup을 실행합니다. 지금 시작을 클릭합니다.
  6. Cloud Station Server를 실행 중인 Synology NAS의 주소(또는 QuickConnect ID), 사용자 이름 및 패스워드를 입력합니다. (오른쪽에서 검색 아이콘을 클릭하여 Cloud Station Backup이 LAN 내에서 다른 Synology NAS를 자동으로 검색할 수도 있습니다.) 다음을 클릭합니다.
  7. 백업 소스를 선택하고 동기화하지 않을 하위 폴더의 선택을 취소합니다.
  8. 선택을 클릭하여 Synology NAS의 폴더를 선택합니다. 선택한 대상에 사용자의 컴퓨터 이름을 가진 백업 폴더가 생성됩니다.
  9. 동기화 필터 또는 규칙을 설정하려면 백업 규칙을 클릭합니다.
  10. 다음을 클릭하여 백업 설정을 살펴보고 완료를 클릭합니다.
  11. 주 앱에서 백업 진행 상태를 볼 수 있습니다.

Cloud Station Server에서 동기화된 파일의 이전 버전 다운로드 또는 복원

  1. Cloud Station Server > 버전 기록으로 이동한 다음 복원할 파일 또는 폴더를 찾습니다.
  2. 검색할 파일을 선택하고 작업을 클릭합니다. 이전 버전 찾아보기를 클릭합니다. 파일이 컴퓨터에서 실수로 삭제되거나 제거된 경우 복원을 클릭하여 복원할 수 있습니다.
  3. 다운로드할 파일의 버전을 선택하고 다운로드를 클릭합니다. 또는, 선택한 버전을 복원하려는 것이 확실한 경우, 복원을 클릭하면 현재 버전을 덮어씁니다.

5. 데이터를 추가로 보호

데이터 보호 및 백업에 대한 자세한 내용은 여기를 클릭하여 DSM에서 제공되는 다른 백업 서비스에 관해 상세히 다룬 자습서를 참조하십시오.
