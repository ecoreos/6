로컬 네트워크 내의 Mac 컴퓨터에서 Synology NAS에 파일 저장

개요

Synology NAS는 로컬 네트워크 내에서 파일을 빠르고 간편하게 저장 및 공유하도록 설계되었으므로, 사용자는 매번 DSM에 로그인하는 번거로움 없이 Synology NAS의 공유된 폴더와 파일에 직접 액세스할 수 있습니다. 예를 들어, 다른 네트워크 장치와 마찬가지로 Finder를 사용하여 Synology NAS에 파일을 저장할 수 있습니다.

Mac OS를 사용하여 공유된 폴더에 액세스

  1. Mac 컴퓨터에서 Finder를 엽니다.
  2. 화면의 위쪽을 따라 있는 메뉴 표시줄을 찾습니다.
  3. 가기를 클릭하고 서버에 연결를 선택합니다.
  4. "afp://"로 시작되는 Synology NAS의 IP 주소를 입력합니다(예: "afp://192.168.59.121"). 연결을 클릭해서 계속 진행합니다.
    5. 즐겨찾는 서버: 더하기 기호(+) 단추를 누르면 나중에 사용할 수 있도록 IP 주소가 즐겨찾는 서버로 지정됩니다. 또한, 시계 단추를 클릭하면 최근에 연결한 적이 있는 서버의 목록을 볼 수 있습니다.
  5. DSM 사용자 이름과 패스워드를 입력합니다. 사용자 계정은 액세스할 공유 폴더에 대한 적절한 읽기/쓰기 권한을 가지고 있어야 합니다. 연결을 클릭해서 계속 진행합니다.
  6. 최종적으로, 사용자 이름 자격 증명이 올바르면 이제 Finder 창의 공유 섹션에 Synology NAS가 나열된 것을 볼 수 있습니다. 공유 폴더를 두번 클릭하여 내용을 탐색합니다.
  • 이 항목의 이전: 로컬 네트워크 내의 Windows PC에서 Synology NAS에 파일 저장
