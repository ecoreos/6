로컬 네트워크 내의 Windows PC에서 Synology NAS에 파일 저장

개요

Synology NAS는 로컬 네트워크 내에서 파일을 빠르고 간편하게 저장 및 공유하도록 설계되었으므로, 사용자는 매번 DSM에 로그인하는 번거로움 없이 Synology NAS의 공유된 폴더와 파일에 직접 액세스할 수 있습니다. 예를 들어, 다른 네트워크 장치와 마찬가지로 Windows 탐색기를 사용하여 Synology NAS에 파일을 저장할 수 있습니다.

Windows 탐색기와 매핑

  1. Windows 탐색기 창을 열고 컴퓨터로 이동합니다.
  2. 네트워크 드라이브 연결을 클릭합니다. 이렇게 하면 네트워크 드라이브 연결 창이 표시됩니다.
  3. 네트워크 드라이브 연결 창의 드라이브 드롭다운 메뉴에서 드라이브 문자를 선택합니다.
  4. 폴더 필드에서 Synology NAS의 서버 이름과 공유 폴더 이름을 앞에 백슬래시를 넣고 구분하여 입력합니다. 예를 들어, Synology NAS 장치 이름이 "DiskStation"이고 공유 폴더의 이름이 "Share1"이면, "\\DiskStation\Share1"을 입력해야 합니다. 준비가 되었으면 종료를 클릭합니다.
  5. DSM 사용자 이름과 패스워드를 입력합니다.
    6. 공유 폴더에 액세스할 수 없습니까? 여기에서 입력한 사용자 계정은 액세스하려는 공유 폴더에 대한 접근 권한을 가지고 있어야 합니다.
  6. 마지막으로, Windows 탐색기 창이 표시됩니다. 이제 공유 폴더가 매핑되어 컴퓨터 아래에서 사용할 수 있습니다.
