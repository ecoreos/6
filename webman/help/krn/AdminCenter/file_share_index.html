파일 색인

공유 폴더 또는 해당 하위 폴더에서 색인을 지정하면 파일 검색 중 포함된 파일과 내용을 손쉽게 찾는 데 유용합니다.

참고:

DiskStation에 마운트된 원격 폴더, 클라우드 폴더 또는 .iso 폴더는 색인을 지정할 수 없습니다.

폴더 및 파일 내용 색인하기:

폴더를 색인한 경우, 그러한 폴더 및 그 안에 포함된 파일들은 파일 이름별로 자동으로 색인됩니다. 색인된 폴더 내에서 파일 유형별로 파일 내용/메타데이터를 추가로 색인할 수 있습니다. 파일 내용 색인은 색인된 파일 내용/메타데이터를 빠르고 심도있게 검색할 수 있게 합니다.

  1. 제어판 > 색인 서비스 > 파일 색인으로 가서 색인된 폴더 목록을 클릭합니다.
  2. 팝업 창에서, 만들기 > 선택을 클릭하여 색인할 준비가 된 폴더를 표시합니다.
  3. 원하는 폴더를 선택하고 선택을 클릭합니다. 색인을 위한 새 폴더를 만들려면, 폴더 생성을 클릭하고 진행합니다.
  4. 색인하기를 원하는 폴더를 선택하고 선택을 클릭합니다.
  5. 특정 유형의 파일 내용을 색인하려면, 파일 유형으로 가서 색인을 위한 이미지, 오디오, 비디오 또는 문서를 선택합니다.
  6. 확인을 클릭하여 색인을 마칩니다.

참고:

  • 색인된 폴더의 최대 수는 1000개 입니다.
  • File Station에서 파일 내용 검색 활성화를 선택하면, 색인된 폴더 내의 파일 내용을 검색할 수 있습니다.
  • 색인된 폴더가 있는 볼륨에는 10 MB 이상의 사용 가능한 공간이 있어야 합니다. 그렇지 않으면 색인이 일시 중지됩니다. 색인을 재개하려면, 볼륨을 확장하고 색인 재설정을 클릭합니다.
  • 폴더 및 해당 하위 폴더를 동시에 색인할 경우, 시스템은 하위 폴더만을 색인하여 시스템 리소소를 절약합니다.
  • 폴더를 완전히 색인하고 SMB 또는 AFP 프로토콜을 통해 Mac 컴퓨터에 마운트한 후, Mac 사용자는 Finder의 Spotlight를 사용하여 파일 내용을 검색할 수 있습니다.

색인된 폴더 삭제하기:

  1. 제어판 > 색인 서비스 > 파일 색인 > 색인된 폴더 목록으로 가서 원하는 색인된 폴더를 선택합니다.
  2. 삭제를 클릭하여 폴더에 대한 색인을 중지합니다.

파일 다시 색인

일반적인 상황에서는 파일을 다시 색인할 필요가 없습니다. 그러나, 폴더 내의 파일이 특정 검색 기준에 일치하지만 어떤 이유로 파일 검색 중 누락된 것으로 나타나는 경우, 파일을 다시 색인하여 색인 구조를 다시 구성해야 할 수도 있습니다.

공유 폴더의 파일을 다시 색인하기:

  1. 제어판 > 색인 서비스 > 파일 색인 > 색인된 폴더 목록으로 갑니다.
  2. 나열된 폴더 중 다시 색인할 폴더를 선택합니다.
  3. 색인 재설정을 클릭합니다.

참고:

File Station은 원격 폴더 및 외부 저장소 장치(예: USB 드라이브/SD 카드)에 대해서는 파일 색인을 지원하지 않습니다.

데스크톱을 통해 파일 색인 관리

파일 색인 작업이 시작될 때, 작업 표시줄에서 색인 진행 (부유식 돋보기 아이콘)을 볼 수 있습니다. 색인에 더 많은 CPU 리소스가 필요하므로, 이를 통해 CPU 리소스를 절약할 수 있습니다.

  • 색인을 일시 중지/다시 시작하려면 재생 단추를 클릭합니다.
  • 적절한 시간까지 파일 색인을 지연시키려면, 드롭다운 메뉴에서 원하는 옵션을 선택합니다.
  • 색인 중인 폴더를 보려면 색인된 폴더 목록 보기를 클릭합니다.

참고:

  • 색인 작업을 일시 중지하는 경우, 바탕화면 작업 표시줄에서 재생 단추를 클릭해야만 색인 작업을 다시 시작할 수 있습니다.
