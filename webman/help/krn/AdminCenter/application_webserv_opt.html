HTTP 서비스

기본 포트 번호 80 이외에 Photo Station 및 Web Station을 사용할 수 있도록 다른 포트를 추가할 수 있습니다.

HTTP 접근을 위해 포트 번호 추가하기:

  1. HTTP 포트 추가를 선택합니다.
  2. 필드에 포트 번호를 입력합니다. 1~65535 중에서 HTTP 접근을 위한 포트 번호를 추가할 수 있습니다.

다음 포트는 제외되어야 합니다.

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule 기본 포트: 4662 (TCP), 4672(UDP) 및 FTP 기본 포트 범위: 실제 범위는 모델에 따라 다를 수 있습니다.

    참고:
  1. HTTP 접근을 위한 포트 번호는 FTP 서비스를 위한 포트 번호와 동일할 수 없습니다.
  2. HTTP 포트와 HTTPS 포트는 서로 충돌할 수 없습니다.
