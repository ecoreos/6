자동 차단

자동 차단 기능을 사용하면 승인되지 않은 로그인을 차단할 수 있습니다. 서비스를 활성화한 후 로그인 시도 실패 횟수가 너무 많으면 IP 주소가 차단됩니다. SSH, Telnet, rsync, 네트워크 백업, 공유 폴더 동기화, FTP, WebDAV, Synology 모바일 응용 프로그램, File Station 또는 DSM 를 통해 실패한 로그인 시도가 모두 누적됩니다. 차단되지 않을 IP 주소를 포함하는 허용 목록을 가질 수도 있습니다.

자동 차단 설정하기:

  1. 일반 탭으로 이동한 다음 자동 차단 활성화를 선택합니다.
  2. 지정된 시간(분) 이내에 실패한 로그인 시도 횟수에 도달했을 때 IP 주소를 차단하려면 로그인 시도 횟수와 다음 시간 이내(분)에 숫자를 입력합니다.
  3. 지정된 일 수 이후 차단된 IP 주소를 제거하려면 차단 만료일 활성화를 선택하고 숫자를 입력합니다. 이 옵션을 선택하지 않았거나 다음 일 수 이후 차단 해제를 0으로 설정한 경우 목록에서 직접 제거하기 전까지 차단 목록에 차단된 IP 주소가 무기한으로 남아 있습니다.
  4. 적용을 클릭합니다.

차단된 IP 주소 추가하기:

차단 목록 탭을 클릭하고 생성 드롭다운 메뉴에서 다음 중 하나를 선택합니다:

  • IP 주소 추가

    • IP 주소를 입력하고 만료 시간을 설정합니다.

  • IP 주소 목록 가져오기

    • IP 주소가 나열된 텍스트 파일을 가져와서 여러 IP 주소를 추가하고 한꺼번에 만료 시간을 설정합니다. 기존 IP 주소를 덮어쓰려면 적절한 선택 상자를 선택합니다. 이 옵션을 선택하지 않으면 중복 IP 주소를 건너뜁니다.

    파일을 가져오려면 파일이 다음 조건을 충족해야 합니다:

    • 파일이 일반 텍스트 파일이어야 합니다.
    • 파일의 각 행에 하나의 IP 주소만 포함합니다.
    • 주석 행은 #(파운드 기호)로 시작됩니다.

      • 선택한 파일의 형식이 올바르면 미리보기 영역에서 모든 IP 주소를 확인할 수 있습니다. 확인을 클릭하여 가져옵니다.

차단된 IP 주소 제거하기:

  1. 차단된 IP 주소 목록을 보려면 차단 목록 탭을 클릭합니다.
  2. 목록에서 제거하려는 IP 주소를 선택하고 제거를 클릭합니다.

허용된 IP 주소 추가하기:

허용 목록 탭을 클릭하고 생성 드롭다운 메뉴에서 다음 중 하나를 선택합니다:

  • IP 주소 추가

    • IP 주소를 입력합니다.

  • IP 주소 목록 가져오기

    • IP 주소가 나열된 텍스트 파일을 가져와서 한꺼번에 여러 IP 주소를 추가합니다. 기존 IP 주소를 덮어쓰려면 적절한 선택 상자를 선택합니다. 이 옵션을 선택하지 않으면 중복 IP 주소를 건너뜁니다.

    파일을 가져오려면 파일이 다음 조건을 충족해야 합니다:

    • 파일이 일반 텍스트 파일이어야 합니다.
    • 파일의 각 행에 하나의 IP 주소만 포함합니다.
    • 주석 행은 #(파운드 기호)로 시작됩니다.

      • 선택한 파일의 형식이 올바르면 미리보기 영역에서 모든 IP 주소를 확인할 수 있습니다. 확인을 클릭하여 가져옵니다.

허용된 IP 주소 제거하기:

  1. 허용된 IP 주소 목록을 보려면 허용 목록 탭을 클릭합니다.
  2. 목록에서 제거하려는 IP 주소를 선택하고 제거를 클릭합니다.
