HDD 대기 기능

HDD 대기 기능 모드로 들어가기 전에 하드 디스크가 유휴 상태로 남아 있는 기간을 지정할 수 있습니다. HDD 대기 기능 동안 하드 디스크는 회전을 멈추고, 전력 소비를 줄이고 하드 디스크의 수명을 연장합니다. 내부 디스크와 외부 eSATA/USB 하드 디스크에 대해 서로 다르게 기간을 지정할 수 있습니다.

참고:

  • 액세스하면 HDD 대기 기능이 중지됩니다.
  • 특정 외부 USB 저장 장치만 HDD 대기 기능을 지원합니다.

DiskStation이(가) 대기하지 않는 이유가 무엇입니까?

아래 서비스가 DiskStation을(를) 대기 기능 모드로 들어가지 못하게 한다는 점에 유의하십시오. 또한, 시스템은 예약된 작업 2분 전에 대기 기능 모드를 종료합니다.

패키지:

  • Download Station: Download Station이 eMule, RSS 예약 다운로드 또는 활성 작업을 실행 중이면 DiskStation이(가) 대기 상태로 들어갈 수 없습니다.
  • Mail Station: Mail Station이 POP3에서 페치되는 경우 DiskStation이(가) 대기 상태로 들어갈 수 없습니다.
  • Surveillance Station: Surveillance Station이 녹화 중이면 DiskStation이(가) 대기 상태로 들어갈 수 없습니다.
  • 미디어 서버: 미디어 서버 설정에서 DMA 로그가 활성화된 경우 DiskStation이(가) 대기 상태로 들어갈 수 없습니다(메인 메뉴 > 미디어 서버 > 미디어 서버 > 진단으로 이동).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • SABnzbd 같은 기타 타사 소프트웨어 또는 옵트웨어

프로세스:

  • 섬네일 및 색인 지정: 섬네일을 생성 중이거나 멀티미디어 파일 포스트-업그레이드의 색인을 재지정하는 동안 DiskStation이(가) 대기 상태로 들어갈 수 없습니다. 이 작업 중에는 "synoindexd" 프로세스가 "실행 중"으로 나열됩니다(메인 메뉴 > 리소스 모니터 > 프로세스로 이동).
  • Local Master Browser: Local Master Browser가 활성화되어 있으면 DiskStation이(가) 대기 상태로 들어갈 수 없습니다(제어판 > Win/Mac/NFS > Windows 파일 서비스 > Local Master Browser 활성화로 이동).
  • Windows Media Player: Windows Media Player의 네트워크 공유 서비스가 LAN에서 활성화되어 있으면 DiskStation이(가) 대기 상태로 들어갈 수 없습니다.
  • 여러 개의 동시 프로세스가 스왑을 요청하는 경우 DiskStation이(가) 대기 상태로 들어갈 수 없습니다. 이 때 RAM 성능이 초과되면 HDD가 읽기/쓰기 작업에 일시적으로 사용됩니다.

기타

  • USB 장치: USB 장치가 연결되어 있으면 DiskStation이(가) 시스템 대기 기능 상태로 들어갈 수 없습니다.

깊은 수면

페어링된 DiskStation 인터페이스를 통해 확장 유닛에 대한 깊은 수면을 활성화할 수 있습니다. 확장 유닛과 페어링된 DiskStation에 대해 서로 다른 기간에 유휴하도록 구성할 수 있습니다. 따라서, 페어링된 DiskStation이(가) 온라인 상태로 남아 있는 동안 확장 유닛이 깊은 수면에 들어가도록 설정할 수 있습니다.

깊은 수면 모드 동안:

  • 확장 유닛의 모든 LED가 꺼집니다.
  • 확장 유닛의 하드 디스크에 대한 전원이 차단됩니다.
  • 확장 유닛의 팬이 중지됩니다.

참고:

  • 액세스하면 깊은 수면이 중지됩니다.
  • 확장 유닛에서 깊은 수면 기능을 사용할 때 정상적인 작동을 보장하기 위해서는 후면 패널에 있는 Auto/Manual 스위치를 Auto으로 밀어야 합니다.

자동 전원 끄기

DiskStation의 전원이 자동으로 차단되기 전에 내부 디스크가 대기 기능 상태를 유지하는 시간을 지정할 수 있습니다.

참고:

  • 먼저 내부 HDD 대기 기능을 활성화해야 합니다.
  • DiskStation을(를) 원격으로 켤 수 있도록 먼저 WOL 서비스를 활성화합니다.
