PPPoE

DSL 또는 케이블 모뎀을 이용해서 인터넷에 연결할 경우 인터넷 서비스 제공업체(ISP)가 PPPoE(Point to Point Protocol over Ethernet)를 이용하면, 제어판 > 네트워크 > 네트워크 인터페이스에서 PPPoE를 활성화시켜서 계정 정보를 입력하고 라우터 없이 바로 인터넷에 연결할 수 있습니다.

자세한 PPPoE 정보는 ISP나 네트워크 관리자에게 문의해 주십시오.

IPv6 설정

DiskStation은(는) IPv4/IPv6 이중 스택 구성표를 지원합니다. 즉, DiskStation에서 IPv4와 IPv6를 동시에 작동할 수 있습니다. IPv6의 기본 설정은 자동입니다. 기존 네트워크 인터페이스를 선택하고, 편집 단추를 클릭하고, IPv6 탭으로 이동하여 설정을 변경합니다.

IPv6 전환 매커니즘은 IPv6가 IPv4를 완전히 대체할 때까지 IPv4 주소를 가진 호스트가 IPv6 서비스에 도달하도록 하는데 필요합니다. 여러 IPv6 전환 매커니즘 중에서 선택합니다:

  • 6in4: 터널 브로커를 통해 IPv6 네트워크에 연결합니다. IPv6 주소, 접두어 길이, 접두어 및 터널 브로커 IPv4 주소를 채워야 합니다. 이것은 공용 IPv4 주소를 요구합니다.
  • 6to4: 임의 캐스트를 사용하는 터널 브로커를 통해 IPv6 네트워크에 연결합니다. 이것은 공용 IPv4 주소를 요구합니다.
  • DHCP-PD: LAN 내에서 접두어 위임을 통해 DHCP 서버로부터 IPv6 접두어를 얻습니다. 이것은 ISP 지원을 요구하며 라우터 모드에서만 작동합니다.
