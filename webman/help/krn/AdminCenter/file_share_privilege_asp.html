고급 공유 폴더 권한 관리

제어판 > 공유 폴더 > 편집 > 고급에 있는 페이지에는 공유 폴더의 액세스 권한을 좀더 미세 조정할 수 있는 다음과 같은 옵션이 포함되어 있습니다.

고급 공유 권한

고급 공유 권한은 공유 폴더의 액세스 권한을 관리하도록 추가 컨트롤 레이어를 제공합니다. 활성화되면, 사용자 또는 그룹에게 고급 공유 권한 및 Windows ACL 권한(공유 폴더 > 편집 > 권한에 위치)이 둘다 부여된 경우에만 해당 사용자 및 그룹이 공유 폴더의 내용을 보거나 수정할 수 있습니다.

고급 공유 권한 활성화하기:

  1. 고급 공유 권한 활성화 상자를 선택합니다.
  2. 고급 공유 권한을 클릭하여 공유 폴더의 고급 공유 권한을 수정합니다.

참고:

  • 사용자가 다음 파일 서비스를 통해 공유 폴더에 액세스할 때 고급 공유 권한이 적용됩니다: Windows 파일 공유, Apple 파일 공유, File Station, FTP 및 WebDAV.

고급 설정

필요하면, File Station, FTP 또는 WebDAV를 통해 공유 폴더에 액세스할 때 사용자에게 추가적인 제한을 적용할 수 있습니다.

  • 디렉터리 검색 비활성화: 이 옵션을 활성화하면 사용자가 공유 폴더의 내용을 볼 수 없도록 제한합니다.
  • 기존 파일 수정 비활성화: 이 옵션을 활성화하면 사용자가 공유 폴더의 파일을 이동, 삭제 또는 수정할 수 없도록 제한합니다. 사용자는 여전히 공유 폴더의 내용을 보고, 다운로드/업로드, 복사 또는 압축 풀기 등을 수행할 수 있습니다.
  • 파일 다운로드 비활성화: 이 옵션을 활성화하면 사용자가 공유 폴더의 내용을 다운로드할 수 없도록 제한합니다.

참고:

위 제한은 administrators 그룹에 속하는 사용자에게는 적용되지 않습니다.
