라우터 포트

인터넷 서비스(예: Google Docs 또는 Pixlr)가 DiskStation에 액세스하도록 하려면 제어판 > 외부 액세스 > 고급에서 공용 외부 호스트 이름과 포트 번호를 지정할 수 있습니다.

기본적으로, 호스트 이름은 공용 IP 주소(또는 제어판 > 외부 액세스 > DDNS에 설정한 호스트 이름)이며, 포트 번호는 5000(HTTPS 연결의 경우 5001)입니다. 그러나 DDNS 페이지에 구성되지 않은 다른 호스트 이름이 있거나, 포트 전달 규칙을 설정할 때 라우터에 다른 포트 번호를 지정했을 경우, 공유한 파일을 인터넷 서비스에서 성공적으로 액세스할 수 있도록 보장하기 위해 호스트 이름 또는 라우터 포트 번호를 입력해야 합니다.

라우터 포트 구성 활성화하기:

  1. 호스트 이름 또는 정적 IP 필드에 DiskStation의 호스트 이름 또는 공용 IP 주소를 입력합니다.
  2. 라우터에서 DSM의 포트 전달 규칙을 설정할 때 DSM(HTTP) 또는 DSM(HTTPS) 필드에 지정한 라우터 포트 번호를 입력합니다.
  3. 적용을 클릭합니다.
