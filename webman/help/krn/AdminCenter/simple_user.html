사용자

유연한 사용자 관리 옵션을 사용하여 가족 또는 회사의 각 구성원에 대해 사용자 계정을 생성하고 각자에 대해 공유 폴더 액세스 권한을 정의할 수 있습니다.

사용자 생성하기:

  1. 제어판(단순 모드) > 사용자로 이동하고 왼쪽 패널에서 + 아이콘을 클릭합니다.
  2. 사용자 이름과 패스워드를 지정합니다.
  3. 새 사용자가 속할 그룹을 지정합니다.
  4. 적절한 공유 폴더 권한을 새 사용자에게 할당합니다. 권한 충돌이 발생한 경우 권한 우선 순위는 다음과 같습니다: 접근 금지 > 읽기/쓰기 > 읽기 전용.
    • 미리보기: 이 열은 사용자가 속한 그룹과 현재 설정에 따라 해당 사용자의 실제 공유 폴더 접근 권한을 표시합니다.
    • 읽기/쓰기: 사용자가 해당 공유 폴더에 있는 파일과 하위 폴더에 접근하여 변경 내용을 적용할 수 있습니다.
    • 읽기 전용: 사용자가 해당 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 있습니다.
    • 접근 금지: 사용자가 해당 공유 폴더에 있는 파일 또는 하위 폴더에 접근할 수 없습니다.
  5. 적용을 클릭하여 설정을 저장합니다.

    6. 사용자 이름 제한:

    • 사용자 이름은 대소문자를 구분하지 않으며, 1~64자의 Unicode 문자를 포함하지만 다음 기호는 포함되지 않습니다:
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • 첫번째 문자는 뺄셈 기호 또는 공백 문자를 사용할 수 없으며, 마지막 문자는 공백 문자를 사용할 수 없습니다.
    • 다음 사용자 이름은 시스템 용도로 예약되어 있습니다:
      mailer-daemon, postmaster

    패스워드 제한:

    • 패스워드는 대소문자를 구분하며 문자, 숫자, 기호, 공백 문자를 포함, 표시 가능한 127자까지 입력 가능합니다.

사용자 편집하기:

  1. 제어판(단순 모드) > 사용자로 이동합니다.
  2. 왼쪽 패널에서 원하는 사용자를 선택합니다.
  3. 이름, 패스워드, 그룹 및 공유 폴더 권한 같은 사용자의 일반 정보를 편집합니다.
  4. 계정의 기능을 중지하려면 이 계정 비활성화를 선택합니다.
  5. 적용을 클릭하여 설정을 저장합니다.

사용자 제거하기:

  1. 제어판(단순 모드) > 사용자로 이동합니다.
  2. 왼쪽 패널에서 원하는 사용자를 선택합니다.
  3. 왼쪽 패널에 있는 휴지통 아이콘을 클릭하여 선택한 사용자를 삭제합니다.
