공유 폴더

공유 폴더는 DiskStation에 파일과 폴더를 저장할 수 있는 기본 디렉토리입니다. DiskStation에 파일을 저장하기 전에 최소 하나의 공유 폴더를 생성해야 합니다. 공유 폴더에 저장된 데이터는 사용자 지정된 액세스 권한에 따라 특정 사용자 또는 사용자 그룹과 공유하거나 개인적으로 보유할 수 있습니다. 추가로, 공유 폴더는 생성 과정에 암호화하여 보호 레이어를 추가할 수 있습니다.

기본 공유 폴더

아래는 특정 응용 프로그램, 서비스 또는 패키지를 설치하거나 활성화할 때 자동으로 생성되는 공유 폴더 목록입니다.

이름 설명
public public 폴더는 처음 DiskStation을(를) 설치하면 자동으로 생성됩니다.*
web web 폴더는 DiskStation에 호스팅된 웹사이트의 콘텐츠를 위한 기본 저장소 위치입니다. 이 폴더는 Web Station이 활성화되면 자동으로 생성됩니다.
photo photo 폴더는 Photo Station과 공유할 사진 및 비디오를 저장하는 데 사용됩니다. 이 폴더는 Photo Station 또는 DLNA/UPnP 미디어 서버가 활성화되면 자동으로 생성됩니다.
music music 폴더는 Audio Station을 사용하여 들을 음악을 저장하는 데 사용됩니다. 이 폴더는 Audio Station이 활성화되면 자동으로 생성됩니다.
video video 폴더는 DLNA/UPnP 장치에서 탐색할 비디오 파일을 저장하는 데 사용됩니다. 이 폴더는 미디어 서버가 활성화되면 자동으로 생성됩니다.
surveillance surveillance 폴더는 Surveillance Station 기록을 저장하는 데 사용됩니다. 이 폴더는 Surveillance Station이 활성화되면 자동으로 생성됩니다. 이 폴더는 읽기 전용이며 시스템 관리자만 접근할 수 있습니다.
home home 폴더는 각 사용자가 데이터를 저장하는 개인 공간을 제공하며, 이 공간은 해당 사용자만 액세스할 수 있습니다. 이 폴더는 사용자 홈이 활성화되면 자동으로 생성됩니다.
homes homes 폴더에는 모든 사용자의 홈 폴더가 들어 있습니다. 이 폴더는 사용자 홈이 활성화되면 자동으로 생성됩니다. 시스템 관리자에 한해 모든 사용자의 home 폴더에 접근하고 관리할 수 있습니다.
NetBackup NetBackup 폴더는 네트워크 백업이 활성화되면 자동으로 생성됩니다.
usbshare usbshare[number] 폴더는 DiskStation의 USB 포트에 USB 외부 드라이브 장치가 연결되면 자동으로 생성됩니다.*
satashare satashare 폴더는 DiskStation의 eSATA 포트에 eSATA 외부 드라이브 장치가 연결되면 자동으로 생성됩니다.*

____
* 특정 모델에서만 사용할 수 있습니다.
