이메일

이메일 알림을 활성화하면 시스템 오류, 네트워크 연결 손실, 팬 동작, 정전 또는 백업 작업 실패 등, 지정된 이벤트가 발생할 때 DiskStation이(가) 사용자의 이메일로 메시지를 보내도록 할 수 있습니다.

이메일 알림을 활성화하면 시스템 오류, 네트워크 연결 손실, 전원 오류 또는 백업 작업 실패 등, 지정된 이벤트가 발생할 때 DiskStation이(가) 사용자의 이메일로 메시지를 보내도록 할 수 있습니다.

이메일 알림 활성화하기:

  1. 제어판 > 알림 > 이메일로 이동합니다.
  2. 이메일 알림 활성화를 선택합니다.
  3. 수신자 필드에 하나 이상의 이메일 주소를 입력합니다. 시스템 알림이 여기에 입력한 주소로 전달됩니다. 두 개의 주소가 허용됩니다. 여러 개의 주소를 세미콜론(;)으로 구분할 수 있습니다.
  4. 필요하면, 제목 접두어 필드에 제목 접두어를 추가합니다. 이 접두어는 DiskStation에서 전송하는 각 메시지의 제목에 추가되어 메시지를 식별하고 필터링하는 데 도움을 줍니다.
  5. 타사 이메일 서버(예: Gmail, Outlook, Yahoo!)를 사용하여 메시지를 전달하기를 원하는 경우, 아래 "타사 이메일 서비스" 섹션을 참조하십시오. 사용자 지정 서버를 사용하기를 원하는 경우에는 "사용자 지정 SMTP 서버" 섹션을 참조하십시오.

타사 이메일 서비스 사용하기:

  1. 서비스 제공업체 메뉴에서 Gmail, Yahoo, Outlook 같은 타사 이메일 서비스를 선택합니다.
  2. 선택한 이메일 서비스의 사용자 이름과 패스워드를 입력합니다.
  3. 적용을 클릭하여 설정을 저장합니다.
  4. 테스트 전자 메일 보내기를 클릭하여 설정이 올바른지 확인할 수 있습니다.

사용자 지정 SMTP 서버 사용하기:

  1. 서비스 제공업체 메뉴에서 사용자 지정 SMTP 서버를 선택합니다.
  2. SMTP 서버 필드에 SMTP 서버 이름 또는 IP 주소를 입력합니다.
  3. SMTP 포트 필드에 SMTP 포트 번호를 입력합니다. (포트 전달 규칙을 설정하는 과정에서 SMTP 포트를 다르게 지정한 경우 그 번호를 입력하십시오.)
  4. SMTP 서버에서 인증을 요구하는 경우, 인증 필요를 선택하고 이메일 서버의 사용자 이름과 패스워드를 입력합니다.
  5. SMTP 서버에 SSL/TLS 연결이 필요한 경우 보안 연결(SSL/TLS)이 필요합니다를 선택합니다.
  6. DiskStation에서 전송하는 이메일 메시지에 표시되는 보낸 사람의 이름을 사용자 지정하려면 보낸 사람 이름 필드에 이름을 입력합니다. 보내는 사람의 이메일 주소를 사용자 지정하려면 보낸 사람 이메일 필드에 이메일 주소를 입력합니다. 필드가 비어 있으면 수신자 필드에 있는 첫 번째 이메일 주소가 사용됩니다.
  7. 적용을 클릭하여 설정을 저장합니다.
  8. 테스트 전자 메일 보내기를 클릭하여 설정이 올바른지 확인할 수 있습니다.

참고:

  • 새로 생성된 각 사용자 계정에 환영 메시지를 보내고 싶으면 새 사용자에게 환영 메시지 보내기를 선택합니다. 그러면 각 사용자의 계정이 만들어질 때 사용자가 환영 메시지를 받게 됩니다.
  • 이메일 알림의 언어는 알림 언어에서 설정할 수 있습니다. 이메일 알림 언어를 변경하려면 메인 메뉴 > 제어판 > 지역 옵션으로 가서 언어 탭을 클릭하고, 알림 언어 섹션의 드롭다운 메뉴에서 원하는 언어를 선택합니다.
  • Gmail의 SMTP 서버를 사용하고 있고 2단계 검증을 활성화한 경우, SMTP 인증을 위해 Google에서 제공한 "응용 프로그램 고유의 패스워드"를 반드시 사용하십시오.

SMTP 서버 도메인 이름:

SMTP 서버는 IP 주소 또는 도메인 이름일 수 있습니다. 도메인 이름일 경우 올바른 DNS 서버 IP가 메인 메뉴 > 제어판 > 네트워크에 입력되었으며 서버가 인터넷 상에 있는지 확인합니다.
