그룹 만들기

  1. 제어판 > 그룹으로 이동합니다.
  2. 생성을 클릭합니다. 그러면 사용자 생성 마법사가 시작됩니다.
  3. 그룹 이름과 설명을 입력합니다. (자세한 정보는 아래의 섹션을 참조하십시오). 다음을 클릭합니다.
  4. 마법사의 나머지 단계를 따라 그룹 생성을 마칩니다.
  5. 적용을 클릭해서 종료합니다.

명명 규칙:

  • 그룹 이름은 대소문자를 구분하며 다음 기호를 제외한 1~32자의 Unicode 문자로 구성되어야 합니다:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • 첫번째 문자는 뺄셈 기호 또는 공백 문자를 사용할 수 없으며, 마지막 문자는 공백 문자를 사용할 수 없습니다.
  • 그룹 설명은 대소문자를 구분하며 최대 64자의 표시 가능한 Unicode 문자로 한정됩니다.
