푸시 서비스

푸시 서비스를 통해 DiskStation이(가) 특정 이벤트나 오류 발생 시 알림을 모바일 장치 또는 웹 브라우저에 전송할 수 있습니다.

포트:

푸시 서비스를 활성화하기 전에, DiskStation이(가) 포트 8089를 사용하여 인터넷에 액세스할 수 있는지 확인하십시오.

Synology에서 제공하는 이메일 알림 활성화하기:

  1. Synology의 이메일 서버를 통해 시스템 상태에 관한 알림을 보냅니다를 선택합니다.
  2. 수신자 필드에 하나 이상의 이메일 주소를 입력합니다. 시스템 알림이 여기에 입력한 주소로 전달됩니다. 두 개의 주소가 허용됩니다. 여러 주소를 세미콜론(;)으로 구분합니다.
  3. 적용을 클릭하면 수신자 필드에 지정된 주소로 확인 메일이 전달됩니다. 확인 메일에 있는 단계를 따르십시오.
  4. 검증을 마친 후, 테스트 메시지 전송을 클릭한 다음 확인을 클릭하여 테스트 알림을 보낼 수 있습니다.

모바일 기기 알림 활성화하기:

  1. 제어판 > 알림 > 푸시 서비스로 이동합니다.
  2. 모바일 기기 알림 활성화 상자를 선택한 다음 적용을 클릭합니다.
  3. 모바일 기기에 DS finder를 설치하고 실행한 다음 DiskStation을(를) 즐겨찾기 목록에 추가하여 모바일 장치와 DiskStation을(를) 페어링합니다.
  4. 테스트 메시지 전송을 클릭한 다음 확인을 클릭하여 테스트 알림을 보냅니다. 그런 다음 모바일 기기에서 DS finder를 실행하여 테스트 메시지가 수신되는지 여부를 확인합니다.

페어링된 모바일 장치 관리하기:

  1. 페어링된 장치 관리를 클릭하여 페어링된 모든 모바일 기기 목록을 살펴봅니다.
  2. 페어링을 해제하고 싶은 장치가 있으면 이를 선택하고 페어링 해제를 클릭합니다. (여러 항목을 선택하려면 Shift 또는 Ctrl 키를 누른 상태에서 선택하십시오.) 다시 페어링할 때까지 그 장치에는 알림이 전송되지 않습니다.

브라우저 알림 활성화하기:

  1. 모바일 기기 알림 활성화를 누른 다음 적용을 클릭하여 이 기능을 활성화합니다.
  2. 브라우저 알림 활성화를 클릭하여 DiskStation을(를) 브라우저와 페어링합니다.
  3. 나타나는 창에서 다음을 수행하십시오:
    • Chrome: 이미 설치하지 않은 경우 지금 설치를 클릭하여 Chrome 실행 파일을 설치한 다음 Google 계정에 로그인합니다.
    • Safari: 허용을 클릭하여 알림 서비스를 인증합니다.
  4. 다시 브라우저 알림 페이지로 리디렉션되면 페어링된 장치 탭을 클릭하여 페어링된 장치의 정보를 확인하거나 브라우저에서 DiskStation의 페어링을 해제합니다. 또는 이벤트 탭을 클릭하여 지난 7일 동안의 브라우저 알림 기록을 확인합니다.

참고:

브라우저 알림 서비스는 Chrome(35 이상) 및 Safari(OS X v10.9 이상)에서 지원됩니다.
