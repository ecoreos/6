핫스폿*

Wi-Fi USB 어댑터를 설치한 경우 또는 DiskStation에 Wi-Fi가 내장되어 있는 경우, DiskStation 장치는 인터넷 연결을 다른 장치와 무선으로 공유할 수 있는 핫스폿 만들기를 지원합니다. 인터넷 사용을 원하는 방식에 따라 DiskStation을(를) 무선 액세스 포인트(AP)나 무선 라우터로 설정할 수 있습니다.

핫스폿 모드 설명
무선 AP
  • LAN 및 Wi-Fi 네트워크를 함께 브리지합니다.
  • LAN 및 무선 클라이언트가 동일한 로컬 네트워크에 있게 됩니다.
  • 이미 라우터를 가지고 있고 DiskStation을(를) 기존 네트워크에 추가적인 액세스 포인트로 사용하려는 경우에 적용할 수 있습니다.
  • DLNA 장치, 프린터, Synology Web Assistant, 네트워크 브로드캐스트에 의존하는 기타 서비스 같은 네트워크 리소스를 공유할 수 있습니다.
  • 무선 AP 역할을 하는 DiskStation 장치에서 DHCP 서버는 비활성화됩니다. DiskStation에 대해 DHCP 서버를 다시 활성화하려면, Wi-Fi 동글을 빼고 DiskStation을(를) 다시 시작합니다.
  • LAN 포트가 여러 개인 모델의 경우, LAN 1만이 무선 AP와 브리지하는 데 사용됩니다.
무선 라우터
  • 장치가 비공개 서브넷에 위치하는 무선 네트워크를 설정합니다.

참고:

  1. 일부 모델은 무선 동글 연결을 통해 핫스폿을 생성해야 합니다. 지원되는 전체 무선 동글 목록은 www.synology.com을 참조하십시오.
  2. DS213air 이외 모델의 경우, DSM 4.3에 있던 무선 핫스폿 옵션이 DSM 5.0 이상에서는 "무선 라우터"로 이름이 바뀌었습니다.

핫스폿 설정 관리

이 섹션에서는 DiskStation의 핫스폿 설정을 활성화하고 관리하는 방법을 설명합니다.

DiskStation을(를) 핫스팟으로 변환하기:

  1. 제어판 > 무선 > 네트워크 맵으로 이동합니다. 무선 AP 또는 무선 라우터를 선택합니다.
  2. Wi-Fi 탭으로 이동하고 편집을 클릭하여 다음 설정을 관리합니다:
    • 이름 (SSID): 이 라디오 무선 네트워크의 이름을 입력합니다.
    • 표시/숨김: 다른 장치에게 무선 네트워크를 표시할지 여부를 선택합니다.
    • 무선 모드: 무선 표준을 선택합니다. 각 표준은 다양한 대역폭과 주파수를 지원합니다.
    • 채널 폭: 채널 폭을 선택합니다(802.11n 무선 모드 사용 시만 해당).
    • 채널: 무선 네트워크의 주파수를 선택합니다. 보다 나은 성능을 위해서는 주변 네트워크가 사용하는 것과는 다른 주파수를 선택하는 것이 좋습니다.
    • 전송력: 무선 네트워크의 신호 강도를 선택합니다.
    • 최대 연결: 네트워크에 얼마나 많은 클라이언트 장치가 연결할 수 있는지 선택합니다. 최대 연결 설정은 256입니다.
    • WMM QoS 활성화: Wi-Fi를 통한 Wi-Fi 멀티미디어(WMM) QoS (서비스 품질)입니다. 이 옵션을 이용하면 무선 네트워크에서 트래픽 우선순위를 정하고 응용 프로그램 간에 리소스 공유 방식을 최적화할 수 있습니다.

무선 일정 설정하기:*

무선 네트워크 일정을 활성화하면 DiskStation의 무선 네트워크가 언제 활성 상태이고 액세스 가능한 지를 제어할 수 있습니다.

  1. Wi-Fi 탭으로 이동하고 편집을 클릭합니다.
  2. 일정 활성화 상자를 선택합니다.
  3. 예약 설정을 클릭합니다.
  4. 예약 화면에서 무선 네트워크가 언제 활성 상태인지 선택합니다.

보안 설정

제어판 > 무선 > Wi-Fi에서는 무선 네트워크 보안 설정을 편집하는 옵션을 제공합니다.

무선 보안을 설정하려면:

  1. 편집을 클릭합니다.
  2. 보안 설정 드롭다운 메뉴에서 다음 보안 프로토콜 중 하나를 선택합니다:
    • WEP: 기본 보안 프로토콜로, 무선 네트워크에 액세스하기 위해서는 사용자가 패스워드를 입력해야 하지만 최소의 보호를 제공합니다.
    • WPA/WPA2 개인: 권장됨. 무선 네트워크에 액세스하기 위해서는 사용자가 패스워드를 입력해야 하며 WEP보다 더 안전합니다.
    • WPA/WPA2 엔터프라이즈:* 기업용으로 설계된 보안 프로토콜로, 외부 인증 서버를 통해 사용자를 인증해야 합니다.

Wi-Fi Protected Setup 활성화하기:*

Wi-Fi Protected Setup은 사용자를 위해 무선 보안 설정을 단순화하도록 Wi-Fi 연합이 제안한 표준입니다. 누름 단추 또는 PIN 번호를 사용하여 Wi-Fi 네트워크를 안전하게 지킬 수 있습니다. WPS를 클릭하여 Wi-Fi Protected Setup을 활성화합니다.

게스트 네트워크*

DiskStation에서는 호스트 무선 네트워크를 만드는 것 뿐만 아니라 게스트 무선 네트워크도 만들 수 있게 지원합니다. 게스트 네트워크의 설정은 사용자의 액세스를 제한하는 것 밖에는 할 수 없지만, 방문자에게 무선 네트워크를 제공하면서 호스트 네트워크에는 액세스하지 못하게 하고자 하는 기업이나 기관에게 이는 매우 유용합니다.

게스트 네트워크 활성화하기:

  1. 제어판 > 무선 > Wi-Fi로 이동합니다.
  2. 편집을 클릭하여 게스트 네트워크 탭에서 다음 설정을 관리합니다:
    • 이름 (SSID): 이 라디오 무선 네트워크의 이름을 입력합니다.
    • 표시/숨김: 다른 장치에게 무선 네트워크를 표시할지 여부를 선택합니다.
    • 최대 연결: 네트워크에 얼마나 많은 클라이언트 장치가 연결할 수 있는지 선택합니다. 가장 큰 최대 연결 설정은 256입니다.

게스트 네트워크 보안 설정하기:

보안 설정 드롭다운 메뉴에서 다음 보안 프로토콜 중 하나를 선택합니다:

  • WEP: 기본 보안 프로토콜로, 무선 네트워크에 액세스하기 위해서는 사용자가 패스워드를 입력해야 하지만 최소의 보호를 제공합니다.
  • WPA/WPA2 개인: 권장됨. 무선 네트워크에 액세스하기 위해서는 사용자가 패스워드를 입력해야 하며 WEP보다 더 안전합니다.

게스트 네트워크 일정 설정하기:

게스트 네트워크 일정을 활성화하면 DiskStation의 게스트 무선 네트워크가 언제 활성 상태이고 액세스 가능한 지를 제어할 수 있습니다.

  1. 게스트 네트워크 일정 활성화 상자를 선택합니다.
  2. 게스트 예약 설정을 클릭합니다.
  3. 예약 화면에서 게스트 무선 네트워크가 언제 활성 상태인지 선택합니다.

참고:

호스트 무선 네트워크가 비활성화된 기간 동안은 게스트 무선 네트워크가 활성 상태일 수 없습니다.

MAC 필터 설정*

MAC 필터(제어판 > 무선)는 무선 네트워크에 대한 액세스가 허용되는 MAC 주소를 지정하여, 사용자가 특정 장치에 대한 액세스를 차단 또는 승인할 수 있습니다. 이 기능은 네트워크 액세스만 제어하며 기타 암호화 설정은 제어하지 않습니다.

MAC 필터 규칙 생성하기:

  1. 장치 설명 및 MAC 주소를 입력합니다.
  2. 액세스 정책을 선택합니다:
    • 거부: 지정한 MAC 주소가 무선 네트워크에 액세스하지 못하게 제한합니다.
    • 허용: 지정한 MAC 주소가 무선 네트워크에 액세스하게 허용합니다.

부모 통제*

부모 통제 기능을 사용하여 DiskStation의 로컬 네트워크에 연결된 각 클라이언트 장치에서 인터넷에 액세스할 수 있는 시간을 제어할 수 있습니다. 예를 들어, 아이들의 컴퓨터나 모바일 장치가 DiskStation을 통해 인터넷에 연결된 경우, 특정 장치와 시간대에 준해 인터넷 사용을 제한하도록 부모 통제 설정을 구성할 수 있습니다.

부모 통제 활성화하기:

  1. 제어판 > 무선 > 부모 통제 설정으로 이동합니다.
  2. 추가 단추를 클릭하여 클라이언트 장치를 추가합니다.
    • 고객 이름: DiskStation에 연결된 장치를 선택하거나 직접 이름을 입력합니다(예: Louisa의 랩탑).
    • MAC 주소: 제어하고 싶은 인터넷 사용 장치의 MAC 주소를 입력합니다.
    • 예약 설정: DiskStation을(를) 통해 인터넷에 액세스하도록 허용된 특정 장치의 시간대를 선택합니다.

연결 목록*

연결 목록(제어판 > 무선에 있음)은 DiskStation에 연결된 장치 관련 정보를 표시합니다. 여기에서 MAC 필터 목록에 장치를 직접 추가할 수 있습니다.

MAC 필터 목록에 장치 추가하기:

  1. MAC 필터에 추가하려는 장치 옆의 상자를 선택합니다.
  2. MAC 필터 목록에 추가를 클릭합니다.
