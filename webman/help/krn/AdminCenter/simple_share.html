공유 폴더

공유 폴더는 DiskStation에 파일과 폴더를 저장할 수 있는 기본 디렉토리입니다. DiskStation에 파일을 저장하기 전에 DSM에 최소 하나의 공유 폴더를 생성해야 합니다. 공유 폴더에 저장된 데이터는 사용자 지정된 액세스 권한에 따라 특정 사용자 또는 사용자 그룹과 공유하거나 개인적으로 보유할 수 있습니다.

아래는 특정 응용 프로그램, 서비스 또는 패키지를 설치하거나 활성화할 때 자동으로 생성되는 기본 공유 폴더 목록입니다.

이름 설명
public public 폴더는 처음 DiskStation을(를) 설치하면 자동으로 생성됩니다.*
web web 폴더는 DiskStation에 호스팅된 웹사이트의 콘텐츠를 위한 기본 저장소 위치입니다. 이 폴더는 Web Station이 활성화되면 자동으로 생성됩니다.
photo photo 폴더는 Photo Station과 공유할 사진 및 비디오를 저장하는 데 사용됩니다. 이 폴더는 Photo Station 또는 DLNA/UPnP 미디어 서버가 활성화되면 자동으로 생성됩니다.
music music 폴더는 Audio Station을 사용하여 들을 음악을 저장하는 데 사용됩니다. 이 폴더는 Audio Station 또는 DLNA/UPnP 미디어 서버가 활성화되면 자동으로 생성됩니다.
video video 폴더는 DLNA/UPnP 장치에서 탐색할 비디오 파일을 저장하는 데 사용됩니다. 이 폴더는 Video Station 또는 DLNA/UPnP 미디어 서버가 활성화되면 자동으로 생성됩니다.
surveillance surveillance 폴더는 Surveillance Station 기록을 저장하는 데 사용됩니다. 이 폴더는 Surveillance Station이 활성화되면 자동으로 생성됩니다. 이 폴더는 읽기 전용이며 시스템 관리자만 접근할 수 있습니다.
home home 폴더는 각 사용자가 데이터를 저장하는 개인 공간을 제공하며, 이 공간은 해당 사용자만 액세스할 수 있습니다. 이 폴더는 사용자 홈이 활성화되면 자동으로 생성됩니다.
homes homes 폴더에는 모든 사용자의 홈 폴더가 들어 있습니다. 이 폴더는 사용자 홈이 활성화되면 자동으로 생성됩니다. 시스템 관리자에 한해 모든 사용자의 home 폴더에 접근하고 관리할 수 있습니다.
NetBackup NetBackup 폴더는 네트워크 백업이 활성화되면 자동으로 생성됩니다.
usbshare usbshare[number] 폴더는 DiskStation의 USB 포트에 USB 외부 드라이브 장치가 연결되면 자동으로 생성됩니다.*
satashare satashare 폴더는 DiskStation의 eSATA 포트에 eSATA 외부 드라이브 장치가 연결되면 자동으로 생성됩니다.*

참고:

  • 단순 모드에서는 사용자의 공유 폴더 권한만 편집할 수 있습니다. 사용자 그룹을 생성하거나 공유 폴더에 대한 그룹의 액세스 권한을 관리하려면 제어판(고급 모드) > 그룹으로 이동하십시오.

공유 폴더 생성하기:

  1. 제어판(단순 모드) > 공유 폴더로 이동하고 왼쪽 패널에서 + 아이콘을 클릭합니다.
  2. 이름을 지정하고 공유 폴더를 생성할 볼륨을 선택합니다.
  3. 필요하면 휴지통 활성화를 선택합니다. 이 옵션을 활성화할 경우 공유 폴더에서 파일을 삭제하면 파일이 #recycle 폴더로 이동됩니다. administrators 그룹에 속하는 사용자로 휴지통에 대한 액세스를 제한할 수 있습니다.
  4. 각 사용자에 대한 적절한 상자를 선택 또는 선택 해제하여 공유 폴더에 대한 접근 권한을 사용자 지정합니다:
    • 읽기/쓰기: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근하여 변경 내용을 적용할 수 있습니다.
    • 읽기 전용: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 있습니다.
    • 접근 금지: 사용자가 공유 폴더에 있는 파일 또는 하위 폴더에 접근할 수 없습니다.
  5. 적용을 클릭하여 설정을 저장합니다.

참고:

  • 공유 폴더 이름은 대소문자를 구분하지 않으며 1~32자여야 합니다. 이름에 다음 문자는 포함할 수 없습니다:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • 다음 이름은 시스템 용도로 예약되어 있습니다:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • 공유 폴더 이름의 첫 번째 문자는 빼기(-) 기호 또는 빈칸일 수 없으며 마지막 문자는 빈칸으로 둘 수 없습니다.
  • 권한 충돌이 발생한 경우 권한 우선 순위는 다음과 같습니다: 접근 금지 > 읽기/쓰기 > 읽기 전용.
  • 새 공유 폴더를 생성할 때, administrators 그룹에 속한 사용자의 권한이 접근 금지로 설정되어 있으면, 이 사용자는 제어판 > 공유 폴더에서 공유 폴더를 볼 수만 있습니다.
  • File Station에서 #recycle 폴더를 마우스 오른쪽 단추로 클릭하여 휴지통 폴더의 권한 설정을 구성할 수 있습니다. 속성을 선택하고 권한 탭을 클릭합니다.

공유 폴더 편집하기:

  1. 제어판(단순 모드) > 공유 폴더로 이동합니다.
  2. 왼쪽 패널에서 원하는 공유 폴더를 선택합니다.
  3. 이름, 위치, 사용자 권한 같은 공유 폴더의 일반 정보를 편집합니다.
  4. 이미 휴지통 활성화를 선택한 경우 휴지통 비우기를 클릭하여 휴지통에서 삭제된 모든 파일을 영구 제거할 수 있습니다.
  5. 적용을 클릭하여 설정을 저장합니다.

공유 폴더 삭제하기:

  1. 제어판(단순 모드) > 공유 폴더로 이동합니다.
  2. 왼쪽 패널에서 원하는 공유 폴더를 선택합니다.
  3. 왼쪽 패널에 있는 휴지통 아이콘을 클릭하여 선택한 공유 폴더를 삭제합니다.

____
* 특정 모델에서만 사용할 수 있습니다.
