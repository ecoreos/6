WebDAV

WebDAV는 사용자가 원격 서버에 저장되어 있는 파일을 편집 및 관리할 수 있도록 디자인된 HTTP의 확장 기능입니다. WebDAV 서비스를 활성화한 경우, 특정 Windows 응용 프로그램, Mac OS Finder, Linux 파일 브라우저와 같은 WebDAV를 지원하는 클라이언트 프로그램은 로컬 네트워크 드라이브에 액세스하는 것처럼 DiskStation에 원격 액세스할 수 있습니다.

WebDAV 활성화하기:

  1. HTTPS를 통한 WebDAV 연결을 활성화하려면 WebDAV 활성화 또는 WebDAV HTTPS 연결 활성화 상자를 선택합니다. WebDAV의 기본 포트는 5005이고 WebDAV HTTPS는 5006입니다.
  2. 적용을 클릭합니다

속도 제한

속도 제한을 활성화하면 특정 사용자 또는 그룹이 WebDAV를 통해 파일을 전송할 때 사용할 수 있는 최대 대역폭을 제어할 수 있습니다.

속도 제한 활성화하기:

  1. 속도 제한 활성화 상자를 선택하거나, 특정 시간에만 속도 제한을 구현하려는 경우에는 예약에 따라 속도 제한 활성화를 선택합니다. 이는 서버 수준 설정이며, 개별 사용자 및 그룹의 설정에 우선합니다. 속도 제한 없음을 선택하면 개별 사용자 및 그룹의 속도 제한 설정이 비활성화됩니다.
  2. 속도 제한 설정을 클릭하여 개별 사용자 또는 그룹에 대한 설정을 수정합니다. 각 사용자 및 그룹에 대해 다음 중 하나를 선택할 수 있습니다:
    • 그룹 설정 적용: 속도 제한 설정은 사용자가 속하는 그룹의 설정과 동일하게 됩니다.
    • 없음: 그룹 구성원이 서비스를 사용하여 파일을 전송할 때 속도 제한 없음이 적용됩니다.
    • 상시: 사용자 및 그룹 구성원에게 고정된 일정한 속도 제한이 적용됩니다. 왼쪽의 필드에 업로드 및 다운로드 속도 제한을 지정할 수 있습니다.
    • 가변: 서로 다른 두 가지 속도 제한을 지정하여 사용자 또는 그룹 구성원에게 예약에 따라 적용할 수 있습니다. 사용자 지정 단추를 클릭하여 속도 제한 설정을 수정하고 일정을 설정합니다.
  3. 적용을 클릭합니다.

참고:

  • 그룹 속도 제한에 대한 자세한 내용은 여기를 참조하십시오.
  • 사용자에 대해 속도 제한을 설정할 때, 결과 섹션에는 사용자의 개인 및 그룹 설정에 따라 실제 속도 제한이 표시됩니다. 예를 들어, 사용자 "Achilles"의 개인 다운로드 제한이 100 KB/s지만, 다운로드 제한이 500 KB/s인 그룹에 속하는 경우, 결과 섹션에는 "0/500"이 표시됩니다.

고급 설정

고급 설정 단추를 클릭하여 다음 옵션을 수정할 수 있습니다.

  • Anonymous WebDAV 활성화: WebDAV를 통해 연결할 때 클라이언트에게 "anonymous" 사용자 이름을 사용하여 공유 폴더에 접근할 수 있도록 허용합니다. "Anonymous FTP/WebDAV" 사용자가 공유 폴더에 대한 적절한 접근 권한을 할당 받도록 확인하십시오.
  • DavDepthInfinity 활성화: 클라이언트 도구를 사용할 때, 이 옵션은 사용자가 파일 경로 길이 제한 없이 파일에 액세스하거나 삭제할 수 있게 합니다.
  • WebDAV 로그 활성화: 이 옵션은 WebDAV와 관련된 이벤트를 기록합니다. 로그 보기를 클릭하면 팝업 창에 로그가 표시됩니다.

다양한 운영 체제에 WebDAV 사용하기

DiskStation에서 WebDAV를 활성화하고 나면, 아래 지침을 따라 Windows, Mac, Linux 장치 또는 특정 Synology 스마트폰 응용 프로그램을 사용하여 파일에 대한 액세스를 시작합니다.

참고:

  • 인코딩 오류를 피하기 위해서는 WebDAV 서비스에 연결할 때 UTF-8-지원이 있는 WebDAV 클라이언트를 사용하는 것이 좋습니다.

Windows의 경우

WebDAV 프로토콜을 사용하여 Synology 제품의 파일에 액세스하려는 Windows 사용자는 비상업 용도인 경우 무료로 이용할 수 있는 드라이브 매핑 응용 프로그램인 BitKinex를 사용하는 것이 좋습니다. 또는 인터넷에서 기타 여러 가지 WebDAV 클라이언트 응용 프로그램을 구할 수도 있습니다.

참고:

  • Windows의 네트워크 드라이브 연결 기능을 사용하여 WebDAV를 통해 DiskStation에 있는 파일에 액세스하려는 경우, 유효한 SSL 인증서를 얻고 HTTPS 연결을 통해 DiskStation에 액세스하십시오. Windows 환경에서 WebDAV 서비스를 사용할 때 다른 사용자 계정으로 전환하려면 먼저 Windows 시스템에서 로그아웃해야 합니다.

Mac의 경우

Mac 사용자는 간단히 Finder를 사용하여 DiskStation에 WebDAV를 통해 연결할 수 있습니다.

Mac에서 WebDAV 사용하기:

  1. Mac 컴퓨터에서 Finder > 서버에 연결로 갑니다.
  2. 서버 주소 필드에 DiskStation IP 주소를 입력하거나, 도메인 이름을 입력하고 앞에 "http://"를 붙인 다음 뒤에 ":5005"(WebDAV 서비스를 활성화하는 동안 지정한 다른 포트 번호)를 추가합니다. 예: http://pm.synology.com:5005/.
  3. DSM 사용자 이름과 패스워드를 입력합니다.
  4. Connect을 클릭합니다.

참고:

  • WebDAV는 Bonjour 브로드캐스트를 지원합니다.

Linux의 경우

Ubuntu 또는 기타 Linux 사용자의 경우, 아래 지침을 따라 DiskStation의 파일에 WebDAV를 통해 액세스하십시오.

Ubuntu에서 WebDAV 사용하기:

  1. 장소 > 서버에 연결을 클릭합니다.
  2. 드롭다운 메뉴에서 WebDAV(HTTP) 또는 WebDAV(HTTPS)를 선택합니다.
  3. 서버 위치, 포트, 폴더 및 사용자 계정을 포함하여 필요한 정보를 입력합니다. 연결을 클릭해서 계속 진행합니다.
  4. 패스워드를 입력하고 연결을 클릭합니다.

Linux 명령줄에서 WebDAV 사용하기:

명령줄을 사용하려는 경우 명령줄 WebDAV 클라이언트 소프트웨어인 cadaver를 사용하여 접근을 설정할 수 있습니다.

CalDAV 클라이언트를 사용하여 달력 등록

컴퓨터 또는 모바일 장치의 CalDAV 클라이언트를 사용하여 DiskStation에 저장된 달력에 등록한 다음, 달력 이벤트, 메모, 할 일 등을 관리할 수 있습니다. 달력 클라이언트의 예로 Sunbird (Windows, Mac, Linux), Chandler (Windows, Mac, Linux), Evolution (Linux), iCal (Mac, iOS 장치)가 있습니다.

CalDAV 활성화하기:

  1. 먼저 WebDAV 또는 WebDAV HTTPS 연결을 활성화합니다.
  2. CalDAV 활성화를 선택합니다.
  3. 적용을 클릭합니다.

달력 목록 관리하기:

  1. 달력 목록 보기를 클릭합니다.
  2. 나타나는 창에서 다음을 수행합니다:
    • 새 달력을 생성하려면 추가를 클릭하고 새 달력의 이름을 지정한 다음(예: "cal1"), 달력을 저장할 대상을 지정(예: "calendar" 공유 폴더)하고 확인을 클릭합니다.
    • 달력 이름과 대상을 변경하려면 달력 목록에서 해당 달력을 선택하고 편집을 클릭해서 변경한 다음, 확인을 클릭합니다.
    • DiskStation에서 달력을 삭제하려면 달력 목록에서 해당 항목을 선택하고 삭제를 클릭합니다.
  3. 닫기를 클릭합니다.

참고:

  • DiskStation에 저장된 달력은 DiskStation에서 폴더로 나타납니다.

클라이언트 응용 프로그램(예: Sunbird)을 사용하여 달력 등록하기:

  1. 컴퓨터에 Sunbird를 다운로드해서 설치합니다.
  2. 파일 > 원격 달력 등록으로 이동합니다.
  3. 나타나는 창에서 네트워크에서를 선택하고 다음을 클릭합니다.
  4. CalDAV를 선택하고 위치 필드에 http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name을 입력한 후(예: "http://demo.no-ip.info:5005/calendar/cal1"), 다음을 클릭합니다.
  5. 달력 이름을 지정하고 다음을 클릭합니다.
  6. 인증을 위해 DSM 사용자 이름과 패스워드를 입력하고 확인을 클릭한 다음, 완료를 클릭합니다.

모바일 장치(예: iOS 장치)을 사용하여 달력 등록하기:

  1. iOS 장치에서 설정 > 메일, 연락처, 달력으로 이동합니다.
  2. 계정에서 계정 추가 > 기타로 이동한 다음, CalDAV 계정 추가를 누릅니다.
  3. 필드에 DiskStation의 IP 주소(또는 도메인 이름), 자격 증명을 입력하고 다음을 누릅니다. 메시지가 나타나면 계속을 누릅니다.
  4. 고급 설정을 누르고 다음을 수행합니다.
    1. 포트 필드에 5005(또는 WebDAV를 활성화할 때 지정한 다른 포트 번호)를 입력합니다.
    2. 계정 URL 필드의 URL을 http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name(예: "http://demo.no-ip.info:5005/calendar/cal1")으로 수정합니다.
    3. CalDAV를 누릅니다.
  5. 다음을 눌러 계정 정보를 확인하고 저장을 눌러 계정 설정을 완료합니다.
  6. iOS 장치의 홈 화면으로 돌아가 달력을 열고 달력을 관리합니다.

참고:

  • 달력 이벤트, 메모 또는 할 일을 관리하는 방법에 대한 자세한 정보는 각 클라이언트 응용 프로그램의 도움말 문서를 참조하십시오.
