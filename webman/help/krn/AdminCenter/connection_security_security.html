보안

제어판 > 보안 > 보안에서 DSM 검색 세션에 대해 다음 보안 설정을 수정할 수 있습니다.

  • 로그아웃 타이머(분): 여기에서 지정한 시간대에 사용자가 사용하지 않으면 자동으로 DSM에서 로그아웃됩니다. 1 ~ 65535 사이의 값을 입력합니다.
  • IP 검사를 건너뛰기 함으로써 브라우저 호환성 개선: HTTP 프록시를 통해 DiskStation에 접근할 때 임의 로그아웃 문제가 발생한 경우 이 옵션을 활성화하여 IP 검사를 건너뛸 수 있습니다.
  • 사이트 간 위장 공격에 대한 보호 기능 향상: 이 옵션은 교차 사이트 스크립팅 공격으로부터 시스템의 보호 기능을 강화합니다. 이 옵션은 다음에 DSM에 로그인할 때 적용됩니다.
  • HTTP 콘텐츠 보안 정책(CSP) 헤더로 보안 향상: 이 옵션은 트러스트된 소스의 데이터만 허용하고 인라인 스크립트 실행을 제한하여 교차 사이트 스크립팅(XSS) 공격으로부터 시스템의 보안을 강화합니다.
  • iFrame과 함께 DSM이 포함되도록 허용하지 않음: 옵션을 이용하면 다른 웹사이트가 iFrame을 사용하여 다른 웹 페이지에 DSM을 삽입하지 못하도록 제한하여 악의적인 웹사이트로부터 특정 공격을 방지할 수 있습니다. 허용된 웹사이트 단추를 클릭하여 특정 웹사이트에게 iFrame을 통해 DSM을 삽입하도록 허용할 수 있습니다.
