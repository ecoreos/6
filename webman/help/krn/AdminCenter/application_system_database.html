시스템 데이터베이스

DiskStation은(는) 시스템 데이터베이스를 사용하여 usbshare1 같은 외부 저장소 장치에 생성된 공유 폴더에 저장되어 있는 패키지 데이터와 시스템 로그 항목을 추적합니다.

중요: DiskStation에 연결된 외부 저장소 장치에 저장되어 있는 패키지 데이터에 액세스하려는 경우나 메인 메뉴 > 로그 센터에서 DSM의 로그 기록을 보려는 경우, 먼저 시스템 데이터베이스를 활성화해야 합니다. 그렇지 않으면 모든 관련 서비스에 액세스할 수 없습니다.

시스템 데이터베이스 활성화하기:

  1. 시스템 데이터베이스 활성화를 선택합니다.
  2. 다음 중 하나를 수행하십시오.
    • DSM이 시스템 데이터베이스 저장을 위한 공유 폴더를 자동으로 선택하게 하려면 자동 구성을 선택합니다. 공유 폴더가 포함되어 있는 외부 저장 장치를 제거한 경우, DSM이 데이터베이스 저장을 위해 자동으로 다른 공유 폴더를 선택합니다.
    • 현재 데이터베이스를 선택한 공유 폴더로 이동합니다를 선택한 다음, 드롭다운 메뉴에서 선택하여 현재 데이터베이스를 외부 장치에 생성된 다른 공유 폴더로 이동합니다.
    • 기존 데이터베이스 중 하나를 선택합니다를 선택한 다음 DiskStation에 하나 이상 연결된 외부 장치가 있을 경우 드롭다운 메뉴에서 기존의 데이터베이스를 선택합니다.
  3. 적용을 클릭합니다.

경고: 데이터베이스 손상을 방지하려면 DiskStation에서 장치를 제거하기 전에 오른쪽 위 구석에서 외부 장치 아이콘을 클릭하거나 메인 메뉴 > 제어판 > 외부 장치 > 외부 장치에서 저장 장치를 꺼내도록 하십시오.
