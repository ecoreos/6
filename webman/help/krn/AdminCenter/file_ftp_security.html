보안 설정

선택한 사용자의 루트를 사용자 홈으로 변경

여기서 선택한 사용자는 FTP를 통해 Synology DiskStation에 로그인한 경우에만 홈 폴더에 액세스할 수 있습니다

익명 FTP 활성화

이 기능을 활성화하면 모든 사람이 DiskStation의 FTP 서비스에 로그인할 수 있습니다(익명 또는 ftp 계정 사용).

익명 사용자의 권한 구성하기:

메인 메뉴 > 제어판 > 공유 폴더로 이동한 다음, 권한 설정을 클릭합니다.

익명 루트 변경

이 기능을 활성화하면 FTP 사용자의 로그인 디렉터리를 변경할 수 있습니다. 드롭다운 목록에서 공유 폴더를 선택합니다.
