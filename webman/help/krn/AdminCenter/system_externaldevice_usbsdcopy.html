USBCopy

제어판 > 외부 장치 > USBCopy/SDCopy로 이동하여 대상 폴더를 선택하여 USBCopy 기능을 사용할 수 있습니다.

USBCopy 활성화하기:

  1. USBCopy 활성화를 선택합니다.
  2. 드롭다운 메뉴에서 대상 공유 폴더를 선택합니다.

USB 장치(디지털 카메라 등)를 Synology 제품의 전면 패널에 있는 USB 포트에 연결하면 전면 패널에 있는 복사 단추를 눌러서 USB 장치에 있는 데이터를 이곳에 할당된 대상 공유 폴더 목표에 복사할 수 있습니다. USBCopy 대상 폴더가 삭제되면 시스템이 공용 공유 폴더로 자동으로 교체합니다.

제한 사항

  1. FAT, NTFS, ext3 및 ext4 형식의 USB 장치만 지원됩니다.
  2. USB 장치를 DiskStation 장치의 전면 패널에 있는 USB 포트에 연결해야 합니다.
  3. admin 또는 administrators 그룹 사용자만 USB 장치에 백업 작업을 수행할 수 있는 적절한 액세스 권한을 갖습니다.
  4. 대상 공유 폴더의 사용 가능한 공간은 USB 장치의 사용된 공간보다 더 커야 합니다.

SDCopy*

제어판 > 외부 장치 > USBCopy/SDCopy로 이동하여 대상 폴더를 선택하여 SDCopy 기능을 사용할 수 있습니다.

Synology 제품의 전면 패널에 있는 SD 슬롯에 SD 장치를 꽂은 후 전면 패널에 있는 복사 단추를 눌러서 SD 장치에 있는 데이터를 이곳에 할당된 대상 공유 폴더로 복사할 수 있습니다. SDCopy 대상 폴더가 삭제되면 시스템이 공용 공유 폴더로 자동으로 교체합니다.

제한 사항

  1. FAT, NTFS, ext3 및 ext4 형식의 SD 장치만 지원됩니다.
  2. SD 장치를 Synology 제품의 전면 패널에 있는 SD 슬롯에 연결해야 합니다.
  3. admin 또는 administrators 그룹 사용자만 SD 장치에 백업 작업을 수행할 수 있는 적절한 액세스 권한을 갖습니다.
  4. 대상 공유 폴더의 사용 가능한 공간은 SD 장치의 사용된 공간보다 더 커야 합니다.
