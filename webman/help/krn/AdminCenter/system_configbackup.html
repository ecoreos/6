구성 백업

제어판 > 업데이트 및 복원으로 이동하여 시스템 구성을 즉시 로컬 컴퓨터에 백업합니다. 이를 통해 다양한 설정(예: 사용자, 그룹, 파일 서비스 등)을 저장하고 나중에 이 설정을 복원할 수 있습니다. 시스템 구성은 .dss 형식 파일로 저장됩니다.

시스템 구성 백업하기:

  1. 구성 백업을 클릭합니다.
  2. 구성 파일은 컴퓨터에 다운로드됩니다(파일 이름은 [Synology 제품 이름]_[생성된 날짜].dss).

시스템 구성 복원하기:

  1. 구성 복원을 클릭합니다.
  2. 찾아보기를 클릭하고 컴퓨터에 저장된 구성 파일을 선택합니다.
  3. 복원할 구성에 해당하는 선택 상자를 선택한 다음 확인을 클릭합니다. 또한 충돌하는 설정 덮어쓰기 옵션을 선택하여 기존 사용자, 그룹, 공유 폴더 및 서비스의 선택에 대한 현재 설정을 덮어쓸 수 있습니다.
  4. 경고 프롬프트가 나타납니다. 계속하려면 예를 클릭합니다.
  5. 복원 기능이 완료된 후, 프롬프트가 나타납니다. 확인을 클릭하여 종료합니다.

다음 설정이 백업됩니다:

  • 사용자, 그룹 및 공유 폴더
  • 작업 그룹, 도메인 및 LDAP
  • 파일 공유(예: Windows 파일 서비스, Mac 파일 서비스, NFS 서비스, FTP, WebDAV, rsync) 및 백업 서비스
  • 기타(예: 사용자 홈, 패스워드 설정, 작업 스케줄러)
  • 사용자, 그룹 및 공유 폴더
  • 파일 공유(예: Windows 파일 서비스, Mac 파일 서비스, NFS 서비스, FTP, WebDAV)
  • 기타(예: 사용자 홈, 패스워드 설정, 작업 스케줄러)

참고:

  • 백업 서비스는 백업 작업에 대한 상세 정보는 제외하고 기본 서비스 설정을 포함합니다.
  • 사용자 홈은 homes 공유 폴더를 제외한 기본 서비스 설정을 포함합니다. Hyper Backup으로 이동하여 homes 공유 폴더와 관련 데이터를 백업하고 복원하십시오.
  • 작업 스케줄러는 다른 응용 프로그램에서 생성된 작업(예: Hyper Backup 및 디스크 사용 보고서 등)을 제외한 작업 스케줄러에서 생성할 수 있는 작업을 포함합니다.
  • 복원된 사용자 또는 그룹의 이름은 이제 시스템 용도로만 예약되어 다른 이름으로 복원됩니다.
