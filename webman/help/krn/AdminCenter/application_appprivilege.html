사용 권한

각 서비스 및 응용 프로그램에 대한 사용자 및 그룹의 권한을 보고 편집할 수 있습니다. DSM은 응용 프로그램 권한을 유연하게 제어하는 설정을 제공하여, 여러 응용 프로그램에 대한 사용자 및 그룹의 액세스를 손쉽게 관리할 수 있도록 해줍니다.

사용자 및 그룹의 응용 프로그램 권한 수정하기:

  1. 제어판 > 사용 권한으로 이동합니다.
  2. 수정할 응용 프로그램을 선택합니다. 편집을 클릭합니다.
  3. 사용자를 클릭하여 개인 권한을 보거나 그룹을 클릭하여 그룹 권한을 확인합니다.
  4. 각 사용자 또는 그룹에 대해, 허용을 선택하여 액세스를 허용하거나 거부를 눌러 선택한 응용 프로그램에 대한 액세스를 거부합니다. 응용 프로그램에 따라 IP 기준을 선택하여 IP 주소에 기반하여 액세스를 허용/거부할 수도 있습니다.
  5. 확인을 클릭하여 변경 사항을 저장합니다.

여러 네트워크 IP 주소를 가진 사용자의 응용 프로그램 권한 확인하기:

  1. 제어판 > 사용 권한으로 이동합니다.
  2. 권한 뷰어를 클릭합니다.
  3. 사용자 이름에서 사용자를 선택한 다음 IP 주소에 권한을 확인할 IP 주소를 입력합니다.

모든 사용자의 기본 응용 프로그램 권한 설정하기:

  1. 제어판 > 사용 권한으로 이동합니다.
  2. 수정할 응용 프로그램을 선택합니다. 편집을 클릭합니다.
  3. 기본 권한을 클릭하고 모든 사람에게 이 권한 부여를 선택합니다.
  4. 확인을 클릭하여 변경 사항을 저장합니다.

참고:

  • admin 및 guest의 권한은 여기에서 수정할 수 없습니다.
  • 권한 충돌이 발생한 경우 권한 우선 순위는 다음과 같습니다: 거부 > 허용.
  • 모든 패키지와 서비스가 IP 기준 액세스 권한 설정을 지원하지는 않습니다.
