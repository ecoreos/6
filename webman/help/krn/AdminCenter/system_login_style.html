로그인 스타일

다양한 템플릿 중에서 선택하고, 배경 이미지를 변경하거나 로고를 추가하여 DSM 로그인 화면의 모양을 사용자 지정할 수 있습니다. 이렇게 하려면 제어판 > 로그인 스타일로 이동합니다.

DSM 로그인 화면 사용자 지정하기:

로그인 스타일 페이지에 다음 옵션이 제공됩니다:

  • 로그인 페이지 제목: 여기에 입력한 텍스트가 로그인 화면에 나타납니다.
  • 시간 정보: 로그인 화면에서 지역 시간 정보를 표시하거나 숨길 수 있습니다.
  • 템플릿: 색상 구성표와 기본 배경 이미지를 선택합니다. 선택한 템플릿은 다음에 DiskStation에 로그인할 때 적용됩니다.
  • 배경: 로그인 화면에 표시되는 배경 이미지를 변경하려면 기존 이미지를 클릭하거나 선택한 이미지가 없는 경우 더하기 기호(+)를 클릭합니다. 기본 배경 무늬, DiskStation에 저장된 이미지 또는 이전에 사용했던 이미지 중에 선택할 수 있습니다. JPE, JPEG, JPG, BMP, GIF, PNG 이미지 파일 형식만 지원됩니다.
  • 로고: 로그인 화면에 로고를 추가하려면 기존 로고를 클릭하거나 선택한 이미지가 없는 경우 더하기 기호(+)를 클릭합니다. 크기가 1 MB 이하인 이미지를 업로드하거나, DiskStation에 저장된 이미지에서 선택하거나, 이전에 사용했던 이미지를 선택할 수 있습니다.
