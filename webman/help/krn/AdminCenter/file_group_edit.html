그룹 편집

그룹의 정보와 공유 폴더에 대한 접근 권한을 편집할 수 있습니다.

그룹 정보 편집하기:

  1. 편집하려는 그룹을 선택합니다.
  2. 편집을 클릭하고 그룹 정보를 편집합니다.
  3. 확인을 클릭하여 종료합니다.

공유 폴더에 대한 그룹의 접근 권한 편집하기:

  1. 편집하려는 그룹을 선택합니다.
  2. 편집 및 권한을 클릭합니다.
  3. 그룹에 접근 권한을 할당하려면 다음의 권한을 선택 또는 선택 취소합니다.
    1. 읽기/쓰기: 그룹이 공유 폴더에 접근하고 이를 변경할 수 있습니다.
    2. 읽기 전용: 그룹이 공유 폴더에 접근할 수 있지만 변경할 수는 없습니다.
    3. 접근 금지: 그룹이 공유 폴더에 접근할 수 없습니다.
  4. 확인을 클릭하여 종료합니다.

참고:

권한 충돌이 발생한 경우 권한 우선 순위는 다음과 같습니다: 접근 금지 > 읽기/쓰기 > 읽기 전용.

그룹의 사용 할당량 편집하기:

  1. 편집하려는 그룹을 선택합니다.
  2. 편집을 클릭하고 할당향 탭으로 이동합니다.
  3. 각 볼륨/공유 폴더에 대해 할당량 활성화 아래에 있는 상자를 선택합니다. 그런 다음 할당량 필드에 사용 할당량을 입력합니다(예: 10GB).

참고:

  • 각 그룹 멤버는 지정된 그룹 사용 할당량을 상속합니다. 그룹 사용 할당량은 각 그룹 멤버 간에 공유되지 않습니다.
  • 각 그룹에 대해 입력할 수 있는 최대 저장소 사용 할당량은 대략 4TB(4095GB)입니다.

응용 프로그램에 대한 그룹의 액세스 권한 편집하기:

  1. 편집하려는 그룹을 선택합니다.
  2. 편집을 클릭하고 응용 프로그램 탭으로 이동합니다.
  3. 각 서비스에 대해 다음 옵션 중 하나를 선택할 수 있습니다:
    • 허용: 그룹이 응용 프로그램에 액세스할 수 있습니다.
    • 거부: 그룹이 응용 프로그램에 액세스할 수 없습니다.
    • 사용자 지정: IP 주소로 액세스 권한을 관리합니다.
  4. 확인을 클릭하여 종료합니다.

참고:

  • 권한 충돌이 발생한 경우 권한 우선 순위는 다음과 같습니다: 거부 > 허용.
  • 모든 패키지와 서비스가 IP 기준 액세스 권한 설정을 지원하지는 않습니다.

그룹의 속도 제한 편집하기:

다양한 서비스(예: File Station, FTP 등)에 대해 그룹 속도 제한을 활성화하여 각 서비스를 통해 파일을 전송할 때 그룹 구성원이 소비하는 대역폭의 양을 제한할 수 있습니다. 그룹 속도 제한은 모든 그룹 구성원이 공유합니다.

  1. 편집하려는 그룹을 선택합니다.
  2. 편집을 클릭하고 속도 제한 탭으로 이동합니다.
  3. 각 서비스에 대해 다음 옵션 중 하나를 선택할 수 있습니다:
    • 없음: 그룹 구성원이 서비스를 사용할 때 속도 제한 없음이 적용됩니다.
    • 상시: 그룹 구성원에게 고정된 일정한 속도 제한이 적용됩니다. 왼쪽 필드에서 업로드 및 다운로드 속도 제한을 지정합니다.
    • 가변: 서로 다른 두 가지 속도 제한을 지정하여 그룹 구성원에게 예약에 따라 적용할 수 있습니다. 사용자 지정 단추를 클릭하여 속도 제한 설정 및 일정을 수정합니다.
  4. 확인을 클릭하여 종료합니다.

여러 그룹에 속하는 사용자의 속도 제한

한 사용자가 여러 그룹에 속해 있을 때 그 사용자의 속도 제한은 파일 전송 시 사용 가능한 최고의 대역폭을 가진 그룹을 기반으로 결정됩니다. 대역폭은 현재 파일을 전송 중인 사용자 간에 균등하게 나누어지기 때문에, 사용 가능한 대역폭은 현재 얼마나 많은 사용자가 그룹의 공유 대역폭을 사용하고 있는지에 따라 바뀝니다.

예를 들어, 사용자 "John"은 "그룹 1"(속도 제한 500KB/s; 총 사용자 5명)과 "그룹 2"(속도 제한 400KB/s; 총 사용자 2명)에 속합니다.

  • 그룹 1 및 그룹 2의 사용자들이 모두 동시에 파일을 전송하고 있을 경우 John의 속도 제한은 200KB/s이고, 사용할 수 있는 대역폭은 그룹 2의 공유 대역폭 할당량에 따라 정해집니다.
  • 그룹 1의 다른 사용자들이 파일 전송을 마쳤을 때에는 John의 속도 제한이 즉시 500KB/s으로 변경되고, 사용할 수 있는 대역폭은 그룹 1의 공유 대역폭 할당량에 따라 정해집니다.
