라우터 구성

DiskStation 장치가 로컬 영역 네트워크에 있는 경우, 인터넷의 다른 장치를 연결할 수 없습니다.제어판 > 외부 액세스 > 라우터 구성에서 포트 전달 규칙을 설정하여 DiskStation 장치에 인터넷을 통해 액세스하게 만들 수 있습니다.

라우터 설정

포트 전달 규칙을 추가하기 전에, DiskStation 장치와 라우터 간의 연결을 설정해야 합니다.

참고:

  1. 포트 전달 규칙을 구성하고 고정 IP 주소를 할당하려면 라우터에 대해 관리 권한을 갖고 있어야 합니다.
  2. DiskStation 장치에서 UPnP 라우터로의 연결을 설정하기 전에, DSM이 DiskStation의 외부 IP 주소를 성공적으로 가져왔는지 확인하십시오(제어판 > 외부 액세스 > DDNS).그렇지 않으면 DiskStation 장치 및 UPnP 라우터의 네트워크와 방화벽 설정을 확인하십시오.

DiskStation 장치의 연결을 라우터로 설정하려면 다음과 같이 하십시오.

  1. 라우터 설정을 클릭하여 설정 마법사를 실행하십시오.DSM은 다음 단계를 계속하기 전에 아래 네트워크 컴포넌트/설정을 테스트하는 것으로 시작합니다.
    • 네트워크 인터페이스
    • 게이트웨이 연결
    • 인터넷 연결
    • 포트 전달
    • DNS
  2. 라우터 유형에 따라 다음 중 하나를 수행하십시오.
    • 라우터에서 UPnP를 지원하지 않으면 3단계를 참조하십시오.
    • UPnP 라우터가 DiskStation 장치와 호환되는지 테스트한 경우 라우터의 정보 개요가 표시되면 적용을 클릭하십시오.DiskStation 장치에서 라우터로의 연결이 성공적으로 설정됩니다.
    • UPnP 라우터가 DiskStation 장치와 호환되는지 테스트하지 않은 경우 다음 중 하나를 수행하십시오.
      • 라우터로 연결을 자동으로 설정하려면 자동을 선택하고 다음을 클릭하여 시스템에서 호환성 테스트를 수행하도록 하십시오.
        • 라우터가 DiskStation 장치 라우터 구성 서비스와 호환 테스트를 마친 경우 확인란을 선택하여 테스트 보고서를 보내어 Synology에서 라우터 호환성 데이터베이스를 업데이트하도록 한 다음 적용을 클릭하십시오.DiskStation 장치에서 라우터로의 연결이 성공적으로 설정됩니다.
        • 라우터를 테스트한 결과 DiskStation 장치의 라우터 구성 서비스와 호환되지 않으면 확인란을 선택하여 테스트 보고서를 보내어 Synology에서 라우터 호환성 데이터베이스를 업데이트하도록 한 다음 닫기를 클릭하여 설정 마법사를 닫으십시오.

          • 참고:

          여기에서 라우터를 테스트한 결과 호환되지 않는 경우에도 라우터의 사용자 인터페이스에서 포트 전달 규칙을 구성할 수 있습니다.자세한 정보는 여기를 참조하십시오.

      • 라우터에 대한 연결을 수동으로 설정하려면 수동을 선택하고 다음을 클릭한 후에 3단계를 참조하십시오.
  3. 라우터로의 DiskStation 장치의 연결을 수동으로 설정하려면 다음과 같이 하십시오.
    1. 라우터 드롭 다운 메뉴에서 라우터를 선택하십시오.

      2. 참고:

      여기에서 라우터를 찾을 수 없는 경우 업데이트 목록을 클릭하여 Synology로부터 최신 목록을 받으십시오.

    2. 라우터의 프로토콜과 포트를 설정합니다.프로토콜과 포트는 해당 라우터 모델에 따라 매우 다양할 수 있습니다.기본 프로토콜(HTTP) 및 포트(80)를 사용하여 연결할 수 없는 경우, 라우터와 함께 제공된 문서를 참조하여 올바를 설정을 확인하십시오.
    3. 라우터에 사용자 지정 계정이 필요한 경우 사용자 지정 계정 사용을 선택하고 계정 및 패스워드 필드를 입력합니다.
    4. 적용을 클릭해서 종료합니다.

포트 전달 규칙 관리

포트 전달 규칙을 통해 라우터의 특정 포트로부터 패키지를 수신하는 DiskStation 대상 포트를 지정할 수 있습니다.

DiskStation에서 라우터를 통해 인터넷에 연결하는 경우, 생성을 클릭하여 포트 전달 설정 마법사를 엽니다.

포트 전달 규칙을 추가하는 방법은 두 가지가 있습니다.

  • 시스템 서비스(예: FTP, Web Station 등)에 대한 포트 전달 규칙을 생성하려면 내장 응용 프로그램을 선택합니다.
  • 다른 서비스에 대한 포트 전달 규칙을 생성하려면 사용자 지정 포트를 선택합니다.

다음 구문을 사용하여 포트 번호와 범위를 지정할 수 있습니다.

  • 포트 범위를 설명하려면 “-”을 사용합니다. 예: 6881-6890.
  • 포트를 구분하려면 “,”을 사용합니다. 예: “21,22”, “21,55536-55663”.

참고:

  • 게이트웨이 라우터의 각 포트는 하나의 전달 규칙에만 사용할 수 있습니다.둘 이상의 규칙에 대해 하나의 라우터 포트를 사용하면 충돌이 발생할 수 있으며, 화면에 경고 메시지가 나타납니다.이러한 충돌을 피하려면 규칙에 다른 포트를 설정해야 합니다.
  • 일부 라우터는 시스템 서비스 전달에 대해 사용자 지정 포트를 허용하지 않습니다.

규칙 저장하기:

규칙을 생성한 후 저장을 클릭합니다.그렇지 않으면 규칙이 적용되지 않습니다.

규칙을 저장한 후, 라우터 모델에 따라 규칙이 작동하기 전에 몇 분 정도 대기해야 할 수 있습니다.

규칙의 작동 여부 테스트하기:

테스트할 규칙을 선택하고 연결 테스트를 클릭합니다.연결 테스트 결과 열에 결과가 표시됩니다.

무선 라우터 구성*

DiskStation이(가) 무선 라우터 역할을 하는 경우 포트 전달 규칙은 DiskStation에 의해 직접 이행됩니다.

포트 전달 규칙 추가하기:

  1. 생성을 클릭하여 설정 창을 엽니다.

  2. 서비스 이름, 프로토콜(TCP 또는 UDP), 개인 IP 주소, 전용 포트 및 공용 포트를 지정합니다.다음 구문을 사용하여 포트 번호와 범위를 지정할 수 있습니다.
    • 포트 범위를 설명하려면 “-”을 사용합니다. 예: 6881-6890.
    • 포트를 구분하려면 “,”을 사용합니다. 예: “21,22”, “21,55536-55663”.
  3. 아래에 있는 생성을 클릭하여 설정을 완료합니다.

참고:

대부분의 라우터는 포트 전달 규칙를 구성할 때 로컬 컴퓨터 포트와 라우터 포트가 서로 다른 것을 허용합니다.사용 중인 라우터가 다음 기능을 지원하는 경우:

  • 라우터 포트를 이미 기존 규칙에서 사용하고 있으면 규칙을 저장할 때 새로운 라우터 포트가 할당됩니다.
  • 규칙을 추가 또는 편집할 때, 라우터 포트 번호를 직접 지정할 수 있습니다.

DMZ*

네트워크 맵 구성이 무선 라우터일 때 DiskStation에서 DMZ을 활성화할 수 있습니다.DMZ의 호스트는 방화벽의 제한과 보호에 구애받지 않고 직접 외부 네트워크에 연결될 수 있습니다.이는 보통 서버를 설정하는 데 사용됩니다.

DMZ 활성화하기:

  1. DMZ 설정을 클릭하여 설정 창을 엽니다.
  2. DMZ 활성화 선택 상자를 선택합니다.
  3. DMZ 호스트 IP 주소를 입력합니다.
  4. 적용을 클릭합니다.

참고:

외부 네트워크에서는 DMZ의 호스트에 연결하기 위해 DiskStation에 의해 검색되는 외부 IP 주소가 필요합니다.

____
* 환경에 따라 일부 모델에서만 사용할 수 있습니다.
