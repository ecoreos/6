DSM 업데이트

Synology에서는 주기적으로 DSM 무료 업데이트를 제공합니다. 업데이트에는 새로운 기능, 기능 개선, 성능 강화 등이 포함될 수 있습니다.

중요: 일반적으로 DSM 업그레이드는 DiskStation 상의 데이터에 영향을 주지 않습니다. 그렇지만 만약을 위해서 업그레이드 전에 데이터를 백업하는 것이 바람직합니다.

수동 DSM 업데이트 수행

Synology의 다운로드 센터에서 .pat 업데이트 파일을 다운로드하여 수동으로 DSM 업데이트를 수행할 수 있습니다.

수동으로 DSM 업데이트 수행하기:

  1. 로컬 컴퓨터에 최신 DSM 업데이트를 다운로드하려면 www.synology.com을 방문합니다. .pat 업데이트 파일을 얻기 위해서 파일의 압축을 풉니다.
  2. 수동 DSM 업데이트를 클릭하고 수동 DSM 업데이트 페이지로 들어갑니다.
  3. 찾아보기를 클릭해서 업데이트 파일을 선택합니다.
  4. 적용을 클릭해서 DSM을 업데이트합니다. 업데이트가 완료되면 시스템이 자동으로 다시 시작됩니다.

문제 해결:

DSM을 업데이트할 수 없으면 다음 제안이 도움이 될 수도 있습니다:

  • 업그레이드 과정이 사용자에 의해서 중단되거나 전원 차단 문제 등과 같은 예상하지 못한 다른 문제 때문에 중단되었는지 확인합니다.
  • DSM 버전을 확인합니다. 더 새로운 DSM 버전으로만 시스템을 업데이트할 수 있습니다.
  • 업데이트 파일이 손상되었을 수도 있기 때문에 DSM 업데이트를 다시 다운로드하십시오.

문제가 계속 발생하면 Synology의 기술 지원부에 연락하십시오.

자동 DSM 업데이트 수행

수동 업데이트와 달리, DSM 업데이트로 이동할 때마다 시스템이 Synology를 확인하여 새로운 DSM을 사용할 수 있는지 여부를 확인한 다음, 결과를 녹색 또는 빨간색 텍스트로 표시합니다. 최신 DSM을 다운로드할 준비가 된 것으로 결과가 나타나면 시스템이 자동으로 최신 DSM을 다운로드하여 업데이트를 수행하도록 설정할 수 있습니다.

시스템이 최신 DSM 을 다운로드하여 업데이트를 수행하도록 설정하기:

  1. 최신 DSM을 다운로드할 준비가 될 때 시스템이 자동으로 최신 DSM을 다운로드하게 하려면 다운로드 링크를 클릭합니다.
  2. 시스템이 다음의 바탕화면 알림을 표시하여 다운로드의 성공 여부를 알려줍니다: DSM 다운로드 완료 또는 DSM 다운로드 실패.
  3. 바탕화면 알림 창을 클릭해서 DSM 업데이트로 이동합니다.
  4. 다음 중 하나를 수행하십시오.
    • DSM 다운로드가 완료되면 지금 업데이트를 클릭하여 DSM 를 즉시 업데이트합니다.
    • DSM 다운로드가 실패한 경우 시스템이 Synology 에서 최신 DSM 을 다시 다운로드하게 하려면 재시도를 클릭합니다.

정기적으로 업데이트 확인하기:

  1. 중요한 DSM 업데이트 자동 확인을 선택합니다.
  2. 업데이트 계획을 선택하고 설치 일정을 설정합니다.
  3. 확인을 클릭합니다.

DSM 업데이트가 다운로드되고 설치 준비가 완료되면 바탕화면 알림이 표시됩니다. DSM이 자동 업데이트되는 것을 중지하려면 바탕화면 알림이나 업데이트 설정을 변경하여 이를 취소할 수 있습니다. 전원 예약에서 종료를 활성화한 경우, 예약된 시스템 종료 전에 DSM 업데이트가 설치됩니다.

문제 해결:

최신 DSM 확인 또는 다운로드 중 오류 메시지(빨간색 텍스트)가 표시되는 경우, 다음의 제안 사항을 수행하면 문제를 해결하는 데 도움이 될 수 있습니다.

  • DiskStation이 인터넷에 연결되어 있는지 확인하십시오.
  • Synology의 다운로드 서버가 현재 사용량이 많을 수 있습니다. 나중에 다운로드를 다시 시도하면 문제를 해결할 수도 있습니다.

문제가 계속 발생하면 Synology의 기술 지원부에 연락하십시오.
