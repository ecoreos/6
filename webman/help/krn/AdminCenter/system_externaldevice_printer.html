프린터

DiskStation은(는) 인쇄 서버가 되어 클라이언트 컴퓨터나 모바일 장치에서 DiskStation에 연결된 프린터에 액세스할 수 있습니다. 또한, Apple 무선 인쇄 및 Google Cloud Print를 지원하므로 iOS 장치나 Google 제품 및 서비스를 사용하여 원격으로 인쇄할 수 있습니다.

호환되는 프린터를 보려면 www.synology.com을 방문하십시오.

프린터 설정

아래 지침을 참조하여 USB 또는 네트워크 프린터를 설정하고 DiskStation에 연결하십시오.

USB 프린터 설정하기:

  1. 전원을 켜고 DiskStation의 USB 포트에 프린터를 연결합니다.
  2. 제어판 > 외부 장치 > 프린터로 이동합니다.
  3. 설정할 프린터를 선택합니다. DiskStation에서는 USB 허브를 통해 연결된 프린터는 인식할 수 없다는 점에 유의하십시오.
  4. 프린터 관리자를 클릭하고 프린터 설정을 선택합니다.
  5. 대화 상자에서 다음 모드 중 하나를 선택합니다:
    • 네트워크 MFP: 로컬 네트워크 내의 클라이언트가 다목적 프린터의 인쇄, 스캔 및 팩스 기능을 사용할 수 있습니다. 스캔과 팩스 기능은 Windows 컴퓨터에서만 사용할 수 있습니다.
    • 네트워크 프린터: 인쇄만 지원하는 프린터를 사용하는 경우나 클라이언트에게 인쇄 기능만 사용할 수 있게 하려는 경우 이 모드를 선택합니다.
  6. Apple 무선 인쇄 또는 Google Cloud Print 설정을 구성합니다(옵션).
  7. 저장을 클릭하십시오.

참고:

네트워크 MFP에는 하나의 클라이언트에서만 동시에 액세스할 수 있습니다.

네트워크 프린터를 설치하려면:

  1. 전원을 켜고 네트워크 프린터를 DiskStation와(과) 동일한 로컬 네트워크에 연결합니다.
  2. 제어판 > 외부 장치 > 프린터로 이동합니다.
  3. 네트워크 프린터 추가를 클릭하고 설정 마법사의 지침을 따릅니다.

참고:

  • 프린터 이름: 프린터 이름은 설정하는 동안 입력해야 하며 문자, 숫자, 밑줄을 포함 32자까지 사용할 수 있습니다.
  • 프로토콜: 일반적으로 HP 네트워크 프린터는 소켓, Canon 네트워크 프린터는 BJNP, 기타 대부분은 LPR을 사용합니다.
  • 대기열 이름: LPR 및 IPP 네트워크 프린터에 대기열 이름을 입력해야 하지만, 소켓이나 BJNP 프린터에는 적용되지 않습니다.
  • 네트워크 MFP 없음: DiskStation에 연결된 네트워크 프린터는 현재 네트워크 MFP 모드를 사용하지 않습니다.
  • USB 프린터 없음: Synology 제품에 의해 공유되는 USB 프린터는 네트워크 프린터로 설정할 수 없습니다.

프린터 관리

프린터 탭에는 DiskStation에 연결된 USB 또는 네트워크 프린터를 관리하는 다음 옵션이 있습니다.

  • 정보 얻기: 장치 이름, 제조업체, 상태 등과 같은 선택한 프린터와 관련된 정보를 표시합니다.
  • 제거: DiskStation에서 프린터를 제거하거나 연결을 끊습니다.
  • 프린터 관리자: 다음 옵션을 포함합니다:
    • 프린터 설정: Apple 무선 인쇄 또는 Google Cloud Print 같은 프린터 설정을 편집합니다(지침은 아래 참조).
    • 스풀 공간 정리: 모든 프린터의 모든 인쇄 작업을 취소합니다.
    • 테스트 페이지 인쇄: 테스트 페이지를 인쇄하여 구성이 올바른지 확인합니다.
    • 프린터 잠금 해제: 프린터와 클라이언트의 연결을 해제합니다(네트워크 MFP 모드에만 해당). 클라이언트가 다목적 프린터에 연결하여 팩스를 받거나 파일을 스캔하면 작업이 오래 걸릴 수 있습니다. 필요한 경우, 대기 중인 다음 클라이언트가 프린터를 이용하도록 연결을 중지할 수 있습니다.

Apple 무선 인쇄 활성화하기:

  1. 제어판 > 외부 장치 > 프린터로 이동합니다.
  2. 장치 목록에서 프린터를 선택합니다.
  3. 프린터 관리자 > 프린터 설정을 클릭합니다.
  4. Apple 무선 인쇄 활성화 상자를 선택합니다.
  5. 프린터의 브랜드와 드라이버를 선택합니다.
  6. 저장을 클릭하십시오.

설정이 완료되면 프린터 관리자 > 테스트 페이지 인쇄를 클릭하여 테스트 페이지를 인쇄할 수 있습니다.

참고:

  • Apple 무선 인쇄를 활성화하면 DSM에 대한 Bonjour 프린터 브로드캐스트도 활성화됩니다.
  • Apple 무선 인쇄는 iOS 4.2 이상에서 사용할 수 있습니다.

Google Cloud Print 활성화하기:

  1. 제어판 > 외부 장치 > 프린터로 이동합니다.
  2. 장치 목록에서 프린터를 선택합니다.
  3. 프린터 관리자 > 프린터 설정을 클릭합니다.
  4. Google Cloud Print 활성화 상자를 선택합니다.
  5. 다음 세부 사항을 입력합니다:
    • Google 계정 정보.
    • 프린터의 브랜드 및 드라이버.
  6. 저장을 클릭하십시오.

설정이 완료되면 프린터 관리자 > 테스트 페이지 인쇄를 클릭하여 테스트 페이지를 인쇄할 수 있습니다.

Google Cloud Print 작업을 관리하려면 Google Cloud Print 링크를 클릭합니다.

프린터에 액세스

PC 또는 Mac을 사용하여 프린터에 액세스하려면 아래 섹션을 참조하십시오.

Windows PC를 사용하여 프린터에 액세스하기:

DiskStation 장치의 인쇄 서비스에 액세스하는 가장 좋은 방법은 Synology Assistant를 사용하여 프린터에 연결을 설정하는 것입니다. 자세한 지침은 해당 도움말을 참조하십시오.

Synology Assistant가 컴퓨터에 설치되지 않은 경우 아래의 다른 방법에 따라 DiskStation 장치의 인쇄 서비스에 액세스하십시오.

유형 1. 공유 네트워크 프린터

  1. PC의 파일 탐색기에서 네트워크를 열고 DiskStation을(를) 더블 클릭합니다.
  2. usbprinter 또는 네트워크 프린터 이름을 더블 클릭합니다. 시스템에서 올바른 드라이버를 설치하라는 메시지가 표시되면 예를 클릭합니다.
  3. 제조업체의 CD나 웹사이트에서 프린터 드라이버를 가져옵니다. 프린터의 제조업체와 모델을 선택하거나 디스크 있음을 클릭하여 드라이버의 위치를 선택합니다. 확인을 클릭합니다.
  4. 시스템이 네트워크 프린터 설치를 시작합니다. 설정이 완료되면 프린터를 사용할 수 있습니다.

유형 2. LPR 포트 (권장)

  1. PC에서 시작을 클릭하고 장치 및 프린터로 이동합니다.
  2. 프린터 추가를 클릭합니다.
  3. 네트워크, 무선 또는 Bluetooth 프린터 추가를 선택합니다.
  4. 원하는 프린터가 목록에 없습니다를 선택합니다.
  5. TCP/IP 주소 또는 호스트 이름을 사용하여 프린터 추가를 선택합니다. 다음을 클릭하여 계속 진행합니다.
  6. 장치 유형이 TCP/IP 포트 장치가 되도록 선택합니다.
  7. DiskStation의 IP 주소를 입력합니다. 다음을 클릭하여 계속 진행합니다.
  8. 장치 유형: 사용자 지정 > 설정 > …을 선택합니다. 다음을 클릭하여 계속 진행합니다.
  9. 프로토콜에서 LPR을 선택하고 대기열 이름에 usbprinter 또는 네트워크 프린터 이름을 입력한 후, LPR 바이트 계산 사용 선택 상자를 선택합니다. 확인을 클릭해서 계속 진행합니다.
  10. 제조업체의 CD나 웹사이트에서 프린터 드라이버를 가져옵니다. 프린터의 제조업체와 모델을 선택하거나 디스크 있음을 클릭하여 드라이버의 위치를 선택합니다. 확인을 클릭합니다.
  11. 시스템이 네트워크 프린터 설치를 시작합니다. 설정이 완료되면 프린터를 사용할 수 있습니다.

Mac을 사용하여 프린터에 액세스하기:

Mac 사용자는 시스템 설정에서 DSM Bonjour 프린터 브로드캐스트를 활성화하고 프린터를 추가함으로써 DiskStation 장치의 인쇄 서비스에 액세스할 수 있습니다.

계속하기 전에 DiskStation이(가) Postscript 프린터를 사용 중인지 확인하십시오.

단계 1. DSM의 Bonjour 프린터 브로드캐스트 활성화

  1. DSM에 admin 또는 administrators 그룹에 속하는 사용자로 로그인합니다.
  2. 제어판 > 파일 서비스 > Win/Mac/NFS로 이동합니다.
  3. Bonjour 프린터 브로드캐스트 활성화를 선택하십시오.

단계 2. Mac을 사용하여 프린터에 액세스

  1. Mac에서 시스템 설정 > 인쇄 및 스캔으로 이동한 다음 +를 클릭합니다.
  2. 드롭다운 메뉴에서 usbprinter[number] 또는 네트워크 프린터 이름을 선택합니다. Mac에서 프린터를 자동으로 설정합니다.
