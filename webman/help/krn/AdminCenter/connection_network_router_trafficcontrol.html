라우터 트래픽 제어

라우터 트래픽 제어는 DiskStation이(가) 라우터 모드일 때 활성화됩니다. 라우터 트래픽 제어(제어판 > 네트워크 > 라우터 트래픽 제어)는 DiskStation 개인 네트워크에서 실행되는 업로드 및 다운로드 트래픽을 관리합니다. 규칙을 생성할 때 보장된 대역폭 및 최대 대역폭을 지정하여 업로드 및 다운로드 트래픽을 제한할 필요가 있습니다.

용어 설명

  • 보장된 대역폭: 전체 시스템 대역폭이 충분할 때 서비스가 보장하는 나가는 트래픽을 정의합니다.
  • 최대 대역폭: 전체 시스템 대역폭이 충분하고 시스템 잔여 대역폭이 있을 때 이 서비스가 이용할 수 있는 나가는 트래픽을 정의합니다. 먼저 시스템 출력 대역폭을 계산하십시오. 그런 다음, 각 장치의 보장된 대역폭 합계가 시스템 출력 대역폭보다 크지 않은지 또는 설정이 올바르게 작동하지 않는지 확인합니다.

설정 공식

  • 시스템 출력 대역폭 = 모든 서비스의 보장된 대역폭 합계 + 시스템 잔여 대역폭
  • 각 서비스의 보장된 대역폭 ≤ 각 서비스의 최대 대역폭

참고:

나가는 패킷이 라우터 트래픽 제어 규칙과 일치하지 않으면 기본 동작에 제한이 없게 됩니다. 모두 포트를 선택하여 모든 규칙 끝에 자신의 기본 라우터 트래픽 제어 규칙을 추가하고, 보장된 대역폭 및 최대 대역폭을 지정할 수 있습니다.

라우터 트래픽 제어 규칙 생성하기:

  1. 오른쪽 위 구석에 있는 드롭다운 메뉴에서 네트워크 인터페이스를 선택합니다.
  2. 생성을 클릭합니다.
  3. 포트 섹션에서 다음 중 하나를 선택합니다:
    • 규칙을 모든 포트에 적용하려면 모두를 선택합니다.
    • 사용자 지정을 선택하고 사용자 지정을 클릭합니다. 최대 15개의 포트를 쉼표로 구분하여 입력하거나 이 규칙을 적용할 포트 범위를 지정합니다.
    • 또한 프로토콜도 지정할 수 있습니다.
  4. 소스 IP 섹션에서 다음 중 하나를 선택합니다:
    • 이 규칙을 모든 IP 주소에 적용하려면 모두를 선택합니다.
    • 특정 IP를 선택하고 선택을 클릭합니다. 이 규칙을 적용할 IP 주소 또는 IP 범위를 지정할 수 있습니다.

이제 규칙 목록에서 생성한 라우터 트래픽 제어 규칙을 확인할 수 있습니다.

라우터 트래픽 제어 규칙을 삭제하려면 다음 중 하나를 수행합니다.

  • 활성화 열 목록에서 규칙의 선택 상자 선택을 해제합니다.
  • 라우터 트래픽 제어 규칙을 삭제합니다.

참고:

  • 각 네트워크 인터페이스에 대해 최대 100개의 규칙을 만들 수 있습니다.
  • 끌어다 놓기로 목록에서 규칙을 재정렬합니다.
  • 규칙은 목록에서의 위치에 따라 우선 순위가 부여됩니다.
  • 지정된 포트는 로컬 네트워크 밖에 있는 인터넷 서비스의 포트 번호를 말합니다.
  • 소스 IP는 라우터로 Synology NAS를 사용하는 장치의 IP 주소를 말합니다.
