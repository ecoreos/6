사용자 가져오기

사용자 목록을 가져와서 일괄처리 방식으로 사용자 계정을 생성할 수 있습니다.

사용자 목록 생성하기:

방법 1. 텍스트 편집기 사용:

  1. 텍스트 편집기로 새 파일을 엽니다.
  2. 다음 순서로 사용자 정보를 입력하고 TAB으로 구분합니다:
    • 사용자 이름
    • 패스워드
    • 설명
    • 이메일
    • 볼륨 1의 할당량 (MB)(볼륨이 두 개 이상일 경우 이 단계를 반복하십시오)

      참고:

      Btrfs 파일 시스템과 외부 저장소 장치(예: USB/eSATA 드라이브)에서는 사용자의 할당량이 지원되지 않습니다.

  3. 모든 사용자를 입력할 때까지 다음 줄로 이동해서 2단계를 반복합니다. 한 줄당 사용자 정보 하나씩 포함됩니다.
  4. 파일을 UTF-8 인코딩으로 저장합니다.

방법 2. Microsoft Excel 사용:

  1. Excel에서 새 파일을 엽니다.
  2. 다음 순서대로 사용자 정보를 입력하고 한 행에서 각 열별로 사용자를 구분합니다:
    • 사용자 이름
    • 패스워드
    • 설명
    • 이메일
    • 볼륨 1의 할당량 (MB)(볼륨이 두 개 이상일 경우 이 단계를 반복하십시오)

      참고:

      Btrfs 파일 시스템과 외부 저장소 장치(예: USB/eSATA 드라이브)에서는 사용자의 할당량이 지원되지 않습니다.

  3. 모든 사용자를 입력할 때까지 다음 행으로 이동해서 2단계를 반복합니다. 한 행당 사용자 정보가 하나씩 포함됩니다.
  4. 텍스트 편집기로 새 파일을 엽니다. 스프레드시트에서 항목들을 복사해서 텍스트 파일로 붙여 넣습니다.
  5. 파일을 UTF-8 인코딩으로 저장합니다.

참고:

  • 사용자 이름을 제외하고 모든 필드는 선택 사항입니다.
  • 텍스트 파일의 예는 다음과 같습니다.
    • rosy 1234 어머니 rosy@abc.com 80000
    chaddy 5678 형제 chaddy@abc.com 80000
    danny 8901 사촌 danny@abc.com 20000
    anny 2345 자매 anny@abc.com 20000

사용자 목록 가져오기:

  1. 생성 > 사용자 가져오기를 선택합니다.
  2. 기존의 중복 계정을 사용자 목록에 있는 계정으로 바꾸려면 중복 계정 덮어쓰기를 선택합니다.
  3. 찾아보기를 클릭하고 사용자 목록의 파일을 선택합니다. 파일을 선택하면 미리보기 영역에 가져온 사용자의 목록이 표시됩니다. 비정상적인 상태의 사용자는 가져오지 않습니다.
  4. 업로드를 클릭하면 사용자 계정이 생성됩니다.

참고:

사용자 목록의 파일은 UTF-8 형식이어야 합니다.
