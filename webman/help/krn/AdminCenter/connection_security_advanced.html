고급

제어판 > 보안 > 고급에서 DSM 검색 세션에 대해 다음 고급 설정을 수정할 수 있습니다.

HTTP 압축

DiskStation에 대해 HTTP 압축을 활성화할 수 있습니다. 네트워크 트래픽의 대역폭을 낮춤으로써 웹페이지의 로딩 속도를 높일 수 있습니다.

HTTP 압축 활성화/비활성화하기:

  • 제어판 > 보안 > 고급에서 HTTP 압축을 선택/선택 해제합니다.

TLS / SSL 프로파일

HTTPS 암호화된 연결의 보안 수준을 선택할 수 있습니다. 다음과 같은 3가지 수준을 제공합니다:

  • 최신 호환성 - 최신 브라우저 및 모바일 장치와만 호환됩니다.
  • 중간 호환성 - 최상 및 최저 보안 수준 사이입니다. 이는 시스템 기본 값입니다.
  • 이전 역방향 호환성 - 안전하지 않지만, 대부분의 브라우저 및 모바일 장치를 지원합니다.
