멀티미디어 데이터베이스

DiskStation은(는) 멀티미디어 데이터베이스를 사용하여 미디어 소스(usbshare1과 같은 외부 저장 장치에 생성한 공유 폴더)에 저장된 멀티미디어 파일과 로그 항목을 추적합니다.

중요: DSM 멀티미디어 서비스(예: 미디어 서버, iTunes, Audio Station 및 Download Station)를 통해 DiskStation의 USB 포트에 연결된 외부 저장 장치에 저장되어 있는 멀티미디어 파일에 접근하거나 DSM 로그 서비스를 활성화하고 메인 메뉴 > 시스템 정보 > 로그에서 해당 로그 기록을 조회하려면 먼저 멀티미디어 데이터베이스를 활성화해야 합니다. 그렇지 않으면 모든 관련 서비스가 비활성화됩니다.

멀티미디어 데이터베이스 활성화하기:

  1. 멀티미디어 데이터베이스 활성화를 선택합니다.
  2. 다음을 수행하십시오.
    • DSM이 멀티미디어 데이터베이스 저장을 위한 공유 폴더를 자동으로 선택하게 하려면 자동 구성을 선택합니다. 공유 폴더가 포함되어 있는 외부 저장 장치를 제거한 경우, DSM이 데이터베이스 저장을 위해 자동으로 다른 공유 폴더를 선택합니다.
    • 데이터베이스를 저장할 공유 폴더를 선택하십시오.를 선택한 다음, 드롭다운 메뉴에서 데이터베이스를 저장하기 위한 공유 폴더를 직접 선택합니다.
  3. OK를 클릭합니다.

경고: 데이터베이스 손상을 방지하려면 DiskStation에서 장치를 제거하기 전에 메인 메뉴 > 제어판 > 외부 장치에서 저장 장치를 꺼내도록 하십시오.

멀티미디어 소스에서 공유 폴더 포함/제외시키기:

  1. 멀티미디어 소스를 클릭합니다.
  2. 공유 폴더 옆에 있는 선택 상자를 선택하거나 선택 취소하여 원하는 항목을 선택합니다.
  3. OK를 클릭합니다.

선택한 공유 폴더에 있는 멀티미디어 파일과 로그 기록에 색인이 지정되고, 색인이 지정된 정보가 멀티미디어 데이터베이스에 추가됩니다.

멀티미디어 데이터베이스 새로 고치기:

  1. 데이터베이스 새로 고침을 클릭합니다. 해당 멀티미디어 데이터베이스에 있는 모든 정보가 현재 멀티미디어 소스에 대해 다시 색인이 지정된 정보로 대체됩니다.
  2. OK를 클릭합니다.

참고: 멀티미디어 데이터베이스 중 상당량의 데이터가 더 이상 사용되지 않는 경우(예: DiskStation과(와) 함께 다시 사용하지 않을 USB 디스크 연결을 해제함으로써 데이터베이스에 색인 정보가 중복됨), 멀티미디어 데이터베이스를 새로 고치면 사용되지 않는 모든 데이터를 삭제하거나 데이터베이스 크기를 축소할 수 있습니다. 그러나 이렇게 할 경우 디스크를 반복해서 읽고 써야 하기 때문에 대량의 시스템 리소스가 소모되고, 그 결과 디스크 수명이 줄어들 수 있습니다. 이 기능을 사용할 때는 각별한 주의를 기울이십시오.
