사용자 홈

사용자 홈에서 guest를 제외한 각 사용자에 대한 개인 home 폴더를 만들 수 있습니다. 이렇게 하면 모든 사용자가 CIFS, AFP, FTP 또는 File Station을 통해 자신의 home 폴더에 접근할 수 있는 권한을 갖게 됩니다.

관리자는 homes 기본 공유 폴더에 있는 모든 개인 폴더에 접근할 수 있습니다. home 폴더의 이름은 사용자 계정과 동일합니다.

사용자 홈 서비스 활성화하기:

  1. 사용자 홈을 클릭합니다.
  2. 사용자 홈 서비스 활성화를 선택합니다.
  3. Synology DiskStation에 두 개 이상의 볼륨이 있는 경우 homes 폴더를 저장할 위치를 선택합니다.
  4. 확인을 클릭합니다.

참고: 로컬 사용자 홈 서비스가 비활성화되면 도메인 사용자 홈 서비스 또한 동시에 비활성화됩니다.

사용자 홈 서비스 비활성화하기:

  1. 사용자 홈을 클릭합니다.
  2. 사용자 홈 서비스 활성화 선택을 취소합니다.

사용자 홈 서비스가 비활성화된 후 admin이 액세스할 수 있도록 homes 공유 폴더가 그대로 유지됩니다. 사용자 홈 서비스가 다시 활성화되면 사용자가 홈 폴더에 다시 접근할 수 있습니다.

    참고:
  1. homes 공유 폴더를 삭제하려면 먼저 사용자 홈 서비스를 비활성화해야 합니다.
  2. 로컬 사용자 홈 서비스가 비활성화된 경우 도메인 사용자 홈 서비스도 비활성화됩니다.
