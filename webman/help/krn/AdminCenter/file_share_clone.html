공유 폴더 복제*

Btrfs 볼륨에 있는 공유 폴더를 복제하여 기존 공유 폴더의 복사본을 거의 즉석에서 만들 수 있습니다. 공유 폴더 복제는 생성 시 저장소를 거의 소비하지 않습니다.

공유 폴더 복제하기:

  1. 제어판 > 공유 폴더로 이동합니다.
  2. Btrfs 파일 시스템에서 원하는 공유 폴더를 선택합니다.
  3. 생성 > 복제를 클릭합니다.
  4. 새 공유 폴더의 이름과 설명을 지정하고, 필요에 따라 기타 옵션을 수정합니다. (이러한 옵션에 대한 자세한 내용은 공유 폴더 생성을 참조하십시오.)

참고:

  • Btrfs 볼륨의 공유 폴더만 복제할 수 있습니다.
  • 복제된 공유 폴더는 원래 공유 폴더와 동일한 볼륨에 생성됩니다.
  • 고급 데이터 무결성 보호의 설정은 복제된 공유 폴더에 적용되며 변경이 불가능합니다.

____
*일부 모델에서만 사용할 수 있습니다.
