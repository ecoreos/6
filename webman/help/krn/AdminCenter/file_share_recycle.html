휴지통 활성화

공유 폴더에서 휴지통을 활성화하면 삭제된 파일을 저장하기 위한 하위 폴더가 생성됩니다. 휴지통에서 삭제된 파일을 검색할 수 있습니다. 파일을 영구 삭제하려면 휴지통을 비웁니다.

휴지통 활성화하기:

  1. 공유 폴더를 선택한 다음 편집을 클릭합니다.
  2. 휴지통 활성화 선택 상자를 선택합니다.
  3. 휴지통에 관리자만 액세스할 수 있도록 하려면 관리자 전용으로 액세스 제한을 선택합니다.
  4. 확인을 클릭하여 설정을 저장합니다.

휴지통 비우기:

휴지통 비우기 단추를 클릭하면 휴지통에 있는 모든 삭제 파일이 영구적으로 제거됩니다.

모든 휴지통 비우기:

작업 드롭다운 메뉴에서 모든 휴지통 비우기를 선택하여 모든 휴지통에서 삭제된 파일을 영구 제거합니다.

참고: File Station에서 #recycle 폴더를 마우스 오른쪽 단추로 클릭하여 휴지통 폴더의 권한 설정을 구성할 수 있습니다. 속성을 선택하고 권한 탭을 클릭합니다.
