시간

현재 시간을 확인하고 서버의 날짜와 시간을 직접 설정하거나 네트워크 시간 서버를 이용해서 자동으로 설정할 수 있습니다.

현재 시간을 확인하고, 서버의 날짜와 시간을 설정할 수 있습니다.

표준 시간대 설정하기:

표준 시간대 드롭다운 메뉴에서 옵션을 선택합니다.

지역 날짜와 시간을 수동으로 설정하기:

수동을 선택하고 날짜와 시간을 선택합니다.

날짜와 시간을 NTP 서버와 동기화하기:

  1. NTP 서버와 동기화를 선택하고 네트워크 시간 서버의 주소를 입력하거나 드롭다운 메뉴에서 서버를 선택합니다.
  2. 지금 업데이트를 클릭하여 DiskStation에서 서버와 시간을 동기화했는지 확인합니다.
  3. 적용을 클릭합니다.

참고: 네트워크 시간 서버와 날짜 및 시간을 동기화하려면 메인 메뉴 > 제어판 > 네트워크에서 올바른 DNS 서버 IP를 설정했는지와 DiskStation을(를) 인터넷에 연결했는지 확인합니다.

네트워크 시간 프로토콜(NTP) 서버 관련 정보

네트워크 시간 프로토콜(NTP)의 주요 용도는 데이터 네트워크에서 컴퓨터 시스템의 시간을 동기화하는 것입니다.

다음에는 몇 가지 권장되는 NTP 서버가 나와 있습니다.

  • time.apple.com
  • time.asia.apple.com
  • time.euro.apple.com
  • time.nist.gov

NTP 서버에 대한 자세한 정보는 http://www.eecis.udel.edu/~mills/ntp/servers.html을 참조하십시오.
