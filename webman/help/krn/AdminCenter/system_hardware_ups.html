UPS

UPS(무정전 전원 공급 장치)는 정전이 발생했을 때 DiskStation이(가) 짧은 시간 동안 계속 작동할 수 있도록 해주는 백업 전원 장치입니다. 제어판 > 하드웨어 및 전원 > UPS에 있는 이 기능은 전원이 끊기고 종료되기 전에 DiskStation이 데이터를 저장하고 볼륨의 마운트를 해제할 수 있도록 충분한 시간을 제공하여 데이터 손실을 방지하는 데 유용합니다.

UPS(무정전 전원 공급 장치)는 정전이 발생했을 때 DiskStation이(가) 짧은 시간 동안 계속 작동할 수 있도록 해주는 백업 전원 장치입니다. 제어판 > 하드웨어 및 전원 > UPS에 있는 이 기능은 전원이 끊기고 중지되기 전에 DiskStation이 데이터를 저장하고 볼륨의 마운트를 해제할 수 있도록 충분한 시간을 제공하여 데이터 손실을 방지하는 데 유용합니다.

안전 모드

DiskStation은(는) 안전 모드에 들어가면 UPS 장치에 전원이 부족할 때 데이터 손실을 방지하고 안전하게 종료될 수 있도록 모든 서비스를 중지하고 볼륨의 마운트를 해제합니다. 기본적으로, UPS 장치가 낮은 전력 상태에서 실행되기 시작하면 시스템이 안전 모드로 들어갑니다. 또한 정전이 발생했을 때 DiskStation이(가) 안전 모드에 들어가기까지의 시간을 지정할 수도 있습니다. 그러나 지정한 시간 전에 UPS 장치의 배터리가 부족해지면 시스템이 즉시 안전 모드로 들어갑니다.

DiskStation은(는) 안전 모드에 들어가면 UPS 장치에 전원이 부족할 때 데이터 손실을 방지하고 안전하게 중지될 수 있도록 모든 서비스를 중지하고 볼륨의 마운트를 해제합니다. 기본적으로, UPS 장치가 낮은 전력 상태에서 실행되기 시작하면 시스템이 안전 모드로 들어갑니다. 또한 정전이 발생했을 때 DiskStation이(가) 안전 모드에 들어가기까지의 시간을 지정할 수도 있습니다. 그러나 지정한 시간 전에 UPS 장치의 배터리가 부족해지면 시스템이 즉시 안전 모드로 들어갑니다.

안전 모드에 있을 때 DiskStation이(가) 종료되는 경우, 전원 차단 후 자동으로 다시 시작 옵션(제어판 > 하드웨어 및 전원 > 일반에 위치)을 활성화해 놓으면 전원이 복구될 때 자동으로 켜집니다.

로컬 UPS 지원

하드웨어 설치에 대한 자세한 내용은 UPS 장치의 사용 설명서를 참조하십시오. 일단 설치가 완료되고 DiskStation이(가) UPS 장치에 연결되면 아래 단계를 따라 UPS 지원을 활성화할 수 있습니다.

로컬 UPS 지원 활성화하기:

  1. UPS 장치를 DiskStation의 USB 포트에 연결합니다.
  2. UPS 탭에서 UPS 지원 활성화 선택 상자를 선택합니다.
  3. 정전이 발생한 후 DiskStation이(가) 안전 모드로 들어가기까지의 시간을 선택합니다.
  4. 시스템이 안전 모드로 들어갈 때 UPS 장치로 종료 신호를 전송할지 여부를 선택합니다. 대부분의 UPS 장치는 전원이 복구되면 자동으로 켜집니다.
  5. 적용을 클릭합니다.

SNMP UPS 지원

DiskStation은(는) 또한 SNMP(단순 네트워크 관리 프로토콜) UPS 장치 연결을 지원합니다. 하드웨어 설치에 대한 자세한 내용은 SNMP UPS 장치의 사용 설명서를 참조하십시오. 일단 DiskStation을(를) 장치에 꽂고 동일한 네트워크에 연결되면 아래 단계를 따라 UPS 지원을 활성화할 수 있습니다.

SNMP UPS 지원 활성화하기:

  1. UPS 탭에서 UPS 지원 활성화 선택 상자를 선택합니다.
  2. 네트워크 UPS 유형 드롭다운 메뉴에서 SNMP UPS를 선택합니다.
  3. SNMP UPS IP 주소 필드에 SNMP UPS 장치의 IP 주소를 입력합니다.
  4. 정전이 발생한 후 DiskStation이(가) 안전 모드로 들어가기까지의 시간을 선택합니다.
  5. SNMP UPS 장치의 커뮤니티 문자열을 SNMP 커뮤니티 필드에 입력합니다.
  6. 적용을 클릭합니다.

참고:

  • Synology High Availability(SHA) 구축을 위해 SNMP UPS를 사용할 것을 강력하게 권장합니다.
  • DiskStation은(는) SNMPv1 및 SNMPv2c UPS 장치를 지원합니다.

네트워크 UPS 지원

두 개의 Synology 제품이 단일 USB 포트를 통해 UPS 장치를 사용하는 경우 하나의 DiskStation만 UPS 장치 상태에 대한 정보를 얻을 수 있습니다. 이 경우, 하나의 DiskStation은(는) "네트워크 UPS 서버"로 작동하여 (장치에 따라) USB 또는 네트워크를 통해 UPS 장치에 연결될 수 있습니다. 네트워크 UPS 서버는 UPS 정보를 얻고 이를 다른 "클라이언트" DiskStation에 중계합니다.

네트워크 UPS 지원 활성화하기:

  1. (USB 또는 네트워크를 통해) UPS 장치에 연결된 DiskStation에 로그인합니다. 로컬 UPS 지원이 이 DiskStation에서 아직 활성화되지 않은 경우 위 섹션을 참조하십시오.
  2. UPS 탭에서 네트워크 UPS 서버 활성화 상자를 선택하여 DiskStation이(가) UPS 정보를 클라이언트 DiskStation에 중계할 수 있도록 합니다.
  3. 허용된 DiskStation 장치 단추를 클릭합니다.
  4. 나타나는 창에서 USB 또는 네트워크를 통해 연결되어 있지 않고 UPS 장치를 사용하는 다른 DiskStation의 IP 주소를 입력합니다. 확인을 클릭합니다.
  5. 적용을 클릭하여 설정을 저장합니다.

이제 네트워크 UPS 서버는 UPS 정보를 얻어 이를 UPS 장치를 사용하는 다른 DiskStation에 중계할 수 있습니다. 아래 지침을 참조하여 다른 DiskStation에서 UPS 지원을 활성화하십시오.

클라이언트 DiskStation에서 UPS 지원 활성화하기:

  1. USB 또는 네트워크를 통해 연결되어 있지 않고 UPS 장치를 사용하는 다른 DiskStation에 로그인합니다.
  2. UPS 탭에서 UPS 지원 활성화 상자를 선택합니다.
  3. 네트워크 UPS 유형 드롭다운 메뉴에서 Synology UPS 서버를 선택합니다.
  4. 네트워크 UPS 서버 IP 필드에 네트워크 UPS 서버(즉, UPS 장치에 연결된 DiskStation)의 IP 주소를 입력합니다.
  5. 정전이 발생한 후 DiskStation이(가) 안전 모드로 들어가기까지의 시간을 선택합니다. 안전 모드 설정을 간소화하기 위해 서버와 동일을 선택할 수 있습니다.
  6. 적용을 클릭하여 설정을 저장합니다.

네트워크 UPS 요구 사항:

  • UPS 장치 정보를 중계하려면 각 DiskStation이(가) 동일한 네트워크에 연결되어 있어야 합니다.
  • 각 DiskStation이(가) 연결되어 있는 네트워크 장치(예: 스위치 또는 허브)는 동일한 UPS 장치에 연결되어야 합니다. 그렇지 않으면 DiskStation이(가) 정전이 발생했을 때 UPS 장치 정보를 중계하지 못합니다.
