사용자 편집

사용자 계정을 찾고(검색 필드에서 사용자 이름 입력) 편집을 더블 클릭한 다음 계정 데이터를 편집할 수 있습니다.

참고:
  1. 계정을 비활성화하면 계정 기능이 정지됩니다.
  2. 할당량은 0이상의 정수로, 디스크/볼륨 용량보다는 작아야 합니다. 할당량 값이 0이면 효과적인 할당량은 그룹 제한에 따라 결정됩니다.
  3. 서버에 볼륨/공유 폴더이 여러 개인 경우, 모든 볼륨/공유 폴더에서 사용자에 대한 할당량을 설정할 수 있습니다.
  4. "알림" 페이지의 이메일 정보는 시스템이 계정 정보가 생성 또는 수정된 다음 이메일로 사용자에게 그 사실을 알리기 위해 사용되기 때문에 올바로 입력되어야 합니다.
  5. 사용자에 대한 속도 제한을 설정할 때, 해당 사용자에게 적용된 그룹 속도 제한이 이미 있는 경우 결과 아래에 숫자가 표시됩니다.
  6. 사용자에게 할당된 속도 제한은 사용자가 속한 그룹의 속도 제한에는 포함되지 않습니다.
