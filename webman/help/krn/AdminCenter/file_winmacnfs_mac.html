Mac 파일 서비스

제어판 > 파일 서비스 > Win/Mac/NFS에서 Mac 파일 서비스를 활성화하여 DiskStation을(를) Mac 파일 서버(AFP 서버)로 설정할 수 있습니다.

Mac 파일 서비스 활성화하기:

  1. DiskStation을(를) Mac 파일 서버(AFP 서버)로 설정하려면 Mac 파일 서비스 활성화를 선택합니다.
  2. 필요하면 Time Machine 백업 폴더를 설정합니다. (자세한 정보는 아래의 섹션을 참조하십시오.)
  3. 적용을 클릭합니다.

참고:

  • Time Machine에서 DiskStation(으)로 데이터를 백업하려면 드롭다운 메뉴에서 공유 폴더를 선택합니다.
  • 다른 서비스에서 사용 중인 공유 폴더는 Time Machine에서 사용할 수 없습니다.

전송 로깅 활성화

이 옵션을 선택하면 AFP(Apple Filing Protocol)를 통한 파일 또는 폴더의 생성, 삭제, 읽기, 쓰기, 이동 및 이름 변경을 비롯하여 파일 액세스 작업에 대한 로깅이 활성화됩니다.

고급 설정

기본 UNIX 권한 적용:

이 옵션을 활성화하면 파일 및 폴더를 업로드하거나 생성할 때 기본 UNIX 권한에 적용됩니다. 적용된 권한은 UNIX command umask에 의해 적용된 권한과 동일합니다. 이 옵션을 활성화하면, UNIX 권한은 파일에 대해서는 644이고 폴더에 대해서는 755입니다. 이 옵션을 비활성화하면, UNIX 권한은 파일에 대해서는 666이고 폴더에 대해서는 777입니다. 기본 umask 값은 022입니다.

참고:

이 옵션을 활성화하면 다른 프로토콜 사이에서 권한 문제가 일관되지 않을 수 있습니다. 일관성을 유지하려면 이 옵션을 비활성화 상태를 유지하는 것이 좋습니다.

연결이 끊긴 후 즉시 리소스 해제:

이 옵션을 활성화하면 AFP 서버 연결이 예기치 않게 끊길 때 리소스를 즉시 해제합니다. Time Machine을 사용하는 동안 "볼륨이 사용 중" 오류 메시지가 발생하면, 이 옵션을 활성화하십시오.

Bonjour 프린터 브로드캐스트 활성화하기:

Mac 클라이언트가 DiskStation에 연결된 프린터를 찾고 검색할 수 있게 하려면 Bonjour 프린터 브로드캐스트 활성화를 선택합니다.

Finder의 Spotlight를 통해 파일 내용 검색하기:

폴더에 전적으로 색인을 지정하고 Mac 컴퓨터에 마운트한 경우, Finder의 Spotlight를 사용하여 그 안에 포함된 파일과 내용을 검색할 수 있습니다.

  1. 제어판 > 색인 서비스 > 파일 색인 > 색인된 폴더 목록 > 생성으로 가서 특정 폴더와 그 안에 포함된 파일을 파일 유형별로 색인합니다.
  2. 확인을 클릭하여 색인을 수행합니다.
  3. 색인된 폴더를 Mac 컴퓨터에 마운트합니다.
  4. Mac 컴퓨터의 Finder 왼쪽 패널에서 원하는 마운트된 폴더를 클릭하고, Finder에서 Spotlight를 사용하여 그 안에 포함된 파일을 검색합니다.

참고:

  • 마운트된 폴더 내에서 검색하려면, Finder의 Spotlight를 사용합니다. Mac 데스크톱의 Spotlight는 로컬 파일에 대해서만 검색을 허용합니다.
  • Mac OS X 10.9 이상에서는 Finder의 Spotlight가 태그 이름 및 범주를 기준으로 마운트된 폴더 내에서의 검색을 지원합니다.
