파일 시스템 단편화 제거*

데이터가 지속적으로 수정되는 경우, 파일 시스템에서 단편화된 파일의 양이 증가하여 결과적으로 하드 드라이브의 파일에 대한 액세스 성능이 느려집니다. 파일 시스템 단편화 제거를 수행하여 성능을 향상시킬 수 있습니다.

파일 시스템 단편화 제거 수행:

  1. 저장소 관리자를 엽니다.
  2. 볼륨으로 이동합니다.
  3. 파일 시스템 단편화 제거를 수행할 볼륨을 선택합니다.
  4. 관리를 클릭합니다.
  5. 파일 시스템 단편화 제거를 선택합니다.

참고:

  • 파일 시스템 단편화 제거는 Btrfs 파일 시스템의 볼륨에만 사용할 수 있습니다.
  • 볼륨의 상태가 정상일 때만 파일 시스템 단편화 제거를 수행할 수 있습니다.
  • 파일 시스템 단편화 제거가 완료되기 전에 시스템을 종료하지 마십시오. 완료 전에 시스템이 종료된 경우 파일 시스템 단편화 제거를 처음부터 다시 시작해야 합니다.
  • 볼륨에 공유 폴더의 스냅샷이 포함된 경우 볼륨 사용량이 증가할 수 있습니다.

____
*일부 모델에서만 사용할 수 있습니다.
