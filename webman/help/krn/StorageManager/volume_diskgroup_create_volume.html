볼륨 생성

이 게시물은 DiskStation에 볼륨을 생성하는 방법을 설명합니다. 새 볼륨을 생성할 때 다음 방법 중 하나를 선택할 수 있습니다:

  • 빠름: 이것은 초보 사용자에게 좋은 옵션입니다. RAID 수준 같은 기술적인 측면에 대해 걱정하지 않고 저장 용량과 성능을 자동으로 최적화하고 싶은 경우 이 방법을 선택합니다.
  • 사용자 지정: 볼륨을 생성할 때 RAID 유형과 용량 할당 같은 좀더 세부적인 사항을 제어하고 싶은 고급 사용자를 위한 옵션입니다. 이 옵션은 다양한 RAID 유형을 지원하므로 사용자가 하나 또는 여러 개의 볼륨을 생성할 수 있으며, 각 볼륨에 특정 용량을 할당하는 옵션을 제공합니다(RAID에 여러 개의 볼륨을 생성할 경우에만).

RAID에 단일 또는 복수 볼륨?

사용자 지정 모드에서 볼륨을 생성할 때 다음 옵션 중 하나를 선택할 수 있습니다:

  • RAID에 단일 볼륨
    • 선택한 하드 드라이브의 사용 가능한 모든 용량을 사용하여 하나의 볼륨을 생성합니다.
    • 빠른 속도와 향상된 성능을 제공합니다.
    • 다양한 RAID 유형을 지원합니다.
  • RAID에 복수 볼륨
    • 디스크 그룹에 복수의 볼륨을 생성할 수 있습니다.
    • 각 볼륨에 특정 분량의 용량을 할당할 수 있습니다.
    • 보다 유연하고 향상된 저장소 관리 기능을 제공합니다.
    • 다양한 RAID 유형을 지원합니다.

시작하기 전에:

  • 시작하기 전에 하드 드라이브에 중요한 데이터가 있는지 확인합니다. 생성 과정에서 기존 데이터가 모두 삭제됩니다.
  • HDD/SSD 페이지로 이동하여 각 하드 드라이브 상태가 정상 또는 초기화되지 않음인지 확인합니다.

참고:

  • Btrfs 볼륨의 최소 크기는 10GB입니다.
  • Btrfs 볼륨은 메타데이터 공간 중 4%를 차지합니다.

볼륨 생성하기:

  1. 저장소 관리자를 엽니다.
  2. 볼륨으로 이동하여 생성을 클릭합니다.
  3. 볼륨 생성 마법사가 나타납니다. 필요에 따라 빠름 또는 사용자 지정을 선택합니다. 그런 다음 마법사의 단계를 따라 볼륨 생성을 마칩니다.
