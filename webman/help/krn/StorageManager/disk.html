HDD/SSD

이 섹션에서는 DiskStation에 설치된 HDD/SSD의 상태를 모니터링할 수 있도록 HDD/SSD 성능 및 상태를 관리하고 분석하는 옵션을 제공합니다. 저장소 관리자 > HDD/SSD로 이동하여 이러한 옵션을 관리합니다.

HDD/SSD 정보 확인하기:

  1. HDD/SSD 탭으로 이동합니다.
  2. 목록에서 드라이브를 선택하여 확장합니다. 여기에서 S.M.A.R.T. 상태, 볼륨 정보, 모델 이름, 일련 번호, 펌웨어 버전 및 온도를 확인할 수 있습니다.

상태 정보 확인하기:

  1. HDD/SSD 탭으로 이동합니다.
  2. 드라이브를 선택합니다.
  3. 상태 정보를 클릭합니다.
  4. 개요 탭으로 이동합니다. 여기에서 드라이브가 몇 시간 동안 켜져 있었는지, 현재 온도, 재연결/재식별 수, 불량 섹터 수 및 전체 S.M.A.R.T. 상태를 알 수 있습니다.
  5. S.M.A.R.T. 정보 탭으로 이동합니다. 제시된 번호는 드라이브의 상태를 자세히 진단하는 데 유용합니다.

참고:

SAS 드라이브는 S.M.A.R.T. 정보를 지원하지 않습니다.

S.M.A.R.T. 테스트 시작하기:

S.M.A.R.T. 테스트는 드라이브 상태를 검사한 후 보고하여 발생 가능한 드라이브 문제를 알려줍니다. 오류가 발견되면 즉시 드라이브를 바꾸는 것이 좋습니다.

  1. HDD/SSD 탭으로 이동합니다.
  2. 드라이브를 선택합니다.
  3. S.M.A.R.T. 테스트를 클릭합니다.
  4. 테스트 유형을 선택하고 시작을 클릭하여 테스트를 시작합니다.
  5. 최신 테스트 결과가 테스트 결과 섹션에 표시됩니다.

참고:

테스트 시간은 시스템 환경에 따라 달라질 수 있습니다.

S.M.A.R.T. 테스트 예약하기:

작업을 생성하여 S.M.A.R.T. 테스트가 자동으로 실행되도록 예약할 수 있습니다.

  1. 테스트 스케줄러 탭으로 이동합니다.
  2. 만들기를 클릭하여 새 작업을 설정합니다.
  3. 일반 탭에서 다음 설정을 수정합니다:
    • 작업: 작업에 대한 이름을 입력합니다.
    • S.M.A.R.T. 테스트 유형: 빠른 테스트 또는 확장 테스트를 수행하도록 선택합니다.
    • S.M.A.R.T. 테스트 범위: S.M.A.R.T. 테스트를 수행할 때마다 테스트할 HDD/SSD를 선택합니다.
  4. 예약 탭에서 다음 설정을 수정합니다.
    • 날짜: S.M.A.R.T. 테스트를 수행할 시기를 지정합니다. 특정 요일(예: 매일, 주말 또는 매주 일요일 등) 또는 구체적인 날짜(예: 2015/10/21)에 실행되도록 S.M.A.R.T. 테스트를 예약할 수 있습니다.
    • 시간: S.M.A.R.T. 테스트가 수행되는 횟수 및 매일 첫 번째와 마지막 시간을 지정합니다.
  5. 확인을 클릭하여 종료합니다.

Secure Erase 수행하기:

  1. HDD/SSD 탭으로 이동합니다.
  2. 드라이브를 선택하고 Secure Erase를 클릭합니다.

참고:

  • 확장 유니트의 드라이브에 Secure Erase를 수행할 수 없습니다.
  • Secure Erase는 드라이브에 저장된 모든 데이터를 영구 삭제합니다. 이를 수행하기 전에 선택한 드라이브가 사용 중이 아닌지, 데이터를 백업했는지 확인하십시오.
  • 시스템이 Secure Erase를 수행 중이면, 선택한 드라이브가 일시적으로 잠긴 다음 작업이 완료된 후 자동으로 잠금이 해제됩니다.
  • 작업 중 Secure Erase가 부적절하게 중지된 경우 사용자 인터페이스에서 직접 Secure Erase를 다시 실행할 수 있습니다. 드라이브가 여전히 잠궈진 상태에서 이를 제거 또는 사용할 경우 패스워드 "synology"를 사용하여 드라이브의 잠금을 해제하십시오.

HDD에 대해 쓰기 캐시 지원 활성화

쓰기 캐시 지원을 활성화하여 DiskStation의 성능을 강화합니다. 특정 HDD 모델에서만 이 기능을 지원합니다. 쓰기 캐시 지원을 사용하는 동안 데이터를 확실히 보호할 수 있도록 UPS 장치를 사용하는 것이 권장됩니다. 또한 매번 사용 후에는 시스템을 적절하게 종료해야 합니다. 쓰기 캐시를 비활성화하면 비정상적으로 전원이 중단될 가능성은 줄어들지만 시스템 성능이 저하됩니다.

쓰기 캐시 지원 활성화하기:

  1. 캐시 관리 탭으로 이동합니다.
  2. 쓰기 캐시 활성화를 선택합니다. 기본적으로 이 기능은 활성화되어 있습니다.
  3. 적용을 클릭합니다.
