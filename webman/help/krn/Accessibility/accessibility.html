접근성

DiskStation Manager는 내장된 접근성 기능을 통해 시각 장애가 있는 사용자가 DSM 서비스/설정을 탐색하고 관리할 수 있도록 안내하는 음성 신호를 제공합니다.

DSM 접근성 설정

접근성을 최대로 활용하기 위해서는 운영 환경을 다음과 같이 설정하는 것이 좋습니다.

  • 로컬 컴퓨터에 NVDA 화면 읽기 프로그램(Windows 환경에서만 사용 가능)을 설치합니다.
  • 최신 Mozilla Firefox 웹 브라우저를 설치합니다.
  • 헤드셋 또는 스피커를 사용하고 필요에 따라 볼륨을 조정합니다.
  • 포커스 모드로 NVDA를 사용합니다(모드를 전환하려면 NVDA 키 + 스페이스바 누름). 기본 NVDA 키는 Insert 키입니다.
  • 다음 핫키를 사용하여 DSM 탐색하기:
    핫키 설명
    Tab 다른 사용자 인터페이스 부분으로 전환
    화살표 키 같은 사용자 인터페이스 부분 내의 항목으로 전환
    Enter 응용 프로그램/창 열기
    Esc 응용 프로그램/창 닫기

이제 DSM의 다음 영역에서 접근성 기능을 사용할 수 있습니다.

  • DSM 로그인 페이지, 바탕화면 및 작업 표시줄
  • 메인 메뉴
  • 제어판
  • File Station

참고:

NVDA 작업에 대한 자세한 내용은 사용자 가이드를 참조하십시오.

DSM 키보드 바로가기 안내

DSM은 접근성 기능과 함께 유연하고 완벽한 관리를 구현하기 위해 다양한 키보드 바로가기를 제공합니다. 바로가기를 찾으려면 Shift + ?을 누릅니다.

  • 바탕화면 키보드 바로가기: 창 작업 등을 위한 핫키
  • 구성 요소 키보드 바로가기: 사용자 인터페이스 구성 요소(예: 목록, 단추 및 검색 필드)를 제어하기 위한 핫키

바탕화면과 관련된 바로가기의 사용을 중단하려면 바탕화면 키보드 바로가기 활성화 옵션을 취소합니다.
