이메일 계정

개인용 > 이메일 계정에서 이메일 계정을 추가하고, 여기에서 추가한 이메일 주소에서 File Station에 저장된 파일을 첨부 파일로 보낼 수 있습니다.

지원되는 이메일 서비스 제공업체는 Gmail, Outlook, Yahoo!, AOL 및 사용자 지정 SMTP 서버입니다.

이메일 계정 추가하기:

  1. 추가를 클릭하고 설정할 이메일 계정의 서비스 제공업체를 선택합니다.
  2. 이메일 주소, 패스워드 및 보내는 사람 이름을 입력합니다.
  3. 여러 계정이 설정되어 있는 경우 File Station이 기본적으로 이 이메일 계정을 사용하도록 하려면 이 이메일 계정을 기본값으로 사용을 선택합니다.
  4. 이메일 서비스 제공업체(Gmail, Outlook 및 Yahoo!)가 지원하는 경우 연락처 동기화를 클릭하여 이메일 연락처를 DiskStation에 동기화합니다. 동기화된 연락처는 이전 단계에서 사용자가 입력한 이메일 계정이 아니라 표시된 인증 창에 제공한 이메일 계정에서 온 것임에 유의하십시오. 따라서, 인증 창에 다른 계정이나 원치 않는 계정이 표시되는 경우 수동으로 원하는 이메일 계정이나 이전에 입력한 계정으로 전환하십시오.
  5. 적용을 클릭해서 설정을 저장합니다.

이메일 계정 편집하기:

목록에서 원하는 이메일 계정을 선택한 다음 편집을 클릭합니다.

이메일 계정 삭제하기:

목록에서 원하는 이메일 계정을 선택한 다음 삭제를 클릭합니다.

참고:

  • 다른 기존 계정이 있지 않으면 기본 계정을 삭제할 수 없습니다. 현재 기본적으로 설정된 이메일 계정을 삭제하려면 기본 계정을 다른 것으로 변경한 다음 작업을 계속하십시오.

기본 계정 변경하기:

목록에서 기본으로 설정할 이메일 계정을 선택한 다음 기본값으로 설정을 클릭합니다.

첨부물로 파일 보내기:

  1. File Station에서 파일을 마우스 오른쪽 단추로 클릭하고 이메일 첨부 파일로 보내기를 선택합니다.
  2. 여러 개의 이메일 계정을 추가한 경우, 드롭다운 목록에서 원하는 보내는 사람 이메일 주소를 선택하여 해당 파일을 특정 수신인에게 보낼 수 있습니다.
