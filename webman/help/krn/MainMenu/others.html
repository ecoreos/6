기타

개인용 > 기타에서 다른 개인 계정 옵션을 사용자 지정할 수 있습니다.

기타 개인 옵션 사용자 지정하기:

  1. 다음을 선택합니다:
    • DSM에서 나갈 때 확인 대화 상자 표시 안 함: 브라우저 창을 닫을 때 확인 대화 상자를 건너뜁니다.
    • 로그인 시 DSM을 이전 로그아웃 상태로 계속하기: 열려 있는 응용 프로그램이나 패키지를 기억하고 다음에 로그인하거나 브라우저 창을 새로 고칠 때 이를 복원합니다.
    • 작업 표시줄에서 즉시 미리보기 활성화: 작업 표시줄의 응용 프로그램 아이콘 위에 마우스를 놓을 때 열린 응용 프로그램에 대한 빠른 미리보기를 표시합니다. Internet Explorer는 최신 버전이 아닌 경우 이 기능을 완전히 지원하지 않을 수도 있습니다.
    • 로그인 시 항상 DSM 도움말 자동 실행 안 함: DSM 도움말 페이지는 DSM에 로그인할 때 자동으로 열리지 않습니다.
    • 바탕화면 키보드 바로가기 활성화: 키보드의 바로가기를 사용하여 여러 유형의 데스크톱 및 DSM 작업을 수행할 수 있습니다(예: 활성 창 간에 전화, 옵션 선택/선택 취소 등). 키보드의 바로가기를 모두 보려면 Shift + ?를 누릅니다.
    • 접근성 비활성화(음성 신호 사용 불가능): 화면 읽기 프로그램이 켜졌을 때 음성 신호를 들을 수 없도록 DSM에서 접근성 기능을 보류합니다.
    • 브라우저 바탕화면 알림 활성화: 웹 브라우저에서 DSM 바탕화면에 알림을 표시합니다. 브라우저 바탕화면 알림은 Google Chrome 및 Safari에서만 지원됩니다. 웹 브라우저의 버전에 따라 알림 설정 페이지가 다를 수 있습니다. 알림 설정을 관리하려면 브라우저의 설정 페이지로 이동하십시오.
    • 작업 표시줄에서 파일럿 보기 단추 활성화: 파일럿 보기 단추를 표시하여 열린 모든 응용 프로그램을 미리보기 보드에서 볼 수 있게 합니다.
  2. 확인을 클릭합니다.
