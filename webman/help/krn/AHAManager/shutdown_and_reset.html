서버 종료 및 교체

아래 지침에서는 high-availability 클러스터에서 서버를 종료 또는 변경하는 과정을 안내합니다.

High Availability Manager를 사용하여 high-availability 클러스터 종료하기:

새로운 실제 위치로 서버를 이동하거나 유지 관리를 수행하기 위해 high-availability 클러스터를 종료하려는 경우 다음 지침을 따르십시오.

  1. DSM에 로그인합니다.
  2. High Availability Manager로 이동합니다.
  3. 관리라고 표시된 단추를 클릭합니다.
  4. High-availability 클러스터 종료를 선택합니다.
  5. 확인 메시지가 나타납니다. 확인을 클릭하여 확인합니다.

High-availability 클러스터 수동으로 종료하기:

High-availability 클러스터는 각 서버에서 전원 단추를 사용하여 수동으로 종료할 수도 있습니다.

  1. 신호음이 들리고 수동 서버가 완전히 종료될 때까지 수동 서버에서 전원 단추를 누르고 있습니다.
  2. 신호음이 들리고 활성 서버가 완전히 종료될 때까지 수동 서버에서 전원 단추를 누르고 있습니다.

참고:

  • 서버 켜기:
    서버를 다시 켜려면, 활성 서버와 수동 서버를 동시에 켜거나 5분 간격 이내에 켜야 합니다. 서버 중 하나만 켜면 LAN 깨우기 메시지가 브로드캐스트되어 상대편 서버의 전원을 켜려고 시도합니다.
  • 비정상 종료:
    high-availability 클러스터에서 전원 차단 후 다시 시작 옵션이 활성화됩니다. 전원 차단으로 인해 서버가 비정상적으로 종료될 경우 전원이 복구되면 자동으로 서버가 켜집니다.

High-availability 클러스터에서 서비스 교체하기:

서버 하나를 제거하고 다른 서버를 설정하려면 다음 지침을 따르십시오.

  1. 개요 페이지로 이동합니다.
  2. 관리 단추를 클릭합니다.
  3. 활성 서버 종료 또는 수동 서버 종료를 선택하여 교체하려는 서버를 종료합니다.
  4. 새 서버 연결을 확인합니다. 새 서버가 켜지면 High Availaibility 관리자에 "알 수 없음"으로 표시됩니다.
  5. 개요 페이지로 다시 이동합니다.
  6. 관리 단추를 클릭합니다.
  7. 알 수 없는 서버 복구를 선택합니다. 그러면 "알 수 없는 서버"가 high-availability 클러스터에 추가되고 수동 서버로 인식됩니다.
