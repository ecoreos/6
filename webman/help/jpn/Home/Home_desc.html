DSM を始める

DSM のデスクトップと、DiskStation にインストールされているサービスやパッケージを実行する方法を紹介します。

オプション

パスワード、説明、2 ステップ認証など、個人のアカウント設定を変更します。

コントロール パネル

ファイル共有、システム、ネットワーク サービスなどに関連する DSM の設定を表示、変更します。

File Station

DiskStation に保管されているフォルダやファイルを整理したり、アップロード/ダウンロードします。

ストレージマネージャ

異なるレベルのデータ保護を備えたストレージ スペースを作成したり、ハードディスクの状態を確認したり、iSCSI target と LUN をマップしたりします。

バックアップと複製

DiskStation でデータのバックアップを作成し、タスクを復元します。

パッケージ センター

Synology 社製またはサードパーティのアプリケーションをインストール、アップデート、管理します。

EZ-Internet

簡単なセットアップ ウィザードで、インターネットを介して DiskStation にアクセスできるようにします。

リソース モニタ

CPU の使用状況、メモリの使用状況、ネットワークフロー、ボリュームの使用状況を監視します。

CPU 使用状況、メモリ使用状況、ネットワークフローを監視を行います。

ログ センター

DiskStation のシステム ログを表示したり、ログの送受信を設定したりします。
