DSM mobile

DSM mobile は、DSM admin (または administrators グループに属するユーザー) が iOS® (iPhone®、iPad®、iPod touch®) または Android® で DSM 設定を管理したり、DSM 情報をチェックしたりするのを支援します。この DSM mobile のヘルプでは、使用開始するのに役立つ以下のトピックやリソースが用意されています。

ログインとログアウト

iOS や Android デバイスで Web ブラウザを使用して DSM に接続すると、ログイン画面が表示されます。DSM アカウント情報を入力します。

DSM mobile にログインする：

DSM ユーザー資格情報を入力して [ログイン] をタップします。ログイン情報を保存して次回から自動的にログインするようにするには、ログインする前に [この情報を保存する] を有効にします。

DSM mobile からログアウトする：

  1. ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) の右上隅にある [オプション] ボタンをタップします。
  2. [設定] をタップし、[ログアウト] をタップします。

DSM ユーザーの管理

[ユーザーとグループ] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップしてユーザーリストを表示します。

ユーザーを作成する：

  1. + をタップして新しいユーザーのユーザー名、パスワード、電子メールアドレスを入力します。
  2. [通知の送信] を有効にすると、DSM ユーザーアカウントを作成するとユーザーの電子メールアドレスに通知のメールが送信されます。
    注意:通知を送信する前に、通知の設定が Synology NAS で完了しているか確認してください。コンピュータの Web ブラウザで DSM のデスクトップエディションにログインし、[メイン メニュー] > [コントロール パネル] > [通知] > 電子メール] の順に選択して設定を変更します。
  3. [終了] をタップして確定します。または、キャンセルするには左矢印をタップします。

ユーザーをクイック検索する

ユーザー名を検索フィールドに入力します。

ユーザーの情報を変更する

  1. ユーザーの名前をタップして、以下のいずれかを行います。
    • [名前] または [パスワード] 欄で、ユーザーの名前やパスワードを変更します。
    • [無効] をオンにしてユーザーを無効にします。設定はそのままで後でユーザーを有効にすることができます。
    • [管理者] をオンにして admin グループにユーザーを追加します。admin グループに属しているユーザーに管理者権限を与えて、DSM 設定を DSM mobile やコンピュータの Web ブラウザから変更できるようにします。
  2. [特権] から以下のいずれかを実行して、[戻る] をタップします：
    • [共有フォルダ] をタップし、共有フォルダの名前をタップしてユーザーのアクセス権限を変更します：読み込み/書き込み：ユーザーは、共有フォルダのファイルやサブフォルダに変更を行うことができます。読込み専用：ユーザーは、共有フォルダのファイルやサブフォルダにアクセスできますが、変更を加えることはできません。アクセスなし：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできません。
      注意：権限の競合が発生した場合は、権限の優先順位は次のようになります：アクセスなし > 読み取り/書き込み > 読み取り専用です。
    • [アプリケーション] をタップして、DSM アプリケーションのユーザーのアクセス権限を変更します。
  3. [終了] をタップして確定します。または、キャンセルするには左矢印をタップします。

共有フォルダの管理

[共有フォルダ] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして Synology NAS の共有フォルダのリストを表示します。

共有フォルダを作成する：

  1. + をタップして新しいフォルダの名前を入力します。
  2. [終了] をタップして確定します。または、キャンセルするには左矢印をタップします。

共有フォルダを変更する

共有フォルダの名前をタップして、以下のいずれかを行います。

  • [名前] 欄の共有フォルダの名前を変更します。
  • [ユーザー権限] をタップし、ユーザーをタップして以下の共有フォルダのアクセス権限を変更します：読み込み/書き込み：ユーザーは、共有フォルダのファイルやサブフォルダに変更を行うことができます。読込み専用：ユーザーは、共有フォルダのファイルやサブフォルダにアクセスできますが、変更を加えることはできません。アクセスなし：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできません。
    注意：権限の競合が発生した場合は、権限の優先順位は次のようになります：アクセスなし > 読み取り/書き込み > 読み取り専用です。
  • [グループの権限] をタップして、グループの権限を設定します。
    注意：グループ権限は、ユーザー権限よりも優先度が高くなります。

サービスを有効にする

[サービス] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして Synology NAS サービスを有効または無効にします。

Synology NAS の LAN サービスを管理する：

  • 以下のいずれかを有効または無効にします。
    • Windows のファイルサービスでは、Synology NAS を LAN 内の Windows (CIFS) ファイルサーバーにすることができます。
    • Mac ファイルサービスでは、Synology NAS を LAN 内の Mac (AFP) ファイルサーバーにすることができます。
    • NFS サービスは LAN 内の Linux クライアントが Synology NAS データへアクセスできるようにします。
    • Bonjour プリンタは Mac クライアントが Synology NAS へ接続した USB プリンタに印刷できるようにします。
    • メディアサーバーは LAN 内の DLNA/UPnP ホームデバイスが Synology NAS のコンテンツを参照および再生できるようにします。
    • iTunes は LAN 内の iTunes クライアントが Synology NAS の音楽やビデオを参照および再生できるようにします。

Synology NAS のインターネット サービスを管理する：

  • 以下のいずれかを有効または無効にします。
    • FTP は Synology NAS を FTP サーバーにすることができます。
    • Web サービスは Synology NAS で Web サイトをホストすることができます (PHP+MySQL のサポートで)。
    • WebDAV はクライアントのプログラムが HTTP プロトコルを介して Synology NAS を管理および編集できるようにします。
    • WebDAV (HTTPS) はクライアントのプログラムが HTTP プロトコルを介して Synology NAS を管理および編集できるようにします。
    • File Station は Web ベースのアプリケーションで、DSM にログインしないでカスタマイズポートからの Synology NAS ファイルの管理を可能にします。
    • File Station (HTTPS) は File Station に HTTPS プロトコルを介して接続します。
    • Surveillance Station は、IP カメラを管理して自宅や会社の環境を守るウェブベースのアプリケーションです。
    • Audio Station は Web ベースのアプリケーションで、Synology NAS の音楽ライブラリを利用することを可能にします。
    • Photo Station は、Synology NAS に保存されているブログと一体化したオンラインフォトアルバムです。
    • Download Station (BT/HTTP/FTP/NZB/RSS) は Web ベースのアプリケーションで、ファイルをインターネットから Synology NAS にダウンロードすることができます。
    • eMule は、Download Station の eMule サポートを有効にします。

注意：

サービスの詳細については、以下のいずれかを参照してください。

  • DSM ヘルプ (コンピュータの Web ブラウザで DSM のデスクトップエディションにログインし、[メイン メニュー] > [DSM ヘルプ] の順に選択)
  • Synology DSM ユーザーガイド（Synology ダウンロードセンターより入手できます）

システムリソースの監視

[リソース モニタ] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして Synology NAS の CPU とメモリの使用状況を監視します。

システム情報の表示

[システム情報] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして Synology NAS の状態を表示します。

Synology NAS のシステム情報を表示する：

  1. 以下のいずれかを行います。
    • [全般] をタップして、シリアル番号、モデル名、実メモリの容量、DSM バージョン、時間情報、温度の状態など、Synology NAS の基本情報を表示する。
    • [ネットワーク] をタップして Synology NAS のネットワーク設定と各インターフェイスの状態を表示する。
    • [ストレージ] をタップして Synology NAS のボリューム使用状況を表示する。
  2. 右上隅の [最新の情報に更新] ボタンをタップしてシステム情報を最新にする。

ファームウエア

DSM バージョンを表示、更新するには、[ファームウェア] をタップします。

DSM アップグレード設定を行う:

  1. [DSM のアップグレード] をタップします。
  2. 以下のいずれかのオプションを選びます：
    • 重要なアップデートのみ
    • 最新の DSM と全アップデート
  3. [完了] をタップします。

使用可能なアップデートをダウンロードする:

  1. [状態] で DSM バージョンを確認します。
  2. DSM が最新版ではない場合は、[状態] セクションの下にある [ダウンロード] ボタンをタップして DSM を更新します。

IP ブロックの管理

[IP ブロック] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして、不正なログインを防止します。

IP ブロックの管理：

  1. 右上にある [設定] ボタンをタップします。
  2. [自動ブロックを有効にする] をオンにして、IP ブロックを有効にします。
  3. [ログイン回数] および [有効時間（分）] を入力して、指定した時間（分）以内に、特定回数以上ログインに失敗した IP アドレスをブロックします。
  4. [ブロックの有効期限を有効にする] をオンにして、[この日数後にブロック解除] を指定すると、指定した日数が経過した後は IP アドレスのブロックが解除されます。ここで「0」を入力すると、ブロックされた IP アドレスはブロック リストから除去するまで継続してブロックされたままとなります。
  5. [終了] をタップします。

IP アドレスをホワイト/ブラック リストに追加する:

  1. 許可/拒否する IP をリストに追加するには、右上にある [追加] ボタンをタップして、[ホワイトリスト] または [ブラックリスト] をタップします。
  2. IP アドレスを入力した後は、[終了] をタップします。

パッケージ センター の管理

Synology は、サードパーティや Synology デザインのアプリケーションを簡単にインストールできるようにパッケージに統合し、Synology NAS で簡単に有効にできるようにしました。

パッケージ センター の管理：

ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) で [パッケージ センター] をタップして、パッケージ情報を表示したり、パッケージを実行したり、停止したり、最新版に更新したりすることができます。

インストールしたパッケージを起動または停止する：

パッケージの横にあるスイッチをタップして、それを起動または中止します。

パッケージを見る：

パッケージをタップすると、情報が表示されます。

パッケージの更新：

  1. [更新] タブをタップして、現在使用可能なアップデート バージョンを表示します。
  2. パッケージの右側にあるボタンをタップすると、最新版にアップデートできます。

ログまたは現在の接続を表示する

[ログ] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして、現在の接続状況をモニターしたり、Synology NAS サービスのログを見たりすることができます。

現在の接続状況をモニターする：

  • [概要] セクションで [最新の [number] ログ] をタップして、Synology NAS サービスに最近アクセスしたユーザーをリストします。

特定のサービスのログ記録を見る：

  • イベント ログまたはファイル転送ログセクションのアイテムをタップすると、関連するログのリストが表示されます。

携帯アプリの起動

DSM mobile には、iOS や Android デバイスにインストールした Synology 携帯アプリを起動する便利な機能名装備されています。

Synology 携帯アプリを DSM mobile で起動する：

ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) の下部にある以下のいずれかのアプリをタップします：

  • DS photo（Photo Station サービスが有効時にのみ使用可能）
  • DS audio（Audio Station サービスが有効時にのみ使用可能）
  • DS cam（Surveillance Station サービスが有効時にのみ使用可能）
  • DS file（WebDAV サービスが有効時に使用可能）

注意：

  • これらのアプリの詳細については、アプリのヘルプ ファイルを参照してください。

Synology NAS の操作設定
(再起動、シャットダウン、ビープ音)

DSM mobile を使用すると、iOS や Android モバイルデバイスを使って Synology NAS の再起動やシャットダウンを行うことができます。さらに、近くに複数の Synology NAS がある場合は、ビープ音を出させることで、見つけたい Synology NAS を簡単に探し出すことができます。

操作中に Synology NAS を設定する：

  1. ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) の右上隅にある [オプション] ボタンをタップします。
  2. 以下のいずれかを行います。
    • [再起動]、[はい] の順にタップして Synology NAS をシャットダウンする。
    • [シャットダウン]、[はい] の順にタップして Synology NAS をシャットダウンする。
    • [サーバー検出] をタップして、Synology NAS に警告音を鳴らさせます。[OK] をタップすると、警告音が停止します。

DSM をデスクトップモードに切り替える

設定アイコンをタップし、[デスクトップ モード] を選択すると、DSM のデスクトップ モードに切り替わります。

ストレージ マネージャを使う

ストレージ マネージャでは、システム ボリュームとディスクの使用状況と健康状態をすばやく見ることができます。

ボリュームの情報を見るには：

  1. [ストレージ マネージャ] をホームページ (iPhone/iPod touch または Androd 携帯電話)、または左パネル (iPad または Android タブレット) からタップします。
  2. [ボリューム] タブをクリックします。
  3. リストのボリュームをタップすると、詳細情報を見ることができます。

ディスクの情報を見るには：

  1. [ストレージ マネージャ] をホームページ (iPhone/iPod touch または Androd 携帯電話)、または左パネル (iPad または Android タブレット) からタップします。
  2. [HDD/SDD] タブをタップします。
  3. リストのディスクをタップすると、その健康状態と S.M.A.R.T. の情報を見ることができます。

S.M.A.R.T. テストを開始または中止する：

  1. [ストレージ マネージャ] をホームページ (iPhone/iPod touch または Androd 携帯電話)、または左パネル (iPad または Android タブレット) からタップします。
  2. [HDD/SDD] タブをタップします。
  3. リストのディスクをタップすると、情報を見ることができます。
  4. [S.M.A.R.T. テスト] をタップすると 、 S.M.A.R.T. テストを実行します。中止するには、[中止] をタップします。
