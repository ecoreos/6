アクセス性

内蔵のアクセス機能を活用すると、視覚障碍者の方のために DiskStation Manager で音声案内による DSM サービス/設定をナビゲートしたり、管理したりすることができます。

DSM のアクセス性の設定

アクセス性を最大化するには、操作環境を次のように設定するようお勧めします。

  • ローカル コンピュータに NVDA スクリーン リーダー（Windows 環境のみ）をインストールします。
  • 最新の Mozilla Firefox ウェブブラウザをインストールします。
  • ヘッドセットまたはスピーカーを使って、音量を調整します。
  • フォーカス モードで NVDA を使います（モードを切り替えるには、NVDA キーと Space を同時に押します）。デフォルトの NVDA キーが挿入されています。
  • 次のホットキーで DSM をナビゲートします：
    ホットキー 説明
    タブ 別なユーザー インターフェイス セクションに切り替え
    矢印キー 同じユーザー インターフェイス セクション内のアイテムに切り替え
    Enter アプリケーション/ウィンドウを開く
    Esc アプリケーション/ウィンドウを閉じる

アクセス性機能は、DSM の次のエリアで機能します：

  • DSM ログイン ページ、デスクトップ、タスクバー
  • メイン メニュー
  • コントロール パネル
  • File Station

注意：

NVDA の操作については、ユーザーガイドをご参照ください。

DSM キーボード ショートカット ガイド

DSM にはアクセス機能に加え、スムーズで簡単に操作を実行できるようにキーボード ショートカットが用意されています。Shift + ? でショートカットを検索します。

  • デスクトップ キーボードのショートカット：ウィンドウ操作などのホットキー
  • コンポーネント キーボードのショートカット：ユーザー インターフェイス コンポーネント操作用のホットキー（例：リスト、ボタン、検索欄）

デスクトップ関連のショートカットを保留するには、[デスクトップ キーボードのショートカットを有効にする] オプションをキャンセルしてください。
