電子メールアカウント

[個人] > [電子メールアカウント] を選択して Eメール アカウントを追加し、File Station に保管されているファイルをここで追加したEメール アドレスから添付ファイルとして送信することができます。

Gmail、Outlook、Yahoo!、AOL、カスタマイズした SMTP サーバーなどのEメール サービス プロバイダがサポートされています。

Eメール アカウントを追加する：

  1. [追加] をクリックし、設定したい Eメール アカウントのサービス プロバイダーを選択します。
  2. Eメール アドレス、パスワード、送信者名を入力してください。
  3. 複数のアカウントが設定されている場合、File Station にこのEメール アカウントをデフォルトで使用させるには、[この電子メールアカウントをデフォルトとして使う] にチェックマークを付けます。
  4. Eメール サービス プロバイダー (Gmail、Outlook、Yahoo!) がこの機能に対応している場合は、[連絡先の同期] をクリックするとEメールの連絡先が DiskStation と同期されます。連絡先は、前のステップで入力したEメール アカウントではなく、認証画面で指定したEメール アカウントから同期されます。したがって、認証画面に別のアカウントや意図しないアカウントが表示されたら、任意のEメール アカウントに手動で切り替えるか、前にステップで指定したアカウントに切り換えてください。
  5. [適用] をクリックして設定を保存します。

Eメール アカウントを編集する：

リストから任意のEメール アカウントを選択し、[編集] をクリックします。

Eメール アカウントを削除する：

リストから任意のEメール アカウントを選択し、[削除] をクリックします。

注：

  • 他に既存のアカウントがない場合を除き、デフォルトのアカウントを削除することはできません。現在デフォルトとして設定されているEメール アカウントを削除するには、デフォルトのアカウントを別のアカウントに変更してください。

デフォルトのアカウントを変更する：

リストからデフォルトに設定したいEメール アカウントを選択し、[デフォルトに設定] をクリックします。

ファイルを添付ファイルとして送信する：

  1. File Station でファイルを右クリックし、[添付ファイルとして送信] を選択します。
  2. 複数のEメール アカウントが追加されている場合は、任意の送信者のEメール アドレスをドロップダウン リストから選択することができます。
