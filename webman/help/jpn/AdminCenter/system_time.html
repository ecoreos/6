時間

現在の時刻を確認したり、サーバーの日付と時刻を手動で設定する、またはネットワークタイムサーバーを使用して自動的に設定したりできます。

You can check the current time, set the server's date and time.

タイムゾーンを設定する：

タイムゾーン ドロップダウンメニューでオプションを選択します。

手動で日付と時間を手動で設定する：

Choose Manually and select the date and time.

日付と時刻を NTP サーバーと同期する：

  1. [ネットワークタイムサーバーとの同期] を選択し、ネットワーク タイム サーバーのアドレスを入力するか、ドロップダウン メニューからサーバーを選択します。
  2. [今すぐ更新] をクリックして、DiskStation がサーバーと時刻を同期していることを確認してください。
  3. [適用] をクリックします。

注意： ネットワークタイムサーバーとデータと時間を同期させるには、有効な DNS サーバー IP がメイン メニュー > [コントロール パネル] > [ネットワーク]で指定されていること、および DiskStation がインターネットに接続されていることを確認してください。

ネットワークタイムプロトコル (NTP) サーバーについて

Network Time Protocol (NTP) の主な目的は、データネットワーク上のコンピュータシステムのクロックを同期することです。

以下に、いくつかの推奨されるNTPサーバーを示します。

  • time.apple.com
  • time.asia.apple.com
  • time.euro.apple.com
  • time.nist.gov

NTP サーバーについての詳細は、http://www.eecis.udel.edu/~mills/ntp/servers.html をご覧ください。
