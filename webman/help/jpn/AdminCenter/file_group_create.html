グループの構築

  1. [コントロール パネル] > [グループ] の順に選択します。
  2. [作成] をクリックします。これを実行すると、ユーザー作成ウィザードが起動します。
  3. グループの名前と説明を入力します（ 詳細は下のセクションを参照してください)。[次へ] をクリックします。
  4. ウィザードの手順に従ってグループの構築を完了してください。
  5. [適用] をクリックして終了します。

名前付けの規則：

  • グループ名は大文字と小文字を区別します。また、以下の記号を除き、1～32 の Unicode 文字で構成される必要があります：

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • 最初の文字にはマイナス記号やスペースは使えません。最後の文字にスペースは使用できません。
  • グループの説明は大文字と小文字が区別され、最高で 64 の表示可能な Unicode 文字で構成される必要があります。
