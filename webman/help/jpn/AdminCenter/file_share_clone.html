共有フォルダのクローン*

Btrfs ボリュームに共有フォルダをクローンして、ほぼ同時に既存の共有フォルダのコピーを作成することができます。共有フォルダはクローン時にわずかなスペースしか使用しません。

共有フォルダをクローンする：

  1. [コントロール パネル] > [共有フォルダ] を選択します。
  2. Btrfs ファイル システムで任意の共有フォルダを選択します。
  3. [作成] > [クローン] をクリックします。
  4. 新しい共有フォルダの名前と説明を指定し、必要であれば他のオプションも変更してください。(これらのオプションについての詳細は、共有フォルダの作成をお読みください)

注意：

  • Btrfs ボリュームの共有フォルダのみクローンできます。
  • クローンした共有フォルダは、元の共有フォルダと同じボリュームに作成されます。
  • 高度なデータ整合性保護の設定は、クローンした共有フォルダに適用され、変更することはできません。

____
* 一部のモデルでのみ対応しています。
