DC 出力*

[コントロール パネル] > [ハードウェアと電源] > [DC 出力] を選択すると、指定した時間に DC 出力を自動的にオン/オフにするように設定できます。

新しい出力オン/オフ予約を追加する：

  1. [作成] ボタンをクリックして [スケジュールを追加] ダイアログを開きます。
  2. [出力オン] か [出力オフ] を選択します。
  3. 出力オン/オフ予約の [日にち] と [時間] を設定します。
  4. [OK] をクリックします。
  5. [保存] をクリックします。

新しいスケジュールがスケジュールリストに追加されます。

スケジュールサマリーを表示する：

[概要] をクリックすると、出力オン/オフ予約の週間予定を参照できます。

注：

  • ひとつの DiskStation 対して最大 100 件のスケジュールが指定できます。
  • 次のような場合には、システムは DC 出力自動的にオフにできないかもしれません。
    • Storage Manager がタスクを実行中のとき
    • システムが共有フォルダを暗号化しているとき
    • システムがハードドライブ上で Secure Erase を行っているとき
    • DSM をアップグレードしているとき
    • パッケージセンターがタスクを実行しているとき
    • システムがデータベースをアップグレードしているとき
    • システムがデータバックアップ タスクを実行しているとき

____
* 特定のモデルでのみ使用できます。
