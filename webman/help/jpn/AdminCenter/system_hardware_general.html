全般

[コントロール パネル] > [ハードウェアと電源] > [全般] を選択すると、電源復旧、WOL（ウェイク オン ラン）、メモリ圧縮を有効にするオプションがあります。また、警告音の設定やファン速度モードも変更できます。

電源復旧

停電などが原因でシステムが突然シャットダウンした場合、電源復旧機能が電気復旧時に自動的にシステムを再起動します。

WOL（ウェイク オン ラン）*

Wake on LAN (WOL) を有効にすると、Synology Assistant、DS finder、その他の WOL アプリケーションを使って DiskStation の IP アドレス（または DDNS ホスト名）と MAC アドレスを入力して、ＬＡＮ やインターネットを介して DiskStation をオンにすることができます。

WOL（ウェイク オン ラン）は、実際の [電源] ボタンを押すか、DSM インターフェイスで [シャットダウン] をクリックして、DiskStation が正常にシャットダウンした後でなければ機能しません。シャットダウンした後も、正常に使えるコンセントに電源コードを差し込んだままにしておいてください。

注：

WOL 機能は 10GbE LAN ポート（DiskStation に搭載されている場合）ではサポートされていません。

  • WOL 機能は 10GbE LAN ポート（DiskStation に搭載されている場合）ではサポートされていません。
  • デフォルトでは、WOL 機能は DiskStation の 1GbE LAN ポートで有効になっており、無効にすることはできません。

ファン速度モード*

次のような速度オプションがあります。

  • 冷却モード：ファンは高速で稼働して DiskStation を冷やしますが、ノイズが大きくなります。
  • 低ノイズモード：ファンは低速で稼働するためノイズは低くなりますが、DiskStation が熱くなります。
  • 低電力：ファンは低速で稼働するためノイズは低くなりますが、DiskStation が熱くなります。また、DiskStation が冷却されると節電のためにファンが停止します。低電力モードは一部のモデルでしかご使用いただけません。

警告音の設定

あるイベントやエラーが発生すると、DiskStation が警告音を発します。リストされたイベントにチェックマークを付けると、エラーが発生したときに DiskStation が警告します。

警告音を止める：

  • DiskStation が警告音を発したら、[警告音停止] ボタンをクリックすると警告音が止まります。

メモリ圧縮*

メモリ圧縮を有効にすると、メモリ内で一番最近使用したデータを圧縮することで、読み込み時のシステムの全体的な応答性を向上できます。

複数のアプリケーションが実行中であり、メモリに書き込みが行われている場合は、ダイナミック メモリ圧縮でメモリ内のあまり使用しないアイテムのサイズを縮小し、メモリの使用量を最小化します。再びそのアイテムが必要になったときには、即座に解凍されます。これにより、ディスク上でのメモリ スワップ ファイルの読み書きの必要性が軽減されるため、効率と応答性を飛躍的に高めることができます。

メモリ圧縮を有効にするには、チェックマークを付けてください。

LED*

[コントロール パネル] > [ハードウェアと電源] > [全般] > [LED 輝度コントロール] を選択すると、DiskStation の LED インジケータの輝度を調整できます。

注意：

輝度設定は、DiskStation の LED インジケータに適用されます (電源、状態、LAN、ドライブの状態など)。

LED の輝度を調整する：

DiskStation の LED インジケータは、4 段階に輝度を調整できます。

  1. 輝度スケールのポインタをいずれかの方向へクリック・アンド・ドラッグします。
    • 右へ：LED の輝度を上げます。
    • 左へ：LED の輝度を下げます。
  2. [適用] をクリックして設定を保存します。

予定に則って LED の輝度を調整する：

LED インジケータは、予約時間になると輝度レベルを変えることができます。

  1. [予約の設定] をクリックします。
  2. 輝度タイプをクリックします。
    • デフォルトの輝度：最大輝度
    • 調整済みの輝度：輝度スケールで指定された輝度レベル
  3. 期間をクリックして、輝度予定をカスタマイズします。
  4. [OK] をクリックし、[適用] をクリックして設定を保存します。
