インデックス付きフォルダ

デフォルトでは、photo、video、および music などの共有フォルダ内のメディア ファイルにはメディア インデックス機能によりインデックスが付けられています。DMA デバイスと DSM のフォト、ビデオ、ミュージック アプリケーションに DiskStation 上の他のファイルにもアクセスさせるには、ここでインデックス付きのフォルダを追加することができます。

インデックス付きフォルダを追加する

  1. ［作成］ をクリックします。
  2. 表示されるウィンドウで、DMA デバイスと DMA アプリケーションで表示される名前を [名前] フィールドに入力します。
  3. [選択]をクリックしてインデックスを付けるフォルダを検索して、[OK] をクリックします。
  4. ファイルタイプセクションで、チェックボックスを選択してインデックスを付けるメディアファイルのタイプを指定します。DMA デバイスと DMA アプリケーションは、指定したファイルタイプに一致するメディアファイルを参照して再生することしかできません。
  5. [OK] をクリックして [保存] をクリックします。
    注：
  • File Station を使用してマウントしたリモート フォルダは、インデックス付きフォルダとして追加することはできません。
  • DiskStation では、最大 100 のインデックス付きフォルダーを追加することができます。

インデックス付きフォルダを編集する

  1. ユーザーリストのフォルダを選択し、 [編集] をクリックします。
  2. 設定を変更します(詳細については、インデックス付けフォルダを追加する を参照してください。)
  3. [OK] をクリックして [保存] をクリックします。

インデックス付きフォルダのリストからフォルダを削除する

  1. フォルダリストからフォルダを選択して、[削除] をクリックします。（Shift または Ctrl を同時に押すと、複数選択できます。）
  2. [はい]をクリックしてから、[保存] をクリックします。
    注：
  • フォルダを削除しても、実際には DiskStation から削除されません。DMA デバイスと DMA アプリケーションがコンテンツを参照したり、再生したりしないように保護するだけです。
