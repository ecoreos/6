メディア インデックス サービス

メディア インデックス サービスは DiskStation に保存されているフォト、ミュージック、ビデオなどのマルチメディア ファイルを自動的にスキャンして、マルチメディア アプリケーションがインデックス化できるようにメディア ライブラリにコンパイルします。アプリケーションは、メディアサーバー、Video Station、Audio Station、Photo Station、iTunes Server に含まれるパッケージ センターから選択できます。これらが DiskStation にインストールされると、「アプリケーション」欄に表示されます。

メディアファイルを再インデックス付けする：

インデックス付けされたフォルダ（デフォルトのフォルダとユーザー自身で作成したフォルダの両方）内のすべてのメディア ファイルのインデックスの構造を再構成するには、[再インデックス] をクリックします。この処理は背景で実行されます。

注意：

NFS プロトコルを使用するインデックス化したフォルダに追加したメディア ファイルは、自動スキャンされません。[再インデックス] をクリックして、手動で再インデックスする必要があります。
