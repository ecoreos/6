プリンタ

DiskStation をプリンタ サーバーとして使用すると、クライアント コンピュータやモバイル デバイスから DiskStation に接続されたプリンタにアクセスできます。また、Apple ワイヤレス印刷や Google Cloud Print にも対応していますので、iOS デバイスや Google 製品やサービスを使って遠隔地から印刷することもできます。

互換性のあるプリンタについては、www.synology.com をご覧ください。

プリンタの設定

下記の説明をお読みになり、USB プリンタまたはネットワーク プリンタを DiskStation に接続し、設定してください。

USB プリンタを設定する：

  1. プリンタを DiskStation の USB ポートに繋ぎ、電源を入れてください。
  2. [コントロール パネル] > [外部デバイス] > [プリンタ] を選択します。
  3. 設定したいプリンタを選択します。DiskStation は、USB ハブを介して接続されているプリンタは認識しませんのでご注意ください。
  4. [プリンタ マネージャ] をクリックして、[プリンタの設定] を選択します。
  5. ダイアログボックスで、次のモードの中から選択してください。
    • ネットワーク MFP： ローカル ネットワーク内のクライアントに、多機能プリンタの印刷、スキャン、FAX 機能を使用させます。Windows コンピュータでのみスキャン機能とファックス機能を使用できます。
    • ネットワークプリンタ： 印刷しかできないプリンタを使用する場合や、クライアントに印刷機能だけを使用させたい場合は、このモードを選択してください。
  6. Apple ワイヤレス印刷または Google Cloud Print を設定してください（任意）。
  7. [保存] をクリックします。

注意：

ネットワーク MFP には、一度に 1 台のクライアントしかアクセスできません。

ネットワーク プリンタを設定する：

  1. DiskStation と同じローカル ネットワークにネットワーク プリンタを接続し、電源を入れます。
  2. [コントロール パネル] > [外部デバイス] > [プリンタ] を選択します。
  3. [ネットワーク プリンタを追加] をクリックし、セットアップ ウィザードの指示に従ってください。

注意：

  • プリンタ名 プリンタ名は、セットアップ中に入力しなければなりません。最大 32 個の文字、数字、アンダーバーを使用できます。
  • プロトコル： 一般的に、HP ネットワーク プリンタは Socket、Canon ネットワーク プリンタは BJNP、その他は LPR となっています。
  • キュー名： LPR ネットワーク プリンタと IPP ネットワーク プリンタにはキュー名を入力する必要がありますが、Socket または BJNP プリンタには必要ありません。
  • ネットワーク MFP なし： DiskStation に接続したネットワーク プリンタは、現時点でネットワーク MFP モードを使用することはできません。
  • USB プリンタなし： Synology 製品で共有される USB プリンタをネットワーク プリンタとして設定することはできません。

プリンタの管理

「プリンタ」タブには、DiskStation に接続された USB やネットワーク プリンタを管理するためのオプションがあります。

  • 情報の取得： 選択したプリンタに関する情報（デバイス名、メーカー、状態など）が表示されます。
  • 削除： DiskStation からプリンタを削除し、接続を解除します。
  • プリンタ マネージャ： 次のオプションがあります。
    • プリンタの設定：Apple ワイヤレス印刷や Google Cloud Print などのプリンタの設定を行います（手順は下記の説明をお読みください）。
    • スプールスペースをクリーンアップ： すべてのプリンタの印刷タスクをすべてキャンセルします。
    • テストページを印刷する： テストページを印刷して、設定が正しくできているかどうかを確認します。
    • プリンタを開放する： プリンタとクライアントの接続を解除します（ネットワーク MFP モードのみ）。クライアントが多機能プリンタに接続して FAX を受信したり、ファイルをスキャンしている間は、タスクを遂行するのに時間がかかるかもしれません。必要であれば、接続を中止して、待機中の次のクライアントに先にアクセスさせることができます。

Apple ワイヤレス印刷を有効にする：

  1. [コントロール パネル] > [外部デバイス] > [プリンタ] を選択します。
  2. デバイス リストからプリンタを選択します。
  3. ｢プリンタ マネージャ] > [プリンタの設定] をクリックします。
  4. [Apple ワイヤレス印刷を有効にする] チェックボックスにチェックマークを付けます。
  5. プリンタのブランドとドライバを選択します。
  6. [保存] をクリックします。

設定が完了したら、テスト ページを印刷することができます。[プリンタ マネージャ] > [テストページを印刷する] をクリックしてください。

注意：

  • Apple ワイヤレス印刷を有効にすると、DSM 用 Bonjour プリンタ ブロードキャストも有効になります。
  • Apple ワイヤレス印刷は iOS 4.2 以降でご利用いただけます。

Google Cloud Print を有効にする：

  1. [コントロール パネル] > [外部デバイス] > [プリンタ] を選択します。
  2. デバイス リストからプリンタを選択します。
  3. ｢プリンタ マネージャ] > [プリンタの設定] をクリックします。
  4. [Google Cloud Print を有効にする] チェックボックスにチェックマークを付けます。
  5. 次の情報を入力します。
    • Google アカウント情報。
    • プリンタのブランドとドライバ。
  6. [保存] をクリックします。

設定が完了したら、テスト ページを印刷することができます。[プリンタ マネージャ] > [テストページを印刷する] をクリックしてください。

Google Cloud Print のタスクを管理するには、Google Cloud Print リンクをクリックしてください。

プリンタにアクセスする

PC または Mac を使って、アプリンタにアクセスするには、下のセクションをご覧ください。

Windows PC からプリンタにアクセスする：

DiskStation デバイスの印刷サービスにアクセスするには、Synology Assistant でプリンタに接続することが最良の方法です。詳しい手順は、ヘルプを参照してください。

Synology Assistant がコンピュータにインストールされていない場合は、他の方法で DiskStation デバイスの印刷サービスにアクセスしてください。

タイプ 1共有ネットワーク プリンタ

  1. お使いの PC のエクスプローラｰで [ネットワーク] を開き、DiskStation をダブルクリックします。
  2. [usbprinter] または [ネットワーク プリンタ名] をダブルクリックします。システムが正しいドライバをインストールするよう要求しますので、[はい] をクリックします。
  3. CD またはメーカーのウェブサイトからプリンタ ドライバを入手してください。メーカー名とプリンタの型番を選択します。または、[ディスク使用] をクリックして、ドライバがある場所を指定します。続いて [OK] をクリックします。
  4. システムがネットワークプリンタのセットアップを開始します。セットアップが完了したら、プリンタを使用することができます。

タイプ 2LPR ポート（推奨）

  1. お使いの PC で [スタート] をクリックし、[デバイスとプリンタ]</b> を選択します。
  2. [プリンタの追加] をクリックします。
  3. [ネットワーク、ワイヤレス、Bluetooth プリンタを追加する] を選択します。
  4. [必要なプリンタがリストされていません] を選択します。
  5. [TCP/IP アドレスまたはホスト名を使ってプリンタを追加する] を選択します。続行するには、[次へ]をクリックしてください。
  6. TCP/IP ポート デバイスにするデバイス タイプを選択します。
  7. DiskStation の IP アドレスを入力します。続行するには、[次へ]をクリックしてください。
  8. [デバイスの種類]：[カスタム] >　[設定] を選択します。続行するには、[次へ]をクリックしてください。
  9. プロトコルに LPR を選択し、キュー名に usbprinter または ネットワーク プリンタ名 を入力し、[LPR バイトカウントを有効にする] チェックボックスを選択します。[OK] をクリックして次に進みます。
  10. CD またはメーカーのウェブサイトからプリンタ ドライバを入手してください。メーカー名とプリンタの型番を選択します。または、[ディスク使用] をクリックして、ドライバがある場所を指定します。続いて [OK] をクリックします。
  11. システムがネットワークプリンタのセットアップを開始します。セットアップが完了したら、プリンタを使うことができます。

Mac を使ってプリンタにアクセスする：

Mac ユーザーは、DSM の Bonjour プリンタ ブロードキャストを有効にして、プリンタをシステム環境設定に追加することにより、DiskStation デバイスの印刷サービスにアクセスすることができます。

処理を行う前に、DiskStation が Postscript プリンタを使用しているか確認してください。

ステップ 1.DSM の Bonjour プリンタ ブロードキャストを有効にする

  1. admin または administrators グループに属するユーザーとして DSM にログインします。
  2. [コントロール パネル] > [ファイル サービス] > [Win/Mac/NFS] を選択します。
  3. [Bonjour プリンタのブロードキャストを有効にする] にチェックマークを付けます。

ステップ 2.Mac を使ってプリンタにアクセスする

  1. Mac コンピュータで [システム環境設定] > [印刷とスキャン] を選択し、次に + をクリックします。
  2. ドロップダウン メニューから usbprinter[number] または ネットワーク プリンタ名 を選択します。すると Mac が自動的にプリンタを設定します。
