グループメンバーの追加と削除

DiskStation ユーザーを作成したグループに追加したり、グループからメンバーを削除したりすることができます。

グループメンバーをグループに追加する：

  1. ユーザーを追加するグループを選択してから [メンバーの編集] をクリックします。
  2. [追加] をクリックします。
  3. メンバーリストからユーザーを選択し、[OK] をクリックします。（複数選択する場合は、[Shift] または [Ctrl] キーを押しながらユーザーをクリックします。）
  4. [終了]をクリックして設定を適用します。

    5. 注意： Administrators グループのメンバーは、DSM/CIFS/FTP/AFP/WebDAV などのアプリケーションを含め、DiskStation 上の権利を管理することができます。

グループからメンバーを削除する:

  1. ユーザーを削除するグループを選択してから [メンバーの編集] をクリックします。
  2. メンバーリストからユーザーを選択します。
  3. [はい] をクリックします。
  4. [終了]をクリックして設定を適用します。

    5. 注意： Administrators から削除されたユーザーは、DiskStation を管理することはできません。
