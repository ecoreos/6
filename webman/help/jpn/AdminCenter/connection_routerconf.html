ルーターの設定

DiskStation デバイスがLAN上にある場合は、他のデバイスをインターネットに接続できません。[コントロール パネル] > [外部アクセス] > [ルーターの設定] を選択してポート転送規則を設定すると、インターネットを介して DiskStation デバイスにアクセスできるようになります。

ルーターの設定

ポート転送規則を追加する前に、DiskStation デバイスとルーターの間で接続を設定する必要があります。

注：

  1. ポート転送規則を設定して、固定 IP アドレスを割り当てるには、ルーターへの管理者権限が必要です。
  2. DiskStation デバイスと UPnP ルーターを接続する前に、DSM が DiskStation の外部 IP アドレスを取得していることを確認してください（[コントロール パネル] > [外部アクセス] > [DDNS])。まだ取得していない場合は、DiskStation デバイスと UPnP ルーターのネットワークおよびファイヤーウォールの設定を確認してください。

DiskStation デバイスとルーターを接続する：

  1. [ルーターの設定] をクリックしてセットアップ ウィザードを起動します。次のステップに進む前に、DSM が次のネットワーク コンポーネント/設定をテストします。
    • ネットワーク インターフェイス
    • ゲートウェイ接続
    • インターネット接続
    • ポート転送
    • DNS
  2. ルーターのタイプによって、次のいずれかを行います：
    • ルーターが UPnP に対応していない場合は、ステップ 3 を参照してください。
    • UPnP ルーターと DiskStation デバイスとの互換性が確認されている場合は、ルーター情報の概要が表示されたら [適用] をクリックします。以上で DiskStation デバイスとルーターが接続されます。
    • UPnP ルーターと DiskStation デバイスの互換性が確認されていない場合は、次のいずれかを行ってください。
      • ルーターとの接続を自動的に設定するには、[自動] を選択して [次へ] をクリックしてシステムに互換性テストを実行させます。
        • DiskStation デバイスのルーター構成機能でルーターとの互換性が確認されている場合は、チェックボックスにチェックマークを付けてSynology にテストレポートを送信していただきルーターの互換性データベースを更新できるようご協力ください。その後で [適用] をクリックします。以上で DiskStation デバイスとルーターが接続されます。
        • DiskStation デバイスのルーター構成機能でルーターとの互換性がないことが確認されている場合は、チェックボックスにチェックマークを付けてSynology にテストレポートを送信していただきルーターの互換性データベースを更新できるようご協力ください。その後で [閉じる] をクリックしてセットアップ ウィザードを終了します。

          • 注：

          ルーターに互換性がないことが確認されている場合は、ルーターのユーザー インターフェイスでポート転送規則を構成することができます。詳しくは、ここを参照してください。

      • ルーターとの接続を手動で設定するには、[手動] を選択し、[次へ] をクリックした後、ステップ 3 に進んでください。
  3. DiskStation デバイスとルーターの接続を手動で設定する場合は、次の手順に従ってください。
    1. [ルーター] ドロップダウン メニューからルーターを選択します。

      2. 注：

      ルーターが見つからない場合は、[更新リスト] をクリックして、Synology から最新のリストを取得します。

    2. プロトコルとルーターのポートを設定します。プロトコルとポートは、使用するルーターの型式により異なります。デフォルトのプロトコル（HTTP）とポート（80）に接続できない場合は、ルーターに付属の説明書で正しい設定を確認してください。
    3. ルーターがカスタム アカウントを要求する場合は、[カスタム アカウントを使う] を選択し、[アカウント] と [パスワード] を入力します。
    4. [適用] をクリックして終了します。

ポート転送規則の管理

ポート転送規則は、使用するルーター専用のポートからパッケージを受信する DiskStation のポートを指定します。

ルーターで DiskStation をインターネットに接続している場合は、[作成] をクリックしてポート転送設定ウィザードを起動してください。

ポート転送ルールへの追加には 2 つの方法があります。

  • システムサービス（FTP、Web Station など）のポート転送ルールを作成する場合は、[組み込み式アプリケーション] を選択します。
  • その他のサービスのポート転送規則を作成する場合は、[カスタムポート] を選択します。

以下のシンタックスを使ってポート番号と範囲を指定します。

  • 「-」を使ってポートの範囲を指定します。例：6881-6890。
  • 「,」を使ってポートを区別します。例：“21,22”、“21,55536-55663”。

注：

  • 各ポートのゲートウェイルーターは、ひとつのフォワーディングに対してのみ使用します。ひとつのルーターポートで複数のルールを使うと競合の原因となり、画面に警告メッセージが表示されます。ルールで別のポートを設定して競合を避ける必要があります。
  • ルーターには、システムサービスフォワーディングにカスタムポートが使用できないものがあります。

ルールを保存する：

ルールを作成したら [保存] をクリックします。保存しない限りそのルールは適用されません。

ルールを保存した後は、適用されるまで数分待ちます（ルーターのモデルにより異なる）。

ルールが機能しているかテストする：

テストするルールを選択して、[テスト接続] をクリックします。テストの結果は、接続テストの結果カラムに表示されます。

ワイヤレス ルーターの設定*

DiskStation がワイヤレス ルーターとして使用されている場合、ポート転送ルールが DiskStation によって直接実行されます。

ポート転送規則を追加する：

  1. [作成] をクリックしてセットアップ ウィンドウを開きます。

  2. サービス名、プロトコル（TCP または UDP）、プライベート IP アドレス、プライベート ポート、パブリック ポートを指定します。以下のシンタックスを使ってポート番号と範囲を指定します。
    • 「-」を使ってポートの範囲を指定します。例：6881-6890。
    • 「,」を使ってポートを区別します。例：“21,22”、“21,55536-55663”。
  3. 一番下にある [作成] をクリックすると、設定が完了します。

注：

ポート転送規則を設定するときには、ほとんどのルーターでローカル マシンのポートとルーター ポートを変えることができます。お使いのルーターがこの機能に対応している場合：

  • 規則を保存するとき、ルーター ポートが既に既存の規則で使用されていると、新しいルーター ポートが割り当てられます。
  • 規則を追加、または編集するとき、ルーターのポート番号を直接指定できます。

DMZ*

ネットワーク マップの設定がワイヤレス ルーターに設定されているときには、DiskStation で DMZ を有効にすることができます。DMZ 内のホストは、制限やファイヤウォールの保護を受けずに、外部ネットワークに直接接続できます。これは通常サーバーの設定に使用されます。

DMZ を有効にする：

  1. [DMZ 設定] をクリックしてセットアップ ウィンドウを開きます。
  2. [DMZ を有効にする] のチェックボックスを選択します。
  3. DMZ ホスト IP アドレスを入力します。
  4. [適用] をクリックします。

注：

DMZ のホストに接続するには、外部ネットワークから、DiskStation により取得された外部 IP アドレスが必要です。

____
* 特定モデル、または特定環境でしか使用できません。
