DDNS

DDNS (Dynamic Domain Name Service) は、ホスト名をその IP アドレスにマップすることで、インターネットを使って簡単に DiskStation に接続できるようにします。例えば、DDNS はドメイン名 (例: www.john.synology.me) を使って DiskStation へのアクセスを許可します。IP アドレス (例: 172.16.254.1) を覚えておく必要はありません。

DDNS の設定

DiskStation の IP アドレスの既存のホスト名を指定するか、Synology または他の DDNS プロバイダが提供する IP アドレスを登録することができます。ホスト名の登録については、それぞれのプロバイダにお尋ねください。

DDNS ホスト名を設定する:

  1. [追加] ボタンをクリックします。
  2. 次の設定を行うためのダイアログ ボックスが表示されます。
    • サービスプロバイダ：サービスプロバイダを選択します。Synology が提供する無料のホスト名を登録するには、ドロップダウン メニューから [Synology] を選択します。[今すぐ登録] をクリックして、指示に従い情報を入力してください。
    • ホスト名：登録した DDNS ホスト名 (john.synology.me など)を入力します。
    • ユーザー名：DDNS プロバイダのユーザー名を入力します。
    • パスワード：DDNS プロバイダのパスワードを入力します。
    • 外部アドレス：ホスト名を使用する DiskStation の外部 IP アドレスを入力します。
    • ハートビート：マップしたホスト名の状態について、警告を発します (Synology が提供するホスト名のみ)。
  3. [テスト接続] をクリックして、正しく設定できているかを確認します。
  4. [OK] をクリックして、設定を保存後に終了します。

設定が完了したら、ウェブ ブラウザで DDNS ホスト名を入力すると、インターネットを介して DiskStation にアクセスできます。

条件と制限

  • DDNS サービス プロバイダのサーバーが作動していること。
  • DiskStation がインターネットに接続可能であること。
  • DDNS はプロキシ サーバーを必要としないネットワーク環境でのみ使用できます。
  • 1 つの DDNS プロバイダにつき、1 個のホスト名しか入力できません。

ホスト名の管理

次のボタンを使って、既存のホスト名を管理してください。

  • 削除：選択したホスト名をリストから削除します。
  • 編集：選択したホスト名の設定を変更します。
  • 今すぐアップデート：DDNS サービスが有効になっている、すべてのホスト名の状態をアップデートします。

カスタマイズした DDNS プロバイダーの設定

プロバイダーがサービス プロバイダー一覧に表示されない場合は、カスタマイズした DDNS プロバイダーを追加できます。

カスタマイズした DDNS プロバイダーを設定する：

  1. [カスタマイズ]をクリックします。
  2. 次の設定を行うためのダイアログ ボックスが表示されます。
    • サービスプロバイダ：カスタマイズしたサービス プロバイダーの名前を入力します。
    • クエリー URL：カスタマイズしたサービス プロバイダーの URL を入力します。
  3. [保存] をクリックして、カスタマイズしたサービス プロバイダーを DDNS サービス プロバイダーの一覧に追加してください。

条件と制限

  • クエリー URL はサービス プロバイダーが指定したプロトコルを遵守しなければなりません。この URL は、プロバイダーの DDNS サーバーに IP 情報を更新するために使用されます。
  • クエリー URL には、通常次の4つのパラメータが含まれています。クエリー URL を入力する時には、次のパラメータを相当する変数で置き換えてください。
    • ユーザー名：__USERNAME__
    • パスワード：__PASSWORD__
    • ホスト名：__HOSTNAME__
    • 外部アドレス：__MYIP__
  • クエリー URL にはスペースやセミコロンを含めることはできません。
