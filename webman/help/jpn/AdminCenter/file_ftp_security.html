セキュリティ設定

指定したユーザーのルートをユーザーのホームに変更

ここで選択したユーザーは、FTP経由で Synology DiskStation にログインした場合のみ自分のホームフォルダにアクセスできます

匿名 FTP を有効にする

この機能を有効にすると、誰でも DiskStation の FTP サービス（匿名または ftp アカウント）にログインできるようになります。

匿名ユーザーのアクセス権を設定する

メインメニュー > コントロール パネル > 共有フォルダ の順に進み、特権のセットアップをクリックします。

匿名ルートの変更

この機能を有効にすると、匿名の FTP ユーザーのログインを直接変更します。ドロップダウンメニューから共有フォルダを選択します。
