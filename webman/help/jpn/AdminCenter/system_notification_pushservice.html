プッシュ サービス

プッシュ　サービスは、特定のイベントやエラーが発生したときに､DiskStation からモバイル デバイスまたはウェブブラウザに向けて通知を送信させます。

ポート：

プッシュ サービスを有効にする前に、ポート 8089 を使って DiskStation がインターネットにアクセスできるか確認してください。

Synology から提供される Eメール通知を有効にする：

  1. [Synology のEメール サーバーを介して、システムの状態に関する通知を送信する] にチェックマークを付けます。
  2. [受信者] 欄には複数の Eメール アドレスを入力できます。通知はここで入力したアドレスに送信されます。2 つのアドレスを指定できます。複数のアドレスはセミコロン (;) で区切ってください。
  3. [適用] をクリックすると、[受信者] 欄で指定したアドレスに通知が送信されます。通知メールの手順に従ってください。
  4. 確認した後、[テスト メッセージの送信] をクリックした後、[OK] をクリックしてテスト通知を送信します。

モバイル デバイス通知を有効にする：

  1. [コントロール パネル] > [通知] > [プッシュ サービス] を選択します。
  2. [モバイル デバイス通知を有効にする] にチェックマークを付けて、[適用] をクリックします。
  3. モバイル機器に DS finder をインストールして起動し、DiskStation を [お気に入り] リストに追加してモバイル機器と DiskStation をペアリングします。
  4. [テスト メッセージの送信] をクリックした後、[OK] をクリックしてテスト通知を送信します。次に、モバイル機器で DS finder を起動してテスト メッセージを受信できることを確認してください。

ペアリングしたモバイル機器を管理する：

  1. [ペアリングしたデバイスを管理] をクリックして、ペアリングしたすべてのモバイル機器をリストします。
  2. デバイスのペアリングを解除するには、[ペアリング解除] をクリックします（Shift または Ctrl を押しながら選択すると、複数選択ができます）。再びそれらのデバイスをペアリングしなければ、通知は送信されません。

ブラウザ通知を有効にする：

  1. [モバイル デバイス通知を有効にする] にチェックマークを付けて、[適用] をクリックします。
  2. [ブラウザの通知を有効にする] をクリックして、DiskStation をブラウザとペア化します。
  3. 開いた画面で次のいずれかを行ってください。
    • Chrome の場合：[今すぐインストール] をクリックして Chrome 拡張をインストールし、まだ Google アカウントがない場合はサインインしてください。
    • Safari の場合：[許可] をクリックして通知サービスを認証します。
  4. ブラウザの通知ページに転送された場合は、[ペア化したデバイス] タブをクリックしてその情報を見たり、ブラウザから DiskStation を解除したりします。または、過去7日分のブラウザ通知履歴を見たい場合は、[イベント] タブをクリックします。

注意：

ブラウザ通知機能は、Chrome（35 以上）および Safari（OS X v10.9 以上）で対応しています。
