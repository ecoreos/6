詳細設定

DSM ブラウズセッションのために、次の詳細設定を [コントロール パネル] > [セキュリティ] > [詳細設定] で変更できます。

HTTP 圧縮

DiskStation に [HTTP 圧縮] を有効にすることができます。ネットワーク トラフィックの帯域を下げることによって、ウェブページの読み込み速度を高めることができます。

HTTP 圧縮を有効/無効にする：

  • [コントロール パネル] > [セキュリティ] > [詳細設定] で [HTTP 圧縮] にチェックマークを付けます/外します。

TLS / SSL プロファイル

HTTPS 暗号化接続のセキュリティ レベルを選択することができます。次の3つのレベルから選択できます。

  • 最新の互換性 - 最新のブラウザとモバイル機器にしか対応しません。
  • 標準の互換性 - 中間レベルのセキュリティです。これがシステムのデフォルト値になっています。
  • 上位互換性 - 安全ではありませんが、ほとんどのブラウザとモバイル機器に対応しています。
