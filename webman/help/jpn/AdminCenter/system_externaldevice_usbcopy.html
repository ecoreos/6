USBCopy

USBCopy 機能のターゲットロケーションを選択および設定できます。

USBCopyを有効にする：

  1. USBCopy を有効にするを選択します。
  2. ドロップダウンメニューから保存先の共有フォルダを選択します。
  3. OK をクリックします。

Synology DiskStation のフロントパネルの USB ポートに USB デバイス(デジタルカメラなど）を接続すると、前面パネルの[コピー]ボタンを押して、USBデバイスのデータを割り当てられた保存先の共有フォルダにコピーすることができます。USBCopyの保存左記フォルダが削除された場合は、システムにより自動的にパブリック共有フォルダに置き換えられます。

    制限:
  1. FAT または NTFS フォーマットの USB デバイスのみに対応しています。
  2. 必ず USB デバイスを Synology DiskStation のフロントパネルの USB ポートに挿入してください。
  3. 利用する共有フォルダで利用可能な容量はは、必ず USB デバイスで使用する容量より大きい必要があります。
