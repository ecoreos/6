PPPoE

DSL またはケーブルモードでインターネットに接続する場合、インターネットサービスプロバイダ（ISP) は PPPoE (Point to Point Protocol over Ethernet) を使用するため、[コントロール パネル] > [ネットワーク] > [ネットワーク インターフェイス] を選択して PPPoE を有効化し、アカウント情報を入力して、ルーターなしに直接インターネットに接続することができます。

PPPoE の詳細情報に関しては、ISPまたはネットワーク管理者にお問い合わせください。

IPv6 設定

DiskStation は IPv4/IPv6 デュアルスタックテーマをサポートしています。つまり、IPv4 および IPv6 の両方が DiskStation で同時に機能します。IPv6 のデフォルト設定は自動です。既存のネットワーク インターフェイスを選択し、[編集] ボタンをクリックして IPv6 タブを開き、設定を変更できます。

IPv6 が完全に IPv4 に引き継がれるまで、IPv4 アドレスを持つホストが IPv6 サービスにアクセスできるように、IPv6 トランジション メカニズムが必要となります。異なる IPv6 トランジション メカニズムから選ぶ：

  • 6in4：トンネル ブローカーから IPv6 ネットワークに接続します。IPv6 アドレス、Prefix の長さ、Prefix、トンネル ブローカーの IPv4 アドレスを入力する必要があります。これにはパブリック IPv4 アドレスが必要です。
  • 6to4：エニーキャストを使って、トンネル ブローカーから IPv6 ネットワークに接続します。これにはパブリック IPv4 アドレスが必要です。
  • DHCP-PD：LAN 内の prefix delegation を介して、DHCP サーバーから IPv6 prefix を取得します。これには ISP サポートが必要であり、ルーター モードでのみ作動します。
