SNMP

SNMP サービスは、ネットワーク管理ソフトウェアを使って DiskStation を監視します。[コントロール パネル] > [端末と SNMP] > [SNMP] を選択してください。現在、SNMPv1、SNMPv2c、SNMPv3 プロトコルに対応しています。

SNMP サービスを有効にする

[SNMP サービスを有効にする] を選択し、次の手順に従ってください。

SNMPv1、SNMPv2c サービスを有効にする：

  1. [SNMPv1、SNMPv2c サービス] にチェックマークを付けます。
  2. コミュニティ名を入力します。
  3. [適用] をクリックします。

SNMPv3 サービスを有効にする：

  1. SNMPv3 サービス にチェックマークを付けます。
  2. ユーザー名 と パスワード を入力します。もう一度パスワードを入力し、パスワードを確定します。
  3. [適用] をクリックします。

さらに、名前、場所、連絡先を入力すると、DiskStation の識別に役立ちます。このフィールドはオプションです。

注：

  1. SNMP サービスは、IPv4 と IPv6 に対応しています。
  2. SNMP サービスは、SNMPv1、SNMPv2c、SNMPv3 に同時に対応します。

SNMPv1、SNMPv2c コミュニティの制限：

コミュニティ名は、1 個から 64 個までの半角英数字に限られます。次の文字は使用できません。
" ' \ およびスペース。

SNMPv3 ユーザー名の制限：

ユーザー名は、1 個から 64 個までの半角英数字に限られます。次の文字は使用できません。
" ' \ およびスペース。

SNMPv3 パスワードの制限：

パスワードは半角英数字、記号、スペースを使用した 8～127 文字まで登録可能です。英字の大文字 小文字も区別されるので、気を付けてください。次の文字は使用できません。
" ' \ およびスペース。
