タスクスケジューラ

タスク スケジューラーは、指定された時間にサービスやスクリプトを実行する機能です。

サポートされるタスク

次のようなタスクを作成できます。

  • ユーザー指定のスクリプト： ユーザーが指定したスクリプトやコマンドを実行します。
  • ごみ箱： 特定の共有フォルダのごみ箱を空にします。また、保持ポリシーを設定しておくと、ごみ箱を空にするときにどのファイルを完全に削除するかを制御できます。例えば、サイズ、削除時間、ファイル形式に応じて削除したり、維持したりすることが可能です。
  • 警告音の設定： DiskStation の警告音を設定します。
  • サービス： Windows ファイル サービス、FTP、自動ブロックなどのネットワーク サービスを開始または中止します。
  • ユーザー指定のスクリプト： ユーザーが指定したスクリプトやコマンドを実行します。
  • ごみ箱： 特定の共有フォルダのごみ箱を空にします。また、保持ポリシーを設定しておくと、ごみ箱を空にするときにどのファイルを完全に削除するかを制御できます。例えば、サイズ、削除時間、ファイル形式に応じて削除したり、維持したりすることが可能です。
  • サービス： Windows ファイル サービス、FTP、自動ブロックなどのネットワーク サービスを開始または中止します。

タスク管理

タスク スケジューラのメイン ページには既存のタスクが表示され、タスクを作成、編集、起動するオプションを設定できます。

予約タスクを作成する：

予約タスクは指定した日付、時間、間隔で実行されるタスクです。

  1. [コントロールパネル] > [タスク スケジューラ] を選択し、[作成] をクリックした後で [予約タスク] を選択します。
  2. 予約するタスクのタイプを選択：[ユーザー定義したスクリプト]、[ごみ箱]、[警告音の設定]、[サービス]。
  3. [タスクの作成] 画面が開いたら、[全般] をクリックして次の設定を変更してください。
    • 全般設定： タスク名を入力し、タスクを実行するユーザーを選択します。次に、タスクを有効または無効にしてください。
  4. [スケジュール] をクリックし、次の設定を編集します。
    • 日付： タスクを実行する日を指定します。タスクは特定の曜日（毎日、週末、毎週日曜日など）、または特定の日（2015/10/21）に実行するように予約することができます。
    • 回数： 毎日何回タスクを実行するのか、また初回、および最終回の時刻を指定します。
  5. タスク専用の設定を整理するには、[タスクの設定] をクリックします（特定のタスクのみ）。
    • タスク実行の詳細を受け取るには、[Eメールで詳細な実行情報を送信] を選択します。
    • 異常が発生したときだけ通知を受け取るには、[スクリプトが異常終了した場合にのみ詳細な実行情報を送信] を選択します。
    • タスクの内容をカスタマイズするには、[ユーザー定義したスクリプト] にスクリプトを入力します。
    • ステップ 2 で [ごみ箱] または [サービス] を選択した場合は、いつ、どのようにごみ箱をクリアするか、サービスを開始/終了するかを決定することができます。
  6. [OK] をクリックして、設定を保存します。

発動したタスクを作成する：

発動したタスクは、特定のイベントが完了する前/後に行われたタスクです。

  1. [コントロールパネル] > [タスク スケジューラ] を選択し、[作成] をクリックした後で [トリガーされたタスク] を選択します。
  2. [ユーザー指定のスクリプト] を選択します。
  3. [タスクの作成] 画面が開いたら、[全般] をクリックして次の設定を変更してください。
    • 全般設定： タスク名を入力し、タスクを実行するユーザーを選択します。次に、タスクを発動させるイベント/プリタスクを選択し、タスクを有効または無効にしてください。
  4. タスク専用の設定を整理するには、[タスクの設定] をクリックします（特定のタスクのみ）。
    • タスク実行の詳細を受け取るには、[Eメールで詳細な実行情報を送信] を選択します。
    • 異常が発生したときだけ通知を受け取るには、[スクリプトが異常終了した場合にのみ詳細な実行情報を送信] を選択します。
    • タスクの内容をカスタマイズするには、[ユーザー定義したスクリプト] にスクリプトを入力します。
  5. [OK] をクリックして、設定を保存します。

注意：

  • 発動したタスクを作成するときには、そのプリタスクも指定する必要があります。作成したタスクは、プリタスクが実行されてから、その後で実行されます。
  • [スクリプトが異常終了した場合にのみ詳細な実行情報を送信] を有効にすると、スクリプトが 0 で終わるとシステムは詳細を配信しません。

タスク設定を編集する：

  1. 既存のタスクを選択します。
  2. [編集] をクリックします。
  3. すると、タスク設定を編集できる画面が開きます。

タスクを削除する：

  1. 既存のタスクを選択します。
  2. [操作] > [削除] をクリックします。

タスクを有効/無効にする：

  1. タスク スケジューラのメイン ページで、[有効] からオプションを選択/選択解除してタスクを有効/無効にします。
  2. [保存] をクリックします。

手動でタスクを実行する：

  1. 1 つ以上のタスクを選択します。
  2. [実行] をクリックします。

タスク スケジューラの制限：

  • タスク名には 512 個の表示可能な文字を使用できます。
  • タスクの最大数は：1024。
  • スクリプトパスまたはコマンド ラインの長さは：4095。
  • 警告音を鳴らす時間は、1 から 600 秒の範囲で設定できます。
