QuickConnect

QuickConnect を活用すると、ポート転送規則を設定しなくてもインターネットを介してアプリケーションが DiskStation に接続できます。

注意：

QuickConnect は DSM および、Synology が開発した Audio Station、Video Station、Download Station、Surveillance Station、Photo Station、File Station、Cloud Station などのパッケージや、モバイル アプリケーションと使用できます。

QuickConnect を有効にする：

  1. [コントロールパネル（シンプル モード）] > [QuickConnect] の順に選択します。
  2. [QuickConnect を有効にする] を選択します。
  3. 既存のSynology アカウント情報を入力するか、新しいアカウントを登録します。
  4. 新しい QuickConnect ID を指定します。いつでも DiskStation にアクセスできるように、覚えやすいものにしてください。
  5. [適用] をクリックします。

注意：

  • クライアント アプリケーション用に QuickConnect を有効にする前に、必要な DSM サービスが既に有効になっていることを確認してください。例えば、DS file 用 QuickConnect を有効にするには、DiskStation で WebDAV を有効にする必要があります。
  • QuickConnect ID には、アルファベット、数字、ハイフン（[-]）を使用できます。必ず文字を先頭に使用してください。
  • QuickConnect の性能を高めるには、[コントロールパネル（詳細設定モード）] > [外部アクセス]> [ルーター設定] を選択し、各サービスのポート転送を構成してください。
    • DSM：5000 (HTTP)、5001 (HTTPS)
    • Photo Station：80 (HTTP)、443 (HTTPS)
    • Cloud Station：6690

トラブルシューティング

「ネットワーク エラーが発生しました」というメッセージが表示された場合は、QuickConnect サービスがネットワーク エラーにより停止したことを意味しています。次のことを確認してください。

  • DiskStation がアクティブなネットワークに接続されていること（[コントロール パネル（詳細設定モード）] > [ネットワーク] > [ネットワーク インターフェイス]）。
  • DNS サーバーとデフォルトのゲートウェイ設定が正しく設定されていること（[コントロール パネル（詳細設定モード）] > [ネットワーク] > [全般]）。
  • 次のアクセス先ポートを使って DiskStation がインターネットにアクセスできること。22, 80, 443, 8888. 

ネットワーク エラーを修正できたら、QuickConnect 機能が数分以内に自動的に再開するはずです。
