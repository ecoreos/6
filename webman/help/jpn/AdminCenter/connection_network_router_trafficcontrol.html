ルーターのトラフィックコントロール

ルーターのトラフィックコントロールは、DiskStation がルーター モードのときに有効にできます。ルーターのトラフィックコントロールは（[コントロール パネル] > [ネットワーク] > [ルーターのトラフィックコントロール]）は、DiskStation プライベート ネットワークで行われているアップロードおよびダウンロードのトラフィックを制御します。規則を作成するときには、[保証帯域] と [最大帯域] を指定してアップロードとダウンロードのトラフィック制限を設定する必要があります。

用語説明

  • 保証帯域：保証帯域は、システム全体の帯域が十分にあるとき、このサービスが保証する送信トラフィックです。
  • 最大帯域：最大帯域は、システム全体の帯域が十分にあり、システムの余剰帯域があるとき、このサービスが使用できる送信トラフィックです。先にシステム出力帯域を計算するようお薦めします。次に、各サービスの保証帯域の合計がシステム出力帯域を超えていないか確認してください。そうしなければ、設定が正しく機能しない場合があります。

設定公式

  • システム出力帯域 = 各サービスの保証帯域の合計 + システムの余剰帯域
  • 各サービスの保証帯域 ≤ 各サービスの最大帯域

注意：

送信パケットがどのルーターのトラフィック コントロール規則にも一致しない場合は、デフォルトの動作には制限がありません。すべてのポートを選択し、保証帯域と最大帯域を指定すると、すべての規則の最後にユーザー独自のデフォルトのルーターのトラフィック コントロール規則を追加することができます。

ルーターのトラフィック コントロール規則を作成する：

  1. 右上にあるドロップダウン メニューからネットワーク インターフェイスを選択します。
  2. [作成] をクリックします。
  3. [ポート]セクションで、以下のいずれかを選択します。
    • [すべて] を選択して、すべてのポートに規則を適用します。
    • [カスタム] を選択し、[カスタム] をクリックします。最大で 15 個のポートをカンマでくぐってください。または、この規則を適用したいポート範囲を指定してください。
    • プロトコルを指定することもできます。
  4. [ソース IP]セクションで、以下のいずれかを選択します。
    • [すべて]を選択して、すべての IP アドレスにこの規則を適用します。
    • 特定 IP を選択し、[選択] をクリックします。この規則を適用する IP アドレス、またはIP 範囲を指定できます。

これでルールリストに作成したルーターのトラフィック コントロール規則が表示されます。

ルーターのトラフィックコントロール規則を削除するには、次のいずれかを行ってください。

  • リストの [有効] 欄でルールのチェックボックスを選択解除します。
  • ルーターのトラフィック コントロール規則を削除してください。

注意：

  • 各ネットワーク インターフェイス用に、最大 100 の規則を作成できます。
  • ドラッグ アンド ドロップ操作でリストの規則を並べ替えることができます。
  • 規則はリスト上の位置によって、優先順位を上げることができます。
  • 指定したポートは、ローカル ネットワーク外のインターネット サービスのポート番号を参照します。
  • ソース IP は、Synology NAS がルーターとして使用されるデバイスの IP アドレスを参照します。
