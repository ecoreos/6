同意/拒否

組み込み式ファイアウォールは、不正ログインを防ぎ、アクセスサービスを制御します。さらに、特定の IP アドレスからのネットワークポートへのアクセスを許可/拒否を選択することができます。

ファイアウォールルールを作成する：

  1. [コントロールパネル] > [ファイアウォール] を選択し、左側の欄からネットワーク接続の種類に基づいてネットワークインターフェイスを選択します。
  2. ［作成］ をクリックします。
  3. ［ポート］ セクションで、以下のいずれかを選択します:
    • ［すべて］ を選択して、すべてのポートにルールを適用します。
    • 組み込みアプリケーションの一覧から選択を選択します。
    • カスタマイズを選択してからカスタマイズをクリックして、最大 15 ポート（コンマ区切り）まで入力する、またはポート範囲を指定します。
    • プロトコルを指定することもできます。
  4. ［ソース IP］ セクションで、以下のいずれかを選択します:
    • [すべて] を選択してすべての IP アドレスを許可/拒否します。
    • [単一ホスト IP アドレス] を入力します。
    • [サブネット] を入力します。
  5. [アクション] セクションで、以下のいずれかを選択します:
    • [許可] を選択すると、上記のポートとソース IP の設定が一致するアクセスを許可します。
    • [拒否] を選択すると、上記のポートとソース IP の設定が一致するアクセスを拒否します。

これでルールリストに作成したファイアウォールが表示されます。

ファイヤーウォール ルールを削除するには、以下のいずれかを行います:

  • リストの [有効] 欄でルールのチェックボックスを選択解除します
  • ファイヤーウォール ルールを削除します

ルールリストの一番下にある [アクセスを許可する] たは [アクセスを拒否する] を選択して、ルールに一致しないアクセスを許可したり、拒否したりすることができます。

    注：
  1. 1 つのネットワークインタフェースに対して最大 100 種類のルールが作成できます。ドラッグ・アンド・ドロップでルールの順序を整理できます。
  2. ルールはリスト上の位置によって、優先順位を上げることができます。
  3. システムに同じサブネットに接続している 複数 のネットワークポートがあると、ファイアウォールルールが正しく機能しない場合があります。*
  4. Link Aggregation で複数 LAN ポートを結合すると、最初のネットワークの規則がファイアウォールに適用されます。2 つ目のネットワークインタフェースの規則は保存されます。

____
* 特定のモデルのみに適用されます。
