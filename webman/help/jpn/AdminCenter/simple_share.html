共有フォルダ

共有フォルダは、DiskStation でファイルやフォルダを保存するための基本的なディレクトリです。DiskStation にファイルを保管する前に、最低 1 個以上の共有フォルダを DSM に作成してください。共有フォルダに保管したデータは、カスタマイズ可能なアクセス権により個人的に保管するか、特定ユーザーやユーザー グループと共有することができます。

以下は、特定のアプリケーション、サービス、パッケージがインストールまたは有効になっているときに自動的に作成されるデフォルトの共有フォルダの例です。

名前 説明
public public フォルダは、DiskStation を初めて設定した後で自動的に作成されます。*
web web フォルダは、DiskStation にホストされるウェブサイトのコンテンツを保管しておくためのデフォルトのフォルダです。これは、Web Station を有効にすると、自動生成されます。
photo photo フォルダには、Photo Station で共有する写真やビデオが格納されます。これは、Photo Station または DLNA/UPnP メディア サーバーを有効にすると自動生成されます。
music music フォルダは、Audio Station で聴きたい音楽を保管しておくためのフォルダです。これは、Audio Station または DLNA/UPnP メディア サーバーを有効にすると自動生成されます。
video video フォルダは、DLNA/UPnP デバイスを介して見たいビデオを保管するためのフォルダです。これは、Video Station または DLNA/UPnP メディア サーバーを有効にすると自動生成されます。
surveillance surveillance フォルダには、Surveillance Station の記録が保存されます。これは、Surveillance Station を有効にすると、自動生成されます。これは読み込み専用であり、システム管理者しかアクセスできません。
home home フォルダは、各ユーザーが自分しかアクセスできないデータを格納するための個人用スペースです。これは、ユーザー ホーム機能を有効にすると自動的に作成されます。
homes homes フォルダには、すべてのユーザーのホームフォルダが格納されます。これは、ユーザー ホーム機能を有効にすると自動的に作成されます。システム管理者だけがアクセスでき、すべてのユーザーの home フォルダを管理します。
NetBackup NetBackup フォルダは、ネットワーク バックアップを有効にすると自動生成されます。
usbshare usbshare[number] フォルダは、USB 外部ドライブを DiskStation の USB ポートに接続すると、自動的に作成されます。*
satashare satashare フォルダは、eSATA 外部ドライブを DiskStation の eSATA ポートに接続すると、自動的に作成されます。*

注：

  • シンプルモードでは、ユーザーの共有フォルダの権限しか編集できません。ユーザー グループを作成したり、グループの共有フォルダへのアクセス権を管理したりするには、[コントロール パネル（詳細モード）] > [グループ] を選択してください。

共有フォルダを作成する：

  1. [コントロール パネル（シンプルモード）] > [共有フォルダ] を選択して、左側のパネルで + アイコンをクリックします。
  2. 名前を指定して、共有フォルダを作成するボリュームを選択します。
  3. 必要であれば、[ごみ箱を有効にする] にチェックマークを付けてください。このオプションを有効にすると、共有フォルダ内のファイルを削除したときに、それらのファイルは #recycle というフォルダに移動されます。ごみ箱にアクセスできるユーザーは、administrators グループに属するユーザーだけです。
  4. ユーザー個人の共有フォルダに対するアクセス権をカスタマイズするには、相当するボックスにチェックマークを付けるか、またはチェックマークを外してください。
    • 読み込み/書き込み：共有フォルダのファイルやサブフォルダにアクセスして変更を加えることができます。
    • 読み取り専用：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできます。
    • アクセス不可：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできません。
  5. [適用] をクリックして設定を保存します。

注：

  • 共有フォルダ名は大文字と小文字の区別があり、1～32 文字で構成される必要があります。名前に次の文字を使用することはできません。
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • 次の名前は、システム用に予約されています。
    .、..、global、home、homes、printers、public、satashare、surveillance、usbbackup、usbshare
  • 共有フォルダ名の最初の文字にマイナス記号またはスペースは使用できません。また、最後の文字にスペースは使用できません。
  • 権限が一致しない場合の優先順位は以下のとおりです。[アクセスなし] > [読み取り/書き込み] > [読み取り専用] です。
  • 新しい共有フォルダを作成するとき、administrators グループに属するユーザーの権限が「アクセスなし」に設定されている場合は、これらのユーザーは [コントロール パネル] > [共有フォルダ] でしか共有フォルダを見れません。
  • 「#recycle」フォルダを右クリックすると、File Station で「ごみ箱」フォルダの権限を設定できます。[プロパティ] を選択し、[許可] タブをクリックします。

共有フォルダを編集する：

  1. [コントロール パネル（シンプルモード）] > [共有フォルダ] を選択します。
  2. 左側のパネルで共有フォルダを選択します。
  3. 共有フォルダの名前、場所、ユーザー権限など一般的な情報を編集します。
  4. すでに [ごみ箱を有効にする] にチェックマークが付いている場合は、削除したすべてのファイルを永久にごみ箱に捨てるには、[ごみ箱を空にする] をクリックします。
  5. [適用] をクリックして設定を保存します。

共有フォルダを削除する：

  1. [コントロール パネル（シンプルモード）] > [共有フォルダ] を選択します。
  2. 左側のパネルで共有フォルダを選択します。
  3. 左側のパネルで [ごみ箱] アイコンをクリックして三択した共有フォルダを削除します。

____
* 特定のモデルのみに使用できます。
