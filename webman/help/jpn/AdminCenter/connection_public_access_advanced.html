ルーターポート

インターネット サービス（Google Docs や Pixlr）に DiskStation にアクセスさせるには、[コントロール パネル] > [外部アクセス] > [詳細設定] を選択して、パブリック、外部ホスト名とポート番号を指定することができます。

デフォルトにより、ホスト名はパブリック IP アドレス（または [コントロール パネル] > [外部アクセス] > [DDNS]）となり、ポート番号は 5000（または HTTPS 接続の場合は 5001）となります。ただし、DDNS ページで設定されていない別のホスト名がある場合、またはルーターでポート転送規則を設定するときに別のポート番号が指定された場合は、ホスト名とルーターのポート番号を入力して、インターネット サービスで共有するファイルに確実にアクセスできるようにしてください。

ルーターポート設定を有効にする：

  1. [ホスト名または静的 IP] 欄に DiskStation のホスト名またはパブリック IP アドレスを入力します。
  2. [DSM (HTTP)] 欄や [DSM (HTTPS)] 欄でルーターに DSM のポート転送ルールを設定した時に指定したルーターポート番号を入力します。
  3. [適用] をクリックします。
