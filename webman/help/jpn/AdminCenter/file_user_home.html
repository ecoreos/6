ユーザーホーム

ユーザーホームを有効にすると、guest を除く、それぞれのユーザーに専用のホームを作成できます。ユーザーはすべて、CIFS、AFP、FTP、File Station 経由で自分のフォルダにアクセスすることができます。

管理者は、 homes のデフォルト共有フォルダにあるすべての個人フォルダにアクセスできます。ホームフォルダの名前はユーザーアカウントと同じです。

ユーザーホームサービスを有効にする

  1. ユーザーホームをクリックします。
  2. ユーザーホームサービスを有効にするをチェックします。
  3. Synology DiskStation に1つ以上のボリュームがある場合は、 homes フォルダを保存するボリュームを選択します。
  4. OK をクリックします。

注意： ローカルユーザーのホームサービスを無効にすると、ドメインユーザーのホームサービスも同時に無効になります。

ユーザーホームサービスを無効にする

  1. ユーザーホームをクリックします。
  2. ユーザーホームサービスを有効にするのチェックを外します。

ユーザーホームサービスを無効にした後、 homes 共有フォルダは admin のみがアクセスできます。ユーザーホームサービスを再度有効にすると、ユーザーは自分のホームフォルダにアクセスできるようになります。

    注：
  1. homes 共有フォルダを削除するには、ユーザーホームサービスを無効にしてから行ってください。
  2. ローカルユーザーのホームサービスを無効にするとドメインユーザーのホームサービスも無効になります。
