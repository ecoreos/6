ユーザーのインポート

ユーザーリストをインポートしてユーザーアカウントの作成をバッチ処理することができます。

ユーザーリストを作成する：

方法 1テキスト エディタを使う：

  1. テキストエディタで新規ファイルを開きます。
  2. 次の順序でユーザーの情報をタブ区切りで入力します。
    • ユーザー名
    • パスワード
    • 説明
    • 電子メール
    • ボリューム 1 の容量（MB）（複数ボリュームを使用している場合はボリューム分のステップを繰り返してください）
    • ボリューム1の容量（MB）（複数ボリュームを使用している場合はボリューム分のステップを繰り返してください）

      注意：

      Btrfs ファイル システムと外部ストレージ デバイス (例：USB/eSATA ドライブ) では、ユーザーの割り当て量に対応していません。

  3. 次のラインに移動して、ステップ2 を全ユーザー分繰り返します。各ラインが 1 ユーザーの情報に対応します。
  4. UTF-8 エンコードでファイルを保存します。

方法 2Microsoft Excel を使う：

  1. Excel で新規ファイルを開きます。
  2. 次の順序でユーザーの情報を1行にコラム区切りで入力します。
    • ユーザー名
    • パスワード
    • 説明
    • 電子メール
    • ボリューム 1 の容量（MB）（複数ボリュームを使用している場合はボリューム分のステップを繰り返してください）

      注意：

      Btrfs ファイル システムと外部ストレージ デバイス (例：USB/eSATA ドライブ) では、ユーザーの割り当て量に対応していません。

  3. 次の行に移動し、ステップ 2 を全ユーザー分繰り返します。各行が 1 ユーザーの情報に対応します。
  4. テキストエディタで新規ファイルを開きます。スプレッドシートの内容をコピーしてテキストファイルに貼り付けます。
  5. UTF-8 エンコードでファイルを保存します。

注意：

  • ユーザー名を除くすべての欄は任意で入力してください。
  • 以下にテキストファイルの例を紹介します。
    • rosy 1234 母親 rosy@abc.com 80000
    chaddy 5678 兄弟 chaddy@abc.com 80000
    danny 8901 いとこ danny@abc.com 20000
    anny 2345 姉妹 anny@abc.com 20000

ユーザーリストをインポートする

  1. [作成] > [ユーザーのインポート] の順に選択します。
  2. ユーザーリストに重複するアカウントがある場合に上書きする場合は、[重複アカウントを上書きする] を選択します。
  3. [参照] をクリックして、ユーザーリストのファイルを選択します。ファイルを選択すると、プレビューエリアにインポートしたユーザーのリストが表示されます。内容が不正なユーザーのアカウントはインポートされません。
  4. [アップロード] をクリックしてユーザーアカウントを作成します。

注意：

ユーザーリストのファイルは、必ず UTF-8 形式にしてください。
