セキュリティ

このタブでは、各ネットワーク インターフェイスのセキュリティ設定を変更できます。

DoS 保護

DoS（サービス拒否攻撃）保護は、インターネット上の悪意のある攻撃を防御します。

DoS 保護を有効にするには：

  1. [DoS 保護を有効にする] にチェックマークを付けます。
  2. [適用] をクリックします。

VPN Pass-through*

DiskStation をルーターとして使用する場合、VPN pass-through はクライアントに DiskStation NAT を通過させ、VPN 接続のアウトバウンドとインバウンドを確立します。内部設定に VPN サーバーが含まれている場合は、[コントロール パネル] > [ルーターの設定] を選択し、ポート転送規則を設定して VPN サーバーに VPN 接続を確立させます。

VPN pass-through の設定を編集するには：

  1. ここでは、次の設定を有効/無効にできます。
    • PPTP Pass-through：PPTP (Point-to-Point Tunneling Protocol) トラフィックを通過させて、接続を確立します。
    • IPSec Pass-through：IPSec (IP Security Protocol) トラフィックを通過させて、接続を確立します。
    • L2TP Pass-through：L2TP (Layer 2 Tunneling Protocol) トラフィックを通過させて、接続を確立します。
  2. 設定を変更したら、[適用] をクリックします。

____
*一部のモデルでのみ対応しています。
