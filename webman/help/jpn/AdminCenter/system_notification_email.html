電子メール

E メール通知を有効にすると、システム エラー、ネットワーク接続の切断、ファンの異常、停電、バックアップ タスクの失敗などが発生した場合に、DiskStation から Eメールで通知が送られてきます。

電子メール通知を有効にすると、システム エラー、ネットワーク接続の切断、停電、バックアップ タスクの失敗などが発生した場合に、DiskStation から Eメールで通知が送られてきます。

E メール通知を有効にする：

  1. [コントロール パネル] > [通知] > [Eメール] を選択します。
  2. [E メール通知を有効にする] にチェックマークを付けます。
  3. [受信者] 欄には複数の Eメール アドレスを入力できます。通知はここで入力したアドレスに送信されます。2 つのアドレスを指定できます。複数のアドレスはセミコロン（;）で区切ってください。
  4. 必要であれば、[対象接頭辞] 欄に対象接頭辞を追加してください。この接頭辞は、DiskStation から送信されるメッセージの件名に追加されます。メッセージを識別したり、フィルタ化したりするのに便利です。
  5. サードパーティのEメール サーバー（Gmail、Outlook、Yahoo!）を使用してメッセージをリレーするには、下記の「サードパーティのEメール サービス」セクションをご覧ください。.カスタム サーバーを使用する場合は、「カスタム SMTP サーバー」セクションをご覧ください。

サードパーティの Eメール サービスを使う：

  1. [サービスプロバイダ] メニューで Gmail、Yahoo、Outlook など、サードパーティの Eメール サービスを選択します。
  2. 選択したEメール サービスのユーザー名とパスワードを入力します。
  3. [適用] をクリックして、設定を保存します。
  4. [テスト メールの送信] をクリックして正しく設定されたかどうかを確認してください。

カスタム SMTP サーバーを使う：

  1. [サービス プロバイダー] メニューから [カスタム SMTP サーバー] を選択します。
  2. [SMTP サーバ] 欄に SMTP サーバー名または IP アドレスを入力します。
  3. [SMTP ポート] 欄に SMTP ポート番号を入力します（ポート転送規則の設定時に別の SMTP ポートを指定した場合は、その番号を入力してください）。
  4. SMTP サーバーに認証が必要な場合は、[要認証] を選択して、E メールサーバーの [ユーザー名] と [パスワード] を入力します。
  5. SMTP サーバーに SSL/TLS 接続が必要な場合は、[安全な接続 (SSL/TLS) が必要] を選択します。
  6. DiskStation が送信するEメール メッセージに表示される送信者の名前をカスタマイズしたい場合は、[送信者の名前] 欄に名前を入力してください。信者のEメール アドレスをカスタマイズするには、[送信者のEメール] 欄にEメール アドレスを入力します。この欄を無記入にすると、[受信者] 欄の最初の Eメール アドレスが使用されます。
  7. [適用] をクリックして、設定を保存します。
  8. [テスト メールの送信] をクリックして正しく設定されたかどうかを確認してください。

注意：

  • 新しく作成されたユーザー アカウントにようこそメッセージを送信したい場合は、[新しいユーザーにようこそメッセージを送信する] にチェックマークを付けます。
  • Eメール通知の言語は、[通知言語] で設定できます。電子メール通知の言語を変更するには、[メイン メニュー] > [コントロール パネル] > [地域オプション] を選択し、[言語] タブをクリックした後、[通知言語]< セクションのドロップダウンメニューから優先言語を選択してください。
  • Gmail の SMTP サーバーを使用しており、2 段階認証を有効にしてある場合は、Google が SMTP 認証のために提供する「アプリケーション固有のパスワード」をご使用ください。

SMTP サーバーのドメイン名：

SMTP サーバは IP アドレスまたはドメイン名で指定できます。ドメイン名を使用する場合は、有効は DNS サーバー IP が [メイン メニュー] > [コントロール パネル] > [ネットワーク] ページに入力されていることを確認し、サーバーがインターネット上にあることを確認してください。
