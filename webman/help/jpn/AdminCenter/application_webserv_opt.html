HTTP サービス

デフォルトポート番号 80 に加えて、Photo Station および Web Station 用の別ポートを追加できます。

HTTP アクセスにポート番号を追加する：

  1. HTTP ポートの追加を選択します。
  2. フィールドにポート番号を入力します。HTTP アクセス用の追加ポートは 1 ～ 65535 で指定します。

以下のポートは使えません。

20、21、22、23、25、110、137、138、139、143、199、443、445、515、543、548、587、873、993、995、3306、3689、5000、5001、5005、5006、5335、5432、6881、8080、7000、7001、8081、9997、9998、9999、50001、50002、eMule デフォルトポート：4662 (TCP)、4672 (UDP)、FTP デフォルトポート範囲：実際の範囲は、モデル毎に異なります。

    注：
  1. HTTP アクセス用ポート番号は、FTP/BT サービスのポート番号と同じにできません。
  2. HTTP ポートと HTTPS ポートは互いに衝突しないようにしてください。
