USBCopy

USBCopy のコピー先を選択できます。[コントロール パネル] > [外部デバイス] > [USBCopy/SDCopy] の順に選択してください。

USBCopy を有効にする：

  1. [USBCopy を有効にする] を選択します。
  2. ドロップダウンメニューから保存先の共有フォルダを選択します。

Synology 製品のフロントパネルの USB ポートに USB デバイス(デジタルカメラなど）を接続すると、前面パネルの[コピー]ボタンを押して、USBデバイスのデータを割り当てられた保存先の共有フォルダにコピーすることができます。USBCopy の保存左記フォルダが削除された場合は、システムにより自動的にパブリック共有フォルダに置き換えられます。

制限事項

  1. FAT、NTFS、ext3、ext4 フォーマットの USB デバイスのみ対応しています。
  2. 必ず USB デバイスを DiskStation デバイスのフロントパネルの USB ポートに挿入してください。
  3. admin または administrators グループのユーザーだけが、USB デバイスにバックアップするのに必要な適切なアクセス権があります。
  4. 利用する共有フォルダで利用可能な容量は、必ず USB デバイスで使用する容量よりも大きくなければなりません。

SDCopy*

SDCopy のコピー先を選択できます。[コントロール パネル] > [外部デバイス] > [USBCopy/SDCopy] の順に選択してください。

SDCopy を有効にする

  1. [SDCopy を有効にする] を選択します。
  2. ドロップダウンメニューから保存先の共有フォルダを選択します。

Synology 製品のフロントパネルの SD スロットに SD デバイスを接続すると、前面パネルの[コピー]ボタンを押して、SD デバイスのデータをここで割り当てた保存先の共有フォルダにコピーすることができます。SDCopy の保存先フォルダが削除された場合は、システムにより自動的にパブリック共有フォルダに置き換えられます。

制限事項

  1. FAT、NTFS、ext3、ext4 フォーマットの SD デバイスのみ対応しています。
  2. 必ず SD デバイスを Synology 製品のフロントパネルの SD スロットに挿入してください。
  3. admin または administrators グループのユーザーだけが、SD デバイスにバックアップするのに必要な適切なアクセス権があります。
  4. 利用する共有フォルダで利用可能な容量は、必ず SD デバイスで使用する容量よりも大きくなければなりません。

____
* 特定のモデルでのみ使用できます。
