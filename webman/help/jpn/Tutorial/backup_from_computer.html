Cloud Station で PC/Mac のデータをバックアップする

概要

Cloud Station Server はプラットフォームをまたぐ同期ソフトウェアだけではなく、パソコンのスマートなバックアップ方法としても機能します。 リアルタイムのバックアップ ソリューショである Cloud Station Backup は、1個のファイルに対して最大 32 個の履歴バージョンを保持しています。

Synology NAS に Cloud Station Server を設定する

  1. [パッケージセンター] を選択し、[Cloud Station Server] を探したら、[インストール] をクリックします。
  2. Cloud Station Server を起動します。
  3. [概要] を選択し、[Cloud Station Backup] で [今すぐダウンロード] をクリックします(Synology Download Center から Cloud Station Backup をダウンロードすることができます)。
  4. コンピュータでインストーラを実行し、画面の指示に従って Cloud Station Backup をインストールしてください
  5. インストール後は、コンピュータで Cloud Station Backup を実行します。[今すぐスタート] をクリックします。
  6. Cloud Station Server を起動している Synology NAS のアドレス（または QuickConnect ID）、ユーザー名、パスワードを入力します（右側の検索アイコンをクリックすると、Cloud Station Backup が LAN 内にある別の Synology NAS を検索します）。[次へ] をクリックします。
  7. バックアップ ソースを選択し、同期したいサブフォルダのチェックマークを外します。
  8. Synology NAS でフォルダを選択し、[]選択 をクリックします。Cloud Station Backup は、ご使用中のコンピュータ名を使って選択した場所にバックアップ フォルダを作成します。
  9. 同期フォルダか規則を設定するには、[バックアップ規則] をクリックします。
  10. [次へ] をクリックしてバックアップ設定を確認し、 [完了] をクリックします。
  11. メイン アプリでバックアップの進行状況を確認してください。

同期したファイルの前のバージョンを Cloud Station からダウンロードまたは復元する

  1. [Cloud Station Server] > [バージョン履歴] を選択し、復元したいファイルやフォルダを探します。はせ
  2. 取得するファイルを選択し、［操作］をクリックします。[以前のバージョンを参照] をクリックします。ファイルが誤ってコンピュータ上から削除された場合は、[復元] をクリックします。
  3. ダウンロードしたいファイルのバージョンを選択し、[ダウンロード] をクリックします。または、選択したバージョンを復元したい場合は、[復元] をクリックすると、現在のバージョンに上書きされます。
