Hot Spare*

「Hot Spare ディスク」とは、劣化したボリューム/ディスク グループ/iSCSI LUN で故障したディスクを自動的に交換して補修するための予備のハードディスクのことです。Hot Spare ディスクは、ボリューム/ディスク グループ/iSCSI LUN が次の条件を満たす場合、全体的に割り当てて、DiskStation 内のボリューム/ディスク グループ/iSCSI LUN を保護することができます。

  • ボリューム/ディスク グループ/iSCSI LUN の RAID タイプが、データ保護を備えていること（例：RAID 1、RAID 5、RAID 6、RAID 10）。
  • Hot Spare ディスクのサイズが、必ずボリューム/ディスク グループ/iSCSI LUN で最小のディスクのサイズと同じ、またはそれ以上のサイズであること。

Hot Spare ディスクを設定する：

  1. [管理] をクリックしてください。
  2. 開いた画面に、各ボリューム/ディスク グループ/iSCSI LUN の Hot Spare の状態と、使用可能なディスクが表示されます。ボリューム/ディスク グループ/iSCSI LUN の状態には、次のような情報が含まれます。
    • 割り当て済み：ボリューム/ディスク グループ/iSCSI LUN は、最低1 台の Hot Spare ディスクで保護されています。
    • 利用できます：ボリューム/ディスク グループ/iSCSI LUN を保護できるディスクがありますが、Hot Spare が割り当てられていません。
    • 利用できません：ボリューム/ディスク グループ/iSCSI LUN を保護できる Hot Spare ディスクがありません。
  3. [有効なディスク] セクションにホットスペア ディスクとして割り当てることができるハードディスクが表示されます。ホットスペア ディスクを割り当てるには、ホットスペアとして割り当てようと思うディスクの隣のボックスにチェックマークを入れてください。
  4. [適用] をクリックして、設定を保存します。

注意：

  • あるハードディスクを Hot Spare ディスクとして割り当てると、そのディスクのデータはすべて消去されます。Hot spare ディスクを割り当てる前に、ディスクに重要なデータがないことを確認してください。
  • HDD によってアセンブルされた RAID は、SSD ホットスペア ディスクによって自動的に修復されず、その逆も同じです。

ボリューム/ディスク群/iSCSI LUN のためにホットスペアを設定するには：

  1. [設定] をクリックしてください。
  2. [Hot Spare ターゲット] タブで、ホットスペア ディスクによって保護することができるボリューム/ディスク群/iSCSI LUN が一覧表示されます。ボックスのチェックを外すと、対応するボリューム/ディスク群/iSCSI LUN は、劣化した場合、ホットスペア ディスクによって自動的に修復されることはありません。
  3. [クロスデバイス Hot Spare] タブで、別のデバイスにある劣化した RAID を修復するために、1 つのデバイスをホットスペア ディスクにすることができます。
  4. [OK] をクリックして、設定を保存してください。

____
*特定の機種でのみ利用できます。
