ファイルのダウンロード

ファイルやフォルダーを DiskStation からコンピュータにダウンロードするには、次の手順に従ってください。

ファイルをローカル コンピュータにダウンロードするには：

  1. ダウンロードするファイルまたはフォルダを選択します（Shift または Ctrl を押しながら選択すると、複数選択ができます）。
  2. [操作] > [ダウンロード] をクリックし、お使いのウェブブラウザーに応じて、必要なダウンロードモードを選択してください：
    • Google Chrome：個々のファイル（圧縮されていない）をダウンロードするには、[ダウンロード] をクリックします。Zipファイルにされた複数のファイルをダウンロードするには、[ .zipとしてダウンロード] をクリックします（[設定] > [全般] で指定される File Station コードページを使用して）。
    • その他のブラウザー：Zipファイルにされた1つのファイル、または複数のファイルをダウンロードするには、[ダウンロード] をクリックします（[設定] > [全般] で指定される File Station コードページを使用して）。
    • [キューに追加] - [上書き] または [キューに追加[ - [スキップ]：ファイルやフォルダを目的先のファイルやフォルダに上書きするかどうかにより、いずれかのオプションを選択してファイルをダウンロード キュー (右上隅にあるダウンロード矢印をクリックすると表示されます) に追加します｡
    注意：
  1. Windows 環境では、ファイル名に含まれた \ * ?" < > | ：の文字は、ダウンロード後に - に置き換えられます。
  2. Chrome が自動的に複数のファイルをダウンロードするのを止めるには、設定に進み、プライバシー セクションのコンテンツの設定を変更します。
  3. 匿名モードの Chrome は、圧縮されていないファイルとしての個々のファイルのダウンロードに対応していません。

ファイル/フォルダのダウンロードリンクを取得

ファイルまたはフォルダダウンロードリンクを取得して、他のユーザーとリンクを共有できます。

ファイル/フォルダのダウンロードリンクを取得する：

  1. ダウンロードリンクを取得するファイルまたはフォルダを選択します。
  2. 以下のいずれかを行ってプロパティウィンドウを開きます。
    • [操作] メニューから [プロパティ] を選択します。
    • ファイルまたはフォルダーの1つを右クリックし、[プロパティ] を選択します。
  3. ダウンロードリンクを右クリックして、ショートカットメニューから付属する機能を選択してダウンロードリンクにコピーします。機能の名前はご利用のウェブブラウザにより異なる場合があります。
  4. E メールまたはインスタントメッセージで他のユーザーにリンクを送信します。受信者は、自分のユーザー名とパスワードを入力して、リンクされているファイルやフォルダをダウンロードできます。

注意：ダウンロードリンクからファイルまたはフォルダーをダウンロードするには、有効なユーザー名とパスワードが必要です。
