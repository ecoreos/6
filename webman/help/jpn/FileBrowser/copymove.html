ファイル/フォルダのコピーと移動

File Station では、ファイル/フォルダを 2 台の DiskStation で、1 台の DiskStation 内で、あるいはローカルコンピュータに転送することができます。

ブラウザ間でファイル/フォルダをドラッグ・アンド・ドロップ

複雑なファイル転送プロトコルを設定しなくても、File Station を使ってファイル/フォルダを 1 台の DiskStation から別の DiskStation に直接アップロードすることができます。

ブラウザ間でのドラッグ・アンド・ドロップを有効にする：

  1. [設定] > [全般] > [File Station] を設定します。
  2. [ブラウザ間でのドラッグ・アンド・ドロップを有効にする] にチェックマークを付けます。
  3. [OK] をクリックします。
  4. 同じ種類の 2 つのウェブ ブラウザ（Firefox など）を開き、それぞれをソースとターゲットの DiskStation と接続します。
  5. 両方の DiskStation で File Station を起動します。以上で、File Station を使ってそれらの間でファイル/フォルダを直接ドラッグ・アンド・ドロップできるようになりました。

注：

  • 同じブラウザ内の 2つのタブの間でファイル/フォルダをドラッグ・アンド・ドラッグすることもできます。
  • Internet Explorer はブラウザ間、またはタブ間でのドラッグ・アンド・ドロップ操作に対応していません。
  • 複数のファイル/フォルダをドラッグ・アンド・ドロップする場合、ファイル/フォルダはアップロードしやすいように1つのアーカイブに圧縮されます。
  • ドラッグ・アンド・ドロップのファイルサイズの制限は、ウェブブラウザ毎に異なります：Safari (1 GB)。

DiskStation 内で、またはローカル コンピュータにファイル/フォルダをコピーまたは移動する

ファイルやフォルダを DiskStation 内で、またはローカルコンピュータに移動またはコピーするには、次の手順に従ってください。

方法 1：

  1. 移動またはコピーするファイル/フォルダを選択します。（Shift または Ctrl を押しながら選択すると、複数選択ができます。）
  2. 操作ドロップダウンメニューから切り取りまたはコピーを選択します。
  3. DiskStation またはコンピュータの目的のフォルダに移動します。
  4. 目的先のフォルダ内にあるファイルやフォルダと同じで名前で上書きするかどうかによって、操作 ドロップダウンメニューから貼り付け - 上書き または 貼り付け - スキップ を選択します。

方法 2：

  1. 移動またはコピーするファイル/フォルダを選択します。（Shift または Ctrl を押しながら選択すると、複数選択ができます。）
  2. フォルダまたはファイルをメインセクションから左側のパネルの別のフォルダにドラッグします。
  3. 開いたポップアップ メニューからコピー - 上書き、コピー - スキップ、移動 - 上書き、移動 - スキップを選択して、目的先のフォルダにある同じ名前のファイルやフォルダに上書きするか、あるいはスキップします。

方法 3

  1. 移動またはコピーするファイル/フォルダを選択します。（Shift または Ctrl を押しながら選択すると、複数選択ができます。）
  2. ファイル/フォルダを切り取る場合は Ctrl + X を押します。コピーする場合は Ctrl + C を押します。Mac の場合は、Command + X と Command + C を使用します。
  3. DiskStation またはコンピュータの目的のフォルダに移動します。
  4. ファイル/フィルタを貼り付ける場合は、Ctrl + V を押します（Mac の場合は Command + V。）
  5. 目的のフォルダに同じ名前を持つファイルがある場合、ファイルを上書きするか、スキップするかを開いたダイアログで選択してください。

キーボードのショートカット：

次のキーボード ショートカットを利用すると、File Station でファイルやフォルダを移動/コピーするのに便利です。

  • F2：別名を付ける
  • Delete：削除
  • Ctrl + A：すべて選択
  • Crtl + X：切り取り
  • Crtl + C：コピー
  • Crtl + V：貼り付け

注：

  • ファイル/フォルダを FAT ファイルシステムのディレクトリにコピーまたは移動する場合、ファイル名の「\ *?">|」の文字は「–」に置き換えられ、Mac ファイルの Metadata は削除されます。
  • ファイルをローカル コンピュータから切り取り、それらを DiskStation フォルダに貼り付けた場合は、ファイルはそのままローカル コンピュータに維持されます。
  • ファイルやフォルダを動かすときには、「所有者」はそのまま変わりません。ただし、ファイルやフォルダをコピーするときには、「所有者」はそれをコピーするユーザーに変わります。
