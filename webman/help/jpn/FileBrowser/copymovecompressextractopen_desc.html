ファイルの読込みと移動

File Station を使ってファイルを移動および読込みます。

ファイルまたはフォルダの圧縮

該当ディレクトリのファイルを圧縮します

ファイル/フォルダのコピーと移動

ファイル構造を認識します

ファイルまたはフォルダのアップロード

ローカルコンピュータから DiskStation へデータを移動します

ファイルのダウンロード

DiskStation からローカルコンピュータへファイルを移動します

アーカイブの解凍

選択したディレクトリのアーカイブを解凍します

新しいウィンドウでファイルを開く

表示中のブラウザを変更せずに選択したファイルを開きます

文書、写真、ビデオの表示

ビデオプレーヤーや写真ビューアでビデオを再生したり、写真をプレビューします

写真を編集

フォトを Web ベースのエディタで編集します

お気に入りまたはデスクトップ ショートカットを作成

フォルダをお気に入りに追加したり、デスクトップにファイルやフォルダのショートカットを作成したりします
