写真を編集

File Station では、Aviary Editor というウェブベースの外部フォト エディタを使って写真を編集することができます。

Aviary Editor で写真を編集する：

  1. 写真が保存されているフォルダを検索します。
  2. 写真を右クリックし、[Aviary で編集] を選択します。
  3. 新しいブラウザ画面に Aviary Editor が開きます。すぐに写真が表示されますので、提供されているツールで編集することができます。
  4. 編集が完了したら、ツールバーにある [保存] ボタンをクリックすると、すべての変更内容が保存されます。

注意：

  • Aviary Editor を活用すると、次のフォーマットの画像を編集できます。.jpg、.jpeg、.jpe、.png
  • Aviary Editor で対応できる最大ファイルサイズは 5 MB です。
  • Aviary Editor を正しく構成するためには、DiskStation のネットワーク設定が必要です。問題を解決するために、次の設定を確認してください。
    • QuickConnect で DiskStation に接続できるか([コントロール パネル] > [QuickConnect])、外部 IP アドレスを取得できるか ([コントロール パネル] > [外部アクセス] > [DDNS] で行うか、[コントロール パネル] > [ネットワーク] > [ネットワーク インターフェイス] > [PPPoE] で行います) を確認してください 。
    • DiskStation がルーターの後ろに配置されている場合は、ポート転送ルールが正しく指定されているか確認してください ([コントロール パネル] > [外部アクセス] > [ルーターの設定] )。
    • それでも Aviary Editor を開けない場合は、[コントロール パネル] > [外部アクセス] > [詳細設定] の順に選択して DiskStation のルーター ポートを指定する必要があります。
