ファイルまたはフォルダの圧縮

  1. メインエリアの DiskStation でファイルまたはフォルダを選択します。
  2. 操作ドロップダウンメニューの [“(ファイル名)”.zip に圧縮] を選択して圧縮を開始します。
  3. 圧縮が完了しても zip ファイルが見えない場合は、操作ドロップダウンメニューの最新の情報に更新を選択します。
  4. 名前の変更、アーカイブの暗号化、または圧縮レベル/モードを変えて圧縮する場合は、アーカイブに追加...を選択します。
    注：
  1. File Station を使ってローカルコンピュータのファイルまたはフォルダは圧縮できません。
  2. 対応アーカイブのフォーマット：.zip、.7z。
  3. [アーカイブに追加...] で .zip を アーカイブフォーマット として選択する場合、そのファイルのコードページを選択することができます。
