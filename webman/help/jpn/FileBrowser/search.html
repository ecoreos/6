ファイルやフォルダの検索

File Station のフィルタを使用すると、指定した条件に合ったファイルかフォルダだけを検索し、表示します。現在のフォルダ内のファイルやフォルダをフィルタにかけることができます。あるいは、高度な検索を行うと、すべてのサブフォルダもカバーしてより正確な結果を得ることができます。

現在のフォルダのファイルやフォルダを検索する：

  1. File Station を使用して検索するコンテンツを含んでいるフォルダに移動します。
  2. 右上隅の [フィルタ] 欄にキーワードを入力すると、ウィンドウのメインエリアにフィルタ化された結果が表示されます。

詳細検索を実施する：

  1. File Station を使用して検索するコンテンツを含んでいるフォルダに移動します。
  2. 右上にある［フィルタ］欄の隣の拡大鏡をクリックすると、詳細検索メニューが開きます。
  3. 下の検索条件を指定するか、ドロップダウン メニューから選択してください。
    • キーワード：キーワードを入力して一致する名前のファイルやフォルダを検索します。

      注意：

      • キーワードにワイルドカード [*] （0または1文字以上を意味します）と [?] を使用できます （1個のランダムな文字を意味します）。たとえば、[a*n] は [action]、[attention]、[an] などを代表します。
      • 複数のキーワードを分ける場合は、スペースをお使いください。例えば、[File Station] というキーワードを入力すると、[File]、[Station]、あるいはその両方が含まれる文字列が検出されます。
      • キーワードの先頭と最後に二重引用符 ["] を追加して完全一致、大文字を小文字を区別する検索を行うことができます。例えば、["Synology Inc."] は [Synology Inc.] を代表できますが、[Synology Incorporation] は代表できません。
      • 次の演算子は高度な検索結果を行いたい場合に使用します。
        演算子 例 使用
        && [X] && [Y]
      • [X] と [Y] の両方が含まれる結果が検出されます
        		•
        || [X] || [Y]
      • [X] と [Y]、またはその両方が含まれる文字列が検出されます
      • [X] と [Y] の両方が含まれる文字列は、一番上に表示されます
        		•
        + +[X]
      • [X] が含まれる文字列が検出されます
        		•
        - -[X]
      • [X] が含まれない文字列が検出されます
        		•
        ! ![X]
      • [X] が含まれない文字列が検出されます
        		•
        \ \[SpecialCharacter]
      • 特殊文字:   +   -   &&   ||    !    (    )    {    }    [    ]    ^    "    ~    *    ?    :   \
      • 上記のターゲットとする特殊文字が含まれた文字列が検出されます
        		•
    • ファイルコンテンツ検索を有効にする: キーワードが含まれるファイル コンテンツやメタデータを持つインデックスを付けたファイルを検索します。
    • 場所：高度な検索の対象範囲と深さをドロップダウン メニューから選択します。その他の共有フォルダを選択すると、検索範囲をカスタマイズすることもできます。
    • ファイルタイプ：拡張子を選択して特定のファイルタイプに検索範囲を制限するか、任意のフォルダを選択して検索対象のフォルダのみを選択します。
    • 日付：[変更日]、[作成日]、または [最後にアクセスした日時] を選択し、カレンダーから検索する日付を選択します。
    • サイズ (MB)：同等、以上、および以下から選択して、サイズを入力して一致するサイズのファイルを検索します。（同等を選択すると、指定されたサイズとほぼ同じサイズのファイルが検索結果に含まれます）。
    • 所有者/グループ: [所有者] または [グループ] を選択して、ドロップダウンメニューから選択して選択した所有者やグループに属するファイルやフォルダを検索します。

      注意：

      • 高度な検索を行うには、位置と最低1個は別の検索条件を指定する必要があります。
      • ISO フォルダまたはリモート フォルダがインデックス付きの共有フォルダにマウントされている場合は、高度な検索を使ってこの中にあるファイルを見ることはできません。
  4. 上で設定した条件に基づいて検索を開始する場合は検索、検索条件を指定し直す場合はリセットをクリックします。検索が完了すると、一致したファイルが File Station のメインエリアに表示されます。
