ファイルまたはフォルダのアップロード

以下の方法のいずれかを使ってファイルまたはフォルダをコンピュータから DiskStation にアップロードします。

下に示すアップロード機能を使用するためには、最新の Java Runtime Environment (JRE) と Flash Player を先にインストールしてください。

方法 1：

  1. 左側パネルの上部で Synology DiskStation の保存フォルダを選択します。
  2. [アップロード] をクリックして [アップロード - スキップ] または [アップロード - 上書き] を選択します。次にローカル コンピュータからファイルかフォルダを選択します。

方法 2：

  1. [設定] を選択し、[File Station でローカル コンピュータからファイルを参照] にチェックマークを付けてウェブページを更新すると、ローカル コンピュータのフォルダが左側のパネルの下側に表示されます。

    2. 注：

    [File Station でローカル コンピュータからファイルを参照] オプションは、コンピュータに JRE をインストールして有効にしなければ使用できません。

  2. 左側パネルの下部分で、ローカルコンピュータのフォルダを選択します。メインセクションに選択したフォルダのコンテンツが表示されます。
  3. メインセクションでファイルまたはフォルダを選択してから、[アップロード] を選択します。
  4. 表示されるウィンドウの DiskStation の保存先フォルダを選択します。
  5. 保存先フォルダと同じ名前のファイルまたはフォルダを上書きする場合は、画面右上の [上書き] を選択します。それ以外の場合は、[スキップ] を選択します。
  6. [OK] をクリックすると、アップロードが開始されます。

方法 3：

  1. [設定] を選択し、[File Station でローカル コンピュータからファイルを参照] にチェックマークを付けてウェブページを更新すると、ローカル コンピュータのフォルダが左側のパネルの下側に表示されます。
  2. 左側パネルで、コンピュータ（下部）からフォルダを DiskStation（上部）のフォルダにドラッグします。
  3. 開いたポップアップ メニューで、目的のフォルダに同じ名前を持つファイル/フォルダがある場合、それを上書きするか、スキップするかを選択してください。

方法 4：

  1. 左側パネルの上部で Synology DiskStation の保存フォルダを選択します。
  2. コンピュータのデスクトップかファイル エクスプローラー（Windows Explorer や Mac Finder など）でファイルを選択して、ウェブ ブラウザで File Station にドラッグします。
  3. 目的のフォルダに同じ名前を持つファイルがある場合、ファイルを上書きするか、スキップするかを開いたダイアログで選択してください。

    4. 注：

    • 方法 4 は、次のブラウザでしか使用できません。Chrome、Firefox、Safari 5.1、またはそれ以降です。ただし、[File Station でローカル コンピュータからファイルを参照] にチェックマークが付いているときは、Mac ではサポートされません。
    • フォルダをドラッグ アンド ドロップしたり、ファイルを空にしたりすることはできますが、上記のブラウザに対して [File Station でローカル コンピュータからファイルを参照] が選択されている場合に限ります。それ以外は、方法 4 でフォルダをアップロードしたり、ファイルを空にしたりすることはできません。
    • Chrome for Linux を使って HTTPS プロトコルで DSM に接続する場合は、信頼できる証明書をインポートして（[コントロール パネル] > [セキュリティ] > [証明書]）方法 4を使ってアップロードできるようにする必要があります。

注：

  1. アップロード キューには、最大で 1000 個までファイルを追加できます。
  2. Google Chrome、Firefox、Safari 5.1 またはそれ以上のバージョンを使用するか、[File Station でローカル コンピュータからファイルを参照] が選択されている場合は、4GB 以上のファイルをアップロードできます。それ以外は、1 回に最大 2GB のファイルしかアップロードできません。
  3. 非暗号化した共有フォルダ内のファイル/フォルダ名は最大 255 文字（ラテン系言語以外は最大 80 文字）とし、暗号化した共有フォルダ内では最大 143 文字（ラテン系言語以外は最大 47 文字）としてください。
  4. ファイル/フォルダ名には、 \ / : * ? " > | などの文字を使用することはできません。
  5. ファイル/フォルダ名を「._」で開始することはできません。
  6. 0KB のファイルは、JRE をインストールしなければアップロードできません。

トラブルシューティング

Safari を使って Mac に保管されているローカル アイテムを参照するには、次の手順に従ってください。

  1. Mac で Safari を開き、Safari メニューから [環境設定] を開きます。
  2. [セキュリティ] タブで [ウェブサイト設定を管理] をクリックします。
  3. [Java] で DSM のウェブサイトを選択します。
  4. [危険なモードで実行] を選択します。
