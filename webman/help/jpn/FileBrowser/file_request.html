ファイル要請の作成

ファイル要請は、非 DSM ユーザーにファイルをアップロードしてもらうために送信する招待状です。ファイル要請リンクを受け取った非 DSM ユーザーは、相当するユーザー アカウントやファイル アップロード権がなくても、複数のファイルを直接 DiskStation 上の File Station にアップロードすることができます。

ファイル要請リンクを作成する：

  1. [File Station] を選択し、アップロード先としてのフォルダを選択します。
  2. [操作] をクリックするか、ディスティネーション フォルダを右クリックして、[ファイル要請を作成] を選択します。
  3. ポップアップ画面で、次の情報を見たり/編集したりすることができます。
    • ファイル パス：ディスティネーションのディレクトリ パスをリストします。
    • リンク：ターゲットのユーザーと共有するファイル要請リンクがリストされます。
    • お名前：要請者の ID を示す名前を入力します。デフォルト値はあなたのユーザー名です。
    • メッセージ：ターゲット ユーザーへのメッセージをカスタマイズします。
    • パスワード保護を有効にする：パスワードを知っているユーザーだけが作成したリンクを介してファイルをアップロードできます。下のボックスにパスワードを指定します。
    • 有効期間：このファイル要請リンクの有効期間をカスタマイズします。
      • 停止時刻の設定：ファイルをアップロードできる期限を日付と時刻で指定します。
      • 開始時刻の設定：ファイルをアップロードできるようになる日付と時刻を指定します。
      • 許可されたアクセスの数：リンクに何回アクセスできるかを指定します。
  4. [保存] をクリックして、設定を完了します。

ファイル要請リンクを共有する：

ファイル要請 URL を探すには、[File Station] > [ツール] > [Shared Links Manager] を選択して任意のリンクをダブルクリックします。次のいずれかの方法でリンクを共有できます。

  • URL をターゲットのユーザーにコピーします。
  • デフォルトのEメール（あなたの DSM Eメール アカウント）か メール（サードパーティのEメール アプリケーション）でリンクを送信します。初めての方は、ウィザードの指示にしたがって使用する Eメール アカウントを設定してください。

ファイルをアップロードする：

リンクを介してファイルをアップロードするよう要求されたら、次の操作を行ってください。

  1. ローカル コンピュータかモバイル デバイス（Android か iOS）で与えられた URL を開きます。要求されたらパスワードを入力してください。
  2. アップロード ページであなたのIDを示す名前を入力します。
  3. ファイルをドラッグ・アンド・ドロップしてアップロードするか、[ファイルを追加] をクリックしてファイルを選択します。
  4. [アップロード] をクリックして、要請を完了します。

注意：

  • フォルダはファイル要請リンクを介してアップロードすることはできません。
  • [ウェブ] 共有フォルダ内のフォルダをディスティネーション フォルダとして指定することはできません。
  • アップロード機能はウェブ ブラウザごとに異なります。
    • Internet Explorer 8 および Internet Explorer 9 は、ドラッグ・アンド・ドロップ操作によるファイル アップロードには対応していません。
    • iOS デバイス上の Chrome は、画像のアップロードのみ対応しています。
    • iOS デバイス上の Safari は、画像と動画のアップロードのみ対応しています。
    • Android デバイスのウェブ ブラウザは、ファイルタイプの制限を受けません。

アップロードしたファイルを検索する：

同一ユーザーがアップロードしたファイルは、すべてディスティネーション内の同じフォルダに保管されます。デフォルトでは、ファイル名は Uploader_Name (Requestor_Name) となっています。

例えば、ファイルが Watson から Holmes にアップロードされた場合、ファイルは Watson (Holmes) という名前のフォルダにあります。

ファイル要請リンクの管理

ファイル要請リンクを変更する：

ファイル要請リンクを作成した後は、[File Station] > [ツール] > [Shared Links Manager] で設定を変更することができます。

  1. 任意のリンクをダブルクリックするか、またはリンクを選択し、[編集] をクリックします。
  2. ポップアップ画面で関連する設定を変更してください。
  3. [保存] をクリックして、新しい設定を適用します。

ファイル要請リンクを削除する：

  1. [File Station] > [ツール] > [Shared Links Manager] を選択します。
  2. 削除したいリンクを選択し、[削除] をクリックします。相当するファイル要請 URL が直ちに機能しなくなります。

リクエスト申請権限を割り当てる：

admin または administrators グループに属するユーザーは、ファイル要請リンクを作成できるように、ユーザーに権限を割り当てることができます。

  1. [File Station] > [設定] > [共有リンク] > [ファイル要請リンク] を選択します。
  2. リクエスト申請権限を割り当てるユーザー タイプを選択します。
  3. [OK] をクリックして、設定を保存します。
