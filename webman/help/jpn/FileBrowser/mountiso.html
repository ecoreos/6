仮想ドライブをマウント

File Station を使用するとディスクイメージのコンテンツにアクセスするために、仮想ドライブをマウントすることができます。

仮想ドライブをマウントするには：

  1. ディスクイメージを選択し、[ツール] > [仮想ドライブをマウント] を選択します。
  2. [参照] をクリックして、仮想ドライブをマウントするために DiskStation に空白のマウント先フォルダを選択または作成します。
  3. システムを起動またはリブートする度に、DiskStation を仮想ドライブにマウントするには、[スタートアップで自動的にマウントする] にチェックマークを付けてください。[マウント] をクリックして続行します。以上で仮想ドライブがマウントされているマウント先フォルダをナビゲートして、ディスクイメージのコンテンツにアクセスすることができます。

注意：

仮想ドライブおよび .iso ファイルは、共有フォルダの中のサブフォルダにしかマウントできません。

仮想ドライブをマウント解除するには：

  1. 仮想ドライブがマウントされるマウント先フォルダを右クリックして、[マウント解除] を選択します。
    注意：
  1. サポートされるディスクイメージ形式は、ISO 9660 および UDF ファイルシステムを使用した「.iso」ファイルです。
  2. 同時に稼働可能な仮想ドライブの数は 16 台です。
  3. 仮想ドライブのマウントは、高権限モードでのみサポートされています。この機能を使用するには、ホスト DSM で Docker パッケージを選択し、暗号化した共有フォルダを有効にする [Docker DSM] を選択します。次に、[編集] をクリックし、[高権限実行コンテナを使う] にチェックマークを付けます。
