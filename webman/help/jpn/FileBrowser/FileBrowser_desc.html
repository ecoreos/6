File Station

File Station は、DSM ユーザーが自分のウェブブラウザを使ってインターネット経由で DiskStation のフォルダにアクセスできるようにするファイル管理ツールです。2 つ以上の File Station を開き、すべての DiskStation データを File Station 間でドラッグ アンド ドロップして管理します。

File Station の要件

コンピュータで File Station を操作するために必用な基本システム

属性と特権の表示

データに関する情報を表示

ファイル構造の変更

ファイルまたはフォルダの構造を変更

ファイルの読込みと移動

File Station を使ってファイルを移動および読込みます

ファイルまたはフォルダの検索

ファイルやフォルダをフィルタするか、詳細検索を実行します

リモートフォルダをマウント

リモートフォルダをマウントして使用可能なネットワーク リソースをナビゲートします

仮想ドライブをマウント

仮想ドライブをマウントしてディスクイメージ コンテンツにアクセスします

ファイル共有リンク

リンクを共有して、DSM ユーザー クレデンシャルを持たない人も DiskStation ファイルをダウンロードできるようにします。
