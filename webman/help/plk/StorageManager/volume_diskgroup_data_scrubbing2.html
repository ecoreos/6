Czyszczenie danych*

Czyszczenie danych to funkcja do zarządzania danymi, która sprawdza woluminy lub grupy dysków. Istnieją dwa rodzaje czyszczenia danych:

  • Czyszczenie RAID: Funkcji tej można użyć dla wolumenów lub grup dysków, które korzystają z macierzy SHR (z trzech lub więcej dysków twardych), RAID 5 lub RAID 6. Naprawia ona wszelkie wykryte niespójności danych.
  • Czyszczenie systemu plików: Funkcji tej można użyć dla woluminów z systemem plików Btrfs. Po wykryciu niespójnych danych z sumą kontrolną ścieżka pliku zostaje zapisana w centrum logów. Do naprawy danych stanowczo zaleca się używanie kopii zapasowej.

Zaleca się regularne wykonywanie czyszczenia danych w celu zapewnienia spójności danych i uniknięcia utraty danych w razie awarii dysku twardego.

Aby przeprowadzić czyszczenie danych:

  1. Otwórz aplikację Zarządzanie przechowywaniem.
  2. Przejdź do Wolumin lub Grupa dysków.
  3. Wybierz wolumin lub grupę dysków, dla której chcesz przeprowadzić czyszczenie danych.
  4. Kliknij przycisk Zarządzaj.
  5. Wybierz Czyszczenie danych.
  6. Wybierz rodzaj czyszczenia danych. (Jeśli dostępny jest tylko jeden rodzaj, system automatycznie pominie ten krok.)

Uwaga:

  • Czyszczenie danych można przeprowadzić tylko wtedy, gdy stan woluminu lub grupy dysków jest Normalny.
  • Jeśli system zostanie zamknięty przed zakończeniem, należy ręcznie uruchomić czyszczenie danych ponownie od początku.

____
*Ta funkcja jest dostępna tylko w niektórych modelach.
