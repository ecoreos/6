Klonowanie folderu współdzielonego*

Możesz sklonować folder współdzielony na woluminie Btrfs, aby stworzyć niemal bezzwłoczne kopie istniejącego folderu współdzielonego. Klonowanie folderu współdzielonego w momencie utworzenia pochłania jedynie niewielką część pamięci.

Aby sklonować folder współdzielony:

  1. Wybierz polecenie Panel sterowania > Folder współdzielony.
  2. Wybierz żądany folder w systemie plików Btrfs.
  3. Kliknij pozycje Utwórz > Klonuj.
  4. Określ nazwę i opis nowego folderu współdzielonego i zmodyfikuj inne opcje zgodnie z potrzebą. (Szczegółowe informacje o tych opcjach można znaleźć w artykule Tworzenie folderu współdzielonego).

Uwaga:

  • Klonować można tylko foldery współdzielone znajdujące się na woluminach Btrfs.
  • Sklonowane foldery współdzielone zostaną utworzone na tym samym woluminie co oryginalny folder współdzielony.
  • Ustawienia zaawansowanej ochrony spójności danych zostaną zastosowane do sklonowanych folderów współdzielonych i nie można ich będzie zmienić.

____
*Ta funkcja jest dostępna tylko w niektórych modelach.
