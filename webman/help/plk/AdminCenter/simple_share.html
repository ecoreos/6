Folder współdzielony

Foldery współdzielone to podstawowe katalogi, w których można przechowywać pliki i foldery na serwerze DiskStation. Przed zapisaniem plików w urządzeniu DiskStation należy utworzyć w systemie DSM co najmniej jeden folder współdzielony. Dane w folderach współdzielonych mogą być poufne lub udostępniane określonym użytkownikom albo grupom w zależności od dostosowywanych uprawnień.

Poniżej przedstawiono listę domyślnych folderów współdzielonych, które są tworzone automatycznie po zainstalowaniu lub włączeniu pewnych aplikacji, usług lub pakietów.

Nazwisko Opis
public Folder public jest tworzony automatycznie po skonfigurowaniu serwera DiskStation po raz pierwszy.*
web Folder web to domyślna lokalizacja pamięci masowej na zawartość witryny internetowej hostowanej na serwerze DiskStation. Jest on tworzony automatycznie po włączeniu aplikacji Web Station.
photo Folder photo zawiera zdjęcia i filmy wideo udostępniane w aplikacji Photo Station. Jest on tworzony automatycznie po włączeniu aplikacji Photo Station lub Serwera multimediów DLNA/UPnP.
music Folder music zawiera muzykę, której chcesz słuchać w aplikacji Audio Station. Jest on tworzony automatycznie po włączeniu aplikacji Audio Station lub Serwera multimediów DLNA/UPnP.
video Folder video zawiera pliki wideo, które można przeglądać przy użyciu urządzeń DLNA/UPnP. Jest on tworzony automatycznie po włączeniu aplikacji Video Station lub Serwera multimediów DLNA/UPnP.
surveillance Folder surveillance służy do przechowywania nagrań z aplikacji Surveillance Station. Jest on tworzony automatycznie po włączeniu aplikacji Surveillance Station. Jest to folder tylko do odczytu, dostępny tylko dla administratora systemu.
home Foldery home udostępniają każdemu użytkownikowi prywatną przestrzeń do przechowywania danych, do której dostęp ma tylko dany użytkownik. Jest on tworzony automatycznie po włączeniu funkcji Użytkownik domowy.
homes Folder homes zawiera foldery główne wszystkich użytkowników. Jest on tworzony automatycznie po włączeniu funkcji Użytkownik domowy. Tylko administrator systemu ma dostęp do folderów domowych wszystkich użytkowników i może nimi zarządzać.
NetBackup Folder NetBackup jest tworzony automatycznie po włączeniu usługi Sieciowa kopia zapasowa.
usbshare Folder usbshare[numer] jest tworzony automatycznie po podłączeniu zewnętrznego dysku USB do portu USB serwera DiskStation.*
satashare Folder satashare jest tworzony automatycznie po podłączeniu zewnętrznego dysku eSATA do portu eSATA serwera DiskStation.*

Uwaga:

  • W trybie prostym można edytować tylko uprawnienia użytkownika do folderu współdzielonego. Aby utworzyć grupy użytkowników lub zarządzać ich uprawnieniami do folderu współdzielonego, przejdź do opcji Panel sterowania (tryb zaawansowany) > Grupa.

Aby utworzyć folder współdzielony:

  1. Przejdź do pozycji Panel sterowania (tryb prosty) > Folder współdzielony i kliknij ikonę + w panelu po lewej stronie.
  2. Wybierz wolumen, na którym chcesz utworzyć folder współdzielony, i podaj jego nazwę.
  3. Aby włączyć obsługę kosza na folderze współdzielonym, zaznacz opcję Włącz kosz. Po włączeniu tej opcji pliki usunięte z folderu współdzielonego są przenoszone do folderu o nazwie #recycle. Dostęp do kosza można ograniczyć do użytkowników należących do grupy administrators.
  4. Zaznacz lub usuń zaznaczenie pól wyboru poszczególnych użytkowników, aby dostosować ich uprawnienia dostępu do folderu współdzielonego:
    • Odczyt/Zapis:Użytkownik może zmieniać pliki i podfoldery w folderze współdzielonym.
    • Tylko do odczytu: Użytkownik może uzyskiwać dostęp do plików i podfolderów w folderze współdzielonym.
    • Brak dostępu: Użytkownik nie może uzyskiwać dostępu do plików i podfolderów w folderze współdzielonym.
  5. Kliknij Zastosuj, aby zapisać ustawienia.

Uwaga:

  • W nazwach folderów współdzielonych nie jest istotna wielkość liter i mogą one mieć od 1 do 32 znaków. Nazwy nie mogą zawierać następujących znaków:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Następujące nazwy są zarezerwowane dla systemu:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare.
  • Pierwszym znakiem nazwy folderu współdzielonego nie może być znak minus ani spacja, a ostatni znak nie może być spacją.
  • W razie wystąpienia konfliktu uprawnień będą one mieć następujący priorytet: Brak dostępu > Odczyt/zapis > Tylko do odczytu.
  • Kiedy podczas tworzenia nowego folderu współdzielonego uprawnienia użytkowników należących do grupy administrators zostaną ustawione na Brak dostępu, tacy użytkownicy będą mogli wyświetlić ten folder współdzielony tylko na stronie Panel sterowania > Folder współdzielony.
  • Ustawienia uprawnień dotyczące Kosza można ustawić w aplikacji File Station, klikając prawym przyciskiem myszy folder #recycle. Wybierz polecenie Właściwości, a następnie kliknij kartę Uprawnienia.

Aby edytować folder współdzielony:

  1. Przejdź do pozycji Panel sterowania (tryb prosty) > Folder współdzielony.
  2. W panelu po lewej stronie wybierz odpowiedni folder współdzielony.
  3. Zmodyfikuj informacje ogólne o folderze współdzielonym, takie jak nazwa, lokalizacja i uprawnienia użytkownika.
  4. Jeśli opcja Włącz kosz została już zaznaczona, kliknij przycisk Opróżnij kosz, aby trwale usunąć wszystkie usunięte pliki z Kosza.
  5. Kliknij Zastosuj, aby zapisać ustawienia.

Aby usunąć folder współdzielony:

  1. Przejdź do pozycji Panel sterowania (tryb prosty) > Folder współdzielony.
  2. W panelu po lewej stronie wybierz odpowiedni folder współdzielony.
  3. Kliknij ikonę Kosz na panelu po lewej stronie, aby usunąć wybrany folder współdzielony.

* Funkcja dostępna tylko w niektórych modelach.
