Folder współdzielony

Foldery współdzielone to podstawowe katalogi, w których można przechowywać pliki i foldery na serwerze DiskStation. Przed zapisaniem plików na serwerze DiskStation należy utworzyć co najmniej jeden folder współdzielony. Dane w folderach współdzielonych mogą być poufne lub udostępniane określonym użytkownikom albo grupom w zależności od dostosowywanych uprawnień. Foldery współdzielone można także szyfrować podczas ich tworzenia, zwiększając w ten sposób ochronę danych.

Domyślne foldery współdzielone

Poniżej przedstawiono listę folderów współdzielonych, które są tworzone automatycznie po zainstalowaniu lub włączeniu pewnych aplikacji, usług lub pakietów.

Nazwa Opis
public Folder public jest tworzony automatycznie po skonfigurowaniu serwera DiskStation po raz pierwszy.*
web Folder web to domyślna lokalizacja pamięci masowej na zawartość witryny internetowej hostowanej na serwerze DiskStation. Jest on tworzony automatycznie po włączeniu aplikacji Web Station.
photo Folder photo zawiera zdjęcia i filmy wideo udostępniane w aplikacji Photo Station. Jest on tworzony automatycznie po włączeniu aplikacji Photo Station lub Serwera multimediów DLNA/UPnP.
music Folder music zawiera muzykę, której chcesz słuchać w aplikacji Audio Station. Jest on tworzony automatycznie po włączeniu aplikacji Audio Station.
video Folder video zawiera pliki wideo, które można przeglądać przy użyciu urządzeń DLNA/UPnP. Jest on tworzony automatycznie po włączeniu funkcji Serwer multimediów.
surveillance Folder surveillance służy do przechowywania nagrań z aplikacji Surveillance Station. Jest on tworzony automatycznie po włączeniu aplikacji Surveillance Station. Jest to folder tylko do odczytu, dostępny tylko dla administratora systemu.
home Foldery home udostępniają każdemu użytkownikowi prywatną przestrzeń do przechowywania danych, do której dostęp ma tylko dany użytkownik. Jest on tworzony automatycznie po włączeniu funkcji Użytkownik domowy.
homes Folder homes zawiera foldery główne wszystkich użytkowników. Jest on tworzony automatycznie po włączeniu funkcji Użytkownik domowy. Tylko administrator systemu ma dostęp do folderów domowych wszystkich użytkowników i może nimi zarządzać.
NetBackup Folder NetBackup jest tworzony automatycznie po włączeniu usługi Sieciowa kopia zapasowa.
usbshare Folder usbshare[numer] jest tworzony automatycznie po podłączeniu zewnętrznego dysku USB do portu USB serwera DiskStation.*
satashare Folder satashare jest tworzony automatycznie po podłączeniu zewnętrznego dysku eSATA do portu eSATA serwera DiskStation.*

____
* Funkcja dostępna tylko w niektórych modelach.
