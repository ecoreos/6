Usługa HTTP

Oprócz domyślnego numeru portu 80 możesz dodać inny port dla Photo Station i Web Station.

Aby dodać dodatkowy numer portu dla dostępu HTTP:

  1. Zaznacz Dodaj dodatkowy port HTTP.
  2. Wprowadź w polu numer portu. Do celów dostępu przez HTTP możesz dodać dodatkowy numer portu od 1 do 65535.

Następujące porty należy pominąć:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, domyślne porty eMule: 4662 (TCP), 4672(UDP) i domyślny zakres portów FTP: rzeczywisty zakres zależy od modelu.

    Uwaga:
  1. Numery portów dostępu HTTP muszą być inne niż numery portów usługi FTP/BT.
  2. Porty HTTP i HTTPS nie mogą być ze sobą w konflikcie.
