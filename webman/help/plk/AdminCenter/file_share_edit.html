Edycja informacji ogólnych o folderze współdzielonym

Informacje ogólne o folderze współdzielonym, takie jak nazwa, opis i inne, można edytować.

Aby edytować informacje ogólne o folderze współdzielonym:

  1. Przejdź do pozycji Panel sterowania > Folder współdzielony.
  2. Wybierz folder współdzielony.
  3. Kliknij przycisk Edytuj.
