Usługa katalogowa

DiskStation może dołączyć do istniejącej już usługi katalogowej jako klient domeny Windows lub usługi LDAP. Istnieje możliwość zarządzania domeną lub uprawnieniami dostępu użytkowników LDAP do folderów współdzielonych i aplikacji DSM, a także włączania dla nich usługi folderu domowego. Wybierz Panel sterowania > Usługa katalogowa, aby dołączyć DiskStation do usługi katalogowej.
