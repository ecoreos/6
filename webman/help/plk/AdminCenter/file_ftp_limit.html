Ograniczenie połączenia

Zezwól tylko na połączenia SSL/TLS

Kiedy ta funkcja jest włączona, serwer akceptuje od klientów FTP tylko żądania połączeń SSL lub TLS.

Limit połączeń według IP

Kiedy ta funkcja jest włączona, serwer ograniczy maksymalną liczbę połączeń na IP. Włącz tę funkcję oraz Ograniczenie szerokości łącza FTP aby efektywnie zarządzać przepustowością łącza.

Włącz ograniczenie łącza FTP

Włącz tę funkcję, aby ograniczyć przepustowość pojedynczego połączenia FTP.
