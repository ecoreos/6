Tworzenie grupy

  1. Przejdź do strony Panel sterowania > Grupa.
  2. Kliknij przycisk Utwórz. Uruchomi się Kreator tworzenia użytkownika.
  3. Wprowadź nazwę i opis grupy. (Aby uzyskać więcej informacji, zobacz sekcję poniżej). Kliknij przycisk Dalej.
  4. Postępuj zgodnie z instrukcjami kreatora, aby ukończyć tworzenie grupy.
  5. Kliknij Zastosuj, aby zakończyć.

Konwencje nazewnicze:

  • W nazwie grupy ważna jest wielkość liter. Powinna zawierać od 1 do 32 znaków Unicode z wyłączeniem symboli:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Pierwszym znakiem nie może być minus ani spacja, a ostatnim znakiem nie może być spacja.
  • W opisie grupy ważna jest wielkość liter i powinien on zawierać do 64 wyświetlanych znaków Unicode.
