Włączanie Kosza

Po włączeniu Kosza w folderze współdzielonym zostanie utworzony podfolder do przechowywania usuniętych plików. Usunięte pliki można odzyskać z Kosza. Jeżeli chcesz usunąć pliki nieodwracalnie, opróżnij Kosz.

Aby włączyć Kosz:

  1. Wybierz folder współdzielony, a następnie kliknij Edytuj.
  2. Zaznacz pole wyboru Włącz kosz.
  3. Zaznacz pole wyboru Ogranicz dostęp wyłącznie do administratorów, jeżeli Kosz ma być dostępny wyłącznie dla administratorów.
  4. Kliknij OK aby zapisać ustawienia.

Aby opróżnić Kosz:

Kliknij przycisk Opróżnij kosz, aby trwale usunąć wszystkie usunięte pliki z Kosza.

Aby opróżnić wszystkie Kosze:

Wybierz pozycję Opróżnij wszystkie Kosze z listy rozwijanej Akcja, aby nieodwracalnie usunąć pliki przeniesione do wszystkich Koszy.

Uwaga: Ustawienia uprawnień dotyczące Kosza można ustawić w aplikacji File Station, klikając prawym przyciskiem myszy folder #recycle. Wybierz polecenie Właściwości, a następnie kliknij kartę Uprawnienie.
