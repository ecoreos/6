Automatyczne blokowanie

Automatyczne blokowanie pozwala zapobiegać nieautoryzowanemu logowaniu. Po włączeniu tej usługi adres IP zostanie zablokowany, jeśli miał zbyt wiele nieudanych prób logowania. Wszystkie nieudane próby logowania przez SSH, Telnet, rsync, usługi sieciowej kopii zapasowej, synchronizacji folderów współdzielonych, FTP, WebDAV, aplikacje mobilne Synology, File Station lub DSM są sumowane. Można także skonfigurować Listę dozwolonych zawierającą adresy IP, które nigdy nie będą blokowane.

Aby skonfigurować automatyczne blokowanie:

  1. Przejdź do karty Ogólne i zaznacz pole wyboru Włącz automatyczne blokowanie.
  2. Wprowadź wartość Próby logowania oraz W ciągu (minut), aby zablokować adres IP po ustawionej liczbie nieudanych prób logowania w określonej liczbie minut.
  3. Zaznacz Włącz wygaśnięcie blokowania i wprowadź liczbę, aby usunąć zablokowany adres IP po określonej liczbie dni. Zablokowane adresy IP pozostaną na liście blokady do ich ręcznego usunięcia, jeżeli ta opcja nie zostanie wybrana lub jeżeli ustawienie Odblokuj po (dni) będzie mieć wartość 0.
  4. Kliknij przycisk Zastosuj.

Aby dodać zablokowane adresy IP:

Kliknij kartę Lista blokowania i wybierz dowolną z następujących opcji z menu rozwijanego Utwórz:

  • Dodaj adres IP

    • Wprowadź adres IP i ustaw czas wygaśnięcia.

  • Importuj listę adresów IP

    • Zaimportuj plik tekstowy z listą adresów IP, aby jednocześnie dodać większą liczbę adresów IP i ustawić czas wygaśnięcia. Zaznacz odpowiednie pole wyboru, jeżeli chcesz zastąpić istniejące już adresy IP. Jeżeli ta opcja nie zostanie zaznaczona, zdublowane adresy IP będą pomijane.

    Aby można było zaimportować plik, musi on spełniać następujące kryteria:

    • Musi to być plik zawierający zwykły tekst.
    • Każdy wiersz pliku może zawierać tylko jeden adres IP.
    • Wiersze komentarzy muszą zaczynać się od znaku # (krzyżyka).

      • Jeżeli wybrany plik ma prawidłowy format, wszystkie adresy IP zostaną wyświetlone w obszarze podglądu. Kliknij przycisk OK, aby zaimportować.

Aby usunąć zablokowane adresy IP:

  1. Kliknij kartę Lista blokowania, aby wyświetlić listę zablokowanych adresów IP.
  2. Wybierz adresy IP, które chcesz usunąć z listy, i kliknij Usuń.

Aby dodać dozwolone adresy IP:

Kliknij kartę Lista dozwolonych, a następnie wybierz dowolną z następujących opcji z menu Utwórz:

  • Dodaj adres IP

    • Wprowadź adres IP.

  • Importuj listę adresów IP

    • Zaimportuj plik tekstowy z listą adresów IP, aby jednocześnie dodać większą liczbę adresów IP. Zaznacz odpowiednie pole wyboru, jeżeli chcesz zastąpić istniejące już adresy IP. Jeżeli ta opcja nie zostanie zaznaczona, zdublowane adresy IP będą pomijane.

    Aby można było zaimportować plik, musi on spełniać następujące kryteria:

    • Musi to być plik zawierający zwykły tekst.
    • Każdy wiersz pliku może zawierać tylko jeden adres IP.
    • Wiersze komentarzy muszą zaczynać się od znaku # (krzyżyka).

      • Jeżeli wybrany plik ma prawidłowy format, wszystkie adresy IP zostaną wyświetlone w obszarze podglądu. Kliknij przycisk OK, aby zaimportować.

Aby usunąć dozwolone adresy IP:

  1. Kliknij kartę Lista dozwolonych, aby wyświetlić listę dozwolonych adresów IP.
  2. Wybierz adresy IP, które chcesz usunąć z listy, i kliknij Usuń.
