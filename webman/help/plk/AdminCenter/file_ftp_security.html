Ustawienia bezpieczeństwa

Zmień wybrany root użytkownika dla użytkownika domowego

Wybrani tu użytkownicy mogą mieć dostęp do swoich folderów domowych tylko podczas logowania do Synology DiskStation przez FTP.

Włącz anonimowe FTP

Włącz tę funkcję, aby zezwolić wszystkim na logowanie w usłudze FTP DiskStation (przy użyciu konta anonymous lub ftp).

Aby skonfigurować uprawnienia użytkownika anonimowego:

Przejdź do Menu głównego > Panel sterowania > Folder współdzielony, a następnie kliknij Konfiguracja uprawnień.

Zmień anonimowy root

Włącz tę funkcję, aby zmienić katalog logowania anonimowego użytkownika FTP. Wybierz folder współdzielony z listy rozwijanej.
