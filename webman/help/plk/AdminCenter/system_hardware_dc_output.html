Wyjście zasilania prądem stałym*

Na stronie Panel sterowania > Sprzęt i zasilanie > Wyjście zasilania prądem stałym można ustawić system tak, aby włączał i wyłączał wyjście zasilania prądem stałym o określonej porze.

Aby dodać nowy harmonogram włączania/wyłączania wyjścia zasilania:

  1. Kliknij przycisk Utwórz, aby otworzyć okno dialogowe Dodaj harmonogram.
  2. Wybierz Wyjście włączone lub Wyjście wyłączone.
  3. Ustaw parametry Data i Czas harmonogramu włączania/wyłączania.
  4. Kliknij przycisk OK.
  5. Kliknij przycisk Zapisz.

Nowy harmonogram zostanie dodany do listy harmonogramów.

Aby wyświetlić podsumowanie harmonogramu:

Kliknij Podsumowanie, aby przeglądać tygodniowy harmonogram włączania/wyłączania wyjścia zasilania.

Uwaga:

  • Dla jednego serwera DiskStation można określić maksymalnie 100 harmonogramów.
  • W następujących przypadkach system może nie wyłączyć wyjścia zasilania prądem stałym automatycznie:
    • Zarządzanie przechowywaniem wykonuje zadania.
    • System szyfruje foldery współdzielone.
    • System wykonuje operację Secure Erase na dyskach twardych.
    • System DSM jest aktualizowany.
    • Centrum pakietów wykonuje zadania.
    • System aktualizuje bazy danych.
    • System wykonuje zadania kopii zapasowej danych.

____
*Funkcja dostępna tylko w niektórych modelach.
