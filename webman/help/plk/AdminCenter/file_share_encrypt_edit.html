Edytowanie folderu zaszyfrowanego

Po zaszyfrowaniu folderu, możesz użyć listy rozwijanej Szyfrowanie, aby edytować folder.

Opcje widoczne w menu zależą od stanu wybranego folderu współdzielonego:

  • Jeśli wybierzesz nieszyfrowany folder współdzielony, lista rozwijana Szyfrowanie jest wyłączona.
  • Jeśli wybierzesz zaszyfrowany folder współdzielony, który jest zamontowany, możesz wybrać Odmontuj z listy rozwijanej Szyfrowanie, aby odmontować folder, lub Eksportuj klucz, aby zapisać klucz szyfrowania.
  • Jeśli wybierzesz zaszyfrowany folder współdzielony, który nie jest zamontowany, możesz wybrać Montuj z listy rozwijanej Szyfrowanie i wprowadzić lub zaimportować klucz szyfrowania, aby zamontować folder i uzyskać do niego dostęp.
    Uwaga:
  1. To bardzo ważne, aby zapamiętać klucz szyfrowania. Bardzo zalecamy wyeksportowanie i zapisanie klucza szyfrowania. Jeśli zapomnisz lub zgubisz klucz, nie będzie można odzyskać zaszyfrowanych danych.
  2. Jeśli zaszyfrowany folder współdzielony jest zamontowany, nie można zmienić jego nazwy ani lokalizacji jego wolumenu. Zmian można dokonać tylko po jego odmontowaniu.
