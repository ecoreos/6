Zarządzanie uprawnieniami do folderu współdzielonego

Na stronie Panel sterowania > Folder współdzielony > Edytuj > Zaawansowane znajdują się poniżej wymienione opcje umożliwiające doprecyzowanie uprawnień dostępu do folderu współdzielonego.

Zaawansowane uprawnienia do udostępniania

Zaawansowane uprawnienia do udostępniania zapewniają dodatkową warstwę kontroli do celów zarządzania uprawnieniami dostępu do folderów współdzielonych. Po ich włączeniu użytkownicy i grupy mogą wyświetlać lub modyfikować zawartość folderu współdzielonego, pod warunkiem że użytkownikowi lub grupie przyznano zarówno zaawansowane uprawnienia do udostępniania, jak i uprawnienia Windows ACL (na stronie Folder współdzielony > Edytuj > Uprawnienia).

Aby włączyć zaawansowane uprawnienia do udostępniania:

  1. Zaznacz pole wyboru Włącz zaawansowane uprawnienia do udostępniania.
  2. Kliknij Zaawansowane uprawnienia do udostępniania, aby zmodyfikować zaawansowane uprawnienia do udostępniania dotyczące danego folderu współdzielonego.

Uwaga:

  • Zaawansowane uprawnienia do udostępniania stosowane są wtedy, gdy użytkownik uzyskuje dostęp do folderu współdzielonego za pośrednictwem następujących usług plików: Windows File Sharing, Apple File Sharing, File Station, FTP i WebDAV.

Ustawienia zaawansowane

W razie potrzeby można nałożyć pewne ograniczenia na użytkowników, kiedy uzyskują dostęp do folderu współdzielonego za pośrednictwem usług File Station, FTP lub WebDAV.

  • Wyłącz przeglądanie katalogu: po włączeniu tej opcji użytkownicy nie będą mogli wyświetlać zawartości folderu współdzielonego.
  • Wyłącz modyfikacje istniejących plików: po włączeniu tej opcji użytkownicy nie będą mogli przenosić, usuwać ani modyfikować plików z folderu współdzielonego. Uwaga — użytkownicy nadal będą mogli wyświetlać, pobierać, przesyłać, kopiować lub rozpakowywać zawartość folderu współdzielonego.
  • Wyłącz pobieranie plików: po włączeniu tej opcji użytkownicy nie będą mogli pobierać zawartości folderu współdzielonego.

Uwaga:

Powyższe ograniczenia nie dotyczą użytkowników należących do grupy administrators.
