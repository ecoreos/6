Odzyskiwanie

Na tej stronie administratorzy mogą przeprowadzać odzyskiwanie danych. Aby zapewnić użytkownikom dostęp do zrzutów obrazu folderu udostępnionego, zaznacz opcję Ustaw widoczność migawki dla poszczególnych folderów współdzielonych. Dzięki temu użytkownicy będą mogli uzyskać dostęp do zrzutów obrazu za pomocą aplikacji File Station lub Eksploratora plików systemu Windows.

Folder współdzielony

Aby przywrócić poszczególne pliki/foldery z folderu współdzielonego:

  1. Wybierz folder współdzielony, którego zawartość ma zostać przywrócona.
  2. Kliknij polecenie Odzyskaj.
  3. Wybierz wersję zrzutu obrazu, która ma zostać użyta do przywracania.
  4. Kliknij przycisk Przeglądaj i skopuj pliki/foldery, które mają zostać przywrócone do folderu docelowego.

Aby przywrócić folder udostępniony:

  1. Wybierz folder współdzielony, którego zawartość ma zostać przywrócona.
  2. Kliknij polecenie Odzyskaj.
  3. Wybierz wersję zrzutu obrazu, która ma zostać użyta do przywracania.
  4. Kliknij pozycję Akcja i wybierz polecenie Przywróć na miejsce.
  5. Zostanie wyświetlony monit o potwierdzenie. Kliknij przycisk Tak, aby potwierdzić czynność.

Aby sklonować zrzut obrazu do nowego folderu współdzielonego:

  1. Wybierz folder współdzielony, którego zawartość ma zostać sklonowana.
  2. Kliknij polecenie Odzyskaj.
  3. Wybierz wersję zrzutu obrazu, która ma zostać użyta do klonowania do nowego folderu udostępnionego.
  4. Kliknij tytuł menu Akcja i wybierz polecenie Klonuj z nową nazwą.
  5. Zostanie wyświetlona strona Panel sterowania > Folder współdzielony > Utwórz >Klonuj. Wprowadź nazwę sklonowanego folderu udostępnionego i kliknij przycisk OK, aby zakończyć.

Uwaga:

  • Nie można przywracać folderu współdzielonego „homes”.
  • Nie można szyfrować sklonowanych folderów współdzielonych.
  • Jeżeli zrzut obrazu zostanie wykonany w trakcie transferu plików, dane mogą być niespójne.
  • Ograniczenia szyfrowania folderów współdzielonych:
    • Do zamontowanych zaszyfrowanych folderów współdzielonych nie można używać żadnego typu odzyskiwania.
    • Nie można używać opcji Klonuj z nową nazwą do przywracania zaszyfrowanych folderów współdzielonych.
    • Odmontowane zaszyfrowane foldery współdzielone można przywracać z poprzednio wykonanych zrzutów obrazu, używając opcji Przywróć na miejsce.

Aby przełączyć replikację folderu współdzielonego:

  1. Zaznacz folder współdzielony, którego zawartość ma zostać przełączona.
  2. Kliknij Przełącz.
  3. Jeśli istnieje wiele zadań replikacji folderu współdzielonego, wybierz nowy serwer docelowy.
  4. Kliknij przycisk Przełącz, aby rozpocząć działanie.

Aby przełączyć awaryjnie replikację folderu współdzielonego:

  1. Zaznacz zreplikowany folder udostępniony, który zostanie przełączony awaryjnie.
  2. Kliknij Przełączanie awaryjne > Przełączanie awaryjne.
  3. Wybierz wersję zrzutu obrazu, która ma zostać przywrócona.
  4. Kliknij przycisk Tak, aby rozpocząć działanie.

Uwaga:

  • Przełączenie awaryjne można uruchomić tylko w przypadku awarii serwera źródłowego.

Aby ponownie zabezpieczyć replikację folderu współdzielonego:

Podczas przełączania awaryjnego na serwer docelowy zadanie replikacji przestanie synchronizować dane między dwoma serwerami Po przywróceniu oryginalnego serwera źródłowego należy ponownie zabezpieczyć zadanie replikacji.

  1. Wybierz przełączony awaryjnie folder współdzielony, który chcesz ponownie zabezpieczyć.
  2. Kliknij Przełączanie awaryjne > Zabezpiecz ponownie.
  3. Wybierz nowy serwer źródłowy.
  4. Jeśli istnieje wiele zadań replikacji folderu współdzielonego, wybierz nowy serwer docelowy.
  5. Kliknij przycisk Tak, aby potwierdzić czynność.

Uwaga:

  • Dane z powielonego folderu istniejące na nowym serwerze docelowym zostaną nadpisane danymi z nowego serwera źródłowego.
  • Jeśli wybrana do przełączenia awaryjnego wersja zrzutu obrazu nie istniała na serwerze partnerskim, proces ponownego zabezpieczania może potrwać dłużej.

Aby wykonać testowe przełączenie awaryjne powielonego folderu współdzielonego:

  1. Zaznacz folder współdzielony, którego zawartość ma zostać awaryjnie przełączona.
  2. Kliknij pozycje Test przełączania > Test przełączania
  3. Jeśli istnieje wiele zadań replikacji folderu współdzielonego, wybierz Test serwera.
  4. Wypełnij pole Nazwa testowego przełączenia awaryjnego folderu współdzielonego i wybierz wersję migawki, która zostanie przetestowana.
  5. Kliknij Test przełączania, aby rozpocząć działanie.

Uwaga:

  • Wykonanie przełączenia / przełączenia awaryjnego spowoduje wyczyszczenie testu przełączania awaryjnego.
  • Nie można utworzyć zadania replikacji folderu współdzielonego testu przełączania awaryjnego.

Aby wyczyścić test przełączania awaryjnego dla powielonego folderu współdzielonego:

  1. Wybierz powielony folder współdzielony, dla którego chcesz wyczyścić test przełączania awaryjnego.
  2. Kliknij Test przełączania > Wyczyść testowe przełączenie awaryjne.
  3. Zaznacz pole przy testowym przełączeniu awaryjnym, które chcesz wyczyścić.
    Uwaga: Zaznacz opcję Usuń dane testowego przełączenia awaryjnego, jeśli chcesz usunąć folder współdzielony utworzony podczas testowego przełączenia awaryjnego.
  4. Kliknij Wyczyść testowe przełączenie awaryjne, aby rozpocząć działanie.

iSCSI LUN

Aby przywrócić jednostkę iSCSI LUN:

  1. Wybierz jednostkę iSCSI LUN, która ma zostać przywrócona.
  2. Kliknij polecenie Odzyskaj.
  3. Wybierz wersję zrzutu obrazu, która ma być użyta do przywracania.
  4. Kliknij pozycję Akcja i wybierz polecenie Przywróć na miejsce.
  5. Zostanie wyświetlony monit o potwierdzenie. Kliknij przycisk Tak, aby potwierdzić czynność.

Aby sklonować zrzut obrazu do nowej jednostki iSCSI LUN:

  1. Wybierz jednostkę iSCSI LUN, która ma zostać sklonowana.
  2. Kliknij polecenie Odzyskaj.
  3. Wybierz wersję zrzutu obrazu, która ma być użyta do klonowania do nowej jednostki iSCSI LUN.
  4. Kliknij tytuł menu Akcja i wybierz polecenie Klonuj z nową nazwą.
  5. Zostanie wyświetlona strona Zarządzanie przechowywaniem > iSCSI LUN >Klonuj. Wprowadź nazwę sklonowanej jednostki iSCSI LUN i kliknij przycisk OK, aby zakończyć.

Uwaga:

  • Jeżeli zrzut obrazu zostanie wykonany w trakcie transferu plików, dane mogą być niespójne.

Aby przełączyć replikację folderu współdzielonego:

  1. Wybierz powieloną jednostkę iSCSI LUN, która ma zostać przywrócona awaryjnie.
  2. Kliknij Przełącz.
  3. Jeśli istnieje wiele zadań replikacji jednostki iSCSI LUN, wybierz nowy serwer docelowy.
  4. Kliknij przycisk Przełącz, aby rozpocząć działanie.

Aby przełączyć awaryjnie replikację jednostki iSCSI LUN:

  1. Wybierz powieloną iSCSI LUN, która ma zostać przełączona awaryjnie.
  2. Kliknij Przełączanie awaryjne > Przełączanie awaryjne.
  3. Wybierz wersję zrzutu obrazu, która ma zostać przywrócona.
  4. Kliknij przycisk Tak, aby rozpocząć działanie.

Uwaga:

Aby wykonać testowe przełączenie awaryjne powielonej jednostki iSCSI LUN:

  1. Wybierz jednostkę iSCSI LUN, która ma zostać przetestowana.
  2. Kliknij pozycje Test przełączania > Test przełączania
  3. Jeśli istnieje wiele zadań replikacji jednostki iSCSI LUN, wybierz Test serwera.
  4. Wypełnij pole Nazwa testowego przełączenia awaryjnego przy jednostce iSCSI LUN i wybierz wersję zrzutu obrazu, która zostanie przetestowana.

  2. Kliknij Test przełączania > Wyczyść testowe przełączenie awaryjne.
  3. Zaznacz pole przy testowym przełączeniu awaryjnym, które chcesz wyczyścić.

  4. Kliknij Wyczyść testowe przełączenie awaryjne, aby rozpocząć działanie.
