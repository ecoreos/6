Informacje ogólne

Możesz sprawdzić bieżący stan DSM, przeskanować system, sprawdzić postęp skanowania oraz podsumowanie wyników. Program Security Advisor skanuje system DSM oraz urządzenie DiskStation, sprawdzając różnego rodzaju reguły zabezpieczeń. Liczba i rodzaje reguł zależą od konfiguracji Poziomu odniesienia zabezpieczeń. Poziom odniesienia zabezpieczeń zależy od potrzeb użytkownika. Czy korzystasz z DiskStation do zastosowań służbowych i biznesowych czy też do użytku domowego i osobistego? Podczas pierwszego uruchomienia skanowania pojawi się pytanie o wybór Poziomu odniesienia zabezpieczeń. Można to później zmienić na stronie Zaawansowane.

Stan systemu

Program Security Advisor posiada łącznie siedem stanów. Po zakończeniu skanowania zostanie wyświetlone podsumowanie wyników w pięciu kategoriach. Wyświetlane są następujące stany:

  • Gotowość do skanowania: Ten stan jest wyświetlany, gdy nie rozpoczęto jeszcze skanowania DiskStation. Kliknij Skanuj, aby rozpocząć skanowanie.
  • Trwa skanowanie: Skanowanie jest w toku. Możesz wyświetlić postęp skanowania. Kliknij Zatrzymaj, aby zatrzymać skanowanie.
  • Dobry: Bieżące skanowanie zostało ukończone. Program Security Advisor nie znalazł żadnych słabości w ustawieniach ani czynników ryzyka. Urządzenie DiskStation jest obecnie bezpieczne.
  • Ostrzeżenie: Bieżące skanowanie zostało ukończone. Ze względu na pewne luki w ustawieniach zabezpieczeń program Security Advisor zgłosił czynniki ryzyka na niskim lub średnim poziomie. Aby zwiększyć bezpieczeństwo urządzenia DiskStation, zaleca się wykonanie wskazanych czynności.
  • Ryzyko: Bieżące skanowanie zostało ukończone. Ze względu na pewne luki w ustawieniach zabezpieczeń program Security Advisor zgłosił czynniki ryzyka na wysokim poziomie. Aby zwiększyć bezpieczeństwo urządzenia DiskStation, zdecydowanie zaleca się wykonanie wskazanych czynności.
  • Niebezpieczeństwo: Bieżące skanowanie zostało ukończone. Program Security Advisor znalazł szkodliwe oprogramowanie w urządzeniu DiskStation. Aby zapobiec dalszym niepożądanym konsekwencjom dla urządzenia DiskStation, należy wykonać zalecane działania.
  • Nieaktualny: Używana wersja systemu DSM jest nieaktualna. Zaleca się aktualizację systemu DSM w celu uzyskania najnowszych funkcji, usprawnień i poprawek zabezpieczeń, które usprawnią użytkowanie i zapewnią bezpieczeństwo urządzenia DiskStation.

Program Security Advisor dzieli testy na pięć kategorii:

  • Szkodliwe oprogramowanie: sprawdza, czy w urządzeniu DiskStation istnieje szkodliwe oprogramowanie oraz czy urządzenie DiskStation jest obiektem ataków.
  • System: wykrywa ustawienia zabezpieczeń związanych z systemem.
  • Konto: wykrywa ustawienia zabezpieczeń kont i haseł.
  • Sieć: wykrywa ustawienia zabezpieczeń związanych z siecią.
  • Aktualizacja: wykrywa, czy system DSM i inne zainstalowane pakiety są aktualne, a także sprawdza ustawienia związane z aktualizacjami.

Po zakończeniu skanowania można wyświetlić szczegółowe wyniki poszczególnych raportów, klikając Wyświetl wyniki.
