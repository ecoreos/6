Wyniki

Szczegółowe informacje dotyczące ukończonego skanowania można przeglądać według raportów. Strona ma następujący układ:

  • Istotność: Ogólny poziom ryzyka, jaki ocenia program Security Advisor w przypadku poszczególnych reguł zabezpieczeń. Istnieją cztery poziomy: Normalne, Ostrzeżenie, Alarm i Krytyczne.
  • Opis: Ustawienia wyszukiwane przez program Security Advisor na podstawie określonej reguły mogą prowadzić do luk w zabezpieczeniach urządzenia DiskStation, jeżeli nie zostaną podjęte żadne działania.
  • Ostatnia aktualizacja: Kiedy ostatnio program Security Advisor sprawdził daną regułę zabezpieczeń.
  • Stan: Aktualny stan poszczególnych reguł zabezpieczeń. Program Security Advisor podaje wyniki poszczególnych reguł zabezpieczeń w następującej postaci: Powodzenie lub Niepowodzenie. Niekiedy mogą być także wyświetlane stany takie jak Błąd, Niesprawdzone, Działa lub Pominięto. Wyniki można filtrować u góry strony.

Aby sprawdzić szczegóły raportu dotyczącego dowolnej reguły zabezpieczeń:

Wybierz regułę, a następnie kliknij Wyświetl. Oprócz pól Istotność i Opis każdy raport zawiera następujące szczegóły:

  • Szczegóły: Krótki opis słabych stron, zalecenia programu Security Advisor, a także korzyści wynikające z eliminacji tych słabych stron.
  • Zalecane działanie: Szczegółowa instrukcja eliminacji zgłoszonych słabych stron zabezpieczeń.
  • Łącze: W przypadku reguł zabezpieczeń wykonanych z niepowodzeniem można kliknąć opcję Łącze, która spowoduje przeniesienie do miejsca DSM umożliwiającego wykonanie zalecanych działań.
  • Napraw: Kliknięcie Napraw po wykryciu szkodliwego oprogramowania powoduje usunięcie tego oprogramowania przez program Security Advisor.

Aby ponownie skanować na podstawie określonej reguły:

Wybierz elementy do ponownego skanowania, a następnie kliknij Skanuj.
