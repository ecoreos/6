Zaawansowane

Pozwala sprawdzić i zmienić zaawansowane ustawienia programu Security Advisor.

Poziom odniesienia zabezpieczeń

Zaznaczenie pola Poziom odniesienia zabezpieczeń pozwala określić reguły zabezpieczeń sprawdzane przez program Security Advisor. Liczba i rodzaje reguł różnią się w zależności od Poziomu odniesienia zabezpieczeń. Wybierz odpowiednie opcje w zależności od potrzeb. Czy korzystasz z DiskStation do zastosowań służbowych i biznesowych czy też do użytku domowego i osobistego? Możesz wybrać następujące opcje:

  • Do użytku domowego i osobistego: Program Security Advisor przeprowadzi podstawowe testy chroniące urządzenie DiskStation przed typowymi zagrożeniami.
  • Do zastosowań służbowych i biznesowych: Program Security Advisor przeprowadzi wszechstronne testy pod kątem zgodności z wymaganiami organizacji.
  • Niestandardowy: Wybierz reguły zabezpieczeń, które ma sprawdzić program Security Advisor.

Harmonogram skanowania

Aby określić harmonogram regularnego skanowania, kliknij opcję Włącz harmonogram regularnego skanowania i wybierz porę, o jakiej program Security Advisor ma przeprowadzać skanowanie. Program Security Advisor sprawdzi reguły w zależności od wybranej opcji Poziom odniesienia zabezpieczeń.
