Wymagania aplikacji File Station

Do pełnego wykorzystania możliwości aplikacji File Station potrzebna jest najnowsza wersja przeglądarki internetowej, środowiska Java i programu Flash Player.

  • Przeglądarka: Internet Explorer 8 lub nowsza wersja, Firefox 3.6 lub nowsza wersja, Safari 5.0 lub nowsza wersja, Chrome; z obsługą skryptów JavaScript
  • Java (do przeglądania folderów na komputerze lokalnym): Java Runtime Environment (JRE) 5 lub nowsza wersja
  • Technologia Flash: Flash Player 9.0.28 lub nowsza wersja
