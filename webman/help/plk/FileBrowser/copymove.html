Kopiowanie lub przenoszenie plików i folderów

Program File Station zapewnia różne metody przenoszenia plików/folderów pomiędzy dwoma DiskStation, w ramach jednego DiskStation lub do lokalnego komputera.

Przeciąganie i upuszczanie plików/folderów pomiędzy przeglądarkami

Można bezpośrednio przesyłać pliki/foldery poprzez program File Station z jednego DiskStation do drugiego bez skomplikowanej konfiguracji protokołów przesyłania plików.

Aby włączyć przeciąganie i upuszczanie pomiędzy przeglądarkami:

  1. Przejdź do pozycji Ustawienia > Ogólne > File Station.
  2. Dotknij opcję Włącz przeciąganie i upuszczanie pomiędzy przeglądarkami.
  3. Kliknij OK.
  4. Otwórz dwie przeglądarki internetowe tego samego rodzaju (np. Firefox) i połącz je odpowiednio z lokalizacją źródłową i docelową DiskStation.
  5. Uruchom program File Station w obu oknach przeglądarek DiskStation. Teraz można bezpośrednio przeciągać i upuszczać pliki/foldery pomiędzy nimi za pośrednictwem programu File Station.

Uwaga:

  • Można także przeciągać i upuszczać pliki/foldery pomiędzy dwiema kartami tej samej przeglądarki.
  • Przeglądarka Internet Explorer nie obsługuje przeciągania i upuszczania między przeglądarkami lub kartami.
  • Jeśli wybrano wiele plików/folderów do przeciągnięcia i upuszczenia, pliki/foldery te zostaną skompresowane do jednego archiwum do przesłania.
  • Limit rozmiaru pliku do przeciągnięcia i upuszczenia zależy od przeglądarki internetowej: Safari (1 GB).

Kopiowanie lub przenoszenie plików/folderów w ramach DiskStation lub do lokalnego komputera

Wykonaj poniższe kroki, aby przenieść lub skopiować pliki lub foldery w ramach DiskStation lub do lokalnego komputera.

Sposób 1:

  1. Wybierz pliki lub foldery, które chcesz przenieść lub skopiować. (Naciśnij i przytrzymaj klawisz Shift lub Ctrl, aby wybrać wiele elementów)
  2. Wybierz Wytnij lub Kopiuj z menu rozwijanego Akcja.
  3. Przejdź do folderu docelowego w DiskStation lub na komputerze.
  4. W zależności od tego, czy chcesz nadpisać pliki lub foldery o tej samej nazwie w folderze docelowym, wybierz Wklej – Nadpisz lub Wklej – Pomiń na liście rozwijanej Akcja.

Sposób 2:

  1. Wybierz pliki lub foldery, które chcesz przenieść lub skopiować. (Naciśnij i przytrzymaj klawisz Shift lub Ctrl, aby wybrać wiele elementów)
  2. Przeciągnij pliki lub foldery z sekcji głównej do innego folderu na lewym panelu.
  3. W wyświetlonym menu podręcznym wybierz Kopiuj – Nadpisz, Kopiuj – Pomiń, Przenieś – Nadpisz lub Przenieś – Pomiń, aby nadpisać lub pominąć pliki albo foldery o tej samej nazwie w folderze docelowym.

Sposób 3

  1. Wybierz pliki lub foldery, które chcesz przenieść lub skopiować. (Naciśnij i przytrzymaj klawisz Shift lub Ctrl, aby wybrać wiele elementów)
  2. Naciśnij klawisze Ctrl + X, aby wyciąć lub Ctrl + C, aby skopiować pliki/foldery. W przypadku użytkowników komputerów Mac należy nacisnąć klawisze Command + X i Command + C.
  3. Przejdź do folderu docelowego w DiskStation lub na komputerze.
  4. Naciśnij klawisze Ctrl + V, aby wkleić pliki/foldery. (Command + V w przypadku komputerów Mac)
  5. W pojawiającym się oknie dialogowym wybierz, czy chcesz nadpisać lub pominąć pliki o tej samej nazwie w folderze docelowym.

Skróty klawiszowe:

Przydatne mogą być poniższe skróty klawiaturowe podczas przenoszenia/kopiowania plików i folderów w programie File Station:

  • F2: Zmień nazwę
  • Del: Usuń
  • Ctrl + A: Wybierz wszystko
  • Ctrl + X: Wytnij
  • Ctrl + C: Kopiuj
  • Ctrl + V: Wklej

Uwaga:

  • Po skopiowaniu lub przeniesieniu plików/folderów do katalogu w systemie plików FAT, znaki \ *?"| w nazwie pliku zostaną zamienione na –, a pliki metadanych Mac zostaną usunięte.
  • W przypadku wycięcia plików z lokalnego komputera i wklejenia ich do folderu DiskStation, pliki będą nadal przechowywane na lokalnym komputerze.
  • Podczas przenoszenia pliku lub folderu „właściciel” nie zmienia się. Jednak w przypadku kopiowania pliku lub folderu, „właścicielem” staje się użytkownik, który go skopiował.
