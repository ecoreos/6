Kompresowanie plików i folderów

  1. W głównym obszarze wybierz pliki lub foldery z serwera DiskStation.
  2. Z listy rozwijanej Akcja wybierz polecenie Kompresuj do „(nazwa pliku)”.zip, aby rozpocząć kompresję.
  3. Jeżeli po zakończeniu kompresji nie widać pliku zip, z listy rozwijanej Akcja wybierz polecenie Odśwież, aby wyświetlić plik.
  4. Aby zmienić nazwę, zaszyfrować lub zarchiwizować archiwum z innym poziomem lub trybem kompresji, wybierz polecenie Dodaj do archiwum.
    Uwaga:
  1. Nie można kompresować plików lub folderów na komputerze lokalnym, używając aplikacji File Station.
  2. Obsługiwane formaty archiwów: .zip, .7z.
  3. Jeżeli w oknie Dodaj do archiwum dla ustawienia Format archiwum zostanie wybrana wartość .zip, można wybrać żądaną stronę kodową pliku .zip.
