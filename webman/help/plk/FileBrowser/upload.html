Przesyłanie plików lub folderów

Użyj dowolnej z następujących metod, aby przesłać pliki lub foldery ze swojego komputera na serwer DiskStation.

Aby uzyskać pełny dostęp do poniższych funkcji przesyłania, zainstaluj najpierw najnowszą wersję środowiska Java Runtime Environment (JRE) oraz programu Flash Player.

Sposób 1:

  1. W górnej części lewego panelu wybierz folder docelowy na serwerze DiskStation.
  2. Kliknij Prześlij, wybierz Prześlij - Pomiń lub Prześlij - Nadpisz i wybierz pliki lub foldery z komputera lokalnego.

Sposób 2:

  1. Przejdź do pozycji Opcje, zaznacz pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station i odśwież stronę internetową, aby wyświetlić foldery z komputera lokalnego w dolnej części lewego panelu.

    2. Uwaga:

    Opcja Przeglądaj pliki z komputera lokalnego za pomocą programu File Station działa dopiero po zainstalowaniu i włączeniu środowiska JRE na komputerze.

  2. W dolnej części lewego panelu wybierz folder znajdujący się na komputerze lokalnym. W sekcji głównej zostanie wyświetlona zawartość wybranego folderu.
  3. Wybierz pliki lub foldery w sekcji głównej, a następnie kliknij przycisk Prześlij.
  4. W wyświetlonym oknie wybierz folder docelowy na serwerze DiskStation.
  5. W prawym górnym rogu okna wybierz Nadpisz, jeżeli chcesz nadpisać pliki lub foldery o tej samej nazwie w folderze docelowym. W przeciwnym razie wybierz opcję Pomiń.
  6. Kliknij przycisk OK, aby rozpocząć przesyłanie.

Sposób 3:

  1. Przejdź do pozycji Opcje, zaznacz pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station i odśwież stronę internetową, aby wyświetlić foldery z komputera lokalnego w dolnej części lewego panelu.
  2. W lewym panelu przeciągnij folder z komputera (w dolnej części) do dowolnego folderu na serwerze DiskStation (w górnej części).
  3. W wyświetlonym menu podręcznym wybierz, czy chcesz nadpisać, czy też pominąć pliki lub foldery o takich samych nazwach w folderze docelowym.

Sposób 4:

  1. W górnej części lewego panelu wybierz folder docelowy na serwerze DiskStation.
  2. Zaznacz pliki na pulpicie komputera lub w eksploratorze plików (np. w programach Eksplorator Windows lub Mac Finder), a następnie przeciągnij je i upuść na aplikację File Station w przeglądarce internetowej.
  3. W wyświetlonym oknie dialogowym wybierz, czy chcesz nadpisać, czy też pominąć pliki o takich samych nazwach w folderze docelowym.

    4. Uwaga:

    • Sposób 4 jest obsługiwany tylko w niektórych przeglądarkach: Chrome, Firefox, Safari 5.1 i nowszych wersjach. Nie jest jednak obsługiwana na komputerach Mac, kiedy zaznaczone jest pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station.
    • Można także przeciągać i upuszczać foldery lub puste pliki, ale tylko gdy zaznaczone jest pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station i używa się wymienionych powyżej przeglądarek. W przeciwnym razie nie można przesyłać folderów i pustych plików, stosując Sposób 4.
    • Jeżeli łączysz się z DSM przy użyciu protokołu HTTPS i używasz przeglądarki Chrome dla systemu Linux, musisz zaimportować zaufany certyfikat (Panel sterowania > Bezpieczeństwo > Certyfikat). W przeciwnym razie przesyłanie materiałów za pomocą metody 4 nie będzie możliwe.

Uwaga:

  1. Do kolejki przesyłania można dodać maksymalnie 1000 plików.
  2. Pliki o rozmiarze większym niż 4 GB można przesyłać, jeżeli używa się przeglądarki Google Chrome, Firefox, Safari 5.1 lub ich nowszych wersji lub gdy jest zaznaczone pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station. W przeciwnym razie można przesyłać jedynie do 2 GB plików naraz.
  3. W niezaszyfrowanym folderze współdzielonym nazwa pliku lub folderu powinna mieć do 255 znaków (w alfabetach niełacińskich do około 80 znaków); w zaszyfrowanym folderze współdzielonym powinna mieć do 143 znaków (w alfabetach niełacińskich do około 47 znaków).
  4. Nazwa pliku/folderu nie powinna zawierać następujących znaków: \ / : * ? " > |.
  5. Nazwa pliku/folderu nie może zaczynać się od "._".
  6. Pliki o rozmiarze 0 KB można przesyłać tylko po zainstalowaniu środowiska JRE.

Rozwiązywanie problemów

Jeśli nie możesz wyświetlać poprawnie w przeglądarce Safari elementów lokalnych zapisanych na komputerze Mac, wykonaj następujące czynności:

  1. Na komputerze Mac otwórz przeglądarkę Safari i wybierz polecenie Preferencje z menu Safari.
  2. Na karcie Bezpieczeństwo kliknij polecenie Zarządzaj ustawieniami strony internetowej.
  3. W sekcji Java wybierz „witrynę” DSM.
  4. Wybierz opcję Uruchom w trybie niebezpiecznym.
