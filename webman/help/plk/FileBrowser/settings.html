Zarządzanie ustawieniami aplikacji File Station

Aby zarządzać ustawieniami aplikacji File Station, otwórz aplikację File Station i kliknij przycisk Ustawienia.

Ogólne

Tutaj można włączyć rejestr programu File Station w celu rejestrowania działań wszystkich użytkowników. Działania użytkowników to Utwórz folder, Prześlij, Pobierz, Usuń, Zmień nazwę, Przenieś, Kopiuj i Właściwość ustawiona.

Aby włączyć rejestr aplikacji File Station:

  1. Jeżeli chcesz rejestrować działania wszystkich użytkowników, zaznacz pole Włącz rejestr programu File Station.
  2. Kliknij przycisk OK.

Aby wyświetlić rejestr programu File Station:

  1. Kliknij Zobacz logi, aby przejść do pozycji Centrum logów.
  2. Kliknij kartę Transfer plików.

Aby automatycznie nadpisać pliki przesyłane za pomocą metody przeciągania i upuszczania lub za pomocą skrótów klawiszowych:

Zaznacz opcję Zawsze zastępuj pliki, kiedy przeciągam je i upuszczam na przeglądarkę internetową lub wklejam je przy użyciu skrótów klawiszowych, jeśli chcesz automatycznie nadpisać pliki o tej samej nazwie, gdy przeciągasz je i upuszczasz je z pulpitu komputera do przeglądarki lub wklejasz je przy użyciu skrótów klawiszowych w aplikacji File Station.

Aby włączyć inteligentne przeciąganie i upuszczanie:

Po przeciągnięciu i upuszczeniu plików w aplikacji File Station ta funkcja automatycznie określi, czy to zadanie przesyłania jest kopiowaniem czy przenoszeniem.

  1. Zaznacz opcję Włącz inteligentne przeciąganie i upuszczanie.
  2. Wybierz akcję, która będzie wykonywana, jeśli w tym samym czasie inny użytkownik będzie przenosił pliki posiadające tę samą nazwę do tej samej lokalizacji:
    • Nadpisz: przesyłane pliki nadpiszą pliki posiadające tę samą nazwę.
    • Pomiń: przesyłane pliki zostaną pomięte i nie nadpiszą plików posiadających tę samą nazwę.
  3. Kliknij przycisk OK.

Uwaga:

Jeśli chcesz zmienić funkcję kopiowania lub przenoszenia określoną przez inteligentną funkcję przeciągania i upuszczania, dostosuj według potrzeb następujące klawisze skrótu:

  • Aby przełączyć do funkcji kopiowania: Podczas przeciągania i upuszczania wciśnij klawisz Ctrl.
  • Aby przełączyć do funkcji przenoszenia: Podczas przeciągania i upuszczania wciśnij klawisz Shift.

Aby przeglądać pliki z komputera lokalnego i zarządzać nimi za pomocą programu File Station:

  1. Zaznacz pole wyboru Przeglądaj pliki z komputera lokalnego za pomocą programu File Station.
  2. Kliknij przycisk OK.

Aby udostępnić listę użytkowników DSM użytkownikom innym niż administratorzy:

Dzięki tej opcji użytkownicy niebędący administratorami mogą uzyskać dostęp do pełnej listy użytkowników systemu DSM w przypadku, gdy chcą wyszukać pliki należące do określonych użytkowników/grupy systemu DSM lub jeśli chcą udostępniać łącza do plików innym użytkownikom systemu DSM.

  1. Zaznacz opcję Udostępnij listę użytkowników DSM użytkownikom niebędącym administratorami.
  2. Kliknij przycisk OK.

Stosowanie domyślnych uprawnień systemu UNIX

Kliknij przycisk Ustawienia zaawansowane i zaznacz tą opcję, aby zastosować domyślne ustawienia systemu UNIX podczas przesyłania lub tworzenia plików i folderów. Nadane uprawnienia są identyczne z uprawnieniami nadanymi przez parametr umask polecenia UNIX. Po włączeniu tej opcji uprawnienia systemu UNIX są ustawione na 644 dla plików i 755 dla folderów. Po wyłączeniu tej opcji uprawnienia systemu UNIX są ustawione na 666 dla plików i 777 dla folderów. Domyślna wartość umask to 022.

Uwaga:

  • W przypadku folderów współdzielonych z obsługą listy kontroli dostępu systemu Windows (ACL) (wszystkie udostępnienia z wyłączeniem udostępnień „photo” i udostępnień na zewnętrznych dyskach) należy uruchomić polecenie chmod w kliencie Linux lub FTP, aby zmienić rodzaje uprawnień dotyczące folderów i plików z listy kontroli dostępu systemu Windows (ACL) na UNIX.
  • Włączenie tej opcji może spowodować problemy z niespójnością uprawnień między różnymi protokołami. Aby uniknąć niespójności, nie zaleca się włączania tej opcji.

Montowanie/połączenia

Wszyscy użytkownicy mogą domyślnie montować zdalne foldery lub dyski wirtualne. Tutaj można ograniczyć dostęp do zarządzania zdalnymi folderami lub dyskami wirtualnymi tylko do administratorów.

Uwaga:

Użytkownicy inni niż administratorzy mogą montować zdalne foldery tylko w swoim folderze home.

Aby pozwolić wszystkim użytkownikom domyślnie montować zdalne foldery lub dyski wirtualne:

  1. Wybierz Wszyscy użytkownicy lub Administratorzy w sekcji Folder zdalny lub Dysk wirtualny.
  2. Kliknij przycisk OK.

Aby zezwolić różnym użytkownikom na montowanie serwerów i usług chmury:

  1. Wybierz Wszyscy użytkownicy, Administratorzy lub Określony użytkownik.
  2. Wybranie opcji Określony użytkownik spowoduje otwarcie okna, które umożliwi skonfigurowanie uprawnień montowania dla różnych użytkowników. Aby umożliwić określonemu użytkownikowi dostęp, upewnij się, że zaznaczone jest pole Włączone. Aby ograniczyć określonemu użytkownikowi dostęp, upewnij się, że pole Włączone nie jest zaznaczone. Po zakończeniu działań kliknij przycisk OK, aby zamknąć okno.

    3. Uwaga:

    • Użytkownicy należący do grupy administratorzy mogą używać tej funkcji, ale nie mogą wprowadzać żadnych zmian. Użytkownicy logujący się jako gość nie mogą używać tej funkcji i wprowadzać żadnych zmian. Służy do określania uprawnień dla wszystkich innych użytkowników
  3. Kliknij przycisk OK.

Udostępnione łącza

Domyślnie wszyscy użytkownicy mogą udostępniać łącza do plików. Tutaj można ograniczyć dostęp do zarządzania udostępnionymi łączami tylko do administratorów.

Aby zezwolić różnym użytkownikom na udostępnianie łączy do plików:

  1. Wybierz Wszyscy użytkownicy, Administratorzy lub Określony użytkownik.
  2. Kliknij przycisk OK.

Aby otworzyć pliki HTML jako zwykły tekst:

  1. Zaznacz pole Zawsze otwieraj pliki HTML w formacie zwykłego tekstu. Gdy ta opcja jest włączona, pliki HTML (np. HTM, HTML, JSP, XHTML) są wyświetlane jako zwykły tekst, gdy użytkownicy otwierają je za pomocą łącza udostępniania plików. Ta opcja chroni przed przechwyceniem sesji logowania przy użyciu osadzonego skryptu w plikach HTML.
  2. Kliknij przycisk OK.

Limit prędkości

Możesz ustawić limit prędkości, aby kontrolować maksymalną przepustowość łącza dla użytkowników lub grup, którzy przesyłają dane za pośrednictwem aplikacji File Station.

Aby włączyć limit prędkości:

  1. Zaznacz pole Włącz limit prędkości lub Włącz limit prędkości zgodnie z harmonogramem, jeśli chcesz włączyć limit prędkości tylko w niektórych okresach. Jest to globalne ustawienie serwera, które zastąpi ustawienia poszczególnych użytkowników i grup. Wybranie opcji Brak limitu prędkości dezaktywuje ustawienia limitu prędkości dla wszystkich użytkowników i grup.
  2. Kliknij przycisk Ustawienia limitu prędkości, aby zmodyfikować ustawienia dla poszczególnych użytkowników lub grup. W przypadku każdego użytkownika lub grupy można wybrać jedną z następujących opcji:
    • Zastosuj ustawienia grupy: ustawienia limitu prędkości będą takie same jak w przypadku grupy, do której należy użytkownik.
    • Brak limitu prędkości: brak limitu prędkości, gdy członkowie grupy przesyłają pliki za pomocą tej usługi.
    • Stały: zastosowanie stałego limitu prędkości dla użytkownika lub członków grupy. Ograniczenia prędkości przesyłania i pobierania można określić w polach po lewej stronie.
    • Zmienny: określanie dwóch różnych limitów szybkości oraz stosowanie ich dla użytkownika i członków grupy zgodnie z harmonogramem. Kliknij przycisk Dostosuj, aby zmodyfikować ustawienia limitu prędkości i ustalić harmonogram.
  3. Kliknij przycisk Zastosuj.

Uwaga:

  • Więcej szczegółów dotyczących dynamiki limitów prędkości w grupach można znaleźć tutaj.
  • Podczas ustawiania ograniczania prędkości dla użytkownika w części Rezultat będzie można zobaczyć rzeczywiste ograniczenie prędkości zgodne z ustawieniami indywidualnymi lub grupy. Jeśli na przykład użytkownik Achilles ma przypisany limit pobierania 100 KB/s, ale należy do grupy z limitem pobierania 500 KB/s, w części Wyniki wyświetlana będzie wartość „0/500”.

Bezpieczeństwo

Tutaj można znaleźć poniższe ustawienia zabezpieczeń.

Aby otworzyć pliki HTML jako zwykły tekst:

  1. Zaznacz pole Zawsze otwieraj pliki HTML w formacie zwykłego tekstu. Gdy ta opcja jest włączona, pliki HTML (np. HTM, HTML, JSP, XHTML) są wyświetlane jako zwykły tekst, gdy użytkownicy otwierają je za pomocą aplikacji File Station. Ta opcja chroni przed przechwyceniem sesji logowania przy użyciu osadzonego skryptu w plikach HTML.

    2. Uwaga:

    • Nie jest to ustawienie globalne. Każdy użytkownik może włączyć lub wyłączyć tę opcje stosownie do swoich potrzeb.
  2. Kliknij przycisk OK.
