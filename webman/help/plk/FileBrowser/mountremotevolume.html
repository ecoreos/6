Montowanie folderów zdalnych

Za pomocą aplikacji File Station można montować zdalne foldery ze zdalnych serwerów (takich jak komputery w sieci lub inne urządzenia Synology NAS) w celu nawigacji po zamontowanych folderach i zarządzania nimi.

Aby zamontować zdalny folder współdzielony CIFS*:

  1. Kliknij Narzędzia > Montuj folder zdalny > Folder współdzielony CIFS.
  2. W polu Folder określ ścieżkę folderu zdalnego w formacie \\Remote_Server\Remote_Folder, np. \192.168.0.2\share lub \\chad\cheeseburger\pictures.
  3. Jeśli serwer zdalny wymaga uprawnień dostępu do folderu zdalnego, wprowadź poświadczenia użytkownika zdalnego serwera w polach Nazwa konta i Hasło.
  4. Kliknij Przeglądaj, aby wybrać lub utworzyć pusty folder docelowy w urządzeniu DiskStation w celu zamontowania folderu zdalnego.
  5. Zaznacz opcję Zamontuj automatycznie przy uruchamianiu, aby serwer DiskStation montował ten folder zdalny przy każdym uruchomieniu lub ponownym uruchomieniu systemu.
  6. Kliknij Montuj, aby zamontować folder zdalny w folderze docelowym. Teraz możesz nawigować po folderze zdalnym i zarządzać nim z poziomu DiskStation.

Aby zamontować zdalny folder współdzielony NFS:

  1. Kliknij Narzędzia > Montuj folder zdalny > Folder współdzielony NFS.
  2. W polu Folder określ ścieżkę folderu zdalnego w formacie Remote_Server:/Remote_Folder_Path, np. 192.168.0.2/volume1/share.
  3. Wybierz żądaną wersję NFS i protokołu transportowego.
  4. Kliknij Przeglądaj, aby wybrać lub utworzyć pusty folder docelowy w urządzeniu DiskStation w celu zamontowania folderu zdalnego.
  5. Zaznacz opcję Montuj automatycznie przy uruchamianiu, aby serwer DiskStation montował ten folder zdalny przy każdym uruchomieniu lub ponownym uruchomieniu systemu.
  6. Kliknij Montuj, aby zamontować folder zdalny w folderze docelowym. Teraz możesz nawigować po folderze zdalnym i zarządzać nim z poziomu DiskStation.

Uwaga:

NFSv4 obsługuje tylko protokół TCP.

Aby odmontować folder zdalny:

  1. Wybierz folder zdalny, który ma być odmontowany, a następnie kliknij Narzędzia > Odmontuj.
  2. Ewentualnie kliknij prawym przyciskiem myszy folder zdalny, a następnie wybierz opcję Odmontuj.

Uwaga:

  • Maksymalna liczba równoczesnych folderów zdalnych to 100.
  • Foldery zdalne można montować wyłącznie z serwera zdalnego obsługującego protokoły CIFS lub NFS.
  • Foldery zdalne mogą być zamontowane tylko do podfolderów w folderach współdzielonych.
  • Montowanie folderów zdalnych jest obsługiwane tylko w trybie wysokich uprawnień. Aby użyć tej funkcji, przejdź do strony Docker w systemie DSM hosta, wybierz system Docker DSM, dla którego chcesz włączyć szyfrowane foldery współdzielone, kliknij Edytuj, a następnie zaznacz pole obok Uruchom kontener przy użyciu wysokich uprawnień.

* Funkcja dostępna tylko w niektórych modelach.
