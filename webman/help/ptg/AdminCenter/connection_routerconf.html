Configuração do Router

Se o seu DiskStation estiver numa rede de área local, outros dispositivos na Internet não se podem ligar ao mesmo.Pode configurar as regras de reencaminhamento de portas em Painel de Controlo > Acesso Externo > Configuração do Router para tornar o seu dispositivo DiskStation acessível através da Internet.

Configurar o Router

Antes de adicionar as regras de reencaminhamento de portas, é necessário configurar uma ligação entre o seu dispositivo DiskStation e o router.

Nota:

  1. Para configurar regras de reencaminhamento de portas e atribuir um endereço IP estático, precisará de privilégios de administrador para aceder ao router.
  2. Antes de configurar a ligação do dispositivo DiskStation a um router UPnP, certifique-se de que o DSM recuperou com êxito o endereço IP externo do seu DiskStation (em Painel de Controlo > Acesso Externo > DDNS).Caso contrário, verifique as configurações de rede e da firewall do seu dispositivo DiskStation e do router UPnP.

Para configurar a ligação do dispositivo DiskStation ao router:

  1. Clique em Configurar router para iniciar o assistente de configuração.O DSM vai começar por testar os seguintes componentes de rede/definições antes de prosseguir para o passo seguinte.
    • Interface de rede
    • Ligação Gateway
    • Ligação à Internet
    • Reencaminhamento de porta
    • DNS
  2. Dependendo do tipo de router que estiver a utilizar, proceda de uma das seguintes formas:
    • Caso o seu router não suporte UPnP, consulte o passo 3.
    • Se o router UPnP tiver sido testado e considerado compatível com o dispositivo DiskStation, clique em Aplicar quando visualizar um resumo das informações do router.A ligação do seu dispositivo DiskStation ao router foi configurada com êxito.
    • Se o seu router UPnP não tiver sido considerado compatível com o dispositivo DiskStation, proceda de uma das seguintes formas:
      • Se pretende configurar automaticamente a ligação ao router, escolha Automático e clique em Seguinte para que o sistema execute um teste de compatibilidade:
        • Se o router tiver sido testado e considerado compatível com o serviço de configuração do router do dispositivo DiskStation, assinale a caixa de verificação para enviar o relatório de teste para ajudar a Synology a atualizar a base de dados de compatibilidade do router e, em seguida, clique em Aplicar.A ligação do seu dispositivo DiskStation ao router foi configurada com êxito.
        • Se o router tiver sido testado e considerado incompatível com o serviço de configuração do router do dispositivo DiskStation, assinale a caixa de verificação para enviar o relatório de teste para ajudar a Synology a atualizar a base de dados de compatibilidade do router e, em seguida, clique em Fechar para sair do assistente de configuração.

          • Nota:

          Se o router for testado e considerado incompatível aqui, poderá continuar a configurar as regras de reencaminhamento de portas na interface de utilizador do router.Para obter mais informações, consulte aqui.

      • Se pretende configurar manualmente a ligação ao router, escolha Manual, clique em Seguinte e consulte o passo 3.
  3. Se for necessário configurar manualmente a ligação do dispositivo DiskStation ao router, proceda da seguinte forma:
    1. Escolha o seu router no menu pendente Router.

      2. Nota:

      Se não encontrar o seu router aqui, clique em Atualizar Lista para obter a lista mais recente da Synology.

    2. Configure o protocolo e a porta do seu router.O protocolo e a porta variam consoante o modelo do seu router.Se não conseguir ligar utilizando o Protocolo (HTTP) e a Porta (80), consulte a documentação fornecida com o seu router para obter as definições corretas.
    3. Se o seu router necessitar de uma conta personalizada, marque Utilizar conta personalizada e preencha os campos Conta e Palavra-passe.
    4. Clique em Aplicar para concluir.

Gerir Regras de Reencaminhamento de Portas

As regras de reencaminhamento de portas podem especificar as portas de destino do DiskStation que irão receber os pacotes de portas específicas do router.

Se o DiskStation estabelecer ligação à Internet através de um router, clique em Criar para abrir o assistente de configuração do reencaminhamento de portas.

Existem duas formas de adicionar regras de reencaminhamento de portas:

  • Se pretende criar regras de reencaminhamento de portas para serviços de sistema (por exemplo, FTP, Web Station, etc.), selecione Aplicação integrada.
  • Se pretende criar regras de reencaminhamento de portas para outros serviços, selecione Personalizar porta.

Pode utilizar a sintaxe seguinte para especificar os números e intervalos de portas:

  • Utilize “-” para descrever um intervalo de portas, por exemplo: 6881-6890.
  • Utilize “,” para separar portas, por exemplo: “21,22”, “21,55536-55663”.

Nota:

  • Cada porta do gateway do router pode ser utilizada apenas para uma regra de reencaminhamento.A utilização de uma porta do router para duas ou mais regras irá provocar conflitos e aparecerá uma mensagem de aviso no ecrã.É necessário definir outra porta na regra para evitar conflitos.
  • Alguns routers não permitem a utilização de portas personalizadas para o reencaminhamento de serviços do sistema.

Para guardar regras:

Clique em Guardar depois de criar as regras.Se não o fizer, as regras não serão aplicadas.

Depois de guardar as regras, pode ser necessário aguardar alguns minutos antes de as regras começarem a funcionar, dependendo do modelo do seu router.

Para testar se as suas regras estão ou não a funcionar:

Escolha as regras que pretende testar e clique em Testar Ligação.O resultado será exibido na coluna Resultado da Ligação de Teste.

Para Configurações do Router Sem Fios*

Quando o seu DiskStation funciona como router sem fios, as regras de reencaminhamento de portas serão diretamente aplicadas pelo DiskStation.

Para adicionar regras de reencaminhamento de portas:

  1. Clique em Criar para abrir a janela de configuração.

  2. Especifique o nome do Serviço, o Protocolo (TCP ou UDP), o Endereço IP Privado, a Porta Privada e a Porta Pública.Pode utilizar a sintaxe seguinte para especificar os números e intervalos de portas:
    • Utilize “-” para descrever um intervalo de portas, por exemplo: 6881-6890.
    • Utilize “,” para separar portas, por exemplo: “21,22”, “21,55536-55663”.
  3. Clique em Criar na parte inferior para concluir as definições.

Nota:

A maioria dos routers permitem portas da máquina local e portas do router serem diferentes quando configura regras de reencaminhamento de portas.Se o seu router suporta esta funcionalidade:

  • Quando guarda uma regra, se uma porta do router já estiver em utilização por uma regra existente, será atribuída uma nova porta de router.
  • Quando adiciona ou edita regras, pode especificar directamente o número de porta do router.

DMZ*

Pode ativar o DMZ no seu DiskStation quando a Configuração do Mapa de Rede for Router Sem Fios.Um anfitrião no DMZ pode ligar diretamente à rede externa, sem a limitação e proteção da firewall.É normalmente utilizado para configurar servidores.

Para ativar o DMZ:

  1. Clique em Definições do DMZ para abrir a janela de configuração.
  2. Assinale a caixa de verificação Ativar DMZ.
  3. Introduza o seu endereço DMZ Host IP.
  4. Clique em Aplicar.

Nota:

A partir da rede externa, será necessário um endereço IP externo obtido por DiskStation para ligar a um anfitrião no DMZ.

* Disponível apenas em determinados modelos ou consoante o seu ambiente.
