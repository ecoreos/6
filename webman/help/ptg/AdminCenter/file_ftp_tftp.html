TFTP

TFTP (Protocolo Trivial de Transferência de Ficheiros) é um simples protocolo de transferência de ficheiros, normalmente utilizado para transferir ficheiros de configuração ou arranque, quando não é necessária autenticação. Ativar o serviço TFTP permite ao DiskStation transferir carregadores de arranque para dispositivos cliente.

Para configurar as definições TFTP:

  1. Vá para Painel de Controlo > Serviços de Ficheiros > TFTP, e selecione Serviço TFTP.
  2. No campo Pasta de raiz TFTP, especifique que pasta no DiskStation pode ser acedida por clientes TFTP.
  3. Clique em Definições avançadas para configurar as seguintes definições:
    • Ativar o registo de transferência do ficheiro TFTP: Mantém um registo dos dados transferidos por TFTP.
    • Permissão de Cliente TFTP: Decide as permissões do cliente TFTP:
      • Só de leitura: Os clientes TFTP apenas podem ver o conteúdo da pasta de raiz.
      • Gravável: Os clientes TFTP apenas podem modificar o conteúdo da pasta de raiz.
    • Clientes permitidos: Escolha que clientes podem ligar através de TFTP:
      • Todas as ligações: Todos os clientes podem ligar-se por TFTP.
      • Permitir apenas a seguinte amplitude de endereços IP: Só os clientes no intervalo IP especificado podem ligar-se por TFTP.
    • Tempo limite: Se o servidor TFTP (por exemplo, o seu DiskStation) não responder durante um período de tempo especificado, a ligação TFTP será interrompida.
  4. Clique em Aplicar para guardar as definições.
