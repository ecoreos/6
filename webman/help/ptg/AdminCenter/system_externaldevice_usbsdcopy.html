USBCopy

Pode selecionar uma pasta de destino para a função USBCopy, acedendo a Painel de Controlo > Dispositivos Externos > USBCopy/SDCopy.

Para ativar a USBCopy:

  1. Assinale a caixa de verificação Ativar USBCopy.
  2. Selecione uma pasta partilhada de destino no menu pendente.

Quando liga um dispositivo USB (por exemplo, uma câmara digital) à porta USB no painel frontal do produto Synology, pode premir o botão (Copy) Copiar no painel frontal para copiar os dados do dispositivo USB para a pasta partilhada de destino atribuída aqui. Se a pasta de destino USBCopy for apagada, o sistema irá substituí-la automaticamente pela pasta partilhada pública.

Restrições

  1. Apenas são suportados dispositivos USB nos formatos FAT, NTFS, ext3 e ext4.
  2. Os dispositivos USB devem ser ligados à porta USB no painel frontal do dispositivo DiskStation.
  3. Apenas o admin ou utilizadores no grupo administrators dispõem dos privilégios de acesso adequados para realizarem tarefas de cópia de segurança para dispositivos USB.
  4. O espaço disponível na pasta partilhada de destino deve ser superior ao espaço utilizado do dispositivo USB.

SDCopy*

Pode selecionar uma pasta de destino para a função SDCopy, acedendo a Painel de Controlo > Dispositivos Externos > USBCopy/SDCopy.

Para ativar a SDCopy:

  1. Assinale a caixa de verificação Ativar SDCopy.
  2. Selecione uma pasta partilhada de destino no menu pendente.

Quando liga um dispositivo SD na ranhura SD do painel frontal do produto Synology, pode premir o botão Copy (Copiar) no painel frontal para copiar os dados no dispositivo SD para a pasta partilhada de destino designada aqui. Se a pasta de destino SDCopy for apagada, o sistema irá substituí-la automaticamente pela pasta partilhada pública.

Restrições

  1. Apenas são suportados dispositivos SD nos formatos FAT, NTFS, ext3 e ext4.
  2. Os dispositivos SD devem ser encaixados na ranhura SD no painel frontal do produto Synology.
  3. Apenas o admin ou utilizadores no grupo administrators dispõem dos privilégios de acesso adequados para realizarem tarefas de cópia de segurança para dispositivos SD.
  4. O espaço disponível na pasta partilhada de destino deve ser superior ao espaço utilizado do dispositivo SD.

____
* Apenas disponível em modelos específicos.
