Bloqueio Automático

A funcionalidade de bloqueio automático ajuda a melhorar a segurança do seu DiskStation, bloqueando os endereços IP de clientes com demasiadas tentativas falhadas de início de sessão. Isto ajuda a reduzir o risco de contas serem assaltadas através de ataques violentos.

Também pode criar e gerir uma lista autorizada para adicionar endereços IP nos quais confia ou uma lista de bloqueio para impedir sempre que determinados endereços IP iniciem sessão.

Nota:

Vários serviços e pacotes suportam o bloqueio automático, como os seguintes: DSM, SSH, Telnet, rsync, cópia de segurança de rede, sincronização de pasta partilhada, FTP, WebDAV, File Station, Photo Station, Audio Station, Video Station, Download Station, Mail Server, Mail Station, Time Backup, VPN Server, Cloud Station e aplicações móveis da Synology.

Para ativar o bloqueio automático:

  1. Abra Painel de Controlo e aceda a Segurança > Bloqueio Automático.
  2. Assinale a caixa de verificação Ativar bloqueio automático.
  3. Introduza um número de tentativas falhadas de início de sessão no campo Tentativas de início de sessão e um número de minutos no campo No espaço de (minutos). Um endereço IP será bloqueado quando exceder o número de tentativas falhadas de início de sessão no número de minutos especificado.
  4. Assinale a caixa de verificação Ativar validade do bloqueio e introduza um número para remover um endereço IP bloqueado após o número especificado de dias.
  5. Clique em Aplicar.

Gerir Lista de Bloqueio

Clique em Lista Autorizada/de Bloqueio e vá para o separador Lista de Bloqueio.

Para adicionar endereços IP:

Escolha qualquer uma das seguintes opções a partir do menu pendente Criar:

  • Adicionar endereço IP
    Introduza um endereço IP e defina a hora de expiração.
  • Importar lista de endereços IP
    Importe um ficheiro de texto que liste os endereços IP para adicionar um número de endereços IP e defina a hora de expiração ao mesmo tempo. Assinale a caixa de verificação se pretender substituir endereços IP existentes. Se esta opção não estiver seleccionada, serão ignorados os endereços IP duplicados.
    Se o ficheiro selecionado tiver um formato correto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

    • Nota:

    Para importar um ficheiro, este deverá cumprir os seguintes critérios:

    • O ficheiro deverá ser um ficheiro de texto simples.
    • Cada linha do ficheiro pode apenas conter um endereço IP.
    • As linhas de comentários começam com um # (cardinal).

Para remover endereços IP bloqueados:

Selecione os endereços IP que pretende remover da lista e clique em Remover.

Gerir Lista Autorizada

Clique em Lista Autorizada/de Bloqueio e vá para o separador Lista Autorizada.

Para adicionar endereços IP:

Escolha qualquer uma das seguintes opções a partir do menu pendente Criar:

  • Adicionar endereço IP
    Introduza um endereço IP.
  • Importar lista de endereços IP
    Importe um ficheiro de texto que liste os endereços IP para adicionar um número de endereços IP ao mesmo tempo. Assinale a caixa de verificação se pretender substituir endereços IP existentes. Se esta opção não estiver seleccionada, serão ignorados os endereços IP duplicados.
    Se o ficheiro selecionado tiver um formato correto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

Nota:

Para importar um ficheiro, este deverá cumprir os seguintes critérios:

  • O ficheiro deverá ser um ficheiro de texto simples.
  • Cada linha do ficheiro pode apenas conter um endereço IP.
  • As linhas de comentários começam com um # (cardinal).

Para remover endereços IP:

Selecione os endereços IP que pretende remover da lista e clique em Remover.

Nota:

Se o seu dispositivo DiskStation estiver atrás de um servidor proxy inverso, adicione o endereço IP desse servidor proxy inverso à lista autorizada.
