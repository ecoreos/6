Base de dados de sistema

O DiskStation utiliza uma base de dados de sistema para acompanhar os dados de pacotes e entradas de registo guardados nas pastas partilhadas nos seus dispositivos de armazenamento externo, como usbshare1.

Importante: Se pretender aceder aos pacotes de dados armazenados nos dispositivos de armazenamento externo ligados ao DiskStation, ou se pretender visualizar registos do DSM em Menu Principal > Centro de Registos, primeiro tem de activar a base de dados de sistema. Caso contrário, todos os serviços relacionados estarão inacessíveis.

Para activar a base de dados de sistema:

  1. Assinale Activar a base de dados de sistema.
  2. Execute qualquer dos seguintes procedimentos:
    • Escolha Configuração automática se pretender que o DSM seleccione automaticamente uma pasta partilhada para armazenamento da base de dados de sistema. Se o dispositivo de armazenamento externo que contenha a pasta partilhada for removido, o DSM irá seleccionar automaticamente qualquer outra pasta partilhada para armazenamento da base de dados.
    • Escolha Mova a base de dados actual para a pasta partilhada seleccionada e de seguida escolha o menu pendente para mover a base de dados actual para outra pasta partilhada criada no seu dispositivo externo.
    • Seleccione Escolher uma das bases de dados existentes e de seguida escolha a partir do menu pendente para seleccionar uma base de dados existente se existir mais do que um dispositivo externo lugado ao seu DiskStation.
  3. Clique em Aplicar.

Aviso: Para evitar que a base de dados fique corrompida, certifique-se de que ejecta o dispositivo de armazenamento em Menu Principal > Painel de Controlo > Dispositivos e Impressoras > Dispositivos Externos ou clicando no ícone de dispositivos externos no canto superior direito antes de o remover do DiskStation.
