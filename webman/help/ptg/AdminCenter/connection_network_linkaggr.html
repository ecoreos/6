Link Aggregation*

DiskStation é compatível com multi-LAN, permitindo-lhe unificar essas interfaces LAN usando a tecnologia Link Aggregation (Agregação de ligações). Link Aggregation aumenta a largura de banda do seu DiskStation agregando múltiplas interfaces de rede e proporciona transferência de tráfego para manter a ligação de rede caso a ligação não esteja a funcionar.

Depois de unificadas as suas interfaces LAN verá uma nova interface com o nome Bond no Painel de Controlo > Rede > Interface de rede, que pode ser configurada tal como outras interfaces (como LAN ou PPPoE).

Nota:

  • Com Link Aggregation activado, o tráfego de rede será automaticamente ajustado e equilibrado para cada dispositivo ligado.
  • Se as interfaces de rede estiverem ligadas a múltiplos switches, certifique-se que as portas do switch se encontram na mesma LAN ou VLAN.

Para unificar várias interfaces LAN com o Link Aggregation:

  1. Aceda a Painel de Controlo > Rede > Interface de rede. Clique em Criar > Criar Bond.
  2. Seleccione o modo que deseja.

    Descrições do modo:

    • Adaptive Load Balancing: Este modo optimiza o tráfego de rede recebido e enviado pela sua DiskStation quer o switch suporte ou não Link Aggregation. Para evitar erros inesperados, não active Link Aggregation no switch mesmo quando seja suportado.
    • Link Aggregation Dinâmica IEEE 802.3ad: Este modo optimiza o tráfego de rede recebido e enviado pela sua DiskStation, e requer que IEEE 802.3ad (Dynamic) Link Aggregation (LACP, 802.1AX) esteja activado no(s) switch(es). Se forem usados múltiplos switches, estes têm de ser empilháveis e estar devidamente configurados.
    • Equilíbrio XOR: Este modo equilibra o tráfego de rede recebido e enviado pela sua DiskStation, e requer que Static Link Aggregation esteja activado no(s) switch(es). Se forem usados múltiplos switches, estes têm de ser empilháveis e estar devidamente configurados. Se Link Aggregation não fo suportado pelo switch, o tráfego enviado pela sua DiskStation pode ainda ser equilibrado. Com este modo, as ligações perdidas podem ser detectadas mas não a causa (por ex., quando os cabos Ethernet são desligados ou o sitch está mal configurado).
    • Activo/Em espera: Este modo proporciona à sua DiskStation tolerância de falhas de rede. Ou seja, quando a interface de rede activa avaria, uma das outras portas LAN assume o controlo para manter a ligação de rede. Pode escolher este modo sem configurar o switch ou com um switch que suporte IEEE 802.3ad Link Aggregation.
  3. Seleccione as interfaces para criar o Link Aggregation.
  4. Configure as definições IP. Pode também activar VLAN se necessário. Clique em Aplicar para concluir a configuração.
  5. Assim que terminar, pode ver a nova interface com o nome Bond 1 na lista Interfaces de rede.

Para editar as definições de Link Aggregation:

  1. Para alterar o modo Link Aggregation ou visualizar as informações dos dispositivos físicos, aceda ao Painel de Controlo > Rede > Interface de Rede.
  2. Seleccione a interface Bond 1, e clique no botão Editar.
  3. Aceda aos separadores correspondentes para modificar as definições desejadas.

Para remover o Link Aggregation:

  1. Aceda a Painel de Controlo > Rede > Interface de rede.
  2. Seleccione a interface Bond 1, e clique no botão Eliminar.
  3. Uma vez terminado, verá que Bond 1 será agora substituído por duas interfaces LAN.
