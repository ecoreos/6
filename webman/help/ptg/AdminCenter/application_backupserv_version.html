Serviço de Cópia de Segurança

Serviço de Cópia de Segurança permite a outros utilizadores salvaguardar dados dos dispositivos Synology NAS através do pacote Cópia de segurança e Restauro para o seu Synology NAS.

Ativar o Synology Backup Server

Para ativar o serviço de cópia de segurança:

Pode transformar o seu Synology NAS no destino de dados/cópia de segurança LUN para outros Synology NAS remotos.

  1. Aceda ao Painel de Controlo > Serviço de Cópia de Segurança.
  2. Clique Activar o destino de cópia de segurança para servidores Synology remotos.
  3. Clique em Aplicar para guardar as definições.
  4. Aceda a Painel de controlo > Pasta Partilhada, seleccione uma pasta partilhada, clique emEditar > Permissões e conceda o privilégio Leitura/Escrita aos utilizadores desejados. Repita este passo para incluir mais pastas partilhadas que possam ser utilizadas como destinos da cópia de segurança.

Para atribuir privilégios de cópia de segurança:

Atribua aos utilizadores privilégios de salvaguarda adequados, para que estes utilizadores possam salvaguardar dados de um Synology NAS de origem para o Synology NAS de destino.

  1. Aceda a Painel de Controlo > Privilégios, clique em Cópia de segurança Synology > Editar.
  2. Atribuir Permitir privilégios aos utilizadores desejados, ou especifique os endereços IP permitidos para utilizadores específicos.
