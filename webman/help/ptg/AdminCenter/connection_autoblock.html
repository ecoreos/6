Bloqueio Automático

O bloqueio automático permite-lhe evitar qualquer início de sessão não autorizado. Depois de ativar o serviço, o endereço IP será bloqueado se tiver demasiadas tentativas para iniciar a sessão falhadas. As tentativas para iniciar a sessão falhadas através de SSH, Telnet, rsync, Cópia de Segurança na Rede, Sincronização de Pasta Partilhada, FTP, WebDAV, aplicações móveis Synology, File Station ou DSM serão todas adicionadas. Pode também ter uma Lista Permitida que inclui os endereços IP que nunca serão bloqueados.

Para configurar o bloqueio automático:

  1. Aceda ao separador Geral e assinale Activar bloqueio automático.
  2. Introduza um número para Tentativas para iniciar sessão e No espaço de (minutos) para bloquear um endereço IP após um número definido de tentativas para iniciar sessão falhadas no espaço do número especificado de minutos.
  3. Assinale a caixa de verificação Activar validade do bloqueio e introduza um número para remover um endereço IP bloqueado após o número especificado de dias. Endereços IP bloqueados serão indefinidamente mantidos na lista de bloqueio, excepto se forem manualmente removidos da lista se esta opção não for seleccionada ou se Desbloquear após (dias) estiver definido para 0.
  4. Clique em Aplicar.

Para adicionar endereços IP bloqueados:

Clique no separador Lista de Bloqueio e selecione uma das seguintes opções a partir do menu Criar:

  • Adicionar Endereço IP

    • Introduza um endereço IP e defina a hora de validade.

  • Importar lista de endereço IP

    • Importe um ficheiro de texto com os endereços IP listados para adicionar vários endereços IP e definir a hora de validade ao mesmo tempo. Assinale a caixa de verificação adequada caso deseje substituir endereços IP existentes. Se esta opção não estiver seleccionada, serão ignorados endereços IP duplicados.

    Para importar um ficheiro, este deverá cumprir os seguintes critérios:

    • O ficheiro deverá ser um ficheiro de texto simples.
    • Cada linha do ficheiro pode apenas conter um endereço IP.
    • As linhas de comentários começam com um # (cardinal).

      • Se o ficheiro seleccionado tiver um formato correcto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

Para remover endereços IP bloqueados:

  1. Clique no separador Lista de bloqueio para verificar a lista de endereços IP bloqueados.
  2. Seleccione os endereços IP que deseja remover da lista, e clique em Remover.

Para adicionar endereços IP autorizados:

Clique no separador Lista Permitida e selecione uma das seguintes opções a partir do menu Criar:

  • Adicionar Endereço IP

    • Introduza um endereço de IP.

  • Importar lista de endereço IP

    • Importe um ficheiro de texto com os endereços IP listados para adicionar vários endereços IP ao mesmo tempo. Assinale a caixa de verificação adequada caso deseje substituir endereços IP existentes. Se esta opção não estiver seleccionada, serão ignorados endereços IP duplicados.

    Para importar um ficheiro, este deverá cumprir os seguintes critérios:

      • Se o ficheiro seleccionado tiver um formato correcto, verá todos os endereços IP na área de pré-visualização. Clique em OK para importar.

  1. Clique no separador Lista Permitida para verificar a lista de endereços IP autorizados.
  2. Seleccione os endereços IP que deseja remover da lista, e clique em Remover.
