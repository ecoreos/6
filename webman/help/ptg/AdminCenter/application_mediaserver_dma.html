Compatibilidade com DMA

Pode melhorar a compatibilidade com DMA para se certificar de que os conteúdos multimédia na sua DiskStation são reproduzidos de forma suave nos seus dispositivos DMA.

Para activar as definições do transcoder:

  1. Assinale as caixas de verificação seguintes para activar os transcoders se o seu DMA não suportar os seguintes formatos de áudio: FLAC, APE, AAC, OGG e AIFF.
  2. Clique em OK.

Para ajudar os DMA a identificar os formatos de ficheiros multimédia:

  1. Assinale a caixa de verificação Activar tipos MIME personalizados e introduza Tipos MIME personalizados.
  2. Clique em OK.

Acerca dos Tipos de MIME

Os tipos de MIME, também conhecidos como Tipos de multimédia da Internet, podem melhorar a compatibilidade dos ficheiros multimédia com DMA especificando o tipo e sub-tipo de ficheiros multimédia. Pode personalizar os tipos MIME para ajudar o seu dispositivo DMA a identificar os formatos de ficheiro correctos.

Para gerir a lista de dispositivos DMA:

  1. Clique em Lista de dispositivos para visualizar a lista de dispositivos DMA encontrados pela DiskStation.
  2. Proceda de uma das seguintes formas:
    • Assinale ou retire a selecção das caixas de verificação para permitir ou negar o acesso dos dispositivos ao Servidor Multimédia.
    • Clique duas vezes nos campos em Perfil e seleccione o tipo correcto para os seus dispositivos a partir do menu pendente.
      Nota: O Perfil Predefinido será utilizado se o Servidor Multimédia não conseguir identificar o tipo do seu dispositivo DMA. Se o seu dispositivo não estiver listado aqui, pode tentar utilizar o DLNA Genérico (para dispositivos que suportem DLNA) ou Não-DLNA Genérico (para dispositivos que não suportem DLNA).
    • Clique em Actualizar se pretender certificar-se de que a lista de dispositivos está actualizada.
  3. Clique em Guardar e Cancelar para guardar as alterações e fechar as janelas da Lista de dispositivos, ou clique em Cancelar directamente para sair sem guardar as alterações.
