Pasta Partilhada

As pastas partilhadas são os diretórios básicos onde pode armazenar ficheiros e pastas em DiskStation. Antes de armazenar ficheiros no seu DiskStation, é necessário criar, pelo menos, uma pasta partilhada no DSM. Os dados armazenados em pastas partilhadas podem ser mantidos como privados ou partilhados com utilizadores ou grupos de utilizadores específicos de acordo com as permissões de acesso personalizáveis.

Em baixo está uma lista de pastas partilhadas predefinidas que foram criadas automaticamente quando determinados serviços, aplicações ou pacotes foram instalados ou ativados.

Nome Descrição
public A pasta public foi criada automaticamente após a configuração pela primeira vez de DiskStation.*
web A pasta web é a localização de armazenamento predefinida para o conteúdo de um website alojado em DiskStation. É criada automaticamente quando a Web Station for ativada.
photo A pasta photo é utilizada para armazenar as fotografias e os vídeos que pretende partilhar com a Photo Station. É criada automaticamente quando a Photo Station ou Servidor Multimédia DLNA/UPnP for ativado.
music A pasta music é utilizada para armazenar a música que pretende ouvir com a Audio Station. É criada automaticamente quando a Audio Station ou Servidor Multimédia DLNA/UPnP for ativado.
video A pasta video é utilizada para armazenar os ficheiros de vídeo que pretende pesquisar através dos dispositivos DLNA/UPnP. É criada automaticamente quando a Video Station ou Servidor Multimédia DLNA/UPnP for ativado.
surveillance A pasta surveillance é utilizada para armazenar as gravações da Surveillance Station. É criada automaticamente quando a Surveillance Station for ativada. É apenas de leitura e só pode ser acedida pelo administrador do sistema.
home As pastas home fornecem espaços privados para que cada utilizador armazene dados que só podem ser acedidos por esse utilizador. É criada automaticamente quando a funcionalidade Home do Utilizador for ativada.
homes A pasta homes contém as pastas home de todos os utilizadores. É criada automaticamente quando a funcionalidade Home do Utilizador for ativada. Apenas o administrador do sistema pode aceder e gerir as pastas home de todos os utilizadores.
NetBackup A pasta NetBackup é criada automaticamente quando a Cópia de Segurança de Rede for ativada.
usbshare A pasta usbshare[número] é criada automaticamente quando liga um dispositivo de unidade externa USB à porta USB de DiskStation.*
satashare A pasta satashare é criada automaticamente quando liga um dispositivo de unidade externa eSATA à porta eSATA de DiskStation.*

Nota:

  • Apenas pode editar as permissões de pasta partilhada de utilizadores no modo simples. Se pretender criar grupos de utilizadores ou gerir os privilégios de acesso dos grupos a pastas partilhadas, aceda a Painel de Controlo (Modo Avançado) > Grupo.

Para criar uma pasta partilhada:

  1. Aceda a Painel de Controlo (Modo Simples) > Pasta Partilhada e clique no ícone + no painel esquerdo.
  2. Especifique um nome e selecione um volume para criar a pasta partilhada.
  3. Marque Ativar Reciclagem se necessário. Se esta opção estiver ativada, quando os ficheiros da pasta partilhada forem eliminados, serão movidos para uma pasta com o nome #recycle. O acesso à reciclagem pode estar limitado aos utilizadores pertencentes ao grupo administrators.
  4. Marque ou desmarque as caixas adequadas de cada utilizador para personalizar as respetivas permissões de acesso à pasta partilhada:
    • Leitura/Escrita: O utilizador pode aceder e efetuar alterações aos ficheiros e subpastas na pasta partilhada.
    • Só de leitura: O utilizador pode aceder aos ficheiros e subpastas na pasta partilhada.
    • Sem acesso: O utilizador não pode aceder aos ficheiros ou subpastas na pasta partilhada.
  5. Clique em Aplicar para guardar as definições.

Nota:

  • Os nomes das pastas partilhadas não são sensíveis a maiúsculas e minúsculas e podem incluir 1 a 32 caracteres. Os nomes não podem incluir os seguintes caracteres:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Os seguintes nomes estão reservados para utilização do sistema:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • O primeiro caractere do nome de uma pasta partilhada não pode ser um sinal de subtração ou um espaço e o último caractere não pode ser um espaço.
  • Quando ocorrerem conflitos de permissões, a prioridade das permissões é a seguinte: Sem acesso > Leitura/Escrita > Só de leitura.
  • Quando criar uma nova pasta partilhada, se as permissões de acesso para os utilizadores pertencentes ao grupo administrators estiverem definidas para Sem acesso, estes utilizadores apenas podem visualizar a pasta partilhada em Painel de Controlo > Pasta Partilhada.
  • Pode configurar as definições de privilégios da pasta Reciclagem em File Station clicando com o botão direito na pasta #recycle. Escolha Propriedades e clique no separador Permissões.

Para editar uma pasta partilhada:

  1. Aceda a Painel de Controlo (Modo Simples) > Pasta Partilhada.
  2. Selecione a pasta partilhada pretendida no painel esquerdo.
  3. Edite as informações gerais da pasta partilhada, como o nome, a localização e as permissões de utilizador.
  4. Se a opção Ativar Reciclagem já estiver marcada, pode clicar em Esvaziar Reciglagem para remover permanentemente todos os ficheiros eliminados que se encontrem na reciclagem.
  5. Clique em Aplicar para guardar as definições.

Para eliminar uma pasta partilhada:

  1. Aceda a Painel de Controlo (Modo Simples) > Pasta Partilhada.
  2. Selecione a pasta partilhada pretendida no painel esquerdo.
  3. Clique no ícone Lixo no painel esquerdo para eliminar a pasta partilhada selecionada.
