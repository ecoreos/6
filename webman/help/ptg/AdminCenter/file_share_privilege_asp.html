Gerir permissões avançadas de pasta partilhada

A página em Painel de Controlo > Pasta Partilhada > Editar > Avançado inclui as opções seguintes para permitir definir mais detalhadamente as permissões de acesso de uma pasta partilhada.

Permissões de Partilha avançadas

As permissões de partilha avançadas proporcionam um patamar adicional de controlo para gerir as permissões de acesso de pastas partilhadas. Quando activada, os utilizadores podem visualizar ou modificar o conteúdo de uma pasta partilhada apenas se tiverem sido concedidas ao utilizador ou grupo as permissões de partilha avançadas e as permissões Windows ACL (localizadas em Pasta Partilhada > Editar > Permissões).

Para activar as permissões de partilha avançadas:

  1. Assinale a caixa Activar permissões de partilha avançadas.
  2. Clique em Permissões de Partilha Avançadas para modificar as permissões de partilha avançadas para a pasta partilhada.

Nota:

  • As permissões de partilha avançadas são aplicadas quando um utilizador acede a uma pasta partilhada através dos seguintes serviços: Windows File Sharing, Apple File Sharing, File Station, FTP e WebDAV.

Definições Avançadas

Se necessário, pode aplicar mais restrições a utilizadores quando acedem a uma pasta partilhada através da File Station, FTP ou WebDAV.

  • Desactivar pesquisa de directórios: A activação desta opção restringe os utilizadores de visualizarem o conteúdo da pasta partilhada.
  • Desactivar modificação de ficheiros existentes: A activação desta opção restringe os utilizadores de moverem, eliminarem ou modificarem os ficheiros na pasta partilhada. Tenha em atenção que os utilizadores ainda conseguirão visualizar, transferir/carregar, copiar ou descompactar o conteúdo da pasta partilhada.
  • Desactivar transferência de ficheiros: A activação desta opção restringe os utilizadores de transferirem o conteúdo da pasta partilhada.

Nota:

As restrições anteriores não se aplicam aos utilizadores que pertencem ao grupo administrators.
