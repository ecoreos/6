Serviço de Ficheiros Mac

Em Painel de Controlo > Serviços de Ficheiros > Win/Mac/NFS, pode ativar o serviço de ficheiros Mac para tornar o DiskStation num servidor de ficheiros Mac (servidor AFP).

Para ativar o serviço de ficheiros do Mac:

  1. Marque Ativar serviço de ficheiros Mac para tornar o DiskStation num servidor de ficheiros Mac (servidor AFP).
  2. Defina uma pasta de cópia de segurança Time Machine, se necessário. (Consulte a secção abaixo para obter mais informações.)
  3. Clique em Aplicar.

Nota:

  • Para efetuar uma cópia de segurança da Time Machine para o DiskStation, selecione uma pasta partilhada no menu pendente.
  • As pastas partilhadas que estiverem a ser utilizadas por outros serviços não estarão disponíveis para a Time Machine.

Ativação de Registo de Transferência

Marcar esta opção ativará operações de acesso a registo do ficheiro, incluindo criar, eliminar, ler, escrever, mover e renomear ficheiros ou pastas através de AFP, também conhecido como protocolo de Apple Filing Protocol (Protocolo de Ficheiros Apple).

Definições avançadas

Aplicar permissões de UNIX predefinidas:

Ao ativar esta opção, aplica as permissões UNIX predefinidas quando estiver a carregar ou a criar ficheiros e pastas. As permissões aplicadas são as mesmas permissões aplicadas pelo comando UNIX umask. Quando esta opção estiver ativada, as permissões UNIX são 644 para ficheiros e 755 para pastas. Quando esta opção estiver desativada, as permissões UNIX são 666 para ficheiros e 777 para pastas. O valor umask predefinido é 022.

Nota:

Ativar esta opção pode provocar questões de inconsistência de permissões entre diferentes protocolos. Para evitar inconsistências, sugerimos que deixe esta opção desativada.

Libertar recursos de imediato após desligar:

Ativar esta opção liberta de imediato recursos quando a ligação ao servidor AFP é desligada de forma inesperada. Caso receba mensagens de erro "Volume em utilização" quando utiliza o Time Machine, ative esta opção.

Para ativar a emissão de impressora Bonjour:

Assinale Ativar Emissão de Impressora Bonjour para permitir que os clientes Mac pesquisem e localizem a impressora ligada ao DiskStation.

Para procurar o conteúdo do ficheiro através do Spotlight no Finder:

Depois de totalmente indexadas as pastas e montadas no seu computador Mac, pode usar Spotlight no Finder para pesquisar os ficheiros contidos e seus conteúdos.

  1. Aceda ao Painel de Controlo > Serviço de Indexação > Indexação de Ficheiros > Lista de Pastas Indexadas > Criar, para indexar pastas específicas e ficheiros contidos por tipo de ficheiro.
  2. Clique em OK para realizar a indexação.
  3. Monte as pastas indexadas no seu computador Mac.
  4. No seu computador Mac, clique na pasta montada desejada no painel esquerdo do Finder, e use Spotlight no Finder para procurar os ficheiros contidos.

Nota:

  • Para procurar dentro de pastas montadas, use Spotlight no Finder. Spotlight no ambiente de trabalho Mac apenas permite a procura de ficheiros locais.
  • Nas versões Mac OS X 10.9 e posteriores, Spotlight no Finder é compatível com a pesquisa dentro de pastas montadas identificadas por nome e categoria.
