Avançado

Pode modificar as seguintes definições avançadas em Painel de Controlo > Segurança > Avançadas para as sessões de navegação do DSM:

Compressão HTTP

Pode ativar Compressão HTTP para DiskStation. Reduzindo a largura de banda do tráfego de rede, pode aumentar a velocidade de carregamento da página da web.

Para ativar/desativar a Compressão HTTP:

  • Marque/desmarque Compressão HTTP em Painel de Controlo > Segurança > Avançadas.

Perfil TLS/SSL

Pode escolher o nível de segurança das ligações encriptadas HTTPS. Tem os seguintes 3 níveis:

  • Compatibilidade moderna - Só compatível com os navegadores e dispositivos móveis mais recentes.
  • Compatibilidade intermédia - Entre o nível de segurança mais alto e mais baixo. É o valor predefinido do sistema.
  • Retrocompatibilidade antiga - Não segura, mas suporta a maioria dos navegadores e dispositivos móveis.
