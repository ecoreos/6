Editar um Utilizador

Procure a conta de utilizador (introduzindo o nome de utilizador no campo Pesquisa), clique duas vezes ou clique em Editar e, de seguida, poderá editar os dados dessa conta.

Nota:
  1. Se desativar uma conta, a conta deixará de funcionar.
  2. Quota deve ser um número inteiro maior ou igual a 0, e menor que a capacidade do disco/volume. Um valor de quota de 0 significa que Quota efectiva será determinada pelo Limite do grupo.
  3. Se houver mais que um volume/pasta partilhada no servidor, pode definir a quota para o utilizador em cada volume/pasta partilhada.
  4. A informação do e-mail na página de "Notificação" deve ser definida corretamente para que o sistema possa notificar o utilizador por e-mail depois de a informação da conta ser criada ou modificada.
  5. Quando configurar o limite de velocidade para um utilizador, verá números em Resultado se um limite de velocidade de grupo já estiver aplicado ao utilizador.
  6. O limite de velocidade atribuído a um utilizador não será incluído no limite de velocidade do grupo ao qual o utilizador pertence.
