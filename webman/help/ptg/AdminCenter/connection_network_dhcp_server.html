DHCP Server

O seu DiskStation pode atuar como servidor DHCP e atribui endereços IP dinâmicos a clientes DHCP na rede local.

Para ativar o servidor DHCP:

  1. Vá para Painel de Controlo > DHCP Server > Interface de Rede.
  2. Selecione uma interface de rede (por exemplo, LAN 1) à qual atribuir endereços IP, e clique em Editar.
  3. Na janela de contexto, clique no separador DHCP Server e selecione Ativar o servidor DHCP.
  4. Especifique as seguintes definições DHCP:
    • Tempo de validade de endereço (minutos)
    • DNS Primário
    • DNS Secundário (opcional)
    • Nome de Domínio (opcional)
    • Ativar a Deteção Automática Web Proxy (opcional)
  5. Especifique, pelo menos, uma sub-rede. Para configuração da sub-rede, clique em Adicionar para especificar a seguinte informação:
    • Endereço IP inicial
    • Endereço IP final
    • Máscara de rede
    • Gateway
  6. Clique em Criar para mostrar a sub-rede criada na lista.
  7. Depois de criar sub-redes suficientes, não se esqueça de ativar, pelo menos, uma sub-rede.
  8. Clique em OK.

Nota:

  • Pode criar tantas sub-redes quantas pretender.
  • Se coordenar com um servidor de relé DHCP, o seu DiskStation como DHCP pode atribuir endereços IP a dispositivos noutra sub-rede.

Clientes DHCP

Pode ver uma lista de clientes de DHCP e as respetivas configurações de rede (como endereço MAC, endereço IP, nome de anfitrião e quanto tempo resta até expirar a concessão DHCP e o estado da reserva).

  1. Vá para Painel de Controlo > DHCP Server > Interface de Rede e selecione uma interface desejada.
  2. Clique em Editar > Clientes DHCP para encontrar os clientes e os detalhes.

Para gerir endereços IP para clientes DHCP:

Pode forçar um cliente a receber sempre o mesmo endereço IP durante a renovação da concessão DHCP, e gerir estes clientes.

  • Para adicionar clientes DHCP: Clique em Adicionar para atribuir o endereço IP ao dispositivo do cliente.
  • Para reservar o endereço IP: Clique no ícone da fechadura para reservar o endereço IP para o atual dispositivo.
  • Para cancelar a reserva: Clique no ícone da fechadura para que o atual endereço IP possa ser utilizado por outro dispositivo depois do tempo de validade do endereço expirar.

Nota:

Se um dispositivo pode ter tanto endereços IPv4 como IPv6, pode reservar um endereço IPv4 e um endereço IPv6 para o dispositivo.

Para pesquisar clientes DHCP:

  1. Clique no ícone do funil no canto superior direito.
  2. Seleccione um dos critérios de pesquisa em baixo:
    • Tudo: Apresenta todos os clientes DHCP e as reservas DHCP.
    • Clientes DHCP: Apresenta todos os clientes DHCP com endereços IP reservados e não reservados.
    • Reservas DHCP: Apresenta todos os endereços IP reservados. Ainda não é possível emparelhar um endereço IP reservado com qualquer dispositivo.

Opções DHCP

Depois de especificar as opções DHCP, o seu DiskStation como servidor DHCP pode enviar informações relacionadas aos clientes DHCP numa certa subrede.

Para especificar as opções DHCP para uma subrede:

  1. Vá para Painel de Controlo > DHCP Server > Interface de Rede.
  2. Selecione a interface de rede desejada e clique em Editar.
  3. Aceda ao DHCP Server e aceda à Lista de subrede.
  4. Execute uma das ações abaixo:
    • Para especificar as opções DHCP para uma subrede existente: Selecione a interface desejada e clique em Editar.
    • Para criar uma subrede e especificar as opções DHCP: Clique em Adicionar e especifique as definições da subrede.
  5. Selecione as opções DHCP desejadas; para especificar os valores correspondentes, clique nos campos em branco para editar.
  6. Clique em OK.

Fabricante

Clientes DHCP de diferentes fabricantes podem necessitar de informações DHCP adicionais. Ao configurar opções DHCP para os fabricantes, o seu DiskStation pode fornecer tais detalhes aos clientes.

Para saber quais opções DHCP compatíveis com cada fabricante, contacte o fabricante em causa, para mais detalhes.

Para adicionar fabricantes:

  1. Vá para Painel de Controlo > DHCP Server > Fabricante.
  2. Clique em Criar para especificar o fabricante e as suas opções DHCP:
    • Nome: Especifique o nome do fabricante.
    • Código: Especifique o código numérico da opção DHCP.
    • Valor: Especifique o valor da opção.
    • +: Adicionar uma opção DHCP.
    • -: Eliminar uma opção DHCP.
  3. Clique em Criar para concluir a configuração.

Para editar opções DHCP para fabricantes:

  1. Vá para Painel de Controlo > DHCP Server > Fabricante.
  2. Selecione o fabricante desejado e clique em Editar para gerir as opções DHCP.
  3. Clique em OK.

PXE

O seu DiskStation pode atuar como servidor DHCP para implementar a configuração de PXE (Pre-Boot Execution Environment). PXE (Pre-boot Execution Environment) é um ambiente de rede que permite que dispositivos cliente arranquem utilizando uma interface de rede em vez de um dispositivo de armazenamento de dados (por exemplo, discos rígidos).

Quando implementa PXE, pode ser armazenado um carregador de arranque no DiskStation ou servidor TFTP remoto, e os dispositivos cliente podem arrancar através da rede utilizando o carregador de arranque.

Para usar o seu DiskStation como servidor de carregador de arranque:

  1. Certifique-se de que selecionou Ativar serviço TFTP (em Painel de Controlo > Serviços de Ficheiro > TFTP).
  2. Selecione um ficheiro que aloje o ficheiro carregador de arranque como pasta raiz TFTP.
  3. Certifique-se que tem pelo menos uma subrede DHCP ativada (em Painel de Controlo > DHCP Server > Interface de Rede).
  4. Vá para Painel de Controlo > DHCP Server > PXE.
  5. Selecione Ativar PXE (Pre-boot Execution Environment).
  6. Selecione Servidor TFTP Local e selecione o ficheiro de carregador de arranque.
  7. Clique em Apply para guardar as definições.

Para usar um servidor TFTP remoto como servidor de carregador de arranque:

Ao usar um servidor TFTP remoto como servidor de carregador de arranque, pode permitir aos clientes arrancar sem ter de alterar a infraestrutura de rede atual.

  1. Certifique-se que tem pelo menos uma subrede DHCP ativada (em Painel de Controlo > DHCP Server > Interface de Rede).
  2. Vá para Painel de Controlo > DHCP Server > PXE.
  3. Selecione Ativar PXE (Pre-boot Execution Environment).
  4. Selecione Servidor TFTP remoto, e especifique o endereço IP do servidor e o caminho de ficheiro do carregador de arranque.
  5. Clique em Apply para guardar as definições.
