Saída DC*

Em Painel de Controlo > Hardware e Energia > Saída DC, pode definir o sistema para ligar ou desligar a saída DC automaticamente para uma hora especificada.

Para adicionar uma nova agenda para ligar/desligar a saída:

  1. Clique no botão Criar para abrir a caixa de diálogo Adicionar Agendamento.
  2. Escolha Saída ligada ou Saída desligada.
  3. Defina a Data e a Hora para o agendamento de ligar/desligar saída.
  4. Clique em OK.
  5. Clique em Guardar.

O novo agendamento será adicionado à lista de agendamentos.

Para ver o resumo de um agendamento:

Clique em Resumo para navegar no agendamento semanal de ligar/desligar saída.

Nota:

  • Pode especificar até 100 agendamentos para um DiskStation.
  • Nos seguintes casos, pode ocorrer uma falha do sistema para ligar/desligar automaticamente a saída DC:
    • O Gestor de Armazenamento está a executar tarefas.
    • O sistema está a encriptar pastas partilhadas.
    • O sistema está a executar o Secure Erase em discos rígidos.
    • O DSM está a ser atualizado.
    • O Centro de Pacotes está a executar tarefas.
    • O sistema está a atualizar bases de dados.
    • O sistema está a executar tarefas de cópia de segurança de dados.

* Apenas disponível em modelos específicos.
