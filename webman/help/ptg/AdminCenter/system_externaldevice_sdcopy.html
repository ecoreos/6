SDCopy*

Pode seleccionar e configurar uma localização alvo para a função SDCopy.

Para activar a SDCopy:

  1. Assinale a caixa de verificação Activar SDCopy.
  2. Seleccione uma pasta partilhada de destino no menu pendente.
  3. Clique em OK.

Quando liga um dispositivo SD na ranhura SD do painel frontal da Synology DiskStation, pode premir o botão Copy (Copiar) no painel frontal para copiar os dados no dispositivo SD para a pasta partilhada de destino designada aqui. Se a pasta de destino SDCopy for apagada, o sistema irá substituí-la automaticamente pela pasta partilhada pública.

    Limitações:
  1. Apenas são suportados dispositivos SD no formato FAT ou NTFS.
  2. Os dispositivos SD devem ser encaixados na ranhura SD no painel frontal do Synology DiskStation.
  3. O espaço disponível na pasta partilhada de destino deve ser superior ao espaço utilizado do dispositivo SD.

____
* Apenas disponível em modelos específicos.
