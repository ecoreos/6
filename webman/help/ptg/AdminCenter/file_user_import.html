Importar Utilizadores

Pode criar contas de utilizador numa base de lotes, importando uma lista de utilizadores.

Para criar a lista de utilizadores:

Método 1. Use um editor de texto:

  1. Abra um novo ficheiro com o editor de texto.
  2. Introduza a informação do utilizador pela seguinte ordem e separe-a com TAB:
    • Nome de Utilizador
    • Palavra-passe
    • Descrição
    • E-mail
    • Quota (MB) no volume 1 (repetir este passo se tiver mais do que um volume)

      Nota:

      A quota do utilizador não é suportada no sistema de ficheiros Btrfs e dispositivos de armazenamento externos (por ex., unidadades USB/eSATA).

  3. Vá para a próxima linha e repita o passo 2 até introduzir todos os utilizadores. Cada linha indica a informação de um utilizador.
  4. Guarde o ficheiro na codificação UTF-8.

Método 2. Use o Microsoft Excel:

  1. Abra um novo ficheiro com o Excel.
  2. Introduza a informação dos utilizadores pela seguinte ordem e separe-a por colunas numa única linha:
    • Nome de Utilizador
    • Palavra-passe
    • Descrição
    • E-mail
    • Quota (MB) no volume 1 (repetir este passo se tiver mais do que um volume)

      Nota:

      A quota do utilizador não é suportada no sistema de ficheiros Btrfs e dispositivos de armazenamento externos (por ex., unidadades USB/eSATA).

  3. Vá para a próxima linha e repita o passo 2 até introduzir todos os utilizadores. Cada linha indica a informação de um utilizador.
  4. Abra um novo ficheiro com o editor de texto. Copie as entradas da folha de cálculo e cole-as no ficheiro de texto.
  5. Guarde o ficheiro na codificação UTF-8.

Nota:

  • Todos os campos são opcionais, exceptuando o nome de utilizador.
  • Em baixo encontra um exemplo do ficheiro de texto:
    • rosy 1234 mãe rosy@abc.com 80000
    chaddy 5678 irmão chaddy@abc.com 80000
    danny 8901 primo danny@abc.com 20000
    anny 2345 irmã anny@abc.com 20000

Para importar a lista de utilizadores:

  1. Escolha Criar > Importar Utilizadores.
  2. Assinale a caixa de verificação Substituir contas duplicadas se pretender substituir as contas duplicadas pelas existentes na lista de utilizadores.
  3. Clique em Pesquisar e seleccione o ficheiro da lista de utilizadores. Após o ficheiro ser seleccionado, a região de pré-visualização irá exibir a lista de utilizadores importados. Utilizadores com status irregulares não serão importados.
  4. Clique em Carregar para criar as contas de utilizador.

Nota:

O ficheiro da lista de utilizadores deve estar no formato UTF-8.
