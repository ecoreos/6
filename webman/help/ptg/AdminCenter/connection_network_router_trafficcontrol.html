Controlo de Tráfego do Router

O controlo de tráfego do router está activo quando DiskStation está no modo router. O Controlo de tráfego do router (em Painel de Controlo > Rede > Controlo de Tráfego do Router) gere o tráfego de carregamento e transferência em execução na sua rede privada DiskStation. Quando criar regras, necessita de especificar a Largura de Banda Garantida e a Largura de Banda Máxima para limitar o tráfego de carregamento e transferência.

Explicação dos Termos

  • Largura de Banda Garantida: Define o tráfego de saída que este serviço garante quando toda a largura de banda do sistema for suficiente.
  • Largura de Banda Máxima: Define o tráfego de saída que este serviço pode utilizar quando toda a largura de banda do sistema for suficiente e existir Largura de Banda Restante do Sistema. Sugerimos que calcule primeiro a Largura de Banda de Saída do Sistema. Em seguida, certifique-se de que a soma da Largura de Banda Garantida de cada serviço não é maior que a Largura de Banda de Saída do Sistema ou a definição poderá não funcionar corretamente.

Fórmula de Definição

  • Largura de Banda de Saída do Sistema = Total Largura de Banda Garantida de todos os serviços + Largura de Banda Restante do Sistema
  • Largura de Banda Garantida para cada serviço ≤ Largura de Banda Máxima para cada serviço

Nota:

Se os pacotes de saída não corresponderem a qualquer regra de controlo de tráfego do router, o comportamento predefinido será sem limite. Pode adicionar a sua própria regra de controlo de tráfego do router predefinida no fim de todas as regras escolhendo Todas as portas e especifique Largura de Banda Garantida e Largura de Banda Máxima.

Para criar uma regra de controlo de tráfego do router:

  1. Selecione uma interface de rede no menu pendente, no canto superior direito.
  2. Clique em Criar.
  3. Na secção Portas, selecione uma das seguintes opções:
    • Escolha Tudo para aplicar a regra a todas as portas.
    • Escolha Personalizar e clique em Personalizar. Introduza até 15 portas separadas por vírgulas ou especifique um intervalo de portas para aplicar esta regra.
    • Também pode especificar o protocolo.
  4. Na secção IP de origem, selecione uma das seguintes opções:
    • Selecione Tudo para aplicar esta regra a todos os endereços IP.
    • Escolha IP Específico e clique em Selecionar. Pode especificar um endereço IP ou um intervalo de IP para aplicar esta regra.

Agora pode ver a regra de controlo de tráfego do router criada na lista de regras.

Para eliminar as regras de controlo de tráfego do router, proceda de uma das seguintes formas:

  • Desmarque a caixa de verificação da regra na coluna Ativado da lista.
  • Elimine a regra de controlo de tráfego do router.

Nota:

  • Pode criar até 100 regras para cada interface de rede.
  • Recorra à função arrastar e largar para reorganizar as regras na lista.
  • As regras são priorizadas de acordo com a sua posição na lista.
  • As portas especificadas referem-se aos números das portas dos serviços de Internet externos à rede local.
  • Os IPs de origem referem-se aos endereços IP dos dispositivos que utilizam o Synology NAS como router.
