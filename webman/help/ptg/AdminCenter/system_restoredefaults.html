Restaurar Predefinições

Pode executar uma reposição para as definições de origem em Painel de Controlo > Atualizar e Restaurar > Restaurar Predefinições para restaurar o seu dispositivo DiskStation para as definições originais do fabricante. Todos os dados do utilizador guardados nas unidades serão apagados e todo o sistema será restaurado para as predefinições.

Nota:

  • A opção Restaurar Predefinições fica desativada quando o DiskStation pertence a um cluster de high-availability. Para restaurar as definições predefinidas de um servidor pertencente a um cluster de high-availability, deve remover em primeiro lugar o cluster de high-availability em High Availability > Visão Geral > Gerir.
