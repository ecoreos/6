Hibernação

Pode especificar o período de tempo que os discos rígidos permanecem inativos antes de entrarem no modo de hibernação de HDD. Durante a hibernação de HDD, os discos rígidos param de rodar, reduzindo o consumo de energia e prolongando a durabilidade dos mesmos. Pode especificar diferentes períodos de tempo para discos internos e discos eSATA/USB externos.

Nota:

  • A hibernação de HDD termina após o acesso.
  • Apenas certos dispositivos de armazenamento USB externo suportam hibernação de HDD.

Porque é que o meu DiskStation não hiberna?

Atenção que os seguintes serviços irão evitar que o DiskStation entre no modo de hibernação. Para além disso, o sistema irá sair do modo de hibernação dois minutos antes de qualquer tarefa agendada.

Pacotes:

  • Download Station: DiskStation não pode hibernar se a Download Station estiver com eMule em execução, transferência RSS agendada ou tarefas ativas.
  • Mail Station: DiskStation não pode hibernar se a Mail Station estiver a obter dados de POP3.
  • Surveillance Station: DiskStation não pode hibernar se a Surveillance Station estiver a gravar.
  • Servidor multimédia: DiskStation não pode hibernar se o registo DMA estiver ativo nas definições do Servidor Multimédia (Aceda a Menu Principal > Servidor Multimédia > Servidor Multimédia > Diagnosticar).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Outro software de terceiros ou optware como SABnzbd

Processos:

  • Miniaturas e indexação: DiskStation não pode hibernar enquanto gera miniaturas ou reindexa ficheiros multimédia após atualização. Durante este tempo, o processo "synoindexd" será listado como "Em execução" (Aceda a Menu Principal > Monitor de Recursos > Processo).
  • Local Master Browser: DiskStation não pode hibernar se Local Master Browser estiver ativo (Aceda a Painel de Controlo > Win/Mac/NFS > Serviço de Ficheiros do Windows > Ativar Local Master Browser).
  • Windows Media Player: DiskStation não pode hibernar se o Serviço de Partilha de Rede do Windows Media Player estiver ativo na LAN.
  • DiskStation não pode hibernar se um número de processos em simultâneo requerer troca, onde a capacidade RAM foi ultrapassada e os HDDs são usados temporariamente para operações de leitura/escrita.

Outros

  • Dispositivos USB: DiskStation não pode entrar em Hibernação de Sistema se algum dispositivo USB estiver ligado à unidade.

Hibernação de sistema

Pode configurar o DiskStation para entrar em hibernação de sistema um minuto após os discos rígidos entrarem em hibernação. A hibernação de sistema reduz o consumo de energia de DiskStation.

Durante o modo de hibernação de sistema:

  1. O botão de energia entra no modo respiração (ciclo lento de ligar/desligar). As outras LED desligam-se.
  2. É cortada a energia para os discos rígidos.
  3. A CPU pára.
  4. A ventoinha externa pára.
  5. As LED nas portas LAN desligam-se.

Nota:

  • Qualquer pacote SMB/CIFS em transmissão poderá evitar que o servidor entre em hibernação de sistema, incluindo navegadores de ficheiros do Windows a funcionar na rede. Se desejar ignorar pacotes de transmissão CIFS ao entrar em hibernação de sistema, selecione Ignorar pacotes em transmissão do Explorador do Windows. Tenha em conta que uma vez selecionada esta opção, os pacotes SMB/CIFS não podem ser usados para sair da hibernação de sistema. Ligar dispositivos USB irá evitar também que o servidor entre em hibernação de sistema.

Suspensão Total

Sono profundo para unidades de expansão pode ser ativado através da interface DiskStation emparelhada. Pode configurar a unidade de expansão e o DiskStation emparelhado para inatividade por diferentes períodos de tempo. Por isso, pode configurar unidades de expansão para entrar em Sono Profundo enquanto o DiskStation emparelhado permanece online.

Durante o modo sono profundo:

  • Todos os LEDs da unidade de expansão estarão desligados.
  • A energia dos discos rígidos da unidade de expansão desliga-se.
  • A ventoinha na unidade de expansão pára.

Nota:

  • Sono profundo termina após o acesso.
  • Para garantir o normal funcionamento quando utiliza a função sono profundo numa unidade de expansão, deslize o comutador Auto/Manual, no painel traseiro, para Auto.

Desligar Automático

Pode especificar a duração do período de tempo em que o disco interno se mantém em hibernação antes do seu DiskStation se desligar automaticamente.

Nota:

  • Deve ativar primeiro a hibernação da unidade de disco rígido (HDD) interno.
  • Deve ativar primeiro o serviço WOL para poder ligar remotamente o seu DiskStation.
