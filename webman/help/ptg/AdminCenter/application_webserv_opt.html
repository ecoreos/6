Serviço HTTP

Além da porta número 80 predefinida, pode adicionar outra porta para a Photo Station e para a Web Station.

Para adicionar um número de porta adicional para acesso HTTP:

  1. Assinale a opção Adicionar uma porta HTTP suplementar.
  2. Introduza o número da porta no campo. Pode adicionar um número de porta adicional de 1 até 65535 para acesso HTTP.

As seguintes portas devem ser excluídas:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, portas predefinidas do eMule: 4662 (TCP), 4672(UDP) e série predefinida da porta FTP: o alcance efetivo poderá depender dependendo dos diferentes modelos.

    Nota:
  1. Os números de portas para acesso HTTP não podem ser os mesmos que o número de porta para o serviço FTP/BT.
  2. A porta HTTP e a porta HTTPS não podem entrar em conflito uma com a outra.
