UPS

Uma UPS (Fonte de Alimentação Ininterrupta) é um dispositivo de energia de recurso que permite à DiskStation continuar a funcionar por mais algum tempo caso ocorra uma falha de energia. Esta função, em Painel de Controlo > Hardware e Energia > UPS, ajuda a evitar perdas de dados, dando ao DiskStation tempo suficiente para poupar dados e desmontar volumes antes de perder energia e encerrar.

Modo de Segurança

Quando o DiskStation entra no Modo de Segurança, pára todos os serviços e desmonta volumes para evitar perda de dados e encerrar com segurança quando o dispositivo UPS fica sem energia. Por predefinição, o sistema entra no Modo de Segurança quando o dispositivo UPS começa a ficar sem energia. Pode também especificar o tempo até o DiskStation entrar no Modo de Segurança quando ocorre uma falha de energia. No entanto, se o dispositivo UPS ficar com bateria fraca antes do tempo especificado, o sistema entra de imediato no Modo de Segurança.

Em situações em que o DiskStation é encerrado durante o Modo de Segurança, será ligado automaticamente quando a energia é restaurada se tiver ativado a opção Reiniciar automaticamente após uma quebra de energia (localizada em Painel de Controlo > Hardware e Energia > Geral).

Suporte UPS Local

Para detalhes referentes à instalação do hardware, consulte o manual de utilizador do seu dispositivo UPS. Uma vez concluída a instalação e ligado o DiskStation ao dispositivo UPS, pode seguir os passos abaixo para ativar o suporte UPS.

Para ativar o suporte UPS local:

  1. Ligue o dispositivo UPS a uma porta USB no DiskStation.
  2. No separador UPS, assinale a caixa de verificação Ativar o suporte UPS.
  3. Selecione quando o DiskStation deverá entrar no Modo de Segurança quando ocorre uma falha de energia.
  4. Escolha entre enviar ou não um sinal para encerrar para a UPS quando o sistema entrar em Modo de Segurança. Quase todos os dispositivos UPS podem ser ligados automaticamente quando é reposta a energia.
  5. Clique em Aplicar.

Suporte UPS SNMP

O seu DiskStation suporta também a ligação a dispositivos UPS SNMP (Protocolo de Gestão de Rede Simples). Para detalhes referentes à instalação do hardware, consulte o manual de utilizador do seu dispositivo UPS SNMP. Uma vez concluída a instalação e ligado o DiskStation ao dispositivo e à mesma rede, pode seguir os passos abaixo para ativar o suporte UPS.

Para ativar o suporte UPS SNMP:

  1. No separador UPS, assinale a caixa de verificação Ativar o suporte UPS.
  2. Selecione UPS SNMP no menu pendente Tipo de UPS de rede.
  3. No campo Endereço IP de UPS SNMP, introduza o endereço IP do dispositivo UPS SNMP.
  4. Selecione quando o DiskStation deverá entrar no Modo de Segurança quando ocorre uma falha de energia.
  5. Introduza a linha de comunidade para o dispositivo UPS SNMP no campo Comunidade SNMP.
  6. Clique em Aplicar.

Nota:

  • Recomendamos vivamente que use SNMP UPS para implementações Synology High Availability (SHA).
  • O DiskStation suporta dispositivos UPS SNMPv1 e SNMPv2c.

Suporte UPS de Rede

Se existirem dois produtos Synology a utilizar um dispositivo UPS com uma só porta USB, então apenas uma DiskStation pode obter informações relativamente ao estado do dispositivo UPS. Neste caso, um DiskStation pode funcionar como "servidor UPS de rede" e ligar ao dispositivo UPS por USB ou rede (dependendo do dispositivo). O servidor UPS de rede obtém informações UPS e transmite-as para o outro DiskStation "cliente".

Para ativar o suporte UPS de rede:

  1. Inicie a sessão no DiskStation que está ligado ao dispositivo UPS (através de USB ou rede). Se o suporte UPS local ainda não estiver ativado neste DiskStation, veja a secção acima.
  2. No separador UPS, assinale a caixa Ativar servidor UPS de rede para permitir ao DiskStation transmitir informações UPS para o DiskStation cliente.
  3. Clique no botão Dispositivos DiskStation permitidos.
  4. Na janela que aparece, introduza os endereços IP do outro DiskStation que está a utilizar o dispositivo UPS, mas não está ligado por USB ou rede. Clique em OK.
  5. Clique em Aplicar para guardar as definições.

Agora o servidor UPS de rede pode obter informações UPS e transmiti-las para o outro DiskStation que está a utilizar o dispositivo UPS. Consulte as instruções abaixo para ativar o suporte UPS no outro DiskStation.

Para ativar o suporte UPS no DiskStation cliente:

  1. Inicie sessão no outro DiskStation que está a utilizar o dispositivo UPS mas não está ligado através de USB ou rede.
  2. No separador UPS, assinale a caixa Ativar o suporte UPS.
  3. Selecione Servidor UPS Synology no menu pendente Tipo de UPS de rede.
  4. Introduza o endereço IP do servidor UPS de rede (ou seja, o DiskStation ligado ao dispositivo UPS) no campo IP de servidor UPS de Rede.
  5. Selecione quando o DiskStation deverá entrar no Modo de Segurança quando ocorre uma falha de energia. Para simplificar as definições do Modo de Segurança, pode selecionar Como o servidor.
  6. Clique em Aplicar para guardar as definições.

Requisitos de UPS de Rede:

  • Para retransmitir as informações do dispositivo UPS, cada DiskStation deverá estar ligado à mesma rede.
  • O dispositivo de rede (por exemplo, um switch ou hub) ao qual cada DiskStation está ligado, deverá estar ligado ao mesmo dispositivo UPS. Caso contrário, o DiskStation não conseguirá retransmitir informações do dispositivo UPS quando ocorre uma falha de energia.
