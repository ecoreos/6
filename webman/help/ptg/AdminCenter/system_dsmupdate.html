Atualização do DSM

Periodicamente, a Synology lança atualizações gratuitas para o DSM. As atualizações podem incluir novas funcionalidades, melhoramentos de funções e aperfeiçoamentos de desempenho.

Importante: Normalmente, a atualização do DSM não afeta os dados na DiskStation. No entanto, para garantir a segurança dos dados, recomendamos que faça uma cópia de segurança dos mesmos antes de começar.

Executar a Atualização Manual do DSM

Pode transferir o ficheiro de atualização .pat do Centro de Transferências da Synology e executar a atualização manual do DSM.

Para executar a atualização manual do DSM:

  1. Visite www.synology.com para transferir a atualização mais recente do DSM para o seu computador local. Descompacte o ficheiro para obter um ficheiro de atualização .pat.
  2. Clique em Atualização Manual do DSM para entrar na Página de Atualização Manual do DSM
  3. Clique em Pesquisar para selecionar o ficheiro de atualização.
  4. Clique em Aplicar para começar a atualizar o seu DSM. O sistema reiniciar-se-á automaticamente quando a atualização terminar.

Resolução de problemas:

Se não conseguir atualizar o DSM, as seguintes sugestões podem ajudá-lo a resolver o problema:

  • Verifique se o processo de atualização foi interrompido por utilizadores ou outros eventos inesperados, tais como falha de energia.
  • Verifique a versão do DSM. Só pode atualizar o sistema com uma versão de DSM mais recente.
  • Faça a transferência da atualização do DSM novamente, pois o ficheiro de atualização pode estar corrompido.

Se o problema persistir, contacte o Suporte Técnico da Synology.

Executar a Atualização Automática do DSM

Além da atualização manual, sempre que acede à Atualização do DSM, o sistema irá verificar na Synology se existe alguma atualização do DSM disponível para transferência e, de seguida, exibe o resultado em texto vermelho ou verde. Se o resultado lhe indicar que está disponível para transferir a versão mais recente do DSM, o sistema pode transferir o DSM mais recente para si e de seguida executar a atualização.

Para que o sistema transfira o DSM mais recente e execute a atualização:

  1. Se o DSM mais recente estiver pronto para transferência, clique no link Transferir para o sistema transferir o DSM mais recente.
  2. O sistema irá exibir uma das seguintes notificações no ambiente de trabalho para o informar se a transferência foi ou não concluída com sucesso: Transferência do DSM Concluída ou Transferência do DSM Falhou.
  3. Clique na janela de notificação no ambiente de trabalho para aceder a Atualização do DSM.
  4. Execute qualquer dos seguintes procedimentos:
    • Se a transferência do DSM estiver concluída, clique em Atualizar agora para começar a atualizar o DSM imediatamente.
    • Se a transferência do DSM tiver falhado, clique em Repetir para o sistema transferir novamente o DSM mais recente da Synology.

Para verificar as atualizações regularmente:

  1. Assinale Verificar atualizações do DSM regularmente.
  2. Selecione um plano de atualização e configure o Agendamento da Instalação.
  3. Clique em OK.

Será exibida uma notificação no ambiente de trabalho quando for transferida uma atualização do DSM e estiver pronta para ser instalada. Para parar a atualização automática do DSM, pode cancelá-la na notificação no ambiente de trabalho ou alterando as Definições de atualização. As atualizações do DSM serão instaladas antes do encerramento agendado, se a opção tiver sido ativada em Encerrar em Agendamento de Energia.

Resolução de problemas:

Se visualizar mensagens de erro (em texto vermelho) durante a verificação ou transferência do DSM mais recente, as seguintes sugestões podem ajudar a resolver o seu problema:

  • Certifique-se de que a sua DiskStation está ligada à Internet.
  • O servidor de transferências da Synology pode estar ocupado no momento. Tentar transferir novamente mais tarde pode resolver o problema.

Se o problema persistir, contacte o Suporte Técnico da Synology.
