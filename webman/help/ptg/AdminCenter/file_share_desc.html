Pasta Partilhada

As pastas partilhadas são os directórios básicos onde pode armazenar ficheiros e pastas em DiskStation. Antes de armazenar ficheiros em DiskStation, é necessário criar, pelo menos, uma pasta partilhada. Os dados armazenados em pastas partilhadas podem ser mantidos como privados ou partilhados com utilizadores ou grupos de utilizadores específicos de acordo com as permissões de acesso personalizáveis. Além disso, as pastas partilhadas podem ser encriptadas durante a criação, adicionando uma camada adicional de protecção.

Pastas Partilhadas Predefinidas

Em baixo está uma lista de pastas partilhadas que foram criadas automaticamente quando determinados serviços, aplicações ou pacotes foram instalados ou activados.

Nome Descrição
public A pasta public foi criada automaticamente após a configuração pela primeira vez de DiskStation.*
web A pasta web é a localização de armazenamento predefinida para o conteúdo de um website alojado em DiskStation. É criada automaticamente quando a Web Station for activada.
photo A pasta photo é utilizada para armazenar as fotografias e os vídeos que pretende partilhar com a Photo Station. É criada automaticamente quando a Photo Station ou Servidor Multimédia DLNA/UPnP for activado.
music A pasta music é utilizada para armazenar a música que pretende ouvir com a Audio Station. É criada automaticamente quando a Audio Station for activada.
video A pasta video é utilizada para armazenar os ficheiros de vídeo que pretende pesquisar através dos dispositivos DLNA/UPnP. É criada automaticamente quando o Servidor Multimédia for activado.
surveillance A pasta surveillance é utilizada para armazenar as gravações da Surveillance Station. É criada automaticamente quando a Surveillance Station for activada. É apenas de leitura e só pode ser acedida pelo administrador do sistema.
home As pastas home fornecem espaços privados para que cada utilizador armazene dados que só podem ser acedidos por esse utilizador. É criada automaticamente quando a funcionalidade Home do Utilizador for activada.
homes A pasta homes contém as pastas home de todos os utilizadores. É criada automaticamente quando a funcionalidade Home do Utilizador for activada. Apenas o administrador do sistema pode aceder e gerir as pastas home de todos os utilizadores.
NetBackup A pasta NetBackup é criada automaticamente quando a Cópia de Segurança de Rede for activada.
usbshare A pasta usbshare[número] é criada automaticamente quando liga um dispositivo de unidade externa USB à porta USB de DiskStation.*
satashare A pasta satashare é criada automaticamente quando liga um dispositivo de unidade externa eSATA à porta eSATA de DiskStation.*

____
* Apenas disponível em determinados modelos.
