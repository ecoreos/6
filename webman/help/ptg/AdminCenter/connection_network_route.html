Encaminhamento de rede

O DSM disponibiliza diversas opções de encaminhamento de rede para vários requisitos de configuração de rede.

Para separar o fluxo de rede das interfaces de rede na mesma LAN:

  1. Aceda a Painel de Controlo > Rede > Avançadas.
  2. Em Geral, assinale Responder a pedidos ARP se o endereço IP alvo for o endereço local configurado na interface de entrada.
  3. Clique em Aplicar para guardar as suas definições.

Desta forma fica estabelecida a regra de encaminhamento predefinida para cada interface de rede, o que irá provocar que os pacotes de rede de cada interface sejam transmitidos da sua própria interface de rede física correspondente.

Além disso, se tiver mais do que duas interfaces de rede na mesma LAN, a ativação desta opção dá origem a que o fluxo de rede de cada interface seja transportado separadamente. Pode, então, aplicar regras de firewall separadas para cada interface.

Para encaminhar cada interface de rede através das respetivas gateways:

  1. Aceda a Painel de Controlo > Rede.
  2. Em Geral, assinale Ativar múltiplas gateways.
  3. Clique em Aplicar para guardar as suas definições.

Isto origina que o encaminhamento de cada interface de rede na sua rede passe através das respetivas gateways.

Para alterar a gateway predefinida:

  1. Aceda a Painel de Controlo > Rede.
  2. Em Interface de Rede, clique em Ordem de Serviço.
  3. Reorganize as suas gateways selecionando a gateway e clicando em Para Cima ou Para Baixo.
  4. Clique em OK.

A gateway no topo da ordem irá funcionar como a gateway predefinida para a sua LAN.

Para configurar o encaminhamento estático:

  1. Aceda a Painel de Controlo > Rede.
  2. Em Rota Estática, clique em Criar.
  3. Em Rede de Destino, introduza o endereço IP do seu destino.
  4. Em Máscara de Rede, introduza a máscara de rede do seu destino.
  5. Em Gateway, introduza o endereço da sua gateway de destino.
  6. Em Interface, selecione a sua interface de destino
  7. Clique em OK.

Para visualizar o quadro de encaminhamento de IP:

  1. Aceda a Painel de Controlo > Rede.
  2. Em Rota Estática, clique em Tabela de Encaminhamento IP.
  3. Selecione uma interface de rede para visualizar a respetiva Tabela de encaminhamento IP na lista pendente no canto superior esquerdo.
