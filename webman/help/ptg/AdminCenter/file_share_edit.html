Editar Informações Gerais da Pasta Partilhada

Pode editar as informações gerais de uma pasta partilhada, nomeadamente o nome, a descrição, etc.

Para editar as informações gerais de uma pasta partilhada:

  1. Vá para Painel de Controlo > Pasta Partilhada.
  2. Seleccione uma pasta partilhada.
  3. Clique em Editar.
