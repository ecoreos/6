Serviços na Internet

O DiskStation é fornecido com funcionalidades de alojamento na Internet apelativas. Com a Web Station pode alojar e publicar facilmente o próprio website com suporte VHost e definições HTTP/HTTPS adicionais. Em alternativa, pode implementar o suporte PHP incorporado e criar websites dinâmicos e orientados para base de dados para o seu negócio ou utilização pessoal.
