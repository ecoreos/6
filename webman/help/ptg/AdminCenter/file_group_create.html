Criar um grupo

  1. Aceda a Painel de Controlo > Grupo.
  2. Clique em Criar. Esta acção inicia o Assistente de Criação de Utilizador.
  3. Introduza o nome e a descrição do grupo. (Consulte a secção em baixo para obter mais informações.) Clique em Seguinte.
  4. Siga os restantes passos indicados pelo assistente para concluir a criação do grupo.
  5. Clique em Aplicar para concluir.

Convenções de nomes:

  • O nome do grupo é sensível a maiúsculas e deve conter entre 1 e 32 caracteres Unicode, excluindo os seguintes símbolos:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • O primeiro caractere não pode ser um sinal de subtracção ou um espaço e o último caractere não pode ser um espaço.
  • A descrição do grupo é sensível a maiúsculas e limitada até 64 caracteres Unicode exibíveis.
