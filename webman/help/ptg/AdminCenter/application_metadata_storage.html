Base de dados de multimédia

O DiskStation utiliza uma base de dados de multimédia para acompanhar os ficheiros multimédia e entradas de registo guardados no dispositivo de origem de multimédia (pastas partilhadas nos seus dispositivos de armazenamento externo, como usbshare1).

Importante: Se pretende serviços de multimédia do DSM (como o Servidor multimédia, iTunes, Audio Station e Download Station) para aceder aos ficheiros de multimédia guardados nos dispositivos de armazenamento externos ligados na porta USB do seu DiskStation ou se pretende activar os serviços de registo do DSM e consultar os registos no Menu Principal > Informações do sistema > Registo, a base de dados de multimédia deve ser activada em primeiro lugar. Caso contrário, todos os serviços relacionados serão desactivados.

Para activar a base de dados de multimédia:

  1. Assinale Activar a base de dados de multimédia.
  2. Proceda de uma das seguintes formas:
    • Escolha Configuração automática se pretender que o DSM seleccione automaticamente uma pasta partilhada para armazenamento da base de dados de multimédia. Se o dispositivo de armazenamento externo que contenha a pasta partilhada for removido, o DSM irá seleccionar automaticamente qualquer outra pasta partilhada para armazenamento da base de dados.
    • Escolha Seleccione uma pasta partilhada para armazenar a sua base de dados e, de seguida, escolha no menu pendente a opção para seleccionar manualmente uma pasta partilhada para armazenar a base de dados.
  3. Clique em OK.

Aviso: Para impedir que a base de dados seja corrompida, garanta a ejecção do respectivo dispositivo de armazenamento em Menu Principal > Painel de Controlo > Dispositivos externos, antes de o remover do DiskStation.

Para incluir ou excluir pastas partilhadas da origem de multimédia:

  1. Clique em Origem de Multimédia.
  2. Assinale ou retire a selecção das caixas de verificação próximas da(s) pasta(s) partilhada(s) para efectuar a sua selecção.
  3. Clique em OK.

Os ficheiros de multimédia e os registos nas pastas partilhadas seleccionadas serão indexados e a informação indexada será adicionada à base de dados de multimédia.

Para actualizar a base de dados de multimédia:

  1. Clique em Actualizar Base de dados. Todos os dados na base de dados de multimédia serão substituídos pela informação reindexada acerca da origem de multimédia actual.
  2. Clique em OK.

Nota: Se uma grande parte dos dados na base de dados de multimédia já não for utilizada (por exemplo, se desligar um Disco USB que não voltará a ser utilizado novamente com o DiskStation, resultando numa informação indexada redundante na base de dados), a actualização da base de dados de multimédia pode ajudar a eliminar todos os dados não utilizados ou a reduzir o respectivo tamanho. No entanto, esta actualização requer leitura e escrita repetitiva no disco, o que pode consumir recursos de sistema e resultar numa redução da vida útil do disco. Utilize esta função com ponderação.
