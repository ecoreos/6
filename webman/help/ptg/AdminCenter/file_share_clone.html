Clonar uma Pasta Partilhada*

Pode clonar uma pasta partilhada em volumes Btrfs para criar cópias quase instantâneas de uma pasta partilhada existente. Clonar uma pasta partilhada consome apenas uma fração do armazenamento aquando da criação.

Para clonar uma pasta partilhada:

  1. Aceda a Painel de Controlo > Pasta Partilhada.
  2. Selecione a pasta partilhada desejada no sistema de ficheiros Btrfs.
  3. Clique em Criar > Clonar.
  4. Especifique um nome e descrição para a nova pasta partilhada, e modifique outras opções, se necessário. (Para informações detalhadas acerca destas opções, consulte Criar uma Pasta Partilhada.)

Nota:

  • Apenas pastas partilhadas em volumes Btrfs podem ser clonadas.
  • A pasta partilhada clonada será criada no mesmo volume que a pasta partilhada original.
  • As definições de proteção avançada da integridade de dados serão aplicadas à pasta partilhada clonada, e não podem ser modificadas.
