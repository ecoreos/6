Activar Reciclagem

Quando activar a Reciclagem numa pasta partilhada, será criada uma sub-pasta para armazenar ficheiros eliminados. Pode recolher os ficheiros eliminados da Reciclagem. Esvazie a Reciclagem se pretender remover permanentemente os ficheiros.

Para activar a Reciclagem:

  1. Seleccione uma pasta partilhada e depois clique em Editar.
  2. Marque a caixa de verificação Activar Reciclagem.
  3. Marque Restringir acesso apenas a administradores se desejar tornar a Reciclagem acessível só a administradores.
  4. Clique em OK para guardar as definições.

Para esvaziar a Reciclagem:

Clique no botão Esvaziar Reciclagem para remover todos os ficheiros eliminados da reciclagem permanentemente.

Para esvaziar todas as Reciclagens:

Seleccione Esvaziar todas as Reciclagens no menu pendente Acção para remover ficheiros eliminados permanentemente de todas as reciclagens.

Nota: Pode configurar as definições de privilégios da pasta Reciclagem em File Station clicando com o botão direito na pasta #recycle. Seleccione Propriedades e clique no separador Permissão.
