QuickConnect

O QuickConnect permite que os aplicativos clientes se conectem ao seu DiskStation através da Internet sem o inconveniente da configuração das regras do encaminhamento de portas.

Nota:

O QuickConnect pode funcionar com o DSM e alguns pacotes desenvolvidos pela Synology, como Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Cloud Station e aplicativos móveis.

Para ativar o QuickConnect:

  1. Vá para Painel de controle (modo simples) > QuickConnect.
  2. Marque a caixa Ativar QuickConnect.
  3. Digite as informações de sua conta Synology existente ou inscreva-se em uma nova conta.
  4. Especifique um novo QuickConnect ID. Certifique-se de que ele é fácil de lembrar, assim você será capaz de acessar seu DiskStation a qualquer momento.
  5. Clique em Aplicar.

Nota:

  • Antes de ativar o QuickConnect para os aplicativos clientes, verifique se os serviços DSM necessários já estão ativados. Por exemplo, o WebDAV deve estar ativado no DiskStation para que seja possível ativar o QuickConnect para o DS file.
  • Um QuickConnect ID personalizado pode incluir letras, números e hífenes ("-") e deve começar com uma letra.
  • Para obter um desempenho melhor do QuickConnect, recomendamos que você vá para Painel de controle (modo avançado) > Acesso externo > Configuração do roteador para configurar o encaminhamento de porta para cada serviço:
    • DSM: 5000 (HTTP); 5001 (HTTPS)
    • Photo Station: 80 (HTTP); 443 (HTTPS)
    • Cloud Station：6690

Solução de problemas

Se a mensagem Ocorreu um erro de rede for exibida, significa que o serviço QuickConnect foi interrompido devido a erros de rede. Verifique o seguinte:

  • Seu DiskStation está conectado a uma rede ativa (em Painel de controle (modo avançado) > Rede > Interface de rede).
  • As configurações de servidor DNS e gateway padrão estão definidas corretamente (em Painel de controle (modo avançado) > Rede > Geral).
  • Certifique-se de que o DiskStation possa acessar a Internet usando as seguintes portas de destino: 22, 80, 443, 8888. 

Quando os erros da rede forem reparados, o serviço QuickConnect deve retomar automaticamente em alguns minutos.
