USBCopy

Pode seleccionar e configurar uma localização alvo para a função USBCopy.

Para activar a USBCopy:

  1. Assinale a caixa de verificação Activar USBCopy.
  2. Seleccione uma pasta partilhada de destino no menu pendente.
  3. Clique em OK.

Quando liga um dispositivo USB (por exemplo, uma câmara digital) à porta USB no painel frontal do Synology DiskStation, pode premir o botão (Copy) Copiar no painel frontal para copiar os dados do dispositivo USB para a pasta partilhada de destino atribuída aqui. Se a pasta de destino USBCopy for apagada, o sistema irá substituí-la automaticamente pela pasta partilhada pública.

    Limitações:
  1. Apenas são suportados dispositivos USB no formato FAT ou NTFS.
  2. Os dispositivos USB devem ser ligados à porta USB no painel frontal do Synology DiskStation.
  3. O espaço disponível na pasta partilhada de destino deve ser superior ao espaço utilizado do dispositivo USB.
