Agendamento de Tarefas

O Agendamento de Tarefas permite-lhe agendar e executar serviços ou scripts a horas predefinidas.

Tarefas suportadas

As tarefas podem ser criadas para fazer o seguinte:

  • Script definido pelo utilizador: Execute scripts ou comandos definidos pelo utilizador.
  • Reciclagem: Esvazie a Reciclagem de pastas partilhadas específicas. Além disso, as definições de Política de Retenção permitem controlar quais os ficheiros que são eliminados de forma permanente quando esvazia a Reciclagem. Por exemplo, os ficheiros podem ser eliminados ou mantidos de acordo com o tamanho, a hora da eliminação ou o formato do ficheiro.
  • Controlo de Sinal Sonoro: DiskStation emite um sinal sonoro.
  • Serviço: Inicia ou pára os serviços de rede, tais como, Serviço de Ficheiros Windows, FTP, Bloqueio Automático, etc.
  • Script definido pelo utilizador: Execute scripts ou comandos definidos pelo utilizador.

Gerir Tarefas

A página principal do Agendamento de Tarefas apresenta todas as tarefas existentes e opções para criar, editar ou executar tarefas.

Para criar uma tarefa agendada:

Tarefas agendadas são tarefas que são executadas numa data, hora ou intervalo específicos.

  1. Aceda ao Painel de Controlo > Agendamento de Tarefas, clique em Criar, e selecione Tarefa agendada.
  2. Selecione o tipo desejado das tarefas a agendar: Script definido pelo utilizador, Reciclagem, Controlo de Aviso Sonoro, ou Serviço.
  3. Quando é exibida a janela Criar tarefa, clique em Geral e modifique as seguintes definições:
    • Definições Gerais: Introduza um nome de tarefa, selecione um utilizador para quem será executada a tarefa, e ative ou desative a tarefa.
  4. Clique em Agendamento, e modifique as seguintes definições:
    • Data: Especifique quando executar a tarefa. As tarefas podem ser agendadas para execução em determinados dias da semana (por exemplo, diariamente, aos fins-de-semana ou a cada Domingo, etc.) ou em datas específicas (por exemplo, 21/10/2015).
    • Vezes: Especifique quantas vezes a tarefa é executada cada dia, bem como a primeira e última execução.
  5. Clique em Definições de Tarefa para ordenar definições específicas de tarefas (apenas para certas tarefas).
    • Para receber detalhes de execução da tarefa, selecione Enviar detalhes de execução por email.
    • Para ser apenas notificado quando ocorrerem anormalidades, selecione Enviar detalhes de execução apenas quando o script terminar de forma anormal.
    • Para personalizar o conteúdo da tarefa, entre no script em Script definido pelo utilizador.
    • Se tiver selecionado Reciclagem ou Serviço no Passo 2, pode decidir como e quando a Reciclagem é limpa, ou que serviço iniciar/interromper.
  6. Clique em OK para guardar as definições.

Para criar uma tarefa acionada:

Tarefas acionadas são tarefas que são executadas antes/depois da conclusão de eventos específicos.

  1. Aceda ao Painel de Controlo > Agendamento de Tarefas, clique em Criar, e selecione Tarefa Acionada.
  2. Selecione Script definido pelo utilizador.
  3. Quando é exibida a janela Criar tarefa, clique em Geral e modifique as seguintes definições:
    • Definições Gerais: Introduza um nome de tarefa, selecione um utilizador para quem será executada a tarefa, selecione um evento/pré-tarefa que aciona a tarefa, e ative ou desative a tarefa.
  4. Clique em Definições de Tarefa para ordenar definições específicas de tarefas (apenas para certas tarefas).
    • Para receber detalhes de execução da tarefa, selecione Enviar detalhes de execução por email.
    • Para ser apenas notificado quando ocorrerem anormalidades, selecione Enviar detalhes de execução apenas quando o script terminar de forma anormal.
    • Para personalizar o conteúdo da tarefa, entre no script em Script definido pelo utilizador.
  5. Clique em OK para guardar as definições.

Nota:

  • Quando criar uma tarefa acionada, poderá ter de especificar a(s) pré-tarefa(s). A tarefa criada apenas será executada depois de concluída a pré-tarefa.
  • Depois de ativar Enviar detalhes de execução apenas quando o script termina anormalmente, o sistema não irá apresentar detalhes se terminar com 0.

Para editar as definições de tarefa:

  1. Selecione uma tarefa existente.
  2. Clique em Editar.
  3. Aparece uma janela onde pode modificar as definições da tarefa.

Para eliminar uma tarefa:

  1. Selecione uma tarefa existente.
  2. Clique em Ação > Eliminar.

Para ativar ou desativar uma tarefa:

  1. Na página principal do Agendamento de Tarefas, assinale ou desmarque a caixa de verificação adequada em Ativado para ativar ou desativar uma tarefa.
  2. Clique em Guardar.

Para executar manualmente uma tarefa:

  1. Selecione uma ou mais tarefas existentes.
  2. Clique em Executar.

Limitações do Agendamento de Tarefas:

  • Os nomes de tarefas podem incluir 512 caracteres visíveis.
  • Número máximo de tarefas: 1024.
  • Comprimento máximo do caminho do script ou da linha de comandos: 4095.
  • A duração do sinal sonoro pode ser de 1 a 600 segundos.
