Restrições de ligação

Permitir apenas ligações SSL/TLS

Quando esta função estiver activada, o servidor apenas aceitará os pedidos de ligação SSL ou TLS de clientes de FTP.

Limitar ligações por IP

Quando esta função estiver activada, o servidor estabelecerá um limite máximo para as ligações por IP. Active esta função e a restrição à largura de banda do FTP, para uma gestão mais eficaz da largura de banda.

Activar restrição de largura de banda de FTP

Active esta função para restringir a largura de banda de uma ligação FTP única.
