Serviço de Diretório

O seu DiskStation pode aderir a um serviço directório existente como um Domínio Windows ou cliente LDAP. Pode gerir os privilégios de acesso dos utilizadores LDAP ou do domínio para partilhar pastas e aplicações DSM e activar o serviço home para eles. Aceda a Painel de controlo > Serviço de Directório para associar o seu DiskStation a um serviço de directório.
