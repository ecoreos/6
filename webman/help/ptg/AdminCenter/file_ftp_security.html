Definições de segurança

Altere o root de utilizador seleccionado para base do utilizador

Os utilizadores aqui seleccionados podem aceder às respectivas pastas home, apenas quando iniciarem a sessão no Synology DiskStation através do FTP.

Activar FTP Anónimo

Active esta função para permitir que todos iniciem sessão no serviço FTP da DiskStation (com a conta anonymous (anónimo) ou ftp).

Para configurar os privilégios do utilizador anónimo:

Aceda a Menu Principal > Painel de Controlo > Pasta Partilhada e clique em Configurar privilégios.

Alterar root Anónimo

Active esta função para alterar o directório de inicio de sessão FTP do utilizador anónimo. Seleccione a pasta partilhada no menu pendente.
