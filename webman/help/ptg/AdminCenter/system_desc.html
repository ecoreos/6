Sistema

Esta secção permite-lhe gerir as definições de sistema para o seu DiskStation. Pode personalizar o ecrã de início de sessão, mudar o fuso horário e as definições de idioma, agendar um arranque/encerramento automático, gerir dispositivos externos, atualizar a versão DSM ou realizar uma reposição para as definições de origem.
