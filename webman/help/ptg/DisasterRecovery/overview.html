Visão Geral

Esta página mostra o estado geral dos destinos de instantâneos e dos seus instantâneos.

Sumário

Esta secção mostra três estados listados sob os destinos de instantâneos (incluindo as pastas partilhadas e iSCSI LUNs). O tamanho total dos destinos de instantâneos nos estados respectivos são apresentados também nesta secção.

  • Instantâneo agendado: Agendado o tirar de instantâneo para destinos de instantâneos.
  • Não agendado: Foram tirados instantâneos para os destinos de instantâneos, mas não está agendado o tirar de instantâneo.
  • Não protegida: Nenhum instantâneo tirado para os destinos de instantâneos.

Instantâneos recentes

Esta secção apresenta uma cronologia de instantâneos categorizados para facilitar a recuperação dos seus dados.

Para restaurar/clonar as suas pastas partilhadas ou iSCSI LUNs de instantâneos recentes:

  1. Clique no ícone calendário, e seleccione a data para a qual deseja restaurar os dados.
  2. Arraste a cronologia para recuar ou avançar, e seleccione um ponto específico no tempo, no qual foram tirados instantâneos.
  3. Seleccione uma pasta partilhada ou iSCSI LUN que deseja restaurar/clonar.
  4. Será redireccionado para a página Recuperação. Para mais informações sobre os passos seguintes, consulte Recuperação.
