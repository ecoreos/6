Recuperação

Esta página permite aos administradores de sistema realizar uma recuperação. Para permitir que utilizadores acedam a instantâneos de pastas partilhadas, active Tornar visível instantâneo para cada pasta partilhada, para que os utilizadores possam aceder aos instantâneos através da File Station ou Explorador de Ficheiros do Windows.

Pasta Partilhada

Para restaurar ficheiros/pastas individuais numa pasta partilhada:

  1. Selecione a pasta partilhada que deseja restaurar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo da qual deseja restaurar.
  4. Clique em Percorrer, e copie os ficheiros/pastas que deseja restaurar para o destino.

Para restaurar uma pasta partilhada:

  1. Selecione a pasta partilhada que deseja restaurar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo da qual deseja restaurar.
  4. Clique em Acção, e seleccione Restaurar em substituição da lista de selecção.
  5. Verá uma mensagem de confirmação. Clique em Sim para confirmar a acção.

Para clonar um instantâneo para uma nova pasta partilhada:

  1. Selecione uma pasta partilhada que deseja clonar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo que deseja clonar para uma nova pasta partilhada.
  4. Clique em Acção, e seleccione Clonar com um novo nome da lista de selecção.
  5. Será redireccionado para a página Clonar no Painel de Controlo > Pasta Partilhada > Criar. Introduza o nome da pasta partilhada clonada e clique em OK para terminar.

Nota:

  • Não é possível restaurar os "homes" da pasta partilhada.
  • A pasta partilhada clonada não pode estar encriptada.
  • Se for tirado um instantâneo quando está a decorrer uma transferência de quaisquer ficheiros, não pode ser garantida a consistência dos dados.
  • Limitações em pastas partilhadas encriptadas:
    • Nenhum tipo de recuperação está disponível para pastas partilhadas montadas e encriptadas.
    • Clonar com um novo nome não está disponível para pastas partilhadas encriptadas.
    • Pastas partilhadas encriptadas e desmontadas podem ser restauradas a partir de instantâneos tirados anteriormente, usando Restaurar em substituição.

Para parar realizar um switchover da replicação de pasta partilhada:

  1. Selecione a pasta partilhada que deseja realizar o switchover.
  2. Clique em Mudança.
  3. Selecione o novo servidor de destino caso existam múltiplas tarefas de replicação da pasta partilhada.
  4. Clique em Switchover para iniciar a ação.

Para realizar um failover de replicação de uma pasta partilhada:

  1. Selecione a pasta partilhada replicada sobre a qual deseja realizar o switchover.
  2. Clique em Failover > Failover.
  3. Selecione a versão do instantâneo a restaurar.
  4. Clique em Failover para iniciar a ação.

Nota:

  • Pode apenas iniciar um failover se houver uma falha no servidor de origem.

Para parar proteger novamente a replicação de uma pasta partilhada:

Quando se inicia a transição por falha para o servidor de destino, a tarefa de replicação irá parar de sincronizar os dados entre os dois servidores. Terá de proteger novamente a tarefa de replicação depois de recuperado o servidor original.

  1. Selecione a pasta partilhada transitada por falha que deseja proteger novamente.
  2. Clique em Failover > Proteger novamente.
  3. Selecione o novo servidor de origem.
  4. Selecione o novo servidor de destino caso existam múltiplas tarefas de replicação da pasta partilhada.
  5. Clique em Proteger novamente para iniciar a ação.

Nota:

  • Os dados da pasta partilhada replicada que existirem no novo servidor de destino serão substituídos pelos dados do novo servidor de origem.
  • Se a versão selecionada de instantâneo para failover não existir no servidor parceiro, poderá demorar mais a proteger novamente.

Para realizar um failover de teste para uma pasta partilhada replicada:

  1. Selecione a pasta partilhada replicada que deseja testar.
  2. Clique em Failover de Teste > Failover de Teste.
  3. Selecione o Servidor de Teste caso existam múltiplas tarefas de replicação da pasta partilhada.
  4. Introduza no Nome para Failover de Teste da pasta partilhada, e selecione a versão do instantâneo a testar.
  5. Clique em Failover de Teste para iniciar a ação.

Nota:

  • Realizar um switchover/failover de uma replicação irá limpar o failover de teste.
  • Não pode criar uma tarefa de replicação para um failover de teste de pasta partilhada.

Para limpar um failover de teste para uma pasta partilhada replicada:

  1. Selecione a pasta partilhada replicada da qual deseja limpar o failover de teste.
  2. Clique em Failover de Teste > Limpar Failover de Teste.
  3. Assinale o failover de teste que deseja limpar.
    Nota: Assinale Eliminar dados de failover de teste caso não queira mais manter a pasta partilhada criada para o failover de teste.
  4. Clique em Limpar Failover de Teste para iniciar a ação.

iSCSI LUN

Para restaurar um iSCSI LUN:

  1. Selecione o iSCSI LUN que pretende restaurar.
  2. Clique em Recuperação.
  3. Selecione a versão de instantâneo da qual deseja restaurar.
  4. Clique em Acção, e seleccione Restaurar em substituição da lista de selecção.
  5. Verá uma mensagem de confirmação. Clique em Sim para confirmar a acção.

Para clonar um instantâneo para um novo iSCSI LUN:

  1. Selecione o iSCSI LUN que pretende clonar.
  2. Clique em Recuperação.
  3. Selecione a versão de instantâneo que deseja clonar para um novo iSCSI LUN.
  4. Clique em Acção, e seleccione Clonar com um novo nome da lista de selecção.
  5. Será redireccionado para a página Clonar no Gestor de Armazenamento > iSCSI LUN. Introduza o nome do iSCSI LUN clonado e clique em OK para terminar.

Nota:

  • Se for tirado um instantâneo quando está a decorrer uma transferência de quaisquer ficheiros, não pode ser garantida a consistência dos dados.
