Instantâneos

Esta página permite-lhe tirar, remover e editar instantâneos, bem como configurar o agendamento e definições de retenção de instantâneos

Pode armazenar até 256 cópias de instantâneos de uma só vez para cada pasta partilhada ou iSCSI LUN. Instantâneos minimizam o consumo do disco, preservando blocos individuais em vez de ficheiros inteiros. Pode ser usado espaço extra quando os ficheiros em pastas partilhadas ou blocos em iSCSI LUNs são actualizados ou removidos.

Pasta Partilhada

Para tirar um instantâneo para uma pasta partilhada:

  1. Seleccione a pasta partilhada da qual deseja tirar um instantâneo.
  2. Clique em Tirar um Instantâneo.
  3. Introduza a descrição na Descrição para que o instantâneo seja tirado.
  4. Assinale/desmarque a caixa de verificação Bloquear para determinar se o instantâneo irá ignorar as definições de retenção quando a remoção automática é activada.
  5. Clique em OK para começar a tirar um instantâneo.

Nota:

  • De entre as definições da página Painel de Controlo > Pasta Partilhada > Editar, apenas a permissão Windows ACL será gravada num instantâneo.

Para percorrer os instantâneos de pastas partilhadas:

  1. Seleccione uma pasta partilhada na qual deseja percorrer os instantâneos.
  2. Clique em Lista de instantâneos.
  3. Seleccione um instantâneo que percorrer e clique em Percorrer.

Nota:

  • Percorrer instantâneo não está disponível para pastas partilhadas encriptadas.
  • Se não conseguir percorrer o instantâneo, aceda ao Painel de Controlo > Pasta Partilhada > Editar para activar Tornar visível instantâneo.

Para remover os instantâneos de pastas partilhadas:

  1. Seleccione uma pasta partilhada da qual deseja remover os instantâneos.
  2. Clique em Lista de instantâneos.
  3. Seleccione um ou mais instantâneos que deseja remover, e clique em Remover. (Mantenha premida a tecla Shift ou Ctrl enquanto efetua seleções múltiplas.)
  4. Verá uma mensagem de confirmação. Clique em Eliminar para confirmar a remoção.

Para editar os instantâneos de pastas partilhadas:

  1. Seleccione uma pasta partilhada da qual deseja editar os instantâneos.
  2. Clique em Lista de instantâneos.
  3. Seleccione um instantâneo que deseja editar e clique em Editar.
  4. Pode modificar a descrição do instantâneo na Descrição, e assinalar/desmarcar a caixa de verificação Bloquear para determinar se o instantâneo irá ignorar as definições de retenção quando a remoção automática for para ser activada.
  5. Clique em OK para guardar as suas definições.

Para configurar instantâneos agendados:

Com instantâneos agendados, Instantâneo e Replicação irá tirar automaticamente instantâneos no momento e com a frequência predeterminadas, poupando-lhe o trabalho de tirar manualmente o instantâneo.

  1. Seleccione uma pasta partilhada.
  2. Clique em Definir agendamento.
  3. No separador Agendamento, assinale a caixa de verificação Activar agendamento de instantâneo, e configure o momento e frequência para tirar instantâneos. Para cancelar instantâneos agendados para esta pasta partilhada, desmarque a caixa de verificação.
  4. Clique em OK para guardar as suas definições.
  5. Verá uma mensagem de confirmação. Clique em OK para confirmar a configuração.

Nota:

  • A configuração de instantâneos agendados não será incluída na funcionalidade Painel de Controlo > Actualizar e Restaurar > Cópia de Segurança de Configuração.

Para configurar as definições de retenção para os instantâneos agendados:

  1. Seleccione uma pasta partilhada.
  2. Clique em Definir agendamento.
  3. Seleccione o separador Retenção.
  4. Configure as suas definições de retenção. Escolha uma das seguintes opções:
    • Remover instantâneos antigos assim que o limite seja atingido: Quando é atingido o número máximo de instantâneos, os instantâneos mais antigos serão substituídos por outros novos.
    • Reter sempre instantâneos: Quando é atingido o número máximo de instantâneos, o Instantâneo e Replicação irá parar de tirar novos instantâneos sem remover os que já foram tirados.
    • Manter instantâneos com a seguinte política: Pode configurar as regras de retenção para blocos de tempo especificados. Para mais informações, consulte a nota relativa à Política de Retenção.
  5. Clique em OK para guardar as suas definições.

Nota:

  • Alterar as definições de retenção irá afetar as definições equivalentes na página Replicação.

iSCSI LUN

Para tirar um instantâneo de um iSCSI LUN:

  1. Seleccione o iSCSI LUN do qual deseja tirar um instantâneo.
  2. Clique em Tirar um Instantâneo.
  3. Introduza a descrição na Descrição para que o instantâneo seja tirado.
  4. A partir da lista de selecção Bloquear, seleccione Sim/Não para determinar se o instantâneo irá ignorar as definições de retenção quando for activada a remoção automática.
  5. Clique em OK para começar a tirar um instantâneo.

Para remover um instantâneo de um iSCSI LUN:

  1. Seleccione um iSCSI LUN do qual deseja remover os instantâneos.
  2. Clique em Lista de instantâneos.
  3. Seleccione um ou mais instantâneos que deseja remover, e clique em Remover. (Mantenha premida a tecla Shift ou Ctrl enquanto efetua seleções múltiplas.)
  4. Verá uma mensagem de confirmação. Clique em Sim para confirmar a remoção.

Para editar um instantâneo de um iSCSI LUN:

  1. Seleccione um iSCSI LUN do qual deseja editar os instantâneos.
  2. Clique em Lista de instantâneos.
  3. Seleccione um instantâneo que deseja editar e clique em Editar.
  4. Pode fazer qualquer um dos seguintes para editar o instantâneo:
    • Altere a descrição do instantâneo na Descrição.
    • Seleccione o Estado consistente caso esteja activado o instantâneo consistente com aplicação.
    • Alterne entre Sim/Não a partir da lista de selecção Bloquear para determinar se o instantâneo irá ignorar as definições de retenção quando for activada a remoção automática.
  5. Clique em OK para guardar as suas definições.

Para configurar instantâneos agendados:

Com instantâneos agendados, Instantâneo e Replicação irá tirar automaticamente instantâneos no momento e com a frequência predeterminadas, poupando-lhe o trabalho de tirar manualmente o instantâneo.

  1. Seleccione um iSCSI LUN.
  2. Clique em Definir agendamento.
  3. No separador Agendamento, assinale a caixa de verificação Activar agendamento de instantâneo, e configure o momento e frequência para tirar instantâneos. Para cancelar instantâneos agendados para esta pasta partilhada, desmarque a caixa de verificação.
  4. Clique em OK para guardar as suas definições.

Para configurar as definições de retenção para os instantâneos agendados:

  1. Seleccione um iSCSI LUN.
  2. Clique em Definir agendamento.
  3. Seleccione o separador Retenção.
  4. Configure as suas definições de retenção. Escolha uma das seguintes opções:
    • Remover instantâneos antigos assim que o limite seja atingido: Quando é atingido o número máximo de instantâneos, os instantâneos mais antigos serão substituídos por outros novos.
    • Reter sempre instantâneos: Quando é atingido o número máximo de instantâneos, o Instantâneo e Replicação irá parar de tirar novos instantâneos sem remover os que já foram tirados.
    • Manter instantâneos com a seguinte política: Pode configurar as regras de retenção para blocos de tempo especificados. Para mais informações, consulte a nota relativa à Política de Retenção.
  5. Clique em OK para guardar as suas definições.
  6. Verá uma mensagem de confirmação. Clique novamente em OK para confirmar a configuração.

Para configurar o estado consistente dos instantâneos iSCSI LUN:

  1. Seleccione um iSCSI LUN.
  2. Clique em Definir agendamento.
  3. Seleccione o separador Aplicação.
  4. Assinale/desmarque a caixa de verificação Activar instantâneos consistentes com aplicação para alternar entre os seguintes estados consistentes:
    • Consistente com a Aplicação Instantâneos neste estado consistente só podem ser obtidos com o Synology Snapshot Manager instalado no seu servidor Windows ou VMware vCenter Server, dependendo do seu ambiente operativo para comunicar com o DSM. Quando são acionados instantâneos consistentes com a aplicação no DSM, são notificadas as aplicações e todos os dados da memória são enviados para o LUN do qual o instantâneo é obtido, para garantir a consistência dos dados.

      Nota:

      • Se o Synology Snapshot Manager não tiver sido instalado no seu ambiente operativo, o instantâneo consistente com a aplicação permanecerá desactivado mesmo que a caixa de verificação esteja assinalada.
    • Consistente com Avaria Se o Synology Snapshot Manager não foi instalado em qualquer dos seus clientes, pode apenas capturar instantâneos consistentes com avarias de LUNs. Quando são accionados instantâneos consistentes com avarias, as aplicações não serão notificadas para enviar os dados da memória para o LUN e, dessa forma, não é garantida a consistência dos dados.
  5. Clique em OK para guardar as suas definições.

Nota:

  • Alterar as definições do estado consistente irá afetar as definições equivalentes na página Replicação.

Política de Retenção

Esta opção proporciona uma solução ideal para o ajudar a especificar os montantes máximos de versões de instantâneos para poupar espaço no seu volume, mas poderá ter de reter os seus instantâneos durante períodos maiores de tempo.

A Synology emprega a política de retenção GFS, ou Grandfather-Father-Son Pode configurar os montantes máximos de versões de instantâneos a ser retidos para os seguintes blocos de tempo, respectivamente: a cada hora, diário, semanal, mensal, e anual.

Instantâneo e Replicação irá reter o número máximo de intantâneos configurados para cada bloco de tempo. Se mais que uma versão de instantâneo existir dentro de um bloco de tempo, apenas será mantido o mais recente. Por exemplo, se configurar uma política de 10 instantâneos semanais, o Instantâneo e Replicação irá reter o instantâneo mais recente (caso seja tirado mais que um instantâneo numa semana) para cada uma das mais recentes 10 semanas.

Mais ainda, todas as versões de instantâneos com menos de uma hora relativamente ao instantâneo mais recente serão mantidas por predefinição, para ser mais fácil encontrar e restaurar de versões recentes de instantâneos. Pode também bloquear um instantâneo para evitar a sua remoção automática segundo as suas regras de retenção.
