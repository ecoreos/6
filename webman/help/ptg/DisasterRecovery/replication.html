Replicação

Esta página permite aos administradores gerir replicações. Pode criar, editar e eliminar tarefas de replicação. Pode também executar/interromper a sincronização e visualizar as informações das suas tarefas de replicação.

Pasta Partilhada

Para criar uma tarefa de replicação para uma pasta partilhada:

  1. Aceda ao separador Pasta Partilhada.
  2. Clique em Criar.
  3. Verá o assistente Criar Tarefa de Replicação. Siga os passos do assistente para criar uma replicação de pasta partilhada.

Nota:

  • Não é possível replicar os "homes" da pasta partilhada.
  • Não pode replicar uma pasta partilhada encriptada.
  • A pasta partilhada tem de estar localizada num volume Btrfs.
  • Pode criar replicações um-para-muitas/expandidas para uma pasta partilhada em diferentes servidores de destino. No entanto, pode apenas criar uma replicação para um servidor de destino para cada pasta partilhada.

Para eliminar uma replicação de pasta partilhada:

  1. Selecione a tarefa de replicação de pasta partilhada que deseja eliminar.
  2. De seguida, clique em Eliminar.
  3. Verá uma mensagem de confirmação. Clique em Eliminar para confirmar a eliminação.

Para editar uma replicação de pasta partilhada:

  1. Selecione a tarefa de replicação de pasta partilhada que deseja editar.
  2. Clique em Editar.
  3. Verá uma janela com os seguintes separadores:
    • Servidor Parceiro
    • Agendamento
    • Retenção
  4. No separador Servidor Parceiro, pode editar as informações do servidor parceiro incluindo o nome do anfitrião, porta, nome de utilizador e palavra-passe.
  5. No separador Agendamento, assinale a caixa de verificação Activar agendamento de instantâneo, e configure o momento e frequência para tirar instantâneos. Pode desmarcar a caixa de verificação para cancelar instantâneos agendados para a pasta partilhada.
  6. No separador Retenção, escolha qualquer das seguintes opções:
    • Manter os 256 mais recentes instantâneos: Quando é atingido o número máximo (256) de instantâneos, os instantâneos mais antigos serão substituídos por outros novos.
    • Reter sempre instantâneos: Quando é atingido o número máximo de instantâneos, o Instantâneo e Replicação irá parar de tirar novos instantâneos sem remover os que já foram tirados.
    • Manter instantâneos com a seguinte política: Pode configurar as regras de retenção para blocos de tempo especificados.
  7. Clique em OK para aplicar as novas definições.

Nota:

  • Pode apenas editar o agendamento da tarefa de replicação no servidor de origem.
  • Alterar as definições de retenção irá afetar as definições equivalentes na página Instantâneo.

Para ver as informações de uma replicação de pasta partilhada:

  1. Selecione a tarefa de replicação de pasta partilhada que deseja visualizar.
  2. Clique em Informação.
  3. Verá uma janela com as seguintes informações:
    • Estado desta replicação.
    • Estatísticas desta replicação.

Para realizar a sincronização de uma replicação de pasta partilhada:

  1. Selecione a replicação de pasta partilhada que deseja sincronizar.
  2. Clique em Executar.
  3. Introduza a descrição do instantâneo a tirar.
  4. Clique em OK para confirmar a ação.

Para parar a sincronização de uma replicação de pasta partilhada:

  1. Selecione a replicação de pasta partilhada sincronizada que deseja interromper.
  2. Clique em Parar.
  3. Verá uma mensagem de confirmação. Clique em Parar para confirmar a acção.

iSCSI LUN

Para tirar uma tarefa de replicação de um iSCSI LUN:

  1. Aceda ao separador iSCSI LUN.
  2. Clique em Criar.
  3. Verá o assistente Criar Tarefa de Replicação. Siga os passos do assistente para criar uma replicação iSCSI LUN.

Nota:

  • O iSCSI LUN tem de ser criado como um LUN avançado.
  • Pode criar replicações um-para-muitas/expandidas para um iSCSI LUN em diferentes servidores de destino. No entanto, pode apenas criar uma replicação para um servidor de destino para cada iSCSI LUN.

Para eliminar uma replicação iSCSI LUN:

  1. Selecione a tarefa de replicação iSCSI LUN que deseja eliminar.
  2. De seguida, clique em Eliminar.
  3. Verá uma mensagem de confirmação. Clique em Eliminar para confirmar a acção.

Para editar uma replicação iSCSI LUN:

  1. Selecione a tarefa de replicação iSCSI LUN que deseja editar.
  2. Clique em Editar.
  3. Verá uma janela com os seguintes separadores:
    • Servidor Parceiro
    • Agendamento
    • Retenção
  4. No separador Servidor Parceiro, pode editar as informações do servidor parceiro incluindo o nome do anfitrião, porta, nome de utilizador e palavra-passe.
  5. No separador Agendamento, assinale a caixa de verificação Activar agendamento de instantâneo, e configure o momento e frequência para tirar instantâneos. Pode desmarcar a caixa de verificação para cancelar instantâneos agendados para iSCSI LUN.
  6. No separador Retenção, escolha qualquer das seguintes opções:
    • Manter os 256 mais recentes instantâneos: Quando é atingido o número máximo (256) de instantâneos, os instantâneos mais antigos serão substituídos por outros novos.
    • Reter sempre instantâneos: Quando é atingido o número máximo de instantâneos, o Instantâneo e Replicação irá parar de tirar novos instantâneos sem remover os que já foram tirados.
    • Manter instantâneos com a seguinte política: Pode configurar as regras de retenção para blocos de tempo especificados.
  7. No separador Aplicação, assinale Ativar instantâneos consistentes com aplicações para ativar instantâneos consistentes com aplicações. Para cancelar instantâneos consistentes com aplicações, desmarque a caixa de verificação.
  8. Clique em OK para aplicar as novas definições.

Nota:

  • Pode apenas editar o agendamento na origem da replicação.
  • Alterar as definições de retenção/aplicação irá afetar as definições equivalentes na página Instantâneo.

Para ver as informações de uma replicação iSCSI LUN:

  1. Selecione a tarefa de replicação iSCSI LUN que deseja visualizar.
  2. Clique em Informação.
  3. Verá uma janela com as seguintes informações:
    • Estado desta replicação.
    • Estatísticas desta replicação.

Para executar a sincronização de uma replicação iSCSI LUN:

  1. Selecione a tarefa de replicação iSCSI LUN que deseja sincronizar.
  2. Clique em Executar.
  3. Introduza a descrição do instantâneo a tirar.
  4. Clique em OK para confirmar a ação.

Para interromper a execução de uma replicação iSCSI LUN:

  1. Selecione a tarefa de replicação de sincronização iSCSI LUN que deseja interromper.
  2. Clique em Parar.
  3. Verá uma mensagem de confirmação. Clique em Parar para confirmar a acção.

Um-para-muitos/Replicação expandida

Pode criar múltiplas replicações para uma pasta partilhada/iSCSI LUN para múltiplos servidores de destino, formando um esquema um-para-muitos. Pode também criar uma replicação para uma pasta partilhada replicada/iSCSI LUN, formando um esquema expandido.

Nota:

  • Quer seja um esquema um-para-muitos ou expandido, qualquer tarefa de replicação pode ter apenas três servidores de destino no máximo.

Política de Retenção

Esta opção proporciona uma solução ideal para o ajudar a especificar os montantes máximos de versões de instantâneos para poupar espaço no seu volume, mas poderá ter de reter os seus instantâneos durante períodos maiores de tempo.

A Synology emprega a política de retenção GFS, ou Grandfather-Father-Son Pode configurar os montantes máximos de versões de instantâneos a ser retidos para os seguintes blocos de tempo, respectivamente: a cada hora, diário, semanal, mensal, e anual.

O Gestor de Protecção de Dados irá reter o número máximo de intantâneos configurados para cada bloco de tempo. Se mais que uma versão de instantâneo existir dentro de um bloco de tempo, apenas será mantido o mais recente. Por exemplo, se configurar uma política de 10 instantâneos semanais, o Gestor de Protecção de Dados irá reter o instantâneo mais recente (caso seja tirado mais que um instantâneo numa semana) para cada uma das mais recentes 10 semanas.

Mais ainda, todas as versões de instantâneos com menos de uma hora relativamente ao instantâneo mais recente serão mantidas por predefinição, para ser mais fácil encontrar e restaurar de versões recentes de instantâneos. Pode também bloquear um instantâneo para evitar a sua remoção automática segundo as suas regras de retenção.
