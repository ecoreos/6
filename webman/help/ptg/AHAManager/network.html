Esta página contém o diagrama e tabela de estado das interfaces de rede.

Diagrama da Interface

Mostra o item seleccionado na tabela, e são usadas cores diferentes para apresentar interfaces de rede 1Gbps e 10Gbps.

Tabela de Estado da Interface

Mostra o estado da interface em ambos servidores, e se a interface está ou não monitorizada. Esta tabela mostra as seguintes informações:

  • Interface de Rede: Mostra as interfaces físicas de rede nos servidores.
  • Servidor Activo: Mostra o estado da interface e velocidade de ligação no servidor activo.
  • Servidor Passivo/Servidor Desconhecido: Mostra o estado da interface e velocidade de ligação no servidor passivo. Esta coluna irá mostrar o nome correspondente ao estado do servidor.
  • Bond: Mostra as definições da ligação de agregação.
  • Estado: Mostra o estado de rede se a interface estiver monitorizada. Poderá também mostrar os seguintes estados:
    • Aviso: A ligação falha no servidor passivo, ou a interface tem diferentes velocidades de ligação nos dois servidores;
    • Erro: A ligação falha no servidor activo;
    • Normal: Nenhuma falha.
  • Monitorizar: Mostra se a interface está monitorizada. Aceda a Monitorizar > Interface de Rede para assinalar/desmarcar a caixa de verificação Monitorizar.
