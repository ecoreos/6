Acessibilidade

Com a funcionalidade de acessibilidade integrada, DiskStation Manager proporciona pistas auditivas para ajudar utilizadores com dificuldades visuais a navegar e gerir os serviços/definições DSM.

Configuração de Acessibilidade DSM

Para maximizar a acessibilidade, sugerimos que configure o ambiente de funcionamento da seguinte forma:

  • Instale o leitor de ecrã NVDA (disponível apenas no ambiente Windows) no seu computador local.
  • Instale o navegador web Mozilla Firefox mais recente.
  • Use auscultadores ou altifalantes, e ajuste o volume para se ajustar às suas necessidades.
  • Use NVDA no modo de focagem (prima a tecla NVDA + Espaço para a comutação de modo). A tecla NVDA predefinida é Insert.
  • Navegue pelo DSM com as seguintes teclas de atalho:
    Tecla de atalho Descrição
    Separador Mude para outras secções da interface do utilizador
    Teclas de seta Mude para itens dentro da mesma secção de interface do utilizador
    Introduza Abrir as aplicações/janelas
    Esc Fechar aplicações/janelas

A funcionalidade de acessibilidade está agora operacional nas seguintes áreas do DSM:

  • Página de início de sessão DSM, ambiente de trabalho e barra de tarefas
  • Menu Principal
  • Painel de Controlo
  • File Station

Nota:

Para detalhes acerca do funcionamento NVDA, consulte o respetivo guia do utilizador.

Guias de atalhos de teclado DSM

Juntamente com a funcionalidade de acessibilidade, DSM proporciona um vasto conjunto de atalhos de teclado para implementar uma gestão eficiente e sem sobressaltos. Prima Shift + ? para localizar os atalhos:

  • Atalhos de teclado do ambiente de trabalho: Teclas de atalho para funcionamento com janelas e mais
  • Atalhos de teclado componente: Teclas de atalho para controlar componentes da interface do utilizador (por ex., listas, botões e campos de pesquisa)

Para suspender atalhos relativos ao ambiente de trabalho, cancele a opção Ativar atalhos de teclado do ambiente de trabalho.
