Análise de RAID*

A análise de RAID é uma funcionalidade de manutenção de dados que inspeciona volumes ou grupos de discos e corrige quaisquer inconsistências de dados detetadas. Esta função pode ser usada com volumes ou grupos de discos que implementem SHR (com três ou mais discos) RAID 5 ou RAID 6.

Recomendamos a realização de uma análise de RAID regular, para manter a consistência de dados e evitar perdas de dados críticos no caso de falha do disco rígido.

Para realizar a análise RAID:

  1. Abra o Gestor de Armazenamento.
  2. Aceda a Volume ou Grupo de Discos.
  3. Selecione o volume ou grupo de discos para realizar a análise de RAID.
  4. Clique em Gerir.
  5. Selecione Análise RAID.

Nota:

  • A análise de RAID apenas pode ser realizada quando o estado do volume ou grupo de discos for Normal.
  • Não desligue o sistema antes de terminar a análise RAID. Se o sistema for encerrado antes da conclusão, a análise RAID deverá ser reiniciada.
