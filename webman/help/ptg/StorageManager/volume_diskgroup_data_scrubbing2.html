Análise de Dados*

Análise de dados é uma funcionalidade de manutenção que inspeciona volumes ou grupos de discos. Existem dois tipos de análise de dados:

  • Análise de RAID: Esta função pode ser usada com volumes ou grupos de discos que implementem SHR (com três ou mais discos) RAID 5 ou RAID 6. Pode reparar inconsistências detetadas de dados.
  • Análise do sistema de ficheiros: Esta função pode ser usada com volumes no sistema de ficheiros Btrfs. Se for detetada alguma inconsistência de dados com o checksum, o caminho do ficheiro será registado no Centro de Registos. Recomendamos vivamente que use uma cópia de segurança para reparar os dados.

Recomendamos a realização de uma análise de dados regular, para garantir a consistência de dados e evitar perdas de dados no caso de falha do disco rígido.

Para realizar a análise de dados:

  1. Abra o Gestor de Armazenamento.
  2. Aceda a Volume ou Grupo de Discos.
  3. Seleccione o volume ou grupo de discos para realizar a análise de dados.
  4. Clique em Gerir.
  5. Seleccione Análise de dados.
  6. Selecione o tipo de análise de dados desejado. (O sistema irá ignorar automaticamente este passo se houver apenas um tipo disponível.)

Nota:

  • A análise de dados apenas pode ser realizada quando o estado do volume ou grupo de discos for Normal.
  • Se o sistema for encerrado antes da sua conclusão, terá de reiniciar manualmente a análise de dados do início.
