Criar um Grupo de Discos

Este artigo explica como criar um grupo de discos no seu DiskStation.

Antes de Começar:

  • Certifique-se de que não existem dados importantes nos discos rígidos. Todos os dados existentes serão eliminados ao criar um novo grupo de discos.
  • Aceda à página HDD/SSD e certifique-se de que o estado de cada um dos discos rígidos é Normal ou Não inicializado.

Para criar um grupo de discos:

  1. Abra o Gestor de Armazenamento.
  2. Aceda a Grupo de Discos e clique em Criar.
  3. Quando é exibido o assistente de criação do grupo de discos, escolha quais os discos rígidos que devem ser utilizados para criar o grupo de discos. Depois, siga os passos indicados pelo assistente para concluir.
