Avançado

Pode ver e alterar as configurações avançadas para o Consultor de segurança.

Base de referência de segurança

Pode escolher que regras de segurança o Consultor de Segurança verificará, selecionando uma Base de segurança. O número e tipos de regras irá variar para cada Base de segurança. Efetue a sua escolha consoante as suas necessidades. Está a usar o seu DiskStation para trabalho e negócios ou lazer e fins pessoais? Pode escolher entre as seguintes opções:

  • Para uso doméstico e pessoal: O Consultor de Segurança irá realizar verificações essenciais para proteger o seu DiskStation contra ataques cibernautas comuns.
  • Para trabalho e negócios: O Consultor de Segurança irá realizar verificações completas para cumprir os requisitos de segurança para organizações.
  • Personalizar: Selecione as regras de segurança que pretende que o Consultor de segurança verifique.

Agendar análise

Pode definir uma agenda de análise regular ao clicar em Ativar agenda de análise regular e escolher a hora a que pretende que o Consultor de segurança realize uma análise. O Consultor de segurança irá verificar as regras de acordo com a Base de referência de segurança que escolheu.
