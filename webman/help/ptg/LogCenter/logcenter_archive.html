Definições do arquivo

A página Definições de Arquivo fornece opções de configuração do modo e do local onde são arquivados os registos recebidos no Centro de Registos. Utilize as opções abaixo para especificar a localização e as regras de armazenamento para criar automaticamente arquivos de registos.

Destino de armazenamento para arquivos

O destino refere-se à pasta onde os registos recebidos no Centro de Registos são armazenados e arquivados. Clique em Selecionar localização para escolher a pasta.

Se pretender que os registos locais do DiskStation sejam arquivados no mesmo destino, assinale a caixa marcada Arquivar registos locais na localização de armazenamento especificada acima. Na maioria dos casos, deste modo, é possível obter mais espaço de armazenamento para os registos locais, permitindo-lhe manter registos do sistema mais completos.

Nota:

Os registos locais (registos de Sistema, Ligação e Transferência de ficheiros) são guardados numa partição de sistema com capacidade de armazenamento limitada. Ao ativar Arquivar registos locais na localização de armazenamento especificada acima, todos os registos serão arquivados no destino especificado, em conformidade com as Regras do arquivo. Para pesquisar registos arquivados, clique em Abrir arquivo de registo e selecione SYNOSYSLOGDB__LOCALARCH.DB.

Regras do arquivo

Os registos são arquivados automaticamente de acordo com os critérios definidos na secção Regras do arquivo. É possível arquivar os registos automaticamente quando a base de dados de registos atinge um tamanho especificado, os registos individuais atingem um número especificado ou os registos atingem uma duração especificada.

Formato de arquivo

Esta secção inclui as seguintes opções.

  • Arquivar registos como formato de texto para além do formato predefinido: Guarda os arquivos como ficheiros de texto simples para além dos ficheiros SQLite predefinidos.
  • Comprimir arquivos de registos: Guarda os arquivos de registos como ficheiros de formato ZIP comprimidos.
  • Arquivar registos separadamente de acordo com o dispositivo: Guarda arquivos separados para dispositivos cliente individuais que enviam registos para o DiskStation.
