Criar Pedidos de ficheiros

Pedidos de ficheiros são convites de carregamento de ficheiros enviados a utilizadores não DSM. Com ligações de pedidos de ficheiros, utilizadores não DSM podem carregar diretamente múltiplos ficheiros para File Station no seu DiskStation, sem precisar de deter privilégios correspondentes sore contas de utilizador e carregamento de ficheiros.

Para criar ligações de pedidos de ficheiros:

  1. Aceda a File Station e selecione uma pasta como destino de carregamento.
  2. Clique em Ação ou clique com o botão direito na pasta de destino, e selecione Criar pedido de ficheiro.
  3. Na janela que surge, pode encontrar/modificar as informações que se seguem:
    • Caminho de ficheiro: Listar o caminho do diretório de destino.
    • Ligação: Listar a ligação de pedido de ficheiro a partilhar com utilizadores de destino.
    • O seu nome: Introduza um nome para indicar a identidade de quem solicita. O valor predefinido é o seu nome de utilizador.
    • Mensagem: Personalizar a mensagem para utilizadores de destino.
    • Ativar protecção por palavra-passe: Selecione esta opção para que apenas utilizadores com a palavra-passe possam carregar ficheiros através da ligação criada. Especifique a palavra-passe na caixa abaixo.
    • Período de validade: Clique neste botão para personalizar a duração desta ligação de pedido de ficheiro:
      • Configurar hora de fim: Selecione esta opção para determinar pela data e hora quando a ligação para de funcionar para carregar ficheiros.
      • Configurar hora de início: Selecione esta opção para determinar pela data e hora quando a ligação começa a funcionar para carregar ficheiros.
      • Número de acesso permitido: Selecione esta opção para determinar quantas vezes pode ser acedida a ligação.
  4. Clique em Guardar para concluir as definições.

Para partilhar ligações de pedidos de ficheiros:

Para encontrar o URL de pedido de ficheiro, aceda File Station > Ferramentas > Gestor de Ligações Partilhadas e faça duplo clique na ligação desejada. Pode partilhar a ligação de uma das formas abaixo:

  • Basta copiar o URL para utilizadores de destino.
  • Envie a ligação através do Email Predefinido (a sua conta de email DSM) ou através de Correio (uma aplicação de email de terceiros). Para quem utiliza pela primeira vez, siga as instruções do assistente para configurar uma conta de email para utilização.

Para carregar ficheiros:

Se lhe for pedido que carregue ficheiros através da ligação, faça o seguinte:

  1. Abra o URL dado no computador local ou dispositivo móvel (Android ou iOS). Introduza a palavra-passe se necessário.
  2. Na página de carregamento, introduza um nome que indica a sua identidade.
  3. Basta arrastar e largar o(s) ficheiro(s) a carregar, ou clique em Adicionar Ficheiros para selecionar o(s) ficheiro(s).
  4. Clique em Carregar para terminar o epdido.

Nota:

  • Não pode carregar pastas através de uma ligação de pedido de ficheiro.
  • Pastas dentro da pasta partilhada "web" não podem ser especificadas como pasta de destino.
  • A função de carregamento varia consoante navegadores web:
    • Internet Explorer 8 e Internet Explorer 9 não suportam o carregamento de ficheiros através de arrastar e largar.
    • Chrome em dispositivos iOS apenas é compatível com o carregamento de imagens.
    • Safari em dispositivos iOS apenas é compatível com o carregamento de imagens e vídeo.
    • Navegadores web em dispositivos Android não estão sujeitos a qualquer tipo de limite de ficheiros.

Para encontrar ficheiros carregados:

Ficheiros carregados pelo mesmo utilizador aparecem todos numa pasta idêntica dentro do destino. Por predefinição, o nome da pasta será Nome_Carregador (Nome_Solicitador).

Por exemplo, se os ficheiros forem carregados por Watson para Holmes, pode encontrar os ficheiros na pasta com o nome Watson (Holmes).

Gerir Ligações de Pedido de Ficheiros

Para modificar ligações de pedidos de ficheiros:

Depois de criar uma ligação de pedido de ficheiro, pode modificar as suas definições na File Station > Ferramentas > Gestor de Ligações Partilhadas.

  1. Clique duas vezes na ligação desejada, ou seleccione a ligação e clique em Editar.
  2. Na janela de contexto, modifique as definições relevantes.
  3. Clique em Guardar para aplicar as novas definições.

Para eliminar ligações de pedidos de ficheiros:

  1. Aceda a File Station > Ferramentas Gestor de Ligações Partilhadas.
  2. Selecione a(s) ligação(ões) partilhada(s) desejada e clique em Eliminar. O(s) URL correspondente(s) ao pedido de ficheiro param imediatamente de funcionar.

Para atribuir privilégios de arquivamento de pedido:

Como admin ou utilizador pertencente ao grupo administrators, pode atribuir privilégios a utilizadores para que possam criar ligações de pedidos de ficheiros.

  1. Aceda a File Station > Definições > Ligações Partilhadas > Ligações de Pedido de Ficheiros.
  2. Selecione o tipo de utilizadores para lhe conferir privilégios de arquivamento de pedidos.
  3. Clique em OK para guardar as definições.
