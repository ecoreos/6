Montar pastas remotas

Pode montar pastas remotas de servidores remotos (como computadores de rede ou outros dispositivos Synology NAS) para navegar e gerir pastas montadas através da File Station.

Para montar uma pasta remota partilhada CIFS*:

  1. Clique em Ferramentas > Montar Pasta Remota > Pasta Partilhada CIFS..
  2. No campo Pasta, especifique o caminho da pasta remota no formato \\Remote_Server\Remote_Folder, como \\192.168.0.2\share ou \\chad\cheeseburger\pictures.
  3. Se forem necessários privilégios de acesso à pasta remota pelo servidor remoto, introduza as suas credenciais de utilizador do servidor remoto nos campos Nome da conta e Palavra-passe.
  4. Clique em Procurar para seleccionar ou criar uma pasta de destino vazia em DiskStation para montar a pasta remota.
  5. Seleccione Montar automaticamente ao iniciar caso deseje que DiskStation monte a pasta remota a cada início ou reinício de sistema.
  6. Clique em Montar para que a pasta remota seja montada na pasta de destino. Pode agora navegar e gerir a pasta remota do seu DiskStation.

Para montar uma pasta remota partilhada NFS:

  1. Clique em Ferramentas > Montar Pasta Remota > Pasta Partilhada NFS.
  2. No campo Pasta, especifique o caminho da pasta remota no formato Remote_Server:/Remote_Folder_Path, como por exemplo 192.168.0.2:/volume1/share.
  3. Selecione a versão NFS e protocolo de transporte desejados.
  4. Clique em Procurar para seleccionar ou criar uma pasta de destino vazia em DiskStation para montar a pasta remota.
  5. Assinale Montar automaticamente ao iniciar caso deseje que DiskStation monte a pasta remota a cada início ou reinício de sistema.
  6. Clique em Montar para que a pasta remota seja montada na pasta de destino. Pode agora navegar e gerir a pasta remota do seu DiskStation.

Nota:

NFSv4 suporta apenas o protocolo TCP.

Para desmontar uma pasta remota:

  1. Seleccione a pasta remota que deseja desmontar, e clique em Ferramentas > Desmontar.
  2. Como alternativa, basta clicar com o botão direito na pasta remota e seleccionar Desmontar.

Nota:

  • O número máximo de pastas remotas em simultâneo é de 100.
  • Só poderá montar pastas remotas de um servidor remoto compatível com os protocolos CIFS e NFS.
  • Pastas remotas podem apenas ser montadas em subpastas dentro de pastas partilhadas.
  • Montar pastas remotas é apenas suportado no modo de privilégio elevado. Para usar esta funcionalidade, aceda ao pacote Docker no seu DSM anfitrião, selecione o Docker DSM que deseja para ativar pastas partilhadas encriptadas, clique em Editar e assinale a caixa junto de Executar contentor usando privilégios elevados.

* Apenas disponível em modelos específicos.
