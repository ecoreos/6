Ver documentos, fotos ou vídeos

Com o File Station você pode ver documentos, vídeos ou fotos facilmente.

Para mudar a forma como os itens são exibidos:

Você pode mudar entre a visualização de lista e visualização de ícone para mudar a forma como os itens são exibidos na área principal. Clique no botão Visualização de lista no canto superior direito para organizar os arquivos em uma lista ou clique no botão Visualizar ícone para exibi-los com ícones e texto.

Você verá miniaturas de fotos no ícone de visualização nas condições a seguir:

  • As fotos estão localizadas nas pastas indexadas listadas em Painel de controle > Serviço de indexação > Indexação de mídia > Pasta indexada e você já instalou o Photo Station ou o Servidor de Mídia em seu dispositivo DiskStation.
  • Os dados da miniatura armazenados nas entradas EXIF criados por sua câmera digital estão disponíveis.

Para ver os documentos no Google docs:

Basta clicar com o botão direito no documento que deseja abrir e selecionar Ver no Google docs.

Formatos suportados:

O Visualizador do Google docs suporta os seguintes formatos: doc, docx, ppt, xlsx, pdf, tif

Nota:

Para visualizar documentos no Google docs, verifique se o dispositivo DiskStation está totalmente acessível através da Internet. Se o DiskStation estiver localizado atrás de um roteador, verifique se foi especificada uma regra de encaminhamento de portas para o DSM em Painel de controle > Acesso Externo > Configuração do Roteador. Os números de porta padrão para o DSM são 5000 (HTTP) e 5001 (HTTPS). Caso ainda tenha problemas para ver documentos no Google docs, você pode ter de especificar a porta do roteador para DSM em Painel de controle > Acesso externo > Avançado.

Para ver os documentos no Office Online:

Basta clicar com o botão direito no documento que deseja abrir e selecionar Visualizar no Office Online.

Ao usar o Office Viewer para visualizar documentos do Microsoft Office hospedados no dispositivo DiskStation, você pode acessar esses documentos com recursos do Office intactos em um navegador da Web, sem a necessidade de ter o Office instalado no computador.

Formatos suportados:

O Office Viewer suporta os seguintes formatos:

  • Word: docx, docm, dotm, dotx
  • Excel: xlsx, xlsb, xls, xlsm
  • PowerPoint: pptx, ppsx, ppt, pps, pptm, potm, ppam, potx, ppsm

Nota:

  • Para visualizar documentos do Office online, certifique-se de ter ativado o QuickConnect. Caso contrário, faça login no DSM como um usuário pertencente ao grupo administrators e vá para Painel de Controle > QuickConnect.
  • Os documentos do Word e do PowerPoint devem ser inferiores a 10 MB; do Excel devem ser inferiores a 5 MB.
  • O nome do arquivo do documento deve conter apenas letras e números.

Para visualizar fotos:

  1. Clique com o botão direito em foto e selecione Visualizar para abrir a janela do Visualizador de fotos.
  2. Passe o cursor do mouse sobre a parte inferior do Visualizador de fotos para mostrar a barra deslizante de miniaturas para navegação rápida.

Formatos suportados:

O Visualizador de fotos suporta os seguintes formatos: jpg, jpeg, jpe, bmp, gif, png, tiff, tif, arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, 3fr, erf, mos, orf, rw2, raw, psd

Para reproduzir (visualizar) arquivos de som:

  • Para reproduzir arquivos de som: Clique duas vezes em um ou mais arquivos de som para abri-los com o Reprodutor de Áudio. Como alternativa, você também pode reproduzir arquivos de som clicando com o botão direito do mouse em um arquivo de som e selecionando Reproduzir.*
  • Para reproduzir arquivos de som por pasta: Clique em uma única pasta ou pressione Ctrl (para usuários Windows)/command (para usuários Mac) e clique em várias pastas. Em seguida clique com o botão direito nas pastas selecionadas e escolha Reproduzir para abri-las com o Reprodutor de áudio.*

Formatos suportados:

O Reprodutor de Áudio aceita os seguintes formatos:

  • mp3, m4a, m4b, aac, ogg, wav, flac, ape, aiff, aif, wma

Nota:

Você pode reproduzir até 1024 arquivos de som por vez.

Para reproduzir vídeos:

  1. Clique com o botão direito no vídeo e selecione Reproduzir para abrir a janela do Reprodutor de vídeo.
  2. O vídeo deve começar a ser exibido imediatamente. Se isso não acontecer, verifique se o VLC Player (Windows) foi instalado no computador local.

    3. Nota:

    • Durante a instalação do VLC Player, certifique-se de marcar o plugin Mozilla e o plugin ActiveX.
    • O VLC é compatível com os seguintes navegadores:
      Windows: Internet Explorer, Firefox, Google Chrome
    • Se o VLC estiver instalado no computador e você estiver se conectando ao DSM com um navegador da web compatível, o Reprodutor de vídeo suporta os seguintes formatos: avi, dat, mkv, vob, asf, wmv, mov, qt, divx, flv, swf, rm, rmvb, mpg, mpeg, mpeg1, m1v, mpeg2, m2v, mp2, dvr-ms, m2ts, mts, m2t, ts, tp, trp, mpeg4, m4v, mp4, 3g2, 3gp, 3gp2, webm, mpe, xvid
    • Se o VLC não estiver instalado no computador local, o reprodutor de vídeo reproduzirá os vídeos suportados pelo HTML5. Veja aqui para saber quais vídeos HTML5 são suportados pelos diversos navegadores da web.
    • Se tiver se conectado ao DSM através do protocolo HTTPS, você deverá importar um certificado confiável (no Painel de controle > Segurança > Certificado) para poder reproduzir vídeos com o File Station.
    • O VLC não suporta a transmissão HTTPS.
  3. Se você tiver um arquivo de legenda com o mesmo nome e na mesma pasta que o vídeo, ou se o vídeo tiver legendas incorporadas, poderá ver as legendas durante a reprodução do vídeo. Se quiser usar outra legenda, selecione-a no menu suspenso.
