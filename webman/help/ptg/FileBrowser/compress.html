Comprimir Ficheiros ou Pastas

  1. Seleccione esses ficheiros ou pastas a partir da área principal do DiskStation.
  2. Seleccione Comprimir para “(nome de ficheiro)”.zip a partir do menu pendente Acção para começar a comprimir.
  3. Se não conseguir visualizar o ficheiro zip quando a compressão tiver sido concluída, escolha Actualizar a partir do menu pendente Acção para visualizar o ficheiro.
  4. Se pretender renomear, encriptar o arquivo ou arquivar utilizando diferentes níveis ou modos de compressão, seleccione Adicionar ao arquivo..., alternativamente.
    Nota:
  1. Não pode comprimir ficheiros ou pastas no seu computador local utilizando a File Station.
  2. Formatos de arquivo suportados: .zip, .7z.
  3. Quando escolhe um .zip como o Formato do arquivo em Adicionar ao arquivo..., pode seleccionar um codepage desejado para o ficheiro .zip.
