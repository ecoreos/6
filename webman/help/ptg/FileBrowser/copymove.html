Copiar ou Mover Ficheiros ou Pastas

File Station proporciona-lhe diferentes métodos de transferência de ficheiros/pastas entre dois DiskStation, dentro de um DiskStation, ou para um computador local.

Arrastar e Largar Ficheiros/Pastas entre Navegadores

Sem a configuração complexa de protocolos de transferência de ficheiros, pode carregar directamente ficheiros/pastas através de File Station de um DiskStation para outro.

Para activar arrastar e largar entre navegadores:

  1. Aceda a Definições > Geral > File Station.
  2. Assinale Activar arrastar e largar entre navegadores.
  3. Clique em OK.
  4. Abra dois navegadores web do mesmo tipo (por exemplo, Firefox), e ligue-os respectivamente ao DiskStation de origem e destino.
  5. Abra File Station em ambos DiskStation. Pode agora arrastara e largar directamente ficheiros/pastas entre eles através de File Station.

Nota:

  • Pode também arrastar e largar ficheiros/pastas entre dois separadores do mesmo navegador.
  • O Internet Explorer não é compatível com o arrastar e largar entre navegadores ou entre separadores.
  • Se múltiplos ficheiros/pastas estiverem seleccionados para arrastar e largar, os ficheiros/pastas serão comprimidos num arquivo para carregamento.
  • O limite do tamanho de ficheiro para arrastar e largar depende dos navegadores web: Safari (1 GB).

Copiar ou Mover Ficheiros/Pastas dentro de DiskStation ou para o Computador Local

Siga os passos abaixo para mover ou copiar ficheiros ou pastas de DiskStation para o seu computador local.

Método 1:

  1. Seleccione os ficheiros ou pastas que pretende mover ou copiar. (Mantenha premida a tecla Shift ou Ctrl enquanto efetua seleções múltiplas.)
  2. Escolha Cortar ou Copiar a partir do menu pendente Acção.
  3. Aceda à pasta de destino na DiskStation ou no seu computador.
  4. Escolha Colar - Substituir ou Colar - Ignorar a partir do menu pendente Acção, dependendo se pretende substituir ficheiros ou pastas com o mesmo nome na pasta de destino.

Método 2:

  1. Seleccione os ficheiros ou pastas que pretende mover ou copiar. (Mantenha premida a tecla Shift ou Ctrl enquanto efetua seleções múltiplas.)
  2. Arraste as pastas ou ficheiros da secção principal para outra pasta no painel esquerdo.
  3. No menu instantâneo que aparece, escolha Copiar - Substituir, Copiar - Ignorar, Mover - Substituir ou Mover - Ignorar para substituir ou ignorar ficheiros ou pastas com o mesmo nome na pasta de destino.

Método 3

  1. Seleccione os ficheiros ou pastas que pretende mover ou copiar. (Mantenha premida a tecla Shift ou Ctrl enquanto efetua seleções múltiplas.)
  2. Prima Ctrl + X para cortar ou Ctrl + C para copiar os ficheiros/pastas. Para utilizadores Mac, prima Command + X e Command + C.
  3. Aceda à pasta de destino na DiskStation ou no seu computador.
  4. Prima Ctrl + V para colar os ficheiros/pastas. (Command + V para Mac.)
  5. Na janela de diálogo que aparece, escolha entre substituir ou ignorar os ficheiros com o mesmo nome na pasta de destino.

Atalhos de teclado:

Os seguintes atalhos de teclado podem ser úteis quando move/copia ficheiros e pastas na File Station:

  • F2: Mudar o nome
  • Del: Eliminar
  • Ctrl + A: Seleccionar tudo
  • Crtl + X: Cortar
  • Crtl + C: Copiar
  • Crtl + V: Colar

Nota:

  • Ao copiar ou mover ficheiros/pastas para um directório no sistema de ficheiros FAT, os caracteres \ *?">| no nome do ficheiro serão substituídos por – e a Metadata dos ficheiros Mac será apagada.
  • Se cortar ficheiros do seu computador local, e depois colá-los numa pasta DiskStation, os ficheiros irão ser retidos no seu computador local.
  • Quando move um ficheiro ou pasta, o "proprietário" permanece o mesmo. No entanto, quando copia um ficheiro ou pasta, o "proprietário" é alterado para o utilizador que o copiou.
