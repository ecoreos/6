Criar atalhos Os Meus Favoritos ou Ambiente de Trabalho

Com a File Station, pode adicionar pastas DiskStation aos Meus Favoritos, ou criar atalhos de ambiente de trabalho para ficheiros ou pastas DiskStation.

Adicionar Pastas aos Meus Favoritos

Para subpastas usadas frequentemente na sua DiskStation, pode adicionar os respetivos atalhos à secção Os meus favoritos no topo do painel esquerdo, para acesso mais rápido.

Para adicionar uma pasta DiskStation aos Meus Favoritos:

  1. Selecione a pasta, e depois faça qualquer dos seguintes:
    • Escolha o menu Acção ou clique com o botão direito na pasta, e depois escolha Criar atalho > Os meus favoritos.
    • Se já tiver adicionado outra pasta aos Meus Favoritos, verá a secção Os meus favoritos no canto superior esquerdo. Arraste a pasta até essa secção para criar um atalho.
  2. Na janela que aparece, atribua um nome ao atalho e clique em OK.

Para reordenar um atalho dos Meus Favoritos:

Arraste e largue o atalho na secção Os meus favoritos para reordenar.

Para mudar o nome de um atalho:

  1. Clique com o botão direito do rato no atalho e escolha Renomear.
  2. Na janela que aparece, mude o nome do seu favorito e clique em OK.

Para eliminar um atalho:

Clique com o botão direito num atalho, escolha Remover dos Meus Favoritos, e depois clique em Sim.

Para limpar todos os favoritos que apontam a caminhos de pastas inexistentes:

Clique com o botão direito em Os Meus Favoritos no topo do painel esquerdo, e depois escolha Limpar favoritos inválidos.

Criar Atalhos no Ambiente de Trabalho para Ficheiros ou Pastas

Para ficheiros ou subpastas DiskStation usados frequentemente, pode criar atalhos no ambiente de trabalho para acesso mais rápido.

Para criar um atalho no ambiente de trabalho para um ficheiro ou pasta DiskStation:

Clique com o botão direito no ficheiro ou pasta na área principal, e depois faça qualquer dos seguintes:

  • Se estiver a criar um atalho no ambiente de trabalho para um ficheiro, escolha Criar atalho no ambiente de trabalho.
  • Se estiver a criar um atalho no ambiente de trabalho para uma pasta, escolha Criar atalho > Ambiente de trabalho DSM.
