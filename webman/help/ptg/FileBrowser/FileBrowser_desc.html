File Station

File Station é uma ferramenta de gestão de ficheiros, de fácil utilização, permitindo que os utilizadores do DSM acedam através da Internet às pastas da DiskStation com os seus navegadores de Internet. Ao iniciar dois ou mais File Stations, pode gerir todos os seus dados da DiskStation arrastando e largando as mesmas entre diferentes File Stations.
