Gerir Definições File Station

Para gerir as definições da File Station, abra a File Station e clique no botão Definições.

Geral

Aqui pode ativar os registos da File Station para registar as atividades de todos os utilizadores. As atividades do utilizador incluem Criar pasta, Carregar, Transferir, Eliminar, Renomear, Mover, Copiar e Propriedade definida.

Para ativar o registo da File Station:

  1. Se pretender registar as atividades de todos os utilizadores, assinale a caixa de verificação Ativar o registo da File Station.
  2. Clique em OK.

Para visualizar o registo da File Station:

  1. Clique em Visualizar Registos, que o encaminha para Centro de Registos.
  2. Clique no separador Transferência de Ficheiros.

Para substituir automaticamente os ficheiros com carregamento através de arrastar e largar ou atalhos de teclado:

Assinale Substituir ficheiros sempre que os arrasto e largo no navegador Web ou colo com atalhos de teclado se pretender substituir automaticamente os ficheiros com o mesmo nome quando os arrasta e larga do seu computador para o navegador Web ou cola ficheiros com atalhos de teclado na File Station.

Para ativar arrastar e largar inteligente:

Quando arrasta e larga ficheiros dentro da File Station, esta funcionalidade irá determinar automaticamente se a transferência é uma ação de cópia ou deslocação.

  1. Selecione Ativar arrastar e largar inteligente.
  2. Selecione uma das ações a realizar sempre caso haja outro utilizador a transferir ficheiros em simultâneo com o mesmo nome, para o mesmo local:
    • Substituir: Os ficheiros que transferir irão substituir os ficheiros com o mesmo nome.
    • Ignorar: Os ficheiros que transferir irão ignorados e não irão substituir os ficheiros com o mesmo nome.
  3. Clique em OK.

Nota:

Se desejar alterar a ação de cópia ou deslocação determinada pelo arrastar e largar inteligente, aplique as seguintes teclas de atalho para se ajustar às suas necessidades:

  • Para mudar para uma ação de cópia: Prima Ctrl enquanto realiza o arrastar e largar.
  • Para mudar para uma ação de deslocação: Prima Shift enquanto realiza o arrastar e largar.

Para percorrer e gerir ficheiros do computador local com File Station:

  1. Assinale Pesquisar ficheiros do computador local com a File Station.
  2. Clique em OK.

Para partilhar a lista de utilizadores DSM com utilizadores não admin:

Esta opção permite aos utilizadores não admin acederem à lista completa de utilizadores DSM caso desejem procurar ficheiros de utilizadores/grupos DSM específicos, ou caso desejem partilhar ligações de ficheiros com outros utilizadores DSM.

  1. Selecione Partilhe a lista de utilizadores DSM com utilizadores não admin.
  2. Clique em OK.

Aplicar Permissões UNIX Predefinidas

Clique em Definições Avançadas e assinale a opção para aplicar as permissões UNIX predefinidas quando carregar ou criar ficheiros e pastas. As permissões aplicadas são as mesmas permissões aplicadas pelo comando UNIX umask. Quando esta opção estiver ativada, as permissões UNIX são 644 para ficheiros e 755 para pastas. Quando esta opção estiver desativada, as permissões UNIX são 666 para ficheiros e 777 para pastas. O valor umask predefinido é 022.

Nota:

  • Para pastas partilhadas com ativação ACL do Windows (todas as partilhas, excluindo "photo" e partilhas em discos externos) execute o comando chmod no seu Linux ou cliente FTP para alterar os tipos de permissão de pasta e ficheiro do ACL do Windows para UNIX.
  • Ativar esta opção pode provocar questões de inconsistência de permissões entre diferentes protocolos. Para evitar inconsistências, sugerimos que deixe esta opção desativada.

Montagem/Ligações

Por predefinição, todos os utilizadores podem montar pastas remotas ou unidades virtuais. Aqui poderá limitar o acesso a uma pasta remota ou gestão de unidade virtual apenas para administradores.

Nota:

Os utilizadores que não são administradores só podem montar pastas remotas na pasta home.

Para permitir a diferentes utilziadores montar pastas remotas ou unidades virtuais:

  1. Escolha Todos os utilizadores ou Administradores na secção Pasta Remota ou Unidade Virtual.
  2. Clique em OK.

Para permitir a diferentes utilizadores montar servidores e serviços na nuvem:

  1. Escolha entre Todos os utilizadores, Administrators ou Utilizador específico.
  2. Ao selecionar Utilizador específico, irá abrir uma janela que lhe permite configurar as permissões de montagem para diferentes utilizadores. Para ativar o acesso a um utilizador específico, certifique-se de que a caixa está assinalada na coluna Ativado. Para desativar o acesso a um utilizador específico, certifique-se de que a caixa não está assinalada na coluna Desativado. Depois de concluído, clique em OK para fechar esta janela.

    3. Nota:

    • Os utilizadores pertencentes ao grupo administrators podem utilizar esta funcionalidade mas não podem efetuar quaisquer alterações. Os utilizadores com início de sessão como guest não podem utilizar esta funcionalidade nem efetuar quaisquer alterações. As permissões para todos os outros utilizadores podem ser definidas aqui.
  3. Clique em OK.

Ligações partilhadas

Por predefinição, todos os utilizadores podem partilhar ligações de ficheiros. Aqui poderá limitar o acesso à gestão de ligações partilhadas apenas para administradores.

Para permitir que diferentes utilizadores partilhem ligações de ficheiros:

  1. Escolha Todos os utilizadores, Administrators ou Utilizadores específicos.
  2. Clique em OK.

Para abrir ficheiros HTML como texto simples:

  1. Assinale a caixa Abrir sempre os ficheiros HTML utilizando o formato de texto simples. Se esta opção estiver ativada, os ficheiros HTML (ou seja, os ficheiros HTM, HTML, JSP e XHTML) são apresentados com o formato de texto simples quando são abertos pelos utilizadores através de uma ligação de partilha de ficheiro. Esta opção evita que terceiros, com intenções maliciosas, se apropriem indevidamente dos inícios de sessão do utilizador com scripts integrados em ficheiros HTML.
  2. Clique em OK.

Limite de Velocidade

Pode ativar um limite de velocidade para controlar a largura de banda máxima utilizável para utilizadores e grupos que transferem dados através da File Station.

Para ativar o limite de velocidade:

  1. Selecione a caixa marcada Ativar limite de velocidade ou Ativar limite de velocidade de acordo com um agendamento se pretender implementar um limite de velocidade apenas em horas específicas. Esta é uma definição ao nível do servidor e irá substituir as definições de grupos e utilizadores individuais. A escolha de Sem limite de velocidade desativa as definições do limite de velocidade de todos os utilizadores e grupos.
  2. Clique em Definições de Limite de Velocidade para modificar as definições de grupos ou utilizadores individuais. Para cada utilizador e grupo, pode selecionar uma das seguintes opções:
    • Aplicar Definições de Grupo: As definições do limite de velocidade serão iguais às definições do grupo ao qual pertence o utilizador.
    • Sem limite de velocidade: Não será aplicado qualquer limite de velocidade quando os membros do grupo transferirem ficheiros com o serviço.
    • Constante: Será aplicado um limite de velocidade fixo e constante ao utilizador ou membros do grupo. Os limites de velocidade de carregamento e transferência podem ser especificados nos campos à esquerda.
    • Variável: É possível especificar e aplicar dois limites de velocidade diferentes aos membros do grupo ou utilizadores de acordo com um agendamento. Clique no botão Personalizar para modificar as definições do limite de velocidade e definir um agendamento.
  3. Clique em Aplicar.

Nota:

  • Para obter mais detalhes sobre a dinâmica do limite de velocidade de grupos, consulte aqui.
  • Quando definir o limite de velocidade para um utilizador, a secção Resultado apresenta o limite de velocidade real do utilizador de acordo com as definições de grupo e individuais. Por exemplo, se um utilizador "Achilles" tiver um limite de transferência individual de 100 KB/s, mas pertencer a um grupo de utilizadores com um limite de transferência de 500 KB/s, a secção Resultados apresenta "0/500".

Segurança

Aqui pode consultar as definições de segurança seguintes.

Para abrir ficheiros HTML como texto simples:

  1. Assinale a caixa Abrir sempre os ficheiros HTML utilizando o formato de texto simples. Se esta opção estiver ativada, os ficheiros HTML (ou seja, os ficheiros HTM, HTML, JSP e XHTML) são apresentados com o formato de texto simples quando são abertos na File Station. Esta opção evita que terceiros, com intenções maliciosas, se apropriem indevidamente dos inícios de sessão do utilizador com scripts integrados em ficheiros HTML.

    2. Nota:

    • Esta não é uma definição global. Cada utilizador pode escolher ativar ou não esta opção, de acordo com as suas necessidades individuais.
  2. Clique em OK.
