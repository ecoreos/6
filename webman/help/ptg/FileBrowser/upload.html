Carregar Ficheiros ou Pastas

Utilize um dos métodos seguintes para carregar ficheiros ou pastas do seu computador para a DiskStation.

Para ter acesso total às funções de carregamento abaixo mostradas, instale primeiro o Java Runtime Environment (JRE) e Flash Player mais recente.

Método 1:

  1. A partir da parte superior do painel esquerdo, seleccione uma pasta de destino na DiskStation.
  2. Clique em Carregar, escolha Carregar - Ignorar ou Carregar - Sobrepor e seleccione os ficheiros ou pastas do seu computador local.

Método 2:

  1. Aceda a Definições, assinale Percorrer ficheiros do computador local com a File Station e actualize a página web para ver pastas do seu computador local na parte inferior do painel esquerdo.

    2. Nota:

    A opção Percorrer ficheiros do computador local com a File Station tem efeito apenas depois de instalar e activar JRE no seu computador.

  2. A partir da parte inferior do painel esquerdo, selecione uma pasta no seu computador local. A secção principal exibe agora o conteúdo da pasta selecionada.
  3. Selecione os ficheiros ou pastas que pretende na área principal e de seguida selecione Carregar.
  4. Seleccione uma pasta de destino na DiskStation, a partir da janela que aparece.
  5. No canto superior direito da janela, selecione Sobrepor se pretender substituir ficheiros ou pastas com o mesmo nome na pasta de destino. Caso contrário, selecione Ignorar.
  6. Clique em OK para começar a carregar.

Método 3:

  1. Aceda a Definições, assinale Percorrer ficheiros do computador local com a File Station e actualize a página web para ver pastas do seu computador local na parte inferior do painel esquerdo.
  2. No painel esquerdo, arraste uma pasta do seu computador (na parte inferior) para qualquer pasta na DiskStation (na parte superior).
  3. No menu de selecção que aparece, escolha entre substituir ou ignorar os ficheiros/pastas com o mesmo nome na pasta de destino.

Método 4:

  1. A partir da parte superior do painel esquerdo, seleccione uma pasta de destino na DiskStation.
  2. Selecione os ficheiros no ambiente de trabalho do seu computador ou através do explorador de ficheiros (por exemplo, Explorador do Windows ou Mac Finder) e de seguida arraste e largue-os na File Station no navegador web.
  3. Na janela de diálogo que aparece, escolha entre substituir ou ignorar os ficheiros com o mesmo nome na pasta de destino.

    4. Nota:

    • O Método 4 apenas é suportado por navegadores específicos: Chrome, Firefox, Safari 5.1 ou superior. No entanto, não é suportado para Mac quando Percorrer ficheiros do computador local com a File Station está assinalado.
    • Pode também arrastar e largar pastas ou ficheiros vazios, mas apenas quando Percorrer ficheiros do computador local com a File Station está assinalado para os navegadore listados acima. Caso contrário, não pode carregar pastas e ficheiros vazios usando o Método 4.
    • Se estiver a utilizar o Chrome para Linux e tiver iniciado sessão no DSM através do protocolo HTTPS, é necessário importar um certificado de confiança (em Painel de Controlo > Segurança > Certificado) para ser possível carregar através do Método 4.

Nota:

  1. Pode adicionar até 1000 ficheiros à sua fila de upload.
  2. Pode carregar ficheiros com mais de 4GB se estiver a usar o Google Chrome, Firefox, Safari 5.1 ou superior, ou quando Percorrer ficheiros do computador local com a File Station está assinalado. Caso contrário, apenas pode carregar até 2GB de ficheiros de cada vez.
  3. Numa pasta partilhada não encriptada, o nome do ficheiro/pasta deve ter entre 255 caracteres (até cerca de 80 caracteres para idiomas não Latinos); numa pasta partilhada encriptada, deve ser composto pelo máximo de 143 caracteres (até cerca de 47 caracteres para idiomas não Latinos).
  4. O nome do Ficheiro/Pasta deve excluir os seguintes caracteres: \ / : * ? " > |.
  5. O nome do ficheiro/pasta não pode começar com "._".
  6. Pode carregar ficheiross que tenham 0KB de tamanho apenas quando tiver instalado JRE.

Resolução de problemas

Se não for possível pesquisar os itens locais guardados no seu Mac utilizando adequadamente o Safari, proceda da seguinte forma:

  1. No seu Mac, abra o Safari e escolha Preferências no menu do Safari.
  2. Clique em Gerir Definições do Website no separador Segurança.
  3. Em Java, escolha o seu "website" do DSM.
  4. Seleccione Executar em Modo não Seguro.
