Montar unidades virtuais

Pode montar unidades virtuais para aceder a conteúdos de imagens de disco com a File Station.

Para montar uma unidade virtual:

  1. Selecione uma imagem de disco e clique em Ferramentas > Montar Unidade Virtual.
  2. Clique em Pesquisar para selecionar ou criar uma pasta de destino vazia em DiskStation para montar a unidade virtual.
  3. Selecione Montar automaticamente ao iniciar caso deseje que o seu DiskStation monte a unidade virtual a cada início ou reinício de sistema. Clique em Montar para continuar. Poderá agora navegar até à pasta de destino onde a unidade virtual está montada e aceder aos conteúdos da imagem de disco.

Nota:

As unidades virtuais e ficheiros .iso apenas podem ser montados em subpastas dentro de pastas partilhadas.

Para desmontar uma unidade virtual:

  1. Clique no botão direito do rato sobre a pasta de destino onde a unidade virtual está montada e selecione Desmontar.
    Nota:
  1. Formatos de imagem de disco suportados: Ficheiros ".iso" utilizando o sistema de ficheiros ISO 9660 e UDF.
  2. O número máximo de unidades virtuais em simultâneo é de 16.
  3. Montar unidades virtuais é apenas suportado no modo de privilégio elevado. Para usar esta funcionalidade, aceda ao pacote Docker no seu DSM anfitrião, selecione o Docker DSM que deseja para ativar pastas partilhadas encriptadas, clique em Editar e assinale a caixa junto de Executar contentor usando privilégios elevados.
