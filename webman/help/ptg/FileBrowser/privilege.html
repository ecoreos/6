Visualizar ou Alterar Permissões dos Ficheiros/Pastas

Com a File Station, pode definir privilégios de acesso a ficheiros e subpastas. Esta função é especialmente útil quando pretende permitir que diferentes grupos de utilizadores acedam a diferentes subpastas numa pasta partilhada.

Para permitir o acesso de utilizadores e grupos a ficheiros e subpastas:

  1. Selecione os ficheiros e pastas aos quais pretende definir privilégios de acesso.
  2. Clique com o botão direito nos ficheiros ou pastas ou escolha o menu Ação e, de seguida, Propriedades.
  3. Na secção Proprietário, escolha o proprietário dos ficheiros e das pastas selecionados a partir do menu pendente Proprietário.
  4. Clique na tabela Permissões e selecione um utilizador ou grupo. Clique em Editar para abrir o Inspetor de Permissões.
  5. Proceda de qualquer uma das formas seguintes na janela Editor de Permissões para gerir as permissões ACL para o ficheiro ou pasta:
    1. Utilizador ou grupo: Especifique o utilizador ou grupo cujas permissões deseja personalizar.
    2. Herdado de: Apenas para visualização. Veja aqui as informações para verificar se a permissão é herdada (de uma pasta superior) ou explícita (mostrada como Nenhuma).
    3. Tipo: Escolha Permitir ou Negar para conceder ou negar a permissão para o utilizador ou grupo.
    4. Aplicar a: Se estiver a criar uma entrada de permissão para uma pasta, assinale as caixas de verificação para aplicar a entrada nesta pasta, nas pastas (ou Pastas Inferiores) ou ficheiros (ou Ficheiros Inferiores) nesta pasta ou em todos os ficheiros e pastas contidos nesta pasta (ou Todos os descendentes).
    5. Administração: Assinale Ler permissões, Alterar permissões ou Assumir propriedade para especificar as definições de permissão de acesso do utilizador ou grupo para a entrada.
    6. Leitura ou Escrita: Assinale as caixas de verificação nestas secções para modificar as definições de permissão do utilizador ou grupo para o ficheiro ou pasta.
  6. Clique em OK.

Nota: Não lhe é permitido editar os privilégios de acesso a ficheiros e subpastas na pasta partilhada photo aqui.
