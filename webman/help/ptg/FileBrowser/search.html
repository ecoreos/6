Procurar Ficheiros ou Pastas

O Filtro da File Station fornece opções para pesquisar e exibir ficheiros ou pastas que cumpram os seus critérios. Pode filtrar ficheiros ou pastas na pasta acutal, ou realizar uma pesquisa avançada que abrange todas as subpastas para resultados de pesquisa mais detalhados.

Para procurar ficheiros ou pastas na pasta atual:

  1. Utilize a File Station para aceder à pasta que contém os conteúdos que pretende procurar.
  2. No campo Filtro, no canto superior direito, introduza uma palavra-chave e visualize os resultados filtrados na área principal da janela.

Para executar uma procura avançada:

  1. Utilize a File Station para aceder à pasta que contém os conteúdos que pretende procurar.
  2. Clique no ícone da lupa junto do campo Filtro no canto superior direito para revelar o menu de pesquisa avançada.
  3. Especifique abaixo os critérios de pesquisa ou escolha de entre os menus pendentes:
    • Palavra-chave: Introduza uma palavra-chave para pesquisar ficheiros ou pastas com nomes coincidentes com o mesmo.

      Nota:

      • Pode utilizar caracteres universais [*] (significando zero ou mais caracteres) e [?] (significando um caracter aleatório) nas suas palavras-chave. Por exemplo, [a*o] pode representar [acção], [atenção], ou [ao].
      • Pode utilizar espaços para separar várias palavras-chave. Por exemplo, a palavra-chave [File Station] conduz a resultados de pesquisa que contenham [File], [Station], ou ambos.
      • Pode utilizar aspas ["] no início e no final da palavra-chave para pesquisar correspondências exactas, sem ligar a maiúsculas ou minúsculas. Por exemplo, ["Synology Inc."] pode representar [Synology Inc.], mas não pode representar [Synology Incorporation].
      • Os operadores que se seguem podem ser usados para resultados de pesquisa mais refinados:
        Operador Exemplo Utilização
        && [X] && [Y]
      • Listar resultados que contenham ambos [X] e [Y]
        		•
        || [X] || [Y]
      • Listar resultados que contenham [X], [Y] ou ambos
      • Resultados que contenham ambos [X] e [Y] serão listados no topo
        		•
        + +[X]
      • Listar resultados que contenham [X]
        		•
        - -[X]
      • Listar resultados que não contenham [X]
        		•
        ! ![X]
      • Listar resultados que não contenham [X]
        		•
        \ \[SpecialCharacter]
      • Caracteres especiais:   +   -   &&   ||    !    (    )    {    }    [    ]    ^    "    ~    *    ?    :   \
      • Lista resultados que contenham um caracter especial acima
        		•
    • Ativar pesquisa de conteúdo de ficheiros: Pesquisar ficheiros indexados com o conteúdo de ficheiro ou metadados que corresponda às palavras-chave.
    • Localização: Seleccione no menu pendente a opção para configurar o alcance e dimensão da pesquisa avançada. Também pode personalizar o tipo de pesquisa, seleccionando outras pastas partilhadas na lista.
    • Tipo de Ficheiro: Escolha Extensão para limitar a área de pesquisa a tipos de ficheiros específicos ou escolha Qualquer Pasta para pesquisar apenas pastas.
    • Data: Escolha Data de Modificação, Data de Criação ou Último Acesso e depois escolha uma data no calendário para pesquisar ficheiros com datas correspondentes.
    • Tamanho (MB): Escolha entre igual, é superior a e é inferior a e, de seguida, introduza um tamanho para pesquisar ficheiros com tamanhos coincidentes. (Quando escolher igual, os resultados da pesquisa irão incluir ficheiros com o tamanho aproximado do ficheiro especificado.)
    • Proprietário/Grupo: Escolha Proprietário ou Grupo e, de seguida, escolha a partir do menu pendente para pesquisar ficheiros ou pastas pertencentes ao grupo ou proprietário seleccionado.

      Nota:

      • Para realizar uma pesquisa avançada, tem de especificar a Localização e pelo menos um critério de pesquisa.
      • Se uma pasta ISO ou pasta remota for associada numa pasta partilhada indexada, não consegue encontrar ficheiros lá dentro através da pesquisa avançada.
  4. Clique em Procurar para iniciar a pesquisa ou clique em Repor e especifique o critério de pesquisa novamente. Quando a pesquisa estiver concluída, os ficheiros coincidentes serão listados na área principal da File Station.
