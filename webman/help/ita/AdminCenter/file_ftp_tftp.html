TFTP

TFTP (Trivial File Transfer Protocol) è un semplice protocollo di trasferimento dei file, utilizzato in genere per il trasferimento dei file di configurazione o di avvio quando non è necessaria l'autenticazione. L'abilitazione del servizio TFTP consente a DiskStation di trasferire i caricatori di avvio ai dispositivi client.

Per configurare le impostazioni TFTP:

  1. Andare su Pannello di controllo > Servizi File > TFTP e selezionare Abilita il servizio TFTP.
  2. Nel campo Cartella radice TFTP, specificare a quale cartella nella DiskStation è possibile accedere tramite i client TFTP.
  3. Toccare Impostazioni avanzate per configurare le seguenti impostazioni.
    • Abilitare Registro di trasferimento file TFTP: Tenere traccia dei dati trasferiti tramite TFTP.
    • Autorizzazione client TFTP: Stabilire le autorizzazioni client TFTP:
      • Sola lettura: i client TFTP possono solo visualizzare i contenuti della cartella radice.
      • Scrivibile: i client TFTP possono modificare i contenuti della cartella radice.
    • Client consentiti: scegliere i client che possono connettersi tramite TFTP.
      • Tutte le connessioni: tutti i client possono connettersi tramite TFTP.
      • Consenti solo il seguente intervallo di indirizzi IP: solo i client nell'intervallo IP specificato possono connettersi tramite TFTP.
    • Timeout: se il server TFTP (es. DiskStation) non risponde alla scadenza del periodo specificato, la connessione TFTP verrà arrestata.
  4. Fare clic su Applica per salvare le impostazioni.
