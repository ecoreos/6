Win/Mac/NFS

Gli utenti Windows, Mac e Linux possono accedere a DiskStation dalla rete una volta che sono state abilitate le opzioni in Pannello di controllo > Servizi file > Win/Mac/NFS.

Sono supportati i seguenti protocolli di condivisione dei file:

  • Per Windows: SMB/CIFS (Risorse di rete), FTP, WebDAV
  • Per Mac: SMB, FTP, AFP, WebDAV
  • Per Linux: SMB, FTP, NFS, WebDAV
