USBCopy

È possibile selezionare una cartella di destinazione per la funzione USBCopy andando a Pannello di controllo > Dispositivi esterni > USBCopy/SDCopy.

Per eseguire USBCopy:

  1. Selezionare Abilita USBCopy.
  2. Selezionare una cartella condivisa di destinazione dall'elenco a discesa.

Quando viene collegata una periferica USB (come ad esempio una fotocamera digitale) alla porta USB sul pannello anteriore del prodotto Synology, è possibile premere il pulsante Copia sul pannello anteriore per copiare i dati presenti all'interno della periferica USB nella cartella condivisa di destinazione assegnata. Se la cartella di destinazione della USBCopy è stata rimossa, il sistema la sostituirà automaticamente con la cartella condivisa pubblica.

Limitazioni

  1. Sono supportati solo i dispositivi USB in formato FAT, NTFS, ext3 ed ext4.
  2. I dispositivi USB devono essere connessi alla porta USB sul pannello anteriore del dispostivo DiskStation.
  3. Solo admin o gli utenti nel gruppo administrators dispongono di privilegi di accesso adatti per eseguire le attività di backup nei dispositivi USB.
  4. Lo spazio disponibile della cartella condivisa di destinazione deve essere maggiore allo spazio utilizzato dal dispositivo USB.

SDCopy*

È possibile selezionare una cartella di destinazione per la funzione SDCopy andando a Pannello di controllo > Dispositivi esterni > USBCopy/SDCopy.

Per abilitare SDCopy:

  1. Selezionare Abilita SDCopy.
  2. Selezionare una cartella condivisa di destinazione dall'elenco a discesa.

Quando viene collegata una periferica SD nell'alloggiamento SD del pannello anteriore del prodotto Synology, è possibile premere il pulsante Copia posto sul pannello anteriore per copiare i dati presenti all'interno del dispositivo SD nella cartella condivisa di destinazione assegnata. Se la cartella di destinazione della SDCopy è stata eliminata, il sistema la sostituirà automaticamente con la cartella condivisa pubblica.

Limitazioni

  1. Sono supportati solo i dispositivi SD in formato FAT, NTFS, ext3 ed ext4.
  2. I dispositivi SD devono essere connessi alla porta SD sul pannello anteriore del prodotto Synology.
  3. Solo l'utente admin o gli utenti nel gruppo administrators dispongono di privilegi di accesso adatti per eseguire le attività di backup nei dispositivi SD.
  4. Lo spazio disponibile della cartella condivisa di destinazione deve essere maggiore allo spazio utilizzato dal dispositivo SD.

____
* Disponibile solo su alcuni modelli.
