Impostazioni DSM

Su Pannello di controllo > Rete > Impostazioni DSM, è possibile modificare il numero di porta che i client utilizzano per accedere a DiskStation.

Per cambiare il numero di porta predefinito:

Entrambe le connessioni HTTP e HTTPS sono sempre abilitate in DSM. La porta HTTP predefinita è 5000 mentre la porta HTTPS predefinita è 5001.

  1. Immettere il numero della porta nel campo HTTP o HTTPS.
  2. Fare clic su Applica.

Per reindirzzare le connessioni da HTTP a HTTPS:

È possibile reindirizzare i client alla porta HTTPS quando tentano di accedere a DiskStation tramite la porta HTTP.

  1. Selezionare Reindirizza automaticamente le connessioni HTTP in HTTPS (escluso Web Station e Photo Station).
  2. Fare clic su Applica.

Porte utilizzate per la connessione HTTPS:

  • DSM: il numero di porta predefinito è 5001, quindi è possibile accedere a DSM inserendo il seguente indirizzo: https://server-hostname:5001/
  • Web Station e Photo Station: Il numero di porta predefinito è 443, quindi è possibile accedere in modo sicuro a Web Station e Photo Station inserendo il seguente indirizzo: https://server-hostname/

Per rilevare DiskStation nella rete:

È possibile individuare DiskStation tramite il rilevamento rete Windows o il rilevamento servizio Bonjour.

  1. Selezionare un servizio per individuare DiskStation nella rete:
    • Abilita Scoperta rete Windows
    • Abilitare il rilevamento servizio Bonjour
  2. Fare clic su Applica.

Dominio

In Dominio, è possibile specificare un nome dominio personalizzato per accedere a DiskStation.

Per abilitare il dominio personalizzato:

  1. Spuntare Abilita dominio personalizzato.
  2. Specificare un nome dominio applicato per DiskStation.
  3. È possibile anche spuntare Abilita HSTS per forzare le connessioni crittate HTTPS in questo dominio.
  4. Fare clic su Applica.

Nota:

  • ogni nome dominio può essere usato solo per una distinta applicazione in DiskStation.
  • Se il nome dominio non supera 15 caratteri in lunghezza ed è conforme alle convenzioni di denominazione NetBIOS, il sistema registrerà e trasmetterà il nome dominio (es. MyDSM) automaticamente. Gli utenti Windows possono accedere al desktop DSM nella stessa LAN con un URL simile: http://MyDSM o https://MyDSM
  • Quando si accede alle applicazioni tramite gli URL che utilizzano nomi dominio, verrà attraversata la porta HTTP standard (80) o la porta HTTPS standard (443).
