Sede utente

Abilita home utente per creare una cartella personale home per ogni utente, tranne per guest. Tutti gli utenti possono accedere alla propria cartella principale via CIFS, AFP, FTP o File Station.

Admin può accedere a tutte le cartelle personali all'interno della cartella condivisa preimpostata homes. Il nome della cartella principale è lo stesso dell'account utente.

Per abilitare il servizio sede utente:

  1. Fare clic su Sede Utente.
  2. Selezionare Abilita il servizio sede utente.
  3. Se vi è più di un volume su Synology DiskStation, selezionare la destinazione dove si desidera archiviare la cartella homes.
  4. Fare clic su OK.

Nota: se si disattiva il servizio sede utente locale, verrà disattivato anche il servizio sede cliente di dominio.

Per disabilitare il servizio sede utente:

  1. Fare clic su Sede Utente.
  2. Deselezionare Abilita il servizio sede utente.

Dopo avere disabilitato il servizio sede utente, admin potrà accedere alle cartelle condivise homes. Gli utenti saranno nuovamente in grado di accedere alle loro cartelle principali quando verrà riabilitato il servizio sede utente.

    Nota:
  1. Per eliminare la cartella condivisa homes, occorre prima disabilitare il servizio sede utente.
  2. Il servizio sede utente di dominio sarà disattivato se il servizio sede locale è disattivato.
