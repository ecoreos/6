Impostazioni PHP

Una volta abilitata la Web Station, le impostazioni PHP descritte di seguito possono essere configurate andando su Pannello di controllo > Servizi web > Impostazioni PHP.

Per attivare cache per PHP

Selezionare Attiva cache per PHP se si desidera migliorare le prestazioni PHP. Tuttavia, non sempre questo si verifica.

Per visualizzare un messaggio di errore PHP sulla pagina web

Selezionare Abilita display_errors per visualizzare il messaggio di errore PHP se si desidera che i messaggio di errore PHP vengano visualizzati sulla pagina web.

Per personalizzare open_basedir PHP

  1. Selezionare Personalizza PHP open_basedir se si desidera personalizzare il proprio open_basedir.
  2. Digitare il percorso personalizzato nel campo di testo e fare clic su Applica. Se si desidera utilizzare le impostazioni predefinite di sistema, deselezionare Personalizza PHP open_basedir e fare clic su Applica.

Nota:

  • Usare i due punti (:) per separare i percorsi. Ad esempio /etc:/tmp:/usr/bin/php.
  • Per motivi di sicurezza, i percorsi per la cartella condivisa homes (ad esempio /volume1/homes) non possono essere aggiunti a open_basedir.

Per selezionare PHP extension

  1. Fare clic su Seleziona PHP extension per personalizzare la propria PHP extension.
  2. Nella finestra di dialogo Elenco PHP extension, selezionare l'estensione che di desidera attivare.
  3. è anche possibile fare clic su Predefinito in alto nella finestra per tornare alle impostazioni predefinite.
  4. Fare clic su OK per salvare le modifiche.
