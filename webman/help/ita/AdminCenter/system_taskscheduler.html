Utilità di pianificazione

Utilità di pianificazione consente di pianificare e eseguire i servizi o script in momenti predefiniti.

Attività supportate

È possibile creare le attività nei seguenti modi:

  • Script definito dall’utente: eseguire gli script o comandi definiti dall'utente.
  • Cestino: vuotare il cestino delle cartelle condivise specificate. Inoltre, le impostazioni di Politica di conservazione consentono di controllare quali file sono eliminati in modo permanente quando si svuota il Cestino. Ad esempio, i file possono essere eliminati o conservati in base a dimensione, tempo di cancellazione o formato file.
  • Controllo acustico: consente a DiskStation di emettere un bip.
  • Servizio: avviare o interrompere i servizi di rete, ad esempio il Servizio file Windows, FTP, Blocco automatico, ecc.

Gestione delle attività

La pagina principale dell'Utilità di pianificazione visualizza tutte le attività esistenti e consente di creare, modificare o eseguire le attività.

Per creare un'utilità di pianificazione:

Le attività pianificate sono attività che vengono eseguite in una specifica data, ora o intervallo.

  1. Andare su Pannello di controllo > Utilità di pianificazione e fare clic su Crea, quindi selezionare Attività pianificata.
  2. Selezionare il tipo desiderato di attività da pianificare: Script definito dall'utente, Cestino, Controllo promemoria acustico o Servizio.
  3. Quando viene visualizzata la finestra Crea attività, fare clic su Generale e modificare le seguenti impostazioni:
    • Impostazioni generali: Inserire un nome attività, selezionare un utente per cui eseguire l'attività oppure disattivare l'attività.
  4. Fare clic su Programmazione e modificare le seguenti impostazioni:
    • Data: specificare quando eseguire l'attività. È possibile pianificare l'esecuzione delle attività in giorni specifici della settimana (ad esempio ogni giorno, nei fine settimana o ogni domenica, ecc.) o in date specifiche (ad esempio 21/10/2015).
    • Ora: specificare il numero di volte che l'attività deve essere eseguita ogni giorno, oltre alla prima e ultima ora di esecuzione.
  5. Fare clic su Impostazioni attività per organizzare le impostazioni specifiche per attività (solo per determinate attività).
    • Per ricevere i dettagli di esecuzione dell'attività, selezionare Invia dettagli esecuzioni per e-mail.
    • Per ricevere la notifica solo in caso di anomalie, selezionare Invia dettagli esecuzione quando lo script termina in modo anomalo.
    • Per personalizzare il contenuto dell'attività, inserire lo script in Script definito dall'utente.
    • Se è stato selezionato Cestino o Servizio nel passaggio 2, è possibile stabilire come e quando svuotare i Cestini oppure quale servizio avviare/arrestare.
  6. Fare clic su OK per salvare le impostazioni.

Per creare un'attività azionata:

Le attività azionate sono attività eseguite prima/dopo il completamento di eventi specifici.

  1. Andare su Pannello di controllo > Utilità di pianificazione e fare clic su Crea, quindi selezionare Attività azionata.
  2. Selezionare Script definito dall’utente.
  3. Quando viene visualizzata la finestra Crea attività, fare clic su Generale e modificare le seguenti impostazioni:
    • Impostazioni generali: Inserire un nome attività, selezionare un utente per cui eseguire l'attività, selezionare un evento/attività preliminare che aziona l'attività e attivare o disattivare l'attività.
  4. Fare clic su Impostazioni attività per organizzare le impostazioni specifiche per attività (solo per determinate attività).
    • Per ricevere i dettagli di esecuzione dell'attività, selezionare Invia dettagli esecuzioni per e-mail.
    • Per ricevere la notifica solo in caso di anomalie, selezionare Invia dettagli esecuzione quando lo script termina in modo anomalo.
    • Per personalizzare il contenuto dell'attività, inserire lo script in Script definito dall'utente.
  5. Fare clic su OK per salvare le impostazioni.

Nota:

  • Quando viene creata un'attività azionata, potrebbe essere necessario specificarne le attività preliminari. L'attività creata verrà eseguita solo dopo l'esecuzione dell'attività preliminare.
  • Dopo avere abilitato Invia dettagli esecuzione quando lo script termina in modo anomalo, il sistema non fornirà i dettagli se lo script finisce con 0.

Per modificare le impostazioni dell'attività:

  1. Selezionare un'attività esistente.
  2. Fare clic su Modifica.
  3. Viene visualizzata una finestra dove è possibile modificare l'attività.

Per eliminare un'attività:

  1. Selezionare un'attività esistente.
  2. Fare clic su Azione > Elimina.

Per abilitare o disabilitare l'attività:

  1. Nella pagina principale di Utilità di pianificazione, selezionare o deselezionare le caselle di controllo appropriate in Abilitato per abilitare o disabilitare l'attività.
  2. Fare clic su Salva.

Per eseguire manualmente l'attività:

  1. Selezionare una o più attività esistenti.
  2. Fare clic su Esegui.

Limitazioni dell'Utilità di pianificazione:

  • I nomi delle attività possono includere 512 caratteri visualizzabili.
  • Numero massimo di attività: 1024.
  • Lunghezza massima del percorso script o linea di comando: 4095.
  • La durata bip può essere impostata da 1 a 600 secondi.
