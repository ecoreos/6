Avanzate

È possibile modificare le seguenti impostazioni avanzate in Pannello di controllo > Sicurezza > Avanzate per le sessioni di ricerca DSM:

Compressione HTTP

È possibile abilitare Compressione HTTP per DiskStation. Riducendo la larghezza di banda per il traffico di rete, è possibile incrementare la velocità di caricamento della pagina Web.

Per abilitare/disabilitare la compressione HTTP:

  • Selezionare/deselezionare la Compress HTTP su Pannello di controllo > Sicurezza > Avanzate.

Profilo TLS/SSL

È possibile scegliere il livello di sicurezza delle connessioni crittate HTTPS. Fornisce 3 livelli come segue:

  • Compatibilità moderna - Compatibile solo con i più recenti browser e dispositivi mobile.
  • Compatibilità intermedia - Tra il livello di sicurezza massimo e minimo. È il valore predefinito di sistema.
  • Compatibilità precedente obsoleta - Non certo, ma supporta la maggior parte dei browser e dei dispositivi mobile.
