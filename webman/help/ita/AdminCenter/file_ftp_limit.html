Limitazioni di connessione

Consenti soltanto connessione SSL/TLS

Quando questa funzione è attiva, il server accetterà soltanto richieste di connessione SSL o TLS dai client FTP.

Limitare le connessioni per IP

Quando questa funzione è attiva, il server limiterà il numero massimo di connessioni per IP. Attivare questa funzione e Restrizione di banda FTP per avere una gestione efficace dell'ampiezza di banda.

Attivare la restrizione di banda per FTP

Attivare questa funzione per limitare l'ampiezza di banda della singola connessione FTP.
