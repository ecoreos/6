Generale

Opzioni di rete generiche come gateway predefinito, server DNS, proxy e tunneling IPv6 sono disponibili su Pannello di controllo > Rete > Generale.

Rinominare il dispositivo DiskStation

Per rinominare il dispositivo DiskStation, inserire il nuovo nome del server e fare clic su Applica.

Il nome del server non fa distinzione tra maiuscole e minuscole e può contenere da 1 a 15 caratteri, che possono essere lettere, numeri, trattino basso o segni meno. Il primo carattere del nome deve essere una lettera.

Specificare gateway predefinito

Basta inserire l'indirizzo IP del gateway predefinito e fare clic su Applica per salvare le impostazioni.

Specificare indirizzo IP server DNS

È possibile scegliere di configurare manualmente l'indirizzo IP del server DNS. In questo caso, è necessario inserire l'indirizzo IP del server DNS preferito. È possibile inserire anche un indirizzo IP di un server DNS alternativo da utilizzare per il backup in caso di problemi con il server DNS preferito. Una volta configurato l'indirizzo IP del server DNS, fare clic su Applica per salvare le impostazioni.

Nota.

I campi Server DNS e gateway predefinito verranno impostati automaticamente e non possono essere modificati nel caso vengano configurate automaticamente tutte le interfacce di rete o se si sta utilizzando la connessione PPPoE.

Collegamento tramite Server Proxy

Per consentire al dispositivo DiskStation di collegarsi tramite un server proxy, spuntare Connetti tramite server proxy, immettere l'Indirizzo e la Porta e quindi fare clic su Applica.

Nota.

  • Fare clic su Impostazioni avanzate per configurare il proxy HTTP, proxy HTTPS e le informazioni di autenticazione.
  • Come impostazione predefinita, le impostazioni proxy HTTPS sono le stesse del proxy HTTP.
  • Se Connetti tramite server proxy ha un segno di spunta, Aggiornamento DSM e Centro pacchetti insieme ad altri pacchetti si connetteranno a Internet tramite un proxy.
  • Se Bypassa server proxy per indirizzi locali, ha un segno di spunta, assicurarsi che il server DNS usato da DiskStation può risolvere le query per i dispositivi nella rete locale.

Abilitare Tunneling IPv6

Finché IPv6 soppianta completamente IPv4, il tunneling IPv6 è necessario per abilitare gli host con indirizzi IPv6 per raggiungere i dispositivi IPv4. Questa tecnica fornisce un modo per portare il traffico IPv6 nelle reti IPv4 esistenti.
Se non si è sicuri di come configurare IPv6 o si riscontrano problemi di connessione, contattare il proprio provider di servizi Internet.

Per abilitare IPv6 Tunnel:

  1. Mettere il segno di spunta sulla casella Abilita tunnel per abilitare il supporto di tunnel.
  2. In Indirizzo server, inserire il nome host o l'indirizzo IP del provider del servizio tunnel, ad esempio: anonymous.freenet6.net.
  3. Selezionare se si desidera la Connessione anonima o Connettere a un account esistente.

Nota.

Il proprio indirizzo IPv6 esterno potrebbe essere aggiornato automaticamente dal provider del servizio. Tuttavia, DiskStation non rimuoverà immediatamente il vecchio indirizzo IPv6.
