Generale

Su Pannello di controllo > Servizi file > FTP, è possibile attivare limite di velocità, accesso per trasferimento file, impostazioni di sicurezza e altre impostazioni avanzate nella sezione Generale.

Abilitazione accesso per trasferimento file FTP

Fare clic su Impostazioni avanzate per abilitare questa funzione e registrare i registri di file scaricati o caricati.

Nota:

L'abilitazione dell'accesso per il trasferimento di file FTP influenzerà le prestazioni del sistema.

Modificare le directory root dell'utente

Fare clic su Impostazioni avanzate e mettere il segno di spunta su Modifica le directory root dell'utente. In questo modo, gli utenti selezionati possono accedere solo alle cartelle specificate quando accedono a DSM via FTP.

Per creare una regola di modifica della directory root:

  1. In Impostazioni avanzate, selezionare Seleziona utente.
  2. Fare clic su Aggiungi.
  3. In Utente o gruppo, selezionare a chi questa regola sarà applicata.
  4. Selezionare quale directory root sarà disponibile quando l'utente effettua l'accesso. Scegliere tra:
    • Sede utente: Gli utenti arrivano alla rispettiva home directory dopo l'accesso.
    • Altra directory: Scegliere una cartella condivisa come home directory e fare clic su Seleziona. I permessi cartella per l'utente o il gruppo devono essere selezionati prima. Un utente o gruppo con permessi cartella insufficienti non potrà effettuare l'accesso.
  5. Fare clic su OK.
  6. Dopo avere aggiunto tutte le regole, fare clic su Applica per salvare le regole.

Nota:

  • La priorità delle regole viene stabilita in base alla posizione nell'elenco.
  • Trascinare per riordinare le regole nell'elenco.

Abilitazione FTP anonimo

Fare clic su Impostazioni avanzate, e spuntare l'opzione per consentire a tutti di accedere al servizio FTP del dispositivo DiskStation (con account anonimo o ftp).

Modifica root anonime

Fare clic su Impostazioni avanzate e mettere il segno di spunta sull'opzione per modificare la directory di accesso dell'utente FTP anonimo. Selezionare una cartella condivisa dall'elenco a discesa.

Applica autorizzazioni UNIX predefinite

Fare clic su Impostazioni avanzate e mettere il segno di spunta sull'opzione per applicare le autorizzazioni UNIX predefinite quando si caricano o creano file e cartelle. Le autorizzazioni applicate sono le stesse applicate dal comando UNIX umask. Quando questa opzione è abilitata, le autorizzazioni UNIX sono 644 per i file e 755 per le cartelle. Quando questa opzione è disabilitata, le autorizzazioni UNIX sono 666 per i file e 777 per le cartelle. Il valore unmask predefinito è 022.

Nota:

  • Per le cartelle condivise abilitate per Windows ACL (tutte le condivisioni escluse le foto e le condivisioni sulle unità esterne), eseguire il comando chmod sul client Linux o FTP per cambiare la cartella e i tipi di permesso dei file da Windows ACL a UNIX.
  • L'abilitazione di questa funzione può causare problemi di autorizzazioni non coerenti tra diversi protocolli. Per evitare incoerenze, si consiglia di lasciare questa opzione disabilitata.

Abilitazione limite velocità

Abilitare un limite di velocità che consente di controllare la larghezza di banda massima utilizzabile per utenti e gruppi specifici durante il trasferimento di dati tramite FTP.

Per abilitare il limite di velocità:

  1. Spuntare la casella contrassegnata Abilita limite velocità o Abilita il limite di velocità in base a una pianificazione se si desidera implementare un limite di velocità solo in orario specifico. Questa è un'impostazione del server e annullerà le impostazioni dei singoli utenti e gruppi. Scegliendo Nessun limite di velocità si disabilita l'impostazione del limite di velocità di tutti gli utenti e gruppi.
  2. Fare clic su Impostazioni limite velocità per modificare le impostazioni per i singoli utenti o gruppi. Per ogni utente e gruppo, è possibile selezionare una delle seguenti:
    • Applica impostazioni gruppo: le impostazioni del limite di velocità saranno le stesse del gruppo a cui l'utente appartiene.
    • Nessuno: nessun limite di velocità sarà applicato quando i membri del gruppo trasferiscono file con il servizio.
    • Costante: Un limite di velocità costante fisso sarà applicato all'utente o membri del gruppo. I limiti di velocità di upload e download possono essere specificati nei campi a sinistra.
    • Personalizzato: Due diversi limiti di velocità possono essere specificati e applicati all'utente o ai membri del gruppo in base alla programmazione. Fare clic sul pulsante Personalizza per modificare le impostazioni del limite di velocità e impostare una programmazione.
  3. Fare clic su Applica.

Nota:

  • Per ulteriori informazioni sulle dinamiche dei limiti di velocità gruppo, vedere qui.
  • Quando viene impostato il limite di velocità per un utente, la sezione Risultato visualizza il limite di velocità attuale dell'utente in base alle sue impostazioni individuali e di gruppo. Ad esempio, se l'utente "Achille" ha un limite di download individuale di 100 KB/s, ma appartiene a un gruppo utente con un limite di download di 500 KB/s, la sezione Risultati mostrerà "0/500."
