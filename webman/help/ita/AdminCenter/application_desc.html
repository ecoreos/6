Applicazioni

Questa sezione consente di gestire le impostazioni generali relative alle applicazioni progettate da Synology, come privilegi accesso utenti e impostazioni di connessione per determinate applicazioni, indicizzazione multimediale per l'utilizzo con applicazioni multimediali. Inoltre, ricchi strumenti di servizio web e opzioni consentono di ospitare facilmente un sito web sul proprio DiskStation.
