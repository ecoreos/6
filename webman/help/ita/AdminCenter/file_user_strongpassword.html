Impostazioni password

Fare clic su Impostazioni password per visualizzare e modificare le impostazioni descritte di seguito.

Ripristino password dimenticate

Questa sezione include un'opzione contrassegnata Consentire agli utenti non amministratore di ripristinare le password dimenticate via e-mail. se questa opzione è abilitata, un link contrassegnato Dimenticato la password? apparirà sulla pagine di accesso del DSM. Se un utente dimentica la password, egli può fare clic su questo link e immettere il suo nome utente. In questo caso, il sistema invierà un messaggio all'utente contenente un link per ripristinare la sua password dimenticata.

Nota:

  • Prima di abilitare questa opzione, assicurarsi di avere già abilitato le notifiche e-mail SMTP. Per far ciò, andare su Pannello di controllo > Notifiche > Generale. Se le notifiche non sono abilitate, il sistema non sarà in grado di inviare messaggi agli utenti che desiderano ripristinare le password.
  • Per riceveree i messaggi dal sistema, gli utenti devono fornire un indirizzo e-mail nella sezione informazioni utente dei loro account.
  • Gli utenti che appartengono al gruppo amministratori non possono ripristinare le password con questa opzione.
  • Gli utenti del dominio non possono ripristinare le password con questa opzione.
  • Gli utenti LDAP possono ripristinare le password con questa funzione, finché DiskStation funge da directory server LDAP.

Resistenza password

È possibile impostare le regole di forza della password per gli utenti su DiskStation.

Nota:

Le regole di forza della password si applicano solo a nuove password. Ad esempio, le regole di forza della nuova password si applicano solo quando si crea un nuovo account utente o quando un utente esistente modifica la sua password. Le password esistenti e quelle relative ad account utenti importati sono escluse dalle regole della nuova password.

Abilitare le regole di forza della password:

  1. Spuntare la casella Applicare le regole di forza della password per attivare le limitazioni della password.
  2. Selezionare più di una limitazione della password dalle regole di seguito:
    • Escludere nome e descrizione dell'utente dalla password: la password non deve contenere il nome utente o la descrizione dell'utente. Sono esclusi i caratteri codificati UTF-8.
    • Consenti maiuscole e minuscole: nella password è possibile utilizzare sia lettere maiuscole che minuscole.
    • Carattere numerico incluso: la password deve contenere almeno un carattere numerico (0~9).
    • Carattere speciale incluso: la password deve contenere almeno un carattere speciale ASCII (p.e., ~, `, !, @, #, $, %, ^, &, *, (, ), -, _, =, +, [, {, ], }, \, |, ;, :, ', ", <, >, /, ?).
    • Lunghezza minima della password: è necessario specificare almeno un valore per la password nel campo di testo. La lunghezza deve essere un numero compreso tra 6 e 127.
  3. Fare clic su Applica per salvare le impostazioni.
