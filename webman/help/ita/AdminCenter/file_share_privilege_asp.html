Gestisci permessi cartella condivisa avanzati

La pagina Pannello di controllo > Cartella condivisa > Modifica > Avanzate include le seguenti opzioni al fine di perfezionare ulteriormente i permessi di una cartella condivisa.

Permessi condivisi avanzati

I permessi condivisi avanzati offrono un livello aggiuntivo di controllo per gestire i permessi di accesso delle cartelle condivise. Quando abilitata, utenti e gruppi possono visualizzare o modificare il contenuto di una cartella condivisa solo se l'utente o il gruppo hanno ottenuto permessi di condivisione avanzati e permessi Windows ACL (in Cartella condivisa > Modifica > Permessi).

Per abilitare i permessi condivisi avanzati:

  1. Spuntare la casella Abilita permessi condivisi avanzati.
  2. Fare clic su Permessi condivisi avanzati per modificare i permessi condivisi avanzati per la cartella condivisa.

Nota:

  • I permessi condivisi avanzati sono applicati quando un utente accede a una cartella condivisa mediante i seguenti servizi di file: Windows File Sharing, Apple File Sharing, File Station, FTP e WebDAV.

Impostazioni avanzate

Se necessario, è possibile porre ulteriori restrizioni agli utenti quando accedono alla cartella condivisa via File Station, FTP o WebDAV.

  • Disabilita esplorazione directory: abilitando questa opzione si restringe agli utenti la visualizzazione del contenuto della cartella condivisa.
  • Disabilita modifica di file esistenti: abilitando questa opzione si restringe agli utenti il movimenti, la cancellazione o la modifica di file nella cartella condivisa. Si noti che gli utenti saranno ancora in grado di visualizzarer, scaricare/caricare, copiare o decomprimere il contenuto della cartella condivisa.
  • Disabilita download di file: abilitando questa opzione si restringe agli utenti il download del contenuto della cartella condivisa.

Nota:

Le suddette restrizioni non si applicano agli utenti che appartengono al gruppo amministratori
