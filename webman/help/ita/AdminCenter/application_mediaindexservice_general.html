Servizio di indicizzazione multimediale

Il servizio di indicizzazione multimediale consente di eseguire la scansione automatica di file come foto, musica e video archiviati su DiskStation, compilandoli nella biblioteca multimediale per essere indicizzati dalle applicazioni multimediali. Le applicazioni disponibili in Centro pacchetti comprendono Server Media, Video Station, Audio Station, Photo Station e iTunes Server. Una volta installate sulla DiskStation, queste saranno visualizzate nel campo Applicazione.

Per reindicizzare i file multimediali:

Fare clic su Ri-indicizzare per ricostruire la struttura dell'indice di tutti i file multimediali nelle cartelle indicizzate, sia le cartelle predefinite sia quelle create dall'utente. Il processo verrà eseguito in background.

Nota:

I file multimediali aggiunti alle cartelle indicizzate utilizzando il protocollo NFS non verranno scansionati automaticamente. Sarà necessario eseguire nuovamente l'indicizzazione manualmente facendo clic su Ri-indicizzare.
