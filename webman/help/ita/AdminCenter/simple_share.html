Cartella condivisa

Le cartelle condivise sono directory di base dove è possibile salvare i file e le cartelle sulla DiskStation. Prima di salvare i file su DiskStation, è necessario creare almeno una cartella condivisa in DSM. I dati salvati sulle cartelle condivise possono essere privati o condivisi con utenti specifici o gruppi di utenti in base alle autorizzazioni di accesso personalizzabili.

Di seguito un elenco delle cartelle condivise predefinite che sono create automaticamente quando alcune applicazioni, servizi o pacchetti sono installati o abilitati.

Nome Descrizione
public La cartella public verrà creata automaticamente dopo la prima configurazione di DiskStation.*
web La cartella web è la posizione di archiviazione predefinita per i contenuti di un sito web ospitato su DiskStation. Essa è creata automaticamente quando la Web Station è abilitata.
photo La cartella photo è usata per archiviare foto e video che si desiderano condividere su Photo Station. Verrà creata automaticamente all'attivazione di Photo Station o del Server media DLNA/UPnP.
music La cartella music è usata per archiviare la musica che si desidera ascoltare con Audio Station. Verrà creata automaticamente all'attivazione di Audio Station o del Server media DLNA/UPnP.
video La cartella video è usata per archiviare i file video che si desidera sfogliare attraverso i dispositivi DLNA/UPnP. Verrà creata automaticamente all'attivazione di Video Station o del Server media DLNA/UPnP.
surveillance La cartella surveillance è usata per archiviare le registrazioni di Surveillance Station. Essa è creata automaticamente quando la Surveillance Station è abilitata. È una cartella di sola lettura alla quale può accedere solo l'amministratore di sistema.
home Le cartelle home forniscono spazi privati per ciascun utente per archiviare i dati a cui solo tale utente può accedere. Essa è creata automaticamente quando la funzione Sede utente è abilitata.
homes La cartella homes contiene le cartelle home di tutti gli utenti. Essa è creata automaticamente quando la funzione Sede utente è abilitata. Solo l'amministratore di sistema può accedere e gestire tutte le cartelle home degli utenti.
NetBackup La cartella NetBackup è creata automaticamente all'attivazione del Backup di rete.
usbshare La cartella usbshare[numero] viene creata automaticamente quando si connette un dispositivo esterno USB alla porta USB di DiskStation.*
satashare La cartella satashare viene creata automaticamente quando si connette un dispositivo esterno eSATA alla porta eSATA di DiskStation.*

Nota:

  • È possibile solo modificare permessi cartella condivisa degli utenti in modalità semplice. Se si vogliono creare gruppi utente o gestire privilegi di accesso dei gruppi a cartelle condivise, andare su Pannello di controllo (Modalità avanzata) > Gruppo.

Per creare una cartella condivisa:

  1. Andare su Pannello di controllo (Modalità semplice) > Cartella condivisa e fare clic sull'icona + sul pannello di sinistra.
  2. Specificare un nome e selezionare un volume su cui creare la cartella condivisa.
  3. Spuntare Abilita cestino, se necessario. Se questa opzione è abilitata, quando i file nella cartella condivisa sono cancellati, essi sono spostati in una cartella chiamata #recycle. L'accesso al cestino può essere limitato agli utenti appartenenti al gruppo administrators.
  4. Selezionare o deselezionare le caselle appropriate per ciascun utente per personalizzare i permessi di accesso alla cartella condivisa:
    • Lettura/Scrittura: l'utente può accedere e apportare modifiche ai file e alle sottocartelle nella cartella condivisa.
    • Sola lettura: l'utente può accedere a file e sottocartelle nella cartella condivisa.
    • Nessun accesso: l'utente non può accedere a file o sottocartelle nella cartella condivisa.
  5. Fare clic su Applica per salvare le impostazioni.

Nota:

  • I nomi della cartella condivisa non fanno la distinzione tra lettere minuscole e maiuscole e possono comprendere tra 1-32 caratteri. I nomi non possono contenere i seguenti caratteri:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • I seguenti nomi sono riservati per il sistema:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • Il primo carattere di una cartella condivisa non può essere un segno meno o uno spazio; l'ultimo carattere non può essere uno spazio.
  • quando si incontrano conflitti di permessi, la priorità di permessi è la seguente: Nessun accesso > Lettura/Scrittura > Sola lettura.
  • Quando si crea una nuova cartella condivisa, se i permessi per gli utenti che appartengono al gruppo administrators sono impostati su Nessun accesso, questi utenti potranno vedere solo la cartella condivisa in Pannello di controllo > Cartella condivisa.
  • È possibile configurare le impostazioni dei privilegi della cartella Cestino in File Station facendo clic sulla cartella #recycle. Scegliere Priorità e fare clic sulla scheda Autorizzazioni.

Per modificare una cartella condivisa:

  1. Andare su Pannello di controllo (Modalità semplice) > Cartella condivisa.
  2. Selezionare la cartella condivisa desiderata sul pannello di sinistra.
  3. Modificare le informazioni generali di una cartella condivisa, come nome, località e permessi utente.
  4. Se Abilita cestino è stato già selezionato, è possibile fare clic su Svuota cestino per rimuovere tutti i file cancellati dal cestino in modo permanente.
  5. Fare clic su Applica per salvare le impostazioni.

Per eliminare una cartella condivisa:

  1. Andare su Pannello di controllo (Modalità semplice) > Cartella condivisa.
  2. Selezionare la cartella condivisa desiderata sul pannello di sinistra.
  3. Fare clic sull'icona Cestino sul pannello di sinistra per cancellare la cartella condivisa selezionata.

* Disponibile solo su alcuni modelli.
