Impostazioni di sicurezza

Trasforma le root degli utenti selezionati in sede utente

Gli utenti qui selezionati potranno accedere alle loro cartelle principali solo dopo avere effettuato l'accesso con Synology DiskStation tramite FTP.

Attivare FTP anonimo

Abilitare questa funzione per permettere a chiunque di accedere al servizio FTP di DiskStation (con account anonimo o ftp).

Per configurare i privilegi di utente anonimo:

Andare su Menu principale > Pannello di controllo > Cartella condivisa e fare clic su Impostazione privilegi.

Cambia root anonime

Abilitare questa funzione per cambiare la directory di accesso degli utenti FTP anonimi. Selezionare una cartella condivisa dall'elenco a discesa.
