DHCP Server

DiskStation può agire come server DHCP e assegnare indirizzi IP dinamici ai client DHCP all'interno della LAN.

Per abilitare il server DHCP:

  1. Andare su Pannello di controllo > DHCP Server > Interfaccia di rete.
  2. Selezionare un'interfaccia di rete (es. LAN 1) a cui assegnare indirizzi IP e fare clic su Modifica.
  3. Nella finestra a comparsa, fare clic sulla scheda DHCP Server e selezionare Abilita DHCP Server.
  4. Specificare le impostazioni DHCP seguenti:
    • Durata lease indirizzo (minuti)
    • DNS principale
    • DNS secondario (opzionale)
    • Nome Dominio (opzionale)
    • Abilita scoperta automatica Web proxy (opzionale)
  5. Specificare almeno una sottorete. Per la configurazione della sottorete, fare clic su Aggiungi per specificare le informazioni seguenti:
    • Indirizzo IP iniziale
    • Indirizzo IP finale
    • Subnetmask
    • Gateway
  6. Fare clic su Crea per mostrare la sottorete creata nell'elenco.
  7. Dopo avere creato un numero sufficiente di sottoreti, abilitare almeno una sottorete.
  8. Fare clic su OK.

Nota:

  • è possibile creare tutte le sottoreti desiderati.
  • Se coordinato con un server relay DHCP, DiskStation come server DHCP può assegnare indirizzi IP ai dispositivi in un'altra sottorete.

Client DHCP

È possibile visualizzare un elenco dei client DHCP e le relative configurazioni di rete (ad esempio, indirizzo MAC, indirizzo IP, nome host e tempo rimasto prima della scadenza del lease DHCP, nonché stato di prenotazione).

  1. Andare su Pannello di controllo > DHCP Server > Interfaccia di rete e selezionare l'interfaccia desiderata.
  2. Fare clic su Modifica > Client DHCP per trovare client e dettagli.

Per gestire indirizzi IP per client DHCP:

È possibile consentire che un client riceva sempre lo stesso indirizzo IP durante il rinnovo del lease DHCP e gestire questi clienti.

  • Per aggiungere un client DHCP: Fare clic su Aggiungi per assegnare gli indirizzi IP al dispositivo client.
  • Per prenotare l'indirizzo IP: Fare clic sull'icona del blocco per prenotare l'indirizzo IP per il dispositivo corrente.
  • Per annullare la prenotazione: Fare clic sull'icona del blocco per consentire l'uso dell'indirizzo IP corrente da parte di un altro dispositivo dopo la scadenza del tempo di lease.

Nota:

se un dispositivo presenta indirizzi IPv4 e IPv6, è possibile prenotar un indirizzo IPv4 e un indirizzo IPv6 per il dispositivo.

Per cercare client DHCP:

  1. Fare clic sull'icona dell'imbuto nell'angolo in alto a destra.
  2. Selezionare uno dei criteri di ricerca di seguito:
    • Tutto: elencare tutti i client DHCP e tutte le prenotazioni DHCP.
    • Client DHCP: elencare tutti i client DHCP con indirizzi IP prenotati o non prenotati.
    • Prenotazioni DHCP: elencare tutti gli indirizzi IP prenotati. Un indirizzo IP prenotato potrebbe non essere ancora associato ad alcun dispositivo.

Opzioni DHCP

Dopo avere specificato le opzioni DHCP, DiskStation, come server DHCP, può inviare le informazioni correlate ai client DHCP in una determinata sottorete.

Per specificare le opzioni DHCP per una sottorete:

  1. Andare su Pannello di controllo > DHCP Server > Interfaccia di rete.
  2. Selezionare un'interfaccia di rete esistente e fare clic su Modifica.
  3. Andare su DHCP Server e passare all'Elenco subnet.
  4. Eseguire una delle azioni seguenti:
    • Per specificare le opzioni DHCP per una sottorete esistente: Selezionare l'interfaccia desiderata e fare clic su Modifica.
    • Per creare una sottorete e specificarne le opzioni DHCP. Fare clic su Aggiungi e specificare le impostazioni della sottorete.
  5. Selezionare le opzioni DHCP desiderate; per specificare i valori corrispondenti fare clic sui campi vuoti per la modifica.
  6. Fare clic su OK.

Fornitore

Client DHCP di differenti vendor potrebbero richiedere altre informazioni DHCP. Attraverso la configurazione di opzioni DHCP per i vendor, DiskStation può fornire questi dettagli ai client.

Per conoscere le opzioni DHCP supportate da ogni vendor, contattare gli specifici vendor per ulteriori dettagli.

Per aggiungere vendor:

  1. Andare su Pannello di controllo > DHCP Server > Vendor.
  2. Fare clic su Crea per specificare iò vendor e le relative opzioni DHCP:
    • Nome: specificare il nome del vendor.
    • Codice: specificare il codice numerico dell'opzione DHCP.
    • Valore: specificare il valore dell'opzione.
    • +: aggiungere un'opzione DHCP.
    • -: eliminare un'opzione DHCP.
  3. Fare clic su Crea per terminare la configurazione.

Per modificare le opzioni DHCP per i vendor:

  1. Andare su Pannello di controllo > DHCP Server > Vendor.
  2. Selezionare il vendor desiderato e fare clic su Modifica pere gestirne le opzioni DHCP.
  3. Fare clic su OK.

PXE

DiskStation può funzionare da server DHCP per implementare la configurazione di PXE (Pre-Boot Execution Environment). PXE è un ambiente di rete che consente ai dispositivi client di avviarsi utilizzando un'interfaccia di rete al posto di un dispositivo di archiviazione dei dati (ad esempio i dischi rigidi).

Quando PXE viene implementato, un caricatore di avvio può essere archiviato su DiskStation o un server TFTP. mentre i dispositivi client possano essere avviati tramite la rete utilizzando il caricatore di avvio.

Per usare DiskStation come server del caricatore di avvio:

  1. Verificare di avere selezionato Abilita servizio TFTP (su Pannello di controllo > Servizi file > TFTP).
  2. Selezionare una cartella che ospiti il file del programma di caricamento di avvio, come la cartella radice TFTP.
  3. Verificare che almeno una sottorete DHCP sia abilitata (su Pannello di controllo > DHCP Server > Interfaccia di rete).
  4. Andare su Pannello di controllo > DHCP Server > PXE.
  5. Selezionare Abilita PXE (pre-avvio ambiente di esecuzione).
  6. Selezionare Server TFTP locale e selezionare il file del caricatore di avvio.
  7. Fare clic su Applica per salvare le impostazioni.

Per usare il server TFTP come server del caricatore di avvio:

Utilizzando un server TFTP remoto come server caricatore di avvio, è possibile avviare i client senza modificare l'infrastruttura di rete corrente.

  1. Verificare che almeno una sottorete DHCP sia abilitata (su Pannello di controllo > DHCP Server > Interfaccia di rete).
  2. Andare su Pannello di controllo > DHCP Server > PXE.
  3. Selezionare Abilita PXE (pre-avvio ambiente di esecuzione).
  4. Selezionare Server TFTP remoto e specificare l'indirizzo IP del server e il percorso del file del caricatore di avvio.
  5. Fare clic su Applica per salvare le impostazioni.
