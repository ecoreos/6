Sicurezza

È possibile modificare le seguenti impostazioni di sicurezza su Pannello di controllo > Sicurezza > Sicurezza per le sessioni di navigazione di DSM:

  • Timer di disconnessione (minuti): verrà effettuata la disconnessione automatica dal DSM se sono inattivi per il periodo di tempo specificato qui. Inserire un valore compreso tra 1 e 65535.
  • Migliora la compatibilità del browser ignorando il controllo IP: se si accede a DiskStation tramite proxy HTTP e si verificano disconnessioni casuali, è possibile abilitare questa opzione per ignorare il controllo IP.
  • Migliora la protezione contro attacchi caratterizzati da false richieste di accesso al sito: questa opzione migliora la protezione del sistema contro attacchi con script di accesso al sito. Questa opzione avrà effetto all'accesso successivo a DSM.
  • Migliorare la sicurezza con l'intestazione HTTP Content Security Policy (CSP): uesta opzione migliora la protezione del sistema contro attacchi con script di accesso al sito (XSS) consentendo solo i dati di fonti affidabili e limitando l'esecuzione di script in linea.
  • Non consentire al DSM di integrarsi con iFrame: È possibile abilitare questa opzione per limitare gli altri siti Web dal nascondere DSM da altre pagine Web con iFrame, quindi prevenire alcuni tipi di attacchi da siti Web malevoli. Siti Web specifici possono nascondere il DSM con iFrame facendo clic sul pulsante Siti Web consentiti.
