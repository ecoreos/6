UPS

Un UPS (gruppo di continuità) è un dispositivo di alimentazione di backup che consente a DiskStation di continuare a funzionare per un breve periodo in caso di guasti elettrici. Questa funzione in Pannello di controllo > Hardware e Alimentazione > UPS aiuta a prevenire la perdita di dati fornendo a DiskStation tempo a sufficienza per salvare i dati e smontare i volumi prima di rimanere senza alimentazione e spegnersi.

Modalità sicura

Quando DiskStation entra in modalità sicura, tutti i servizi vengono interrotti e i volumi vengono smontati modo da evitare la perdita dei dati ed eseguire lo spegnimento sicuro quando si riduce l'alimentazione fornita dall'UPS. Per impostazione predefinita, il sistema entra in Modalità sicura quando si riduce l'alimentazione dell'UPS. È anche possibile specificare quanto tempo prima il DiskStation deve accedere alla Modalità sicura in caso di assenza di corrente. Tuttavia, se la batteria dell'UPS si scarica prima del periodo specificato, il sistema entra subito in Modalità sicura.

Supporto UPS locale

Per informazioni relative all'installazione hardware, consultare il manuale utente dell'UPS. Una volta completata l'installazione e la DiskStation è stata collegata all'UPS, è possibile attenersi alle procedure di seguito per abilitare il supporto UPS.

Per abilitare il supporto UPS locale:

  1. Collegare l'UPS alla porta USB della DiskStation.
  2. Nella scheda UPS, selezionare la casella di controllo Attiva il supporto UPS.
  3. Selezionare quando DiskStation deve accedere alla Modalità sicura in caso di assenza di alimentazione.
  4. Scegliere se inviare o meno un segnale di chiusura all'UPS quando il sistema entra in Modalità sicura. Molti UPS si accendono automaticamente quando viene ripristinata l'alimentazione.
  5. Fare clic su Applica.

Supporto UPS SNMP

DiskStation supporta inoltre il collegamento agli UPS SNMP (Simple Network Management Protocol). Per informazioni relative all'installazione hardware, consultare il manuale utente dell'UPS SNMP. Una volta collegato DiskStation al dispositivo e alla stessa rete, è possibile attenersi alle procedure di seguito per abilitare il supporto UPS.

Per abilitare il supporto UPS SNMP:

  1. Nella scheda UPS, selezionare la casella di controllo Attiva il supporto UPS.
  2. Selezionare UPS SNMP nel menu a discesa Tipo UPS in rete.
  3. Nel campo Indirizzo IP UPS SNMP, inserire l'indirizzo IP del dispositivo UPS SNMP.
  4. Selezionare quando DiskStation deve accedere alla Modalità sicura in caso di assenza di alimentazione.
  5. Inserire la stringa della comunità per il dispositivo UPS SNMP nel campo Comunità SNMP.
  6. Fare clic su Applica.

Nota:

  • Si consiglia vivamente di utilizzare SNMP UPS per le distribuzioni SHA (Synology High Availability).
  • DiskStation supporta i dispositivi UPS SNMPv1 e SNMPv2c.

Supporto UPS di rete

Se sono presenti due prodotti Synology che utilizzano un UPS con una sola porta USB, solo un DiskStation può ottenere le informazioni relative allo stato dell'UPS. In questo caso, DiskStation può agire da "server UPS di rete" e connettersi all'UPS tramite USB o rete (in relazione al dispositivo). Il server UPS di rete ottiene le informazioni UPS e le trasmette a DiskStation "client".

Per abilitare il supporto UPS in rete:

  1. Accedere a DiskStation collegato all'UPS (tramite USB o rete). Se il supporto UPS locale ancora non è stato abilitato su questa DiskStation, consultare la sezione precedente.
  2. Nella scheda UPS, selezionare la casella Abilita il server di rete UPS per consentire a DiskStation di trasmettere le informazioni dell'UPS a DiskStation client.
  3. Fare clic sul pulsante Dispositivi DiskStation consentiti.
  4. Nella finestra visualizzata, inserire gli indirizzi IP del DiskStation che sta utilizzando l'UPS, ma non è connesso tramite USB o rete. Fare clic su OK.
  5. Fare clic su Applica per salvare le impostazioni.

A questo punto il server UPS di rete può ottenere le informazioni dell'UPS e trasmetterle a DiskStation utilizzando l'UPS. Consultare le istruzioni di seguito per abilitare il supporto UPS sull'altro DiskStation.

Per abilitare il supporto UPS su DiskStation client:

  1. Accedere all'altro DiskStation che sta utilizzando l'UPS, ma non tramite USB o rete.
  2. Nella scheda UPS, selezionare la casella di controllo Attiva il supporto UPS.
  3. Selezionare Server Gruppo di continuità Synology nel menu a discesa Tipo Gruppo di continuità di rete.
  4. Inserire l'indirizzo IP del server UPS di rete (ad esempio la DiskStation collegata all'UPS) nel campo IP del server di rete UPS.
  5. Selezionare quando DiskStation deve accedere alla Modalità sicura in caso di assenza di alimentazione. Per rendere più semplici le impostazioni della Modalità sicura, è possibile selezionare Come il server.
  6. Fare clic su Applica per salvare le impostazioni.

Requisiti dell'UPS di rete:

  • Per trasmettere le informazioni dell'UPS, ciascun DiskStation deve essere collegato alla stessa rete.
  • Il dispositivo di rete (ad esempio switch o hub) al quale è collegato ciascun DiskStation deve essere collegato allo stesso UPS. In caso contrario DiskStations non sarà in grado di trasmettere le informazioni dell'UPS in caso di assenza di corrente.
