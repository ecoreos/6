Ora

È possibile controllare l'ora attuale, impostare la data e l'ora del server manualmente o far sì che si impostino automaticamente utilizzando un server dell'ora di rete.

È possibile controllare l'ora corrente e impostare la data e l'ora del server.

Per impostare il fuso orario:

Scegliere l'opzione Fuso orario dal menu.

Per impostare data e ora manualmente:

scegliere Manualmente e selezionare la data e l'ora.

Per sincronizzare la data e l'ora con il server NTP:

  1. Scegliere Sincronizza con server NTP, quindi immettere l'indirizzo del server orario di rete o scegliere un server dal menu a discesa.
  2. Fare clic su Aggiorna adesso per accertarsi che DiskStation si sincronizzi correttamente l'ora con il server.
  3. Fare clic su Applica.

Nota: per sincronizzare la data e l'ora con un server orario di rete, assicurati che sia stato inserito un IP valido del server DNS su Menu principale > Pannello di controllo > Rete e che DiskStation sia collegata ad Internet.

Sul server Network Time Protocol (NTP)

Lo scopo principale del Network Time Protocol (NTP) è la sincronizzazione degli orologi del sistema del computer sulla rete di dati.

Di seguito sono elencati alcuni server NTP consigliati:

  • time.apple.com
  • time.asia.apple.com
  • time.euro.apple.com
  • time.nist.gov

Per maggiori informazioni sul server NTP, visitare http://www.eecis.udel.edu/~mills/ntp/servers.html
