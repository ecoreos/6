Sospensione

È possibile specificare un periodo di tempo in cui i dischi rigidi restano inattivi prima di accedere alla modalità di sospensione HDD. Durante la sospensione HDD, i dischi rigidi smettono di ruotare, riducendo il consumo energetico e prolungando la durata dei dischi rigidi stessi. È possibile specificare periodi di tempo diversi per i dischi rigidi interni e i dischi rigidi eSATA/USB esterni.

Nota.

  • La sospensione HDD termina al momento dell'accesso.
  • Solo alcuni dispositivi di archiviazione USB esterni supportano la sospensione HDD.

Perché la DiskStation non accede alla sospensione?

Notare che i seguenti servizi non consentono a DiskStation di accedere alla modalità di sospensione. Inoltre, il sistema esce dalla modalità di sospensione due minuti prima di qualsiasi attività pianificata.

Pacchetti:

  • Download Station: DiskStation non accede alla sospensione se Download Station sta eseguendo eMule, il download programmato RSS o attività attive.
  • Mail Station: DiskStation non accede alla sospensione se Mail Station sta recuperando informazioni da POP3.
  • Surveillance Station: DiskStation non accede alla sospensione se Surveillance Station è in registrazione.
  • Server media: DiskStation non accede alla sospensione se è abilitato il log DMA nelle impostazioni del Server media (andare su Menu principale > Server media > Server media > Diagnosi).
  • Directory Server
  • Webalizer
  • Server ® Media Logitech
  • Plex Media Server
  • Altri software o optware di terze parti come SABnzbd

Processi:

  • Miniature e indicizzazione: DiskStation non accede alla sospensione durante la generazione di miniature o la re-indicizzazione dei file multimediali dopo l'aggiornamento. Durante questo periodo, il processo “synoindexd” sarà indicato come “In esecuzione” (andare su Menu principale > Controllo risorse > Processo).
  • Local Master Browser: DiskStation non accede alla sospensione se è abilitato il Local Master Browser (andare su Pannello di controllo > Win/Mac/NFS > Servizio file Windows > Abilita Local Master Browser).
  • Windows Media Player: DiskStation non accede alla sospensione se il servizio di condivisione di rete di Windows Media Player è abilitato nella LAN.
  • DiskStation non accede alla sospensione se il numero di processi simultanei richiede lo scambio, la capacità della RAM è stata superata e gli HDD vengono usati temporaneamente per le operazioni di lettura/scrittura.

Altri

  • Dispositivi USB: DiskStation non può accedere alla sospensione sistema se dei dispositivi USB sono collegati all'unità.

Sospensione sistema

È possibile impostare DiskStation per accedere alla sospensione del sistema un minuto dopo che i dischi rigidi accedano alla sospensione. La sospensione del sistema riduce il consumo energetico di DiskStation.

Durante la modalità di sospensione del sistema:

  1. Il pulsante di accensione lampeggia (si accende/spegne ciclicamente in modo lento). Gli altri LED sono spenti.
  2. L'alimentazione per i dischi rigidi viene scollegata.
  3. La CPU si arresta.
  4. La ventola esterna si arresta.
  5. I LED sulle porte LAN si spengono.

Nota.

  • Qualsiasi pacchetto di trasmissione SMB/CIFS può evitare che il server entri in modalità Sospensione sistema, inclusi i browser file Windows utilizzati in rete. Per ignorare i pacchetti di trasmissione CIFS durante l'accesso alla modalità Sospensione sistema, selezionare Ignora trasmissione pacchetti da Esplora risorse. Una volta selezionata questa opzione, i pacchetti SMB/CIFS non possono essere usati per uscire dalla Sospensione sistema. Il collegamento di qualsiasi dispositivo USB evita anche al server di accedere alla Sospensione sistema.

Sospensione profonda

È possibile abilitare la sospensione profonda per le unità di espansione tramite l'interfaccia DiskStation accoppiata. È possibile configurare l'unità di espansione e il DiskStation accoppiato per essere inattivi in diversi periodi di tempo. Pertanto, le unità di espansione possono essere impostate per accedere alla Sospensione profonda mentre i DiskStation restano online.

In modalità sospensione profonda:

  • Tutti i LED dell'unità di espansione sono spenti.
  • L'alimentazione dei dischi rigidi dell'unità di espansione non viene erogata.
  • La ventola dell'unità di espansione si arresta.

Nota.

  • La Sospensione profonda termina al momento dell'accesso.
  • Per garantire un funzionamento normale durante l'uso della funzione Sospensione profonda sull'unità di espansione, è necessario spostare l'interruttore Auto/Manual sul pannello posteriore su Auto.

Spegnimento automatico

È possibile specificare la lunghezza del periodo della sospensione del disco interno prima di spegnere automaticamente DiskStation.

Nota.

  • Prima è necessario abilitare la sospensione dell'HDD interno.
  • È necessario prima abilitare il servizio WOL così da poter accendere in remoto DiskStation.
