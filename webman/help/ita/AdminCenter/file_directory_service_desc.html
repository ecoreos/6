Servizio Directory

DiskStation può unirsi a un servizio di directory esistente come un dominio Windows o client LDAP. ﻿È possibile gestire i privilegi di accesso dell'utente del dominio o LDAP alle cartelle condivise e alle applicazioni DSM e abilitarne il servizio sede utente. Andare su Pannello di controllo > Servizio directory per unire DiskStation a un servizio directory.
