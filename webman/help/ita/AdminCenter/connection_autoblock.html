Blocco automatico

Il blocco automatico ti consente di evitare accessi non autorizzati. Una volta abilitato il servizio, un indirizzo IP viene bloccato se vi sono stati troppi tentativi di accesso non riusciti. I tentativi di accesso non riusciti attraverso SSH, Telnet, rsync, Backup di rete, Sincronizzazione cartelle condivise, FTP, WebDAV, dispositivi mobili Synology, File Station o DSM sono cumulativi. È anche possibile avere un Elenco consentiti che include gli indirizzi IP che non saranno mai bloccati.

Per impostare il blocco automatico:

  1. Passare alla scheda Generale e selezionare Attiva blocco automatico.
  2. Inserire un numero per i Tentativi di accesso e Entro (minuti) per bloccare un indirizzo IP dopo un determinato numero di accessi non riusciti entro il numero di minuti indicato.
  3. Selezionare Attiva scadenza del blocco e inserire un numero per rimuovere un indirizzo IP bloccato dopo il numero di giorni indicato. Gli indirizzi IP bloccati rimarranno indefinitivamente nell'elenco blocchi fino a quando non vengono rimossi manualmente dall'elenco se questa opzione non è selezionata o se Sblocca dopo (giorni) è impostato su 0.
  4. Fare clic su Applica.

Per aggiungere gli indirizzi IP bloccati:

Fare clic sulla scheda Elenco blocchi e selezionare una delle seguenti voci dal menu a discesa Crea:

  • Aggiungi indirizzo IP

    • Inserire l'indirizzo IP e impostare la data di scadenza.

  • Importa elenco indirizzi IP

    • Importare un file di testo che elenca gli indirizzi IP per aggiungere tutti gli indirizzi IP e impostare tutte le date di scadenza. Selezionare le caselle di controllo appropriate per sovrascrivere gli indirizzi IP esistenti. Se l'opzione non viene selezionata, gli indirizzi IP duplicati saranno ignorati.

    Per importare un file, il file deve soddisfare i seguenti criteri:

    • Il file deve essere un file di solo testo.
    • Ciascuna riga del file deve contenere un solo indirizzo IP.
    • Le righe di commento devono iniziare con # (simbolo del cancelletto).

      • Se il file selezionato è nel formato corretto, gli indirizzi IP saranno visualizzati nell'area di anteprima. Fare clic su OK per importare.

Per rimuovere gli indirizzi IP bloccati:

  1. Fare clic sulla scheda Elenco blocchi per visualizzare l'elenco degli indirizzi IP bloccati.
  2. Selezionare gli indirizzi IP da rimuovere dall'elenco e fare clic su Rimuovi.

Per aggiungere gli indirizzi IP consentiti:

Fare clic sulla scheda Elenco consentiti e selezionare una delle seguenti voci dal menu a discesa Crea:

  • Aggiungi indirizzo IP

    • Immettere un indirizzo IP.

  • Importa elenco indirizzi IP

    • Importare un file di testo che elenca gli indirizzi IP per aggiungere tutti gli indirizzi IP. Selezionare le caselle di controllo appropriate per sovrascrivere gli indirizzi IP esistenti. Se l'opzione non viene selezionata, gli indirizzi IP duplicati saranno ignorati.

    Per importare un file, il file deve soddisfare i seguenti criteri:

    • Il file deve essere un file di solo testo.
    • Ciascuna riga del file deve contenere un solo indirizzo IP.
    • Le righe di commento devono iniziare con # (simbolo del cancelletto).

      • Se il file selezionato è nel formato corretto, gli indirizzi IP saranno visualizzati nell'area di anteprima. Fare clic su OK per importare.

Per rimuovere gli indirizzi IP consentiti:

  1. Fare clic sulla scheda Elenco consentiti per visualizzare l'elenco degli indirizzi IP consentiti.
  2. Selezionare gli indirizzi IP da rimuovere dall'elenco e fare clic su Rimuovi.
