Cartella condivisa

Le cartelle condivise sono directory di base dove è possibile salvare i file e le cartelle sulla DiskStation. Prima di salvare i file su DiskStation, è necessario creare almeno una cartella condivisa. I dati salvati sulle cartelle condivise possono essere privati o condivisi con utenti specifici o gruppi di utenti in base alle autorizzazioni di accesso personalizzabili. Inoltre, le cartelle condivise possono essere crittografate durante la creazione, aggiungendo un ulteriore livello di protezione.

Cartelle condivise predefinite

Di seguito un elenco delle cartelle condivise che sono create automaticamente quando alcune applicazioni, servizi o pacchetti sono installati o abilitati.

Nome Descrizione
public La cartella public verrà creata automaticamente dopo la prima configurazione di DiskStation.*
web La cartella web è la posizione di archiviazione predefinita per i contenuti di un sito web ospitato su DiskStation. Essa è creata automaticamente quando la Web Station è abilitata.
photo La cartella photo è usata per archiviare foto e video che si desiderano condividere su Photo Station. Verrà creata automaticamente all'attivazione di Photo Station o del Server media DLNA/UPnP.
music La cartella music è usata per archiviare la musica che si desidera ascoltare con Audio Station. Essa è creata automaticamente quando la Audio Station è abilitata.
video La cartella video è usata per archiviare i file video che si desidera sfogliare attraverso i dispositivi DLNA/UPnP. Essa è creata automaticamente quando il Server Media è abilitato.
surveillance La cartella surveillance è usata per archiviare le registrazioni di Surveillance Station. Essa è creata automaticamente quando la Surveillance Station è abilitata. È una cartella di sola lettura alla quale può accedere solo l'amministratore di sistema.
home Le cartelle home forniscono spazi privati per ciascun utente per archiviare i dati a cui solo tale utente può accedere. Essa è creata automaticamente quando la funzione Sede utente è abilitata.
homes La cartella homes contiene le cartelle home di tutti gli utenti. Essa è creata automaticamente quando la funzione Sede utente è abilitata. Solo l'amministratore di sistema può accedere e gestire tutte le cartelle home degli utenti.
NetBackup La cartella NetBackup è creata automaticamente all'attivazione del Backup di rete.
usbshare La cartella usbshare[numero] viene creata automaticamente quando si connette un dispositivo esterno USB alla porta USB di DiskStation.*
satashare La cartella satashare viene creata automaticamente quando si connette un dispositivo esterno eSATA alla porta eSATA di DiskStation.*

____
* Disponibile solo su alcuni modelli.
