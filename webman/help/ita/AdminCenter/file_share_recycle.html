Abilita cestino

Quando viene abilitato il Cestino in una cartella condivisa, viene creare una sottocartella per archiviare i file eliminati. I file eliminati presenti nel Cestino possono essere recuperati. Vuotare il Cestino per rimuovere definitivamente i file.

Per abilitare il Cestino:

  1. Selezionare una cartella condivisa, quindi fare clic su Modifica.
  2. Selezionare la casella Abilita cestino.
  3. Selezionare Limita accesso ai soli amministratori per rendere il Cestino accessibile solamente agli amministratori.
  4. Fare clic su OK per salvare le impostazioni.

Per vuotare il Cestino:

Fare clic sul pulsante Vuota cestino per rimuovere definitivamente i file eliminati dal cestino.

Per vuotare tutti i Cestini:

Scegliere Vuota tutti i cestini dal menu a discesa Azione per rimuovere definitivamente tutti i file eliminati dai cestini.

Nota: è possibile configurare le impostazioni dei privilegi della cartella Cestino in File Station facendo clic sulla cartella #recycle. Scegliere Proprietà e fare clic sulla scheda Autorizzazione.
