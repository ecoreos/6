Database di sistema

DiskStation utilizza un database di sistema per tenere traccia dei dati pacchetto e record sul registro archiviati nelle cartelle condivise create su dispositivi di archiviazione esterni, come usbshare1.

Importante: Se si desidera accedere ai dati pacchetto archiviati sui dispositivi di archiviazione esterna a DiskStation, o se si desidera di vedere i record di registro di DSM su Menu principale > Log Center, deve essere prima abilitato il database di sistema. Diversamente, tutti i servizi correlati saranno inaccessibili.

Per abilitare il database di sistema:

  1. Selezionare Abilita il database di sistema.
  2. Eseguire una delle seguenti azioni:
    • Scegliere Configurazione automatica se si desidera che il DSM selezioni automaticamente una cartella condivisa per l'archiviazione del database di sistema. Se il dispositivo di archiviazione esterna contenente la cartella condivisa viene rimosso, il DSM selezionerà automaticamente un'altra cartella condivisa per l'archiviazione del database.
    • Scegliere Sposta il database attuale alla cartella condivisa selezionata, e quindi scegliere dal menu a discesa per spostare il database attuale in un'altra cartella condivisa creata sul proprio dispositivo esterno.
    • Scegliere Scegli uno dei database esistenti, e quindi scegliere dal menu a discesa per selezionare un database esistente se c'è più di un dispositivo esterno collegato a DiskStation.
  3. Fare clic su Applica.

Avvertenza: Per evitare danni al database, assicurarsi di aver estratto il dispositivo di archiviazione da Menu principale > Pannello di controllo > Dispositivi esterni > Dispositivi esterni o facendo clic sull'icona dei dispositivi esterni nell'angolo in alto a destra prima di rimuoverlo da DiskStation.
