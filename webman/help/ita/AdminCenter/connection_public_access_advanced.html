Porta del router

Se si desidera che servizi Internet (quali Google Docs o Pixlr) accedano a DiskStation, è possibile specificare il nome host esterno pubblico e il numero della porta su Pannello di controllo > Accesso esterno > Avanzate.

Come impostazione predefinita, il nome host è il proprio indirizzo IP pubblico (o nome host impostato nel Pannello di controllo > Accesso esterno > DDNS), e il numero di porta è 5000 (o 5001 per connessioni HTTPS). Tuttavia, se si ha un altro host name che non è configurato nella pagina DDNS, o si hanno numeri di porta diversi specificati sul router quando si configurano le regole di invio porta, sarà necessario immettere il nome host o i numeri di porta router per assicurarsi che si possa accedere con successo ai file condivisi nei servizi Internet.

Per abilitare la configurazione di porta del router:

  1. Inserire il nome host o l'indirizzo IP pubblico del DiskStation nel campo Nomehost o IP statico.
  2. Inserire il numero di porta del router specificato quando si configura la regola di invio di porta di DSM sul router nei campi DSM (HTTP) o DSM (HTTPS).
  3. Fare clic su Applica.
