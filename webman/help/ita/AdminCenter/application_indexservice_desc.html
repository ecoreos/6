Indicizzazione servizio

Con lIndicizzazione multimediale è possibile eseguire la scansione automatica di file come foto, musica e video archiviati su DiskStation, compilandoli nella libreria multimediale per essere utilizzati dalle applicazioni multimediali. È anche possibile impostare la qualità della miniatura per foto e video, gestire le impostazioni di conversione per video, e creare cartelle indicizzate a cui è possibile accedere dai dispositivi DMA e delle applicazioni multimediali di DSM.

Indicizzazione file consente di indicizzare le cartelle e i file contenuti in DiskStation in base al nome file oltre a indicizzare il contenuto/i metadati del file per tipo di file. Questa funzione di indicizzazione dettagliata velocizzerà la ricerca consentendo corrispondenze più precise.
