Rete

In questa pagina sono riportati lo schema e la tabella di stato delle interfacce di rete.

Schema interfacce

Visualizza l'elemento selezionato nella tabella, mentre dei colori diversi sono usati per presentare le interfacce di rete da 1 GBPS e 10 Gbps.

Tabella stato interfaccia

Visualizza lo stato dell'interfaccia su entrambi i server e se l'interfaccia è monitorata. In questa tabella sono mostrate le seguenti informazioni:

  • Interfaccia di rete: Sono visualizzate le interfacce di rete fisica sui server.
  • Server attivo: Visualizza lo stato dell'interfaccia e la velocità di connessione sul server attivo.
  • Server passivo/Server sconosciuto: Visualizza lo stato dell'interfaccia e la velocità di connessione sul server passivo. In questa colonna è mostrato il nome corrispondente allo stato del server.
  • Bond: Visualizza le impostazioni di aggregazione dei collegamenti.
  • Stato: Visualizza lo stato di rete se l'interfaccia è monitorata. Potrebbe mostrare gli stati seguenti:
    • Avvertenza: La connessione ha esito negativo sul server passivo oppure l'interfaccia presenta velocità di connessione diverse sui due server;
    • Errore: La connessione ha esito negativo sul server attivo;
    • Normale: Nessun errore.
  • Monitor: Visualizza se l'interfaccia è monitorata. Andare su Monitor > Interfaccia di rete per spuntare/deselezionare la casella di controllo Monitor.
