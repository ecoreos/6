Cache SSD*

La cache SSD è in grado di migliorare le prestazioni dell'accesso casuale archiviando i dati di accessi frequenti nella cache SSD. La cache SSD può essere montata su un volume o iSCSI LUN (livello blocco). Esistono due tipi di cache SSD:

  • Cache di sola lettura è composta da 1 a 12 SSD montabili rispettivamente in una configurazione base o RAID 0 per migliorare la velocità di lettura dello spazio di archiviazione in cui viene montata.

    Nota:

    La cache di sola lettura SSD archivia le copie di dati dal volume; in questo modo, non si verificherà alcuna perdita di dati anche in caso di blocco della cache sola lettura.

  • Cache lettura-scrittura può essere montata in una configurazione RAID 1 / RAID 5 / RAID 6, a seconda del numero di SSD (fino a 12 SSD) per migliorare le prestazioni di lettura e scrittura dello spazio di archiviazione su cui è montata.

Entrambi i tipi di cache SSD implementano un algoritmo LRU (Least Recently Used) per lo scambio dei dati nella cache.

Importante:

Prima di installare qualsiasi SSD per l'utilizzo come cache, verificare le note seguenti:

  • L'obiettivo dell'abilitazione della cache SSD è aumentare le prestazioni dell'accesso casuale a una piccola porzione di dati con accesso frequente nello spazio di archiviazione. Ad esempio, entrambe le operazioni di lettura o scrittura sequenziali grandi (es., streaming video HD) e i modelli di lettura dati interamente casuali mancano di modelli di rilettura, quindi non beneficieranno particolarmente della cache SSD. Per applicazioni generali si consiglia di abilitare l'opzione Salta I/O sequenziale, in modo che l'I/O possa ancora attraversare le unità dello spazio di archiviazione.
  • fare riferimento all'elenco di compatibilità sul sito Web di Synology per scegliere quale SSD può essere usato come cache su DiskStation.
  • La cache SSD richiede una determinata quantità di memoria di sistema a seconda delle dimensioni della cache, quindi il montaggio di unità cache SSD di dimensioni più grandi potrebbe richiedere l'aggiornamento della memoria del DiskStation. Per mantenere il sistema stabile e veloce, solo 1/4 della memoria del sistema preinstallata può essere usata per la cache SSD. Ogni 1 GB di cache SSD richiede circa 416 KB di memoria del sistema (memoria espandibile può essere considerata). Quindi, se la configurazione di cache di sola lettura è 2 x 128 GB SSD, la dimensione della cache totale sarà di 256 GB e richiede almeno 104 MB di memoria; se la configurazione cache di sola scrittura è 2 x 128 GB SSD, la dimensione della cache totale sarà di 128 GB e richiede almeno 52 MB di memoria. Se la memoria è insufficiente, la creazione della cache SSD sarà limitata o le dimensioni della cache SSD saranno limitate.
  • Ogni 1 GB di cache SSD richiede circa 416 KB di memoria di sistema (memoria espandibile inclusa) e la dimensione massima della cache è di 930 GB. Quindi, se la configurazione di cache di sola lettura è 2 x 128 GB SSD, la dimensione della cache totale sarà di 256 GB e richiede almeno 104 MB di memoria; se la configurazione cache di sola scrittura è 2 x 128 GB SSD, la dimensione della cache totale sarà di 128 GB e richiede almeno 52 MB di memoria. Se la memoria è insufficiente, la creazione della cache SSD sarà limitata o le dimensioni della cache SSD saranno limitate.
  • Fare riferimento alle specifiche del prodotto del modello sul sito Web di Synology per la dimensione massima della cache SSD consentita su DiskStation.
  • Quando una cache SSD lettura-scrittura è danneggiata, la funzione della cache verrà disabilitata e il sistema avvierà la scrittura (ad es. la sincronizzazione) dei dati cache dalla SSD ai dischi rigidi. Al termine della sincronizzazione, non si verificherà alcuna perdita di dati anche se la cache lettura-scrittura si è bloccata.
  • La cache lettura-scrittura non può essere usata insieme a Time Backup. Ad esempio, quando si crea una cache SSD lettura-scrittura, non sarà possibile eseguire Time Backup. Viceversa, se si sta eseguendo Time Backup, non si sarà in grado di creare una cache SSD lettura-scrittura.

Per montare la cache SSD:

  1. Installare l'SSD sulla DiskStation. Consultare prima le note precedenti.
  2. Andare su Gestore archiviazione > Cache SSD. Fare clic su Crea.
  3. Selezionare la modalità cache (disponibile solo per modelli che supportano la cache di lettura/scrittura).
  4. Trovare il menu a discesa e selezionare il volume o iSCSI LUN (livello blocco) su cui montare la cache SSD.
  5. Scegliere le due SSD nell'elenco. I dati sulle SSD saranno cancellati, quindi assicurarsi che non vengano eliminati dati importanti.
  6. Fare clic su Applica.

Per interrompere il caching I/O sequenziale:

se il proprio volume di destinazione o iSCSI LUN (livello blocco) ha prestazioni I/O sequenziali eccellenti, è possibile interrompere la cache SSD bloccando I/O sequenziale e accedendo direttamente ai dati dalle proprie unità per velocità più alta e prestazioni migliori.

  1. Fare clic su Configura per aprire la finestra di configurazione.
  2. Spuntare Salta I/O sequenziale.

Per rimuovere la cache SSD:

  1. Fare clic su Rimuovi per rimuovere in modo permanente questa cache SSD da DiskStation.
  2. Per la cache lettura-scrittura, la si deve rimuovere attraverso questa procedura (anche se DiskStation è spento), altrimenti si verificherà un crash perché dati parziali risiedono solo sulla cache.

Nota:

  • Quando viene montata o rimossa la cache SSD, il sistema interrompe tutti i servizi correlati (ad esempio le cartelle condivise, i pacchetti installati, le connessioni correnti, ecc.)
  • Se una cache lettura-scrittura su uno spazio di archiviazione non è rilevata al riavvio, il sistema non monterà tale spazio di archiviazione. Trovare e installare nuovamente le SSD in DiskStation e riavviare per montare lo spazio di archiviazione. Se è impossibile trovare le SSD, potrebbe essere necessario forzare il montaggio dello spazio di archiviazione. Fare riferimento a Volume o iSCSI LUN.

Per riparare la cache SSD:

  • Per cache di lettura/scrittura: Se una delle SSD è in crash, procedere come segue per ripararla.
    1. Sostituire la SSD guasta con una nuova (la nuova SSD deve essere dello stesso modello e della stessa dimensione della vecchia).
    2. Fare clic sul pulsante Riparazione.
    3. Selezionare la SSD dall'elenco.
    4. Fare clic su Applica.
  • Per cache di sola lettura: Se una delle SSD è in crash, rimuovere la cache e sostituire la SSD guasta. Quindi fare clic su Crea per creare una nuova configurazione della cache.

Segnalatore Cache SSD

Questa funzione consente di calcolare e visualizzare la dimensione totale e il numero di file a cui è stato eseguito l'accesso di recente in un volume specifico. Se le cache SSD non sono state ancora create sul volume selezionato, si può fare riferimento a questa informazione quando si calcola la quantità di cache SSD adatta al sistema. Se in precedenza è stata creata una cache SSD sul volume selezionato, anche l'analisi e l'utilizzo della cache verranno mostrati qui.

Per utilizzare SSD Cache Advisor:

  1. fare clic su SSD Cache Advisor per aprire la procedura guidata.
  2. Scegliere un volume per il calcolo, quindi fare clic su Avanti.
  3. La procedura guidata consentirà di avviare automaticamente la dimensione e del numero di file a cui è eseguito l'accesso di recente sul volume selezionato. La durata del calcolo varia in relazione al singolo ambiente.
