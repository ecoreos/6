Pulitura dati*

La pulitura dati è una funzione di manutenzione che consente di ispezionare volumi o gruppi disco. Esistono due tipi di pulizia dati:

  • Pulitura RAID: Questa funzione può essere utilizzata con volumi e gruppi disco che implementano SHR (con tre o meno dischi), RAID 5 o RAID 6. Consente di riparare tutti le inconsistenze di dati rilevate.
  • Pulitura del file system: Questa funzione può essere utilizzata con volumi in file system Btrfs. In caso di rilevamento di inconsistenze di dati con il checksum, il percorso file sarà registrato in Log Center. Si consiglia di eseguire un backup per proteggere i dati.

Si consiglia di eseguire regolarmente la pulitura dei dati in modo da garantire la consistenza dei dati ed evitare di perdere i dati in caso di guasto del disco.

Per eseguire la pulitura dei dati:

  1. Aprire Gestione archiviazione.
  2. Andare Volume o Gruppo Disco.
  3. Selezionare il volume o gruppo disco sul quale eseguire la pulitura dei dati.
  4. Fare clic su Gestisci.
  5. Selezionare Pulitura dati.
  6. Selezionare il tipo di pulitura dati desiderato. Il sistema ignorerà automaticamente questo passaggio se è disponibile un solo tipo.

Nota:

  • la pulizia dati può essere eseguita solamente quando lo stato del volume o Gruppo disco è Normale.
  • Se il sistema viene spento prima del termine, sarà necessario riavviare manualmente dall'inizio la pulizia dei dati.

* Disponibile solo su alcuni modelli.
