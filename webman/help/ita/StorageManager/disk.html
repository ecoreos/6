Disco rigido/SSD

Questa sezione consente all'utente di monitorare lo stato dei dischi rigidi/SSD installati sul DiskStation offrendo le opzioni per la gestione e analisi delle prestazioni e dell'integrità dei dischi rigidi/SSD. Andare a Gestione archiviazione > Disco rigido/SSD per gestire queste opzioni.

Per controllare le informazioni del disco rigido/SSD:

  1. Andare alla scheda HDD/SSD.
  2. Selezionare un'unità dall'elenco ed espanderla. Qui è possibile vedere lo stato di S.M.A.R.T., informazioni volume, nome modello, numero di serie, versione firmware e temperatura.

Per controllare le informazioni sulla salute:

  1. Andare alla scheda HDD/SSD.
  2. Selezione un'unità.
  3. Fare clic su Informazioni salute.
  4. Per passare nella scheda Panoramica. Qui troverete per quante ore l'unità è stata accesa, la temperatura corrente, conteggio riconnessione/reidentificazione, conteggio settore sbagliato e stato S.M.A.R.T. generale.
  5. Andare sulla scheda Informazioni S.M.A.R.T.. I numeri elencati possono aiutare ad eseguire la diagnosi dello stato dell'unità.

Nota:

le unità SAS non supportano le Informazioni S.M.A.R.T.

Per eseguire un test S.M.A.R.T:

Il test S.M.A.R.T. del disco aiuterà ad esaminare e riferire lo stato delle unità per possibili guasti delle unità. Nel caso vengano rilevati errori si consiglia di cambiare subito le unità.

  1. Andare alla scheda HDD/SSD.
  2. Selezione un'unità.
  3. Fare clic su test S.M.A.R.T.
  4. Scegliere un tipo di test e fare clic su Avvia per iniziare il test.
  5. I risultati dell'ultimo test sono mostrati nella sezione Risultato test.

Nota:

la durata del test varia in relazione all'ambiente di sistema.

Per programmare i test S.M.A.R.T.:

È possibile programmare i test S.M.A.R.T. per essere eseguiti automaticamente creando delle attività.

  1. Andare alla scheda Programmazione test.
  2. Fare clic su Crea per impostare una nuova attività.
  3. Modificare le seguenti impostazioni nella scheda Generale:
    • Attività: immettere un nome per l'attività.
    • Tipo test S.M.A.R.T.: scegliere se eseguire un test rapido o un test esteso.
    • Intervallo test S.M.A.R.T.: scegliere di quali dischi rigidi/SSD eseguire il test tutte le volte che viene eseguito un test S.M.A.R.T..
  4. Nella scheda Programmazione, modificare le seguenti impostazioni:
    • Data: specificare quando eseguire un test S.M.A.R.T. È possibile programmare i test S.M.A.R.T. per essere eseguiti in dati giorni della settimana (ad esempio ogni giorno, nei fine settimana o ogni Domenica, ecc.) o in date specifiche (ad esempio 2015/10/21).
    • Ora: specificare quante volte eseguire il test S.M.A.R.T. o la prima e l'ultima volta durante il giorno.
  5. Fare clic su OK per terminare.

Per eseguire Secure Erase:

  1. Andare alla scheda HDD/SSD.
  2. Selezionare un'unità e fare clic su Secure Erase.

Nota:

  • Secure Erase non può essere eseguito su dischi nell'unità di espansione.
  • Secure Erase cancellerà in modo permanente tutti i dati archiviati sulla propria unità. Assicurarsi che l'unità scelta non sia in uso o sia stato effettuato il backup dei dati prima di procedere.
  • Quando il sistema sta eseguendo Secure Erase, l'unità scelta sarà temporaneamente bloccata e quindi automaticamente sbloccata quando l'operazione è in completata.
  • Se Secure Erase è interrotto in modo errato durante l'operazione, è possibile eseguire nuovamente Secure Erase direttamente sull'interfaccia utente. Se le unità sono rimosse o usate altrove quando sono ancora bloccate, inserire la password "synology" per sbloccare le unità.

Abilitazione del supporto scrittura cache per dischi rigidi

L'abilitazione del supporto per la cache di scrittura migliora le prestazioni della DiskStation. Solamente alcuni modelli di dischi rigidi supportano questa funzione. Per garantire la protezione dei dati durante l'uso del supporto cache di scrittura, si consiglia di utilizzare un dispositivo UPS. Si consiglia inoltre di spegnere correttamente il sistema dopo l'uso. Disattivando la cache di scrittura si riduce la possibilità di perdita dei dati causata dalle interruzioni di corrente, riducendo anche le prestazioni del sistema.

Abilitare il supporto scrittura cache:

  1. Andare alla scheda Gestione cache.
  2. Selezionare Attiva scrittura cache. Per impostazione predefinita, la funzione è disattivata.
  3. Fare clic su Applica.
