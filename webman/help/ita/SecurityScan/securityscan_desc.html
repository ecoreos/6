Segnalatore sicurezza

Security Advisor è un'applicazione per la sicurezza DSM che scansiona le impostazioni di DSM e DiskStation. Security Advisor consente di controllare le impostazioni e suggerisce le modifiche che contribuiscono alla sicurezza di DiskStation. Di seguito è riportato un elenco delle funzioni offerte:

  • Security Advisor eseguirà diversi controllo per rilevare le impostazioni di DSM che potrebbero presentare rischi per la sicurezza e qualsiasi attività sospetta.
  • Security Advisor fornisce risultati dettagliati, con report che spiegano le azioni consigliate da intraprendere e i vantaggi associati.
  • Il database delle definizioni di sicurezza interna viene aggiornato automaticamente, restando aggiornato sulle nuove minacce cibernetiche.

Per lanciare Segnalatore sicurezza procedere come segue:

  • Andare su Menu principale > Segnalatore sicurezza.
