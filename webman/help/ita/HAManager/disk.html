Stato disco

Questa pagina contiene le schede che visualizzano lo stato dei dischi in un cluster high-availability. Ciascuna scheda del rispettivo server mostra le seguenti informazioni.

  • Numero: indica quale vano è occupato da ciascun disco. I dischi nei server attivo e passivo che occupano che occupano i vani corrispondenti devono avere la stessa capacità.
  • Dimensioni disco: indica la capacità di ciascun disco.
  • Temperatura: indica la temperatura del disco.
  • Stato S.M.A.R.T.: indica lo stato S.M.A.R.T. di ciascun disco.
  • Stato: visualizza le informazioni sul disco e le notifiche di errore.

Per sostituire i dischi non funzionanti:

  1. Verificare il numero del disco.
  2. Sostituire il disco non funzionante con uno nuovo. Per le procedure relative all'installazione del disco rigido, consultare la Guida di installazione rapida del prodotto Synology.
  3. Andare su Stato archiviazione.
  4. Selezionare il volume danneggiato.
  5. Fare clic su Ripara per riparare il volume.

Nota:

la capacità dei dischi rigidi di ricambio deve essere uguale o superiore a quella del disco rigido non funzionante.

Per aggiungere nuovi dischi al cluster high-availability:

Per espandere la capacità, è possibile aggiungere dei dischi rigidi al cluster high-availability. Prima di aggiungere i dischi rigidi, fare riferimento di seguito.

  • Si consiglia di aggiungere i dischi rigidi quando sia il server attivo che il server passivo sono online.
  • La capacità dei dischi rigidi sul server passivo deve essere uguale o superiore a quella dei dischi sul server attivo.

Per aggiungere i dischi rigidi:

  1. Inserire i dischi rigidi nei vani corrispondenti sul server attivo e sul server passivo. Ad esempio, se viene inserito un disco rigido nel vano 1 del server attivo, allora è necessario inserire il disco rigido corrispondente nel vano 1 del server passivo.
  2. Andare su Gestione archiviazione > Volume (o Gruppo disco/iSCSI LUN/RAID Group a seconda della situazione e del modello). Selezionare il volume da espandere, fare clic sul pulsante Gestisci e seguire le istruzioni per aggiungere i dischi rigidi.
  3. Dopo avere aggiunto correttamente i dischi rigidi, tornare a High Availability Manager > Stato archiviazione per controllare se lo stato è Normale.
