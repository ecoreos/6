DSM mobile

DSM mobile consente all'admin DSM (o a un utente appartenente al gruppo administrators) di gestire le impostazioni DSM e di controllarne le informazioni DSM con un dispositivo iOS® (iPhone®, iPad® o iPod touch®) o Android®. Questa Guida a DSM mobile fornisce i seguenti argomenti e risorse.

Accesso e disconnessione

Dopo la connessione a DSM con il browser Web sul dispositivo iOS o Android, sarà visibile la schermata di accesso dove inserire le informazioni dell'account DSM.

Per accedere a DSM mobile:

Inserire le credenziali utente DSM e toccare Accedi. Per salvare le informazioni di accesso e accedere automaticamente la volta successiva, attivare l'interruttore Memorizza prima di accedere.

Per uscire da DSM mobile:

  1. Toccare il pulsante di opzione in alto a destra sulla home page (telefono iPhone/iPod touch o Android) o nel pannello a sinistra (tablet iPad o Android).
  2. Toccare Impostazioni e Disconnessione.

Gestione utenti DSM

Toccare Utente e Gruppo sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per vedere l'elenco utenti.

Per creare un utente:

  1. Toccare + e inserire il nome utente e la password e l'indirizzo e-mail del nuovo utente.
  2. Attivare Invia notifica per inviare una mail di notifica all'indirizzo e-mail dell'utente quando è stato creato l'account dell'utente DSM.
    Nota: prima di inviare la notifica, verificare di aver completato le impostazioni di notifica su NAS Synology. Accedere all'edizione desktop del DSM con il browser web del computer e selezionare Menu principale > Pannello di controllo > Notifica > E-mail per modificare le impostazioni.
  3. Toccare Fine per confermare oppure toccare la freccia sinistra per annullare.

Per trovare velocemente un utente:

Inserire il nome utente nel campo di ricerca.

Per modificare le informazioni di un utente:

  1. Toccare il nome dell'utente e procedere come segue:
    • Modificare il nome o la password dell'utente nel campo Nome o Password.
    • Attivare Disabilita per disabilitare l'utente. È possibile scegliere di abilitare l'utente in un secondo momento senza perdere le sue impostazioni.
    • Attivare Amministratore per aggiungere l'utente al gruppo admin. Un utente qualsiasi appartenente al gruppo admin avrà i privilegi amministrativi per modificare le impostazioni DSM tramite DSM mobile o il browser Web del computer.
  2. In Privilegi, eseguire quanto segue e toccare Indietro:
    • Toccare Cartella condivisa e toccare il nome della cartella condivisa per modificare il privilegio di accesso dell'utente: Lettura/Scrittura: l'utente può apportare modifiche a file e sottocartelle nella cartella condivisa. Sola lettura: l'utente può accedere a file e sottocartelle nella cartella condivisa, ma non apportarvi modifiche; Nessun accesso: l'utente non può accedere ai file e alle sottocartelle nella cartella condivisa.
      Note: se si verificano conflitti di privilegio, la priorità sarà: Nessun accesso > Lettura/Scrittura > Sola lettura.
    • Toccare Applicazione per modificare il privilegio di accesso dell'utente alle applicazioni DSM.
  3. Toccare Fine per confermare oppure toccare la freccia sinistra per annullare.

Gestire le cartelle condivise

Toccare Cartella condivisa sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per vedere l'elenco delle cartelle condivise su NAS Synology.

Per creare una cartella condivisa:

  1. Toccare + e inserire il nome della nuova cartella.
  2. Toccare Fine per confermare oppure toccare la freccia sinistra per annullare.

Per modificare una cartella condivisa:

Toccare il nome della cartella condivisa e procedere come segue:

  • Modificare il nome della cartella condivisa nel campo Nome.
  • Toccare Autorizzazioni utente e poi un utente per modificarne i privilegi di accesso alla cartella condivisa: Lettura/Scrittura: l'utente può apportare modifiche a file e sottocartelle nella cartella condivisa. Sola lettura: l'utente può accedere a file e sottocartelle nella cartella condivisa, ma non apportarvi modifiche; Nessun accesso: l'utente non può accedere ai file e alle sottocartelle nella cartella condivisa.
    Note: se si verificano conflitti di privilegio, la priorità sarà: Nessun accesso > Lettura/Scrittura > Sola lettura.
  • È possibile anche toccare Autorizzazioni gruppo per configurare i privilegi per i gruppi.
    Nota: le autorizzazioni gruppi hanno una priorità più elevata rispetto alle autorizzazioni utente.

Abilitare i servizi

Toccare Servizio sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per abilitare o disabilitare i servizi NAS Synology.

Per gestire i servizi LAN di NAS Synology:

  • Abilitare o disabilitare quanto segue:
    • Servizio file Windows consente a NAS Synology di diventare un server file Windows (CIFS) nella LAN.
    • Servizio file Mac consente a NAS Synology di diventare un server file Mac (AFP) nella LAN.
    • Servizio NFS consente ai client Linux nella LAN di accedere ai dati di NAS Synology.
    • Stampante Bonjour consente ai client Mac di stampare sulle stampanti USB connesse a NAS Synology.
    • Server media consente ai dispositivi domestici DLNA/UPnP nella LAN di cercare e riprodurre i contenuti multimediali su NAS Synology.
    • iTunes consente ai client iTunes nella LAN di cercare e riprodurre musica e video su NAS Synology.

Per gestire i servizi Internet di NAS Synology:

  • Abilitare o disabilitare quanto segue:
    • FTP permette a NAS Synology di diventare un server FTP.
    • Servizi Web consente di ospitare siti (con supporto PHP+MySQL) su NAS Synology.
    • WebDAV consente ai programmi client di gestire e modificare i file NAS Synology tramite protocollo HTTP.
    • WebDAV (HTTPS) consente ai programmi client di gestire e modificare i file NAS Synology tramite protocollo HTTPS.
    • File Station File Station è un'applicazione basata su Web che permette la gestione dei file NAS Synology tramite porta personalizzata senza connettersi a DSM.
    • File Station (HTTPS) consente la connessione a File Station tramite protocollo HTTPS.
    • Surveillance Station è un'applicazione basata su Web in grado di gestire le telecamere IP per il controllo dell'ambiente di casa o lavorativo.
    • Audio Station è un'applicazione su basata su Web che consente di accedere alla libreria musicale su NAS Synology.
    • Photo Station è un album fotografico online con blog integrato salvato su NAS Synology.
    • Download Station (BT/HTTP/FTP/NZB/RSS) è un'applicazione basata su Web che aiuta a scaricare i file da Internet su NAS Synology.
    • eMule abilita il supporto eMule di Download Station.

Nota:

Per maggiori informazioni sui servizi, consultare quanto segue:

  • Guida DSM (disponibile collegandosi all'edizione desktop del DSM con il browser del computer e andando su Menu principale > Guida DSM)
  • Guida dell'utente Synology DSM (disponibile al Centro download Synology)

Controllo delle risorse di sistema

Toccare Controllo risorse sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per controllare utilizzo della memoria e della CPU di NAS Synology.

Visualizzazione informazioni di sistema

Toccare Informazioni di sistema sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per ottenere lo stato di NAS Synology.

Per visualizzare le informazioni di sistema di NAS Synology

  1. Eseguire una delle seguenti azioni:
    • Toccare Generale per visualizzare le informazioni di base su NAS Synology, incluso numero di serie, nome modello, quantità di memoria fisica, versione DSM, e informazioni sull'ora e lo stato termico.
    • Toccare Rete per visualizzare la configurazione di rete di NAS Synology e lo stato di ogni interfaccia di rete.
    • Toccare Archiviazione per visualizzare l'utilizzo di volume di NAS Synology.
  2. Toccare il pulsante Aggiorna nell'angolo a destra per mantenere il sistema aggiornato.

Firmware

Toccare Firmware per visualizzare e aggiornare la versione DSM.

Per configurare le impostazioni di aggiornamento DSM:

  1. Toccare Aggiornamento DSM.
  2. Eseguire una delle seguenti opzioni:
    • Solo aggiornamenti importanti
    • DSM più recente e tutti gli aggiornamenti
  3. Toccare Fine.

Per scaricare gli aggiornamenti disponibili:

  1. Controllare la versione DSM in Stato.
  2. Se la versione DSM non è aggiornata, toccare il pulsante Scarica che apparirà nella sezione Stato per aggiornare DSM.

Gestire il blocco IP

Toccare Blocco IP sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per evitare l'accesso non autorizzato.

Per gestire il blocco IP:

  1. Toccare il pulsante di configurazione nell'angolo in alto a destra.
  2. Attivare Attiva blocco automatico per abilitare il Blocco IP.
  3. Inserire un numero per i Tentativi di accesso e Entro (minuti) per bloccare un indirizzo IP dopo un determinato numero di accessi non riusciti entro il numero di minuti indicato.
  4. È possibile attivare Attiva scadenza blocco e specificare Sblocca dopo (giorni) per sbloccare un indirizzo IP dopo il numero di giorni specificato. Se viene inserito "0", l'indirizzo IP bloccato resterà indefinitivamente bloccato fino a quando non viene rimosso dall'elenco bloccati.
  5. Toccare Fine.

Per toccare gli indirizzi IP per l'elenco consentiti/bloccati:

  1. Per aggiungere l'indirizzo IP consentito/bloccato, toccare il pulsante per aggiungere nell'angolo in alto a destra e toccare Elenco consentiti o Elenco bloccati.
  2. Dopo avere inserito l'indirizzo IP, toccare Fine.

Gestire Centro pacchetti

Synology ha integrato applicazioni progettate da Synology o da terzi in pacchetti che possono essere installati e abilitati su NAS Synology.

Per gestire Centro pacchetti:

Toccare Centro pacchetti sulla home page (iPhone/iPod touch o Android phone) o sul pannello di sinistra (tablet iPad o Android) per visualizzare le informazioni sul pacchetto, eseguire o arrestare pacchetti oppure aggiornare alle ultime versione.

Per eseguire o arrestare i pacchetti installati:

Toccare lo switch accanto a un pacchetto per arrestarlo o eseguirlo.

Per visualizzare i pacchetti:

Toccare un pacchetto per visualizzarne le informazioni.

Per aggiornare i pacchetti:

  1. Toccare la scheda Aggiorna per visualizzare le versioni di aggiornamento attualmente disponibili.
  2. Fare clic sul pulsante a destra di un pacchetto per aggiornare all'ultima versione.

Visualizzare registri o connessioni correnti

Toccare Registro sulla home page (iPhone/iPod touch o Android phone) o sul pannello di sinistra (tablet iPad o Android) per monitorare le connessioni correnti o per visualizzare i registri di servizi di Synology NAS.

Per monitorare le connessioni concorrenti:

  • Toccare Ultimi [numero] registri nella sezione Panoramica per visualizzare l'elenco di utenti che accedono in ritardo ai servizi NAS Synology.

Per vedere le registrazioni di log di un servizio specifico:

  • Toccare un elemento qualsiasi nella sezione Registro eventi o Registro trasferimento file per visualizzare un elenco di registri.

Avviare applicazioni mobile

DSM mobile fornisce un luogo comodo dove avviare le applicazioni mobile Synology installate sul dispositivo iOS o Android.

Per avviare le app mobile Synology con DSM mobile:

Toccare una delle seguenti applicazioni nella sezione in basso della home page (telefono iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android):

  • DS photo (disponibile quando il servizio Photo Station è abilitato)
  • DS audio (disponibile quando il servizio Audio Station è abilitato)
  • DS cam (disponibile quando il servizio Surveillance Station è abilitato)
  • DS file (disponibile quando il servizio WebDAV è abilitato)

Nota:

  • per informazioni su queste applicazioni mobile, consultare i file della Guida allegati alle applicazioni.

Azionare NAS Synology
(Riavvio, Spegnimento o Avviso acustico)

DSM mobile consente di riavviare o spegnere NAS Synology con il dispositivo iOS o Android. Inoltre, quando sono presenti numerosi NAS Synology, è possibile identificare rapidamente il proprio richiedendo che emetta un avviso acustico.

Per azionare NAS Synology:

  1. Toccare il pulsante di opzione in alto a destra sulla home page (telefono iPhone/iPod touch o Android) o nel pannello a sinistra (tablet iPad o Android).
  2. Eseguire una delle seguenti azioni:
    • Toccare Riavvia e Sì per riavviare NAS Synology.
    • Toccare Spegnimento e Sì per spegnere NAS Synology.
    • Toccare Trovami per richiedere l'emissione di avvisi sonori da NAS Synology. Toccare OK per interrompere l'avviso acustico.

Passare alla modalità desktop di DSM

Fare clic sull'icona delle impostazioni e selezionare Modalità desktop per passare alla modalità desktop di DSM.

Uso di Storage Manager

Storage Manager consente di visualizzare rapidamente l'uso e lo stato di integrità dei volumi di sistema e dei dischi.

Per visualizzare le informazioni di volume:

  1. Toccare Gestione archiviazione sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android).
  2. Toccare la scheda Volume.
  3. Toccare un volume nell'elenco per visualizzare le informazioni dettagliate.

Per visualizzare le informazioni della foto:

  1. Toccare Gestione archiviazione sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android).
  2. Toccare la scheda HDD/SDD.
  3. Toccare un disco nell'elenco per visualizzare le informazioni sull'integrità e S.M.A.R.T.

Per avviare o arrestare un test S.M.A.R.T.:

  1. Toccare Gestione archiviazione sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android).
  2. Toccare la scheda HDD/SDD.
  3. Toccare un disco nell'elenco per visualizzare la pagina informazioni.
  4. Toccare S.M.A.R.T. Test per avviare un test S.M.A.R.T. o Stop per arrestarlo.
