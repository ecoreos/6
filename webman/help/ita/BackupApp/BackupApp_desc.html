Backup e Ripristino

Verrà descritto come utilizzare Backup e Ripristino nei modi seguenti:

  • Backup dei dati nei dispositivi Synology NAS locali remoti
  • Backup dei dati in un server rsync remoto
  • Backup di dati su cloud pubblici (es. Amazon S3 e Microsoft Azure)
  • Backup di iSCSI LUN* su dispositivi di archiviazione esterni e su altri dispositivi Synology NAS
  • Ripristino di file di backup
