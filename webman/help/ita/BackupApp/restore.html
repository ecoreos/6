Ripristina

Andare su Backup e Ripristino e fare clic su Ripristina, nell'angolo in basso a sinistra, per ripristinare i dati di backup.

Per ripristinare il backup di dati/LUN*:

  1. Fare clic su Ripristina, nell'angolo in basso a sinistra, selezionare Dati o LUN* e selezionare un'attività di backup da ripristinare.
  2. Verrà richiesto di selezionare la configurazione del sistema, versioni diverse di dati di backup o altro. Dipende dal tipo di attività di backup da ripristinare.
  3. Se l'attività di backup è crittografata, sarà necessaria la password/chiave di crittografia per il corretto ripristino.
  4. Seguire la procedura guidata per completare il ripristino.

Nota:

  • quando si ripristina un pacchetto non installato, il sistema installa automaticamente la stessa o l'ultima versione dell'applicazione. Se l'applicazione viene installata ma in un versione più vecchia di quella di cui si è eseguito il backup, verrà aggiornato automaticamente alla versione di cui si dispone il backup.
  • Le applicazioni di cui è stato effettuato il backup in DSM 5.1 o superiore non possono essere ripristinate in DSM 5.0 o inferiore.

Nota:

  • Al termine del ripristino, entrambe le LUN di livello blocco e di livello file saranno ripristinate nelle LUN di livello file.*
  • Quando si ripristina un pacchetto, se non è installato, verrà installata automaticamente la stessa o l'ultima versione dell'applicazione. Se l'applicazione viene installata ma in un versione più vecchia di quella di cui si è eseguito il backup, verrà aggiornato automaticamente alla versione di cui si dispone il backup.
  • Le applicazioni di cui è stato effettuato il backup in DSM 5.1 o superiore non possono essere ripristinate in DSM 5.0 o inferiore.

Per ripristinare un singolo file o cartella per l'attività di backup multi-versione:

  1. Selezionare l'attività di backup desiderata nel pannello di sinistr e lanciare Backup di Explorer (facendo clic sull'icona della lente di ingrandimento). Se l'attività di backup è criptata, è necessaria la password/chiave di crittografia.
  2. Individuare la versione di backup desiderata con la barra della cronologia o mediante l'icona del calendario.
  3. Selezionare un file o una cartella contenuti e fare clic su Ripristina, Copia in... o Scarica per recuperare il file/la cartella selezionata. Le tre opzioni sono disponibili anche facendo clic con il tasto destro del mouse sul file/cartella.

Per cercare una versione specifica:

Per aumentare la velocità di ricerca file, è possibile usare le parole chiave per filtrare il file/cartella desiderata. I file/cartelle non corrispondenti saranno nascosti.

  1. Lanciare Backup di Explorer e fare clic sull'icona dell'imbuto nell'angolo in alto a destra.
  2. Nel menu a discesa, è possibile filtrare i file/cartelle in base ai criteri seguenti:
    • Parola chiave: inserire il nome di un qualsiasi file o cartella Nelle parole chiave, è possibile utilizzare i caratteri jolly [*] (per indicare uno o più caratteri) e il simbolo [?] (per indicare un carattere casuale). Ad esempio, [a*n] può rappresentare [action], [attention] o [an].
    • Tipo di file: scegliere fra Qualsiasi, Qualsiasi cartella o Qualsiasi file per ridefinire la ricerca.
    • Dimensioni (KB): scegliere Qualsiasi o selezionare è uguale a, è maggiore di, è minore di, quindi inserire un valore.
    • Data modificata: scegliere tra Oggi, Ieri, Settimana scorsa o Mese scorso o Personalizza per inserire una data di inizio e di fine per trovare le versioni che rientrano in uno specifico periodo.
  3. Per applicare i filtri, fare clic su Filtro.
  4. Per applicare gli stessi filtri in un'altra versione di backup, è sufficiente passare alla versione desiderata con la barra della timeline.
  5. Per reimpostare la ricerca filtrata, fare clic sul pulsante Reset.

Nota:

  • Backup di Explorer è disponibile solo su attività di backup multiversione.
  • Backup di Explorer presenta funzioni limitate quando la destinazione di backup è impostata in DSM 5.1 o versioni inferiori.
  • La configurazione di sistema e le applicazioni possono essere ripristinate mediante Ripristina.
  • La funzione di filtro è disponibile solo su attività di backup non crittografate.
  • I file/cartelle recuperati utilizzando Copia in... o Scarica perderanno le impostazioni di privilegi originali.

Per ripristinare attività di backup da un'altra destinazione di backup:

  1. Fare clic su Ripristina, nell'angolo in basso a sinistra, e selezionare Dati o LUN*.
  2. È possibile selezionare e ripristinare il tipo di attività di backup:
    • Per ripristinare il backup di dati: Fare clic su Dati > Ripristina da repository esistenti e selezionare la destinazione di backup desiderata per ripristinare i dati.
    • Per ripristinare il backup LUN: Fare clic su LUN*.
  3. Seguire le istruzioni della procedura guidata. Se l'attività da ripristinare è criptata, è necessaria la password/chiave di crittografia.
  4. Fare clic su Applica per terminare il ripristino.

Nota:

  • Con Embedded DataStation è possibile solo ripristinare le cartelle condivise nei dispositivi esterni EXT3/EXT4.
  • Embedded DataStation non supporta il ripristino di applicazioni senza un dispositivo esterno EXT3/EXT4.

____
* Disponibile solo su alcuni modelli.
