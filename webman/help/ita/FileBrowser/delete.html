Eliminazione di file o cartelle

  1. Selezionare i file e le cartelle che si desidera eliminare. (tenere premuto i tasti "Shift" o "Ctrl" mentre si effettua la selezione.)
  2. Eseguire una delle seguenti azioni:
    • Scegliere Elimina dal menu Azione.
    • Fare clic con il pulsante destro su file o cartelle e scegliere Elimina.
