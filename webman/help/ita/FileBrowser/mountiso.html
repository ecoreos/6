Montare unità virtuali

È possibile montare le unità virtuali per accedere ai contenuti delle immagini disco con il File Station.

Per montare una unità virtuale:

  1. Selezionare un'immagine del disco e fare clic su Strumenti > Monta unità virtuale.
  2. Fare clic su Sfoglia per selezionare o creare una cartella di destinazione vuota su DiskStation per il montaggio dell'unità virtuale.
  3. Selezionare Montaggio automatico all'avvio per consentire a DiskStation di montare l'unità virtuale tutte le volte che il sistema viene avviato o riavviato. Fare clic su Monta per continuare. A questo punto è possibile passare alla cartella di destinazione, dove è montata l'unità virtuale, e accedere i contenuti dell'immagine disco.

Nota:

le unità virtuali e i file .iso possono essere montati solo sulle sottocartelle nelle cartelle condivise.

Per smontare una unità virtuale:

  1. Fare clic con il tasto destro del mouse sulla cartella di destinazione in cui è montata l'unità virtuale, quindi selezionare Smonta.
    Nota:
  1. Formati immagine disco supportati: file ".iso" utilizzando il file system ISO 9660 e UDF.
  2. Il numero massimo di unità virtuali simultanee è 16.
  3. Il montaggio delle unità virtuali è supportato solo nella modalità privilegi elevati. Per usare questa funzione, andare nel pacchetto Docker del DSM host, selezionare il Docker DSM per cui abilitare le cartelle condivise crittografate, fare clic su Modifica e spuntare le caselle accanto Esegui contenitore utilizzando i privilegi elevati.
