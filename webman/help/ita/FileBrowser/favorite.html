Creare i preferiti o i collegamenti del desktop

Con File Station è possibile aggiungere le cartelle di DiskStation ai Preferiti, o creare collegamenti del desktop per i file o cartelle di DiskStation.

Aggiungere cartelle ai Preferiti

Per le sottocartelle di DiskStation utilizzate con maggiore frequenza è possibile aggiungere dei collegamenti nella sezione Preferiti nella parte in alto a sinistra del riquadro per un accesso più rapido.

Per aggiungere una cartella di DiskStation ai Preferiti:

  1. Selezionare la cartella, quindi procedere come segue:
    • Scegliere il menu Azione o fare clic con il tasto destro sulla cartella, quindi scegliere Crea collegamento > Preferiti.
    • Se in precedenza è già stata aggiunta una cartella ai Preferiti, la sezione Preferiti sarà visualizzata nell'angolo superiore sinistro. Trascinare la cartella sulla sezione per creare un collegamento.
  2. Nella finestra che appare, nominare il collegamento e fare clic su OK.

Per riorganizzare l'ordine dei collegamenti in Preferiti:

Trascinare il collegamento nella sezione Preferiti per posizionarlo in un ordine diverso.

Per rinominare un collegamento:

  1. Fare clic con il pulsante destro sul collegamento e scegliere Rinomina.
  2. Nella finestra che appare, rinominare il preferito e fare clic su OK.

Per eliminare un collegamento:

Fare clic con il pulsante destro sul collegamento, scegliere Rimuovi dai Preferiti, quindi fare clic su Sì.

Per cancellare tutti i collegamenti che collegano a percorsi di cartelle non esistenti:

Fare clic con il pulsante destro su Preferiti nella parte superiore del riquadro di sinistra, quindi scegliere Cancella preferiti non validi.

Creare collegamenti del desktop per file e cartelle

Per i file o le sottocartelle di DiskStation utilizzati con maggiore frequenza è possibile creare dei collegamenti sul desktop per un accesso più rapido.

Per creare un collegamento sul desktop per un file o cartella di DiskStation:

Fare clic con il pulsante destro sul file o cartella nell'area principale, quindi procedere come segue:

  • Se si sta creando un collegamento del desktop per un file, scegliere Crea collegamento sul desktop.
  • Se si sta creando un collegamento del desktop per una cartella, scegliere Crea collegamento > Desktop DSM.
