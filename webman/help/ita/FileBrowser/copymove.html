Copiare o spostare file o cartelle

File Station fornisce vari metodi di trasferimento di file/cartelle tra due DiskStation, in un DiskStation, oppure su un computer locale.

Trascinare e rilasciare file/cartelle tra browser

Senza alcuna complessa configurazione di protocolli di trasferimento di file, è possibile caricare direttamente file/cartelle mediante File Station da un DiskStation a un altro.

Per abilitare il trascinamento tra browser:

  1. Andare su Impostazioni > Generale > File Station.
  2. Spuntare Abilita trascinamento della selezione tra browser.
  3. Fare clic su OK.
  4. Aprire due browser Web dello stesso tipo (es., Firefox) e connetterli direttamente al DiskStation sorgente e di destinazione.
  5. Avviare File Station su entrambi i DiskStation. Adesso sarà possibile trascinare e rilasciare direttamente file/cartelle tramite File Station.

Nota:

  • È possibile anche trascire file/cartelle tra due schede dello stesso browser.
  • Internet Explorer non supporta la funzione di trascinamento tra bowser o tra schede.
  • se più file/cartelle sono selezionati per il trascinamento, verranno compressi in un archivio per il caricamento.
  • Il limite di dimensioni del file per il trascinamento varia in base ai browser Web: Safari (1 GB).

Copiare o spostare file/cartelle in DiskStation o su un computer locale

Seguire i passaggi di seguito per spostare o copiare file o cartelle in DiskStation o sul computer locale.

Metodo 1:

  1. Selezionare i file o le cartelle che si desidera spostare o copiare. Tenere premuto i tasti "Shift" o "Ctrl" mentre si effettua la selezione.
  2. Scegliere Taglia o Copia dal menu a discesa Azione.
  3. Andare alla cartella di destinazione su DiskStation o sul computer.
  4. Scegliere Incolla - Sovrascrivi o Incolla - Ignora dal menu a discesa Azione in relazione a cosa si desidera fare, se sovrascrivere i file o le cartelle con lo stesso nome nella cartella di destinazione o meno.

Metodo 2:

  1. Selezionare i file o le cartelle che si desidera spostare o copiare. Tenere premuto i tasti "Shift" o "Ctrl" mentre si effettua la selezione.
  2. Trascinare qualsiasi file o cartella dalla sezione principale a un'altra cartella nel pannello a sinistra.
  3. Nel menu pop-up visualizzato, scegliere Copia - Sovrascrivi, Copia - Ignora, Sposta - Sovrascrivi o Sposta - Ignora per sovrascrivere i file o le cartelle con lo stesso nome nella cartella di destinazione.

Metodo 3

  1. Selezionare i file o le cartelle che si desidera spostare o copiare. Tenere premuto i tasti "Shift" o "Ctrl" mentre si effettua la selezione.
  2. Premere Ctrl + X per tagliare o Ctrl + C per copiare i file/cartelle. Per gli utenti Mac, premere Comando + X e Comando + C.
  3. Andare alla cartella di destinazione su DiskStation o sul computer.
  4. Premere Ctrl + V per incollare i file/cartelle. (Comando + V per Mac.)
  5. Nella finestra di dialogo visualizzata, scegliere se sovrascrivere o ignorare i file con lo stesso nome nella cartella di destinazione.

Collegamenti da tastiera:

Quando vengono spostati/copiati i file e le cartelle in File Station potrebbero essere utili i seguenti collegamenti da tastiera:

  • F2: Rinomina
  • Del: Elimina
  • Ctrl + A: Seleziona tutto
  • Crtl + X: Taglia
  • Crtl + C: Copia
  • Crtl + V: Incolla

Nota:

  • quando si copiano o spostano file o cartelle in una directory di un file system FAT, i caratteri \ *?">| del nome file saranno sostituiti dal simbolo –, e i Metadati dei file Mac saranno eliminati.
  • Se i file vengono tagliati dal computer locale e incollati in un delle cartelle di DiskStation, i file saranno conservati ancora nel computer locale.
  • Quando si sposta un file o una cartella, il "proprietario" resta lo stesso. Tuttavia, quando si copia un file o una cartella, il "proprietario" cambia nell'utente che ha eseguito la copia.
