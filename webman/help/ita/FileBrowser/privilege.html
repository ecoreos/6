Vedere o cambiare le autorizzazioni di File/Cartelle

Con File Station, è possibile impostare i privilegi di accesso a file e sottocartelle. Questa funzione è particolarmente utile quando si desidera consentire l'accesso a gruppi di utenti diversi a sottocartelle differenti in una cartella condivisa.

Per consentire a utenti e gruppi di accedere a file e sottocartelle:

  1. selezionare i file e le cartelle per le quali si desidera impostare i privilegi di accesso.
  2. Fare clic con il tasto destro su uno di file o cartelle o scegliere il menu Azione, quindi scegliere Proprietà.
  3. Nella sezione Proprietario , scegliere il proprietario di file e cartelle selezionati dal menu a discesa Proprietario.
  4. Fare clic sulla scheda Permesso e selezionare un utente o un gruppo. Fare clic su Modifica per aprire Controllo autorizzazione.
  5. Per gestire i permessi ACL per il file o la cartella, eseguire una delle seguenti azioni nella finestra Editori autorizzazioni:
    1. Utente o gruppo: Specificare l'utente o gruppo di cui si desidera personalizzare i permessi.
    2. Eredita da: Solo per visualizzazione. Visualizza le informazioni per verificare se l'autorizzazione è ereditata (da una cartella principale) o esplicita (indicato con Nessuno).
    3. Tipo: scegliere Consenti o Rifiuta per concedere o negare l'autorizzazione all'utente o gruppo.
    4. Applica a: se si sta creando una voce di autorizzazione per una cartella, selezionare le caselle di controllo per applicare la voce alla cartella, alle cartelle (o Cartelle secondarie) o file (o File secondari) in questa cartella o tutti i file e cartelle contenuti nella cartella (o Tutti i discendenti).
    5. Amministrazione: selezionare Leggi autorizzazioni, Cambia autorizzazioni o Diventa proprietario per specificare le impostazioni del permesso di accesso al gruppo o all'utente per la voce.
    6. Lettura o Scrittura: selezionare le caselle di controllo in queste sezioni per modificare l'autorizzazione dell'utente o del gruppo per il file o cartella.
  6. Fare clic su OK.

Nota. Non è consentito modificare i privilegi di accesso a file e sottocartelle nella cartella condivisa photo.
