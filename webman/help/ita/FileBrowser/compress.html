Compressione di file o cartelle

  1. Scegliere file o cartelle da DiskStation nell'area principale.
  2. Scegliere Comprimi in “(nome file)”.zip dal menu a discesa Azione per avviare la compressione.
  3. Se non è possibile visualizzare il file zip al termine della compressione, scegliere Aggiorna dal menu a discesa Azione per vedere il file.
  4. Se si desidera rinominare, codificare l'archivio o archiviarlo utilizzando livelli o modalità di compressione differenti, scegliere invece Aggiungi all'archivio....
    Nota:
  1. non è possibile comprimere file o cartelle sul proprio computer utilizzando File Station.
  2. Formati di archiviazione supportati: .zip, .7z.
  3. Quando si sceglie l'estensione .zip come Formato dell'archivio su Aggiungi all'archivio..., è possibile selezionare un codice pagina desiderato per il file .zip.
