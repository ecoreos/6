Modifica delle foto

Su File Station, è possibile modificare foto con Aviary Editor, un editor foto esterno basato su Web.

Per modificare una foto con Aviary Editor:

  1. Accedere alla cartella dove è posizionata la foto.
  2. Con il tasto destro del mouse, fare clic sulla foto e scegliere Modifica in Aviary.
  3. Aviary Editor verrà aperto in una nuova finestra del browser. È possibile vedere la foto e modificarla con gli strumenti forniti.
  4. Al termine del photo-editing, fare clic sul pulsante in alto nella barra degli strumenti per salvare tutte le modifiche apportate alla foto.

Nota:

  • Con Aviary Editor è possibile modificare le foto nei seguenti formati: .jpg, .jpeg, .jpe, .png
  • Le dimensioni file massime consentite da Aviary Editor sono di 5 MB.
  • Aviary Editor richiede le impostazioni di rete per consentire la corretta configurazione di DiskStation. Per la risoluzione dei problemi, consultare le impostazioni seguenti:
    • Verificare che DiskStation sia accessibile tramite QuickConnect (su Pannello di controllo > QuickConnect) oppure consenta di recuperare un indirizzo IP esterno (su Pannello di controllo > Accesso esterno > DDNS, o su Pannello di controllo > Rete > Interfaccia di rete > PPPoE).
    • Se DiskStation è posizionato dietro un router, accertarsi di aver specificato le corrette regole di inoltro porta (su Pannello di controllo > Accesso esterno > Configurazione router).
    • Se si riscontrano ancora problemi per aprire gli editor di foto, potrebbe essere necessario specificare la porta del router per DiskStation (su Pannello di controllo > Accesso esterno > Avanzate).
