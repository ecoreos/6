Requisiti File Station

Per utilizzare appieno le caratteristiche di File Station, occorrono il più recente browser web Java e Flash Player.

  • Browser: Internet Explorer 8 o successivo, Firefox 3.6 o successivo, Safari 5.0 o successivo, Chrome; con AJAX e JavaScript abilitati
  • Java (per sfogliare le cartelle sul computer locale): Java Runtime Environment (JRE) 5 o successivo
  • Flash: Flash Player 9.0.28 o successivo
