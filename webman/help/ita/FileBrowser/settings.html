Gestione delle impostazioni di File Station

Per gestire le impostazioni di File Station, aprire File Station e fare clic sul pulsante Impostazioni.

Generale

Da qui è possibile abilitare i registri File Station per registrare le attività di tutti gli utenti. Le attività dell'utente includono: Crea cartella, Carica, Download, Elimina, Rinomina, Sposta, Copia e Insieme proprietà.

Per attivare il registro File Station:

  1. Per registrare le attività di tutti gli utenti, selezionare Attiva registro File Station.
  2. Fare clic su OK.

Per visualizzare il registro File Station:

  1. Fare clic su Visualizza registri che riporta al Log Center.
  2. Fare clic sulla scheda Trasferimento file.

Per sovrascrivere automaticamente i file utilizzando il caricamento per trascinamento o i collegamenti da tastiera:

Selezionare Sovrascrive sempre i file quando vengono trascinati sul browser web o incollati con i collegamenti da tastiera per sovrascrivere automaticamente i file con lo stesso nome quando i file vengono trascinati dal desktop del computer al browser web, o i file vengono incollati in File Station utilizzando i collegamenti da tastiera.

Per abilitare la funzione intelligente di trascinamento e rilascio:

Quando si trascinano e rilasciano file in File Station, questa funzione determinerà automaticamente se il trasferimento è un'azione di copia o di spostamento.

  1. Selezionare Abilitare un trascina e rilascia intelligente.
  2. Selezionare un'azione da eseguire sempre nel caso in cui un altro utente, contemporaneamente, trasferisce file con lo stesso nome nella stessa posizione:
    • Sovrascrivi: i file trasferiti sovrascriveranno i file con lo stesso nome.
    • Ignora: i file trasferiti saranno ignorati e non sovrascriveranno i file con lo stesso nome.
  3. Fare clic su OK.

Nota:

per modificare l'azione di copia o di spostamento determinata con il trascinamento e rilascio intelligente, è possibile applicare i seguenti collegamenti rapidi a seconda delle specifiche esigenze:

  • Per passare a un'azione di copia: Premere Ctrl durante il trascinamento e il rilascio.
  • Per passare a un'azione di spostamento: Premere Maiusc durante il trascinamento e il rilascio.

Per scorrere e gestire i file dal computer locale con File Station:

  1. Selezionare Scorri file dal computer locale con File Station.
  2. Fare clic su OK.

Condividere l'elenco utenti DSM con utenti non-admin:

Quest'opzione consente a utenti non-admin di accedere all'elenco completo di utenti DSM per ricercare i file di specifici utenti/gruppi DSM o per condividere link di file con altri utenti DSM.

  1. Selezionare Condividere l'elenco utenti DSM con utenti non-admin.
  2. Fare clic su OK.

Applicazione di autorizzazioni UNIX predefinite

Fare clic su Impostazioni avanzate e mettere il segno di spunta sull'opzione per applicare le autorizzazioni UNIX predefinite quando si caricano o creano file e cartelle. Le autorizzazioni applicate sono le stesse applicate dal comando UNIX umask. Quando questa opzione è abilitata, le autorizzazioni UNIX sono 644 per i file e 755 per le cartelle. Quando questa opzione è disabilitata, le autorizzazioni UNIX sono 666 per i file e 777 per le cartelle. Il valore unmask predefinito è 022.

Nota:

  • per le cartelle condivise abilitate per Windows ACL (tutte le condivisioni escluse le "foto" e le condivisioni sulle unità esterne), eseguire il comando chmod sul client Linux o FTP per cambiare la cartella e i tipi di permesso dei file da Windows ACL a UNIX.
  • l'abilitazione di questa funzione può causare problemi di autorizzazioni non coerenti tra diversi protocolli. Per evitare incoerenze, consigliamo di lasciare questa opzione disabilitata.

Montaggio/Connessioni

Per impostazione predefinita, tutti gli utenti possono montare cartelle remote e unità virtuali. Qui è possibile limitare l'accesso alla gestione della cartella remota o dell'unità virtuale ai soli amministratori.

Nota:

utenti non-amministratori possono montare le cartelle remote solo all'interno della rispettiva cartella home.

Per consentire a utenti differenti di montare cartelle remote o unità virtuali:

  1. Scegliere Tutti gli utenti o Administrators nella sezione Cartella remota o Unità virtuale.
  2. Fare clic su OK.

Per consentire ai seguenti utenti di montare i server e i servizi cloud:

  1. Scegliere Tutti gli utenti, Administrators o Utente specifico.
  2. Scegliendo Utente specifico verrà visualizzata una finestra nella quale sarà possibile configurare i permessi di montaggio per diversi utenti. Per abilitare l'accesso per un utente specifico, verificare che la casella sia selezionata nella colonna Abilitato. Per disabilitare l'accesso per un utente specifico, verificare che la casella sia deselezionata nella colonna Abilitato Al termine, fare clic su OK per chiudere la finestra.

    3. Nota:

    • gli utenti che appartengono al gruppo administrators possono utilizzare questa funzione, ma non possono apportare modifiche. Gli utenti che effettuano l'accesso usando guest non possono né utilizzare questa funzione né apportare modifiche. I permessi per tutti gli altri utenti possono essere impostati qui.
  3. Fare clic su OK.

Collegamenti condivisi

Per impostazione predefinita, tutti gli utenti possono condividere collegamenti ai file. Da qui è possibile limitare l'accesso alla gestione dei collegamenti condivisi ai soli amministratori.

Per consentire a diversi utenti di condividere i link dei file:

  1. Scegliere Tutti gli utenti, Amministratori o Utenti specifici.
  2. Fare clic su OK.

Per aprire i file HTML come testo semplice:

  1. Selezionare la casella Apri sempre i file HTML usando il formato testo semplice. Se questa opzione è abilitata, i file HTML (ad es. i file HTM, HTML, JSP, XHTML) sono visualizzati usando il formato testo semplice quando gli utenti li aprono mediante un link di condivisione di file. Questa opzione aiuta a prevenire che parti malintenzionate possano prendere il controllo delle sessioni di accesso dell'utente usando script nascosti nei file HTML.
  2. Fare clic su OK.

Limite velocità

L'abilitazione del limite di velocità consente di controllare la larghezza di banda massima utilizzabile per utenti e gruppi specifici durante il trasferimento di dati tramite File Station.

Per abilitare il limite di velocità:

  1. Spuntare la casella contrassegnata Abilita limite velocità o Abilita il limite di velocità in base a una pianificazione se si desidera implementare un limite di velocità solo in orario specifico. Si tratta di un'impostazione del server e annullerà le impostazioni dei singoli utenti e gruppi. Scegliendo Nessun limite di velocità si disabilita l'impostazione del limite di velocità di tutti gli utenti e gruppi.
  2. Fare clic su Impostazioni limite velocità per modificare le impostazioni per i singoli utenti o gruppi. Per ogni utente e gruppo, è possibile selezionare una delle seguenti:
    • Applica impostazioni gruppo: le impostazioni del limite di velocità saranno le stesse del gruppo a cui l'utente appartiene.
    • Nessun limite di velocità: nessun limite di velocità sarà applicato quando i membri del gruppo trasferiscono file con il servizio.
    • Costante: un limite di velocità costante fisso sarà applicato all'utente o ai membri del gruppo. I limiti di velocità di upload e download possono essere specificati nei campi a sinistra.
    • Variabile: due diversi limiti di velocità possono essere specificati e applicati all'utente o ai membri del gruppo in base alla programmazione. Fare clic sul pulsante Personalizza per modificare le impostazioni del limite di velocità e impostare una programmazione.
  3. Fare clic su Applica.

Nota:

  • Per ulteriori informazioni sulle dinamiche dei limiti di velocità gruppo, consultare qui.
  • Quando viene impostato il limite di velocità per un utente, la sezione Risultato visualizza il limite di velocità attuale dell'utente in base alle sue impostazioni individuali e di gruppo. Ad esempio, se l'utente "Achille" ha un limite di download individuale di 100 KB/s, ma appartiene a un gruppo utente con un limite di download di 500 KB/s, la sezione Risultati mostrerà "0/500".

Sicurezza

Qui è possibile trovare le seguenti impostazioni di sicurezza.

Per aprire i file HTML come testo semplice:

  1. Selezionare la casella Apri sempre i file HTML usando il formato testo semplice. Se quest'opzione è abilitata, i file HTML (ad es. i file HTM, HTML, JSP, XHTML) sono visualizzati usando il formato testo semplice quando aperti in File Station. Questa opzione aiuta a prevenire che parti malintenzionate possano prendere il controllo delle sessioni di accesso dell'utente usando script nascosti nei file HTML.

    2. Nota:

    • non è un'impostazione globale. Ogni utente può scegliere se abilitare o meno questa opzione in base alle proprie necessità.
  2. Fare clic su OK.
