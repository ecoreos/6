Apri un file in una nuova finestra

Aprire un file su DiskStation in una nuova scheda o finestra del browser web per vedere l'anteprima.

Per aprire un file in una nuova finestra:

  1. Selezionare il file di cui si desidera vedere l'anteprima.
  2. Eseguire una delle seguenti azioni:
    • Scegliere Apri in una nuova finestra dal menu Azione.
    • Fare clic con il pulsante destro e scegliere Apri in una nuova finestra.

Nota: le impostazioni del browser decideranno quale tipo di file è possibile visualizzare direttamente.
