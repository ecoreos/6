Caricare file o cartelle

Usare uno dei seguenti metodi per caricare file o cartelle dal computer su DiskStation.

Per disporre dell'accesso completo alle funzioni di caricamento descritte di seguito, installare la versione più recente di Java Runtime Environment (JRE) e Flash Player.

Metodo 1:

  1. Dalla parte superiore del pannello a sinistra, selezionare una cartella di destinazione su DiskStation.
  2. Fare clic su Carica, scegliere Carica - Ignora o Carica - Sovrascrivi e selezionare i file o cartelle dal computer locale.

Metodo 2:

  1. Andare su Impostazioni, selezionare Scorri file dal computer locale con File Station e aggiornare la pagina web per visualizzare le cartelle del computer locale nella parte inferiore del pannello di sinistra.

    2. Nota:

    L'opzione Scorri file dal computer locale con File Station diventa effettiva solo dopo aver installato e abilitato JRE sul computer.

  2. Nella parte inferiore del pannello di sinistra, selezionare una cartella dal computer locale. L'area principale mostrerà i contenuti della cartella selezionata.
  3. Scegliere i file o le cartelle che si desiderano nell'area principale, quindi selezionare Carica.
  4. Scegliere una cartella di destinazione su DiskStation nella finestra che appare.
  5. In alto a destra nella finestra, selezionare Sovrascrivi se si desidera sovrascrivere i file o le cartelle con lo stesso nome nella cartella di destinazione. Altrimenti, scegliere Ignora.
  6. Fare clic su OK per avviare il caricamento.

Metodo 3:

  1. Andare su Impostazioni, selezionare Scorri file dal computer locale con File Station e aggiornare la pagina web per visualizzare le cartelle del computer locale nella parte inferiore del pannello di sinistra.
  2. Nel pannello a sinistra, trascinare una cartella dal computer (la parte inferiore) su qualsiasi cartella su DiskStation (parte superiore).
  3. Nella menu popup visualizzato, scegliere se sovrascrivere o ignorare i file/cartelle con lo stesso nome nella cartella di destinazione.

Metodo 4:

  1. Dalla parte superiore del pannello a sinistra, selezionare una cartella di destinazione su DiskStation.
  2. Selezionare i file sul desktop o nella ricerca file (ad esempio, Esplora risorse o Mac Finder), quindi trascinarli in File Station nel browser web.
  3. Nella finestra di dialogo visualizzata, scegliere se sovrascrivere o ignorare i file con lo stesso nome nella cartella di destinazione.

    4. Nota:

    • Metodo 4 supportato solo su browser specifici: Chrome, Firefox, Safari 5.1 o superiori. Tuttavia, non è supportato per Mac quando Scorri file dal computer locale con File Station è selezionato.
    • È anche possibile trascinare le cartelle o i file vuoti, ma solo quando Scorri file dal computer locale con File Station è selezionato per il browser elencati in precedenza. In caso contrario, non è possibile caricare le cartelle e i file vuoti utilizzando il Metodo 4.
    • Se si utilizza Chrome per Linux e ci si connette al DSM via protocollo HTTPS, occorre importare un certificato di autorizzazione (su Pannello di controllo > Sicurezza > Certificato) per effettuare il caricamento usando Metodo 4.

Nota:

  1. è possibile aggiungere fino a 1000 file alla coda di caricamento.
  2. È possibile caricare file con dimensioni superiori a 4GB quando si utilizza Google Chrome, Firefox, Safari 5.1 o versioni successive, o quando Scorri file dal computer locale con File Station è selezionato. Altrimenti, è possibile caricare fino a 2 GB di file alla volta.
  3. Nella cartella condivisa non-crittata, il nome del file/cartella deve essere compreso entro 255 caratteri (fino a 80 caratteri per le lingue non latine); nella cartella condivisa crittata, invece, entro 143 caratteri (fino a 47 caratteri per le lingue non latine).
  4. Il nome file o cartella esclude i seguenti caratteri: \ / : * ? " > |.
  5. Un nome di file/cartella non può cominciare con "._".
  6. È possibile caricare file con dimensioni di 0KB solo dopo aver installato JRE.

Risoluzione dei problemi

Se non è possibile sfogliare gli elementi locali archiviati sul proprio Mac usando Safari correttamente, procedere come segue:

  1. Sul proprio Mac, aprire Safari e scegliere Preferenze dal menu Safari.
  2. Fare clic su Gestione impostazioni sito web nella scheda Sicurezza.
  3. In Java, scegliere il proprio "sito web" DSM.
  4. Selezionare Esegui in modalità insicura.
