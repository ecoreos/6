Creare richieste file

Le richieste file sono inviti di caricamento file inviati a utenti non DSM. Con i collegamenti delle richieste file, gli utenti non-DSM possono caricare direttamente più file su File Station in DiskStation, senza dovere possedere i corrispondenti account utente e privilegi di caricamento file.

Per creare i collegamenti richiesta file:

  1. Andare su File Station e selezionare una cartella come destinazione di caricamento.
  2. Fare clic su Azione oppure fare clic con il tasto destro del mouse sulla cartella di destinazione e selezionare Crea richiesta file.
  3. Nella finestra a comparsa, è possibile trovare/modificare le informazioni seguenti:
    • Percorso file: elencare il percorso della directory di destinazione.
    • Collega: elencare il collegamento richiesta file da condividere con gli utenti di destinazione.
    • Nome: inserire un nome per indicare l'identità del richiedente. Il valore predefinito è il nome utente.
    • Messaggio: personalizzare il messaggio per gli utenti di destinazione.
    • Abilita protezione con password: selezionare quest'opzione per consentire solo agli utenti con password di caricare i file mediante il link creato. Specificare la password nella casella seguente.
    • Periodo validità: fare clic su questo pulsante per personalizzare la durata del collegamento richiesta file:
      • Configura ora di fine: selezionare quest'opzione per stabilire, in base a data ed ora, quando il link smette di funzionare per il caricamento file.
      • Configura ora di inizio: selezionare quest'opzione per stabilire, in base a data ed ora, quando il link inizia a funzionare per il caricamento file.
      • Numero di accessi consentiti: selezionare quest'opzione per stabilire quante volte è possibile accedere al link.
  4. Fare clic su Salva per terminare le impostazioni.

Per condividere i collegamenti di richiesta file:

Per individuare l'URL di richiesta file, andare su File Station > Strumenti > Gestione collegamenti condivisi e fare doppio clic sul link desiderato. Il collegamento può essere condiviso in uno dei modi seguenti:

  • Copiare semplicemente l'URL agli utenti di destinazione.
  • Inviare il link tramite E-mail predefinita (account e-mail DSM) oppure tramite Mail (applicazione e-mail di terze parti). Per gli utenti nuovi, seguire le istruzioni della procedura guidata per configurare un account e-mail per l'uso.

Per caricare i file:

Se viene richiesto di caricare i file tramite il link, procedere come segue:

  1. Aprire l'URL specificato nel computer locale o un dispositivo mobile (Android o iOS). Se richiesto, inserire la password.
  2. Nella pagina di caricamento, inserire un nome per indicare la propria identità.
  3. Semplicemente trascinare e rilasciare i file da caricare oppure fare clic su Aggiungi file per selezionare i file.
  4. Fare clic su Carica per completare la richiesta.

Nota:

  • le cartelle non possono essere caricate tramite un collegamento richiesta file.
  • Le cartelle all'interno della cartella condivisa "Web" non devono essere specificate come cartella di destinazione.
  • La funzione di caricamento varia con i browser Web:
    • Internet Explorer 8 e Internet Explorer 9 non supportano il caricamento file tramite trascinamento.
    • Chrome sui dispositivi iOS supporta solo il caricamento immagine.
    • Safari sui dispositivi iOS supporta solo il caricamento immagine e video.
    • I browser Web su dispositivi Android non sono soggetti ad alcuna limitazione di tipi di file.

Per individuare i file caricati:

i file caricati dallo stesso utente appariranno in una cartella identica all'interno della destinazione. Per impostazione predefinita, il nome cartella sarà Nome_Uploader (Nome_Richiedente).

Ad esempio, se i file sono caricati da Watson a Holmes, è possibile trovare i file nella cartella denominata Watson (Holmes).

Gestione dei collegamenti richiesta file

Per modificare i collegamenti richiesta file:

Dopo la creazione di un collegamento richiesta file, è possibile modificarne le impostazioni su File Station > Strumenti > Gestione collegamenti condivisi.

  1. Fare doppio clic sul link desiderato oppure selezionare il link e fare clic su Modifica.
  2. Nella finestra a comparsa, modificare le impostazioni pertinenti.
  3. Fare clic su Salva per applicare le nuove impostazioni.

Per eliminare i collegamenti richiesta file:

  1. Andare su File Station > Strumenti > Gestione collegamenti condivisi.
  2. Selezionare i collegamenti desiderati e fare clic su Elimina. Gli URL di richiesta file corrispondenti smetteranno immediatamente di funzionare.

Per assegnare i privilegi di compilazione richiesta:

Come admin o come utente appartenente al gruppo administrators, è possibile assegnare i privilegi a utenti per consentire a questi di creare i collegamenti richiesta file.

  1. Andare su File Station > Impostazioni > Collegamenticondivisi > Collegamenti richiesta file.
  2. Selezionare il tipo di utenti per consentire i privilegi di file richiesta.
  3. Fare clic su OK per salvare le impostazioni.
