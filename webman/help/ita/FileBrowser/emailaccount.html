Invia come allegati e-mail

È possibile inviare direttamente file come allegati e-mail nella File Station facendo clic con il tasto destro sui file scelti e quindi selezionando Invia come allegati e-mail. Verrà visualizzata un'e-mail con alcuni importanti elementi:

  • A: Se si sono sincronizzati dei contatti per l'account e-mail, verrà visualizzato un elenco dei contatti in cui verranno visualizzati i contatti sincronizzati associati all'account e-mail.
  • Da: È possibile selezionare l'account e-mail predefinito o un altro account e-mail aggiunto. L'elenco dei contatti passerà automaticamente all'account e-mail selezionato.
  • Allegati: Gli allegati selezionati verranno visualizzati nella parte inferiore dell'e-mail. È possibile aggiungere altri allegati con il pulsante degli allegati.

Nota.

  • Solo i file possono essere aggiunti come allegati.
  • È possibile caricare fino a 10 file alla volta.
  • La dimensione totale dei file massima di tutti gli allegati in un'unica e-mail dipende dal servizio e-mail in uso. Se non si è sicuri, si consiglia di limitare la dimensione totale dei file di tutti gli allegati a 25 MB.
