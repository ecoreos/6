File Station

File Station è uno strumento di gestione file facile da usare che permette agli utenti DSM in Internet di accedere alle cartelle su DiskStation con i propri browser web. Aprendo due o più finestre File Station, è possibile gestire tutti i dati su DiskStation trascinandoli tra File Station differenti.
