Ricerca di file o cartelle

Il Filtro di File Station consente di cercare e visualizzare i file o le cartelle che soddisfano i criteri necessari. È possibile filtrare i file o le cartelle nella cartella corrente o eseguire una ricerca avanzata in tutte le sottocartelle per ottenere risultati della ricerca più dettagliati.

Per cercare file o cartelle nella cartella corrente:

  1. Utilizzare File Station per accedere alla cartella con i contenuti da cercare.
  2. Nel campo Filtro, nell'angolo in alto a destra, inserire una parola chiave e vedere i risultati filtrati nell'area principale della finestra.

Per eseguire una ricerca avanzata:

  1. Utilizzare File Station per accedere alla cartella con i contenuti da cercare.
  2. Fare clic sull'icona della lente di ingrandimento accanto al campo Filtro, nell'angolo in alto a destra, per aprire il menu di ricerca avanzata.
  3. Specificare i criteri di ricerca seguenti oppure sceglierli dai menu a discesa:
    • Parola chiave: inserire una parola chiave per cercare file o cartelle con nomi corrispondenti.

      Nota:

      • è possibile utilizzare simboli jolly [*] (che indica zero o più caratteri) e [?] (che indica un carattere casuale) nella parola chiave. Ad esempio, [a*n] può rappresentare [action], [attention] o [an].
      • È possibile utilizzare gli spazi per separare le parole chiave. Ad esempio, la parola chiave [File Station] porta risultati di ricerca contenenti [File], [Station], o entrambi.
      • È possibile inserire le virgolette ["] all'inizio e alla fine della parola chiave per eseguire la ricerca esatta del termine, con le corrispondenze di maiuscole/minuscole. Ad esempio, ["Synology Inc."] rappresenta [Synology Inc.], ma non può rappresentare [Synology Incorporation].
      • Gli operatori seguenti possono essere usati per ridefinire i risultati di ricerca:
        Operatore Esempio Uso
        && [X] && [Y]

        || [X] || [Y]
      • Elencare le corrispondenze contenenti [X], [Y] o entrambi
      • Le corrispondenze contenenti entrambi [X] e [Y] saranno elencate nella parte superiore

        + +[X]

        - -[X]

    • Abilita ricerca contenuto file: ricercare i file indicizzati con il contenuto file o i metadati corrispondenti alle parole chiave.
    • Posizione: scegliere dal menu a discesa per impostare l'intervallo e la profondità della ricerca avanzata. È anche possibile personalizzare l'ambito della ricerca selezionando qualsiasi altra cartella condivisa elencata.
    • Tipo di file: scegliere Estensione per limitare la ricerca a tipi di file specifici oppure scegliere Qualsiasi cartella per ricercare solo le cartelle.
    • Data: scegliere Data ultima modifica, Data creazione o Ultimo accesso, quindi scegliere una data dal calendario per cercare i file con le date corrispondenti.
    • Dimensioni (MB): scegliere tra uguale, è maggiore di e è minore di, quindi inserire una dimensione per cercare i file con dimensioni corrispondenti. (Scegliendo uguale, nei risultati della ricerca saranno inclusi anche i file con approssimativamente la stessa dimensione specificata.)
    • Proprietario/Gruppo: scegliere Proprietario o Gruppo e quindi scegliere dal menu a discesa i file o le cartelle che appartengono al gruppo o al proprietario selezionato.

      Nota:

      • Per eseguire la ricerca avanzata, è necessario specificare la Posizione e almeno un altro criterio di ricerca.
      • Se una cartella ISO o una cartella remota viene montata in una cartella condivisa indicizzata, non sarà possibile individuare i file interni attraverso una ricerca avanzata.
  4. Fare clic su Cerca per avviare la ricerca, oppure fare clic su Reimposta e specificare di nuovo il criterio di ricerca. Al termine della ricerca, i file corrispondenti verranno elencati nell'area principale di File Station.
