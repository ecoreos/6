Accedere a Synology NAS da Internet e condividere i file

Panoramica

Con QuickConnect, è possibile connettersi facilmente alla Synology NAS da Internet senza dover impostare le regole di invio porta o altre impostazioni di rete complicate. Con QuickConnect è possibile connettersi tramite un indirizzo facilmente personalizzabile come Quickconnect.to/example.

1. Configura QuickConnect

  1. Andare su Pannello di controllo > QuickConnect.
  2. Selezionare la casella Abilita QuickConnect.
  3. Se non si dispone di un account Synology, fare clic su Accedere o registrare un account Synology. Inserire le informazioni necessarie e fare clic su OK. Se si dispone già di un account Synology, inserire le informazioni dell'account.
  4. Creare il QuickConnect ID nel campo QuickConnect ID. Quindi fare clic su Applica.
  5. Verranno visualizzate le informazioni sul collegamento QuickConnect. Utilizzare questi collegamenti per accedere a Synology NAS. In questo esempio, l'indirizzo per DSM è http://Quickconnect.to/YourID.
  6. Se non si visualizza il collegamento DSM QuickConnect è possibile fare clic su Avanzate e assicurarsi che DSM sia abilitato.
  7. Adesso è possibile utilizzare il collegamento DSM QuickConnect per accedere a Synology NAS.

2. Condividere file su NAS utilizzando QuickConnect

Abilitando QuickConnect è possibile condividere facilmente i file archiviati su Synology NAS con chiunque senza alcuna preoccupazione relativa all'invio porta.

  1. Prima abilitare la connessione HTTPS per condividere i file in modo sicuro. Andare su Pannello di controllo > Rete.
  2. Fare clic sulla casella Abilita connessione HTTPS, quindi fare clic su Applica.
  3. Abilitare quindi la condivisione di file per QuickConnect. Andare su Pannello di controllo > QuickConnect.
  4. Fare clic su Avanzate e assicurarsi che la casella di Condivisione file abbia il segno di spunta.

    5. Nota:

    Quando vengono configurate impostazioni QuickConnect avanzate, si consiglia di accedere a DSM con l'utilizzo di un indirizzo IP locale o del nome di dominio del Synology NAS.

  5. Si può ora condividere file utilizzando QuickConnect. Aprire File Station e fare clic con il destro sul file o sulla cartella da condividere. Quindi fare clic su Condividi collegamenti file nel menu contestuale.
  6. Viene creato un collegamento per la condivisione del file. Condividere questo collegamento con la famiglia e gli amici per scaricare la cartella/file.
