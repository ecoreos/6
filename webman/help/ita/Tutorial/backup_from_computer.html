Backup dei dati su PC/Mac utilizzando Cloud Station

Panoramica

Cloud Station Server non è soltanto un sofisticato software di sincronizzazione multipiattaforma: gli utenti possono utilizzare come mezzo intuitivo di backup per i propri personal computer. Come soluzione di backup in tempo reale, Cloud Station Backup è immediato e ottimizza la larghezza di banda, offrendo una migliore protezione con un massimo di 32 versioni cronologiche per singolo file.

Configurare Cloud Station Server su Synology NAS

  1. Andare su Centro pacchetti, individuare Cloud Station e fare clic su Installa.
  2. Avviare Cloud Station Server.
  3. Andare su Panoramica e fare clic su Scarica adesso in Cloud Station Backup. È anche possibile scaricare Cloud Station   dal Centro download Synology.
  4. Avviare l'installer sul computer e seguire le istruzioni per installare Cloud Station Backup.
  5. Dopo l'installazione, eseguire Cloud Station Backup sul computer. Fare clic su Avvia adesso.
  6. Inserire l'indirizzo per il Synology NAS che esegue Cloud Station Server (o l'ID QuickConnect), nome utente e password. È possibile anche fare clic sull'icona di ricerca a destra per consentire a Cloud Station Backup di ricercare automaticamente un altro Synology NAS nella LAN in uso. Fare clic su Avanti.
  7. Selezionare la fonte del backup e deselezionare tutte le sottocartelle da non sincronizzare.
  8. Selezionare una cartella in Synology NAS facendo clic su Seleziona. Cloud Station Backup creerà una cartella di backup nella destinazione selezionata con il nome del computer.
  9. Fare clic su Regole di backup per impostare i filtri o le regole di sincronizzazione.
  10. Fare clic su Avanti per rivedere le impostazioni di backup, quindi fare clic su  Fine.
  11. È possibile visualizzare l'avanzamento del backup nell'app principale.

Scaricare o ripristinare una versione precedente di un file sincronizzato da Cloud Station Server

  1. Andare su Cloud Station Server > Cronologia versione e individuare il file o la cartella da ripristinare.
  2. Selezionare il file recuperare e fare clic su Azione. Fare clic su Sfoglia le versioni precedenti. In caso di eliminazione o rimozione casuale di un file nel computer, fare clic su Ripristina per ripristinarlo.
  3. Scegliere la versione del file da scaricare e fare clic su Scarica. Oppure, per ripristinare la versione selezionata, fare clic su Ripristina e la versione corrente verrà sovrascritta.
