Archiviare i file sul Synology NAS da un computer Mac senza rete locale

Panoramica

Synology MAS è stato progettato per rendere più semplice e rapida l’archiviazione e condivisione dei file nella LAN, consentendo di accedere direttamente alle cartelle condivise e ai file sul Synology NAS senza dover eseguire tutte le volte l’accesso al DSM. Ad esempio, sarà possibile accedere ai file sul Synology NAS con Finder proprio come con altri dispositivi di rete.

Accedere alle cartelle condivise con Mac OS

  1. Aprire Finder sul computer Mac.
  2. Trovare la barra dei menu nella parte superiore dello schermo.
  3. Fare clic su Vai e selezionare Connetti al server.
  4. Immettere l’indirizzo IP per il Synology NAS, preceduto da “afp://”, ad esempio "afp://192.168.59.121". Fare clic su Connetti per continuare.
    5. Server preferiti: Premendo il pulsante con il simbolo più (+) è possibile designare gli indirizzi IP come server preferiti per usi futuri. Inoltre, è possibile fare clic sul pulsante dell’orologio per visualizzare un elenco di server verso cui è stata eseguita la connessione di recente.
  5. Immettere il nome utente e la password del DSM. L’account utente deve disporre dei privilegi di lettura/scrittura per la cartella condivisa a cui si desidera accedere. Fare clic su Connetti per continuare.
  6. In conclusione, se le credenziali del nome utente sono corrette, il Synology NAS sarà presente nell’elenco nella sezione Condivisi della finestra del Finder. Fare doppio clic sulla cartella condivisa per esplorarne i contenuti.
