Visualizzare i contenuti multimediali archiviati sul Synology NAS su schermi di grandi dimensioni

Panoramica

Il Synology NAS può essere utilizzato come server multimediale nella LAN consentendo ai dispositivi domestici, ad esempio TV/impianti stereo e Sony PlayStation3/Xbox 360, di riprodurre i proprio contenuti multimediali. Questi dispositivi sono adattatori multimediali digitali (DMA) conformi DLNA/UPnP. Per trasmettere i contenuti multimediali dal Synology NAS ai dispositivi DMA per la riproduzione, è necessario configurare le impostazioni di riproduzione multimediali. Ad esempio, la gestione delle impostazioni di navigazione dei file multimediali e il miglioramento della compatibilità dei file. Questa funzione è disponibile nel pacchetto Server media.

1. Installare Server media

Per installare Server media, andare su Centro pacchetti, trovare Server media, quindi fare clic su Installa.

2. Caricare i file multimediali sul Synology NAS

Le cartelle condivise predefinite, photo, music e video, vengono create e indicizzate automaticamente al termine dell'installazione di Server media sul NAS. Basta solamente caricare i file multimediali, ad esempio immagini, canzoni e video nelle cartelle corrispondenti per rendere accessibili questi file dai dispositivi DMA.

Nota: le cartelle condivise montate dai server remoti non possono essere aggiunte come cartelle indicizzate. È possibile aggiungere al Synology NAS fino a 100 cartelle indicizzate.

3. Configurare le impostazioni di navigazione per il lettore multimediale

È possibile decidere come scorrere i file multimediali sul Synology NAS utilizzando i dispositivi DMA.

  1. Andare su Menu principale > Server media.
  2. Fare clic su Impostazioni navigazione nel pannello sinistro.
  3. Per ulteriori informazioni, fare clic sul pulsante Guida nell'angolo superiore destro della finestra di Server media.

4. Compatibilità DMA migliorata

Se le TV DLNA/UPnP o i dispositivi DMA non riescono a riprodurre i contenuti multimediali archiviati sul NAS, è possibile provare a configurare le impostazioni di Compatibilità DMA del Server media.

Nota:
  1. Per verificare se il dispositivo DMA è installato correttamente e rilevato dalla NAS nella LAN, fare clic su Elenco dispositivi. Viene visualizzato un elenco di dispositivi DMA con le relative informazioni dispositivo.
  2. Assicurarsi di non aver selezionato Limita l'accesso al Server media per i nuovi dispositivi UPnP/DLNA rilevati. In caso contrario il Server media non sarà accessibile ai dispositivi DMA.

Per ulteriori informazioni relative al supporto tecnico per i file multimediali di altri formati, fare clic sul pulsante Guida nell'angolo superiore destro della finestra di Server media.

5. Per accedere ai file multimediali archiviati sul NAS con una TV DLNA

I dispositivi DLNA/UPnP e DMA possono essere collegati in modo semplice alla rete domestica. Una volta collegato sia il Synology NAS che i dispositivi DLNA alla rete domestica, è possibile scorrere le cartelle selezionate e ascoltare la musica o guardare i video dal NAS su uno schermo di grandi dimensioni.

Nota: per un elenco di TV DLNA/UPnP e dispositivi DMA supportati che sono stati verificati da Synology, consultare qui.

Il processo di impostazioni dipende dai dispositivi DMA conformi DLNA. Per ulteriori informazioni sull'accesso ai file multimediali archiviati sul NAS tramite DLNA, seguire le istruzioni dei dispositivi client.
