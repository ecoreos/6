Моментальные снимки

На данной странице можно делать, удалять и редактировать моментальные снимки, а также настраивать расписание создания моментальных снимков и параметры их хранения.

Для каждой общей папки или iSCSI LUN можно сохранять до 256 копий моментальных снимков одновременно. Моментальные снимки минимизируют потребление дискового пространства, сохраняя отдельные блоки, а не целые файлы. Дополнительное пространство может потребляться, когда обновляются или удаляются файлы в общих папках или блоках в ISCSI LUN.

Общая папка

Чтобы сделать моментальный снимок общей папки:

  1. Выберите общую папку, мгновенный снимок которой вы хотите сделать.
  2. Нажмите Сделать моментальный снимок.
  3. Введите описание снимка, который нужно сделать, в поле Описание.
  4. Установите/снимите флажок в поле Блокировать, чтобы определить, будет ли снимок игнорировать параметры хранения, когда будет запущено автоматическое удаление.
  5. Нажмите ОК, чтобы сделать снимок.

Примечание.

  • Среди параметров в меню Панель управления > Общая папка > Редактирование в моментальном снимке будут записаны только права доступа Windows ACL.

Для просмотра моментальных снимков общих папок:

  1. Выберите общую папку, моментальные снимки которой вы хотите просмотреть.
  2. Нажмите Список моментальных снимков.
  3. Выберите снимок и нажмите Просмотреть.

Примечание.

  • Просмотр моментальных снимков недоступен для зашифрованных общих папок.
  • Если вы не можете просмотреть снимок, выберите Панель управления > Общая папка > Редактирование и включите параметр Сделать моментальный снимок видимым.

Для удаления моментальных снимков общих папок:

  1. Выберите общую папку, моментальные снимки которой вы хотите удалить.
  2. Нажмите Список моментальных снимков.
  3. Выберите один или несколько снимков, которые вы хотите удалить, и нажмите Удалить. (Чтобы выбрать несколько элементов, нажмите и удерживайте клавишу Shift или Ctrl.)
  4. Появится сообщение с подтверждением. Нажмите Удалить для подтверждения удаления.

Для редактирования моментальных снимков общих папок:

  1. Выберите общую папку, моментальные снимки которой вы хотите отредактировать.
  2. Нажмите Список моментальных снимков.
  3. Выберите снимок, который необходимо отредактировать, и нажмите Редактировать.
  4. Можно изменить описание моментального снимка в поле Описание и установить/снять флажок в поле Блокировать, чтобы определить, будет ли снимок игнорировать параметры хранения, когда будет запущено автоматическое удаление.
  5. Нажмите OK, чтобы сохранить изменения.

Настройка расписания создания моментальных снимков:

В случае запланированных моментальных снимков инструмент "Моментальные снимки и репликация" будет автоматически делать снимки в заданное время и с заданной частотой, и вам не придется делать снимки вручную.

  1. Выберите общую папку.
  2. Нажмите Настроить расписание.
  3. На вкладке Расписание установите флажок в поле Включить расписание создания моментальных снимков и установите время и частоту создания моментальных снимков. Чтобы отменить создание моментальных снимков по расписанию для этой общей папки, снимите флажок.
  4. Нажмите OK, чтобы сохранить изменения.
  5. Появится сообщение с подтверждением. Нажмите OK для завершения настройки.

Примечание.

  • Настроенное расписание моментальных снимков не предусмотрено в меню Панель управления > Обновление и восстановление > Резервное копирование конфигурации.

Настройка параметров хранения запланированных моментальных снимков:

  1. Выберите общую папку.
  2. Нажмите Настроить расписание.
  3. Откройте вкладку Хранение.
  4. Настройте параметры хранения. Выберите один из следующих параметров:
    • Удалять старые моментальные снимки при достижении лимита: По достижению максимального количества моментальных снимков самые старые снимки будут перезаписаны более новыми.
    • Всегда сохранять моментальные снимки: По достижении максимального количества моментальных снимков инструмент "Моментальные снимки и репликация" прекратит создание новых снимков и не будет удалять старые.
    • Сохранять моментальные снимки в соответствии со следующей политикой: Можно настроить правила хранения для установленных диапазонов времени. Для получения более подробной информации см. раздел Политика хранения.
  5. Нажмите OK, чтобы сохранить изменения.

Примечание.

  • Изменение параметров хранения повлияет на эквивалентные параметры на странице Репликация.

iSCSI LUN

Создание моментального снимка iSCSI LUN:

  1. Выберите iSCSI LUN, моментальный снимок которого вы хотите сделать.
  2. Нажмите Сделать моментальный снимок.
  3. Введите описание снимка, который нужно сделать, в поле Описание.
  4. В раскрывающемся списке Блокировать выберите Да/Нет, чтобы определить, будет ли снимок игнорировать параметры хранения, когда будет запущено автоматическое удаление.
  5. Нажмите ОК, чтобы сделать снимок.

Удаление моментального снимка iSCSI LUN:

  1. Выберите iSCSI LUN, моментальные снимки которого вы хотите удалить.
  2. Нажмите Список моментальных снимков.
  3. Выберите один или несколько снимков, которые вы хотите удалить, и нажмите Удалить. (Чтобы выбрать несколько элементов, нажмите и удерживайте клавишу Shift или Ctrl.)
  4. Появится сообщение с подтверждением. Нажмите Да для подтверждения удаления.

Редактирование моментального снимка iSCSI LUN:

  1. Выберите iSCSI LUN, моментальные снимки которого вы хотите редактировать.
  2. Нажмите Список моментальных снимков.
  3. Выберите снимок, который необходимо отредактировать, и нажмите Редактировать.
  4. Для редактирования моментального снимка можно выполнить любое из следующих действий:
    • Измените описание моментального снимка в поле Описание.
    • Выберите Состояние с учетом, если включено создание моментальных снимков с учетом используемого приложения.
    • Переключайте значения Да/Нет в раскрывающемся списке Блокировать, чтобы определить, будет ли снимок игнорировать параметры хранения, когда будет запущено автоматическое удаление.
  5. Нажмите OK, чтобы сохранить изменения.

Настройка расписания создания моментальных снимков:

В случае запланированных моментальных снимков инструмент "Моментальные снимки и репликация" будет автоматически делать снимки в заданное время и с заданной частотой, и вам не придется делать снимки вручную.

  1. Выберите iSCSI LUN.
  2. Нажмите Настроить расписание.
  3. На вкладке Расписание установите флажок в поле Включить расписание создания моментальных снимков и установите время и частоту создания моментальных снимков. Чтобы отменить создание моментальных снимков по расписанию для этой общей папки, снимите флажок.
  4. Нажмите OK, чтобы сохранить изменения.

Настройка параметров хранения запланированных моментальных снимков:

  1. Выберите iSCSI LUN.
  2. Нажмите Настроить расписание.
  3. Откройте вкладку Хранение.
  4. Настройте параметры хранения. Выберите один из следующих параметров:
    • Удалять старые моментальные снимки при достижении лимита: По достижению максимального количества моментальных снимков самые старые снимки будут перезаписаны более новыми.
    • Всегда сохранять моментальные снимки: По достижении максимального количества моментальных снимков инструмент "Моментальные снимки и репликация" прекратит создание новых снимков и не будет удалять старые.
    • Сохранять моментальные снимки в соответствии со следующей политикой: Можно настроить правила хранения для установленных диапазонов времени. Для получения более подробной информации см. раздел Политика хранения.
  5. Нажмите OK, чтобы сохранить изменения.
  6. Появится сообщение с подтверждением. Нажмите OK еще раз для подтверждения настройки.

Чтобы настроить состояние с учетом для снимков iSCSI LUN:

  1. Выберите iSCSI LUN.
  2. Нажмите Настроить расписание.
  3. Откройте вкладку Приложение.
  4. Установите/снимите флажок в поле Включить создание моментальных снимков с учетом используемого приложения для переключения между следующими состояниями с учетом:
    • С учетом используемого приложения. Данный тип моментальных снимков доступен, только если на сервер Windows или VMware vCenter Server установлен Synology Snapshot Manager для обмена данными с DSM. Когда моментальные снимки с учетом используемого приложения будут запущены на DSM, на приложения отправится уведомление и все данные из памяти будут перенаправлены на LUN, снимки которых должны быть созданы, что обеспечит согласованность данных.

      Примечание.

      • Если в рабочей среде не установлен Synology Snapshot Manager, функция создания снимка с учетом используемого приложения останется выключенной, даже если установить флажок.
    • С учетом сбоя. Если ни на одном клиенте не установлен Synology Snapshot Manager, вы можете создавать моментальные снимки LUN с учетом сбоя. Когда будут запущены моментальные снимки с учетом сбоя, на приложения не будет отправлено уведомление по перенаправлению данных из памяти на LUN, поэтому согласованность данных не гарантируется.
  5. Нажмите OK, чтобы сохранить изменения.

Примечание.

  • Изменение параметров состояния с учетом повлияет на эквивалентные параметры на странице Репликация.

Политика хранения

Это идеальное решение по определению максимального количества версий моментальных снимков для экономии места в томе, но при этом, возможно, придется хранить снимки дольше.

В компании Synology принята политика хранения данных GFS (Grandfather-Father-Son). Можно настроить хранение максимального количества версий моментальных снимков в течение следующих диапазонов времени соответственно: каждый час, ежедневно, еженедельно, ежемесячно и ежегодно.

Инструмент "Моментальные снимки и репликация" будет сохранять установленное максимальное количество моментальных снимков для каждого диапазона времени. При наличии нескольких версий снимка в каком-либо временном диапазоне сохраняется только более ранняя. Например, если политикой предусмотрено 10 еженедельных снимков, инструмент "Моментальные снимки и репликация" будет сохранять самый ранний снимок (если в течение недели было создано несколько снимков) для каждой из последних 10 недель.

Более того, все версии моментальных снимков в течение одного часа с момента создания последнего снимка будут сохраняться по умолчанию, чтобы упростить поиск и восстановление из последних версий моментальных снимков. Также можно заблокировать моментальный снимок, чтобы предотвратить его автоматическое удаление согласно правилам хранения.
