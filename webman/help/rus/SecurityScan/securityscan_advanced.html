Дополнительно

Вы можете просматривать и изменять дополнительные параметры Security Advisor.

Базовая безопасность

Выбрав Базовую безопасность, можно определить, какие правила безопасности будет проверять Security Advisor. Количество и типы правил отличаются для каждой Базовой безопасности. Сделайте выбор в зависимости от ваших целей. Вы используете DiskStation для работы или для личного использования? Можно выбрать следующие варианты:

  • Для дома и личного использования: Security Advisor выполнит основные проверки для защиты устройства DiskStation от основных кибер-атак.
  • Для работы и бизнеса: Security Advisor выполнит комплексные проверки в соответствии с требованиями безопасности для организаций.
  • Пользовательский: определите, какие правила безопасности будет проверять Security Advisor.

График сканирования

Можно задать график регулярного сканирования, щелкнув Включить график регулярного сканирования и выбрав время выполнения проверки Security Advisor. Security Advisor будет проверять правила в соответствии с выбранной Базовой безопасностью.
