Security Advisor

Security Advisor — это приложение безопасности DSM, которое сканирует настройки DSM и DiskStation. Security Advisor проверяет ваши настройки и рекомендуемые изменения, которые помогают обеспечивать безопасность DiskStation. Ниже приведен список предоставляемых возможностей:

  • Security Advisor выполняет различные проверки для поиска настроек DSM, которые могут представлять риски безопасности, а также подозрительные события.
  • Security Advisor создает подробные отчеты с описанием рекомендуемых действий и преимуществ от их выполнения.
  • Внутренняя база данных определений безопасности автоматически обновляется, что позволяет следить за изменением кибер-атак.

Для запуска Security Advisor выполните следующие действия:

  • Перейдите в Главное меню > Security Advisor.
