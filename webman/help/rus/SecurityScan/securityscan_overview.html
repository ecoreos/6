Обзор

Вы можете проверить текущее состояние DSM, выполнить сканирование системы, а также проверить ход выполнения сканирования и сводную информацию о результатах. Security Advisor сканирует DSM и DiskStation, проверяя количество правил безопасности. Количество и типы правил зависят от того, как настроена Базовая безопасность. Базовая безопасность определяется вашими потребностями. Вы используете DiskStation для работы или для личного использования? При первом запуске сканирования вам будет предложено выбрать Базовую безопасность. Позднее этот параметр можно изменить на странице Дополнительно.

Состояние системы

Security Advisor имеет семь состояний. После завершения сканирования появится сводная информация о результатах с разделением на пять категорий. Различные состояния:

  • Готов к сканированию: это состояние отображается, когда сканирование DiskStation еще не началось. Щелкните Сканировать для начала сканирования.
  • Сканирование: сканирование выполняется. Можно посмотреть ход выполнения сканирования. Нажмите Остановить, чтобы остановить сканирование.
  • Хорошее: текущее сканирование завершено. Приложение Security Advisor не обнаружило уязвимостей или рисков безопасности. В настоящее время DiskStation в безопасности.
  • Предупреждение! Текущее сканирование завершено. Из-за некоторых уязвимых настроек безопасности приложение Security Advisor сообщило о низких/умеренных рисках безопасности. Для повышения безопасности DiskStation выполните рекомендуемые действия.
  • В зоне риска: текущее сканирование завершено. Из-за некоторых уязвимых настроек безопасности приложение Security Advisor сообщило о высоких рисках безопасности. Для повышения безопасности DiskStation обязательно выполните рекомендуемые действия.
  • Опасность: текущее сканирование завершено. Приложение Security Advisor обнаружило вредоносное ПО на устройстве DiskStation. Необходимо выполнить рекомендуемые действия, чтобы предотвратить дальнейшие нежелательные последствия для DiskStation.
  • Устаревший: ваша версия DSM устарела. Для удобства работы и обеспечения безопасности DiskStation рекомендуется обновить DSM для получения новых функций, улучшений и исправлений безопасности.

Security Advisor делит проверки на пять категорий:

  • Вредоносное ПО: определяет, есть ли вредоносное ПО на DiskStation и подвергается ли DiskStation атаке
  • Система: определяет настройки безопасности системы.
  • Учетная запись. определяет настройки безопасности учетной записи и пароля.
  • Сеть: определяет настройки безопасности сети.
  • Обновить: определяет, используется ли последняя версия DSM и других установленных пакетов, а также проверяет настройки обновлений.

После завершения сканирования можно посмотреть подробные результаты в форме отчета, щелкнув Просмотреть результаты.
