Результаты

Дополнительную информацию о выполненном сканировании можно посмотреть по отчетам. Эта страница состоит из следующих разделов:

  • Уровень серьезности: общий уровень риска каждого правила безопасности по оценке Security Advisor. Существует четыре уровня: Нормальный, Предупреждение, Тревога и Критичный.
  • Описание: настройки, которые ищет Security Advisor в определенном правиле и которые могут привести к уязвимости DiskStation, если не предпринять никаких действий.
  • Последнее обновление: время последней проверки этого правила безопасности, выполненной Security Advisor.
  • Состояние: текущее состояние каждого правила безопасности. Security Advisor сообщает один из двух результатов каждого правила безопасности: Проверено или Сбой. Иногда могут отображаться состояния Ошибка, Не проверено, Выполняется и Пропущено. В верхней части страницы можно отфильтровать результаты.

Проверка данных отчета любого правила безопасности:

Выберите правило и нажмите Редактировать. Помимо характеристик Уровень серьезности и Описание каждый отчет содержит следующие сведения:

  • Сведения: краткое описание всех уязвимостей, которые Security Advisor рекомендует устранить, а также преимущества, которые можно получить при устранении этих уязвимостей.
  • Рекомендуемое действие: подробные инструкции о том, как устранить описанные в отчете уязвимости системы безопасности.
  • Подключение: для любого давшего сбой правила безопасности нажмите Подключение, чтобы перейти к местоположению в DSM и выполнить рекомендуемые действия.
  • Исправить: при обнаружении вредоносного ПО щелкните Исправить, и Security Advisor удалит это ПО.

Повторное сканирование определенных правил:

Выберите элементы, которые необходимо отсканировать, и нажмите кнопку Сканировать.
