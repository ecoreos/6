Hot Spare

Диски горячего резерва Hot Spare — это резервные жесткие диски, которые могут использоваться для замены неисправного блока RAID Group с помощью автоматической замены отказавшего устройства. Диски Hot Spare можно назначить глобально для восстановления любого блока RAID Group в DiskStation. Диск Hot Spare можно использовать для замены блока RAID Group, если он отвечает следующим критериям:

  • Тип RAID блока RAID Group поддерживает защиту данных (т. е. RAID 1, RAID 5, RAID 6, RAID 10).
  • Типы жестких дисков Hot Spare должны совпадать с типами составляющих дисков RAID Group (например, SATA, SAS).
  • Размер диска Hot Spare не меньше, чем размер самого малого диска в RAID Group.

Для управления дисками Hot Spare выполните следующие действия.

  1. Нажмите Управление.
  2. Чтобы назначить диск Hot Spare, найдите раздел Доступные диски с правой стороны и установите флажок рядом с диском, который будет выполнять функцию Hot Spare. Для удаления Hot Spare оставьте этот флажок. Нажмите Применить, чтобы сохранить настройки.

Примечание.

  • Когда диск Hot Spare восстановит неисправный блок RAID Group, все данные на диске Hot Spare будут удалены. Перед назначением диска Hot Spare убедитесь, что на диске нет важных данных.
  • Раздел RAID, созданный на основе жестких дисков, не будет автоматически восстанавливаться твердотельными дисками (SSD) Hot Spare, и наоборот.

Настройка параметров Hot Spare для RAID Group:

  1. Нажмите Настройка.
  2. На вкладке Целевые устройства Hot Spare отображается список блоков RAID Group, которые могут быть защищены дисками Hot Spare. Если снять флажки, поврежденные данные соответствующих блоков RAID Group не будут автоматически восстанавливаться дисками Hot Spare.
  3. На вкладке Hot Spare для разных устройств можно разрешить дискам Hot Spare одного устройства восстанавливать поврежденные RAID на другом устройстве.
  4. Нажмите кнопку OK, чтобы сохранить изменения.
