HDD/SSD

В этом разделе можно отслеживать состояние жестких дисков, установленных в DiskStation, а также организовывать и анализировать производительность и состояние дисков. Выберите Диспетчер хранилища > HDD/SSD для управления этими параметрами.

Проверка информации о дисках HDD/SSD

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск из списка и разверните. Здесь отображается состояние S.M.A.R.T., информация о томе, название модели, серийный номер, версия микропрограммы и температура.

Просмотр информации об исправности

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск.
  3. Нажмите Информация о состоянии.
  4. Перейдите на вкладку Обзор. Здесь отображается, сколько часов диск включен, текущая температура, количество повторных подключений и повторных идентификаций, количество неисправных секторов и общее состояние S.M.A.R.T.
  5. Выберите вкладку Информация S.M.A.R.T.. Выведенные на экран цифры помогут точнее определить состояние диска.

Примечание.

Диски SAS не поддерживают информацию S.M.A.R.T.

Запуск тестирования S.M.A.R.T.

Тест S.M.A.R.T. для дисков изучает жесткий диск и сообщает о его состоянии, предупреждая о возможных сбоях диска. При обнаружении ошибок рекомендуется сразу заменить диск.

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск.
  3. Нажмите Проверка S.M.A.R.T..
  4. Выберите тип теста и нажмите Пуск, чтобы запустить тестирование.
  5. Результаты последнего тестирования отображаются в разделе Результат теста.

Примечание.

Время тестирования зависит от среды.

Расписание тестирования S.M.A.R.T.

Можно запланировать автоматическое выполнение тестов S.M.A.R.T., создав задачи.

  1. Перейдите на вкладку Расписание задач.
  2. Нажмите Создать, чтобы настроить новую задачу.
  3. На вкладке Общие измените следующие параметры.
    • Задача: Введите имя для задачи.
    • Тип теста S.M.A.R.T.: выберите для выполнения быстрого или расширенного тестирования.
    • Диапазон теста S.M.A.R.T.: выберите, какой диск HDD/SSD будет проверяться при каждом выполнении тестирования S.M.A.R.T.
  4. На вкладке Расписание измените следующие параметры.
    • Дата: задайте дату выполнения тестирования S.M.A.R.T. Можно запланировать запуск тестов S.M.A.R.T. на определенные дни недели (ежедневно, выходной, каждое воскресенье и т. д.) или на определенные даты (например, 2015/10/21).
    • Время: укажите количество запусков тестов S.M.A.R.T. в день, а также время первого и последнего запусков.
  5. Нажмите OK для завершения.

Выполнение безопасного удаления

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск и нажмите Безопасное удаление.

Примечание.

  • Невозможно выполнить безопасное удаление на дисках в модулях расширения.
  • Безопасное удаление приведет к необратимому удалению всех данных, которые хранятся на диске. Перед выполнением операции убедитесь, что выбранный диск не используется либо что создана резервная копия данных.
  • Когда система выполняет безопасное удаление, выбранный диск будет временно заблокирован, а затем автоматически разблокируется после завершения операции.
  • Если безопасное удаление будет неправильно остановлено при выполнении, можно выполнить его повторно прямо в интерфейсе пользователя. Если диски удалены или используются где-либо еще во время блокировки, введите пароль "synology", чтобы разблокировать их.

Включение поддержки записи в кэш для жестких дисков

Включение поддержки записи в кэш повышает производительность DiskStation. Эта функция поддерживается только для некоторых моделей жестких дисков. Для обеспечения защиты данных при поддержке записи в кэш настоятельно рекомендуется использовать устройство ИБП. Также рекомендуется правильно выключать систему после каждого использования. Отключение записи в кэш уменьшит вероятность потери данных при сбое питания, но сократит производительность.

Включение поддержки записи в кэш

  1. Перейдите на вкладку Управление кэшем.
  2. Установите флажок Включить запись в кэш. По умолчанию данная функция включена.
  3. Нажмите Применить.
