Создание тома

В этой статье описывается процесс создание тома на DiskStation. Выберите один из способов создания нового тома.

  • Быстрый. Лучший вариант для новичков. Выберите этот способ, чтобы автоматически оптимизировать емкость и производительность хранилища, не заботясь о технических аспектах, например уровнях RAID.
  • Пользовательский. Этот способ подходит продвинутым пользователям, которые хотят выборочно управлять параметрами создания тома, например типами RAID и распределением емкости. Эта опция поддерживает различные типы RAID, позволяет создавать один том или несколько томов, а также предоставляет возможность определять объем емкости для каждого тома (только при создании нескольких томов на RAID).

Один или несколько томов на RAID?

При создании тома в пользовательском режиме можно выбрать один из вариантов ниже.

  • Один том на RAID
    • Создание единого тома, включающего все пространство выбранных жестких дисков.
    • Обеспечивает лучшую производительность и скорость.
    • Поддерживает различные типы RAID.
  • Несколько томов на RAID
    • Позволяет создавать несколько томов в группе дисков.
    • Поддерживает распределение емкости между томами.
    • Обеспечивает большую гибкость в управлении хранилищем.
    • Поддерживает различные типы RAID.

Перед началом работы

  • Сначала проверьте, что на жестких дисках нет важных данных. При создании все имеющиеся данные будут удалены.
  • Откройте страницу HDD/SSD и убедитесь, что для каждого жесткого диска установлен статус Нормальный режим или Диск не инициализирован.

Примечания

  • Минимальный размер тома Btrfs — 10 ГБ.
  • Тома Btrfs сохранят 4% пространства для метаданных.

Создание тома:

  1. Откройте Диспетчер хранения.
  2. Выберите Том и нажмите Создать.
  3. Отобразится мастер создания томов. Нажмите Быстрый или Пользовательский на свой выбор. Следуйте инструкциям мастера для завершения создания тома.
