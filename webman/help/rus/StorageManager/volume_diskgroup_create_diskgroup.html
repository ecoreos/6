Создание группы дисков

В этой статье описывается процесс создание группы дисков на DiskStation.

Перед началом работы:

  • Убедитесь, что на жестких дисках нет важных данных. При создании новой группы дисков все существующие данные будут удалены.
  • Откройте страницу HDD/SSD и убедитесь, что для каждого жесткого диска установлен статус Нормальный режим или Диск не инициализирован.

Создание группы дисков:

  1. Откройте Диспетчер хранения.
  2. Выберите Группа дисков и нажмите Создать.
  3. Когда отобразится мастер создания группы дисков, выберите жесткие диски, которые будут использованы для создания группы дисков. Выполните инструкции мастера для завершения операции.
