Исправление ошибок данных*

Исправление ошибок данных — это функция обслуживания данных, проверяющая тома или группы дисков. Существует два типа исправления ошибок данных.

  • Исправление ошибок данных RAID. Эту функцию можно использовать с томами или группами дисков, использующими SHR (из трех или более дисков), RAID 5 или RAID 6. Она может исправить обнаруженные несоответствия данных.
  • Исправление ошибок данных файловой системы. Эту функцию можно использовать с томами в файловой системе Btrfs. При обнаружении несоответствия данных с контрольной суммой путь доступа к файлу будет записан в Центр журналов. Настоятельно рекомендуется исправить данные с помощью резервной копии.

Рекомендуется регулярно выполнять исправление ошибок данных, чтобы поддерживать их согласованность и избежать утери критически важных данных в случае отказа диска.

Исправление ошибок данных:

  1. Откройте Диспетчер хранения.
  2. Выберите Том или Группа дисков.
  3. Выберите том или группу дисков, в которой необходимо выполнить исправление ошибок данных.
  4. Нажмите Управление.
  5. Выберите Исправление ошибок данных.
  6. Выберите нужный тип исправления ошибок данных. (Система автоматически пропустит этот шаг, если доступен только один тип.)

Примечание.

  • Исправление ошибок данных можно выполнить, только когда том или группа дисков находится в состоянии Нормальный.
  • Если отключить систему до завершения процесса, потребуется вручную перезапустить исправление ошибок данных с самого начала.

____
* Данная функция доступна только в некоторых моделях.
