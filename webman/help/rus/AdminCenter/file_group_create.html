Создание группы

  1. Выберите Панель управления > Группа.
  2. Нажмите Создать. Данное действие позволит запустить Мастер создания пользователей.
  3. Введите имя и описание группы. (Для получения дополнительной информации см. раздел ниже). Нажмите Далее.
  4. Выполните остальные действия мастера для завершения создания группы.
  5. Нажмите Применить, чтобы завершить настройку.

Правила наименования:

  • Название группы вводится с учетом регистра клавиатуры и должно содержать от 1 до 32 символов в кодировке Unicode, исключая следующие символы:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Первый символ не должен быть знаком минуса или пробелом, а последний символ не может быть пробелом.
  • Свойства группы вводятся с учетом регистра клавиатуры и ограничены 64 отображаемыми символами в кодировке Unicode.
