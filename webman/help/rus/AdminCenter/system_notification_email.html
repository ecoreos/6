Электронная почта

Включение уведомлений по электронной почте позволяет DiskStation отправлять сообщения на ваш адрес электронной почты в случае возникновения определенных событий, например системных ошибок, сбоя сетевого подключения, изменений в работе вентилятора, сбоя питания или неудачной попытки резервирования.

Включение уведомлений по электронной почте позволяет DiskStation отправлять сообщения на ваш адрес электронной почты в случае возникновения определенных событий, например системных ошибок, сбоя сетевого подключения, сбоя питания или неудачной попытки резервирования.

Чтобы включить уведомления по электронной почте, выполните следующие действия.

  1. Выберите Панель управления > Уведомления > Электронная почта.
  2. Установить флажок Включить уведомления по электронной почте.
  3. Укажите один или несколько адресов электронной почты в поле Получатели. Системные уведомления будут отправлены по введенным здесь адресам. Допускается два адреса. Вы можете использовать двоеточия (;) для разделения нескольких адресов.
  4. При необходимости добавьте префикс к теме в поле Префикс к теме. Этот префикс будет добавлен к теме каждого сообщения, которое отправляется с DiskStation, чтобы помочь определить и отфильтровать сообщения.
  5. Чтобы использовать сторонний сервер электронной почты (например, Gmail, Outlook, Yahoo!) для передачи сообщений, см. раздел "сторонняя почтовая служба" ниже. Чтобы использовать пользовательский сервер, см. раздел "пользовательский SMTP-сервер" ниже.

Чтобы использовать стороннюю почтовую службу, выполните следующие действия.

  1. В меню Поставщик услуг выберите стороннюю почтовую службу, например Gmail, Yahoo или Outlook.
  2. Введите имя пользователя и пароль для выбранной почтовой службы.
  3. Нажмите Применить, чтобы сохранить настройки.
  4. Нажмите Отправить пробное письмо для проверки правильности настроек.

Чтобы использовать пользовательский SMTP-сервер, выполните следующие действия.

  1. В меню Поставщик услуг выберите Пользовательский SMTP-сервер.
  2. Введите имя SMTP-сервера или IP-адрес в поле SMTP-сервер.
  3. Введите в поле Порт SMTP номер порта SMTP. (Если в правилах переадресации порта указан другой порт SMTP, укажите номер этого порта.)
  4. Если сервер SMTP требует аутентификации, установите флажок Требуется аутентификация, введите Имя пользователя и Пароль для сервера электронной почты.
  5. Если ваш сервер SMTP требует подключения SSL/TLS, установите флажок Требуется защищенное соединение (SSL/TLS).
  6. Если вы хотите настроить имя отправителя, которое отображается в электронных письмах, отправленных с DiskStation, введите имя в поле Имя отправителя. Если вы хотите настроить адрес электронной почты отправителя, введите адрес электронной почты в поле Электронный адрес отправителя. Если отставить поле пустым, будет использоваться первый адрес электронной почты в поле Получатели.
  7. Нажмите Применить, чтобы сохранить настройки.
  8. Нажмите Отправить пробное письмо для проверки правильности настроек.

Примечание.

  • При необходимости отправлять приветственное сообщение каждому пользователю, создавшему новую учетную запись, установите флажок Отправить приветственное сообщение новым пользователям, и при создании учетной записи каждый пользователь будет получать приветственное сообщение.
  • Язык уведомления электронной почты можно установить в разделе Язык уведомлений. Чтобы изменить язык уведомлений электронной почты, перейдите в Главное меню > Панель управления > Региональные параметры, выберите вкладку Язык и из раскрывающегося меню выберите необходимый язык в разделе Язык уведомлений.
  • При использовании SMTP-сервера Gmail и включении двухэтапной проверки следует использовать "пароль приложения", предоставляемый Google для аутентификации SMTP.

Имя домена сервера SMTP

В качестве SMTP-сервера можно указать либо IP-адрес, либо имя домена. Если это имя домена, убедитесь, что в пункте Главное меню > Панель управления > Сеть введен правильный IP-адрес сервера DNS и сервер находится в Интернете.
