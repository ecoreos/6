Сетевая маршрутизация

DSM предоставляет несколько опций сетевой маршрутизации для различных конфигураций сети.

Разделение сетевого потока в сетевых интерфейсах в одной локальной сети:

  1. Выберите Панель управления > Сеть > Дополнительно.
  2. В меню Общие, установите флажок Ответить на запросы ARP, если целевой IP-адрес является локальным адресом, настроенным на входящем интерфейсе.
  3. Нажмите Применить, чтобы сохранить настройки.

Это позволяет установить правило переадресации по умолчанию для каждого сетевого интерфейса и отправлять сетевые пакеты каждого интерфейса из соответствующего интерфейса физической сети.

Кроме того, если у вас установлено более двух интерфейсов в одной локальной сети, включение данной опции обеспечит раздельную передачу сетевого потока на каждом интерфейсе. Для каждого интерфейса можно применить отдельные правила брандмауэра.

Направление каждого сетевого интерфейса через соответствующие шлюзы:

  1. Выберите Панель управления > Сеть.
  2. В меню Общие установите флажок Включить несколько шлюзов.
  3. Нажмите Применить, чтобы сохранить настройки.

Это позволит направить маршрутизацию для каждого сетевого интерфейса в сети через соответствующие шлюзы.

Изменение шлюза по умолчанию:

  1. Выберите Панель управления > Сеть.
  2. В меню Сетевой интерфейс нажмите Порядок использования.
  3. Измените положение шлюзов с помощью кнопок Вверх или Вниз.
  4. Нажмите OK.

Шлюз в верхней части порядка будет являться шлюзом по умолчанию для локальной сети.

Настройка статической маршрутизации:

  1. Выберите Панель управления > Сеть.
  2. В меню Статическая маршрутизация нажмите Создать.
  3. В пункте Сеть назначения введите IP-адрес места назначения.
  4. В пункте Маска сети введите маску сети места назначения.
  5. В пункте Шлюз введите адрес шлюза места назначения.
  6. В пункте Интерфейс выберите интерфейс места назначения.
  7. Нажмите OK.

Просмотр таблицы маршрутизации IP-адресов:

  1. Выберите Панель управления > Сеть.
  2. В меню Статическая маршрутизация нажмите Таблица маршрутизации IP-адресов.
  3. Выберите сетевой интерфейс для просмотра таблицы маршрутизации IP-адресов в левом верхнем раскрывающемся списке.
