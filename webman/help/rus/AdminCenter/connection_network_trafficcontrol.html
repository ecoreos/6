Управление трафиком

Управление трафиком (Панель управления > Сеть > Управление трафиком) направлено на управление исходящим трафиком служб, работающих на DiskStation. После выбора службы необходимо настроить параметры Гарантированная ширина пропускания и Максимальная ширина пропускания для ограничения исходящего трафика.

Объяснение терминов

  • Гарантированная ширина пропускания: Задает исходящий трафик, который данная служба гарантированно предоставляет при достаточном значении полосы пропускания всей системы.
  • Максимальная ширина пропускания: Задает исходящий трафик, который данная служба может предоставить при достаточном значении полосы пропускания всей системы и наличии Остаточной ширины пропускания системы. Рекомендуется сначала подсчитать значение Исходящая ширина пропускания системы. Затем убедитесь, что сумма Гарантированной ширины пропускания для каждой службы не превышает значение Исходящей ширины пропускания системы, в противном случае параметр может работать неисправно.

Формула настройки

  • Исходящая ширина пропускания системы = сумма Гарантированной ширины пропускания всех служб + Остаточная ширина пропускания системы
  • Гарантированная ширина пропускания каждой службы ≤ Максимальной ширины пропускания каждой службы

Примечание.

Если исходящие пакеты не соответствуют правилам управления трафиком, ограничение установлено не будет. Вы можете добавить собственное правило управления трафиком по умолчанию в конце всех правил, выбрав Все порты и указав параметры Гарантированная ширина пропускания и Максимальная ширина пропускания.

Создание правила управления трафиком:

  1. Выберите сетевой интерфейс в раскрывающемся меню в правом верхнем углу.
  2. Нажмите Создать.
  3. В разделе Порты выберите для правила один из вариантов:
    • Выберите Все, чтобы применить правило ко всем портам.
    • Выберите Выбрать из списка встроенных приложений.
    • Выберите Пользовательский и нажмите Пользовательский, а затем введите не более 15 портов, разделяя их запятой, либо укажите диапазон портов.
    • Можно также указать протоколы.

Теперь вы можете просматривать созданное правило управления трафиком в списке правил.

Чтобы удалить правила управления трафиком, выполните следующие действия:

  • Снимите флажок правила в столбце Включено списка.
  • Удалите правило управления трафиком.

Примечание.

  1. Вы можете создать до 100 правил для сетевого интерфейса.
  2. Выполните перетаскивание для переупорядочивания правил в списке.
  3. Правила получают приоритет согласно их положению в списке.
  4. Если в системе используются несколько сетевых портов, подключенных к одной подсети, правила брандмауэра могут работать неправильно.*
  5. Если DiskStation подключается к Интернету с помощью PPPoE, соответствующие правила управления трафиком должны быть настроены в соответствующем интерфейсе PPPoE.

____
* Поддерживается только на некоторых моделях.
