Расширенные настройки

Для изменения нижеприведенных параметров сеансов просмотра DSM перейдите в Панель управления > Безопасность > Дополнительно:

Сжатие HTTP

Можно включить параметр Сжатие HTTP для DiskStation. Уменьшение пропускной способности сетевого трафика увеличит скорость загрузки веб-страницы.

Включение/отключение сжатия HTTP

  • Установите/снимите флажок с параметра Сжатие HTTP, перейдя на Панель управления > Безопасность > Дополнительно.

Профиль TLS/SSL

Выберите уровень безопасности зашифрованных подключений HTTPS. Существуют 3 уровня:

  • Современная совместимость: поддерживает браузеры и мобильные устройства только последних версий.
  • Промежуточная совместимость: уровень безопасности между высоким и низким уровнями. Этот уровень установлен по умолчанию.
  • Совместимость со старыми версиями: небезопасный, но поддерживает большинство браузеров и мобильных устройств.
