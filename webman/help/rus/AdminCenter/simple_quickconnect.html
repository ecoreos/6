QuickConnect

Служба QuickConnect позволяет клиентским приложениям подключаться к DiskStation через Интернет без необходимости настройки правил переадресации портов.

Примечание.

QuickConnect может также работать с DSM и некоторыми пакетами Synology (например, Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Cloud Station и мобильные приложения).

Включение QuickConnect:

  1. Выберите Панель управления (Простой режим) > QuickConnect.
  2. Установите флажок Включить QuickConnect.
  3. Введите данные вашей учетной записи Synology или зарегистрируйте новую учетную запись.
  4. Укажите новый идентификатор QuickConnect ID. Выберите легкий для запоминания код, чтобы обращаться к DiskStation в любое время.
  5. Нажмите Применить.

Примечание.

  • Перед включением QuickConnect на клиентских приложениях убедитесь, что включены необходимые службы DSM. Например, прежде чем включить QuickConnect для DS file необходимо включить службу WebDAV на DiskStation.
  • Идентификатор QuickConnect ID должен начинаться с буквы и может содержать буквы, числа и знак дефиса ("-").
  • Для повышения производительности QuickConnect рекомендуется выбрать Панель управления (Расширенный режим) > Внешний доступ > Конфигурация маршрутизатора, чтобы настроить переадресацию порта для каждой службы:
    • DSM: 5000 (HTTP), 5001 (HTTPS)
    • Photo Station: 80 (HTTP), 443 (HTTPS)
    • Cloud Station: 6690

Устранение неполадок

Если появляется сообщение Произошла ошибка сети, это означает, что служба QuickConnect была остановлена из-за ошибки сети. Проверьте следующее:

  • Устройство DiskStation подключено к активной сети (Панель управления (Расширенный режим) > Сеть > Сетевой интерфейс).
  • Сервер DNS и шлюз по умолчанию настроены правильно (Панель управления (Расширенный режим) > Сеть > Общие).
  • Убедитесь, что DiskStation работает в Интернете, используя следующие порты назначения: 22, 80, 443, 8888. 

После устранения ошибок сети служба QuickConnect должна автоматически возобновиться в течение нескольких минут.
