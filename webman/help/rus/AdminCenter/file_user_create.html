Создание пользователя

Вы можете создать несколько пользовательских учетных записей и управлять ими на DiskStation.

Создание пользователя:

  1. Выберите Панель управления > Пользователь.
  2. Нажмите кнопку Создать и выберите "Создать пользователя". Данное действие запускает Мастер создания пользователей.
  3. На странице Информация о пользователе укажите следующие параметры.
    • Имя. Введите имя пользователя. Данное поле должно быть заполнено.
    • Описание. Введите краткое описание пользователя.
    • Эл. почта. Введите адрес электронной почты пользователя. Уведомления системы, например сообщения о сбросе пароля, будут отправляться на указанный здесь адрес.
    • Отправить уведомление по электронной почте новому пользователю. Если данный параметр включен, система отправит пользователю сообщение с уведомлением о новой учетной записи. Для данного параметра требуется включение уведомлений электронной почты в пункте Главное меню > Панель управления > Уведомление > Эл. почта. По умолчанию пароль пользователя не отображается в уведомлении. Если вы хотите отобразить в уведомлении пароль, установите флажок Отобразить пароль пользователя в уведомлении.
    • Запретить пользователям изменять пароль учетной записи. Если включен данный параметр, пользователь не сможет изменять пароль для входа в DSM.
  4. На странице Группа > Создать > Мастер создания групп укажите группы, к которым будет принадлежать новый пользователь (например, пользователи, администраторы и др.).
  5. На странице Назначить разрешения к общим папкам выберите общие папки, к которым новый пользователь будет иметь доступ. Приоритет прав доступа выглядит следующим образом: Нет доступа > Чтение/запись > Только для чтения.
    • Предварительный просмотр. В данном столбце отображаются права доступа пользователя к папке общего доступа в соответствии с текущими настройками и группами, к которым принадлежит пользователь.
    • Разрешения группы. В данном столбце отображаются разрешения пользователя к папке общего доступа в соответствии с группами, к которым принадлежит пользователь.
  6. На странице Настройки дисковых квот пользователей можно управлять объемом хранения, который может использовать новый пользователь. Для ограничения объема хранения, который может использовать пользователь, установите флажок Включить квоту для каждого тома/папки общего доступа. Затем введите максимальный объем места для использования в поле Квота (например, 10 ГБ).
  7. На странице Назначить разрешения к приложениям можно управлять службами, к которым пользователь будет иметь доступ.
  8. На странице Настройка ограничения скорости для группы вы можете активировать ограничение скорости для различных служб (например, File Station, FTP и др.) для ограничения полосы пропускания, потребляемой пользователем при передаче файлов. Для каждой службы можно выбрать один из следующих параметров.
    • Нет. Настройки ограничения скорости будут совпадать с настройками группы, к которой принадлежит пользователь.
    • Постоянная. Будет применяться фиксированное ограничение скорости. Вы можете указать ограничение скорости на выгрузку и загрузку в полях слева.
    • Переменная. К пользователю могут применяться два различных ограничения скорости по расписанию. Нажмите кнопку Настроить, чтобы изменить настройки ограничения скорости и настроить расписание.
  9. Подтвердите настройки и для завершения нажмите Применить.

Ограничения для имени пользователя:

  • Имя пользователя вводится без учета регистра клавиатуры и должно содержать от 1 до 64 символов в кодировке Unicode, исключая следующие символы:
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • Название не может начинаться с дефиса или пробела и не может оканчиваться пробелом.
  • Следующие имена зарезервированы для системного использования: mailer-daemon, postmaster.
  • Описание пользователя может содержать не более 64 отображаемых символов Unicode.

Ограничения для пароля:

  • Пароль вводится с учетом регистра клавиатуры и может содержать до 127 отображаемых символов, включая буквы, цифры, знаки и пробелы.

Ограничение по квоте:

  • Максимальная квота для каждого пользователя составляет примерно 4 ТБ (4095 ГБ).
