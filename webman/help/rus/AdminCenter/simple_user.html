Пользователь

Благодаря гибким параметрам управления пользователями можно создавать учетные записи для ваших друзей или коллег, а также настраивать права доступа к папкам общего доступа для каждого пользователя.

Создание пользователя:

  1. Выберите Панель управления (Простой режим) > Пользователь и нажмите значок + на левой панели.
  2. Укажите имя и пароль пользователя.
  3. Укажите группы, к которым будет принадлежать новый пользователь.
  4. Назначьте пользователю соответствующие права доступа к папке общего доступа. В случае возникновения конфликта прав доступа будет действовать следующий приоритет: Нет доступа > Чтение/запись > Только для чтения.
    • Предварительный просмотр: В данном столбце отображаются права доступа пользователя к папке общего доступа в соответствии с текущими настройками и группами, к которым принадлежит пользователь.
    • Чтение/запись: Пользователи могут получать доступ и изменять файлы и подпапки в соответствующей папке общего доступа.
    • Только для чтения: Пользователь имеет доступ к файлам и подпапкам в соответствующей папке общего доступа.
    • Нет доступа: Пользователь имеет доступ к файлам и подпапкам в соответствующей папке общего доступа.
  5. Нажмите Применить, чтобы сохранить настройки.

    6. Ограничения для имени пользователя:

    • Имя пользователя вводится без учета регистра клавиатуры и должно содержать от 1 до 64 символов в кодировке Unicode, исключая следующие символы:
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • Первый символ не должен быть знаком минуса или пробелом, а последний символ не может быть пробелом.
    • Это имя зарезервировано для системного использования.
      mailer-daemon, postmaster

    Ограничения для пароля:

    • Пароль вводится с учетом регистра клавиатуры и может быть пустым или содержать до 127 отображаемых символов, включая буквы, цифры, знаки и пробелы.

Редактирование пользователя:

  1. Выберите Панель управления (Простой режим) > Пользователь.
  2. В левой панели выберите необходимого пользователя.
  3. Измените общую информацию о пользователе, например имя, пароль, группу и права доступа.
  4. Установите флажок Отключить данную учетную запись, если вы хотите прекратить работу учетной записи.
  5. Нажмите Применить, чтобы сохранить настройки.

Удаление пользователя:

  1. Выберите Панель управления (Простой режим) > Пользователь.
  2. В левой панели выберите необходимого пользователя.
  3. Нажмите значок Корзина в левой панели, чтобы удалить выбранного пользователя.
