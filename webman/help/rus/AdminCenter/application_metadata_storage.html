Мультимедийная база данных

DiskStation использует мультимедийную базу данных для отслеживания мультимедийных файлов и записей журналов, хранящихся на мультимедийном источнике (папках общего доступа, созданных на внешних устройствах хранения, например usbshare1).

Внимание! Чтобы мультимедийные службы DSM (например Сервер мультимедиа, Сервис iTunes, Audio Station и Download Station) имели доступ к мультимедийным файлам, хранящимся на внешних устройствах хранения, подключенных к порту USB на DiskStation, или для включения служб журналов DSM и просмотра записей в пункте Основное меню > Сведения о системе > Журнал, сначала необходимо включить мультимедийную базу данных. В противном случае все соответствующие службы будут отключены.

Включение мультимедийной базы данных:

  1. Установите флажок Включить мультимедийную базу данных.
  2. Выполните одно из следующих действий:
    • Выберите Автоматическая конфигурация, чтобы программа DSM автоматически выбрала папку общего доступа для хранения мультимедийной базы данных. Если внешнее устройство хранения, на котором находится папка общего доступа, будет удалено, DSM автоматически выберет другую папку общего доступа для хранения базы данных.
    • Выберите пункт Выберите папку общего доступа для хранения базы данных и укажите в раскрывающемся меню выбор места хранения базы данных вручную.
  3. Нажмите кнопку OK.

Внимание! Для предотвращения повреждения базы данных убедитесь, что вы извлекли устройство хранения в меню Основное меню > Панель управления > Внешние устройства перед отсоединением его от DiskStation.

Чтобы включить или удалить папки общего доступа из источника мультимедиа:

  1. Нажмите Источник мультимедиа.
  2. Установите или снимите флажок напротив необходимых папок общего доступа.
  3. Нажмите кнопку OK.

Мультимедийные файлы и записи журналов в выбранных папках общего доступа будут проиндексированы, и соответствующие сведения добавлены в базу мультимедиа.

Обновление мультимедийной базы данных:

  1. Нажмите Обновить базу данных. Все данные в мультимедийной базе данных будут заменены переиндексированной информацией о текущем источнике мультимедиа.
  2. Нажмите кнопку OK.

Примечание. Если большая часть данных в мультимедийной базе данных больше не используется (например, при отключении диска USB, который больше не будет использоваться с DiskStation, в базе данных образуется избыточная индексированная информация), обновление базы данных позволяет очистить все неиспользуемые данные и уменьшить размер базы данных. Однако для этого требуется постоянное чтение и запись на диск, что может задействовать системные ресурсы и снизить срок службы диска. Используйте данную функцию осторожно.
