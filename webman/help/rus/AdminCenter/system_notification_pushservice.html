Push-служба

Push-служба позволяет DiskStation отправлять уведомления на мобильные устройства или в веб-браузеры в случае возникновения особых событий или ошибок.

Порты:

Перед включением Push-службы убедитесь, что DiskStation может получить доступ к Интернету через порт 8089.

Чтобы включить уведомления по электронной почте от компании Synology, выполните следующие действия:

  1. Установите флажок Отправить уведомления о состоянии системы с помощью сервера электронной почты Synology.
  2. Укажите один или несколько адресов электронной почты в поле Получатели. Системные уведомления будут отправлены по введенным здесь адресам. Допускается два адреса. Используйте двоеточия (;) для разделения нескольких адресов.
  3. Щелкните Применить для отправки проверочного сообщения на адреса, указанные в поле Получатели. Следуйте инструкциям, приведенным в проверочном сообщении.
  4. После проверки щелкните Отправить тестовое сообщение и нажмите OK, чтобы отправить тестовое уведомление.

Включение уведомления по мобильному устройству:

  1. Выберите Панель управления > Уведомления > Push-служба.
  2. Установите флажок Включить уведомление по мобильному устройству и нажмите Применить.
  3. Установите и запустите DS finder на мобильном устройстве, а затем добавьте DiskStation в список Избранное, чтобы подключить мобильное устройство к DiskStation.
  4. Нажмите Отправить тестовое сообщение и нажмите OK, чтобы отправить тестовое уведомление. Теперь запустите DS finder на мобильном устройстве и проверьте наличие тестового сообщения.

Управление подключенными мобильными устройствами:

  1. Нажмите Управление подключенными устройствами для просмотра списка всех подключенных мобильных устройств.
  2. Чтобы отключить устройства, выберите их и нажмите Отключить. (Чтобы выбрать несколько элементов, нажмите и удерживайте клавишу Shift или Ctrl.) На них не будут отправляться уведомления до их повторного подключения.

Чтобы включить уведомления браузера, выполните следующие действия.

  1. Установите флажок Включить уведомление по мобильному устройству и нажмите Применить для активации этой функции.
  2. Установите флажок Включить уведомления браузера, чтобы подключить DiskStation к браузеру.
  3. В отобразившемся окне выполните одно из следующих действий.
    • Для Chrome: Щелкните Установить прямо сейчас для установки расширения Chrome и затем выполните вход в учетную запись Google, если вход не был выполнен ранее.
    • Для Safari: Щелкните Разрешить, чтобы авторизовать службу уведомлений.
  4. Когда вы будете перенаправлены обратно на страницу уведомлений браузера, щелкните вкладку Подключенное устройство, чтобы просмотреть информацию о подключенных устройствах или отключить DiskStation от браузера, или щелкните вкладку Событие, чтобы просмотреть журнал уведомлений браузера за последние семь дней.

Примечание.

Служба уведомлений браузера поддерживается на Chrome (35 и выше) и Safari (OS X 10.9 и выше).
