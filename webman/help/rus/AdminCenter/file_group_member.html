Добавление/удаление членов группы

Вы можете добавить пользователей DiskStation в созданные группы или удалить их из групп.

Добавление в группу новых участников:

  1. Выберите группу, в которую хотите добавить пользователей, и нажмите Редактировать участников группы.
  2. Нажмите Добавить.
  3. Выберите пользователей из списка участников и нажмитеOK. (Чтобы выбрать нескольких участников, нажмите и удерживайте клавишу "Shift" или "Ctrl" и нажимайте соответствующих пользователей.)
  4. Нажмите кнопку Готово, чтобы применить настройки.

    5. Примечание. Члены группы Administrators имеют административные полномочия на DiskStation, включая приложения DSM/CIFS/FTP/AFP/WebDAV.

Удаление участников из группы:

  1. Выберите группу, из которой хотите удалить пользователей, и нажмите Редактировать участников группы.
  2. Выберите пользователей из списка участников.
  3. Нажмите Да.
  4. Нажмите кнопку Готово, чтобы применить настройки.

    5. Примечание. Пользователи, удаленные из группы Administrators, не могут выполнять действия администратора DiskStation.
