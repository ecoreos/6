Пользователи и группы домена/LDAP

Если устройство DiskStation присоединено к службе каталогов, например домену Windows или клиенту LDAP, вы можете настроить или изменить права пользователей или групп домена/LDAP к папкам общего доступа DiskStation и приложениям DSM, а также включить службу главной папки для всех пользователей домена/LDAP.

Пользователь домена/LDAP

В зависимости от выбранного типа службы каталога нажмите вкладку Пользователи домена или Пользователь LDAP, чтобы настроить разрешения к папкам общего доступа и приложениям, а также квоту емкости диска для каждого пользователя.

В зависимости от выбранного типа службы каталога нажмите вкладку Пользователи домена или Пользователь LDAP, чтобы настроить разрешения к папкам общего доступа и приложениям для каждого пользователя.

Правила настройки прав доступа включают следующее:

  • В случае возникновения конфликта прав доступа будет действовать следующий приоритет. Нет > Чтение-Запись > Только чтение.
  • Предварительный просмотр прав: Здесь будут показаны права пользователя, которые установлены на основе настройки прав пользователя и групп, присоединенных пользователем.
  • Права группы: Присвойте права группам для доступа к папкам общего доступа.

Главная папка пользователя

Как и локальные пользователи, пользователи домена/LDAP могут войти в папку home через CIFS, AFP, FTP или File Station.

Пользователи с правами администратора имеют доступ ко всем личным папкам, расположенным в папке общего доступа homes. Все главные папки пользователей расположены в одной из подпапок в папке "@DH-имя домена" (для пользователей домена) или "@LH-имя FQDN" (для пользователей LDAP). Имя главной папки содержит учетную запись домена плюс уникальный номер.

    Примечание.
  1. Чтобы удалить папку общего доступа homes, необходимо отключить службу главной папки пользователя.
  2. При включении службы главной папки пользователей домена/LDAP также активируется служба главной папки локальных пользователей.

Пользователь домена/LDAP

В зависимости от выбранного типа служб каталогов нажмите вкладку Группа доменов или Группа LDAP для настройки прав доступа групп домена/LDAP к папкам общего доступа. Приоритет прав выглядит следующим образом: Нет > Чтение-Запись > Только чтение.
