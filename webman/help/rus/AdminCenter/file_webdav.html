WebDAV

WebDAV — это расширение протокола HTTP, позволяющее пользователям редактировать файлы, хранящиеся на удаленных серверах, и управлять этими файлами. Если служба WebDAV включена, то клиентские программы с поддержкой WebDAV (например, некоторые приложения Windows, Mac OS Finder или Linux File Browser) могут получать удаленный доступ к DiskStation так же, как при использовании локальных сетевых дисков.

Включение WebDAV:

  1. Установите флажок в поле Включить WebDAV или Включить соединения WebDAV HTTPS, если требуется разрешить подключение WebDAV по протоколу HTTPS. По умолчанию для WebDAV используется порт 5005, а для WebDAV HTTPS — порт 5006.
  2. Нажмите Применить

Ограничение скорости

Включение ограничения скорости позволяет управлять максимальной используемой шириной пропускания сети для определенных пользователей и групп при передаче данных с помощью WebDAV.

Включение ограничения скорости:

  1. Установите флажок Включить ограничение скорости или Включить ограничение скорости согласно расписанию. В последнем случае ограничение скорости будет применяться только в определенное время. Эта настройка применяется на всех серверах и переопределяет настройки отдельных пользователей и групп. Если выбран параметр Нет ограничения скорости, ограничения скорости для пользователей и групп отсутствуют.
  2. Нажмите кнопку Настройки ограничения скорости, чтобы изменить настройки для отдельных пользователей или групп. Для каждого пользователя или группы можно выбрать один из следующих параметров.
    • Применить настройки группы: Настройки ограничения скорости будут совпадать с настройками группы, к которой принадлежит пользователь.
    • Нет. Ограничение скорости не действует, когда участники группы передают файлы с помощью этой службы.
    • Постоянная. К отдельным пользователям или участникам группы будет применяться фиксированное постоянное ограничение скорости. В полях слева можно указать ограничения скорости при выгрузке и загрузке файлов.
    • Переменная. К отдельным пользователям или участникам группы применяются два различных ограничения скорости, которые включаются по расписанию. Нажмите кнопку Настроить, чтобы изменить настройки ограничения скорости и настроить расписание.
  3. Нажмите Применить.

Примечание.

  • Дополнительные сведения о динамике ограничений скорости группы см. здесь.
  • При установке ограничения скорости для пользователя в разделе Результат отображается фактическое ограничение скорости пользователя с учетом индивидуальных и групповых настроек. Например, если для пользователя "Achilles" установлено индивидуальное ограничение скорости загрузки 100 КБ/с, но этот пользователь является участником группы с ограничением скорости загрузки 500 КБ/с, в разделе Результаты будет отображаться значение "0/500."

Дополнительные настройки

Нажав кнопку Дополнительные настройки, можно изменить следующие параметры.

  • Включить анонимную учетную запись WebDAV: Позволяет клиентам получать доступ к папкам общего доступа с помощью WebDAV от имени пользователя anonymous. Убедитесь, что учетной записи пользователя "Анонимный FTP/WebDAV" предоставлены соответствующие права доступа на странице Папка общего доступа.
  • Включить DavDepthInfinity: При использовании клиентских программ этот параметр позволяет пользователям получать доступ к файлам или удалять файлы без ограничений на длину пути к файлу.
  • Включить журнал WebDAV: Этот параметр позволяет регистрировать события, связанные с WebDAV. Нажмите Смотреть журнал событий, чтобы просмотреть журналы в отобразившемся окне.

Использование WebDAV в различных операционных системах

После включения службы WebDAV на устройстве DiskStation рекомендуется ознакомиться с приведенными ниже инструкциями по получению доступа к файлам с устройств Windows, Mac, Linux или из определенных приложений Synology для смартфонов.

Примечание.

  • При подключении с помощью служб WebDAV рекомендуется использовать клиент WebDAV с поддержкой UTF-8. Это позволит избежать ошибок кодировки.

Для Windows

При использовании протокола WebDAV для получения доступа к файлам в продуктах Synology, пользователям ОС Windows рекомендуется использовать BitKinex — бесплатное приложение для подключения дисков. Также можно загрузить другие клиентские приложения WebDAV, доступные в Интернете.

Примечание.

  • Чтобы использовать функцию Windows Подключение сетевого диска для доступа к файлам на DiskStation по протоколу WebDAV, необходимо получить действующий сертификат SSL и настроить доступ к устройству DiskStation через подключение HTTPS. При использовании службы WebDAV в среде Windows для переключения учетной записи пользователя необходимо выполнить выход из системы Windows.

Для Mac

На компьютерах Mac подключиться к DiskStation по протоколу WebDAV можно с помощью Finder.

Использование WebDAV на Mac:

  1. На компьютере Mac выберите Finder > Подключиться к серверу.
  2. В поле Адрес сервера ведите IP-адрес DiskStation или имя домена с префиксом "http://" и постфиксом ":5005" (можно указать другой номер порта, заданный при включении службы WebDAV). Например: http://pm.synology.com:5005/.
  3. Введите имя пользователя в ОС DSM и пароль.
  4. Нажмите Подключиться.

Примечание.

  • WebDAV поддерживает вещание Bonjour.

Для Linux

Для доступа к файлам на DiskStation по протоколу WebDAV в Ubuntu и других операционных системах Linux следуйте приведенным ниже инструкциям.

Использование WebDAV на Ubuntu:

  1. Нажмите Места > Подключиться к серверу.
  2. В раскрывающемся меню выберите WebDAV (HTTP) или WebDAV (HTTPS).
  3. Введите необходимую информацию, включая местоположение сервера, порт, папку и учетную запись пользователя. Чтобы продолжить, нажмите Подключиться.
  4. Введите пароль и нажмите Подключиться.

Работа с WebDAV с помощью командной строки Linux:

В командной строке можно использовать cadaver – команду клиентского программного обеспечения WebDAV для настройки доступа.

Подписка на календари с помощью клиентов CalDAV

Вы можете использовать клиенты CalDAV на компьютере или мобильном устройстве для подписки на календари, хранящиеся на DiskStation, а также для управления событиями, заметками и задачами. Примеры клиентов календарей: Sunbird (Windows, Mac, Linux), Chandler (Windows, Mac, Linux), Evolution (Linux) и iCal (Mac, устройства iOS).

Включение CalDAV:

  1. Сначала включите WebDAV- или WebDAV HTTPS-подключение.
  2. Установите флажок Включить CalDAV.
  3. Нажмите Применить.

Управление списком календаря:

  1. Нажмите Просмотр списка календаря.
  2. В отобразившемся окне выполните одно из следующих действий:
    • Чтобы создать новый календарь, нажмите Добавить, введите имя нового календаря (например, "cal1") и укажите место назначения для сохранения (например, папка общего доступа "календарь"), а затем нажмите OK.
    • Если вы хотите изменить имя или местоположение календаря, выберите календарь в списке, нажмите Редактировать, чтобы внести изменения, и нажмите OK.
    • Чтобы удалить календарь из DiskStation, выберите его в списке календарей и нажмите Удалить.
  3. Нажмите Закрыть.

Примечание.

  • Календари, расположенные на DiskStation, будут отображаться в виде папки на DiskStation.

Подписка на ваш календарь с помощью клиентского приложения (например, Sunbird):

  1. Загрузите и установите Sunbird на компьютер.
  2. Выберите Файл > Подписка на удаленный календарь.
  3. В отобразившемся окне выберите По сети и нажмите Далее.
  4. Выберите CalDAV, введите http://DiskStation_имя_хоста_DDNS:номер_порта_WebDAV/Местоположение_календаря/Имя_календаря в поле Местоположение (например, "http://demo.no-ip.info:5005/calendar/cal1") и нажмите Далее.
  5. Укажите имя календаря и нажмите Далее.
  6. Введите имя пользователя и пароль DSM для аутентификации, нажмите OK и Выполнено.

Подписка на ваш календарь с помощью мобильного устройства (например, устройства iOS):

  1. На устройстве iOS выберите Настройки > Почта, Контакты, Календари.
  2. В меню Учетные записи выберите Добавить учетную запись > Другое и нажмите Добавить учетную запись CalDAV.
  3. В полях введите IP-адрес DiskStation (или имя домена), регистрационные данные пользователя и нажмите Далее. Если отображается сообщение, нажмите Продолжить.
  4. Нажмите Дополнительные настройки и выполните следующие действия:
    1. Введите 5005 (или другой номер порта, указанный при включении службы WebDAV) в поле Порт.
    2. Измените URL-адрес в поле URL-адрес учетной записи на http://DiskStation_Имя_хоста_DDNS:Номер_порта_WebDAV/Местоположение_календаря/Имя_календаря (например, "http://demo.no-ip.info:5005/calendar/cal1").
    3. Нажмите CalDAV.
  5. Нажмите Далее, чтобы проверить учетные данные, затем нажмите Сохранить, чтобы завершить настройку учетной записи.
  6. Вернитесь на главный экран устройства iOS и откройте Календарь для управления календарем.

Примечание.

  • Для получения дополнительной информации об управлении событиями календаря, заметками и задачами см. справочную документацию для каждого клиентского приложения.
