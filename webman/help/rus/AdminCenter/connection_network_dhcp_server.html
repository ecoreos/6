Сервер DHCP

Ваше устройство DiskStation может выступать в качестве сервера DHCP и назначать динамические IP-адреса клиентам DHCP в вашей локальной сети.

Включение сервера DHCP:

  1. Выберите Панель управления > DHCP Server > Сетевой интерфейс.
  2. Выберите сетевой интерфейс (например, LAN 1), чтобы назначить ему IP-адреса, и нажмите Редактировать.
  3. Во всплывающем окне перейдите на вкладку Сервер DHCP и выберите Включить сервер DHCP.
  4. Задайте следующие настройки DHCP:
    • Время аренды адреса (в минутах)
    • Основной DNS
    • Дополнительный DNS (необязательно)
    • Имя домена (необязательно)
    • Включить автоматическое обнаружение прокси-сервера в Интернете (не обязательно)
  5. Настройте хотя бы одну подсеть. Чтобы настроить подсеть, нажмите Добавить и укажите следующую информацию:
    • Начальный IP-адрес
    • Конечный IP-адрес
    • Маска подсети
    • Шлюз
  6. Нажмите Создать, чтобы отобразить созданную подсеть в списке.
  7. Создав достаточное количество подсетей, необходимо включить хотя бы одну из них.
  8. Нажмите OK.

Примечания

  • Можно создать неограниченное количество подсетей.
  • При координации с сервером ретрансляции DHCP ваше устройство DiskStation, выступающее в качестве сервера DHCP, может назначать IP-адреса устройствам в другой подсети.

Клиенты DHCP

Можно просмотреть список клиентов DHCP и их сетевая конфигурация (MAC-адрес, IP-адрес, имя хоста, оставшееся количество времени перед окончанием аренды DHCP и состояние резервирования).

  1. Нажмите Панель управления > DHCP Server > Сетевой интерфейс и выберите один из интерфейсов.
  2. Нажмите Редактировать > Клиенты DHCP для поиска клиентов и сведений о них.

Управление IP-адресами для клиентов DHCP

Чтобы клиент всегда получал один и тот же IP-адрес во время обновления аренды DHCP и для управления этими клиентами, нужно добавить клиентов.

  • Добавление клиентов DHCP Нажмите Добавить, чтобы назначить IP-адрес клиентскому устройству.
  • Резервирование IP-адреса: Нажмите на значок блокировки (замок), чтобы зарезервировать IP-адрес для текущего устройства.
  • Отмена резервирования: Нажмите на значок блокировки (замок), чтобы разрешить другому устройству использовать текущий IP-адрес, когда пройдет время аренды адреса.

Примечание

Если устройство может иметь одновременно адреса IPv4 и IPv6, можно зарезервировать для него один адрес IPv4 и один адрес IPv6.

Поиск клиентов DHCP:

  1. Нажмите значок трубы в верхнем правом углу.
  2. Выберите один из критериев поиска:
    • Все. Список всех клиентов DHCP и резервирований DHCP.
    • Клиенты DHCP. Список всех клиентов DHCP с зарезервированными или незарезервированными IP-адресами.
    • Резервирование DHCP. Список всех зарезервированных IP-адресов. Зарезервированный IP-адрес может быть еще не связан с устройством.

Параметры DHCP

После настройки параметров DHCP ваш DiskStation в качестве сервера DHCP сможет отправлять информацию клиентам DHCP по определенной подсети.

Настройка параметров DHCP для подсети

  1. Выберите Панель управления > DHCP Server > Сетевой интерфейс.
  2. Выберите сетевой интерфейс и щелкните Редактировать.
  3. Перейдите в DHCP Server и затем в Список подсети.
  4. Выполните одно из приведенных ниже действий.
    • Настройка параметров DHCP для имеющейся подсети Выберите интерфейс и щелкните Редактировать.
    • Создание подсети и настройка параметров DHCP Нажмите Добавить и настройте параметры подсети.
  5. Выберите требуемые параметры DHCP. Чтобы указать их значения, нажмите на соответствующие пустые поля.
  6. Нажмите OK.

Поставщик

Для клиентов DHCP от разных поставщиков может потребоваться указать дополнительную информацию. Настроив параметры DHCP для поставщиков, DiskStation сможет предоставить такую информацию клиентам.

Чтобы узнать, какие параметры DHCP поддерживаются определенным поставщиком, обратитесь к соответствующему поставщику.

Добавление поставщиков

  1. Выберите Панель управления > DHCP Server > Поставщик.
  2. Нажмите Создать, чтобы указать поставщика и его параметры DHCP.
    • Название: имя поставщика.
    • Код: числовой код параметра DHCP.
    • Значение: значение параметра.
    • +: добавление параметра DHCP.
    • -: удаление параметра DHCP.
  3. Нажмите Создать, чтобы завершить настройку.

Изменение параметров DHCP для поставщиков

  1. Выберите Панель управления > DHCP Server > Поставщик.
  2. Выберите поставщика и нажмите Редактировать для управления параметрами DHCP.
  3. Нажмите OK.

PXE

Ваше устройство DiskStation может выступать в качестве сервера DHCP для реализации настройки PXE (среды удаленной загрузки Pre-Boot Execution Environment). PXE — это сетевая среда, позволяющая клиентским устройствам загружаться по сетевому интерфейсу, а не с помощью устройства хранения данных (например, жестких дисков).

Если реализована среда PXE, загрузчик может храниться на DiskStation или удаленном сервере TFTP, а клиентские устройства могут запускаться по сети с помощью этого загрузчика.

Использование DiskStation в качестве загрузчика сервера

  1. Убедитесь, что установлен флажок Включить службу TFTP (Панель управления > Файловые службы > TFTP).
  2. Выберите папку, в которой находится файл загрузчика, в качестве корневой папки TFTP.
  3. Убедитесь, что активирована хотя бы одна подсеть DHCP (Панель управления > DHCP Server > Сетевой интерфейс).
  4. Выберите Панель управления > DHCP Server > PXE.
  5. Выберите Включить PXE (среда удаленной загрузки).
  6. Нажмите Локальный сервер TFTP и выберите файл загрузчика.
  7. Нажмите Применить, чтобы сохранить настройки.

Использование удаленного сервера TFTP в качестве загрузчика сервера

Использование удаленного сервера TFTP в качестве загрузчика сервера позволяет клиентам загружаться без необходимости изменения текущей сетевой инфраструктуры.

  1. Убедитесь, что активирована хотя бы одна подсеть DHCP (Панель управления > DHCP Server > Сетевой интерфейс).
  2. Выберите Панель управления > DHCP Server > PXE.
  3. Выберите Включить PXE (среда удаленной загрузки).
  4. Нажмите Удаленный сервер TFTP и укажите IP-адрес, а также путь к файлу загрузчика.
  5. Нажмите Применить, чтобы сохранить настройки.
