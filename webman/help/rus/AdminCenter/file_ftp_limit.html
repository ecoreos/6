Ограничение подключения

Разрешить только SSL/TLS-соединения

Когда данная функция включена, сервер принимает от клиентов FTP только запросы соединения SSL или TLS.

Ограничивать число подключений по IP

Когда данная функция включена, сервер будет ограничивать максимальное количество соединений по IP. Для эффективного управления полосой пропускания включите эту функцию, а также активируйте параметр Ограничение полосы пропускания FTP.

Включить ограничение ширины FTP-канала

Включите эту функцию, чтобы ограничить полосу пропускания для отдельного подключения через FTP.
