Права доступа

Права доступа пользователей и групп к отдельным службам и приложениям можно просматривать и редактировать. DSM предоставляет гибкие настройки прав доступа пользователей к приложению, позволяя легко управлять доступом пользователей и групп к различным приложениям.

Изменение прав доступа к приложению для пользователей и групп:

  1. Выберите Панель управления > Права доступа.
  2. Выберите приложение, которое необходимо изменить. Нажмите Редактировать.
  3. Нажмите Пользователь для просмотра отдельных прав доступа или Группа для просмотра прав доступа группы.
  4. Для каждого пользователя или группы установите Разрешить, чтоб разрешить доступ, или Запретить, чтобы запретить доступ к выбранному приложению. В зависимости от приложения можно также выбрать По IP-адресу, чтобы разрешить/запретить доступ на основе IP-адреса.
  5. Нажмите OK, чтобы сохранить изменения.

Просмотр прав доступа к приложению для пользователей с различными IP-адресами:

  1. Выберите Панель управления > Права доступа.
  2. Нажмите Средство просмотра разрешений.
  3. Выберите пользователя в пункте Имя пользователя и введите IP-адреса, для которых вы хотите просмотреть разрешения, в пункте IP-адреса.

Установка прав доступа к приложению по умолчанию для всех пользователей:

  1. Выберите Панель управления > Права доступа.
  2. Выберите приложение, которое необходимо изменить. Нажмите Редактировать.
  3. Нажмите Права доступа по умолчанию и установите флажок Предоставить эти права доступа всем.
  4. Нажмите OK, чтобы сохранить изменения.

Примечание.

  • Здесь невозможно выполнить редактирование прав доступ для учетных записей admin и guest.
  • В случае возникновения конфликта прав доступа будет действовать следующий приоритет: Запретить > Разрешить.
  • Не все пакеты и службы поддерживают настройки прав доступа По IP-адресу.
