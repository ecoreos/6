USBCopy

На странице Панель управления > Внешние устройства > USBCopy/SDCopy можно выбрать папку назначения для функции USBCopy.

Включение функции USBCopy:

  1. Установите флажок Включить USBCopy.
  2. В раскрывающемся меню выберите целевую папку общего доступа.

При подключении USB-устройства (например, цифровой камеры) к порту USB на передней панели продукта Synology можно нажать кнопку "Копировать", расположенную на передней панели, чтобы скопировать данные с USB-устройства в соответствующую общую целевую папку. Если удалить общую целевую папку USBCopy, система автоматически заменит ее папкой общего доступа public.

Ограничения

  1. Поддерживаются только устройства USB в форматах FAT, NTFS, ext3 и ext4.
  2. Устройства USB должны быть подключены к порту USB, расположенному на передней панели DiskStation.
  3. Только пользователи с именем admin или пользователи, принадлежащие к группе administrators, имеют соответствующие права доступа для выполнения задач резервирования на устройствах USB.
  4. Доступное пространство в целевой папке общего доступа должно быть больше используемого места на устройстве USB.
