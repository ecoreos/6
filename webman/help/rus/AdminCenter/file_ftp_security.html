Настройки безопасности

Изменить выбранные каталоги пользователей на главные папки

Выбранные пользователи имеют доступ к главным папкам при входе Synology DiskStation через FTP.

Включить анонимную учетную запись FTP

Данная функция позволяет всем пользователям выполнять вход в службу FTP на DiskStation (с учетными записями anonymous или ftp).

Настройка прав доступа для анонимного пользователя:

Выберите Основное меню > Панель управления > Папка общего доступа и нажмите Настройка прав доступа.

Изменение анонимного каталога

Данная функция позволяет изменять анонимный регистрационный каталог для FTP. В раскрывающемся меню выберите папку общего доступа.
