Автоматическая блокировка

Автоматическая блокировка позволяет предотвратить несанкционированный доступ. После включения данной службы IP-адрес будет заблокирован, если будет выполнено много неудачных попыток входа. Будут фиксироваться все неудачные попытки входа при помощи служб SSH, Telnet, rsync, Резервное копирование по сети, Синхронизация общих папок, FTP, WebDAV, программ Synology для мобильных устройств, File Station или DSM. Вы также можете использовать Список разрешенных, который содержит всегда разрешенные IP-адреса.

Настройка автоматической блокировки:

  1. Выберите вкладку Общие и установите флажок Включить автоматическую блокировку.
  2. Введите Попытки входа и В течение (минут), чтобы заблокировать IP-адрес после превышения числа установленных попыток входа за указанное время.
  3. Установите флажок Включить дату истечения блокировки и введите число для удаления заблокированного IP-адреса после указанного количества дней. Заблокированный IP-адрес остается неограниченное время в списке блокировок, до тех пор пока он не будет удален вручную, если не выбран соответствующий параметр или параметр Разблокировать через (дней) не установлен на 0.
  4. Нажмите Применить.

Добавление заблокированных IP-адресов:

Нажмите вкладку Список блокированных и выберите один из следующих элементов в раскрывающемся меню Создать:

  • Добавить IP-адрес

    • Введите IP-адрес и установите срок действия блокировки.

  • Импортировать список IP-адресов

    • Импортируйте текстовый файл со списком IP-адресов, чтобы добавить IP-адреса и установить срок действия блокировки. Установите соответствующий флажок, чтобы перезаписать существующие IP-адреса. Если параметр не выбран, дублирующиеся IP-адреса будут пропущены.

    Чтобы импортировать файл, он должен удовлетворять следующим критериям:

    • Файл должен иметь обычный текстовый формат.
    • Каждая строка файла должна содержать только один IP-адрес.
    • Строки комментариев должны начинаться с # (pешетка).

      • Если выбранный файл имеет правильный формат, в области просмотра будут отображаться все IP-адреса. Нажмите OK для импортирования.

Удаление заблокированных IP-адресов:

  1. Нажмите вкладку Список блокированных, чтобы просмотреть список заблокированных IP-адресов.
  2. Выберите IP-адреса для удаления из списка и нажмите Удалить.
