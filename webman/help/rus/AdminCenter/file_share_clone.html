Клонирование папки общего доступа*

Папку общего доступа можно клонировать в разделах Btrfs для создания почти мгновенных копий существующей папки общего доступа. Клонирование папки общего доступа использует лишь незначительную часть хранилища во время создания.

Клонирование папки общего доступа:

  1. Выберите Панель управления > Папка общего доступа.
  2. Выберите нужную папку общего доступа в файловой системе Btrfs.
  3. Нажмите Создать > Клонировать.
  4. Укажите имя и описание новой папки общего доступа и измените другие параметры при необходимости. (Для получения дополнительной информации об этих параметрах см. Создание папки общего доступа.)

Примечание.

  • В разделах Btrfs можно клонировать папки общего доступа.
  • Клонированная папка общего доступа будет создана в том же разделе, что и исходная папка общего доступа.
  • Настройки улучшенной защиты целостности данных будут применены к клонированной папке общего доступа и не могут быть изменены.

____
*Данная функция доступна только в некоторых моделях.
