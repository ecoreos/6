Настройки преобразования

При выгрузке фотографий и видео для использования в сервере мультимедиа и Photo Station будут созданы соответствующие значки. Вы можете настроить качество эскиза и следить за процессом преобразования.

Если вы хотите просмотреть видео в папке общего доступа photo Photo Station на мобильном устройстве, можно включить преобразование видео и следить за ходом его выполнения.

Настройка качества эскиза:

  1. Выберите качество в раскрывающемся меню.
  2. Нажмите Применить.
Примечание.
  • Создание эскизов высокого качества занимает больше времени.

Включение преобразования видео для мобильных устройств:

  1. Установите флажок Включить преобразование видео для мобильных устройств.
  2. Нажмите Применить, чтобы просматривать видео на мобильном устройстве везде, где бы вы ни находились.
Примечание.
  • Только загруженные в папку общего доступа photo Photo Station видео можно преобразовать в формат, поддерживаемый мобильными устройствами.
  • Файлы MP4 не будут преобразованы.

Просмотр хода выполнения преобразования фото/видео:

  1. При выгрузке фото или видео в DiskStation нажмите значок Ход выполнения преобразования в правом верхнем углу панели задач.
  2. В отобразившемся окне выполните одно из следующих действий:
    • В разделе Ход выполнения преобразования можно следить за процессом создания эскизов в фоновом режиме.
    • Приостановка процесса преобразования:
      1. Выберите Задержка на [количество] часов в раскрывающемся меню и нажмите Пауза, чтобы приостановить процесс. (По умолчанию выбран параметр Задержка на 1 час.)
      2. Процесс преобразования возобновится после указанного времени задержки, либо нажмите Продолжить, чтобы возобновить процесс немедленно.
      Примечание.
      • Поскольку процесс преобразования требует больших ресурсов процессора, приостановка процессора позволяет использовать ресурсы системы для других выполняющихся задач.
  3. После завершения процесса преобразования значок на панели задач исчезнет.
Примечание.
  • Поскольку Photo Station Uploader предоставляет улучшенную производительность для создания значков, рекомендуется использовать Photo Station Uploader для выгрузки большого числа фото и видео в библиотеку Photo Station.
