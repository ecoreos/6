Включить корзину

При активации корзины в папке общего доступа будет создана подпапка для хранения удаленных файлов. Вы можете извлечь удаленные файлы из корзины. Чтобы окончательно удалить файлы, очистите корзину.

Включение корзины:

  1. Выберите папку общего доступа и нажмите Редактировать.
  2. Установите флажок Включить корзину.
  3. Установите флажок Ограничить доступ, разрешив его только администраторам, чтобы корзина была доступна только для администраторов.
  4. Нажмите кнопку OK, чтобы сохранить изменения.

Очистка корзины:

Нажмите Очистить корзину, чтобы окончательно удалить все файлы из корзины.

Очистка всех корзин:

Нажмите Очистить все корзины а раскрывающемся меню Действие, чтобы окончательно удалить все файлы из всех корзин.

Примечание. Вы можете настроить параметры прав доступа корзины в File Station, нажав папку #Корзина правой кнопкой мыши. Выберите Свойства и нажмите вкладку Разрешение.
