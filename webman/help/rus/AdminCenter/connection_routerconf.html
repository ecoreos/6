Конфигурация маршрутизатора

Если устройство DiskStation находится в локальной сети, другие устройства в Интернете не смогут к нему подключиться. Можно настроить правила переадресации портов в разделе Панель управления > Внешний доступ > Конфигурация маршрутизатора, чтобы обеспечить доступ к своему устройству DiskStation через Интернет.

Настройка маршрутизатора

Перед добавлением правил переадресации портов необходимо настроить подключение между устройством DiskStation и маршрутизатором.

Примечание.

  1. Чтобы настроить правила переадресации портов и назначить статический IP-адрес, нужно обладать правами администратора для доступа к маршрутизатору.
  2. Прежде чем настраивать подключение устройства DiskStation к маршрутизатору UPnP, проверьте, что программа DSM успешно извлекла внешний IP-адрес устройства DiskStation (для этого откройте Панель управления > Внешний доступ > DDNS). В противном случае проверьте настройки сети и брандмауэра своего устройства DiskStation и маршрутизатора UPnP.

Порядок настройки подключения устройства DiskStation к маршрутизатору:

  1. Нажмите Настроить маршрутизатор, чтобы запустить мастер установки. Перед переходом к следующему шагу DSM будет тестировать следующие сетевые компоненты и параметры.
    • Сетевой интерфейс
    • Подключение к шлюзу
    • Подключение к Интернету
    • Переадресация портов
    • DNS
  2. В зависимости от типа вашего маршрутизатора выполните одно из следующих действий.
    • Если ваш маршрутизатор не поддерживает UPnP, то перейдите к шагу 3.
    • Если ваш маршрутизатор UPnP протестирован на совместимость с устройством DiskStation, то нажмите Применить, когда увидите сводные сведения о маршрутизаторе. На данном этапе успешно завершена настройка подключения устройства DiskStation к маршрутизатору.
    • Если ваш маршрутизатор UPnP не протестирован на совместимость с устройством DiskStation, выполните одно из следующих действий.
      • Если хотите автоматически настроить подключение к маршрутизатору, то выберите Автоматически и нажмите Далее, чтобы система провела тест на совместимость:
        • Если маршрутизатор протестирован и признан совместимым со службой конфигурирования маршрутизаторов устройства DiskStation, установите флажок, чтобы отправить отчет о тестировании в компанию Synology для обновления базы данных совместимых маршрутизаторов, а затем нажмите Применить. На данном этапе успешно завершена настройка подключения устройства DiskStation к маршрутизатору.
        • Если маршрутизатор протестирован и признан несовместимым со службой конфигурирования маршрутизаторов устройства DiskStation, установите флажок, чтобы отправить отчет о тестировании в компанию Synology для обновления базы данных совместимых маршрутизаторов, а затем нажмите Закрыть, чтобы завершить работу мастера настройки.

          • Примечание.

          Если на данном этапе маршрутизатор протестирован и признан несовместимым, вы, тем не менее, можете продолжить настройку правил переадресации портов в пользовательском интерфейсе маршрутизатора. Дополнительную информацию можно найти здесь.

      • Если хотите вручную настроить подключение к маршрутизатору, то выберите Вручную, нажмите Далее и перейдите к шагу 3.
  3. Чтобы вручную настроить подключение устройства DiskStation к маршрутизатору, выполните следующее.
    1. В раскрывающемся меню Маршрутизатор выберите маршрутизатор.

      2. Примечание.

      Если не найдете свой маршрутизатор в этом списке, то нажмите Обновить список, чтобы получить новейший список от компании Synology.

    2. Настройте протокол и порт своего маршрутизатора. Протокол и порт могут изменяться в зависимости от модели маршрутизатора. Если вы не можете подключиться используя Протокол (HTTP) и Порт (80) по умолчанию, см. документацию к маршрутизатору для информации о правильных настройках.
    3. Если для маршрутизатора требуется особая учетная запись, установите флажок Использовать пользовательскую учетную запись и заполните поля Учетная запись и Пароль.
    4. Нажмите Применить, чтобы завершить настройку.

Управление правилами переадресации портов

Правила переадресации портов позволяют указать порты назначения для устройства DiskStation, которое будет получать пакеты от определенных портов маршрутизатора.

Если устройство DiskStation подключено к Интернету через маршрутизатор, нажмите Создать, чтобы открыть мастер установки для переадресации портов.

Добавить правила переадресации порта можно двумя способами:

  • Чтобы создать правила переадресации портов для системных служб (например, FTP, Web Station и др.), выберите Встроенное приложение.
  • Чтобы создать правила переадресации портов для других служб, выберите Пользовательский порт.

Вы можете использовать следующий синтаксис для указания номеров порта и диапазона:

  • Используйте "-", чтобы описать диапазон портов, например: 6881-6890.
  • Используйте "-" для разделения портов, например: "21,22", "21,55536-55663".

Примечание.

  • Каждый порт маршрутизатора шлюза можно использовать только для одного правила переадресации. Использование порта маршрутизатора для одного и более правил может привести к конфликтам, в результате чего на экране будут отображены предупреждения. Чтобы избежать конфликтов, необходимо установить другой порт в правиле.
  • На некоторых маршрутизаторах не поддерживается использование пользовательских портов для переадресации системных служб.

Сохранение правил:

После создания правил нажмите Сохранить. В противном случае правила применены не будут.

В зависимости от модели маршрутизатора после сохранения правил, возможно, придется подождать несколько минут, прежде чем правила вступят в силу.

Тестирование работы правил:

Выберите правило, которое необходимо протестировать, и нажмите кнопку Тестировать подключение. Результат будет отображен в столбце Результат тестирования подключения.

Для конфигураций маршрутизатора с беспроводным подключением*

Когда устройство DiskStation работает как беспроводной маршрутизатор, правила переадресации портов выполняет само устройство DiskStation.

Добавление правил переадресации порта:

  1. Нажмите Создать, чтобы открыть окно установки.

  2. Укажите имя службы в поле Служба, Протокол (TCP или UDP), Личный IP-адрес, Личный порт и Общий порт. Вы можете использовать следующий синтаксис для указания номеров порта и диапазона:
    • Используйте "-", чтобы описать диапазон портов, например: 6881-6890.
    • Используйте "-" для разделения портов, например: "21,22", "21,55536-55663".
  3. Нажмите Создать внизу, чтобы завершить настройки.

Примечание.

Большинство маршрутизаторов разрешают использовать другие порты локальных машин и порты маршрутизатора при настройке правил переадресации портов. Если маршрутизатор поддерживает эту возможность:

  • Если порт маршрутизатора уже используется существующим правилом, при сохранении правила будет назначен новый порт маршрутизатора.
  • Добавляя или редактируя правила, можно напрямую указать номер порта маршрутизатора.

DMZ*

DMZ можно включить на устройстве DiskStation, когда конфигурацию карты сети представляет беспроводной маршрутизатор. Хост на DMZ может подключаться непосредственно к внешней сети вне зависимости от ограничений и защиты брандмауэра. Он обычно используется для настройки серверов.

Включение DMZ:

  1. Нажмите Настройки DMZ, чтобы открыть окно установки.
  2. Установите флажок Включить DMZ.
  3. Введите IP-адрес хоста DMZ.
  4. Нажмите Применить.

Примечание.

Во внешней сети для подключения к хосту на DMZ вам потребуется внешний IP-адрес, извлеченный устройством DiskStation.

* Доступно только с конкретными моделями или в соответствии с вашей средой.
