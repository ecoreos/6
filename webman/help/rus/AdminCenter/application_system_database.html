База данных системы

DiskStation использует системную базу данных для отслеживания данных пакетов и записей журналов, хранящихся в папках общего доступа, созданных на внешних устройствах хранения, например usbshare1.

Внимание! Чтобы получить доступ к данным пакетам, расположенным на внешних устройствах хранения, подключенных к DiskStation, или чтобы просмотреть записи журналов DSM в меню Главное меню > Центр журналов, сначала необходимо включить базу данных системы. В противном случае все соответствующие службы будут недоступны.

Включение базы данных системы:

  1. Установите флажок Включить базу данных системы.
  2. Выполните одно из следующих действий:
    • Выберите Автоматическая конфигурация, чтобы программа DSM автоматически выбрала папку общего доступа для хранения базы данных системы. Если внешнее устройство хранения, на котором находится папка общего доступа, будет удалено, DSM автоматически выберет другую папку общего доступа для хранения базы данных.
    • Выберите Переместите текущую базу данных в выбранную папку общего доступа и в раскрывающемся меню переместите текущую базу данных в другую папку общего доступа, созданную на внешнем устройстве.
    • Выберите пункт Выберите одну из существующих баз данных, затем в раскрывающемся меню выберите базу данных, если к DiskStation подключено более одного внешнего устройства.
  3. Нажмите Применить.

Внимание! Во избежание повреждения базы данных убедитесь, что вы извлекли устройство хранения в пункте Главное меню > Панель управления > Внешние устройства > Внешние устройства, или нажав значок внешнего устройств в правом верхнем углу перед его извлечением из DiskStation.
