Настройки PHP

После включения Web Station параметры PHP, описанные ниже, могут быть настроены в пункте Панель управления > Веб-службы > Настройки PHP.

Включение кэша PHP

Установите флажок Включить кэш PHP, чтобы увеличить производительность PHP. Но в некоторых случаях данный параметр не поможет.

Отображение сообщения об ошибке PHP на веб-странице

Установите флажок Включите display_errors, чтобы отобразить сообщение об ошибке PHP, чтобы отобразить сообщения об ошибке PHP на веб-странице.

Настройка PHP-параметра open_basedir

  1. Чтобы настроить параметр open_basedir, установите флажок Настроить PHP open_basedir.
  2. Введите путь в текстовом поле и нажмите Применить. Если вы хотите использовать настройки системы по умолчанию, снимите флажок Настроить PHP open_basedir и нажмите Применить.

Примечание.

  • Используйте двоеточие (:) для разделения нескольких путей. Например: /etc:/tmp:/usr/bin/php.
  • В целях безопасности пути для главных папок общего доступа (например /volume1/homes) невозможно добавить в open_basedir.

Выбор расширения PHP

  1. Выберите Выберите PHP extension, чтобы настроить собственное расширение PHP.
  2. В раскрывающемся диалоговом окне Список PHP extension выберите расширение, которое хотите включить.
  3. Вы также можете нажать По умолчанию в верхней части диалогового окна для возврата к настройкам по умолчанию.
  4. Нажмите OK, чтобы сохранить изменения.
