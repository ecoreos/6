Win/Mac

Пользователи Windows, Mac и Linux могут получить доступ к DiskStation из сети, включив параметры в разделе Панель управления > Файловые службы > Win/Mac.

Поддерживаемые протоколы обмена файлами:

  • Для Windows: SMB/CIFS (Сетевое окружение), FTP, WebDAV
  • Для Mac: SMB, FTP, AFP, WebDAV
  • Для Linux: SMB, FTP, WebDAV
