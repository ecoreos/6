Служба резервирования

Служба резервирования позволяет выполнять резервное копирование данных с одного устройства Synology NAS на другое через пакет Резервирование и восстановление.

Включение сервера резервного копирования Synology

Включение службы резервирования:

Synology NAS можно превратить в место назначения резервирования данных/LUN для другого удаленного устройства Synology NAS.

  1. Выберите Панель управления > Служба резервирования.
  2. Выберите Включить место назначения резервирования для удаленных серверов Synology.
  3. Нажмите Применить, чтобы сохранить изменение.
  4. Откройте Панель управления > Папка общего доступа, выберите общую папку, нажмите Редактировать > Права доступа и предоставьте нужному пользователю право на Чтение/запись. Повторите этот шаг, чтобы добавить другие общие папки для использования в качестве целевых папок для архивирования.

Примечание.

  • Удаленное устройство Synology NAS должно использовать пакет Резервирование и восстановление для резервирования данных/LUN на устройство Synology NAS.
  • Чтобы разрешить резервирование LUN из удаленного устройства Synology NAS, выберите Панель управления > Файловые службы > rsync и нажмите Включить службу на целевой стороне.

Назначение права доступа на резервирование:

Назначьте пользователям соответствующие права доступа для резервного копирования с исходного устройства Synology NAS на целевое устройство.

  1. Выберите Панель управления > Права доступа, нажмите Резервирование Synology > Редактировать.
  2. Назначьте нужным пользователям права доступа Разрешить, при необходимости укажите разрешенные IP-адреса для конкретных пользователей.
