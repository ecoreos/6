Резервирование конфигурации

Чтобы создать резервную копию конфигурации на локальном компьютере, откройте Панель управления > Обновить и восстановить. Вы сможете сохранить различные настройки (пользователей, группы, настройки файловой службы и т. д.) для последующего восстановления. Системная конфигурация сохраняется в файл формата .dss.

Резервирование системных конфигураций:

  1. Нажмите Резервировать конфигурацию.
  2. Файл конфигурации будет загружен на ваш компьютер (с именем [Synology Product Name]_[Created Date].dss).

Восстановление системных конфигураций:

  1. Нажмите Восстановить конфигурацию.
  2. Нажмите Обзор и выберите конфигурационный файл, сохраненный на компьютере.
  3. Установите флажки, чтобы выбрать конфигурации для восстановления, а затем нажмите OK. Также вы можете выбрать параметр Перезаписать конфликтные настройки, чтобы перезаписать текущие настройки для выбранных пользователей, групп, общих папок и служб.
  4. Отобразится окно с предупреждением. Чтобы продолжить, нажмите кнопку Да.
  5. По завершении процесса восстановления отобразится окно с подтверждением. Нажмите OK для завершения.

Будет выполнено резервирование следующих настроек:

  • Пользователи, группы и папки общего доступа
  • Рабочая группа, домен и LDAP
  • Обмен файлами (например, файловая служба Windows, файловая служба Mac, служба NFS, FTP, WebDAV и rsync) и служба резервирования
  • Другие (например, главная папка пользователя, настройки паролей и планировщик задач)
  • Пользователи, группы и папки общего доступа
  • Обмен файлами (например, файловая служба Windows, файловая служба Mac, служба NFS, FTP, WebDAV)
  • Другие (например, главная папка пользователя, настройки паролей и планировщик задач)

Примечание.

  • Служба резервирования содержит базовые настройки служб, но не включает в себя информацию о задачах резервирования.
  • Главная папка пользователя содержит базовые настройки служб, но не включает в себя общую папку homes. Выберите пункт Hyper Backup для резервирования и восстановления общей папки homes и ее содержимого.
  • Планировщик задач содержит задачи, которые можно создать в Планировщике задач, но не включает в себя задачи, созданные другими приложениями (например, Hyper Backup и Отчет об использовании диска)
  • Если имена восстановленных пользователей или групп зарезервированы для системного использования, при восстановлении они будут изменены.
  • Главная папка пользователя содержит базовые настройки служб, но не включает в себя общую папку homes. Выберите пункт Hyper Backup для резервирования и восстановления общей папки homes и ее содержимого.
  • Планировщик задач содержит задачи, которые можно создать в Планировщике задач, но не включает в себя задачи, созданные другими приложениями (например, Hyper Backup и Отчет об использовании диска)
  • Если имена восстановленных пользователей или групп зарезервированы для системного использования, при восстановлении они будут изменены.
