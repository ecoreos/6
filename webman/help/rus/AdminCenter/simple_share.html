Папка общего доступа

Папки общего доступа являются базовыми каталогами, где можно хранить файлы и папки на DiskStation. Перед хранением файлов на DiskStation необходимо создать минимум одну папку общего доступа в DSM. Данные в папках общего доступа могут быть личными или общими для определенных пользователей или групп пользователей согласно настраиваемым правам доступа.

Ниже приведен список папок общего доступа по умолчанию, которые создаются автоматически при установке или активации определенных приложений, служб или пакетов.

Имя Описание
public Папка public создается автоматически при первой установке DiskStation.*
web Папка web является местом хранения по умолчанию для содержимого веб-сайтов, расположенных на DiskStation. Папка будет создана автоматически после включения службы Web Station.
photo В папке photo содержатся фотографии и видео для обмена с другими пользователями на Photo Station. Она будет создана автоматически при включении Photo Station или Сервера мультимедиа DLNA/UPnP.
music Папка music используется для хранения музыки для прослушивания с помощью Audio Station. Она будет создана автоматически при включении Audio Station или Сервера мультимедиа DLNA/UPnP.
video В папке video содержатся видео для просмотра с помощью устройства DLNA/UPnP. Она будет создана автоматически при включении Video Station или Сервера мультимедиа DLNA/UPnP.
surveillance В папке surveillance содержатся записи Surveillance Station. Папка будет создана автоматически после включения службы Surveillance Station. Она предназначена только для чтения и может использоваться только системным администратором.
home Папки home предоставляют каждому пользователю личное место для хранения данных, к которому имеет доступ только данный пользователь. Эти папки создаются автоматически при активации функции Главная папка пользователя.
homes В папке homes содержатся главные папки всех пользователей. Эти папки создаются автоматически при активации функции Главная папка пользователя. Только администратор может получать доступ и управлять всеми главными папками пользователей.
NetBackup Папка NetBackup создается автоматически при включении службы Сетевое резервирование.
usbshare Папка usbshare[номер] создается автоматически при подключении внешнего устройства USB к порту USB устройства DiskStation.*
satashare Папка satashare создается автоматически при подключении внешнего устройства eSATA к порту eSATA устройства DiskStation.*

Примечание.

  • В простом режиме вы можете только редактировать права доступа пользователей к папкам общего доступа. Если вы хотите создать группы пользователей или настроить права доступа групп к папкам общего доступа, выберите Панель управления (Расширенный режим) > Группа.

Создание папки общего доступа:

  1. Выберите Панель управления (Простой режим) > Папка общего доступа и нажмите значок + на левой панели.
  2. Укажите имя и выберите раздел, на котором будет создана папка общего доступа.
  3. При необходимости установите флажок Включить корзину. Если данная опция включена, при удалении файлов из папки общего доступа они будут перемещены в папку #recycle. Доступ к корзине можно ограничить пользователями, принадлежащими группе administrators.
  4. Установите или снимите соответствующие флажки для каждого пользователя, чтобы настроить их разрешения к папке общего доступа:
    • Чтение/запись: Пользователи могут получать доступ и изменять файлы и подпапки в папке общего доступа.
    • Только для чтения: Пользователь имеет доступ к файлам и подпапкам в папке общего доступа.
    • Нет доступа: Пользователь имеет доступ к файлам и подпапкам в папке общего доступа.
  5. Нажмите Применить, чтобы сохранить настройки.

Примечание.

  • Названия папок общего доступа вводятся с учетом регистра клавиатуры и могут быть длиной от 1 до 32 символов. Имя не может содержать следующие символы:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Следующие имена зарезервированы для системного использования:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • Первый символ в названии папки общего доступа должен быть знаком минуса или пробелом, и последний символ также не может быть пробелом.
  • В случае возникновения конфликта прав доступа будет действовать следующий приоритет: Нет доступа > Чтение/запись > Только для чтения.
  • Если при создании новой общей папки разрешения пользователя, принадлежащего группе administrators, будут установлены на Нет доступа, данный пользователь сможет просмотреть папку общего доступа в пункте Панель управления > Папка общего доступа.
  • Вы можете настроить параметры прав доступа корзины в File Station, нажав папку #recycle правой кнопкой мыши. Выберите Свойства и нажмите вкладку Разрешения.

Изменение папки общего доступа:

  1. Выберите Панель управления (Простой режим) > Папка общего доступа.
  2. В левой панели выберите необходимую папку общего доступа.
  3. Измените общую информацию о папке общего доступа, например имя, местоположение и разрешения пользователей.
  4. Если параметр Включить корзину уже активирован, нажмите Очистить корзину, чтобы удалить все удаленные файлы из корзины.
  5. Нажмите Применить, чтобы сохранить настройки.

Удаление папки общего доступа:

  1. Выберите Панель управления (Простой режим) > Папка общего доступа.
  2. В левой панели выберите необходимую папку общего доступа.
  3. Нажмите значок Корзина в левой панели, чтобы удалить выбранную папку общего доступа.

* Поддерживается только в некоторых моделях.
