Настройки DSM

В разделе Панель управления > Сеть > Настройки DSM можно изменить номер порта, который используется клиентом для доступа к DiskStation.

Изменение номера порта по умолчанию:

Оба соединения — HTTP и HTTPS — всегда включены на DSM. Порт HTTP по умолчанию — 5000, а HTTPS — 5001.

  1. Введите номер порта в поле HTTP или HTTPS.
  2. Нажмите Применить.

Перенаправление с HTTP на HTTPS

При попытке клиентов получить доступ к DiskStation через порт HTTP их можно перенаправлять на порт HTTPS.

  1. Выберите Автоматически перенаправлять подключения HTTP на HTTPS (кроме Web Station и Photo Station).
  2. Нажмите Применить.

Порты, используемые при подключении HTTPS:

  • DSM: По умолчанию номер порта – 5001, поэтому безопасный доступ к DSM можно получить, указав следующий адрес: https://server-hostname:5001/
  • Web Station и Photo Station: По умолчанию номер порта – 443, поэтому безопасный доступ к Web Station и Photo Station можно получить, указав следующий адрес: https://server-hostname/

Обнаружение DiskStation в сети

Найти DiskStation можно с помощью функции сетевого обнаружения Windows или функции обнаружения службы Bonjour.

  1. Выберите функцию для поиска вашего DiskStation в сети:
    • Включить обнаружение сети Windows
    • Включить функцию обнаружения службы Bonjour
  2. Нажмите Применить.

Домен

В разделе Домен можно указать настраиваемое имя домена для доступа к DiskStation.

Включение настраиваемого домена

  1. Установите флажок Включить настраиваемый домен.
  2. Укажите применяемое имя домена для своего DiskStation.
  3. Кроме того, можно установить флажок Включить HSTS, чтобы принудительно применять зашифрованные подключения HTTPS в этом домене.
  4. Нажмите Применить.

Примечание.

  • Каждое имя домена может быть использовано только для одного отдельного приложения на DiskStation.
  • Если домен не превышает в длину 15 символов и соответствует правилам наименования NetBIOS, система автоматически зарегистрирует и распространит имя домена (например, MyDSM). Пользователи Windows могут получать доступ к рабочему столу DSM через ту же локальную сеть с таким же URL-адресом: http://MyDSM или https://MyDSM
  • При доступе к приложениям по URL-адресам, использующим имена доменов, вы будете направлены через стандартный порт HTTP (80) или HTTPS (443).
