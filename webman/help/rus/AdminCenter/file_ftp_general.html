Общие

В меню Панель управления > Файловые службы > FTP вы можете включить ограничение скорости, ведение журналов передачи файлов, настройки безопасности, а также ряд других параметров в разделе Общие.

Включение ведения журнала передачи файлов по FTP

Нажмите Дополнительные настройки для включения этой функции и записи журналов загруженных и выгруженных файлов.

Примечание.

Включение журнала передачи файлов по FTP может повлиять на производительность системы.

Изменение корневых каталогов пользователя

Нажмите Дополнительные настройки и установите флажок Изменить корневые каталоги пользователя. После этого выбранным пользователям, выполнившим вход в DSM по протоколу FTP, будут доступны только определенные папки.

Создание правила изменения корневого каталога:

  1. В разделе Дополнительные настройки установите флажок Выбрать пользователя.
  2. Нажмите Добавить.
  3. В разделе Пользователь или группа укажите, к кому будет применяться это правило.
  4. Выберите корневой каталог, который будет доступен пользователю после входа в систему. Выберите один из следующих вариантов.
    • Главная папка пользователя. После входа в систему пользователь попадает в свою главную папку.
    • Другой каталог. Выберите общую папку, которая будет использоваться в качестве корневого каталога, и нажмите Выбрать. Примечание. Сначала необходимо выбрать разрешения для пользователя или группы. При отсутствии достаточных разрешений пользователь или группа не сможет войти в систему.
  5. Нажмите OK.
  6. После добавления правил нажмите Применить, чтобы сохранить правила.

Примечание.

  • Приоритет правил соответствует их позициям в списке.
  • Изменить порядок правил можно путем перетаскивания.

Включение анонимной учетной записи FTP

Нажмите Дополнительные настройки и включите параметр, позволяющий любому пользователю использовать службу FTP для входа в DiskStation (с учетной записью anonymous или ftp).

Изменение анонимного корневого каталога

Нажмите Дополнительные настройки и установите флажок на опции для изменения каталога для входа анонимного пользователя или пользователя FTP. В раскрывающемся меню выберите папку общего доступа.

Применение разрешений UNIX по умолчанию

Нажмите Дополнительные настройки и включите параметр, позволяющий применять разрешения UNIX по умолчанию при выгрузке или создании папок и файлов. Те же разрешения можно применить с помощью команды UNIX umask. Если этот параметр включен, для файлов используется разрешение UNIX 644, а для папок — разрешение 755. Если этот параметр отключен, для файлов используется разрешение UNIX 666, а для папок — разрешение 777. По умолчанию параметр unmask имеет значение 022.

Примечание.

  • Для всех папок общего доступа с включенными разрешениями Windows ACL (кроме папки "photo" и папок на внешних дисках) необходимо выполнить команду chmod на клиенте Linux или FTP. В результате типы разрешений для папок и файлов изменятся с Windows ACL на UNIX.
  • Включение данного параметра может привести к несогласованности разрешений между различными протоколами. Во избежание несогласованности рекомендуется не включать данный параметр.

Включение ограничения скорости

Включите ограничение скорости, чтобы управлять максимальной используемой шириной пропускания сети для пользователей и групп, передающих данные с помощью FTP.

Включение ограничения скорости:

  1. Установите флажок Включить ограничение скорости или Включить ограничение скорости согласно расписанию. В последнем случае ограничение скорости будет применяться только в определенное время. Эта настройка применяется на всех серверах и переопределяет настройки отдельных пользователей и групп. Если выбран параметр Нет ограничения скорости, ограничения скорости для пользователей и групп отсутствуют.
  2. Нажмите кнопку Настройки ограничения скорости, чтобы изменить настройки для отдельных пользователей или групп. Для каждого пользователя или группы можно выбрать один из следующих параметров.
    • Применить настройки группы: настройки ограничения скорости будут совпадать с настройками группы, к которой принадлежит пользователь.
    • Нет: ограничение скорости не действует, когда участники группы передают файлы с помощью этой службы.
    • Постоянная: к отдельным пользователям или участникам группы будет применяться фиксированное постоянное ограничение скорости. В полях слева можно указать ограничения скорости при выгрузке и загрузке файлов.
    • Пользовательский: к отдельным пользователям или участникам группы применяются два различных ограничения скорости, которые включаются по расписанию. Нажмите кнопку Настроить, чтобы изменить настройки ограничения скорости и настроить расписание.
  3. Нажмите Применить.

Примечание.

  • Дополнительные сведения о динамике ограничений скорости группы см. здесь.
  • При установке ограничения скорости для пользователя в разделе Результат отображается фактическое ограничение скорости пользователя с учетом индивидуальных и групповых настроек. Например, если для пользователя "Achilles" установлено индивидуальное ограничение скорости загрузки 100 КБ/с, но этот пользователь является участником группы с ограничением скорости загрузки 500 КБ/с, в разделе Результаты будет отображаться значение "0/500."
