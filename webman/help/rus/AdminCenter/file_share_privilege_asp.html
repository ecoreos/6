Управление расширенными правами доступа к папке общего доступа

На странице Панель управления > Папка общего доступа > Редактировать > Дополнительно содержатся следующие параметры для более широкой настройки прав доступа к папке общего доступа.

Расширенные права доступа к папке общего доступа

Расширенные права доступа предоставляют дополнительный уровень управления правами доступа для папок общего доступа. При включении пользователи и группы могут просматривать и изменять содержимое папки общего доступа только если пользователь или группа имеют расширенные права общего доступа и права доступа Windows ACL (Папка общего доступа > Редактировать > Права доступа).

Включение расширенных прав общего доступа:

  1. Установите флажок Включить расширенные права общего доступа.
  2. Нажмите Расширенные права общего доступа, чтобы изменить расширенные права общего доступа для папки общего доступа.

Примечание.

  • Расширенные права общего доступа применяются, если пользователь осуществляет доступ к папке общего доступа с помощью следующих файловых служб: Windows File Sharing, Apple File Sharing, File Station, FTP и WebDAV.

Дополнительные настройки

При необходимости можно создать дополнительные ограничения для пользователей, когда они осуществляют доступ к папке общего доступа с помощью File Station, FTP или WebDAV.

  • Отключить просмотр каталога: Включение данного параметра запрещает пользователям просмотр содержимого папок общего доступа.
  • Отключить изменение существующих файлов: Включение данного параметра запрещает пользователям перемещение, удаление и изменение файлов в папке общего доступа. Обратите внимание, что пользователи смогут просматривать, загружать/выгружать, копировать и распаковывать содержимое папки общего доступа.
  • Отключить загрузку файлов: Включение данного параметра запрещает пользователям загрузку содержимого папок общего доступа.

Примечание.

Эти ограничения не применяются к пользователям, принадлежащим группе administrators.
