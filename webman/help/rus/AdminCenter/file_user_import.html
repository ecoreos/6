Импорт пользователей

Можно создавать учетные записи пользователей в пакетном режиме. Для этого используется функция импорта списка пользователей.

Создание списка пользователей:

Метод 1. Использование текстового редактора:

  1. Откройте новый файл в текстовом редакторе.
  2. Введите информацию о пользователе в следующем порядке со знаком табуляции в качестве разделителя:
    • Имя пользователя
    • Пароль
    • Описание
    • Электронная почта
    • Квота (МБ) в томе 1 (при наличии нескольких томов повторите этот шаг)

      Примечание.

      Выделенная пользователю квота не поддерживается в файловой системе Btrfs и на внешних устройствах хранения данных (например, диски USB/eSATA).

  3. Перейдите на следующую строку и повторяйте шаг 2 до тех пор, пока не будут введены все пользователи. Одна строка предназначена для ввода информации об одном пользователе.
  4. Сохраните файл в кодировке UTF-8.

Метод 2. Использование Microsoft Excel:

  1. Откройте новый файл в программе Excel.
  2. Введите информацию о пользователях в следующем порядке, разделяя данные в строке пустым столбцом:
    • Имя пользователя
    • Пароль
    • Описание
    • Электронная почта
    • Квота (МБ) в томе 1 (при наличии нескольких томов повторите этот шаг)

      Примечание.

      Выделенная пользователю квота не поддерживается в файловой системе Btrfs и на внешних устройствах хранения данных (например, диски USB/eSATA).

  3. Перейдите на следующую строку и повторяйте шаг 2 до тех пор, пока не будут введены все пользователи. Одна строка предназначена для ввода информации об одном пользователе.
  4. Откройте новый файл в текстовом редакторе. Скопируйте записи из листа Excel и вставьте их в текстовый файл.
  5. Сохраните файл в кодировке UTF-8.

Примечание.

  • Все поля, за исключением поля "Имя пользователя", являются необязательными.
  • Пример текстового файла:
    • rosy 1234 мать rosy@abc.com 80000
    chaddy 5678 брат chaddy@abc.com 80000
    danny 8901 кузен danny@abc.com 20000
    anny 2345 сестра anny@abc.com 20000

Импорт списка пользователей:

  1. Выберите Создать > Импортировать пользователей.
  2. Установите флажок Перезаписать дублирующиеся учетные записи, если вы хотите заменить существующие дублирующиеся учетные записи на записи из списка пользователей.
  3. Нажмите Обзор и выберите файл списка пользователей. После выбора файла в области предварительного просмотра будет отображен список импортированных пользователей. Пользователи с непредусмотренным статусом не будут импортированы.
  4. Чтобы создать учетные записи пользователей, нажмите Отправить.

Примечание.

Файл списка пользователей должен иметь формат UTF-8.
