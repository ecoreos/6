Общие

Общие параметры сети, например шлюз по умолчанию, сервер DNS, прокси и тунеллирование IPv6 доступны в пункте Панель управления > Сеть > Общие.

Переименование устройства DiskStation

Чтоб переименовать DiskStation, введите новое имя сервера и нажмите Применить.

Имя сервера вводится без учета регистра клавиатуры и может содержать от 1 до 15 знаков, в числе которых могут быть буквы, цифры, нижние подчеркивания и знаки минус. Первый символ имени должен быть буквой.

Выбор шлюза по умолчанию

Введите IP-адрес для шлюза по умолчанию и нажмите Применить, чтобы сохранить настройки.

Указание IP-адреса сервера DNS

Вы можете настроить IP-адрес сервера DNS вручную. В этом случае необходимо ввести IP-адрес необходимого сервера DNS. Можно также ввести IP-адрес сервера DNS на случай, если на нужном сервере DNS обнаружатся проблемы. После настройки IP-сервера сервера DNS нажмите Применить, чтобы сохранить настройки.

Примечание.

Если все интерфейсы настраиваются автоматически или если используется подключение PPPoE, поля "Сервер DNS" и "Шлюз по умолчанию" будут заполнены автоматически и не могут быть изменены.

Подключение с помощью прокси-сервера

Для подключения DiskStation с помощью прокси-сервера установите флажок Подключиться с помощью прокси-сервера, введите Адрес и Порт, а затем нажмите Применить.

Примечание.

  • Нажмите Дополнительные настройки, чтобы настроить прокси HTTP, HTTPS и информацию об аутентификации.
  • По умолчанию настройки прокси-сервера HTTPS совпадают с настройками прокси-сервера HTTP.
  • При активации параметра Подключиться с помощью прокси-сервера функции Обновление DSM и Центр пакетов вместе с другими пакетами будут подключены к Интернету через прокси.
  • Если установлен флажок Обходить прокси-сервер для локального адреса, убедитесь, что сервер DNS, используемый DiskStation, может разрешать запросы для устройств в локальной сети.

Включение тунеллирования IPv6

Пока IPv6 полностью не вытеснит IPv4, тунеллирование IPv6 требуется для доступа хостов с адресами IPv6 к службам IPv4. Данная технология позволяет перераспределять трафик IPv6 в рамках существующих сетей IPv4.
Если вы не знаете, как настраивать IPv6 или устранять неисправности подключения, обратитесь к провайдеру услуг Интернет.

Включение IPv6 Tunnel:

  1. Установите флажок Включить tunnel, чтобы включить поддержку tunnel.
  2. В поле Адрес сервера введите имя хоста или IP-адрес провайдера услуг tunnel, например: anonymous.freenet6.net.
  3. Выберите Подключаться анонимно или Подключиться с помощью существующей учетной записи.

Примечание.

Внешний адрес IPv6 может обновляться провайдером услуг автоматически. Однако DiskStation не сразу выполнит удаление старого адреса IPv6.
