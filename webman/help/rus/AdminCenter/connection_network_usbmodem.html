Мобильная связь*

Если устройство DiskStation оснащено ключом мобильного широкополосного доступа, оно может подключаться к Интернету через беспроводную сеть 3G/LTE. Настройки в пункте Панель управления > Сеть > Сетевой интерфейс > Мобильная связь помогут установить и изменить беспроводное интернет-подключение с помощью ключа мобильного широкополосного доступа.

Примечание:

Для получения полного списка ключей мобильного широкополосного доступа, поддерживаемых продуктами Synology посетите веб-сайт www.synology.com.

Настройка мобильной сети:

После подключения ключа мобильного широкополосного доступа к одному из портов USB на DiskStation устройство DiskStation выполнит попытку автоматического подключения. Если подключение невозможно установить автоматически или вы хотите изменить настройки подключения, см. следующее:

  • Разблокировка/настройка SIM-карты: Если SIM-карта защищена PIN-кодом, ее необходимо разблокировать, чтобы устройство DiskStation могло подключиться к сети. После разблокировки карты в данном разделе будут приведены параметры для настройки параметров PIN-кода.
  • Редактирование: Можно вручную настроить подключение или выбрать имеющиеся в соответствии с местоположением и провайдером. Настройка подключения вручную требует ввода APN, номера телефона, имени пользователя и пароля. Если вам неизвестна данная информация, обратитесь к провайдеру мобильной связи.
  • Подключение/отключение: Данные кнопки позволяют подключать и отключать DiskStation от сети 3G/LTE.
  • Параметры: В данном разделе содержится параметр, который позволяет настроить интерфейс ключа мобильного широкополосного доступа в качестве шлюза по умолчанию для устройства DiskStation. При настройке мобильной сети в качестве шлюза по умолчанию интернет-трафик с устройства DiskStation будет проходить через мобильную, а не через проводную сеть.

Примечание:

  1. При подключении к Интернету некоторые аппаратные ключи мобильного широкополосного доступа не отображают силу сигнала, если данная функция не поддерживается.
  2. Для некоторых аппаратных ключей мобильного широкополосного доступа невозможно изменить настройки SIM при подключении к Интернету.
