Обновление DSM

Периодически компания Synology выпускает бесплатные обновления для DSM. Обновления могут включать в себя новые функции, усовершенствования функций и решения для повышения производительности.

Важно! Обычно обновление DSM не влияет на данные в DiskStation. Однако, чтобы обеспечить безопасность данных, настоятельно рекомендуется зарезервировать данные перед началом процесса обновления.

Обновление DSM вручную

Вы можете загрузить файл обновления .pat в Центре загрузок Synology и выполнить обновление DSM вручную.

Обновление DSM вручную:

  1. Посетите сайт www.synology.com, чтобы загрузить последнюю версию DSM на свой локальный компьютер. Распакуйте файл, чтобы извлечь файл обновления .pat.
  2. Нажмите Обновление DSM вручную для перехода на страницу обновления DSM вручную
  3. Нажмите кнопку Обзор, чтобы выбрать файл обновления.
  4. Нажмите кнопку Применить, чтобы начать обновление DSM. Система перезагрузится автоматически по завершении обновления.

Устранение неполадок:

Если не удается выполнить обновление микропрограммы системы, следующие предложения могут помочь в решении данной проблемы:

  • Проверьте, не был ли процесс обновления прерван пользователями или в результате непредвиденных событий, например, отключения питания.
  • Проверьте версию DSM. Обновление возможно только при наличии более поздней версии DSM.
  • Еще раз загрузите обновление DSM, поскольку файл обновления мог быть поврежден.

Если проблема все еще не устранена, обратитесь в Службу технической поддержки Synology.

Автоматическое обновление DSM

Помимо обновления вручную при каждом посещении страницы Обновление DSM система будет проверять наличие новых версий DSM от Synology и отображать результаты в виде зеленого или красного текста. Если последняя версия DSM готова к загрузке, вы можете позволить системе выполнить загрузку DSM автоматически, а затем осуществить обновление.

Автоматическая загрузка DSM и обновление:

  1. Если последняя версия DSM готова к загрузке, нажмите Загрузить, чтобы система автоматически загрузила последнюю версию DSM.
  2. Система отобразит одно из следующих уведомлений о состоянии загрузки: Загрузка DSM завершена или Не удалось загрузить DSM.
  3. Нажмите окно уведомления для перехода к пункту Обновление DSM.
  4. Выполните одно из следующих действий:
    • Если загрузка DSM завершена, нажмите Обновить сейчас, чтобы начать обновление DSM немедленно.
    • Если загрузка DSM не удалась, нажмите Повторить, чтобы система выполнила загрузку последней версии DSM еще раз.

Регулярная проверка обновлений:

  1. Установите флажок Проверять наличие обновлений DSM автоматически.
  2. Выберите план обновлений и настройте График установки.
  3. Нажмите OK.

На рабочем столе отобразится уведомление, когда обновление DSM будет загружено и готово к установке. Чтобы отключить автоматическое обновление DSM, отмените его в уведомлении на рабочем столе или изменив параметры в пункте Настройки обновления. Обновления DSM будут установлены до настроенного выключения компьютеры, если включен пункт Выключение в меню График включения и отключения питания.

Устранение неполадок:

При отображении сообщении об ошибке (красный текст) во время проверки или загрузки последней версии DSM следующие действия могут помочь устранить проблему:

  • Убедитесь, что DiskStation подключен к Интернету.
  • Сервер загрузок Synology в данный момент может быть перегружен. Попробуйте повторить загрузку позже.

Если проблема все еще не устранена, обратитесь в Службу технической поддержки Synology.
