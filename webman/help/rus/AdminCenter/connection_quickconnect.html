QuickConnect

Служба QuickConnect позволяет клиентским приложениям подключаться к DiskStation через Интернет без необходимости настройки правил переадресации портов. QuickConnect может также работать с пакетами Synology (например, Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Note Station, CMS, Cloud Station и мобильные приложения).

Включение QuickConnect:

  1. Выберите Панель управления > QuickConnect > Общие.
  2. Установите флажок Включить QuickConnect.
  3. Введите данные вашей учетной записи Synology или зарегистрируйте новую учетную запись.
  4. Укажите новый идентификатор QuickConnect ID. Выберите легкий для запоминания код, чтобы обращаться к DiskStation в любое время.
  5. Нажмите Применить.

Примечание

Идентификатор QuickConnect ID должен начинаться с буквы и может содержать буквы, числа и знак дефиса ("-").

Улучшение качества соединения QuickConnect

DiskStation предлагает расширенные возможности, благодаря которым QuickConnect адаптируется к различным сетевым средам.

  1. Выберите Панель управления > QuickConnect > Дополнительно.
  2. Выберите наиболее подходящие параметры ниже.
    • Включить службу ретрансляции QuickConnect: подключение QuickConnect будет ретранслироваться через сервер Synology Relay Server, если прямой доступ к DiskStation в текущей сетевой среде недоступен.
    • Автоматически создавать правила переадресации портов: если маршрутизатор DiskStation поддерживает протокол UPnP, этот маршрутизатор информируется о необходимости создания правил переадресации портов для QuickConnect.
  3. Нажмите Применить.

Примечания

  • Ретранслированное соединение QuickConnect может выполняться дольше из-за задержек в сети.
  • Когда маршрутизатору разрешено создавать правила переадресации портов для QuickConnect, это создает определенный риск нарушения безопасности DiskStation.

Включение и отключение QuickConnect для определенных приложений или сервисов

  1. Выберите Панель управления > QuickConnect > Дополнительно.
  2. Выберите приложения или сервисы, для которых необходимо включить QuickConnect.
  3. Нажмите Применить.

Примечание

Для повышения производительности QuickConnect рекомендуется выбрать Панель управления > Внешний доступ > Конфигурация маршрутизатора, чтобы настроить переадресацию порта для каждой службы:

  • DSM: 5000 (HTTP), 5001 (HTTPS)
  • Photo Station: 80 (HTTP), 443 (HTTPS)
  • Cloud Station: 6690

Устранение неполадок

Если появляется сообщение Произошла ошибка сети, это означает, что служба QuickConnect была остановлена из-за ошибки сети. Проверьте следующее:

  • Устройство DiskStation подключено к активной сети (Панель управления > Сеть > Сетевой интерфейс).
  • Сервер DNS и шлюз по умолчанию настроены правильно (Панель управления > Сеть > Общие).
  • Убедитесь, что DiskStation работает в Интернете, используя следующие порты назначения: 80, 443, 8888.

После устранения ошибок сети служба QuickConnect должна автоматически возобновиться за несколько минут.
