Файловая служба Mac

В меню Панель управления > Файловые службы > Win/Mac/NFS вы можете включить файловую службу Mac, чтобы сделать DiskStation файловым сервером Mac (AFP-сервером).

Включение файловой службы Mac:

  1. Установите флажок Включить файловую службу Mac, чтобы сделать DiskStation файловым сервером Mac (AFP-сервером).
  2. При необходимости укажите папку для резервного копирования Time Machine. (Для получения дополнительной информации см. раздел ниже.)
  3. Нажмите Применить.

Примечания

  • Чтобы выполнить резервное копирование данных из Time Machine на DiskStation, выберите общую папку из раскрывающегося меню.
  • Общие папки, используемые другими службами, будут недоступны для Time Machine.

Включения журнала передачи

Выбор этого параметра включит ведение журнала операций доступа к файлам, включая создание, удаление, чтение, запись, перемещение и переименование файлов или папок по протоколу AFP, известному как протокол передачи файлов Apple.

Дополнительные настройки

Применение разрешений UNIX по умолчанию.

Включение данного параметра позволяет применять разрешения UNIX по умолчанию при загрузке или создании папок и файлов. Те же разрешения можно применить с помощью команды UNIX umask. Если этот параметр включен, для файлов используется разрешение UNIX 644, а для папок — разрешение 755. Если этот параметр отключен, для файлов используется разрешение UNIX 666, а для папок — разрешение 777. По умолчанию параметр unmask имеет значение 022.

Примечание

Включение данного параметра может привести к несогласованности разрешений между различными протоколами. Во избежание несогласованности рекомендуется не включать данный параметр.

Немедленное освобождение ресурсов после отключения.

Включение данного параметра позволяет немедленно освобождать ресурсы при непредвиденном отключении сервера AFP. Включите этот параметр, если вы получаете сообщение об ошибке "Том используется" при использовании Time Machine.

Включение вещания принтера Bonjour:

Установите флажок Включить вещание принтера Bonjour, чтобы разрешить клиентам Mac просматривать и находить принтер, подключенный к DiskStation.

Поиск содержимого файлов с помощью Spotlight в Finder

После завершения индексирования папок и размещения их на компьютере Mac, можно использовать Spotlight в Проводнике для поиска файлов и их содержимого.

  1. Выберите Панель управления > Служба индексации > Индексирование файлов > Список проиндексированных папок > Создать для индексации определенных папок и содержащихся в них файлов по типу.
  2. Для выполнения индексирования нажмите OK.
  3. Подключите проиндексированные папки к компьютеру Mac.
  4. На компьютере Mac нажмите на нужную подключенную папку на панели Finder слева и Spotlight в Finder для поиска файлов в этой папке.

Примечания

  • Для поиска в подключенных папках используйте Spotlight в Finder. Spotlight на рабочем столе Mac выполняет поиск только локальных файлов.
  • В Mac OS X 10.9 и более поздних версиях Spotlight в Finder поддерживает поиск подключенных папок по имени ярлыка и по категории.
