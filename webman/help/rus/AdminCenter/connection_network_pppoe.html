Настройки PPPoE

Если для подключения к Интернету используется DSL или кабельный модем, и ваш провайдер услуг Интернет (ISP) использует PPPoE (Протокол точка-точка по Ethernet), можно включить PPPoE в пункте Панель управления > Сеть > Сетевой интерфейс и вводить информацию учетной записи для прямого подключения к Интернету без маршрутизатора.

Для получения дальнейшей информации о PPPoE обратитесь к своему интернет-провайдеру или к администратору сети.

Настройка IPv6

Устройства DiskStation поддерживают двойную схему блоков IPv4/IPv6. Поэтому IPv4 и IPv6 могут работать на DiskStation одновременно. По умолчанию в параметрах IPv6 установлено Авто. Выберите существующий сетевой интерфейс, нажмите кнопку Редактировать и выберите вкладку IPv6 для изменения настроек.

Пока IPv6 полностью не вытеснит IPv4, механизмы перехода IPv6 требуются для доступа хостов с адресами IPv4 к службам IPv6. Выберите один из механизмов перехода IPv6:

  • 6in4: Выполните подключение к сети IPv6 с помощью сервиса туннелей. Вам необходимо ввести данные в поля Адрес IPv6, Длина префикса, Префикс и Адрес IPv4 сервиса туннелей. Для этого требуется публичный адрес IPv4.
  • 6to4: Выполните подключение к сети IPv6 через сервис туннелей с помощью произвольной рассылки. Для этого требуется публичный адрес IPv4.
  • DHCP-PD: Получите префикс IPv6 от сервера DHCP с помощью делегирования префикса в рамках ЛВС. Для этого требуется поддержка ISP; функция работает только в режиме маршрутизатора.
