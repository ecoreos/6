Редактирование группы

Вы можете изменять информацию о группе и права доступа к общим папкам.

Редактирование информации о группе:

  1. Выберите группу для редактирования.
  2. Нажмите Редактировать и отредактируйте информацию о группе.
  3. Нажмите OK для завершения.

Редактирование прав доступа группы к общим папкам:

  1. Выберите группу для редактирования.
  2. Нажмите Редактировать и Разрешения.
  3. Установите или снимите флажок со следующих прав доступа, чтобы назначить права доступа группе:
    1. Чтение/Запись: группа может получать доступ и вносить изменения в общую папку.
    2. Только для чтения: группа может получать доступ к общей папке, но не может вносить в нее изменения.
    3. Нет доступа: группа не имеет права доступа к общей папке.
  4. Нажмите OK для завершения.

Примечание.

В случае конфликта прав доступа будет действовать следующий приоритет: Нет доступа > Чтение/запись > Только для чтения.

Редактирование квоты использования группы:

  1. Выберите группу для редактирования.
  2. Нажмите Редактировать и перейдите на вкладку Квота.
  3. Для каждого тома/общей папки установите флажок под полем Включить квоту. Введите квоту использования в поле Квота (например, 10 ГБ).

Примечание.

  • Каждый член группы наследует указанную квоту использования группы. Квота использования группы не действует на всех участников группы.
  • Максимальная квота использования хранилища для каждой группы составляет примерно 4 ТБ (4095 ГБ).

Редактирование прав доступа группы к приложениям:

  1. Выберите группу для редактирования.
  2. Нажмите Редактировать и перейдите на вкладку Приложения.
  3. Для каждой службы можно выбрать один из следующих параметров:
    • Разрешить: Группа может осуществлять доступ к приложению.
    • Запретить: Группа не может осуществлять доступ к приложению.
    • Пользовательский: Управление правами доступа по IP-адресу.
  4. Нажмите OK для завершения.

Примечание.

  • В случае конфликта прав доступа будет действовать следующий приоритет: Запретить > Разрешить.
  • Не все пакеты и службы поддерживают настройки прав доступа По IP-адресу.

Редактирование ограничения скорости группы:

Вы можете активировать ограничение скорости групп для различных служб (например, File Station, FTP и др.) для ограничения ширины полосы пропускания, потребляемой участниками группы, которые выполняют передачу файлов с помощью каждой службы. Ограничения скорости группы действуют на всех участников группы.

  1. Выберите группу для редактирования.
  2. Нажмите Редактировать и выберите вкладку Ограничение скорости.
  3. Для каждой службы можно выбрать один из следующих параметров:
    • Нет. Ограничение скорости не будет применяться, если участники группы используют эту службу.
    • Постоянная. К участникам группы будет применяться фиксированное постоянное ограничение скорости. Ограничение скорости на загрузку и отправку можно указать в полях слева.
    • Переменная. К участникам группы могут применяться два различных ограничения скорости по расписанию. Нажмите кнопку Настроить, чтобы изменить настройки ограничения скорости и настроить расписание.
  4. Нажмите OK для завершения.

Ограничение скорости для пользователей, принадлежащих нескольким группам

Если пользователь принадлежит нескольким группам, его ограничение скорости рассчитывается на основе группы с наиболее высокой полосой пропускания на момент передачи файлов. Поскольку полоса пропускания равномерно распределяется между пользователями, выполняющими передачу файлов, доступная полоса пропускания меняется в зависимости от количества пользователей, использующих общую полосу пропускания группы.

Например, пользователь "Джон" принадлежит "Группе 1" (ограничение скорости 500 Кбит/с; всего пять пользователей) и "Группе 2" (ограничение скорости 400 Кбит/с; всего два пользователя).

  • Если все пользователи Группы 1 и Группы 2 одновременно выполняют передачу файлов, то скорость Джона будет не более 200 Кбит/с, а его трафик будет рассчитываться на основе квоты общей полосы пропускания Группы 2.
  • Если другие пользователи Группы 1 завершили передачу файлов, то скорость Джона повысится до 500 Кбит/с, а его трафик будет рассчитываться на основе квоты общей полосы пропускания Группы 1.
