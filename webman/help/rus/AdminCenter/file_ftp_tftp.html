TFTP

TFTP (Trivial File Transfer Protocol) — это простой протокол передачи файлов, который обычно используется для передачи конфигураций или файлов загрузки, когда не требуется аутентификация. Включение службы TFTP позволяет устройству DiskStation передавать загрузчики на клиентские устройства.

Настройка параметров TFTP:

  1. Выберите Панель управления > Файловые службы > TFTP и нажмите Включить службу TFTP.
  2. В поле Корневая папка TFTP укажите, к какой папке на устройстве DiskStation могут иметь доступ клиенты TFTP.
  3. Нажмите Дополнительные настройки, чтобы настроить следующие параметры:
    • Включить журнал передачи по TFTP. Отслеживание данных, переданных по протоколу TFTP.
    • Разрешение клиента TFTP. Определение разрешения клиента TFTP.
      • Только для чтения: Клиенты TFTP могут только просматривать содержимое корневой папки.
      • Доступно для записи: Клиенты TFTP могут изменять содержимое корневой папки.
    • Разрешенные клиенты. Выберите, каким клиентам разрешено подключаться по протоколу TFTP.
      • Все подключения: Все клиенты могут подключаться по TFTP.
      • Разрешить только следующий диапазон IP-адресов: Только клиенты в указанном IP-диапазоне могут подключаться по TFTP.
    • Время ожидания. Если сервер TFTP (т.е. ваше устройство DiskStation) не отвечает в течение указанного промежутка времени, подключение TFTP будет прервано.
  4. Нажмите Применить, чтобы сохранить настройки.
