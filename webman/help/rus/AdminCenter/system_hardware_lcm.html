Дисплей LCM*

Вы можете установить пароль дисплея LCM для предотвращения изменения настроек программы неавторизованными пользователями с помощью дисплея LCM.

Активация защиты дисплея LCM с помощью пароля:

  1. Установите флажок Активация защиты дисплея LCM с помощью пароля.
  2. Введите Пароль и подтвердите пароль в поле Подтвердите пароль.
  3. Нажмите Применить.

____
* Доступно только на некоторых моделях.
