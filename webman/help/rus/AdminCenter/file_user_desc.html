Пользователь

Благодаря гибким параметрам управления пользователями можно создавать учетные записи для ваших друзей или коллег. Создание пользователей позволяет задавать и управлять настройками каждого пользователя, например правами доступа к общей папке, квотами для хранилища/общей папки и ограничением полосы пропускания.
