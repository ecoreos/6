Спящий режим

Вы можете указать время неактивности жесткого диска, по истечение которого он перейдет в спящий режим. Находясь в спящем режиме прекращается вращение жестких дисков, снижается потребление питания и увеличивается срок их службы. Вы можете указать различные периоды времени для внутренних жестких дисков и внешних жестких дисков eSATA/USB.

Примечание.

  • Спящий режим отключается при доступе к жесткому диску.
  • Только некоторые внешние USB-диски поддерживают спящий режим.

Почему DiskStation не переходит в спящий режим?

Обратите внимание, что следующие службы не позволяют DiskStation переходить в спящий режим. Кроме того, система выйдет из спящего режима за две минуты до начала запланированной задачи.

Пакеты:

  • Download Station: DiskStation не сможет перейти в спящий режим, если на Download Station запущено eMule, запланирована загрузка RSS или есть активные задачи.
  • Mail Station: DiskStation не сможет перейти в спящий режим, если Mail Station выполняет сбор почты с POP3.
  • Surveillance Station: DiskStation не сможет перейти в спящий режим, если Surveillance Station выполняет запись.
  • Сервер мультимедиа: DiskStation не сможет перейти в спящий режим, если журнал DMA активирован в параметрах сервера мультимедиа (перейдите на страницу Главное меню > Сервер мультимедиа > Сервер мультимедиа > Диагностика).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Другие программные приложения сторонних производителей, например SABnzbd

Процессы:

  • Значки и индексирование: DiskStation не сможет перейти в спящий режим при создании значков и переиндексировании мультимедийных файлов после обновления. При этом процесс “synoindexd” будет иметь состояние “Выполняется” (перейдите на страницу Главное меню > Мониторинг ресурсов > Процесс).
  • Local Master Browser: DiskStation не сможет перейти в спящий режим, если включен Local Master Browser (перейдите на страницу Панель управления > Win/Mac/NFS > Файловая служба Windows > Включить Local Master Browser).
  • Проигрыватель Windows Media: DiskStation не сможет перейти в спящий режим, если включена служба общего доступа к проигрывателю Windows Media по сети.
  • DiskStation не сможет перейти в спящий режим, если несколько одновременно выполняемых процессов требуют переключения, при котором емкость ОЗУ превышена, а для операций чтения/записи временно используются жесткие диски.

Другие

  • USB-устройства: DiskStation не сможет перейти в спящий режим системы, если к устройству подключены USB-устройства.

Спящий режим системы

Вы можете настроить DiskStation для перехода в спящий режим через одну минут после жестких дисков. Спящий режим снижает потребление питания устройством DiskStation.

В спящем режиме:

  1. Кнопка питания медленно включается и выключается. Другие индикаторы отключены.
  2. Электропитание жестких дисков отключено.
  3. Работа процессора остановлена.
  4. Работа внешнего вентилятора остановлена.
  5. Индикаторы порта LAN отключены.

Примечание.

  • Пакет вещания SMB/CIFS может ограничивать переход сервера в спящий режим системы, включая работающие проводники Windows в сети. Если вы хотите игнорировать пакеты вещания CIFS при входе в спящий режим системы, выберите Игнорировать пакеты вещания из Проводника Windows. После выбора этой опции пакеты SMB/CIFS будет невозможно использовать для выхода из спящего режима системы. При подключении USB-устройств спящий режим системы на сервере активирован не будет.

Глубокий сон

Глубокий сон для устройств расширения можно активировать с помощью сопряженного DiskStation. Вы можете настроить устройство расширения и сопряженное устройство DiskStation на переход в неактивный режим через разные промежутки времени. Поэтому устройства расширения могут переходить в режим глубокого сна, когда DiskStation остается активным.

В режиме глубокого сна:

  • Все индикаторы на устройстве расширения гаснут.
  • Электропитание жестких дисков устройства расширения отключается.
  • Вентилятор устройства расширения останавливается.

Примечание.

  • Режим глубокого сна отключается при доступе к жесткому диску.
  • Для обеспечения нормальной работы при использовании функции глубокого сна на устройстве расширения необходимо перевести переключатель Авто/Вручную на его задней панели в положение Авто.

Автоматическое отключение питания

Вы можете указать продолжительность временного периода, в течение которого внутренний диск остается в спящем режиме до автоматического отключения питания DiskStation.

Примечание.

  • Сначала необходимо активировать спящий режим внутреннего жесткого диска.
  • Необходимо сначала активировать службу WOL, чтобы удаленно включать питание DiskStation.
