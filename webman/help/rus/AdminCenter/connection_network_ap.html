Зона доступа*

DiskStation поддерживает создание зоны доступа для предоставления общего доступа к интернет-подключению другим устройствам беспроводным способом, если установлен USB-адаптер Wi-Fi или на DiskStation имеется встроенный Wi-Fi. Вы можете сделать DiskStation беспроводной точкой доступа или беспроводным маршрутизатором, в зависимости от того, как вы хотите использовать интернет-подключение.

Режим зоны доступа Описание
Беспроводная точка доступа
  • Соединение ЛВС и сети Wi-Fi.
  • ЛВС и беспроводные клиенты будут находиться в одной локальной сети.
  • Это применимо, если у вас есть маршрутизатор и вы хотите использовать DiskStation в качестве дополнительной точки доступа в существующей сети.
  • Позволяет осуществлять общий доступ к сетевым ресурсам, например к устройствам DLNA, принтерам, Synology Web Assistant или другим службам, основанным на сетевом вещании.
  • Сервер DHCP будет отключен на DiskStation, которые выступают в качестве беспроводных точек доступа. Чтобы заново активировать сервер DHCP для DiskStation, удалите аппаратные ключи Wi-Fi и перезапустите DiskStation.
  • Если модель имеет несколько портов LAN, только LAN 1 будет использоваться для подключения с беспроводной точкой доступа.
Беспроводной маршрутизатор
  • Позволяет настроить беспроводную сеть, где устройства находятся в личной подсети.

Примечание.

  1. Некоторые модели требуют подключения беспроводных аппаратных ключей для создания зон доступа. Для получения полного списка поддерживаемых беспроводных аппаратных ключей посетите веб-сайт www.synology.com.
  2. Для моделей, отличных от DS213air, параметр "Беспроводная зона доступа" в DSM 4.3 будет называться "Беспроводной маршрутизатор" в DSM 5.0 и последующих версиях.

Управление настройками зоны доступа

В данном разделе описано включение и управление параметрами зоны доступа DiskStation.

Использование DiskStation в качестве зоны доступа:

  1. Выберите Панель управления > Беспроводное подключение > Карта сети. Выберите Беспроводная точка доступа или Беспроводной маршрутизатор.
  2. Выберите вкладку Wi-Fi и нажмите Редактировать для управления следующими настройками:
    • Имя (SSID): Введите имя беспроводной сети.
    • Отобразить/Скрыть: Укажите, будет ли беспроводная сеть видна для других устройств.
    • Режим беспроводного подключения: Выберите стандарт беспроводного подключения. Каждый стандарт поддерживает различные полосы пропускания и частоты.
    • Ширина пропускания канала: Выберите ширину канала (доступно только при использовании беспроводного режима 802.11n).
    • Канал: Выберите частоту для беспроводной сети. Для улучшения производительности выберите частоту, отличную от частоты, используемой соседними сетями.
    • Мощность передачи: Выберите силу сигнала беспроводной сети.
    • Макс. кол-во подключений: Выберите количество клиентских устройств, которое можно подключить к сети. Максимальное число подключений составляет 256.
    • Включить WMM QoS: Wi-Fi Multimedia (WMM) QoS (Quality of Service) через Wi-Fi. Данный параметр позволяет беспроводной сети создавать приоритет для трафика и оптимизировать распределение ресурсов между приложениями.

Настройка расписания беспроводной сети:*

Включение расписания беспроводной сети позволяет управлять включением и доступностью беспроводной сети DiskStation.

  1. Выберите вкладку Wi-Fi и нажмите Редактировать.
  2. Установите флажок Включить расписание.
  3. Нажмите Настроить расписание.
  4. В окне расписания выберите время, когда беспроводная сеть будет активна.

Настройки безопасности

Пункт Панель управления > Беспроводное подключение > Wi-Fi содержит параметры для редактирования настроек безопасности беспроводной сети.

Настройка безопасности беспроводной сети:

  1. Нажмите Редактировать.
  2. Выберите один из следующих протоколов безопасности в раскрывающемся меню Настройки безопасности:
    • WEP: Базовый протокол безопасности, который требует ввода пароля для доступа к беспроводной сети, но обеспечивает минимальный уровень защиты.
    • Личный WPA/WPA2: Рекомендуется. Требует ввода пароля для доступа к беспроводной сети и обеспечивает более высокий уровень безопасности, чем WEP.
    • Корпоративный WPA/WPA2:* Протокол безопасности, разработанный для корпораций, требующий проверки подлинности пользователей с помощью внешнего сервера проверки подлинности.

Настройка Wi-Fi Protected Setup:*

Wi-Fi Protected Setup — это стандарт Wi-Fi Alliance, позволяющий упростить настройки безопасности беспроводного подключения для пользователей. Можно использовать кнопку Push или PIN-код для обеспечения защиты сети Wi-Fi. Нажмите WPS, чтобы включить Wi-Fi Protected Setup.

Гостевая сеть*

Помимо создания беспроводной хост-сети, ваше устройство DiskStation также поддерживает создание гостевой беспроводной сети. Настройки гостевой сети позволяют ограничить доступ пользователей, что удобно для корпораций или других предприятий, которым необходимо предоставить беспроводную сеть для посетителей, но закрыть от них доступ к хост-сети.

Активация гостевой сети:

  1. Выберите Панель управления > Беспроводное подключение > Wi-Fi.
  2. Нажмите Редактировать для управления следующими настройками на вкладке Гостевая сеть:
    • Имя (SSID): Введите имя беспроводной сети.
    • Отобразить/Скрыть: Укажите, будет ли беспроводная сеть видна для других устройств.
    • Макс. кол-во подключений: Выберите количество клиентских устройств, которое можно подключить к сети. Максимальное число подключений составляет 256.

Настройка безопасности гостевой сети:

Выберите один из следующих протоколов безопасности в раскрывающемся меню Настройки безопасности:

  • WEP: Базовый протокол безопасности, который требует ввода пароля для доступа к беспроводной сети, но обеспечивает минимальный уровень защиты.
  • Личный WPA/WPA2: Рекомендуется. Требует ввода пароля для доступа к беспроводной сети и обеспечивает более высокий уровень безопасности, чем WEP.

Настройка расписания беспроводной сети:

Включение расписания гостевой беспроводной сети позволяет управлять включением и доступностью беспроводной сети DiskStation.

  1. Установите флажок Включить расписание гостевой сети.
  2. Нажмите Настроить гостевое расписание.
  3. В окне расписания выберите время, когда гостевая беспроводная сеть будет активна.

Примечание.

Гостевая беспроводная сеть не может быть активна, когда отключена беспроводная хост-сеть.

Настройка фильтра MAC*

Настройки фильтра MAC (Панель управления > Беспроводное подключение) позволяют указать MAC-адреса, с которых разрешен доступ к беспроводной сети, и блокируют либо предоставляют доступ определенным устройствам. Данная функция позволяет управлять только сетевым доступом, а не другими настройками защиты.

Создание правил фильтра MAC:

  1. Введите описание и MAC-адрес устройства.
  2. Выберите политику доступа:
    • Запретить: Запрещает некоторым MAC-адресам доступ к беспроводной сети.
    • Разрешить: Разрешает определенным MAC-адресам доступ к беспроводной сети.

Родительский контроль*

Функция родительского контроля позволяет управлять доступом в Интернет каждого клиентского устройства, подключенного к локальной сети DiskStation. Например, если компьютеры или мобильные устройства ваших детей подключены к Интернету с помощью DiskStation, вы можете настроить параметры родительского контроля для ограничения доступа в Интернет с определенных устройств или в определенное время.

Включение родительского контроля:

  1. Выберите Панель управления > Беспроводное подключение > Настройки родительского контроля.
  2. Нажмите кнопку Добавить, чтобы добавить клиентское устройство.
    • Имя клиента: Выберите устройство, подключенное к DiskStation, либо введите его имя вручную (например, ноутбук Маши).
    • MAC-адрес: Введите MAC-адрес устройства, с которого вы хотите контролировать доступ в Интернет.
    • Настройка расписания: Укажите время, когда указанные устройства смогут иметь доступ к Интернету через DiskStation.

Список подключений*

Список подключений (Панель управления > Беспроводное подключение) отображает информацию об устройствах, подключенных к DiskStation. Отсюда устройства можно непосредственно добавить в список фильтров MAC.

Добавление устройств в список фильтров MAC:

  1. Установите флажок напротив устройства, которое вы хотите добавить в фильтр MAC.
  2. Нажмите Добавить в список фильтров MAC.
