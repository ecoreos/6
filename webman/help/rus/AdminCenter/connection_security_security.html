Безопасность

Следующие настройки безопасности можно изменить в разделе Панель управления > Безопасность > Безопасность для сеансов работы с DSM:

  • Таймер выхода (минут): Будет осуществлен автоматический выход пользователей из DSM, если они были неактивны в течение указанного здесь времени. Введите значение в диапазоне от 1 до 65535.
  • Улучшить совместимость браузера, пропустив проверку IP-адреса: При доступе к DiskStation через прокси-сервер HTTP и возникновении произвольных завершений сеанса вы можете включить этот параметр, чтобы пропустить проверку IP-адресов.
  • Повысить защиту от подделки межсайтовых запросов: Этот параметр повышает уровень защиты системы от атак, связанных с межсайтовым скриптингом. Этот параметр вступит в силу при следующем входе в DSM.
  • Повысить уровень безопасности с помощью HTTP-заголовка Content Security Policy (CSP): Этот параметр повышает уровень защиты системы от атак, связанных с межсайтовым скриптингом, загружая данные только из надежных источников и ограничивая выполнение встроенных скриптов.
  • Запретить на DSM встраивание iFrame: Включите это параметр, чтобы ограничить встраивание DSM в другие веб-страницы с помощью iFrame и таким образом предотвратить определенные виды атак с вредоносных веб-сайтов. Чтобы разрешить встраивание DSM с помощью iFrame для отдельных веб-сайтов, нажмите кнопку Разрешенные веб-сайты.
