SFTP

SFTP — это протокол передачи файла, встроенный в качестве дополнения в протокол Secure Shell (SSH). Для SFTP требуется только один номер порта TCP. Кроме того, для проверки подлинности пользователей без применения пароля можно использовать личные и общие ключи. Однако скорость передачи снижена и занимает меньше системных ресурсов из-за функций защиты.

Включение службы SFTP:

  1. Нажмите Включить службу SFTP.
  2. Нажмите Применить.

Номер порта

Настройте номер порта SFTP-службы.

Примечание.

Номер порта по умолчанию для службы SFTP - 22. Пользователь может задать диапазон от 1 до 65535, исключая следующие цифры:
20, 21, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002 и порты eMule по умолчанию: 4662(TCP), 4672(UDP)
