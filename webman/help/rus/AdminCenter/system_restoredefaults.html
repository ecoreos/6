Восстановить значения по умолчанию

Вы можете выполнить сброс до заводских настроек в меню Панель управления > Обновить и восстановить > Восстановить значения по умолчанию, чтобы восстановить DiskStation до исходных настроек производителя. Все пользовательские данные на жестком диске будут удалены и все параметры системы будут возвращены в значения по умолчанию.

Примечание.

  • Параметр Восстановить значения по умолчанию отображается, если DiskStation относится к кластеру high-availability. Чтобы восстановить значения по умолчанию для сервера, принадлежащего к кластеру high-availability, требуется сначала удалить кластер high-availability в разделе High Availability > Обзор > Управление.
