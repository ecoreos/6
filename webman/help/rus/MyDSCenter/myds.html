Учетная запись Synology

В разделе Учетная запись Synology можно с легкостью получить доступ и настроить параметры служб учетной записи Synology и eNews, связанные с DiskStation и пакетами.

  • Учетная запись: Просмотр и редактирование информации учетной записи Synology.
  • История покупок: Просмотр и поиск действий, связанных с покупками пакетов.
  • eNews: Выберите одну или несколько категорий eNews, которые вы хотите получать от Synology, и настройте содержимое подписки.
  • Выход: Завершите все службы учетной записи Synology, связанные с DiskStation.

Вход в учетную запись Synology:

  1. Перейдите в учетную запись Synology.
  2. Во всплывающем окне Вход в учетную запись Synology введите адрес электронной почты и пароль, которые использовались для регистрации учетной записи Synology.
    • Если у вас нет учетной записи Synology, щелкните Требуется учетная запись Synology? для регистрации.
    • Если вы забыли пароль, щелкните Забыли свой пароль?, чтобы получить новый пароль по электронной почте.
  3. Щелкните Вход для доступа к службам учетной записи Synology.

Просмотр и редактирование учетной записи Synology:

  1. Щелкните Учетная запись на левой панели для просмотра информации об учетной записи.
  2. Щелкните поле для изменения информации об учетной записи.
  3. Для сохранения настроек нажмите Применить.

Просмотр Журнала покупок DiskStation:

  1. Нажмите Журнал покупок на левой панели.
  2. Выберите один из способов поиска истории покупок пакетов на DiskStation.
    • Чтобы сделать это с помощью ключевых слов, выберите Поиск по ключевым словам.
      • Введите ключевой слово и выберите соответствующий фильтр.
    • Чтобы сделать это с помощью даты приобретения, выберите Поиск по дате.
      • Щелкните значки календаря, чтобы указать время приобретения DiskStation или пакетов.
  3. Нажмите кнопку Поиск.
  4. Появится запрашиваемая информация о приобретении пакетов (например, дата приобретения и цена).

Подписка на Synology eNews:

  1. Нажмите eNews на левой панели.
  2. В разделе Настройки eNews проверьте одну или несколько категорий eNews, которые вы хотите получать от Synology.
  3. Вы можете более точно настроить параметры eNews в раскрывающихся списках:
    • Выпуски новых продуктов и функций: Выберите язык подписки.
    • Информация из первых рук о местных рекламных акциях: Выберите страну, чтобы получать информацию о локальных событиях, проводимых Synology.
    • Важные обновления ОС DSM: Выберите одну или несколько версий DSM для получения соответствующих обновлений.
  4. Для сохранения настроек нажмите Применить.

Выход из учетной записи Synology:

  1. Нажмите кнопку Выход из системы в левом верхнем углу.
  2. После выхода все службы учетной записи Synology (например, DDNS и QuickConnect) и пакеты, связанные с учетной записью для входа, будут недоступны на DiskStation.

Примечание.

  1. После изменения адреса электронной почты, сообщение с подтверждением изменения будет отправлено на новый адрес.
  2. После выхода из системы пакеты останутся на DiskStation, но срок действия их лицензий истечет.
