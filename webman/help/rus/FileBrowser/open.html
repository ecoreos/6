Открыть файл в новом окне

Откройте файл DiskStation в новой вкладке или окне веб-обозревателя для его просмотра.

Открытие файла в новом окне:

  1. Выберите файл, который необходимо просмотреть.
  2. Выполните одно из следующих действий:
    • Выберите Открыть в новом окне в меню Действие.
    • Нажмите файл правой кнопкой мыши и выберите Открыть в новом окне.

Примечание: Настройки обозревателя определяют, файлы каких типов можно просматривать напрямую.
