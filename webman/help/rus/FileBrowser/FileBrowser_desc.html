File Station

File Station — это инструмент управления файлами DSM, позволяющий пользователям DSM получать доступ к папкам на DiskStation по Интернету с помощью веб-обозревателей. Открыв два или более окон File Stations, вы можете управлять всеми данными на DiskStation, перетаскивая их между различными File Station.
