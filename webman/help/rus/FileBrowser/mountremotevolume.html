Подключение удаленных папок

Можно подключать удаленные папки на удаленных серверах (таких как сетевые компьютеры или другие устройства Synology NAS) для навигации и управления подключенными папками при помощи File Station.

Подключение удаленной общей папки CIFS*

  1. Выберите Инструменты > Подключить удаленную папку > Общая папка CIFS.
  2. В поле Папка укажите путь к удаленной папке в формате \\удаленный_сервер\удаленная_папка (например, \\192.168.0.2\share или \\chad\cheeseburger\pictures).
  3. Если удаленный сервер требует разрешение на доступ к удаленной папке, введите свои учетные данные для удаленного сервера в полях Имя учетной записи и Пароль.
  4. Нажмите Обзор, чтобы выбрать или создать пустую целевую папку в вашем устройстве DiskStation для подключения удаленной папки.
  5. Установите флажок Подключать автоматически при запуске, если хотите, чтобы устройство DiskStation подключало эту удаленную папку при каждом запуске или перезагрузке системы.
  6. Нажмите Подключить, чтобы подключить удаленную папку к папке назначения. Теперь можно выполнять навигацию по удаленной папке и управлять ею с DiskStation.

Подключение удаленной общей папки NFS

  1. Выберите Инструменты > Подключить удаленную папку > Общая папка NFS.
  2. В поле Папка укажите путь к удаленной папке в формате \удаленный_сервер:/путь_к_удаленной_папке (например, 192.168.0.2:/volume1/share).
  3. Выберите предпочитаемую версию NFS и транспортный протокол.
  4. Нажмите Обзор, чтобы выбрать или создать пустую целевую папку в вашем устройстве DiskStation для подключения удаленной папки.
  5. Установите флажок Подключать автоматически при запуске, если хотите, чтобы устройство DiskStation подключало эту удаленную папку при каждом запуске или перезагрузке системы.
  6. Нажмите Подключить, чтобы подключить удаленную папку к папке назначения. Теперь можно выполнять навигацию по удаленной папке и управлять ею с DiskStation.

Примечание

NFSv4 поддерживает только протокол TCP.

Отключение удаленной папки:

  1. Выберите удаленную папку, которую необходимо отключить, и нажмите Инструменты > Отключить.
  2. Можно также щелкнуть правой кнопкой мыши удаленную папку и выбрать Отключить.

Примечания

  • Одновременно можно подключить максимум 100 удаленных папок.
  • Подключать удаленные папки можно только с удаленного сервера, который поддерживает протокол CIFS или NFS.
  • Удаленные папки могут быть подключены только к подпапкам в составе общих папок.
  • Подключение удаленных папок поддерживается только в режиме доступа высокого уровня. Чтобы использовать данную функцию, откройте пакет Docker в DSM хост-сервера, выберите Docker DSM, для которого необходимо включить зашифрованные общие папки, выберите Редактировать и установите флажок Использовать контейнер с правами доступа высокого уровня.

* Доступно только в некоторых моделях.
