Отправка файлов как вложений электронной почты

Вы можете отправлять файлы как вложения электронной почты в File Station, нажав файлы правой кнопкой мыши и выбрав Отправить как вложения электронной почты. Отобразится окно электронной почты с рядом удобных функций:

  • Кому: Если вы выполнили синхронизацию любых контактов для учетной записи электронной почты, отобразится список синхронизированных контактов, связанных с вашей учетной записью электронной почты.
  • От: Можно установить учетную запись электронной почты по умолчанию или другую добавленную запись электронной почты. Список контактов будет автоматически переключен на выбранную учетную запись электронной почты.
  • Вложения: Выбранные вложения будут отображаться в нижней части письма. Вы можете добавить дополнительные вложения с помощью кнопки вложений.

Примечание.

  • В качестве вложений можно добавить только файлы.
  • К одному письму можно прикрепить до 10 файлов.
  • Максимальный общий размер всех вложений для одного письма будет зависеть от вашей службы электронной почты. Если вы не знаете точное значение, рекомендуется не использовать вложения объемом более 25 Мб.
