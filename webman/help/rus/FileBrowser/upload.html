Загрузка файлов и папок

Используйте любой из следующих способов для выгрузки файлов или папок с компьютера на DiskStation.

Чтобы получить полный доступ к указанным выше функциям выгрузки, установите последнюю версию Java Runtime Environment (JRE) и Flash Player.

Метод 1:

  1. В верхней части левой панели выберите целевую папку на DiskStation.
  2. Нажмите Выгрузить, выберите Выгрузить - Пропустить или Выгрузить - Перезаписать и выберите файлы или папки на локальном компьютере.

Метод 2:

  1. Выберите Настройки, установите флажок Просмотр файлов с локального компьютера с помощью File Station и обновите веб-страницу для просмотра папок на локальном компьютере в нижней части левой панели.

    2. Примечание.

    Параметр Просмотр файлов с локального компьютера с помощью File Station вступает в силу только после установки и активации JRE на компьютере.

  2. В нижней части левой панели выберите папку на локальном компьютере. В основном разделе будет отображено содержимое выбранной папки.
  3. Выберите необходимые файлы или папки в основном разделе и нажмите Выгрузить.
  4. Выберите папку назначения на DiskStation в отобразившемся окне.
  5. В правом верхнем углу окна выберите Перезаписать, чтобы перезаписать файлы или папки с одинаковым именем в целевой папке. В противном случае выберите Пропустить.
  6. Нажмите OK для начала загрузки.

Метод 3:

  1. Выберите Настройки, установите флажок Просмотр файлов с локального компьютера с помощью File Station и обновите веб-страницу для просмотра папок на локальном компьютере в нижней части левой панели.
  2. В левой панели перетащите папку с компьютера (в нижней части) в любую папку на DiskStation (в верхней части).
  3. Во всплывающем меню выберите, следует ли перезаписать или пропустить файлы/папки с одинаковым именем в целевой папке.

Метод 4:

  1. В верхней части левой панели выберите целевую папку на DiskStation.
  2. Выберите файлы на рабочем столе компьютера или в проводнике (например, Проводник Windows или Mac Finder), затем перетащите их в File Station в веб-обозревателе.
  3. Во всплывающем меню выберите, следует ли перезаписать или пропустить файлы с одинаковым именем в целевой папке.

    4. Примечание.

    • Метод 4 поддерживается только в определенных браузерах: Chrome, Firefox, Safari 5.1 и более поздней версии. Однако этот метод не поддерживается для Mac, если установлен флажок Просмотр файлов с локального компьютера с помощью File Station.
    • Вы также можете перетащить папки или пустые файлы, но только в том случае, если установлен флажок Просмотр файлов с локального компьютера с помощью File Station для обозревателей, указанных выше. В противном случае, вы не сможете выгрузить папки и пустые файлы с помощью Метода 4.
    • Если вы используете Chrome для Linux и выполняете подключение к DSM через протокол HTTPS необходимо импортировать сертификат, выпущенный надежным органом, (Панель управления > Безопасность > Сертификат) для выполнения выгрузки с помощью Метода 4.

Примечание.

  1. В очередь выгрузки можно добавить до 1000 файлов.
  2. При использовании Google Chrome, Firefox, Safari 5.1 или более поздней версии, а также если установлен флажок Просмотр файлов с локального компьютера с помощью File Station, вы можете выгружать файлы размером более 4 ГБ. В противном случае вы можете одновременно загружать не более 2 ГБ.
  3. В незашифрованной папке общего доступа имя файла/папки не должно превышать 255 символов (до 80 символов для языков, в которых не используется латинский алфавит); в зашифрованной папке общего доступа имя не должно превышать 143 символов (до 47 символов для языков, в которых не используется латинский алфавит).
  4. Имя файла/папки не должно содержать следующие символы: \ / : * ? " > |.
  5. Имя файла/папки не может начинаться с "._".
  6. Вы можете выгружать файлы размером 0 КБ, только если установлено JRE.

Устранение неполадок

Если вы не можете выполнить просмотр локальных элементов на Mac с помощью Safari, выполните следующее:

  1. На Mac откройте Safari и выберите Предпочтения в меню Safari.
  2. Нажмите Управление настройками веб-сайта на вкладке Безопасность.
  3. В пункте Java выберите "веб-сайт" DSM.
  4. Выберите Выполнить в небезопасном режиме.
