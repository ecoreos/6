Загрузка файлов

Выполните указанные ниже действия для загрузки файлов или папок с устройства DiskStation на компьютер.

Загрузка файлов на локальный компьютер:

  1. Выберите файлы или папки для загрузки. (Чтобы выбрать несколько элементов, нажмите и удерживайте клавишу Shift или Ctrl.)
  2. Нажмите Действие > Загрузить и выберите нужный режим загрузки для вашего веб-браузера:
    • Google Chrome: Нажмите Загрузить для загрузки отдельных файлов (без сжатия). Нажмите Загрузить как .zip для загрузки нескольких файлов в файл .zip (при помощи кодовой страницы File Station, указанной в разделе Параметры > Общие).
    • Другие браузеры: Нажмите Загрузить для загрузки одного или нескольких файлов в файл .zip (при помощи кодовой страницы File Station, указанной в разделе Параметры > Общие).
    • Добавить в очередь - Перезаписать или Добавить в очередь - Пропустить: В зависимости от того, хотите ли вы перезаписать файлы или папки с одинаковым именем в месте назначения, выберите один из параметров для добавления файлов в очередь загрузки (которая отображается при нажатии стрелки вниз в правом верхнем углу).
    Примечание.
  1. В среде ОС Windows символы \ * ? " < > | : , содержащиеся в имени файла, будут заменены на - после загрузки.
  2. Чтобы отключить автоматическую загрузку нескольких файлов в Chrome, откройте страницу настроек и измените настройки контента в разделе "Личные данные".
  3. Chrome в режиме инкогнито не поддерживает загрузку отдельных файлов без сжатия.

Получение ссылки на файл или папку для загрузки

Вы можете получить ссылку для загрузки файла или папки и поделиться ссылкой с другими пользователями.

Получение ссылки для загрузки файла или папки:

  1. Выберите файл или папку, ссылку для загрузки которой необходимо получить.
  2. Выполните одно из следующих действий, чтобы открыть окно "Свойства":
    • Выберите Свойства в меню Действие.
    • Правой кнопкой мыши нажмите файл или папку, а затем выберите Свойства.
  3. Нажмите правой кнопкой мыши ссылку для загрузки и выберите функцию в меню ярлыков для копирования ссылки. Название функции может изменяться в зависимости от используемого веб-браузера.
  4. Отправьте ссылку другим пользователям по электронной почте или с помощью программ обмена мгновенными сообщениями. Они смогут загрузить файл или папку по указанной ссылке, введя собственные имя пользователя и пароль.

Примечание. Чтобы загрузить файл или папку с помощью ссылки для загрузки, требуется действующие имя пользователя и пароль.
