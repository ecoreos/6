Создание запросов файлов

Запросы файлов — это приглашения на передачу файлов, отправленные пользователям устройств, отличных от DSM. С помощью ссылок на запросы файлов такие пользователи могут передавать несколько файлов напрямую на File Station устройства DiskStation без соответствующих учетных записей и прав доступа на передачу файлов.

Создание ссылок на запросы файлов

  1. Выберите File Station, затем выберите папку, в которую будет выполняться передача.
  2. Нажмите Действие или щелкните правой кнопкой мыши папку назначения и выберите Создать запрос файла.
  3. Во всплывающем окне можно найти или изменить следующую информацию.
    • Путь к файлу. Путь к каталогу назначения.
    • Ссылка. Ссылка на запрос файла, которой следует поделиться с целевыми пользователями.
    • Ваше имя. Введите имя для идентификации запрашивающего лица. Значение по умолчанию — ваше имя пользователя.
    • Сообщение. Настройка сообщения для целевых пользователей.
    • Включить защиту пароля. Выберите этот параметр, чтобы разрешить передачу файлов по созданной ссылке только пользователям с паролем. Задайте пароль в поле ниже.
    • Срок действия. Нажмите эту кнопку, чтобы настроить срок действия ссылки на запрос файла.
      • Настройка времени остановки. Выберите этот параметр, чтобы задать дату и время, когда ссылка перестанет действовать для передачи файла.
      • Настройка времени запуска. Выберите этот параметр, чтобы задать дату и время, когда ссылка начнет действовать для передачи файла.
      • Количество разрешенных сеансов доступа. Выберите этот параметр, чтобы задать, сколько раз можно получить доступ по ссылке.
  4. Нажмите кнопку Сохранить, чтобы завершить настройку.

Обмен ссылками на запросы файлов

Чтобы найти URL запроса файла, выберите File Station > Инструменты > Диспетчер ссылок общего доступа и дважды щелкните нужную ссылку. Поделиться ссылкой можно следующими способами.

  • Просто копируйте URL целевым пользователям.
  • Отправьте ссылку по электронной почте по умолчанию (электронная почта вашего устройства DSM) или по электронной почте (приложение электронной почты стороннего поставщика). Если вы впервые используете функцию, следуйте указаниям мастера, чтобы настроить учетную запись электронной почты.

Передача файлов

Если у вас запросили передачу файлов по ссылке, выполните следующие действия.

  1. Откройте URL на локальном компьютере или мобильном устройстве (с ОС Android или iOS). Введите пароль при необходимости.
  2. На странице передачи введите имя, чтобы идентифицировать себя.
  3. Перетащите файлы для передачи или нажмите Добавить файлы и выберите файлы.
  4. Нажмите Передать, чтобы завершить запрос.

Примечание.

  • Невозможно передать папки по ссылке на запрос файла.
  • Невозможно задать в качестве папки назначения папки в составе сетевой папки общего доступа.
  • Функция передачи зависит от веб-браузера.
    • Internet Explorer 8 и Internet Explorer 9 не поддерживают передачу файлов перетаскиванием.
    • Chrome на устройствах с ОС iOS поддерживает только передачу изображений.
    • Safari на устройствах с ОС iOS поддерживает только передачу изображений и видео.
    • На веб-браузеры на устройствах с ОС Android не распространяются никакие ограничения по типу файла.

Поиск переданных файлов

Файлы, переданные одним пользователем, будут отображаться в одной папке в каталоге назначения. По умолчанию имя папки Имя_Передавшего (Имя_Запросившего).

Например, если файлы были переданы Петровым Сергееву, они будут находиться в папке Петров (Сергеев).

Управление ссылками на запросы файлов

Изменение ссылок на запросы файлов

После создания ссылки на запрос файла можно изменить ее настройки, выбрав File Station > Инструменты > Диспетчер ссылок общего доступа.

  1. Дважды щелкните нужную ссылку или выберите ссылку и нажмите кнопку Редактировать..
  2. Во всплывающем окне измените нужные настройки.
  3. Нажмите кнопку Сохранить, чтобы применить новые настройки.

Удаление ссылок на запросы файлов

  1. Выберите File Station > Инструменты > Диспетчер ссылок общего доступа.
  2. Выберите нужные ссылки и нажмите Удалить. URL-адреса этих запросов файлов перестанут действовать.

Назначение прав доступа на отправку запросов

Пользователь admin или из группы administrators может назначать права доступа пользователям, чтобы они могли создавать ссылки на запросы файлов.

  1. Выберите File Station > Настройки > Ссылки общего доступа > Ссылки на запросы файлов.
  2. Выберите тип пользователей, которым следует разрешить отправку запросов.
  3. Нажмите кнопку OK, чтобы сохранить изменения.
