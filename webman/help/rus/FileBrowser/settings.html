Управление параметрами File Station

Чтобы управлять параметрами File Station, откройте File Station и нажмите кнопку Настройки.

Общие

Здесь можно включить журналы File Station для записи действий всех пользователей. Действия пользователя могут быть следующими: Создать папку, Загрузить, Скачать, Удалить, Переименовать, Переместить, Копировать и Свойство установлено.

Включение журнала File Station

  1. Чтобы вести запись действий всех пользователей, установите флажок Включить журнал File Station.
  2. Нажмите OK.

Просмотр журнала File Station

  1. Нажмите Смотреть журналы событий, чтобы открыть Центр журналов.
  2. Выберите вкладку Передача файла.

Порядок автоматической перезаписи файлов с помощью загрузки перетаскиванием или с помощью клавиш быстрого доступа.

Установите флажок Всегда перезаписывать файлы при перетаскивании в браузер или вставке с помощью сочетаний клавиш, чтобы автоматически перезаписывать файлы с одинаковым именем при перетаскивании с рабочего стола компьютера в браузер или при вставке в File Station с помощью сочетаний клавиш.

Активация смарт-перетаскивания

При перетаскивании файлов в File Station эта функция автоматически определяет, является ли передача действием копирования или перемещения.

  1. НажмитеВключить смарт-перетаскивание.
  2. Если возможны случаи, при которых другой пользователь может одновременно передавать файлы с тем же именем и в то же местоположение, выберите одну из опций ниже, чтобы всегда выполнять это действие.
    • Перезаписать: ваши переданные файлы перезапишут файлы с теми же именами.
    • Пропустить: ваши переданные файлы будут пропущены и не перезапишут файлы с теми же именами.
  3. Нажмите OK.

Примечания

Чтобы изменить действие копирования или перемещения, определенное функцией смарт-перетаскиванием, воспользуйтесь необходимыми сочетаниями клавиш ниже.

  • Чтобы изменить действие на копирование, во время перетаскивания нажмите Ctrl.
  • Чтобы изменить действие на перемещение, во время перетаскивания нажмите Shift.

Просмотр и управление файлами с локального компьютера с помощью File Station

  1. Установите флажок Просмотр файлов с локального компьютера с помощью File Station.
  2. Нажмите OK.

Предоставление списка пользователей DSM пользователям без прав администратора

Данная опция предоставляет пользователям без прав администратора доступ к полному списку пользователей DSM. С его помощью пользователи могут выполнять поиск файлов по пользователям или группам DSM, а также отправлять ссылки на файлы другим пользователям DSM.

  1. Нажмите Предоставить список пользователей DSM пользователям без прав администратора.
  2. Нажмите OK.

Применение разрешений UNIX по умолчанию

Нажмите Дополнительные настройки и включите параметр, позволяющий применять разрешения UNIX по умолчанию при загрузке или создании папок и файлов. Те же разрешения можно применить с помощью команды UNIX umask. Если этот параметр включен, для файлов используется разрешение UNIX 644, а для папок — разрешение 755. Если этот параметр отключен, для файлов используется разрешение UNIX 666, а для папок — разрешение 777. По умолчанию параметр unmask имеет значение 022.

Примечания

  • Для всех общих папок с включенными разрешениями Windows ACL (кроме папки "photo" и папок на внешних дисках) необходимо выполнить команду chmod на клиенте Linux или FTP. В результате типы разрешений для папок и файлов изменятся с Windows ACL на UNIX.
  • Включение данного параметра может привести к несогласованности разрешений между различными протоколами. Во избежание несогласованности рекомендуется не включать данный параметр.

Подключения

По умолчанию все пользователи могут выполнять подключение удаленных папок или виртуальных дисков. Здесь вы можете ограничить доступ к удаленной папке или управлению виртуальным диском, разрешив его только администраторам.

Примечания

Не принадлежащие к группе администраторов пользователи могут подключать удаленные папки только в своей папке home.

Порядок разрешения пользователям выполнять подключение удаленных папок или виртуальных дисков.

  1. Выберите Все пользователи или Администраторы в разделе Удаленная папка или Виртуальный диск.
  2. Нажмите OK.

Предоставление различным пользователям разрешений на подключение серверов и облачных служб

  1. Выберите Все пользователи, Администраторы или Конкретный пользователь.
  2. При выборе пункта Конкретный пользователь откроется окно, в котором можно настроить разрешения на подключение для разных пользователей. Чтобы разрешить доступ для конкретного пользователя, установите флажок в столбце Включить. Чтобы отключить доступ для конкретного пользователя, снимите флажок в столбце Включить. После завершения настройки нажмите OK, чтобы закрыть окно.

    3. Примечания

    • Пользователи из группы administrators могут использовать эту функцию, не могут вносить изменения. Пользователи из группы guest не могут использовать эту функцию и не могут вносить изменения. Здесь можно настроить разрешения для всех других пользователей.
  3. Нажмите OK.

Ссылки общего доступа

По умолчанию все пользователи могут обмениваться ссылками на файлы. Здесь вы можете ограничить доступ к управлению ссылками общего доступа только для администраторов.

Порядок разрешения пользователям обмениваться ссылками на файлы.

  1. Выберите Все пользователи, Администраторы или Конкретные пользователи.
  2. Нажмите OK.

Открытие файлов HTML как обычного текста

  1. Установите флажок Всегда открывать файлы HTML с помощью обычного текстового формата. Если этот параметр включен, файлы HTML (т. е. файлы HTM, HTML, JSP, XHTML) отображаются в формате обычного текста, когда пользователи открывают их по ссылке для обмена файлами. Это позволяет предотвратить взлом сеансов входа пользователя с помощью сценариев, встроенных в файлы HTML.
  2. Нажмите OK.

Ограничение скорости

Можно включить ограничение скорости, чтобы управлять максимальной используемой шириной пропускания сети для пользователей и групп, передающих данные с помощью File Station.

Включение ограничения скорости

  1. Установите флажок Включить ограничение скорости или Включить ограничение скорости согласно расписанию. В последнем случае ограничение скорости будет применяться только в определенное время. Эта настройка применяется на всех серверах и переопределяет настройки отдельных пользователей и групп. Если выбран параметр Нет ограничения скорости, ограничения скорости для пользователей и групп отсутствуют.
  2. Нажмите кнопку Настройки ограничения скорости, чтобы изменить настройки для отдельных пользователей или групп. Для каждого пользователя или группы можно выбрать один из следующих параметров.
    • Применить настройки группы: настройки ограничения скорости будут совпадать с настройками группы, к которой принадлежит пользователь.
    • Нет ограничения скорости: ограничение скорости не действует, когда участники группы передают файлы с помощью этой службы.
    • Постоянная: к отдельным пользователям или участникам группы будет применяться фиксированное постоянное ограничение скорости. В полях слева можно указать ограничения скорости при загрузке и скачивании файлов.
    • Переменная: к отдельным пользователям или участникам группы могут применяться два различных ограничения скорости, которые включаются по расписанию. Нажмите кнопку Настроить, чтобы изменить настройки ограничения скорости и настроить расписание.
  3. Нажмите Применить.

Примечания

  • Дополнительные сведения о динамике ограничений скорости группы см. здесь.
  • При установке ограничения скорости для пользователя в разделе Результат отображается фактическое ограничение скорости пользователя с учетом индивидуальных и групповых настроек. Например, если для пользователя "Achilles" установлено индивидуальное ограничение скорости скачивания 100 КБ/с, но этот пользователь является участником группы с ограничением скорости скачивания 500 КБ/с, в разделе Результаты будет отображаться значение "0/500."

Безопасность

Здесь можно найти следующие настройки безопасности.

Открытие файлов HTML как обычного текста

  1. Установите флажок Всегда открывать файлы HTML с помощью обычного текстового формата. Если этот параметр включен, файлы HTML (т. е. файлы HTM, HTML, JSP, XHTML) отображаются в формате обычного текста при открытии в File Station. Это позволяет предотвратить взлом сеансов входа пользователя с помощью сценариев, встроенных в файлы HTML.

    2. Примечания

    • Это не глобальный параметр. Каждый пользователь может выбрать, включать ли этот параметр.
  2. Нажмите OK.
