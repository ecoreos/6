Подключение виртуальных дисков

Можно подключить виртуальные диски для доступа к содержимому образов дисков при помощи File Station.

Подключение виртуального диска:

  1. Выделите образ диска и выберите Инструменты > Подключить виртуальный диск.
  2. Нажмите Обзор, чтобы выбрать или создать пустую целевую папку в вашем устройстве DiskStation для подключения виртуального диска.
  3. Установите флажок Подключать автоматически при запуске, если хотите, чтобы устройство DiskStation подключало этот виртуальный диск при каждом запуске или перезагрузке системы. Чтобы продолжить, нажмите Подключить. Теперь можно открыть целевую папку, в которой подключен виртуальный диск, и просмотреть содержимое образа диска.

Примечания

Виртуальные диски и файлы .iso можно подключать только к подпапкам в составе общих папок.

Отключение виртуального диска:

  1. Правой кнопкой нажмите целевую папку, в которой подключен виртуальный диск, и выберите Отключить.
    Примечание.
  1. Поддерживаемые форматы образов дисков: формат файла ".iso", использующий файловую систему ISO 9660 и UDF.
  2. Одновременно можно подключить максимум 16 виртуальных дисков.
  3. Подключенные виртуальные диски поддерживаются только в режиме доступа высокого уровня. Чтобы использовать данную функцию, откройте пакет Docker в DSM хост-сервера, выберите Docker DSM, для которого необходимо включить зашифрованные общие папки, выберите Редактировать и установите флажок Использовать контейнер с правами доступа высокого уровня.
