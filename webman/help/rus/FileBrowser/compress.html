Сжатие файлов и папок

  1. Выберите файлы или папки в основной области DiskStation.
  2. Выберите Сжать в архив (имя файла).zip в раскрывающемся меню Действие для начала сжатия.
  3. Если архив не отображается после завершения сжатия, выберите Обновить в раскрывающемся меню Действие, чтобы отобразить файл.
  4. Чтобы переименовать или зашифровать архив, либо выполнить архивацию с помощью различных уровней и режимов сжатия, выберите Добавить в архив...
    Примечание:
  1. С помощью File Station невозможно сжать файлы или папки на локальном компьютере.
  2. Поддерживаемые форматы архива: .zip, .7z.
  3. Ели выбрать .zip в качестве Формата архива в пункте Добавить в архив..., можно выбрать кодовую страницу для файла .zip.
