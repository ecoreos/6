Учетная запись электронной почты

Вы можете добавлять учетные записи электронной почты в пункте Личные > Учетная запись электронной почты и отправлять файлы, расположенные на File Station, как вложения с электронных адресов, указанных здесь.

К поддерживаемым провайдерам электронной почты относятся Gmail, Outlook, Yahoo!, AOL и пользовательский SMTP-сервер.

Добавление учетной записи электронной почты:

  1. Нажмите Добавить и выберите провайдера службы электронной почты, учетную запись которой вы хотите настроить.
  2. Введите адрес электронной почты, пароль и имя отправителя.
  3. Установите флажок Использовать данную учетную запись электронной почты как запись по умолчанию, чтобы File Station использовал данную учетную запись электронной почты по умолчанию при настройке нескольких учетных записей.
  4. Нажмите Синхронизировать контакты, чтобы синхронизировать контакты электронной почты с DiskStation, если ваш провайдер службы электронной почты (Gmail, Outlook и Yahoo!) поддерживает данную функцию. Обратите внимание, что контакты будут синхронизированы с учетной записи электронной почты, указанной в окне аутентификации, а не с указанной в предыдущем шаге. Поэтому, если в окне аутентификации отображается другая или непросматриваемая учетная запись, вручную переключитесь на необходимую учетную запись электронной почты или на запись, введенную в предыдущем шаге.
  5. Нажмите Применить, чтобы сохранить настройки.

Редактирование учетной записи электронной почты:

Выберите необходимую учетную запись электронной почты из списка и нажмите Редактировать.

Удаление учетной записи электронной почты:

Выберите необходимую учетную запись электронной почты из списка и нажмите Удалить.

Примечание.

  • Учетную запись по умолчанию невозможно удалить, если нет других учетных записей. Если вы хотите удалить учетную запись электронной почты, установленную по умолчанию, перед продолжением назначьте другую учетную запись по умолчанию.

Изменение учетной записи по умолчанию:

Выберите из списка учетную запись электронной почты, которую вы хотите установить по умолчанию, затем нажмите Установить по умолчанию.

Отправка файлов как вложений:

  1. Нажмите правой кнопкой файлы в File Station и выберите Отправить как вложение электронной почты.
  2. Если вы добавили несколько учетных записей электронной почты, вы можете выбрать необходимый электронный адрес отправителя в раскрывающемся списке, чтобы отправить файлы определенному получателю.
