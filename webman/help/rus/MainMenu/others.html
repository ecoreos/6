Другие

Настроить другие параметры личной учетной записи можно в Личные > Другие.

Настройка других личных параметров:

  1. Выберите один из следующих параметров:
    • Не показывать окно подтверждения при выходе из DSM: диалоговое окно подтверждения не отображается при закрытии браузера.
    • Возвращать DSM в состояние при последнем выходе: запоминает открытые приложения и пакеты и восстанавливает их при следующем входе или обновлении окна браузера.
    • Включить мгновенный просмотр на панели задач: предварительный просмотр открытого приложения при наведении курсора на значок приложения на панели задач. Если браузер Internet Explorer не последней версии, он может поддерживать эту функцию не в полной мере.
    • Не запускать справку DSM автоматически при каждом входе в систему: страница помощи по DSM не будет автоматически открываться при входе в DSM.
    • Включение клавиш быстрого доступа на настольной клавиатуре: можно использовать клавиши быстрого доступа для выполнения операций рабочего стола и DSM различных типов (например, переключение между активными окнами, выбор/отмена выбора опций и т. д.). Для просмотра клавиш быстрого доступа нажмите Shift + ?.
    • Отключить специальные возможности (аудиоподсказки больше не будут воспроизводиться): отключение функции специальных возможностей на DSM. Аудиоподсказки больше не будут воспроизводиться при включенном чтении с экрана.
    • Включить уведомления рабочего стола браузера: на рабочем столе DSM отображаются уведомления от вашего веб-браузера. Уведомления браузера на рабочем столе поддерживаются только Google Chrome и Safari. Страница настроек уведомлений может выглядеть по-разному в зависимости от версии вашего веб-браузера. Перейдите на страницу настроек браузера, на которой можно управлять настройками уведомлений.
    • Включить кнопку Pilot View на панели задач: отображается кнопка Pilot View, позволяющая просматривать все открытые приложения в режиме предварительного просмотра.
  2. Нажмите OK.
