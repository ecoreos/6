Центр журналов

Центр журналов — это приложение для централизованного управления журналами. Оно предлагает комплексные инструменты для простого и эффективного выполнения следующих задач. Примечание. Для получения доступа к расширенным функциям необходимо перейти в Центр пакетов и установить пакет Центр журналов.

Базовые функции

  • Поиск и фильтрование локальных журналов
  • Мониторинг объема журналов по времени
  • Отправка уведомлений администратору при наступлении определенных событий

Расширенные функции

  • Отправка и получение журналов с других сетевых устройств по стандартному протоколу syslog
  • Мониторинг объема журналов по устройству
  • Задание места назначения архивирования журналов и создание правил для автоматического запуска архивирования
  • Поиск и фильтрование журналов, полученных с других сетевых устройств
