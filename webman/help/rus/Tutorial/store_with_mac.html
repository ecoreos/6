Сохранение файлов на Synology NAS с Mac-компьютера в локальной сети

Обзор

Устройство Synology NAS разработано для ускорения и упрощения хранения и общего доступа к файлам в локальной сети. Оно обеспечивает прямой доступ к общим папкам и файлам на Synology NAS без выполнения каждый раз входа в DSM. Например, вы сможете сохранять файлы на Synology NAS с помощью Finder точно так же, как и на другие сетевые устройства.

Доступ к общим папкам из Mac OS

  1. Откройте Finder на Mac-компьютере.
  2. Найдите строку меню наверху экрана.
  3. Нажмите Перейти, затем выберите Подключиться к серверу.
  4. Введите IP-адрес для Synology NAS с префиксом "afp://". Например, "afp://192.168.59.121". Нажмите кнопку Подключиться для продолжения.
    5. Избранные серверы: Нажатие кнопки "плюс" (+) назначает IP-адреса как избранные серверы для будущего использования. Кроме того, можете нажать кнопку часов, чтобы просмотреть список серверов, к которым вы недавно подключались.
  5. Введите имя пользователя и пароль для входа в DSM. Учетная запись пользователя должна иметь правильные привилегии для чтения/записи для общей папки, к которой требуется доступ. Нажмите кнопку Подключиться для продолжения.
  6. Наконец, если учетные данные пользователя правильны, вы будете теперь видеть Synology NAS в разделе Общедоступные окна Finder. Дважды щелкните общую папку, чтобы просмотреть ее содержимое.
