Доступ к NAS-устройству Synology по Интернету и обмен файлами

Обзор

С помощью службы QuickConnect можно с легкостью подключаться к Synology NAS по Интернету без необходимости настройки правил переадресации портов или выполнения сложных настроек сети. Служба QuickConnect позволяет устанавливать подключение с использованием простого настраиваемого адреса, например, Quickconnect.to/example.

1. Настройка QuickConnect

  1. Выберите Панель управления > QuickConnect.
  2. Установите флажок Включить QuickConnect.
  3. Если у вас нет учетной записи Synology, нажмите Вход или регистрация учетной записи Synology. Введите требуемую информацию и нажмите OK. Если у вас уже есть учетная запись Synology, укажите учетные данные.
  4. Создайте собственный идентификатор QuickConnect ID в поле QuickConnect ID. Нажмите Применить.
  5. Появится информация о ссылке на ваш QuickConnect. Используйте эти ссылки для доступа к Synology NAS. В данном примере адресом DSM является http://Quickconnect.to/YourID.
  6. Если ссылка на QuickConnect DSM не отображается, нажмите Дополнительно и проверьте, что служба DSM включена.
  7. Вы можете теперь использовать ссылку на QuickConnect DSM для доступа к Synology NAS.

2. Обмен файлами на устройстве NAS с помощью службы QuickConnect

Если QuickConnect включена, вы можете с легкостью обмениваться с другими пользователями файлами, которые хранятся на устройстве Synology NAS, и не беспокоиться о переадресации порта.

  1. Сначала необходимо включить соединения HTTPS для безопасного обмена файлами. Выберите Панель управления > Сеть.
  2. Установите флажок Включить соединения HTTPS и нажмите Применить.
  3. Затем включите функцию обмена файлами для QuickConnect. Выберите Панель управления > QuickConnect.
  4. Нажмите Дополнительно и установите флажок Обмен файлами.

    5. Примечание.

    При настройке дополнительных параметров QuickConnect рекомендуется войти в DSM, используя локальный IP-адрес или доменное имя устройства NAS Synology.

  5. Вы готовы начать обмен файлами с помощью QuickConnect. Откройте File Station и щелкните правой кнопкой мыши на файле или папке, чтобы предоставить общий доступ к ним. Затем в контекстном меню выберите Обмен ссылками на файлы.
  6. Ссылка для обмена файлами создана. Просто поделитесь этой ссылкой с друзьями и членами своей семьи для загрузки папки/файла с общим доступом.
