Наслаждайтесь просмотром мультимедийного содержимого, сохраненного на Synology NAS, на большом экране

Обзор

Synology NAS может служить в качестве сервера мультимедиа в локальной сети, позволяя домашним устройствам, например телевизорам/стереосистемам и Sony PlayStation3/Xbox 360 воспроизводить мультимедийное содержимое. Эти устройства являются совместимыми с DLNA/UPnP цифровыми мультимедийными адаптерами (DMA). Для потоковой передачи мультимедийного содержимого с Synology NAS для воспроизведения на устройствах DMA необходимо настроить параметры воспроизведения мультимедийных данных. Например, параметры управления просмотром медиафайлов и расширения совместимости файлов. Эта функция имеется в пакете Сервер мультимедиа.

1. Установка сервера мультимедиа

Чтобы установить сервер мультимедиа, перейдите на страницу Центр пакетов, выберите Сервер мультимедиа и затем нажмите Установить.

2. Выгрузка мультимедийных файлов на Synology NAS

Применяемые по умолчанию общие папки photo, music и video автоматически создаются и индексируются после установки сервера мультимедиа на NAS. Вам остается просто выгрузить медиафайлы — изображения, песни или видеозаписи — в соответствующие папки, после чего эти файлы будут доступны с DMA-устройств.

Примечание. Созданные с удаленных серверов папки общего доступа невозможно добавить как проиндексированные. На Synology NAS можно добавить до 100 индексированных папок.

3. Настройка параметров просмотра для мультимедийного проигрывателя

Вы можете указать способ просмотра на DMA-устройствах медиафайлов, хранимых на Synology NAS.

  1. Выберите Главное меню > Сервер мультимедиа.
  2. В левой панели выберите Настройки просмотра.
  3. Для получения дополнительных сведений нажмите кнопку Справка в правом верхнем углу окна Сервер мультимедиа.

4. Расширение совместимости с DMA

Если ваш телевизор с поддержкой DLNA/UPnP или DMA-устройство плохо воспроизводит мультимедийное содержимое, сохраненное на NAS, можно попробовать настроить параметры Совместимость с DMA сервера мультимедиа.

Примечание.
  1. Чтобы проверить, правильно ли установлено DMA-устройство и найдено ли оно NAS в локальной сети, нажмите Список устройств. Появится список DMA-устройств с информацией о каждом устройстве.
  2. Флажок Ограничить доступ к серверу мультимедиа для новых обнаруженных устройств UPnP/DLNAне должен быть установлен. В противном случае сервер мультимедиа для DMA-устройств не будет доступен.

Для получения дополнительных сведений о технической поддержке мультимедийных файлов других форматов нажмите кнопку справки в правом верхнем углу окна сервера мультимедиа.

5. Просмотр медиафайлов, сохраненных на NAS, с помощью телевизора с поддержкой DLNA

Устройства с поддержкой DLNA/UPnP и DMA можно легко подключить к домашней сети. После установления подключений устройств DLNA и Synology NAS к домашней сети вы сможете просматривать выбранные папки и слушать музыку или смотреть видеозаписи с NAS на большом экране.

Примечание. Список поддерживаемых телевизоров с поддержкой DLNA/UPnP и устройств DMA, проверенных компанией Synology, см. здесь.

Процесс настройки зависит от DLNA-совместимых устройств DMA. Для получения дополнительных сведений о доступе к медиафайлам, сохраненным на NAS, через DLNA, см. в инструкциях клиентских устройств.
