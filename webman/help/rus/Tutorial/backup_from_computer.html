Резервирование данных на ПК/Mac с помощью Cloud Station

Обзор

Cloud Station Server – современная кроссплатформенная программа синхронизации, которая может использоваться как гибкий инструмент резервирования пользовательских данных. Решение Cloud Station Backup мгновенно выполняет резервное копирование в режиме реального времени и требуют меньшей ширины пропускания. Оно обеспечивает лучшую защиту и сохраняет до 32 версий одного файла.

Настройка Cloud Station Server на Synology NAS

  1. Откройте Центр пакетов, найдите Cloud Station Server и нажмите Установить.
  2. Запустите Cloud Station Server.
  3. Выберите Обзор и нажмите Загрузить сейчас в разделе Cloud Station Backup. (Вы также можете загрузить Cloud Station Backup из Центра загрузок Synology.)
  4. Запустите на компьютере средство установки и следуйте инструкциям по установке Cloud Station Backup.
  5. После установки запустите Cloud Station Backup на компьютере. Нажмите Запустить сейчас.
  6. Введите адрес Synology NAS, на котором установлено приложение Cloud Station Server (или QuickConnect ID), имя пользователя и пароль. (Можно также нажать на значок поиска справа, и Cloud Station Backup автоматически начнет искать другие Synology NAS в вашей локальной сети.) Нажмите Далее.
  7. Выберите источник резервирования и отмените выбор папок, которые не нужно синхронизировать.
  8. Выберите папку на Synology NAS, нажав Выбрать. Cloud Station Backup создаст папку резервирования в выбранном месте назначения с именем компьютера.
  9. Нажмите Правила синхронизации, чтобы установить фильтры и правила синхронизации.
  10. Нажмите Далее для предварительного просмотра настроек резервирования и выберите Готово.
  11. В главном приложении можно посмотреть ход выполнения резервирования.

Загрузка и восстановление предыдущей версии синхронизированного файла из Cloud Station Server

  1. Выберите Cloud Station Server> Журнал версий и найдите файл или папку для восстановления.
  2. Выберите файл, который нужно извлечь, и нажмите Действие. Выберите Обзор предыдущих версий. Если файл был случайно удален с компьютера, его можно восстановить, нажав Восстановить.
  3. Выберите версию файла, которую вы хотите загрузить, и нажмите Загрузить. Если вы хотите восстановить выбранную версию файла, нажмите Восстановить, чтобы перезаписать текущую версию файла.
