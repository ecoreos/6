Защитите свое устройство Synology NAS

Обзор

DSM содержит новую функцию, Помощник по безопасности, которая помогает поддерживать безопасность Synology NAS. Помощник по безопасности проверяет общую конфигурацию Synology NAS и предоставляет детальные отчеты о состоянии безопасности устройства. Кроме того, Помощник по безопасности показывает, как устранить выявленные риски безопасности. Откройте Помощник по безопасности и узнайте, как:

  • выполнить проверку безопасности;
  • просмотреть отчеты по правилам безопасности;
  • управлять рисками безопасности.

1. Выполнение проверки безопасности

  1. При первом открытии Помощника по безопасности программа спрашивает, для каких целей вы используете Synology NAS. В зависимости от ситуации выберите Дом или Работа и нажмите кнопку OK. Вы всегда сможете изменить этот параметр позже на странице Дополнительно.
  2. Помощник по безопасности начнет проверять конфигурацию Synology NAS. Общее состояние безопасности, ход процедуры проверки и сводку результатов можно просмотреть на странице Обзор.
  3. После завершения проверки отображаются результаты для каждой категории правил безопасности. Помощник по безопасности сообщает общее состояние для каждой категории и предоставляет информацию по выявленным рискам безопасности.

2. Просмотр отчетов по правилам безопасности

  1. На странице Обзор нажмите Просмотр результатов. Вы сможете по очереди просмотреть подробные отчеты о завершенном просмотре.
  2. Дважды щелкните на нужное правило безопасности, чтобы открыть подробный отчет. Каждый отчет для каждого правила безопасности может содержать информацию в разделах Серьезность, Описание, Сведения и Рекомендуемое действие.

3. Управление рисками безопасности

Рассмотрим пример нарушения правила безопасности и путей решения этой проблемы.

  1. Помощник по безопасности выявит потенциальные риски безопасности для любого нарушения правил безопасности. В данном примере Помощник по безопасности выявляет инцидент "Автоблокировка отключена".
  2. Откройте нарушенное правило безопасности, чтобы просмотреть подробную информацию. В разделе Рекомендуемое действие Помощник по безопасности предоставляет инструкции по устранению этого риска безопасности.
  3. Нажмите Открыть, чтобы перейти непосредственно к соответствующей странице настроек, и выполните Рекомендуемое действие.
  4. На странице Результаты повторите проверку нарушенного правила безопасности, выбрав правило и нажав Проверить.
  5. Для этого правила безопасности риски безопасности не обнаружены. Помощник по безопасности будет и дальше помогать выявлять потенциальные риски безопасности и рекомендовать действия по управлению этими рисками. Поддерживайте безопасность своего устройства Synology NAS, регулярно выполняя проверки с помощью Помощника по безопасности!
