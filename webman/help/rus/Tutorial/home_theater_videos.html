Распределение по категориям и воспроизведение видеозаписей с помощью Video Station

Обзор

С помощью Video Station можно легко организовывать и категорировать видеоколлекции по разным применяемым по умолчанию или пользовательским библиотекам. Это помогает намного более эффективно находить нужные видеозаписи. Кроме того, воспроизведение и публикация видеозаписей также выполняется очень просто и интуитивно — всего несколькими простыми щелчками.

1. Установка Video Station

Чтобы установить Video Station, перейдите на страницу Центр пакетов, выберите Video Station и затем нажмите Установить.

2. Сохранение и распределение видеозаписей в библиотеки по умолчанию

Video Station предоставляет три применяемые по умолчанию библиотекам, представленные на левой панели: Фильм, ТВ-передача и Домашнее видео. Чтобы видеозаписи были правильно проиндексированы и категорированы, необходимо для каждой библиотеки добавить папки видео и сначала сохранить файлы в соответствующих папках видео. По умолчанию для трех библиотек по умолчанию в общей папке video создается три папки видео: Фильм, ТВ-передача и Домашнее видео. Однако, выполнив следующие действия, в дополнение к этим трем папкам для библиотеки можно также добавить дополнительные папки видео.

Примечание.

Для двух типов видео (фильм и телепередача) система будет автоматически искать в Интернете связанные метаданные и прочие сведения (например, изображение афиши). Таким образом, чтобы для видеозаписи могла быть найдена правильная информация, настоятельно рекомендуется, чтобы видеозаписи относились к указанному типу видео соответствующих библиотек.

Добавление папок видео для библиотек по умолчанию

  1. Выберите Настройки > Библиотека.
  2. Нажмите Создать > Добавить папку, чтобы выбрать папку на Synology NAS, и выберите целевую библиотеку. Для одной библиотеки можно добавить до 100 папок видео, но после выбора папки ни папка, ни ее вложенные папки уже не могут быть выбраны для другой библиотеки.
  3. Сохраните видеофайлы в папках видео, чтобы распределить видеозаписи по соответствующим библиотекам.
  4. Теперь можно находить и воспроизводить нужные видеозаписи из библиотек!

3. Сохранение и распределение видеозаписей в пользовательские библиотеки

Кроме библиотек по умолчанию, к которым есть доступ у каждого пользователя, вы как администратор DSM с учетной записью admin или пользователи, принадлежащие к группе administrators, могут создавать пользовательские библиотеки и предоставлять права доступа другим пользователям.

3.1 Создание пользовательских библиотек

  1. Выберите Настройки > Библиотека и нажмите Создать > Создать библиотеку.
  2. Введите название для новой библиотеки, выберите тип видеозаписей, которые будут храниться в папке, и затем выберите Общие, если хотите, чтобы библиотека была доступна всем пользователям, или Личные, чтобы ограничить доступ только пользователями, которые имеют соответствующие разрешения. Примите во внимание: чтобы гарантировать точность информации о видеозаписи, собираемой из Интернета, добавляемые в папку видеозаписи должны соответствовать выбранному здесь типу.
  3. Если выбрано Личные, нажмите кнопку Назначить разрешения и установите флажки, чтобы предоставить доступ пользователям.

3.2 Добавление папок видео для пользовательских библиотек

  1. Выберите Настройки > Библиотека.
  2. Нажмите Создать > Добавить папку, чтобы выбрать папку на Synology NAS, и выберите целевую библиотеку. Для одной библиотеки можно добавить до 100 папок видео, но после выбора папки ни папка, ни ее вложенные папки уже не могут быть выбраны для другой библиотеки.
  3. Сохраните видеофайлы в папках видео, чтобы распределить видеозаписи по соответствующим библиотекам.
  4. Теперь можно находить и воспроизводить нужные видеозаписи из библиотек!

4. Воспроизведение видео

После распределения видеофайлов по разным библиотекам можно, следуя приведенным ниже шагам, воспроизводить видеозаписи на Video Station. Настоятельно рекомендуется сначала установить на компьютере проигрыватель мультимедиа VLC (Windows, Linux). Кроме воспроизведения видеозаписей на локальном компьютере, можно также выполнять потоковое воспроизведение видео на DMA-устройствах с поддержкой DLNA/UPnP или устройствах, совместимых с AirPlay.

Воспроизведение видеозаписей с помощью встроенного видеопроигрывателя

  1. Выберите нужный видеоклип и затем нажмите значок воспроизведения на изображении видеоклипа или выберите Действие > Воспроизвести.
  2. Воспроизведение видео должно начаться сразу же. Если оно не началось, убедитесь, что формат видеозаписи поддерживается веб-браузером, и установлен ли на компьютере подключаемый модуль VLC.
  3. При необходимости используйте расположенные внизу видеопроигрывателя экранные кнопки и шкалу выполнения. Можно также регулировать громкость, выбирать качество воспроизведения видео и аудиодорожку, включать субтитры или выполнять потоковую передачу видео на другое устройство (если оно поддерживает такой режим).
Примечание.

Подробные сведения о подключаемом модуле VLC и поддерживаемых устройствах см. в разделе Справка DSM > Video Station > Воспроизведение видео.

5. Публикация видеозаписей

Функция общего доступа позволяет легко обмениваться видеозаписями с другими людьми с помощью Video Station. Все общедоступные видеозаписи доступны все пользователям по ссылке; ими можно управлять на странице Коллекция.

Открытие доступа к видеозаписям

  1. Перейдите к нужной библиотеке видео и нажмите Все в правом верхнем углу, чтобы просмотреть все видеозаписи в этой библиотеке.
  2. Щелкните значок Режим выбора или нажмите и удерживайте клавишу Ctrl или Shift, чтобы выбрать несколько видеозаписей. Затем выберите Действие > Открыть доступ.
    • Если вы выбрали для публикации только одну видеозапись, она будет добавлена к общедоступной коллекции Общедоступные видеозаписи. Здесь можно управлять всеми общедоступными видеозаписями.
    • Если вы выбрали для публикации несколько видеозаписей, они будут сгруппированы в новую общедоступную коллекцию.
  3. В качестве альтернативы можно выбрать Действие > Коллекция, чтобы добавить выбранные видеозаписи к существующей общедоступной коллекции (если это применимо).
  4. При необходимости укажите срок действия.
  5. Чтобы поделиться видеозаписями с другими пользователями, просто копируйте и вставляйте ссылку в сообщения эл. почты, мгновенные сообщения и веб-страницы.

6. Организация и просмотр видеозаписей на мобильных устройствах

Чтобы подключиться к Video Station через мобильное устройство, можно загрузить DS video на устройство iOS/Android или Windows Phone с помощью приведенных ниже QR-кодов. После этого можно будет воспроизводить видеозаписи и организовывать коллекции на мобильных устройствах или просматривать потоковое видео на устройствах DMA или AirPlay — как дома, так и в любом другом месте.
