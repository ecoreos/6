Резервирование LUN

На вкладке Резервирование в меню Резервирование и репликация можно использовать функцию Локальное резервирование LUN для выполнения резервирования iSCSI LUN устройства DiskStation в локальные папки общего доступа. Вы также можете использовать Сетевое резервирование LUN для выполнения резервирования в другое устройство DiskStation.

Резервирование iSCSI LUN:

  1. Выберите Создать > Задача резервирования LUN, чтобы открыть Мастер резервирования LUN, и нажмите Далее.
  2. При отображении запроса на выбор типа местоположения выберите одно из следующих:
    • Локальное резервирование LUN: Выберите данный параметр, чтобы выполнить резервирование LUN с устройства DiskStation в локальную папку общего доступа или внешний диск, подключенный к порту USB или eSATA устройства DiskStation.
    • Сетевое резервирование LUN: Выберите данный параметр для резервирования LUN с устройства DiskStation на другое устройство DiskStation.
  3. При отображении запроса укажите место для резервирования и исходный LUN, затем настройте расписание резервирования.
  4. Следуйте инструкциям мастера настройки.

После создания задачи резервирования устройство DiskStation выполнит задачу по расписанию и сохранит резервный LUN в указанном местоположении. Резервный LUN используется только в целях резервирования. Для получения доступа к резервному LUN см. раздел Восстановление резервного LUN, чтобы получить дополнительную информацию.

Примечание.

  • LUN обычных файлов и на уровне блоков будут зарезервированы в указанном местоположении как образы файлов. При восстановлении все LUN на уровне блоков будут изменены на LUN файлов.
  • Каждая задача резервирования может содержать только один LUN в качестве источника резервирования.
  • Резервирование LUN не может выполняться на внешний диск с помощью формата FAT, FAT32, HFS, HFS Plus и NTFS.
  • Перед выполнением сетевого резервирования LUN выполните вход в необходимое устройство DiskStation и убедитесь, что служба сетевого резервирования включена в пункте Резервирование и репликация > Службы резервирования.
  • Если устройство DiskStation подключается к сети с помощью маршрутизатора, перед выполнением сетевого резервирования LUN убедитесь, что вы указали правило переадресации порта для службы iSCSI (порт TCP 3260) на маршрутизаторе.

О полном и инкрементном резервировании:

DiskStation выполнит полное резервирование выбранного LUN в первый раз, а затем будет выполнять инкрементное резервирование в дальнейшем. Однако если имя папки общего доступа или имя каталога места назначения резервирования изменено, устройство DiskStation выполнит полное резервирование еще раз.
