Специальные возможности

Благодаря встроенной функции поддержки специальных возможностей DiskStation Manager воспроизводит звуковые подсказки, упрощая навигацию и управление службами/настройками DSM для пользователей с нарушениями слуха.

Настройка специальных возможностей DSM

Чтобы максимально использовать специальные возможности, рекомендуется настроить рабочую среду следующим образом:

  • Установите приложение для чтения с экрана NVDA (доступно только в среде Windows) на локальном компьютере.
  • Установите последнюю версию веб-браузера Mozilla Firefox.
  • Используйте гарнитуры или динамики и настройте том в соответствии со своими потребностями.
  • NVDA используется в режиме фокусировки (для переключения режима нажмите клавишу NVDA + пробел). В качестве клавиши NVDA по умолчанию используется Insert.
  • Для навигации по DSM используйте следующие сочетания кнопок:
    Сочетание кнопок Описание
    Tab Переключение на другой пользовательский интерфейс
    Клавиши со стрелками Переключение на элементы в одном разделе пользовательского интерфейса
    Ввод Открытие приложений/окон
    Esc Закрытие приложений/окон

Функция поддержки специальных возможностей теперь работает в следующих областях DSM:

  • Страница входа в DSM, рабочий стол и панель задач
  • Основное меню
  • Панель управления
  • File Station

Примечание:

Сведения о работе NVDA см. в руководстве пользователя.

Руководство по клавишам быстрого доступа в DSM

Помимо поддержки специальных возможностей DSM позволяет реализовать гибкое и простое управление с помощью клавиш быстрого доступа. Для поиска клавиш быстрого доступа нажмите Shift + ?:

  • Клавиши быстрого доступа на настольной клавиатуре: Клавиши быстрого доступа для операций в окне и других действий
  • Клавиши быстрого доступа на системной клавиатуре: Клавиши быстрого доступа для управления компонентами пользовательского интерфейса (списками, кнопками, полями поиска и т.д.)

Чтобы приостановить действие ярлыков рабочего стола, отключите параметр Включение клавиш быстрого доступа на настольной клавиатуре.
