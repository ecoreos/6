限制和系统要求

Synology High Availability 功能需要使用两台相同的 Synology 服务器作为活动和无源服务器。在将两台服务器组合为 high-availability 集群前，系统将检查以下硬件和软件限制，确保 high-availability 集群可以成功创建。

系统要求

请注意以下软硬件说明：

  • Synology 服务器：活动和无源服务器都必须是相同型号。
  • 扩充设备：活动和无源服务器必须正确连接到相同的扩充设备。
  • 硬盘数量和种类：硬盘类型必须只能为 SAS。安装 DSM 之前，必须在扩充设备中安装至少 3 个硬盘。
  • USB 扩充设备：仅支持 USB 存储器。不支持其它 USB 外接设备（例如，UPS、无线网上、扬声器、打印机、蓝牙适配器和 DVB 适配器）。USB 存储器仅用于活动服务器，不可切换到无源服务器。

网络环境

若要构建 high-availability 集群，请注意以下有关网络设置的说明。

  • 网络设置：两个服务器之间的 Heartbeat 连接必须设置正确。两个服务器必须属于相同子网，需在安装后配置静态 IP 地址。
  • 局域网端口数量：两个服务器必须有相同数量的局域网端口。如果 DiskStation 型号之一配备了其它网卡接口，则另一个必须配置相同的网卡接口。
  • 连接限制：High-availability 集群不支持 Wi-Fi 网络。

注：

  • 活动服务器上的 NTP 服务器将被自动启用。

套件和软件限制

请参阅下列套件和软件相关限制：

  • High-availability 集群不支持以下套件和功能。
    • DVBLink Server
    • 在 CMS 中重启/关闭 high-availability 集群的功能
    • Time Backup
  • 带共享存储空间的 High-availability 集群无法被配置为或与基于复制的 high-availability 集群结合使用。

特定扩充设备

活动和无源服务器必须正确连接到特定的扩充设备。在每个扩充设备上有两个模块。

若要设置/添加扩充设备：

  • 一个模块含有两个端口；输入端口在左侧，输出端口在右侧。将 SAS 线从扩充设备一个模块的输出端口连接到下一个模块的输入端口。
  • 扩充设备必须与各服务器相应一侧的模块串联。
  • 活动和无源服务器必须按相同顺序连接到共享扩充设备。

注：

有关扩充设备安装的详细信息，请参见服务器和扩充设备快速安装指南。
