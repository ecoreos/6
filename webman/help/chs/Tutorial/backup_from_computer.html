使用 Cloud Station 在我的 PC/Mac 上备份数据

总览

Cloud Station Server 不仅是一个复杂的跨平台同步软件，用户可将其用作更智能的备份其个人计算机的方法。Cloud Station Backup 作为一个实时备份解决方案，具有即时和节省带宽的特点，可通过单个文件高达 32 个的历史版本提供更好的保护。

在 Synology NAS 上设置 Cloud Station Server

  1. 进入套件中心，找到 Cloud Station Server，然后单击安装。
  2. 启动 Cloud Station Server。
  3. 进入 Cloud Station 下的 概述并单击下载。(您还可从 Synology 下载中心下载 Cloud Station Backup。)
  4. 启动计算机上的安装程序并按说明安装 Cloud Station Backup。
  5. 安装后，在计算机上运行 Cloud Station Backup。单击立即开始。
  6. 输入运行 Cloud Station Server 的 Synology NAS 地址（或 QuickConnect ID）、用户名和密码。（您还可单击右侧的搜索图标，Cloud Station Backup 将自动搜索 LAN 中的其它 Synology NAS）。单击下一步。
  7. 选择备份源并取消勾选您不要同步的任何子文件夹。
  8. 在 Synology NAS 上单击选择以选择一个文件夹。Cloud Station Backup 将在所选目的地以您计算机的名称创建备份文件夹。
  9. 如果您要设置同步过滤器或规则，单击备份规则。
  10. 单击下一步以查看您的备份设置，然后单击完成。
  11. 您可在主应用程序域中查看备份进程。

下载或还原 Cloud Station Server 上同步文件的旧版本

  1. 进入 Cloud Station Server> 版本历史并找到您要还原的文件或文件夹。
  2. 请选择您要检索的文件，并单击操作。单击浏览以前的版本。如果计算机上的文件被意外删除或移动，可单击还原来恢复。
  3. 选择您要下载的文件版本，并单击下载。或者，如果您确认要还原选择的版本，单击还原，它将覆盖当前版本。

5. 进一步保护您的数据
