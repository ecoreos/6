备份服务

备份和复制 > 备份服务中的选项可让其他用户将数据从 DiskStation 设备或 rsync 兼容客户端备份至 DiskStation。

启用网络备份服务

如果您要将 DiskStation 或 rsync 兼容服务器上的数据备份到另一台 DiskStation，或允许另一台 DiskStation 设备执行远程备份任务并将备份数据存储到 DiskStation 设备的卷上，则必须在创建备份任务之前先启用目标服务器上的网络备份服务。您可进入控制面板 > 权限，向用户授予网络备份目的地权限。然后，您可以为您 DiskStation 上的一位 DSM 用户创建适当的应用程序及共享文件夹权限。设置完成时，其他人便可利用该用户的权限，从他们的 DiskStation 运行网络备份到您 DiskStation 的目标共享文件夹中。

若要启用单个版本备份服务：

  1. 进入备份服务 > 网络备份目的地 ，然后勾选启用网络备份服务。
  2. 单击应用。
  3. 进入控制面板 > 共享文件夹，选择共享文件夹，单击编辑 > 权限，然后将读取/写入权限授予给用户。重复运行此步骤来加入更多的共享文件夹，这些文件夹将可作为备份目的地。

若要启用多重版本备份服务：

  1. 进入备份服务 > 网络备份卷目的地，然后勾选启用网络备份卷目的地的复选框。
  2. 进入备份目的地并选择与本地卷相关联的备份目的地。
  3. 单击编辑 > 允许列表 > 添加来添加用户或群组到列表。

注：

  • 如果其他人想要从非 DiskStation 的客户端设备执行网络备份，他们仅能利用您 DiskStation 的 admin 或 rsync 权限执行此操作。如果您不想与他人共享 admin 凭据，可进入控制面板 > 用户来创建一个名为 rsync 的用户，并修改其权限，然后与他人共享其用户凭据。
  • 如果您从运行 DSM 3.0 之前版本的 DiskStation 或不是 DiskStation 的客户端，且要保留源数据的所有者和群组信息，必须将备份帐户添加到 administrators 群组，并将数据备份到 NetBackup 共享文件夹。

使用自定义的 rsync 配置来启用网络备份服务

您可以使用自定义的 rsync 配置来启用网络备份服务，并在您的 DiskStation 配置用户权限和备份目的地。当设置完成时，其他人可以从他们的 DiskStation 设备或 rsync 兼容客户端运行 rsync 备份到您 DiskStation 上的目标共享文件夹（或称“备份模块”）。

若要使用自定义的 rsync 配置来启用网络备份服务：

  1. 请进入备份和复制 > 网络备份目的地。
  2. 请勾选启用网络备份服务。
  3. 勾选使用自定义的 rsync 配置。
  4. 单击应用。

若要在您的 DiskStation 上设置用户权限和备份模块：

  1. 请通过 Telnet 或 SSH 登录 DiskStation 设备。
  2. 在 rsync 配置文件“/etc/rsyncd.conf”中指定备份模块和您要授权的用户。如需详细信息，请参阅 rsync 客户端设备随附的 rsyncd.conf 说明页面或此处。

注：

  • 如果自定义 rsync 配置已被使用，DiskStation 将无法用作共享文件夹同步、Time Backup 和 LUN 备份的目标服务器。

启用速度限制

您可启用速度限制来控制用户和群组通过网络备份传输数据的最大可用带宽。

若要启用速度限制：

  1. 请进入备份和复制 > 网络备份目的地。如果您只要在特定时间启用速度限制，请勾选标示为启用速度限制或根据计划启用速度限制的复选框。此将应用至整个服务器，并覆盖服务器中的各用户和群组的设置。选择无速度限制可禁用所有用户和群组的速度限制设置。
  2. 单击速度限制设置可修改单个用户或群组的速度限制。您可为每个用户和群组选择下列其中一个选项：
    • 应用群组设置：速度限制设置将与用户所属群组的设置相同。
    • 无：当群组成员用此服务传输文件时不应用速度限制。
    • 恒定：一个固定速度限制常量将应用到用户或群组成员。可在左侧栏中指定上传与下载的速度限制。
    • 可变：可根据计划指定两种不同的速度限制并应用到用户或群组成员。单击自定义按钮可修改速度限制设置并设置计划。
  3. 单击应用。

注：

设置用户的速度限制后，结果区域会根据单个和群组设置显示用户的实际速度限制。例如，如果用户 “Achilles” 有 100 KB/秒的具体下载限制，但该用户所属的群组的下载限制为 500 KB/秒，则结果区域将显示 “0/500”。

启用 Time Backup 服务

Time Backup 服务可让用户从另一台 DiskStation 远程备份多个版本的文件夹到 DiskStation。这需要启用网络备份服务及 Time Backup 服务。

若要启用网络备份服务和 Time Backup 服务：

请进入备份和复制 > Time Backup。勾选适当的复选框。

注：

只有 admin 帐户才能为远程 Time Backup 创建备份连接。
