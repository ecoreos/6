还原

进入备份和还原，并单击左下角的还原可还原备份数据。

若要还原数据/LUN* 备份：

  1. 单击左下角的还原，然后选择数据或 LUN，并选择要还原的备份任务。
  2. 将提示您选择系统配置、不同版本的备份数据等，取决于您要还原的备份任务类型。
  3. 如果备份任务被加密，您需要密码/密钥才能成功还原。
  4. 按向导完成还原。

注：

  • 还原套件时，如果该套件尚未安装，系统会自动安装相同的或最新版本的应用程序。如果该套件已安装，但其比备份版本旧，系统会自动升级到原备份版本。
  • 在 DSM 5.1 或以上版本中备份的应用程序无法在 DSM 5.0 或以下版本中还原。

注：

  • 还原完成后，块级和文件级 LUN 都将被还原为文件级 LUN。*
  • 还原套件时，如果该套件尚未安装，系统会自动安装相同的或最新版本的应用程序。如果该套件已安装，但其比备份版本旧，系统会自动升级到原备份版本。
  • 在 DSM 5.1 或以上版本中备份的应用程序无法在 DSM 5.0 或以下版本中还原。

若要还原多版本备份任务的单个文件或文件夹：

  1. 在左侧面板中选择所需的备份任务，并启动备份资源管理器 (通过单击放大镜图标)。如果备份任务被加密，则需要密码/加密密钥。
  2. 用时间轴或通过日历图标找到所需的备份版本。
  3. 选择包含的文件或文件夹，然后单击还原、复制到...或下载以恢复所选的文件/文件夹。右键单击文件/文件夹也可使用这三个选项。

若要搜索特定版本：

若要加快文件搜索速度，您可使用关键字来过滤所需文件/文件夹。不匹配的文件/文件夹将被隐藏起来。

  1. 启动备份资源管理器，并单击右上角的漏斗图标。
  2. 在下拉菜单中，您可按以下条件过滤文件/文件夹：
    • 关键字：输入任何文件或文件夹的名称。您可在关键字中可使用统配符 [*] (亦即零个或多个字符) 和 [?] (亦即任意随机字符)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
    • 文件类型：从任何、任何文件夹或任何文件中进行选择以微调搜索。
    • 大小 (KB)：选择任何、或选择等于、大于、小于，然后输入一个值。
    • 修改日期：从今天、昨天、上周或下月或自定义中进行选择，以输入起始与结束日期，从而在特定范围内寻找版本。
  3. 若要应用过滤器，单击过滤器。
  4. 若要将同一过滤器应用到另一个备份版本，只需通过时间轴切换到所需版本即可。
  5. 若要重新设置过滤搜索，单击重置按钮。

注：

  • 备份资源管理器仅对多版本备份任务可用。
  • 备份资源管理器在备份目的地在 DSM 5.1 或以下版本设置时有功能限制。
  • 系统配置和应用程序只能通过还原功能来还原。
  • 滤器功能仅在未加密备份任务中可用。
  • 使用复制到...或下载恢复的文件/文件夹将丟失其原始权限设置。

若要从其他备份目的地还原备份任务：

  1. 单击左下角的还原，然后选择数据或 LUN*。
  2. 您可选择和还原这两种备份任务之一：
    • 若要还原数据备份：单击数据 > 从已有存储库还原，然后选择所需备份目的地以还原数据。
    • 若要还原 LUN 备份：单击 LUN*。
  3. 请按向导指示操作。如果要还原的任务被加密，则需要密码/加密密钥。
  4. 单击应用完成还原。

注：

  • 内置的 DataStation 仅将共享文件夹还原到 ext3/ext 外接设备。
  • Embedded DataStation 不支持无 ext3/ext 外接设备的应用程序还原功能。

____
* 仅适用于特定机型。
