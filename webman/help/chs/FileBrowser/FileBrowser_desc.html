File Station

File Station 是一款易于使用的文件管理工具，可让 Internet 上的 DSM 用户使用他们自己的网页浏览器访问 DiskStation 文件夹。您可以启动两个或两个以上 File Station，在不同的 File Station 之间以拖放操作来管理 DiskStation 上的所有数据。
