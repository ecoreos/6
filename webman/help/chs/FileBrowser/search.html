搜索文件或文件夹

File Station 的过滤器为您提供了搜索并显示符合条件的文件或文件夹的选项。您可过滤当前文件夹中的文件或文件夹，或执行涵盖所有子文件夹的高级搜索以更精确地搜索结果。

若要搜索当前文件夹中的文件或子文件夹：

  1. 用 File Station 进入包含您想要搜索内容的文件夹。
  2. 在右上角的过滤器栏中，输入关键字并在窗口的主要区域中查看过滤的结果。

若要执行高级搜索：

  1. 用 File Station 进入包含您想要搜索内容的文件夹。
  2. 单击右上角过滤器栏旁边的放大镜以显示高级搜索菜单。
  3. 指定以下搜索条件，或从下拉菜单中进行选择：
    • 关键字：输入关键字来搜索名称相符的文件或文件夹。

      注：

      • 您可在关键字中可使用统配符 [*] (亦即零个或多个字符) 和 [?] (亦即任意随机字符)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
      • 您可用空格来分开多个关键字。例如，关键词 [File Station] 可搜索包含 [File]、[Station] 或两者的结果。
      • 您可以在关键字的开头和结尾处加入双引号 ["] 进行精确、区分大小写的匹配项。例如，[“Synology Inc”] 可代表 [Synology Inc.]，但不能代表 [Synology Incorporation]。
      • 以下运算符可让搜索结果更精确：
        运算符 例如 使用方式
        && [X] && [Y]

    • 启用文件内容搜索：搜索索引文件，以找到匹配关键词的文件内容或元数据。
    • 位置：从下拉菜单中以设置高级搜索的范围和深度。您还可选择列示的任何其它共享文件夹以自定义搜索范围。
    • 文件类型：选择文件扩展名来将搜索范围限制为特定类型的文件，或是选择任何文件夹来将搜索范围限制为仅搜索文件夹。
    • 日期：选择修改日期、创建日期或最近访问，然后从日历中选择日期以搜索符合日期的文件。
    • 大小 (MB)：您可以选择等于、大于或小于，然后输入大小来搜索条件相符的文件。(如果您选择等于，则搜索结果将包含与指定大小近似的文件。)
    • 所有者/群组：选择所有者或群组，然后从下拉菜单中进行选择，搜索所选用户或群组拥有的文件或文件夹。

      注：

      • 要执行高级搜索，需指定位置用至少一个其它搜索条件。
      • 如果在索引的共享文件夹中有 ISO 文件夹或远程文件夹，您将无法通过高级搜索找到其中的文件。
  4. 单击搜索来开始搜索，或是单击重置并再次指定搜索条件。搜索完成之后，条件相符的文件会列在 File Station 的主要区域中。
