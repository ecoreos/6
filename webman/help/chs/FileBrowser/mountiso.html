装载虚拟驱动器

您可以装载虚拟驱动器，在 File Station 上访问光盘映像内容。

若要装载虚拟驱动器：

  1. 选择光盘映像文件并单击工具 > 装载虚拟驱动器。
  2. 单击浏览选择一个目的地，或在 DiskStation 上添加一个空的目的地文件夹来装载虚拟驱动器。
  3. 如果您想要让 DiskStation 在每次系统开机或重新启动时自动装载此虚拟驱动器，请勾选开机时自动装载。单击装载继续。现在您可以前往已装载虚拟驱动器的目的地文件夹，并且访问光盘映像内容。

注：

虚拟驱动器和 .iso 文件只能装置到共享文件夹下的子文件夹。

若要卸载虚拟驱动器：

  1. 右键单击已装载虚拟驱动器的目的地文件夹并选择卸载。
    注：
  1. 支持的光盘映像文件格式：".iso“文件，使用 ISO 9660 和 UDF 文件系统。
  2. 最多可同时装载 16 个虚拟驱动器。
  3. 仅在高权限模式下才支持装载虚拟盘。若要使用此功能，请进入主 DSM 的 Docker 套件，选择您要为其启用加密共享文件夹的 Docker DSM，单击编辑， 然后勾选使用高权限执行容器旁的复选框。
