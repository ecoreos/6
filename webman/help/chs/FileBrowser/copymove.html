复制或移动文件或文件夹

File Station 可向您提供在两个 DiskStation 中、一个 DiskStation 中传输文件/文件夹或传输至本地计算机的各种方法。

在浏览器之间拖放文件/文件夹

无需复杂的文件传输协议设置，您就可通过 File Station 从一个 DiskStation 将文件/文件夹传输到另一个 DiskStation。

若要在浏览器之间启用拖放操作：

  1. 进入设置 > 常规 > File Station。
  2. 勾选在浏览器之间启用拖放操作。
  3. 单击确定。
  4. 打开两个相同类型的网页浏览器（如 Firefox），并将其分别连接到来源和目标 DiskStation。
  5. 在两个 DiskStation 上启动 File Station。您现在可通过 File Station 直接在它们之间拖放文件/文件夹。

注：

  • 您还可在同一浏览器的两个选项卡之间拖放文件/文件夹。
  • Internet Explorer 不支持浏览器之间或选项卡之间的拖放操作。
  • 如果选择多个文件/文件夹进行拖放操作，这些文件/文件夹将被压缩到一个档案以进行上传。
  • 拖放操作的文件大小限制视网页浏览器而有所不同：Safari（1 GB）。

在 DiskStation 之间复制或移动文件/文件夹或至本地计算机

按照以下步骤在 DiskStation 中移动或复制文件或文件夹，或至您的计算机。

方法 1：

  1. 选择您想移动或复制的文件或文件夹。（多选时请按住 Shift 或 Ctrl 键。）
  2. 从操作下拉菜单中选择剪切或复制。
  3. 进入 DiskStation 或计算机中的目标文件夹。
  4. 根据您是否想要以同样的名字覆盖目标文件夹中的文件或文件夹，从操作下拉菜单中选择粘贴 - 覆盖或粘贴 - 跳过。

方法 2：

  1. 选择您想移动或复制的文件或文件夹。（多选时请按住 Shift 或 Ctrl 键。）
  2. 将文件夹或文件从主要区块拖放至左侧面板上的其它文件夹中。
  3. 在显示的下拉菜单中，选择复制 - 覆盖、复制 - 跳过、移动 - 覆盖或移动 - 跳过以同样名字覆盖或跳过目标文件夹中的文件或文件夹。

方法 3

  1. 选择您想移动或复制的文件或文件夹。（多选时请按住 Shift 或 Ctrl 键。）
  2. 按 Ctrl + X 来剪切或按 Ctrl + C 来复制文件/文件夹。Mac 用户，请按 Command + X 和 Command + C。
  3. 进入 DiskStation 或计算机中的目标文件夹。
  4. 按 Ctrl + V 来粘贴文件/文件夹。（Mac 用户为 Command + V。）
  5. 在显示的弹出式菜单中，选择覆盖或跳过目的地文件夹中名称相同的文件或文件夹。

键盘快捷方式：

在 File Station 中移动/复制文件和文件夹时，以下快捷键可派上用场：

  • F2：重命名
  • Del：删除
  • Ctrl + A：全选
  • Crtl + X：剪切
  • Crtl + C：复制
  • Crtl + V：粘贴

注：

  • 当文件/文件夹复制或移动的目的地为 FAT 文件系统的目录时，文件名中的 \ *?">| 字符将会由 – 字符替换，且 Mac 文件中的元数据会被删除。
  • 如果您从本地计算机剪切文件并粘贴到 DiskStation 文件夹，文件将仍然留在您的本地计算机中。
  • 当移动文件或文件夹时，“所有者”不会改变。但在复制文件或文件夹时，“所有者”将更改为对其进行复制的用户。
