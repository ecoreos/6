管理 File Station 设置

若要管理 File Station 设置，打开 File Station，然后单击设置按钮。

常规设置

您可以在此启用 File Station 日志来记录所有用户的操作。包括创建文件夹、上传、下载、删除、重命名、移动、复制和属性设置等用户操作。

若要启用 File Station 日志：

  1. 如果您想查看所有用户的操作记录，请勾选启用 File Station 日志。
  2. 单击确定。

若要查看 File Station 日志：

  1. 请单击查看日志，将引导您前往日志中心。
  2. 单击文件传输选项卡。

若要使用拖放操作上传或键盘快捷键自动覆盖文件：

如果您在从计算机桌面将文件拖放到网页浏览器或用键盘快捷键在 File Station 中粘贴文件时要自动覆盖名称相同的文件，请勾选当我将这些文件拖放到网页浏览器或用键盘快捷键粘贴这些文件时始终覆盖文件。

若要启用智能拖放：

当您在 File Station 中进行拖放文件时，此功能将自动决定传输的是复制还是移动操作的问题。

  1. 选择启用智能拖放。
  2. 选择如果恰好有另一个用户同时传输相同名称的文件至相同位置时始终执行的操作：
    • 覆盖：您传输的文件将覆盖有相同名称的文件。
    • 跳过：您传输的文件将被跳过，且不会覆盖有相同名称的文件。
  3. 单击确定。

注：

如果您要更改智能拖放决定的复制或移动操作，请应用以下热键来满足您的需要：

  • 若要切换到复制操作：按 Ctrl 同时执行拖放操作。
  • 若要切换到移动操作：按 Shift 同时执行拖放操作。

若要通过 File Station 浏览和管理本地计算机中的文件：

  1. 请勾选通过 File Station 浏览本地计算机中的文件。
  2. 单击确定。

若要与非 admin 用户共享 DSM 用户列表：

此选项可让非 admin 用户在需要搜索特定 DSM 用户/群组拥有的文件或要与其他 DSM 用户共享文件链接时访问完整的 DSM 用户列表。

  1. 选择与非 admin 用户共享 DSM 用户列表。
  2. 单击确定。

应用默认的 UNIX 权限

单击高级设置并勾选此选项，在上传或创建文件或文件夹时应用默认的 UNIX 权限。所应用的权限将与 UNIX umask 命令所应用的权限相同。启用此选项后，文件的 UNIX 权限为 644，而文件夹为 755。禁用此选项后，文件的 UNIX 权限为 666，而文件夹为 777。默认 umask 值为 022。

注：

  • 对于启用 Windows ACL 的共享文件夹（外部设备上的 "照片" 和共享文件除外），请在 Linux 或 FTP 客户端上运行 chmod 命令，以将 Windows ACL 上的文件夹和文件权限类型更改到 UNIX。
  • 启动此选项可能会导致不同协议之间发生权限不一致的问题。为避免不一致问题，建议您保持禁用此选项。

装载/连接

默认时，所有用户可以装载远程文件夹或虚拟驱动器。您可在此将远程文件夹或虚拟驱动器的管理权限设置为仅限管理员。

注：

非管理员用户只能在 home 文件夹中装载远程文件夹。

若要允许不同用户装载远程文件夹或虚拟驱动器：

  1. 请在远程文件夹或虚拟硬盘区块中选择所有用户或管理员。
  2. 单击确定。

若要允许不同用户装载服务器和云服务：

  1. 从所有用户、管理员或特定用户中进行选择。
  2. 选择特定用户将打开一个窗口，让您为不同用户配置装载权限。若要启用特定用户的访问权限，请确认已勾选已启用栏下的复选框。若要禁用特定用户的访问权限，请确认未勾选已启用栏下的复选框。完成后，单击确定以关闭窗口。

    3. 注：

    • 属于 administrators 群组的用户可使用此功能，但不可进行任何修改。使用访客身份登录的用户既不能使用此功能，也不可进行任何更改。在此可设置任何其他用户的权限。
  3. 单击确定。

共享链接

默认时，所有用户皆可共享文件链接。您可在此将共享链接管理的访问权限设置为仅限管理员。

若要允许不同用户共享文件链接：

  1. 请选择所有用户、管理员或特定用户。
  2. 单击确定。

若要以纯文本格式打开 HTML 文件：

  1. 勾选始终以纯文本格式打开 HTML 文件复选框。如果启用此选项，当用户通过文件共享链接打开 HTML 文件（即 HTM、HTML、JSP、XHTML 文件）时，这些文件都将以纯文本格式显示。此选项有助于防止恶意者使用 HTML 文件中内嵌的脚本黑入用户的登录会话。
  2. 单击确定。

速度限制

您可启用速度限制来控制用户和群组通过 File Station 传输数据的最大可用带宽。

若要启用速度限制：

  1. 如果您只要在特定时间启用速度限制，请勾选标示为启用速度限制或根据计划启用速度限制的复选框。此将应用至整个服务器，并覆盖服务器中的各用户和群组的设置。选择无速度限制可禁用所有用户和群组的速度限制设置。
  2. 单击速度限制设置可修改单个用户或群组的速度限制。您可为每个用户和群组选择下列其中一个选项：
    • 应用群组设置：速度限制设置将与用户所属群组的设置相同。
    • 禁用速度限制：当群组成员用此服务传输文件时不应用速度限制。
    • 恒定：一个固定速度限制常量将应用到用户或群组成员。可在左侧栏中指定上传与下载的速度限制。
    • 可变：可根据计划指定两种不同的速度限制并应用到用户或群组成员。单击自定义按钮可修改速度限制设置并设置计划。
  3. 单击应用。

注：

  • 要了解有关群组速度限制动态详情，请参阅此处。
  • 设置用户的速度限制后，结果区域会根据单个和群组设置显示用户的实际速度限制。例如，如果用户 “Achilles” 有 100 KB/秒的具体下载限制，但该用户所属的群组的下载限制为 500 KB/秒，则结果区域将显示 “0/500”。

安全性

您可在此找到以下安全设置。

若要以纯文本格式打开 HTML 文件：

  1. 勾选始终以纯文本格式打开 HTML 文件复选框。如果启用此选项，当在 File Station 中打开 HTML 文件（即 HTM、HTML、JSP、XHTML 文件）时，这些文件都将以纯文本格式显示。此选项有助于防止恶意者使用 HTML 文件中内嵌的脚本黑入用户的登录会话。

    2. 注：

    • 这不会更改全局设置。每个用户都可根据自身需要选择是否启用此选项。
  2. 单击确定。
