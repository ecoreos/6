共享文件链接

保存在 DiskStation 中的文件或文件夹可轻松快速地进行共享。只需右键单击文件或文件夹，并选择共享文件链接即可。此时会生成一个链接和 QR 代码，您可将其发送给好友或其他用户。他们可以通过点按该链接来下载所选文件或文件夹，即使他们没有 DSM 帐户。

若要分享文件链接：

  1. 请选择您想分享的文件或文件夹。
  2. 右键单击所选文件或文件夹，或单击操作菜单。然后选择共享文件链接。
  3. 在出现的窗口中将显示共享链接与 QR 代码。
  4. 您还可修改以下选项：
    • 有效期：您可为共享链接自定义有效访问期和访问时间。
      • 自定义有效期:您可确定共享链接在创建后保持活动的时间。链接在特定时间过后将变为无效。
      • 自定义有效访问次数：您可确定共享链接可被访问的次数。链接在特定次数过后将变为无效。
    • 与 DSM 用户共享：您可从下拉列表中选择要与 DSM 用户或群组共享的链接。请注意，您无法与访客帐户或用户群组共享。
    • 与他人共享：您可与他人（包括非 DSM 用户）共享受密码保护的链接。

若要启用 gofile.me 链接：

  1. 请进入控制面板 > QuickConnect 并确保勾选启用 QuickConnect。若您尚未设置 QuickConnect，可在此页面设置。
  2. 下一步，单击高级并确认已勾选文件共享。
  3. 您无需 DSM 帐户即可通过访问 gofile.me 共享链接下载共享文件。

自定义共享链接页面

File Station 向您提供了不同的选项以使共享链接页面风格化。

  1. 请进入 File Station > 设置 > 共享链接 > 自定义。
  2. 选择自定义共享链接页面并单击自定义设置。
  3. 您可通过以下方式自定义该页面：
    • 模板：选择默认模板之一。
    • 背景：单击缩略图或 + 以选择默认壁纸、DiskStation 上的图像或之前应用的图像。
    • 标志：单击缩略图或 + 以选择上传的图像（小于 1 MB）、DiskStation 上的图像或之前应用的图像。标志可在四角之一进行定位。
  4. 单击预览可看到是否需要任何调整。
  5. 单击确定来应用设置。

注：

  1. 自定义设置将应用到所有的共享链接而也其创建时间无关。
  2. 您只能在壁纸/标志中应用以下格式的图像。.jpe、.jpeg、.jpg、.bmp、.gif 和 .png

管理共享链接

您可进入工具 > 共享链接管理器来管理已有的共享链接。在出现的窗口中，会有选项可让您对已有的共享链接进行编辑、共享或删除。

注：

  1. 您最多可创建 1000 个共享链接。
  2. 每个共享链接的状态区域将会显示下列其一状态：
    • 有效的：该共享链接正常工作。
    • 已过期：表示该共享链接已超过有效期。
    • 无效：表示该共享文件或文件夹已不存在，或您没有访问该文件或文件夹的权限。
    • 尚未生效：表示该共享链接尚未激活。
