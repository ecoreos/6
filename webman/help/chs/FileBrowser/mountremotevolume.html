装载远程文件夹

您可以通过 File Station，从远程服务器 (例如网络上的计算机或其他 Synology NAS 设备) 装载远程文件夹来浏览和管理装载的文件夹。

若要装载远程 CIFS 共享文件夹*：

  1. 单击工具 > 装载远程文件夹 > CIFS 共享文件夹。
  2. 在文件夹栏中，以 \\Remote_Server\Remote_Folder 的格式来指定远程文件夹路径，如 \\192.168.0.2\share 或 \\chad\cheeseburger\pictures。
  3. 如果远程服务器需要访问权限才能访问远程文件夹，请在帐户名称及密码栏中输入远程服务器的用户证书信息。
  4. 单击浏览选择一个目的地，或在 DiskStation 上添加一个空的目的地文件夹来装载远程文件夹。
  5. 如果您想要让 DiskStation 在每次系统开机或重新启动时自动装载此远程文件夹，请勾选开机时自动装载。
  6. 单击装载可将远程文件夹装载到目标文件夹。您现在可从 DiskStation 浏览和管理远程文件夹。

若要装载远程 NFS 共享文件夹：

  1. 单击工具 > 装载远程文件夹 > NFS 共享文件夹。
  2. 在文件夹栏中，以 Remote_Server:/Remote_Folder_Path 的格式来指定远程文件夹路径，如 192.168.0.2:/volume1/share。
  3. 选择所需的 NFS 版本和传输通讯协议。
  4. 单击浏览选择一个目的地，或在 DiskStation 上添加一个空的目的地文件夹来装载远程文件夹。
  5. 如果您想要让 DiskStation 在每次系统开机或重新启动时自动装载此远程文件夹，请勾选开机时自动装载。
  6. 单击装载可将远程文件夹装载到目标文件夹。您现在可从 DiskStation 浏览和管理远程文件夹。

注：

NFSv4 仅支持 TCP 协议。

若要卸载远程文件夹：

  1. 选择要卸载的远程文件夹，并单击工具 > 卸载。
  2. 或者只需右键单击远程文件夹并选择卸载。

注：

  • 同时间可装载的最大远程文件夹数量为 100。
  • 只有当远程服务器支持 CIFS 或 NFS 通讯协议时，您才可以装载它的远程文件夹。
  • 远程文件夹只能装置到共享文件夹下的子文件夹。
  • 仅在高权限模式下才支持装载远程文件夹。若要使用此功能，请进入主 DSM 的 Docker 套件，选择您要为其启用加密共享文件夹的 Docker DSM，单击编辑， 然后勾选使用高权限执行容器旁的复选框。

____
* 仅适用于特定机型。
