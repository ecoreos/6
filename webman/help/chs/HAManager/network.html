网络设置

本页包含显示 high-availability 集群、活动成员服务器和无源成员服务器网络设置的三个选项卡。

HA 集群选项卡

创建 high-availability 集群时，系统为 high-availability 集群生成一个共同的 IP 地址，进行传入连接的接收。必要时可以创建另外的 high-availability 集群 IP 地址。活动服务器上已激活网络接口将在此选项卡上显示（检测信号连接所使用的网络接口将不会在本页显示，因为所述接口将用于在两个服务器之间进行数据复制）。该选项卡显示以下信息：

  • 网络接口：活动服务器上已激活网络接口。
  • 连接：显示该接口是否已连接。
  • IP 地址：创建 high-availability 集群时，系统将为 high-availability 集群生成一个共同的 IP 地址，进行传入连接的接收。如果您想要登录 DiskStation Manager（DSM），您可以用该 IP 地址连接到 high-availability 集群，不论哪个 Synology 服务器是活动服务器。这样在进行移转时就无需连接到不同的 IP 地址。
  • 子网掩码：所给定 high-availability 集群 IP 地址的相应子网掩码。

要编辑 IP 地址和子网掩码：

  1. 选择想要编辑的 IP 地址或子网掩码，点击编辑或双击字段开始编辑。
  2. 勾选启用复选框，启用或禁用每个接口。您可以启用一个以上的集群网络接口，但是创建集群时的原始网络接口无法禁用。

要更改 high-availability 集群服务器名称：

  • 点击更改网络设置，查看 high-availability 集群名称并更改其设置。
  • 点击应用，保存当前 high-availability 集群名称和 IP 地址设置。
  • 要放弃当前更改，点击取消。

活动和无源服务器选项卡（显示为服务器名称）

这些选项卡显示活动或无源服务器上网络接口状态。其中，检测信号连接将占据每个活动和无源服务器上的一个网络接口。选项卡显示以下信息：

  • 网络接口：服务器上物理网络接口。
  • IP 地址：给定网络接口所分配的 IP 地址。
  • 子网掩码：所给定 IP 地址的相应子网掩码。
  • 说明：显示给定网络接口的信息及其分配的 IP 地址可访问性。本字段将显示以下可能的状况：
    1. 可访问原始 IP 地址: 分配给该活动服务器上接口的原始 IP 地址可以进行数据和服务访问。
    2. 不可访问原始 IP 地址: 分配给该无源服务器上接口的原始 IP 地址不能访问。
    3. 检测信号连接 IP，无法从内部网访问: 自动分配给检测信号连接的原始 IP 地址不能进行数据和服务访问。
