无论您确切知道要查找哪些说明，或者只想浏览需要的内容，DSM 帮助都能为您提供快速及简易的方式来找到下列主题及资源。

主菜单

启动 DSM 服务及安装在 DiskStation 中的套件。

选项

修改帐户设置，如密码、描述或 2 步骤验证。

控制面板

查看并修改各种有关文件共享、系统、网络服务及其它的 DSM 设置。

File Station

组织、上传或下载存储在 DiskStation 的文件夹和文件。

备份和还原

备份或还原数据、iSCSI LUN 或您的 DiskStation 系统配置。

备份或还原您 DiskStation 的数据或系统配置。

套件中心

安装、更新及管理 Synology 或第三方应用程序。

EZ-Internet

使用简易的设置向导来通过 Internet 访问 DiskStation。

资源监视器

监视 CPU 使用、内存使用、网络流和存储空间。

监控 CPU 使用率、内存使用率及网络流量。

系统信息

查看 DiskStation 及其他连接设备的整体状态。
