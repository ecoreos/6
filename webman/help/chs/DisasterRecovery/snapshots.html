快照

本页面可让您拍摄、删除和编辑快照，以及配置快照计划和保留设置。

您一次可为每个共享文件夹或 iSCSI LUN 存储最多达 256 张快照副本。快照可通过保留具体数据块而不是整个文件，从而使硬盘空间占用达到最小。当 iSCSI LUN 中共享文件夹或数据块中的文件更新或删除时，可能会消耗额外的空间。

共享文件夹

若要对共享文件夹拍摄快照：

  1. 选择您要拍摄快照的共享文件夹。
  2. 单击拍摄快照。
  3. 在描述中为要拍摄的快照输入描述。
  4. 勾选/取消勾选锁定复选框以确定在触发自动删除时该快照是否忽视保留设置。
  5. 单击确定开始拍摄快照。

注：

  • 在控制面板 > 共享文件夹 > 编辑页面的设置中，仅 Windows ACL 权限会被录制到快照中。

若要浏览共享文件夹的快照：

  1. 选择您要浏览其快照的共享文件夹。
  2. 单击快照列表。
  3. 选择您要浏览的快照，并单击浏览。

注：

  • 对加密共享文件夹，快照浏览无法执行。
  • 如果您无法浏览快照，请进入控制面板 > 共享文件夹 > 编辑以启用让快照可见。

若要删除共享文件夹的快照：

  1. 选择您要删除其快照的共享文件夹。
  2. 单击快照列表。
  3. 选择您要删除的一张或多线快照，并单击删除。（多选时请按住 Shift 或 Ctrl 键。）
  4. 您将看到确认信息。单击删除以确定删除。

若要编辑共享文件夹的快照：

  1. 选择您要编辑其快照的共享文件夹。
  2. 单击快照列表。
  3. 选择您要编辑的快照，并单击编辑。
  4. 您可在描述中修改快照的描述，并勾选/取消锁定复选框以确定在触发自动删除时该快照是否忽视保留设置。
  5. 单击确定来保存设置。

若要配置计划快照：

通过计划快照，快照和复制将在预告确定的时间和频率自动拍摄快照，此功能可减轻您手动拍摄快照的工作量。

  1. 选择共享文件夹。
  2. 单击设置计划。
  3. 在计划选项卡中，勾选启用快照计划复选框，设置拍摄快照的时间和频率。若要取消此共享文件夹的计划快照，请取消勾选此复选框。
  4. 单击确定来保存设置。
  5. 您将看到确认信息。单击确定以确认配置。

注：

  • 计划快照的配置不包括在控制面板 > 更新和还原 > 配置备份功能中。

若要为计划快照配置保留设置：

  1. 选择共享文件夹。
  2. 单击设置计划。
  3. 选择保留选项卡。
  4. 配置保留设置。请选择下列任何项目：
    • 达到上限后删除旧快照：当快照数量达到上限时，最旧的快照将被新拍摄的快照覆盖。
    • 始终保留快照：当快照数量达到上限时，“快照和复制”将停止拍摄新快照而不删除已拍摄的快照。
    • 按以下政策保留快照： 您可为具体时间范围配置保留规则。详情请参阅文件保留规则注意事项。
  5. 单击确定来保存设置。

注：

  • 更改保留设置将影响复制页面中的等效设置。

iSCSI LUN

若要为 iSCSI LUN 拍摄快照：

  1. 选择您要拍摄快照的 iSCSI LUN。
  2. 单击拍摄快照。
  3. 在描述中为要拍摄的快照输入描述。
  4. 从锁定下拉列表中，选择是/否以确定在触发自动删除时该快照是否忽视保留设置。
  5. 单击确定开始拍摄快照。

若要删除 iSCSI LUN 的快照：

  1. 选择您要删除其快照的 iSCSI LUN。
  2. 单击快照列表。
  3. 选择您要删除的一张或多线快照，并单击删除。（多选时请按住 Shift 或 Ctrl 键。）
  4. 您将看到确认信息。单击是确认删除。

若要编辑 iSCSI LUN 的快照：

  1. 选择您要编辑其快照的 iSCSI LUN。
  2. 单击快照列表。
  3. 选择您要编辑的快照，并单击编辑。
  4. 您可执行下列任一操作来编辑快照：
    • 在描述中更改快照的描述。
    • 如果应用程序一致性已启用，选择一致性状态。
    • 从锁定下拉列表中的是/否之间切换可确定在触发自动删除时该快照是否忽视保留设置。
  5. 单击确定来保存设置。

若要配置计划快照：

通过计划快照，快照和复制将在预告确定的时间和频率自动拍摄快照，此功能可减轻您手动拍摄快照的工作量。

  1. 选择 iSCSI LUN。
  2. 单击设置计划。
  3. 在计划选项卡中，勾选启用快照计划复选框，设置拍摄快照的时间和频率。若要取消此共享文件夹的计划快照，请取消勾选此复选框。
  4. 单击确定来保存设置。

若要为计划快照配置保留设置：

  1. 选择 iSCSI LUN。
  2. 单击设置计划。
  3. 选择保留选项卡。
  4. 配置保留设置。请选择下列任何项目：
    • 达到上限后删除旧快照：当快照数量达到上限时，最旧的快照将被新拍摄的快照覆盖。
    • 始终保留快照：当快照数量达到上限时，“快照和复制”将停止拍摄新快照而不删除已拍摄的快照。
    • 按以下政策保留快照： 您可为具体时间范围配置保留规则。详情请参阅文件保留规则注意事项。
  5. 单击确定来保存设置。
  6. 您将看到确认信息。再次单击确定以确认配置。

若要配置 iSCSI LUN 快照的一致性状态：

  1. 选择 iSCSI LUN。
  2. 单击设置计划。
  3. 选择应用程序选项卡。
  4. 勾选/取消勾选启用程序一致性快照复选框可在以下一致性状态之间切换：
    • 应用程序一致性：此一致性状态中的快照仅在 Windows server 或 VMware vCenter Server 根据您与 DSM 通信的操作环境安装了 Synology Snapshot Manager 后才能拍摄。触发 DSM 中的应用程序一致性快照后，应用程序将收到通知，并将内存中的所有数据储存到要对其拍摄快照的 LUN，从而确保数据的一致性。

      注：

      • 如果您的操作环境未安装 Synology Snapshot Manager，即便已勾选此复选框，启用程序一致性快照也将保持禁用。
    • 崩溃一致性：如果客户端之一未安装 Synology Snapshot Manager，您仅可捕捉 LUN 的崩溃一致性快照。应用崩溃一致性快照触发后，应用程序将不会获得通知以将内存中的数据存储到 LUN，因此无法保证数据一致性。
  5. 单击确定来保存设置。

注：

  • 更改一致性状态设置将影响复制页面中的等效设置。

文件保留规则

此选项为帮助您指定快照版本的数量上限以节省卷空间提供了一个理想的解决方案，但您仍需保留快照较长一段时间。

Synology 部署了 GFS（或三代）保留规则。您可为以下时间范围分配配置快照版本的最大保留数量。每小时、每天、每周、每月和每年。

“快照和复制”会保留为每个时间范围配置的最大快照数量。如果在一个时间范围内有多个快照版本，将仅保留最早的版本。例如，如果您将规则设置为 10 个每周快照，则“快照和复制”将为最近 10 周内的每个快照保留最早的快照版本 (如果一周内拍摄多个快照)。

此外，自拍摄最新快照后一个小时内的所有快照版本默认保存，以方便您查找和还原最近的快照版本。您还可锁定快照以防止其被您的保留规则自动删除。
