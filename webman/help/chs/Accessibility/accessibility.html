辅助功能

DiskStation Manager 通过内置的辅助功能，提供了听觉提示，为有视觉障碍的用户在导航和管理 DSM 服务/设置中提供引导。

DSM 辅助功能设置

若要使辅助功能发挥最佳作用，建议您建立以下工作环境：

  • 在本地计算机中安装 NVDA 屏幕阅读器 (仅对 Windows 环境可用)。
  • 安装最新的 Mozilla Firefox 网页浏览器。
  • 使用耳麦和耳机并调整音量以满足您的需要。
  • 在对焦模式下使用 NVDA (按 NVDA 键 + 空格进行模式切换)。默认 NVDA 键为插入。
  • 通过以下热键导航 DSM：
    热键 描述
    Tab 切换到另一个用户用户界面区域
    箭头键 切换到同一用户用户界面区域中的项目
    Enter 打开应用程序/窗口
    Esc 关闭应用程序/窗口

辅助功能目前在以下 DSM 区域中工作：

  • DSM 登录页面、桌面和任务栏
  • 主菜单
  • 控制面板
  • File Station

注：

要了解更多有关 NVDA 操作的信息，请参见用户指南。

DSM 键盘快捷方式指南

除辅助功能外，DSM 还提供了丰富的键盘快捷方式以进行顺畅、易操作的管理。按 Shift + ? 可找到快捷方式：

  • 桌面键盘快捷方式：窗口操作等热键
  • 组合键盘快捷方式：控制用户界面组件的热键 (如列表、按钮和搜索栏等)

若要暂停桌面相关的快捷方式，请取消启用桌面键盘快捷方式选项。
