DSM 手机

DSM mobile 可让 DSM admin（或属于 administrators 组的用户）管理 DSM 设置并用 iOS® (iPhone®、iPad® 或 iPod touch®) 或 Android®设备检查 DSM 信息。此 DSM mobile 帮助为您提供下列主题及资源，帮助您上手。

登录和注销

使用 iOS 或 Android 设备的网页浏览器连接到 DSM 后，您可以看到登录画面，可在其中输入 DSM 帐户信息。

若要登录 DSM mobile：

输入您的 DSM 用户凭据并点按登录。如果您要存储登录信息并在下次自动登录，请在登录之前打开记住我的登录信息开关。

若要注销 DSM mobile：

  1. 在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 的右上角点按选项按钮。
  2. 点按设置，并点按注销。

管理 DSM 用户

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按用户和群组来查看用户列表。

若要创建用户：

  1. 点按 +，然后输入新用户的用户名称、密码及电子邮件地址。
  2. 启动发送通知来在 DSM 用户的帐户创建时寄送通知信到他/她的电子邮件地址。
    注：发送通知前，请确定您已完成 Synology NAS 的通知设置。用您计算机的网页浏览器登录至 DSM 桌面版，并进入主菜单 > 控制面板 > 通知 > 电子邮件可修改设置。
  3. 点按完成以确认，或点按左箭头以取消。

若要快速查找用户：

在搜索字段中输入用户名。

若要修改用户信息：

  1. 点按用户名并运行下列任何操作：
    • 在名称或密码栏中修改用户名和密码。
    • 打开禁用可禁用用户。您可以选择稍后启用用户，这样做并不会遗失该用户的设置。
    • 打开管理员可将用户添加到 admin 群组。属于 admin 群组的任何用户皆拥有管理权限，可以使用 DSM mobile 或计算机的网页浏览器修改 DSM 设置。
  2. 在权限下，执行下列任何操作，然后点按返回：
    • 点按共享文件夹并点按共享文件夹的名称，修改用户访问权限：读取/写入：用户可以访问并更改此共享文件夹中的文件及子文件夹；只读：用户可以访问此共享文件夹中的文件及子文件夹，但是无法进行更改；禁止访问：用户无法访问此共享文件夹中的文件及子文件夹。
      注：如果发生权限冲突，权限优先将为：禁止访问 > 读取/写入 > 只读。
    • 点按应用程序来修改用户对 DSM 应用程序的访问权限。
  3. 点按完成以确认，或点按左箭头以取消。

管理共享文件夹

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按共享文件夹来查看 Synology NAS 上的共享文件夹列表。

若要创建共享文件夹：

  1. 点按 +，然后输入新文件夹的名称。
  2. 点按完成以确认，或点按左箭头以取消。

若要修改共享文件夹：

点按共享文件夹的名称并运行下列任何操作：

  • 在名称栏中修改共享文件夹的名称。
  • 点按用户权限并点按用户可修改其对共享文件夹的访问权限：读取/写入：用户可以访问并更改此共享文件夹中的文件及子文件夹；只读：用户可以访问此共享文件夹中的文件及子文件夹，但是无法进行更改；禁止访问：用户无法访问此共享文件夹中的文件及子文件夹。
    注：如果发生权限冲突，权限优先将为：禁止访问 > 读取/写入 > 只读。
  • 您还可点按群组权限为群组进行配置。
    注：群组权限的优先权高于用户权限。

启用服务

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按服务来启动或禁用 Synology NAS 服务。

若要管理 Synology NAS 的局域网服务：

  • 启用或禁用下列任何项目：
    • Windows 文件服务可让 Synology NAS 成为局域网中的 Windows (CIFS) 文件服务器。
    • Mac 文件服务可让 Synology NAS 成为局域网中的 Mac (AFP) 文件服务器。
    • NFS 服务可让局域网中的 Linux 客户端访问 Synology NAS 的数据。
    • Bonjour 打印机可让 Mac 客户端使用 Synology NAS 上连接的 USB 打印机进行打印。
    • 媒体服务器可让局域网中的 DLNA/UPnP 家用设备浏览并播放 Synology NAS 上的多媒体内容。
    • iTunes 可让局域网中的 iTunes 客户端浏览并播放 Synology NAS 上的音乐或视频。

若要管理 Synology NAS 的 Internet 服务：

  • 启用或禁用下列任何项目：
    • FTP 可让您的 Synology NAS 成为 FTP 服务器。
    • 网页服务可让您在 Synology NAS 上架设网站 (支持 PHP+MySQL)。
    • WebDAV 可让客户端程序通过 HTTP 通讯协议管理并编辑 Synology NAS 文件。
    • WebDAV (HTTPS) 可让客户端程序通过 HTTPS 通讯协议管理并编辑 Synology NAS 文件。
    • File Station 是网页界面的应用程序，可让您无须登录 DSM 即可通过自定义端口管理 Synology NAS 文件。
    • File Station (HTTPS)> 可让您通过 HTTPS 协议连接到 File Station。
    • Surveillance Station 是一个基于网页的应用程序，其可管理网络摄像机来保卫家庭或办公室环境的安全。
    • Audio Station 是一个基于网页的应用程序，可让您访问 Synology NAS 上的音乐库。
    • Photo Station 是整合了博客功能的在线相册 (存放在 Synology NAS 上)。
    • Download Station (BT/HTTP/FTP/NZB/RSS) 是基于 Web 的应用程序，可辅助您将 Internet 上的文件下载到 Synology NAS 上。
    • eMule 可启动 Download Station 的 eMule 支持功能。

注：

以上服务详情参看以下：

  • DSM 说明 (您可以使用计算机的网页浏览器登录 DSM 的一般版，并前往主菜单 > DSM 说明进行访问)
  • Synology DSM 用户指南 (可在 Synology 的下载中心获得)。

监控系统资源

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按资源监视器来监控 Synology NAS 的 CPU 及内存用量。

查看系统信息

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按系统信息来取得 Synology NAS 的状态总览。

若要查看 Synology NAS 的系统信息：

  1. 执行下列任何操作：
    • 点一下常规来查看 Synology NAS 的基本信息，包含序列号、型号、物理内存大小、DSM 版本、时间信息以及散热状态。
    • 点按网络来查看 Synology NAS 的网络设置以及各个网络接口的状态。
    • 点按存储空间来查看 Synology NAS 的存储空间用量。
  2. 点按右上角的刷新按钮来让系统信息保持最新状态。

固件

点按固件可查看并更新 DSM 版本。

若要配置 DSM 更新设置：

  1. 点按 DSM 升级。
  2. 选择以下选项之一：
    • 仅重要更新
    • 最新版本的 DSM 和所有更新
  3. 点按完成。

若要下载可用更新：

  1. 在状态中检查您的 DSM 版本。
  2. 如果您的 DSM 版本不是最新的，点按下载按钮 (该按钮将出现在状态区域下) 可升级 DSM。

管理 IP 封锁

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 上点按 IP 封锁可防止未授权登录。

若要管理 IP 封锁：

  1. 点按右上角的配置按钮。
  2. 打开启用自动锁定可启用 IP 封锁。
  3. 在登录尝试和几分钟内后面输入数字，系统将会封锁在设置分钟内登录失败超过尝试登录次数的 IP 地址。
  4. 您可打开启用封锁过期，并指定封锁天数，可在指定天数后解除封锁 IP 地址。如果您在此输入“0”，锁定的 IP 地址将永久锁定，直到您从锁定列表中将其删除。
  5. 单击完成。

若要将 IP 地址添加到允许/封锁列表：

  1. 如果您要将允许/封锁 IP 添加到列表，按住右上角的添加按钮，并点按允许列表或封锁列表。
  2. 输入 IP 地址后，点按完成。

管理套件中心

Synology 已将第三方或是 Synology 自行设计的应用程序整合到套件中，可在 Synology NAS 上安装并轻易启用。

若要管理套件中心：

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 上点按套件中心来查看套件信息、运行或停止套件，或更新至最新版本。

若要运行或停止安装的套件：

点按套件下面的切换开关可运行或停止操作。

若要查看套件：

点按套件可查看其信息。

若要更新套件：

  1. 点按更新选项卡可查看当前可用的更新版本。
  2. 点按套件右侧的按钮可更新到最新版本。

查看日志或当前连接

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 上点按日志来监控当前连接或查看 Synology NAS 服务日志。

欲监控当前连接：

  • 点按概述区域中的最新 [几个] 日志可查看最新访问 Synology NAS 服务的用户列表。

查看指定服务的日志记录：

  • 点按事件日志或文件传输日志区域中的任何项目可查看相关日志列表。

开通移动应用程序

DSM mobile 为您提供一个便利的位置来打开 iOS 或 Android 设备上安装的 Synology 移动应用程序。

若要使用 DSM mobile 开通 Synology 移动应用程序：

在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 的下半部点按下列任何应用程序：

  • DS photo (启动 Photo Station 服务后才可访问)
  • DS audio (启动 Audio Station 服务后才可访问)
  • DS cam (启动 Surveillance Station 服务后才可访问)
  • DS file (启动 WebDAV 服务后才可访问)

注：

  • 如需更多使用这些应用程序的相关信息，请参阅应用程序随附的帮助文件。

设置操作中的 Synology NAS
(重启、关机或哔声)

DSM mobile 可让您使用 iOS 或 Android 设备为 Synology NAS 运行重新启动或关机。此外，当您身边有多部 Synology NAS 时，您也可以要求您的 Synology NAS 播放哔声以进行快速识别。

若要设置 Synology NAS 动作：

  1. 在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板) 的右上角点按选项按钮。
  2. 执行下列任何操作：
    • 点按重新启动然后点按是以重启 Synology NAS。
    • 点按关机然后点按是以关闭 Synology NAS。
    • 点按找到我可要求 Synology NAS 播放哔声。点按确定停止发出哔声。

切换至 DSM 桌面模式

点按设置图标并选择桌面模式可切换到 DSM 的桌面模式。

使用存储管理器

存储管理器可让您快速查看系统卷和硬盘的使用和运行状况。

若要查看卷信息：

  1. 在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按存储管理器。
  2. 点按卷选项卡。
  3. 点按列表中的卷可查看其详细信息。

若要查看硬盘信息：

  1. 在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按存储管理器。
  2. 点按 HDD/SDD 选项卡。
  3. 点按列表中的硬盘可查看其运行状况和 S.M.A.R.T. 信息。

若要开始或停止 S.M.A.R.T. 检测：

  1. 在主页 (iPhone/iPod touch 或 Android 手机) 或左侧面板 (iPad 或 Android 平板设备) 上点按存储管理器。
  2. 点按 HDD/SDD 选项卡。
  3. 点按列表中的硬盘可查看其信息页面。
  4. 点按 S.M.A.R.T. 检测可开始 S.M.A.R.T. 检测或停止以停止它。
