Mac 文件服务

在控制面板 > 文件服务 > Win/Mac/NFS 中，您可以启用 Mac 文件服务，使得 DiskStation 成为一台 Mac 文件服务器 (AFP 服务器)。

若要启用 Mac 文件服务：

  1. 请勾选启用 Mac 文件服务，使得 DiskStation 成为一台 Mac 文件服务器 (AFP 服务器)。
  2. 如需要，为 Time Machine 设置一个备份文件夹。(详情请参阅以下部分。)
  3. 单击应用。

注：

  • 若要将数据从 Time Machine 备份至 DiskStation，请从下拉菜单中选择一个共享文件夹。
  • 其他服务所使用的共享文件夹无法用在 Time Machine 上。

启用传输日志

勾选这个选项，系统将会记录通过 AFP (所谓的苹果文件协议) 访问文件的记录，包括对文件及文件夹的创建、删除、读取、写入、移动及重命名。

高级设置

应用默认的 UNIX 权限：

启动此选项在上传或创建文件和文件夹时应用默认的 UNIX 权限。所应用的权限将与 UNIX umask 命令所应用的权限相同。启用此选项后，文件的 UNIX 权限为 644，而文件夹为 755。禁用此选项后，文件的 UNIX 权限为 666，而文件夹为 777。默认 umask 值为 022。

注：

启动此选项可能会导致不同协议之间发生权限不一致的问题。为避免不一致问题，建议您保持禁用此选项。

在断开连接后立即释放资源：

立即启用此选项可在 AFP 服务器连接意外断开连接时释放资源。如果您在使用 Time Machine 时收到“卷在使用”的错误信息，请启用此选项。

若要启用 Bonjour 打印机共享：

勾选启用 Bonjour 打印机共享来允许 Mac 客户端浏览并找到 DiskStation 上连接的打印机。

若要通过 Finder 上的 Spotlight 搜索文件内容：

文件夹全面建索并装载到 Mac 计算机后，您可用 Finder 上的 Spotlight 来搜索所含的文件及其内容。

  1. 请进入控制面板 > 索引服务 > 文件索引 > 索引文件夹列表 > 创建来按文件类型对特定文件夹及其所含文件进行索引。
  2. 单击确定以执行索引。
  3. 将索引文件夹装载到您的 Mac 计算机。
  4. 在您的 Mac 计算机上，单击 Finder 左侧面板上需装载的文件夹，使用 Finder 中的 Spotlight 可搜索所含的文件。

注：

  • 若要在装载的文件夹内进行搜索，请使用 Finder 中的 Spotlight。Mac 桌面上的 Spotlight 仅可搜索本地文件。
  • 在 Mac OS X 10.9 及以上版本，Finder 上的 Spotlight 支持按标签名称和类别在装载的文件夹中进行搜索。
