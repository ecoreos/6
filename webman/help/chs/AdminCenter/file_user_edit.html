编辑用户

选择您要编辑的用户帐户后（可在搜索栏中输入关键字查找），双击它，或按编辑按钮修改帐户数据。

注：
  1. 如果您禁用某个帐户，该帐户将停止运作。
  2. 配额须为大于或等于 0 GB 的整数，且小于硬盘/卷容量的大小。\配额值为 0 意味着有效配额将由群组限制决定。
  3. 如果服务器上的多个卷/共享文件夹，您可为每个卷/共享文件夹上的用户设置配额。
  4. “通知” 页面上的电子邮件信息必须正确设置，以使系统在创建或修改帐户信息后通过电子邮件通知用户。
  5. 设置用户速度限制时，如果该用户已经应用了群组速度限制，您将在结果下方看到数值。
  6. 指定给用户的速度限制将不包括在用户所属群组的速度限制中。
