创建群组

  1. 请进入控制面板 > 群组。
  2. 单击创建。此操作将启动用户创建向导。
  3. 输入用户组名及群组说明。（请参阅以下区块来了解更多信息）。单击下一步。
  4. 然后按照向导的剩余步骤完成创建群组。
  5. 单击应用完成。

命名规则：

  • 群组名称有大小写区分，可包含 1 到 32 个 Unicode 字符，但不可为下列符号︰

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • 第一个字符不可为减号或空格，最后一个字符不可为空格。
  • 群组说明有大小写区分，最多可输入 64 个可显示的 Unicode 字符。
