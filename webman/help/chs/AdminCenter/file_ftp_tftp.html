TFTP

TFTP (Trivial File Transfer Protocol) 是一个简单文件传输协议，它一般用于在无需验证的情况下传输配置或启动文件。启用 TFTP 服务后，DiskStation 就能将启动加载器文件传送到客户端设备。

若要配置 TFTP 设置：

  1. 请进入控制面板 > 文件服务 > TFTP，并选择启用 TFTP 服务。
  2. 于 TFTP 根文件夹栏中，指定 TFTP 客户端在 DiskStation 上可访问的文件夹。
  3. 单击高级设置配置以下设置：
    • 启用 TFTP 文件传输日志：通过 TFTP 保持传输数据跟踪。
    • TFTP 客户端权限：决定 TFTP 客户端权限：
      • 只读：TFTP 客户端只能查看根文件夹的内容。
      • 可写：TFTP 客户端可修改根文件夹的内容。
    • 允许的客户端：选择允许通过 TFTP 连接的客户端：
      • 所有连接：所有客户端都可通过 TFTP 进行连接。
      • 仅允许以下 IP 地址范围：只有特定 IP 范围内的客户端可通过 TFTP 进行连接。
    • 超时：如果 TFTP 服务器 (即您的 DiskStation) 在特定时间内仍无响应，则 TFTP 连接将停止。
  4. 单击应用以保存该设置。
