USBCopy

您可进入控制面板 > 外部设备 > USBCopy/SDCopy USBCopy 功能选择目标文件夹。

若要启用 USBCopy：

  1. 勾选启用 USBCopy。
  2. 请从下拉菜单中选择目标共享文件夹。

在您将一个 USB 设备（例如数码摄像机）链接到 Synology DiskStation 前方面板的 USB 端口后，您可以按压前面板的 Copy 按钮来将 USB 设备上的数据复制到您指定在 Synology 产品上的目的地共享文件夹。如果 USBCopy 目标文件夹被删除，则系统会自动以 public 共享文件夹替换它。

限制

  1. 仅支持格式为 FAT、NTFS、ext3 和 ext4 的 USB 设备。
  2. USB 设备必须插入 DiskStation 设备前方面板的 USB 端口。
  3. 只有 admin 或 administrators 群组中的用户有适当的访问权限对 USB 设备执行备份任务。
  4. 目标共享文件夹的可用空间必须大于 USB 设备的已使用空间。

SDCopy*

进入控制面板 > 外部设备 > USBCopy/SDCopy 为 SDCopy 功能选择目标文件夹。

若要启用 SDCopy：

  1. 勾选启用 SDCopy。
  2. 请从下拉菜单中选择目标共享文件夹。

在您将一个 SD 设备插入 Synology 产品前方面板的 SD 插槽时，您可以按压前方面板的 Copy 按钮来将 SD 设备上的数据复制到您指定的目标共享文件夹。如果 SDCopy 目标文件夹被删除，系统将会自动以 public 共享文件夹替换它。

限制

  1. 仅支持格式为 FAT、NTFS 和 ext3 的 SD 设备。
  2. SD 设备必须插入 Synology 产品前方面板的 SD 插槽。
  3. 只有 admin 或 administrators 群组中的用户有适当的访问权限对 SD 设备执行备份任务。
  4. 目标共享文件夹的可用空间必须大于 SD 设备的已使用空间。

____
* 仅适用于特定机型。
