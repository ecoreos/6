管理高级共享文件夹权限

控制面板 > 共享文件夹 > 编辑 > 高级页面中包括以下选项，可进一步微调共享文件夹的访问权限。

高级共享权限

高级共享权限提供了一个额外的控制层，管理共享文件夹的访问权限。启用后，只有当用户或组被授予高级共享权限以及 Windows ACL 权限后（位于共享文件夹 > 编辑 > 权限），则用户或组才可查看或修改共享文件夹的内容。

若要启用高级共享权限：

  1. 勾选启用高级共享权限复选框。
  2. 单击高级共享权限可修改共享文件夹的高级共享权限。

注：

  • 当用户通过以下文件服务访问共享文件夹时，将应用高级共享权限。Windows 文件共享、Apple 文件共享、File Station、FTP 和 WebDAV。

高级设置

如需要，您可对通过 File Station、FTP 或 WebDAV 对访问共享文件夹的用户设置进一步的限制。

  • 禁用浏览目录内容：启动此选项可限制用户查看共享文件夹中的内容。
  • 禁用修改现有文件内容：启动此选项可限制用户移动、删除或修改共享文件夹中的文件。请注意，用户仍可查看、下载/上传、复制或解压缩共享文件夹中的内容。
  • 禁用文件下载：启动此选项可限制用户下载共享文件夹中的内容。

注：

上述限制不适用属于 administrators 群组的用户。
