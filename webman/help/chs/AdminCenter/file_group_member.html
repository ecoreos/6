添加/删除群组成员

您可以将 DiskStation 用户加到您创建的群组，或从群组中删除。

若要加入群组成员：

  1. 请选择您要添加用户的群组，然后单击编辑群组成员。
  2. 单击添加。
  3. 从用户列表中选择要加入的用户，然后单击确定。（若您想要一次选择多个用户，请按住“Shift”或“Ctrl”键并单击用户。）
  4. 单击完成来应用设置。

    5. 注：Administrators 群组中的成员将具有 DiskStation 的管理权限，包括 DSM/CIFS/FTP/AFP/WebDAV 应用程序。

若要删除中的群组成员：

  1. 请选择您想要删除用户的群组，然后单击编辑群组成员。
  2. 从用户列表中选择要删除的用户。
  3. 单击是。
  4. 单击完成来应用设置。

    5. 注：如果用户从 Administrators 群组中删除，这些用户将无法管理 DiskStation。
