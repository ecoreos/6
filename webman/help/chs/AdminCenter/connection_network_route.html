网络路由

DSM 为不同的网络配置需求提供多种网络路由选项。

若要分开同一局域网中不同网络接口的网络流量：

  1. 请进入控制面板 > 网络 > 高级。
  2. 在常规下，勾选当目标 IP 地址与传入接口的本地地址相同时，回复 ARP 请求。
  3. 单击应用来保存设置。

这会为每个网络接口建立默认的路由规则，让来自各接口的网络套件从其自己的相应物理网络接口发送出去。

此外，如果同一局域网中有两个以上的网络接口，启用此选项将分别传输各接口的网络流量。您可以为各接口应用独立的防火墙规则。

若要通过相应网关来路由各网络接口：

  1. 请进入控制面板 > 网络。
  2. 在常规下，勾选启用多网关。
  3. 单击应用来保存设置。

这会让网络中的各网络接口通过其相应的网关进行路由。

若要更改默认网关：

  1. 请进入控制面板 > 网络。
  2. 在网络接口下，单击服务顺序。
  3. 选择网关并单击上或下，来重新排列网关。
  4. 单击确定。

顺序顶部的网关将成为局域网的默认网关。

若要设置静态路由：

  1. 请进入控制面板 > 网络。
  2. 在静态路由下，单击创建。
  3. 在目的地网络中输入目的地 IP 地址。
  4. 在网络掩码中输入目的地的网络掩码。
  5. 在网关中输入目的地的网关地址。
  6. 在接口中选择您目的地的接口。
  7. 单击确定。

若要查看 IP 路由表：

  1. 请进入控制面板 > 网络。
  2. 在静态路由下，单击 IP 路由表。
  3. 从左上角的下拉列表中选择要查看其 IP 路由表的网络接口。
