路由器端口

如果您要让 Internet 服务（如 Google Docs 或 Pixlr）访问 DiskStation，可在控制面板 > 外部访问 > 高级中指定公共、外部主机名和端口号。

默认时，主机名为您的公用 IP 地址（或为控制面板 > 外部访问 > DDNS 中设置的主机名），而端口号为 5000（HTTPS 连接则为 5001）。但如果您有另一个主机名却未在 DDNS 页面中进行配置，或者您在设置端口转送规则时于路由器上指定了其它端口号，则需要输入主机名或路由器端口号以确保可成功访问共享到 Internet 服务的文件。

若要启用路由器端口配置：

  1. 请在主机名或静态 IP 栏中输入 DiskStation 的主机名或公用 IP 地址。
  2. 当您在 DSM（HTTP）或 DSM（HTTPS）栏中设置您路由器上 DSM 端口转发规则时，输入您指定的路由器端口号。
  3. 单击应用。
