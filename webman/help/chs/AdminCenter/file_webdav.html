WebDAV

WebDAV 是一个 HTTP 的扩充服务，可让用户编辑和管理存储在远程服务器上的文件。启用 WebDAV 服务后，支持 WebDAV 的客户端程序（如某些 Windows 应用程序、Mac OS Finder、Linux 资源管理器）能够远程访问 DiskStation，如同访问本地网络硬盘。

若要启用 WebDAV：

  1. 如果您要启用通过 HTTPS 的 WebDAV 连接，请勾选启用 WebDAV 或启用 WebDAV HTTPS 连接的复选框。WebDAV 默认端口号为 5005，而 WebDAV HTTPS 为 5006。
  2. 单击应用。

速度限制

启用速度限制后，您可控制特定用户或群组在通过 WebDAV 传输文件时的最大可用带宽。

若要启用速度限制：

  1. 如果您只要在特定时间启用速度限制，请勾选标示为启用速度限制或根据计划启用速度限制的复选框。此将应用至整个服务器，并覆盖服务器中的各用户和群组的设置。选择无速度限制可禁用所有用户和群组的速度限制设置。
  2. 单击速度限制设置可修改单个用户或群组的速度限制。您可为每个用户和群组选择下列其中一个选项：
    • 应用群组设置：速度限制设置将与用户所属群组的设置相同。
    • 无：当群组成员用此服务传输文件时不应用速度限制。
    • 恒定：一个固定速度限制常量将应用到用户或群组成员。可在左侧栏中指定上传与下载的速度限制。
    • 可变：可根据计划指定两种不同的速度限制并应用到用户或群组成员。单击自定义按钮可修改速度限制设置并设置计划。
  3. 单击应用。

注：

  • 要了解有关群组速度限制动态详情，请参阅此处。
  • 设置用户的速度限制后，结果区域会根据单个和群组设置显示用户的实际速度限制。例如，如果用户 “Achilles” 有 100 KB/秒的具体下载限制，但该用户所属的群组的下载限制为 500 KB/秒，则结果区域将显示 “0/500”。

高级设置

您可单击高级设置按钮来修改以下选项：

  • 启用 Anonymous WebDAV：通过 WebDAV 连接时，允许客户端使用 “anonymous” 用户名来访问共享文件夹。请确认您已经在共享文件夹页面中正确设置 “Anonymous FTP/WebDAV” 用户的访问权限。
  • 启用 DavDepthInfinity： 使用客户端工具时，此功能可让用户访问或删除文件而没有文件路径的长度限制。
  • 启用 WebDAV 日志：该选项用于记录 WebDAV 相关事件。单击查看日志，您就能从弹出的窗口中查看日志。

在不同的操作系统中使用 WebDAV

在启用 DiskStation 的 WebDAV 服务后，请按以下说明来开始使用 Windows、Mac、Linux 设备或某些 Synology 智能手机应用程序访问文件。

注：

  • 在连接 WebDAV 服务时，建议您使用支持 UTF-8 的 WebDAV 客户端以避免编码错误。

适用于 Windows

若您是 Windows 用户且想要使用 WebDAV 协议来访问 Synology 产品中的文件，建议您使用 BitKinex（一种驱动器映射应用程序，仅对非商业目的免费提供）。或者，您可也在 Internet 上找到更多其它的 WebDAV 客户端应用程序。

注：

  • 如果您想使用 Windows 映射网络驱动器来通过 WebDAV 访问 DiskStation 上的文件，请取得有效的 SSL 证书，并通过 HTTPS 连接访问 DiskStation。在 Windows 环境下使用 WebDAV 服务时，如果您想要变更用户帐户，您必须注销并重新登录 Windows 系统。

适用于 Mac

若您是 Mac 用户，您只需使用 Mac Finder 即可通过 WebDAV 连接到 DiskStation。

若要在 Mac 上使用 WebDAV：

  1. 请进入 Mac 电脑中的 Finder > 连接到服务器。
  2. 在服务器地址栏中，输入 DiskStation IP 地址或域名，开头为“http://”，且结尾为“:5005”（或是您在启动 WebDAV 服务时所指定的端口编号）。例如：http://pm.synology.com:5005/。
  3. 输入您的 DSM 用户名和密码。
  4. 单击连接。

注：

  • WebDAV 支持 Bonjour 播送功能。

对于 Linux

若您是 Ubuntu 或其他 Linux 的用户，请按以下说明来通过 WebDAV 访问 DiskStation 上的文件。

若要在 Ubuntu 使用 WebDAV：

  1. 请单击位置 > 连接至服务器。
  2. 从下拉菜单中选择 WebDAV（HTTP）或 WebDAV（HTTPS）。
  3. 输入必要信息，包含服务器位置、端口、文件夹以及用户帐户。单击连接来继续。
  4. 输入您的密码并单击连接。

若要在 Linux 通过命令行使用 WebDAV：

若您想使用命令行，您可以使用 cadaver（为一套支持命令行的 WedDAV 客户端软件）来设置访问。

使用 CalDAV 客户端程序来订阅日历

您可以使用计算机或移动设备上的 CalDAV 客户端程序来订阅 DiskStation 上存储的日历，并管理日历的事件、备忘录或待办事项。日历客户端程序包括：Sunbird（适用于 Windows、Mac、Linux）、Chandler（适用于 Windows、Mac、Linux）、Evolution（适用于 Linux）或 iCal（适用于 Mac、iOS 设备）。

若要启用 CalDAV：

  1. 请先启用 WebDAV 或 WebDAV HTTPS 连接。
  2. 勾选启用 CalDAV。
  3. 单击应用。

若要管理日历列表：

  1. 请单击查看日历列表。
  2. 在显示的窗口中，执行下列任何操作：
    • 若要创建新的日历，请单击添加，命名新创建的日历（例如，“cal1”）并指定其储存目的地（例如，“calendar”共享文件夹），然后单击确定。
    • 如果您要变更日历名称及储存目的地，请选择日历列表中的日历，单击编辑来进行变更，然后单击确定。
    • 如果您想要将日历从 DiskStation 中删除，请选择日历列表中的日历，然后单击删除。
  3. 单击关闭。

注：

  • 存放在 DiskStation 上的日历会以文件夹形式来显示。

若要使用客户端应用程序来订阅您的日历（此处以 Sunbird 为例）：

  1. 请下载 Sunbird 并安装在计算机。
  2. 进入文件 > 订阅远程日历。
  3. 在出现的窗口中，选择在网络上，然后单击下一步。
  4. 选择 CalDAV，在位置栏中输入 http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name（例如，“http://demo.no-ip.info:5005/calendar/cal1”），然后单击下一步。
  5. 为日历命名，并单击下一步。
  6. 输入您的 DSM 用户名称和密码来进行验证，单击确定，然后单击完成。

若要使用移动设备来订阅您的日历（此处以 iOS 设备为例）：

  1. 请进入 iOS 设备中的设置 > 邮件、联系人、日历。
  2. 在帐户下，进入添加帐户 > 其它，然后点按添加 CalDAV 帐户。
  3. 将 DiskStation 的 IP 地址（或域名）以及您的用户帐户信息输入在各列后，点按下一步。如果画面上显示信息，请点按继续。
  4. 点按高级设置，并运行下列操作：
    1. 在端口栏中输入 5005（或是您在启用 WebDAV 时指定的其他端口号）。
    2. 将帐户网址栏中的网址修改为 http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name（例如，“http://demo.no-ip.info:5005/calendar/cal1”）。
    3. 点按 CalDAV。
  5. 点按下一步来验证您的帐户信息，然后点按保存以完成帐户设置。
  6. 回到 iOS 设备的主屏幕，并打开日历来管理您的日历。

注：

  • 要了解更多管理日历事件、备忘录或待办事项的相关信息，请参阅客户端应用程序随附的帮助文件。
