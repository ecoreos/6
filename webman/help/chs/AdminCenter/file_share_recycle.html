启用回收站

当您启用共享文件夹中的回收站后，将创建一个子文件夹存储删除的文件。您可以将文件从回收站中还原。如果您要永久删除文件，可清空回收站。

启用回收站：

  1. 选择共享文件夹，然后单击编辑。
  2. 勾选启用回收站复选框。
  3. 如果您要仅让管理员可访问回收站，勾选仅限管理员访问。
  4. 单击确定以保存设置。

清空回收站：

单击清空回收站按钮可永久删除回收站中的全部文件。

清空所有回收站：

从操作下拉菜单中选择清空所有回收站可永久性删除所有回收站中的文件。

注：您可右键 #recycle 件夹以配置 File Station 中回收站文件夹的权限设置。选择属性，然后单击权限选项卡。
