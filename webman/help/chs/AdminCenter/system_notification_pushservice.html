推送服务

推送服务可让 DiskStation 在特定事件或错误发生时发送通知到您的移动设备或网页浏览器。

端口：

启用推送服务之前，请确认 DiskStation 可使用端口 8089 访问 Internet。

启用 Synology 提供的电子邮件通知：

  1. 勾选通过 Synology 的电子邮件服务器发送有关系统状态的通知。
  2. 在接收人栏内输入一个或多个电子邮件地址。系统通知将发送给此处输入的电子邮件地址。允许输入两个地址。多个电子邮件地址可用分号（;）分开。
  3. 单击应用，一条验证邮件会发送到接收人栏指定的地址。请按照验证邮件所列的步骤操作。
  4. 验证后，单击发送测试信息，然后单击确定发送测试通知。

若要启用移动设备通知：

  1. 请进入控制面板 > 通知 > 推送服务。
  2. 勾选启用移动设备通知复选框，然后单击应用。
  3. 在您的移动设备上安装并启动 DS finder，然后将您的 DiskStation 添加至收藏夹列表以将您的移动设备与 DiskStation 进行配对。
  4. 单击发送测试信息，然后单击确定发送测试通知。然后启动移动设备上的 DS finder 以检查是否收到该测试信息。

若要管理配对的移动设备：

  1. 请单击管理配对设备查看所有配对移动设备列表。
  2. 如果您想要取消任何设备的配对，选择并单击解除配对。（多选时请按住 Shift 或 Ctrl 键。）在您再次将它们配对之前，不会有任何通知发送给它们。

若要启用浏览器通知：

  1. 勾选启用移动设备通知，然后单击应用激活此项功能。
  2. 单击启用浏览器通知让您的 DiskStation 与浏览器配对。
  3. 在出现的窗口中执行下列任何操作：
    • 对于 Chrome:单击立即安装来安装 Chrome extension，如果您尚未登录 Google 帐户，请立即登录。
    • 对于 Safari:单击允许来授权通知服务。
  4. 当您重新导向回浏览器通知页面后，单击已配对设备选项卡以查看已配对设备或取消 DiskStation 与浏览器的配对，或单击事件选项卡以查看过去七天的浏览器通知历史。

注：

Chrome（35 或更新版本）及 Safari（OS X v10.9 或更新版本）都支持浏览器通知服务。
