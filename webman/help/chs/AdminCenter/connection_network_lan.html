局域网/Internet

根据 DiskStation 的型号和网络设置，您可进入控制面板 > 网络 > 网络接口来配置局域网或 Internet 的 DHCP 设置。DHCP 是管理自动 IP 配置的网络协议。如果您希望 DHCP 指派 IP 配置给系统使用，请单击编辑并选择 IPv4 选项卡下的自动获取网络配置（DHCP）。

如果您使用的是网络服务供应商提供的 IP 地址，请选择使用手动配置并手动输入 IP 地址和子网掩码。如果您不知道 DNS 服务器或默认网关的设置，则保持该栏空白。

注：

  1. 为了确保系统所发出的电子邮件通知可以顺利由 SMTP 服务器送达系统管理员的电子邮件邮箱，请正确填写 DNS 服务器的 IP 地址。
  2. Jumbo Frame 是“最大传输单元”（MTU）大于标准 1500 比特的以太网络框架，可让以太网络传输大文件更有效率。只有在 Gigabit 网络环境下才能启用此功能。如果要确保 Jumbo Frame 正常运作，网络上所有访问 DiskStation 的计算机及设备必须支持此功能，并且也使用相同的 MTU 值。

VLAN

当网络环境需要 VLAN 设置时，您可在“编辑”窗口的 IPv4 选项卡下启用 VLAN 选项并为网络环境输入 VLAN ID。

注：

  • 请确认您的网络交换器已正确配置，否则可能无法访问 DiskStation 的服务。
  • 如果您在修改上述设置后仍无法连接到 DiskStation，请按 DiskStation 上的 RESET 按钮来恢复默认设置。

IPv6 设置

DiskStation 设备支持 IPv4/IPv6 双重架构机制。表示 IPv4 和 IPv6 能同时运作于 DiskStation 之上。IPv6 的默认设置为自动。选择已有的网络接口，单击编辑按钮并进入 IPv6 选项卡以更改设置。

若要配置 IPv6 设置：

  1. 请进入控制面板 > 网络 > 网络接口。
  2. 请选择已有的网络接口并单击编辑。
  3. 在 IPv6 选项卡下，从 IPv6 设置中选择设置类型：
    • 自动：您的 DiskStation 会根据 LAN 路由器通告 (RA) 自动生成 IPv6 地址。
      • 无状态模式：允许连接到路由器的设备自动配置 IPv6 地址而无需设置 IPv6 DNS 服务器。
      • 无状态 DHCPv6 模式：允许连接到路由器的设备通过 DHCPv6 自动配置 IPv6 地址并获取 DNS 信息。
      • 有状态模式：允许连接到路由器的设备通过 DHCPv6 获取 IPv6 地址和 DNS 信息。在此模式中，您必须启用 DHCPv6 服务器并设置其 起始 IP 地址和结束 IP 地址。
    • 手动：输入 IPv6 地址、前缀长度和默认网关等信息来手动设置 IPv6。
    • 关闭：禁用 IPv6 支持。

在 IPv6 完全取代 IPv4 之前，您需通过 IPv6 转换机制来让拥有 IPv4 地址的主机连接到 IPv6 服务。请在不同的 IPv6 转换机制之间选择：

  • 6in4：通过隧道代理连接到 IPv6 网络。您需填入 IPv6 地址、前缀长度、前缀和隧道代理 IPv4 地址。此操作需要公共 IPv4 地址。
  • 6to4：通过使用 anycast 的隧道代理连接到 IPv6 网络。此操作需要公共 IPv4 地址。
  • DHCP-PD：通过 LAN 中的前缀授权从 DHCP 服务器获取 IPv6 前缀。此操作需要 ISP 支持，且仅在路由器模式中工作。
  • IPv6 Tunneling (TSP)：请进入控制面板 > 网络 > 常规配置此选项。

802.1X

802.1X 是由 IEEE 为用户连接网络而开发的验证标准。802.1X 在网络中使用，以对客户端进行验证，不允许未通过验证的客户端访问受保护的网络。

您可让 Synology NAS 连接到受 802.1X 验证保护的网络。

若要启用 802.1X 验证：

  1. 请进入控制面板 > 网络 > 网络接口。
  2. 选择有线网络接口（即局域网）并单击编辑。
  3. 进入 802.1X 选项卡，并勾选启用 802.1X 验证。
  4. 从验证下拉菜单中选择下列协议之一：如果您不确定最适用的协议，请选择自动。Â Â Â Â 
    • 受保护的可扩展验证协议 (PEAP)使用 CA 认证 (可选) 建立安全连接。此安全连接将被内部验证使用。
    • Tunneled TLS (TTLS)：使用 CA 认证 (可选) 建立安全连接。此安全连接将被内部验证使用。
    • TLS：使用 TLS (传输层安全) 进行验证。
  5. 如果您选择 TTLS 或 PEAP，从操作下拉菜单中选择内部验证协议之一。
    • MS CHAP v2：验证信息将以 Microsoft CHAP v2 加密。
    • GTC：验证信息将使用 GTC (通用令牌卡) 进行加密。
  6. 输入您的用户名及密码。
  7. 如需要，单击导入，然后单击浏览可找到并导入从 Radius server (如 ca.crt) 导出的证书。
  8. 单击确定来保存设置。

注：

  • 若要为无线网络配置 802.1X 验证，请进入控制面板 > 无线 > Wi-Fi。
  • 验证设置必须与设置 Radius Server 相同。要了解更多信息，请联系 Radius Server 管理员。

无线局域网*

当 DiskStation 用作无线路由器时，您可以在控制面板 > 网络 > 网络接口中配置以下无线局域网设置。

IPv4 设置

IP 地址和子网掩码

您可以在此设置 DiskStation 的 IP 地址和子网掩码。

DHCP Server

勾选 启用 DHCP Server 选项可让 DiskStation 将 IP 地址分配给连接本地网络（由 DiskStation 提供）的客户端设备。

PPPoE 中继

启用 PPPoE 中继可让连接到 DiskStation 的设备来通过 PPPoE 连接到 Internet。

配置为 UPnP Internet 网关

启用 UPnP 服务可让 DiskStation 成为 UPnP Internet 网关设备，来使所有连接到 DiskStation 本地网络的设备能够通过 UPnP 配置端口转送设置。

IPv6 设置

您可从带 6in4、6to4 或 DHCP-PD 的 LAN/Internet 获取 IPv6 地址和前缀。通过此信息，您可在此为 DiskStation 配置 IPv6 路由选项。

  • IPv6 地址：输入从 LAN/Internet 获取的 IPv6 地址。
  • 前缀长度：输入从 LAN/Internet 获取的 IPv6 地址的前缀长度，一般为 64。
  • 主 DNS：让设备连接到路由器的主 DNS 服务器。在无状态模式中需要设置此项。
  • 从 DNS：让设备连接到路由器的从 DNS 服务器。在无状态模式中需要设置此项。

有三个路由器通告 (RA) 模式可供选择：

  • 无状态模式：允许连接到路由器的设备自动配置 IPv6 地址而无需设置 IPv6 DNS 服务器。
  • 无状态 DHCPv6 模式：允许连接到路由器的设备通过 DHCPv6 自动配置 IPv6 地址并获取 DNS 信息。
  • 有状态模式：允许连接到路由器的设备通过 DHCPv6 获取 IPv6 地址和 DNS 信息。在此模式中，您必须启用 DHCPv6 服务器并设置其 起始 IP 地址和结束 IP 地址。

DHCP 客户端

您可在此查看 DHCP 客户端列表及其网络配置（如 MAC 地址、IP 地址、主机名以及剩余的 DHCP 租用时间）。

您可过滤 IPv4/IPv6 客户端下拉菜单以分别显示使用 IPv4 和 IPv6 的客户端。选择客户端并单击添加到地址保留以将客户端添加到地址保留列表。

DHCP 保留

如果您要让客户端在 DHCP 租用延期内始终收到相同的 IP 地址，可以将该客户端添加到地址保留列表中。

您可过滤 IPv4/IPv6 地址保留下拉菜单。IPv4 地址可按 MAC 地址识别，而 IPv6 地址则按 DUID 识别。

若要将客户端添加到地址保留列表：

  • 选择客户端并单击添加。
  • 指定客户端的 MAC 地址或 DUID、您要指派给该客户端的 IPv4/IPv6 地址以及该客户端主机名。
  • 单击确定来保存设置。
