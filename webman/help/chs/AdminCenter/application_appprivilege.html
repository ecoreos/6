权限

用户和群组可查看和编辑具体服务和应用程序的权限。DSM 提供了灵活的设置来控制应用程序权限，从而能够轻松管理用户和群组对不同应用程序的访问。

若要修改用户和群组的应用程序权限：

  1. 请进入控制面板 > 权限。
  2. 请选择要修改的应用程序。单击编辑。
  3. 单击用户来查看单一用户的权限，或单击群组来查看群组的权限。
  4. 勾选允许/拒绝来允许访问或拒绝用户或群组访问所选的应用程序。根据应用程序，您可能还可选择按 IP，基于 IP 地址来允许/拒绝访问。
  5. 单击确定来保存更改。

若要查看有不同网络 IP 地址的用户的应用程序权限：

  1. 请进入控制面板 > 权限。
  2. 单击权限查看器。
  3. 从用户名选择用户，然后在 IP 地址中输入您要查看权限的 IP 地址。

若要为所有用户设置默认的应用程序权限：

  1. 请进入控制面板 > 权限。
  2. 请选择要修改的应用程序。单击编辑。
  3. 单击默认权限并勾选向任何人授予此权限。
  4. 单击确定来保存更改。

注：

  • admin 和 guest 帐户的权限无法在此修改。
  • 遇到访问权限冲突时，权限设置的优先顺序为：拒绝 > 允许。
  • 并非所有的套件和服务都支持按 IP 访问权限设置。
