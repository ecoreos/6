共享文件夹

共享文件夹是 DiskStation 基本的目录结构，您可在其中存储文件和文件夹。存储文件前，您必须先在 DiskStation 的 DSM 上创建至少一个共享文件夹。存储在共享文件夹的数据可根据可自定义的访问权限，设为不公开，或与特定用户或用户群组共享。

以下是部分应用程序、服务或套件安装或启用后自动创建的默认共享文件夹列表。

名称 描述
public DiskStation 安装后，会自动创建 public 文件夹。*
web web 文件夹是架设于 DiskStation 的网站内容默认存储位置。Web Station 启用后，会自动创建此共享文件夹。
photo photo 文件夹用于存储您要与 Photo Station 共享的照片和视频。Photo Station 或 DLNA/UPnP 媒体服务器启用时，会自动创建此共享文件夹。
music music 文件夹用于存储您要用 Audio Station 聆听的音乐。Audio Station 或 DLNA/UPnP 媒体服务器启用时，会自动创建此共享文件夹。
video video 文件夹用于存储您要使用 DLNA/UPnP 设备浏览的视频文件。Video Station 或 DLNA/UPnP 媒体服务器启用时，会自动创建此共享文件夹。
surveillance surveillance 文件夹用于存储 Surveillance Station 录制的影片。Surveillance Station 启用后，会自动创建此共享文件夹。此共享文件夹的权限为只读，且仅有系统管理员可以访问。
home home 文件夹为每位用户提供私密空间，让他们放置只有自己可以访问的数据。用户主目录功能启用后，会自动创建此共享文件夹。
homes homes 文件夹包含所有用户的 home 文件夹。用户主目录功能启用后，会自动创建此共享文件夹。仅系统管理员可以访问并管理所有用户的 home 文件夹。
NetBackup 网络备份启用后，会自动创建 NetBackup 文件夹。
usbshare 当您将外接 USB 硬盘连接到 DiskStation 的 USB 端口后，会自动创建 usbshare[编号] 文件夹。*
satashare 当您将外接 eSATA 硬盘连接到 DiskStation 的 eSATA 端口后，会自动创建 satashare 文件夹。*

注：

  • 您只能在简单模式下编辑用户的共享文件夹访问权限。如果您要创建用户组或管理群组对共享文件夹的访问权限，请进入控制面板（高级模式） > 群组。

若要创建共享文件夹：

  1. 进入控制面板（简单模式） > 共享文件夹然后单击左侧面板上的 + 图标。
  2. 指定名称并选择在其中创建共享文件夹的卷。
  3. 如需要，可勾选启用回收站。如果启用此选项，则当共享文件夹中的文件被删除后，这些文件将被移动到名为 #recycle 的文件夹中。访问回收站的权限仅限属于 administrators 群组的用户。
  4. 勾选或取消勾选每个用户的适当复选框来自定义共享文件夹的访问权限：
    • 可读写：用户可以访问并变更此共享文件夹中的文件及子文件夹。
    • 只读：用户可访问此共享文件夹中的文件及子文件夹。
    • 禁止访问：用户无法访问此共享文件夹中的文件或子文件夹。
  5. 单击应用来保存设置。

注：

  • 共享文件夹名称不分大小写，最多可以包含 1-32 个字符。但名称不得包含以下字符：
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • 以下名称保留给系统所用：
    .、..、global、home、homes、printers、public、satashare、surveillance、usbbackup、usbshare
  • 共享文件夹名称的第一个字符不可为减号及空白，最后一个字符不可为空白。
  • 遇到访问权限冲突时，权限设置的优先顺序为：禁止访问 > 读取/写入 > 只读。
  • 创建新的共享文件夹时，若属于 administrators 组的用户的权限设置为禁止访问，则该用户只能在控制面板 > 共享文件夹中看到此共享文件夹。
  • 您可右键 #recycle 件夹以配置 File Station 中回收站文件夹的权限设置。选择属性，然后单击权限选项卡。

若要编辑共享文件夹：

  1. 请进入控制面板（简单模式） > 共享文件夹。
  2. 在左侧面板中选择要编辑的共享文件夹。
  3. 编辑该共享文件夹的常规信息，如名称、位置和用户访问权限。
  4. 如果已勾选启用回收站，您可单击清空回收站来永久删除回收站中所有已删除文件。
  5. 单击应用来保存设置。

若要删除共享文件夹：

  1. 请进入控制面板（简单模式） > 共享文件夹。
  2. 在左侧面板中选择要删除的共享文件夹。
  3. 单击左侧面板中的垃圾桶图标来删除所选共享文件夹。

____
* 仅适用于部分机型。
