HTTP 服务

除了预设的 80 端口，您可以添加其他端口供 Photo Station 和 Web Station 使用。

添加额外的 HTTP 端口号：

  1. 勾选添加额外的 HTTP 端口。
  2. 在栏中输入端口号。您可以额外添加端口号（1 到 65535）供 HTTP 连接使用。

下列端口不可使用：

20、21、22、23、25、110、137、138、139、143、199、443、445、515、543、548、587、873、993、995、3306、3689、5000、5001、5005、5006、5335、5432、6881、8080、7000、7001、8081、9997、9998、9999、50001、50002、eMule 默认端口：4662 (TCP)、4672 (UDP) 和 FTP 默认端口范围：实际范围视不同的型号而定。

    注：
  1. HTTP 服务使用的端口号不可与 FTP/BT/PT 服务所使用的端口号一样。
  2. HTTP 与 HTTPS 的端口号不可重复。
