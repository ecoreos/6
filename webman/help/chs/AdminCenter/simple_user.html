用户

通过灵活的用户管理选项，您可以为家庭或企业的每个成员创建用户帐户，并为每个人定义共享文件夹访问权限。

若要创建用户：

  1. 进入控制面板（简单模式） > 用户然后单击左侧面板上的 + 图标。
  2. 为用户指定名称和密码。
  3. 指定新用户属于的群组。
  4. 为新用户指定合当的共享文件夹访问权限。遇到访问权限冲突时，权限设置的优先顺序为：禁止访问 > 读取/写入 > 只读。
    • 预览：根据当前设置及用户所属的群组，此列将显示用户的实际共享文件夹访问权限。
    • 可读写：用户可以访问并变更相应共享文件夹中的文件及子文件夹。
    • 只读：用户可访问相应共享文件夹中的文件及子文件夹。
    • 禁止访问：用户无法访问相应共享文件夹中的文件或子文件夹。
  5. 单击应用来保存设置。

    6. 用户名称限制：

    • 用户名称不分大小写，可包含 1 到 64 个 Unicode 字符，但不可为下列字符：
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • 第一个字符不可为减号或空格，最后一个字符不可为空格。
    • 以下用户名称保留给系统所用：
      mailer-daemon、postmaster

    密码限制：

    • 密码有区分大小写，最多可输入 127 个可显示的字符，包含英文字母、数字、符号和空格键等。

若要编辑用户：

  1. 请进入控制面板（简单模式） > 用户。
  2. 在左侧面板中选择要编辑的用户。
  3. 编辑该用户的常规信息，如名称、密码、群组和共享文件夹访问权限。
  4. 如果您要停用该帐户，请勾选禁用此用户帐户。
  5. 单击应用来保存设置。

若要删除用户帐户：

  1. 请进入控制面板（简单模式） > 用户。
  2. 在左侧面板中选择要删除的用户。
  3. 单击左侧面板中的垃圾桶图标来删除所选用户。
