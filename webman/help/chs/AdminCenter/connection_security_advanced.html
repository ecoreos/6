高级

您可以在控制面板 > 安全性 > 高级中为 DSM 浏览作业阶段修改以下高级设置：

HTTP 压缩

您可为 DiskStation 启用 HTTP 压缩。通过降低网络流量的带宽，可增加加裁网页的速度。

若要启用 / 禁用 HTTP 压缩：

  • 勾选 / 取消勾选 HTTP 压缩 (在控制面板 > 安全性 > 高级)。

TLS / SSL 配置文件

您可选择 HTTPS 加密连接的安全级别。它提供以下 3 层等级：

  • 现代兼容性 - 仅与最新浏览器和移动设备兼容。
  • 中等兼容性 - 最高和最低安全级别之间。它是系统默认值。
  • 回溯兼容性 - 不安全，但支持大多数浏览器和移动设备。
