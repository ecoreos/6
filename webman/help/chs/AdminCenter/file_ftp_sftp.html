SFTP

SFTP 是文件传输协议，是对安全外壳（SSH）协议的扩展。SFTP 仅需要一个 TCP 端口号。此外，私钥和公共密钥可用于验证用户且无需密码。但由于加密，传输速度较慢且消耗的系统资源较多。

若要启用 SFTP 服务：

  1. 请单击启用 SFTP 服务。
  2. 单击应用。

端口号码

设置 SFTP 服务的端口号。

注：

SFTP 服务的默认端口号为 22。您可以将其设置为 1 至 65535，但不可为下列编号：
20、21、23、25、80、110、137、138、139、143、199、443、445、515、543、548、587、873、993、995、3306、3689、5000、5001、5005、5006、5335、5432、6881、8080、7000、7001、8081、9997、9998、9999、50001、50002 以及 eMule 默认端口：4662（TCP）、4672（UDP）
