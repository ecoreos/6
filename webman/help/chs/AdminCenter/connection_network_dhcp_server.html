DHCP Server

DiskStation 用作 DHCP 服务器并将动态 IP 地址指派给本地网中的 DHCP 客户端。

若要启用 DHCP 服务器：

  1. 进入控制面板 > DHCP Server > 网络接口。
  2. 选择向其指派 IP 地址的网络接口 (例如：LAN 1) 并单击编辑。
  3. 在弹出窗口中，单击 DHCP Server 选项卡并选择启用 DHCP 服务器。
  4. 指定以下 DHCP 设置：
    • 地址租用时间 (分钟)
    • 主 DNS
    • 从 DNS (可选)
    • 域名 (可选)
    • 启用 Web 代理服务器自动发现 (可选)
  5. 设置至少一个子网。要设置子网，单击添加可指定以下信息：
    • 起始 IP 地址
    • 结束 IP 地址
    • 子网掩码
    • 网关
  6. 单击创建可在列表中显示创建的子网。
  7. 创建足够的子网后，记得启用至少一个子网。
  8. 单击确定。

注：

  • 您可按需要数量创建子网。
  • 如果与 DHCP 中继服务器协调，您的 DiskStation 作为 DHCP 服务器可将 IP 地址分派给另一个子网中的设备。

DHCP 客户端

您可查看 DHCP 客户端列表及其网络配置 (如 MAC 地址、IP 地址、主机名以及剩余的 DHCP 租用时间和保留状态)。

  1. 进入控制面板 > DHCP Server > 网络接口并选择一个所需的接口。
  2. 单击编辑 > DHCP 客户端可找到客户端和详情。

若要管理 DHCP 客户端的 IP 地址：

您可让客户端在 DHCP 租用延期内始终接收相同的 IP 地址并管理这些客户端。

  • 若要添加 DHCP 客户端：单击添加可将 IP 地址指派给客户端设备。
  • 若要保留 IP 地址：单击锁定图标可为当前设备保留 IP 地址。
  • 若要取消保留：单击锁定图标可在地址租用时间过后让当前 IP 地址被另一个设备所用。

注：

如果设备有 IPv4 和 IPv6 地址，您可为设备保留一个 IPv4 地址和一个 IPv6 地址。

若要搜索 DHCP 客户端：

  1. 单击右上角的漏斗图标。
  2. 选择以下搜索条件之一：
    • 全部：列示所有的 DHCP 客户端和 DHCP 保留。
    • DHCP 客户端：列示所有的 DHCP 客户端及保留或未保留的 IP 地址。
    • DHCP 保留地址：列示所有保留的 IP 地址。保留的 IP 地址可能尚未与任何设备配对。

DHCP 选项

指定 DHCP 选项后，作为 DHCP 服务器的 DiskStation 可将相关的信息发送到特定子网的 DHCP 客户端。

若要为子网指定 DHCP 选项：

  1. 进入控制面板 > DHCP Server > 网络接口。
  2. 请选择所需的网络接口并单击编辑。
  3. 进入DHCP Server，并进入子网列表。
  4. 执行以下操作之一：
    • 若要为已有的子网指定 DHCP 选项：请选择所需的接口并单击编辑。
    • 若要创建子网并指定其 DHCP 选项：单击添加并指定子网设置。
  5. 选择所需的 DHCP 选项；指定相应的值，单击空白栏以进行编辑。
  6. 单击确定。

供应商

不同供应商的 DHCP 客户端可能需要其它 DHCP 信息。通过为供应商设置 DHCP 选项，您的 DiskStation 可向客户端提供此类详情。

若要了解各供应商支持的 DHCP 选项，请联系特定供应商以获得详情。

若要添加供应商：

  1. 请进入控制面板 > DHCP Server > 供应商。
  2. 单击创建可指定供应商及其 DHCP 选项：
    • 名称：指定供应商的名称
    • 代码：指定 DHCP 选项的数字代码。
    • 值：指定选项值。
    • +：添加 DHCP 选项。
    • -：删除 DHCP 选项。
  3. 单击创建完成设置。

若要为供应商编辑 DHCP 选项：

  1. 请进入控制面板 > DHCP Server > 供应商。
  2. 选择所需的供应商并单击编辑可管理其 DHCP 选项。
  3. 单击确定。

PXE

您的 DiskStation 可用作 DHCP 服务器以执行 PXE (预启动执行环境) 的设置。预启动执行环境 (PXE) 是一个网络环境，客户端可使用网络接口而非数据存储设备 (如硬盘) 来启动其设备。

当执行 PXE 后，启动加载器可存储在 DiskStation 或远程 TFTP 服务器上，而客户端设备可使用启动加载器来通过网络进行启动。

若要将您的 DiskStation 用作启动加载服务器：

  1. 请确认您已选择启用 TFTP 服务 (在控制面板 > 文件服务 > TFTP)。
  2. 选择一个文件夹，将启动加载器文件托管为 TFTP 根文件夹。
  3. 确认启用至少一个 DHCP 子网 (在控制面板 > DHCP Server > 网络接口)。
  4. 请进入控制面板 > DHCP Server > PXE。
  5. 选择启用 PXE (预启动执行环境)。
  6. 选择本地 TFTP 服务器并选择启动加载器文件。
  7. 单击应用以保存该设置。

若要将远程 TFTP 服务器用作启动加载服务器：

通过将远程 TFTP 服务器用作启动加载服务器，您可让客户端启动而无需更改当前网络架构。

  1. 确认启用至少一个 DHCP 子网 (在控制面板 > DHCP Server > 网络接口)。
  2. 请进入控制面板 > DHCP Server > PXE。
  3. 选择启用 PXE (预启动执行环境)。
  4. 选择远程 TFTP 服务器，指定服务器 IP 地址和启动加载器文件的文件路径。
  5. 单击应用以保存该设置。
