路由器配置

如果您的 DiskStation 设备是位于局域网，则 Internet 上的其他设备无法进行链接。您可以在控制面板 > 外部访问 > 路由器配置中设置端口转送规则以使 DiskStation 设备可通过 Internet 访问。

设置路由器

添加端口转发规则之前，您必须先设置 DiskStation 设备与路由器之间的连接。

注：

  1. 若要配置端口转发规则并指派静态 IP 地址，您需要拥有此路由器的管理权限。
  2. 在设置 DiskStation 设备与 UPnP 路由器之间的连接之前，请先确定 DSM 已成功采集 DiskStation 的外部 IP 地址（位于控制面板 > 外部访问 > DDNS）。如果信息不正确，请检查 DiskStation 设备及 UPnP 路由器的网络及防火墙设置。

若要设置 DiskStation 设备与路由器之间的链接：

  1. 请单击设置路由器以启动设置向导。在继续到下一步之前，DSM 将对以下网络组件/设置进行测试。
    • 网络接口
    • 网关连接
    • Internet 连接
    • 端口转发
    • DNS
  2. 视您的路由器类型而定，运行以下其一：
    • 如果您的路由器不支持 UPnP，请参考步骤 3。
    • 如果您的 UPnP 路由器经测试与 DiskStation 设备兼容，请在您看到路由器的总览信息时单击应用。DiskStation 设备与路由器之间的链接已成功设置。
    • 如果您的 UPnP 路由器尚未通过 DiskStation 的兼容性测试，请执行下列任何操作：
      • 如果您想要自动连接至路由器，请选择自动并单击下一步来让系统运行兼容性测试：
        • 如果路由器经测试与 DiskStation 设备的路由器设置服务兼容，请勾选复选框来传送测试报告以辅助 Synology 更新路由器兼容性数据库，然后单击应用。DiskStation 设备与路由器之间的链接已成功设置。
        • 如果路由器经测试与 DiskStation 设备的路由器设置服务不兼容，请勾选复选框来传送测试报告以辅助 Synology 更新路由器兼容性数据库，然后单击关闭。

          • 注：

          如果路由器经测试为不兼容，您仍可在路由器的用户界面中配置端口转送规则。参阅此处以了解更多信息。

      • 如果您想要手动设置与路由器之间的连接，请选择手动，单击下一步，然后参阅步骤 3。
  3. 如果您需要手动设置 DiskStation 设备与路由器之间的连接，请执行下列操作：
    1. 从路由器下拉菜单中选择您的路由器。

      2. 注：

      如果找不到您的路由器，请单击更新列表可从 Synology 取得最新列表。

    2. 设置路由器的通讯协议和端口。通讯协议和端口视路由器型号而定。如果您无法使用预设的通讯协议 (HTTP) 和端口 (80)，请参阅路由器随附的说明文件来了解正确设置。
    3. 如果路由器需自定义帐户，勾选使用自定义帐户，然后填写帐户和密码栏。
    4. 单击应用完成。

管理端口转送规则

端口转送规则可以指定 DiskStation 的目标端口，其会接收来自路由器特定端口的套件。

如果您的 DiskStation 通过路由器连接到 Internet，请单击创建以打开端口转送设置向导。

您可以使用两种方式来添加端口转送规则：

  • 如果您要为系统服务（例如，FTP、Web Station 等）创建端口转送规则，请选择内置应用程序。
  • 如果您要为其他服务创建端口转送规则，请选择自定义端口。

您可以使用下列语法来指定端口编号和范围：

  • 使用“-”来描述端口范围，例如：6881-6890。
  • 使用“,”来分开端口，例如：“21,22”、“21,55536-55663”。

注：

  • 每个网关路由器的端口只能使用于一个转送规则。如果路由器的端口使用于两个或两个以上规则，会造成冲突，且画面上会显示一个警告消息。您需要在规则中设置另一个端口才能避免冲突。
  • 部分路由器并不允许使用自定义端口来转送系统服务。

若要保存规则：

创建规则后，请单击保存。否则，将不会应用规则。

保存规则后，您可能会需要稍待数分钟，规则才会开始生效，视路由器的型号而定。

若要测试规则是否正常运作：

选择您要测试的规则，并单击测试连接。结果会显示在连接测试结果栏中。

无线路由器配置*

当您的 DiskStation 作为无线路由器使用时，DiskStation 将直接执行端口转送规则。

若要添加端口转送规则：

  1. 请单击创建来打开设置窗口。

  2. 指定服务名称、协议（TCP 或 UDP）、专用 IP 地址、专用端口和公用端口。您可以使用下列语法来指定端口编号和范围：
    • 使用“-”来描述端口范围，例如：6881-6890。
    • 使用“,”来分开端口，例如：“21,22”、“21,55536-55663”。
  3. 单击底部的创建以完成该设置。

注：

大多数路由器允许在配置端口转送规则时允许本地机端口和路由器端口不同。如果您的路由器支持此功能：

  • 如果路由器端口已被已有规则使用，在保存规则时，将被指派新的路由器端口。
  • 在添加或编辑规则时，您可直接指定路由器端口号。

DMZ*

当“网络映射配置”为无线路由器时，您可以启用 DiskStation 上的 DMZ。DMZ 中的主机可直接连接到外部网络，没有防火墙的限制和保护。它一般用于设置服务器。

若要启用 DMZ：

  1. 请单击 DMZ 设置来打开设置窗口。
  2. 勾选启用 DMZ 复选框。
  3. 输入 DMZ 主机 IP 地址。
  4. 单击应用。

注：

若使用外部网络，您将需要通过 DiskStation 取得外部 IP 地址才能连接到 DMZ 中的主机。

____
* 仅适用于特定机型或根据您的环境而定。
