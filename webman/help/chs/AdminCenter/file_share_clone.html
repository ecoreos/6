克隆共享文件夹*

您可在 Btrfs 卷上克隆一个共享文件夹以即时创建已有共享文件夹的副本。克隆共享文件夹在创建的过程也只消耗少量的存储空间。

若要克隆共享文件夹：

  1. 请进入控制面板 > 共享文件夹。
  2. 在 Btrfs 文件系统中选择所需的共享文件夹。
  3. 单击创建 > 克隆。
  4. 为新共享文件夹指定名称和描述，如需要可修改其它选项。(要了有关解此选项的详细信息，请参见创建共享文件夹。)

注：

  • 仅 Btrfs 卷上的共享文件夹可克隆。
  • 克隆的共享文件夹将在原始共享文件夹的同一卷上创建。
  • 高级数据完整性保护的设置将应用到克隆共享文件夹且无法更改。

____
* 仅适用于特定机型。
