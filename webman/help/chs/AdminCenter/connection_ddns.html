DDNS

DDNS（动态域名服务）通过将主机映射到其 IP 地址来简化通过 Internet 与 DiskStation 的连接。例如，DDNS 可让您使用域名（如 www.john.synology.me）来访问 DiskStation，而无需记住 IP 地址（如 172.16.254.1）。

设置 DDNS

您可将一个已有的主机名指定到 DiskStation 的 IP 地址，或注册由 Synology 或多个其它 DDNS 供应商提供一个新的主机名。请咨询各供应商以了解有关主机名注册的详情。

若要设置 DDNS 主机名：

  1. 单击添加按钮。
  2. 出现对话框，提示您编辑以下设置：
    • 服务供应商：选择服务供应商。若要注册以获得 Synology 提供的免费主机名，从下拉菜单中选择 Synology，单击立即注册，然后按说明输入详情。
    • 主机名：输入注册的 DDNS 主机名，如 john.synology.me。
    • 用户名：输入 DDNS 供应商的用户名。
    • 密码：输入 DDNS 供应商的密码。
    • 外接地址：输入将使用该主机名的 DiskStation 的外接 IP 地址。
    • Heartbeat：启用此选项可接收有关映射主机名状态的报警（仅适用于 Synology 提供的主机名）。
  3. 单击测试连接以查看设置是否正确。
  4. 单击确定来保存并结束设置。

设置完成后，您可在网页浏览器中输入 DDNS 主机名以通过 Internet 访问 DiskStation。

要求和限制

  • DDNS 服务供应商的服务器正在运作。
  • DiskStation 可以连接到 Internet。
  • 只有在不需代理服务器的网络环境下，DDNS 才可运作。
  • 一个 DDNS 供应商只能输入一个主机名。

管理主机名

使用以下按钮可管理已在的主机名条目：

  • 删除：从列表中删除所选主机名条目。
  • 编辑：更改所选主机名条目的设置。
  • 立即更新：更新所有已启用 DDNS 服务的主机名条目状态。

设置自定义 DDNS 供应商

如果您的供应商未出现在服务供应商列表中，您可添加自定义 DDNS 供应商。

若要设置自定义 DDNS 供应商：

  1. 单击自定义。
  2. 出现对话框，提示您输入以下设置：
    • 服务供应商：输入自定义服务供应商的名称。
    • Query URL：输入自定义服务供应商的 URL 。
  3. 单击保存可将自定义服务供应商添加到 DDNS 服务供应商的列表中。

要求和限制

  • 查询 URL 必须遵守服务供应商指定的协议。此 URL 用于将 IP 信息更新到您供应前几年 DDNS 服务器。
  • 查询 URL 一般包含以下四个参数。当输入查询 URL 时，请以相关变量替换以下参数。
    • 用户名：__USERNAME__
    • 密码：__PASSWORD__
    • 主机名：__HOSTNAME__
    • 外接地址：__MYIP__
  • 查询 URL 不可包含空格或分号。
