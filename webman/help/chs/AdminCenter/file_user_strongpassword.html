密码设置

单击密码设置按钮可查看并修改下述设置。

重置忘记的密码

您会此区域中看到允许非管理员用户在忘记密码时通过电子邮件重置的选项。当您启用此功能后，DSM 登录页面将会出现忘了密码？的链接。当用户忘记密码时，可单击此链接并输入其用户名。此时系统会向用户发送一则信息，当中含有重置忘记密码的链接。

注：

  • 启用该选项之前，请先确认您已启用 SMTP 电子邮件通知。若要进行此操作，请进入控制面板 > 通知 > 常规。如果未启用通知，则系统将无法对要重置密码的用户发送信息。
  • 若要接收系统信息，用户必须在其帐户的用户信息区域中提供电子邮件地址。
  • 属于 administrators 群组的用户无法使用此选项来重置密码。
  • 域用户无法使用此选项来重置密码。
  • 只要 DiskStation 作为 LDAP 目录服务器，LDAP 用户就可使用此选项来重置密码。

密码强度

您可以为 DiskStation 中的用户设置密码强度规则。

注：

密码强度规则仅适用于之后新创建的密码。例如，仅当创建新的用户帐户或当现有用户更改其密码时才能应用新设置的密码强度规则。因此现有密码以及属于导入用户帐户的密码将排除在新设置的密码规则外。

若要启用密码强度规则：

  1. 请勾选应用密码强度规则复选框来启用密码限制。
  2. 从下列规则中选择多个密码限制。
    • 从密码中排除用户名及描述：密码中不能包含用户名或用户描述。若用户名称或描述为 UTF-8 字符则不在此限制中。
    • 允许混合大小写：密码中可以包含大小写字符。
    • 包含数字字符：密码中至少要包含一个 0 ~ 9 的数字字符。
    • 包含特殊字符：密码需包含至少一个 ASCII 特殊字符（即 ~、`、!、@、#、$、%、^、&、*、(、)、-、_、=、+、[、{、]、}、\、|、;、:、'、"、<、>、/、?）。
    • 最短密码长度：最短密码长度不可小于此栏中指定的值。最短密码长度应介于 6 到 127 之间的数字。
  3. 单击应用来保存设置。
