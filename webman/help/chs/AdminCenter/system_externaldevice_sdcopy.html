SDCopy*

您可以选择设置一个共享文件夹作为 SDCopy 功能的目的地文件夹。

若要启用 SDCopy：

  1. 勾选启用 SDCopy。
  2. 请从下拉菜单中选择目的地共享文件夹。
  3. 单击确定。

在您将一个 SD 设备（例如数码摄像机）插入 Synology DiskStation 前方面板的 SD 插槽时，您可以按压前方面板的 Copy 按钮来将 SD 设备上的数据复制到您指定的目的地共享文件夹。如果 SDCopy 的目的地文件夹被删除，系统将会自动以 public 共享文件夹替换它。

    限制：
  1. 仅支持格式为 FAT 或 NTFS 的 SD 设备。
  2. SD 设备必须插入 Synology DiskStation 前方面板的 SD 插槽。
  3. 目的地共享文件夹的可用空间必须大于 SD 设备的已使用空间。

____
* 仅适用于特定机型。
