常规设置

控制面板 > 网络 > 常规中提供通用的网络选项，如默认网关、DNS 服务器、代理服务器和 IPv6 隧道等。

重命名 DiskStation 设备

若要重命名 DiskStation 设备，请输入新服务器名称并单击应用。

服务器名称没有大小写区分，可包含 1 到 15 个字符，由英文字母、数字、下划线和减号组成。名称的第一个字符必须为字母。

指定默认网关

请输入默认网关的 IP 地址并单击应用来保存设置。

指定 DNS 服务器 IP 地址

您可选择手动配置 DNS 服务器的 IP 地址。若选择此方法，您必须输入首选 DNS 服务器的 IP 地址。您也可以输入备选 DNS 服务器的 IP 地址，如此一来，如果首选 DNS 服务器遇到问题备选 DNS 服务器就可作为备用。DNS 服务器 IP 地址按要求配置后，请单击应用以保存设置。

注：

当您使用 PPPoE 连接或是让系统自动设置所有的网络接口时，系统会一并自动设置 DNS 服务器和默认网关等栏，且您将无法变更这些栏的数值。

通过代理服务器连接

若要通过代理服务器让 DiskStation 设备进行连接，请勾选通过代理服务器连接，输入地址和端口，然后单击应用。

注：

  • 单击高级设置可配置 HTTP 代理、HTTPS 代理和验证信息。
  • HTTPS 代理设置与 HTTP 代理设置默认相同。
  • 如果勾选通过代理服务器连接，则 DSM 更新、套件中心及部分其他套件将始终通过代理服务器连接 Internet。
  • 如果勾选了对于本地地址不使用代理服务器，请确认 DiskStation 使用的 DNS 服务器可为本地网络中的设备解析查询。

启用 IPv6 隧道

在 IPv6 完全取代 IPv4 之前，您需通过 IPv6 隧道来让拥有 IPv6 地址的主机连接到 IPv4 服务。此项技术提供了一种在现有 IPv4 网络中传输 IPv6 流量的方法。
如果您对于设置 IPv6 有任何疑问或遇到链接上的问题，请联系您的网络服务供应商（ISP）。

若要启用 IPv6 隧道：

  1. 勾选启用 tunnel 复选框来启用 tunnel 支持。
  2. 在服务器地址中，输入 tunnel 服务供应商的主机名或 IP 地址，例如：anonymous.freenet6.net。
  3. 选择匿名连接或以现有的帐户连接。

注：

服务供应商可能会主动更新您的外部 IPv6 地址。但 DiskStation 设备不会立刻删除原有的 IPv6 地址。
