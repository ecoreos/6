QuickConnect

QuickConnect 允许客户端应用程序通过 Internet 连接到 DiskStation，而无需设置端口转发规则。

注：

QuickConnect 可以与 DSM 及一些 Synology 开发的套件一起工作，如 Audio Station、Video Station、Download Station、Surveillance Station、Photo Station、File Station、Cloud Station 和移动设备应用程序。

若要启用 QuickConnect：

  1. 请进入控制面板（简单模式） > QuickConnect。
  2. 勾选启用 QuickConnect 复选框。
  3. 输入现有 Synology 帐户信息或注册新帐户。
  4. 指定一个新的、好记的 QuickConnect ID，这样您就可以随时访问您的 DiskStation。
  5. 单击应用。

注：

  • 在为客户端应用程序启用 QuickConnect 之前，请确认所需的 DSM 服务均已启用。例如，在启用 QuickConnect 来连接 DS file 之前，您必须先启用在 DiskStation 上的 WebDAV 服务。
  • 自定义 QuickConnect ID 可包含英文字母、数字和连字符（-），且开头必须为字母。
  • 为使 QuickConnect 有更好的性能，建议您进入控制面板（高级模式） > 外部访问 > 路由器配置中来为各项服务配置端口转送规则：
    • DSM：5000（HTTP）；5001（HTTPS）
    • Photo Station：80（HTTP）；443（HTTPS）
    • Cloud Station：6690

疑难解答

如果您看到发生网络错误信息，表示 QuickConnect 服务因网络错误而中止。请检查以下内容：

  • 您的 DiskStation 连接到活动网络（在控制面板（高级模式） > 网络 > 网络接口）。
  • 已适当配置 DNS 服务器和默认网关设置（在控制面板（高级模式） > 网络 > 常规）。
  • 确认 DiskStation 可通过以下目标端口访问 Internet：22、80、443、8888。

网络错误修复后，QuickConnect 服务会在数分钟内自动恢复运行。
