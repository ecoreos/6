UPS

UPS（Uninterruptible Power Supply，不间断电源）是备用电源设备，断电时它可让 DiskStation 继续运作一小段时间。在控制面板 > 硬件和电源 > UPS 中的此功能有助于 DiskStation 在断电和关机之前有足够时间来保存数据和卸载卷，以避免数据丢失。

安全模式

当 DiskStation 进入安全模式时，它会停止所有服务并卸载卷，以避免数据丢失，并在 UPS 设备消耗完电力后安全关机。当 UPS 设备在电量不足的情况下开始运行时，系统会默认进入安全模式。您还可以指定断电发生后多久让 DiskStation 进入安全模式。但如果 UPS 设备在指定时间之前电量已不足，系统会立即进入安全模式。

DiskStation 在安全模式下关闭的情况下，如果您已启用断电后自动重启选项（位于控制面板 > 硬件和电源 > 常规），则在供电恢复时它会自动开启。

本地不间断电源系统支持

如需有关硬件安装的详情，请参阅不间断电源系统设备的用户手册。安装完成且 DiskStation 已插入不间断电源系统设备后，您可以按照下列步骤启用不间断电源系统支持。

若要启用本地不间断电源系统支持：

  1. 请将不间断电源系统设备连接到 DiskStation 的 USB 端口。
  2. 请在不间断电源系统选项卡中，勾选启用不间断电源系统支持复选框。
  3. 选择在断电后多久让 DiskStation 进入安全模式。
  4. 选择是否要在系统进入安全模式时，发送关机信号至不间断电源系统设备。大多数不间断电源系统设备会在电源恢复时自动开启。
  5. 单击应用。

SNMP 不间断电源系统支持

您的 DiskStation 还支持连接到 SNMP（Simple Network Management Protocol，简单网络管理协议）不间断电源系统设备。如需有关硬件安装的详情，请参阅 SNMP 不间断电源系统设备的用户手册。在 DiskStation 插入不间断电源系统设备且连接到同一网络后，您可以按照下列步骤启用不间断电源系统支持。

若要启用 SNMP 不间断电源系统支持：

  1. 请在不间断电源系统选项卡中，勾选启用不间断电源系统支持复选框。
  2. 从网络不间断电源系统类型下拉菜单中选取 SNMP 不间断电源系统。
  3. 在 SNMP 不间断电源系统 IP 地址栏中，输入 SNMP 不间断电源系统设备的 IP 地址。
  4. 选择在断电后多久让 DiskStation 进入安全模式。
  5. 将 SNMP 不间断电源系统设备的共同体字符串输入到 SNMP 共同体字段。
  6. 单击应用。

注：

  • 强烈建议使用 SNMP UPS 以进行 Synology High Availability (SHA) 部署。
  • DiskStation 支持 SNMPv1 和 SNMPv2c 不间断电源系统设备。

网络不间断电源系统支持

如果有两台 Synology 产品共同使用一台不间断电源系统设备，而该设备只有一个 USB 端口，则只有一台 DiskStation 可以获取该不间断电源系统设备的状态信息。在这种情况下，一台 DiskStation 可用作“网络不间断电源系统服务器”并通过 USB 或网络（取决于设备）连接到不间断电源系统设备。网络不间断电源系统服务器将获取不间断电源系统信息并将其信息转发至另一个“客户端” DiskStation。

若要启用网络不间断电源系统支持：

  1. 请登录到连接不间断电源系统设备的 DiskStation（通过 USB 或网络）。如果未在 DiskStation 上启用本地不间断电源系统支持，请参阅上述章节。
  2. 在不间断电源系统选项卡中，勾选启用网络不间断电源系统服务器框来让 DiskStation 将不间断电源系统信息转发至客户端 DiskStation。
  3. 单击允许的 DiskStation 设备按钮。
  4. 在出现的窗口中，输入另一台非通过 USB 或网络连接使用不间断电源系统设备的 DiskStation 其 IP 地址。单击确定。
  5. 单击应用来保存设置。

现在，网络不间断电源系统服务器可以获取不间断电源系统信息并将信息转发至另一台正在使用不间断电源系统设备的 DiskStation。请按照下列说明来启用另一台 DiskStation 上的不间断电源系统支持。

若要启用客户端 DiskStation 的不间断电源系统支持：

  1. 请登录到另一台正在使用不间断电源系统设备的 DiskStation（非通过 USB 或网络连接）。
  2. 在不间断电源系统选项卡中，勾选启用不间断电源系统支持框。
  3. 从网络不间断电源系统类型下拉菜单中选择 Synology 不间断电源系统服务器。
  4. 在网络不间断电源系统服务器 IP 栏中输入网络不间断电源系统服务器的 IP 地址（即连接到不间断电源系统设备的 DiskStation）。
  5. 选择在断电后多久让 DiskStation 进入安全模式。若要简化安全模式设置，您可以选择与服务器相同。
  6. 单击应用来保存设置。

网络不间断电源系统需求：

  • 若要转发不间断电源系统设备信息，每台 DiskStation 都应连接至相同的网络。
  • 各 DiskStation 连接的网络设备（例如交换器或集线器）应插入同一台不间断电源系统设备。否则，DiskStation 将无法在断电时转发不间断电源系统设备信息。
