路由器流量控制

当 DiskStation 处于路由器模式时，路由器流量控制被启用。路由器流量控制 (位于控制面板 > 网络 > 路由器流量控制) 管理着 DiskStation 专用网络中上传和下载的流量。在创建规则时，您需指定保障带宽和最大带宽来限制上传与下载流量。

名词解释

  • 保障带宽：代表整个系统带宽足够时，该服务保证取得的出站通信量。
  • 最大带宽：代表整个系统带宽足够且有系统剩余带宽时，该服务可利用的出站通信量。建议您先计算系统输出带宽，并确认各项服务的保障带宽总量不大于系统输出带宽，否则该设置可能无法正常运作。

设置公式

  • 系统输出带宽 = 所有服务保障带宽的总量 + 系统剩余带宽
  • 各项服务的保障带宽 ≤ 各项服务的最大带宽

注：

如果出站包与任何路由器流量控制规则不匹配，则默认行为没有限制。您可在所有规则的结尾添加自已的默认路由器流量控制规则，方法是选择全部端口，然后指定保障带宽和最大带宽。

若要创建路由器流量控制规则：

  1. 请从右上角的下拉菜单中选择网络接口。
  2. 单击创建。
  3. 在端口区域，选择以下内容之一：
    • 选择全部来将规则应用到所有端口。
    • 选择自定义，然后单击自定义。最多可输入 15 个端口，并用逗号隔开或指定要对其应用该规则的端口范围。
    • 您也可指定端口的通讯协议。
  4. 在来源 IP 区块，选择以下内容之一：
    • 选择全部将此规则应用到所有 IP 地址。
    • 选择特定 IP 地址，然后单击选择。您可以指定 IP 地址或要对其应用此规则的 IP 地址范围。

现在您可以在规则列表中看到创建的路由器流量控制规则。

若要删除路由器流量控制规则，请执行以下任何操作：

  • 请取消勾选列表启用栏中的规则复选框。
  • 删除路由器流量控制规则。

注：

  • 每个网络接口最多创建 100 项规则。
  • 您可通过拖放操作来重新排序列表中的规则。
  • 规则的优先顺序将根据在列表中排序的位置。
  • 指定的端口是指外网 Internet 服务的端口号。
  • 来源 IP 地址是指将 Synology NAS 用作路由器的设备 IP 地址。
