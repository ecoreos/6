热点*

如果您已安装 Wi-Fi USB 适配器或 DiskStation 有内置 Wi-Fi，您的 DiskStation 设备将支持创建热点以来其它设备无线共享其 Internet 连接。您可设置 DiskStation 为无线接入点（AP）或无线路由器，取决于您要使用 Internet 的方式。

热点模式 描述
无线接入点
  • 桥接您的局域网和 Wi-Fi 网络。
  • 您的局域网和无线客户端将位在同一本地网络中。
  • 当您有路由器并想要将 DiskStation 设为已有网络环境的额外接入点时可用。
  • 允许网络资源共享，如 DLNA 设备、打印机、Synology Web Assistant 或其它服务，取决于网络广播。
  • DHCP 服务器在作为无线接入点的 DiskStation 设备上将被禁用。若要重新激活 DiskStation 上的 DHCP 服务器，请移除无线网卡并重启 DiskStation。
  • 对于配备多个 LAN 端口的型号，只有 LAN 1 将用来与无线接入点桥接。
无线路由器
  • 在专用子网中的设备上设置无线网络。

注：

  1. 部分型号需要连接无线网卡来创建热点。要了解完整的支持无线网卡清单，请参见 www.synology.com。
  2. 在 DSM 5.0 和更新版本中，对于非 DS213air 型号，DSM 4.3 中的无线热点选项被重命名为“无线路由器”。

管理热点设置

此章节介绍如何启用和管理 DiskStation 的热点设置。

若要将 DiskStation 转为热点：

  1. 请进入控制面板 > 无线 > 网络映射。选择无线接入点或无线路由器.
  2. 进入 Wi-Fi 选项卡并单击编辑来管理下列设置：
    • 名称（SSID）：输入无线网络名称。
    • 显示/隐藏：选择无线网络是否可被其它设备访问。
    • 无线模式：选择无线标准。每个标准都支持不同的带宽和频率。
    • 通道宽度：选择通道宽度（仅在使用 802.11n 无线模式时可用）。
    • 频道：选择无线网络频率。为获得更好的性能，选择附近网络使用的不同频率。
    • 传输功率：选择无线网络的信号强度。
    • 最大连接数：选择可连接到网络的客户端设备数量。最大连接设置为 256。
    • 启用 WMM QoS：通过 WLAN 的 WLAN 多媒体（WMM）QoS（服务质量）。此选项可让无线网络优先通信并优化资源在应用程序之间共享的方式。

若要设置无线计划：*

启用无线网络计划可让您控制 DiskStation 被激活和可访问的时间。

  1. 请进入 Wi-Fi 选项卡并单击编辑。
  2. 勾选启用计划框。
  3. 单击设置计划。
  4. 在计划屏幕，选择无线网络被激活的时间。

安全性设置

在控制面板 > 无线 > Wi-Fi 提供了一些选项，让您编辑无线网络的安全性设置。

若要设置无线安全性：

  1. 单击编辑。
  2. 从安全性设置下拉菜单中选择下列安全协议之一：
    • WEP：需要用户输入密码以访问无线网络的基本安全协议，但提供最低限度的保护。
    • WPA/WPA2 个人：推荐。需要用户输入密码以访问无线网络，比 WEP 更安全。
    • WPA/WPA2 企业：* 为企业指定的安全协议，需要用外部验证服务器来验证用户。

若要启用 Wi-Fi Protected Setup：*

Wi-Fi Protected Setup 是 Wi-Fi Alliance 推广的标准，可简化用户的无线安全设置。您可使用按下按钮或 PIN 码来维护 Wi-Fi 网络安全。请单击 WPS 来启用 Wi-Fi Protected Setup。

访客网络*

除创建主无线网络外，您的 DiskStation 还支持创建访客无线网络。可将访客网络的设置限制为禁止用户的访问，这对于要为访问者提供无线网络但不让其访问主网络的企业或其它机构非常有用。

若要启用访客网络：

  1. 请进入控制面板 > 无线 > Wi-Fi。
  2. 单击编辑来管理以下访客网络选项卡中的设置：
    • 名称（SSID）：输入无线网络名称。
    • 显示/隐藏：选择无线网络是否可被其它设备访问。
    • 最大连接数：选择可连接到网络的客户端设备数量。最大连接设置为 256。

若要设置访客网络安全：

从安全性设置下拉菜单中选择下列安全协议之一：

  • WEP：需要用户输入密码以访问无线网络的基本安全协议，但提供最低限度的保护。
  • WPA/WPA2 个人：推荐。需要用户输入密码以访问无线网络，比 WEP 更安全。

若要设置访客网络计划：

启用访客网络计划可让您控制 DiskStation 被激活和可访问的访客无线网络时间。

  1. 请勾选启用访客网络计划框。
  2. 单击设置访客计划。
  3. 在计划屏幕，选择访客无线网络被激活的时间。

注：

访客无线网络在主无线网络停用的时段中无法激活。

MAC 过滤器设置*

MAC 过滤器（在控制面板 > 无线）可让您指定允许访问无线网络的 MAC 地址、锁定或向特定设备授予访问权限。此功能仅能控制网络访问，而不是其它加密设置。

若要创建 MAC 过滤器规则：

  1. 请输入设备的描述和 MAC 地址。
  2. 选择访问策略：
    • 拒绝：限制指定的 MAC 地址访问无线网络。
    • 允许：允许指定的 MAC 地址访问无线网络。

家长控制*

您可使用家长控制功能来控制每个连接 DiskStation 本地网络的客户端设备其访问 Internet 的时间。例如，如果您孩子的计算机或移动设备是通过 DiskStation 连接 Internet，则您可配置家长控制设置，根据特定设备和时间来限制其 Internet 的使用。

若要启用家长控制：

  1. 请进入控制面板 > 无线 > 家长控制设置。
  2. 单击添加按钮来添加客户端设备。
    • 客户端名称：选择连接到 DiskStation 的设备或手动输入名称（如 Louisa's laptop）。
    • MAC 地址：输入您要控制其 Internet 使用的设备的 MAC 地址。
    • 设置计划：指定设备可通过 DiskStation 访问 Internet 的时间。

连接列表*

连接列表（在控制面板 > 无线）显示了有关连接到 DiskStation 的设备信息。设备可从此处直接添加到 MAC 过滤器列表。

若要将设备添加到 MAC 过滤器列表：

  1. 请勾选要添加到 MAC 过滤器设备旁的复选框。
  2. 单击添加到 MAC 过滤器列表。
