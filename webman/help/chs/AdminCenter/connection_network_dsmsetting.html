DSM 设置

在控制面板 > 网络 > DSM 设置中，您可以更改客户端用于访问 DiskStation 的端口号。

若要更改默认端口号：

DSM 中始终启用 HTTP 和 HTTPS 连接。默认的 HTTP 端口为 5000，默认的 HTTPS 端口为 5001。

  1. 请在 HTTP 或 HTTPS 栏中输入端口号。
  2. 单击应用。

若要将 HTTP 重新导向到 HTTPS 连接：

如果客户端尝试通过 HTTP 端口访问 DiskStation，您可将客户端重新导向到 HTTPS 端口。

  1. 选择将 HTTP 连接自动重新导向到 HTTPS (Web Station 和 Photo Station 除外)。
  2. 单击应用。

HTTPS 连接使用的端口：

  • DSM：默认的端口号为 5001，因此您可通过输入以下地址安全访问 DSM：https://server-hostname:5001/
  • Web Station 和 Photo Station：默认的端口号为 443，因此您可通过输入下列地址以安全地访问 Web Station 和 Photo Station：https://server-hostname/

若要找到网络中的 DiskStation：

您可通过 Windows 网络发现或 Bonjour 服务发现来找到您的 DiskStation。

  1. 选择一个服务可在网络上找到您的 DiskStation：
    • 启用 Windows 网络发现
    • 启用 Bonjour 服务发现
  2. 单击应用。

域

在域下，您可指定自定义的域名以访问 DiskStation。

若要启用自定义域：

  1. 勾选启用自定义域。
  2. 为 DiskStation 指定启用的域名。
  3. 您还可勾选启用 HSTS 以在此域上强制 HTTPS 加密。
  4. 单击应用。

注：

  • 每个域名只能用于 DiskStation 中一个不同的应用程序。
  • 如果域名的长度未超出 15 个字符并符合 NetBIOS 命名规则，系统将自动注册和播送域名 (如 MyDSM)。Windows 用户可通过类似的 URL 访问同一本地网络中的 DSM 桌面：http://MyDSM 或 https://MyDSM
  • 当通过使用域名的 URL 访问应用程序时，您会通过标准的 HTTP 端口 (80) 或标准的 HTTPS 端口 (443)。
