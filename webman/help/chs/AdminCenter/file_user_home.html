用户家目录

启用用户家目录功能来为每一位用户 (guest除外) 创建个人的home文件夹 (家目录)。所有的用户都可以通过 CIFS、AFP、FTP 及 File Station 来访问自己的家目录。

系统管理者 (admin) 可以访问所有用户的家目录，并藉由家目录共享文件给特定用户。 所有用户的家目录统一集中储存于系统默认共享文件夹homes。家目录的名称与用户帐号相同。

若要启动用户家目录服务：

  1. 点选家目录按钮。
  2. 勾选启动家目录服务。
  3. 若您有多个储存空间，请选择您希望将 homes (家目录) 储存在哪个储存空间。
  4. 单击确定。

注意：停用本地用户的家目录服务后，域用户的家目录服务也会一并停用。

若要停用使用家目录服务：

  1. 点选家目录按钮。
  2. 取消勾选启动家目录服务。

关闭家目录功能后，所有用户的 homes (家目录) 将被保留下来仅供系统管理者 (admin) 访问。重新打开家目录功能后，各用户仍可访问其家目录的所有文件。

    注意：
  1. 若要删除 homes 共享文件夹，您必须先停用家目录服务。
  2. 如果本地用户的家目录服务已停用，域用户的家目录服务也会一并停用。
