打印机

DiskStation 可以成为打印服务器，让客户端计算机或移动设备可以访问连接到 DiskStation 的打印机。此外，对 Apple 无线打印和 Google Cloud Print 的支持可让您使用 iOS 设备或 Google 产品和服务进行远程打印。

请访问 www.synology.com 以了解兼容打印机。

设置打印机

参见以下说明以设置并将 USB 或网络打印机与 DiskStation 进行连接。

若要设置 USB 打印机：

  1. 请开启打印机并将其连接到 DiskStation 上的 USB 端口。
  2. 进入控制面板 > 外部设备 > 打印机。
  3. 选择要设置的打印机。请注意，DiskStation 无法识别通过 USB 集线器连接的打印机。
  4. 单击打印机管理器并选择设置打印机。
  5. 在对话框中，选择以下模式之一：
    • 网络 MFP：可让本地网络中的客户端使用多功能打印机的打印、扫描和传真功能。只有 Windows 计算机支持扫描及传真的功能。
    • 网络打印机：为仅支持打印的打印机或如果您要让客户端仅使用打印功能选择此模式。
  6. 配置 Apple 无线打印和 Google Cloud Print 设置（可选）。
  7. 单击保存。

注：

网络 MFP 仅可由一个客户端同时访问。

若要设置网络打印机：

  1. 请开启网络打印机并将其连接到与 DiskStation 相同的本地网络。
  2. 进入控制面板 > 外部设备 > 打印机。
  3. 单击添加网络打印机并按设置向导说明进行操作。

注：

  • 打印机名称：在设置过程必须输入打印机名称，并可含有最多达 32 个字符，包括字母、数字和下划线。
  • 协议：一般而言，HP 网络打印机使用 Socket；Canon 网络打印机使用 BJNP；其它多数使用 LPR。
  • 队列名称：LPR 和 IPP 网络打印机需要输入队列名称，但不适用于 Socket 或 BJNP 打印机。
  • 无网络 MFP：连接到 DiskStation 的网络打印机目前无法使用网络 MFP 模式。
  • 无 USB 打印机：由 Synology 产品共享的 USB 打印机无法设置为网络打印机。

管理打印机

打印机选项卡包含以下选项，可管理连接到 DiskStation 的USB 或网络打印机。

  • 获取信息：显示与所选打印机相关的信息，如设备名称、制造商、状态等。
  • 删除：将打印机从 DiskStation 上删除和断开连接。
  • 打印机管理器：包含以下选项：
    • 设置打印机：编辑打印机设置，如 Apple 无线打印或 Google Cloud Print（参见以下说明）。
    • 清除缓存区：取消所有打印机上的全部打印任务。
    • 打印测试页：打印测试页以确定配置正确。
    • 解除打印机：断开打印机与客户端的连接（仅适用于网络 MFP 模式）。当客户端连接到多功能打印机以接收传真或扫描文件时，任务可能需要较长时间。如果需要，连接可停止以让下一个客户端访问打印机。

若要启用 Apple 无线打印：

  1. 进入控制面板 > 外部设备 > 打印机。
  2. 从设备列表中选择打印机。
  3. 单击打印机管理器 > 设置打印机。
  4. 勾选启用 Apple 无线打印框。
  5. 选择打印机的品牌和驱动程序。
  6. 单击保存。

设置完成后，您可单击打印机管理器 > 打印测试页以打印测试页。

注：

  • 启用 Apple 无线打印同时将启用 DSM 的 Bonjour 打印机共享。
  • iOS 4.2 或以上版本支持 Apple 无线打印。

若要启用 Google Cloud Print：

  1. 进入控制面板 > 外部设备 > 打印机。
  2. 从设备列表中选择打印机。
  3. 单击打印机管理器 > 设置打印机。
  4. 勾选启用 Google 云打印框。
  5. 输入以下信息：
    • 您的 Google 帐户信息。
    • 您的打印机品牌和驱动程序。
  6. 单击保存。

设置完成后，您可单击打印机管理器 > 打印测试页以打印测试页。

若要管理 Google 云打印任务，单击 Google 云打印链接。

访问打印机

以下章节介绍如何使用 PC 或 Mac 计算机访问打印机。

若要使用 Windows PC 访问打印机：

访问 DiskStation 设备打印服务的最佳方式是通过 Synology Assistant 设置打印机链接。请参阅其帮助文件以了解详细的指示说明。

如果您的计算机上没有安装 Synology Assistant，请参阅下方说明以了解其他访问 DiskStation 设备打印服务的方式。

类型一、共享网络打印机

  1. 在计算机上，打开文件资源管理器中的网络，并双击 DiskStation。
  2. 双击 usbprinter 或网络打印机名称；系统将提示您安装正确的驱动程序，然后单击是。
  3. 请从 CD 或制造商网站上获取打印机驱动程序。选择打印机的制造商和型号，或单击从磁盘安装以选择驱动程序的位置。然后单击确定。
  4. 系统会开始设置网络打印机。设置完成后，您就可以开始使用该台打印机。

类型二、LPR Port（建议选项）

  1. 在计算机上，单击开始并进入设备和打印机。
  2. 单击添加打印机。
  3. 选择添加网络、无线或蓝牙打印机。
  4. 选择我所需的打印机未列示。
  5. 选择使用 TCP/IP 地址或主机名添加打印机。单击下一步继续。
  6. 选择要成为 TCP/IP 端口设备的设备类型。
  7. 输入 DiskStation 的 IP 地址。单击下一步继续。
  8. 选择设备类型：自定义 > 设置 > ….单击下一步继续。
  9. 选择 LPR 作为协议；输入 usbprinter 或网络打印机名称作为队列名称，并勾选启用 LPR 字节计数复选框。单击确定继续。
  10. 请从 CD 或制造商网站上获取打印机驱动程序。选择打印机的制造商和型号，或单击从磁盘安装以选择驱动程序的位置。然后单击确定。
  11. 系统会开始设置网络打印机。设置完成后，您就可以开始使用该台打印机。

若要使用 Mac 访问打印机：

Mac 用户可以启动 DSM 的 Bonjour 打印机共享功能并在 [系统偏好设置] 中添加打印机，藉此访问 DiskStation 设备的打印服务。

继续进行之前，请先确定您的 DiskStation 是使用 Postscript 打印机。

步骤 1. 启用 DSM 的 Bonjour 打印机共享

  1. 请以 admin（或属于 administrators 群组的用户）身份登录 DSM。
  2. 请进入控制面板 > 文件服务 > Win/Mac/NFS。
  3. 勾选启动 Bonjour 打印机共享。

步骤 2. 使用 Mac 访问打印机

  1. 在 Mac 计算机上，进入系统偏好设置 > 打印与扫描，然后单击 +。
  2. 从下拉菜单中选择 usbprinter[编号]或网络打印机名称。您的 Mac 会自动设置打印机。
