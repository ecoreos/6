USBCopy

您可以选择设置一个共享文件夹作为 USBCopy 功能的目的地文件夹。

若要启动 USBCopy：

  1. 请勾选启动 USBCopy。
  2. 请从下拉式选单中选择目的地共享文件夹。
  3. 单击确定。

在您将一个 USB 设备（例如数码相机）链接到 Synology DiskStation 前方面板的 USB 界面后，您可以按压前面板的 Copy 按钮来将 USB 设备上的数据复制到您指定的目的地共享文件夹。如果目的地共享文件夹被删除，则系统将会自动变更 public 为 USBCopy 的目的地共享文件夹。

    限制：
  1. 仅支持格式为 FAT 或 NTFS 的 USB 设备。
  2. USB 设备必须插入 Synology DiskStation 前方面板的 USB 埠。
  3. 目的地共享文件夹的可用空间必须大于 USB 设备的已使用空间。
