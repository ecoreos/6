休眠

您可指定在硬盘休眠模式之前保持闲置的时间长度。在硬盘休眠过程中，硬盘将停止运转，以降低功耗并延长硬盘的使用寿命。您可为内部硬盘或外接的 eSATA/USB 硬盘指定不同的时间。

注：

  • 访问硬盘后，硬盘休眠即中止。
  • 仅部分特定外接 USB 存储设备支持硬盘休眠功能。

为什么我的 DiskStation 不会进入休眠模式？

请注意，当使用下列服务时，DiskStation 将不会进入休眠模式。此外，系统会在运行任何计划任务前 2 分钟，自动退出休眠模式。

套件：

  • Download Station：当 Download Station 正在运行电驴、RSS 计划下载或活动任务时，DiskStation 无法进入休眠模式。
  • Mail Station：当 Mail Station 正从 POP3 收取邮件时，DiskStation 无法进入休眠模式。
  • Surveillance Station：当 Surveillance Station 正在进行录制时，DiskStation 无法进入休眠模式。
  • 媒体服务器：当启用“媒体服务器”设置中的 DMA 日志时（于主菜单 > 媒体服务器 > 媒体服务器 > 诊断中），DiskStation 无法进入休眠模式。
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • 其它第三方软件或 optware，如 SABnzbd

进程：

  • 创建缩略图和索引：当 DiskStation 正在生成缩略图或重建多媒体文件升级后的索引时，其无法进入休眠模式。此时，“synoindexd”程序会列示为“正在运行”（进入主菜单 > 资源监视器 > 进程）。
  • Local Master Browser：当启用 Local Master Browser 时（于控制面板 > Win/Mac/NFS > Windows 文件服务 > 启用 Local Master Browser 中），DiskStation 无法进入休眠模式。
  • Windows Media Player：当在局域网中启用 Windows Media Player 的网络共享服务时，DiskStation 无法进入休眠模式。
  • 如果同时有多个进程需要交换执行，则 DiskStation 无法进入休眠模式。代表 DiskStation 的内存容量已经超出上限，且硬盘暂时用于执行读/写操作。

其他

  • USB 设备：当 DiskStation 与任何 USB 设备连接时，其无法进入系统休眠模式。

系统休眠

DiskStation 可以设置为当硬盘开始休眠 1 分钟后，就会进入系统休眠模式。这样做可降低 DiskStation 的功耗。

在系统休眠模式过程中：

  1. 电源按钮将进入间歇缓变模式（慢开/关循环）。将关闭其它 LED 指示灯。
  2. 将切断硬盘电源。
  3. 将停止 CPU 的运作。
  4. 将停止外接风扇的运作。
  5. 将关闭 LAN 端口的 LED 指示灯。

注：

  • 任何 SMB/CIFS 广播数据包都可能造成服务器无法进入系统休眠，包括网络中运行的任何 Windows 文件浏览器。如果要忽略 CIFS 广播数据包来让服务器进入系统休眠，请选择忽略 Windows 资源管理器的广播数据包。请注意当该选项选定后，SMB/CIFS 将无法使用数据包来退出系统休眠。此外，连接任何 USB 设备也会造成服务器无法进入系统休眠模式。

深度睡眠

扩充设备的深度睡眠功能可以通过配对的 DiskStation 服务器接口来启用。您可为扩充设备以及配对的 DiskStation 配置不同的闲置时间长度。因此，当配对的 DiskStation 保持在线时，扩充设备可设置为进入“深度睡眠”模式。

在深度睡眠模式过程中：

  • 将关闭扩充设备上所有的 LED 指示灯。
  • 将切断扩充设备的硬盘电源。
  • 将停止扩充设备风扇的运作。

注：

  • 访问硬盘后，深度睡眠即中止。
  • 为了确保扩充设备在使用深度睡眠功能时可正常运作，请将其后面板上的 Auto/Manual 开关滑动到 Auto。

自动关机

您可指定内部硬盘在 DiskStation 自动关机之前的休眠时间长度。

注：

  • 您必须先启用硬盘休眠。
  • 您必须先启用网络唤醒服务才能远程开启 DiskStation。
