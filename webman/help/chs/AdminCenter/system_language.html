语言

您可为 DSM 用户界面、Google Chrome 桌面通知、电子邮件、短信、即时通讯通知和应用程序代码页选择希望的语言。

显示语言

您可以为 DSM 和 Google Chrome 桌面通知选择您偏好的语言，或是选择让界面显示的语言和默认浏览器的语言设置相同。

通知语言

您可以为电子邮件、短信、Windows Live Messenger 和 Skype 通知设置您希望的语言。它还将应用于 Windows Phone 上接收的通知。

但 iOS 或 Android 设备上的通知语言取决于移动设备上的语言设置。

代码页

针对会转换非 Unicode 字符串为 Unicode 的应用程序，如 FTP 服务、UPnP 支持、创建音乐文件索引等，请在代码页下拉菜单中选择适当的代码页。
