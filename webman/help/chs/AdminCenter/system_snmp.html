SNMP

SNMP 服务（在控制面板 > 终端机和 SNMP > SNMP 中）可让用户通过网络管理软件来监控 DiskStation。目前支持的 SNMP 协议为 SNMPv1、SNMPv2c 以及 SNMPv3。

启动 SNMP 服务

勾选启用 SNMP 服务并按以下步骤操作。

若要启用 SNMPv1、SNMPv2c 服务：

  1. 请勾选 SNMPv1、SNMPv2c 服务复选框。
  2. 输入社群名称。
  3. 单击应用。

若要启用 SNMPv3 服务：

  1. 请勾选 SNMPv3 服务复选框。
  2. 输入用户名和密码。再输入一次来确认密码。
  3. 单击应用。

此外，输入名称、位置和联系人将有助您识别 DiskStation。这些栏为可选填栏。

注：

  1. SNMP 服务支持 IPv4 和 IPv6。
  2. SNMP 服务同时支持 SNMPv1、SNMPv2c 和 SNMPv3 协议。

SNMPv1、SNMPv2c 共同体限制：

共同体名称必须为 1 至 64 个可显示的字符。不允许下列字符：
" ' \ 和空格。

SNMPv3 用户名限制：

用户名必须为 1 至 64 个可显示的字符。不允许下列字符：
" ' \ 和空格。

SNMPv3 密码限制：

密码有区分大小写，长度为 8 至 127 个可显示的字符，包含字母、数字和符号。不允许下列字符：
" ' \ 和空格。
