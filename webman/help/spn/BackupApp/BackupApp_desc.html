Copia de seguridad y restauración

Aprenderá a usar la opción Copia de seguridad y restauración de las siguientes maneras:

  • Realizar copias de seguridad de datos en dispositivos Synology NAS locales/remotos
  • Realizar copias de seguridad de datos en un servidor remoto de rsync
  • Realizar copias de seguridad de datos en nubes públicas (p. ej., Amazon S3 y Microsoft Azure)
  • Realizar copias de seguridad de iSCSI LUN* en dispositivos de almacenamiento externos y en otros dispositivos Synology NAS
  • Restaurar archivos de copia de seguridad
