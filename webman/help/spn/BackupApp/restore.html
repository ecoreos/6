Restaurar

Vaya a Copia de seguridad y restauración y haga clic en Restaurar, en la esquina inferior izquierda, para restaurar los datos de una copia de seguridad.

Para restaurar una copia de seguridad de datos/LUN*:

  1. Haga clic en Restaurar, en la esquina inferior izquierda, seleccione Datos o LUN* y, a continuación, seleccione una tarea de copia para restaurarla.
  2. Se le pedirá que seleccione la configuración del sistema, diferentes versiones de datos de copias de seguridad o más. Dependerá de qué tipo de tarea de copia de seguridad desee restaurar.
  3. Si la tarea de copia de seguridad está codificada, será necesaria la contraseña o la clave de cifrado para realizar la tarea de restauración con éxito.
  4. Siga el asistente para completar el proceso de restauración.

Nota:

  • Al restaurar un paquete que no esté instalado, el sistema instalará automáticamente la misma versión de la aplicación o una versión posterior. Si la aplicación está instalada pero la versión es anterior a la que se ha utilizado para realizar la copia de seguridad, el sistema la actualizará automáticamente a la versión original de la que se hizo la copia de seguridad.
  • Aplicaciones de las que se haya realizado copia de seguridad en DSM 5.1 o versiones posteriores no se pueden restaurar en DSM 5.0 o versiones anteriores.

Nota:

  • Una vez finalizada la restauración, se restaurarán los LUN tanto de nivel de bloque como de nivel de archivo en los LUN de nivel de archivo.*
  • Cuando restaure un paquete, si el paquete no está instalado, el sistema instalará automáticamente la misma versión, o una posterior, que la de la aplicación. Si la aplicación está instalada pero la versión es anterior a la que se ha utilizado para realizar la copia de seguridad, el sistema la actualizará automáticamente a la versión original de la que se hizo la copia de seguridad.
  • Aplicaciones de las que se haya realizado copia de seguridad en DSM 5.1 o versiones posteriores no se pueden restaurar en DSM 5.0 o versiones anteriores.

Para restaurar un archivo o carpeta individual con tareas de copia de seguridad de varias versiones:

  1. Seleccione la tarea de copia de seguridad que desee en el panel izquierdo e inicie el Explorador de copia de seguridad (clic en el icono de lupa). Será necesaria la contraseña o la clave de cifrado si la tarea de copia de seguridad está cifrada.
  2. Busque la versión de la copia de seguridad que desee con la barra de línea temporal o a través del icono de calendario.
  3. Seleccione una carpeta o un archivo donde esté almacenada la copia de seguridad y, a continuación, haga clic en Restaurar, Copiar a... o Descargar para recuperar la carpeta o el archivo seleccionado. Las tres opciones están también disponibles si se hace clic con el botón derecho en el archivo o la carpeta.

Para buscar una versión específica:

Para aumentar la velocidad de búsqueda de archivos, puede utilizar palabras clave para filtrar la carpeta o el archivo que desee. Las carpetas o los archivos que no coincidan con los criterios de búsqueda se ocultarán.

  1. Inicie el Explorador de copia de seguridad y haga clic en el icono con forma de embudo situado en la esquina superior derecha.
  2. En el menú desplegable puede filtrar carpetas/archivos según los criterios que se especifican a continuación:
    • Palabra clave: introduzca el nombre de cualquier archivo o carpeta. Puede utilizar comodines [*] (para representar uno o varios caracteres) y [?] (para representar un carácter aleatorio) en las palabras clave. Por ejemplo, [a*n] puede representar [acción], [atención], o [an].
    • Tipo de archivo: elija Cualquiera, Cualquier carpeta o Cualquier archivo para delimitar la búsqueda.
    • Tamaño (KB): elija Cualquiera, o seleccione igual a, es mayor que, es menor que y, a continuación, introduzca un valor.
    • Fecha de modificación: elija entre Hoy, Ayer, Última semana, Último mes o Personalizar para introducir una fecha de inicio y una fecha de finalización para encontrar versiones dentro de un intervalo de fechas específico.
  3. Para aplicar los filtros haga clic en Filtrar.
  4. Para aplicar los mismos filtros en otra versión de copia de seguridad, solo tiene que cambiar a la versión deseada con la barra de línea temporal.
  5. Para restablecer la búsqueda filtrada, haga clic en el botón de Restablecer.

Nota:

  • El Explorador de copia de seguridad solo está disponible para las tareas de copia de seguridad de varias versiones.
  • El Explorador de copia de seguridad tiene funciones limitadas cuando el destino de copia de seguridad se establece en DSM 5.1 o inferior.
  • La configuración del sistema y las aplicaciones solo se pueden restaurar a través de Restaurar.
  • La función de filtro solo está disponible para tareas de copia de seguridad sin cifrar.
  • Las carpetas o los archivos recuperados con Copiar a... o Descargar perderán los ajustes de privilegios originales.

Para restaurar tareas de copia de seguridad desde otro destino de copia de seguridad:

  1. Haga clic en Restaurar, en la esquina inferior izquierda, y seleccionar Datos o LUN*.
  2. Puede seleccionar y restaurar cualquier tipo de tarea de copia de seguridad:
    • Para restaurar una copia de seguridad de datos: Haga clic en Datos > Restaurar desde repositorios existentes y seleccione el destino de la copia de seguridad que desee para restaurar los datos.
    • Para restaurar una copia de seguridad de LUN: Haga clic en LUN*.
  3. Siga las instrucciones del asistente. Será necesaria la contraseña o clave de cifrado si la tarea que se va a restaurar está cifrada.
  4. Haga clic en Aplicar para finalizar la restauración.

Nota:

  • Embedded DataStation solo puede restaurar carpetas compartidas a dispositivos externos ext3/ext4.
  • Embedded DataStation no es compatible con la restauración para aplicaciones sin un dispositivo externo ext3/ext4.

____
* Solo disponible en determinados modelos.
