Accesibilidad

Gracias a la función de accesibilidad integrada, DiskStation Manager ofrece señales acústicas para guiar a los usuarios con discapacidad visual por la navegación y la gestión de la configuración/los servicios de DSM.

Configuración de accesibilidad de DSM

A fin de maximizar la accesibilidad, recomendamos que configure el entorno operativo como se indica a continuación:

  • Instale el lector de pantalla NVDA (solo disponible para el entorno Windows) en su ordenador local.
  • Instale la última versión del explorador web Mozilla Firefox.
  • Use auriculares o altavoces para ajustar el volumen según sus preferencias.
  • Use NVDA en el modo de enfoque (pulse la tecla NVDA + Espacio para cambiar el modo). La tecla NVDA predeterminada es Insert.
  • Navegue por DSM con las siguientes teclas de acceso rápido:
    Tecla de acceso rápido Descripción
    Tabulador Cambia a otras secciones de la interfaz de usuario
    Teclas de dirección Cambia entre los elementos de una misma sección de la interfaz de usuario
    Intro Abre ventanas/aplicaciones
    Esc Cierra ventanas/aplicaciones

La función de accesibilidad ya funciona en las siguientes áreas de DSM:

  • Página de inicio de sesión de DSM, escritorio y barra de tareas
  • Menú principal
  • Panel de control
  • File Station

Nota:

Para obtener más información sobre el funcionamiento de NVDA, consulte la guía de usuario.

Guías sobre accesos directos de teclado de DSM

Junto con la función de accesibilidad, DSM ofrece un completo conjunto de accesos directos de teclado para optimizar y dar fluidez a la gestión. Pulse Mayús + ? para consultar los accesos directos:

  • Accesos directos de teclado de escritorio: teclas de acceso rápido para operaciones de ventanas y otros
  • Accesos directos de teclado de componentes: Teclas de acceso rápido para controlar los componentes de la interfaz de usuario (p. ej. listas, botones y campos de búsqueda)

Para suspender los accesos directos relacionados con el escritorio, deshabilite la opción Habilitar accesos directos de teclado de escritorio.
