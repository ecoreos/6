Recuperación

Esta página permite a los usuarios administrators del sistema realizar la recuperación. Para permitir que los usuarios accedan a las instantáneas de carpetas compartidas, habilite Hacer visibles las instantáneas para cada carpeta compartida de manera que los usuarios puedan acceder a las instantáneas mediante File Station o el explorador de archivos de Windows.

Carpeta compartida

Para restaurar archivos individuales/carpetas de una carpeta compartida:

  1. Seleccione la carpeta compartida que desea restaurar.
  2. Haga clic en Recuperar.
  3. Seleccione la versión de la instantánea desde la que desea restaurar.
  4. Haga clic en Examinar y copie los archivos/carpetas que desea restaurar al destino.

Para restaurar una carpeta compartida:

  1. Seleccione la carpeta compartida que desea restaurar.
  2. Haga clic en Recuperar.
  3. Seleccione la versión de la instantánea desde la que desea restaurar.
  4. Haga clic en Acción y seleccione Restaurar in situ en la lista desplegable.
  5. Verá un mensaje de confirmación. Haga clic en Sí para confirmar la acción.

Para clonar una instantánea a una carpeta compartida nueva:

  1. Seleccione la carpeta compartida que desea clonar.
  2. Haga clic en Recuperar.
  3. Seleccione la versión de instantánea que desea clonar a una carpeta compartida nueva.
  4. Haga clic en Acción y seleccione Clonar con un nombre nuevo en la lista desplegable.
  5. Se le redirigirá a la página Clonar en Panel de control > Carpeta compartida > Crear. Introduzca el nombre de la carpeta compartida clonada y haga clic en OK para finalizar.

Nota:

  • La carpeta compartida "homes" no se puede restaurar.
  • Las carpetas compartidas clonadas no se pueden cifrar.
  • Si se toma una instantánea cuando todavía se están transfiriendo algunos archivos, no se podrá garantizar la consistencia de los datos.
  • Limitaciones de carpetas compartidas cifradas:
    • Ninguno de los tipos de recuperación está disponible para carpetas compartidas cifradas montadas.
    • Clonar con un nombre nuevo no está disponible para carpetas compartidas cifradas.
    • Las carpetas compartidas cifradas desmontadas se pueden restaurar a partir de instantáneas tomadas previamente con Restaurar in situ.

Para cambiar la réplica de una carpeta compartida:

  1. Seleccione la carpeta compartida replicada que desea cambiar.
  2. Haga clic en Cambio.
  3. Seleccione el nuevo servidor de destino en el caso de que haya varias tareas de réplica de la carpeta compartida.
  4. Haga clic en Cambio para iniciar la acción.

Para realizar la conmutación por error de una réplica de una carpeta compartida:

  1. Seleccione la carpeta compartida replicada de la que desea realizar la conmutación por error.
  2. Haga clic en Conmutación por error > Conmutación por error.
  3. Seleccione la versión de instantánea que desea restaurar.
  4. Haga clic en Conmutación por error para iniciar la acción.

Nota:

  • La conmutación por error solo se puede iniciar tras un error del servidor de origen.

Para proteger de nuevo la réplica de una carpeta compartida:

Cuando se esté realizando la conmutación por error al servidor de destino, la tarea de réplica detendrá la sincronización de datos entre los dos servidores. Deberá proteger de nuevo la tarea de réplica una vez que el servidor de origen original se recupere.

  1. Seleccione la carpeta conmutada por error que desea volver a proteger.
  2. Haga clic en Conmutación por error > Proteger de nuevo.
  3. Seleccione el nuevo servidor de origen.
  4. Seleccione el nuevo servidor de destino en el caso de que haya varias tareas de réplica de la carpeta compartida.
  5. Haga clic en Proteger de nuevo para iniciar la acción.

Nota:

  • Los datos de la carpeta compartida replicada existentes en el nuevo servidor de destino quedarán sobrescritos por los del nuevo servidor de origen.
  • Si la versión de instantánea seleccionada para la conmutación por error no existía en el servidor asociado, es posible que tarde más tiempo en protegerse de nuevo.

Para realizar una conmutación por error de prueba de una carpeta compartida replicada:

  1. Seleccione la carpeta compartida replicada con la que desea hacer la prueba.
  2. Haga clic en Conmutación por error de prueba > Conmutación por error de prueba.
  3. Seleccione el Servidor de prueba en el caso de que haya varias tareas de réplica de la carpeta compartida.
  4. Introduzca el Nombre de la conmutación por error de prueba de la carpeta compartida y, a continuación, seleccione una versión de instantánea para la prueba.
  5. Haga clic en Conmutación por error de prueba para iniciar la acción.

Nota:

  • Si se lleva a cabo un cambio o conmutación por error de una réplica, la conmutación por error de prueba se borrará.
  • No puede crear una tarea de réplica para una carpeta compartida de conmutación por error de prueba.

Para borrar una conmutación por error de prueba de una carpeta compartida replicada:

  1. Seleccione la carpeta compartida replicada de la que desea borrar la conmutación por error de prueba.
  2. Haga clic en Conmutación por error de prueba > Borrar conmutación por error de prueba.
  3. Marque la conmutación por error de prueba que desea borrar.
    Nota: Marque Eliminar datos de conmutación por error de prueba si ya no desea mantener la carpeta compartida creada para la conmutación por error de prueba.
  4. Haga clic en Borrar conmutación por error de prueba para iniciar la acción.
