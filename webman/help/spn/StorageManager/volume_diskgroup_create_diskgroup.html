Crear un Grupo de discos

Este articulo explica cómo crear un grupo de discos en DiskStation.

Antes de comenzar:

  • Asegúrese de que no hay datos importantes en las unidades de disco duro. Todos los datos existentes se eliminarán al crear un nuevo grupo de discos.
  • Vaya a la página HDD/SSD y asegúrese de que el estado de cada unidad de disco duro es Normal o No iniciado.

Para crear un grupo de discos:

  1. Abra Administrador de almacenamiento.
  2. Vaya a Grupo de discos y haga clic en Crear.
  3. Cuando aparezca el asistente de creación de grupos de discos, elija las unidades de disco duro que se deberán utilizar para crear el grupo de discos. A continuación siga los pasos del asistente para finalizar.
