iSCSI Target*

iSCSI es un tipo de servicio de red de área de almacenamiento (SAN, Storage Area Networking) que permite acceder a un almacenamiento de datos consolidado de nivel de bloques. Facilita la transferencia de datos a través de intranets y la gestión del almacenamiento en largas distancias. Asignando iSCSI LUN a iSCSI Target, los servidores cliente pueden acceder a los espacios de almacenamiento mediante su iniciador iSCSI como si fueran discos locales. Dependiendo del modelo, los productos de Synology permiten crear múltiples iSCSI Target en los que se pueden asignar iSCSI LUN.

Nota: El número de iSCSI Target compatibles varía según el modelo del producto de Synology. Para obtener las especificaciones detalladas del producto, consulte el sitio web de Synology.

Para crear un iSCSI Target:

  1. Haga clic en el botón Crear.
  2. Introduzca la información del iSCSI Target, incluido Nombre, IQN y si se habilita o no Autenticación CHAP.
  3. Haga clic en Siguiente.
  4. Configurar la asignación de iSCSI LUN. Seleccione Crear un nuevo iSCSI LUN o Asignar iSCSI LUN existentes.
  5. Haga clic en Siguiente.
  6. Haga clic en Atrás para modificar o en Aplicar para confirmar la configuración.

Explicación de los términos

  • IQN: IQN (iSCSI Qualified Name) es un conjunto de nombre exclusivo para un iSCSI Target. No debe entrar en conflicto con otros IQN de iSCSI Target de otros servidores. IQN debe tener menos de 128 caracteres y puede incluir letras (sin tener en cuenta mayúsculas y minúsculas), números, dos puntos, guiones y puntos. El formato de nomenclatura sugerido es iqn.[fecha].[dominio]:[identificador del dispositivo], por ejemplo: iqn.2000-04.com.synology:storage.diskarray.1.
  • Autenticación CHAP: si selecciona CHAP, el servidor podrá verificar la identidad del cliente. El nombre de usuario puede estar formado por 1 ~ 12 letras y números, mientras que la contraseña puede estar formada por 12 ~ 16 letras y números.
  • Autenticación CHAP mutua: En CHAP mutua, tanto el cliente como el servidor se autenticarán entre sí. Cualquier contraseña CHAP empleada para una autenticación de cliente no deberá configurarse para la autenticación de cualquier servidor, y cualquier contraseña CHAP empleada para la autenticación del servidor no deberá configurarse para la configuración de un cliente.

Para quitar un iSCSI Target:

Puede quitar un iSCSI Target y cualquier iSCSI LUN. Tenga en cuenta que, al quitar un iSCSI LUN, se eliminarán todos los datos del iSCSI LUN.

  1. Seleccione el iSCSI Target que desea quitar.
  2. Haga clic en Quitar.
  3. Si desea quitar los iSCSI LUN asignados, pulse Eliminar también los iSCSI LUN asignados y seleccione los que desee quitar.
  4. Haga clic en OK.

Para editar un iSCSI Target:

  1. Seleccione el iSCSI Target que desea editar.
  2. Haga clic en Editar.
  3. Edite la configuración según sus preferencias. Haga clic en OK para guardar los cambios.

Explicación de los términos

  • Suma de comprobación CRC: habilite Suma de comprobación CRC si se ha producido algún error durante la transmisión de datos.
  • Varias sesiones: permita que varias sesiones de uno o varios iniciadores se conecten al mismo iSCSI Target al mismo tiempo. Para proteger sus archivos de daños o pérdida inesperada de datos, se sugiere encarecidamente que use un sistema de archivos compartido o un sistema de clúster tal como VMware Virtual Machine File System (VMFS) y Oracle Cluster File System (OCFS).
  • Enmascaramiento: el enmascaramiento permite que a un iSCSI Target accedan exclusivamente determinados IQN del iniciador iSCSI.

Para habilitar o deshabilitar un iSCSI Target:

  1. Seleccione el iSCSI Target que desea habilitar o deshabilitar.
  2. Haga clic en Habilitar o en Deshabilitar.

Para registrar iSCSI Target con un servidor iSNS

iSNS (Internet Storage Name Service) ofrece el mecanismo de gestiones centralizadas. Puede registrar todos los iSCSI Target con un servidor iSNS.

  1. Haga clic en el botón iSNS y marque la casilla de verificación de Habilitar iSNS.
  2. Escriba la dirección IP del servidor iSNS. Haga clic en OK para aplicar la configuración.
