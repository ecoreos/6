Depuración de datos*

La depuración de datos es una función de mantenimiento de datos que inspecciona los volúmenes o los grupos de discos. Hay dos opciones de depuración de datos:

  • Depuración de RAID: esta función se puede usar con volúmenes o grupos de discos que implementen SHR (compuestos por tres o más discos), RAID 5 o RAID 6. Puede reparar cualquier inconsistencia detectada en los datos.
  • Depuración del sistema de archivos: esta función se puede utilizar con volúmenes en el sistema de archivos Btrfs. Si se detectan datos que no son coherentes con la suma de comprobación, la ruta de archivo se registrará en el Centro de registros. Recomendamos encarecidamente que utilice una copia de seguridad para reparar los datos.

Le recomendamos que realice regularmente una depuración de datos para garantizar la consistencia de los datos y evitar la pérdida de datos importantes en caso de que una unidad de disco duro falle.

Para realizar la depuración de datos:

  1. Abra Administrador de almacenamiento.
  2. Vaya a Volumen o Grupo de discos.
  3. Seleccione el volumen o grupo de discos en el que desea realizar una depuración de datos.
  4. Haga clic en Administrar.
  5. Seleccione Depuración de datos.
  6. Seleccione el tipo de depuración de datos que desee (el sistema omitirá automáticamente este paso si solo hay un tipo disponible).

Nota:

  • La depuración de datos solo se puede realizar cuando el estado del volumen o grupo de discos sea Normal.
  • Si el sistema se apaga antes de que el proceso termine, tendrá que reiniciar manualmente la depuración de datos desde el principio.
