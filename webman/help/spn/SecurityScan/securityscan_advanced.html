Avanzado

Puede visualizar y cambiar la configuración avanzada para el Consejero de seguridad.

Referencia de seguridad

Puede elegir qué reglas de seguridad comprobará el Consejero de seguridad seleccionando una Referencia de seguridad. La cantidad y el tipo de reglas varía para cada Referencia de seguridad. Realice la selección en función de sus necesidades. ¿Va a utilizar DiskStation para fines profesionales o domésticos? Puede elegir las siguientes opciones:

  • Para el uso doméstico y personal: El Consejero de seguridad realizará las comprobaciones básicas para proteger su DiskStation frente a las amenazas de Internet.
  • Para trabajo y negocios: El Consejero de seguridad realizará comprobaciones completas para satisfacer los requisitos de cumplimiento de las organizaciones.
  • Personalizado: Seleccione las reglas de seguridad que desea que compruebe el Consejero de seguridad.

Programa de exploración

Puede definir un programa de exploración regular si hace clic en Habilite un programa regular de exploración y elige el momento en el que desea que el Consejero de seguridad realice la exploración. El Consejero de seguridad comprobará las reglas de acuerdo con la Referencia de seguridad que ha elegido.
