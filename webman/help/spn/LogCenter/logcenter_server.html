Recibiendo registros

DiskStation puede servir de Syslog Server para recibir registros de dispositivos de cliente en la red. Existen tres formatos compatibles para recibir registros: BSD (RFC 3164), IETF (RFC 5424) y Personalizado. Todos los registros recibidos se almacenan en la base de datos de registros.

Antes de comenzar:

Vaya a Centro de registros > Configuración de archivo y elija una ubicación para almacenar y archivar registros.

Para configurar la recepción de registros estándar:

  1. Vaya a Centro de registros > Recibiendo registros > Estándar.
  2. Puede habilitar la recepción de registros BSD y IETF si marca la opción Recibir registros de otros dispositivos en el formato que necesite.
  3. Seleccione UDP o TCP en Protocolo de transferencia.
  4. Indique un número de puerto para recibir mensajes de syslog en Puerto. El número de puerto predeterminado es 514.
  5. Si desea transferir registros de manera segura, marque la opción Habilitar conexión segura (SSL). Esta opción solo está disponible cuando TCP está seleccionado.
  6. Haga clic en Aplicar para guardar la configuración.

Para configurar la recepción de registros personalizada:

  1. Vaya a Centro de registros > Recibiendo registros > Personalizado.
  2. Haga clic en Crear.
  3. Escriba un nombre de regla único en Nombre.
  4. Escriba una regla de análisis en Parámetros de regla. Consulte la nota que aparece más abajo para obtener información detallada acerca de la creación de analizadores de patrones.
  5. Seleccione UDP o TCP en Protocolo de transferencia.
  6. Indique un número de puerto para recibir mensajes de syslog en Puerto. El número de puerto predeterminado es 514.
  7. Si desea transferir registros de manera segura, marque la opción Habilitar conexión segura (SSL). Esta opción solo está disponible cuando TCP está seleccionado.
  8. Haga clic en OK para guardar la configuración.

Nota:

Obtenga información detallada acerca de los analizadores de patrones aquí. También puede descargar un patrón de muestra a modo de referencia.

Para administrar las reglas personalizadas:

  1. Seleccione una regla personalizada existente.
  2. Elija de entre las siguientes:
    • Editar: edita la regla de personalización seleccionada.
    • Eliminar: elimina la regla de personalización seleccionada.
    • Habilitar: habilita la regla de personalización seleccionada.
    • Deshabilitar: deshabilita la regla de personalización seleccionada.

Exportar certificado

Si ha marcado la casilla Habilitar conexión segura (SSL), puede hacer clic en el botón Exportar certificado para exportar el certificado que ha utilizado para autenticar otros dispositivos al enviar registros a DiskStation.
