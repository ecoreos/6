Biblioteca multimedia

El servicio de biblioteca multimedia puede examinar automáticamente archivos multimedia, tales como fotos, música y vídeos en el DiskStation, y compilarlos en la biblioteca multimedia para que aplicaciones multimedia los utilicen.

Con biblioteca multimedia también puede configurar la calidad de miniaturas para fotos y vídeos, administrar configuraciones de conversión para vídeos y crear carpetas indexadas a las que podrán acceder dispositivos DMA y aplicaciones multimedia de DSM.
