Idioma

Puede seleccionar el idioma que prefiera para la interfaz de usuario de DSM, las notificaciones de escritorio de Google Chrome, el correo electrónico, los SMS, las notificaciones de mensajería instantánea y la página de código de la aplicación.

Idioma de visualización

Puede elegir el idioma de visualización que prefiera para las notificaciones de escritorio de DSM y Google Chrome, o hacer que coincida con el valor por defecto del navegador.

Idioma de notificación

Puede ajustar el idioma que prefiera para las notificaciones de correo electrónico, SMS, Windows Live Messenger y Skype. También se aplicará a las notificaciones recibidas en Windows Phone.

Sin embargo, el idioma de notificación en un dispositivo iOS o Android depende de la configuración de idioma de su dispositivo móvil.

Página de código

Para aplicaciones que convierten cadenas no Unicode en Unicode, como el servicio FTP, la compatibilidad con UPnP o la indexación de metadatos de música, deberá elegir la página de código adecuada para ellas en el menú desplegable Página de código.
