Clonar una carpeta compartida*

Puede clonar una carpeta compartida en volúmenes Btrfs para crear de forma casi instantánea copias de una carpeta compartida existente. La clonación de una carpeta compartida consume únicamente una parte del almacenamiento en el momento de su creación.

Para clonar una carpeta compartida:

  1. Vaya a Panel de control > Carpeta compartida.
  2. Seleccione la carpeta compartida que desee del sistema de archivos Btrfs.
  3. Haga clic en Crear > Clonar.
  4. Especifique un nombre y una descripción para la nueva carpeta compartida y modifique otras opciones si es necesario. (Para obtener más información sobre estas opciones, consulte Crear una carpeta compartida).

Nota:

  • Solo se pueden clonar las carpetas compartidas en volúmenes Btrfs.
  • La carpeta compartida clonada se creará en el mismo volumen que la carpeta compartida original.
  • La configuración de protección avanzada de la integridad de los datos se aplicará a la carpeta compartida clonada y no puede cambiarse.

____
*Solo disponible en ciertos modelos.
