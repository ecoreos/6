Administrar permisos avanzados de carpetas compartidas

La página en Panel de control > Carpeta compartida > Editar > Avanzada incluye las siguientes opciones para ajustar aun más los permisos de acceso de una carpeta compartida.

Permisos avanzados de uso compartido

Los permisos avanzados de uso compartido ofrecen una capa de control adicional para administrar los permisos de acceso de carpetas compartidas. Cuando se habilitan, usuarios y grupos pueden ver o modificar los contenidos de una carpeta compartida solo si el usuario o grupo han recibido los permisos avanzados de uso compartido y los permisos de Windows ACL (situados en Carpeta compartida > Editar > Permisos).

Para habilitar los permisos avanzados de uso compartido

  1. Marque la casilla Habilitar permisos avanzados de uso compartido.
  2. Haga clic en Permisos avanzados de uso compartido para modificar los permisos avanzados de uso compartido de la carpeta compartida.

Nota:

  • Los permisos avanzados de uso compartido se aplican cuando un usuario accede a una carpeta compartida a través de los siguientes servicios de archivos: Windows File Sharing, Apple File Sharing, File Station, FTP y WebDAV.

Configuración avanzada

Si fuera necesario, puede colocar mayores restricciones a los usuarios cuando accedan a carpetas compartidas a través de File Station, FTP o WebDAV.

  • Deshabilitar búsqueda de directorios: Cuando se habilita esta opción se limitan los contenidos de la carpeta compartida que pueden visualizar los usuarios.
  • Deshabilitar modificación de archivos existentes: Cuando se habilita esta opción se limitan los archivos de la carpeta compartida que los usuarios pueden mover, eliminar o modificar. Observe que los usuarios todavía podrán visualizar, descargar/cargar, copiar o descomprimir los contenidos de la carpeta compartida.
  • Deshabilitar descarga de archivos: Cuando se habilita esta opción se limitan los contenidos de la carpeta compartida que pueden descargar los usuarios.

Nota:

Las restricciones anteriores no se aplicarán a usuarios que pertenezcan al grupo de administrators.
