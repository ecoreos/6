Privilegios

Aquí se pueden ver y editar los privilegios de usuarios y grupos en servicios y aplicaciones individuales. DSM proporciona una configuración flexible para controlar los privilegios en aplicaciones, facilitando la administración del acceso de usuarios y grupos a diferentes aplicaciones.

Para modificar los privilegios de usuarios y grupos en aplicaciones:

  1. Vaya al Panel de control > Privilegios.
  2. Seleccione la aplicación que desea modificar. Haga clic en Editar.
  3. Haga clic en Usuario para visualizar los privilegios individuales o Grupo para visualizar los de un grupo.
  4. Para cada usuario o grupo, marque Permitir para permitir el acceso o Denegar para denegar el acceso a la aplicación seleccionada. Según la aplicación, podrá seleccionar Por IP para permitir/denegar acceso en base a la dirección IP.
  5. Haga clic en Aceptar para guardar sus cambios.

Para ver los privilegios de aplicación de usuarios con diferentes direcciones IP de red:

  1. Vaya al Panel de control > Privilegios.
  2. Haga clic en Visor de permisos.
  3. Seleccione un usuario en Nombre de usuario y a continuación introduzca la dirección IP para la que desea visualizar los permisos en Dirección IP.

Para configurar los privilegios de aplicación de todos los usuarios:

  1. Vaya al Panel de control > Privilegios.
  2. Seleccione la aplicación que desea modificar. Haga clic en Editar.
  3. Haga clic en Privilegios predeterminados y marque Otorgar este privilegio a todos.
  4. Haga clic en Aceptar para guardar sus cambios.

Nota:

  • Los privilegios de admin y guest no se pueden modificar aquí.
  • Cuando encuentre conflictos de privilegios, la prioridad de los privilegios es la siguiente: Denegar > Permitir.
  • No todos los paquetes y servicios son compatibles con la configuración de privilegios de acceso Por IP.
