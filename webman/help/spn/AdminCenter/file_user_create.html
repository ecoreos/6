Crear un usuario

Se pueden crear y administrar varias cuentas de usuarios en DiskStation

Para crear un usuario:

  1. Vaya a Panel de control > Usuario.
  2. Haga clic en el botón Crear y a continuación cree el usuario. Esta acción abrirá el Asistente de creación de usuario.
  3. En la página de Información de usuario especifique las siguientes opciones:
    • Nombre: introduzca un nombre para el usuario. Esta sección es obligatoria.
    • Descripción: escriba una breve descripción para el usuario.
    • Correo electrónico: escriba una dirección de correo electrónico para el usuario. Notificaciones del sistema, como mensajes de restablecimiento de contraseñas, se enviarán a la dirección introducida aquí.
    • Enviar un correo electrónico de notificación al usuario recién creado: si esta opción está habilitada, el sistema enviará un mensaje al usuario para informarle de la nueva cuenta. Esta opción requiere que se habilite el sistema de notificaciones por correo electrónico en Menú principal > Panel de control > Notificación > Correo electrónico. De forma predeterminada, la contraseña del usuario no se muestra en mensaje de notificación. Sin embargo, si desea mostrar la contraseña en el mensaje de notificación, marque la casilla Mostrar la contraseña de usuario en el correo de notificación.
    • No permitir que el usuario cambie la contraseña de la cuenta: si se habilita esta opción, el usuario no será capaz de cambiar la contraseña utilizada para iniciar sesión en DSM.
  4. En la página de Grupo > Crear > Asistente de creación de grupo, especifique los grupos a los que pertenecerá el nuevo usuario (por ejemplo, usuarios, administrators, etc.).
  5. En la página de Asignar permisos de carpetas compartidas, elija las carpetas a las que puede acceder el nuevo usuario modificando sus privilegios de acceso. La prioridad de los privilegios es la siguiente: Sin acceso > Lectura/Escritura > Sólo lectura.
    • Vista previa: esta columna muestra los privilegios reales de acceso a la carpeta compartida del usuario de acuerdo con la configuración actual y los grupos a los que pertenece el usuario.
    • Permisos de grupo: esta columna muestra los privilegios de acceso a la carpeta compartida asignados de acuerdo con los grupos de usuarios a los que pertenece el usuario.
  6. En la página de Configuración de cuota de usuariopuede controlar el volumen de almacenamiento que puede utilizar el nuevo usuario. Para restringir la cantidad de espacio disponible para el usuario, marque la casilla debajo de Habilitar cuota para cada volumen/carpeta compartida. A continuación introduzca la cantidad máxima de espacio que se puede utilizar en el campo Cuota (por ejemplo 10 GB).
  7. En la página Asignar permisos de aplicaciones puede controlar los servicios a los que puede acceder el usuario.
  8. En la página Ajuste del límite de velocidad de grupo puede habilitar un límite de velocidad para los diferentes servicios (por ejemplo, File Station, FTP, etc.) para restringir la cantidad de ancho de banda consumida por el usuario al transferir archivos. Puede elegir una de las siguientes opciones para cada servicio:
    • Ninguno: los ajustes del límite de velocidad serán los mismos que los del grupo al que pertenece el usuario.
    • Constante: se aplicará un límite de velocidad fijo constante. Podrá especificar los límites de velocidad de carga y descarga en los campos de la izquierda.
    • Variable: se pueden especificar dos límites de velocidad diferentes para el usuario de acuerdo con un programa. Haga clic en el botón Personalizar para modificar el ajuste del límite de velocidad y el programa.
  9. Tras confirmar el ajuste haga clic en Aplicar para terminar.

Limitación de nombre de usuario:

  • El nombre de usuario no distingue entre mayúsculas y minúsculas y puede incluir de 1 a 64 caracteres Unicode, excluyendo los siguientes símbolos:
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • El primer carácter no puede ser un signo menos o un espacio y el último no puede ser un espacio.
  • Los siguientes nombres de usuario están reservados para el uso del sistema: mailer-daemon, postmaster.
  • La descripción del usuario puede incluir hasta 64 caracteres Unicode que se pueden mostrar.

Limitación de la contraseña:

  • La contraseña distingue entre mayúsculas y minúsculas y está limitada a 127 caracteres que se puedan mostrar, incluidas letras, números, signos y espacios.

Limitación de cuota:

  • La cuota máxima de almacenamiento que se puede introducir para cada usuario es de aproximadamente 4 TB (4095 GB).
