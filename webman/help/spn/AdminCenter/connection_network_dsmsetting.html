Configuración de DSM

En Panel de control > Red > Configuración de DSM, puede cambiar el número de puerto que los clientes utilizan para acceder a DiskStation.

Para cambiar el número de puerto predeterminado:

Las conexiones HTTP y HTTPS siempre están activas en DSM. El puerto HTTP predeterminado es 5000 y el del puerto HTTPS, 5001.

  1. Escriba el número de puerto en el campo HTTP o HTTPS.
  2. Haga clic en Aplicar.

Para redirigir las conexiones HTTP a HTTPS:

Puede redirigir a los clientes al puerto HTTPS cuando intenten acceder a DiskStation a través del puerto HTTP.

  1. Seleccione Redirigir automáticamente las conexiones HTTP a HTTPS (se excluyen Web Station y Photo Station).
  2. Haga clic en Aplicar.

Puertos usados para la conexión HTTPS:

  • DSM: el número de puerto predeterminado es 5001, de manera que puede acceder a DSM introduciendo la siguiente dirección: https://server-hostname:5001/
  • Web Station y Photo Station: el número de puerto predeterminado es 443, de manera que puede acceder a Web Station y Photo Station introduciendo la siguiente dirección: https://server-hostname/

Para detectar un DiskStation en la red:

Encuentre su DiskStation a través de la detección de redes de Windows o la detección del servicio Bonjour.

  1. Seleccione un servicio para detectar su DiskStation en la red:
    • Activar la detección de redes de Windows
    • Activar detección del servicio Bonjour
  2. Haga clic en Aplicar.

Dominio

En Dominio, puede especificar un nombre de dominio personalizado para acceder a DiskStation.

Para habilitar un dominio personalizado:

  1. Marque Habilitar dominio personalizado.
  2. Especifique un nombre de dominio aplicado para su DiskStation.
  3. También puede marcar Habilitar HSTS para forzar las conexiones cifradas HTTPS en este dominio.
  4. Haga clic en Aplicar.

Nota:

  • Los nombres de dominio solo pueden utilizarse para una única aplicación en su DiskStation.
  • Si el nombre de dominio no supera los 15 caracteres de longitud y cumple con las convenciones de nomenclatura de NetBIOS, el sistema registrará y transmitirá el nombre del dominio (p. ej., MyDS) de forma automática. Los usuarios de Windows pueden acceder al escritorio de DSM en la misma red local utilizando una URL parecida: http://MyDSM o https://MyDSM
  • Para acceder a aplicaciones a través de URL que utilicen nombres de dominio, utilice el puerto HTTP estándar (80) o el puerto HTTPS estándar (443).
