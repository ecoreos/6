Base de datos multimedia

DiskStation usa una base de datos multimedia para realizar un seguimiento de los archivos multimedia y de las entradas del registro almacenadas en el origen multimedia (carpetas compartidas creadas en sus dispositivos de almacenamiento externos, tales como usbshare1).

Importante: Si desea que los servicios multimedia de DSM (tales como Servidor multimedia, iTunes, Audio Station y Download Station) accedan a los archivos multimedia almacenados en los dispositivos de almacenamiento externos conectados al puerto USB de su DiskStation, o si desea habilitar los servicios de registro de DSM y ver los registros en Menú principal > Información del sistema > Registro, primero es necesario habilitar la base de datos multimedia. En caso contrario, se desactivarán todos los servicios relacionados.

Para habilitar la base de datos multimedia:

  1. Marque Habilitar la base de datos multimedia.
  2. Realice cualquiera de las siguientes operaciones:
    • Elija Configuración automática si desea que DSM seleccione automáticamente una carpeta compartida para almacenar la base de datos multimedia. Si se quita el dispositivo de almacenamiento externo que contiene la carpeta compartida, DSM seleccionará automáticamente cualquier otra carpeta compartida para almacenar la base de datos.
    • Elija Seleccione una carpeta compartida para almacenar su base de datos y, a continuación, seleccione del menú desplegable para seleccionar una carpeta compartida para almacenar la base de datos.
  3. Haga clic en OK.

Advertencia: Para impedir que se dañe la base de datos, asegúrese de expulsar su dispositivo de almacenamiento en Menú principal > Panel de control > Dispositivos externos antes de quitarla de DiskStation.

Para incluir o excluir carpetas compartidas del origen de multimedia:

  1. Haga clic en Origen de multimedia.
  2. Marque o deje sin marcar las casillas de verificación situadas junto a la(s) carpeta(s) compartida(s) para realizar su selección.
  3. Haga clic en OK.

Se indexarán los archivos multimedia y los registros de las carpetas compartidas seleccionadas y la información indexada se añadirá a la base de datos multimedia.

Para actualizar la base de datos multimedia:

  1. Haga clic en Actualizar base de datos. Se sustituirán todos los datos de la base de datos por la información reindexada acerca del origen de multimedia actual.
  2. Haga clic en OK.

Observación: Si ya no se usa una enorme cantidad de datos de la base de datos multimedia (por ejemplo, si desconecta un disco USB que ya no se va a volver a usar con DiskStation, con lo que habrá una información indexada redundante en la base de datos), actualizar la base de datos multimedia puede ayudar a borrar todos los datos sin usar o a reducir el tamaño de la base de datos. No obstante, esto requiere reiteradas lecturas y escrituras en el disco, lo cual podría consumir recursos del sistema y reducir la vida útil del disco. Use esta función con prudencia.
