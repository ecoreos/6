Programador de tareas

El Programador de tareas le permite programar y ejecutar servicios o scripts a horas predefinidas.

Tareas compatibles

Se pueden crear tareas para hacer lo siguiente:

  • Script definido por usuario: ejecute scripts o comandos definidos por el usuario.
  • Papelera de reciclaje: vacíe la papelera de reciclaje de ciertas carpetas compartidas específicas. Además, la configuración de la Política de retención le permitirá controlar los archivos que se eliminarán permanentemente al vaciar la papelera de reciclaje. Por ejemplo, los archivos se pueden eliminar o conservar de acuerdo con su tamaño, hora de eliminación o formato de archivo.
  • Control de pitido: hace que DiskStation emita un pitido.
  • Servicio: inicie o detenga los servicios de red, como el Servicio de archivos Windows, FTP, Bloqueo automático, etc.

Administrar tareas

La página principal del Programador de tareas muestra todas las tareas existentes y ofrece opciones para crear, editar o ejecutar tareas.

Para crear una tarea programada:

Las tareas programadas son tareas que se ejecutan en una fecha, una hora o un intervalo determinados.

  1. Vaya a Panel de control > Programador de tareas, haga clic en Crear y seleccione Tareas programadas.
  2. Seleccione el tipo de tareas que desee programar: Script definido por usuario, Papelera de reciclaje, Control de pitido o Servicio.
  3. Cuando aparezca la ventana Crear tarea, haga clic en General y modifique los siguientes ajustes:
    • Configuración general: escriba un nombre para la tarea, seleccione un usuario para el que se ejecutará la tarea y active o desactive la tarea.
  4. Haga clic en Programa y modifique los siguientes ajustes:
    • Fecha: indique cuándo ejecutar la tarea. Las tareas se pueden programar para que se ejecuten determinados días de la semana (por ejemplo, diariamente, los fines de semana o todos los domingos, etc.) o en determinadas fechas (p. ej., 21/10/2015).
    • Hora: indique cuántas veces se debe ejecutar la tarea cada día, así como la primera y la última hora de ejecución.
  5. Haga clic en Configuración de tarea para organizar las opciones específicas de la tarea (solo para determinadas tareas).
    • Para recibir los detalles de ejecución de la tarea, seleccione Enviar detalles de ejecución por correo electrónico.
    • Para recibir notificaciones solo en caso de que se produzcan anomalías, seleccione Enviar detalles de ejecución solo cuando el script finalice de forma anómala.
    • Para personalizar el contenido de la tarea, escriba la secuencia en Script definido por usuario.
    • Si ha seleccionado Papelera de reciclaje o Servicio en el paso 2, puede decidir cómo y cuándo se vaciarán las papeleras de reciclaje o qué servicio se debe iniciar/detener.
  6. Haga clic en OK para guardar la configuración.

Para crear una tarea desencadenada:

Las tareas desencadenadas son tareas que se ejecutan antes/después de la finalización de eventos específicos.

  1. Vaya a Panel de control > Programador de tareas, haga clic en Crear y seleccione Tareas activadas.
  2. Seleccione Script definido por usuario.
  3. Cuando aparezca la ventana Crear tarea, haga clic en General y modifique los siguientes ajustes:
    • Configuración general: escriba un nombre para la tarea, seleccione un usuario para el que se ejecutará la tarea, seleccione un evento o una pretarea que activará la tarea, y active o desactive la tarea.
  4. Haga clic en Configuración de tarea para organizar las opciones específicas de la tarea (solo para determinadas tareas).
    • Para recibir los detalles de ejecución de la tarea, seleccione Enviar detalles de ejecución por correo electrónico.
    • Para recibir notificaciones solo en caso de que se produzcan anomalías, seleccione Enviar detalles de ejecución solo cuando el script finalice de forma anómala.
    • Para personalizar el contenido de la tarea, escriba la secuencia en Script definido por usuario.
  5. Haga clic en OK para guardar la configuración.

Nota:

  • Al crear una tarea desencadenada, es posible que tenga que especificar sus pretareas. La tarea creada se ejecutará solo después de que la pretarea se haya llevado a cabo.
  • Una vez haya habilitada la función Enviar detalles de ejecución solo cuando el script finalice de forma anómala, el sistema no enviará los detalles si el script termina en 0.

Para editar la configuración de la tarea:

  1. Seleccione una tarea existente.
  2. Haga clic en Editar.
  3. Aparece una ventana en la que puede modificar la configuración de la tarea.

Para eliminar una tarea:

  1. Seleccione una tarea existente.
  2. Haga clic en Acción > Eliminar.

Para habilitar o deshabilitar una tarea:

  1. En la página principal del Programador de tareas, active o desactive la casilla de verificación adecuada bajo Habilitado para habilitar o deshabilitar una tarea.
  2. Haga clic en Guardar.

Para ejecutar manualmente una tarea:

  1. Seleccione una o varias tareas existentes.
  2. Haga clic en Ejecutar.

Limitaciones del Programador de tareas:

  • Los nombres de las tareas pueden incluir 512 caracteres visibles.
  • Número máximo de tareas: 1024.
  • Longitud máxima de la ruta del script o de la línea de comando: 4095.
  • La duración del pitido se puede ajustar entre 1 y 600 segundos.
