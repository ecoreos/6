Base de datos del sistema

DiskStation usa una base de datos del sistema para realizar un seguimiento de los datos del paquete y de las entradas del registro almacenadas en las carpetas compartidas creadas en sus dispositivos de almacenamiento externos, tales como usbshare1.

Importante: Si desea acceder a datos de paquetes almacenados en los dispositivos externos de almacenamiento unidos a su DiskStation, o si desea ver los registros de DSM en Menú principal > Centro de registros, primero deberá habilitar la base de datos del sistema. En caso contrario, no se podrá acceder a ninguno de los servicios relacionados.

Para habilitar la base de datos del sistema:

  1. Marque Habilitar la base de datos del sistema.
  2. Realice cualquiera de las siguientes operaciones:
    • Elija Configuración automática si desea que DSM seleccione automáticamente una carpeta compartida para almacenar la base de datos del sistema. Si se quita el dispositivo de almacenamiento externo que contiene la carpeta compartida, DSM seleccionará automáticamente cualquier otra carpeta compartida para almacenar la base de datos.
    • Elija Mueva la base de datos actual a la carpeta compartida seleccionada, y a continuación elija la opción del menú desplegable para mover la base de datos actual a otra carpeta compartida creada en su dispositivo externo.
    • Elija Elegir una de las bases de datos existentes, y a continuación elija la opción del menú desplegable para seleccionar una base de datos existente si hay más de un dispositivo externo conectado a su DiskStation.
  3. Haga clic en Aplicar.

Advertencia: Para evitar que la base de datos se dañe, asegúrese de extraer el dispositivo de almacenamiento en Menú principal > Panel de control > Dispositivos externos > Dispositivos externos o haga clic en el icono de dispositivos externos en la esquina superior derecha antes de quitarlos de DiskStation.
