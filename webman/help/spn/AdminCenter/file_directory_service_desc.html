Servicio de directorio

Su DiskStation puede unirse a un servicio de directorio existente, como un dominio de Windows o cliente de LDAP. Puede administrar los privilegios de acceso de los usuarios de dominio o LDAP a carpetas compartidas y aplicaciones de DSM, y habilitar algunos servicios de inicio para ellos. Vaya a Panel de control > Servicio de directorio para vincular su DiskStation a un servicio de directorio.
