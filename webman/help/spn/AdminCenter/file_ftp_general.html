General

En Panel de control > Servicios de archivos > FTP, puede habilitar el límite de velocidad, el registro de transferencia de archivos, los parámetros de seguridad y otras configuraciones avanzadas en la sección General.

Habilitar el registro de transferencia de archivos vía FTP

Haga clic en Configuración avanzada para habilitar esta función y grabar los registros de archivos cargados o descargados.

Nota:

Habilitar el registro de transferencia de archivos vía FTP puede afectar al rendimiento del sistema.

Cambio de los directorios root de usuario

Haga clic en Configuración avanzada y marque Cambiar directorios root de usuario. Así, los usuarios seleccionados sólo pueden acceder a las carpetas especificadas cuando inician sesión en DSM a través de FTP.

Para crear una regla de cambio de directorio root:

  1. En Configuración avanzada marque Seleccionar usuario.
  2. Haga clic en Agregar.
  3. En Usuario o grupo seleccione a quién se le aplicará esta regla.
  4. Seleccione el directorio root que estará disponible cuando el usuario inicie sesión. Elija entre:
    • Inicio de usuario: el usuario llega a su directorio inicial al iniciar sesión.
    • Otro directorio: elija una carpeta compartida como directorio root y haga clic en Seleccionar. Tenga en cuenta que los permisos de carpeta para el usuario o grupo se deben comprobar primero. Un usuario o grupo sin suficientes permisos de carpeta no podrán iniciar sesión.
  5. Haga clic en OK.
  6. Tras agregar todas las reglas, haga clic en Aplicar para guardar las reglas.

Nota:

  • El orden de prioridad de las reglas se establece en función de sus posiciones en la lista.
  • Arrastre y suelte para reordenar las reglas de la lista.

Habilitar FTP anónimo

Haga clic en Configuración avanzada y marque la casilla para permitir que cualquiera inicie sesión en el servicio FTP del dispositivo DiskStation (con la cuenta anonymous o ftp).

Cambiar root anónima

Haga clic en Configuración avanzada y marque la opción para cambiar el directorio de inicio de sesión del usuario de FTP anónimo. Seleccione una carpeta compartida desde el menú desplegable.

Aplicar permisos UNIX predeterminados

Haga clic en Configuración avanzada y marque la opción para aplicar permisos UNIX predeterminados al cargar o crear archivos y carpetas. Los permisos aplicados son los mismos que los aplicados por el comando UNIX umask. Cuando se habilita esta opción, los permisos UNIX son 644 para archivos y 755 para carpetas. Cuando se deshabilita esta opción, los permisos UNIX son 666 para archivos y 777 para carpetas. El valor predeterminado de umask es 022.

Nota:

  • Para las carpetas compartidas con Windows ACL (todos los recursos compartidos excepto "photo" y los recursos compartidos en discos externos), ejecute el comando chmod en el cliente Linux o FTP para cambiar los tipos de permisos de carpeta y de archivo de Windows ACL a UNIX.
  • Si se habilita esta opción se pueden provocar problemas de permisos que no mantienen la consistencia entre protocolos diferentes. Para evitar inconsistencias, sugerimos no habilitar esta opción.

Habilitar el límite de velocidad

Habilite un límite de velocidad para controlar el ancho de banda utilizable máximo para usuarios y grupos al transferir datos a través de FTP.

Para habilitar el límite de velocidad:

  1. Marque la casilla con la indicación de Habilitar límite de velocidad o Habilitar el límite de velocidad de acuerdo con un programa si desea imponer un límite de velocidad solo en determinados momentos. Este ajuste se realiza a nivel de servidor y anulará los ajustes de usuarios individuales y de grupos. Si elige Sin límite de velocidad deshabilitará los ajustes del límite de velocidad para todos los usuarios y grupos.
  2. Haga clic en Ajustes del límite de velocidad para modificar la configuración para determinados usuarios o grupos. Para cada usuario y grupo puede elegir una de las siguientes opciones:
    • Aplicar configuración de grupo: los ajustes del límite de velocidad serán los mismos que los del grupo al que pertenece el usuario.
    • Ninguno: no se aplica ningún límite de velocidad cuando miembros del grupo transfieren archivos con el servicio.
    • Constante: se aplicará un límite de velocidad fijo constante para el usuario o los miembros del grupo. Los límites de velocidad de carga y descarga se pueden especificar en los campos a la izquierda.
    • Personalizado: se pueden especificar dos límites de velocidad diferentes al usuario o los miembros del grupo de acuerdo con un programa. Haga clic en el botón Personalizar para modificar los ajustes del límite de velocidad y definir un programa.
  3. Haga clic en Aplicar.

Nota:

  • Para obtener más información sobre la dinámica del límite de velocidad de grupo, consulte aquí.
  • Cuando esté definiendo el límite de velocidad para un usuario, la sección Resultado mostrará el límite de velocidad actual del usuario de acuerdo con el ajuste individual y de grupo. Por ejemplo, si el usuario "Aquiles" tiene un límite individual de descarga de 100 kB/s, pero pertenece a un grupo de usuarios con un límite de descarga de 500 KB/s, entonces la sección de Resultados mostrará "0/500".
