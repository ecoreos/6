Editar una carpeta cifrada

Una vez cifrada la carpeta, puede usar el menú desplegable Cifrado para editar la carpeta.

Los elementos mostrados en el menú dependen del estado de la carpeta compartida seleccionada:

  • Si selecciona una carpeta compartida sin cifrar, se desactivará el menú desplegable Cifrado.
  • Si selecciona una carpeta compartida cifrada que está montada, puede elegir Desmontar en el menú desplegable Cifrado para desmontar la carpeta, o bien Exportar clave para guardar la clave de cifrado.
  • Si selecciona una carpeta compartida cifrada que está montada, puede elegir Montar en el menú desplegable Cifrado y escribir o importar la clave de cifrado para montar la carpeta para acceder a ella.
    Nota:
  1. Es muy importante recordar la clave de cifrado. Se recomienda encarecidamente exportar y guardar la clave de cifrado. Si olvida la clave o la pierde, no podrá de ningún modo rescatar los datos cifrados.
  2. Si la carpeta compartida cifrada está montada, no podrá cambiarla de nombre ni cambiar su ubicación de volumen. Sólo podrá realizar cambios después de desmontarla.
