Hora

Puede comprobar la hora actual, ajustar manualmente la fecha y hora del servidor o hacer que se ajusten automáticamente usando un servidor de hora de la red.

Puede comprobar la hora actual y establecer la fecha y la hora del servidor.

Para configurar la zona horaria:

Elija una opción del menú desplegable Zona horaria.

Para ajustar la fecha y hora de forma manual:

Elija Manualmente y seleccione la fecha y hora.

Para sincronizar la fecha y la hora con un servidor NTP:

  1. Elija Sincronizar con un servidor NTP e introduzca la dirección de su servidor de hora de red o elija un servidor del menú desplegable.
  2. Haga clic en Actualizar ahora para asegurarse de que DiskStation puede sincronizar correctamente la hora con el servidor.
  3. Haga clic en Aplicar.

Nota: Para sincronizar la fecha y la hora con un servidor de hora de la red, asegúrese de haber introducido una dirección IP del servidor DNS válida en Menú principal > Panel de control > Red y de que su DiskStation tiene conexión a Internet.

Acerca del servidor de protocolo de tiempo de redes (NTP)

La finalidad principal del protocolo de tiempo de redes (NTP) es sincronizar relojes de sistemas informáticos a través de la red de datos.

A continuación se indican varios servidores NTP recomendados:

  • time.apple.com
  • time.asia.apple.com
  • time.euro.apple.com
  • time.nist.gov

Para obtener más información acerca del servidor NTP, visite http://www.eecis.udel.edu/~mills/ntp/servers.html
