Copia de seguridad de configuración

Vaya a Panel de control > Actualizar y restaurar para realizar inmediatamente copias de seguridad de las configuraciones del sistema a un equipo local. Esto le permitirá guardar diversas configuraciones (por ejemplo, grupos de usuarios, configuraciones de servicios de archivos, etc.) y restaurar dichas configuraciones más adelante. Las configuraciones del sistema se guardan como archivos con formato .dss.

Para hacer una copia de seguridad de las configuraciones del sistema:

  1. Haga clic en Hacer copia de seguridad de la configuración.
  2. Se descargará un archivo de configuración (con el nombre [Nombre del producto Synology]_[Fecha de creación].dss) en su ordenador.

Para restaurar la configuración del sistema:

  1. Haga clic en Restaurar configuración.
  2. Haga clic en Examinar y seleccione el archivo de configuración almacenado en su equipo.
  3. Marque las casillas de verificación correspondientes de las configuraciones que desea restaurar y luego haga clic en OK. También puede marcar Sobrescribir configuración en conflicto para sobrescribir la configuración actual para cualquier selección de usuarios, grupos, carpetas compartidas y servicios existentes.
  4. Aparecerá un mensaje de advertencia. Haga clic en Sí para continuar.
  5. Una vez que la función de restauración finalice, aparece una notificación. Haga clic en OK para terminar.

Se copiarán las siguientes configuraciones:

  • Usuarios, grupos y carpetas compartidas
  • Grupo de trabajo, Dominio y LDAP
  • Servicios de uso compartido de archivos (p. ej., Servicio de archivos de Windows, Servicio de archivos de Mac, Servicio NFS, FTP, WebDAV y rsync) y Servicio de copia de seguridad
  • Otros (p. ej., Inicio de usuario, Configuración de contraseña y Programador de tareas)
  • Usuarios, grupos y carpetas compartidas
  • Uso compartido de archivos (p. ej., Servicio de archivos de Windows, Servicio de archivos de Mac, Servicio NFS, FTP, WebDAV)
  • Otros (p. ej., Inicio de usuario, Configuración de contraseña y Programador de tareas)

Nota:

  • Servicio de copia de seguridad incluye parámetros de servicios básicos, pero no información sobre tareas de copia de seguridad.
  • Inicio de usuario incluye parámetros de servicios básicos, a la vez que excluye la carpeta compartida homes. Vaya a Hyper Backup para realizar copias de seguridad y restaurar la carpeta compartida homes y sus datos.
  • Programador de tareas incluye tareas que se pueden crear en Programador de tareas mientras que excluye tareas generadas por otras aplicaciones (por ejemplo, Hyper Backup e Informe de uso de disco, etc.)
  • Si los nombres de los usuarios o grupos restaurados ahora están reservados solo para el uso del sistema, se restaurarán como los otros nombres.
