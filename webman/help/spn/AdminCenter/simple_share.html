Carpeta compartida

Las carpetas compartidas son los directorios básicos en los que puede almacenar archivos y carpetas de su DiskStation. Antes de almacenar archivos en su DiskStation, debe crear al menos una carpeta compartida en DSM. Los datos almacenados en carpetas compartidas se pueden mantener privados o se pueden compartir con usuarios específicos o grupos de usuarios de acuerdo con permisos de acceso personalizables.

A continuación se presenta una lista de carpetas compartidas predeterminadas, creadas automáticamente cuando ciertas aplicaciones, servicios o paquetes se instalan o habilitan.

Nombre Descripción
public La carpeta public se crea automáticamente después de configurar DiskStation por primera vez.*
web La carpeta web es la ubicación predeterminada de almacenamiento para los contenidos de un sitio web alojado en DiskStation. Se crea automáticamente cuando se habilita Web Station.
photo La carpeta photo se utiliza para almacenar fotos y vídeos que desee compartir en Photo Station. Se creará automáticamente cuando se habilita Photo Station o el servidor multimedia DLNA/UPnP.
music La carpeta music se utiliza para almacenar la música que desee escuchar con Audio Station. Se crea automáticamente cuando se habilita Audio Station o el servidor multimedia DLNA/UPnP.
video La carpeta video se utiliza para almacenar archivos de vídeo que desee examinar a través de dispositivos DLNA/UPnP. Se crea automáticamente cuando se habilita Video Station o el servidor multimedia DLNA/UPnP.
surveillance La carpeta surveillance se utiliza para almacenar las grabaciones de Surveillance Station. Se crea automáticamente cuando se habilita Surveillance Station. Es de sólo lectura y solamente puede acceder a ella el administrador del sistema.
home La carpeta home proporciona espacios privados para cada usuario, para que pueda almacenar datos donde sólo él pueda acceder. Se crea automáticamente cuando la característica Inicio de usuario se habilita.
homes La carpeta homes contiene las carpetas Inicio de todos los usuarios. Se crea automáticamente cuando la característica Inicio de usuario se habilita. Solamente el administrador del sistema puede acceder a las carpetas home de todos los usuarios y administrarlas.
NetBackup La carpeta NetBackup se crea automáticamente cuando se habilita Copia de seguridad de red.
usbshare La carpeta usbshare[número] se crea automáticamente cuando conecta un dispositivo con unidad USB externa al puerto USB de DiskStation.*
satashare La carpeta satashare[número] se crea automáticamente cuando conecta un dispositivo con unidad eSATA externa al puerto eSATA de DiskStation.*

Nota:

  • Solo podrá editar permisos de carpetas compartidas de usuarios en el modo sencillo. Si desea crear grupos de usuarios o administrar los privilegios de acceso de grupos a carpetas compartidas, vaya a Panel de control (modo avanzado) > Grupo.

Para crear una carpeta compartida:

  1. Vaya a Panel de control (modo sencillo) > Carpeta compartida y haga clic en el símbolo + en el panel izquierdo.
  2. Especifique un nombre y seleccione un volumen en el que crear la carpeta compartida.
  3. Marque Habilitar papelera de reciclaje si fuera necesario. Si se habilita esta opción, cuando se eliminan archivos de la carpeta compartida, se moverán a una carpeta llamada #recycle. El acceso a la papelera de reciclaje estará limitado a usuarios que pertenecen al grupo de administrators.
  4. Marque o desmarque las casillas adecuadas para cada usuario, a fin de personalizar sus permisos de acceso de la carpeta compartida:
    • Lectura/Escritura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida y hacer cambios.
    • Sólo lectura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida.
    • Sin acceso: El usuario no puede acceder a los archivos ni subcarpetas de la carpeta compartida.
  5. Haga clic en Aplicar para guardar la configuración.

Nota:

  • Los nombres de las carpetas compartidas no distinguen entre mayúsculas y minúsculas y pueden incluir de 1 a 32 caracteres. Los nombres no puede incluir los siguientes caracteres:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Los siguientes nombres de están reservados para el uso del sistema:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • El primer carácter de una carpeta compartida no puede ser un signo menos o un espacio y el último no puede ser un espacio.
  • Cuando encuentre conflictos de permisos, la prioridad de los permisos es la siguiente: Sin acceso > Lectura/Escritura > Sólo lectura.
  • Al crear una carpeta compartida nueva, si los permisos del usuario perteneciente al grupo de administrators está configurado como Sin acceso, estos usuarios solo podrán ver carpetas compartidas en Panel de control > Carpeta compartida.
  • Puede ajustar la configuración de privilegios de la carpeta Papelera de reciclaje de File Station haciendo clic con el botón secundario en la carpeta #recycle. Elija Propiedades y haga clic en la pestaña Permiso.

Para editar una carpeta compartida:

  1. Vaya al Panel de control (modo sencillo) > Carpeta compartida.
  2. Seleccione la carpeta compartida deseada en el panel izquierdo.
  3. Edite la información general de la carpeta compartida, como por ejemplo el nombre, la ubicación y los permisos de usuario.
  4. Si ya se ha marcado Habilitar la papelera de reciclaje, puede hacer clic en Vaciar la papelera de reciclaje para quitar permanentemente todos los archivos eliminados que se encuentran en la papelera de reciclaje.
  5. Haga clic en Aplicar para guardar la configuración.

Para eliminar una carpeta compartida:

  1. Vaya al Panel de control (modo sencillo) > Carpeta compartida.
  2. Seleccione la carpeta compartida deseada en el panel izquierdo.
  3. Haga clic en el icono de Papelera del panel de la izquierda para eliminar la carpeta compartida seleccionada.
