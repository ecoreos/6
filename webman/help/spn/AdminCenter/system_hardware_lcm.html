Pantalla LCM*

Puede ajustar una contraseña para la pantalla LCM con el fin de evitar que cualquier usuario no autorizado cambie la configuración del programa utilizando la pantalla LCM.

Para habilitar la protección mediante contraseña para la pantalla LCM:

  1. Marque Habilitar protección mediante contraseña para pantalla LCM.
  2. Introduzca la Contraseña y vuelva a escribirla en Confirmar contraseña.
  3. Haga clic en Aplicar.

____
* Solo disponible en determinados modelos.
