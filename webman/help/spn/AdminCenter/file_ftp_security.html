Parámetros de seguridad

Cambiar la raíz de los usuario seleccionados a inicio de usuario

Los usuarios seleccionados aquí pueden acceder a sus carpetas de inicio solamente si inician sesión en Synology DiskStation mediante FTP.

Habilitar FTP anónimo

Habilite esta funció para permitir que cualquiera inicie sesión en el servicio FTP de DiskStation (con la cuenta anonymous o ftp).

Para configurar los privilegios del usuario anonymous:

Vaya al Menú principal > Panel de control > Carpeta compartida y haga clic en Configuración de privilegios.

Cambiar raíz anónima

Habilite esta función para cambiar el directorio de inicio de sesión del usuario anonymous de FTP. Seleccione la carpeta compartida desde el menú desplegable.
