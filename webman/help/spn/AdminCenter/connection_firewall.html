Permitir / Denegar

El cortafuegos incorporado puede impedir inicios de sesión no autorizados y controlar a qué servicios se puede acceder. Además, puede elegir entre permitir o denegar el acceso a determinados puertos de red desde direcciones IP específicas.

Para crear una regla de cortafuegos:

  1. Vaya a Panel de control > Cortafuegos y seleccione la interfaz de red de la columna izquierda según el tipo de conexión de red.
  2. Haga clic en Crear.
  3. En la sección Puertos, seleccione una de las siguientes opciones para la regla:
    • Elija Todo para aplicar las reglas a todos los puertos.
    • Elija Seleccione de una lista de aplicaciones incorporadas.
    • Elija Personalizado y haga clic en Personalizado y escriba hasta 15 puertos separados por comas o especifique un intervalo de puertos.
    • También puede especificar los protocolos.
  4. En la sección IP origen, seleccione una de las siguientes opciones para la regla:
    • Elija Todo para permitir o denegar todas las direcciones IP.
    • Escriba una Dirección IP de host sencillo.
    • Escriba una Subred.
  5. En la sección Acción, seleccione una de las siguientes opciones para la regla:
    • Elija Permitir para permitir el acceso a los puertos y a la configuración IP de origen anterior.
    • Elija Denegar para denegar el acceso a los puertos y a la configuración IP de origen anterior.

Ahora puede ver la regla de cortafuegos creada en la lista de reglas.

Para eliminar las reglas del cortafuegos, realice una de las siguientes acciones:

  • Desactive la casilla de verificación de la regla en la columna Habilitado de la lista
  • Eliminar la regla del cortafuegos

También puede elegir Permitir acceso o Denegar acceso en la parte inferior de la lista de reglas para permitir o denegar el acceso que no coincide con ninguna regla.

    Nota:
  1. Puede crear hasta 100 reglas para una interfaz de red y reordenar las reglas de la lista con funciones de arrastrar y soltar.
  2. Las reglas se priorizan en función de sus posiciones en la lista.
  3. Si el sistema tiene múltiples puertos de red conectados a la misma subred, es posible que las reglas de cortafuegos no funcionen correctamente.*
  4. Cuando combine múltiples puertos LAN con link aggregation, el cortafuegos aplicará las reglas de la primera interfaz de red y reservará las reglas de la segunda interfaz de red.

____
* Sólo se aplica a determinados modelos.
