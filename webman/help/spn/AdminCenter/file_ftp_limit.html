Restricción de conexión

Sólo permitir conexión SSL/TLS

Si esta función está habilitada, el servidor sólo aceptará solicitudes de conexión SSL o TLS de clientes de FTP.

Limitar conexiones por IP

Con esta función habilitada, el servidor limitará las conexiones máximas para cada dirección IP. Habilite esta función y restricción de ancho de banda de FTP para administrar el ancho de banda de manera efectiva.

Habilitar restricción de ancho de banda de FTP

Habilite esta función para restringir el ancho de banda de una sola conexión de FTP.
