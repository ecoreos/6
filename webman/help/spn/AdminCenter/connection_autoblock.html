Bloqueo automático

Bloqueo automático permite impedir conexiones no autorizadas. Una vez habilitado el servicio, se bloqueará una dirección IP si tiene demasiados intentos de conexión fallidos. Los intentos de conexión fallidos mediante SSH, Telnet, rsync, Copia de seguridad de red, Sincronización de carpetas compartidas, FTP, WebDAV, aplicaciones móviles de Synology, File Station o DSM se irán acumulando. También puede tener una Lista de permiso que incluye las direcciones IP que nunca se bloquearán.

Para configurar el bloqueo automático:

  1. Vaya a la ficha General y marque Habilitar bloqueo automático.
  2. Introduzca un número para Intentos de conexión y En (minutos) para bloquear una dirección IP al cabo de un número definido de intentos de conexión fallidos en un número de minutos especificado.
  3. Marque Habilitar caducidad del bloqueo y introduzca un número para quitar una dirección IP bloqueada al cabo del número de días especificado. Las direcciones IP bloqueadas permanecerán indefinidamente en la lista de bloqueo a menos que se quite de la lista si no se selecciona esta opción o si Desbloquear al cabo de (días) se define en 0.
  4. Haga clic en Aplicar.

Para añadir direcciones IP bloqueadas:

Haga clic en la ficha Lista de bloqueo y seleccione cualquiera de las siguientes opciones del menú desplegable Crear:

  • Añadir dirección IP

    • Introduzca una dirección IP y establezca el tiempo de caducidad.

  • Importar lista de direcciones IP

    • Importe un archivo de texto con las direcciones IP para añadir varias direcciones IP y establecer la fecha de caducidad a la vez. Marque la casilla de verificación adecuada si desea sobrescribir direcciones IP existentes. Si no se selecciona esta opción, se omitirán las direcciones IP duplicadas.

    Para importar un archivo, el archivo debe cumplir los siguientes criterios:

    • El archivo debe ser un archivo de solo texto.
    • Cada línea del archivo debe contener una única dirección IP.
    • Las líneas de comentario deben empezar por # (signo de libra).

      • Si el archivo seleccionado tiene el formato correcto, verá todas las direcciones IP en el área de vista previa. Haga clic en Aceptar para importar.

Para eliminar direcciones IP bloqueadas:

  1. Haga clic en la ficha Lista de bloqueo para ver la lista de direcciones IP bloqueadas.
  2. Seleccione las direcciones IP que desee eliminar de la lista y haga clic en Eliminar.
