Win/Mac

Los usuarios de Windows, Mac y Linux pueden acceder a su DiskStation desde la red cuando habilite las opciones en Panel de control > Servicios de archivos > Win/Mac.

Los siguientes protocolos de uso compartido de archivos son compatibles:

  • Para Windows: SMB/CIFS (Mis sitios de red), FTP, WebDAV
  • Para Mac: SMB, FTP, AFP, WebDAV
  • Para Linux: SMB, FTP, WebDAV
