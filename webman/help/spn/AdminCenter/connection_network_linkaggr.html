Link Aggregation*

DiskStation admite múltiples LAN, permitiendo que combine esas interfaces LAN con ayuda de la tecnología Link Aggregation. Link Aggregation aumenta el ancho de banda del DiskStation mediante la adición de varias interfaces de red y proporciona la función de conmutación por error para el tráfico de manera que la conexión a la red se mantenga en casos de fallo de conexión.

Después de que las interfaces LAN se combinen, verá una nueva interfaz llamada Bond en Panel de control > Red > Interfaz de red, que puede configurarse de la misma forma que lo haría con otras interfaces (como LAN o PPPoE).

Nota:

  • Al habilitar Link Aggregation, el tráfico de red se ajustará y equilibrará automáticamente para cada dispositivo conectado.
  • Si las interfaces de red están conectadas a varios conmutadores, asegúrese de que los puertos de los conmutadores están en el mismo LAN o VLAN.

Para combinar múltiples interfaces LAN con Link Aggregation:

  1. Vaya al Panel de control > Red > Interfaz de red. Haga clic en Crear > Crear Bond.
  2. Seleccione el modo que desee.

    Descripciones de los modos:

    • Adaptive Load Balancing: Este modo optimiza el tráfico de red recibido y enviado por DiskStation sin importar que el conmutador sea compatible con Link Aggregation o no. Para evitar errores inesperados no habilite Link Aggregation en el conmutador incluso si es compatible.
    • IEEE 802.3ad Dinámico Link Aggregation: Este modo optimiza el tráfico de red recibido y enviado por el DiskStation y requiere que se habilite IEEE 802.3ad (Dynamic) Link Aggregation (LACP, 802.1AX) en los conmutadores. Si se utilizan varios conmutadores, estos deben ser apilables y se deben configurar correctamente.
    • Balance XOR: Este modo equilibra el tráfico de red recibido y enviado por el DiskStation y requiere que se habilite Static Link Aggregation en los conmutadores. Si se utilizan varios conmutadores, estos deben ser apilables y se deben configurar correctamente. Aunque el conmutador no sea compatible con Link Aggregation también es posible equilibrar el DiskStation. Con este modo se pueden detectar los enlaces perdidos, pero no la causa (por ejemplo, cuando los cables de Ethernet se desenchufan o el conmutador no está configurado correctamente).
    • Activo/En reposo: Este modo proporciona la característica de network fault tolerance al DiskStation. O sea, cuando la interfaz de la red activa se desconecta, uno de los otros puertos LAN tomará el control para mantener la conexión de red. Puede elegir este modo sin configurar el interruptor o con un interruptor compatible con IEEE 802.3ad Link Aggregation.
  3. Seleccione las interfaces para crear Link Aggregation.
  4. Configure los ajustes de IP. También puede habilitar VLAN en caso necesario. Haga clic en Aplicar para completar la configuración.
  5. Cuando acabe, podrá ver la nueva interfaz denominada Bond 1 en la lista Interfaz de red.

Para editar la configuración de Link Aggregation:

  1. Para cambiar el modo de Link Aggregation o ver la información de los dispositivos físicos, vaya a Panel de control > Red > Interfaz de red.
  2. Seleccione la interfaz Bond 1 y, a continuación, haga clic en el botón Editar.
  3. Vaya a las fichas correspondientes para modificar la configuración deseada.

Para eliminar Link Aggregation:

  1. Vaya al Panel de control > Red > Interfaz de red.
  2. Seleccione la interfaz Bond 1 y, a continuación, haga clic en el botón Eliminar.
  3. Una vez terminado, verá que Bond 1 se ha sustituido por dos interfaces LAN.
