Servicio HTTP

Además del número de puerto predeterminado 80, puede agregar otro puerto para el uso de Photo Station y Web Station.

Para agregar un número de puerto adicional para acceso HTTP:

  1. Marque Agregar puerto HTTP adicional.
  2. Escriba el número de puerto en el campo. Puede agregar un número de puerto adicional de 1 a 65535 para acceso HTTP.

Deben excluirse los siguientes puertos:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, puertos predeterminados de eMule: 4662 (TCP), 4672(UDP), e intervalo de puertos predeterminado para FTP: el intervalo concreto puede variar dependiendo de los distintos modelos.

    Nota:
  1. Los números de puerto para acceso HTTP no pueden coincidir con el número de puerto para el servicio FTP/BT.
  2. Los puertos HTTP y HTTPS no pueden estar en conflicto entre sí.
