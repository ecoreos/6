Grupos o usuarios de Dominio/LDAP

Si su DiskStation se une a un servicio de directorio como un Dominio o cliente LDAP de Windows, puede configurar y modificar el permiso de acceso de grupos o usuarios de Dominio/LDAP en las carpetas compartidas y las aplicaciones DSM de DiskStation y habilitar el servicio de inicio para todos los usuarios de Dominio/LDAP.

Usuario de Dominio/LDAP

En función del tipo de servicio de directorio al que se está uniendo, haga clic en la ficha Usuarios de dominio o Usuario LDAP para configurar la carpeta compartida y los permisos de aplicaciones y la cuota de capacidad del disco de cada usuario.

En función del tipo de servicio de directorio al que se está uniendo, haga clic en la ficha Usuarios de dominio o Usuario LDAP para configurar la carpeta compartida y los permisos de aplicaciones para cada usuario.

Las reglas de ajustes de privilegios incluyen lo siguiente:

  • En el caso de conflictos de privilegios, la prioridad de privilegios será: NA > RW > RO.
  • Vista previa del privilegio: el privilegio del usuario debe mostrarse aquí y se decide en función de la configuración de privilegios del usuario y de los grupos a los que se haya unido el usuario.
  • Privilegios de grupo: asigne privilegios a grupos para acceder a carpetas compartidas.

Inicio de usuario

Al igual que los usuarios locales, todos los usuarios del dominio/LDAP pueden acceder a su propia carpeta home mediante CIFS, AFP, FTP o File Station.

Los usuarios con privilegios administrativos pueden acceder a todas las carpetas personales ubicadas en la carpeta compartida homes. Todas las carpetas home de los usuarios se encuentran en una de las subcarpetas bajo una carpeta denominada "@DH-nombre de dominio" (para usuarios de dominio) o "@LH-FQDN nombre" (para usuarios LDAP). El nombre de la carpeta home del usuario es la cuenta del usuario más un número exclusivo.

    Nota:
  1. Para eliminar la carpeta compartida homes, primero es necesario deshabilitar el servicio de inicio de usuario.
  2. Al habilitar el servicio de inicio para usuarios del dominio/LDAP también se habilitará el servicio de inicio de usuarios locales, si aún no está habilitado.

Grupo de Dominio/LDAP

En función del tipo de servicio de directorio al que se está uniendo, haga clic en la ficha Grupo de dominios o Grupo de LDAP para configurar los privilegios de acceso de grupos de dominio/LDAP en carpetas compartidas. La prioridad de privilegios es: NA > RW > RO.
