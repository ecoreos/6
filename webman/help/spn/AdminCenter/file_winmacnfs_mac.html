Servicio de archivos Mac

En Panel de control > Servicios de archivos > Win/Mac/NFS, puede habilitar el servicio de archivos Mac para que DiskStation actúe como un servidor de archivos Mac (servidor AFP).

Para habilitar el servicio de archivos de Mac:

  1. Marque Habilitar el servicio de archivos de Mac para que DiskStation actúe como un servidor de archivos Mac (servidor AFP).
  2. Configure una carpeta de copia de seguridad de Time Machine si fuera necesario (consulte el siguiente apartado para obtener más información).
  3. Haga clic en Aplicar.

Nota:

  • Para realizar una copia de seguridad desde Time Machine en DiskStation, seleccione una carpeta compartida desde el menú desplegable.
  • Las carpetas compartidas que están siendo utilizadas por otros servicios no estarán disponibles para Time Machine.

Habilitar registro de transferencia

Si activa esta opción, se habilitará el inicio de sesión para operaciones de acceso de archivos, incluyendo crear, eliminar, leer, escribir, mover y renombrar archivos o carpetas a través de AFP, también conocido como Apple Filing Protocol.

Configuración avanzada

Aplicar permisos UNIX predeterminados:

Al habilitar esta opción se aplican los permisos UNIX predeterminados cuando se cargan o crean archivos y carpetas. Los permisos aplicados son los mismos que se aplican con el comando UNIX umask. Cuando se habilita esta opción, los permisos UNIX son 644 para archivos y 755 para carpetas. Cuando se deshabilita esta opción, los permisos UNIX son 666 para archivos y 777 para carpetas. El valor predeterminado de umask es 022.

Nota:

Si se habilita esta opción se pueden provocar problemas de permisos que no mantienen la consistencia entre protocolos diferentes. Para evitar inconsistencias, sugerimos no habilitar esta opción.

Liberar recursos inmediatamente después de la desconexión:

La activación de esta opción libera inmediatamente recursos cuando la conexión del servidor AFP se interrumpe de manera inesperada. Si recibe mensajes de error que indican que el volumen está en uso mientras utiliza Time Machine, active esta opción.

Para habilitar la transmisión de impresora Bonjour:

Marque Habilitar transmisión de impresora Bonjour para permitir que los clientes de Mac examinen y localicen la impresora conectada a DiskStation.

Para buscar contenido de los archivos utilizando Spotlight en Finder:

Tras indexar y montar por completo las carpetas en un ordenador Mac, puede utilizar Spotlight en Finder para buscar en los archivos y su contenido.

  1. Para indexar las carpetas y los archivos por tipo, vaya a Panel de control > Servicio de indexación > Indexado de archivos > Lista de carpetas indexadas > Crear.
  2. Haga clic en OK para llevar a cabo la indexación.
  3. Monte las carpetas indexadas en el ordenador Mac.
  4. En el ordenador Mac, haga clic en la carpeta montada que desee del panel izquierdo de Finder y utilice Spotlight en Finder para buscar en los archivos.

Nota:

  • Para buscar en las carpetas montadas, utilice Spotlight en Finder. Spotlight solo admite la búsqueda de archivos locales desde el escritorio de Mac.
  • En Mac OS X 10.9 y posterior, Spotlight en Finder admite la búsqueda en las carpetas montadas filtrando por nombre y categoría de etiqueta.
