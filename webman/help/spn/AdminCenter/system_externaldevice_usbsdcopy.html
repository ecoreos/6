USBCopy

Puede seleccionar una carpeta de destino para la función USBCopy yendo a Panel de control > Dispositivos externos > USBCopy/SDCopy.

Para habilitar USBCopy:

  1. Marque Habilitar USBCopy.
  2. Elija una carpeta compartida de destino desde el menú desplegable.

Al conectar un dispositivo USB (como una cámara digital) al puerto USB del panel frontal de un producto de Synology, puede pulsar el botón Copy en el panel frontal para copiar los datos en el dispositivo USB a la carpeta compartida de destino asignada aquí. Si se elimina la carpeta de destino de USBCopy, el sistema la sustituirá por la carpeta compartida pública automáticamente.

Limitaciones

  1. Únicamente se admiten los dispositivos USB en formato FAT o NTFS ext3 y ext4.
  2. Los dispositivos USB deben conectarse al puerto USB del panel frontal de DiskStation.
  3. Solo el usuario admin o los usuarios en el grupo administrators tienen los accesos de privilegio adecuados para realizar las tareas de copia de seguridad a dispositivos USB.
  4. El espacio disponible de la carpeta compartida de destino debe ser superior al espacio usado del dispositivo USB.

SDCopy*

Puede seleccionar una carpeta de destino para la función SDCopy yendo a Panel de control > Dispositivos externos > USBCopy/SDCopy.

Para habilitar SDCopy:

  1. Marque Habilitar SDCopy.
  2. Elija una carpeta compartida de destino desde el menú desplegable.

Al conectar un dispositivo SD a la ranura SD del panel frontal de un producto de Synology, puede pulsar el botón Copy en el panel frontal para copiar los datos en el dispositivo SD a la carpeta compartida de destino asignada aquí. Si se elimina la carpeta de destino de SDCopy, el sistema la sustituirá por la carpeta compartida pública automáticamente.

Limitaciones

  1. Únicamente se admiten los dispositivos SD en formato FAT o NTFS ext3 y ext4.
  2. Los dispositivos SD deben conectarse a la ranura SD en el panel frontal de un producto de Synology.
  3. Solo el usuario admin o los usuarios en el grupo administrators tienen los accesos de privilegio adecuados para realizar las tareas de copia de seguridad a dispositivos SD.
  4. El espacio disponible de la carpeta compartida de destino debe ser superior al espacio usado del dispositivo SD.
