Configuración de enrutador

Si su dispositivo DiskStation se encuentra en una red de área local, otros dispositivos de Internet no podrán conectarse a él. Puede definir las reglas de reenvío de puerto en Panel de control > Acceso externo > Configuración del enrutador para que el dispositivo DiskStation sea accesible a través de Internet.

Configuración del enrutador

Antes de agregar las reglas de reenvío de puerto, debe configurar una conexión entre su dispositivo DiskStation y el enrutador.

Nota:

  1. Para configurar reglas de reenvío de puerto y asignar una dirección IP estática, deberá tener privilegios administrativos para el enrutador.
  2. Antes de configurar la conexión del dispositivo DiskStation en un enrutador UPnP, asegúrese de que DSM ha recuperado correctamente la dirección IP externa de su DiskStation (en Panel de control > Acceso externo > DDNS). En caso contrario, compruebe la configuración de red y del cortafuegos de su dispositivo DiskStation y el enrutador UPnP.

Para configurar la conexión del dispositivo DiskStation en su enrutador:

  1. Haga clic en Configurar enrutador para iniciar el asistente de configuración. DSM comenzará comprobando los siguientes componentes/configuraciones de red antes de continuar al paso siguiente.
    • Interfaz de red
    • Conexión de pasarela
    • Conexión a Internet
    • Reenvío de puerto
    • DNS
  2. En función del tipo de enrutador, realice una de las siguientes acciones:
    • Si su enrutador no admite UPnP, consulte el paso 3.
    • Si ha comprobado que su enrutador UPnP es compatible con el dispositivo DiskStation, haga clic en Aplicar cuando vea un resumen de la información del enrutador. La conexión de su dispositivo DiskStation con el enrutador se configurará correctamente.
    • Si no ha comprobado que su enrutador UPnP es compatible con el dispositivo DiskStation, realice una de las siguientes acciones:
      • Si desea configurar automáticamente la conexión con el enrutador, seleccione Automática y haga clic en Siguiente para que el sistema lleve a cabo una prueba de compatibilidad:
        • Si ha comprobado que el enrutador es compatible con el servicio de configuración de enrutador del dispositivo DiskStation, active la casilla de verificación para enviar el informe del test y ayudar a Synology a actualizar la base de datos de enrutadores compatibles, y luego haga clic en Aplicar. La conexión de su dispositivo DiskStation con el enrutador se configurará correctamente.
        • Si ha comprobado que el enrutador no es compatible con el servicio de configuración de enrutador del dispositivo DiskStation, active la casilla de verificación para enviar el informe del test y ayudar a Synology a actualizar la base de datos de enrutadores compatibles, y luego haga clic en Cerrar.

          • Nota:

          Si ha comprobado que el enrutador no es compatible aquí, puede configurar las reglas de reenvío de puerto en la interfaz de usuario del enrutador. Consulte aquí para obtener más información.

      • Si desea configurar manualmente la conexión con el enrutador, seleccione Manual, haga clic en Siguiente y consulte el paso 3.
  3. Si se le pide que configure manualmente la conexión del dispositivo DiskStation al enrutador, haga lo siguiente:
    1. Elija su enrutador desde el menú desplegable Enrutador.

      2. Nota:

      Si no encuentra el enrutador aquí, haga clic en Actualizar lista para obtener de Synology la lista más reciente.

    2. Configure su protocolo y el puerto de su enrutador. El protocolo y el puerto varían en función del modelo de su enrutador. Si no puede conectarse usando el Protocolo (HTTP) y Puerto (80) predeterminados, consulte la documentación incluida en el enrutador para ver la configuración correcta.
    3. Si su enrutador requiere una cuenta personalizada, marque Usar cuenta personalizada y rellene los campos Cuenta y Contraseña.
    4. Haga clic en Aplicar para terminar.

Administrar Reglas de reenvío de puerto

Las reglas de reenvío de puerto pueden especificar puertos de destino de su DiskStation que recibirán paquetes de puertos específicos del enrutador.

Si su DiskStation se conecta a Internet a través de un enrutador, haga clic en Crear para abrir el asistente de configuración de reenvío de puerto.

Existen dos modos para agregar reglas de reenvío de puerto:

  • Si desea crear reglas de reenvío de puerto para servicios de sistema (por ejemplo, FTP, Web Station, etc), seleccione Aplicación incorporada.
  • Si desea crear reglas de reenvío de puerto para otros servicios, seleccione Puerto personalizado.

Puede usar la siguiente sintaxis para especificar números de puerto e intervalo:

  • Utilice “-” para describir el intervalo de puertos, por ejemplo: 6881-6890.
  • Utilice “,” para separar puertos, por ejemplo: “21,22”, “21,55536-55663”.

Nota:

  • Cada puerto del enrutador de pasarela se puede usar únicamente para una regla de reenvío. El uso de un puerto de enrutador para dos o más reglas provocará conflictos y aparecerá en pantalla un mensaje de advertencia. Debe configurar otro puerto en la regla para evitar conflictos.
  • Algunos enrutadores no permiten el uso de puertos personalizados para el reenvío de servicios del sistema.

Para guardar las reglas:

Haga clic en Guardar después de crear las reglas. En caso contrario, no se aplicarán las reglas.

Después de guardar las reglas, es posible que tenga que esperar unos minutos hasta que comiencen a funcionar las reglas, según los modelos de sus enrutadores.

Para probar si funcionan sus reglas o no:

Elija las reglas que desee probar y haga clic en Probar conexión. El resultado se mostrará en la columna Resultado de la prueba de conexión.

Para las configuraciones del enrutador inalámbrico*

Cuando el dispositivo DiskStation actúe como enrutador inalámbrico, las reglas de reenvío de puerto serán aplicadas directamente por la unidad DiskStation.

Para agregar reglas de reenvío de puerto:

  1. Haga clic en Crear para abrir la ventana de configuración.

  2. Indique el nombre del Servicio, el Protocolo (TCP o UDP), la Dirección IP privada, el Puerto privado y el Puerto público. Puede usar la siguiente sintaxis para especificar números de puerto e intervalo:
    • Utilice “-” para describir el intervalo de puertos, por ejemplo: 6881-6890.
    • Utilice “,” para separar puertos, por ejemplo: “21,22”, “21,55536-55663”.
  3. Haga clic en Crear en la parte inferior para completar la configuración.

Nota:

La mayoría de enrutadores permiten que los puertos de los equipos locales sean distintos a los del enrutador para configurar reglas de reenvío de puerto. Si su enrutador es compatible con esta función:

  • Si el puerto de un enrutador ya está en uso por una regla existente, al guardar otra regla, se asignará a un nuevo puerto del enrutador.
  • Al añadir o editar reglas, puede especificar directamente el número de puerto del enrutador.

DMZ*

Puede habilitar DMZ en su DiskStation cuando la Configuración de mapa de red sea Enrutador inalámbrico. Un host del DMZ puede conectarse directamente a la red externa, sin la limitación ni la protección del cortafuegos. Se suele usar para configurar servidores.

Para habilitar DMZ:

  1. Haga clic en Configuración de DMZ para abrir la ventana de configuración.
  2. Marque la casilla de verificación Habilitar DMZ.
  3. Introduzca la dirección IP de su DMZ Host.
  4. Haga clic en Aplicar.

Nota:

Desde la red externa, necesitará una dirección IP externa recuperada por DiskStation para conectarse a un host en el DMZ.

* Solo disponible en determinados modelos o según su entorno.
