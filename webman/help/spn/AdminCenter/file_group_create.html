Crear un grupo

  1. Vaya al Panel de control > Grupo.
  2. Haga clic en Crear. Esta acción abrirá el Asistente de creación de usuario.
  3. Escriba el nombre y la descripción del grupo. (Consulte el siguiente apartado para obtener más información). Haga clic en Siguiente.
  4. Siga los pasos restantes del asistente para finalizar la creación de grupos.
  5. Haga clic en Aplicar para terminar.

Convenciones de nombres:

  • El nombre de grupo distingue entre mayúsculas y minúsculas y puede incluir de 1 a 32 caracteres Unicode, excluyendo los siguientes símbolos:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • El primer carácter no puede ser un signo menos o un espacio y el último no puede ser un espacio.
  • La descripción del grupo distingue entre mayúsculas y minúsculas y está limitada a 64 caracteres Unicode que se pueden mostrar.
