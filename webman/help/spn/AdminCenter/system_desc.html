Sistema

Esta sección le permite administrar la configuración del sistema de DiskStation. Puede personalizar la pantalla de inicio de sesión, cambiar la banda horaria y la configuración de idioma, programar un encendido/apagado automático, administrar dispositivos externos, actualizar la versión de DSM o realizar el restablecimiento a la configuración de fábrica.
