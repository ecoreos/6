Salida de CC*

En Panel de control > Hardware y alimentación > Salida de CC, puede configurar el sistema para que se encienda o se apague automáticamente a una hora determinada.

Para agregar un nuevo programa de salida activada y salida desactivada:

  1. Haga clic en el botón Crear para abrir el cuadro de diálogo Agregar programa.
  2. Elija Salida activada o Salida desactivada.
  3. Ajuste la Fecha y Hora para el programa de salida activada/salida desactivada.
  4. Haga clic en OK.
  5. Haga clic en Guardar.

El nuevo programa se agregará a la lista de programas.

Para ver el resumen del programa:

Haga clic en Resumen para examinar el programa semanal de salida activa/salida desactivada.

Nota:

  • Puede especificar hasta 100 programas para un DiskStation.
  • En los siguientes casos, es posible que el sistema no consiga desactivar la salida de CC automáticamente:
    • El Administrador de almacenamiento está realizando tareas.
    • El sistema está cifrando las carpetas compartidas.
    • El sistema está realizando Secure Erase en los discos duros.
    • DSM se está actualizando.
    • El Centro de paquetes está realizando tareas.
    • El sistema está actualizando bases de datos.
    • El sistema está realizando tareas de copia de seguridad de datos.

____
* Solo disponible en determinados modelos.
