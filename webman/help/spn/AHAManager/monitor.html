Monitor

Esta página contiene tres pestañas que muestran el monitor del sistema de estado, servicio e interfaz de red. Esta página le permite supervisar algunos destinos. Si un destino supervisado se avería en el servidor activo, el sistema conmutará al servidor pasivo en modo de espera (siempre que el servidor pasivo funcione con normalidad). En esta página aparecen las siguientes pestañas:

Pestaña Estado del sistema

El estado del sistema se supervisa de manera predeterminada y no se puede deshabilitar. La conmutación por error se activará cuando se produzcan fallos del sistema en el servidor activo. En esta pestaña aparecen las siguientes secciones:

  • Estado del servidor activo: conmutación por error al producirse la avería del servidor activo para proporcionar los servicios. Por ejemplo, ambas unidades de potencia del servidor activo se averían.
  • Conectividad del almacenamiento: conmutación por error en una situación en la que una unidad está ausente en el servidor activo pero presente en el servidor pasivo.

Pestaña Servicio

Muestra los servicios que se pueden supervisar y su estado. La conmutación por error se activará cuando los servicios sean anormales. En esta pestaña aparecen las siguientes secciones:

  • Servicio: muestra los servicios que se pueden supervisar.
  • Estado: muestra si los servicios están en funcionamiento o no.
  • Monitor: marque la casilla de verificación junto a un servicio para comenzar a supervisarlo. El servicio conmutará si el servicio seleccionado deja de funcionar.

Para supervisar un servicio:

  1. Encuentre el servicio que desea supervisar. Asegúrese de que el estado de servicio es normal.
  2. Marque la casilla de verificación junto a él.
  3. Haga clic en Aplicar para guardar la configuración.

Nota:

Los servicios que se pueden supervisar son los siguientes: Servicios de archivos Windows, iSCSI Target, FTP, Servicio de archivos Mac, Servicio NFS.

Pestaña Interfaz de red

Muestra todas las interfaces de red lógicas que se pueden supervisar. La conmutación por error se activará si la conexión de red es anormal. (Consulte las notas siguientes). En esta pestaña aparecen las siguientes secciones:

  • Interfaz de red: muestra las interfaces que se pueden supervisar.
  • Monitor: marque la casilla de verificación junto a una interfaz para comenzar a supervisarla.

Nota:

Si se marca la casilla de selección de Monitor, el sistema conmutará al servidor pasivo cuando se interrumpa o averíe la conexión de un servidor activo con la red, y la conexión del servidor pasivo a la red todavía funcione.

  • La conmutación se disparará solo si el servidor pasivo tiene una mejor conexión. Por ejemplo, si el servidor activo tiene tres conexiones y una falla, el sistema activará una conmutación solo si el servidor pasivo tiene tres o más conexiones buenas.
  • Las conexiones conectadas con Link Aggregation cuentan como una conexión. Por ejemplo, si dos conexiones se han combinado y una de ellas falla, pero la otra todavía funciona, la conexión todavía se considera válida.

Para supervisar una interfaz:

  1. Encuentre la interfaz que desea supervisar. Asegúrese de que los cables se han conectado correctamente a las interfaces de los servidores activo y pasivo.
  2. Marque la casilla de verificación junto a él.
  3. Haga clic en Aplicar para guardar la configuración.
