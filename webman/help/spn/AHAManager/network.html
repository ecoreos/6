Diagrama de interfaces

Muestra el elemento seleccionado en la tabla y se utilizan colores diferentes para presentar las interfaces de red de 1 Gbps y 10 Gbps.

Tabla de estados de las interfaces

Muestra el estado de la interfaz en ambos servidores y si está supervisada. La tabla muestra la siguiente información:

  • Interfaz de red: muestra las interfaces físicas de red de los servidores.
  • Servidor activo: muestra el estado de la interfaz y la velocidad de conexión en el servidor activo.
  • Servidor pasivo/Servidor desconocido: muestra el estado de la interfaz y la velocidad de conexión en el servidor pasivo. Esta columna muestra el nombre correspondiente al estado del servidor.
  • Bond: muestra la configuración de link aggregation.
  • Estado: muestra el estado de red si la interfaz está supervisada. Puede mostrar los siguientes estados:
    • Advertencia: error de conexión en el servidor pasivo o la interfaz tiene diferentes velocidades de conexión en los dos servidores;
    • Error: error de conexión en el servido activo;
    • Normal: no hay ninguna avería.
  • Monitor: muestra si la interfaz está supervisada. Vaya a Monitor > Interfaz de red para marcar o desmarcar la casilla de verificación Monitor.
