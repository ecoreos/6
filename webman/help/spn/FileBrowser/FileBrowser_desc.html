File Station

File Station es una herramienta de administración de archivos fácil de usar que permite a los usuarios de DSM por Internet acceder a las carpetas de DiskStation con sus navegadores web. Abriendo dos o más File Stations, puede administrar todos sus datos de DiskStation arrastrándolos y soltándolos entre diferentes File Stations.
