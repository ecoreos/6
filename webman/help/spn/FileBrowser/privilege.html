Ver o cambiar permisos de archivos/carpetas

Con File Station, puede configurar privilegios de acceso a archivos y subcarpetas. Esta función resulta especialmente útil si desea permitir que diferentes grupos de usuarios accedan a diferentes subcarpetas de una carpeta compartida.

Para permitir que usuarios o grupos accedan a archivos y subcarpetas:

  1. Seleccione los archivos o carpetas para los que desee configurar privilegios de acceso.
  2. Haga clic con el botón secundario en la carpeta y elija el menú Acción y luego elija Propiedades.
  3. En la sección de Propietario, elija el propietario de los archivos y carpetas seleccionados desde el menú desplegable Propietario.
  4. Haga clic en la pestaña de Permiso y seleccione un usuario o un grupo. Haga clic en Editar para abrir el Inspector de permisos.
  5. Realice cualquiera de las siguientes acciones en la ventana de Editor de permisos para administrar los permisos de ACL para el archivo o carpeta:
    1. Usuario o grupo: Especifique el usuario o grupo que desea personalizar.
    2. Heredar de: Solo para ver. Vea la información aquí para ver si el permiso se ha heredado (de una carpeta principal) o es explícito (mostrado como Ninguno).
    3. Tipo: Elija Permitir o Denegar para otorgar o denegar el permiso al usuario o grupo.
    4. Aplicar a: Si está creando una entrada de permiso para una carpeta, marque las casillas de verificación para aplicar la entrada de esta carpeta, las carpetas (o Carpetas secundarias) o archivos (o Archivos secundarios) en esta carpeta, o todos los archivos y carpetas incluidas en esta carpeta (o Todos los descendientes).
    5. Administración: Marque Leer permisos, Cambiar permisos o Toma de posesión para especificar la configuración de permisos de acceso de usuario o grupo de la entrada.
    6. Leer o Escribir: Marque las casillas de verificación en estos apartados para modificar la configuración del permiso del grupo o usuario en el archivo o carpeta.
  6. Haga clic en OK.

Nota: Aquí no se le permite editar los privilegios de acceso a archivos y subcarpetas de la carpeta compartida photo.
