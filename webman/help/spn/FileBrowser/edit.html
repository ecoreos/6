Editar fotos

En File Station, podrá editar fotos con el editor Aviary, un editor de fotos externo basado en la web.

Para editar una foto con el editor Aviary:

  1. Busque la carpeta donde se encuentra la foto.
  2. Haga clic con el botón secundario en la foto y seleccione Editar en Aviary.
  3. El editor Aviary se abrirá en una nueva ventana del explorador. Podrá visualizar su foto y editarla con las herramientas proporcionadas.
  4. Cuando haya terminado de editar la foto, pulse el botón Guardar que aparece en la barra de herramientas superior para guardar todos los cambios que ha realizado.

Nota:

  • El editor Aviary permite editar los siguientes formatos: .jpg, .jpeg, .jpe, .png
  • El tamaño máximo permitido por el editor Aviary es de 5 MB.
  • El editor Aviary requiere haber ajustado correctamente la configuración de red de DiskStation. Compruebe la siguiente configuración para solucionar los problemas:
    • Asegúrese de que puede acceder a DiskStation a través de (Panel de control > QuickConnect) o que puede recuperar una dirección IP externa (en Panel de control > Acceso externo > DDNS o en Panel de control > Red > Interfaz de red > PPPoE).
    • Si su DiskStation está ubicado detrás de un enrutador, asegúrese de que ha especificado la regla de reenvío de puerto adecuada (Panel de control > Acceso externo > Configuración de enrutador).
    • Si continúa teniendo problemas para abrir el editor Aviary, es posible que deba especificar el puerto del enrutador para DiskStation (Panel de control > Acceso externo > Avanzado).
