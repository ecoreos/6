Requisitos de File Station

Para utilizar File Station con todas sus funciones, necesitará el explorador web más reciente, Java y Flash Player.

  • Explorador: Internet Explorer 8 o posterior, Firefox 3.6 o posterior, Safari 5.0 o posterior, Chrome; con AJAX y JavaScript habilitados
  • Java (para examinar las carpetas del equipo local): Java Runtime Environment (JRE) 5 o posterior
  • Flash: Flash Player 9.0.28 o posterior
