Administrar la configuración de File Station

Para administrar la configuración de File Station, abra File Station y haga clic en el botón Configuración.

General

Aquí puede habilitar los registros de File Station para registrar las actividades de todos los usuarios. Las actividades del usuario incluyen Crear carpeta, Cargar, Descargar, Eliminar, Renombrar, Mover, Copiar y Propiedad fijada.

Para habilitar el registro de File Station:

  1. Haga clic en Habilitar registro de File Station para registrar las actividades de todos los usuarios.
  2. Haga clic en OK.

Para ver el registro de File Station:

  1. Haga clic en Ver registros, lo que le llevará al Centro de registros.
  2. Haga clic en la pestaña Transferencia de archivos.

Para sobrescribir automáticamente archivos con el método de arrastrar y soltar o con las teclas de acceso directo:

Marque Sobrescribir siempre archivos cuando los arrastre y los coloque en el navegador web o los pegue con las teclas de acceso directo si desea sobrescribir automáticamente los archivos con el mismo nombre cuando arrastre y coloque archivos desde su escritorio del ordenador al navegador web, o bien pegue archivos con las teclas de acceso directo en File Station.

Para habilitar la función de arrastrar y soltar inteligente:

Al arrastrar y soltar archivos en File Station, esta función determinará automáticamente si la transferencia es una acción de copiado o de movimiento.

  1. Seleccione Habilitar arrastrar y soltar inteligente.
  2. Seleccionar la acción que se debe llevar a cabo si se da la situación de que haya otro usuario transfiriendo archivos simultáneamente con el mismo nombre en la misma ubicación:
    • Sobrescribir: sus archivos transferidos sobrescribirán a los archivos con el mismo nombre.
    • Omitir: sus archivos transferidos se omitirán y no sobrescribirán a los archivos con el mismo nombre.
  3. Haga clic en OK.

Nota:

Si desea cambiar la acción de copiado o movimiento determinada por la función de arrastrar y soltar inteligente, pulse las siguientes teclas de acceso rápido según sus necesidades:

  • Para cambiar a una acción de copiado: presione Ctrl mientras realiza la acción de arrastrar y soltar.
  • Para cambiar a una acción de movimiento: pulse la tecla Mayús mientras realiza la acción de arrastrar y soltar.

Para examinar y administrar archivos desde el ordenador local con File Station:

  1. Marque Examinar archivos desde ordenador local con File Station.
  2. Haga clic en OK.

Para compartir la lista de usuarios DSM con usuarios que no sean administradores:

Esta opción permite a los usuarios que no sean administradores acceder a la lista completa de usuarios DSM si desean buscar archivos pertenecientes a usuarios o grupos específicos de DSM, o si desean compartir vínculos de archivos con otros usuarios de DSM.

  1. Seleccione Compartir la lista de usuarios DSM con usuarios que no sean administradores.
  2. Haga clic en OK.

Aplicar permisos UNIX predeterminados

Haga clic en Configuración avanzada y marque la opción para aplicar permisos UNIX predeterminados al cargar o crear archivos y carpetas. Los permisos aplicados son los mismos que se aplican con el comando UNIX umask. Cuando se habilita esta opción, los permisos UNIX son 644 para archivos y 755 para carpetas. Cuando se deshabilita esta opción, los permisos UNIX son 666 para archivos y 777 para carpetas. El valor predeterminado de umask es 022.

Nota:

  • Para las carpetas compartidas con Windows ACL (todos los recursos compartidos excepto "photo" y los recursos compartidos en discos externos), ejecute el comando chmod en el cliente Linux o FTP para cambiar los tipos de permisos de carpeta y de archivo de Windows ACL a UNIX.
  • Si se habilita esta opción se pueden provocar problemas de permisos que no mantienen la consistencia entre protocolos diferentes. Para evitar inconsistencias, sugerimos no habilitar esta opción.

Montaje/conexiones

Por defecto, todos los usuarios pueden montar carpetas remotas o unidades virtuales. Aquí puede limitar el acceso a la administración de carpetas remotas o unidades virtuales sólo a los administradores.

Nota:

Los usuarios que no son administradores solo pueden montar carpetas remotas dentro de su carpeta home.

Para permitir que diferentes usuarios puedan montar carpetas remotas o unidades virtuales.

  1. Elija Todos los usuarios o Administrators en el apartado Carpeta remota o Unidad virtual.
  2. Haga clic en OK.

Para permitir que diferentes usuarios monten servidores y servicios en la nube:

  1. Elija en Todos los usuarios, Administrators o Usuario específico.
  2. Al seleccionar Usuario específico se abrirá una ventana que le permitirá configurar los permisos para diferentes usuarios. Para habilitar el acceso de un usuario específico, asegúrese de marcar la casilla en la columna Habilitado. Para deshabilitar el acceso de un usuario específico, asegúrese de desmarcar la casilla en la columna Habilitado. Cuando acabe, haga clic en OK para cerrar esta ventana.

    3. Nota:

    • Los usuarios que pertenecen al grupo administrators pueden utilizar esta función pero no pueden realizar ningún cambio. Los usuarios que inicien sesión con guest no pueden utilizar esta característica ni hacer ningún cambio. Los permisos de todos los demás usuarios se pueden configurar aquí.
  3. Haga clic en OK.

Enlaces compartidos

Por defecto, todos los usuarios pueden compartir vínculos de archivo. Aquí puede limitar el acceso a la administración de enlaces compartidos solo para el grupo administrators.

Para permitir que los siguientes usuarios compartan vínculos de archivo:

  1. Elija Todos los usuarios, Administradores, o Usuario específico.
  2. Haga clic en OK.

Para abrir archivos HTML como solo texto:

  1. Marque la casilla Abrir archivos HTML siempre en formato de solo texto. Si se habilita esta opción, los archivos HTML (o sea, archivos HTM, HTML, JSP, XHTML) se mostrarán utilizando un formato de solo texto cuando los usuarios los abran a través de un enlace de uso compartido. Esta opción ayuda a evitar que usuarios malintencionados se apropien de sesiones de usuario utilizando scripts incrustados en archivos HTML.
  2. Haga clic en OK.

Límite de velocidad

Puede habilitar el límite de velocidad para controlar el ancho de banda útil máximo para usuarios y grupos al transferir datos a través de File Station.

Para habilitar el límite de velocidad:

  1. Marque la casilla con la indicación de Habilitar límite de velocidad o Habilitar el límite de velocidad de acuerdo con un programa si desea imponer un límite de velocidad solo en determinados momentos. Este ajuste se realiza a nivel de servidor y anulará los ajustes de usuarios individuales y de grupos. Si elige Sin límite de velocidad deshabilitará los ajustes del límite de velocidad para todos los usuarios y grupos.
  2. Haga clic en Ajustes del límite de velocidad para modificar la configuración para determinados usuarios o grupos. Para cada usuario y grupo puede elegir una de las siguientes opciones:
    • Aplicar configuración de grupo: los ajustes del límite de velocidad serán los mismos que los del grupo al que pertenece el usuario.
    • Sin límite de velocidad: no se aplica ningún límite de velocidad cuando miembros del grupo transfieren archivos con el servicio.
    • Constante: se aplicará un límite de velocidad fijo constante para el usuario o los miembros del grupo. Los límites de velocidad de carga y descarga se pueden especificar en los campos a la izquierda.
    • Variable: se pueden especificar dos límites de velocidad diferentes al usuario o los miembros del grupo de acuerdo con un programa. Haga clic en el botón Personalizar para modificar los ajustes del límite de velocidad y definir un programa.
  3. Haga clic en Aplicar.

Nota:

  • Para obtener más información sobre la dinámica del límite de velocidad de grupo, consulte aquí.
  • Cuando esté definiendo el límite de velocidad para un usuario, la sección Resultado mostrará el límite de velocidad actual del usuario de acuerdo con el ajuste individual y de grupo. Por ejemplo, si el usuario "Achilles" tiene un límite individual de descarga de 100 kB/s, pero pertenece a un grupo de usuarios con un límite de descarga de 500 KB/s, entonces la sección de Resultados mostrará "0/500".

Seguridad

Aquí podrá encontrar los siguientes parámetros de seguridad.

Para abrir archivos HTML como solo texto:

  1. Marque la casilla Abrir archivos HTML siempre en formato de solo texto. Si se habilita esta opción, los archivos HTML (o sea, archivos HTM, HTML, JSP, XHTML) se mostrarán utilizando un formato de solo texto cuando se abran en File Station. Esta opción ayuda a evitar que usuarios malintencionados se apropien de sesiones de usuario utilizando scripts incrustados en archivos HTML.

    2. Nota:

    • Esta no es una configuración global. Cada usuario puede elegir si desea habilitar o no esta opción de acuerdo con sus necesidades individuales.
  2. Haga clic en OK.
