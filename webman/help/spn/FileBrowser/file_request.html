Crear solicitudes de archivo

Las solicitudes de archivo son invitaciones para la carga de archivos enviadas a usuarios a personas que no son usuarios de DSM. Gracias a los enlaces de solicitud de archivo, aquellas personas que no son usuarios de DSM pueden cargar directamente varios archivos en File Station en su DiskStation, sin la necesidad de contar con las cuentas de usuario ni los privilegios de carga de archivos correspondientes.

Para crear enlaces de solicitud de archivo:

  1. Vaya a File Station y seleccione una carpeta como destino de carga.
  2. Haga clic en Acción, o haga clic con el botón derecho en la carpeta de destino y seleccione Crear solicitud de archivo.
  3. En la ventana emergente, puede buscar o modificar la siguiente información:
    • Ruta de archivo: muestra la ruta del directorio de destino.
    • Vínculo: muestra el enlace de solicitud de archivo para compartirlo con los usuarios de destino.
    • Su nombre: escriba un nombre para indicar la identidad del solicitante. El valor predeterminado es su nombre de usuario.
    • Mensaje: permite personalizar el mensaje para los usuarios de destino.
    • Habilitar protección con contraseña: seleccione esta opción para que solo los usuarios con la contraseña puedan cargar archivos a través del enlace creado. Especifique la contraseña en el cuadro que aparece a continuación.
    • Periodo de validez: haga clic en este botón para personalizar la duración de este enlace de solicitud de archivo:
      • Configurar hora de finalización: seleccione esta opción para determinar la fecha y la hora en que el enlace dejará de funcionar para la carga de archivos.
      • Configurar hora de inicio: seleccione esta opción para determinar la fecha y la hora en que el enlace comenzará a funcionar para la carga de archivos.
      • Número de accesos permitido: seleccione esta opción para determinar la cantidad de veces que se puede acceder al enlace.
  4. Haga clic en Guardar para completar la configuración.

Para compartir enlaces de solicitud de archivo:

Para buscar la URL de solicitud de archivo, vaya a File Station > Herramientas > Administrador de enlaces compartidos y haga doble clic en el enlace que desee. Puede compartir el enlace de una de las dos maneras siguientes:

  • Sencillamente copie la dirección URL en los usuarios de destino.
  • Envíe el enlace a través del correo electrónico predeterminado (su cuenta de correo electrónico de DSM) o a través del correo (una aplicación de correo electrónico de terceros). Si es nuevo usuario, siga las instrucciones del asistente para configurar una cuenta de correo electrónico que pueda utilizar.

Para cargar archivos:

Si se le pide que cargue archivos a través del enlace, realice el siguiente procedimiento:

  1. Abra la dirección URL que se le ha proporcionado en el equipo local o en un dispositivo móvil (Android o iOS). Introduzca la contraseña, si fuera necesario.
  2. En la página de carga, introduzca un nombre que indique su identidad.
  3. Simplemente arrastre y suelte los archivos que desee cargar, o haga clic en Añadir archivos para seleccionar los archivos.
  4. Haga clic en Cargar para terminar la solicitud.

Nota:

  • No pueden cargar carpetas a través de un enlace de solicitud de archivo.
  • Las carpetas que están dentro de la carpeta compartida "web" no pueden especificarse como la carpeta de destino.
  • La función de carga varía en función de los exploradores web:
    • Internet Explorer 8 e Internet Explorer 9 no admiten la carga de archivos a través del método de arrastrar y soltar.
    • Chrome en dispositivos iOS solo admite la carga de imágenes.
    • Safari en dispositivos iOS solo admite la carga de imágenes y vídeos.
    • Los exploradores web de dispositivos Android no están sujetos a ninguna limitación de tipo de archivo.

Para buscar los archivos cargados:

Los archivos cargados por el mismo usuario aparecerán en la misma carpeta en el destino. De forma predeterminada, el nombre de la carpeta será nombre_cargador (nombre_solicitante).

Por ejemplo, si los archivos fueron cargados por Watson a Holmes, podrá encontrar los archivos en la carpeta llamada Watson (Holmes).

Gestionar enlaces de solicitud de archivo

Para modificar enlaces de solicitud de archivo:

Después de crear un enlace de solicitud de archivo, puede modificar su configuración en File Station > Herramientas > Administrador de enlaces compartidos.

  1. Haga doble clic en el enlace que desee, o seleccione el enlace y haga clic en Editar.
  2. En la ventana emergente, modifique los ajustes correspondientes.
  3. Haga clic en Guardar para aplicar la nueva configuración.

Para eliminar enlaces de solicitud de archivo:

  1. Vaya a File Station > Herramientas > Administrador de enlaces compartidos.
  2. Seleccione los enlaces que desee y haga clic en Eliminar. Las URL de solicitud de archivo correspondientes dejarán de funcionar inmediatamente.

Para asignar privilegios para crear enlaces de solicitud de archivo:

Como usuario admin o miembro del grupo administrators, puede asignar privilegios a los usuarios para que puedan crear enlaces de solicitud de archivo.

  1. Vaya a File Station > Configuración > Enlaces compartidos > Enlaces de solicitud de archivo.
  2. Seleccione el tipo de usuarios a los que concederá los privilegios.
  3. Haga clic en OK para guardar la configuración.
