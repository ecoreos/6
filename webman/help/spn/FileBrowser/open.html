Abrir archivo en una nueva ventana

Puede abrir un archivo de DiskStation en una nueva ficha o ventana del explorador web para obtener una vista previa instantáneamente.

Para abrir archivo en una nueva ventana:

  1. Seleccione el archivo del que desea obtener una vista previa.
  2. Realice una de las siguientes operaciones:
    • Elija Abrir en una ventana nueva en el menú Acción.
    • Haga clic con el botón secundario en el archivo y elija Abrir en una ventana nueva.

Nota: La configuración del explorador decidirá los tipos de archivos que se pueden ver directamente.
