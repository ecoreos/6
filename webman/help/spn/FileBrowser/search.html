Buscar archivos o carpetas

El campo Filtrar de File Station le ofrece opciones para buscar y visualizar archivos o carpetas que cumplen sus criterios. Puede filtrar archivos o carpetas en la carpeta actual, o bien realizar una búsqueda avanzada que abarque todas las subcarpetas para obtener unos resultados más refinados.

Buscar archivos o carpetas en la carpeta actual:

  1. Utilice File Station para ir a la carpeta que contiene contenido que desea buscar.
  2. En el campo Filtrar en la esquina superior derecha del campo, escriba una palabra clave y verá los resultados filtrados en el área principal de la ventana.

Para realizar una búsqueda avanzada:

  1. Utilice File Station para ir a la carpeta que contiene contenido que desea buscar.
  2. Haga clic en el icono de lupa que hay junto al campo Filtrar en la esquina superior derecha para ver el menú de búsqueda avanzada.
  3. Especifique los criterios de búsqueda que se indican a continuación o elija en los menús desplegables:
    • Palabra clave: introduzca una palabra clave para buscar archivos o carpetas con los nombres coincidentes.

      Nota:

      • Puede utilizar los comodines [*] (que significa 0 o más caracteres) y [?] (que significa un carácter aleatorio) en los teclados. Por ejemplo, [a*n] puede representar [action], [attention], o [an].
      • Puede utilizar espacios para separar varias palabras clave. Por ejemplo, la palabra clave [File Station] generará resultados de búsqueda que contengan la palabra [File], la palabra [Station], o ambas.
      • Puede añadir comillas ["] al principio y final de las palabras clave para buscar coincidencias exactas que distingan entre mayúsculas y minúsculas. Por ejemplo, ["Synology Inc."] puede representar a [Synology Inc.] pero no a [Synology Incorporation].
      • Los operadores que se mencionan a continuación se pueden utilizar para delimitar los resultados de la búsqueda:

        && [X] && [Y]

    • Habilitar búsqueda de contenido de archivos: busca archivos indexados cuyos metadatos o contenido de archivo coincidan con las palabras clave.
    • Ubicación: Seleccione en el menú desplegable para establecer el rango y la profundidad de la búsqueda avanzada. También puede personalizar el ámbito de búsqueda seleccionando cualquier otra carpeta compartida de la lista.
    • Tipo de archivo: elija Extensión para limitar el ámbito de búsqueda a tipos de archivos específicos; también puede elegir Cualquier carpeta para buscar solo en carpetas.
    • Fecha: elija Fecha de modificación, Fecha de creación o Último acceso y luego elija una fecha del calendario para buscar archivos con fechas coincidentes.
    • Tamaño (MB): elija entre igual a, es mayor que, y es menor que, e introduzca un tamaño para buscar archivos con los tamaños coincidentes (si elige igual a, los resultados de la búsqueda incluirán archivos de aproximadamente el mismo tamaño especificado).
    • Propietario/Grupo: Elija Propietario o Grupo, y luego elija del menú desplegable para buscar archivos o carpetas que pertenezcan al propietario o grupo seleccionados.

      Nota:

      • Para realizar búsquedas avanzadas deberá especificar una Ubicación y al menos otro criterio de búsqueda.
      • Si se monta una carpeta ISO en una carpeta remota o en una carpeta compartida indexada, no podrá encontrar los archivos en su interior con la búsqueda avanzada.
  4. Haga clic en Buscar para iniciar la búsqueda, o haga clic en Restablecer y vuelva a especificar los criterios de búsqueda. Una vez completada la búsqueda, los archivos coincidentes se enumerarán en el área principal de File Station.
