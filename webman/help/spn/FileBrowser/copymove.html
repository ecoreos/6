Copiar o mover archivos o carpetas

File Station le proporciona diversos métodos para transferir archivos/carpetas entre dos DiskStation, dentro de un DiskStation, o a un ordenador local.

Arrastrar y soltar archivos/carpetas entre exploradores

Sin necesidad de configuraciones complejas de protocolos de transferencia de archivos, puede cargar archivos/carpetas directamente desde File Station de un DiskStation a otro.

Para habilitar la opción de arrastrar y soltar entre exploradores:

  1. Acceda a Configuración > General > File Station.
  2. Marque Habilitar arrastrar y soltar entre exploradores.
  3. Haga clic en OK.
  4. Abra dos exploradores web del mismo tipo (p. ej., Firefox) y conéctelos al DiskStation de origen y de destino respectivamente.
  5. Inicie File Station en ambos DiskStation. Ahora puede arrastrar y soltar archivos/carpetas directamente entre ellos mediante File Station.

Nota:

  • También puede arrastrar y soltar archivos/carpetas entre dos fichas del mismo explorador.
  • Internet Explorer no es compatible con la función de arrastrar y soltar entre navegadores y pestañas.
  • Si se seleccionan varias carpetas/archivos para arrastrar y soltar, se comprimirán en un solo archivo para cargar.
  • El límite de tamaño de archivo para la opción de arrastrar y soltar varía según el explorador web: Safari (1 GB).

Copiar o mover archivos/carpetas en DiskStation o a un ordenador local

Siga estos pasos para mover o copiar archivos o carpetas en DiskStation o al ordenador local.

Método 1:

  1. Seleccione los archivos o carpetas que desee mover o copiar. (Pulse y mantenga pulsada la tecla Mayús o Ctrl para la selección de varios elementos).
  2. Elija Cortar o Copiar en el menú desplegable Acción.
  3. Acceda a la carpeta de destino en DiskStation o en su ordenador.
  4. Elija Pegar - Sobrescribir o Pegar - Omitir en el menú desplegable Acción, en función de si desea sobrescribir archivos o carpetas con el mismo nombre en la carpeta de destino.

Método 2:

  1. Seleccione los archivos o carpetas que desee mover o copiar. (Pulse y mantenga pulsada la tecla Mayús o Ctrl para la selección de varios elementos).
  2. Arrastre las carpetas o los archivos desde la sección principal a otra carpeta del panel izquierdo.
  3. En el menú emergente que aparece, elija Copiar - Sobrescribir, Copiar - Omitir, Mover - Sobrescribir o Mover - Omitir para sobrescribir u omitir archivos o carpetas con el mismo nombre en la carpeta de destino.

Método 3

  1. Seleccione los archivos o carpetas que desee mover o copiar. (Pulse y mantenga pulsada la tecla Mayús o Ctrl para la selección de varios elementos).
  2. Pulse Ctrl + X para cortar o Ctrl + C para copiar los archivos/carpetas. Para usuarios de Mac, pulse Command + X o Command + C.
  3. Acceda a la carpeta de destino en DiskStation o en su ordenador.
  4. Pulse Ctrl + V para pegar los archivos/carpetas. (Command + V para Mac).
  5. En el cuadro de diálogo que aparece, elija si desea sobrescribir u omitir archivos con el mismo nombre en la carpeta de destino.

Teclas de acceso directo:

Las siguientes combinaciones de teclas pueden resultar útiles cuando mueva/copie archivos y carpetas en File Station:

  • F2: Renombrar
  • Del: Eliminar
  • Ctrl + A: Seleccionar todo
  • Crtl + X: Cortar
  • Crtl + C: Copiar
  • Crtl + V: Pegar

Nota:

  • Al copiar o mover archivos o carpetas a un directorio que tenga el sistema de archivos FAT, los caracteres \ *?">| del nombre de archivo serán sustituidos por –, y se eliminarán los metadatos de los archivos de Mac.
  • Si corta archivos de su ordenador local, y luego los pega en una carpeta DiskStation, los archivos seguirán conservándose en su ordenador local.
  • Al mover un archivo o carpeta, el "propietario" sigue siendo el mismo. Sin embargo, cuando copie un archivo o carpeta, el "propietario" pasará a ser el usuario que los copió.
