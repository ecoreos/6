Comprimir archivos o carpetas

  1. Elija archivos o carpetas de DiskStation en el área principal.
  2. Elija Comprimir a “(nombre de archivo)”.zip en el menú desplegable Acción para comenzar la compresión.
  3. Si no puede ver el archivo zip cuando termine la compresión, elija Actualizar en el menú desplegable Acción para ver el archivo.
  4. Si desea cambiar de nombre, cifrar el archivo de archivación o archivarlo con diferentes niveles de compresión o modos, elija Agregar a archivación... en su lugar.
    Nota:
  1. Con File Station no puede comprimir los archivos o carpetas de su equipo local.
  2. Formatos de archivación admitidos: .zip, .7z.
  3. Cuando elija .zip como Formato de archivación en Agregar a archivación..., puede seleccionar una página de código para el archivo .zip.
