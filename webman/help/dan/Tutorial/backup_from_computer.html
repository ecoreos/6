Backup af data på min PC/Mac med Cloud Station

Oversigt

Cloud Station Server er ikke kun en sofistikeret synkroniseringssoftware på tværs af platforme. Brugere kan bruge den som en smartere måde at udføre backup af deres personlige computere. Cloud Station Backup er en backupløsning i realtid, som er øjeblikkelig og sparer båndbredde, så du får bedre beskyttelse med op til 32 historiske versioner af én fil.

Konfigurer Cloud Station Server på Synology NAS

  1. Gå til Pakkecenter, søg efter Cloud Station Server, og klik så på Installer.
  2. Start Cloud Station Server.
  3. Gå til Oversigt, og klik på Download nu under Cloud Station Backup. (Du kan også downloade Cloud Station Backup  fra Synologys Downloadcenter).
  4. Start installationsprogrammet på computeren, og følg instruktionerne til installation af Cloud Station Backup.
  5. Efter installationen kører du Cloud Station Backup på computeren. Klik på Start nu
  6. Indtast adressen på den Synology NAS, der kører Cloud Station Server (eller QuickConnect ID), brugernavn og adgangskode. (Du kan også klikke på søgeikonet til højre, hvorefter Cloud Station Backup automatisk vil søge efter andre Synology NAS på dit LAN). Klik på Næste.
  7. Vælg din backupkilde, og afmarker eventuelle undermapper, som du ikke vil synkronisere.
  8. Vælg en mappe på din Synology NAS ved at klikke på Vælg. Cloud Station Backup vil oprette en backupmappe på din valgte destination med dit computernavn.
  9. Klik på Backupregler, hvis du vil angive synkroniseringsfiltre eller -regler.
  10. Klik på Næste for at gennemgå dine indstillinger for backup, og klik på  Udført.
  11. Du kan vise dine backupstatus i hovedprogrammet.

Download eller gendan en tidligere version af en synkroniseret fra Cloud Station Server

  1. Gå til Cloud Station Server > Versionshistorik, og find den fil eller mappe, som du vil gendanne.
  2. Vælg den fil, du vil hente, og klik på Handling. Klik på Gennemse tidligere versioner. Hvis en fil ved et tilfælde bliver slettet eller fjernet fra computeren, kan du klikke på Gendan for at gendanne den.
  3. Vælg den version af den fil, du vil downloade, og klik på Download. Eller, hvis du ikke er sikker på, du vil have den valgte version gendannet, skal du klikke på Gendan, hvorefter den bliver overskrevet med din aktuelle version.
