Opret en gruppe

  1. Gå til Kontrolpanel > Gruppe.
  2. Klik på Opret. Denne handling starter Guide til oprettelse af bruger.
  3. Indtast gruppenavnet og beskrivelsen. (Se i nedenstående afsnit for at få flere oplysninger). Klik på Næste.
  4. Følg de resterende trin i guiden for at afslutte oprettelse af gruppen.
  5. Klik på Anvend for at afslutte.

Navngivningskonventioner:

  • Gruppenavnet skelner mellem store og små bogstaver og skal være mellem 1 og 32 Unicode-tegn. Følgende tegn kan ikke bruges:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Det første tegn må ikke være et minustegn eller et mellemrum, ligesom det sidste tegn ikke må være en tom plads.
  • Gruppebeskrivelsen skelner mellem store og små bogstaver og er begrænset til 64 visbare Unicode-tegn.
