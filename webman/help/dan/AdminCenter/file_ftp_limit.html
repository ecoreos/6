Forbindelsesbegrænsning

Tillad kun SSL/TLS-forbindelse

Når denne funktion er aktiveret, vil serveren kun acceptere SSL eller TLS-forbindelsesanmodninger fra FTP-klienter.

Begræns forbindelser pr. IP

Når denne funktion er aktiveret, vil serveren begrænse maks. forbindelser pr. IP. Aktiver denne funktion og FTP-båndbreddebegrænsning for at få effektiv båndbreddestyring.

Aktiver FTP-båndbreddebegrænsning

Aktiver denne funktion for at begrænse båndbredden for enkelt FTP-forbindelse.
