Katalogtjeneste

Din DiskStation kan være med i en eksisterende katalogtjeneste som Windows-domæne eller LDAP-klient. Du kan administrere brugeres adgangsprivilegier til domæne eller LDAP til delte mapper og DSM-programmer og aktivere starttjenesten for dem. Gå til Kontrolpanel > Katalogtjeneste for at tilmelde din DiskStation til en katalogtjeneste.
