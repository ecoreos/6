Webtjenester

Din DiskStation leveres med rige værtsfunktioner til nettet. Med Web Station kan du nemt være vært for og publicere dit eget websted med VHost-support og yderligere HTTP/HTTPS-indstillinger. Alternativt kan du implementere din indbyggede PHP-support og skabe dynamiske databasedrevne websteder til personlig brug eller forretningen.
