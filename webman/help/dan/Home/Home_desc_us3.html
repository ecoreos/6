Onlineressourcer

Hvis du ikke kan finde det, du har brug for her, skal du kigge i Synologys onlineressourcer nedenfor.

  • Vidensbase: www.synology.com/support.php
  • Forum: Forum.synology.com
  • Downloadcenter: www.synology.com/support/download.php
  • Teknisk support: account.synology.com/support/support_form.php

Lige meget om du nøjagtigt ved, hvad du leder efter eller bare vil søge efter, hvad du har brug for, vil DSM - Hjælp give detaljerede forklaringer om følgende emner og ressourcer.

Hovedmenu

Start DSM-tjenester og pakker, der er installeret på din DiskStation.

Valgmuligheder

Ret dine kontoindstillinger, f.eks. adgangskode, beskrivelse eller 2-trins verificering.

Kontrolpanel

Se og ret forskellige DSM-indstillinger, der er relateret til fildeling, system, netværkstjenester og mm.

File Station

Organiser, upload eller download mapper og filer, der er gemt på din DiskStation.

Backup og gendan

Sikkerhedskopier og gendan data, iSCSI LUN'er eller systemkonfigurationer på din DiskStation.

Backup og gendannelse af data eller systemkonfigurationer på din DiskStation.

Pakkecenter

Installer, opdater og administrer Synology eller tredjepartsprogrammer.

EZ-Internet

Gør din DiskStation åben over nettet med en nem konfigurationsguide.

Ressourceovervågning

Overvåg CPU-brug, hukommelsesbrug, netværks-flow og diskenhedsbrug

Overvåg CPU-brugen, hukommelsesbrugen og netværksflow.

Systemoplysninger

Se den generelle status over din DiskStation og alle andre tilsluttede enheder.
