Switchover

Sørg for fortsat ydelse, og minimer tjenestenedetid ved at skifte tjenester fra den aktive server på en high-availability-klynge til den passive server på standby. Denne funktion kaldes "Switchover" eller i nogle tilfælde "Failover". Dette afsnit indeholder grundlæggende oplysninger om disse to begreber.

Switchover

Switchover refererer til et skift fra den aktive til den passive server på standby i en high-availability-klynge. Under normale forhold styrer den aktive server alle tjenester. I tilfælde af at den aktive server fejler, kan du starte en switchover og overføre alle tjenester til den passive server under forudsætning af, at den passive server fungerer normalt. I det tilfælde påtager den aktive server sig rollen som passiv server og vice versa.

Sådan starter du switchover manuelt:

  1. Gå til siden Oversigt.
  2. Klik på Administrer > Switchover.

Bemærk:

Før manuel switchover anbefales det, at du konfigurerer en statisk IP og logger ind på DSM med denne statiske IP for at undgå DSM-sidetab pga. IP-skifte efter switchover.

Bemærk:

Når du udfører en switchover, påtager den aktive server sig først rollen som den passive server, hvorefter den passive server bliver den aktive. Derfor vil der gå en kort tidsperiode, hvor begge servere bliver passive og ikke kan yde tjenester normalt. Switchover'ens fuldførelsestid varierer, afhængigt af antallet og størrelsen på diskenheder eller iSCSI LUN'er (blokniveau) eller antallet og den samlede belastning af tjenester og pakker i high-availability-klyngen.

Failover

Når visse fejl opstår, kan systemet automatisk påbegynde en switchover af tjenesterne for at minimere tjenestens nedetid. Denne begivenhed kaldes "failover". Systemet kan påbegynde en failover i følgende situationer, hvis den passive server fungerer normalt:

  • Strømnedbrud:
    • Den aktive server er lukket eller genstartet.
    • Begge strømenheder på den aktive server fejlede.
    • Ingen strømforsyning.
  • Diskfejl: Antal diskstier på den aktive server er mindre end på den aktive server.
  • Tjenestefejl: Der opstod en fejl på den overvågede tjeneste.
  • Forbindelsesfejl: Der opstod en forbindelsesfejl på et overvåget interface.

Switchover-fejl

Switchover eller failover fungerer måske ikke i følgende situationer:

  • Unormal Heartbeat-forbindelse: Status for aktiv-passiv serverforbindelse er ikke normal.
  • Hukommelsesstørrelser matcher ikke: Den aktive og passive server har ikke identisk RAM-størrelse.
  • Strømnedbrud:
    • Den passive server er lukket eller genstartet.
    • Begge strømenheder på den passive server fejlede.
    • Ingen strømforsyning.
  • Forbindelsesfejl: Der opstod en forbindelsesfejl på et overvåget interface på den passive server.
  • Tjenestefejl: Der findes fejl på overvågede tjenester på den passive server.
  • Forbindelsesfejl på udvidelsesenhed: Forbindelsen for udvidelsesenheder på de to servere er ikke i samme rækkefølge.
  • Diskfejl: Antal diskstier på den passive server er mindre end på den aktive server.
  • DSM-opdatering: Ved installation af DSM-opdatering standses alle tjenester, og de genoptages, når installationen af DSM-opdateringen er fuldført.

Hvis du manuelt påbegynder en switchover, men tjenesterne kan ikke skiftes til den passive server, vil systemet forsøge at skifte tjenester tilbage til den aktive server.

Uafbrudt strømforsyning

High-availability-klynger kan tilsluttes til netværks-UPS (uninterruptible power supply)-enheder for at reducere risikoen for uventet strømtab. Hvis strømmen går, vil serverne få backup-strøm fra den UPS-enhed, der er tilsluttet til dem. Når den UPS-enhed, der er tilsluttet til den aktive server ikke har så meget strøm tilbage, vil den passive server lukke, og den aktive server vil gå i sikret tilstand. Når strømmen kommer tilbage, vil den aktive server genstarte og forsøge at tænde for den anden server ved at sende en Væk på LAN-meddelelse.

  • Det anbefales at tilslutte begge servere og udvidelsesenheder til den samme UPS som de netværksomskiftere, der er tilsluttet.

Bemærk:

DiskStation understøtter kun tilslutning til netværks-USP-enheder. Yderligere oplysninger findes i hjælpen til Kontrolpanel > Hardware > UPS.
