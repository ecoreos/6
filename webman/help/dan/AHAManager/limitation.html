Begrænsninger og systemkrav

Implementering af Synology High Availability kræver to identiske Synology-servere, der kan være aktiv og passiv server. Før de to servere kombineres til en high-availability-klynge, vil systemet kontrollere følgende hardware- og softwarebegrænsninger for at sikre, at en high-availability-klynge kan oprettes.

Systemkrav

Bemærk følgende retningslinjer vedrørende software og hardware:

  • Synology-server: Både aktiv og passiv server skal være identiske modeller.
  • Udvidelsesenhed(er): Både aktiv og passiv server skal være korrekt tilsluttet til samme udvidelsesenheder.
  • Antal og disktype: Disktype kan kun være SAS. Før installation af DSM skal du installere mindst 3 diske på dine udvidelsesenhed(er).
  • Ekstern USB-enhed: Kun USB-lager understøttes. Andre eksterne USB-enheder (f.eks. UPS, Wi-Fi dongle, Højttaler, Printer, Bluetooth-dongle og DVB-dongle) understøttes ikke. USB-lager kan kun bruges på den aktive server og kan ikke skiftes over til den passive server.

Netværksmiljø

Bemærk følgende retningslinjer vedrørende netværksindstillinger ved forsøg på at danne en high-availability-klynge.

  • Netværksindstillinger: Heartbeat-forbindelsen mellem de to servere skal være konfigureret korrekt. Begge servere skal høre til samme undernet, og det kræves, at der konfigureres statiske IP-adresser efter installation.
  • Antal LAN-porte: Begge servere skal have samme antal LAN-porte. Hvis en af dine DiskStation-modeller er udstyret med yderligere netværkskortinterfacer, skal det andet udstyres med samme netværkskortinterfacer.
  • Forbindelsesbegrænsninger: Wi-Fi-netværk understøttes ikke af high-availability-klyngen.

Bemærk:

  • NTP-server bliver automatisk aktiveret på den aktive server.

Pakke- og softwarebegrænsninger

Se nedenstående pakke- og softwarerelaterede begrænsninger:

  • High-availiability-klyngen understøtter ikke følgende pakker og funktioner:
    • DVBLink Server
    • Funktionen til genstart/nedlukning af high-availability-klyngen i CMS
    • Time Backup
  • High-availability-klynge med delt lager kan ikke konfigureres som eller bruges sammen med en kopieringsbaseret high-availability-klynge.

Særlige udvidelsesenheder

Både aktiv og passiv server skal være tilsluttet til de særlige udvidelsesenheder. Der er to moduler på hver udvidelsesenhed.

Sådan konfigurerer/tilføjer du en udvidelsesenhed:

  • Et modul indeholder to porte: Input-porten til venstre og output-porten til højre. Tilslut et SAS-kabel fra output-porten på modulet til input-porten for den næste udvidelsesenheds modul.
  • Udvidelsesenhederne skal være tilsluttet serielt med modulerne på tilsvarende sider for hver server.
  • Både aktiv og passiv server skal være tilsluttet til de delte udvidelsesenheder i samme rækkefølge.

Noter:

Få detaljerede oplysninger om udvidelsesenheders installation ved at se i kvikinstallationsvejledningen til dine servere og udvidelsesenheder.
