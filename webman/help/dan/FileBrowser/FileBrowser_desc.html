File Station

File Station er et brugervenligt filstyringsværktøj, der sætter DSM-brugere i stand til over internettet at få adgang til DiskStation-mapper med deres webbrowsere. Ved at starte to eller flere File Stations kan du styre alle dine DiskStation-data ved at trække og slippe dem mellem forskellige File Stations.
