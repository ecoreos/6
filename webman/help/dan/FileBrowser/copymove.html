Kopier eller flyt filer eller mapper

File Station giver dig flere metoder til overførsel af filer/mapper mellem to DiskStation, på én DiskStation eller til en lokal computer.

Træk-og-slip filer/mapper mellem browsere

Uden kompleks konfiguration af filoverførselsprotokoller kan du direkte uploade filer/mapper via File Station fra én DiskStation til en anden.

Sådan aktiverer du træk-og-slip mellem browsere:

  1. Gå til Indstillinger > Generelt > File Station.
  2. Marker Aktiver træk-og-slip mellem browsere.
  3. Klik på OK.
  4. Åbn to webbrowsere af samme slags (f.eks. Firefox), og tilslut dem respektivt til kilde- og mål-DiskStation.
  5. Start af File Station på begge DiskStation. Du kan nu direkte trække-og-slippe filer/mapper mellem dem via File Station.

Bemærk:

  • Du kan også trække-og-slippe filer/mapper mellem to faner på den samme browser.
  • Internet Explorer understøtter ikke træk-og-slip mellem browsere eller mellem faner.
  • Hvis der vælges flere filer/mapper til træk-og-slip, bliver filer/mapper komprimeret til ét arkiv til upload.
  • Grænsen på filstørrelse for træk-og-slip varierer med webbrowsere. Safari (1 GB).

Kopier eller flyt filer/mapper i DiskStation eller til lokal computer

Følg nedenstående trin for at flytte eller kopiere filer eller mapper i DiskStation eller til lokal computer.

Metode 1:

  1. Vælg de filer og mapper, du vil flytte eller kopiere. (Tryk og hold på Shift eller Ctrl for at foretage flere valg).
  2. Vælg Klip eller Kopier i rullemenuen Handling.
  3. Gå til destinationsmappen på DiskStation eller computeren.
  4. Vælg Sæt ind - Overskriv eller Sæt ind - Spring over i rullemenuen Handling, afhængigt af, om du vil overskrive filer eller mapper med det samme navn i destinationsmappen.

Metode 2:

  1. Vælg de filer og mapper, du vil flytte eller kopiere. (Tryk og hold på Shift eller Ctrl for at foretage flere valg).
  2. Træk mapperne eller filerne fra hovedafsnittet til en anden mappe i venstre panel.
  3. I den pop op-menu, der vises, skal du vælge Kopier - Overskriv, Kopier - Spring over, Flyt - Overskriv elller Flyt - Spring over for at overskrive eller springe filer eller mapper over med det samme navn i destinationsmappen.

Metode 3

  1. Vælg de filer og mapper, du vil flytte eller kopiere. (Tryk og hold på Shift eller Ctrl for at foretage flere valg).
  2. Tryk på Ctrl + X for at klikke eller Ctrl + C for at kopiere filerne/mapperne. For Mac-brugere skal du trykke på Command + X og Command + C.
  3. Gå til destinationsmappen på DiskStation eller computeren.
  4. Tryk på Ctrl + V for at indsætte filerne/mapperne. (Command + V for Mac.)
  5. I den dialogboks, der vises, vælger du, om du vil overskrive eller springe filer med samme navn over i destinationsmappen.

Tastaturgenveje:

Følgende tastaturgenveje kan være nyttige, år du flytter/kopiere filer og mapper i File Station:

  • F2: Omdøb
  • Del: Slet
  • Ctrl + A: Vælg alt
  • Crtl + X: Klip
  • Crtl + C: Kopier
  • Crtl + V: Sæt ind

Bemærk:

  • Når du kopierer eller flytter filer/mapper til et bibliotek i FAT-filsystemet, bliver tegnene \ *?" | i filnavnet erstattet af –, og Metadata for Mac-filer bliver slettet.
  • Hvis du klipper filer fra den lokale computer og indsætter dem i en DiskStation-mappe, vil filerne stadig bevares på den lokale computer.
  • Ved flytning af en fil eller mappe, forbliver "ejeren" den samme. Ved kopiering af en fil eller mappe ændres "ejeren" dog til den bruger, der kopierede den.
