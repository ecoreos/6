Säkerhet

Denna flik låter dig ändra nätverkssäkerhetsinställningarna för varje nätverkskort.

DoS-skydd

Denial-of-service (DoS)-skydd hjälper dig att förebygga skadliga attacker över Internet.

Aktivera DoS-skydd:

  1. Markera rutan Aktivera DoS-skydd.
  2. Klicka på Tillämpa.

VPN Pass-through*

När DiskStation agerar router låter VPN pass-through klienter att passera genom NAT hos DiskStation och upprätta utgående eller ingående VPN-anslutningar. Om din interna konfiguration inkluderar en VPN-server, vänligen gå till Kontrollpanel > Konfiguration av router för att konfigurera regler för vidarebefordran av portar och tillåta VPN-servern att upprätta VPN-anslutningar.

För att redigera inställningar för VPN pass-through:

  1. Följande inställningar kan aktiveras eller inaktiveras här:
    • PPTP Pass-through: Tillåter PPTP (Point-to-Point Tunneling Protocol)-trafik att passera genom och upprätta anslutningar.
    • IPSec Pass-through: Tillåter IPSec (IP Security Protocol)-trafik att passera genom och upprätta anslutningar.
    • L2TP Pass-through: Tillåter L2TP (Layer 2 Tunneling Protocol)-trafik att passera genom och upprätta anslutningar.
  2. Efter att du ändrat inställningarna, klicka på Tillämpa.

____
* Endast tillgängligt för vissa modeller.
