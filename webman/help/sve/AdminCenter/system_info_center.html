Informationscenter

Informationscenter visar grundläggande information om diverse aspekter för din DiskStation, såsom hårdvaruspecifikationer, nätverk, lagring och tjänster. Informationscenter finns i Kontrollpanel > Informationscenter.

Allmänt

Fliken Allmänt visar grundläggande information om DiskStation, såsom serienummer, modellnamn, mängden fysiskt minne, DSM-version, tidsinformation, värmestatus eller externa enheter.

Temperaturstatus*

Temperaturstatus för DiskStation visas med gröna och orange lampor:

  • Grön: Temperaturstatusen är normal.
  • Orange: Temperaturen för DiskStation är för hög. Om temperaturen för DiskStation fortsätter att vara hög kan det hända att den stänger av på grund av överhettning.

Nätverk

Fliken Nätverk visar nätverksinformation, såsom DNS-server, IP-adress, subnätmask med mera.

Lagring

Fliken Lagring visar information om volymanvändning och hårddiskstatus.

Fliken Lagring visar information om användning av lagringsutrymme och status för externa lagringsenheter.

Hårddiskstatus

Nedan följer de möjliga statustyperna för en intern hårddisk:

  • Ingen hårddisk installerad: Ingen hårddisk finns installerad i motsvarande uttag för hårddisk.
  • Ej initierad: Systemets partitionering och volym har inte fastställts på hårddisken.
  • Normal: Systemets partitionering och volym finns tillgängliga för att användas på hårddisken.
  • Initierad: Systemets partitionering har fastställts, men ingen volym har skapats.
  • Systempartitioneringen misslyckades: Systempartitioneringen på hårddisken är inte tillgänglig.
  • Kraschad: Misslyckades med att få åtkomst till volymen på hårddisken. Vi rekommenderar att du byter hårddisk.

Obs:

För att systemet ska fungera normalt måste det finnas minst en disk med statusen Normal eller Initierad.

Tjänst

Fliken Tjänst visar en lista över tjänster och installerade filpaket, inklusive alternativ för att aktivera eller avaktivera varje tjänst och filpaket. Dessutom finns det för en del tjänster och filpaket alternativ för att testa uppkopplingen eller välja huruvida en tjänst eller ett filpaket kan vara åtkomligt genom brandväggen.

För att ändra inställningar för tjänster eller filpaket:

  1. Följande alternativ finns:
    • Aktivera: Aktivera eller avaktivera tjänsterna eller filpaketen.
    • Tillåt genom brandvägg: Lägg till eller ta bort de portar som används av tjänsten eller applikationen i Kontrollpanel > Extern åtkomst > Konfiguration av router, vilket låter tjänster eller filpaketet att passera genom brandväggen på DiskStation.
    • Inställningar: Välj det nätverkskort vars brandväggsinställningar du vill visa och hantera. Du kan välja Alla för att visa och hantera brandväggsinställningarna för alla nätverkskort.
    • Testa anslutning: Testa de portar som används av tjänster och filpaket. Detta låter dig testa huruvida enheter kan nå tjänster och filpaket från ett externt nätverk, eller inte.
  2. Klicka på Spara för att spara ändringarna eller Återställ för att avvisa ändringar.

Synology-konto

Synology-konto är en välintegrerad webbplattform som hjälper användare att få åtkomst till följande tjänster som Synology erbjuder:

  • Tjänsterna QuickConnect och DDNS för din DiskStation
  • Hantering av programvarulicenser
  • Brådskande Synology-support
  • Prenumeration på eNyheter
  • Inköpshistorik för Synology-produkter

Obs:

Du kan även logga in på ditt Synology-konto med ett konto för socialt nätverk (Facebook, Google+ och LinkedIn).

För att registrera eller logga in på ditt Synology-konto:

  1. Gå till Kontrollpanel > Informationscenter > Synology-konto.
  2. Klicka på Logga in på eller registrera ett Synology-konto.
  3. Specificera användaruppgifterna eller klicka på Registrera dig för att registrera ett Synology-konto. Alternativt, kan du logga in med ett socialt nätverk.

Att hantera ditt Synology-konto:

  1. Klicka på Hantera konto för att komma till webbplattformen.
  2. Följ instruktionerna på webbplatsen för att få åtkomst till att hantera ditt Synology-konto.

Att logga ut från ditt Synology-konto:

Klicka på Ändra konto för att logga ut från ditt Synology-konto.

Obs:

Att logga ut från ditt Synology-konto kan göra att tillhörande Synology-tjänster (t.ex. QuickConnect och DDNS) upphör.
