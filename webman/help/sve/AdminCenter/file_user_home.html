Användarhem

Gör det möjligt för användarhem att skapa en personlig mapp, home, för varje användare, utom för guest. Alla användare kan komma åt sin egen home-mapp via CIFS, AFP, FTP eller File Station.

Admin kan tillträda alla personliga mappar som finns i delad standardmapp homes. Namnet på home-mappen är det samma som användarkontot.

Aktivera tjänsten Användarhem:

  1. Klicka på Användarhem.
  2. Markera Aktivera tjänsten Användarhem.
  3. Om det finns fler än en volym på Synology DiskStation, välj var mappen homes ska förvaras.
  4. Klicka på OK.

Obs! Om tjänsten Användarhem för lokala användare inaktiveras kommer tjänsten Användarhem för domänanvändare också att inaktiveras samtidigt.

Avaktivera tjänsten Användarhem:

  1. Klicka på Användarhem.
  2. Avmarkera Aktivera tjänsten Användarhem.

Efter att tjänsten Användarhem inaktiverats kommer den delade mappen homes att sparas för admin-åtkomst. Användare kommer att kunna tillträda sina home-mappar igen när tjänsten för användarhem aktiveras igen.

    Obs!
  1. För att radera den delade mappen homes måste först tjänsten användarhem inaktiveras.
  2. Tjänsten Användarhem för domänanvändare inaktiveras om tjänsten Användarhem för lokala användare inaktiveras.
