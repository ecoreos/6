PHP-inställningar

När du har aktiverat Web Station kommer inställningarna för PHP som beskrivs här nedanför kunna konfigureras genom att gå till Kontrollpanel > Webbtjänster > PHP-inställningar.

Aktivera PHP-cache

Markera Aktivera PHP-cache om du vill förbättra PHP-prestandan. Under vissa omständigheter ger detta dock inte någon effekt.

Visa PHP-felmeddelande på hemsidan

Markera Aktivera display_errors för att visa PHP-felmeddelanden om du vill visa PHP-felmeddelanden på hemsidan.

Anpassa PHP open_basedir

  1. Markera Anpassa PHP open_basedir om du vill anpassa din egen open_basedir.
  2. Skriv in den anpassade sökvägen i textfältet och klicka sedan på Tillämpa. Om du vill använda standardinställningar, avmarkera Anpassa PHP open_basedir och klicka på Tillämpa.

Obs!

  • Använd ett kolon (:) för att avgränsa varje sökväg. Till exempel:/etc:/tmp:/usr/bin/php.
  • Av säkerhetsskäl kan sökvägen till den delade mappen homes (dvs. /volume1/homes) inte läggas till i open_basedir.

Att välja PHP extension

  1. Klicka på Välj PHP extension för att anpassa din eget PHP extension.
  2. Markera det tillägg du vill aktivera i dialogrutan PHP extension-lista.
  3. Du kan också klicka på Standard överst i rutan för att återgå till standardinställningarna.
  4. Klicka på OK för att spara ändringarna.
