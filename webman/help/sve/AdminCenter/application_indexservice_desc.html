Indexeringstjänst

Mediaindexering kan automatiskt skanna flera multimediafiler som finns lagrade på din DiskStation samtidigt, såsom foton, musik och videor och kompilera dem till ett multimediabiblioteket för att användas av multimediaapplikationer. Du kan även ställa in miniatyrbildernas kvalitet för bilder och videor, hantera konverteringsinställningar för videor och skapa indexerade mappar som kan användas av DMA-enheter och multimediaapplikationer i DSM.

Filindexering gör att du kan indexera mappar och de filer de innehåller, på din DiskStation, efter filnamn och du kan även indexera filens innehåll/metadata efter filtyp. Denna detaljerade funktion för indexering snabbar upp din sökhastighet för mer precisa träffar.
