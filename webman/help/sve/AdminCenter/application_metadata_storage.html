Multimediadatabas

DiskStation använder en multimediadatabas för att spåra multimediafiler och loggfilsposter som finns lagrade på mediakällan (delade mappar som skapats på dina externa lagringsenheter såsom usbshare1).

Viktigt: Om du vill att DSM:s multimediatjänster (såsom Mediaserver, iTunes, Audio Station, och Download Station) ska kunna komma åt lagrade multimediafiler på de externa lagringsenheter som är anslutna till USB-porten på din DiskStation eller om du vill aktivera DSM:s loggtjänster och se loggfilerna via Huvudmeny > Systeminformation > Loggfil, måste multimediadatabasen aktiveras först. Annars kommer alla relaterade tjänster att inaktiveras.

Så här gör du för att aktivera multimediadatabasen:

  1. Markera kryssrutan för Aktivera multimediadatabas.
  2. Gör något av följande:
    • Välj Automatisk konfiguration om du vill att DSM automatiskt ska välja en delad mapp för att lagra multimediadatabasen. Om den externa lagringsenhet som innehåller den delade mappen tas bort kommer DSM automatiskt att välja en annan delad mapp för att lagra databasen.
    • Välj Välj en delad mapp där du vill lagra din databas och välj sedan i rullgardinsmenyn för att manuellt välja en delad mapp där du vill lagra databasen.
  3. Klicka på OK.

Varning: För att förhindra att databasen skadas måste du se till att mata ut dess lagringsenhet via Huvudmeny > Kontrollpanel > Externa enheter innan du tar bort den från DiskStation.

Så här gör du för att inkludera eller utesluta delade mappar från multimediakällan:

  1. Klicka på Multimediakälla.
  2. Markera eller avmarkera kryssrutorna intill de delade mapparna för att göra ditt val.
  3. Klicka på OK.

Multimediafiler och loggfiler i de valda delade mapparna kommer att indexeras och den indexerade informationen kommer att läggas till i multimediadatabasen.

Så här gör du för att uppdatera multimediadatabasen:

  1. Klicka på Uppdatera databas. Alla data i multimediadatabasen kommer att ersättas med omindexerad information om den aktuella multimediakällan.
  2. Klicka på OK.

Obs: Om en stor mängd data i multimediadatabasen inte längre används (till exempel om du kopplar bort en USB-hårddisk som aldrig mer kommer att användas med DiskStation, vilket därmed leder till överflödig indexerad information i databasen), kan du genom att uppdatera multimediadatabasen rensa alla oanvända data eller minska databasens storlek. Detta kräver dock repetitiv läsning från och skrivning till hårddisken, vilket kan konsumera systemresurser och leda till förkortad livslängd för hårddisken. Använd inte den här funktionen i onödan.
