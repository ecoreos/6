Katalogtjänst

Din DiskStation kan ansluta till en befintlig katalogtjänst såsom Windows-domän eller LDAP-klient. Du kan hantera domän- eller LDAP-användares åtkomstprivilegier till delade mappar och DSM-applikationer och aktivera tjänsten Användarhem för dem. Gå till Kontrollpanel > Katalogtjänst för att ansluta din DiskStation till en katalogtjänst.
