Mediaindexeringstjänst

Mediaindexeringstjänst kan automatiskt skanna flera multimediafiler samtidigt såsom foton, musik och videor som finns lagrade på din DiskStation och kompilera dem till mediabiblioteket för att kunna indexeras av multimediaapplikationer. De applikationer som är tillgängliga i Paketcenter är Media Server, Video Station, Audio Station, Photo Station och iTunes Server. När de har installerats på din DiskStation kommer de att visas i fältet Applikation.

Skapa ett nytt index för mediafiler:

Klicka på Skapa nytt index för att återskapa indexstrukturen för alla mediafiler i de indexerade mapparna, både standardmappar och de som du skapat. Processen utförs i bakgrunden.

Obs:

Mediafiler som läggs till indexerade mappar som använder protokollet NFS kommer inte att skannas automatiskt. Du kommer att behöva skapa nytt index manuellt genom att klicka på Skapa nytt index.
