Webbtjänster

Din DiskStation levereras med mångsidiga funktioner för webbvärdskap. Med Web Station kan du enkelt vara värd för och publicera din egen webbplats med stöd för VHost och ytterligare inställningar för HTTP/HTTPS. Alternativt kan du implementera det inbyggda stödet för PHP och skapa dynamiska databasdrivna webbplatser för personligt eller affärsmässigt bruk.
