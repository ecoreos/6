Översikt

Kontrollpanel (Enkelt läge) > Översikt är en snabb och centraliserad gateway som låter dig hantera olika grundläggande inställningar och övervaka systemresurserna för din DiskStation.

Systemhälsa

Det här avsnittet visar den övergripande statusen för ditt system. Om ett fel har inträffat eller om dina lagringsutrymmen har kraschat eller blivit trasiga kan du föra din mus över detta avsnitt och klicka på ikonen Fler överst i det högra hörnet för att öppna motsvarande sidor för reparation och hantering.

DSM-version

Din aktuella DSM-version samt -status kommer att visas i det här avsnittet. När en ny DSM finns tillgänglig för nedladdning kan du uppdatera till den senaste versionen direkt härifrån.

QuickConnect ID

Du kan visa ditt QuickConnect ID här. För mer avancerade inställningar och hantering, för din mus över det här avsnittet och klicka på ikonen Fler i det övre högra hörnet för att öppna sidan QuickConnect.

Lagring

Det här avsnittet visar information om hårdvara och status för dina lagringsutrymmen, såsom hårddiskanvändning och volymanvändning/-kapacitet. Du kan kontrollera hur många hårddiskar som används och av vilka lagringsutrymmen de används. För mer avancerade inställningar och hantering, för din mus över det här avsnittet och klicka på ikonen Fler i det övre högra hörnet för att öppna sidan Lagringshanteraren.

Resursskärm

Du kan övervaka procenten för den övergripande CPU-användningen och hastigheten i KB för data som skickats och tagits emot. Om mer än ett LAN används, klicka på ikonen med en pil i flödesdiagrammet över nätverket för att byta. För mer avancerade inställningar och hantering, för din mus över det här avsnittet och klicka på ikonen Fler i det övre högra hörnet, för att öppna sidan Resursskärm.

Anslutna användare

Här kan du visa och hantera de klienter som för närvarande är anslutna till systemet och använder resurser. Detta avsnitt visar klienter som är anslutna via HTTP, TELNET, SSH, SMB/CIFS, AFP, FTP och NFS. För mer avancerade inställningar och hantering, för din mus över det här avsnittet och klicka på ikonen Fler i det övre högra hörnet, för att öppna sidan Resursskärm.

För att avsluta aktuella anslutningar:

Klicka på ikonen Avsluta anslutning bredvid den önskade användare för att hindra användaren från att få åtkomst till resursen. Den frånkopplade användaren måste blir auktoriserad på nytt för att få tillgång till resursen.
