SFTP

SFTP är ett filöverföringsprotokoll konstruerat som en förlängning av protokollet Secure Shell (SSH). SFTP använder endast ett TCP-portnummer. Dessutom kan privata och publika nycklar används för att autentisera användare utan lösenord. Dock blir överföringshastigheten mer långsam och mer systemresurser används vid kryptering.

Att aktivera tjänsten SFTP:

  1. Klicka på Aktivera STFP-tjänst.
  2. Klicka på Tillämpa.

Portnummer

Ställa in portnummer i SFTP-tjänsten.

Obs!

Standardportnumret för SFTP-tjänsten är 22. Du kan ställa in den från 1 till 65 535 med undantag från följande nummer:
20, 21, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, och eMule standardportar: 4662 (TCP), 4672 (UDP)
