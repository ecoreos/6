DHCP Server

Din DiskStation kan agera som en DHCP-server och tilldela dynamiska IP-adresser till DHCP-klienter inom ditt lokala nätverk.

Att aktivera DHCP-server:

  1. Gå till Kontrollpanel > DHCP Server > Nätverksgränssnitt.
  2. Välj ett nätverksgränssnitt (t.ex. LAN 1) att tilldela IP-adresser till och klicka på Redigera.
  3. I popup-fönstret, klicka på fliken DHCP Server och välj Aktivera DHCP-server.
  4. Specificera de följande inställningarna för DHCP:
    • Adresslånetid (minuter)
    • Primär DNS
    • Sekundär DNS (valfritt)
    • Domännamn (valfritt)
    • Aktivera automatisk upptäckt av webbproxy (valfritt)
  5. Konfigurera minst ett subnät. För konfiguration av subnät, klicka på Lägg till för att specificera följande information:
    • Start-IP-adress
    • Slut-IP-adress
    • Nätmask
    • Gateway
  6. Klicka på Skapa för att visa det subnät som skapats i listan.
  7. Efter att du skapat tillräckligt med subnät, kom ihåg att aktivera minst ett subnät.
  8. Klicka på OK.

Obs:

  • Du kan skapa så många subnät som du vill.
  • Om det koordineras med en DHCP-överföringsserver kan din DiskStation, i egenskap av DHCP-server tilldela IP-adresser till enheter på andra subnät.

DHCP-klienter

Du kan hitta en lista över DHCP-klienter och deras nätverkskonfiguration (t.ex. MAC-adress, IP-adress, värdnamn, tiden som återstår till DHCP-lånet upphör och status för reservationen).

  1. Gå till Kontrollpanel > DHCP Server > Nätverksgränssnitt och välj ett önskat gränssnitt.
  2. Klicka på Redigera > DHCP-klienter för att hitta klienterna och information om dem.

Att hantera IP-adresser för DHCP-klienter:

Du kan göra så att klienterna alltid får samma IP-adress vid förnyelse av DHCP-lån och hantera dessa klienter.

  • Lägga till DHCP-klienter: Klicka på Lägg till för att tilldela IP-adressen till klientenheten.
  • Att reservera IP-adressen: Klicka på ikonen med ett lås för att reservera IP-adressen för den aktuella enheten.
  • Att avbryta reservationen: Klicka på ikonen med ett lås för att tillåta att den aktuella IP-adressen används av en annan enhet efter att adresslånetiden passerat.

Obs:

Om en enhet kan ha både IPv4- och IPv6-adresser kan du reservera en IPv4- och en IPv6-adress för enheten.

Att söka efter DHCP-klienter:

  1. Klicka på ikonen med en tratt i det övre högra hörnet.
  2. Välj ett av nedanstående sökkriterier:
    • Alla: Listar alla DHCP-klienter och DHCP-reservationer.
    • DHCP-klienter: Listar alla DHCP-klienter med reserverade eller oreserverade IP-adresser.
    • DHCP-reservationer: Listar alla reserverade IP-adresser. En reserverad IP-adress kan ännu inte paras ihop med någon enhet.

DHCP-alternativ

Efter att du specificerat DHCP-alternativ kan din DiskStation, som DHCP-server, sända ut relaterad information till DHCP-klienterna i ett visst subnät.

Att specificera DHCP-alternativ för ett subnät:

  1. Gå till Kontrollpanel > DHCP Server > Nätverksgränssnitt.
  2. Välj ett önskat nätverksgränssnitt och klicka på Redigera.
  3. Gå till DHCP Server och gå till Subnätlista.
  4. Utför någon av nedanstående åtgärder:
    • Att specificera DHCP-alternativ för ett befintligt subnät: Välj det önskade gränssnittet och klicka på Redigera.
    • Att skapa ett subnät och specificera dess DHCP-alternativ: Klicka på Lägg till och specificera inställningarna för subnätet.
  5. Välj de önskade DHCP-alternativet; för att specificera det motsvarande värdena, klicka på de blanka fälten och redigera.
  6. Klicka på OK.

Försäljare

DHCP-klienter från olika försäljare kan kräva ytterligare information för DHCP. Genom att ställa in DHCP-alternativ för försäljarna kan din DiskStation tillhandahålla sådan information till klienterna.

För att få veta DHCP-alternativen som stöds av varje försäljare, vänligen kontakta de specifika försäljarna för information.

Att lägga till försäljare:

  1. Gå till Kontrollpanel > DHCP Server > Försäljare.
  2. Klicka på Skapa för att specificera försäljaren och dess DHCP-alternativ:
    • Namn: Specificera försäljarens namn.
    • Kod: Specificera den numeriska koden för DHCP-alternativet.
    • Värde: Specificera alternativet för värdet.
    • +: Lägg till ett DHCP-alternativ.
    • -: Radera ett DHCP-alternativ.
  3. Klicka på Skapa för att slutföra installationen.

Att redigera DHCP-alternativ för försäljare:

  1. Gå till Kontrollpanel > DHCP Server > Försäljare.
  2. Välj den önskade försäljaren och klicka på Redigera för att hantera dess DHCP-alternativ.
  3. Klicka på OK.

PXE

Din DiskStation kan agera som en DHCP-server för att implementera konfigurationen av PXE (Pre-Boot Execution Environment). PXE är en nätverksmiljö som låter klientenheterna starta genom att använda ett nätverksgränssnitt istället för en datalagringsenhet (t.ex. hårddisk).

När PXE är implementerat kan en bootloader lagras på din DiskStation eller en fjärransluten TFTP-server och klientenheter kan starta över nätverket genom att använda bootloadern.

Att använda din DiskStation som en bootloaderserver:

  1. Kontrollera att du har valt Aktivera TFTP-tjänst (i Kontrollpanel > Filtjänster > TFTP).
  2. Välj en mapp som är värd för bootloaderfilen som TFTP-rootmapp.
  3. Kontrollera att minst ett DHCP-subnät är aktiverat (i Kontrollpanel > DHCP Server > Nätverksgränssnitt).
  4. Gå till Kontrollpanel > DHCP Server > PXE.
  5. Välj Aktivera PXE (Pre-boot Execution Environment).
  6. Välj Lokal TFTP-server och välj bootloaderfilen.
  7. Klicka på Tillämpa för att spara inställningarna.

Att använda fjärransluten TFTP-server som en bootloaderserver:

Genom att använda en fjärransluten TFTP-server som en bootloaderserver kan du tillåta att klienter startas utan att behöva ändra infrastrukturen i det aktuella nätverket.

  1. Kontrollera att minst ett DHCP-subnät är aktiverat (i Kontrollpanel > DHCP Server > Nätverksgränssnitt).
  2. Gå till Kontrollpanel > DHCP Server > PXE.
  3. Välj Aktivera PXE (Pre-boot Execution Environment).
  4. Välj Fjärransluten TFTP-server och specificera serverns IP-adress och filsökvägen för bootloaderfilen.
  5. Klicka på Tillämpa för att spara inställningarna.
