HDD-viloläge

Du kan specificera den tidsperiod för vilken hårddiskarna ska vara inaktiva innan de går in i HDD-viloläge. Vid HDD-viloläge slutar hårddiskarna snurra, minskar energiförbrukning och utökar livslängden för hårddiskarna. Du kan specificera olika tidsperioder för interna hårddiskar och externa eSATA-/USB-hårddiskar.

Du kan specificera den tidsperiod för vilken hårddiskarna ska vara inaktiva innan de går in i HDD-viloläge. Vid HDD-viloläge slutar hårddiskarna snurra, minskar energiförbrukning och utökar livslängden för hårddiskarna. Du kan specificera olika tidsperioder för dina externa USB-hårddiskar.

Obs:

  • HDD-viloläge avslutas vid åtkomst.
  • Endast vissa externa USB-lagringsenheter har stöd för HDD-viloläge.

Varför går inte min DiskStation in i viloläge?

Vänligen notera att följande tjänster kommer att förhindra DiskStation från att inta viloläge. Därutöver kommer systemet att gå ut ur viloläge två minuter innan någon schemalagd uppgift.

Filpaket:

  • Download Station: DiskStation kan inte inta viloläge om Download Station kör eMule, RSS schemalagd nedladdning eller aktiva uppgifter.
  • Mail Station: DiskStation kan inte inta viloläge om Mail Station hämtar från POP3.
  • Surveillance Station: DiskStation kan inte inta viloläge om Surveillance Station utför inspelning.
  • Media Server: DiskStation kan inte inta viloläge om loggfiler för DMA är aktiverad i inställningar för Media Server (Gå till Huvudmeny > Media Server > Media Server > Diagnos).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Annan tredjeparts mjukvara eller optware såsom SABnzbd

Processer:

  • Miniatyrbilder och indexering: DiskStation kan inte inta vilolägen vid generering av miniatyrbilder eller ombyggnad av index för multimediafiler efter en uppgradering. Under denna tid kommer processen "synoindexd" att listas som "Körs" (Gå till Huvudmeny > Resursskärm > Process).
  • Local Master Browser: DiskStation kan inte inta viloläge om Local Master Browser är aktiverad (Gå till Kontrollpanel > Win/Mac/NFS > Windows-filservice > Aktivera Local Master Browser).
  • Windows Media Player: DiskStation kan inte inta viloläge om tjänsten för nätverksdelning är aktiverad för LAN i Windows Media Player.
  • DiskStation kan inte inta viloläge om antalet samtidiga processer kräver swap, när RAM-kapaciteten har överskridits och HDD:s tillfälligt används för läs-/skrivoperationer.

Övriga

  • USB-enheter: DiskStation kan inte inta systemvila om någon USB-enhet är ansluten.

Sovläge

Sovläge för expansionsenheter kan aktiveras genom gränssnittet hos den parade DiskStation. Du kan konfigurera att expansionsenheten och den parade DiskStation ska vara inaktiva vid olika tidsperioder. Sålunda kan expansionsenheten ställas in att gå in i sovläge medan den parade DiskStation förblir online.

Under sovläge:

  • Alla lysdioder i utökningsenheten kommer att vara avstängda.
  • Strömmen till hårddiskarna i expansionsenheten kommer att stängas av.
  • Fläkten i utökningsenheten kommer att stängas av.

Obs:

  • Sovläge avslutas vid åtkomst.
  • För att säkerställa funktionalitet när funktionen för sovläge används i utökningsenheten måste du skjuta brytaren Automatiskt/Manuellt på bakpanelen till Automatiskt för normal drift.

Automatisk avstängning

Du kan specificera längden på tidsperioden som de interna diskarna förblir i viloläge innan din DiskStation stänger av automatiskt.

Obs:

  • Du måste först aktivera HDD-viloläge.
  • Du måste först aktivera WOL-tjänsten så att du kan fjärrstarta din DiskStation.
