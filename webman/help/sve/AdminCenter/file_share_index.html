Indexfiler

Att indexera en delad mapp eller dess undermappar kan hjälpa dig att snabbt hitta filer och innehåll under filsökning.

Obs:

Fjärrmappar, molnmappar eller .iso-mappar som monterats på din DiskStation kan inte indexeras.

Att indexera mappar och filinnehåll:

Om du har mappar indexerade blir dessa mappar och de filer de innehåller automatiskt indexerade efter filnamn. Du kan vidare indexera filers innehåll/metadata inom de indexerade mapparna efter filtyp. Indexering av filinnehåll erbjuder snabbare och mer djupgående sökning i den indexerade filernas innehåll/metadata.

  1. Gå till Kontrollpanel > Indexeringstjänst > Filindexering och klicka på Lista över indexerade mappar.
  2. I popup-fönstret, klicka på Skapa > Välj för att visa mappar som är klara att indexeras.
  3. Välj de önskade mapparna och klicka på Välj. För att skapa en ny mapp för indexering, klicka på Skapa mapp och fortsätt.
  4. Klicka på den mapp du vill indexera och klicka på Välj.
  5. För att indexera specifika typer av filinnehåll, gå till Filtyp och välj Bild, Ljud, Video eller Dokument för indexering.
  6. Klicka på OK för att slutföra indexeringen.

Obs:

  • Maximalt antal samtidiga indexerade mappar är 1000.
  • Om du väljer Aktivera sökning av filinnehåll i File Station kan du söka bland filinnehåll i de indexerade mapparna.
  • Volymen som är värd för den indexerade mappen måste ha mer än 10 MB tillgängligt utrymme. I annat fall pausas indexeringen. För att återuppta indexeringen, utöka volymen och klicka på Skapa nytt index.
  • Om du har indexerat en mapp och dess undermapp(ar) på samma gång indexerar systemet endast undermappar(na) för att hushålla med systemresurser.
  • När mappar är fullständigt indexerade och sedan monterade på en Mac-dator med protokollen SMB eller AFP kan de som använder Mac använda Spotlight i Finder för att söka efter innehåll i filerna.

Att radera indexerade mappar:

  1. Gå till Kontrollpanel > Indexeringstjänst > Filindexering > Lista över indexerade mappar och välj önskad indexerad mapp(ar).
  2. Klicka på Radera för att sluta indexera mapp(ar).

Skapa ett nytt filregister

Under normala omständigheter behöver du bara indexera en mapp en gång. Om en fil i en mapp matchar specifika sökkriteria men ändå inte hittas vid en filsökning kan du dock behöva skapa nytt index för filerna för att återskapa indexstrukturen.

Att skapa du ett nytt index i en delad mapp:

  1. Gå till Kontrollpanel > Indexeringstjänst > Filindexering > Lista över indexerade mappar.
  2. Välj listad mapp(ar) att skapa nytt index för.
  3. Klicka på Skapa nytt index.

Obs:

File Station stöder inte indexering av filer på fjärrmappar eller externa lagringsenheter (t.ex. USB-diskar/SD-kort).

Hantera filindexering via skrivbordet

När filindexering börjar ser du Indexeringsförlopp (en ikon med ett flytande förstoringsglas) på aktivitetsfältet. Eftersom filindexering kräver mer CPU-resurser låter detta dig hushålla med CPU-resurser.

  • För att pausa/fortsätta filindexering, klicka på knappen för uppspelning.
  • För att fördröja filindexering till en lämplig tid, välj det önskade alternativet från rullgardinsmenyn.
  • För att visa vilka mappar som har indexerats, klicka på Visa lista över indexerade mappar.

Obs:

  • Om du pausar åtgärden för indexering kan den bara återupptas genom att klicka på knappen för uppspelning på skrivbordets aktivitetsfält.
