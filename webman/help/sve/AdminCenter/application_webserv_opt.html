HTTP-tjänst

Utöver standardportnumret 80, kan du lägga till en annan port för användning med Photo Station och Web Station.

Lägga till ytterligare ett portnummer för HTTP-åtkomst:

  1. Markera Lägg till en ytterligare HTTP-port.
  2. Ange portnumret i fältet. Du kan lägga till ytterligare ett portnummer från 1 till 65535 för HTTP-åtkomst.

Följande portar bör undvikas:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule-standardportar: 4662 (TCP), 4672(UDP), och FTP-standardportintervall: den faktiska räckvidden kan variera beroende på olika modeller.

    Obs:
  1. Portnumren för HTTP-åtkomst kan inte vara samma som FTP/BT-tjänstens portnummer.
  2. Portar för HTTP och HTTPS får inte stå i konflikt med varandra.
