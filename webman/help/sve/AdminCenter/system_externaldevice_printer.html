Skrivare

DiskStation kan bli en skrivarserver, vilket tillåter klientdatorer eller mobila enheter att få åtkomst till skrivare som är ansluta till DiskStation. Dessutom har Apple Wireless Printing och Google Cloud Print stöd för att tillåta dig att göra fjärrutskrift genom att använda iOS-enheter eller Googleprodukter och -tjänster.

Besök www.synology.com för kompatibla skrivare.

Installera skrivare

Se instruktionerna här nedanför för att installera och ansluta en USB- eller nätverksskrivare till din DiskStation.

Installera en USB-skrivare:

  1. Slå på och anslut skrivaren till en USB-port på DiskStation.
  2. Gå till Kontrollpanel > Externa enheter > Skrivare.
  3. Välj den skrivare du vill installera. Vänligen notera att DiskStation inte kan känna igen skrivare som anslutits via USB-hubbar.
  4. Klicka på Skrivarhanterare och välj Installera skrivare.
  5. I dialogrutan, välj ett av följande lägen:
    • Nätverk MFP: Tillåter klienter i det lokala nätverket att använda funktionerna utskrift, skanna och fax från din multifunktionsskrivare. Funktionerna skanna och faxa är endast tillgängliga för Windows-datorer.
    • Nätverksskrivare: Välj detta läge för skrivare som endast har stöd för utskrift om du vill klienterna endast ska kunna använda funktionen för utskrift.
  6. Konfigurera inställningar för Apple Wireless eller Google Cloud Print (valfritt).
  7. Klicka på Spara.

Obs:

Endast en klient i taget kan ha åtkomst till en nätverks-MFP.

Installera en nätverksskrivare:

  1. Slå på och anslut nätverksskrivaren till samma lokala nätverk som din DiskStation.
  2. Gå till Kontrollpanel > Externa enheter > Skrivare.
  3. Klicka på Lägg till nätverksskrivare och följ instruktionerna i installationsguiden.

Obs:

  • Skrivarnamn: Ett skrivarnamn måste anges vid installation och kan innehålla upp till 32 tecken, inklusive bokstäver, siffror och understreck.
  • Protokoll: I allmänhet använder HP:s nätverksskrivare Socket; Canon:s nätverksskrivare använder BJNP; de övriga använder överlag LPR.
  • Könamn: Att ange ett könamn krävs för nätverksskrivare som använder LPR och IPP men är inte applicerbart för skrivare som använder Socket eller BJNP.
  • Inte Nätverk MFP: Nätverksskrivare anslutna till DiskStation kan för närvarande inte använda läget nätverk MFP.
  • Inte USB-skrivare: USB-skrivare som delas med en Synology-produkt kan inte ställas in som nätverksskrivare.

Hantera skrivare

Fliken skrivare innehåller följande alternativ för att hantera USB- eller nätverksskrivare anslutna till DiskStation.

  • Hämta information: Visar information om den valda skrivaren såsom enhetsnamn, tillverkare, status, etc.
  • Ta bort: Tar bort och kopplar från en skrivare från DiskStation.
  • Skrivarhanterare: Innehåller följande alternativ:
    • Installera skrivare: Redigera printerinställningar såsom Apple Wireless Printing eller Google Cloud Print (se här nedanför för instruktioner).
    • Rengöra skrivarens rullar: Avbryter alla utskrifter på alla skrivare.
    • Skriv ut testsida: Skriver ut en testsida för att säkerställa att alla inställningar är korrekta.
    • Frigör skrivare: Kopplar från skrivare och klient (endast för läget nätverk MFP). När en klient ansluter till en multifunktionsskrivare för att ta emot fax eller skanna filer kan denna uppgift ta lång tid. Om nödvändigt kan anslutningen avbrytas för att tillåta nästa klient i kö att få åtkomst till skrivaren.

Att aktivera Apple Wireless Printing:

  1. Gå till Kontrollpanel > Externa enheter > Skrivare.
  2. Välj skrivare från enhetslistan.
  3. Klicka på Skrivarhanterare > Installera skrivare.
  4. Markera kryssrutan Aktivera Apple Wireless Printing.
  5. Välj din skrivares märke och drivrutin.
  6. Klicka på Spara.

När installationen är klar kan du skriva ut en testsida genom att klicka på Skrivarhanterare > Skriv ut testsida.

Obs:

  • Aktivering av Apple Wireless Printing aktiverar också Bonjour Skrivarsändning för DSM.
  • Apple Wireless Printing finns tillgängligt för iOS 4.2 eller senare.

Aktivera Google Cloud Print:

  1. Gå till Kontrollpanel > Externa enheter > Skrivare.
  2. Välj skrivare från enhetslistan.
  3. Klicka på Skrivarhanterare > Installera skrivare.
  4. Markera rutan Aktivera Google Cloud Print.
  5. Ange följande detaljer:
    • Information om ditt Google-konto.
    • Din skrivares märke och drivrutin.
  6. Klicka på Spara.

När installationen är klar kan du skriva ut en testsida genom att klicka på Skrivarhanterare > Skriv ut testsida.

För att hantera uppgifter för Google Cloud Print, klicka på länken Google Cloud Print.

Åtkomst till skrivare

Se avsnittet här nedanför för att få åtkomst till skrivare med PC eller Mac.

Åtkomst till skrivare med en Windowsdator:

Bästa sättet att få åtkomst till utskriftstjänsten i DiskStation-enheten är att upprätta anslutningen till skrivaren med Synology Assistant. Gå till dess Hjälp för detaljerade anvisningar.

Om Synology Assistant inte är installerat i datorn, se här nedanför, för andra sätt att få åtkomst till utskriftstjänsten i DiskStation-enheten.

Typ 1. Delad nätverksskrivare

  1. På din PC, öppna Nätverk i Filutforskaren och dubbelklicka på DiskStation.
  2. Dubbelklicka på usbprinter eller nätverksskrivarnamn; systemet uppmanar dig att installera den korrekta drivrutinen, klicka sedan på Ja.
  3. Vänligen erhåll skrivarens drivrutin från tillverkarens CD eller webbsida. Välj tillverkare och modell för skrivaren, eller klicka på Har disk för att välja drivrutinens plats. Klicka sedan på OK.
  4. Systemet startar nätverksskrivarens installation. När installationen är klar kan du börja använda skrivaren.

Typ 2. LPR-port (rekommenderas)

  1. På din PC, klicka på Start och gå till Enheter och skrivare.
  2. Klicka på Lägg till skrivare.
  3. Välj Lägg till en nätverksskrivare, trådlös skrivare eller Bluetooth-skrivare.
  4. Välj Skrivaren jag vill använda finns inte med i listan.
  5. Välj Lägg till en skrivare med TCP/IP-adress eller värdnamn. Klicka på Nästa för att fortsätta.
  6. Välj att Enheten ska vara TCP/IP-enhet.
  7. Ange IP-adress för DiskStation. Klicka på Nästa för att fortsätta.
  8. Välj Typ av enhet: Anpassa > Inställningar > …. Klicka på Nästa för att fortsätta.
  9. Välj LPR som protokoll; Ange usbprinter eller nätverksskrivarnamn som könamn,och markera kryssrutan Byte-räkning för LPR aktiverad. Klicka på OK för att fortsätta.
  10. Vänligen erhåll skrivarens drivrutin från tillverkarens CD eller webbsida. Välj tillverkare och modell för skrivaren, eller klicka på Har disk för att välja drivrutinens plats. Klicka sedan på OK.
  11. Systemet startar nätverksskrivarens installation. När installationen är klar kan du börja använda skrivaren.

Åtkomst till skrivare med en Mac:

Mac-användare kan få åtkomst till utskriftstjänsten på DiskStation-enheten genom att aktivera DSM:s Bonjour-skrivarsändning och lägga till skrivaren i Systeminställningar.

Se till att din DiskStation använder Postscript-skrivare innan du fortsätter.

Steg 1. Aktivera DSM:s Bonjour-skrivarsändning

  1. Logga in på DSM som admin eller som en användare tillhörande gruppen administrators.
  2. Gå till Kontrollpanel > Filtjänster > Win/Mac/NFS.
  3. Markera Aktivera Bonjour Skrivarsändning.

Steg 2. Åtkomst till skrivare med Mac

  1. På din Mac, gå till Systeminställningar > Skriv ut Skanna och klicka sedan på +.
  2. Välj usbprinter[nummer] eller nätverksskrivarnamn från rullgardinsmenyn. Din Mac konfigurerar skrivaren automatiskt.
