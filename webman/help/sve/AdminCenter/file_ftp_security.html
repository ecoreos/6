Säkerhetsinställningar

Ändra den markerade användarens rot till användarhem

Användare kommer åt sina hem-mappar endast när de loggar in på Synology DiskStation via FTP.

Aktivera anonym FTP

Aktivera denna funktion för att göra det möjligt för alla att logga in på DiskStations FTP-tjänst (med kontot anonym eller ftp).

Konfigurera privilegium för anonym användare:

Gå till Huvudmeny > Kontrollpanel > Delad mapp och klicka på Inställning av privilegier.

Ändra anonym rot

Aktivera denna funktion för att ändra anonyma FTP-användares inloggningskatalog. Välj den delade mappen från rullgardinsmenyn.
