Mac-filservice

I Kontrollpanel > Filtjänster > Win/Mac/NFS kan du aktivera Mac-filservice för att göra DiskStation till en Mac filserver (AFP-server).

Aktivera Mac-filservice:

  1. Markera Aktivera Mac-filservice för att göra DiskStation till en Mac-filserver (AFP-server).
  2. Ställ in en säkerhetskopieringsmapp för Time Machine om det behövs. (Se avsnittet nedan för mer information.)
  3. Klicka på Tillämpa.

Obs:

  • För att säkerhetskopiera data från Time Machine till DiskStation, välj en delad mapp från rullgardinsmenyn.
  • Delade mappar som används av andra tjänster kommer inte att vara tillgängliga för Time Machine.

Aktivera loggning av överföring

Att markera detta alternativ aktiverar loggning av filåtkomstågärder, inklusive skapande, radering, läsning, skrivning, flyttning och namnbyte för filer eller mappar via AFP även känt som Apple Filing Protocol.

Avancerade inställningar

Tillämpa standardrättigheter för UNIX:

Aktivering av detta alternativ applicerar standardrättigheterna för UNIX för att ladda upp eller skapa filer och mappar. Tillämpade behörigheter är de samma som de behörigheter som appliceras av UNIX-kommandot umask. När detta alternativ är aktiverat är behörigheterna för UNIX 644 för filer och 755 för mappar. När detta alternativ är avaktiverat är behörigheterna för UNIX 666 för filer och 777 för mappar. Standardvärdet för umask är 022.

Obs:

Aktivering av detta alternativ kan orsaka problem med inkonsekventa behörigheter mellan olika protokoll. För att undvika inkonsekvens rekommenderar vi att detta alternativ lämnas avaktiverat.

Frigör omedelbart resurser efter frånkoppling:

Aktivering av det här alternativet frigör omedelbart resurser när anslutningen till AFP-server oväntat kopplar från. Om du tar emot felmeddelandet ”Volymen används” när Time Machine används, vänligen aktivera det här alternativet.

Aktivera Bonjour Skrivarsändning:

Markera Aktivera Bonjour Skrivarsändning för att tillåta Mac-klienter att bläddra efter och hitta skrivare som är anslutna till DiskStation.

Att söka efter filers innehåll via Spotlight på Finder: via Spotlight i Finder:

När mappar är fullständigt indexerade och sedan monterade på din Mac-dator kan du använda Spotlight i Finder för att söka efter de filer den innehåller och dessa filers innehåll.

  1. Gå till Kontrollpanel > Indexeringstjänst > Filindexering > Lista över indexerade mappar > Skapa för att indexera specifika mappar och de filer de innehåller efter filtyp.
  2. Klicka på OK för att utföra indexering.
  3. Montera de indexerade mapparna på din Mac-dator.
  4. På din Mac-dator, klicka på den önskade monterade mappen i den vänstra panelen i Finder och använd Spotlight i Finder för att söka efter de filer den innehåller.

Obs:

  • För att söka i monterade mappar, använd Spotlight i Finder. Med Spotlight på skrivbordet i Mac är det endast möjligt att söka efter lokala filer.
  • På Mac OS X 10.9 och senare versioner har Spotlight i Finder stöd för att söka inom monterade mappar per taggnamn och kategori.
