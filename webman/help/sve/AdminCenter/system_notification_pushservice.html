Push-tjänst

Push-tjänst låter DiskStation skicka meddelanden till din mobila enheter eller webbläsare när specifika händelser eller speciellt fel inträffar.

Portar:

Innan aktivering av push-tjänst, vänligen tillse att DiskStation har åtkomst till Internet genom att använda port 8089.

För att aktivera e-postmeddelanden som tillhandahålls av Synology:

  1. Markera Skicka meddelande om systemstatus via Synology:s e-postserver.
  2. Ange en eller fler e-postadresser i fältet Mottagare. Systemmeddelanden kommer att skickas till de adresser som anges här. Två adresser tillåts. Separera flera adresser med ett semikolon (;).
  3. Klicka på Tillämpa, ett verifieringsmeddelande bör ha sänts till adresserna som specificerats i fältet Mottagare. Vänligen följ stegen i verifieringsmeddelandet.
  4. Efter verifiering kan du klicka på Skicka ett testmeddelande och sedan på OK för att skicka ett testmeddelande.

Att aktivera meddelanden för mobila enheter:

  1. Gå till Kontrollpanel > Meddelande > Push-tjänst.
  2. Markera rutan Aktivera mobilenhetsmeddelandetjänsten och klicka sedan på Tillämpa.
  3. Installera och starta DS finder på din mobila enhet och lägg sedan till din DiskStation till listan Favorit för att parkoppla din mobila enhet med DiskStation.
  4. Klicka på Skicka ett testmeddelande och sedan på OK för att skicka ett testmeddelande. Starta sedan DS finder på din mobila enhet och kontrollera om testmeddelandet har mottagits.

Att hantera parkopplade mobila enheter:

  1. Klicka på Hantera parkopplade enheter för att se en lista över alla parkopplade mobila enheter.
  2. Om du vill ta bort parkoppling från någon av enheterna, välj dem och klicka på Ta bort parkoppling. (Tryck och håll nere skift eller ctrl medan du gör flera val.) Inget meddelande kommer att skickas till dem såvida du inte parkopplar dem igen.

För att aktivera meddelanden i webbläsaren:

  1. Markera Aktivera mobilenhetsmeddelandetjänsten och klicka sedan på Tillämpa för att aktivera denna funktion.
  2. Klicka på Aktivera meddelanden för webbläsare för att parkoppla din DiskStation med webbläsaren.
  3. Gör något av följande i fönstret som visas:
    • För Chrome: Klicka på Installera nu för att installera tillägget till Chrome och logga in på ditt konto i Google, om du inte redan har gjort det.
    • För Safari: Klicka på Tillåt för att tillåta meddelandetjänst.
  4. När du blir omdirigerad tillbaka till webbläsarens meddelandesida, klicka på fliken Parkopplad enhet för att visa information för parkopplade enheter eller för att ta bort parkopplingen för din DiskStation från webbläsaren eller klicka på fliken Händelse för att visa din historik över meddelanden för webbläsaren, för de senaste sju dagarna.

Obs:

Meddelanden för webbläsaren stöds på Chrome (35 och senare) och Safari (OS X v10.9 eller senare).
