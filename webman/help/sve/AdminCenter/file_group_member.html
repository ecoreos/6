Lägga till/ta bort gruppmedlemmar

Du kan lägga till DiskStation-användare till den grupp som du har skapat eller ta bort dem från gruppen.

Lägg till medlemmar till en grupp:

  1. Välj den grupp du vill lägga till användare i och klicka på Redigera medlemmar.
  2. Klicka på Lägg till.
  3. Välj användare från medlemslistan och klicka på OK. (För att göra flera val, tryck och håll ned "Skift"- eller "Ctrl"-tangenten och klicka på önskade användare.)
  4. Klicka på Slutför för att använda inställningarna.

    5. Obs: Medlemmar i gruppen Administrators har administratörsrättigheter på DiskStation, inklusive programmen DSM/CIFS/FTP/AFP/WebDAV.

Ta bort medlemmar från en grupp:

  1. Välj den grupp du vill ta bort användare från och klicka på Redigera medlemmar.
  2. Välj användare från medlemslistan.
  3. Klicka på Ja.
  4. Klicka på Slutför för att använda inställningarna.

    5. Obs: Användare kan inte administrera DiskStation om de är raderade från gruppen Administrators.
