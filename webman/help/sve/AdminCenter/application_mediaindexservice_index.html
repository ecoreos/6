Indexerad mapp

Som standard är mediafiler i de delade mapparna photo, video, och music indexerade av Mediaindexeringstjänsten. Om du vill att DMA-enheter och DSM:s foto-, video- eller musikapplikationer ska ha tillgång till fler filer på din DiskStation kan du lägga till fler indexerade mappar här.

Så här gör du för att lägga till en indexerad mapp:

  1. Klicka på Skapa.
  2. I det fönster som visas, ge mappen ett namn som kommer att hittas av DMA-enheter och DSM-applikationer i fältet Namn.
  3. Klicka på Välj, bläddra fram en mapp som du vill indexera och klicka sedan på OK.
  4. I avsnittet Filtyp markerar du kryssrutorna för att specificera vilka typer av mediafiler som ska indexeras. DMA-enheter och DSM-applikationer kan endast söka och spela upp mediafiler som matchar de angivna filtyperna.
  5. Klicka på OK och sedan på Spara.
    Obs!
  • Fjärrmappar som monterats med hjälp av File Station kan inte läggas till som indexerade mappar.
  • Du kan lägga till upp till 100 användare på din DiskStation.

Så här gör du för att redigera en indexerad mapp:

  1. Markera mappen i mapplistan och klicka sedan på Redigera.
  2. Ändra inställningarna. (se Så här gör du för att lägga till en indexerad mapp för detaljerad information.)
  3. Klicka på OK och sedan på Spara.

Så här gör du för att ta bort mappar från listan över indexerade mappar:

  1. Välj en mapp eller flera mappar i mapplistan och klicka sedan på Radera. (Tryck och håll nere skift eller ctrl om vill markera flera val.)
  2. Klicka på Ja och klicka sedan på Spara.
    Obs!
  • Att radera mappar tar egentligen inte bort dem från din DiskStation. Det förhindrar enbart DMA-enheter och DSM-applikationer från att söka och spela upp filernas innehåll.
