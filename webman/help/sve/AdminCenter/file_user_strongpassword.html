Lösenordsinställningar

Klicka på knappen Lösenordsinställningar för att titta på och ändra de inställningar som beskrivs här nedanför.

Återställa glömt lösenord

Detta avsnitt innehåller ett alternativ med texten Tillåt användare som inte är admin att återställa glömda lösenord via e-post. När detta alternativ är aktiverat finns en länk, med texten Glömt lösenordet? på inloggningssidan till DSM. Om en användare glömmer sitt lösenord, kan denne klicka på denna länk och ange sitt användarnamn. I så fall kommer systemet att skicka ett meddelande till användaren, med en länk till återställning av det glömda lösenordet.

Obs:

  • Innan aktivering av detta alternativ, vänlig tillse att du redan aktiverat SMTP för e-postmeddelanden. För att göra detta, gå till Kontrollpanel > Meddelande > Allmänt. Om meddelanden inte är aktiverade kan inte systemet skicka meddelanden till användare som önskar återställa sitt lösenord.
  • För att erhålla meddelande från systemet, måste användare ha angett en e-postadress i avsnittet för användarinformation i sitt konto.
  • Användare som hör till gruppen administrators kan inte återställa lösenord med detta alternativ.
  • Domänanvändare kan inte återställa lösenord med detta alternativ.
  • LDAP-användare kan återställa lösenord med detta alternativ, så länge som DiskStation agerar server för LDAP-katalogen.

Lösenordsstyrka

Du kan ställa in regler för lösenordsstyrka för användare av din DiskStation.

Obs:

Regler för lösenordsstyrka gäller endast nya lösenord. Till exempel, nya regler för lösenordsstyrka appliceras bara när när ett nytt konto skapas eller när en befintligt användare byter sitt lösenord. Befintliga lösenord och de som hör till importerade användarkontot exkluderas från de nya lösenordsreglerna.

Aktivera regler för lösenordsstyrka:

  1. Markera kryssrutan Tillämpa regler för lösenordsstyrka för att aktivera lösenordsskydd.
  2. Välj fler än ett av dessa lösenordsskydd, från reglerna här nedanför:
    • Uteslut namn och beskrivning av användaren från lösenordet: Lösenordet får inte innehålla användarens namn eller användarens beskrivning. UTF-8-kodade tecken är uteslutna.
    • Tillåt både gemener och versaler: Både gemener och versaler är tillåtna i lösenord.
    • Inkludera numeriskt tecken: Lösenordet måste innehålla minst ett numeriskt tecken (0~9).
    • Inkludera specialtecken: Lösenordet måste innehålla minst ett ASCII specialtecken (dvs. ~, `, !, @, #, $, %, ^, &, *, (, ), -, _, =, +, [, {, ], }, \, |, ;, :, ', ", <, >, /, ?).
    • Minsta längd på lösenord: Lösenordet måste vara minst det värde som specificeras i textfältet. Lösenordets längd ska vara mellan 6 och 127 tecken.
  3. Klicka på Tillämpa för att spara inställningarna.
