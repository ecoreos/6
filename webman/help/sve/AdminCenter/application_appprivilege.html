Privilegier

Privilegier för användares och gruppers enskilda tjänster och applikationer kan visas och redigeras. DSM tillhandahåller flexibla inställningar för att kontrollera applikationsprivilegier, vilket gör det enkelt att hantera användares och gruppers åtkomst till olika applikationer.

För att ändra applikationsprivilegier för användare och grupper:

  1. Gå till Kontrollpanel > Privilegier.
  2. Välj den applikation som du vill ändra. Klicka på Redigera.
  3. Klicka på Användare för att visa individuella privilegier eller Grupp för att visa grupprivilegier.
  4. För varje användare eller grupp, markera Tillåt för att tillåta åtkomst eller Neka för att neka åtkomst till den valda applikationen. Beroende på applikationen kan du också välja Efter IP för att tillåta/neka åtkomst baserat på IP-adress.
  5. Klicka på OK för att spara dina ändringar.

För att visa applikationsprivilegier för användare som har olika nätverks-IP-adresser:

  1. Gå till Kontrollpanel > Privilegier.
  2. Klicka på Behörighetsvisare.
  3. Välj en användare från Användarnamn och ange sedan den IP-adress som du vill visa behörigheterna för i IP-adress.

För att ställa in standard för applikationsprivilegier för alla användare:

  1. Gå till Kontrollpanel > Privilegier.
  2. Välj den applikation som du vill ändra. Klicka på Redigera.
  3. Klicka på Standardprivilegier och markera Tilldela det här privilegiet till alla.
  4. Klicka på OK för att spara dina ändringar.

Obs:

  • Privilegier för admin och guest kan inte ändras här.
  • När du stöter på privilegiekonflikter är privilegieprioriteten enligt följande: Neka > Tillåt.
  • Alla filpaket och tjänster stöder inte åtkomsträttigheter Efter IP.
