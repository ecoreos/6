Allmänt

Kontrollpanel > Hårdvara & Ström > Allmänt tillhandahåller alternativ för att aktivera power recovery, Wake on LAN (WOL) och minneskompression. Du kan också ändra ljudkontroll och läge för fläkthastighet här.

Strömåterställning

Om systemet stängs av på grund av strömförlust, gör strömåterställningsfunktionen att systemet kan starta upp automatiskt då strömmen återfås.

Vakna med LAN*

Aktivering av Wake on LAN (WOL) låter dig starta din DiskStation över det lokala nätverket eller Internet genom att använda IP-adressen (eller DDNS-värdnamn) och MAC-adressen för din DiskStation genom att använda en WOL-applikation, såsom Synology Assistant DS finder eller valfri annan WOL-applikation.

Vakna med LAN fungerar endast efter att DiskStation stängts av ordentligt, genom att den fysiska Strömknappen tryckts in eller klickning på Stäng av i gränssnittet för DSM. Ha strömsladden inkopplad i ett fungerande strömuttag efter avstängning.

Obs:

Funktionen WOL har inte stöd på 10GbE LAN-portar (om sådana finns på din DiskStation).

  • Funktionen WOL har inte stöd på 10GbE LAN-portar (om sådana finns på din DiskStation).
  • Funktionen WOL är som standard aktiverad på 1GbE LAN-portar för DiskStation och kan inte avaktiveras.

Fläkthastighetsläge*

Följande alternativ för fläkthastighetsläge finns:

  • Kylningsläge: Fläkten arbetar med hastighet, vilket kyler ner DiskStation mer men låter mer.
  • Tyst läge: Fläkten arbetar med låg hastighet, låter mindre men DiskStation kan komma att bli varmare.
  • Lågeffekt: Fläkten arbetar med låg hastighet, låter mindre men DiskStation kan komma att bli varmare. Dessutom kommer fläkten att stanna när DiskStation är kyld, för att spara ström. Lågeffektläge är bara tillgängligt på vissa modeller.

Ljudkontroll

DiskStation kan avge ett pipljud när speciella händelser eller fel inträffar. Klicka på kryssrutorna för listade händelser för att låta DiskStation göra dig uppmärksam när fel inträffar.

För att stoppa pipljudet:

  • Om DiskStation avger pipljud, klicka på knappen Pip av för att inaktivera det.

Minneskomprimering*

Du kan aktivera Minneskomprimering för att förbättra den allmänna svarbarheten vid last genom att komprimera de senaste använda data i minnet.

När du har flera applikationer igång och minnet börjar bli fullt krymper den dynamiska minneskomprimeringen storlek för sällan använda poster i minnet, vilket minskar minnesanvändning. När posterna behövs igen kan de omedelbart dekomprimeras. Detta minskar behovet av att läsa och skriva minnesswapfiler på disk, vilket förbättrar effektivitet och svarstid.

För att aktivera Minneskompression, markera helt enkelt kryssrutan.

LED*

I Kontrollpanel > Hårdvara & Ström > Allmänt > Kontroll av ljudstyrka på LED kan du justera ljusstyrkan för LED-indikatorerna på din DiskStation.

Obs:

Inställningarna för ljusstyrka tillämpas på alla LED-indikatorer på din DiskStation (dvs. ström, STATUS, LAN, diskstatus, osv.).

Att justera ljusstyrkan för LED:

Det finns fyra tillgängliga nivåer för ljusstyrka för LED-indikatorerna på din DiskStation.

  1. Klicka och dra pekaren på skalan för ljusstyrka i valfri riktning:
    • Höger: Ökar ljusstyrkan för LED.
    • Vänster: Minskar ljusstyrkan för LED.
  2. Klicka på Tillämpa för att spara inställningen.

Att justera ljusstyrkan för LED per schema:

LED-indikatorerna kan ändra sin nivå på ljusstyrka på den schemalagda tiden.

  1. Klicka på Lägg till schema.
  2. Klicka på valfri nedanstående typ av ljusstyrka:
    • Standardljusstyrka: Den högsta ljusstyrkan
    • Justerad ljusstyrka: Den nivå på ljusstyrka som specificerats på skalan för ljusstryka
  3. Klicka på de önskade punkterna för att anpassa schemat för ljusstyrka.
  4. Klicka på OK och Tillämpa för att spara inställningarna.
