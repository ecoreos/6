Skapa en Grupp

  1. Gå till Kontrollpanel > Grupp.
  2. Klicka på Skapa. Denna åtgärd startar Guiden Skapa Användare.
  3. Ange gruppnamnet och beskrivningen. (Se avsnittet här nedanför för mer information). Klicka på Nästa.
  4. Följ de återstående stegen i guiden för att slutföra skapandet av gruppen.
  5. Klicka på Tillämpa för att slutföra.

Namnkonventioner:

  • Gruppnamnet är skiftlägeskänsligt och bör innehålla mellan 1 och 32 Unicode-tecken, med undantag från följande symboler:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Det första tecknet får inte vara ett minustecken eller ett mellanslag och sista tecknet får inte vara ett mellanslag.
  • Gruppbeskrivningen är skiftlägeskänslig och kan innehålla upp till 64 Unicode-tecken.
