Mediabibliotek

Mediabiblioteket kan automatiskt skanna flera multimediafiler samtidigt såsom foton, musik och videor som finns lagrade på din DiskStation och kompilera dem till ett multimediabiblioteket för att användas av multimediaapplikationer.

Med Mediabiblioteket kan du också ställa in miniatyrbildernas kvalitet för bilder och videor, hantera konverteringsinställningar för videor och skapa indexerade mappar som kan användas av DMA-enheter och multimediaapplikationer i DSM.
