Delad mapp

Delade mappar är de grundläggande mapparna där du kan lagra filer och mappar på din DiskStation. Innan du lagrar några filer på din DiskStation måste du skapa minst en delad mapp. Data som lagras i delade mappar kan hållas privata eller delas med specifika användare eller grupper beroende på de anpassningsbara åtkomsträttigheter. Dessutom kan delade mappar krypteras när de skapas vilket tillför ett extra lager av skydd.

Delade mappar som är standard

Här nedanför finns en listan över delade mappar som skapas automatiskt när särskilda applikationer, tjänster eller filpaket installeras eller aktiveras.

Namn Beskrivning
public Mappen public skapas automatiskt när din DiskStation initialt installeras.*
web Mappen web är den lagringsplats som är standard för innehållet hos en webbplats som DiskStation är värd åt. Den skapas automatiskt när funktionen Web Station aktiveras.
photo Mappen photo används för att lagra bilder och videor som du vill dela med Photo Station. Den skapas automatiskt när Photo Station eller DLNA/UPnP Media Server aktiveras.
music Mappen music används för att lagra musik du vill lyssna på med Audio Station. Den skapas automatiskt när Audio Station aktiveras.
video Mappen video används för att lagra de videor som du vill bläddra bland med DLNA/UPnP-enheter. Den skapas automatiskt när Media Server aktiveras.
surveillance Mappen surveillance används för att lagra inspelningar från Surveillance Station. Den skapas automatiskt när Surveillance Station aktiveras. Den är skrivskyddad och kan endast kommas åt av systemadministratören.
home Mappen home ger ett privat utrymme för varje användare att lagra data på en plats som endast användaren kommer åt. Den skapas automatiskt när funktionen Användarhem aktiveras.
homes Mappen homes innehåller alla användares home-mappar. Den skapas automatiskt när funktionen Användarhem aktiveras. Endast systemadministratören kan komma åt och hantera alla användares home-mappar.
NetBackup Mappen NetBackupskapas automatiskt när Nätverkssäkerhetskopiering aktiveras.
usbshare Mappen usbshare[nummer] skapas automatiskt när du ansluter en extern USB-diskenhet till USB-porten på DiskStation.*
satashare Mappen satashare skapas automatiskt när du ansluter en extern eSATA-diskenhet till eSATA-porten på DiskStation.*

____
* Endast tillgängligt för vissa modeller.
