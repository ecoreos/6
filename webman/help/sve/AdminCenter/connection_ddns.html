DDNS

DDNS (Dynamic Domain Name Service) förenklar anslutning av DiskStation över Internet genom att mappa ett värdnamn till dess IP-adress. Till exempel tillåter DDNS dig att få åtkomst till din DiskStation genom att använda ett domännamn (t.ex. www.john.synology.me), utan att behöva komma ihåg en IP-adress (t.ex. 172.16.254.1).

Installera DDNS

Du kan peka ut ett befintligt värdnamn vid IP-adressen för din DiskStation, eller registrera ett nytt som Synology eller flertalet andra DDNS-leverantörer erhåller. Vänligen rådfråga de enskilda leverantörerna för detaljer om registrering av värdnamn.

Sätta upp ett DDNS-värdnamn:

  1. Klicka på knappen Lägg till.
  2. En dialogruta visas och uppmanar dig att redigera följande inställningar:
    • Serviceleverantör: Välj en serviceleverantör. För att registrera ett gratis värdnamn från Synology, välj Synology från rullgardinsmenyn, klicka på Registrera nu och ange information när du uppmanas att göra så.
    • Värdnamn: Ange ett registrerat DDNS-värdnamn, exempelvis john.synology.me.
    • Användarnamn: Ange användarnamnet för din DDNS-leverantör.
    • Lösenord: Ange lösenordet för din DDNS-leverantör.
    • Extern adress: Ange den externa IP-adressen på den DiskStation som ska använda värdnamnet.
    • Heartbeat: Aktivera denna funktion för att få meddelanden om statusen på det mappade värdnamnet (endast för värdnamn som erhållits av Synology).
  3. Klicka på Testa anslutning för att se om inställningarna är korrekta.
  4. Klicka på OK för att spara och avsluta.

När installationen är klar får du åtkomst till din DiskStation över Internet genom att ange DDNS-värdnamnet i en webbläsare.

Krav och begränsningar

  • Servrarna för DDNS-tjänster fungerar.
  • DiskStation kan ansluta till Internet.
  • DDNS fungerar bara i miljöer som inte kräver proxy-server.
  • Du kan bara använda ett värdnamn per DDNS-leverantör.

Hantera värdnamn

Använd följande knappar för att hantera befintliga värdnamnsposter:

  • Radera: Tar bort den valda värdnamnsposten från listan.
  • Redigera: Ändra inställningar för den valda värdnamnsposten.
  • Uppdatera nu: Uppdaterar status för alla värdnamnsposter som har DDNS-tjänsten aktiverad.

Konfigurera en anpassad DDNS-leverantör

Du kan lägga till en anpassad DDNS-leverantör om din leverantör inte redan finns i listan över tjänsteleverantörer.

För att konfigurera en anpassad DDNS-leverantör

  1. Klicka på Anpassa.
  2. En dialogruta visas och uppmanar dig att ange följande inställningar:
    • Serviceleverantör: Ange namnet på din anpassade tjänsteleverantör.
    • Fråge-URL: Ange URL för din anpassade tjänsteleverantör.
  3. Klicka på Spara för att lägga till den anpassade tjänsteleverantören i listan över tjänsteleverantörer för DDNS.

Krav och begränsningar

  • Fråge-URL måste använda det protokoll som specificerats av din tjänsteleverantör. Den här URL:en används för att uppdatera IP-information till din leverantörs DDNS-server.
  • En fråge-URL innehåller vanligtvis de fyra nedanstående parametrarna. När fråge-URL anges, vänligen byt ut de följande parametrarna med motsvarande värde.
    • Användarnamn: __USERNAME__
    • Lösenord: __PASSWORD__
    • Värdnamn: __HOSTNAME__
    • Extern adress: __MYIP__
  • Fråge-URL kan inte innehålla mellanslag eller semikolon.
