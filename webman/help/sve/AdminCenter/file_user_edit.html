Redigera en användare

Hitta användaren (genom att ange användarnamnet i Sök-fältet), dubbelklicka på den eller klicka på Redigera, därefter kan du redigera dennes kontoinformation.

Obs:
  1. Om du avaktiverar ett konto kommer kontot inte längre att fungera.
  2. Kvot ska vara ett heltal större än eller lika med 0 och mindre än hårddiskens/volymens kapacitet. Ett värde på kvot som är 0 betyder att Effektiv kvot kommer att avgöras av Gruppbegränsning.
  3. Om det finns fler än en volym/delad mapp på servern kan du ställa in användarkvoter för varje volym/delad mapp.
  4. E-postinformation på sidan ”Meddelande” måste ställas in korrekt för att systemet ska kunna meddela användaren via e-post efter att kontoinformationen har skapats eller ändrats.
  5. Vid inställning av hastighetsbegränsning för en användare kommer du att se nummer under Resultat om det redan finns en grupphastighetsbegränsning som tillämpas för användaren.
  6. Hastighetsbegränsning som tilldelats en användare inkluderas inte i hastighetsbegränsning för den grupp som användaren tillhör.
