LCM-skärm*

Du kan ställa in lösenord för LCM-skärm för att förhindra obehöriga användare från att ändra programinställningar via LCM-skärm.

För att aktivera lösenordskydd av LCM-skärm:

  1. Bocka i Aktivera lösenordskydd av LCM-skärm.
  2. Ange Lösenord och skriv det igen för att Bekräfta lösenord.
  3. Klicka på Tillämpa.

____
* Endast tillgängligt för vissa modeller.
