Link Aggregation*

DiskStation har stöd för multi-LAN vilket innebär att du kan kombinera dessa LAN-gränssnitt med Link Aggregation-teknologi. Link Aggregation ökar bandbredden på din DiskStation genom att aggregera flera nätverksgränssnitt och tillhandahåller redundans för att upprätthålla nätverksanslutning om anslutningen är nere.

Efter att dina LAN-gränssnitt har kombinerats ser du ett nytt gränssnitt som heter Bond i Kontrollpanel > Nätverk > Nätverksgränssnitt, det kan konfigureras på samma sätt som andra gränssnitt (exempelvis LAN eller PPPoE).

Obs:

  • Med Link Aggregation aktiverat justeras och balanseras nätverkstrafiken automatiskt för varje ansluten enhet.
  • Om nätverksgränssnitten är anslutna till flera switchar, tillse att switchportarna är i samma LAN eller VLAN.

Kombinera flera olika LAN med Link Aggregation:

  1. Gå till Kontrollpanel > Nätverk > Nätverksgränssnitt. Klicka på Skapa > Skapa Bond.
  2. Välj det läge du vill ha.

    Beskrivning av lägen:

    • Adaptive Load Balancing: Det här läget optimerar den nätverkstrafik som tas emot och skickas av din DiskStation oavsett om switchen har stöd för Link Aggregation eller inte. För att undvika oväntade fel, vänligen aktivera inte Link Aggregation på switchen även när det finns stöd.
    • IEEE 802.3ad Dynamic Link Aggregation: Det här läget optimerar den nätverkstrafik som tas emot och skickas av din DiskStation och kräver att IEEE 802.3ad (Dynamisk) Link Aggregation (LACP 802.1AX) har aktiveras på switchen(switcharna). Om flera switchar används måste dessa switchar vara stackningsbara och konfigurerade på rätt sätt.
    • Balansera XOR: Det här läget balanserar nätverkstrafiken som tas emot och skickas av din DiskStation och kräver att Static Link Aggregation är aktiverad på switchen(switcharna). Om flera switchar används måste dessa switchar vara stackningsbara och konfigurerade på rätt sätt. Om Link Aggregation inte stöds av switchen kan trafiken som skickas av din DiskStation fortfarande balanseras. Med det här läget kan förlorade länkar upptäckas men inte det som är orsaken (dvs. när Ethernet-kablar är frånkopplade eller switchen felaktigt konfigurerad).
    • Aktiv/standby: Det här läget tillhandahåller feltolerans för nätverket till din DiskStation. Det betyder att när det aktiva nätverksgränssnittet går ner tar en av de andra LAN-portarna över för att upprätthålla nätverksanslutningen. Du kan välja det här läget utan att konfigurera switchen eller med en switch som har stöd för IEEE 802.3ad Link Aggregation.
  3. Välj gränssnitt för att skapa Link Aggregation.
  4. Konfigurera IP-inställningarna. Du kan även aktivera VLAN om det behövs. Klicka på Tillämpa för att slutföra installationen.
  5. När det är klart kan du se ett nytt gränssnitt med namnet Bond 1 i listan över Nätverksgränssnitt.

Att redigera inställningarna för Link Aggregation:

  1. För att ändra läge för Link Aggregation eller visa informationen för de fysiska enheterna, gå till Kontrollpanel > Nätverk > Nätverksgränssnitt.
  2. Välj gränssnittet Bond 1 och klicka på knappen Redigera.
  3. Gå till motsvarande flikar för att redigera de önskade inställningarna.

Att ta bort Link Aggregation:

  1. Gå till Kontrollpanel > Nätverk > Nätverksgränssnitt.
  2. Välj gränssnittet Bond 1 och klicka på knappen Radera.
  3. När det är klart ser du att Bond 1 nu är ersatts av två LAN-gränssnitt.

* Endast tillgängligt för vissa modeller.
