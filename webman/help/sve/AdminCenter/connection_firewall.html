Tillåt/neka

Den inbyggda brandväggen kan förhindra icke auktoriserade inloggningar och styra tjänstern åtkomst. Vidare kan du välja att tillåta eller neka åtkomsten till vissa nätverksportar från specifika IP-adresser.

Skapa en brandväggsregel:

  1. Gå till Kontrollpanel > Brandvägg och välj nätverksgränssnittet från den vänstra kolumnen beroende på typen av nätverksanslutning.
  2. Klicka på Skapa.
  3. I avsnittet Portar väljer du någon av följande förloppsordningar:
    • Välj Alla för att tillämpa reglerna på alla portar.
    • Välj Välj från en lista över inbyggda applikationer.
    • Välj Anpassa och klicka på Anpassa, och ange därefter 15 portar separerade med komma, eller ange ett portintervall.
    • Du kan även specificera protokollen.
  4. I avsnittet Käll-IP väljer du någon av följande förloppsordningar:
    • Välj Alla för att tillåta eller neka alla IP-adresser.
    • Ange en Enskild värd IP-adress.
    • Ange ett Subnät.
  5. I avsnittet Åtgärd väljer du någon av följande förloppsordningar:
    • Välj Tillåt för att tillåta åtkomst möta portar och käll-IP inställningarna ovan.
    • Välj Neka för att neka åtkomst möta portar och käll-IP inställningarna ovan.

Nu kan du se den skapade i brandväggsregeln i regellistan.

Gör något av följande för att ta bort reglerna för brandväggen:

  • Avmarkera kryssrutan för regeln i Aktivera kolumnen för listan
  • Ta bort regeln för brandväggen

Du kan också välja Tillåt åtkomst eller Neka åtkomst längst ner på regellistan för att tillåta eller neka åtkomsten som inte matchar några regler.

    Obs:
  1. Du kan skapa upp till 100 regler för ett nätverksgränssnitt och du kan omarrangera reglerna i listan med hjälp av dra och släpp-funktionen.
  2. Reglerna prioriteras enligt deras plats i listan.
  3. Om ditt system har flera nätverksportar anslutna till samma subnät kanske brandväggsreglerna inte fungerar korrekt.*
  4. När du kombinerar flera LAN-portar med link aggregation tillämpar brandväggen reglerna från det första nätverket och sparar reglerna för det andra nätverket.

____
* Endast tillgängligt för vissa modeller.
