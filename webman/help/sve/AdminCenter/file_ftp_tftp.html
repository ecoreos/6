TFTP

Trivial File Transfer Protocol (TFTP, mindre avancerat filöverföringsprotokoll) är ett enkelt filöverföringsprotokoll som allmänt används för att överföra konfigurationer eller bootfiler, i situationer där autentisering inte krävs. Aktivering av TFTP-tjänster gör att DiskStation kan överföra bootloaders till klientenheter.

Att konfigurera TFTP-inställningar:

  1. Gå till Kontrollpanel > Filtjänster > TFTP och välj Aktivera TFTP-tjänst.
  2. I fältet TFTP rootmapp, specificera till vilken mapp på DiskStation TFTP-klienter har åtkomst.
  3. Klicka på Avancerade inställningar för att konfigurera följande inställningar:
    • Aktivera loggfil för TFTP-filöverföring: Håll reda på data som överförs med TFTP.
    • TFTP-klientbehörighet: Avgör TFTP-klientbehörigheter:
      • Endast läsbar: TFTP-klienter kan endast läsa innehållet i rootmappen.
      • Skrivbar: TFTP-klienter kan ändra innehållet i rootmappen.
    • Tillåtna klienter: Välj vilka klienter som är tillåtna att ansluta via TFTP:
      • Alla anslutningar: Alla klienter kan ansluta via TFTP.
      • Tillåt bara följande IP-adressintervall: Endast klienter inom det specificerade IP-intervallet kan ansluta via TFTP.
    • Timeout: Om TFTP-servern (dvs. din DiskStation) inte svarar inom det specificerade tidsintervallet stoppas TFTP-anslutningen.
  4. Klicka på Tillämpa för att spara inställningarna.
