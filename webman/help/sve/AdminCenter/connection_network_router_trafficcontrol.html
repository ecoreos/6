Trafikkontroll för router

Trafikkontroll för router är aktiverad när din DiskStation är i läget router. Trafikkontroll för router (i Kontrollpanel > Nätverk > Trafikkontroll för router) hanterar trafiken för den uppladdning och nedladdning som körs på det privata nätverket för din DiskStation. När regler skapas behöver du specificera Aktuell bandbredd och Maximal bandbredd för att begränsa trafiken för uppladdning och nedladdning.

Förklaring av termer

  • Aktuell bandbredd: Bestämmer den utgående trafiken som en tjänst garanterar att erbjuda när hela systemets bandbredd är tillräcklig.
  • Maximal bandbredd: Bestämmer den utgående trafiken som denna tjänst kan använda när hela systemets bandbredd är tillräcklig och det finns Systemets återstående bandbredd. Vi föreslår att Systemets utgående bandbredd beräknas först. Tillse sedan att summan av Aktuell bandbredd för varje tjänst inte är större än Systemets utgående bandbredd, i annat fall kan det hända att inställningen inte fungerar korrekt.

Inställningsformel

  • Systemets utgående bandbredd = Total Aktuell bandbredd för alla tjänster + Systemets återstående bandbredd
  • Aktuell bandbredd för varje tjänst ≤ Maximal bandbredd för varje tjänst

Obs:

Om de utgående paketen inte motsvarar någon regel för trafikkontroll för router är standardbeteendet att det inte är någon begränsning. Du kan lägga till dina egna standardregler för trafikkontroll för router i slutet av alla regler genom att välja Alla portar, och specificera Aktuell bandbredd och Maximal bandbredd.

Skapa en regel för trafikkontroll för router:

  1. Välj ett nätverkskort från rullgardinsmenyn i det övre högra hörnet.
  2. Klicka på Skapa.
  3. I avsnittet Portar välj en av följande:
    • Välj Alla för att tillämpa regeln på alla portar.
    • Välj Anpassa och klicka på Anpassa. Ange upp till 15 portar separerade med komman eller specificera ett portintervall att tillämpa denna regel på.
    • Du kan även specificera protokollet.
  4. I avsnittet Käll-IP välj en av följande:
    • Välj Alla för att tillämpa regeln på alla IP-adresser.
    • Välj Specifik IP och klicka på Välj. Du kan specificera en IP-adress eller ett IP-intervall att tillämpa denna regel på.

Nu kan du se den skapade regeln för trafikkontroll för router i regellistan.

Gör något av följande för att ta bort regler för trafikkontroll för router:

  • Avmarkera kryssrutan för regeln i kolumnen Aktiverad för listan.
  • Ta bort regel för trafikkontroll för router.

Obs:

  • Du kan skapa upp till 100 regler för vardera nätverkgränssnitt.
  • Dra och släpp för att ändra ordning på reglerna i listan.
  • Regler prioriteras enligt deras plats i listan.
  • Portarna som specificerats hör till de portnummer för Internettjänster utanför det lokala nätverket.
  • Källans IP:n hänvisar till de IP-adress för enheterna som använder Synology NAS som router.
