Användare

Med flexibla alternativ för hantering av användare kan du skapa användare för enskilda medlemmar av din familj eller ditt företag. Att skapa användare låter dig definiera och hantera behörigheter för varje person, såsom åtkomstbehörigheter för delade mappar, kvot för lagring/delade mappar eller begränsning av bandbredd.
