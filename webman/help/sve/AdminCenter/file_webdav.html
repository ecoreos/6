WebDAV

WebDAV är ett tillägg till HTTP som är utvecklat för att låta användare ändra och hantera filer som är lagrade på fjärrservrar. När tjänsten WebDAV är aktiverad kan klientprogram med stöd för WebDAV, såsom vissa Windows-applikationer, Mac OS Finder, Linux filbläddrare, få fjärråtkomst till DiskStation, precis som de skulle få till en lokal nätverkshårddisk.

Aktivera WebDAV:

  1. Markera rutan med texten Aktivera WebDAV eller Aktivera WebDAV HTTPS-anslutning om du önskar aktivera WebDAV-anslutningar via HTTPS. Standardport för WebDAV är 5005 och WebDAV HTTPS är 5006.
  2. Klicka på Tillämpa

Hastighetsbegränsning

Att aktivera hastighetsbegränsning låter dig kontrollera den maximala bandbredden för specifika användare eller grupper vid överföring av filer via WebDAV.

För att aktivera hastighetsgräns:

  1. Markera kryssrutan med texten Aktivera hastighetsbegränsning eller Aktivera hastighetsbegränsning i enlighet med ett schema om du vill implementera hastighetsbegränsning endast vid specifika tider. Detta är en inställning som får genomslagskraft över hela servern och kommer att åsidosätta inställningar för enskilda användare och grupper. Att välja Ingen hastighetsbegränsning avaktiverar inställningarna för hastighetsbegränsning, för alla användare och grupper.
  2. Klicka på Inställningar för hastighetsbegränsning för att ändra inställningarna för användare eller grupper. För varje användare eller grupp kan du välja en av de följande:
    • Tillämpa gruppinställningar: Inställningen för hastighetsbegränsning kommer att vara samma som för gruppen användaren tillhör.
    • Ingen: Ingen hastighetsbegränsning kommer att tillämpas när gruppmedlemmar överför filer med tjänsten.
    • Konstant: En fast, konstant hastighetsbegränsning kommer att tillämpas för användaren eller gruppmedlemmarna. Hastighetsbegränsning för uppladdning och nedladdning kan specificeras i fälten till vänster.
    • Variabel: Två olika hastighetsbegränsningar kan specificeras och tillämpas på användaren eller gruppmedlemmar, i enlighet med ett schema. Klicka på knappen Anpassa för att ändra inställningarna för hastighetsbegränsning och ställa in ett schema.
  3. Klicka på Tillämpa.

Obs:

  • För mer detaljer om dynamiken för hastighetsbegränsning för grupper, vänligen se här.
  • Vid inställning av hastighetsbegränsning för en användare kan avsnittet Resultat visa användarens faktiska hastighetsbegränsning, i enlighet med sina individuella eller sin grupps inställningar. Till exempel, Till exempel om användaren ”Achilles” har en individuell nedladdningsgräns på 100 KB/s men hör till en grupp med en nedladdningsgräns på 500 KB/s, kommer avsnittet Resultat att visa ”0/500”.

Avancerade inställningar

Att klicka på knappen Avancerade inställningar låter dig ändra följande alternativ:

  • Aktivera anonym WebDAV: Låter klienter få åtkomst till delade mappar genom att använda användarnamnet "anonymous" när de anslutningen med WebDAV. Se till att användaren "Anonym FTP/WebDAV" tilldelats riktiga åtkomsträttigheter på sidan Delad mapp.
  • Aktivera DavDepthInfinity: Vid användning av ett klientverktyg låter detta användare att få åtkomst till eller radera filer utan begränsningar av filsökväg.
  • Aktivera WebDAV-loggfil: Detta alternativ spelar in händelser som är relaterade till WebDAV. Klicka på Visa loggfiler och du får se loggfilerna i popup-fönstret.

Använda WebDAV med olika operativsystem

När WebDAV har aktiverats på din DiskStation, vänligen följ instruktionerna här nedanför, för att börja få åtkomst till filer med Windows-, Mac-, Linux-enheter eller vissa appar från Synology för smartphone.

Obs:

  • Vid anslutning med tjänster för WebDAV, föreslår vi användning av WebDAV-klient med stöd för UTF-8 för att undvika fel på grund av kodning.

För Windows

För Windows-användare som vill använda protokollet WebDAV för åtkomst till filer på Synology-produkter, föreslår vi BitKinex, en applikation för mappning av enheter som är gratis för icke kommersiell användning. Alternativt finns det många andra klientapplikationer för WebDAV tillgängliga på Internet.

Obs:

  • Om du vill använda Windows Mappad nätverkshårddisk för åtkomst till filer på din DiskStation via WebDAV, vänligen införskaffa ett giltigt SSL-certifikat för åtkomst till din DiskStation via en HTTPS-anslutning. Om du vill växla till ett annat användarkonto när du använder WebDAV-service i Windows-miljö måste du först logga ut ur Windows-systemet.

För Mac

Mac-användare kan ansluta till sin DiskStation med WebDAV, genom att använda Finder.

Använda WebDAV på Mac:

  1. På din Mac-dator, gå till Finder > Anslut till server.
  2. I fältet Serveradress, skriv IP-adressen till din DiskStation eller ditt domännamn med "http://" framför och ":5005" efter (eller något annan portnummer som du specificerat när du aktiverat tjänsten WebDAV). Till exempel: http://pm.synology.com:5005/.
  3. Ange ditt användarnamn och lösenord för DSM.
  4. Klicka på Anslut.

Obs:

  • WebDAV stöder Bonjour-broadcast.

För Linux

För dem som använder Ubuntu eller annan Linuxdistribution, vänlig följ instruktionerna här nedanför för åtkomst till din DiskStation via WebDAV.

Använda WebDAV på Ubuntu:

  1. Klicka på Platser > Anslut till server.
  2. Välj WebDAV (HTTP) eller WebDAV (HTTPS) från rullgardinsmenyn.
  3. Ange den information som krävs, t.ex. serverplats, port, mapp och användare. Klicka på Anslut för att fortsätta.
  4. Ange ditt lösenord och klicka på Anslut.

Använda WebDAV via kommandoraden i Linux:

Om du vill använda kommandoraden kan du använda cadaver, en WebDAV-klientprogramvara för kommandoraden, för att ställa in anslutningen.

Prenumerera på kalendrar med CalDAV-klienter

Du kan använda CalDAV-klienter på din dator eller mobila enhet för att prenumerera på kalendrar som finns lagrade på DiskStation och sedan hantera kalenderhändelser, anteckningar eller att göra-listor. Exempel på kalender-klienter inkluderar Sunbird (Windows, Mac, Linux), Chandler (Windows, Mac, Linux), Evolution (Linux) eller iCal (Mac, iOS-enhet).

Så här gör du för att aktivera CalDAV:

  1. Aktivera antingen WebDAV- eller WebDAV HTTPS-anslutning först.
  2. Markera Aktivera CalDAV.
  3. Klicka på Tillämpa.

Så här gör du för att hantera kalenderlistan:

  1. Klicka på Visa kalenderlista.
  2. Gör något av följande i fönstret som visas:
    • Om du vill skapa en ny kalender, klicka på Lägg till, ge din nya kalender ett namn (till exempel "cal1") och ange ett mål där du vill spara kalendern (till exempel den delade mappen "kalender") och klicka sedan på OK.
    • Om du vill ändra kalenderns namn och mål, välj kalendern i kalenderlistan, klicka på Redigera för att göra ändringar och klicka sedan på OK.
    • Om du vill radera kalendern från din DiskStation, välj den i kalenderlistan och klicka sedan på Ta bort.
  3. Klicka på Stäng.

Obs:

  • Kalendrar lagrade på din DiskStation kommer att visas som en mapp på din DiskStation.

Så här gör du för att prenumerera på din kalender med en klientapplikation (med Sunbird till exempel):

  1. Ladda ned och installera Sunbird på din dator.
  2. Gå till Arkiv > Prenumerera på fjärrkalender.
  3. I det fönster som visas, välj På nätverket och klicka på Nästa.
  4. Välj CalDAV, ange http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name/ i fältet Plats (till exempel "http://demo.no-ip.info:5005/calendar/cal1"), och klicka sedan på Nästa.
  5. Ge din kalender ett namn och klicka på Nästa.
  6. Skriv in ditt DSM-användarnamn och lösenord för autentisering, klicka på OK och sedan på Klart.

Att prenumerera på din kalender med din mobila enhet (med till exempel en iOS-enhet):

  1. Gå till Inställningar > Mail, Kontakter, Kalendrar på din iOS-enhet.
  2. Under Konton, gå till Lägg till konto > Övriga och tryck sedan på Lägg till CalDAV-konto.
  3. I fälten anger du IP-adressen (eller domännamnet) för DiskStation, dina användaruppgifter och trycker sedan på Nästa. Om ett meddelande visas, tryck på Fortsätt.
  4. Tryck på Avancerade inställningar och gör följande:
    1. Skriv 5005 (eller något annat portnummer du angivit vid aktivering av WebDAV) i fältet Port.
    2. Ändra URL-adressen i fältet Konto-URL till http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name/ (till exempel "http://demo.no-ip.info:5005/calendar/cal1").
    3. Tryck på CalDAV.
  5. Tryck på Nästa för att kontrollera din kontoinformation och tryck sedan på Spara för att avsluta kontoinställningen.
  6. Gå tillbaka till iOS-enhetens startskärm och öppna Kalender för att hantera din kalender.

Obs:

  • För mer information om hur du hanterar kalenderhändelser, anteckningar eller att göra-listor, läs hjälpdokumenten för vardera klientapplikation.
