Allmänt

Allmänna nätverksalternativ såsom standardgateway, DNS-server och IPv6-tunnling finns tillgängliga i Kontrollpanel > Nätverk > Allmänt.

Byta namn på DiskStation-enhet

För att byta namn på din DiskStation-enhet, ange det nya servernamnet och klickar på Tillämpa.

Servernamnet är skiftlägeskänsligt och kan vara i intervallet mellan 1 till 15 tecken bestående av bokstäver, siffror, understreck och minustecken. Det första tecknet i namnet måste vara en bokstav.

Specificera standardgateway

Ange helt enkelt IP-adressen för standardgateway och klicka på Tillämpa för att spara inställningen.

Specificera IP-adress för DNS Server

Du kan välja att konfigurera IP-adressen på din DNS Server manuellt. I detta fall måste IP-adress till din föredragna DNS Server anges. En IP-adress för alternativ DNS Server kan också anges, vilket ger en ersättare om föredragen DNS Server får problem. När IP-adresser för DNS Server har konfigurerats så som önskas, vänligen klicka på Tillämpa för att spara inställningarna.

Obs:

DNS-servern och Standardgatewayfälten ställs in automatiskt och kan inte ändras när alla dina nätverksgränssnitt konfigureras automatiskt eller om du använder en PPPoE-anslutning.

Anslutning via en proxyserver

För att tillåta din DiskStation-enhet att ansluta via en proxyserver, markera Anslut via en proxyserver, ange Adress och Port, och klicka sedan på Tillämpa.

Obs:

  • Klicka på Avancerade inställningar för att konfigurera HTTP proxy, HTTPS proxy och autentiseringsinformation.
  • Som standard är inställningarna för HTTPS-proxy samma som HTTP-proxy.
  • Om Anslut via en proxyserver är markerad, kommer DSM-uppdatering och Paketcenter samt några andra filpaket alltid att ansluta till Internet via en proxy.
  • Om Kringgå proxyserver för lokala adresser är markerad, vänligen säkerställ att DNS-servern som används av DiskStation kan slå upp förfrågningar för enheter i det lokala nätverket.

Aktivera IPv6-tunnling

Till dess IPv6 helt och håller ersätter IPv4 behövs IPv6-tunnling för att aktivera värdar med IPv6-adressera att nå IPv4-tjänster. Denna teknik tillhandahåller ett sätt att använda IPv6-trafik i befintliga IPv4-nätverk.
Om du inte är säker på hur du ska konfigurera IPv6 eller om du stöter på problem med anslutningen, kontakta din Internetleverantör.

Att aktivera IPv6-tunnel:

  1. Markera kryssrutan Aktivera tunnel för att aktivera stöd för tunnel.
  2. I Serveradress, ange värdnamnet eller IP-adressen för tunneltjänstleverantören, till exempel: anonym.freenet6.net.
  3. Välj Anslut anonymt eller Anslut med ett existerande konto.

Obs:

Din IPv6-adress kan eventuellt uppdateras automatiskt av tjänsteleverantören. DiskStation-enheten kommer dock inte att ta bort den gamla IPv6-adressen omedelbart.
