Säkerhet

Du kan ändra följande säkerhetsinställningar i Kontrollpanel > Säkerhet > Säkerhet för bläddringssessioner i DSM:

  • Utloggningstimer (minuter): Användarna blir automatiskt utloggade från DSM om de är inaktiva för den tidsperiod som specificeras här. Ange valfritt värde mellan 1 och 65535.
  • Öka webbläsarkompatibiliteten genom att hoppa över kontroll av IP: Om du använder DiskStation via en HTTP-proxy och drabbas av slumpvisa utloggningar kan du aktivera detta alternativ för att hoppa över kontroll av IP.
  • Öka skyddet mot cross-site request forgery attacks: Detta alternativ ökar systemets skydd mot cross-site skriptningsattacker. Detta alternativ träder i kraft nästa gång du loggar in på DSM.
  • Förbättra säkerheten med HTTP Content Security Policy (CSP)-huvud: Det här alternativet förbättrar systemets säkerhet mot attacker med cross-site scripting (XSS) genom att endast tillåta data från betrodda källor och begränsa körning av inlineskript.
  • Tillåt inte att DSM bäddas in med iFrame: Du kan aktivera detta alternativ för att förhindra att andra webbplatser bäddar in DSM på deras webbsidor med iFrame och genom detta förhindra vissa typer av attacker från skadliga webbplatser. Specifika webbplatser kan tillåtas att bädda in DSM med iFrame, genom att klicka på knappen Tillåtna webbplatser.
