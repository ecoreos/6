Blåtand

Denna sida tillhandahåller alternativ för att hantera blåtandadapter och ansluta blåtandsenheter till din DiskStation. Väl anslutna kan en del blåtandsenheter, såsom högtalare eller hörlurar väljas som ljudutgång för Audio Station, vilket gör det enkelt för dig att lyssna på musik som lagrats på din DiskStation.

Komma igång

Ansluta en USB-blåtandadapter till en USB-port på din DiskStation. En lista med blåtandadaptrar visas på den vänstra sidan av huvudsidan för blåtand. Information och alternativ om blåtandsenheter visas till höger.

Aktivera eller inaktivera blåtandadaptrar:

  1. Välj en blåtandadapter.
  2. Klicka på Avancerade inställningar.
  3. Markera kryssrutan Aktivera blåtand.
  4. Klicka på OK för att spara inställningarna.

Hantera blåtandsenheter

Vid anslutning av en ny blåtandenhet, tillse att den går att upptäcka och befinner sig inom 10 meter från DiskStation samt använder A2DP (Advanced Audio Distribution Profile). Se din användarhandbok för blåtandenheten för mer detaljer om installation.

Ansluta en blåtandenhet:

  1. På huvudsidan för Blåtand, välj den blåtandenhet du vill ansluta till DiskStation. Om enheten inte visas, prova att klicka på Uppdatera för att söka efter enheter i närheten.
  2. Välj enheten när den visas.
  3. Klicka på Anslut.
  4. Ange PIN för din enhet om det behövs.

Se information om en blåtandenhet:

  1. Välj en ansluten blåtandenhet.
  2. Klicka på Info.
  3. Ett fönster visas och innehåller information såsom blåtandadress, namn, typ och status.

Ta bort en blåtandenhet:

  1. Välj en blåtandenehet som du inte vill använda med din DiskStation längre.
  2. Klicka på Ta bort.

Obs:

  • När musik spelas genom Blåtandhögtalare eller -hörlurar kan du uppleva problem med uppspelningen om inställningarna för din blåtandadapter är modifierad eller om du ansluter till för många blåtandenheter på samma gång.
  • Beroende på enheten, om ett par blåtandhögtalare som varit anslutna till din DiskStation senare ansluts till en annan DiskStation (eller blåtandmusikspelare) kan högtalarna glömma din DiskStation. När detta händer uppträder ett fel när du försöker spela upp musik med högtalarna. För att lösa detta problem, vänligen ta bort högtalarna och försök att ansluta dem igen.
