PPPoE

Om du använder ett DSL- eller kabelmodem för att ansluta till Internet och din Internetleverantör (ISP) använder PPPoE (Point to Point Protocol over Ethernet), kan du aktivera PPPoE i Kontrollpanel > Nätverk > Nätverksgränssnitt och ange kontoinformationen för att direkt ansluta till Internet utan routern.

För att få mer information om PPPoE, kontakta din ISP eller nätverksadministratör.

IPv6-inställning

DiskStation-enheter har stöd för IPv4/IPv6 dual stack-scheman. Detta innebär att både IPv4 och IPv6 kan arbeta samtidigt på Synology din DiskStation. Standardinställningarna för IPv6 är Automatiskt. Välj ett befintligt nätverksgränssnitt, klicka på knappen Redigera och gå till fliken IPv6 för att ändra inställningarna.

Till dess IPv6 helt och håller ersätter IPv4 behövs övergångsmekanismer för IPv6 för att aktivera värdar med IPv4-adresser att nå IPv6-tjänster. Välj mellan de olika typerna av övergångsmekanismer för IPv6:

  • 6i4: Ansluter till IPv6-nätverket genom en tunnelagent. Du behöver fylla i IPv6-adress, Prefix-längd, Prefix och IPv4-adress för tunnelagent. Det kräver en publik IPv4-adress.
  • 6till4: Anslut till IPv6-nätverket genom en tunnelagent genom att använda anycast. Det kräver en publik IPv4-adress.
  • DHCP-PD: Erhåll ett IPv6-prefix från DHCP-servern via prefixdelegering inom LAN. Det kräver IPS-stöd och fungerar bara i routerläge.
