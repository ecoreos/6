SSO Client

Om din DiskStation har anslutit till samma domän eller LDAP-tjänst som SSO Server kan du ställa in din DiskStation som en SSO Client. Användare kan få åtkomst till tjänsterna som tillhandahålls av din DiskStation när de har loggat in på SSO-servern med sina autentiseringsuppgifter.

Innan du börjar, kontrollera att din DiskStation har anslutit till samma domän eller LDAP-tjänst som SSO-servern.

För att ställa in din DiskStation som en SSO Client:

  1. Gå till Kontrollpanel > Domän/LDAP > SSO Client.
  2. Markera kryssrutan Aktivera tjänsten SSO.
  3. Specificera URL för SSO-server och Applikations-ID i fälten nedanför kryssrutan.

    Obs:

    För att få Applikations-ID, välj något av följande metoder:

    • Metod 1:
      1. Logga in på den DSM som körs som SSO Server.
      2. Lägg till din DiskStation som kommer att tjänstgöra som klientapplikationen i Applikationslista i SSO Server. För detaljerade steg, se hjälpartikeln för SSO Server > Applikationslista.
      3. Kopiera och klistra in Applikations-ID som allokerats av SSO Server när din DiskStation har lagts till i SSO-servern.
    • Metod 2:
      1. Efter att URL för SSO-servern* har lagts till, klicka på Snabbregistrering bredvid fältet Applikations-ID.
      2. I registreringsrutan, ange autentiseringsuppgifterna för ett konto som är administrator på SSO-servern i fälten Server-administrator och Lösenord.
      3. Specificera namnet på din klientapplikation i fältet Applikationens namn.
      4. Kontrollera noga att Omdirigerings-URI är exakt den som leder till inloggningssidan för din DiskStation.
      5. Klicka på OK så läggs din DiskStation. till i SSO Server, med Applikations-ID automatiskt ifyllt.

    ______

    * Det krävs att du anger den identiska SSO server URL (antingen värdnamn eller IP) vid konfiguration av varje SSO-klient.

  4. Klicka på Tillämpa för att spara dina inställningar.

Obs:

  • Om SSO-servern använder HTTPS-protokoll kan inloggningen till SSO blockeras av webbläsare som har betrott certifikat som standard.
  • Vid åtkomst till en SSO-server via HTTP-protokoll, om din DiskStation, (inställd som en SSO-klient) använder HTTPS-anslutning, behöver användare konfigurera inställningarna i deras webbläsare att tillåta användning av en osäker anslutning.
  • IP-adressen i IPv6-formatet för SSO-servern har inte stöd om du har bockat för kryssrutan Förbättra säkerheten med HTTP Content Security Policy (CSP)-huvud i Kontrollpanel > Säkerhet > Säkerhet på din DSM.
