Konfiguration av router

Om din DiskStation-enhet finns i ett lokalt nätverk kan andra enheter på Internet inte ansluta till den. Du kan ställa in regler för vidarebefordran av port i Kontrollpanel > Extern åtkomst > Konfiguration av router för att göra din DiskStation-enhet åtkomlig över Internet.

Ställa in din router

Innan regler för vidarebefordring av port läggs till måste du installera en anslutning mellan din DiskStation-enhet och din router.

Obs:

  1. För att konfigurera regler för vidarebefordring av port och tilldela fast IP-adress, behöver du ha administratörsprivilegier till routern.
  2. Innan du ställer in anslutning för DiskStation-enheten till en UPnP-router, se till att DSM har lyckats erhålla en extern IP-adress för din DiskStation (i Kontrollpanel > Extern åtkomst > DDNS). Om inte, kontrollera nätverks- och brandväggsinställningar för din DiskStation-enhet och UPnP-routern.

Att ställa in DiskStation-enhetens anslutning till din router:

  1. Klicka på Installera din router för att köra konfigurationsguiden. DSM kommer att börja med att testa följande nätverkskomponenter/-inställningar innan nästa steg vidtas.
    • Nätverksgränssnitt
    • Gatewayanslutning
    • Internetanslutning
    • Vidarebefordring av port
    • DNS
  2. Beroende på vilken typ av router du har, gör något av följande:
    • Om din router inte UPnP, se steg 3.
    • Om din UPnP-router har testats och befunnits vara kompatibel med DiskStation-enhet, klicka på Tillämpa när du ser en sammanfattning av routerns information. Din DiskStation-enhets anslutning till routern har då installerats korrekt.
    • Om din UPnP-router inte har testats för kompatibilitet med DiskStation-enheten, gör något av följande:
      • Om du vill konfigurera anslutningen till routern automatiskt väljer du Automatiskt och klickar på Nästa för att låta systemet utföra ett kompatibilitetstest:
        • Om routern befinns vara kompatibel med DiskStation-enhetens tjänst för konfiguration av router, markera kryssrutan för att skicka testrapporten och hjälpa Synology att uppdatera databasen över kompatibla routrar, klicka därefter på Tillämpa. Din DiskStation-enhets anslutning till routern har då installerats korrekt.
        • Om routern befinns vara inkompatibel med DiskStation-enhetens tjänst för konfiguration av router, markera kryssrutan för att skicka testrapporten och hjälpa Synology att uppdatera databasen över kompatibla routrar, klicka därefter på Stäng för att avsluta konfigurationsguiden.

          • Obs:

          Även om din router konstaterades vara inkompatibel här kan du fortfarande konfigurera regler för vidarebefordran av port i routerns användargränssnitt. Se här för mer information.

      • Om du vill konfigurera anslutningen till routern manuellt väljer du Manuellt och klickar på Nästa för att få referens i steg 3.
  3. Om du behöver ställa in DiskStation-enhetens anslutning till routern manuellt, gör följande:
    1. Välj din router från rullgardinsmenyn Router.

      2. Obs:

      Om du inte kan hitta din router här, klicka på Uppdatera lista för att få den senaste listan från Synology.

    2. Ställ in protokoll och port för din router. Protokollet och porten varierar beroende på vilken modell din router är. Om du inte kan ansluta med standard Protokoll (HTTP) och Port (80), se dokumentationen som medföljer din router för rätt inställningar.
    3. Om din router kräver ett skräddarsytt konto, markera Använd anpassat konto och fyll i fälten Konto och Lösenord.
    4. Klicka på Tillämpa för att slutföra.

Hantera regler för vidarebefordring av port

Regel för vidarebefordring av port kan ange målportar för din DiskStation, dessa kommer att ta emot paket från specifika portar hos din router.

Om din DiskStation ansluter till Internet via en router, vänligen klicka på Skapa för att öppna guiden för Vidarebefordring av port.

Det finns två sätt att lägga till regler för vidarebefordring av port på:

  • Om du vill skapa regler för vidarebefordring av port för systemtjänster (t.ex., FTP, Web Station, etc.), välj Inbyggd applikation.
  • Om du vill skapa regler för vidarebefordring av port för andra tjänster, välj Skräddarsydd port.

Du kan använda följande syntax för att ange portnummer och område:

  • Använd “-” för att beskriva portintervall, t.ex.: 6881-6890.
  • Använd “,” för att avgränsa portar, t.ex.: “21,22”, “21,55536-55663”.

Obs:

  • Varje port i gateway-routern kan endast användas för en regel för vidarebefordring. Om du använder en routerport för två eller fler regler kan detta orsaka problem och ett varningsmeddelande kommer att visas på skärmen. Du måste ange en annan port till regeln för att undvika problem.
  • En del routrar tillåter inte bruk av anpassade portar för vidarebefordring av systemtjänster.

Spara regler:

Klicka på Spara efter att du skapat reglerna. Annars kommer reglerna inte att tillämpas.

Efter att du sparat reglerna måste du kanske vänta några minuter innan reglerna börjar att fungera, beroende på vilken modell dina routrar är av.

Testa om dina regler fungerar eller inte:

Välj de regler du vill testa och klicka på Testa anslutning. Resultatet kommer att visas i kolumnen Anslutningstestets resultat.

För konfiguration av trådlös router*

När din DiskStation agerar trådlös router, kommer regler för vidarebefordran av portar utföras direkt av DiskStation.

Lägga till regler för vidarebefordring:

  1. Klicka på Skapa för att öppna inställningsfönstret.

  2. Specificera namnet på Tjänst, Protokoll (TCP eller UDP), Privat IP-adress, Privat port och Publik port. Du kan använda följande syntax för att ange portnummer och område:
    • Använd “-” för att beskriva portintervall, t.ex.: 6881-6890.
    • Använd “,” för att avgränsa portar, t.ex.: “21,22”, “21,55536-55663”.
  3. Klicka på Skapa längst ned för att slutföra inställningarna.

Obs:

De flesta routrar tillåter att lokala maskinportar och routerportar är olika när regler för vidarebefordran av port konfigureras. Om din router har stöd för den här funktionen:

  • När en regel sparas och en routerport redan används av en befintlig regel tilldelas en ny routerport.
  • När regler läggs till eller redigeras kan du specificera routerportnumret direkt.

DMZ*

Du kan aktivera DMZ på din DiskStation när konfiguration av nätverkskarta är trådlös router. En värd i DMZ kan ansluta direkt till det externa nätverket utan begränsningar och utan skydd av brandväggen. Det används vanligtvis för inställning av servrar.

Aktivera DMZ:

  1. Klicka på DMZ-inställningar för att öppna inställningsfönstret.
  2. Markera kryssrutan Aktivera DMZ.
  3. Ange IP-adress för din DMZ-värd.
  4. Klicka på Tillämpa.

Obs:

Från det externa nätverket kommer du att behöva en extern IP-adress, som du får från DiskStation, för att ansluta till en värd i DMZ.
