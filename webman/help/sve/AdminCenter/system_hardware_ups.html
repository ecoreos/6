UPS

En UPS (Uninterruptible Power Supply) är en reservströmkälla som låter DiskStation fortsätta fungera under en kortare tid om ett strömavbrott inträffar. Denna funktion i Kontrollpanel > Hårdvara & Ström > UPS hjälper till att förhindra dataförlust genom att ge DiskStation tillräckligt med tid att spara data och avmonterar alla volymer före den blir strömlös och stängs av.

Felsäkert läge

När DiskStation går in i säkert läge stoppar den alla tjänster och avmonterar volymer för att förhindra dataförlust och stänga ner säkert när UPS-enhetens ström tar slut. Som standard går systemet in i säkert läge när UPS-enheten börjar få ont om ström. Du kan också specificera hur länge det ska dröja innan DiskStation går in i säkert läge när ett strömavbrott inträffar. Om UPS-enheten når låg batterinivå före den specificerade tiden går den emellertid in i säkert läge genast.

I situationer där DiskStation stängs ned i Säkert läge, kommer den automatisk att sättas på när strömmen återställs, om du har aktiverat alternativet Starta om automatiskt efter strömavbrott (finns i Kontrollpanel > Hårdvara & Ström > Allmänt).

Lokalt stöd för SNMP UPS

För detaljer om hårdvaruinstallation, vänligen se användarhandboken för din UPS-enhet. När installationen är komplett och DiskStation har kopplats in i UPS-enheten kan du följa stegen här nedanför för att aktivera stöd för UPS.

Aktivera lokalt stöd för UPS:

  1. Anslut UPS-enheten till en USB-port på DiskStation.
  2. På fliken UPS, markera kryssrutan Aktivera stöd för UPS.
  3. Välj när DiskStation ska gå in i säkert läge efter att ett strömavbrott inträffat.
  4. Välj huruvida du ska sända en signal för att stänga av till UPS-enheten när systemet går in i säkert läge. De flesta UPS-enheter kan automatiskt starta när systemet återställts.
  5. Klicka på Tillämpa.

SNMP UPS Support

Din DiskStation har också stöd för att ansluta till UPS-enheter med SNMP (Simple Network Management Protocol). För detaljer om hårdvaruinstallation, vänligen se användarhandboken för din UPS-enhet med SNMP. När väl DiskStation har kopplats in i enheten och är ansluten till samma nätverk kan du följa stegen här nedanför för att aktivera stöd för UPS.

Aktivera stöd för SNMP UPS:

  1. På fliken UPS, markera kryssrutan Aktivera stöd för UPS.
  2. Välj SNMP UPS från rullgardinsmenyn Nätverkets UPS-typ.
  3. I fältet SNMP UPS IP-adress ange IP-adressen till SNMP UPS-enheten.
  4. Välj när DiskStation ska gå in i säkert läge efter att ett strömavbrott inträffat.
  5. Ange communitysträngen för SNMP UPS-enheten i fältet SNMP Community.
  6. Klicka på Tillämpa.

Obs:

  • Det rekommenderas starkt att SNMP UPS används vid implementering av Synology High Availability (SHA).
  • DiskStation har stöd för SNMPv1 och SNMPv2c UPS-enheter.

Nätverksstöd för UPS

Om det finns två Synology-produkter som använder en UPS-enhet med en enda USB-port kan bara en DiskStation få information om statusen för UPS-enheten. I detta fall kan en DiskStation agera som ”network UPS server” och ansluta till UPS-enheten via USB eller nätverk (beroende på enheten). UPS-nätverksserver innehåller UPS-information som den överför till den andra ”client” DiskStation.

Aktivera stöd för nätverks-UPS:

  1. Logga in på DiskStation som är ansluten till UPS-enheten (via USB eller nätverk). Om lokalt stöd för UPS inte är aktiverat på denna DiskStation ännu, vänligen se avsnittet här ovanför.
  2. På fliken UPS, markera kryssrutan Aktivera nätverkets UPS-server för att tillåta DiskStation att överföra UPS-information till klienten DiskStation.
  3. Klicka på knappen Tillåtna DiskStation-enheter.
  4. I fönstret som visas, ange IP-adresser för de andra DiskStation som använder UPS-enheten men inte anslutna via USB eller nätverket. Klicka på OK.
  5. Klicka på Tillämpa för att spara inställningarna.

Nu kan UPS-nätverksserver få UPS-information och överföra den till de andra DiskStation som fortfarande använder UPS-enheten. Vänligen se instruktionerna här nedanför för att aktivera stöd för UPS på den andra DiskStation.

För att aktivera stöd för UPS på klienten DiskStation:

  1. Logga in på den andra DiskStation som använder UPS-enheten men som inte är ansluten via USB eller nätverk.
  2. På fliken UPS, markera rutan Aktivera stöd för UPS.
  3. Välj Synology UPS Server från rullgardinsmenyn Nätverkets UPS-typ.
  4. Ange IP-adress för UPS-nätverksserver (t.ex. den DiskStation som är ansluten till UPS-enheten), i fältet Nätverkets UPS-server IP.
  5. Välj när DiskStation ska gå in i säkert läge efter att ett strömavbrott inträffat. För att förenkla inställningarna för Säkert läge, kan du välja Samma som server.
  6. Klicka på Tillämpa för att spara inställningarna.

Krav för nätverks-UPS:

  • För att överföra information om UPS-enhet ska varje DiskStation vara ansluten till samma nätverk.
  • Nätverksenheten (t.ex. switch eller hub) till vilken varje DiskStation är ansluten ska vara inkopplad i samma UPS-enhet. I annat fall kommer inte DiskStation:s inte att ha möjlighet att överföra information om UPS-enheten när ett strömavbrott inträffar.
