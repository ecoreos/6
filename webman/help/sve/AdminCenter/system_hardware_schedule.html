Schema för start/avstängning*

DC-utgång*

I Kontrollpanel > Hårdvara & Ström > Schema för start/avstängning kan du ställa in att systemet ska startas eller stängas av automatiskt vid en viss tidpunkt.

I Kontrollpanel > Hårdvara & Ström > DC-utgång kan du ställa in att systemet ska starta eller stänga av DC-utgången automatiskt vid en viss tidpunkt.

Så här lägger du till ett nytt schema för start/avstängning:

För att lägga till en ny utmatning på/utmaning av i schemat:

  1. Klicka på knappen Skapa för att öppna dialogrutan Lägg till schema.
  2. Välj Start eller Stäng av.
  3. Ange Datum och Tid för start eller avstängning.
  4. Klicka på OK.
  5. Klicka på Spara.
  1. Klicka på knappen Skapa för att öppna dialogrutan Lägg till schema.
  2. Välj Utmatning på eller Utmatning av.
  3. Ställ in Datum och Tid för schemat utmatning-på/utmatning-av.
  4. Klicka på OK.
  5. Klicka på Spara.

Det nya schemat kommer att läggas till i listan över scheman.

Visa en sammanfattning av schemat:

Klicka på Sammanfattning för att ta fram veckoschemat för start/avstängning.

Klicka på Sammanfattning för att ta fram veckoschemat för schemat utmatning-på/utmatning-av.

Obs:

  • Du kan ange upp till 100 scheman för en DiskStation.
  • I följande fall kan systemet misslyckas med att stänga av automatiskt:
    • Lagringshanterare utför uppgifter.
    • Systemet krypterar delade mappar.
    • Systemet utför Secure Erase på hårddiskar.
    • DSM uppgraderas.
    • Systemet reparerar RAID-volymer.
    • Paketcenter utför uppgifter.
    • Systemet uppgraderar databaser.
    • Systemet utför säkerhetskopieringsuppgifter.
  • För att konfigurera ett startschema, vänligen säkerställ en kontinuerlig strömförsörjning till DiskStation. Om det inträffar ett tillfälligt strömavbrott för DiskStation blir schema för start/avstängning ogiltigt.
  • Du kan ange upp till 100 scheman för en DiskStation.
  • I följande fall kan systemet misslyckas med att stänga av DC-utgången automatiskt:
    • Lagringshanterare utför uppgifter.
    • Systemet krypterar delade mappar.
    • Systemet utför Secure Erase på hårddiskar.
    • DSM uppgraderas.
    • Paketcenter utför uppgifter.
    • Systemet uppgraderar databaser.
    • Systemet utför säkerhetskopieringsuppgifter.
