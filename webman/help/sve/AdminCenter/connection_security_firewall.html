Brandvägg

I Kontrollpanel > Säkerhet > Brandvägg kan du aktivera brandvägg, skapa brandväggsregler och konfigurera inställningar för brandvägg för att förhindra oauktoriserade inloggningar och kontrollera åtkomst till tjänster. Du kan avgöra huruvida tillåta eller neka åtkomst till vissa nätverksportar från specifika IP-adresser.

Hantering av brandvägg

Du kan skapa brandväggsregler för olika brandväggsprofiler så att det enkelt och snabbt går att växla till och använda den önskade profilen i enlighet med olika säkerhetsbehov.

Att aktivera brandvägg:

  1. Gå till Kontrollpanel > Säkerhet > Brandvägg och markera Aktivera brandvägg.
  2. Klicka på Tillämpa för att spara inställningarna.

Att konfigurera en brandväggsprofil:

  1. I avsnittet Brandväggsprofil, klicka på ikonen + i rullgardinsmenyn och ange ett namn för att skapa den nya profilen.
  2. Välj den önskade profilen från rullgardinsmenyn och klicka på Välj.
  3. Klicka på knappen Redigera regler till höger och klicka på Skapa för att skapa brandväggsregler för den valda profilen. (Vänligen se avsnittet här nedanför för mer information om att skapa brandväggsregler.)
  4. Repetera ovanstående steg tills du har skapat alla profiler och brandväggsregler som du behöver.

Att hantera en brandväggsprofil:

I avsnittet Brandväggsprofil, klicka på motsvarande knappar i rullgardinsmenyn för att skapa, radera, byta namn på eller klona den valda profilen.

Att skapa en brandväggsregel:

  1. I avsnittet Brandväggsprofil, välj en brandväggsprofil från rullgardinsmenyn och klicka på knappen Redigera regler till höger.
  2. Välj ett nätverksgränssnitt från rullgardinsmenyerna i det övre högra hörnet.
  3. Klicka på Skapa.
  4. I avsnittet Portar välj en av följande:
    • Välj Alla för att tillämpa den här brandväggsregeln på alla portar.
    • Välj Välj från en lista över inbyggda applikationer och klicka på Välj. Välj de inbyggda applikationerna som du vill tillämpa den här brandväggsregeln på.
    • Välj Anpassa och klicka på Anpassa. Ange upp till 15 portar separerade med komman eller specificera ett portintervall att tillämpa denna brandväggsregel på.
  5. I avsnittet Käll-IP välj en av följande:
    • Välj Alla för att tillämpa den här brandväggsregeln på alla IP-adresser.
    • Välj Specifik IP och klicka på Välj. Du kan specificera en IP-adress eller ett IP-intervall att tillämpa denna brandväggsregel på.
    • Välj Region och specificera sedan upp till 15 länder att applicera den här brandväggsregeln på.
  6. I avsnittet Åtgärd välj en av följande:
    • Välj Tillåt för att tillåta åtkomst för portar och käll-IP-adresser du har specificerat.
    • Välj Neka för att tillåta åtkomst för portar och käll-IP-adresser du har specificerat.

Du kan visa brandväggsregler för LAN och PPPoE genom att välja dessa gränssnitt från rutan i det övre högra hörnet. Längst ned i listan över regler kan du välja Tillåt åtkomst eller Neka åtkomst för att tillåta eller neka alla åtkomstförfrågningar som inte överensstämmer med en befintlig brandväggsregel för respektive gränssnitt.

Obs:

  • Dra och släpp för att ändra ordning på reglerna i listan.
  • Regler prioriteras enligt deras plats i listan.
  • Om ditt system har flera nätverksportar anslutna till samma subnät kan det hända att brandväggsreglerna inte fungerar korrekt.*
  • När du kombinerar flera LAN-portar med link aggregation tillämpar brandväggen reglerna från det första nätverkskortet och bevarar reglerna för det andra nätverksgränssnittet.
  • Om din DiskStation-enhet ansluter till Internet via PPPoE, måste tillhörande regler för brandvägg konfigureras i motsvarande PPPoE-gränssnitt.

Att tillämpa en brandväggsprofil:

  1. I avsnittet Brandväggsprofil, använd rullgardinsmenyn för att byta till den önskade profilen.
  2. Klicka på Tillämpa för att spara ändringarna. Den valda brandväggsprofilen kommer att tillämpas och användas som den aktiva profilen.

Beteende för brandväggsregel:

DSM-brandväggen kommer att använda regler som överensstämmer enligt prioritet. När en regel väl överensstämmer kommer den att bli framtvingad och DSM-brandväggen fortsätter inte att matcha återstående regler. Om det inte finns några regler som överensstämmer kommer DSM-brandväggen att utföra den standardåtgärd som finns specificerad för varje gränssnitt.

Prioritet för brandväggsregel:

  1. Regler som är definierad i ”Alla gränssnitt”.
  2. Regler som används i respektive gränssnitt som anslutningen hör till.
  3. Standardregler som används i respektive gränssnitt som anslutningen hör till.

* Endast tillgängligt för vissa modeller.
