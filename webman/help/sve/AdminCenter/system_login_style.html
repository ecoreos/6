Inloggningsutseende

Du kan anpassa utseendet på inloggningsskärmen för DSM, genom att välja från ett utbud av mallar, ändra bakgrundsbild eller lägga till en logo. För att göra detta, gå till Kontrollpanel > Inloggningsutseende.

För att anpassa inloggningsskärmen i DSM:

Följande alternativ finns på sidan Inloggningsutseende:

  • Titel för inloggningssida: Text som anges här visas på inloggningsskärmen.
  • Tidsinformation: Välj att visa eller dölja lokal tidsinformation på inloggningsskärmen.
  • Mall: Välj ett färgschema och en standardbakgrundsbild. Den valda mallen börjar gälla nästa gång du loggar in på DiskStation.
  • Bakgrund: För att ändra den bakgrundsbild som visas på inloggningsskärmen, klicka på den befintliga bilden eller plustecknet (+) om ingen bild är vald. Du kan välja från standardbakgrunder, bilder som lagras på DiskStation eller tidigare använda bilder. Det finns bara stöd för filformaten JPE, JPEG, JPG, BMP, GIF eller PNG.
  • Logo: För att lägga till en logotyp till inloggningsskärmen, klicka på den befintliga logotypen eller plustecknet (+) om inte någon bild är vald. Du kan ladda upp en bild som är mindre än 1 MB, välja från bilder som lagras på DiskStation eller välja en tidigare använd bild.
