Hantera avancerade behörigheter för Delad mapp

Sida i Kontrollpanel > Delad mapp > Redigera > Avancerad inkluderar nedanstående alternativ för att ytterligare finjustera åtkomstbehörigheterna för en delad mapp.

Avancerade resursbehörigheter

Avancerade resursbehörigheter tillför ett extra lager av kontroll för att hantera åtkomstbehörigheterna för delade mappar. När aktiverad kan användare och grupper visa eller ändra innehållet i en delad mapp endast om användaren eller gruppen har tilldelats både resursbehörigheter och Windows ACL-behörigheter (finns i Delad mapp > Redigera > Behörigheter).

Att aktivera avancerade resursbehörigheter:

  1. Markera rutan Aktivera avancerade resursbehörigheter.
  2. Klicka på Avancerade resursbehörigheter för att ändra de avancerade resursbehörigheterna för den delade mappen.

Obs!

  • Avancerade resursbehörigheter tillämpas när en användare har åtkomst till en delad mapp via följande filtjänster: Windows File Sharing, Apple File Sharing, File Station, FTP och WebDAV.

Avancerade inställningar

Om nödvändigt kan du ställa in ytterligare restriktioner för användare när de har åtkomst till den delade mappen via File Station, FTP eller WebDAV.

  • Inaktivera mappbläddring: Att aktivera detta alternativ förhindrar användare från att se innehållet i den delade mappen.
  • Inaktivera ändring av befintliga filer: Att aktivera detta alternativ förhindrar användare från att flytta, ta bort, eller ändra filer i den delade mappen. Vänligen observera att användare fortfarande kommer kunna se, ladda ned/ladda upp, kopiera eller extrahera innehållet i den delade mappen.
  • Inaktivera nedladdning av filer: Att aktivera detta alternativ förhindrar användare från att ladda ned innehållet från den delade mappen.

Obs!

Dessa restriktioner tillämpas inte på användare som hör till gruppen administrators.
