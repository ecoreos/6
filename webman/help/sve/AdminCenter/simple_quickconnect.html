QuickConnect

QuickConnect låter klientapplikationer ansluta till din DiskStation via Internet, utan att krångel med att sätta upp regler för vidarebefordran av portar.

Obs:

QuickConnect kan fungera med DSM och en del filpaket som är utvecklade av Synology, såsom Audio Station, Video Station, Download Station, Surveillance Station, Photo Station , File Station, Cloud Station och mobila applikationer.

Att aktivera QuickConnect:

  1. Gå till Kontrollpanel (Enkelt läge) > QuickConnect.
  2. Markera rutan Aktivera QuickConnect.
  3. Ange information för ditt befintliga Synology-konto eller registrera ett nytt konto.
  4. Specificera ett nytt QuickConnect ID. Säkerställ att det är lätt att komma ihåg så du enkelt kan få åtkomst till din DiskStation när som helst.
  5. Klicka på Tillämpa.

Obs:

  • Innan aktivering av QuickConnect för klientapplikationer, tillse att nödvändiga tjänster på DSM redan är aktiverade. Till exempel måste WebDAV vara aktiverad på DiskStation innan du kan aktivera QuickConnect för DS file.
  • Ett anpassat QuickConnect ID kan innehålla bokstäver, siffror och bindestreck (”-”) och måste börja med en bokstav.
  • För bättre QuickConnect-prestanda rekommenderas att du går till Kontrollpanel (Avancerat läge) > Extern åtkomst > Konfiguration av router för att konfigurera vidarebefordring av port för varje tjänst:
    • DSM: 5000 (HTTP); 5001 (HTTPS)
    • Photo Station: 80 (HTTP); 443 (HTTPS)
    • Cloud Station：6690

Felsökning

Om du ser meddelandet Ett nätverksfel inträffade betyder det att tjänsten QuickConnect har avbrutits på grund av nätverksfel. Kontrollera följande:

  • Att din DiskStation är anslutet till ett aktivt nätverk (i Kontrollpanel (Avancerat läge) > Nätverk > Nätverksgränssnitt).
  • Att DNS-server och standardinställningar för gateway är korrekt konfigurerade (i Kontrollpanel (Avancerat läge) > Nätverk > Allmänt).
  • Tillse att DiskStation har åtkomst till Internet med användning av följande målportar: 22, 80, 443, 8888. 

När nätverksfelen är avhjälpta bör tjänsten QuickConnect återupptas automatiskt inom några minuter.
