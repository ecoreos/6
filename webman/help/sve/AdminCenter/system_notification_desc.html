Meddelande

Du kan ställa in din DiskStation att skicka meddelanden via e-post, SMS, mobila enheter eller webbläsare när en specifik händelse eller ett specifikt fel inträffar. Till exempel kan systemet automatiskt skicka dig ett meddelande vid frånkoppling av nätverksanslutning, förändring i beteende för kylfläkt, strömavbrott, början till slut på utrymme eller när säkerhetskopierings- eller återställningsuppgifter inte slutförts.
