QuickConnect

QuickConnect låter klientapplikationer ansluta till din DiskStation via Internet, utan att krångel med att sätta upp regler för vidarebefordran av portar. QuickConnect kan även fungera med en del filpaket som är utvecklade av Synology, såsom Audio Station, Video Station, Download Station, Surveillance Station, Photo Station , File Station, Note Station, CMS, Cloud Station och mobila applikationer.

Att aktivera QuickConnect:

  1. Gå till Kontrollpanel > QuickConnect > Allmänt.
  2. Markera Aktivera QuickConnect.
  3. Ange information för ditt befintliga Synology-konto eller registrera ett nytt konto.
  4. Specificera ett nytt QuickConnect ID. Säkerställ att det är lätt att komma ihåg så du kan få åtkomst till din DiskStation när som helst.
  5. Klicka på Tillämpa.

Obs:

Ett anpassat QuickConnect ID kan innehålla bokstäver, siffror och bindestreck ("-") och måste börja med en bokstav.

Att förbättra anslutningsbarheten för QuickConnect:

Din DiskStation erbjuder avancerade alternativ för att göra QuickConnect-åtkomst mer anpassningsbar för olika nätverksmiljöer.

  1. Gå till Kontrollpanel > QuickConnect > Avancerat.
  2. Välj de nedanstående alternativen som motsvarar dina behov:
    • Aktivera överföringstjänsten QuickConnect: Din QuickConnect-anslutning kommer att överföras via Synology Relay Server när direkt QuickConnect-åtkomst till DiskStation inte är möjligt under aktuell nätverksmiljö.
    • Skapa automatiskt regler för vidarebefordring av port: När din DiskStation finns under en UPnP-aktiverad router blir routern informerad om att skapa regler för vidarebefordring av port för QuickConnect.
  3. Klicka på Tillämpa.

Obs:

  • Överförd QuickConnect-anslutning kan vara långsam på grund av längre nätverksfördröjning.
  • När routern tillåts skapa regler för vidarebefordring av port för QuickConnect kan DiskStation bli utsatt för en säkerhetsrisk.

Att aktivera/avaktivera QuickConnect för specifika applikationer/tjänster:

  1. Gå till Kontrollpanel > QuickConnect > Avancerat.
  2. Välj de applikationer/tjänster du vill aktivera QuickConnect för.
  3. Klicka på Tillämpa.

Obs:

För bättre QuickConnect-prestanda rekommenderas att du går till Kontrollpanel > Extern åtkomst > Konfiguration av router för att konfigurera vidarebefordring av port för varje tjänst:

  • DSM: 5000 (HTTP); 5001 (HTTPS)
  • Photo Station: 80 (HTTP); 443 (HTTPS)
  • Cloud Station：6690

Felsökning

Om du ser meddelandet Ett nätverksfel inträffade betyder det att tjänsten QuickConnect har avbrutits på grund av nätverksfel. Kontrollera följande:

  • Att din DiskStation är anslutet till ett aktivt nätverk (i Kontrollpanel > Nätverk > Nätverksgränssnitt).
  • Att DNS-server och standardinställningar för gateway är korrekt konfigurerade (i Kontrollpanel > Nätverk > Allmänt).
  • Tillse att DiskStation har åtkomst till Internet med användning av följande målportar: 80, 443, 8888.

När nätverksfelen är avhjälpta bör QuickConnect-servicen återupptas automatiskt inom några minuter.
