Aktivera papperskorgen

När du aktiverar en delad mapps papperskorg skapas en undermapp för att lagra borttagna filer. Du kan återhämta borttagna filer från papperskorgen. Töm papperskorgen om du vill ta bort filerna permanent.

Så här gör du för att aktivera papperskorgen

  1. Välj en delad mapp och klicka sedan på Redigera.
  2. Markera kryssrutan Aktivera papperskorgen.
  3. Markera Begränsar åtkomst endast till administratörer om du vill att papperskorgen endast ska vara tillgänglig för administratörer.
  4. Klicka på OK för att spara inställningarna.

Så här gör du för att tömma papperskorgen

klicka på knappen Töm papperskorgen för att ta bort alla borttagna filer i papperskorgen permanent.

Så här gör du för att tömma alla papperskorgar

Välj Töm alla papperskorgar i rullgardinsmenyn Åtgärd för att ta bort alla borttagna filer i papperskorgarna permanent.

Obs: Du kan konfigurera papperskorgs-mappens åtkomsträttigheter i File Station genom att högerklicka på mappen #papperskorg. Välj Egenskaper och klicka på fliken Behörighet.
