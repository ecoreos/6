Anslutningsbegränsning

Tillåt bara SSL/TLS-anslutning

När den här funktionen är aktiverad accepterar servern enbart SSL- eller TLS-anslutningar från FTP-klienterna.

Begränsa antalet anslutningar per IP

När den här funktionen är aktiverad begränsar servern det maximala antalet anslutningar per IP-adress. Aktivera denna funktion och begränsning av FTP-bandbredd för att effektivt kunna hantera bandbredd.

Aktivera begränsning i FTP-bandbreddd

Aktivera denna funktion för att begränsa bandbredden för en enskild FTP-anslutning.
