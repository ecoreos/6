Domän

Din DiskStation kan ansluta till en Windows-domän. Efter att den anslutits till domänen kan användare logga in till DiskStation genom att använda deras domänkonton och -lösenord, vilket låter den få åtkomst till filer och använda program på DSM, utan att behov av att komma ihåg ytterligare en uppsättning användarnamn och lösenord.

Anslut till en domän:

  1. Gå till Kontrollpanel > Katalogtjänst > Domän.
  2. Markera rutan Anslut till domän.
  3. Ange domännamnet och specificera DNS-server i därför avsedda fält.
  4. Om nödvändigt, markera rutan Avancerade domänalternativ. (Se här nedanför för mer information.)
  5. Klicka på Tillämpa.
  6. Ange användarnamnet och lösenordet för domänserverns administratör i popup-fönstret.
  7. Klicka på OK.

DNS Server

Det rekommenderas att du anger en DNS-server för att hjälpa din DiskStation att söka efter domänen. Om domänen inte är registrerad på DNS-servern kommer proceduren att gå med i domänen kanske att misslyckas.

Domänservertyp

Det här fältet visar den typ av domän när din DiskStation ansluter till en domän. Typen av domän kommer att bli "AD-domän".

Hanteringsläge

Det här alternativet kommer att avgöra hur du hanterar privilegierna för domänanvändare och -grupper.

  • Betrodd domän: Du kan hantera användare och grupper i den anslutna domänen såväl som i betrodda domäner. Användare och grupper kan filtreras med rullgardinslistan domän.
  • Enkel domän med OU: Endast användare och grupper i den anslutna domänen kommer att visas i detta läge. Det här läget låter dig filtrera domänanvändare/domängrupper med rullgardinsmenyn OU (Organizational Unit).

Avancerade domänalternativ

I de flesta fall behöver du inte fylla i några av de avancerade domänalternativen. Avancerade domänalternativ krävs endast för specifika domänmiljöer.

  • DC IP/FQDN: Specificera en IP-adress eller FQDN för DC (Domain Controller) så kommer din DiskStation att försöka kommunicera med den. Om det finns fler än en IP-adress eller FQDN, avgränsa med ett komma (,) mellan var och en. Lägg till en asterisk (*) efter den sista IP-adressen eller FQDN för DC om du vill, så kommer din DiskStation att försöka kommunicera med andra DC om alla angivna misslyckats. Du måste lägga till ett komma mellan asterisken och den sista IP-adressen eller FQDN.
  • Domän NetBIOS-namn: Ange NetBIOS-namn för domänen.
  • Domän FQDN (DNS-namn): Ange FQDN (DNS-namn) för domänen.
  • Registrera gränssnitt för DNS: Vid anslutning till en domän kommer det NIC (Network Interface Card, på svenska nätverkskort) som specificerats här registreras med DNS-servern. Vänligen observera att för det fall värdnamnet för din DiskStation innehåller ett understreck (_) kommer registreringen att misslyckas eftersom understreck (_) inte kan användas med DNS.
  • Uppdatera användar-/grupplista: Specificera hur ofta din DiskStation automatiskt uppdaterar domänens lista över användare/grupper. Du kan anpassa tidsperioden så att uppdateringar utförs dagligen, veckovis eller månadsvis. Dessutom kan domänens användar-/grupplista uppdateras manuellt genom att gå till fliken Domänanvändare och klicka på Uppdatera alla domäner. Vänligen observera att automatiska uppdatering påverkar systemets viloläge.

Ansluta Endast läsbar domänkontrollant

För att ansluta din DiskStation till en endast läsbar domänkontrollant (RODC), behöver du specificera "hela datorns namn" (FQDN) om din läs- och skrivbara domänkontrollant (RWDC) i fältet DC IP/FQDN. Hela datorns namn ska anges i följande format: Datornamn.Domän, dvs. "ad01.synology.com".

Obs:

  • Om din domäns användarnamn inkluderar ”%” eller ”$”, kanske du inte kan komma åt din home-mapp. Be din domänadministratör om ett nytt användarnamn.
  • Domängrupperna Domänadministratörer och Företagsadministratörer läggs till den lokala gruppen Administrators automatiskt. Därför kommer domänanvändare i dessa grupper få administrativa privilegier på DIN DiskStation, inklusive åtkomst till filtjänster såsom CIFS/FTP/AFP/WebDAV och andra DSM-applikationer.
  • Du kan också konfigurera domänanvändares åtkomstprivilegier till de delade mapparna på DIN DiskStation. Se här för mer information.

Domänalternativ

Detta avsnitt innehåller följande ytterligare alternativ för domäner.

  • DiskStation synkroniserar med en NTP-server varje gång en domänanvändare loggar in: Att aktivera detta alternativ kommer att framtvinga synkronisering av tid mellan DIN DiskStation och NTP-servern.
  • Aktivera serversignering: Om klienten har klientsignering aktiverat, måste serversignering också vara aktiverat på DIN DiskStation, så att användare kan använda domänens tjänster på ett normalt sätt.
  • Hämta användar-/grupplistor i NT4-kompatibelt läge: Aktivering av detta alternativ låter systemet hämta användar-/grupplistor genom att använda läget NT4 RCP. Detta alternativ kan aktiveras om vissa av domänens användar-/grupplistor inte kan hämtas med standardinställningarna.
  • Avaktivera administratörsbehörigheter för domän- och företagsadministratörer: En användare med administratörsbehörigheter kan ha full kontroll över din DiskStation och dess filer. Aktivering av detta alternativ låter dig begränsa så att behörigheter för domän- och företagsadministratörer är desamma som för en vanlig användare och att de därför ärver alla normala behörigheter för fil/mapp.
  • Räkna upp nästlade gruppnivåer: Specificera antalet nästlade domängruppsanvändare som kan räknas upp.
