NTP Service

Du kan använda din DiskStation som en tidsserver i nätverket för att synkronisera tid mellan olika nätverksenheter och DiskStation över nätverket.

För att aktivera NTP Service:

Markera Aktivera NTP-tjänst och klicka på Tillämpa.

Obs: NTP Service krävs för Surveillance Station och High Availability-kluster. Om du har Surveillance Station eller High Availability Manager installerad och kör den på din DiskStation, kan den därför inte avaktiveras medan paketet körs.
