Delad mapp

Delade mappar är de grundläggande mapparna där du kan lagra filer och mappar på din DiskStation. Innan du lagrar några filer på din DiskStation måste du skapa minst en delad mapp i DSM. Data som lagras i delade mappar kan hållas privata eller delas med specifika användare eller grupper beroende på de anpassningsbara åtkomstbehörigheterna.

Här nedanför finns en listan över delade mappar som är standard och skapas automatiskt när särskilda applikationer, tjänster eller filpaket installeras eller aktiveras.

Namn Beskrivning
public Mappen public skapas automatiskt när din DiskStation initialt installeras.*
web Mappen web är den lagringsplats som är standard för innehållet hos en webbplats som DiskStation är värd åt. Den skapas automatiskt när funktionen Web Station aktiveras.
photo Mappen photo används för att lagra bilder och videor som du vill dela med Photo Station. Den skapas automatiskt när Photo Station eller DLNA/UPnP-mediaserver aktiveras.
music Mappen music används för att lagra musik du vill lyssna på med Audio Station. Den skapas automatiskt när Audio Station eller DLNA/UPnP Media Server aktiveras.
video Mappen video används för att lagra de videor som du vill bläddra bland med DLNA/UPnP-enheter. Den skapas automatiskt när Video Station eller DLNA/UPnP Media Server aktiveras.
surveillance Mappen surveillance används för att lagra inspelningar från Surveillance Station. Den skapas automatiskt när Surveillance Station aktiveras. Den är endast läsbar och kan endast systemadministratören har åtkomst.
home Mappen home ger ett privat utrymme för varje användare att lagra data på en plats som endast användaren kommer åt. Den skapas automatiskt när funktionen Användarhem aktiveras.
homes Mappen homes innehåller alla användares home-mappar. Den skapas automatiskt när funktionen Användarhem aktiveras. Endast systemadministratören kan komma åt och hantera alla användares home-mappar.
NetBackup Mappen NetBackupskapas automatiskt när Nätverkssäkerhetskopiering aktiveras.
usbshare Mappen usbshare[nummer] skapas automatiskt när du ansluter en extern USB-diskenhet till USB-porten på DiskStation.*
satashare Mappen satashare skapas automatiskt när du ansluter en extern eSATA-diskenhet till eSATA-porten på DiskStation.*

Obs:

  • Du kan bara redigera användares behörigheter för delade mappar i enkelt läge. Om du vill skapa användargrupper eller hantera gruppens åtkomsträttigheter till delade mappar, vänligen gå till Kontrollpanel (Avancerat läge) > Grupp.

Så här gör du för att skapa en delad mapp:

  1. Gå till Kontrollpanel (Enkelt läge) > Delad mapp och klicka på ikonen + i den vänstra panelen.
  2. Specificera ett namn och välj en volym på vilken den delade mappen ska skapas.
  3. Markera Aktivera Papperskorgen, om det behövs. Om detta alternativ är aktiverat när filer i den delade mappen tas bort, kommer de att flyttas till en mapp som heter #recycle. Åtkomst till papperskorgen kan ändras för användare som hör till gruppen administrators.
  4. Markera eller avmarkera de lämpliga rutorna för varje användare för att anpassa deras åtkomstbehörigheter till den delade mappen:
    • Läsa/skriva: Användaren kan komma åt och göra ändringar i filerna och undermapparna i de delade mapparna.
    • Endast läsbar: Användaren kan komma åt filerna och undermapparna i den delade mappen.
    • Ingen åtkomst: Användaren kan inte komma åt filerna och undermapparna i den delad mappen.
  5. Klicka på Tillämpa för att spara inställningarna.

Obs:

  • Namn på delade mappar är inte skiftlägeskänsliga och kan innehålla 1-32 tecken. Namnet kan inte innehålla följande tecken:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Följande namn är reserverade för systemanvändning:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • Det första tecknet i en delad mapps namn får inte vara ett minustecken eller ett mellanslag och det sista tecknet får inte heller vara ett mellanslagstecken.
  • När du stöter på behörighetskonflikter är behörighetsprioriteten som följer: Ingen åtkomst > Läsa/skriva > Endast läsbar.
  • Om behörigheterna för användarna som hör till gruppen administrators är satt till Ingen åtkomst, när en ny delad mapp skapas, kommer dessa användare endast kunna se de delade mapparna i Kontrollpanel > Delad mapp.
  • Du kan konfigurera papperskorgs-mappens åtkomsträttigheter i File Station genom att högerklicka på mappen Papperskorg. Välj Egenskaper och klicka på fliken Behörighet.

För att redigera en delad mapp:

  1. Gå till Kontrollpanel (Enkelt läge) > Delad mapp.
  2. Välj den önskade delade mappen i den vänstra panelen.
  3. Redigera den allmänna informationen för den delade mappen, exempelvis namn, plats och användarbehörigheter.
  4. Om Aktivera Papperskorgen redan är markerat kan du klicka på Töm Papperskorgen för att permanent ta bort alla borttagna filer från papperskorgen.
  5. Klicka på Tillämpa för att spara inställningarna.

För att ta bort en delad mapp:

  1. Gå till Kontrollpanel (Enkelt läge) > Delad mapp.
  2. Välj den önskade delade mappen i den vänstra panelen.
  3. Klicka på ikonen Skräp på den vänstra panelen för att ta bort den valda delade mappen.

____
* Endast tillgängligt för vissa modeller.
