Avancerad

Du kan ändra följande avancerade inställningar i Kontrollpanel > Säkerhet > Avancerat för bläddringssessioner i DSM:

HTTP-komprimering

Du kan aktivera HTTP-komprimering för DiskStation. Genom att sänka bandbredden för nätverkstrafiken kan hastigheten med vilken webbsidan läses in ökas.

Att aktivera / avaktivera HTTP-komprimering:

  • Markera / avmarkera HTTP-komprimering i Kontrollpanel > Säkerhet > Avancerat.

Profil för TLS / SSL

Du kan välja säkerhetsnivån på krypteringsalgoritmen för krypterade HTTP-anslutningar. Den ger 3 nivåer enligt följande:

  • Modern kompatibilitet - Endast kompatibel med de senaste webbläsarna och mobila enheterna.
  • Mellanliggande kompatibilitet - Mellan högsta och lägsta säkerhetsnivån. Det är systemets standardvärde.
  • Äldre bakåtkompatibilitet - inte säkert men stöder de flesta webbläsare och mobila enheter.
