Uppdatering av DSM

Emellanåt ger Synology ut gratis DSM-uppdateringar. Uppdateringar kan bestå av nya funktioner, funktionsförbättringar och prestandaförbättringar.

Viktigt: Uppgradering av DSM påverkar vanligtvis inte data på DiskStation. Om du vill tillförsäkra datasäkerheten rekommenderar vi dig dock att säkerhetskopiera dina data innan du börjar.

Manuell uppdatering av DSM

Du kan ladda ner uppdateringsfilen (med filtillägget .pat) från Synologys Download Center och göra en manuell uppdatering av DSM.

Så här gör du en manuell uppdatering av DSM:

  1. Besök www.synology.com för att ladda ned den senaste dsm-uppdateringen till din lokala dator. Packa upp filen för att få fram en .pat-uppdateringsfil.
  2. Klicka på Manuell uppdatering av DSM för att komma till sidan Manuell uppdatering av dsm
  3. Klicka på Bläddra för att välja uppdateringsfilen.
  4. Klicka på Tillämpa för att börja uppdatera ditt systems DSM. Systemet startas om automatiskt när uppdateringen är slutförd.

Felsökning:

Om du inte kan uppdatera DSM kan följande rekommendationer hjälpa dig att lösa ditt problem:

  • Kontrollera om uppgraderingsprocessen avbröts av användare eller andra oväntade händelser såsom ett strömavbrott.
  • Kontrollera DSM-versionen. Du kan endast uppdatera systemet med en nyare version av DSM.
  • Ladda ner dsm-uppdateringen igen eftersom uppdateringsfilen kan vara skadad.

Om problemet kvarstår, kontakta Synologys tekniska support.

Automatisk uppdatering av DSM

Till skillnad från vid manuell uppdatering kommer systemet varje gång du går till Uppdatering av fast programvara att kontrollera med Synology för att se om det finns någon ny DSM-version tillgänglig för nedladdning, och sedan visa resultatet i grön eller röd text. Om det finns en ny DSM-uppdatering att ladda ner kan du låta systemet hämta denna åt dig och sedan utföra uppdateringen.

Så här gör du för att låta systemet hämta den senaste DSM-versionen och utföra uppdateringen:

  1. Om det finns en ny DSM-uppdatering att ladda ner kan du klicka på länken Ladda ner för att låta systemet hämta denna åt dig.
  2. Systemet visar något av följande meddelanden på skrivbordet för att tala om huruvida nedladdningen lyckades eller inte: Nedladdningen av DSM är slutförd eller Nedladdningen av DSM lyckades inte.
  3. Klicka på skrivbordets meddelandefönster för att gå till Uppdatering av fast programvara.
  4. Gör något av följande:
    • Om nedladdningen är klar klickar du på Uppdatera nu för att börja uppdatera DSM omedelbart.
    • Om DSM-hämtningen misslyckades kan du klicka på Försök igen för att låta systemet hämta den senaste DSM-versionen från Synology igen.

Söka efter uppdateringar regelbundet:

  1. Markera Sök automatiskt efter uppdateringar för DSM.
  2. Välj en plan för uppdatering och konfigurera Installationsschemat.
  3. Klicka på OK.

Ett skrivbordsmeddelande visas när en uppdatering för DSM är nedladdad och klar att installeras. För att sluta uppdatera DSM automatiskt kan du avbryta det i skrivbordsmeddelandet eller genom att ändra Uppdateringsinställningar. DSM-uppdateringarna installeras innan den schemalagda avstängningen om du har aktiverat Stäng av i Schema för start/avstängning.

Felsökning:

Om du ser felmeddelanden (i röd text) vid sökning eller hämtning av den senaste DSM-versionen kanske följande förslag kan hjälpa till att lösa ditt problem:

  • Se till att din DiskStation är ansluten till Internet.
  • Synologys hämtningsserver kan vara upptagen för tillfället. Ett sätt att lösa problemet kan vara att försöka hämta igen vid ett senare tillfälle.

Om problemet kvarstår, kontakta Synologys tekniska support.
