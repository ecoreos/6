Schemaläggare för uppgifter

Schemaläggaren för uppgifter låter dig schemalägga och köra tjänster eller skript på fördefinierade tider.

Hantera uppgifter

Huvudsidan för schemaläggare för uppgifter visar alla befintliga uppgifter och tillhandahåller alternativ för att skapa, redigera eller köra uppgifter.

Skapa en schemalagd uppgift:

Schemalagda uppgifter är uppgifter som körs på specificerade datum, tider eller intervall.

  1. Gå till Kontrollpanel > Schemaläggare för uppgifter, klicka på skapa och välj Schemalagd uppgift.
  2. Välj den önskade typ av uppgift som ska schemaläggas: Användardefinierat skript, Papperskorg, Ljudkontroll eller Tjänst.
  3. När fönstret Skapa uppgift visas, klicka på Allmänt och ändra följande inställningar:
    • Allmänna inställningar: Ange ett namn för uppgiften, välj en användare som uppgiften ska köras för och aktivera eller avaktivera uppgiften.
  4. Klicka på Schema och ändra följande inställningar:
    • Datum: Specificera när du vill köra uppgiften. Uppgifter kan schemaläggas att köras vissa dagar i veckan (t.ex. dagligen, veckoslut, varje söndag, etc.) eller på specifika datum (t.ex. 2015-10-21).
    • Tid: Specificera hur många gånger uppgiften ska köras varje dag, så väl som den första och sista tiden för körning.
  5. Klicka på Inställningar för uppgifter för att ordna uppgiftsspecifika inställningar (endast för vissa uppgifter).
    • För att få körinformation för uppgiften, välj Skicka körinformation per e-post.
    • För att få meddelanden endast vid avvikelser, välj Skicka körinformation endast när skriptet avslutas avvikande.
    • För att anpassa uppgiftsinnehållet, ange skriptet under Användardefinierat skript.
    • Om du valde Papperskorg eller Tjänst i steg 2 kan du nu bestämma hur och när Papperskorgar töms eller vilken tjänst som ska startas/stoppas.
  6. Klicka på OK för att spara inställningarna.

Att skapa en utlöst uppgift:

Utlösta uppgifter är uppgifter som körs före/efter slutförandet av specifika händelser.

  1. Gå till Kontrollpanel > Schemaläggare för uppgifter, klicka på skapa och välj Utl√∂st uppgift.
  2. Välj Användardefinierat skript.
  3. När fönstret Skapa uppgift visas, klicka på Allmänt och ändra följande inställningar:
    • Allmänna inställningar: Ange ett namn för uppgiften, välj en användare som uppgiften ska köras för, välj en händelse/för-uppgift som löser ut uppgiften och aktivera eller avaktivera uppgiften.
  4. Klicka på Inställningar för uppgifter för att ordna uppgiftsspecifika inställningar (endast för vissa uppgifter).
    • För att få körinformation för uppgiften, välj Skicka körinformation per e-post.
    • För att få meddelanden endast vid avvikelser, välj Skicka körinformation endast när skriptet avslutas avvikande.
    • För att anpassa uppgiftsinnehållet, ange skriptet under Användardefinierat skript.
  5. Klicka på OK för att spara inställningarna.

Obs:

  • Vid skapandet av en utlöst uppgift behöver du eventuellt specificera dess för-uppgift(er). Den skapade uppgiften körs endast efter att för-uppgiften utförts.
  • Efter att du aktiverat Skicka körinformation endast när skriptet avslutas avvikande skickar systemet inte informationen om skriptet slutat med 0.

Redigera inställningar för uppgifter:

  1. Välj en befintlig uppgift.
  2. Klicka på Redigera.
  3. Ett fönster där du kan ändra inställningarna för uppgifter visas.

Radera en uppgift:

  1. Välj en befintlig uppgift.
  2. Klicka på Åtgärd > Radera.

Aktivera eller avaktivera en uppgift:

  1. På huvudsidan för Schemaläggare för uppgifter, markera eller avmarkera den lämpliga kryssrutan under Aktiverad för att aktivera eller avaktivera en uppgift.
  2. Klicka på Spara.

Manuellt köra en uppgift:

  1. Välj en eller flera befintliga uppgifter.
  2. Klicka på Kör.

Begränsningar för schemaläggaren för uppgifter:

  • Namn på uppgifter kan inte vara längre än 512 visningsbara tecken.
  • Maximalt antal uppgifter: 1024.
  • Maximal längd på sökväg för skript eller kommandorad: 4095.
  • Längd på pipljud kan ställas in från 1 till 600 sekunder.
