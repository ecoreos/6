Nätverksrouting

DSM tillhandahåller flera alternativ för att routa nätverk för olika krav vid nätverkskonfiguration.

För att separera nätverksgränssnittens nätverksflöde i samma LAN:

  1. Gå till Kontrollpanel > Nätverk > Avancerad.
  2. Under Allmänt, markera Svara på ARP-förfrågningar om målets IP-adress är en lokal adress konfigurerad på det inkommande gränssnittet.
  3. Klicka på Tillämpa för att spara dina inställningar.

Det här fastställer standardregeln för routning för varje nätverksgränssnitt, vilket gör att nätverkspaket från varje gränssnitt skickas ut från deras egna respektive fysiska nätverksgränssnitt.

Därutöver, om du har mer än två nätverksgränssnitt på samma LAN kan aktivering av detta alternativ orsaka nätverksflöde på varje gränssnitt för separat transport. Du kan sedan tillämpa separata brandväggsregler för varje gränssnitt.

För att routa varje nätverksgränssnitt genom deras respektive gateways:

  1. Gå till Kontrollpanel > Nätverk.
  2. Under Allmänt, markera Aktivera flera gateways.
  3. Klicka på Tillämpa för att spara dina inställningar.

Det här kommer att göra att routningen för varje nätverksgränssnitt i ditt nätverk går genom sina respektive gateways.

För att ändra standardgateway:

  1. Gå till Kontrollpanel > Nätverk.
  2. Under Nätverksgränssnitt, klicka på Tjänstordning.
  3. Ordna om dina gateways genom att välja gatewayen och klicka på Upp eller Ned.
  4. Klicka på OK.

Gatewayen högst upp i ordningen kommer att tjänstgöra som standardgateway för ditt LAN.

Att ställa in statisk routning:

  1. Gå till Kontrollpanel > Nätverk.
  2. Under Statisk route, klicka på Skapa.
  3. Ange IP-adressen för ditt mål i Målnätverk.
  4. Ange nätmasken för ditt mål i Nätmask.
  5. Ange ditt måls gatewayadress i Gateway.
  6. Välj ditt måls gränssnitt i Gränssnitt.
  7. Klicka på OK.

För att visa IP-routingtabell:

  1. Gå till Kontrollpanel > Nätverk.
  2. Under Statisk route, klicka på IP-routingtabell.
  3. Välj ett nätverksgränssnitt för att vissa dess IP-routingtabell från den övre vänstra rullgardinsmenyn.
