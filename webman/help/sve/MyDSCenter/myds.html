Synology-konto

I Synology-konto kan du enkelt ha åtkomst till och hantera inställningarna för dina tjänster för Synology-konto och eNyheter som är associerade med din DiskStation och filpaket.

  • Konto: Visa och redigera din Synology-kontoinformation.
  • Inköpshistorik: Bläddra eller sök efter inköpsaktivitet för filpaket.
  • eNyheter: Välj en eller flera kategorier med eNyheter att få från Synology och gör prenumerationsinnehållet personligt anpassat.
  • Logga ut: Avsluta alla tjänster i Synology-kontot som är associerade med din DiskStation.

Att logga in på ditt Synology-konto:

  1. Gå till Synology-konto.
  2. I pop-up Logga in på ditt Synology-konto, ange den e-postadress och det lösenord som du använde för att registrera ett Synology-konto.
    • Om du inte har ett Synology-konto, klicka på Behöver du ett Synology-konto? för registrering.
    • Om du glömmer ditt lösenord, klicka på Glömt ditt lösenord? för att få ett nytt lösenord, som skickas via e-post, utfärdat.
  3. Klicka på Logga in för att få åtkomst till tjänsterna för ditt Synology-konto.

Att visa och redigera Synology-konto:

  1. Klicka på Konto på den vänstra panelen för att visa din kontoinformation.
  2. Klicka på fältet för att ändra kontoinformationen.
  3. För att säkra inställningarna, klicka på Tillämpa.

För att se Inköpshistorik för din DiskStation:

  1. Klicka på Inköpshistorik på den vänstra panelen.
  2. Välj valfritt sätt att hitta inköpshistoriken för filpaket för din DiskStation.
    • För att göra detta via nyckelord, välj Sök efter nyckelord.
      • Ange nyckelordet och välj det lämpliga filtret.
    • För att göra detta via inköpsdatum, välj Sök efter inköpsdatum.
      • Klicka på kalenderikonerna för att specificera en tidsperiod när du har köpt din DiskStation eller filpaket.
  3. Klicka på Sök.
  4. Den omfrågade informationen om inköp av filpaket (t.ex. inköpsdatum och pris) visas.

Att prenumerera på Synology eNyheter:

  1. Klicka på eNyheter i den vänstra panelen.
  2. I Inställningar för eNyheter, markera en eller flera kategorier för eNyheter som du vill ta emot från Synology.
  3. Du kan finjustera inställningarna för eNyheter i rullgardinsmenyn:
    • Nya produkter & framtida utgåvor: Välj språk för prenumerationen.
    • Förstahandsinformation om lokala reklamevenemang: Välj ett land för att få veta vilka lokala evenemang som hålls av Synology.
    • Viktiga uppdateringar för DSM: Välj en eller flera versioner av DSM för motsvarande uppdateringar.
  4. För att säkra inställningarna, klicka på Tillämpa.

Att logga ut från Synology-konto:

  1. Klicka på knappen Logga ut i det nedre vänstra hörnet.
  2. Vid utloggning blir alla tjänster för Synology-kontot (t.ex DDNS och QuickConnect) och filpaket som är associerade med det inloggade kontot inte längre tillgängliga för din DiskStation.

Obs:

  1. När du ändrat e-postadressen skickas ett meddelande med bekräftelse till den nya adressen för att verifiera ändringen.
  2. Vid utloggning är filpaketen fortfarande installerade på din DiskStation men deras licens kommer att ha utgått.
