Översikt

Du kan kontrollera aktuell status för DSM, skanna igenom systemet och kontrollera förloppet och resultatsummeringen för skanningen. Security Advisor skannar igenom DSM och din DiskStation genom att kontrollera ett antal säkerhetsregler. Antalet och typer av regler beror på hur du konfigurerar din Baslinje för säkerhet. Din Baslinje för säkerhet avgörs av dina behov. Använder du din DiskStation för arbete och företag eller hemma och personlig användning? Du blir omfrågad att välja en Baslinje för säkerhet första gången du startar en skanning. Det kan senare ändras på sidan Avancerat.

Systemstatus

Security Advisor har totalt sju olika statusar. När skanningen är klar kommer du att se resultatsummeringen fördelad på fem kategorier. De olika statusen omfattar:

  • Klar för att skanna: Den är statusen visas när du inte har sökt igenom din DiskStation ännu. Klicka på Skanna för att påbörja en skanning.
  • Skannar: En skanning pågår. Du kan visa förloppet för skanningen. Klicka på Stopp för att sluta skanna.
  • Bra: Nuvarande skanning har slutförts. Security Advisor hittade inte några sårbara inställningar eller säkerhetsrisker. Din DiskStation är för närvarande säker.
  • Varning: Nuvarande skanning har slutförts. Beroende på en del sårbara säkerhetsinställningar rapporterade Security Advisor några låga/måttliga säkerhetsrisker. Du rekommenderas att vidta följande åtgärder för att göra din DiskStation säkrare.
  • Riskabelt: Nuvarande skanning har slutförts. Beroende på en del sårbara säkerhetsinställningar rapporterade Security Advisor några höga säkerhetsrisker. Du rekommenderas starkt att vidta följande åtgärder för att göra din DiskStation säkrare.
  • Fara: Nuvarande skanning har slutförts. Security Advisor hittade sabotageprogram på din DiskStation. Du måste vidta de rekommenderade åtgärderna för att förhindra alla ytterligare oönskade konsekvenser för din DiskStation.
  • Föråldrad: Din version av DSM är föråldrad. Du rekommenderas att uppdatera din DSM för att få de senaste funktionerna, förbättringarna och säkerhetsfixarna för att förbättra din upplevelse och hålla din DiskStation säker.

Security Advisor grupperar kontrollerna i fem kategorier:

  • Sabotageprogram: Upptäcker om det finns sabotageprogram på din DiskStation och om din DiskStation är under attack
  • System: Upptäck systemrelaterade säkerhetsinställningar.
  • Konto: Upptäck säkerhetsinställningarna för konto och lösenord.
  • Nätverk: Upptäck nätverksrelaterade säkerhetsinställningar.
  • Uppdatera: Upptäck huruvida DSM och andra installerade filpaket är aktuella, såväl som kontrollera uppdateringsrelaterade inställningar.

När skanningen väl är klar kan du se det detaljerade resultatet rapport för rapport genom att klicka på Visa resultat.
