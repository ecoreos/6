Resultat

Du kan visa detaljerad information för din slutförda skanning rapport för rapport. Den här sidan är utformad som följer:

  • Allvarsgrad: Den övergripande risknivån som Security Advisor fastställer för varje säkerhetsregel. Det finns fyra nivåer: Normal, Varning, Larm och Kritisk.
  • Beskrivning: Inställningar som Security Advisor tittar efter för en specifik regel och som kan leda till sårbarheter för din DiskStation om ingen åtgärd vidtas.
  • Senast uppdaterad: Den senaste tidpunkten Security Advisor kontrollerade den här säkerhetsregeln.
  • Status: Den aktuella statusen för varje säkerhetsregel. Security Advisor rapporterar resultatet för varje säkerhetsregel som Godkänd eller Misslyckades. Statusen för Fel, Inte kontrollerad, Körs och Hoppades över kan också alltid visas. Du kan filtrera resultaten högst upp på sidan.

För att kontrollera rapportdetaljerna för valfri säkerhetsregel:

Välj en regel och klicka på Visa. Utöver Allvarsgrad och Beskrivning innehåller varje rapport följande information:

  • Information: En kortfattad förklaring över eventuella sårbarheter, vad Security Advisor rekommenderar för att hantera dem och de fördelar du kan få från att åtgärda sårbarheten.
  • Rekommenderad åtgärd: Detaljerade instruktioner om hur de rapporterade sårbarheterna åtgärdas.
  • Länk: För alla misslyckade säkerhetsregler tar klickning på Länk dig i förekommande fall till platsen i DSM där de rekommenderade åtgärderna kan utföras.
  • Fixa: När sabotageprogram upptäcks kommer att klicka på Fixa ge instruktioner till Security Advisor att ta bort sabotageprogrammet.

För att skanna en specifik regel igen:

Välj de objekt du vill skanna igen och klicka på Skanna.
