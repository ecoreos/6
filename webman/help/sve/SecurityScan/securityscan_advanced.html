Avancerad

Du kan visa och ändra avancerade inställningar för Security Advisor.

Baslinje för säkerhet

Du kan välja vilka säkerhetsregler Security Advisor kontrollerar genom att välja Baslinje för säkerhet. Antalet och typer av regler skiljer sig åt mellan varje Baslinje för säkerhet. Gör ditt val baserat på dina behov. Använder du din DiskStation för arbete och företag eller hemma och personlig användning? Du kan välja från följande alternativ:

  • För hemmabruk och personligt bruk: Security Advisor utför väsentliga kontroller för att skydda din DiskStation mot vanliga cyberhot.
  • För arbete och företag: Security Advisor utför omfattande kontroller för att uppfylla krav på säkerhetstillämpning för organisationer.
  • Anpassad: Välj vilka säkerhetsregler du vill att Security Advisor ska kontrollera.

Skanningsschema

Du kan definiera ett regelbundet skanningsschema genom att klicka på Aktivera schema för regelbunden skanning och välja vilken tid du vill att Security Advisor ska utföra skanningen. Security Advisor kommer att kontrollera regler i enlighet med den Baslinje för säkerhet som du har valt.
