Security Advisor

Security Advisor är en säkerhetsapplikation i DSM som skannar dina inställningar i DSM och för DiskStation. Security Advisor kontrollerar dina inställningar och rekommenderar ändringar som hjälper till att hålla din DiskStation säker. Här är en lista över de funktioner den erbjuder:

  • Security Advisor utför olika kontroller för att upptäcka alla inställningar i DSM som kan utgöra en säkerhetsrisk såväl som misstänkt aktivitet.
  • Security Advisor tillhandahåller även detaljerade resultat i rapportform som förklarar de åtgärder som rekommenderas att vidta och hur du kommer att dra fördel av att göra så.
  • Den interna databasen för säkerhetsdefinitioner uppdaterar automatisk säkerhetsdefinitionerna och hålls uppdaterad om cyberhoten när de förändras.

För att starta Security Advisor, gör följande:

  • Gå till Huvudmeny > Security Advisor.
