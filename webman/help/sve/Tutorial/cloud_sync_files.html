Synka filer mellan Synology NAS och din dator med Cloud Station

Översikt

Cloud Station för Synology NAS är en applikation för att synka filer som låter dig enkelt synkronisera filer på din Synology NAS med andra enheter såsom datorer eller mobila enheter (med DS cloud). Installera Cloud Station Server på din Synology NAS och Cloud Station Drive på din dator för att automatiskt synka filer som lagras på din dator till din Synology NAS.

Ställa in anslutning mellan din Synology NAS och din dator

  1. Gå till Paketcenter, hitta Cloud Station Server och klicka sedan på Installera.
  2. Starta Cloud Station Server.
  3. På fliken Översikt, klicka på  knappen under Cloud Station Drive för att ladda ner Cloud Station Drive för ditt operativsystem. (Du kan också ladda ned Cloud Station från Synology:s nedladdningscenter.)
  4. Starta installationsprogrammet på din dator och följ instruktionerna på skärmen.
  5. Efter installation, kör Cloud Station på din dator. Klicka på Starta nu.
  6. Ange din adressen till den Synology NAS som kör Cloud Station (eller QuickConnect ID), användarnamn och lösenord. Du kan klicka på sökikonen till höger för att söka efter IP-adresser inom ditt LAN. Klicka på Nästa.
  7. Välj de mappar du vill synkronisera på din Synology NAS och din dator genom att klicka på ikonerna Redigera. Du kan sedan klicka på Avancerat för att konfigurera synkinställningarna
  8. Avmarkera alla mappar som du inte vill synka.
  9. Välj en maximal filstorlek eller svartlista filnamn eller filtyper så att de inte synkas.
  10. Välj om du vill synka dubbelriktat eller bara ladda ner data från din Synology NAS.
  11. Klicka på Klart för att slutföra installationen.
  12. Du finner ikonen för Cloud Station i ditt systemverktygsfält.
  13. Klicka på ikonen för verktygsfältsmeny för att öppna verktygsfältsmenyn. Här kan du visa förloppet och status för dina filer. Här kan du visa förloppet och status för dina filer. Klicka på Huvudapp för att starta Cloud Station.
  14. Vid start kommer du att få se några tips om att använda Cloud Station. Klicka på högerpilen för att se fler tips eller klicka på Självstudier online för mer information. Om du inte behöver tipsen, markera rutan bredvid Visa inte det här igen.
  15. Nu kan du hantera dina synkuppgifter i Cloud Station Drive.

Ladda ner en tidigare version av en synkad fil

  1. Klicka på ikonen Cloud Station Drive i ditt systemverktygsfält och klicka på mappens ikon.
  2. Högerklicka på din valda fil och välj Synology Cloud Station > Bläddra i tidigare versioner.
  3. Hitta den version du vill ladda ner och klicka på ikonen för nerladdning.

Ladda ner eller återställ en tidigare version av en synkad fil från Cloud Station Server

  1. Gå till Cloud Station Server > Versionshistorik och lokalisera filen eller mappen som du vill återställa.
  2. Välj den fil du vill hämta och klicka på Åtgärd > Bläddra i tidigare versioner.
  3. Välj den version av filen som du vill ladda ner och klicka på Ladda ner. Eller, om du är säker på att du vill ha den valda versionen återställd, klicka på Återställ och den skriver över din aktuella version.

Viktiga noteringar

  1. Behåll alltid historiska versioner för det fall en fil tas bort eller skrivs över av misstag.
  2. I Globala inställningar kan du konfigurera om en fil ska hämtas igen från din NAS eller raderas på din NAS när du raderar en fil på din dator.

Synka data till dina mobila enheter

DS cloud finns tillgänglig gratis i Apple App Store och Google Play Store. Du kan även skanna QR-koden här nedanför, för att ladda ner direkt.
