Säkerhetskopiera data på min PC/Mac med Cloud Station

Översikt

Cloud Station Server är inte bara en sofistikerad mjukvara för synkning mellan plattformar, användare kan använda den som ett smartare sätt att säkerhetskopiera sina persondatorer. Cloud Station Backup är en lösning för säkerhetskopiering i realtid som är omedelbar och sparar bandbredd samt ger bättre skydd med upp till 32 historiska versioner av en enstaka fil.

Konfigurera Cloud Station Server på Synology NAS

  1. Gå till Paketcenter, hitta Cloud Station Server och klicka sedan på Installera.
  2. Starta Cloud Station Server.
  3. Gå till Översikt och klicka på Ladda ned nu under Cloud Station Backup. (Du kan också ladda ner Cloud Station Backup  från Synology Nerladdningscenter.)
  4. Starta installationsprogrammet på din dator och följ instruktionerna för att installera Cloud Station Backup.
  5. Efter installation, kör Cloud Station Backup på din dator. Klicka på Starta nu.
  6. Ange adressen till den Synology NAS som kör Cloud Station Server (eller QuickConnect ID), användarnamn och lösenord. (Du kan även klicka på sökikonen till höger så kommer Cloud Station Backup automatiskt att söka efter andra Synology NAS inom ditt LAN). Klicka på Nästa.
  7. Välj din säkerhetskopieringskälla och avmarkera alla undermappar som du inte vill synka.
  8. Välj en mapp på din Synology NAS genom att klicka på Välj. Cloud Station Backup skapar en säkerhetskopieringsmapp med ditt datornamn på ditt valda mål.
  9. Klicka på Säkerhetskopieringsregler om du vill ställa in filter eller regler för synkningen.
  10. Klicka på Nästa för att granska dina inställningar för säkerhetskopiering, klicka sedan på  Klar.
  11. Du kan visa förloppet för din säkerhetskopiering i huvudappen.

Ladda ner eller återställ en tidigare version av en synkad fil från Cloud Station Server

  1. Gå till Cloud Station Server > Versionshistorik och lokalisera filen eller mappen som du vill återställa.
  2. Välj den fil du vill återställa och klicka på Åtgärd. Klicka på Bläddra i tidigare versioner. Om en fil oavsiktligt raderats eller flyttas på din dator kan du klicka på Återställ för att återställa den.
  3. Välj den version av filen som du vill ladda ner och klicka på Ladda ner. Eller, om du är säker på att du vill ha den valda versionen återställd, klicka på Återställ och den skriver över din aktuella version.
