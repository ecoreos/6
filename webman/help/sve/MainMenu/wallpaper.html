Skrivbord

Du kan anpassa utseendet på ditt skrivbord.

Så här gör du för att ändra skrivbordets bakgrundsfärg och textfärg:

  1. Klicka på Anpassa färg och använd rullgardinsmenyerna för att ändra textfärg för skrivbordets miniatyrbilder och bakgrundsfärg för skrivbordet i DSM.
  2. Klicka på OK.

Ladda upp en anpassad bild som ska användas som skrivbordsbakgrund:

  1. Markera Anpassa skrivbordsbakgrund och gör sedan något av följande för att välja din bakgrundsbild:
    • Om du vill använda en bild som är lagrad på din DiskStation, klicka på Välj bild för att välja bilden och klicka sedan på Välj.
    • Om du vill använda den standardbakgrundsbild som Synology tillhandahåller eller välja en bild som har använts som bakgrundsbild tidigare, klicka på Välj bild, välj en bild i Mina bilder och klicka sedan på Välj.
    • Om du vill använda en bild som finns lagrad på din dator, klicka på Välj bild och klicka sedan på Ladda upp för att välja bilden.
  2. Använd rullgardinsmenyn för att bestämma hur bakgrundsbilden ska arrangeras på din DSM desktop.
  3. Klicka på OK.
