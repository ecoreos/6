E-postkonto

Du kan lägga till dina e-postkonto i Inst. > E-postkonto och skicka filer som lagras i File Station som bilagor från e-postadresser som läggs till här.

E-postleverantörer som stöds är Gmail, Outlook, Yahoo!, AOL och anpassad SMTP-server.

För att lägga till ett e-postkonto:

  1. Klicka på Lägg till och välj tjänst för det e-postkonto du vill konfigurera.
  2. Ange din e-postadress, ett lösenord och ditt avsändarnamn.
  3. Markera Använd det här e-postkontot som standard om du föredrar att låta File Station använda det här e-postkontot som standard, när du har flera konton konfigurerade.
  4. Klicka på Synka kontakter för att synka dina e-postkontakter till DiskStation om din e-posttjänstleverantör (Gmail, Outlook och Yahoo!) har stöd för denna funktion. Vänligen observera att kontakterna som synkas kommer att vara från det e-postkonto du tillhandahåller i fönstret för autentisering som visas och inte från det du angav i det tidigare steget. Därför, om du finner att ett annat eller oavsett konto visas i fönstret för autentisering, vänligen byt manuellt till det önskade e-postkontot eller något som du tidigare angivit.
  5. Klicka på Tillämpa för att spara inställningarna.

För att redigera ett e-postkonto:

Välj det valda e-postkontot från listan och klicka sedan på Redigera.

För att ta bort ett e-postkonto:

Välj det valda e-postkontot från listan och klicka sedan på Ta bort.

Obs:

  • Standardkontot kan inte tas bort, förutom om det inte finns några andra befintliga konton. Om du vill ta bort ett e-postkonto som för närvarande är satt till standard, vänligen ändra ditt standardkonto till ett annat innan fortsättning.

För att ändra standardkontot:

Välj det e-postkonto du vill ställa in som standard från listan och klicka sedan på Ställ in som standard.

För att skicka filer som bilagor:

  1. Högerklicka på valfria filer i File Station och välj Skicka som e-postbilagor.
  2. Om du har lagt till mer än ett e-postkonto kan du välja din önskade e-postadress i rullgardinsmenyn för att skicka filerna till en specificerad mottagare.
