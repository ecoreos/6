HDD/SSD

Detta avsnitt låter dig övervaka statusen för den HDD/SSD som finns installerad på din DiskStation och tillhandahåller alternativ för att hantera och analysera prestanda och hälsa för HDD/SSD. Gå till Lagringshanterare > HDD/SSD för att hantera dessa alternativ.

För att kontrollera information för HDD/SSD:

  1. Gå till fliken HDD/SSD.
  2. Välj en disk från listan och expandera den. Här kommer du att se status för S.M.A.R.T., volyminformation, modellnamn, serienummer, version av fast programvara och temperatur.

För att kontrollera hälsoinformation:

  1. Gå till fliken HDD/SSD.
  2. Välj en disk.
  3. Kicka på Hälsoinformation.
  4. Gå till fliken Översikt. Här får du reda på hur många timmar disken har varit igång, aktuell temperatur, antal återanslutningar/återidentifieringar, antal skadade sektorer, och övergripande status för S.M.A.R.T.
  5. Gå till S.M.A.R.T. Info. Siffrorna som listas kan hjälpa dig att ytterligare diagnostisera statusen för din disk.

Obs:

SAS-diskar har inte stöd för S.M.A.R.T.-info.

Starta ett S.M.A.R.T.-test:

S.M.A.R.T. Test undersöker och rapporterar status för din diskar och varnar dig vid eventuella diskproblem. Vi rekommenderar att omgående byta diskar om fel upptäcks.

  1. Gå till fliken HDD/SSD.
  2. Välj en disk.
  3. Klicka på S.M.A.R.T. Test.
  4. Välj en typ av test och klicka på Starta för att börja testet.
  5. Det senaste testresultatet visas i avsnittet Testresultat.

Obs:

Tiden för att utföra testet varierar beroende på din systemmiljö.

Schemalägga ett S.M.A.R.T.-test:

S.M.A.R.T-test kan schemaläggas att köras automatiskt genom att skapa uppgifter.

  1. Gå till fliken Schemaläggare för test.
  2. Klicka på Skapa för att göra en ny uppgift.
  3. Ändra följande inställningar på fliken Allmänt:
    • Uppgift: Ange ett namn för uppgiften.
    • S.M.A.R.T. Testtyp: Välj att utföra ett snabbtest eller ett utökat test.
    • S.M.A.R.T. Testintervall: Välj vilka HDD/SSD som ska testas var gång S.M.A.R.T.-test utförs.
  4. På fliken Schema, ändra följande inställningar:
    • Datum: Specificera när S.M.A.R.T.-test ska utföras. S.M.A.R.T. tests kan schemaläggas att köras vissa dagar i veckan (t.ex. dagligen, veckoslut, varje söndag, etc.) eller på specifika datum (t.ex. 2015-10-21).
    • Tid: Specificera hur många gånger S.M.A.R.T.-test ska köras, så väl som den första och sista tiden för körning.
  5. Klicka på OK för att slutföra.

För att utföra Secure Erase:

  1. Gå till fliken HDD/SSD.
  2. Välj en enhet och klicka på Secure Erase.

Obs:

  • Secure Erase kan inte utföras på enheter i utökningsenheter.
  • Secure Erase kommer permanent att utplåna all data som lagras på din enhet. Vänligen säkerställ att den valda disken inte används och att data har säkerhetskopierats innan detta görs.
  • När systemet utför Secure Erase kommer den valda disken att temporärt låsas och sedan automatiskt låsas upp när åtgärden är slutförd.
  • Om Secure Erase stoppas på ett felaktigt sätt under åtgärden kan du köra Secure Erase igen, direkt i användargränssnittet. Om enheterna är borttagna eller används någon annanstans när de fortfarande är låsta, vänligen ange lösenordet ”synology” för att låsa upp enheterna.

Aktivera stöd för skrivcache för HDD

Aktivering av stöd för skrivcache förbättrar prestanda för din DiskStation. Endast vissa HDD-modeller har stöd för denna funktion. För att säkerställa dataskyddet vid användning av stöd för skrivcache rekommenderas användning av en UPS-enhet. Vi rekommenderar också att stänga av systemet ordentligt efter varje användning. Genom att avaktivera skrivcache minskar risken för dataförlust orsakad av onormala strömavbrott men systemets prestanda minskar.

Att aktivera stöd för skrivcache:

  1. Gå till fliken Cache-hantering.
  2. Markera Aktivera skrivcache. Som standard är denna funktion aktiverad.
  3. Klicka på Tillämpa.
