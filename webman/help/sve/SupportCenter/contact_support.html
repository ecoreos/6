Kontakta Support

Synology Supportcenter är den kunskapsresurs som kommer att ge support vid problem med din DiskStation under garantiperioden. Vi finns här för att säkerställa och upprätthålla optimal prestanda för din DiskStation. För att använda denna tjänst och skapa ett supportärende, gå till Supportcenter > Kontakta Support.

Läs detta innan supportärendet skapas:

  • För att spara tid åt dig rekommenderar vi att du söker igenom Kunskapsdatabas innan ett nytt supportärende skapas, eftersom det är troligt att din fråga redan har ställts och att vi har skapat en artikel om det.
  • Du kan använda antingen ditt Synology-konto eller föredragen e-postadress för att starta ett nytt supportärende från DSM.
  • Alla obligatoriska fält måste fyllas i för det nya supportärendet.
  • För att undvika missförstånd, vänligen skicka in supportärendet antingen på engelska eller kinesiska.

Tips för att skapa ett nytt supportärende:

  • Beskriv problemet som du rapporterar och ta med alla åtgärder du själv har vidtagit för att felsöka. Ta med information som kommer att hjälpa tekniska supporttekniker från Synology att förstå problemet. Om så är möjligt, bifoga numrerade steg med tillhörande skärmdumpar som kan återskapa situationen.
  • Vårt supportsystem kommer att ta bort bilagor som är större än 20 MB.

Vad som kan förväntas när supportärendet skickats:

  • En teknisk supporttekniker från Synology kommer att kontakta dig inom kort efter att supportärendet skickats.
  • Om du inte hört från oss är det mycket viktigt att du tillser att dessa e-postmeddelanden inte blockeras av säkerhetsmjukvara på din dator eller i ditt nätverk.
  • Alla supportärendet har ett unikt ID och detta ID återfinns ärendet på det e-postmeddelandets som skickas som svar på supportärendet.
  • Vår tekniska supporttekniker kan be dig aktivera fjärråtkomst på din DiskStation för att kunna köra ytterligare diagnostik, från ett Synologykontor. Vänligen se Supporttjänster för detaljerad information.
