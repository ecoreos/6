Supporttjänster

Supporttjänster inkluderar verktyg för både fjärråtkomst och loggframställning, vilket låter våra tekniska supporttekniker att snabbt granska, diagnostisera och lösa tekniska problem på din DiskStation, från ett Synologykontor. För att göra detta, gå till Supportcenter > Supporttjänster.

För att aktivera fjärråtkomst:

Aktivera fjärråtkomst på din DiskStation endast när du blir ombedd att göra så av teknisk supporttekniker från Synology.

  1. Markera kryssrutan Aktivera fjärråtkomst och klicka sedan på Tillämpa för att ändringen ska börja gälla.
  2. Ge teknisk supporttekniker från Synology lösenordet för admin och identifieringsnyckeln för support per e-post.

Obs!

  • Fjärråtkomst kommer att automatiskt inaktiveras efter 14 dagar. Den ska inte manuellt inaktiveras såvida inte en teknisk supporttekniker från Synology ber dig göra detta.
  • Vänligen avlägsna allt innehåll som du anser vara privat från din DiskStation.
  • Även om Synology inte kan garantera att åtgärden fjärråtkomst kommer att lösa ditt problem kommer Synology att vidta rimliga åtgärder för att utföra supporttjänster på ett professionellt sätt.

För att generera loggfiler:

Generera och tillhandahåll loggfiler endast till teknisk supporttekniker från Synology.

  1. Klicka på Generera loggfiler och vänta sedan i fem minuter, tills nedladdningen är slutförd.
  2. Ge teknisk supporttekniker från Synology loggfilerna per e-post.

Saker du bör veta om våra Supporttjänster

  • Vi respekterar din DiskStation som vore den vår egen.
  • Din DiskStation kommer endast att användas i syfte att diagnostisera systemet. Synology kommer inte att använda eller yppa någon annan personlig information på din DiskStation.
