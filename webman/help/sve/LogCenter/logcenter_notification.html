Meddelande

Du kan aktivera regler för att automatiskt meddelade dig när specifika loggrelaterade händelser inträffar. Till exempel kan meddelande utlösas per allvarlighetsgrad, antal loggfiler per sekund eller när loggfiler innehåll fördefinierade nyckelord.

Obs!

Innan aktivering av denna funktion, vänligen gå till Kontrollpanel > Meddelande > E-post och ställ in e-postmeddelanden. Meddelanden från Loggcentralen skickas till adress som specificerats där.

För att aktivera regler för meddelanden:

  1. Gå till Loggcentral > Meddelanden.
  2. Markera rutor bredvid de regler du vill aktivera.
  3. Specificera regelns kriteria.
  4. Klicka på Tillämpa för att spara inställningarna.
