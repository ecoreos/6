Mottagning av logg

Din DiskStation kan agera som syslog-server för att kunna ta emot loggfiler från klientenheter över nätverket. Tre format har stöd för mottagning av loggfiler: BSD (RFC 3164), IETF (RFC 5424) och Anpassad. Alla mottagna loggfiler lagras i loggfilsdatabasen.

Innan du börjar:

Gå till Loggcentral > Arkivinställningar och välj en plats för att lagra och arkivera loggfiler.

Inställning för mottagning av standardloggfiler:

  1. Gå till Loggcentral > Mottagning av logg > Standard.
  2. Du kan aktivera mottagning av loggfiler med BSD och IEFT genom att markera alternativet Ta emot loggfiler från andra enheter i de(t) format du behöver.
  3. Välj UDP eller TCP från Överföringsprotokoll.
  4. Specificera ett portnummer för att ta emot syslogmeddelanden i Port. Standardportnumret är 514.
  5. Om du vill säkra överföringen av loggfiler, markera alternativet Aktivera säker anslutning (SSL). Det här alternativet är endast tillgängligt när TCP är valt.
  6. Klicka på Tillämpa för att spara inställningarna.

Inställning för mottagning av anpassade loggfiler:

  1. Gå till Loggcentral > Mottagning av logg > Anpassad.
  2. Klicka på Skapa.
  3. Ange ett unikt namn för regeln i Namn.
  4. Ange en regel för parsning i Parameter för regel. Se anteckningen här nedanför för detaljerad information om att skapa parser för mönster.
  5. Välj UDP eller TCP från Överföringsprotokoll.
  6. Specificera ett portnummer för att ta emot syslogmeddelanden i Port. Standardportnumret är 514.
  7. Om du vill säkra överföringen av loggfiler, markera alternativet Aktivera säker anslutning (SSL). Det här alternativet är endast tillgängligt när TCP är valt.
  8. Klicka på OK för att spara inställningarna.

Obs:

Hitta detaljerad information om att skapa parser för mönster här. Du kan också ladda ned ett exempel på mönster för referens.

Att hantera anpassade regler:

  1. Välj en befintlig anpassad regel.
  2. Välj från följande:
    • Redigera: Redigera vald anpassad regel.
    • Radera: Radera vald anpassad regel.
    • Aktivera: Aktivera vald anpassad regel.
    • Avaktivera: Avaktivera vald anpassad regel.

Exportera certifikat

Om du har markerat alternativet Aktivera säker anslutning (SSL) kan du klicka på knappen Exportera certifikat för att exportera certifikatet som användas för att autentisera andra enheter vid sändning av loggfiler till din DiskStation.
