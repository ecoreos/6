Tillgänglighet

Med den inbyggda funktionen för tillgänglighet ger DiskStation Manager hörselvägledning för att användare med försämrad syn ska kunna navigera och hantera tjänster/inställningar i DSM.

Konfiguration av tillgänglighet i DSM

För att maximera tillgängligheten föreslår vi att driftmiljön konfigureras enligt nedanstående:

  • Installera skärmläsaren NVDA (finns bara för Windowsmiljön) på din lokala dator.
  • Installera den senaste versionen av webbläsaren Mozilla Firefox.
  • Använd headsets eller högtalare för att justera volymen så att den passar dina behov.
  • Använd NVDA i fokusläge (tryck på tangenterna NVDA + mellanslag för lägesbyte). Standardtangent för NVDA är Insert.
  • Navigera i DSM med följande snabbtangenter:
    Snabbtangent Beskrivning
    Tabb Byter till andra avsnitt av användargränssnittet
    Piltangenter Byter till objekt inom samma avsnitt av användargränssnittet
    Enter Öppnar applikationer/fönster
    Esc Stänger applikationer/fönster

Funktionen för tillgänglighet fungerar nu i följande områden av DSM:

  • Inloggningssidan för DSM, skrivbord och aktivitetsfält
  • Huvudmeny
  • Kontrollpanel
  • File Station

Obs:

För information om att använda NVDA, vänligen se dess bruksanvisning.

Vägledning om tangentbordsgenvägar i DSM

Tillsammans med funktionen för tillgänglighet erbjuder DSM en omfattande uppsättning tangentbordsgenvägar för att implementera en jämn och strömlinjeformad hantering. Tryck på Shift + ? för att få reda på genvägarna:

  • Tangentbordsgenvägar för skrivbord: Snabbtangenter för fönsteranvändning med mera
  • Tangentbordsgenvägar för komponent: Snabbtangenter för att styra komponenter i användargränssnittet (t.ex. listor, knappar och sökfält)

För att upphäva skrivbordsrelaterade genvägar, avbryt alternativet Aktivera tangentbordsgenvägar för skrivbord.
