Återställ

Gå till Säkerhetskopiera & Återställ och klicka på Återställ i det nedre vänstra hörnet för att återställa säkerhetskopierat data.

Att återställa säkerhetskopierad data/LUN:

  1. Klicka på Återställ i det nedre vänstra hörnet, välj Data eller LUN* och välj en säkerhetskopieringsuppgift att återställa.
  2. Du blir uppmanad att välja systemkonfiguration, olika versioner av säkerhetskopierad data, eller annat. Det beror på vilken typ av säkerhetskopieringsuppgift du vill återställa.
  3. Om säkerhetskopieringsuppgiften är krypterad behöver du lösenordet/krypteringsnyckeln för att återställa med lyckat resultat.
  4. Följ guiden för att slutföra återställningen.

Obs:

  • Vid återställning av ett oinstallerat filpaket installerar systemet automatiskt samma eller senaste versionen av applikationen. Om applikationen är installerad men dess version är äldre än den säkerhetskopierade kommer systemet automatiskt att uppgradera den till den ursprungliga säkerhetskopierade versionen.
  • Applikationer som säkerhetskopierats i DSM 5.1 eller över kan inte återställas i DSM 5.0 eller lägre.

Obs:

  • Efter att återställningen är slutförd återställs LUNs på både blocknivå och filnivå till LUNs på filnivå.*
  • Vid återställning av ett filpaket när ett filpaket inte är installerat kommer systemet automatiskt att installera samma eller senaste versionen av applikationen. Om applikationen är installerad men dess version är äldre än den säkerhetskopierade kommer systemet automatiskt att uppgradera den till den ursprungliga säkerhetskopierade versionen.
  • Applikationer som säkerhetskopierats i DSM 5.1 eller över kan inte återställas i DSM 5.0 eller lägre.

För att återställa en enstaka fil eller mapp för flera versioner av säkerhetskopior:

  1. Välj den önskade säkerhetskopieringsuppgiften i den vänstra panelen och starta Säkerhetskopieringsutforskare (genom att klicka på ikonen med ett förstoringsglas). Lösenordet/krypteringsnyckeln behövs om säkerhetskopieringsuppgiften är krypterad.
  2. Lokalisera den önskade versionen av säkerhetskopia med tidslinjefältet eller via ikonen kalender.
  3. Välj en fil eller mapp som finns i den och klicka på Återställ, Kopiera till... eller Ladda ner för att hämta den valda filen/mappen. De tre alternativen är även tillgängliga om du högerklickar på filen/mappen.

Söka efter en specifik version:

För att snabba upp hastigheten för filsökning kan du använda nyckelord för att filtrera de önskade filerna/mapparna. De filer/mappar som inte matchar är dolda.

  1. Starta Säkerhetskopieringsutforskare och klicka på ikonen med en tratt i det övre högra hörnet.
  2. I rullgardinsmenyn kan du filtrera filer/mappar enligt nedanstående kriteria:
    • Nyckelord: Ange namnet på valfri fil eller mapp. Du kan använda jokertecken [*] (vilket representerar ett eller flera tecken) och [?] (som representerar ett tecken, vilket som helst) i dina nyckelord. Till exempel kan [a*n] representera [aktion], [attention] eller [an].
    • Filtyp: Välj mellan Alla, Alla mappar eller Alla filer för att förfina din sökning.
    • Storlek (KB): Välj Alla eller välj är lika med, är större än, är mindre än och sedan ange ett värde.
    • Ändringsdatum: Välj från Idag, Igår, Förra veckan eller Förra månaden eller Anpassa för att ange ett startdatum och ett slutdatum för att hitta versioner inom ett specifikt intervall.
  3. För att tillämpa dina filter, klicka på Filtrera.
  4. För att tillämpa samma filter på en annan version av säkerhetskopia, byt bara till den önskade versionen med tidslinjefältet.
  5. För att återställa din filtrerade sökning, klicka på knappen Återställ.

Obs:

  • Säkerhetskopieringsutforskare är endast tillgänglig för säkerhetskopieringsuppgifter i flera versioner.
  • Säkerhetskopieringsutforskare har begränsade funktioner när säkerhetskopieringsmålet är inställt till DSM 5.1 eller lägre.
  • Systemkonfiguration och applikationer kan endast bli återställt via Återställ.
  • Funktionen att filtrera är endast tillgänglig för okrypterade säkerhetskopieringsuppgifter.
  • Filer/mappar som hämtats med Kopiera till... eller Ladda ner förlorar sina ursprungliga inställningar för åtkomsträttigheter.

Att återställa säkerhetskopieringsuppgifter från andra säkerhetskopieringsmål:

  1. Klicka på Återställ i det nedre vänstra hörnet och välj Data eller LUN*.
  2. Du kan välja och återställa vardera typ av säkerhetskopieringsuppgift:
    • Att återställa datasäkerhetskopiering: Klicka på Data > Återställ från befintliga dokumentsamlingar och välj det önskade säkerhetskopieringsmålet för att återställa data.
    • Att återställa säkerhetskopierad LUN: Klicka på LUN*.
  3. Följ guidens instruktioner. Lösenordet/krypteringsnyckeln behövs om säkerhetskopieringsuppgiften som ska återställas är krypterad.
  4. Klicka på Tillämpa för att slutföra återställningen.

Obs:

  • Inbäddad DataStation kan bara återställa delade mappar till externa enheter med ext3/ext4.
  • Inbäddad DataStation har inte stöd för återställning av applikationer utan en extern enhet med ext3/ext4.

____
* Endast tillgängligt för vissa modeller.
