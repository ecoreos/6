Se dokument, bilder eller videor

Med File Station kan du enkelt titta på dokument, videor eller bilder.

För att byta sätt som objekt visas:

Du kan växla mellan listvy och ikonvy för att ändra hur objekt visas i huvudområdet. Klicka på knappen Listvy i det övre högra hörnet för att ordna filer i en lista eller klicka på knappen ikonvy för att visa dem med tillhörande ikoner och text.

Du kommer att se miniatyrbilder av bilder i ikonvy vid följande förhållanden:

  • Bilderna finns i indexerade mappar som listas i Kontrollpanel > Indexeringstjänst > Mediaindexering > Indexerad mapp och du har redan installerat Photo Station eller Media Server på din DiskStation-enhet.
  • Data för miniatyrbild som lagras i EXIF-poster som din digitala kamera skapat, är tillgängliga.

För att se dokument i Google docs:

Högerklicka helt enkelt på dokumentet du vill öppna och välj Se i Google Dokument.

Format som stöds:

Google Dokumentvisare stödjer följande format: doc, docx, ppt, xlsx, pdf, tif

Obs:

För att se dokument i Google Dokument, tillse att din DiskStation-enhet är fullständigt åtkomlig över Internet. Om din DiskStation finns bakom en router, se till att du har angett regel för vidarebefordring av port för DSM på Kontrollpanel > Extern åtkomst > Konfiguration av router. Standardportnumret för DSM 5000 (HTTP) och 5001 (HTTPS). Om du fortfarande har problem med att se dokument i Google Dokument kan du behöva specificera routerns port för DSM i Kontrollpanel > Extern åtkomst > Avancerad.

För att se dokument i Office Online:

Högerklicka helt enkelt på dokumentet du vill öppna och välj Se i Office Online.

Genom att använda Officevisaren för att se de Microsoft Office-dokument som din DiskStation-enhet är värd åt, kan du få åtkomst till dessa dokument med Officefunktionerna intakta, i en webbläsare, utan behov av att ha Office installerat på din dator.

Format som stöds:

Officevisaren stöder följande format:

  • Word: docx, docm, dotm, dotx
  • Excel: xlsx, xlsb, xls, xlsm
  • PowerPoint: pptx, ppsx, ppt, pps, pptm, potm, ppam, potx, ppsm

Obs:

  • För att se dokument i Office Online, tillse att du har aktiverat QuickConnect. Om inte, vänligen logga in på DSM som en användare som hör till gruppen administrators och gå till Kontrollpanel > QuickConnect.
  • Word- och PowerPoint-dokument måste vara mindre än 10 MB; Excel-dokument måste vara mindre än 5 MB.
  • Dokumentets filnamn får endast innehålla bokstäver och siffror.

Så här förhandsgranskar du bilder:

  1. Högerklicka på en bild och välj Förhandsgranskning för att öppna fönstret för Bildvisare.
  2. Håll musen längst ned på Bildvisare för att visa skjutreglaget för miniatyrer för snabb navigering.

Format som stöds:

Bildvisare stöder följande format: jpg, jpeg, jpe, bmp, gif, png, tiff, tif, arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, 3fr, erf, mos, orf, rw2, raw, psd

För att spela upp (förhandsgranska) ljudfiler:

  • För att spela upp ljudfiler: Dubbelklicka på en eller fler ljudfiler för att öppna dem med Ljudspelaren. Alternativt kan du också spela upp ljudfiler genom att högerklicka på en ljudfil och välja Spela upp.*
  • För att spela upp ljudfiler per mapp: Klicka på en enskild mapp eller tryck på Ctrl (för användare av Windows)/command (för användare av Mac) och klicka på flera mappar. Högerklicka sedan på vald(a) mapp(ar) och välj Spela upp för att öppna dem med Ljudspelaren.*

Format som stöds:

Ljudspelaren stöder följande format:

  • mp3, m4a, m4b, aac, ogg, wav, flac, ape, aiff, aif, wma

Obs:

Du kan inte spela upp mer än 1024 filer åt gången.

Så här spelar du upp videor:

  1. Högerklicka på en video och välj Spela upp för att öppna Videospelarens fönster.
  2. Videon börjar spelas upp direkt. Om inte, kontrollera att du har installerat VLC Player (Windows) på din lokala dator.

    3. Obs:

    • När du installerar VLC Media Player, se till att du har markerat Mozilla-plugin och ActiveX-plugin.
    • VLC stöds av följande webbläsare:
      Windows: Internet Explorer, Firefox, Google Chrome
    • Om VLC är installerat på din dator och du ansluter till DSM med en webbläsare som stöds stödjer videospelaren följande format: avi, dat, mkv, vob, asf, wmv, mov, qt, divx, flv, swf, rm, rmvb, mpg, mpeg, mpeg1, m1v, mpeg2, m2v, mp2, dvr-ms, m2ts, mts, m2t, ts, tp, trp, mpeg4, m4v, mp4, 3g2, 3gp, 3gp2, webm, mpe, xvid
    • Om VLC inte är installerat på din lokala dator kan videospelaren spela upp videoklipp som stöds av HTML5. Se här för HTML5-videor som stöds av olika webbläsare.
    • Om du ansluter till DSM via HTTPS-protokollet måste du importera ett certifikat utfärdat av en tillförlitlig källa (i Kontrollpanel > Säkerhet > Certifikat) för att kunna spela upp videofiler med File Station.
    • VLC har inte stöd för direktuppspelning av HTTPS.
  3. Om du har en fil med undertexter, med samma namn och i samma mapp som videon eller om din video innehåller inbäddad textning, bör du kunna se undertexter under videouppspelningen. Om du vill kan du välja en annan undertext från rullgardinsmenyn.
