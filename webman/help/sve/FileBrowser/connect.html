Fjärranslutning

I File Station kan du ansluta till publika molntjänster (Box, Dropbox, Google Drive och Microsoft OneDrive) och filservrar med ett antal olika protokoll (FTP, SFTP, WebDAV och WebDAV HTTPS).

Att ansluta till en molntjänst:

  1. Gå till Verktyg > Fjärranslutning > Anslutningskonfiguration.
  2. Välj en önskad molntjänst genom att klicka på ikonen och klicka på Nästa.
  3. I popup-fönstret, följ guidens instruktioner för autentisering av konto.
  4. När installationen är slutförd visas molntjänsten i den vänstra panelen.

Moln-specifika begränsningar:

  • Box:
    • En katalog som delas med andra samarbetspartners får ägarens användarnamn som suffix.
    • Om kataloger som delas med andra samarbetspartners använder ett identiskt namn (en katalog som har ägarens användarnamn som suffix) kommer endast en sådan katalog att visas.
    • Filer med dessa filnamnstillägg går endast att öppna via Box: .gdoc, .gsheet och .boxnote.
    • Filnamn är inte skiftlägeskänsliga.
    • För rätt inställningar av brandvägg, vänligen se den här artikeln.
  • Dropbox:
    • En Dropbox-katalog kan inte visas om den innehåller över 25000 filer.
    • Det går inte att byta namn på en Dropbox-katalog om den innehåller över 10000 filer.
    • Filer som överskrider 10 GB kan inte laddas upp.
    • En uppladdningsuppgift misslyckas om målfilen tar mer än 24 timmar att ladda upp.
    • Filnamn är inte skiftlägeskänsliga.
    • För rätt inställningar av brandvägg, vänligen se den här artikeln.
  • Google Drive:
    • Filer/mappar med en sökväg längre än 2048 tecken visas inte.
    • Filer med ett filnamn längre än 255 tecken visas inte.
    • Filer/kataloger med identiska namn kommer att markeras med individuell tid för skapandet.
    • Om du öppnar en Google-specifik fil (dvs. Google Docs och Google Slides) via File Station visas en URL för dig, för åtkomst till målfilen.
    • Mapparna Min enhet och Stjärnmärkta som visas i File Station motsvarar respektive mappar ”Min enhet” och ”Stjärnmärkta” i Google Drive.
    • För rätt inställningar av brandvägg, vänligen se den här artikeln.
  • Microsoft OneDrive:
    • OneNote-anteckningsfiler visas inte.
    • Filer som överskrider 10 GB kan inte laddas upp.
    • Filer (t.ex. .png, .jpg, .docx, .pdf) kan visa större filstorlekar (ursprunglig storlek plus storlek på miniatyrbild). Det här påverkar inte deras ursprungliga filstorlekar när de laddas ner.
    • Filnamn är inte skiftlägeskänsliga.

För att få reda på fler begränsningar föreslår vi att du besöker den officiella webbplatsen för den berörda molntjänsten.

Att ansluta till en filserver:

  1. Gå till Verktyg > Fjärranslutning > Anslutningskonfiguration.
  2. Välj ett önskat serverprotokoll genom att klicka på ikonen och klicka på Nästa.
  3. Tillhandahåll följande information för att slutföra konfigurationen:
    • Värdnamn eller IP-adress: Ange värdnamn eller IP-adress för målfilservern.
    • Port: Ange portnumret för målfilservern.
    • Användare och Lösenord: Ange den här inloggningsinformationen om målfilservern kan kräva ett konto/lösenord.
    • Teckentabell: Välj teckentabell för målfilservern så att filnamnen visas korrekt.
    • Profilnamn: Ange ett unikt namn för att enkelt identifiera den här filservern.
    • Använd WebDAV HTTPS-anslutning: Det här alternativet är bara tillgängligt när du konfigurerar ett WebDAV-protokoll och både klienten och filserver har stöd för WebDAV HTTPS.
    • Tillåt bara en anslutning per användare: Det här alternativet är bara tillgängligt när du konfigurerar ett FTP- eller SFTP-protokoll. Markera det här alternativet för att optimera anslutningsbarhet när servern har begränsningar för antalet anslutningar.
  4. Klicka på Tillämpa för att spara inställningarna. Målfilservern visas i den vänstra panelen.

Att ändra en profil för en befintlig molntjänst eller filserver:

  1. Gå till Verktyg > Fjärranslutning > Anslutningslista.
  2. Klicka på molntjänsten eller filerserven som är mål för att hantera följande:
    • Redigering: Klicka på Redigera, ändra den nuvarande profilkonfigurationen och klicka på OK.
    • Koppla ifrån: Klicka på Koppla ifrån för att avsluta anslutningen.
    • Återansluta: Klicka på Anslut för att återställa anslutning. (Alternativt, klicka på målet i vänster panel på File Station för att återansluta.)
    • Radering: Klicka på Radera och bekräfta därefter att profilen ska överges. (Alternativt, klicka på målet i vänster panel på File Station och klicka på Radera för radering.)

Obs:

  • Varje användare kan samtidigt ansluta till inte mer än 100 molntjänster och filservrar.
  • Tillse att du väljer korrekt teckentabell för servern, för att undvika oväntade fel. Unicode rekommenderas om du inte känner till serverns teckentabell.
  • Vissa funktioner i File Station (dvs. Komprimera till, Extrahera, Förhandsgranska, osv.) kan inte tillämpas på filer som lagras på molntjänster eller filservrar.
  • Individuella överföringshastigheter för varje ansluten molntjänst eller filserver är föremål för användarens inställningar för hastighetsbegränsning.
  • Varje användare är begränsad till tre samtidiga filöverföringar mellan alla anslutna molntjänster och filservrar.
  • För att konfigurera WebDAV, WebDAV HTTPS och inställningarna för proxy för molntjänster, gå till Kontrollpanel > Nätverk > Allmänt > Proxy. Om du har ändrat inställningar för proxy och varit ansluten till en molntjänst eller filserver, vänligen koppla ifrån och återanslut sedan.
  • Att ändra information för anslutna molntjänster eller filservrar påverkar inte anslutningsstatusen.
  • För att frigöra anslutningsresurser blir en molntjänst/filserver automatiskt frånkopplad om de filer den innehåller inte används på 30 minuter. Klicka på molntjänsten/filsservern för att återställa anslutningen.
  • Systemet kan inte visa filer/mappar som använder sig av följande format på namn:
    • Börjar med ._ eller .SYNOPPSDB
    • Innehåller /
    • Namngiven exakt som .    ..    '    "    .DS_Store    @eaDir    @sharebin    @tmp    .SynologyWorkingDirectory
  • Fjärranslutning stöds endast i läget högt privilegie. För att använda den här funktionen, gå till filpaketet Docker på din DSM som är värd, välj den Docker DSM som du vill aktivera krypterade delade mappar för, klicka på Redigera och markera rutan bredvid Kör container med högt privilegie.
