Montera virtuella enheter

Du kan montera virtuella enheter för att komma åt innehåll i skivbilder med File Station.

Montera en virtuell enhet:

  1. Markera en skivbild och klicka på Verktyg > Montera virtuell enhet.
  2. Klicka på Bläddra för att välja eller skapa en tom destinationsmapp på DiskStation för montering av den virtuella enheten.
  3. Markera Montera automatiskt vid start om du vill att din DiskStation ska montera den virtuella enheten vid varje systemstart eller omstart. Klicka på Montera för att fortsätta. Du kan nu navigera till destinationsmappen där den virtuella enheten är monterad och komma åt innehållet på skivbilden.

Obs:

Virtuella enheter och .iso-filer kan bara monteras till undermappar under delade mapparna.

Att avmontera en virtuell enhet:

  1. Högerklicka på destinationsmappen där den virtuella enheten är monterad och välj Avmontera.
    Obs:
  1. Skivbildsformat som stöds: ”.iso”-filer med filsystem ISO 9660 och UDF.
  2. Max antal samtidiga virtuella enheter är 16.
  3. Montering av virtuella enheter stöds endast i läget högt privilegie. För att använda den här funktionen, gå till filpaketet Docker på din DSM som är värd, välj den Docker DSM som du vill aktivera krypterade delade mappar för, klicka på Redigera och markera rutan bredvid Kör container med högt privilegie.
