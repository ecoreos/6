File Station

File Station är ett lättanvänt verktyg för filhantering, med vars hjälp DSM-användare kan komma åt sina mappar på DiskStation via Internet genom sina webbläsare. Genom att ta fram två eller flera File Stations kan du hantera alla dina DiskStation-data genom att dra och släppa dem mellan olika File Stations.
