Extrahera ett arkiv

  1. Välj ett arkiv som du vill extrahera.
  2. Välj Extrahera från menyn Åtgärd. Du kan se arkivets innehåll i Fillista.
  3. Gå till fliken Alternativ för att ange lösenordet om arkivet är lösenordsskyddat.
  4. Klicka på Mål längst ner och välj en målmapp på DiskStation i fönstret som visas.
  5. Gör något av följande:
    • För att extrahera alla filerna och mapparna, klicka på Extrahera alla.
    • För att extrahera vissa filer eller mappar i arkivet, välj dem i Fillista och klicka på Extrahera markerade.

Anm: Komprimerade filformat som stöds: .zip, .tar, .gz, .tgz, .rar, .7z, .iso (ISO 9660 + joliet).
