Redigera bilder

I File Station kan du redigera bilder med Aviary Editor som är en webbaserad extern bildredigerare.

Att redigera en bild med Aviary Editor:

  1. Bläddra till den mapp där bilden finns.
  2. Högerklicka på bilden och välj Redigera i Aviary.
  3. Aviary Editor öppnas i ett nytt läsarfönster. Du kan se din bild och redigera den med de verktyg som tillhandahålls.
  4. Efter du är klar med bildredigeringen, klicka på knappen spara i det övre verktygsfältet för att spara alla ändringar för din bild.

Obs:

  • Med Aviary Editor kan du redigera bilder i följande format: .jpg, .jpeg, .jpe, .png
  • Den maximala filstorleken som Aviary Editor tillåter är 5 MB.
  • Aviary Editor kräver att nätverksinställningarna för din DiskStation är rätt konfigurerade. För felsökning, kontrollera inställningarna här nedanför:
    • Kontrollera att det finns extern åtkomst till din DiskStation via QuickConnect (i Kontrollpanel > QuickConnect) eller att den kan hämta en extern IP-adress (i Kontrollpanel > Extern åtkomst > DDNS eller i Kontrollpanel > Nätverk > Nätverksgränssnitt > PPPoE).
    • Om din DiskStation finns bakom en router, kontrollera att regler för vidarebefordring av port har specificerats rätt (i Kontrollpanel > Extern åtkomst > Konfiguration av router).
    • Om du fortfarande har problem med att öppna Aviary Editor kan du behöva specificera routerns port för din DiskStation (i Kontrollpanel > Extern åtkomst > Avancerad).
