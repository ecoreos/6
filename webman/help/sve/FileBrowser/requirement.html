File Station Krav

För att använda alla funktioner i File Station behöver du den senaste webbläsaren, Java och Flash Player.

  • Webbläsare: Internet Explorer 8 eller senare, Firefox 3.6 eller senare, Safari 5.0 eller senare, Chrome; med AJAX och JavaScript aktiverat
  • Java (för bläddring i mappar på den lokala datorn): Java Runtime Environment (JRE) 5 eller senare
  • Flash: Flash Player 9.0.28 eller senare
