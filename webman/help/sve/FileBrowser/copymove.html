Kopiera eller flytta filer eller mappar

File Station tillhandahåller olika metoder för att överföra filer/mappar mellan två DiskStation, inom en DiskStation eller till en lokal dator.

Drag och släpp filer/mappar mellan webbläsare

Utan den komplexa installationen av överföringsprotokoll kan du ladda upp mappar/filer direkt via File Station, från en DiskStation till en annan.

För att aktivera drag och släpp mellan webbläsare:

  1. Gå till Inställningar > Allmänt > File Station.
  2. Markera Aktivera drag och släpp mellan webbläsare.
  3. Klicka på OK.
  4. Öppna två webbläsare av samma sort (t.ex. Firefox) och anslut dem till respektive DiskStation som är mål och källa.
  5. Starta File Station på båda DiskStation. Du kan nu direkt använda drag och släpp för filer/mappar mellan dem via File Station.

Obs:

  • Du kan använda drag och släpp för filer/mappar mellan två flikar i samma webbläsare.
  • Internet Explorer stöder inte drag och släpp mellan webbläsare eller mellan flikar.
  • Om flera filer/mappar är valda för drag och släpp komprimeras filerna/mapparna till ett arkiv för uppladdning.
  • Gränsen för filstorlek för drag och släpp skiljer sig åt mellan webbläsare: Safari (1 GB).

Kopiera eller flytta filer/mappar inom DiskStation eller till en lokal dator

Följ anvisningarna nedan för att flytta eller kopiera filer eller mappar inom DiskStation eller till din lokala dator.

Metod 1:

  1. Välj de filer eller mappar du vill flytta eller kopiera. (Tryck och håll nere skift eller ctrl medan du gör flera val.)
  2. Välj Klipp ut eller Kopiera från rullgardinsmenyn Åtgärd.
  3. Gå till målmappen på DiskStation eller din dator.
  4. Välj Klistra in - Skriv över eller Klistra in - Hoppa över från rullgardinsmenyn Åtgärd, beroende på om du vill skriva över filer eller mappar med samma namn i destinationsmappen eller inte.

Metod 2:

  1. Välj de filer eller mappar du vill flytta eller kopiera. (Tryck och håll nere skift eller ctrl medan du gör flera val.)
  2. Dra mapparna eller filerna från huvudavsnittet till en annan mapp i den vänstra panelen.
  3. I popup-menyn som visas väljer du Kopiera - Skriv över, Kopiera - Hoppa över, Flytta - Skriv över eller Flytta - Hoppa över för att skriva över eller hoppa över filer eller mappar med samma namn i destinationsmappen.

Metod 3

  1. Välj de filer eller mappar du vill flytta eller kopiera. (Tryck och håll nere skift eller ctrl medan du gör flera val.)
  2. Tryck Ctrl + X för att klippa eller Ctrl + C för att kopiera filerna/mapparna. Mac-användare trycker Command + X och Command + C.
  3. Gå till målmappen på DiskStation eller din dator.
  4. Tryck Ctrl + V för att klistra in filerna/mapparna. (För Mac Command + V.)
  5. I dialogen som visas, välj huruvida du vill skriva över eller hoppa över filer med samma namn i målmappen.

Tangentbordskommandon:

Följande tangentbordskommando kan komma väl till pass när du flyttar/kopierar filer och mappar i File Station:

  • F2: Döp om
  • Del: Radera
  • Ctrl + A: Välj alla
  • Ctrl + X: Klipp ut
  • Ctrl + C: Kopiera
  • Ctrl + V: Klistra in

Obs:

  • När du kopierar eller flyttar filer/mappar till en katalog på ett FAT-filsystem kommer tecknen \ *?">| i filnamnet att ersättas av –, och metadata i Mac-filer kommer att raderas.
  • Om du klipper ut filer från din lokala dator och klistrar in dem i en mapp på DiskStation kommer filerna fortfarande att vara kvar på din lokala dator.
  • Vid flytt av en fil eller mapp förblir ”ägaren” densamma. Dock ändras ”ägaren” vid kopiering av en fil eller mapp, till den användaren som kopierade den.
