Dela fillänkar

Filer eller mappar som lagras på din DiskStation kan delas snabbt och enkelt. Högerklicka helt enkelt på en mapp och välj Dela fillänkar. Detta genererar en länk och en QR-kod som kan skickas till vänner och andra användare. De kan följa länken för att ladda ned den valda filen eller mappen, oavsett om de har ett DSM-konto eller inte.

För att dela fillänkar:

  1. Välj de filer eller mappar som du vill dela.
  2. Högerklicka på en av de valda filerna eller mapparna eller klicka på menyn Åtgärd. Välj sedan Dela fillänkar.
  3. Delningslänkarna och QR-koder finns i fönstret som visas.
  4. Följande alternativ kan också ändras:
    • Giltighetsperiod: Du kan anpassa giltiga åtkomstperioder och antal åtkomster för delade länkar.
      • Anpassa giltighetsperiod: Du kan bestämma hur länge en delad länk förblir aktiv efter att den skapats. Länken kommer att bli ogiltig när den specificerade perioden förflutit.
      • Anpassa antalet giltiga åtkomster: Du kan bestämma hur många gånger det går att få åtkomst till en delad länk. Länken blir ogiltig när det specificerade antalet gånger uppnåtts.
    • Dela med DSM-användare: Du kan dela länken med DSM-användare eller grupper genom att välja dem från rullgardinsmenyn. Vänligen observera att du inte kan dela med kontot eller användargruppen gäst.
    • Dela med andra: Du kan dela lösenordsskyddade länkar med andra personer, inklusive dem som inte är användare i DSM.

För att aktivera länkar för gofile.me:

  1. Gå till Kontrollpanel > QuickConnect och säkerställ att Aktivera QuickConnect är markerad. Om du inte har installerat QuickConnect kan du göra detta på den här sidan.
  2. Klicka därefter på Avancerad och säkerställ att Fildelning är markerad.
  3. Delade filer kan laddas ned utan DSM-konto genom att använda delningslänken för gofile.me.

Anpassa sida för delad länk

File Station erbjuder olika alternativ för att stilisera sidan för delade länk.

  1. Gå till File Station > Inställningar > Delade länkar > Anpassad.
  2. Välj Anpassa sida för delad länk och klicka på Anpassade inställningar.
  3. Du kan anpassa sidan på nedanstående sätt:
    • Mall: Välj valfri standardmall.
    • Bakgrund: Klicka på miniatyrbild eller + för att välja en standardbakgrundsbild, en bild på din DiskStation eller en tidigare använd bild.
    • Logo: Klicka på miniatyrbild eller + för att välja en uppladdad bild (mindre än 1 MB), en bild på din DiskStation eller en tidigare använd bild. Logotypen kan placeras i något av de fyra hörnen.
  4. Klicka på Förhandsgranskning för att se om det behövs några justeringar.
  5. Klicka på OK för att använda inställningarna.

Obs:

  1. De anpassade inställningar tillämpas på alla delade länkar oavsett den tid de skapats.
  2. Du kan bara använda bilder i de följande formatet som bakgrundsbild/logotyp: .jpe, jpeg, .jpg, .bmp, .gif, and .png

Hantera delade länkar

Befintliga delade länkar kan hanteras genom att gå till Verktyg > Hantering av delade länkar. I fönstret som visas finns det alternativ att ändra, dela eller ta bort befintliga delade länkar.

Obs:

  1. Du kan skapa upp till 1000 delade länkar.
  2. Avsnittet Status för vardera delad länk visar en av följande:
    • Giltig: Den delade länken fungerar.
    • Har upphört att gälla: Giltighetsperioden för den delande länken har upphört att gälla.
    • Ogiltig: Den delade filen eller mappen finns inte längre eller så har du inte rätt rättigheter för åtkomst till fil eller mapp.
    • Inte giltig ännu: Den delade länken har ännu inte blivit aktiv.
