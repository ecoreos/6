Ladda upp filer eller mappar

Använd någon av följande metoder för att ladda upp filer eller mappar från din dator till DiskStation.

För att få full åtkomst till uppladdningsmetoderna som visas här nedanför, installera senaste versionen av Java Runtime Environment (JRE) och Flash Player först.

Metod 1:

  1. Från den övre delen av den vänstra panelen, välj en målmapp på DiskStation.
  2. Klicka på Ladda upp, välj Ladda upp - Hoppa över eller Ladda upp - Skriv över och välj filer eller mappar från din lokala dator.

Metod 2:

  1. Gå till Inställningar, markera Bläddra bland filer från lokal dator med File Station och uppdatera webbsidan för att se filer från din lokala dator på den nedre delen av den vänstra panelen.

    2. Obs!

    Alternativet Bläddra bland filer från lokal dator med File Station kan bara träda i kraft efter att du installerat och aktiverat JRE på din dator.

  2. Från den nedre delen av den vänstra panelen, välj en mapp på din lokala dator. Huvudavsnittet visar nu innehållet i den valda mappen.
  3. Välj de filer eller mappar du vill ha i huvudavsnittet och klicka sedan på Ladda upp.
  4. Välj en målmapp på DiskStation i fönstret som visas.
  5. I det övre högra hörnet av fönstret, välj Skriv över om du vill skriva över filer eller mappar med samma namn i målmappen. I annat fall, välj Hoppa över.
  6. Klicka på OK för att starta uppladdning.

Metod 3:

  1. Gå till Inställningar, markera Bläddra bland filer från lokal dator med File Station och uppdatera webbsidan för att se filer från din lokala dator på den nedre delen av den vänstra panelen.
  2. I den vänstra panelen, drag en mapp från din lokala dator (den nedre delen) till någon mapp på DiskStation (den övre delen).
  3. I pop-up-menyn som visas, välj huruvida du vill skriva över eller hoppa över filer/mappar med samma namn i målmappen.

Metod 4:

  1. Från den övre delen av den vänstra panelen, välj en målmapp på DiskStation.
  2. Välj filerna på din dators skrivbord eller i utforskaren (till exempel Windows Explorer eller Mac Finder) och dra och släpp dem i File Station i webbläsaren.
  3. I dialogen som visas, välj huruvida du vill skriva över eller hoppa över filer med samma namn i målmappen.

    4. Obs!

    • Metod 4 har endast stöd i specifika webbläsare: Chrome, Firefox, Safari 5.1 och högre. Det finns dock inte stöd för Mac när Bläddra bland filer från lokal dator med File Station är markerad.
    • Du kan också dra och släppa mappar eller tomma filer, men endast när Bläddra bland filer från lokal dator med File Station är markerad i de läsare som listas här ovanför. I annat fall kan du inte ladda upp mappar och tomma filer genom att använda Metod 4.
    • Om du använder Chrome för Linux och är inloggad till DSM via HTTPS-protokollet behöver du importera ett betrott certifikat (i Kontrollpanel > Säkerhet > Certifikat) för att kunna ladda upp genom att använda Metod 4.

Obs!

  1. Du kan lägga till upp till 1000 filer i din uppladdningskö.
  2. Du kan ladda upp filer större än 4GB om du använder Google Chrome, Firefox, Safari 5.1 eller senare eller när Bläddra bland filer från lokal dator med File Station är markerad. I annat fall kan endast uppladdning av filer upp till 2GB ske åt gången.
  3. I en icke-krypterad delad mapp, ska fil-/mappnamn vara inom 255 tecken (upp till omkring 80 tecken fick icke-latinska språk); i en krypterad delad mapp ska det vara inom 143 tecken (upp till omkring 47 tecken för icke-latinska språk).
  4. Namn på fil/mapp får inte innehålla följande tecken: \ / : * ? " > |.
  5. Namn på fil/mapp kan inte börja med “._”.
  6. Du kan ladda upp filer som har en storlek på 0KB endast om du har installerat JRE.

Felsökning

Om du inte kan bläddra ordentligt bland dina lokala objekt på din Mac med Safari, vänligen gör följande:

  1. På din Mac, öppna Safari och välj Inställningar från menyn Safari.
  2. Klicka på Hantera webbplatsinställningar på fliken Säkerhet.
  3. Under Java, välj din DSM-"webbplats"
  4. Välj Kör i osäkert läge.
