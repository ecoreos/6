Skicka filer som e-postbilagor

Du kan skicka filer direkt som bilagor i e-postmeddelanden i File Station genom att högerklicka på valda filer och sedan välja Skicka som e-postbilagor. Ett fönster för e-post visas med några användbara funktioner:

  • Till: Om du har synkat några kontakter för ditt e-postkonto visas en kontaktlista med synkade kontakter som är knutna till ditt e-postkonto.
  • Från: Du kan välja det e-postkonto som är standard eller ett annat tillagt e-postkonto. Kontaktlistan kommer automatiskt att byta till det valda e-postkontot.
  • Bilagor: De valda bilagorna visas längst ned i e-postmeddelandet. Du kan lägga till flera bilagor genom att använda knappen bilagor.

Obs:

  • Endast filer kan läggas till som bilagor.
  • Du kan lägga till upp till 10 filer i ett enda e-postmeddelande.
  • Den maximala totala filstorleken för alla bilagor i ett enda e-postmeddelande beror på din e-posttjänst. Om du inte är säker rekommenderar vi att begränsa den totala filstorleken för alla bilagor till 25MB.
