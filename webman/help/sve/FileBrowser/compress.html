Komprimera filer eller mappar

  1. Välj filer eller mappar från DiskStation i huvudområdet.
  2. Välj Komprimera till "(filnamn)".zip från rullgardinsmenyn Åtgärd för att börja komprimera.
  3. Om du inte kan se zip-filen när komprimeringen är färdig, välj Uppdatera i rullgardinsmenyn Åtgärd för att se filen.
  4. Om du vill byta namn på, kryptera arkivet eller arkivera det med andra komprimeringsnivåer eller -lägen, välj Lägg till i arkiv... i stället.
    Obs:
  1. Du kan inte komprimera filer eller mappar på din lokala dator med File Station.
  2. Filformat som stöds: .zip, .7z.
  3. När du väljer .zip som Arkivformat i Lägg till i arkiv... kan du välja en föredragen teckentabell för .zip-filen.
