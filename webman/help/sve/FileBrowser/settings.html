Hantera inställningar för File Station

För att hantera inställningar i File Station, öppna File Station och klicka på knappen Inställningar.

Allmänt

Här kan du aktivera File Stations loggfil för att dokumentera alla användares aktiviteter. Användaraktiviteterna inkluderar Skapa mapp, Ladda upp, Ladda ner, Radera, Döp om, Flytta, Kopiera och Egenskap inställd.

Aktivera File Station loggfil:

  1. Om du vill dokumentera alla användares aktiviteter, markera Aktivera File Station loggfil.
  2. Klicka på OK.

För att se File Stations loggfil:

  1. Klicka på Visa loggfiler, vilket leder dig till Loggcentral.
  2. Klicka på fliken Filöverföring.

För att automatiskt skriva över filer vid uppladdning med dra och släpp eller tangentbordskommandon:

Markera Skriv alltid över filer när jag drar och släpper dem i en webbläsare eller klistrar in med tangentbordskommandon om du vill att filer med samma namn automatiskt skrivs över, när filer dras och släpps från din dators skrivbord till webbläsaren eller när du klistrar in filer med tangentbordskommandon i File Station.

Att aktivera smart dra och släpp:

När du drar och släpper filer i File Station avgör den här funktionen automatiskt om överföringen är en kopierings- eller flyttningsåtgärd.

  1. Välj Aktivera smart dra och släpp.
  2. Välj valfri åtgärd som alltid ska utföras om det händelsevis finns en annan användare som samtidigt överför filer med samma namn till samma plats:
    • Skriv över: Dina överförda filer skriver över filer med samma namn.
    • Hoppa över: Dina överförda filer hoppas över och skriver inte över filer med samma namn.
  3. Klicka på OK.

Obs:

Om du vill ändra kopierings- eller flyttningsåtgärden som avgörs av smart dra och släpp, tillämpa följande snabbtangenter för att passa dina behov:

  • Att byta till en kopieringsåtgärd: Tryck på Ctrl medan dra och släpp utförs.
  • Att byta till en flyttningsåtgärd: Tryck på Shift medan dra och släpp utförs.

Bläddra bland och hantera filer från lokal dator med File Station:

  1. Markera Bläddra bland filer från lokal dator med File Station.
  2. Klicka på OK.

För att dela användarlistan i DSM med användare som inte är admin:

Det här alternativet låter användare som inte är admin få åtkomst till den fullständiga användarlistan i DSM, om de vill söka efter filer som ägs av specifika DSM-användare/-grupper eller om de vill dela fillänkar med andra DSM-användare.

  1. Välj Dela användarlistan i DSM med användare som inte är admin.
  2. Klicka på OK.

Tillämpning av standardbehörigheter för UNIX

Klicka på Avancerade inställningar och markera alternativet att tillämpa standardbehörigheter för UNIX när filer eller mappar laddas upp eller skapas. Tillämpade behörigheter är de samma som de behörigheter som appliceras av UNIX-kommandot umask. När detta alternativ är aktiverat är behörigheterna för UNIX 644 för filer och 755 för mappar. När detta alternativ är avaktiverat är behörigheterna för UNIX 666 för filer och 777 för mappar. Standardvärdet för umask är 022.

Obs:

  • För delade mappar där Windows ACL aktiverats (alla utdelningar förutom "photo" och utdelningar på externa enheter), vänligen kör kommandot chmod på din Linux- eller FTP-klient för att ändra typ av filbehörighet från Windows ACL till UNIX.
  • Aktivering av detta alternativ kan orsaka problem med inkonsekventa behörigheter mellan olika protokoll. För att undvika inkonsekvens rekommenderar vi att detta alternativ lämnas avaktiverat.

Montera/Anslutningar

Som standard kan alla användare montera fjärrmappar eller virtuella diskar. Här kan du begränsa åtkomsten till hantering av fjärrmappar eller virtuella diskar till administrators.

Obs:

Användare som inte är administrator kan endast montera fjärrmappar inom sin mapp home.

Att tillåta olika användare att montera fjärrmappar eller virtuella diskar:

  1. Välj Alla användare eller Administrators i avsnittet Fjärrmapp eller Virtuell enhet.
  2. Klicka på OK.

Att tillåta olika användare att montera servrar och cloud services:

  1. Välj från Alla användare, Administrators, eller Specifik användare.
  2. Att välja Specifik användare öppnar ett fönster som låter dig konfigurera monteringsbehörigheter för olika användare. För att aktivera åtkomst för en specifik användare, tillse att rutan är markerad i kolumnen Aktiverad. För att avaktivera åtkomst för en specifik användare, tillse att rutan är inte markerad i kolumnen Aktiverad. När det är klart, klicka på OK för att stänga det här fönstret.

    3. Obs:

    • Användare som hör till gruppen administrators kan använda den här funktionen men kan inte göra några ändringar. Användare som loggar in med guest kan varken använda den här funktionen och inte heller göra några ändringar. Behörigheter för alla andra användare kan ställas in här.
  3. Klicka på OK.

Delade länkar

Som standard kan alla användare dela fillänkar. Här kan du begränsa åtkomsthantering av delade länkar till enbart administrators.

Att tillåta olika användare att dela fillänkar:

  1. Välj Alla användare, Administrators, eller Specifika användare.
  2. Klicka på OK.

För att alltid öppna HTML-filer i textformat:

  1. Markera rutan Öppna alltid HTML-filer i textformat. Om detta alternativ är aktiverat kommer HTML-filer (dvs. filer i formaten HTM, HTML, JSP, XHTML) visas i textformat när användare öppnar dem via en fildelningslänk. Detta alternativ hjälper till att förhindra att skadliga aktörer manipulerar användares inloggningssessioner genom att använda inbäddade skript i HTML-filer.
  2. Klicka på OK.

Hastighetsbegränsning

Du kan aktivera hastighetsbegränsning för att kontrollera den maximalt användbara bandbredden för användare och grupper som överför data via File Station.

För att aktivera hastighetsgräns:

  1. Markera kryssrutan med texten Aktivera hastighetsbegränsning eller Aktivera hastighetsbegränsning i enlighet med ett schema om du vill implementera hastighetsbegränsning endast vid specifika tider. Detta är en inställning som får genomslagskraft över hela servern och kommer att åsidosätta inställningar för enskilda användare och grupper. Att välja Ingen hastighetsbegränsning avaktiverar inställningarna för hastighetsbegränsning, för alla användare och grupper.
  2. Klicka på Inställningar för hastighetsbegränsning för att ändra inställningarna för användare eller grupper. För varje användare eller grupp kan du välja en av de följande:
    • Tillämpa gruppinställningar: Inställningen för hastighetsbegränsning kommer att vara samma som för gruppen användaren tillhör.
    • Ingen hastighetsgräns: Ingen hastighetsbegränsning kommer att tillämpas när gruppmedlemmar överför filer med tjänsten.
    • Konstant: En fast, konstant hastighetsbegränsning kommer att tillämpas för användaren eller gruppmedlemmarna. Hastighetsbegränsning för uppladdning och nedladdning kan specificeras i fälten till vänster.
    • Variabel: Två olika hastighetsbegränsningar kan specificeras och tillämpas på användaren eller gruppmedlemmar, i enlighet med ett schema. Klicka på knappen Anpassa för att ändra inställningarna för hastighetsbegränsning och ställa in ett schema.
  3. Klicka på Tillämpa.

Obs:

  • För mer detaljer om dynamiken för hastighetsbegränsning för grupper, vänligen se här.
  • Vid inställning av hastighetsbegränsning för en användare kan avsnittet Resultat visa användarens faktiska hastighetsbegränsning, i enlighet med sina individuella eller sin grupps inställningar. Till exempel, Till exempel om användaren ”Achilles” har en individuell nedladdningsgräns på 100 KB/s men hör till en grupp med en nedladdningsgräns på 500 KB/s, kommer avsnittet Resultat att visa ”0/500”.

Säkerhet

Här hittar du nedanstående säkerhetsinställningar.

För att alltid öppna HTML-filer i textformat:

  1. Markera rutan Öppna alltid HTML-filer i textformat. Om detta alternativ är aktiverat kommer HTML-filer (dvs. filer i formaten HTM, HTML, JSP, XHTML) visas i textformat när de öppnas i File Station. Detta alternativ hjälper till att förhindra att skadliga aktörer manipulerar användares inloggningssessioner genom att använda inbäddade skript i HTML-filer.

    2. Obs:

    • Detta är inte en global inställning. Varje användare kan välja om detta alternativ ska aktiveras, enligt dennes personliga behov.
  2. Klicka på OK.
