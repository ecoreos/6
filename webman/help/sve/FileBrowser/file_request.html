Skapa filförfrågningar

Filförfrågningar är inbjudningar till filuppladdning vars mottagare inte är DSM-användare. Med länkar till filförfrågan kan de som inte är DSM-användare direkt ladda upp flera filer till File Station på din DiskStation utan behov av att ha motsvarande privilegier för användare och filuppladdning.

Att skapa länkar till filförfrågan:

  1. Gå till File Station och välj en mapp som mål för uppladdningen.
  2. Klicka på Åtgärd eller högerklicka på målmappen och välj Skapa filförfrågan.
  3. I popup-fönstret kan du hitta/ändra nedanstående information:
    • Filsökväg: Listar målets katalogsökväg.
    • Länk: Listar länken till filförfrågan att dela med målanvändare.
    • Ditt namn: Ange ett namn för att visa identitet på den som utfärdar förfrågan. Standardvärdet är ditt användarnamn.
    • Meddelande: Anpassa meddelandet för målanvändare.
    • Aktivera lösenordskydd: Välj det här alternativet så att endast användare med lösenordet kan ladda upp filer via den skapade länken. Specificera lösenordet i rutan här nedanför.
    • Giltighetsperiod: Klicka på den här knappen för att anpassa varaktigheten för länken till filförfrågan:
      • Ställ in stopptid: Välj det här alternativet för att per datum och tid bestämma när länken slutar fungera för filuppladdning.
      • Ställ in starttid: Välj det här alternativet för att per datum och tid bestämma när länken börjar fungera för filuppladdning.
      • Antal tillåtna åtkomster: Välj det här alternativet för att bestämma hur många gånger det går att få åtkomst till länken.
  4. Klicka på Spara för att avsluta inställningarna.

Att dela länkar till filförfrågan:

För att hitta URL till filförfrågan, gå till File Station > Verktyg > Hantering av delade länkar och dubbelklicka på den önskade länken. Du kan dela länken på något av nedanstående sätt:

  • Kopiera helt enkelt URL till målanvändare.
  • Skicka länken via standard-e-postadressen (ditt e-postkonto i DSM) eller via Mail (eller en 3:e-parts e-postapplikation). För förstagångsanvändare, följ guidens instruktioner för att skapa ett e-postkonto att använda.

Att ladda upp filer:

Om du är ombedd att ladda upp filer via länken, vänligen gör enligt följande:

  1. Öppna angiven URL på den lokala datorn eller en mobil enhet (Android eller iOS). Ange lösenord om det krävs.
  2. På sidan för uppladdning, ange ett namn som indikerar din identitet.
  3. Drag och släpp helt enkelt fil(er) som ska laddas upp eller klicka på Lägg till filer för att välja fil(er).
  4. Klicka på Ladda upp för att slutföra förfrågan.

Obs:

  • Mappar kan inte laddas upp via en länk till filförfrågan.
  • Mappar i den delade mappen ”web” kan inte specificeras som målmapp.
  • Funktionen för uppladdning skiljer sig åt mellan webbläsare:
    • Internet Explorer 8 och Internet Explorer 9 stöder inte uppladdning av filer via drag och släpp.
    • Chrome på iOS-enheter stöder endast uppladdning av bilder.
    • Safari på iOS-enheter stöder endast uppladdning av bilder och video.
    • Webbläsare på Android-enheter har inte några begränsningar beträffande filtyper.

Att hitta uppladdade filer:

Fl(er) som laddats upp av samma användare visas i en identisk mapp på målet. Som standard är mappens namn Uppladdare_Namn (Förfrågare_Namn).

Om filerna till exempel laddas upp av Watson till Holmes återfinner du filerna i mappen med namnet Watson (Holmes).

Hantera länkar till filförfrågan

Att hantera länkar till filförfrågan:

Efter skapandet av en länk till filförfrågan kan du ändra dess inställningar i File Station > Verktyg > Hantering av delade länkar.

  1. Dubbelklicka på den önskade länken eller välj länken och klicka på Redigera.
  2. I popup-fönstret, ändra de relevanta inställningarna.
  3. Klicka på Spara för att tillämpa de nya inställningarna.

Att radera länkar till filförfrågan:

  1. Gå till File Station > Verktyg > Hantering av delade länkar.
  2. Välj önskad länk(ar) och klicka på Radera. Motavarande URL(:er) för filförfrågan slutar att fungera omedelbart.

Att tilldela privilegier för filförfrågan:

Som admin eller en användare som hör till gruppen administrators kan du tilldela privilegier till användare så att de kan skapa länkar för filförfrågan.

  1. Gå till File Station > Inställningar > Delade länkar > Länkar till filförfrågningar.
  2. Välj typ av användare för att ge dem privilegier att göra filförfrågan.
  3. Klicka på OK för att spara inställningarna.
