Montera fjärrmappar

Du kan montera fjärrmappar från fjärrservrar (t.ex. nätverksdatorer eller andra Synology NAS-enheter) för att enkelt navigera och hantera de monterade mapparna via File Station.

Att montera en delad fjärrmapp i CIFS:

  1. Klicka på Verktyg > Montera fjärrmapp > CIFS Delad mapp.
  2. I fältet Mapp, specificera sökvägen till fjärrmappen i formatet \\Remote_Server\Remote_Folder, exempelvis \\192.168.0.2\share eller \\chad\cheeseburger\pictures.
  3. Om fjärrservern kräver åtkomstprivilegier till fjärrmappen, ange dina användaruppgifter för fjärrservern i fälten Kontonamn och Lösenord.
  4. Klicka på Bläddra för att välja eller skapa en tom målmapp på din DiskStation för montering av fjärrmappen.
  5. Markera Montera automatiskt vid start om du vill att DiskStation ska montera den här fjärrmappen vid varje systemstart eller omstart.
  6. Klicka på Montera för att montera fjärrmappen i målmappen. Du kan nu navigera och hantera fjärrmappar från din DiskStation.

Att montera en delad fjärrmapp i NFS:

  1. Klicka på Verktyg > Montera fjärrmapp > NFS Delad mapp.
  2. I fältet Mapp, specificera sökvägen till fjärrmappen i formatet Remote_Server:/Remote_Folder_Path, exempelvis 192.168.0.2:/volume1/share.
  3. Välj den önskade NFS-versionen och transportprotokollet.
  4. Klicka på Bläddra för att välja eller skapa en tom målmapp på din DiskStation för montering av fjärrmappen.
  5. Markera Montera automatiskt vid start om du vill att din DiskStation ska montera den här fjärrmappen vid varje systemstart eller omstart.
  6. Klicka på Montera för att montera fjärrmappen i målmappen. Du kan nu navigera och hantera fjärrmappar från din DiskStation.

Obs:

NFSv4 stöder endast TCP-protokollet.

Avmontera en fjärrmapp:

  1. Välj den fjärrmapp du vill avmontera och klicka på Verktyg > Avmontera.
  2. Alternativt, högerklicka helt enkelt på fjärrmappen och välj Avmontera.

Obs:

  • Max antal samtidiga fjärrmappar är 100.
  • Du får bara montera fjärrmappar från en fjärrserver som stöder CIFS- eller NFS-protokollet.
  • Fjärrmappar kan bara monteras till undermappar under delade mappar.
  • Montering av fjärrmappar stöds endast i läget högt privilegie. För att använda den här funktionen, gå till filpaketet Docker på din DSM som är värd, välj den Docker DSM som du vill aktivera krypterade delade mappar för, klicka på Redigera och markera rutan bredvid Kör container med högt privilegie.
