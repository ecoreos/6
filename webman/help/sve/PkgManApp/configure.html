Konfigurera Paketcenter

För att hantera vilka filpaket som är tillåtna att installeras:

  1. Under fliken Allmänt, välj från de följande alternativen för att ställa in din önskade Förtroendenivå:
    • Synology Inc.: Tillåt endast installation av filpaket som getts ut av Synology Inc.
    • Synology Inc. och betrodda utgivare: Tillåt endast installation av filpaket som getts ut av Synology Inc. och betrodda utgivare. Betrodda utgivare består av utgivare som antingen har blivit betrodda av en tredje parts gemenskap genom att lägga till källkod till filpaketet eller är betrodda av användaren via import av utgivarens certifikat.
    • Alla utgivare: Tillåt installation för alla filpaket oavsett av utgivare och integritet.
  2. Klicka på OK.

För att importera en betrodd utgivares certifikat:

  1. På fliken Certifikat, klicka på Importera.
  2. Klicka på Bläddra och välj den betrodda utgivarens certifikat.
  3. Klicka på OK.

För att uppdatera certifikatet för Synology Inc.:

När du är ansluten till Internet kommer certifikatet från Synology Inc. automatiskt att uppdateras under tiden filpaketet installeras. För att manuellt uppdatera:

  1. På fliken Certifikat, klicka på Importera.
  2. Klicka på Bläddra och välj certifikatet från Synology Inc. som kan laddas ned från Synology:s webbplats.
  3. Klicka på OK.

Så här gör du för att aktivera meddelanden om filpaketuppdateringar:

  1. Klicka på fliken Meddelande och klicka sedan på Aktivera e-postmeddelanden och/eller Aktivera meddelande på skrivbordet. Det första alternativet gör att DSM skickar e-post till dig när det finns uppdateringar, det andra alternativet gör att DSM visar ett meddelande högst upp till höger på DSM desktop.
  2. Klicka på OK.

Obs:

För att e-postmeddelanden ska börja gälla, konfigurera först DSM:s meddelandeinställningar för e-post (i Huvudmeny > Kontrollpanel > Meddelande).

För att lägga till en filpaketresurs som tillhandahålls av en tredjepartscommunity:

  1. Klicka på fliken Filpaketkällor, klicka på Lägg till och ange sedan filpaketkällans namn och plats.
    Platsen kan vara en URL som finns på Internet tillhandahållen av en tredje parts utvecklare/utgivare eller gemenskap.
  2. Klicka på OK. När filpaketkällan lagts till med lyckat resultat kommer du att se en lista över filpaket på fliken Community.
