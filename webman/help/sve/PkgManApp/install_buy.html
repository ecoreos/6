Installera eller köp filpaket

Paketcenter erbjuder ett mångsidigt urval av webbapplikationer som kan utöka funktionaliteten på din DiskStation. Med Paketcenter kan du ladda ned, installera och starta filpaket med några få enkla klick.

Artikeln här nedanför förklarar hur filpaket installeras på din DiskStation genom att använda Paketcenter. För instruktioner om enskilda filpaket, vänligen se här.

Att Installera eller köpa filpaket:

  1. Öppna Paketcenter.
  2. Gå till sidan Alla för att se alla tillgängliga filpaket.

    3. Obs:

    • Om du inte kan hitta filpaketet du letar efter kan det behövas en senare version av DSM eller bara vara tillgänglig på modeller med specifika hårdvarukrav. Kontakta utvecklaren av detta filpaket för mer information.
  3. Hitta det filpaket du vill installera och klicka på Installera. (För betalningsfilpaket, klicka på Köp eller Prova för att ansluta till Synology-konto och köpa med PayPal eller använda testversionen för utvärdering.)
  4. När filpaketet är installerat med lyckat resultat bör det visas i Huvudmeny.

Köpa filpaket:

Innan köp av några filpaket, vänligen se följande noteringar.

  • Registrera ett konto på Synology Synology-konto för att få åtkomst till fler funktioner, såsom tidigare betalningsinformation och köphistorik. Information som du delar används bara för att i framtiden erbjuda bättre service.
  • Paketcenter accepterar för närvarande bara onlinebetalning via PayPal. Om du vill begära en återbetalning, besök PayPal:s officiella webbplats.
  • Filpaketen är för närvarande prissatta i U.S. dollar.

Friskrivningsklausul för valutaomvandling:

För transaktioner som innebär valutaomvandling kan din utfärdare av ditt kredit-/kontokort komma att lägga till extra avgifter som inte avspeglas i priset som visas. Alla köp kan bli föremål för att belastas avgifter från kortets utfärdare för utländsk valuta eller gränsöverskridning. Din kortutfärdare kommer att bestämma alla tillämpliga avgifter för vilka Synology inte kan tillhandahålla ersättning. Vänligen kontakta din kredit-/kontokortsutfärdare för mer information.

Att manuellt installera filpaket:

  1. Om du vill installera ett filpaket genom att ladda upp en fil i formatet .spk (finns tillgängligt på Synology:s Nerladdningscenter eller på respektive tillverkares webbplats), klicka på Manuell installation och följ guiden för att installera det nya filpaketet.

Söka efter ett filpaket:

  1. Ange ett sökord i sökverktygsfältet längst upp i vänstra hörnet och tryck på Enter för att börja söka.

Välj hur filpaket sorteras:

Du kan ändra ordning för hur filpaket visas i Paketcenter.  Välj en kategori, klicka på listikonen i det övre högra hörnet och välj att sortera filpaketen med något av följande alternativ:

  • Sortera efter namn: Klicka för att sortera filpaketen i alfabetisk ordning.
  • Sortera efter senaste popularitet: Klicka för att sortera filpaket efter antalet senast installerade filpaket med de mest populära först.

För att dölja installera filpaket:

Välj valfri kategori och klicka på listikonen i övre högra hörnet samt välj Visa inte installerade filpaket. Detta döljer filpaket som redan blivit installerade och visar dem som ännu inte installerats, uppgraderade eller reparerade.
