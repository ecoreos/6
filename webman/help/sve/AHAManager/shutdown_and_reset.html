Stäng av och byt ut server

Instruktionerna här nedanför kommer att vägleda dig genom förfaranden för att stänga av eller byta ut en server i ett high-availability-kluster.

Att stänga av high-availability-klustret med High Availability Manager:

När du vill stänga av ett high-availability-kluster för att transportera servrar till en ny fysisk plats eller utföra underhåll, vänligen följ dessa instruktioner.

  1. Logga in på DSM.
  2. Gå till High Availability Manager.
  3. Klicka på knappen med texten Hantera.
  4. Välj Stäng av high-availability-kluster.
  5. En meddelande med bekräftelse visas. Klicka på OK för att bekräfta.

Stänga av ett high-availability-kluster manuellt:

High-availability-kluster kan också stängas av manuellt genom att använda strömbrytaren på vardera server.

  1. Tryck och håll kvar på strömknappen på den passiva servern tills den ger ifrån sig ett pipljud och vänta på att servern ska bli helt avstängd.
  2. Tryck och håll kvar på strömknappen på den aktiva servern tills den ger ifrån sig ett pipljud och vänta på att servern ska bli helt avstängd.

Obs:

  • Starta servrarna:
    När du vill starta på servrarna igen ska du starta både den aktiva och den passiva servern samtidigt (inom fem minuter efter att någon av dem startats skall den andra också startas). Om du bara slår på en av servrarna kommer den att försöka slå på den andra servern genom att skicka ett Wake-on-LAN-meddelande.
  • Onormal avstängning:
    I high-availability-klustret finns alternativet starta om efter strömavbrott tillgängligt. Om servrarna stängs av avvikande på grund av strömavbrott kommer de automatiskt att slås på när strömmen kommer tillbaka.

För att byta ut en server i ett high-availability-kluster:

När du vill ta bort en server och installera en annan server, vänligen följ dessa instruktioner.

  1. Gå till sidan Översikt.
  2. Klicka på knappen Hantera.
  3. Välj Stäng av aktiv server eller Stäng av passiv server för att stänga av den server du vill byta ut.
  4. Kontrollera anslutningen för den nya servern. Efter att en ny server startats kommer den att vara markerad som ”okänd” i High Availability Manager.
  5. Gå till sidan Översikt igen.
  6. Klicka på knappen Hantera.
  7. Välj Återställ den okända servern så att den ”okända servern” läggs till i high-availability-klustret och kännas igen som den passiva servern.
