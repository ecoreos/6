Kopiere eller flytte filer eller mapper

File Station gir deg ulike metoder for å overføre filer/mapper mellom to DiskStation, innad i én DiskStation eller til en lokal datamaskin.

Dra-og-slipp filer/mapper mellom lesere

Du kan laste opp filer/mapper via File Station fra én DiskStation til en annen, uten komplisert oppsett av filoverføringsprotokoller.

Slik aktiverer du dra-og-slipp mellom nettlesere:

  1. Gå til Innstillinger > Generelt > File Station.
  2. Merk av for Aktiver dra-og-slipp mellom nettlesere.
  3. Klikk på OK.
  4. Åpne to nettlesere av samme type (f.eks. Firefox) og koble dem til henholdsvis kilde- og mål-DiskStation.
  5. Kjør File Station på begge DiskStation. Nå kan du dra-og-slippe filer/mapper direkte mellom dem via File Station.

Merk:

  • Du kan også dra-og-slippe filer/mapper mellom to faner i den samme nettleseren.
  • Internet Explorer støtter ikke dra-og-slipp mellom nettlesere eller mellom faner.
  • Hvis flere filer/mapper velges for dra-og-slipp, vil filene/mappene bli komprimert i et arkiv for opplasting.
  • Grensen for filstørrelse for dra-og-slipp varierer mellom nettlesere: Safari (1 GB).

Kopier eller flytt filer/mapper innad i DiskStation eller til en lokal datamaskin

Følg fremgangsmåten nedenfor for å flytte eller kopiere filer eller mapper innad i DiskStation eller til den lokale datamaskinen.

Metode 1:

  1. Velg filene eller mappene du vil flytte eller kopiere. (Trykk og hold Shift eller Ctrl nede hvis du vil velge flere.)
  2. Velg Klipp ut eller Kopier fra rullegardinmenyen Handling.
  3. Gå til målmappen på DiskStation eller datamaskinen.
  4. Velg Lim inn - Overskriv eller Lim inn - Hopp over fra rullegardinmenyen Handling, avhengig av om du vil overskrive filer eller mapper med samme navn i målmappen.

Metode 2:

  1. Velg filene eller mappene du vil flytte eller kopiere. (Trykk og hold Shift eller Ctrl nede hvis du vil velge flere.)
  2. Dra mappene eller filene fra hoveddelen til en annen mappe i på det venstre panelet.
  3. I menyen som kommer opp, velger du Kopier - Overskriv, Kopier - Hopp over, Flytt - Overskriv eller Flytt - Hopp over for å overskrive eller hoppe over filer eller mapper med samme navn i målmappen.

Metode 3

  1. Velg filene eller mappene du vil flytte eller kopiere. (Trykk og hold Shift eller Ctrl nede hvis du vil velge flere.)
  2. Trykk på Ctrl + X for klippe eller Ctrl + C for å kopiere filene/mappene. Mac-brukere trykker på Command + X og Command + C.
  3. Gå til målmappen på DiskStation eller datamaskinen.
  4. Trykk på Ctrl + V for å lime inn filene/mappene. (Command + V for Mac.)
  5. I dialogboksen som åpnes velger du om du vil overskrive eller hoppe over filer med samme navn i målmappen.

Tastatursnarveier:

Følgende tastatursnarveier kan være nyttige når du flytter/kopierer filer og mapper i File Station:

  • F2: Gi nytt navn
  • Del: Slett
  • Ctrl + A: Velg alle
  • Crtl + X: Klipp ut
  • Crtl + C: Copy
  • Crtl + V: Lim inn

Merk:

  • Når du kopierer eller flytter filer/mapper til en katalog i FAT-filsystemet, vil tegnene \ *?">| i filnavnet bli erstattet med –, og Metadata for Mac-filer vil bli slettet.
  • Hvis du klipper ut filer fra den lokale datamaskinen og limer dem inn i en DiskStation-mappe, vil filene likevel ikke bli fjernet fra den lokale datamaskinen.
  • Når du flytter en fil eller mappe, forblir "eieren" den samme. Men når du kopierer en fil eller mappe, endres "eieren" til brukeren som kopierte den.
