Søke etter filer eller mapper

File Stations Filter gir deg alternativer for å søke etter og vise filer og mapper som samsvarer med kriteriene dine. Du kan filtrere ut filer og mapper i den åpne mappen eller utføre et avansert søk som dekker alle undermapper for mer presise søkeresultater.

Slik søker du etter filer eller mapper i gjeldende mappe:

  1. Bruk File Station for å gå til mappen med innholdet du vil søke etter.
  2. I feltet Filter øverst til høyre skriver du inn et nøkkelord og ser det filtrerte resultatet i hovedområdet i vinduet.

For å utføre et avansert søk:

  1. Bruk File Station for å gå til mappen med innholdet du vil søke etter.
  2. Klikk på forstørrelsesglassikonet ved siden av feltet Filter øverst til høyre for å vise menyen for avansert søk.
  3. Spesifiser søkekriteriene nedenfor, eller velg fra rullegardinmenyer:
    • Nøkkelord: Skriv inn et nøkkelord for å søke etter filer eller mapper med matchende navn.

      Merk:

      • Du kan bruke jokertegnene [*] (som betyr null eller flere tegn) og [?] (som betyr et vilkårlig tegn) i nøkkelordene dine. For eksempel kan , [a*n] representere [action], [attention], eller [an].
      • Du kan bruke mellomrom til å skille mellom nøkkelord. For eksempel, nøkkelordet [File Station] gir resultater som inneholder [File], [Station] eller begge deler.
      • Du kan legge til dobbelt hermetegn ["] i begynnelsen og slutten av nøkkelord for å søke etter nøyaktige. versalsensitive treff. For eksempel kan ["Synology Inc."] representere [Synology Inc.], men ikke [Synology Incorporation].
      • Bruk operatorene nedenfor til å finjustere søket ditt:
        Operator Eksempel Forbruk
        && [X] && [Y]
      • Lister opp treff som inneholder både [X] og [Y]
        		•
        || [X] || [Y]
      • Lister opp treff som inneholder [X], [Y] eller begge deler
      • Treff som inneholder både [X] og [Y] vil ligge øverst på listen
        		•
        + +[X]
      • Lister opp treff som inneholder [X]
        		•
        - -[X]
      • Lister opp treff som ikke inneholder [X]
        		•
        ! ![X]
      • Lister opp treff som ikke inneholder [X]
        		•
        \ \[SpecialCharacter]
      • spesialtegn:   +   -   &&   ||    !    (    )    {    }    [    ]    ^    "    ~    *    ?    :   \
      • Lister opp treff som inneholder et av spesialtegnene ovenfor
        		•
    • Aktiver søk i filinnhold: Søk etter indekserte filer med filinnhold eller metadata som samsvarer med nøkkelordene.
    • Plassering: Velg fra rullegardinmenyen for å angi område og dybde for avansert søk. Du kan også tilpasse søkeomfanget ved å velge enhver annen delt mappe fra listen.
    • Filtype: Velg Filetternavn for å begrense søkeområdet til bestemte filtyper, eller velg Enhver mappe for å bare søke etter mapper.
    • Dato: Velg Endret dato, Dato opprettet eller Sist brukt og velg en dato fra kalenderen for å søke etter filer med samsvarende datoer.
    • Størrelse (MB): Velg blant er lik, er større enn og er mindre enn, og angi deretter en størrelse for å søke etter filer med matchende størrelse. (Når du velger er lik, vil søkeresultatene inkludere filer på omtrent samme størrelse som den som er spesifisert.)
    • Eier/gruppe: Velg Eier eller Gruppe, og velg deretter fra rullegardinmenyen for å søke etter filer eller mapper som tilhører valgt eier eller gruppe.

      Merk:

      • For å utføre avansert søk, må du spesifisere Plassering og minst ett annet søkekriterium.
      • Hvis en ISO-mappe eller en ekstern mappe er montert i en indeksert delt mappe, finner du ikke filene i den i et avansert søk.
  4. Klikk på Søk for å starte søket, eller klikk på Tilbakestill og angi søkekriterier på nytt. Når søket er fullført, vil matchende filer bli oppført i hovedområdet i File Station.
