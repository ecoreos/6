File Station

File Station er et brukervennlig filadministreringsverktøy, som tillater DSM-brukere over Internett å få tilgang til DiskStation-mapper med nettleseren de bruker. Ved å åpne to eller flere File Station-er, kan du administrere alle dine DiskStation-data ved å dra og droppe dem mellom forskjellige File Station-er.
