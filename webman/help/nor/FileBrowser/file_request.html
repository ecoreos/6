Opprett filforespørsler

Filforespørsler er filopplastingsinvitasjoner som sendes til ikke-DSM-brukere. Med filforespørselskoblinger kan ikke-DSM-brukere laste opp flere filer direkte til File Station på din DiskStation, uten behov for korresponderende brukerkontoer og rettigheter for filopplasting.

Slik oppretter du filforespørselskoblinger:

  1. Gå til File Station og velg en mappe som opplastingsmål.
  2. Klikk på Handling eller høyreklikk på målmappen og velg Opprett filforespørsel.
  3. I vinduet som åpnes kan du finne/endre informasjonen nedenfor:
    • Filbane: Angi målkatalogbanen.
    • Kobling: Angi filforespørselskoblingen som deles med målbrukere.
    • Navnet ditt: Angi navn for å indikere anmoders identitet. Standardverdien er brukernavnet ditt.
    • Melding: Egendefiner meldingen til målbrukere.
    • Aktiver passordbeskyttelse: Velg dette alternativet slik at kun brukere med riktig passord kan laste opp filer via den opprettede koblingen. Spesifiser passordet i boksen nedenfor.
    • Gyldighetsperiode: Klikk på denne knappen for å egendefinere filforespørselskoblingens varighet.
      • Sett opp sluttidspunkt: Velg dette alternativet for å angi dato og tid for når koblingen skal slutte å fungere som filopplaster.
      • Sett opp starttidspunkt: Velg dette alternativet for å angi dato og tid for når koblingen skal begynne å fungere som filopplaster.
      • Antall tillatte tilganger: Velg dette altenativet for å angi hvor mange ganger koblingen kan brukes.
  4. Klikk på Lagre for å fullføre innstillingene.

Slik deler du filforespørselskoblinger:

For å finne URL-adressen til filforespørselen, går du til File Station > Verktøy > Delt koblingsadministrator og dobbeltklikker på ønsket kobling. Du kan dele koblingen på følgende måter:

  • Kopier URL-adressen til målbrukerne.
  • Send ut koblingen via Standarde-post (e-postkontoen din for DSM) eller via Post (et e-postprogram fra en tredjepart). Førstegangsbrukere kan følge veiviserens instruksjoner for å sette opp en e-postkonto.

Slik laster du opp filer:

Hvis du blir bedt om å laste opp filer via koblingen, gjør du som følger:

  1. Åpne aktuell URL på den lokale datamaskinen eller en mobilenhet (Android eller iOS). Angi passordet om nødvendig.
  2. På opplastingssiden angir du et navn som indikerer identiteten din.
  3. Dra og slipp filen(e) for å laste opp, eller klikk på Legg til filer for å velge filer.
  4. Klikk på Last opp for å fullføre forespørselen.

Merk:

  • Mapper kan ikke lastes opp via filforespørselskobling.
  • Mapper i den delte mappen "web" kan ikke brukes som målmappe.
  • Opplastingsfunksjonen varierer i forhold til nettlesere:
    • Internet Explorer 8 og Internet Explorer 9 støtter ikke filopplasting med dra-og-slipp.
    • Chrome på iOS-enheter støtter kun bildeopplasting.
    • Safari på iOS-enheter støtter kun bilde- og videoopplasting.
    • Nettlesere på Android-enheter har ingen filtypebegrensninger.

Slik finner du opplastede filer:

Alle filer som lastes opp av den samme brukeren finner du i en identisk mappe i målet. Som standard vil mappenavnet være Opplaster_Navn (Anmoder_Navn).

For eksempel, hvis filene lastes opp av Watson til Holmes, finner du filene i mappen som heter Watson (Holmes).

Administrere filforespørselskoblinger

Slik endrer du filforespørselskoblinger:

Når du har opprettet en filforespørselskobling, kan du endre innstillinger for den i File Station > Verktøy > Delt koblingsadministrator.

  1. Dobbeltklikk på ønsket kobling, eller velg koblingen, og klikk på Rediger.
  2. I vinduet som åpnes endrer du relevante innstillinger.
  3. Klikk på Lagre for å ta i bruk de nye innstillingene.

Slik sletter du filforespørselskoblinger:

  1. Gå til File Station > Verktøy > Delt koblingsadministrator.
  2. Velg ønsket kobling og klikk på Slett. Korresponderende URL-adresser for filforespørslene vil slutte å fungere umiddelbart.

Slik tilordner du filforespørselsrettigheter:

Som admin eller en bruker som tilhører administrators-gruppen kan du tilordne rettigheter til brukere slik at de kan opprette filforespørselskoblinger.

  1. Gå til File Station > Innstillinger > Delte koblinger > Filforespørselskoblinger.
  2. Velg brukertype for å gi dem filforespørselsrettigheter.
  3. Klikk på OK for å lagre innstillingene.
