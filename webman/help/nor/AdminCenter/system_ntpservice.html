NTP-tjeneste

Du kan bruke din DiskStation som nettverktidsserver for å synkronisere tiden mellom forskjellige nettverksenheter og DiskStation over nettverk.

Slik aktiverer du NTP-tjenesten:

Merk av for Aktiver NTP-tjeneste og klikk på Bruk.

Merk: NTP-tjenesten er påkrevd for Surveillance Station og High Availability-sky. Så hvis Surveillance Station eller High Availability Manager er installert og kjører på din DiskStation, kan ikke tjenesten deaktiveres mens pakken kjører.
