Tilkoblingsbegrensninger

Tillat kun SSL/TSL forbindelse

Når denne funksjonen er aktivert, vil serveren bare akseptere SSL- eller TLS-tilkoblingsforespørsler fra FTP-klienter.

Begrens tilkoblinger per IP

Når denne funksjonen er aktivert, vil serveren begrense maksimalt antall tilkoblinger per IP. Aktiver denne funksjonen og FTP-båndbredderestriksjon for å få effektiv administrasjon av båndbredde.

Aktiver restriksjon på FTP-båndbredden

Aktiver denne funksjonen hvis du vil begrense båndbredden til en enkelt-FTP-tilkobling.
