Endre den valgte brukerens rot til brukerhjem

Brukere som er valgt her kan bare få tilgang til sine hjemmemapper ved pålogging av Synology DiskStation via FTP.

Aktiver anonym FTP

Aktiver denne funksjonen for å tillate alle å logge inn til DiskStations FTP-tjeneste (med anonym eller ftp-konto).

For å konfigurere privilegier for anonyme brukere:

Gå til Hovedmeny > Kontrollpanel > Delt mappe og klikk Innstilling av rettigheter.

Endre anonym rot

Aktiver denne funksjonen for å endre innloggingsregisteret for anonyme FTP-brukre. Velg den delte mappen fra rullegardinmenyen.
