Indekseringstjeneste

Medieindeksering kan automatisk skanne multimediefiler, for eksempel fotografier, musikk og videoer som er lagret på din DiskStation og samle dem i et multimediebibliotek som kan brukes av multimedieprogrammer. Du kan også sette opp miniatyrbildekvalitet for fotografier og videoer, administrere konverteringsinnstillinger for videoer og opprette indekserte mapper som du kan åpne med DMA-enheter og DSMs multimediaprogrammer.

Filindeksering lar deg indeksere mapper og iboende filer på din DiskStation etter filnavn, og du kan ytterligere indeksere filinnhold/-metadata etter filtype. Denne detaljerte indekseringsfunksjonen vil levere raskere resultater når du søker i filene dine.
