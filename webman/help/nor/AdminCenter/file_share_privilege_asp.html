Administrer avanserte tillatelser for delt mappe

Siden i Kontrollpanel > Delt mappe > Rediger > Avansert inkluderer alternativene nedenfor for å finjustere tilgangstillatelsene for en delt mappe ytterligere.

Avanserte deletillatelser

Avanserte deletillatelser gir deg et ytterligere lag med kontroll i administreringen av tilgangstillatelser for delte mapper. Når dette er aktivert, kan brukere og grupper kun vise eller endre innholdet i en delt mappe hvis brukeren eller gruppen har fått tildelt både avanserte deletillatelser og Windows ACL-tillatelser (i Delt mappe > Rediger > Tillatelser).

Slik aktiverer du avanserte deletillatelser:

  1. Merk av for Aktiver avanserte deletillatelser.
  2. Klikk på Avanserte deletillatelser for å endre de avanserte deletillatelsene for den delte mappen.

Merk:

  • Avanserte deletillatelser tas i bruk når en bruker aksesserer en delt mappe via følgende filtjenester: Windows File Sharing, Apple File Sharing, File Station, FTP og WebDAV.

Avanserte innstillinger

Om nødvendig, kan du legge til ytterligere begrensninger for brukere når de aksesserer en delt mappe via File Station, FTP eller WebDAV.

  • Deaktiver blaing: Aktivering av dette alternativet forhindrer brukere i å se innholdet i den delte mappen.
  • Deaktiver endring av eksisterende filer: Aktivering av dette alternativer forhindrer brukere i å flytte, slette eller endre filer i den delte mappen. Vær oppmerksom på at brukere fortsatt vil kunne se, laste ned/opp, kopiere eller pakke ut innholdet i den delte mappen.
  • Deaktiver filnedlasting: Aktivering av dette alternativet forhindrer brukere i å laste ned innholdet i den delte mappen.

Merk:

Begrensningene ovenfor gjelder ikke for brukere som tilhører gruppen administrators.
