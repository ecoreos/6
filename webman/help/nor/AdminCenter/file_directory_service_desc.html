Katalogtjeneste

Din DiskStation kan koble seg til en eksisterende katalogtjeneste som et Windows-domene eller en LDAP-klient. Du kan administrere domene- eller LDAP-brukeres tilgangsrettigheter til delte mapper og DSM-programmer, og aktivere hjemmetjeneste for dem. Gå til Kontrollpanel > Katalogtjeneste for å koble din DiskStation til en katalogtjeneste.
