DSM-innstillinger

I Kontrollpanel > Nettverk > DSM-innstillinger kan du endre portnummeret som klienter bruker til å få tilgang til DiskStation.

For å forandre standard portnummer:

Både HTTP- og HTTPS-tilkoblinger er alltid aktivert i DSM. Standard HTTP-port er 5000 og standard HTTPS-port er 5001.

  1. Skriv inn portnummeret i HTTP eller HTTPS-feltet.
  2. Klikk på Bruk.

Slik viderekobler du HTTP til HTTPS-tilkoblinger:

Du kan viderekoble klienter til HTTPS-porten når de prøver å aksessere din DiskStation via HTTP-porten.

  1. Velg Foreta automatisk omdirigering av HTTP-forbindelser til HTTPS (unntatt Web Station og Photo Station).
  2. Klikk på Bruk.

Porter som brukes for HTTPS-tilkobling:

  • DSM: Standard portnummer er 5001, så du kan få sikker tilgang til DSM ved å angi følgende adresse: https://server-hostname:5001/
  • Web Station og Photo Station: Standard portnummer er 443, så du kan få sikker tilgang til Web Station og Photo Station ved å angi følgende adresse: https://server-hostname/

Slik finner du DiskStation over nettverket:

Du finner DiskStation med Windows-nettverkssøk eller Bonjour-tjenestesøk.

  1. Velg tjenesten du vil bruke til å finne DiskStation over nettverket:
    • Aktiver Windows-nettverkssøk
    • Aktiver Bonjour-tjenestesøk
  2. Klikk på Bruk.

Domene

Under Domene kan du spesifisere et egendefinert domenenavn for tilgang til DiskStation.

Slik aktiverer du egendefinert domene:

  1. Merk av for Aktiver egendefinert domene.
  2. Spesifiser et domenenavn for din DiskStation.
  3. Du kan også merke av for Aktiver HSTS for å påkreve HTTPS-krypterte tilkoblinger på dette domenet.
  4. Klikk på Bruk.

Merk:

  • Hvert domenenvan kan kun brukes til ett bestemt formål på din DiskStation.
  • Hvis domenenavnet ikke overskrider 15 tegn og overholder NetBIOS-navnekonvensjonene, vil systemet automatisk registrere og kringkaste domenavnet (f.eks. MyDSM). Windows-brukere kan aksessere DSM-skrivebordet på samme lokale nettverk med en lignende URL-adresse: http://MyDSM eller https://MyDSM
  • Når du aksesserer programmer via URL-adresser som bruker domenenavn, går du gjennom standard HTTP-port (80) eller standard HTTPS-port (443).
