Opprette en gruppe

  1. Gå til Kontrollpanel > Gruppe.
  2. Klikk på Opprett. Denne handligen starter Veiviser for oppretting av bruker.
  3. Skriv inn gruppenavnet og beskrivelse. (Se seksjonen nedenfor for mer informasjon.) Klikk på Neste.
  4. Følg de gjenværende trinnene i veiviseren for å fullføre opprettingen gruppen.
  5. Klikk på Bruk for å fullføre.

Navngivningskonvensjoner:

  • Gruppenavnet skiller mellom store og små bokstaver og må være mellom 1 til 32 Unicode-tegn, unntatt følgende symboler:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Det første tegnet kan ikke være et minustegn eller et mellomrom, og det siste tegnet kan ikke være et mellomrom.
  • Gruppebeskrivelsen skiller mellom store og små bokstaver og kan bestå av inntil 64 visbare Unicode-tegn.
