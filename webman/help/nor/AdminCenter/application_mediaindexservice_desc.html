Mediebibliotek

Mediebibliotek kan automatisk skanne multimediefiler, for eksempel fotografier, musikk og videoer som er lagret på din DiskStation og samle dem i et multimediebibliotek som kan brukes av multimedieprogrammer.

Med Mediebibliotek kan du også sette opp miniatyrbildekvalitet for fotografier og videoer, administrere konverteringsinnstillinger for videoer og opprette indeksmapper som du kan åpne med DMA-enheter og DSMs multimedieprogrammer.
