Brukerhjem

Aktiver brukerhjem for å opprette en personlig hjem-mappe for hver bruker, unntatt for guest. Alle brukere kan få tilgang til sin egen hjem-mappe via CIFS, AFP, FTP eller File Station.

Admin kan få tilgang til alle tilpassede mapper som er lokalisert i den standard delte mappen homes. Navnet på hjemmappen er det samme som brukerkontoen.

Slik aktiverer du brukerhjemtjenesten:

  1. Klikk på Brukerhjem.
  2. Merk av for Aktiver brukerhjemtjeneste.
  3. Hvis det er mer enn ett volum på Synology DiskStation, velger du hvor du vil at mappen homes skal lagres.
  4. Klikk på OK.

Merk: Så snart den lokale brukerhjemtjenesten er deaktivert, vil hjemtjenesten for domenebrukeren bli deaktivert samtidig.

Slik deaktiverer du brukerhjemtjenesten:

  1. Klikk på Brukerhjem.
  2. Fjern avmerkingen for Aktiver brukerhjemtjeneste.

Hvis brukerhjemtjenesten deaktiveres, vil den delte mappen homes bli reservert for admin-tilgang. Brukere vil få tilgang til sine hjem-mapper på nytt når brukerhjemtjenesten aktiveres igjen.

    Merk:
  1. For å slette homes delte mappe, må brukerhjemtjenesten deaktiveres først.
  2. Brukerhjemtjenestene for domenebruker kan bli deaktivert, dersom den lokale brukerhjemtjenesten er deaktivert.
