Språk

Du kan velge ønsket språk for DSM-brukergrensesnitt, Google Chrome-skrivebordsvarslinger, e-post, SMS, direktemeldingsvarslinger og programkodeside.

Displayspråk

Du kan velge displayspråket du foretrekker for DSM og Google Chrome-skrivebordsvarslinger, eller la det være det samme som standardspråket for nettleseren.

Varslingsspråk

Du kan angi språket du foretrekker for e-post-, SMS-, Windows Live Messenger- og Skype-varslinger. Det vil også gjelde for varslinger som mottas på Windows Phone.

Varslingsspråket på en iOS- eller Android-enhet er imidlertid avhengig av språkinnstillingen for din mobilenhet.

Kodeside

For programmer som konverterer ikke-Unicode-strenger til Unicode, for eksempel FTP-tjenesten, UPnP-støtte, indeksering av musikk-metadata, må du velge riktig kodeside for dem fra rullegardinmenyen Kodeside.
