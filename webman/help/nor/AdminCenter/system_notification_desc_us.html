Varsling

Du kan sette opp DiskStation til å sende deg varslingsmeldinger via e-post, mobilenheter, Windows Live Messenger eller Skype når status endres eller det oppstår feil i systemet, for eksempel: tilkoblingsfeil, endring av viftens atferd, strømbrudd, lagringsvolum som går tomt for plass eller sikkerhetskopierings- eller gjenopprettingsfeil osv.
