TFTP

Trivial File Transfer Protocol (TFTP, enkel filoverføringsprotokoll) er en enkel filoverføringsprotokoll som stort sett blir brukt til å overføre konfigurasjoner eller oppstartsfiler når autentisering ikke er påkrevd. Aktivering av TFTP-tjenesten lar DiskStation overføre oppstartslastere til klientenheter.

Slik konfigurerer du TFTP-innstillinger:

  1. Gå til Kontrollpanel > Filtjenester > TFTP og merk av for Aktiver TFTP-tjeneste.
  2. I feltet TFTP-rotmappe spesifiserer du hvilken mappe på DiskStation TFTP-klienter skal ha tilgang til.
  3. Klikk på Avanserte innstillinger for å konfigurere følgende innstillinger:
    • Aktiver TFTP-filoverføringslogg: Få oversikt over data overført via TFTP.
    • TFTP-klienttillatelse: Sett opp TFTP-klienttillatelser:
      • Skrivebeskyttet: TFTP-klienter kan kun lese innholdet i rotmappen.
      • Skrivbar: TFTP-klienter kan modifisere innholdet i rotmappen.
    • Tillatte klienter: Velg hvilke klienter som skal ha tillatelse til å koble til via TFTP:
      • Alle tilkoblinger: Alle klienter kan koble til via TFTP.
      • Tillat kun følgende IP-adresseområde: Kun klienter i spesifisert IP-område kan koble til via TFTP.
    • Tidsavbrudd: Hvis TFTP-serveren (dvs. din DiskStation) ikke svarer i løpet av den spesifiserte tiden, vil TFTP-tilkoblingen bli stoppet.
  4. Klikk på Bruk for å lagre innstillingene.
