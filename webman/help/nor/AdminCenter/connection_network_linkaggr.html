Link Aggregation*

DiskStation støtter multi-LAN, noe som gir deg muligheten til å kombinere LAN-grensesnitt ved hjelp av Link Aggregation-teknologien. Link Aggregation øker båndbredden til din DiskStation ved å forene flere nettverksgrensesnitt og yter trafikkfeiloverføring for å opprettholde nettverkstilkoblingen hvis tilkoblingen er nede.

Når LAN-grensesnittet er kombinert, vil du se et nytt grensesnitt kalt Bond i Kontrollpanel > Nettverk > Nettverksgrensesnitt, som kan konfigureres på samme måte som andre grensesnitt (f.eks. LAN eller PPPoE).

Merk:

  • Med Link Aggregation aktivert, vil nettverkstrafikken automatisk bli justert og balansert for hver tilkoblede enhet.
  • Hvis nettverksgrensesnittene er koblet til flere switcher, må du kontrollere at switch-portene er på samme LAN eller VLAN.

Slik kombinerer du flere LAN med Link Aggregation:

  1. Gå til Kontrollpanel > Nettverk > Nettverksgrensesnitt. Klikk på Opprett > Opprett Bond.
  2. Velg modusen du vil ha.

    Modusbeskrivelser:

    • Dynamisk belastningsfordeling: Denne modusen optimaliserer mottatt og sendt nettverkstrafikk for din DiskStation, uansett om switchen støtter Link Aggregation eller ikke. For å unngå uventede feil, må du ikke aktivere Link Aggregation på switchen selv når dette støttes.
    • IEEE 802.3ad dynamisk Link Aggregation: Denne modusen optimaliserer mottatt og sendt nettverkstrafikk for din DiskStation, og krever at IEEE 802.3ad (Dynamisk) Link Aggregation (LACP, 802.1AX) aktiveres på switchen(e). Hvis flere switcher er i bruk, må disse switchene kunne stables og være riktig konfigurert.
    • Fordelings-XOR: Denne modusen balanserer mottatt og sendt nettverkstrafikk for din DiskStation, og krever at Statisk Link Aggregation aktiveres på switchen(e). Hvis flere switcher er i bruk, må disse switchene kunne stables og være riktig konfigurert. Hvis Link Aggregation ikke støttes at switchen, kan trafikken som sendes av din DiskStation likevel balanseres. Med denne modusen kan de tapte koblingene oppdages, men ikke årsaken (f.eks. om Ethernet-kabler er frakoblet eller switchen er feilkonfigurert).
    • Aktiv/ventemodus: Denne modusen gir nettverksfeiltoleranse til din DiskStation. Det vil si at når det aktive nettverksgrensesnittet svikter, vil et av de andre LAN-portene ta over for å opprettholde nettverkstilkoblingen. Du kan velge denne modusen uten å konfigurere switchen, eller med en switch som støtter IEEE 802.3ad Link Aggregation.
  3. Velg grensesnittene for å opprette Link Aggregation.
  4. Konfigurer IP-innstillingene. Du kan også aktivere VLAN om nødvendig. Klikk på Bruk for å fullføre oppsettet.
  5. Når du er ferdig, kan du se det nye grensesnittet kalt Bond 1 på Nettverksgrensesnitt-listen.

Slik redigerer du Link Aggregation-innstillinger:

  1. For å endre Link Aggregation-modus eller vise informasjon om fysiske enheter, går du til Kontrollpanel > Nettverk > Nettverksgrensesnitt.
  2. Velg grensesnittet Bond 1 og klikk på knappen Rediger.
  3. Gå til korresponderende faner for å endre ønskede innstillinger.

Slik fjerner du Link Aggregation:

  1. Gå til Kontrollpanel > Nettverk > Nettverksgrensesnitt.
  2. Velg grensesnittet Bond 1 og klikk på knappen Slett.
  3. Når du er ferdig, vil du se at Bond 1 nå er erstattet med to LAN-grensesnitt.
