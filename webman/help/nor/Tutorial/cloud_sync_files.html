Synkroniser filer mellom Synology NAS og datamaskinen med Cloud Station

Oversikt

Cloud Station for Synology NAS er et filsynkroniseringsprogram som enkelt lar deg synkronisere filer på din Synology NAS med andre enheter, som datamaskiner eller mobilenheter (med DS cloud). Installer Cloud Station Server på Synology NAS og Cloud Station Drive på datamaskinen din for å synkronisere filer som er lagret på datamaskinen med din Synology NAS automatisk.

Sett opp tilkobling mellom Synology NAS og datamaskinen

  1. Gå til Pakkesenter, finn Cloud Station Server og klikk på Installer.
  2. Kjør Cloud Station Server.
  3. Under Oversikt-fanen klikker du på  -knappen under Cloud Station Drive for å laste ned Cloud Station Drive for ditt operativsystem. (Du kan også laste ned Cloud Station fra Synology Nedlastingssenter.)
  4. Kjør installasjonsprogrammet på datamaskinen din og følg instruksjonene på skjermen.
  5. Når installasjonen er fullført, kjører du Cloud Station på datamaskinen din. Klikk på Start nå.
  6. Angi adressen til den Synology NAS som kjører Cloud Station (eller QuickConnect ID), brukernavn og passord. Du kan klikke på søkeikonet til høyre for søke etter IP-er på ditt LAN. Klikk på Neste.
  7. Velg mappene du vil synkronisere mellom Synology NAS og datamaskinen ved å klikke på Rediger-ikonet. Deretter kan du klikke på Avansert for å konfigurere innstillinger for synkronisering.
  8. Fjern avmerking for mapper du ikke vil synkronisere.
  9. Du kan angi en maksimal filstørrelse eller svarteliste filnavn eller filtyper slik at de ikke blir synkronisert.
  10. Velg om du vil synkronisere begge retninger eller kun laste ned data fra Synology NAS.
  11. Klikk på Ferdig for å fullføre oppsettet.
  12. Du finner Cloud Station-ikonet i systemstatusfeltet.
  13. Klikk på ikonet i systemstatusfeltet for å åpne menyen. Her kan du vise fremdriften og statusen til filene dine. Her kan du vise fremdriften og statusen til filene dine. Klikk på Hovedapp for å kjøre Cloud Station.
  14. Ved oppstart, vil du få tips om hvordan du bruker Cloud Station. Klikk på høyrepilen for å se flere tips, eller klikk på Nettveiledninger for mer informasjon. Hvis du ikke trenger tips, merker du av for Ikke vis dette igjen.
  15. Nå kan du administrere synkroniseringsoppgavene dine i Cloud Station Drive.

Last ned en tidligere versjon av en synkronisert fil

  1. Klikk på Cloud Station Drive-ikonet i systemstatusfeltet og klikk på mappeikonet.
  2. Høyreklikk på den valgte filen og velg Synology Cloud Station > Bla gjennom tidligere versjoner.
  3. Finn versjonen du ønsker å laste ned og klikk på nedlastingsikonet.

Last ned eller gjenopprett en tidligere versjon av en synkronisert fil fra Cloud Station Server

  1. Gå til Cloud Station Server > Versjonshistorikk og finn filen eller mappen du vil gjenopprette.
  2. Velg filen du vil hente og klikk på Handling > Bla gjennom tidligere versjoner.
  3. Velg filversjonen du vil laste ned og klikk på Last ned. Eller hvis du er sikker på at du vil gjenopprette den valgte versjonen, klikker du på Gjenopprett for å overskrive den gjeldende versjonen.

Viktige meldinger

  1. Du bør alltid beholde historikkversjoner i tilfelle en fil blir fjernet eller overskrevet ved et uhell.
  2. I Globale innstillinger kan du konfigurere om en fil skal hentes på nytt fra din NAS eller slettes på din NAS når du sletter en fil på datamaskinen din.

Synkroniser data til mobilenheter

DS cloud er tilgjengelig gratis i Apple App Store og Google Play Store. Du kan også skanne QR-koden nedenfor for å laste ned med en gang.
