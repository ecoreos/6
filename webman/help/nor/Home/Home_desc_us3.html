Online-ressurser

Hvis du ikke finner det du trenger her, kan du se i Synologys elektroniske ressurser nedenfor.

  • Kunnskapsbase: www.synology.com/support.php
  • Forum: forum.synology.com
  • Nedlastingssenter: www.synology.com/support/download.php
  • Teknisk støtte: account.synology.com/support/support_form.php

Enten du vet nøyaktig hva du leter etter, eller om du vil bla gjennom for å finne det du trenger, gir DSM Hjelp deg detaljerte forklaringer på følgende emner og ressurser.

Hovedmeny

Kjør DSM-tjenester og pakkene som er installer på din DiskStation.

Alternativer

Endre kontoinnstillingene dine, som passord, beskrivelse eller 2-trinnsverifikasjon.

Kontrollpanel

Vis og endre forskjellige DSM-innstillinger vedrørende fildeling, system, nettverkstjenester og mye mer.

File Station

Organiser, last opp eller last ned mapper og filer som er lagret på din DiskStation.

Sikkerhetskopier og gjenopprett

Sikkerhetskopier og gjenopprett data, iSCSI LUN-er eller systemkonfigurasjonene for din DiskStation.

Sikkerhetskopier og gjenopprett data eller systemkonfigurasjoner for din DiskStation.

Pakkesenter

Installer, oppdater og administrer programmer fra Synology eller tredjeparter.

Ez-Internet

Gjør din DiskStation tilgjengelig over Internett med en enkel veiviser for oppsett.

Ressursovervåker

Overvåke CPU-forbruk, minnebruk, nettverksflyt og volumforbruk.

Overvåk CPU-forbruk, minneforbruk og nettverksflyt.

Systeminformasjon

Vis den generelle statusen til din DiskStation og andre tilkoblede enheter.
