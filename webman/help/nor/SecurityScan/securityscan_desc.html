Security Advisor

Security Advisor er et DSM-sikkerhetsprogram som skanner dine DSM-innstillinger og DiskStation. Security Advisor kontrollerer innstillingene dine og anbefaler endringer som bidrar til å holde din DiskStation trygg. Her en liste over tilgjengelige funksjoner:

  • Security Advisor vil utføre ulike kontroller for å finne eventuelle DSM-innstillinger som utgjør en sikkerhetsrisiko i tillegg til andre mistenkelige forhold.
  • Security Advisor gir deg detaljerte resultater med rapporter som forklarer hvilke handlinger som blir anbefalt og hvorfor disse handlingene er fordelaktige.
  • De interne sikkerhetsdefinisjonene oppdaterer seg automatisk og holder seg oppdatert på nye nettrusler.

For å kjøre Security Advisor, gjør du følgende:

  • Gå til Hovedmeny > Security Advisor.
