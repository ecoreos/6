Oversikt

Du kan kontrollerer den gjeldende statusen til DSM, skanne systemet og kontrollere skannefremdriften og lese resultatsammendrag. Security Advisor skanner DSM og din DiskStation ved å kontrollere en rekke sikkerhetsregler. Antallet og typen regler er avhengig av hvordan du konfigurerer Sikkerhetsgrunnlinje. Velg Sikkerhetsgrunnlinje ut ifra dine behov. Bruker du din DiskStation til forretninger på jobben eller til personlig bruk hjemme? Du vil bli bedt om å velge en Sikkerhetsgrunnlinje første gangen du setter i gang en skanning. Denne kan endres senere på Avansert-siden.

Systemstatus

Security Advisor har totalt sju statuser. Når en skanning er fullført, vil du få en resultatoppsummering med fem kategorier. De ulike statusene inkluderer:

  • Klar til å skanne: Denne statusen vises når du ikke har skannet din DiskStation ennå. Klikk på Skann for å sette i gang skanningen.
  • Skanner: En skanning pågår. Du kan vise skanningens fremdrift. Klikk på Stopp for å stoppe skanningen.
  • God: Gjeldende skanning er fullført. Security Advisor fant ingen svake innstillinger eller sikkerhetsrisikoer. Din DiskStation er beskyttet.
  • Advarsel: Gjeldende skanning er fullført. Pga. enkelte sårbare sikkerhetsinnstillinger, rapporterte Security Advisor enkelte lave/moderate sikkerhetsrisikoer. Du rådes til å utføre de anbefalte handlingene for å gjøre din DiskStation sikrere.
  • Utsatt: Gjeldende skanning er fullført. Pga. enkelte sårbare sikkerhetsinnstillinger, rapporterte Security Advisor enkelte høye sikkerhetsrisikoer. Du anbefales sterkt å utføre de anbefalte handlingene for å gjøre din DiskStation sikrere.
  • Fare: Gjeldende skanning er fullført. Security Advisor fant malware på din DiskStation. Du må utføre anbefalte handlinger for å unngå ytterligere uønskede konsekvenser for din DiskStation.
  • Utdatert: DSM-versjonen din er utdatert. Du blir anbefalt å oppdatere DSM for å få de nyeste funksjonene, forbedringene og sikkerhetsreparasjonene for en bedre brukeropplevelse og bedre beskyttelse av din DiskStation.

Security Advisor grupperer kontroller i fem kategorier:

  • Malware: Finner ut om din DiskStation har malware og om din DiskStation er under angrep
  • System: Finner systemrelaterte sikkerhetsinnstillinger.
  • Konto: Finner sikkerhetsinnstillinger for konto og passord.
  • Nettverk: Finner nettverksrelaterte sikkerhetsinnstillinger.
  • Oppdater: Finner ut om DSM og andre installerte pakker er oppdatert, og kontrollerer oppdateringsrelaterte innstillinger.

Når en skanning er fullført, kan du vise detaljerte resultater ved å klikke på Vis resultater.
