Avansert

Du kan vise og endre avanserte innstillinger for Security Advisor.

Sikkerhetsgrunnlinje

Du kan velge hvilke sikkerhetsregler Security Advisor skal kontrollere ved å velge en Sikkerhetsgrunnlinje. Antallet og typen regler vil variere for hver Sikkerhetsgrunnlinje. Foreta valget basert på dine behov. Bruker du din DiskStation til forretninger på jobben eller til personlig bruk hjemme? Du kan velge blant følgende alternativer:

  • For hjemme- og privatbruk: Security Advisor vil utføre essensielle kontroller for å beskytte din DiskStation mot vanlige nettrusler.
  • For arbeid og forretninger: Security Advisor vil utføre omfattende kontroller for å imøtekomme organisasjoners krav til sikkerhet.
  • Egendefinert: Velg hvilke sikkerhetsregler du ønsker Security Advisor skal kontrollere.

Skannetidsplan

Du kan definere en regelmessig skannetidsplan ved å klikke på Aktiver regelmessig skannetidsplan og velge tidspunkt for Security Advisor-skanninger. Security Advisor vil kontrollere regler i henhold til din valgte Sikkerhetsgrunnlinje.
