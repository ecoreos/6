Resultater

Du kan vise detaljert informasjon om den fullførte skanningen, én rapport om gangen. Denne siden er lagt opp som følger:

  • Alvorlighet: Det generelle risikonivået som Security Advisor vurderer for hver sikkerhetsrege. Det finnes fire nivåer: Normal, Advarsel, Alarm og Kritisk.
  • Beskrivelse: Innstillinger som Security Advisor ser etter i en bestemt regel som kan føre til svakheter for din DiskStation hvis ingen handling utføres.
  • Siste oppdatering: Forrige gang Security Advisor kontrollerte denne sikkerhetsregelen.
  • Status: Gjeldende status for hver sikkerhetsregel. Security Advisor rapporterer resultatet for hver sikkerhetsregel som Bestått eller Mislyktes. Noen ganger vil du også få se statusene Feil, Ikke kontrollert, Kjører og Hoppet over. Du kan filtrere resultatene øverst på siden.

Slik kontrollerer du rapportdetaljene for en sikkerhetsregel:

Velg en regel og klikk på Vis. I tillegg til Alvorlighet og Beskrivelse, inneholder alle rapporter følgende detaljer:

  • Detaljer: En kort forklaring av eventuelle svakheter, hva Security Advisor anbefaler for å håndtere dem og fordeler ved å adressere svakheten.
  • Anbefalt handling: Detaljerte instruksjoner om hvordan du skal håndtere den rapporterte sikkerhetssvakheten.
  • Kobling: For mislykkede sikkerhetsregler kan du gå til plasseringen i DSM hvor du må utføre anbefalte handlinger ved å klikke på Kobling.
  • Reparer: Når malware oppdages, vil Reparer instruere Security Advisor om å fjerne malwaren.

Slik skanner du en bestemt regel på nytt:

Velg elementene du vil skanne på nytt, og klikk på Skann.
