Compresser des fichiers ou dossiers

  1. Choisissez des fichiers ou dossiers de DiskStation dans la zone principale.
  2. Choisissez Compresser vers « (nom de fichier) ».zip dans le menu déroulant Action pour lancer la compression.
  3. Si vous ne voyez pas le fichier zip quand la compression est terminée, choisissez Actualiser dans le menu déroulant Action pour voir le fichier.
  4. Si vous voulez renommer, crypter l'archive ou l'archiver à l'aide de différents niveaux ou modes de compression, choisissez Ajouter à l’archive... à la place.
    Remarque :
  1. Vous ne pouvez pas compresser des fichiers et dossiers sur votre ordinateur local à l'aide de File Station.
  2. Formats d'archive pris en charge : .zip, .7z.
  3. Si vous choisissez .zip comme Format d’archive dans Ajouter à l’archive..., vous pouvez sélectionner une page de code de votre choix pour le fichier .zip.
