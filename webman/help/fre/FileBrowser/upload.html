Charger des fichiers ou dossiers

Utilisez l'une des méthodes suivantes pour charger les fichiers et dossiers de votre ordinateur vers le DiskStation.

Pour avoir un accès total aux fonctions de chargement décrites ci-dessous, installez les dernières versions de Java Runtime Environment (JRE) et Flash Player.

Méthode 1 :

  1. Dans la partie supérieure du panneau gauche, choisissez un dossier de destination sur le DiskStation.
  2. Cliquez sur Charger, choisissez Charger - Ignorer ou Charger - Écraser et sélectionnez les fichiers et les dossiers sur votre ordinateur local.

Méthode 2 :

  1. Allez dans Paramètres, cochez Parcourir les fichiers sur l'ordinateur local avec File Station et actualisez la page Web pour afficher les dossiers de votre ordinateur local sur la partie inférieure du volet gauche.

    2. Remarque :

    L'option Parcourir les fichiers sur l'ordinateur local avec File Station ne prend effet qu'après avoir installé et activé JRE sur votre ordinateur.

  2. Dans la partie inférieure du panneau gauche, sélectionnez un dossier sur votre ordinateur local. La section principale affiche maintenant le contenu du dossier sélectionné.
  3. Sélectionnez les fichiers ou dossiers souhaités dans la section principale, puis cliquez sur Charger.
  4. Sélectionnez un dossier de destination sur le DiskStation dans la fenêtre qui apparaît.
  5. Dans le coin supérieur droit de la fenêtre, sélectionnez Écraser si vous voulez écraser les fichiers et dossiers ayant le même nom dans le dossier de destination. Sinon, sélectionnez Ignorer.
  6. Cliquez sur OK pour démarrer le chargement.

Méthode 3 :

  1. Allez dans Paramètres, cochez Parcourir les fichiers sur l'ordinateur local avec File Station et actualisez la page Web pour afficher les dossiers de votre ordinateur local sur la partie inférieure du volet gauche.
  2. Sur le panneau gauche, faites glisser un dossier de votre ordinateur (la partie inférieure) vers n'importe quel dossier du DiskStation (la partie supérieure).
  3. Dans le menu contextuel qui apparaît, choisissez si vous voulez écraser ou ignorer les fichiers/dossiers ayant le même nom dans le dossier de destination.

Méthode 4 :

  1. Dans la partie supérieure du panneau gauche, choisissez un dossier de destination sur le DiskStation.
  2. Sélectionnez les fichiers sur le bureau de votre ordinateur ou sur votre explorateur de fichiers (par exemple, Windows Explorer ou Mac Finder) et faites glisser et déposez-les dans File Station dans le navigateur Web.
  3. Dans le dialogue qui apparaît, choisissez si vous voulez écraser ou ignorer les fichiers ayant le même nom dans le dossier de destination.

    4. Remarque :

    • La Méthode 4 n'est pris en charge que certains navigateurs : Chrome, Firefox, Safari 5.1 ou ultérieur. Cependant, elle n'est pas prise en charge pour Mac quand Parcourir les fichiers sur l'ordinateur local avec File Station est coché.
    • Vous pouvez également faire glisser-déplacer des dossiers ou des fichiers vides, mais seulement quand Parcourir les fichiers sur l'ordinateur local avec File Station est coché pour les navigateurs listés ci-dessus. Dans le cas contraire, vous ne pouvez pas charger des dossiers et des fichiers vides à l'aide de la Méthode 4.
    • Si vous utilisez Chrome pour Linux et vous connectez à DSM via le protocole HTTPS, vous devez importer un certificat de confiance (dans Panneau de configuration > Sécurité > certificat) pour pouvoir charger à l'aide de Method 4.

Remarque :

  1. Vous pouvez ajouter jusqu'à 1000 fichiers à votre file d'attente des chargements.
  2. Vous pouvez charger des fichiers dont la taille dépasse 4 Go si vous utilisez Google Chrome, Firefox, Safari 5.1 ou version ultérieure, ou quand Parcourir les fichiers sur l'ordinateur local avec File Station est coché. Sinon, vous ne pouvez charger des fichiers que de jusqu'à 2 Go à la fois.
  3. Dans un dossier partagé non crypté, le dossier/fichier peut comporter jusqu'à 255 caractères (jusqu'à 80 caractères pour les langues non latines) ; dans un dossier partagé crypté, il peut comporter jusqu'à 143 caractères (jusqu'à 47 caractères pour les langues non latines).
  4. Le nom du dossier/fichier doit exclure les caractères suivants : \ / : * ? " > |.
  5. Le nom du fichier/dossier ne peut commencer par « ._ ».
  6. Vous pouvez charger des fichiers d'une taille de 0 Ko seulement quand vous avez installé JRE.

Dépannage

Si vous ne pouvez pas parcourir des éléments locaux stockés sur votre Mac en utilisant Safari correctement, veuillez procéder comme suit :

  1. Sur votre Mac, ouvrez Safari et choisissez Préférences dans le menu Safari.
  2. Cliquez sur Gérer les paramètres du site web dans l'onglet Sécurité.
  3. Sous Java, choisissez votre "site web" DSM.
  4. Sélectionnez Exécuter en mode non sécurisé.
