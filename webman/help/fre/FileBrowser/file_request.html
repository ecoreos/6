Créer des demandes de fichiers

Les demandes de fichiers sont des invitations à transférer des fichiers envoyés à des utilisateurs non DSM. Avec des liens de demande de fichiers, des utilisateurs non DSM peuvent transférer directement plusieurs fichiers sur File Station sur votre DiskStation, sans qu'il soit nécessaire de posséder de compte utilisateur correspondant ni de privilèges de transfert de fichiers.

Pour créer des liens de demande de fichiers :

  1. Accédez à File Station et sélectionnez un dossier comme destination de transfert.
  2. Cliquez sur Action ou cliquez avec le bouton droit sur le dossier de destination et sélectionnez Créer une demande de fichier.
  3. Dans la fenêtre contextuelle, vous pouvez rechercher/modifier les informations suivantes:
    • Chemin de fichier : Indique le chemin du répertoire de destination.
    • Lien : Indique le lien de la demande de fichiers à partager avec les utilisateurs cible.
    • Votre nom : Saisissez un nom pour indiquer l'identité du demandeur. La valeur par défaut est votre nom d'utilisateur.
    • Message : Personnalisez le message pour des utilisateurs cible.
    • Activer la protection par mot de passe : Sélectionnez cette option afin que seuls les utilisateurs ayant le mot de passe puissent transférer des fichiers via le lien créé. Indiquer le mot de passe dans la boîte qui se trouve en dessous.
    • Période de validité : Cliquez sur ce bouton pour personnaliser la durée de ce lien de demande de fichiers:
      • Configurer l'heure d'arrêt : Sélectionnez cette option pour déterminer le moment où le lien cesse de fonctionner pour le transfert de fichiers, à l'aide de la date et de l'heure.
      • Configurer l'heure de début : Sélectionnez cette option pour déterminer le moment où le lien commence à fonctionner pour le transfert de fichiers, à l'aide de la date et de l'heure.
      • Nombre d'accès autorisés : Sélectionnez cette option pour déterminer le nombre de fois où il est possible d'accéder au lien.
  4. Cliquez sur Enregistrer pour terminer la configuration.

Pour partager des liens de demande de fichiers :

Pour rechercher l'URL de la demande de fichiers, accédez à File Station > Outils > Gestionnaire de liens partagés et double-cliquez sur le lien souhaité. Vous pouvez partager le lien de l'une des manières indiquées ci-dessous :

  • Copiez simplement l'URL vers les utilisateurs cible.
  • Envoyez le lien via l'adresse e-mail par défaut (votre compte e-mail DSM) ou via un courrier (une application de messagerie électronique tierce). Pour les utilisateurs débutants, suivez les instructions de l'assistant pour configurer un compte e-mail à utiliser.

Pour transférer des fichiers :

Si l'on vous a demandé de transférer des fichiers via le lien, procédez comme suit :

  1. Ouvrez l'URL fournie sur l'ordinateur local ou un périphérique mobile (Android ou iOS). Saisissez le mot de passe si nécessaire.
  2. Sur la page de transfert, saisissez un nom qui indique votre identité.
  3. Glissez et déposez simplement le ou les fichiers à transférer ou cliquez sur Ajouter des fichiers pour sélectionner un ou des fichiers.
  4. Cliquez sur Transférer pour terminer la demande.

Remarque :

  • Il est impossible de transférer des dossiers via un lien de demande de fichiers.
  • Les dossiers se trouvent dans le dossier partagé « Web » ne peuvent pas être spécifiés comme dossier de destination.
  • La fonction de transfert varie en fonction des navigateurs Web :
    • Internet Explorer 8 et Internet Explorer 9 ne prennent pas en charge le transfert de fichiers via glisser/déposer.
    • Chrome sur les périphériques iOS ne prend en charge que le transfert d'images.
    • Safari sur les périphériques iOS ne prend en charge que le transfert d'images et de vidéos.
    • Les navigateurs Web sous Android ne sont soumis aucune restriction de type de fichier.

Pour rechercher des fichiers transférés :

Les fichiers transférés par le même utilisateur s'affichent tous dans un dossier identique au sein de la destination. Par défaut, le nom du dossier est Nom_du_télépartageur (Nom_du_demandeur).

Par exemple, si les fichiers sont transférés par Watson à Holmes, vous pouvez rechercher les fichiers dans le dossier nommé Watson (Holmes).

Gestion des liens de demande de fichiers

Pour modifier des liens de demande de fichiers :

Après avoir créé un lien de demande de fichiers, vous pouvez en modifier les paramètres dans File Station > Outils > Gestionnaire de liens partagés.

  1. Double-cliquez sur le lien souhaité ou sélectionnez le lien et cliquez sur Modifier.
  2. Dans la fenêtre contextuelle, modifiez les paramètres correspondants.
  3. Cliquez sur Enregistrer pour appliquer les nouveaux paramètres.

Pour supprimer des liens de demande de fichiers :

  1. Accédez à File Station > Outils > Gestionnaire de liens partagés.
  2. Sélectionnez le ou les liens souhaités et cliquez sur Supprimer. La ou les URL correspondantes à la demande de fichiers cessent immédiatement de fonctionner.

Pour affecter des privilèges de remplissage de demande :

En tant qu'admin ou utilisateur appartenant au groupe administrators vous pouvez affecter des privilèges à des utilisateurs afin qu'ils puissent créer des liens de demande de fichiers.

  1. Accédez à File Station > Paramètres > Liens partagés > Liens de demande de fichiers.
  2. Sélectionnez le type d'utilisateur pour les autoriser à demander des privilèges de remplissage.
  3. Cliquez sur OK pour enregistrer les paramètres.
