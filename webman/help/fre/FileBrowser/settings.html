Gestion des paramètres de File Station

Pour gérer les paramètres File Station, ouvrez File Station et cliquez sur le bouton Paramètres.

Généralités

Ici vous pouvez activer le journal de File Station pour enregistrer les activités de tous les utilisateurs. Les activités de l’utilisateur comprennent Créer dossier, Transférer, Télécharger, Supprimer, Renommer, Déplacer, Copier et Propriété définie.

Pour activer le journal de File Station :

  1. Si vous voulez enregistrer les activités de tous les utilisateurs, cochez Activer le journal de File Station.
  2. Cliquez sur OK.

Pour voir le journal de File Station :

  1. Cliquez sur Afficher les journaux, ce qui affiche le Centre des journaux.
  2. Cliquez sur l’onglet Transfert de fichiers.

Pour écraser automatiquement les fichiers avec le glisser-déposer ou les raccourcis du clavier :

Cochez Toujours écraser les fichiers lorsque je les fais glisser-déplacer sur le navigateur Web ou je les colle avec les raccourcis clavier si vous voulez écraser automatiquement les fichiers portant le même nom lorsque vous glissez et déposez des fichiers depuis le bureau de votre ordinateur sur le navigateur, ou collez des fichiers avec les raccourcis clavier dans File Station.

Pour activer le glisser et déposer intelligent :

Lorsque vous glissez et déposez des fichiers avec File Station, cette fonction détermine automatiquement si le transfert est une action de copie ou de déplacement.

  1. Sélectionnez Activer le glisser et déposer intelligent.
  2. Indiquez si l'action doit toujours être effectuée s'il arrive qu'un autre utilisateur transfère simultanément des fichiers portant le même nom au même emplacement :
    • Écraser : Vos fichiers transférés remplacent les fichiers portant le même nom.
    • Ignorer : Vos fichiers transférés sont ignorés et ne remplacent pas les fichiers portant le même nom.
  3. Cliquez sur OK.

Remarque :

Si vous souhaitez modifier l'action de copie ou de déplacement déterminé par le glisser et déposer intelligent, appliquez les touches de raccourci suivantes selon vos besoins :

  • Pour passer à une action de copie : Appuyez sur la touche Ctrl en effectuant le glisser et déposer.
  • Pour passer à une action de déplacement : Appuyez sur la touche Maj en effectuant le glisser et déposer.

Pour parcourir et gérer les fichiers depuis l'ordinateur local avec File Station :

  1. Cochez Parcourir les fichiers sur l'ordinateur local avec File Station.
  2. Cliquez sur OK.

Pour partager la liste des utilisateurs de DSM avec les utilisateurs non administrateurs :

Cette option permet aux utilisateurs non administrateurs d'accéder à la liste complète des utilisateurs de DSM s'ils souhaitent rechercher des fichiers appartenant à des utilisateurs/groupes spécifiques de DSM ou s'ils souhaitent partager des liens de fichiers avec d'autres utilisateurs de DSM.

  1. Sélectionnez Partager la liste des utilisateurs de DSM avec les utilisateurs non administrateurs.
  2. Cliquez sur OK.

Application des permissions UNIX par défaut

Cliquez sur Paramètres avancés et cochez l'option pour appliquez les permissions UNIX par défaut lors du partage ou la création de fichiers et de dossiers. Les autorisations appliquées sont les mêmes que les autorisations appliquées par la commande UNIX umask. Lorsque cette option est activée, les permissions UNIX sont de 644 pour les fichiers et de 755 pour les dossiers. Lorsque cette option est désactivée, les permissions UNIX sont de 666 pour les fichiers et de 777 pour les dossiers. La valeur umask par défaut est 022.

Remarque :

  • Pour les dossiers partagés avec Windows ACL activé (tous les partages sauf les « photos » et les partages sur des disques externes), veuillez exécuter la commande chmod sur votre client Linux ou FTP pour changer les types de permissions de dossier et de fichier depuis Windows ACL vers UNIX.
  • L'activation de cette option pourrait provoquer des problèmes d'incohérence de permission entre différents protocoles. Afin d'éviter les incohérences, nous suggérons de laisser cette option désactivée.

Montage/Connexions

Par défaut, tous les utilisateurs peuvent monter des dossiers distants ou des lecteurs virtuels. Ici vous pouvez limiter l’accès à la gestion des dossiers distants ou des lecteurs virtuels aux administrateurs seulement.

Remarque :

Les utilisateurs qui ne sont pas administrateurs ne peuvent monter les dossiers distants que dans leur dossier personnel accueil.

Pour autoriser différents utilisateurs à monter des dossiers distants ou des lecteurs virtuels :

  1. Choisissez Tous les utilisateurs ou Administrators dans le Dossier distant ou Lecteur virtuel.
  2. Cliquez sur OK.

Pour autoriser différents utilisateurs à monter des serveurs et des services cloud :

  1. Sélectionnez à partir de Tous les utilisateurs, Administrators, ou Utilisateur spécial.
  2. La sélection de Utilisateur spécial ouvre une fenêtre qui vous permet de configurer les autorisations de montage pour les différents utilisateurs. Pour activer l'accès à un utilisateur spécifique, assurez-vous que la case est cochée dans la colonne Activé. Pour désactiver l'accès à un utilisateur spécifique, assurez-vous que la case n'est pas cochée dans la colonne Activé. Une fois terminé, cliquez sur OK pour fermer cette fenêtre.

    3. Remarque :

    • Les utilisateurs appartenant au groupe administrators peuvent utiliser cette fonction mais ne peuvent pas faire n'importe quels changements. Les utilisateurs connectés en tant que Guest ne peuvent ni utiliser cette fonction ni procéder à la moindre modification. Les autorisations pour tous les autres utilisateurs peuvent être configurées ici.
  3. Cliquez sur OK.

Liens partagés

Par défaut, tous les utilisateurs peuvent partager des liens vers les fichiers. Ici vous pouvez limiter l’accès à la gestion des liens partagés aux administrateurs seulement.

Pour autoriser différents utilisateurs à partager des liens de fichier :

  1. Sélectionnez Tous les utilisateurs, Administrators, ou Utilisateurs spéciaux.
  2. Cliquez sur OK.

Pour ouvrir les fichiers HTML en texte brut :

  1. Cochez la case Toujours ouvrir les fichiers HTML utilisant le format texte brut. Si cette option est activée, les fichiers HTML (c.-à-d. les fichiers HTM, HTML, JSP, XHTML) sont affichés en utilisant le format texte brut lorsque les utilisateurs les ouvrent via un fichier de lien partagé. Cette option empêche les parties malicieuses de pirater des sessions de connexion d'utilisateur en utilisant le script intégré dans les fichiers HTML.
  2. Cliquez sur OK.

Limite de vitesse

Vous pouvez limiter la vitesse pour contrôler le débit maximal utilisable par les utilisateurs et les groupes qui transfèrent des données via File Station.

Pour activer la limite de vitesse :

  1. Cochez la case Activer la limite de vitesse, ou Activer la limite de vitesse selon un programme si vous désirez appliquer une limite de vitesse à certaines heures seulement. Ceci est un paramétrage à l'échelle du serveur qui remplace les paramétrages des utilisateurs individuels et groupes. L'option Pas de limite de vitesse désactive les paramétrages de limite de vitesse de tous les utilisateurs et groupes.
  2. Cliquez sur Paramètres de limite de vitesse pour modifier les paramètres des utilisateurs individuels ou groupes. Pour chaque utilisateur et groupe, vous pouvez sélectionner l'une des options suivantes :
    • Appliquer les paramètres de groupe : Les paramètres de limite de vitesse sont les mêmes que ceux du groupe auquel l'utilisateur appartient.
    • Pas de limite de vitesse : Aucune limite de vitesse ne sera appliquée lorsque les membres du groupe transfèrent les fichiers avec le service.
    • Constante : Une limite de vitesse fixe, constante sera appliquée à l'utilisateur ou aux membres du groupe. Les vitesses de chargement et de téléchargement peuvent être spécifiées dans les champs sur la gauche.
    • Variable : Deux limites de vitesse différentes peuvent être spécifiées et appliquées à l'utilisateur ou aux membres du groupe en fonction d'un horaire. Cliquez sur le bouton Personnaliser pour modifier les paramètres de limite de vitesse et définir un planning.
  3. Cliquez sur Appliquer.

Remarque :

  • Pour plus de détails concernant la dynamique de limitation de la vitesse des groupes, veuillez voir ici.
  • Lorsqu'une limite de vitesse est définie pour un utilisateur, la section Résultat affiche la limite de vitesse réelle de l'utilisateur selon les paramètres individuels et de groupe. Par exemple, si un utilisateur « Achille » est affecté d'une limite de vitesse de téléchargement de 100 Ko/s, mais qu'il appartient à un groupe d'utilisateurs affecté d'une limite de téléchargement de 500 Ko/s, la section Résultats affiche alors « 0/500 ».

Sécurité

Ici vous pouvez trouver les paramètres de sécurité ci-dessous.

Pour ouvrir les fichiers HTML en texte brut :

  1. Cochez la case Toujours ouvrir les fichiers HTML utilisant le format texte brut. Si cette option est activée, les fichiers HTML (c.-à-d. les fichiers HTM, HTML, JSP, XHTML) sont affichés en utilisant le format texte simple lorsqu'ils sont ouverts dans File Station. Cette option empêche les parties malicieuses de pirater des sessions de connexion d'utilisateur en utilisant le script intégré dans les fichiers HTML.

    2. Remarque :

    • Il ne s'agit pas d'un paramètre global. Chaque utilisateur peut choisir s'il active ou pas cette option selon ses besoins individuels.
  2. Cliquez sur OK.
