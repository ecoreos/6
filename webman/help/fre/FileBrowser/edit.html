Modifier des photos

Dans File Station, vous pouvez modifier des photos grâce à Éditeur Aviary, un éditeur Web externe de photos.

Pour modifier une photo avec Éditeur Aviary :

  1. Recherchez le dossier dans lequel se trouve la photo.
  2. Cliquez avec le bouton droit de la souris sur la photo et choisissez Modifier dans Aviary.
  3. Éditeur Aviary est ouvert dans une nouvelle fenêtre de navigation. Vous pouvez voir votre photo et la modifier à l'aide des outils fournis.
  4. Une fois que vous avez terminé l'édition de la photo, cliquez sur le bouton d'enregistrement dans la barre d'outils supérieure pour enregistrer toutes vos modifications dans la photo.

Remarque :

  • Grâce à Éditeur Aviary, vous pouvez modifier des photos dans les formats suivants : .jpg, .jpeg, .jpe, .png
  • La taille des fichier maximal autorisée par Éditeur Aviary est de 5 Mo.
  • Les paramètres réseau de votre DiskStation doivent être correctement configurés pour qu'Éditeur Aviary fonctionne correctement. Vérifiez les paramètres ci-dessous pour le dépannage :
    • Assurez-vous que votre DiskStation est accessible via QuickConnect (dans Panneau de configuration > QuickConnect) ou qu'il est en mesure de récupérer une adresse IP externe (dans Panneau de configuration > Accès externe > DDNS, ou dans Panneau de configuration > Réseau > Interface réseau > PPPoE).
    • Si votre DiskStation dépend d'un routeur, assurez-vous d'avoir bien spécifié les règles de transfert de port (dans Panneau de configuration > Accès externe > Configuration du routeur).
    • Si des problèmes persistent pour ouvrir l'Éditeur Aviary, vous serez peut-être amené à spécifier le port du routeur de votre DiskStation (dans Panneau de configuration > Accès externe > Avancé).
