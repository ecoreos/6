Afficher les attributs supplémentaires

Vous pouvez afficher les attributs supplémentaires des fichiers ou dossiers pour les faire apparaître dans de nouvelles colonnes de la section principale.

Pour afficher les attributs supplémentaires :

  1. Mettez votre curseur sur n'importe quel titre de colonne et cliquez sur le triangle pointant vers le bas qui apparaît à droite.
  2. Dans le menu déroulant qui apparaît, choisissez le sous-menu Colonnes.
  3. Cochez les cases à côté des attributs supplémentaires pour les faire apparaître dans de nouvelles colonnes.
