Modification des informations musicales

Grâce à l'Éditeur d’informations musicales, vous pouvez afficher ou modifier les informations de fichiers musicaux.

Pour afficher ou éditer les informations musicales :

  1. Faites un clic droit sur le fichier musical et choisissez Modifier les informations musicales.
  2. Ici, vous pouvez afficher ou modifier les informations du fichier musical ou choisir une langue dans le menu déroulant Page de code pour convertir le texte non-Unicode en Unicode et afficher le texte dans la langue appropriée. (Paramètres système par défaut est la page de code que vous avez spécifiée dans Menu principal > Panneau de configuration > Options régionales > Langue.)
  3. Cliquez sur Enregistrer pour terminer la configuration.

Remarque : L'éditeur d'informations musicales prend en charge les formats suivants : mp3, ogg.
