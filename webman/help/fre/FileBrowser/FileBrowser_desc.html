File Station

File Station est un outil de gestion des fichiers convivial, permettant aux utilisateurs de DSM sur Internet d'accéder aux dossiers de DiskStation avec leurs navigateurs Web. En lançant deux File Station ou plus, vous pouvez gérer toutes vos données DiskStation par glisser-déplacer entre différents File Station.
