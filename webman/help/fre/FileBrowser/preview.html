Visionner des documents, photos ou vidéos

Avec File Station, vous pouvez visionner facilement des documents, photos ou vidéos.

Pour changer la manière dont les fichiers sont affichés :

Vous pouvez basculer entre la Vue liste et la Vue icônes pour changer la manière dont les fichiers sont affichés dans la zone principale. Cliquez sur le bouton Vue liste dans le coin supérieur droit pour organiser les fichiers en une liste ou cliquez sur le bouton Vue icônes pour les afficher avec des icônes et du texte.

Vous verrez des miniatures des photos en Vue icônes dans les conditions suivantes :

  • Les photos se trouvent dans des dossiers indexés listés sous Panneau de configuration > Service d'indexation > Indexation multimédia > Dossier indexé, et vous avez déjà installé Photo Station ou le Media Server sur votre DiskStation.
  • Les données de miniatures stockées dans les entrées EXIF créées par votre appareil photo numérique sont disponibles.

Pour afficher les documents dans Google docs :

Faites un clic droit sur le document et choisissez Afficher dans Google docs.

Formats pris en charge :

La visionneuse de docs Google prend en charge les formats suivants : doc, docx, ppt, xlsx, pdf, tif

Remarque :

Pour visualiser les documents dans Google docs, assurez-vous que votre machine DiskStation est pleinement accessible via Internet. Si votre DiskStation dépend d'un routeur, assurez-vous d'avoir bien spécifié la règle du port de transmission de DSM dans Panneau de configuration > Accès externe > Configuration du routeur. Les numéros de port par défaut pour DSM sont 5000 (HTTP) et 5001 (HTTPS). Si des problèmes persistent lorsque vous affichez des documents dans Google docs, il vous faudra peut-être spécifier le port du routeur pour DSM dans Panneau de configuration > Accès externe > Avancé.

Pour visualiser les documents dans Office Online :

Faites un clic droit sur le document et choisissez Afficher dans Office Online.

En utilisant Office Viewer pour visualiser les documents Microsoft Office hébergés sur votre machine DiskStation, vous pouvez accéder à ces documents avec les fonctions Office intactes dans un navigateur web sans avoir besoin d'installer Office sur votre ordinateur.

Formats pris en charge :

La visionneuse Office prend en charge les formats suivants :

  • Word : docx, docm, dotm, dotx
  • Excel : xlsx, xlsb, xls, xlsm
  • PowerPoint : pptx, ppsx, ppt, pps, pptm, potm, ppam, potx, ppsm

Remarque :

  • Pour visualiser les documents dans Office Online, assurez-vous que vous avez activé QuickConnect. Si non, veuillez vous connecter dans DSM en tant qu'utilisateur appartenant au groupe administrators et allez dans Panneau de configuration > QuickConnect.
  • Les documents Word et PowerPoint doivent être inférieurs à 10 Mo, et les documents Excel doivent être inférieurs à 5 Mo.
  • Le nom de fichier du document doit contenir uniquement des lettres et les chiffres.

Pour prévisualiser des photos :

  1. Faites un clic droit sur l'une des photos et choisissez Aperçu pour ouvrir la fenêtre Visionneuse photo.
  2. Passez votre souris sur le bas de la visionneuse de photos pour révéler la barre de défilement de miniatures en bas pour une navigation rapide.

Formats pris en charge :

La visionneuse de photos prend en charge les formats suivants : jpg, jpeg, jpe, bmp, gif, png, tiff, tif, arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, 3fr, erf, mos, orf, rw2, raw, psd

Pour lire (apercevoir) les fichiers sonores :

  • Pour lire les fichiers sonores : Double-cliquez sur un ou plusieurs fichiers sonores pour les ouvrir avec le lecteur Audio. Sinon, vous pouvez également lire les fichiers sonores en cliquant avec le bouton droit sur un fichier sonore et en sélectionnant Lire.*
  • Pour lire les fichiers sonores par dossier : Cliquez sur un dossier unique ou appuyez sur la touche Ctrl (pour les utilisateurs de Windows)/command (pour les utilisateurs de Mac) et cliquez sur plusieurs dossiers. Puis cliquez avec le bouton droit de la souris sur le ou les dossiers sélectionnés et choisissez Lire pour les ouvrir dans le Lecteur Audio.*

Formats pris en charge :

Le lecteur Audio prend en charge les formats suivants :

  • mp3, m4a, m4b, aac, ogg, wav, flac, ape, aiff, aif, wma

Remarque :

Vous pouvez lire jusqu’à 1024 fichiers sonores à la fois.

Pour lire des vidéos :

  1. Effectuez un clic droit sur la vidéo et choisissez Lecture pour ouvrir la fenêtre du lecteur vidéo.
  2. La lecture de la vidéo doit commencer immédiatement. Sinon, vérifiez que vous avez bien installé le lecteur VLC Player (Windows) sur votre ordinateur local.

    3. Remarque :

    • En installant VLC Player, veillez à cocher la case du plug-in Mozilla et du plug-in ActiveX.
    • VLC est pris en charge par les navigateurs suivants :
      Windows : Internet Explorer, Firefox, Google Chrome
    • Si VLC est installé sur votre ordinateur, et que vous vous connectez à DSM avec un navigateur Web pris en charge, le lecteur vidéo prend en charge les formats suivants : avi, dat, mkv, vob, asf, wmv, mov, qt, divx, flv, swf, rm, rmvb, mpg, mpeg, mpeg1, m1v, mpeg2, m2v, mp2, dvr-ms, m2ts, mts, m2t, ts, tp, trp, mpeg4, m4v, mp4, 3g2, 3gp, 3gp2, webm, mpe, xvid
    • Si VLC n'est pas installé sur votre ordinateur local, le lecteur vidéo lit les vidéos prises en charge par HTML5. Rendez-vous ici pour des vidéos HTML5 prises en charge par différents navigateurs Web.
    • Si vous vous connectez à DSM via le protocole HTTPS, vous devez importer un certificat de confiance (dans Panneau de configuration > Sécurité > certificat) pour pouvoir lire des vidéos avec File Station.
    • VLC ne prend pas en charge la lecture en continu HTTPS.
  3. Si vous disposez d'un fichier de sous-titres portant le même nom et se trouvant dans le même dossier que la vidéo, ou que votre vidéo contient des sous-titres intégrés, vous serez en mesure de voir les sous-titres pendant la lecture de la vidéo. Si vous souhaitez utiliser un autre fichier de sous-titres, choisissez-le dans le menu déroulant.
