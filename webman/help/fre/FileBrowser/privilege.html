Afficher ou modifier les permissions des fichiers/dossiers

Grâce à File Station, vous pouvez définir les privilèges d'accès aux fichiers et aux sous-dossiers. Cette fonction est particulièrement intéressante lorsque vous voulez autoriser différents groupes d'utilisateurs à accéder aux différents sous-dossiers d'un dossier partagé.

Pour autoriser les utilisateurs et les groupes à accéder aux fichiers et aux sous-dossiers :

  1. Sélectionnez les fichiers et les dossiers pour lesquels vous souhaitez définir les privilèges d'accès.
  2. Effectuez un clic droit sur un des fichiers ou dossiers, ou choisissez le menu Action, puis choisissez Propriétés.
  3. Dans la section Propriétaire,choisissez le propriétaire des fichiers et dossiers sélectionnés dans le menu déroulant Propriétaire.
  4. Cliquez sur l'onglet Autorisation, et sélectionnez un utilisateur ou un groupe. Cliquez sur Modifier pour ouvrir Inspecteur d'autorisation.
  5. Effectuez l'une des actions suivantes dans la fenêtre Modifier l'autorisation pour gérer les autorisations ACL pour le fichier ou le dossier :
    1. Utilisateur ou groupe : Spécifiez l'utilisateur ou le groupe dont vous désirez personnaliser les autorisations.
    2. Hériter de : Pour visualiser seulement. Afficher les informations ici pour voir si l'autorisation est héritée (d'un dossier parent) ou explicite (indiqué par Aucun).
    3. Type : Choisissez Autoriser ou Refuser pour accorder ou refuser l'autorisation à l'utilisateur ou au groupe.
    4. Appliquer à : Si vous créez une entrée d'autorisation pour un dossier, cochez les cases pour appliquer l'entrée à ce dossier, les dossiers (ou Dossiers enfants) ou fichiers (ou Fichiers enfants) dans ce dossier, ou tous les fichiers ou dossiers contenu dans ce dossier (ou Tous les descendants).
    5. Administration : Cochez Lire les autorisations, Changer les autorisations ou Prendre possession pour spécifier le privilège d'accès de l'utilisateur ou du groupe pour l'entrée de l'autorisation.
    6. Lecture ou Écriture : Cochez les cases dans ces sections pour modifier la permission de l'utilisateur ou du groupe pour le fichier ou dossier.
  6. Cliquez sur OK.

Remarque : Vous n’êtes pas autorisé à modifier les privilèges d’accès aux fichiers et sous-dossiers dans le dossier partagé photo.
