Partager les liens de fichier

Les fichiers ou dossiers stockés sur votre DiskStation peuvent être partagés rapidement et facilement. Effectuez un clic droit sur un fichier ou un dossier et sélectionnez Partager les liens de fichier. Ceci génère un lien et un code QR, qui peuvent être envoyés à des amis ou d'autres utilisateurs. Ils peuvent suivre le lien pour télécharger le fichier ou dossier sélectionné, qu'ils possèdent un compte DSM ou non.

Pour partager les liens de fichiers :

  1. Sélectionnez les fichiers et dossiers que vous souhaitez partager.
  2. Effectuez un clic droit sur l'un des fichiers ou dossiers sélectionnés ou cliquez sur le menu Action. Sélectionnez alors Partager les liens de fichier.
  3. Les liens de partage et codes QR sont affichés dans la fenêtre qui apparaît.
  4. Les options suivantes peuvent également être modifiées :
    • Période de validité : Vous pouvez personnaliser et les périodes d'accès et le nombre d'accès valides des liens partagés.
      • Personnaliser la période de validité : Vous pouvez déterminer la durée pendant laquelle un lien partagé demeure actif après sa création. Une fois la période écoulée, le lien devient invalide.
      • Personnaliser le nombre d'accès valides : Vous pouvez déterminer le nombre de fois où il est possible d'accéder à un lien partagé. Lorsque le nombre de fois est dépassé, le lien devient invalide.
    • Partager avec des utilisateurs DSM : Vous pouvez partager le lien avec les utilisateurs ou les groupes DSM en les sélectionnant depuis la liste déroulante. Veuillez remarquer que vous ne pouvez pas partager avec le compte et le groupe d'utilisateurs invité.
    • Partager avec d'autres : Vous pouvez partager des liens protégés par mot de passe avec d'autres personnes, dont des personnes n'utilisant pas DSM.

Pour activer les liens gofile.me :

  1. Allez dans Panneau de configuration > QuickConnect et assurez-vous que Activer QuickConnect est coché. Si vous n'avez pas configuré QuickConnect, vous pouvez le faire sur cette page.
  2. Puis, cliquez sur Avancé et assurez-vous que Partage de fichiers est coché.
  3. Les fichiers partagés peuvent être téléchargés sans compte DSM en accédant au lien de partage gofile.me.

Personnaliser une page de liens partagés

File Station vous offre plusieurs options vous permettant de changer le style d'une page de liens partagés.

  1. Accédez à File Station > Paramètres > Liens partagés > Personnalisé.
  2. Sélectionnez Personnaliser une page de liens partagés et cliquez sur Paramètres personnalisés.
  3. Vous pouvez personnaliser la page de l'une des manières suivantes :
    • Modèle : Choisissez un modèle par défaut :
    • Arrière-plan : Cliquez sur la miniature ou sur + pour choisir un papier peint par défaut, une image sur votre DiskStation ou une image que vous avez précédemment appliquée.
    • Logo : Cliquez sur la miniature ou sur + pour choisir une image transférée (de moins de 1 Mo), une image sur votre DiskStation ou une image précédemment appliquée. Pouvez placer le logo dans l'un des quatre coins.
  4. Cliquez sur Aperçu pour voir si des ajustements sont nécessaires.
  5. Cliquez sur OK pour appliquer les paramètres.

Remarque :

  1. Les paramètres personnalisés sont appliqués à tous les liens partagés, quelles que soit leur heure de création.
  2. Vous ne pouvez appliquer des images en tant que papier peint/logo que dans les formats suivants : .jpe, jpeg, .jpg, .bmp, .gif, and .png

Gestion des liens partagés

Il est possible de gérer les liens partagés dans Outils > Gestionnaires de liens partagés. La fenêtre qui apparaît comporte des options pour modifier, partager ou supprimer les liens partagés existants.

Remarque :

  1. Vous pouvez créer jusqu'à 1000 liens partagés.
  2. La section État de chaque lien partagé affiche l'une des valeurs suivantes :
    • Valide : Le lien partagé fonctionne correctement.
    • Expiré : La période de validité du lien partagé a expiré.
    • Non valide : Les fichiers ou dossiers partagés n'existent plus, ou vous ne possédez pas les privilèges appropriés pour accéder au fichier ou le dossier.
    • Pas encore effectif : Le lien partagé n'est pas encore devenu actif.
