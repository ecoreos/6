Créer Mes favoris ou des raccourcis bureaux

Avec File Station, vous pouvez ajouter des dossiers DiskStation à Mes favoris ou créer des raccourcis du bureau pour les fichiers ou dossiers DiskStation.

Ajouter des dossiers à Mes favoris

Pour les sous-dossiers fréquemment utilisés sur votre DiskStation, vous pouvez ajouter leurs raccourcis à la section Mes favoris en haut du panneau gauche pour un accès plus rapide.

Pour ajouter un dossier DiskStation à Mes favoris :

  1. Sélectionnez le dossier, puis procédez d'une des manières suivantes :
    • Choisissez le menu Action ou cliquez-droit sur le dossier, puis choisissez Créer un raccourci > Mes favoris.
    • Si vous avez déjà ajouté un autre dossier à Mes favoris, vous verrez la section Mes favoris dans le coin supérieur gauche. Faites glisser le dossier vers cette section pour créer un raccourci.
  2. Dans la fenêtre qui apparaît, nommez le raccourci et cliquez sur OK.

Pour changer l'ordre d'un raccourci de Mes favoris :

Faites glisser le raccourci dans la section Mes favoris pour changer son ordre.

Pour renommer un raccourci :

  1. Cliquez-droit sur le raccourci puis choisissez Renommer.
  2. Dans la fenêtre qui apparaît, renommez votre favori et cliquez sur OK.

Pour supprimer un raccourci :

Cliquez-droit sur un raccourci, choisissez Supprimer de Mes favoris, puis cliquez sur Oui.

Pour effacer tous les raccourcis qui lient vers des chemins de dossier non existants :

Cliquez-droit sur Mes favoris en haut du panneau gauche, puis choisissez Effacer les favoris incorrects.

Créer des raccourcis du bureau pour les fichiers ou dossiers

Pour les fichiers ou sous-dossiers fréquemment utilisés de DiskStation, vous pouvez créer des raccourcis du bureau pour un accès plus rapide.

Pour créer des raccourcis du bureau pour un fichier ou dossier DiskStation :

Cliquez-droit sur le fichier ou dossier dans la zone principale, puis procédez d'une des manières suivantes :

  • Si vous créez un raccourci du bureau pour un fichier, choisissez Créer un raccourci du bureau.
  • Si vous créez un raccourci du bureau pour un dossier, choisissez Créer un raccourci > Bureau DSM.
