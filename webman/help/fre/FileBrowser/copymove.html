Copier ou déplacer les fichiers ou dossiers

File Station vous offre diverses méthodes de transfert de fichiers/dossiers entre deux DiskStation, sur un même DiskStation ou sur un ordinateur local.

Glisser et déposer des fichiers/dossiers entre des navigateurs

Vous pouvez, sans aucune configuration complexe de protocol de transfert de fichiers, transférer directement des fichiers/dossiers via File Station depuis un DiskStation vers un autre.

Pour activer le glisser et déposer entre les navigateurs :

  1. Accédez à Paramètres > Général > File Station.
  2. Cochez Activer le glisser et déposer entre les navigateurs.
  3. Cliquez sur OK.
  4. Ouvrez de navigateur Web de même sorte (Par ex., Firefox), connectez-les respectivement aux DiskStation source et cible.
  5. Lancez File Station sur les deux DiskStation. Vous pouvez maintenant glisser et déposer directement des fichiers/dossiers entre eux via File Station.

Remarque :

  • Vous pouvez également glisser et déposer des fichiers/dossiers entre deux onglets du même navigateur.
  • Internet Explorer ne prend pas en charge le glisser-déposer entre navigateurs ou entre onglets.
  • Si vous avez sélectionné plusieurs fichiers/dossiers pour le glisser-déposer, ils sont compressés en une archive pour le transfert.
  • La limite de taille du fichier pour le glisser-déposer varie en fonction des navigateurs : Safari (1 Go).

Copier ou déplacer des fichiers/dossiers dans un même DiskStation ou sur un ordinateur local

Suivez les étapes ci-dessous pour déplacer ou copier des fichiers ou dossiers dans un même DiskStation ou sur votre ordinateur local.

Méthode 1 :

  1. Sélectionnez les fichiers et dossiers que vous souhaitez déplacer ou copier. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.)
  2. Choisissez Couper ou Copier dans le menu déroulant Action.
  3. Allez au dossier de destination sur DiskStation ou votre ordinateur.
  4. Choisissez Coller - Écraser ou Coller - Ignorer dans le menu déroulant Action, selon que vous souhaitez écraser les fichiers ou dossiers avec le même nom dans le dossier de destination.

Méthode 2 :

  1. Sélectionnez les fichiers et dossiers que vous souhaitez déplacer ou copier. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.)
  2. Faites glisser n'importe quels dossiers ou fichiers de la section principale vers un autre dossier du panneau gauche.
  3. Dans le menu contextuel qui s'affiche, choisissez Copier - Écraser, Copier - Ignorer, Déplacer - Écraser ou Déplacer - Ignorer pour écraser ou ignorer les fichiers ou dossiers ayant le même nom dans le dossier de destination.

Méthode 3

  1. Sélectionnez les fichiers et dossiers que vous souhaitez déplacer ou copier. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.)
  2. Appuyez sur Ctrl + X pour couper ou Ctrl + C pour couper les fichiers/dossiers. Pour les utilisateurs Mac, appuyez sur Commande + X et Commande + C.
  3. Allez au dossier de destination sur DiskStation ou votre ordinateur.
  4. Appuyez sur Ctrl + V pour coller les fichiers/dossiers. (Commande + V pour Mac.)
  5. Dans le dialogue qui apparaît, choisissez si vous voulez écraser ou ignorer les fichiers ayant le même nom dans le dossier de destination.

Raccourcis clavier :

Les raccourcis clavier suivants peuvent être utiles lorsque vous transférez/copiez les fichiers et dossiers dans File Station :

  • F2 : Renommer
  • Suppr : Supprimer
  • Ctrl + A : Sélectionner tout
  • Ctrl + X : Couper
  • Ctrl + C : Copier
  • Ctrl + V : Coller

Remarque :

  • Lorsque vous copiez ou déplacez les fichiers/dossiers vers un répertoire sur un système de fichier FAT, les caractères \ *?">| dans le nom du fichier seront remplacés par – et les métadonnées des fichiers Mac seront perdues.
  • Si vous coupez les fichiers de votre ordinateur local, puis les collez dans un dossier de DiskStation, les fichiers seront conservés sur votre ordinateur local.
  • Lorsque vous déplacez un fichier ou un dossier, le "possesseur" reste le même. Cependant, lorsque vous copiez un fichier ou un dossier, le "possesseur" devient l'utilisateur qui l'a copié.
