Monter les dossiers distants

Vous pouvez monter les dossiers distants depuis les serveurs distants (comme les ordinateurs en réseau d'autres Synology NAS) pour les parcourir et gérer les dossiers montés via File Station.

Pour monter un dossier partagé CIFS* :

  1. Cliquez sur Outils > Monter le dossier distant > Dossier partagé CIFS.
  2. Dans le champ Dossier, spécifiez le chemin d'accès du dossier distant au format \\Remote_Server\Remote_Folder, tel que\\192.168.0.2\share ou \\chad\cheeseburger\pictures.
  3. Si le serveur distant nécessite des privilèges d'accès au dossier distant, saisissez vos identifiants de serveur distant dans les champs Nom de compte et Mot de passe.
  4. Cliquez sur Parcourir pour sélectionner ou créer un dossier de destination vide sur votre DiskStation pour monter le dossier distant.
  5. Cochez Monter automatiquement au démarrage si vous souhaitez que DiskStation monte ce dossier distant à chaque démarrage ou redémarrage du système.
  6. Cliquez sur Monter pour que le dossier distant soit monté dans le dossier de destination. Vous pouvez maintenant naviguer dans votre dossier distant et le gérer depuis votre DiskStation.

Pour monter un dossier partagé NFS :

  1. Cliquez sur Outils > Monter le dossier distant > Dossier partagé NFS.
  2. Dans le champ Dossier, spécifiez le chemin d'accès du dossier distant au format Remote_Server:/Remote_Folder_Path, tel que 192.168.0.2:/volume1/share.
  3. Sélectionnez la version NFS souhaitée et le protocole de transport.
  4. Cliquez sur Parcourir pour sélectionner ou créer un dossier de destination vide sur votre DiskStation pour monter le dossier distant.
  5. Cochez Monter automatiquement au démarrage si vous souhaitez que votre DiskStation monte ce dossier distant à chaque démarrage ou redémarrage du système.
  6. Cliquez sur Monter pour que le dossier distant soit monté dans le dossier de destination. Vous pouvez maintenant naviguer dans votre dossier distant et le gérer depuis votre DiskStation.

Remarque :

NFSv4 ne prend en charge le protocole TCP.

Pour démonter un dossier distant :

  1. Sélectionnez le dossier distant que vous souhaitez démonter et cliquez sur Outils > Démonter.
  2. Vous pouvez également tout simplement cliquer sur le bouton droit sur le dossier distant et sélectionner Démonter.

Remarque :

  • Le nombre maximum de dossiers distants simultanés est de 100.
  • Vous êtes seulement autorisé à monter les dossiers distants depuis un serveur distant qui prend en charge le protocole CIFS ou les protocoles NFS.
  • Les dossiers distants ne peuvent être montés que dans des sous-dossiers sous des dossiers partagés.
  • Le montage de dossiers distants n'est pris en charge qu'en mode à privilèges élevés. Pour utiliser cette fonction, accédez au paquet Docker sur votre DSM hôte, sélectionnez le Docker DSM pour lequel vous souhaitez activer des dossiers partagés chiffrés, cliquez sur Modifier et cochez la case en regard de Exécuter le conteneur à l'aide de privilèges élevés.

* Uniquement disponible sur certains modèles.
