Synchroniser des fichiers entre un Synology NAS et votre ordinateur à l'aide de Cloud Station

Vue d'ensemble

Cloud Station pour Synology NAS est une application de synchronisation de fichier qui vous permets de synchroniser facilement les fichiers sur votre Synology NAS avec d'autres appareils tels que les ordinateurs ou les appareils mobiles (avec DS Cloud). Installez Cloud Station Server sur votre Synology NAS et Cloud Station Drive sur votre ordinateur pour synchroniser automatiquement des fichiers stockés sur votre ordinateur sur le Synology NAS.

Configurer la connexion entre votre Synology NAS et l'ordinateur

  1. Accédez à Centre de paquets, recherchez Cloud Station Server et cliquez sur Installer.
  2. Lancez Cloud Station Server.
  3. Dans l'onglet Vue d'ensemble, cliquez sur le bouton  sous Cloud Station Drive pour télécharger Cloud Station Drive pour votre système d'exploitation. (Vous pouvez également télécharger Cloud Station depuis le Centre de téléchargement Synology.)
  4. Lancez l'installateur sur votre ordinateur et suivez les instructions affichées à l'écran.
  5. Après l'installation, lancez Cloud Station sur votre ordinateur. Cliquez sur Démarrer maintenant.
  6. Entrez l'adresse du Synology NAS qui exécute Cloud Station (ou QuickConnect ID), le nom d'utilisateur et le mot de passe. Vous pouvez cliquer sur l'icône de recherche située à droite pour rechercher des IP sur votre LAN. Cliquez sur Suivant.
  7. Sélectionnez les dossiers que vous souhaitez synchroniser sur votre Synology NAS et sur votre ordinateur en cliquant sur les icônes Modifier. Vous pouvez alors cliquer sur Avancé pour configurer les paramètres de synchronisation
  8. Décochez tous les sous-dossiers que vous ne souhaitez pas synchroniser.
  9. Configurez une taille maximale de fichier, des noms de fichier de la liste noire ou des types de fichier qui ne seront pas synchronisés.
  10. Sélectionnez une synchronisation bidirectionnelle ou un simple téléchargement de données depuis votre Synology NAS.
  11. Cliquez sur Effectué pour achever l'installation.
  12. Vous pouvez trouver l'icône de Cloud Station dans la barre d'état système.
  13. Cliquez sur l'icône de la barre d'état système pour ouvrir le menu de la barre d'état. Ici, vous pouvez voir la progression et l'état de vos fichiers. Ici, vous pouvez voir la progression et l'état de vos fichiers. Cliquez sur Application principale pour lancer Cloud Station.
  14. Lors du lancement, des conseils destinés à l'utilisation de Cloud Station s'affichent. Cliquez sur la flèche droite pour voir davantage de conseils ou sur Didact. en ligne pour davantage d'informations. Si vous n'avez pas besoin des conseils, cliquez sur la case en regard de Ne plus afficher ceci.
  15. Vous pouvez maintenant gérer vos tâches de synchronisation dans Cloud Station Drive.

Télécharger une version précédente d’un fichier synchronisé

  1. Cliquez sur l'icône Cloud Station Drive dans la barre d'état de votre système et cliquez sur l'icône du dossier.
  2. Effectuez un clic droit sur votre fichier sélectionné puis choisissez Synology Cloud Station > Parcourir les versions précédentes.
  3. Trouvez la version que vous souhaitez télécharger et cliquez sur l'icône de téléchargement.

Télécharger ou restaurer une version précédente d'un fichier synchronisé depuis Cloud Station Server

  1. Accédez à Cloud Station Server> Historique des versions localisez le fichier ou le dossier que vous souhaitez restaurer.
  2. Sélectionnez le fichier que vous voulez retrouver et cliquez sur Action > Parcourir les versions précédentes.
  3. Sélectionnez les versions du fichier que vous voulez télécharger et cliquez sur Télécharger. Ou si vous êtes sûr que vous voulez que la version sélectionnée soit restaurée, cliquez sur Restaurer et elle écrasera votre version actuelle.

Remarques importantes

  1. Gardez toujours des historiques de versions au cas où un fichier est supprimé ou écrasé par accident.
  2. Dans les Paramètres généraux, vous pouvez configurer si un fichier sera pré-extrait de votre NAS ou supprimé sur votre NAS lorsque vous supprimez un fichier sur votre ordinateur.
