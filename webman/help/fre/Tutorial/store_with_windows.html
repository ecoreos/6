Stocker les fichiers sur le NAS Synology depuis un PC Windows dans le réseau local

Vue d'ensemble

Le NAS Synology est conçu pour rendre le stockage et le partage de fichiers dans votre réseau local simple et rapide, vous permettant d'accéder directement aux dossiers et fichiers partagés sur le NAS Synology sans les inconvénients d'avoir vous connecter à DSM à chaque fois. Par exemple, vous pourrez stocker les fichiers sur votre NAS Synology avec l'Explorateur Windows, tout comme d'autres appareils réseau.

Connecter avec l'Explorateur Windows

  1. Ouvrez une fenêtre de l'Explorateur Windows et allez à Ordinateur.
  2. Cliquez sur Connecter un lecteur réseau. Cette action fait apparaître la fenêtre Connecter un lecteur réseau.
  3. Sur la fenêtre Connecter un lecteur réseau, choisissez une lettre de lecteur dans le menu déroulant Lecteur.
  4. Dans le champ Dossier, saisissez le nom de serveur de votre NAS Synology et le nom de dossier partagé et précédés et séparés des barres obliques inverses. Par exemple, si le nom de serveur de votre NAS Synology est "DiskStation", et le nom de votre volume est "Partage1", alors vous devez entrer "\\DiskStation\Partage1". Cliquez sur Terminer quand vous être prêt.
  5. Saisissez votre nom d'utilisateur DSM et votre mot de passe.
    6. Impossible d'accéder au dossier partagé ? Le compte utilisateur que vous entrez ici doit disposer des privilèges de d'accès pour le dossier partagé auquel vous souhaitez accéder.
  6. Enfin, une fenêtre de l'Explorateur Windows apparaît. Votre dossier partagé sera désormais mappé et accessibles sous Ordinateur.
