Sauvegarder les données sur mon PC/Mac en utilisant Cloud Station

Vue d'ensemble

Cloud Station Server n'est pas seulement un logiciel de synchronisation sophistiqué multi-plateforme. Les utilisateurs peuvent s'en servir comme moyen de sauvegarde plus intelligent de leurs ordinateurs personnels. Véritable solution de sauvegarde en temps réel, Cloud Station Backup est instantané et permet d'économiser de la bande passante tout en offrant une meilleure protection avec un historique pouvant comporter jusqu'à 32 versions dans un seul fichier.

Paramétrer Cloud Station Server sur votre Synology NAS

  1. Accédez à Centre de paquets, recherchez Cloud Station Server et cliquez sur Installer.
  2. Lancez Cloud Station Server.
  3. Accédez à Vue d’ensemble et cliquez sur Télécharger tout de suite dans Cloud Station Backup. (Vous pouvez également télécharger Cloud Station Backup  depuis le Centre de téléchargement Synology.)
  4. Lancez l'installateur sur votre ordinateur et suivez les instructions pour installer Cloud Station Backup.
  5. Après l'installation, lancez Cloud Station Backup sur votre ordinateur. Cliquez sur Démarrer maintenant.
  6. Entrez l'adresse du Synology NAS qui exécute Cloud Station Server (ou QuickConnect ID), le nom d'utilisateur et le mot de passe. (Vous pouvez également cliquer sur l'icône de recherche située à droite et Cloud Station Backup recherchera automatiquement d'autres Synology NAS sur votre LAN.) Cliquez sur Suivant.
  7. Sélectionnez la source de sauvegarde et décochez tous les sous-dossiers que vous ne souhaitez pas synchroniser.
  8. Sélectionnez un dossier sur votre Synology NAS en cliquant sur Sélectionner. Cloud Station Backup crée un dossier de sauvegarde sur la destination sélectionnée avec le nom de votre ordinateur.
  9. Cliquez sur Règles de la sauvegarde si vous souhaitez configurer des filtres ou des règles de synchronisation.
  10. Cliquez sur Suivant pour revoir les paramètres de sauvegarde, puis cliquez sur Effectué.
  11. Vous pouvez consulter la progression de votre sauvegarde dans l'application principale.

Télécharger ou restaurer une version précédente d'un fichier synchronisé depuis Cloud Station Server

  1. Accédez à Cloud Station Server> Historique des versions localisez le fichier ou le dossier que vous souhaitez restaurer.
  2. Sélectionnez le fichier que vous souhaitez récupérer et cliquez sur Action. Cliquez sur Parcourir les versions précédentes. Si le fichier a été supprimé ou effacé par erreur de votre ordinateur, vous pouvez cliquer sur Restaurer pour le restaurer.
  3. Sélectionnez les versions du fichier que vous voulez télécharger et cliquez sur Télécharger. Ou si vous êtes sûr que vous voulez que la version sélectionnée soit restaurée, cliquez sur Restaurer et elle écrasera votre version actuelle.
