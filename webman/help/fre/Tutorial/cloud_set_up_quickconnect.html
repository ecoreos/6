Accéder à votre Synology NAS par Internet et les fichiers de partage

Vue d'ensemble

Avec QuickConnect, vous pouvez facilement connecter votre Synology NAS à Internet sans le casse-tête du paramétrage des règles de transmission de port ou autre paramétrages de réseau compliqués. QuickConnect vous permet de vous connecter via une adresse personnalisable simple telle que Quickconnect.to/example.

1. Configurer QuickConnect

  1. Accédez à Panneau de configuration > QuickConnect.
  2. Cochez la case Activer QuickConnect.
  3. Si vous n'avez pas de compte Compte Synology, cliquez sur Se connecter ou enregistrer un nouveau Compte Synology. Entrez l'information demandée et cliquez sur OK. Si vous possédez déjà un compte Synology, saisissez les informations de votre compte.
  4. Entrez votre QuickConnect ID souhaité dans le champ QuickConnect ID. Puis cliquez sur Appliquer.
  5. Votre information de lien QuickConnect apparaîtra. Utilisez ces liens pour accéder à votre Synology NAS. Dans cette exemple, l'adresse pour DSM est http://Quickconnect.to/YourID.
  6. Si vous ne voyez pas l'adresse pour QuickConnect DSM, vous pouvez cliquer sur Avancé et vous assurer que DSM est activé.
  7. Vous pouvez maintenant utiliser le lien QuickConnect DSM pour accéder à votre Synology NAS !

2. Partagez les fichiers sur votre NAS en utilisant QuickConnect

Avec QuickConnect activé, vous pouvez facilement partager les fichiers stockés sur votre Synology NAS vers n'importe qui sans vous inquiéter de la transmission de port.

  1. D'abord, activez la connexion HTTPS pour partager vos fichiers en toute sécurité. Accédez à Panneau de configuration > Réseau.
  2. Cochez la case Activer les connexions HTTPS et cliquez sur Appliquer.
  3. Puis activez le partage de fichier pour QuickConnect. Accédez à Panneau de configuration > QuickConnect.
  4. Cliquez sur Avancé et assurez-vous que la case Partage de fichiers est coché.

    5. Remarque :

    Lors de la configuration des paramètres avancés QuickConnect, nous recommandons que vous vous identifiez dans DSM en utilisant l'adresse Ip locale ou le nom de domaine de votre Synology NAS.

  5. Vous être prêts à commencer le partage de fichiers en utilisant QuickConnect. Ouvrez File Station et cliquez-droit sur le fichier ou le dossier que vous voulez partager. Puis cliquez sur Partager les liens de fichier depuis le menu de contexte.
  6. Un lien de partage de fichier est généré. Partagez simplement ce lien avec la famille et les amis pour télécharger le dossier/fichier partagé.
