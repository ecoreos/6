Récupération

Cette page permet aux administrateurs système de procéder à une récupération. Pour permettre aux utilisateurs d'accéder aux instantanés des dossiers partagés, activezRendre l'instantané visible pour chaque dossier partagé afin que les utilisateurs puissent accéder aux instantanés via File Station ou Windows File Explorer.

Dossier partagé

Pour restaurer des fichiers/dossiers individuels dans un dossier partagé :

  1. Sélectionnez le dossier partagé que vous souhaitez restaurer.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez restaurer.
  4. Cliquez sur Parcourir et copiez les fichiers/dossier que vous souhaitez restaurer dans la destination.

Pour restaurer un dossier partagé :

  1. Sélectionnez le dossier partagé que vous souhaitez restaurer.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez restaurer.
  4. Cliquez sur Action, et sélectionnez Restaurer en place dans la liste déroulante.
  5. Un message de confirmation s'affiche. Cliquez sur Oui pour confirmer l'action.

Pour cloner un instantané dans un nouveau dossier partagé :

  1. Sélectionnez le dossier partagé que vous souhaitez cloner.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez cloner dans un nouveau dossier partagé.
  4. Cliquez sur Action et sélectionnez Cloner sous un nouveau nom dans la liste déroulante.
  5. Vous êtes redirigé vers la page Cloner de Panneau de configuration > Dossier partagé > Créer. Saisissez le nom du dossier partagé cloné et cliquez sur OK pour terminer.

Remarque :

  • Il est impossible de restaurer le dossier partagé « homes ».
  • Les dossiers partagés clonés ne peuvent pas être chiffrés.
  • La cohérence des données n'est pas garantie si un instantané est pris alors qu'un transfert de fichiers est encore en cours.
  • Limitations relatives aux dossiers partagés chiffrés :
    • Aucun type de récupération n'est disponible pour des dossiers partagés chiffrés montés.
    • Cloner sous un nouveau nom n'est pas disponible pour des dossiers partagés chiffrés.
    • Il est possible de restaurer des dossiers partagés chiffrés non montés à partir d'un instantané pris précédemment à l'aide de l'option Restaurer en place.

Pour basculer la réplication d'un dossier partagé :

  1. Sélectionnez le dossier partagé répliqué que vous souhaitez faire basculer.
  2. Cliquez sur Basculement.
  3. Sélectionnez le nouveau serveur de destination s'il existe plusieurs tâches de réplication du serveur partagé.
  4. Cliquez sur Basculement pour démarrer l'action.

Pour permuter une réplication de dossier partagé :

  1. Sélectionnez le dossier partagé répliqué que vous souhaitez permuter.
  2. Cliquez sur Permutation > Permutation.
  3. Sélectionnez la version de l'instantané à restaurer.
  4. Cliquez sur Permutation pour démarrer l'action.

Remarque :

  • Vous ne pouvez initier une permutation que lors d'une panne du serveur source.

Pour reprotéger la réplication d'un dossier partagé :

Lorsqu'elle est permutée sur le serveur de destination, la tâche de réplication cesse la synchronisation des données entre les deux serveurs. Vous devez reprotéger la tâche de réplication après la récupération du serveur source original.

  1. Sélectionnez le dossier partagé permuté que vous souhaitez reprotéger.
  2. Cliquez sur Permutation > Reprotéger.
  3. Sélectionnez le nouveau serveur source.
  4. Sélectionnez le nouveau serveur de destination s'il existe plusieurs tâches de réplication du serveur partagé.
  5. Cliquez sur Reprotéger pour démarrer l'action.

Remarque :

  • Les données du dossier partagé répliqué qui se trouvent sur le nouveau serveur de destination sont remplacées par celles du nouveau serveur de destination.
  • Si la version de l'instantané sélectionné pour la permutation n'existe pas sur le serveur partenaire, la reprotection peut prendre davantage de temps.

Pour procéder à une prise en charge test pour un dossier partagé répliqué :

  1. Sélectionnez le dossier partagé répliqué que vous souhaitez faire tester.
  2. Cliquez sur Prise en charge test > Prise en charge test.
  3. Sélectionnez Serveur test s'il existe plusieurs tâches de réplication du serveur partagé.
  4. Saisissez le Nom de la prise en charge test du dossier partagé, puis sélectionnez une version d'instantané à tester.
  5. Cliquez sur Prise en charge test pour démarrer l'action.

Remarque :

  • Un basculement/une permutation de réplication nettoie sa prise en charge test.
  • Vous ne pouvez pas créer une tâche de réplication pour un dossier partagé à prise en charge test.

Pour nettoyer une prise en charge test pour un dossier partagé répliqué :

  1. Sélectionnez le dossier partagé répliqué souhaitez nettoyer la prise en charge test.
  2. Cliaquez sur Prise en charge test > Nettoyage de la prise en charge test.
  3. Cochez la prise en charge test que vous souhaitez nettoyer.
    Note : Cochez Supprimer les données de prise en charge test si vous ne souhaitez plus conserver le dossier partagé créé pour la prise en charge test.
  4. Cliquez sur Nettoyage de la prise en charge test pour démarrer l'action.
