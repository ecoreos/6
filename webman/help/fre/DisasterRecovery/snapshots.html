Instantanés

Cette page vous permet de prendre, supprimer et modifier des instantanés ainsi que de configurer les paramètres de planification et de conservation des instantanés.

Vous pouvez stocker jusqu'à 256 copies d'instantanés en une fois pour chaque dossier partagé ou iSCSI LUN. Les instantanés permet de réduire la consommation d'espace disque en préservant des blocs individuels plutôt que des fichiers complets. Il est possible que de l'espace supplémentaire soit utilisé lorsque les fichiers qui se trouvent dans des blocs ou des dossiers partagés dans des iSCSI LUN sont mis à jour ou supprimés.

Dossier partagé

Pour prendre un instantané d'un dossier partagé :

  1. Sélectionnez le dossier partagé dont vous souhaitez prendre un instantané.
  2. Cliquez sur Prendre un instantané.
  3. Saisissez la description de l'instantané à prendre dans Description.
  4. Cochez/décochez la case Verrouiller pour indiquer si l'instantané doit ignorer les paramètres de conservation lorsque la suppression automatique est déclenchée.
  5. Cliquez sur OK pour prendre un instantané.

Remarque :

  • Parmi les paramètres de Panneau de configuration > Dossier partagé > page Modifier, seule la permission de Windows ACL est enregistrée dans un instantané.

Pour parcourir les instantanés des dossiers partagés :

  1. Sélectionnez un dossier partagé dont vous souhaitez parcourir les instantanés.
  2. Cliquez sur Liste d'instantannés.
  3. Sélectionnez un instantané que vous souhaitez parcourir, puis cliquez sur Parcourir.

Remarque :

  • Parcourir l'instantané est désormais disponible pour les dossiers partagés chiffrés.
  • Si vous n'êtes pas en mesure de parcourir l'instantané, accédez à Panneau de configuration > Dossier partagé > Modifier pour activer Rendre l'instantané visible.

Pour supprimer les instantanés de dossiers partagés :

  1. Sélectionnez un dossier partagé dont vous souhaitez supprimer les instantanés.
  2. Cliquez sur Liste d'instantannés.
  3. Sélectionnez un ou plusieurs instantanés que vous souhaitez supprimer et cliquez sur Supprimer. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.)
  4. Un message de confirmation s'affiche. Cliquez sur Supprimer pour confirmer la suppression.

Pour modifier les instantanés de dossiers partagés :

  1. Sélectionnez un dossier partagé dont vous souhaitez modifier les instantanés.
  2. Cliquez sur Liste d'instantannés.
  3. Sélectionnez un instantané que vous souhaitez modifier, puis cliquez sur Modifier.
  4. Vous pouvez modifier la description de l'instantané dans Description, puis cochez/décochez la case Verrouiller pour déterminer si l'instantané doit ignorer les paramètres de conservation lorsque la suppression automatique est déclenchée.
  5. Cliquez sur OK pour enregistrer vos paramètres.

Pour configurer des instantanés programmés :

Avec des instantanés programmés, Instantané et réplication prend automatiquement des instantanés à des moments et à une fréquence prédéterminés, et allège donc votre charge de travail de prise manuelle d'instantanés.

  1. Sélectionnez un dossier partagé.
  2. Cliquez sur Définir la programmation.
  3. Dans l'onglet Programmer, cochez la case Activer le calendrier d'instantané et configurez le moment et la fréquence de prise des instantanés. Pour annuler les instantanés programmés de ce dossier partagé, décochez la case.
  4. Cliquez sur OK pour enregistrer vos paramètres.
  5. Un message de confirmation s'affiche. Cliquez sur OK pour confirmer la configuration.

Remarque :

  • La configuration des instantanés programmés n'est pas incluse dans Panneau de configuration > Mise à jour et restauration > Sauvegarde de configuration.

Pour configurer les paramètres de conservation des instantanés programmés :

  1. Sélectionnez un dossier partagé.
  2. Cliquez sur Définir la programmation.
  3. Sélectionnez l’onglet Conservation.
  4. Configurez vos paramètres de conservation. Choisissez l'une des options suivantes :
    • Supprimer les anciens instantanés lorsque la limite est atteinte : Lorsque le nombre maximum d'instantanés est atteint, les instantanés les plus anciens sont remplacés par les nouveaux instantanés pris.
    • Toujours conserver les instantanés : Lorsque le nombre maximum d'instantanés est atteint, Instantané et réplication cesse de prendre de nouveaux instantanés sans supprimer ceux qui ont déjà été pris.
    • Conserver les instantanés selon la règle suivante : Vous pouvez configurer des règles de conservation pour des plages horaires spécifiées. Pour plus d'informations, veuillez consulter la note Politique de conservation.
  5. Cliquez sur OK pour enregistrer vos paramètres.

Remarque :

  • La modification des paramètres de conservation à une incidence sur les paramètres équivalents de la page Réplication.

iSCSI LUN

Pour prendre l'instantané d'un iSCSI LUN :

  1. Sélectionnez l'iSCSI LUN dont vous souhaitez prendre un instantané.
  2. Cliquez sur Prendre un instantané.
  3. Saisissez la description de l'instantané à prendre dans Description.
  4. Dans la liste déroulante Verrouiller, sélectionnez Oui/Non pour déterminer si l'instantané doit ignorer les paramètres de conservation lorsque la suppression automatique est déclenchée.
  5. Cliquez sur OK pour prendre un instantané.

Pour supprimer l'instantané d'un iSCSI LUN :

  1. Sélectionnez un iSCSI LUN dont vous souhaitez supprimer les instantanés.
  2. Cliquez sur Liste d'instantannés.
  3. Sélectionnez un ou plusieurs instantanés que vous souhaitez supprimer et cliquez sur Supprimer. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.)
  4. Un message de confirmation s'affiche. Cliquez sur Oui pour confirmer la suppression.

Pour modifier l'instantané d'un iSCSI LUN :

  1. Sélectionnez un iSCSI LUN dont vous souhaitez modifier les instantanés.
  2. Cliquez sur Liste d'instantannés.
  3. Sélectionnez un instantané que vous souhaitez modifier, puis cliquez sur Modifier.
  4. Vous pouvez procéder de l'une des manières suivantes pour modifier l'instantané :
    • Saisissez la description de cet instantané dans Description.
    • Sélectionnez l'État cohérent si un instantané conforme à l'application est activé.
    • Basculez entre Oui/Non dans la liste déroulante Verrouiller pour indiquer si l'instantané doit ignorer les paramètres de conservation lorsque la suppression automatique est déclenchée.
  5. Cliquez sur OK pour enregistrer vos paramètres.

Pour configurer des instantanés programmés :

Avec des instantanés programmés, Instantané et réplication prend automatiquement des instantanés à des moments et à une fréquence prédéterminés, et allège donc votre charge de travail de prise manuelle d'instantanés.

  1. Sélectionnez un iSCSI LUN.
  2. Cliquez sur Définir la programmation.
  3. Dans l'onglet Programmer, cochez la case Activer le calendrier d'instantané et configurez le moment et la fréquence de prise des instantanés. Pour annuler les instantanés programmés de ce dossier partagé, décochez la case.
  4. Cliquez sur OK pour enregistrer vos paramètres.

Pour configurer les paramètres de conservation des instantanés programmés :

  1. Sélectionnez un iSCSI LUN.
  2. Cliquez sur Définir la programmation.
  3. Sélectionnez l’onglet Conservation.
  4. Configurez vos paramètres de conservation. Choisissez l'une des options suivantes :
    • Supprimer les anciens instantanés lorsque la limite est atteinte : Lorsque le nombre maximum d'instantanés est atteint, les instantanés les plus anciens sont remplacés par les nouveaux instantanés pris.
    • Toujours conserver les instantanés : Lorsque le nombre maximum d'instantanés est atteint, Instantané et réplication cesse de prendre de nouveaux instantanés sans supprimer ceux qui ont déjà été pris.
    • Conserver les instantanés selon la règle suivante : Vous pouvez configurer des règles de conservation pour des plages horaires spécifiées. Pour plus d'informations, veuillez consulter la note Politique de conservation.
  5. Cliquez sur OK pour enregistrer vos paramètres.
  6. Un message de confirmation s'affiche. Cliquez à nouveau sur OK pour confirmer la configuration.

Pour configurer l'état cohérent de vos instantanés d'iSCSI LUN :

  1. Sélectionnez un iSCSI LUN.
  2. Cliquez sur Définir la programmation.
  3. Sélectionnez l’onglet Application.
  4. Cochez/décochez la case Activer des instantanés conformes à l'application pour basculer entre les étapes cohérents suivants :
    • Cohérent avec l'application : Des instantanés dans cet état cohérent ne peuvent être pris que lorsque Synology Snapshot Manager est installé sur votre serveur Windows ou sur VMware vCenter Server selon votre environnement pour communiquer avec DSM. Lorsqu'un instantané cohérent avec l'application est déclenché sur DSM, les applications sont notifiées et transfèrent toutes les données depuis la mémoire vers les LUN dont un instantané doit être pris, ce qui assure la cohérence des données.

      Remarque :

      • Si Synology Snapshot Manager n'a pas été installé dans votre environnement d'exploitation, l'option instantané cohérent avec l'application demeure désactivée, même si la case est cochée.
    • Cohérent avec le plantage : Si Synology Snapshot Manager n'est installé sur aucun de vos clients, vous ne pouvez capturer que des instantanés de LUN cohérents avec le plantage. Lorsque des instantanés cohérents avec le plantage sont déclenchés, les applications ne sont pas averties de transférer toutes les données de la mémoire dans les LUN, d'où une impossibilité de garantir la cohérence des données.
  5. Cliquez sur OK pour enregistrer vos paramètres.

Remarque :

  • La modification des paramètres d'état cohérent à une incidence sur les paramètres équivalents de la page Réplication.

Politique de conservation

Cette option offre une solution idéale pour vous aider à spécifier le nombre maximum de versions d'instantanés enregistrées dans l'espace de votre volume, mais vous pouvez être amené à conserver vos instantanés pendant une période plus longue.

Synology utilise la politique de conservation GFS, ou Grandfather-Father-Son. Vous pouvez configurer le nombre maximum de versions d'instantanés à conserver pour les plages horaires suivantes, respectivement : horaire, quotidienne, hebdomadaire, mensuelle et annuel.

Instantané et réplication conserve autant d'instantanés que le nombre maximum configuré pour chaque plage horaire. Si plus d'une version d'un instantané existe dans une plage horaire, seule la plus ancienne est conservée. Par exemple, si vous avez configuré une politique de 10 instantané(s) hebdomadaire(s), Instantané et réplication conserve l'instantané le plus ancien (si plus d'un instantané est pris au cours d'une semaine) pour chacune des 10 dernières semaines.

En outre, toutes les versions des instantanés dans une heure, depuis la prise de l'instantané le plus récent, sont conservées par défaut, pour vous aider à procéder commodément à la recherche et à la restauration parmi les versions récentes des instantanés. Vous pouvez également verrouiller un instantané afin d'empêcher sa suppression automatique selon vos règles de conservation.
