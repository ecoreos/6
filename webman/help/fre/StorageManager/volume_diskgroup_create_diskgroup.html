Créer un groupe de disques

Cet article explique comment créer un groupe de disques sur votre DiskStation.

Avant de commencer :

  • Assurez-vous qu'il n'y a aucune données importantes sur les disques durs. Toutes les données existantes seront supprimées lors de la création d'un nouveau groupe de disques.
  • Accédez à la page HDD/SSD et assurez-vous que le statut de chaque disque dur est Normal ou Non initialisé.

Pour créer un groupe de disques :

  1. Ouvrez Gestionnaire de stockage.
  2. Accédez à Groupe de disques et cliquez sur Créer.
  3. Lorsque l'assistant de création de groupe de disques apparaît, choisissez quels disques durs seront utilisés pour créer un groupe de disques. Puis suivez les étapes de l'assistant pour terminer.
