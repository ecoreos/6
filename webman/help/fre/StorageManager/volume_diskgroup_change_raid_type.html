Modifier le type RAID d'un Volume ou d'un groupe de disques

Les volumes et les groupes de disques peut être modifiés d'un type RAID à un autre sans la perte des données existantes. Par exemple, vous pouvez créer un volume RAID 1 sur votre DiskStation, et plus tard passer à RAID 5 si vous installez plusieurs disques durs. Cet article fournit les instructions pour modifier le type RAID des volumes et des groupes de disque existants sur votre DiskStation.

Limitations du type RAID

Seuls certains types RAID peuvent être modifiés à certains autres types RAID. Consultez le tableau ci-dessous pour plus de détails.

Type RAID Peut être changé en…
Basic RAID 1
RAID 5
RAID 1 RAID 5
RAID 5 RAID 6

Exemple :

Le RAID 1 peut être modifié au RAID 5 mais le RAID 5 ne peut pas être modifié en RAID 1.

Exigences des disques durs

Afin de changer le type de RAID d'un volume ou groupe de disques, vous devez avoir un nombre suffisant de disques durs disponibles installés sur votre DiskStation. Veuillez voir ci-dessous le nombre minimum de disques durs requis pour chaque type de RAID.

Type RAID souhaité Disques durs exigés
RAID 1 2 ou 4
RAID 5 3
RAID 6 4

Exemple :

Si vous voulez modifier un volume RAID 1 à RAID 5, vous aurez besoin d'au moins trois disques durs au total (c.-à-d. les deux disques durs appartenant au volume RAID 1, plus un disque dur libre pour ajouter le volume).

Changer le type de RAID

Dans l'exemple ci-dessous, nous modifierons le volume Basic à un volume RAID 1.

Avant de commencer :

  • Assurez-vous que l'état de votre volume ou du groupe de disques est Normal.
  • Lors de l'ajout de nouveaux disques durs à un volume ou groupe de disques, les nouveaux disques durs doivent être plus grands que le plus petit disque dur dans le volume ou le groupe de disques.
  • L'état des nouveaux disques durs doit être Initialisé ou Non initialisé.

Pour changer le type de RAID d'un volume :

  1. Ouvrez Gestionnaire de stockage.
  2. Accédez à Volume. (Si vous voulez modifier le type RAID d'un groupe de disques, allez à l'onglet Groupe de disques.)
  3. Sélectionnez le volume que vous souhaitez changer et cliquez sur Gérer.
  4. Lorsque l'assistant apparaît, sélectionnez Modifier le type de RAID. Puis cliquez sur Suivant.
  5. Sélectionnez le type RAID auquel vous voulez modifier le volume. Cliquez sur Suivant.
  6. Sélectionnez le ou les disque(s) dur que vous voudriez ajouter au volume existant. Puis cliquez sur Suivant.
  7. Suivez les instructions de l'assistant pour terminer. Veuillez noter que les données existantes sur le disque dur ajouté seront supprimées.
