Vue d'ensemble du stockage

La Vue d'ensemble du stockage indique le statut des disques durs installés, les connexions de câbles, et voyants d'état matériel.

Voyants d'état matériel

Les voyants d'état matériel sur la droite présentent les deux couleurs suivantes :

  • Vert : L'état matériel (p.ex. alimentation, ventilateur, connexions entre le serveur et les unités d'expansion, etc.) est normal.
  • Rouge : Une anomalie matérielle s'est produite.

Cliquer sur un voyant d'état ouvrira une liste détaillant l'état de tout le matériel, y compris alimentation, ventilateurs et connexions.

Afficher/Masquer le nombre de disques

Vous pouvez afficher ou masquer les nombres de chacun des disques avec le bouton Afficher/Masquer le nombre de disques.
