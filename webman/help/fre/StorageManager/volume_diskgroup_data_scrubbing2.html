Nettoyage des données*

Le nettoyage des données est un paramètre de maintenance des données qui inspecte les volumes ou groupes de disques. Il y a deux types de nettoyage des données :

  • Nettoyage du RAID : Cette fonction peut être utilisée avec les volumes ou groupes de disques qui implémentent SHR (comportant trois disques ou plus), RAID 5 ou RAID 6. Elle peut réparer toutes les incohérences de données détectées.
  • Nettoyage du système de fichiers : Vous pouvez utiliser cette fonction avec des volumes dans le système de fichiers Btrfs. Si des données sont incohérentes avec la somme de contrôle détectée, le chemin du fichier est enregistré dans le centre des journaux. Nous vous recommandons vivement d'utiliser une sauvegarde pour réparer les données.

Nous recommandons d'effectuer régulièrement le nettoyage des données afin d'assurer la cohérence des données et d'éviter de perdre des données en cas de panne de disque dur.

Pour effectuer le nettoyage des données :

  1. Ouvrez Gestionnaire de stockage.
  2. Accédez à Volume ou Groupe de disques.
  3. Sélectionnez le volume ou groupe de disques sur lequel vous souhaitez effectuer le nettoyage des données.
  4. Cliquez sur Gérer.
  5. Sélectionnez Nettoyage des données.
  6. Sélectionnez le type de nettoyage souhaité pour les données. (Le système ignore automatiquement cette étape si un seul type est disponible.)

Remarque :

  • Le nettoyage des données ne peut être effectué que lorsque le statut du volume ou groupe de disques est Normal.
  • Si le système est arrêté avant la fin, vous devrez redémarrer manuellement le nettoyage des données depuis le début.

____
*Uniquement pris en charge sur certains modèles.
