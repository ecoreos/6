HDD/SSD

Cette section vous permet de suivre le statut du HDD/SSD installé sur votre DiskStation, avec des options pour gérer et analyser les performances et l'état de santé du HDD/SSD. Allez dans Gestionnaire de stockage > HDD/SSD pour gérer ces options.

Pour vérifier les infos HDD/SSD :

  1. Accédez à l'onglet HDD/SSD.
  2. Sélectionnez un lecteur dans la liste et développez-le. Ici, vous verrez l'état S.M.A.R.T., les infos de volume, le nom du modèle, le numéro de série, la version du firmware et la température.

Pour vérifier l'état de santé du disque :

  1. Accédez à l'onglet HDD/SSD.
  2. Sélectionnez un lecteur.
  3. Cliquez sur Infos de santé.
  4. Accédez à l'onglet Vue d'ensemble. Ici, vous allez découvrir combien d'heures le lecteur a été alimenté, la température actuelle, le nombre de reconnections/réidentifications, le nombre de secteurs défectueux, et l'état S.M.A.R.T. général.
  5. Allez dans l'onglet Infos S.M.A.R.T.. Les chiffres répertoriés peuvent vous aider à diagnostiquer l'état de votre disque.

Remarque :

Les disques SAS ne prennent pas en charge les infos S.M.A.R.T.

Pour démarrer un test S.M.A.R.T. :

Le test S.M.A.R.T. examinera et fera un rapport sur l'état de vos disques durs, vous alertant des éventuelles défaillances du disque. Il est recommandé de changer rapidement vos disques si des erreurs sont détectées.

  1. Accédez à l'onglet HDD/SSD.
  2. Sélectionnez un lecteur.
  3. Cliquez sur Test S.M.A.R.T.
  4. Choisissez un type de test et cliquez sur Démarrer pour commencer le test.
  5. Les résultats du test sont affichés dans la section Résultats du test.

Remarque :

La durée du test varie en fonction de votre environnement système.

Pour planifier les tests S.M.A.R.T. :

Les tests S.M.A.R.T. peuvent être planifiés pour exécution automatique en créant des tâches.

  1. Accédez à l'onglet Planificateur de test.
  2. Cliquez sur le bouton Créer pour configurer une nouvelle tâche.
  3. Modifiez les paramètres suivants sur l'onglet Généralités :
    • Tâche : Entrez un nom pour la tâche.
    • Type de test S.M.A.R.T. : Choisissez d'effectuer un test rapide ou un test étendu.
    • Plage de test S.M.A.R.T. : Choisissez les HDD/SSD à tester chaque fois que le test S.M.A.R.T. est effectué.
  4. Sur l'onglet Planification, modifiez les paramètres suivants :
    • Date : Spécifiez quand effectuer le test S.M.A.R.T. Les tests S.M.A.R.T. peuvent être planifiés pour s'exécuter certains jours de la semaine (p.ex. tous les jours, les weekends ou tous les dimanches, etc.) ou à des dates spécifiques (par exemple 21/10/2015).
    • Nbre de fois : Spécifiez combien de fois le test S.M.A.R.T. est effectué, ainsi que la première et la dernière heure d'exécution chaque jour.
  5. Cliquez sur OK pour terminer.

Pour effectuer Secure Erase :

  1. Accédez à l'onglet HDD/SSD.
  2. Sélectionnez un lecteur et cliquez sur Secure Erase.

Remarque :

  • Secure Erase ne peut pas être utilisé sur les lecteurs dans les unités d'expansion.
  • Secure Erase va supprimer de manière permanente les données stockées sur votre lecteur. Assurez-vous que le lecteur choisi n'est pas utilisé ou que les données ont été sauvegardées avant de ce faire.
  • Lorsque le système effectue Secure Erase, le lecteur choisi sera temporairement verrouillé et ensuite déverrouillé automatiquement lorsque l'opération est terminée.
  • Si Secure Erase est arrêté incorrectement pendant l'opération, vous pouvez réexécuter Secure Erase directement sur l'interface utilisateur. Si les lecteurs sont supprimés ou utilisés ailleurs lorsqu'ils sont toujours verrouillés, veuillez entrer le mot de passe "synology" pour déverrouiller les lecteurs.

Activation de la prise en charge du cache en écriture pour le disque dur

L'activation de la prise en charge du cache en écriture améliore les performances de votre DiskStation. Seuls certains modèles de disques durs prennent en charge cette fonctionnalité. Pour assurer la protection des données avec la prise en charge du cache en écriture, l'utilisation d'un onduleur est fortement recommandée. Il est également suggéré que le système soit arrêté correctement à chaque fois après usage. La désactivation du cache en écriture permettra d'éviter les pertes de données causées par des pannes de courant anormales, mais les performances du système diminueront.

Pour activer la prise en charge du cache en écriture :

  1. Accédez à l’onglet Gestion du cache.
  2. Cochez Activer le cache d'écriture. Par défaut, cette fonction est désactivée.
  3. Cliquez sur Appliquer.
