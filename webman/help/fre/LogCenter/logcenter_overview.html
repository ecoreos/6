Aperçu

La page Aperçu du centre de journaux affiche l'activité du journal en temps réel, le nombre de journaux récents, les appareils source des journaux et une prévisualisation rapide d'au moins 50 journaux.

  • Le diagramme affichant l'activité du journal inclut les options pour modifier la période de temps à afficher. Vous pouvez cliquer sur les icônes + ou - dans le coin inférieur droit du diagramme pour définir la période de temps en : 6 dernières minutes, dernière heure, 6 dernières heures et dernier jour.
  • Vous pouvez basculer entre les sources des journaux (par ex. différents appareils DiskStation) pour afficher les menus déroulants situés en-dessous du diagramme.
  • Le graphique circulaire sur le côté supérieur droit illustre la distribution de tous les journaux dans votre base de données actuelle des journaux.
