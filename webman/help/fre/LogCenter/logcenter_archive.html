Paramètres d'archive

La page Paramètres d'archive fournit des options pour configurer comment et où les journaux sont archivés par le centre des journaux. Utilisez les options ci-dessous pour spécifier l'emplacement et les règles de stockage pour créer automatiquement des archives de journaux.

Destination de stockage pour les archives

La destination se réfère au dossier où les journeaux reçus par le centre des journaux sont stockés et archivés. Cliquez sur Sélectionner un emplacement pour choisir votre dossier.

Si vous voulez que les journaux locaux du DiskStation soient archivés au même emplacement, cochez la case marquée Archiver les journaux locaux vers l'emplacement de stockage spécifié ci-dessus. Dans la plupart des cas, ceci fournit plus d'espace de stockage pour les journaux locaux, vous permettant de maintenir plus de journaux systèmes complets.

Remarque :

Les journaux locaux (Système, Connexion et les journaux de transfert de fichier) sont sauvegardés dans une partition système qui a un stockage limité. En activant Archiver les journaux locaux vers l'emplacement de stockage spécifié ci-dessus, tous les journaux seront archivés à la destination spécifiée selon les Règles d'archive. Pour parcourir les journaux locaux archivés, cliquez sur Ouvrir une archive de journal et sélectionnez SYNOSYSLOGDB__LOCALARCH.DB.

Règles d'archive

Les journaux sont automatiquement archivés selon le critère défini dans la section Règles d'archivage. Les journaux peuvent être automatiquement archivés quand la taille de la base de données du journal atteint une taille spécifiée, le nombre de journaux individuels atteint une nombre spécifié ou les journaux atteignent un âge spécifié.

Format d'archive

Cette section inclut les options suivantes :

  • Archivez les journaux en format texte en plus du format par défaut : Sauvegarder les archives en fichiers texte simple en plus des fichiers SQLite par défaut.
  • Compresser les journaux archivés : Sauvegarde les archives de journaux en fichiers de format ZIP compressé.
  • Archiver les journaux séparément selon l'appareil : Sauvegarde les archives séparées pour les appareils client individuels qui envoient les journaux vers le DiskStation.
