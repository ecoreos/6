Réception des journaux

Votre DiskStation peut fonctionner comme un serveur syslog afin de recevoir des journaux d'un appareil client du réseau. Trois formats sont pris en charge pour la réception des journaux : BSD (RFC 3164), IETF (RFC 5424), et Personnalisé. Tous les journaux reçus sont stockés dans la base de données des journaux.

Avant de commencer :

Accédez à Centre des journaux > Paramètres d'archive et choisissez un emplacement de stockage et archivez les journaux.

Pour configurer la réception standard des journaux :

  1. Accédez à Centre des journaux > Réception des journaux > Standard.
  2. Vous pouvez activer la réception de journaux BSD et IETF en cochant l'option Recevoir des journaux d'autres appareils sous le ou les formats nécessaires.
  3. Sélectionnez UDP ou TCP depuis Protocole de transfert.
  4. Spécifiez un numéro de port pour la réception des messages syslog dans Port. Le numéro de port par défaut est 514.
  5. Si vous souhaitrez transférer des journaux en toute sécurité, cochez l'option Activer la connexion sécurisée (SSL). Cette option est disponible uniquement lorsque le protocole TCP est sélectionné.
  6. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour configurer la réception personnalisée des journaux :

  1. Allez dans Centre des journaux > Réception des journaux > Personnalisé.
  2. Cliquez sur Créer.
  3. Entrez un nom de règle unique dans Nom.
  4. Entrez une règle d'analyse dans Paramètres de règle. Veuillez lire la remarque ci-dessous pour obtenir des informations détaillées à propos de la création d'analyseurs de modèle.
  5. Sélectionnez UDP ou TCP depuis Protocole de transfert.
  6. Spécifiez un numéro de port pour la réception des messages syslog dans Port. Le numéro de port par défaut est 514.
  7. Si vous souhaitrez transférer des journaux en toute sécurité, cochez l'option Activer la connexion sécurisée (SSL). Cette option est disponible uniquement lorsque le protocole TCP est sélectionné.
  8. Cliquez sur OK pour enregistrer les paramètres.

Remarque :

Vous trouverez des informations détaillées à propos de analyseurs de modèle ici Vous pouvez également télécharger un exemple de modèle pour référence.

Pour gérer les règles personnalisées :

  1. Sélectionnez une règle personnalisée existante.
  2. Choisissez à partir des éléments suivants :
    • Modifier : Modifier une règle personnalisée.
    • Supprimer : Supprimer une règle personnalisée.
    • Activer : Activer une règle personnalisée.
    • Désactiver : Désactiver une règle personnalisée.

Exporter le certificat

Si vous avez coché l'option Activer la connexion sécurisée (SSL), vous pouvez cliquer sur le bouton Exporter le certificat pour exporter le certificat utilisé pour authentifier d'autres appareils lorsque vous envoyez des journaux dans DiskStation.
