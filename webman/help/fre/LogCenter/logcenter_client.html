Envoi des journaux

Si vous avez un serveur syslog, vous pouvez suivre les instructions ci-dessous pour configurer votre DiskStation en tant que client syslog et transmettre les journaux système locaux de votre DiskStation vers un serveur syslog pour la surveillance et la gestion centralisées.

Pour envoyer des journaux à un serveur syslog :

  1. Accédez à Centre des journaux > Envoi des journaux.
  2. Cochez la case Envoyer des journaux à un serveur syslog.
  3. Entrez le nom d'hôte ou l'adresse IP du serveur syslog dans Serveur.
  4. Spécifiez un numéro de port pour la réception des messages syslog dans Port.
  5. Sélectionnez UDP ou TCP depuis Protocole de transfert. Ceci doit correspondre au protocole utilisé par le serveur syslog.
  6. Sélectionnez BSD (RFC 3164) ou IETF (RFC 5424 depuis Format du journal pour envoyer des journaux en utilisant le format sélectionné.

    7. Remarque :

    • Par défaut, le numéro du port est 514. Les règles de transmission de port appropriées doivent être configurées si vous voulez réassigner un nombre de port différent au serveur syslog.
    • UDP (User Datagram Protocol) est un protocole plus simple de connexions basées sur les messages qui s'avère plus rapide mais auquel la sécurité et la correction des erreurs font défaut. TCP ( Transmission Control Protocol) offre une garantie de livraison dans le cadre de laquelle il utilise un contrôle du flux pour déterminer si tous les paquets ont été expédiés avec succès jusqu'au traitement de nouveaux paquets.
    • La connexion sécurisée (SSL) est disponible uniquement lorsque TCP est utilisé.
    • Si vous choisissez un protocole TCP et que vous voulez sécuriser la connexion syslog avec SSL, vous pouvez cocher l'option Activer la connexion sécurisée (SSL). Puis cliquez sur Importer le certificat et sélectionnez le certificat pour le transfert en sureté des journaux vers le serveur syslog.
  7. Cliquez sur Envoyer un journal de test pour confirmer que le serveur syslog peut recevoir le journal avec succès. Si le serveur syslog ne reçoit pas le journal test, veuillez vous assurer que vos paramètres sont corrects.
  8. Cliquez sur Appliquer pour enregistrer les paramètres.

Filtrage des journaux

Dans l'onglet Filtrer, vous pouvez spécifier quels types de journaux sont envoyés vers le serveur syslog, selon la catégorie (par ex. le système, la connexion, la sauvegarde) ou la sévérité (par ex. l'avertissement, l'erreur, l'information).
