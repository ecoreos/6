Restaurer

Accédez à Sauvegarde et restauration et cliquez sur Restaurer, dans le coin inférieur gauche pour restaurer des données sauvegardées.

Pour restaurer une sauvegarde de données/LUN* :

  1. Cliquez sur Restaurer dans le coin inférieur gauche, sélectionnez Données ou LUN*, et sélectionnez une tâche de sauvegarde à restaurer.
  2. Vous serez invité à sélectionner une configuration système, différentes versions des données de sauvegarde, ou davantage. Cela dépend du type de tâche de sauvegarde que vous souhaitez restaurer.
  3. Si la tâche de sauvegarde est cryptée, vous avez besoin d'un mot de passe/d'une clé de chiffrement pour réussir la restauration.
  4. Suivez l'assistant pour effectuer la restauration.

Remarque :

  • Lors de la restauration d'un paquet désinstallé, le système installe automatiquement la même version, ou la version la plus récente de l'application. Si l'application est installée, mais qu'il s'agit d'une version plus ancienne que celle sauvegardée, le système l'améliore automatiquement avec la version originale sauvegardée.
  • Les applications sauvegardées dans DSM 5.1 ou une version ultérieure ne peuvent pas être restaurées dans DSM 5.0 ou inférieur.

Remarque :

  • Une fois la restauration terminée, les LUN de niveau bloc et de niveau fichier sont restaurés sur des LUN de niveau bloc.
  • Lors de la restauration d'un paquet, si le paquet n'est pas installé, le système installe automatiquement la même version, ou la version la plus récente de l'application. Si l'application est installée, mais qu'il s'agit d'une version plus ancienne que celle sauvegardée, le système l'améliore automatiquement avec la version originale sauvegardée.
  • Les applications sauvegardées dans DSM 5.1 ou une version ultérieure ne peuvent pas être restaurées dans DSM 5.0 ou inférieur.

Pour restaurer un fichier ou un dossier unique pour une tâche de sauvegarde multi-versions :

  1. Sélectionnez la tâche de sauvegarde souhaitée dans le volet de gauche et lancez Explorateur de sauvegardes (en cliquant sur l'icône de la loupe). Le mot de passe/la clé de chiffrement est nécessaire si la tâche est cryptée.
  2. Localisez la version de la sauvegarde souhaitée à l'aide de la barre chronologique ou via l'icône du calendrier.
  3. Sélectionnez un fichier ou un dossier contenu et cliquez sur Restaurer, Copier vers... ou Télécharger pour récupérer le fichier/dossier sélectionné. Les trois options sont également disponibles si vous cliquez avec le bouton droit assurent les fichiers/dossier.

Pour rechercher une version spécifique :

Pour accélérer la vitesse de recherche des fichiers, vous pouvez utiliser des mots clés les fichiers/dossiers. Les fichiers/dossiers qui ne correspondent pas sont masqués.

  1. Lancez Explorateur de sauvegardes, et cliquez sur l'icône de l'entonnoir, dans le coin supérieur droit.
  2. Dans le menu déroulant, vous pouvez filtrer les fichiers/dossiers selon les critères ci-dessous :
    • Mot-clé : Saisissez le nom d'un fichier ou d'un dossier. Vous pouvez utiliser des caractères de remplacement [*] (représente un ou plusieurs caractères) et [?] (représente un caractère quelconque) dans vos mots-clés. Par exemple, [a*n] peut représenter [action], [attention], ou [an].
    • Type de fichier : Choisissez parmi Tout, Tous les dossiers ou Tous les fichiers pour affiner votre recherche.
    • Taille (Ko) : Choisissez Tout, ou sélectionnez équivaut à, est plus grand que, est inférieur à , puis saisissez une valeur.
    • Date de modification : Choisissez entre Aujourd'hui, Hier, La semaine dernière, ouLe mois dernier ou Personnaliser pour entrer une date de début et de fin pour trouver les versions dans une période spécifique.
  3. Pour appliquer vos filtres, cliquez sur Filtre.
  4. Pour appliquer les mêmes filtres à une autre version de sauvegarde, il suffit de passer à la version souhaitée à l'aide de la barre chronologique.
  5. Pour réinitialiser la recherche filtrée, cliquez sur le bouton Réinitialiser.

Remarque :

  • L'explorateur de sauvegardes est uniquement disponible pour les tâches de sauvegarde multiversions.
  • L'explorateur de sauvegardes possède des fonctions limitées lorsque la destination de la sauvegarde est configurée sous DSM 5.1 ou une version antérieure.
  • La configuration système et les applications ne peuvent être restaurées que via Restaurer.
  • La fonction de filtre est uniquement disponible pour l es tâches de sauvegarde non chiffrées.
  • Les fichiers/dossiers récupérés à l'aide de Copier vers... ou Télécharger perdent leurs paramètres de privilèges originaux.

Pour restaurer les tâches de sauvegarde depuis une autre destination de sauvegarde :

  1. Cliquez sur Restaurer dans le coin inférieur gauche, sélectionnez Données ou LUN*.
  2. Vous pouvez sélectionner et configurer l'un ou l'autres des types de tâche de sauvegarde :
    • Pour restaurer une sauvegarde de données : Cliquez sur Données > Restaurer à partir des référentiels existants, et sélectionnez la destination de sauvegarde souhaitée pour restaurer les données.
    • Pour restaurer une sauvegarde de LUN : Cliquez sur LUN*.
  3. Suivez les instructions de l'assistant. Le mot de passe/la clé de chiffrement est nécessaire si la tâche à restaurer est cryptée.
  4. Cliquez sur Appliquer pour terminer la restauration.

Remarque :

  • Embedded DataStation peut uniquement restaurer les dossiers partagés vers des périphériques externes ext3/ext4.
  • Embedded DataStation ne prend pas en charge la restauration des applications sans appareil externe ext3/ext4.

* Uniquement disponible sur certains modèles.
