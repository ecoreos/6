DSM mobile

DSM mobile permet à l'admin DSM (ou à un utilisateur appartenant au groupe administrators) de gérer les paramètres de DSM et de vérifier les informations DSM à l'aide d'un appareil iOS® (iPhone®, iPad®, ou iPod touch®) ou d'un appareil Android®. Cette aide de DSM mobile met à votre disposition les rubriques et ressources suivantes permettant de vous assister au début de votre utilisation.

Connexion et déconnexion

Une fois connecté au DSM à l'aide de votre navigateur Web sur votre appareil iOS ou Android, vous verrez l'écran de connexion dans lequel vous pouvez saisir les informations de votre compte DSM.

Pour se connecter à DSM mobile :

Saisissez vos identifiants d’utilisateur de DSM et appuyez sur Connexion. Si vous souhaitez enregistrer vos informations de connexion et vous connecter automatiquement la prochaine fois, cochez Se souvenir de moi avant de vous connecter.

Pour vous déconnecter de DSM mobile :

  1. Touchez le bouton d'option situé dans le coin supérieur droit de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (Tablette iPad ou Android).
  2. Appuyez sur Paramètres, puis sur Déconnexion.

Gérer les utilisateurs de DSM

Appuyez sur Utilisateur et groupe de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour afficher la liste d'utilisateurs.

Pour créer un utilisateur :

  1. Appuyez sur + puis saisissez le nom de l'utilisateur, son adresse électronique et le mot de passe du nouvel utilisateur.
  2. Activez Envoyer notification pour envoyer un e-mail de notification à l'adresse de l'utilisateur lorsque son compte utilisateur de DSM est créé.
    Remarque : Avant d'envoyer une notification, assurez-vous que les paramètres de notification ont été configurés sur votre Synology NAS. Connectez-vous à l'édition desktop de DSM avec le navigateur web de votre ordinateur, puis sélectionnez Menu principal > Panneau de configuration > Notification > E-mail pour modifier les paramètres.
  3. Appuyez sur Terminer pour confirmer ou sur la flèche gauche pour annuler.

Pour détecter rapidement un utilisateur :

Entrez le nom de l'utilisateur dans le champ Recherche.

Pour modifier les informations d'un utilisateur :

  1. Appuyez sur le nom de l'utilisateur et procédez comme suit :
    • Modifiez le nom de l'utilisateur et le mot de passe dans le champ Nom ou Mot de passe.
    • Appuyez sur Désactiver pour désactiver le compte de l'utilisateur. Vous pouvez choisir d'activer le compte de cet utilisateur plus tard et sans perdre ses paramètres de son compte.
    • Activez Administrateur pour ajouter l'utilisateur au groupe admin. Tous les utilisateurs appartenant au groupe admin bénéficieront de privilège administratif permettant de modifier les paramètres de DSM à l'aide de DSM mobile ou du navigateur Web de leur ordinateur.
  2. Sous Privilèges, effectuez l'une des opérations suivantes puis appuyez sur Retour :
    • Appuyez sur Dossier partagé et appuyez sur le nom du dossier partagé pour modifier le privilège du compte de l'utilisateur : Lecture/écriture : L'utilisateur peut effectuer des modifications sur les fichiers et sous-dossiers contenus dans le dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers contenus dans le dossier partagé, mais ne peut pas effectuer de modifications sur ces derniers. Pas d'accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
      Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Pas d'accès> Lecture/écriture > Lecture seule.
    • Appuyez sur Application pour modifier le privilège d'accès de l'utilisateur aux applications DSM.
  3. Appuyez sur Terminer pour confirmer ou sur la flèche gauche pour annuler.

Gérer des dossiers partagés

Appuyez sur Dossier partagé de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour afficher la liste des dossiers partagés sur votre Synology NAS.

Pour créer un dossier partagé :

  1. Appuyez sur + puis saisissez le nom du nouveau dossier.
  2. Appuyez sur Terminer pour confirmer ou sur la flèche gauche pour annuler.

Pour modifier un dossier partagé :

Appuyez sur le nom du dossier partagé et procédez comme suit :

  • Modifiez le nom du dossier partagé dans le champ Nom.
  • Appuyez sur Permissions utilisateur, puis sur un utilisateur pour modifier ses privilèges d'accès au dossier partagé : Lecture/écriture : L'utilisateur peut effectuer des modifications sur les fichiers et sous-dossiers contenus dans le dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers contenus dans le dossier partagé, mais ne peut pas effectuer de modifications sur ces derniers. Pas d'accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
    Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Pas d'accès> Lecture/écriture > Lecture seule.
  • Vous pouvez également appuyer sur Permissions des groupes pour configurer des privilèges pour des groupes.
    Remarque : Les permissions des groupes ont une priorité supérieure aux permissions des utilisateurs.

Activer les services

Touchez Service de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour activer ou désactiver les services du Synology NAS.

Pour gérer les services LAN du Synology NAS :

  • Activez ou désactiver l'un des services suivants :
    • Service de fichiers Windows permet à votre Synology NAS de devenir un serveur de fichiers Windows (CIFS) sur le réseau local.
    • Service de fichiers Mac permet à votre Synology NAS de devenir un serveur de fichiers Mac (AFP) sur le réseau local.
    • Service NFS permet aux clients Linux sur le réseau local d'accéder aux données du Synology NAS.
    • Imprimante Bonjour permet aux clients Mac d'imprimer vers le(s) imprimante(s) connectée(s) à votre Synology NAS.
    • Serveur multimédia permet aux périphériques domestiques DLNA/UPnP sur le réseau local de parcourir et de lire des contenus multimédias sur votre Synology NAS.
    • iTunes permet aux clients iTunes sur le réseau local de parcourir et de lire de la musique ou des vidéos sur votre Synology NAS.

Pour gérer les services Internet du Synology NAS :

  • Activez ou désactiver l'un des services suivants :
    • FTP permet à votre Synology NAS de devenir un serveur FTP.
    • Services Web vous permet d'héberger des sites Web (à l'aide de la prise en charge PHP+MySQL) sur votre Synology NAS.
    • WebDAV permet aux programmes client de gérer et d'éditer les fichiers du Synology NAS via le protocole HTTP.
    • WebDAV (HTTPS) permet aux programmes client de gérer et d'éditer les fichiers du Synology NAS via le protocole HTTPS.
    • File Station est une application Web vous permettant de gérer les fichiers du Synology NAS via un port personnalisé sans avoir à vous connecter au DSM.
    • File Station (HTTPS) vous permet de vous connecter à File Station via le protocole HTTPS.
    • Surveillance Station est une application Web qui peut gérer des caméras IP pour protéger l'environnement de votre domicile ou de votre bureau.
    • Audio Station est une application Web vous permettant d'accéder à la bibliothèque musicale de votre Synology NAS.
    • Photo Station est un album photo intégré à un blog en ligne enregistré sur votre Synology NAS.
    • Download Station (BT/HTTP/FTP/NZB/RSS) est une application Web vous permettant de télécharger des fichiers depuis Internet vers votre Synology NAS.
    • eMule permet de prendre en charge eMule sur Download Station.

Remarque :

Pour plus d'informations sur les services mentionnés ci-dessus, consultez l'une des méthodes suivantes :

  • Aide de DSM (disponible en vous connectant à l'Édition desktop du DSM à l'aide du navigateur Web de votre ordinateur et accédez à Menu principal > Aide de DSM)
  • Guide de l'utilisateur du DSM de Synology (disponible sur le Centre de téléchargement de Synology)

Contrôle des ressources du système

Appuyez sur Moniteur de ressources de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour surveiller le processeur et l'utilisation de mémoire du Synology NAS.

Afficher les infos système

Appuyez sur Info système de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour obtenir une vue d'ensemble du statut de votre Synology NAS.

Pour afficher les infos système de votre Synology NAS :

  1. Procédez de l'une des manières suivantes :
    • Appuyez sur Général pour afficher les informations de base de votre Synology NAS, notamment le numéro de série, le nom du modèle, la quantité de mémoire physique, la version de DSM, ainsi que les informations sur la durée de fonctionnement et les températures.
    • Appuyez sur Réseau pour afficher la configuration réseau de votre Synology NAS et le statut de chaque interface réseau.
    • Touchez Stockage pour afficher l'utilisation du volume du Synology NAS.
  2. Touchez le bouton Actualiser situé dans le coin supérieur droit pour que les informations système soient mises à jour.

Micrologiciel

Appuyez sur Micrologiciel pour afficher mettre à jour votre version de DSM.

Pour configurer les paramètres de mise à jour de DSM :

  1. Appuyez sur Mise à jour de DSM.
  2. Choisissez l'une des options suivantes :
    • Mises à jour importantes
    • DSM le plus récent et toutes les mises à jour
  3. Appuyez sur Effectué.

Pour télécharger les mises à jour disponibles :

  1. Vérifiez votre version de DSM dans État.
  2. Si votre version de DSM n'est pas à jour, appuyez sur le bouton Télécharger qui s'affiche en dessous de la section État pour mettre à jour DSM.

Gestion du blocage des IP

Appuyez sur Blocage d'une adresse IP de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour empêcher une connexion non autorisée.

Pour gérer le blocage des IP :

  1. Appuyez sur le bouton de configuration, dans le coin supérieur droit.
  2. Activez Activer le blocage auto pour activer le blocage des IP.
  3. Entrez un nombre pour les Tentatives de connexion et Sous (minutes) pour bloquer une adresse IP après un nombre spécifié d'échecs de tentatives de connexion sous le nombre de minutes spécifié.
  4. Vous pouvez activer Activer l'expiration des blocages, et spécifier Débloquer après (jours) pour débloquer une adresse IP après le nombre de jours indiqué. Si vous entrez « 0 » ici, l'Adresse IP bloquée restera bloquée indéfiniment jusqu'à ce que vous la retiriez de la liste des blocages.
  5. Appuyez surTerminer.

Pour ajouter des adresses IP à la liste des autorisations/des blocages :

  1. Si vous souhaitez ajouter l'IP autorisée/bloquée à la liste, appuyez sur le bouton d'ajout, dans le coin supérieur droit, puis sur Liste des permissions ou Liste des blocages.
  2. Après avoir entré l'adresse IP, appuyez sur Terminer.

Gérer le Centre de paquets

Synology a intégré des applications tierces et conçues par Synology dans des paquets pouvant être installés et activés facilement sur Synology NAS.

Pour gérer le Centre de paquets :

Appuyez sur Centre de paquets sur la page d'accueil (iPhone/iPod touch ou téléphone Android) ou sur le panneau gauche (tablette iPad ou Android) pour afficher les informations du paquet, exécuter ou arrêter des paquets ou procéder à la mise à jour avec les dernières versions.

Pour exécuter ou arrêter les paquets installés :

Appuyez sur le commutateur situé en dessous d'un paquet pour exécuter ou l'interrompre.

Pour afficher des paquets :

Appuyez sur un paquet pour afficher ses informations vidéo.

Pour mettre à jour des paquets :

  1. Appuyez sur l'onglet Mettre à jour en bas de la page pour afficher les informations des mises à jour actuellement disponibles.
  2. Appuyez sur le bouton situé à droite d'un paquet pour le mettre à jour avec la dernière version.

Afficher les journaux ou les connexions actuelles

Appuyez sur Journal sur la page d'accueil (iPhone/iPod touch ou téléphone Android) ou sur le panneau gauche (tablette iPad ou Android) pour surveiller les connexions actuelles ou afficher les journaux des services Synology NAS.

Pour surveiller les connexions actuelles :

  • Appuyez sur Les [nombre] derniers journaux dans la section Aperçu pour afficher la liste des derniers utilisateurs ayant accédé aux services du Synology NAS.

Pour consulter les enregistrements du journal d'un service spécifique :

  • Appuyez sur un élément dans la section Journaux d'événements ou Journaux de transfert de fichiers pour voir la liste des journaux en rapport.

Lancer des applications mobiles

DSM mobile fournit un endroit facile d'accès vous permettant de lancer des applications mobiles Synology installées sur votre appareil iOS ou Android.

Pour lancer les applications mobiles Synology avec DSM mobile :

Appuyez sur une des applications suivantes situées dans la section inférieure de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (Tablette iPad ou Android) :

  • DS photo (disponible lorsque le service Photo Station est activé)
  • DS audio (disponible lorsque le service Audio Station est activé)
  • DS cam (disponible lorsque le service Surveillance Station est activé)
  • DS file (disponible lorsque le service WebDAV est activé)

Remarque :

  • Pour plus d'informations concernant ces applications, reportez-vous aux fichiers d'Aide fournis avec les applications.

Configurer Synology NAS dans Action
(redémarrer, éteindre ou émettre des bips)

DSM mobile vous permet de redémarrer ou d'éteindre votre Synology NAS à l'aide d'un appareil iOS ou Android. De plus, lorsque plusieurs Synology NAS sont disponibles, vous pouvez rapidement identifier votre Synology NAS en lui demandant qu'il émette des bips.

Pour configurer votre Synology NAS avec le menu Action :

  1. Touchez le bouton d'option situé dans le coin supérieur droit de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (Tablette iPad ou Android).
  2. Procédez de l'une des manières suivantes :
    • Touchez Redémarrer puis touchez Oui pour redémarrer votre Synology NAS.
    • Touchez Éteindre puis touchez Oui pour éteindre votre Synology NAS.
    • Appuyez sur Me trouver pour demander à votre Synology NAS d'émettre un bip. Appuyez sur OK pour arrêter le signal sonore.

Basculer vers le mode bureau de DSM

Appuyez sur l'icône des paramètres et sélectionnez Mode bureau pour accéder au mode bureau de DSM.

Utilisation du gestionnaire de stockage

Le Gestionnaire de stockage vous permet d'afficher rapidement l'utilisation et l'état de santé des volumes et des disques du système.

Pour afficher les informations des volumes :

  1. Appuyez sur Gestionnaire de stockage de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android).
  2. Appuyez sur l’onglet Volume.
  3. Appuyez sur un volume dans la liste pour afficher ses informations détaillées.

Pour afficher les informations d'un disque :

  1. Appuyez sur Gestionnaire de stockage de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android).
  2. Appuyez sur l’onglet HDD/SDD.
  3. Appuyez sur un disque dans la liste pour afficher son état de santé et les informations S.M.A.R.T.

Pour démarrer ou arrêter le test S.M.A.R.T. :

  1. Appuyez sur Gestionnaire de stockage de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android).
  2. Appuyez sur l’onglet HDD/SDD.
  3. Appuyez sur un disque dans la liste pour afficher sa page d'information.
  4. Appuyez sur Test S.M.A.R.T. pour démarrer à un test S.M.A.R.T. ou sur Arrêt pour l'arrêter.
