Compte Synology

Dans Compte Synology, vous pouvez facilement accéder et gérer les paramètres de vos services de compte Synology et des eNews associées à vos DiskStation et paquets.

  • Compte : Afficher et modifier les informations de votre compte Synology.
  • Historique d'achat : parcourez et recherchez vos activités d'achats de paquets.
  • eNews : choisissez une ou plusieurs catégories d'information à recevoir de la part de Synology et personnalisez le contenu abonné.
  • Déconnexion : fermez tous les services du compte Synology associés à votre DiskStation.

Pour se connecter à votre compte Synology :

  1. Pour accéder au compte Synology.
  2. Dans la fenêtre pop-up, Se connecter à votre compte Synology, entrez l'adresse e-mail et le mot de passe que vous avez utilisé lors de l'enregistrement du compte Synology.
    • Si vous n'avez pas de compte Synology, cliquez sur Besoin d'un compte Synology ? pour l'enregistrement.
    • Si vous oubliez votre mot de passe, cliquez sur Mot de passe oublié ? pour qu'un nouveau mot de passe soit envoyé par e-mail.
  3. Cliquez sur Connexion pour accéder aux services de votre compte Synology.

Pour afficher et modifier le compte Synology :

  1. Cliquez sur Compte sur le panneau de gauche pour afficher l'information de votre compte.
  2. Cliquez sur le champ pour changer l'information du compte.
  3. Pour fixer les paramètres, cliquez sur Appliquer.

Pour afficher l'historique d'achat de votre DiskStation :

  1. Cliquez sur Historique d'achat sur le panneau gauche.
  2. Choisissez l'une des façons de trouver l'historique d'achat des paquets de votre DiskStation.
    • Pour le faire via les mots-clés, sélectionnez Recherche par mot-clé.
      • Entrez le mot-clé et sélectionnez le filtre approprié.
    • Pour le faire via la date d'achat, sélectionnez Recherche par date.
      • Cliquez sur les icônes calendrier pour spécifier une période de temps lorsque vous avez acheté votre DiskStation ou paquets.
  3. Cliquez sur Rechercher.
  4. Les informations demandées à propos des achats (par ex. la date d'achat et le prix) s'affichent.

Pour s’abonner aux eNews Synology :

  1. Cliquez sur eNews sur le panneau gauche.
  2. Dans Paramètres eNews, cochez une ou plusieurs catégories eNews à recevoir de la part de Synology.
  3. Vous pouvez affiner les paramètres des eNews dans les listes déroulantes :
    • Nouveautés sur les produits et fonctionnalités : Choisissez votre langue d'abonnement.
    • Information de première main sur les événements promotionnels locaux : Sélectionnez un pays pour découvrir les événements locaux Synology qui s'y tiennent.
    • Mises à jour importantes pour DSM : Sélectionnez une ou plusieurs version de DSM pour les dates correspondantes.
  4. Pour fixer les paramètres, cliquez sur Appliquer.

Pour vous déconnecter du compte Synology :

  1. Cliquez sur la touche Déconnexion dans le coin inférieur gauche.
  2. Lors de la déconnexion, tous les services du Compte Synology (par ex. DDNS et QuickConnect) et les paquets associés avec le compte d'identification ne seront pas disponibles pour votre DiskStation.

Remarque :

  1. Une fois que vous avez modifié l'adresse email, un message de confirmation sera envoyé à la nouvelle adresse pour vérifier le changement.
  2. Lors de la déconnexion, les packages resteront installés sur votre DiskStation mais leurs licences auront expirées.
