Options

Le menu Options vous permet de modifier les options de compte personnel, de se déconnecter et d'arrêter ou de redémarrer le système. Il peut être trouvé en cliquant sur l'icône en forme de personne dans le coin supérieur droit du bureau DSM. Après avoir affiché le menu, cliquez sur Perso. pour modifier les paramètres de compte personnel.
