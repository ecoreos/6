Menu principal

Le Menu principal inclut une liste d'applications et de paquets qui sont actuellement installés sur votre DiskStation. Cliquez sur l'icône dans le coin supérieur gauche du bureau du DSM pour le faire apparaître et lancer les applications ou les paquets. Vous pouvez également faire glisser les applications ou les paquets vers le côté du menu pour créer des raccourcis bureau.
