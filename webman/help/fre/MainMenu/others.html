Autres

Vous pouvez personnaliser les autres options du compte personnel dans Perso. > Autres.

Pour personnaliser d'autres options personnelles :

  1. Choisissez l’une des options suivantes :
    • Ne pas afficher la boîte de dialogue de confirmation à la fermeture de DSM : Ignorer la boîte de confirmation lorsque vous fermez la fenêtre du navigateur :
    • Reprendre DSM à mon statut de déconnexion précédent lorsque je me connecte : Se souvenir des applications ou des paquets ouverts et les restaurer la prochaine fois que vous vous connectez ou que vous rafraîchissez la fenêtre de navigation.
    • Activer la prévisualisation instantanée sur la barre des tâches :Affichez un aperçu rapide d’une application ouverte quand vous passez sur son icône d’application sur la barre des tâches : Il est possible qu'Internet Explorer ne prenne pas complètement en charge cette fonction s'il ne s'agit pas de la dernière version.
    • Ne pas lancer automatiquement l'aide de DSM à chaque connexion : La page d'aide de DSM n'est pas automatiquement ouverte lorsque vous vous connectez à DSM.
    • Activer les raccourcis clavier du bureau : Vous pouvez utiliser les raccourcis clavier pour effectuer plusieurs types d'opérations du bureau et de DSM (par ex., commutation entre les fenêtres activées, sélectionner/désélectionner des options, etc.). Pour afficher tous les raccourcis clavier, appuyez sur Maj + ?.
    • Désactiver l'accessibilité (pas de conseils audio disponibles) : Suspend la fonction d'accessibilité sous DSM afin que vous n'entendiez pas de conseils audio lorsque le lecteur d'écran est activé.
    • Activer les notifications du navigateur sur le bureau : Afficher les notifications depuis votre navigateur web sur le bureau DSM. Les notifications du navigateur sur le bureau sont prises en charge par Google Chrome et Safari seulement. Selon la version de votre navigateur Web, la page des paramètres de notification peut être différente. Veuillez aller à la page des paramètres de votre navigateur pour gérer les paramètres des notifications.
    • Activer le bouton Pilot View dans la barre de tâches : Affiche le bouton Pilot View pour vous aider à voir les applications ouvertes en mode d'aperçu.
  2. Cliquez sur OK.
