Parcourir le bureau

DiskStation Manager (DSM) est un système d'exploitation basé sur le web conçu spécifiquement pour les produits Synology. Avec DSM, vous pouvez accomplir rapidement et facilement ce qui suit sur votre DiskStation :

  • Modifier et personnaliser les paramètres du système
  • Installer et ouvrir les applications et les paquets
  • Créer des volumes RAID et gérer les autres types d'espaces de stockage
  • ...Et plus encore!

Apprenez-en plus sur DSM.

Pour en savoir plus à propos de toutes les propriétés disponibles sur DSM, veuillez accéder à www.synology.com.

S'identifier dans DSM

Après avoir paramétré et installé DSM sur votre DiskStation, vous pouvez vous identifiez à l'aide d'un navigateur sur votre ordinateur.

Installation de DSM

Pour les instructions étapes par étapes d'installation de DSM sur votre DiskStation, veuillez consulter le guide d'installation rapide de votre produit Synology disponible sur www.synology.com.

Pour vous identifier dans DSM :

  1. Assurez-vous que votre ordinateur et le DiskStation sont connectés au même réseau local.
  2. Ouvrez un navigateur sur votre ordinateur et entrez un des éléments suivants dans la barre d'adresse :
    • find.synology.com (Uniquement si votre ordinateur et le DiskStation sont connectés à Internet)
    • Le nom de serveur de votre DiskStation suivi de :5000, par exemple diskstation:5000 ou rackstation:5000 (Le nom spécifique dépend de vos paramètres configurés pendant l'installation initiale)
    • L'adresse IP de votre DiskStation suivie de :5000, par exemple 192.168.48.14:5000 (L'adresse IP dépend des paramètres configurés pendant l'installation initiale)
  3. Entrez votre nom d'utilisateur et votre mot de passe et cliquez sur Connexion.

Remarque :

Le nom, l'adresse IP et le mot de passe admin pour votre DiskStation sont paramétrés pendant l'installation initiale. Si vous avez des problèmes pour trouver cette information, essayez d'installer Synology Assistant ou accédez à find.synology.com.

Bureau DSM

Après vous être identifié, vous pouvez voir le bureau DSM. À partir de là, vous pouvez commencer à effectuer des tâches telles que gérer les paramètres, utiliser les paquets ou voir les notifications.

Barre des tâches

La barre des tâches est située en haut de l'écran et inclut les éléments suivants :

  1. Afficher le bureau : Minimiser toutes les applications ouvertes et les fenêtres de paquet.
  2. Menu principal : Voir et ouvrir les applications et les paquets installés sur votre DiskStation. Vous pouvez également cliquer et faire glisser pour créer des raccourcis bureau.
  3. Ouvrir les applications : Affiche les applications et les paquets actuellement ouverts. Vous pouvez effectuer un clic droit et épingler les paquets dans la barre des tâches pour un accès plus rapide dans le futur.
  4. File d’attente des chargements : Apparaît lorsque vous commencez à télécharger des fichiers vers le DiskStation. Cliquez ici pour voir plus de détails, tels que la progression et la vitesse de chargement.
  5. Périphériques externes : S'affiche lorsqu'un périphérique externe (par ex., clé USB) est monté sur votre DiskStation.
  6. Notifications : Affiche les notifications telles que les erreurs, les mises à jour de statut et les notifications d'installation de paquet.
  7. Options : Arrêt, redémarrage, déconnexion ou modification des options du compte personnel.
  8. Options : Arrêtez, redémarrez, quittez ou modifiez les options de compte personnel.
  9. Rechercher : Trouver rapidement les applications spécifiques, les paquets ou les articles DSM Help ici.
  10. Widgets : Affiche ou cache les widgets.
  11. Pilot View : Voir une prévisualisation de toute l'application ouverte et des fenêtres de paquet.

Menu principal

Vous trouverez une liste des applications et des paquets installés sur votre DiskStation ici.

Pour créer des raccourcis bureau :

  1. Ouvrir Menu principal.
  2. Cliquer et faire glisser une application ou le paquet sur le côté.
