Photo Station

Photo Station personnel est un album photo en ligne avec blog, détenu et géré par un utilisateur de DSM. DiskStation fournit le dossier home/photopermettant de stocker les photos et les vidéos que vous souhaitez partager. Le système créera automatiquement des miniatures d’index des photos et des vidéos puis vos albums photos pourront être visionnés depuis un navigateur Web.

Pour activer Photo Station personnel :

Cliquez sur Activer Photo Station personnel - [nom d’utilisateur] et cliquez sur OK.

Pour modifier les paramètres de Photo Station personnel :

  1. Cochez Définir les dossiers comme albums publics si vous voulez que les visiteurs aient accès aux albums photos chargés.
  2. Cliquez sur Re-indexation pour reconstituer la structure de l'index des fichiers image si des images sont absentes dans Photo Station.
  3. Cliquez sur OK.

Pour lancer Photo Station, procédez de l’une des manières suivantes :

  • Allez dans Menu principal et cliquez sur Photo Station.
  • Saisissez l'adresse Web suivante dans la barre d'adresse de votre navigateur Web : http://Synology_Server_IP_Address/photo/

Pour lancer Photo Station personnel, procédez d’une des manières suivantes :

  • Cliquez sur Lancer Photo Station.
  • Allez dans Menu principal et cliquez sur Photo Station - [nom d'utilisateur]
  • Saisissez l'adresse Web suivante dans la barre d'adresse de votre navigateur Web : http://Synology_Server_IP_Address/~username/photo/

Prise en charge et limites :

  • Formats d'image pris en charge : BMP, JPEG, GIF, RAW (arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, raw, 3fr, erf, mef, mos, orf, rw2, dng, x3f), TIFF, PNG
  • Prend en charge les formats vidéo : 3G2, 3GP, ASF, AVI, DAT, DivX, FLV, M4V, MOV, MP4, MPEG, MPG, MTS, M2TS, M2T QT, WMV, XviD, RM(1), RMVB(1), VOB(1)
  • Si vous chargez du contenu avec Photo Station personnel, vous pouvez uniquement charger jusqu'à 2 Go de contenu en une fois.

    • Remarque (1) : Les utilisateurs peuvent télécharger les fichiers aux formats spécifiés mais ne peuvent pas les lire dans Photo Station en tant que vidéo flash compressée.

Comment utiliser Photo Station personnel ?

Vous pouvez charger des fichiers images ou vidéo dans le dossier home/photo et créer un blog. Pour ce faire :

  1. Organisez les photos et les vidéos que vous souhaitez partager dans des dossiers. Ces dossiers deviendront des albums dans Photo Station personnel.
  2. Chargez les dossiers album dans le dossier home/photo.
  3. Rendez-vous à l'adresse http://servername/~username/photo/ avec votre navigateur pour voir vos albums.
  4. Pour créer votre blog, cliquez sur Blog dans le coin supérieur droit et identifiez-vous pour pouvoir poster des messages.

Comment publier mes albums et mon blog sur Photo Station personnel ?

Pour publier mes albums et mon blog sur Photo Station personnel et les rendre accessibles depuis Internet, vous devez configurer DiskStation pour utiliser une IP externe.

  1. Connectez-vous à DiskStation en tant qu'admin ou en tant qu'utilisateur appartenant au groupe administrators.
  2. Pour raccorder DiskStation à Internet, vous pouvez appliquer :
    • Soit une IP externe. Accédez à Menu principal > Panneau de configuration > Réseau pour éditer les paramètres réseau du serveur si vous recevez une adresse IP externe spécifique de votre fournisseur de services réseau.
    • Paramétrage de PPPoE. Allez dans PPPoE en passant par Menu principal > Panneau de configuration > Réseau pour activer la connexion PPPoE.
  3. Si vous souhaitez utiliser un compte DDNS enregistré, allez dans Menu principal > Panneau de configuration > Accès externe > DDNS pour activer le service DDNS.
  4. Terminé !

Désormais, les internautes peuvent parcourir vos albums et votre blog en dirigeant leur navigateur sur http://serverIP/~username/photo/ ou http://DDNS_hostname/~username/photo/.
