Compte

L'onglet Compte fournit les options pour modifier les paramètres de compte basiques tels que la description, le mot de passe, l'e-mail et la vérification en 2 étapes. Pour voir cet onglet, cliquez sur l'icône en forme de personne en haut à droite du bureau et sélectionnez Perso..

Modifier les paramètres de compte

Les paramètres de compte suivants peuvent être modifiés :

  • Description : Entrez une description pour votre compte. Les descriptions de l'utilisateur peuvent inclure jusqu'à 64 caractères Unicode affichables.
  • Nouveau mot de passe : Si vous souhaitez changer le mot de passe de votre compte, entrez un nouveau mot de passe ici et confirmez-le dans le champ Confirmer le mot de passe ci-dessous. Les mots de passe doivent respecter la casse et peuvent être composés de caractères affichables (127 maximum), tels que lettres, nombres, symboles et espaces.
  • E-mail : Entrez une adresse e-mail (uniquement pour les utilisateurs non admin).
  • Langue d'affichage : Spécifiez la langue d'affichage pour votre compte. Les paramètres de langue d'affichage que vous spécifiez ici auront la priorité sur ceux spécifiés dans Panneau de configuration > Options régionales > Langue.

Après avoir modifié les paramètres de compte, cliquez sur OKpour enregistrer les modifications.

Vérification en 2 étapes

La vérification en 2 étapes fournit une sécurité améliorée pour votre compte DSM. Si la vérification en 2 étapes est activée, vous devrez entrer votre mot de passe en plus d'un code de vérification unique lorsque vous vous connectez à DSM. Les codes de vérification sont obtenus depuis une application d'authentification installée sur votre appareil mobile. Par conséquent, si quelqu'un veut accéder à votre compte, il devra non seulement posséder votre nom d'utilisateur et mot de passe, mais aussi votre appareil mobile.

Configuration requise :

La vérification en 2 étapes nécessite un appareil mobile et une application d’authentification qui prend en charge le protocole TOTP (Time-based One-Time Password). Les applications d'authentification comprennent Google Authenticator (Android/iPhone/BlackBerry) ou Authenticator (Windows Phone).

Pour activer la vérification en 2 étapes :

  1. Allez dans Perso. > onglet Compte et cochez la case Activez la vérification en 2 étapes pour lancer l'assistant de configuration de la vérification en 2 étapes.
  2. Sur le premier écran, entrez une adresse e-mail. Des codes de vérification d'urgence peuvent être envoyés à cette adresse e-mail au cas où votre appareil mobile serait perdu.
  3. Téléchargez et installez une appli d'authentification, telles que Google Authenticator (Android/iPhone/BlackBerry) ou Authenticator (Windows Phone).
  4. Sur l'écran suivant, ouvrez votre appli d'authentification et numérisez le code QR. Vous pouvez également cliquer sur le lien manuel pour entrer une clé secrète.
  5. Ensuite, votre appli d'authentification génère un code de vérification à 6 chiffres. Entrez ce code dans le champ de texte de l'assistant afin de confirmer que les configurations sont correctes. Si une erreur se produit, veuillez vous assurer que l'heure système de votre appareil mobile est synchronisée avec l'heure système de DSM. En outre, les codes de vérification sont mis à jour régulièrement, il faut donc vous assurer que le code que vous entrez n'est pas expiré.
  6. Une fois l'assistant de configuration terminé, cliquez sur OK pour enregistrer les paramètres.

Pour vous connecter à DSM avec la vérification en 2 étapes :

Quand la vérification en 2 étapes est activée, vous êtes invité à entrer un code de vérification à 6 chiffres lorsque vous vous connectez à DSM.

  1. Sur l'écran de connexion de DSM, entrez votre nom d'utilisateur et le mot de passe comme d'habitude.
  2. Quand vous êtes invité à entrer un code de vérification, ouvrez l'appli d'authentification sur votre appareil mobile.
  3. Trouvez et entrez le code de vérification à 6 chiffres pour votre compte. Si votre appareil mobile est perdu, vous pouvez cliquer sur le lien Vous avez perdu votre téléphone ?, et un code de vérification d'urgence sera envoyé à votre adresse e-mail.

Remarque :

  • Paramètres SMTP : Pour recevoir des codes de vérification d'urgence via e-mail, les paramètres de serveur SMTP situés dans Panneau de configuration > Notification doivent être configurés correctement.
  • Limite du code d’urgence : Chaque utilisateur a une limite de 5 codes d'urgence. Si vous dépassez votre limite, vous devrez désactiver et activer la vérification en 2 étapes avant de pouvoir recevoir d'autres codes d'urgence.
  • Il est possible de forcer la vérification en 2 étapes sous DSM et toutes les applications mobiles Synology connexes (par ex. DSM mobile et DS file).

Pour réinitialiser les paramètres de la vérification en 2 étapes des utilisateurs :

Les utilisateurs appartenant au groupe administrators peuvent réinitialiser les paramètres de la vérification en 2 étapes des autres utilisateurs.

  1. Accédez à Panneau de configuration > Utilisateur.
  2. Double-cliquez sur l'utilisateur dont vous voulez réinitialiser la vérification en 2 étapes.
  3. Sur l'onglet Informations utilisateur, cliquez sur le bouton Réinitialiser la vérification en 2 étapes.

Remarque :

Si un utilisateur appartenant au groupe administrators n'est pas disponible, vous pouvez appuyer sur le bouton de réinitialisation physique sur votre DiskStation pour réinitialiser les paramètres DSM et désactiver la vérification en 2 étapes.

Activité du compte

L'activité du compte indique l'activité de connexion de votre compte DSM, notamment les connexions actuelles, les périphériques fiables et l'historique de connexion.

Pour vérifier l'activité de votre compte :

Cliquez sur le bouton Activité du compte. Une fenêtre contextuelle apparaît affichant les onglets suivants :

  • Utilisateurs connectés Cet onglet affiche les informations suivantes :
    • Date et heure : Quand le compte a été accédé.
    • Nom du client et IP : Nom et adresse IP de l'appareil qui a accédé au compte. Le nom du client apparaît uniquement lorsque le compte est accédé avec le protocole CIFS.
    • Service : Le protocole utilisé pour accéder au compte.
    • Ressources : Les ressources du système utilisées par le compte.
  • Périphériques fiables : Cet onglet affiche la page de gestion des périphériques fiables :
    • Le périphérique actuel n'est pas fiable / est fiable : Cliquez sur Faire confiance à ce périphérique si vous choisissez de faire confiance au périphérique actuel, et la vérification en 2 étapes ne sera plus nécessaire lorsque vous vous connecterez à DSM sur ce périphérique.
    • Ne pas faire confiance à d'autres périphériques : Vous pouvez ne pas faire confiance à d'autres périphériques (par ex., d'autre ordinateur ou des périphériques mobiles) qui étaient fiables. Cela signifie que la vérification 2 étapes sera nécessaire lorsque vous vous connecterez à DSM sur ces périphériques.
  • Historique de connexion : Cet onglet affiche les sessions de connexion récentes et les informations suivantes :
    • Date et heure : Quand le compte a été accédé.
    • Événement : Indique le nom d'utilisateur et l'adresse IP.

Remarque :

Le périphérique que vous avez marqué comme fiable sera mémorisé uniquement lorsque vous vous connecterez avec le même compte et à partir du même navigateur. Vous serez invité à saisir un OTP (mot de passe à usage unique) pour la vérification en 2 étapes dans les scénarios suivants, même si vous avez fait confiance à ce périphérique auparavant :

  • Vous connectez avec le même compte, mais sur un navigateur différent.
  • Vos cookies ont été supprimés du navigateur.
  • Vous faites confiance un périphérique lors de la connexion en tant qu'utilisateur A, et vous faites confiance au même périphérique lorsque vous vous connectez en tant qu'utilisateur B. Lorsque vous vous connectez à nouveau en tant qu'utilisateur A, vous devez encore saisir un mot de passe pour la vérification en 2 étapes.
