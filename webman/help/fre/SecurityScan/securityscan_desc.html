Security Advisor

Security Advisor est une application de sécurité DSM qui analyse vos paramètres de DSM et le DiskStation. Security Advisor vérifie vos paramètres et recommande les changements qui aident à maintenir votre DiskStation protégé. Voici une liste des fonctionnalités qu'il propose :

  • Security Advisor effectue différents vérification pour détecter les paramètres de DSM qui pourraient présenter des risques de sécurité, ainsi qu'une activité suspecte.
  • Security Advisor fournit des résultats détaillés et des rapports qui vous expliquent les actions recommandées à effectuer et ce dont vous bénéficierez ce faisant.
  • La base de données des définitions de sécurité interne se met automatiquement à jour vis-à-vis des cyber-menaces en évolution constante.

Pour lancer Security Advisor, effectuez ce qui suit :

  • Allez dans Menu principal > Security Advisor.
