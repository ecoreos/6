Résultats

Vous pouvez afficher l'information détaillées de votre analyse complète sur une base rapport par rapport. Cette page est présentée comme suit :

  • Gravité : Le niveau de risque global que Security Advisor évalue pour chaque règle de sécurité. Il y a quatre niveaux : Normal, Avertissement, Alarme, et Critique.
  • Description : Paramètres que Security Advisor recherche pour une règle particulière et qui peuvent mener à des vulnérabilités pour votre DiskStation si aucune action n'est prise.
  • Dernière mise à jour : La dernière fois que Security Advisor a vérifié cette règle de sécurité.
  • État : L'état actuel de chaque règle de sécurité. Security Advisor établit un rapport du résultat pour chaque règle de sécurité, Succès ou Échec. À certains moments, l'état Erreur, Non vérifié, En cours d'exécution et Ignoré peut également s'afficher. Vous pouvez filtrer les résultats en haut de cette page.

Pour vérifier les détails du rapport pour n'importe quelle règle de sécurité :

Sélectionnez une règle et cliquez sur Affichage. En plus de Gravité et Description, chaque rapport contient les détails suivants :

  • Détails : Une brève explication de toute faiblesse, que Security Advisor recommande pour la traiter, et les bénéfices que vous pouvez gagner de l'identification de la faiblesse.
  • Action recommandée : Les instructions détaillées sur la façon de signaler la faiblesse de sécurité rapportée.
  • Lien : Pour toutes les règles de sécurité ayant échoué, cliquer sur Lien vous permet d'accéder à l'emplacement de DSM pour procéder aux actions recommandées.
  • Corriger : Lorsqu'un maliciel est détecté, cliquer sur Corriger dira à Security Advisor de supprimer le maliciel.

Pour analyser une nouvelle fois une règle spécifique :

Sélectionnez les articles que vous souhaitez analyser encore une fois et cliquez sur Analyser.
