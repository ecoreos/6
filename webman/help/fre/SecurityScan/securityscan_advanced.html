Avancé

Vous pouvez afficher et changer les paramètres avancés pour Security Advisor.

Ligne de base de sécurité

Vous pouvez choisir les règles de sécurité que Security Advisor vérifie en sélectionnant une Ligne de base de sécurité. Le nombre et le type des règles varient pour chaque Ligne de base de sécurité. Effectuez votre choix en fonction de vos besoins. Utilisez-vous votre DiskStation pour le travail ou dans un cadre domestique et personnel ? Vous pouvez choisir à partir des options suivantes :

  • Pour l'utilisation privée et personnelle : Security Advisor effectue des contrôles essentiels pour protéger votre DiskStation contre les cyber-menaces communes.
  • Pour le travail et l'entreprise : Security Advisor effectue des contrôles complets pour répondre aux exigences de conformité de la sécurité pour les organisations.
  • Personnalisé : Sélectionnez quelles règles de sécurité vous voulez que Security Advisor vérifie.

Programme d'analyse

Vous pouvez définir un programme d'analyse régulière en cliquant sur Activer la planification d'analyse régulière et choisissez l'intervalle de temps auquel vous voulez que Security Advisor effectue une analyse. Security Advisor vérifiera les règles selon la Ligne de base de sécurité que vous avez choisie.
