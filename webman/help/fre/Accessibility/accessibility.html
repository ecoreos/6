Accessibilité

Grâce à la fonctionnalité d'accessibilité intégrée, DiskStation Manager fournit des conseils audio pour aider les utilisateurs déficients visuels à naviguer et à gérer les périphériques/paramètres de DSM.

Configuration de l'accessibilité de DSM

Afin de maximiser l'accessibilité, nous vous conseillons de configurer l'environnement de travail indiqué ci-dessous :

  • Installez le lecteur de l'écran NVDA (disponible pour l'environnement Windows) sur votre ordinateur local.
  • Installez la dernière version du navigateur Mozilla Firefox.
  • Utilisez un casque ou des enceintes et réglez le volume qui vous convient.
  • Utilisez NVDA en mode de mise au point (Appuyez sur la touche NVDA + Espace pour changer de mode). La touche NVDA par défaut est Insérer.
  • Vous pouvez naviguer dans DSM à l'aide des touches de raccourci suivantes :
    Touche raccourci Description
    Onglet Permet de basculer sur d'autres sections de l'interface utilisateur
    Touches fléchées Permet de basculer au sein de la même section de l'interface utilisateur
    Entrée Permet d'ouvrir des applications/fenêtres
    Échap Permet de fermer des applications/fenêtres

La fonctionnalité d'accessibilité est désormais opérationnelle dans les zones suivantes de DSM :

  • Page de connexion, bureau et barre de tâches de DSM
  • Menu principal
  • Panneau de configuration
  • File Station

Remarque :

Pour plus de détails à propos du fonctionnement de NVDA, reportez-vous à son guide de l'utilisateur.

Guide des raccourcis clavier de DSM

Avec la fonctionnalité d'accessibilité, DSM propose une riche palette de raccourcis clavier pour mettre en œuvre une gestion fluide et rationalisée. Appuyez sur Maj. + ? pour rechercher les raccourcis :

  • Raccourcis clavier du bureau : Touche de raccourci pour l'utilisation des fenêtres et plus encore
  • Raccourcis clavier du composant : Touches de raccourci pour contrôler les composants de l'interface utilisateur (par ex., les listes, les boutons et les champs de recherche)

Pour suspendre les raccourcis du bureau, annulez l'option Activer les raccourcis clavier du bureau.
