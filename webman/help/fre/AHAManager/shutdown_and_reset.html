Arrêter et remplacer un serveur

Les instructions ci-dessous vous guideront dans la procédure d'arrêt ou de changement d'un serveur dans un cluster high-availability.

Pour arrêter un cluster high-availability en utilisant le High Availability Manager :

Veuillez suivre ces instructions si vous souhaitez arrêter un cluster high-availability pour transporter les serveurs à un nouvel emplacement physique ou effectuer un entretien.

  1. Connectez-vous à DSM.
  2. Accédez au High Availability Manager.
  3. Cliquez sur le bouton marqué Gérer.
  4. Sélectionnez Arrêter le cluster high-availability.
  5. Un message de confirmation s'affiche. Cliquez sur OK pour confirmer.

Pour arrêter manuellement un cluster high-availability :

Le cluster high-availability peut également être arrêté manuellement en utilisant les boutons d'alimentation situés sur chaque serveur.

  1. Appuyez de manière prolongée sur le bouton d'alimentation du serveur passif jusqu'à ce qu'il émette un bip, et attendez que le serveur soit complètement arrêté.
  2. Appuyez de manière prolongée sur le bouton d'alimentation du serveur actif jusqu'à ce qu'il émette un bip, et attendez que le serveur soit complètement arrêté.

Remarque :

  • Mise sous tension des serveurs :
    Quand vous voulez allumer les serveurs à nouveau, vous devez allumer simultanément le serveur actif et le serveur passif (ou à cinq minutes d'intervalle maximum). Si vous n'allumez que l'un des serveurs, il essaie d'allumer l'autre serveur en diffusant un message Wake-on-LAN.
  • Arrêt anormal :
    Dans un cluster high-availability, l'option de redémarrage après panne est activée. Si les serveurs s'arrêtent de manière anormale suite à une panne, ils s'allument automatiquement une fois le courant rétabli.

Pour remplacer un serveur dans un cluster high-availability :

Si vous souhaitez supprimer un serveur et en configurer un autre, suivez ces instructions.

  1. Accédez à la page Aperçu.
  2. Cliquez sur le bouton Gérer.
  3. Sélectionnez Arrêter le serveur actif ou Arrêter le serveur passif pour arrêter le serveur que vous souhaitez remplacer.
  4. Vérifiez les connexions au nouveau serveur. Une fois le nouveau serveur sous tension, il est marqué comme « inconnu » dans High Availaibility Manager.
  5. Accédez à nouveau à la page Aperçu.
  6. Cliquez sur le bouton Gérer.
  7. Sélectionnez Récupérer le serveur inconnu afin d'ajouter « serveur inconnu » au cluster high-availability et de reconnaître le serveur passif.
