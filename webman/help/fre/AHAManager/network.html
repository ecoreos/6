Diagramme des interfaces

Affiche l'élément sélectionné dans le tableau et différentes couleurs sont utilisées pour présenter les interfaces réseau 1Gbps et 10Gbps.

Tableau de statut des interfaces

Affiche l'état des interfaces sur les de serveur, et indique si l'interface est surveillée. Ce tableau affiche les informations suivantes :

  • Interface réseau : Affiche les interfaces réseau physiques sur les serveurs.
  • Serveur actif : Affiche l'état de l'interface et la vitesse de connexion sur le serveur actif.
  • Serveur passif/serveur inconnu : Affiche l'état de l'interface et la vitesse de connexion sur le serveur passif. Cette colonne indique le nom correspondant à l'état du serveur.
  • Bond : affiche les paramètres link aggregation.
  • État : Affiche l'état du réseau si l'interface est surveillée. Les états suivants peuvent être affichés :
    • Avertissement : Échec de la connexion sur le serveur passif, ou l'interface possède des vitesses de connexion différentes sur les deux serveurs.
    • Erreur : Échec de la connexion sur le serveur actif ;
    • Normal : Aucune panne.
  • Surveiller : Indique si l'interface est surveillée. Accédez à Surveiller > Interface réseau pour cocher/décocher la case Surveiller.
