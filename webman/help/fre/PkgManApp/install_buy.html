Installer ou acheter des paquets

Centre de paquets offre une riche variété d'applications web qui peuvent étendre les fonctionnalités de votre DiskStation. Avec Centre de paquets, vous pouvez télécharger, installer et lancer des paquets en quelques clics seulement.

L'article ci-dessous explique comment installer les paquets sur votre DiskStation en utilisant Centre de paquets. Pour plus de renseignements concernant les paquets Synology, veuillez consulter ici.

Pour installer ou acheter des paquets :

  1. Ouvrez le Centre de paquets.
  2. Allez à la page Tous pour voir tous les paquets disponibles.

    3. Remarque :

    • Si vous ne pouvez pas trouver le paquet que vous recherchez, il peut exiger une version ultérieure de DSM ou n'est disponible que sur les modèles avec une configuration matérielle spécifique. Veuillez contacter le développeur de ce paquet pour plus d'informations.
  3. Trouvez le paquet que vous souhaitez installer et cliquez sur Installer. (Pour des paquets payants, cliquez sur Acheter ou Essayer pour vous connecter à un compte Synology et acheter avec Paypal ou utiliser la version d'essai pour évaluation.)
  4. Dès que le paquet est installé avec succès, il apparaitra dans le Menu principal.

Achat de paquets :

Avant d'acheter tous paquets, veuillez consulter les remarques suivantes.

  • Enregistrez un Compte Synology pour accéder à plus de paramètres, tels que les informations de facturation et l'historique d'achat. L'information que vous partagez est uniquement utilisée pour fournir de meilleurs services dans le futur.
  • Le Centre de paquets accepte uniquement les paiements en ligne offerts par Paypal. Si vous souhaitez demander un remboursement, veuillez visiter le site officiel de PayPal.
  • Les prix des paquets sont actuellement en dollar U.S.

Mentions légales sur la conversion des devises :

Pour les opérations qui incluent la conversion de devises, l'émetteur de votre carte de crédit/débit peut ajouter des frais supplémentaires qui ne sont pas reflétés dans le prix affiché. Tous les achats peuvent être sujets à des frais de change ou commissions transfrontalières par l'émetteur de la carte. L'émetteur de votre carte évaluera tous les frais appliqués, pour lesquels Synology ne peut pas assurer le remboursement. Veuillez contacter l'émetteur de votre carte de crédit/débit pour plus d'informations.

Pour installer les paquets manuellement :

  1. Si vous souhaitez installer un paquet en chargeant le fichier .spk (disponible sur le Centre de téléchargement de Synology ou sur des sites Web tiers), cliquez sur Installation manuelle et suivez les étapes de l'assistant pour installer le nouveau paquet.

Pour rechercher un paquet :

  1. Entrez un mot-clé dans la barre de recherche dans le coin supérieur gauche, et appuyez sur Entrée pour commencer la recherche.

Pour choisir comment les paquets sont triés :

Vous pouvez modifier l'ordre des paquets à afficher dans Centre de paquets.  Sélectionnez une catégorie, cliquez sur la liste des icônes dans le coin supérieur droit et choisissez de trier les paquets par l'une ou l'autre des options suivantes :

  • Trier par nom : Cliquez pour trier les paquets en ordre alphabétique.
  • Trier par popularité récente : Cliquez pour trier les paquets par le nombre d'installations récentes avec les plus populaires devant.

Pour masquer les paquets installés :

Sélectionnez une catégorie, cliquez sur l'icône de liste dans le coin supérieur droit, puis choisissez Ne pas présenter les paquets installés. Ceci masquera les paquets qui ont déjà été installés, et présentera ceux qui ne sont pas encore installés, mis à niveau ou réparés.
