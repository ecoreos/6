Configurer le centre de paquets

Pour gérer quels paquets sont autorisés a être installé :

  1. Sous l'onglet Général, choisissez les options suivantes pour configurer votre Niveau de confiance désiré :
    • Synology Inc.: Autoriser uniquement les paquets fournis par Synology Inc.
    • Synology Inc. et les éditeurs de confiance: Autoriser uniquement les paquets fournis par Synology Inc.et les éditeurs de confiance. Les éditeurs de confiance se compose d'éditeurs qui ont soit obtenu la confiance d'une communauté tierce via l'ajout d'une source de paquet ou sont de confiance par l'utilisateur via l'importation du certificat de l'éditeur.
    • N'importe quel éditeur: Autorise l'installation de n'importe quel paquet en ce qui concerne l'éditeur et l'intégrité.
  2. Cliquez sur OK.

Pour importer un certificat d'un éditeur de confiance :

  1. Sous l'onglet Certificat, cliquez sur Importer.
  2. Cliquez sur Parcourir et sélectionnez le certificat de l'éditeur de confiance.
  3. Cliquez sur OK.

Pour mettre à jour le certificat Synology Inc.'s :

Lorsque vous vous connectez à Internet, le certificat Synology Inc.'s sera mis à jour automatiquement pendant le processus d'installation du paquet. Pour mettre à jour manuellement :

  1. Sous l'onglet Certificat, cliquez sur Importer.
  2. Cliquez sur Parcourir et sélectionnez le certificat Synology Inc.'s qui peut être téléchargé via le site web de Synology.
  3. Cliquez sur OK.

Pour activer les notifications sur les mises à jour des paquets :

  1. Cliquez sur l’onglet Notification, puis cochez Activer la notification par courriel et/ou Activer les notifications sur le bureau. La première option permet à DSM de vous envoyer des e-mails lorsque de nouvelles mises à jour sont disponibles, tandis que la seconde permet à DSM d'afficher un message de notification dans le coin supérieur droit de votre bureau de DSM.
  2. Cliquez sur OK.

Remarque :

Pour que les notifications par courriel prennent effet, configurez d'abord vos paramètres de notification par courriel de DSM (à Menu principal > Panneau de configuration > Notification).

Pour ajouter une ressource de paquet fournie par une communauté tierce :

  1. Cliquez sur l'onglet Sources de paquets, cliquez sur Ajouter, puis entrez le nom et l'emplacement de vos sources de paquet.
    L'emplacement peut être une URL que vous avez trouvée sur Internet fournie par un développeur/éditeur tiers ou une communauté.
  2. Cliquez sur OK. Quand la source du paquet est ajoutée avec succès, vous verrez une liste de paquets dans l'onglet Communauté.
