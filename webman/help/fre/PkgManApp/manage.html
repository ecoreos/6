Gérer les paquets

Après l'installation de paquets, soit en ligne, soit manuellement, vous pouvez les gérer dans le Centre de paquets pour qu'ils restent à jour et fonctionnent correctement. Les instructions ci-dessous vous expliquent comment lancer, mettre à jour, désinstaller, exécuter ou arrêter les paquets, et résoudre les problèmes des paquets.

Pour mettre à jour les paquets, procédez d'une des manières suivantes :

Toujours garder vos paquets à jour est la meilleure pratique pour maintenir votre DiskStation en bon état.

  • Cliquez sur Mettre à jour sur le panneau gauche pour voir les mises à jour disponibles. Cliquez sur les boutons Mettre à jour ou Tout mettre à jour pour mettre à jour les paquets.
  • Cliquez sur Installation manuelle pour installer un paquet en chargeant le fichier .spk (disponible sur le Centre de téléchargement de Synology ou sur des sites Web tiers).
  • Les mises à jour de paquets apparaîtront selon les options dans Paramètres > Mise à jour auto Voir ici pour plus d’informations.

Remarque : Si vous souhaitez recevoir une notification par e-mail ou sur le bureau lorsque les mises à jour sont disponibles, cliquez sur Paramètres > Général > Notification, cochez les case puis cliquez sur OK.

Pour réparer les paquets, procédez d'une des manières suivantes :

En cas d'erreur sur un paquet de votre DiskStation, vous devez le réparer pour le faire fonctionner à nouveau. Cela se produit parfois parce que le paquet est trop obsolète pour être compatible avec la dernière version du DSM, et d'autres fois les fichiers de programme du paquet sont manquants ou endommagés. Les paquets qui doivent être réparés seront inscrits sous la catégorie Installé.  Les paquets qui doivent être réparés seront listés dans la catégorie Installée.

  • Cliquez sur les boutons Réparer pour réparer les paquets.
  • Cliquez sur Installation manuelle pour réinstaller et réparer un paquet en chargeant le fichier .spk (disponible sur le Centre de téléchargement de Synology ou sur des sites Web tiers).

Remarque : En cliquant sur Réparer vous allez installer en fait la dernière version de ce paquet, plutôt que de réparer l'ancienne. Veuillez contacter l'éditeur ou le développeur de ce paquet si vous avez encore des difficultés à installer le paquet.

Pour désinstaller des paquets existants :

Cliquez sur un paquet installé, puis choisissez Désinstallerdans le menu déroulant Action pour désinstaller le paquet.

Pour lancer ou arrêter les services du paquet :

Cliquez sur un paquet installé, puis choisissez Exécuter ou Arrêter dans le menu déroulant Action pour exécuter ou arrêter le paquet.

Pour résoudre les problèmes de paquets :

Si vous trouvez un problème sur un paquet, veuillez essayez ce qui suit :

  • Mettez à jour le paquet à la dernière version.
  • Supprimez le paquet ainsi que sa base de données (le cas échéant), et réinstallez-le.
  • Contactez l'éditeur ou le développeur du paquet de l'aide et du support.

Pour trouver les coordonnées d'un éditeur de paquet :

Certains paquets sur le Centre de paquets de Synology sont conçus et publiés par des développeurs tiers, et par conséquent Synology ne fournit pas de support technique pour les applications individuelles. Veuillez contacter les éditeurs ou développeurs de chaque application pour les demandes de fonctionnalités, questions et autres commentaires. Certains paquets ont un éditeur en plus d'un développeur. L'éditeur intègre et rend le paquet disponible pour les produits Synology, tandis que les fonctionnalités essentielles du paquet sont fournies par le développeur. Si un éditeur est fourni, veuillez contacter l'éditeur pour de l'aide.  Si un éditeur est fourni, veuillez contacter l'éditeur pour de l'assistance.

  1. Aller à la page détaillée d'un paquet installé.
  2. Sur la gauche de la page détaillée, vous trouverez les coordonnées de son éditeur ou développeur.
  3.  Si vous avez des demandes de fonctionnalités, des questions ou des commentaires sur un paquet développé par Synology, vous êtes invité à visiter le Synology Website et à nous contacter directement.

Lancer les applications

Après l'installation et l'exécution des paquets, vous pouvez lancer les applications activées de plusieurs manières.

Pour lancer une application installée et activée, procédez d'une des manières suivantes :

  • Allez dans Menu principal > [Nom de l'application].
  • Ouvrez le Menu principal et faites glisser la miniature de l'application sur le bureau de DSM, puis cliquez sur [Nom de l'application] sur le bureau.
  • Allez dans Menu principal > [Nom de l'application]. Cliquez-droit sur l’icône de l’application dans la barre des tâches, puis choisissez Épingler à la barre des tâches. Lorsque l’application est fermée, son icône restera épinglée dans la barre des tâches, et vous pouvez cliquer dessus pour lancer l’application à nouveau.
