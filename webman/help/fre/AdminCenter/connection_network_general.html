Onglet Général

Les options génériques de réseau telles que la passerelle par défaut, le DNS server, le proxy et le tunneling IPv6 sont disponibles dans Panneau de configuration > Réseau > Général.

Renommer l'appareil DiskStation

Pour renommer votre machine DiskStation, saisissez le nouveau nom du serveur et cliquez sur Appliquer.

Le nom du serveur n'est pas sensible à la casse et peut inclure de 1 à 15 caractères contenant des lettres, des nombres, des traits de soulignement et des signes moins (-). Le premier caractère du nom doit être une lettre.

Spécification de la passerelle par défaut

Saisissez simplement l'adresse IP pour la passerelle par défaut et cliquez sur Appliquer pour sauvegarder les paramètres.

Spécification de l'adresse IP du DNS server

Vous pouvez choisir de configurer manuellement l'adresse IP de votre DNS server. Dans ce cas, l'adresse IP de votre DNS server préféré doit être saisie. Une adresse IP de DNS server auxiliaire peut également être entrée, fournissant une sécurité si le DNS server préféré rencontre des problèmes. Une fois les adresses IP des DNS server configurées comme vous le souhaitez, veuillez cliquer sur Appliquer pour enregistrer vos paramètres.

Remarque :

Les champs DNS server et Passerelle par défaut seront définis automatiquement et ne peuvent pas être changés si toutes vos interfaces réseau sont configurées automatiquement ou quand vous utilisez une connexion PPPoE.

Connexion via un serveur proxy

Pour permettre à votre DiskStation de se connecter via un serveur proxy, cochez Se connecter via un serveur proxy, entrez l'Adresse et le Port, puis cliquez sur Appliquer.

Remarque :

  • Cliquez sur Paramètres avancés pour configurer le proxy HTTP, le proxy HTTPS et les informations d'authentification.
  • Par défaut, les paramétrages du proxy HTTPS sont les mêmes que ceux du proxy HTTP.
  • Si vous cochez Se connecter via un serveur proxy, Mise à jour de DSM et Centre de paquets, avec d'autres paquets, se connecteront toujours à Internet via un proxy.
  • Si Ignorer le serveur proxy pour les adresses locales est coché, assurez-vous que le DNS server utilisé par le DiskStation peut résoudre les demandes d'appareils dans le réseau local.

Activer le tunneling IPv6

Jusqu'à ce que IPv6 supplante complètement IPv4, on utilise le tunneling IPv6 pour permettre aux hôtes fonctionnant en IPv6 d'accéder aux services IPv4. Cette technique fournit une manière de transporter un trafic IPv6 dans des réseaux IPv4 existants.
Si vous n'êtes pas sûr de vous quant à la manière de configurer l'IPv6 ou que vous rencontrez des problèmes de connexion, contactez votre fournisseur d'accès Internet.

Pour activer le Tunnel IPv6 :

  1. Cochez la case Activer le tunnel pour activer la prise en charge du tunnel.
  2. Dans Adresse du serveur, saisissez le nom d'hôte ou l'adresse IP du fournisseur de service tunnel, par exemple : anonymous.freenet6.net.
  3. Choisissez entre Se connecter en anonyme ou Se connecter avec un compte existant selon vos préférences.

Remarque :

Votre adresse IPv6 externe pourra être mise à jour automatiquement par le prestataire de service. Toutefois, l'appareil DiskStation ne supprimera pas l’ancienne adresse IPv6 immédiatement.
