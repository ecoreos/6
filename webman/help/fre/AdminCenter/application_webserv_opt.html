Service HTTP

Vous pouvez ajouter un port supplémentaire en plus du port 80 pour l'utilisation de Photo Station et de Web Station.

Pour ajouter un numéro de port supplémentaire pour l'accès HTTP :

  1. Cochez Ajouter un port HTTP supplémentaire.
  2. Entrez le numéro de port dans le champ. Vous pouvez ajouter un numéro de port supplémentaire compris entre 1 et 65 535 pour l'accès HTTP.

Les ports suivants doivent être exclus :

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, les ports par défaut eMule : 4662 (TCP), 4672(UDP) et la plage par défaut des ports FTP : la plage actuelle peut varier selon les modèles.

    Remarque :
  1. les numéros de ports pour l'accès HTTP ne peuvent pas être les mêmes que les numéros de ports des services FTP/BT.
  2. Les ports HTTP et HTTPS ne peuvent pas être en conflit les uns avec les autres.
