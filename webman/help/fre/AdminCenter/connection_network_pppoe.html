PPPoE

Si vous utilisez un modem câble ou DSL pour vous connecter à Internet et que votre fournisseur de services Internet (ISP) utilise le PPPoE (Point to Point Protocol over Ethernet), vous pouvez activer le PPPoE dans Panneau de configuration > Réseau > Interface réseau et entrez les informations du compte afin de vous connecter directement à Internet sans passer par le routeur.

Pour plus d'informations concernant le PPPoE, contactez votre ISP ou votre administrateur réseau.

Configuration d’IPv6

DiskStation prend en charge le schéma de fonctionnement IPv4/IPv6 dual stack. Cela signifie que l’IPv4 et l’IPv6 peuvent fonctionner simultanément sur votre DiskStation. Le paramètre par défaut de l'IPv6 est Auto. Sélectionnez une interface réseau existante, cliquez sur le bouton Modifier et aller dans l'onglet IPv6 pour modifier les paramètres.

Jusqu'à ce que IPv6 supplante complètement IPv4, les mécanismes de transition IPv6 sont nécessaires pour permettre aux hôtes avec des adresse IPv6 d'atteindre des services IPv6. Choisissez entre les différents types de mécanismes de transition IPv6 :

  • 6in4 : Connectez vous au réseau IPv6 à travers un agent de tunnel. Vous devrez remplir l'Adresse IPv6, la Longueur de préfixe, le Préfixe et l'Adresse IPv4 de l'agent de tunnel. Ceci nécessite une adresse IPv4 publique.
  • 6to4 : Connectez vous au réseau IPv6 à travers un agent de tunnel utilisant anycast. Ceci nécessite une adresse IPv4 publique.
  • DHCP-PD : Obtenez un préfixe IPv6 depuis le serveur DHCP par délégation de préfixe dans le LAN. Ceci requiert une prise en charge ISP et fonctionne uniquement en mode routeur.
