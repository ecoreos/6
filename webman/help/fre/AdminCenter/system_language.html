Langue

Vous pouvez sélectionner votre langue préférée pour l'interface utilisateur de DSM, les notifications du bureau de Google Chrome, les notifications par messagerie instantanée, e-mail, SMS, et la page de code de l'application.

Langue d’affichage

Vous pouvez choisir votre langue d'affichage préférée pour DSM et les notifications du bureau de Google Chrome, ou conserver la même que celle de votre navigateur par défaut.

Langue de notification

Vous pouvez définir votre langue préférée pour les notifications par e-mail, SMS, Windows Live Messenger et Skype. Elle sera aussi appliquée aux notifications reçues sur Windows Phone.

Cependant, la langue de notification sur un appareil iOS ou Android dépend du paramètre de la langue de votre appareil mobile.

Page de code

Pour les applications qui convertissent les chaînes non Unicode en Unicode, telles que le service FTP, la prise en charge UPnP, l'indexation des métadonnées musicales, vous devrez choisir la page de code appropriée depuis le menu déroulant Page de code.
