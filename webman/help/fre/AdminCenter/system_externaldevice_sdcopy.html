SDCopy*

Vous pouvez sélectionner et définir un emplacement cible pour la fonction SDCopy.

Pour activer SDCopy :

  1. Cochez Activer SDCopy.
  2. Sélectionnez un dossier partagé de destination à partir du menu déroulant.
  3. Cliquez sur OK.

Lorsque vous raccordez un périphérique SD dans la fente SD du panneau avant de Synology DiskStation, vous pouvez appuyer sur le bouton Copie de celui-ci pour copier les données du périphérique SD vers le dossier de destination partagé, défini sur celui-ci. Si le dossier de destination SDCopy a été supprimé, le système le remplacera automatiquement par le dossier partagé public.

    Limites :
  1. Seuls les périphériques SD au format FAT ou NTFS sont pris en charge.
  2. Les périphériques SD doivent être branchés dans la fente SD de la façade du Synology DiskStation.
  3. L'espace disponible du dossier partagé de destination doit être supérieur à l'espace utilisé sur le périphérique SD.

____
* Uniquement disponible sur certains modèles.
