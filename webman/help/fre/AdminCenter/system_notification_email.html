Courrier électronique

L'activation des notifications par email permet au DiskStation d'envoyer des messages à votre email lorsque des événements spécifiques se produisent, tels que des erreurs système, une perte de connexion réseau, le comportement du ventilateur, une panne d'alimentation ou un échec d'une tâche de fond.

L'activation des notifications par email permet au DiskStation d'envoyer des messages à votre adresse électronique lorsque des événements spécifiques se produisent, tels que des erreurs système, une perte de connexion réseau, une coupure de courant, ou un échec d'une tâche de fond, etc.

Pour activer la notification par e-mail :

  1. Accédez à Panneau de configuration > Notification > E-mail.
  2. Cochez Activer la notification par e-mail.
  3. Saisissez une ou plusieurs adresses e-mail dans le champ Destinataires. Les notifications système sont envoyées aux adresses saisies ici. Vous pouvez saisir deux adresses. Les comptes multiples peuvent être séparés avec un point-virgule (;).
  4. Si nécessaire, ajoutez un préfixe d'objet dans le champ Préfixe de l'objet. Ce préfixe sera ajouté à l'objet de chaque message envoyé par le DiskStation, vous permettant ainsi d'identifier et de filtrer.
  5. Si vous voulez utiliser un serveur de courriel tiers (p. ex. Gmail, Outlook, Yahoo!) pour relayer les messages, consultez la section "service de courriel tierce" ci-dessous. Si vous voulez utiliser un serveur client, consultez la section "serveur SMTP client".

Pour utiliser un service de messagerie tiers :

  1. dans le menu Fournisseur de service, choisissez un service de messagerie tiers, tel que Gmail, Yahoo ou Outlook.
  2. Saisissez votre nom d'utilisateur et mot de passe pour le service de messagerie sélectionné.
  3. Cliquez sur Appliquer pour enregistrer les paramètres.
  4. Vous pouvez cliquer sur Envoyer un courriel de test pour voir si vos paramètres sont corrects.

Pour utiliser un serveur SMTP personnalisé :

  1. Dans le menu Fournisseur de service, sélectionnez Serveur SMTP personnalisé.
  2. Saisissez le nom du serveur SMTP ou l'adresse IP dans le champ Serveur SMTP
  3. Entrez le numéro de port SMTP dans le champ Port SMTP. (Si vous avez spécifié un port SMTP différent pendant la configuration des règles de transmission de port, veuillez saisir ce numéro.)
  4. Si votre serveur SMTP requiert une authentification SMTP, cochez Authentification requise, puis saisissez votre Nom d'utilisateur et votre Mot de passe associés au serveur de messagerie.
  5. Si votre serveur SMTP requiert une connexion SSL/TLS, cochez Une connexion sécurisée (SSL/TLS) est nécessaire.
  6. Si vous désirez personnaliser le nom de l'expéditeur qui apparaît sur les messages e-mail envoyés par le DiskStation, saisissez ce nom dans le champ Nom de l'expéditeur. Si vous désirez personnaliser l'adresse e-mail de l'expéditeur, saisissez l'adresse e-mail dans le champ Expéditeur de l'email. Si le champ est vide, la première adresse email dans la champ Destinataires sera utilisée.
  7. Cliquez sur Appliquer pour enregistrer les paramètres.
  8. Vous pouvez cliquer sur Envoyer un courriel de test pour voir si vos paramètres sont corrects.

Remarque :

  • Si vous désirez envoyer un message de bienvenue à chaque compte nouvellement créé, cochez sur Envoyer un message de bienvenue aux nouveaux utilisateurs, et chaque utilisateur recevra un message de bienvenue lorsque son compte est créé.
  • La langue de notification par e-mail est définie dans Langue de notification. Pour changer votre langue de notification par e-mail, allez dans Menu principal > Panneau de configuration > Options régionales, cliquez sur l'onglet Langue, et choisissez votre langue préférée dans le menu déroulant de la section Langue de notification.
  • Si vous utilisez le serveur SMTP de GMail et que vous avez activé la vérification en deux étapes, n'oubliez pas d'utiliser votre "mot de passe spécifique à l'application" fourni par Google pour l'authentification SMTP.

Nom de domaine du serveur SMTP :

Le serveur SMTP peut être une adresse IP ou un nom de domaine. S'il s'agit d'un nom de domaine, assurez-vous qu'une IP valide du serveur DNS a bien été entrée dans Menu principal > Panneau de configuration > Réseau et que le serveur se trouve sur Internet.
