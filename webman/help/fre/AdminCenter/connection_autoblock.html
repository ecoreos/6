Blocage auto

Le blocage auto vous permet d'empêcher une connexion non autorisée. Après avoir autorisé le service, une adresse IP sera bloquée si elle a trop d'échecs de tentatives de connexion. Les échecs de tentatives de connexion via SSH, Telnet, rsync, Sauvegarde réseau, Synchro du dossier partagé, FTP, WebDAV, applis mobile de Synology, File Station ou DSM s’ajouteront. Vous pouvez également avoir une Liste des permissions qui inclut les adresses IP qui ne seront jamais bloquées.

Pour configurer le blocage auto :

  1. Allez à l’onglet Général et cochez Activer le blocage auto.
  2. Entrez un nombre pour les Tentatives de connexion et Sous (minutes) pour bloquer une adresse IP après un nombre défini d'échecs de tentatives de connexion sous le nombre de minutes spécifié.
  3. Cochez Activer l’expiration des blocages et entrez un numéro pour supprimer une adresse IP bloquée après le nombre de jours spécifié. Les adresses IP bloquées resteront indéfiniment sur la liste des blocages à moins qu'elles ne soient supprimées manuellement de la liste si cette option n’est pas sélectionnée ou si Débloquer après (jours) est défini sur 0.
  4. Cliquez sur Appliquer.

Pour ajouter des adresses IP bloquées :

Cliquez sur l'onglet Liste blocages et choisissez une des options suivantes dans le menu déroulant Créer :

  • Ajouter une adresse IP

    • Entrez une adresse IP et définissez le délai d'expiration.

  • Importer une liste d'adresses IP

    • Importez un fichier texte listant les adresses IP pour ajouter un certain nombre d'adresses IP et définir le délai d'expiration en même. Cochez la case appropriée si vous souhaitez écraser les adresses IP existantes. Si cette option n'est pas sélectionnée, les adresses IP en double seront ignorées.

    Pour importer un fichier, le fichier doit répondre aux critères suivants :

    • Le fichier doit être un fichier texte.
    • Chaque ligne du fichier ne peut contenir qu’une seule adresse IP.
    • Les lignes de commentaires commencent par un # (dièse).

      • Si le fichier que vous avez sélectionné a un format correct, vous verrez toutes les adresses IP dans la zone d’aperçu. Cliquez sur OK pour importer.

Pour supprimer des adresses IP bloquées :

  1. Cliquez sur l'onglet Liste blocages pour voir la liste des adresses IP bloquées.
  2. Sélectionnez les adresses IP que vous souhaitez supprimer de la liste et cliquez sur Supprimer.

Pour ajouter des adresses IP permises :

Cliquez sur l'onglet Liste des permissions et choisissez une des options suivantes dans le menu déroulant Créer :

  • Ajouter une adresse IP

    • Entrez une adresse IP.

  • Importer une liste d'adresses IP

    • Importez un fichier texte listant les adresses IP pour ajouter un certain nombre d'adresses IP en même. Cochez la case appropriée si vous souhaitez écraser les adresses IP existantes. Si cette option n'est pas sélectionnée, les adresses IP en double seront ignorées.

Pour supprimer des adresses IP permises :

  1. Cliquez sur l'onglet Liste des permissions pour voir la liste des adresses IP permises.
  2. Sélectionnez les adresses IP que vous souhaitez supprimer de la liste et cliquez sur Supprimer.
