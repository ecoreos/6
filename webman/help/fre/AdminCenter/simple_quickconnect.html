QuickConnect

QuickConnect permet aux applications clientes de se connecter à votre DiskStation via Internet sans l'inconvénient d'avoir à configurer les règles de transmission de ports.

Remarque :

QuickConnect peut fonctionner avec DSM et quelques paquets développés par Synology, tels que Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Cloud Station et des applications mobiles.

Pour activer QuickConnect :

  1. Accédez à Panneau de configuration (Mode simple) > QuickConnect.
  2. Cochez la case Activer QuickConnect.
  3. Saisissez les informations de votre compte Synology ou enregistrez un nouveau compte.
  4. Spécifiez un nouveau QuickConnect ID. Assurez-vous qu'il est facile à se rappeler, de sorte que vous pourrez accéder à votre DiskStation à tout moment.
  5. Cliquez sur Appliquer.

Remarque :

  • Avant d’activer QuickConnect pour les applications clientes, assurez-vous que les services DSM en rapport sont déjà activés. Par exemple, WebDAV doit être activé sur DiskStation avant de pouvoir activer QuickConnect pour DS file.
  • Un QuickConnect ID personnalisé peut inclure des lettres, chiffres et tirets ("-"), et doit commencer avec une lettre.
  • Pour une meilleure performance de QuickConnect, il est recommandé d'aller dans Panneau de configuration (Mode avancé) > Accès externe > Configuration du routeur pour configurer la transmission de port pour chaque service :
    • DSM : 5000 (HTTP), 5001 (HTTPS)
    • Photo Station : 80 (HTTP), 443 (HTTPS)
    • Cloud Station：6690

Dépannage

Si vous voyez le message Erreur de réseau, cela signifie que le service QuickConnect a été interrompu en raison d’erreurs de réseau. Veuillez vérifier ce qui suit :

  • Votre DiskStation est connecté à un réseau actif (dans Panneau de configuration (Mode avancé) > Réseau > Interface réseau).
  • Les paramètres de serveur DNS et de passerelle par défaut sont correctement configurés (dans Panneau de configuration (Mode avancé) > Réseau > Général).
  • Assurez-vous que le DiskStation peut accéder à Internet en utilisant les ports de destination suivants : 22, 80, 443, 8888. 

Quand les erreurs de réseau sont corrigées, le service QuickConnect devrait reprendre automatiquement après quelques minutes.
