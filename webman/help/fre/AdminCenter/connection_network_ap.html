Point d'accès*

Votre appareil DiskStation prend en charge la création d'un point d'accès pour partager sa connexion Internet avec les autres appareils sans fil si vous avez installé un adaptateur USB Wi-Fi ou si votre DiskStation a un Wi-Fi intégré. Vous pouvez configurer votre DiskStation pour être un point d'accès sans fil (AP) ou un routeur sans fil, selon la façon dont vous souhaitez utiliser Internet.

Mode de point d'accès Description
AP sans fil
  • Faites un pont avec votre LAN et votre réseau Wi-Fi.
  • Votre LAN et les clients sans fil seront sur le même réseau local.
  • Applicable lorsque vous avez déjà un routeur et que vous voulez utiliser votre DiskStation en tant que point d'accès supplémentaire vers un réseau existant.
  • Permet le partage des ressources réseau telles que les appareils DLNA, les imprimantes, Synology Web Assistant ou les autres services reposant sur les diffusions réseau.
  • DHCP Server sera désactivé sur les appareils DiskStation qui agissent comme des points d'accès sans fil. Pour réactiver DHCP Server pour votre DiskStation, retirez les dongles Wi-Fi et redémarrez DiskStation.
  • Pour les modèles avec plusieurs ports LAN, seul le LAN 1 sera utilisé pour servir de pont au AP sans fil.
Routeur sans fil
  • Configurez un réseau sans fil où les appareils sont sur un sous-réseau privé.

Remarque :

  1. Certains modèles requièrent la connexion de dongles sans fil pour créer des points d'accès. Pour une liste complète des dongles sans fil pris en charge, visitez www.synology.com.
  2. Pour les modèles qui ne sont pas DS213air, l'option Point d'Accès sans fil dans DSM 4.3 est renommé en "Routeur sans fil" dans DSM 5.0 et ultérieur.

Gérer les paramètres du point d'accès

Cette section explique comment activer et gérer les paramètres du point d'accès de votre DiskStation.

Pour transformer le $-DSPRODUCTNAME_$ en point d'accès :

  1. Accédez à Panneau de configuration > Sans fil > Configuration de la carte du réseau. Choisissez Wireless AP ou Wireless Router.
  2. Allez dans l'onglet Wi-Fi et cliquez sur Modifier pour gérer les paramètres suivants :
    • Nom (SSID) : Entrez un nom pour le réseau sans fil.
    • Afficher/Masquer : Choisissez si le réseau sans fil est visible ou non par les autres appareils.
    • Mode sans fil : Sélectionnez un standard sans fil. Chaque standard prend en charge différentes bandes passantes et fréquences.
    • Largeur de canal : Sélectionnez une largeur de canal (disponible uniquement lorsque vous utilisez le mode sans fil 802.11n).
    • Canal : Entrez une fréquence pour le réseau sans fil. Pour de meilleures performances, choisissez une fréquence différente de celles utilisées par les réseaux à proximité.
    • Alimentation de transmission : Entrez une longueur de signal pour le réseau sans fil.
    • Connexions maximum : Choisissez le nombre d'appareils clients qui peuvent se connecter au réseau. Le paramètre de connexion le plus élevé est 256.
    • Activer WMM QoS : Wi-Fi Multimedia (WMM) QoS (qualité de service) via Wi-Fi. Cette option permet au réseau sans fil de hiérarchiser le trafic et d'optimiser la manière dont les ressources sont partagées entre les applications.

Pour configurer un calendrier sans fil :*

Activer un calendrier de réseau sans fil vous permet de contrôler quand le réseau sans fil de votre DiskStation est actif et accessible.

  1. Allez dans l'onglet Wi-Fi et cliquez sur Modifier.
  2. Cochez la case Activer le calendrier.
  3. Cliquez sur Définir le calendrier.
  4. Sur l'écran du calendrier, sélectionnez quand le réseau sans fil est actif.

Paramètres de sécurité

La fonction Panneau de configuration > Sans fil > Wi-Fi fournit des options pour modifier les paramètres de sécurité de votre réseau sans fil.

Pour configurer la sécurité sans fil :

  1. Cliquez sur Modifier.
  2. Choisissez l'un des éléments suivants dans le menu déroulant Paramètres de sécurité :
    • WEP : Un protocole de sécurité de base qui oblige les utilisateurs à entrer un mot de passe pour accéder au réseau sans fil, mais fournit une protection minimale.
    • WPA/WPA2 Personnel : Recommandé. Oblige les utilisateurs à entrer un mot de passe pour accéder au réseau sans fil, et est plus sécurisé que le WEP.
    • WPA/WPA2 Enterprise :* Un protocole de sécurité conçu pour les entreprises, ce qui nécessite l'authentification des utilisateurs avec un serveur d'authentification externe.

Pour configurer WPS (Wi-Fi Protected Setup) :*

WPS (Wi-Fi Protected Setup) est une norme promue par la Wi-Fi Alliance pour simplifier les paramètres de sécurité sans fil pour les utilisateurs. Vous pouvez utiliser un bouton-poussoir ou un code PIN pour sécuriser votre réseau WiFi. Cliquez sur WPS pour activer Wi-Fi Protected Setup.

Réseau invité*

En plus de créer un réseau sans fil hôte, votre DiskStation prend également en charge la création d'un réseau sans fil invité. Les paramètres du réseau invité peuvent être limités à restreindre l'accès des utilisateurs, ce qui est utile pour les entreprises ou d'autres institutions qui souhaitent fournir un réseau sans fil pour les visiteurs, mais en les empêchant d'accéder au réseau hôte.

Pour activer le réseau invité :

  1. Accédez à Panneau de configuration > Sans fil > WiFi.
  2. Cliquez sur Modifier pour gérer les paramètres suivants sur l'onglet Réseau invité :
    • Nom (SSID) : Entrez un nom pour le réseau sans fil.
    • Afficher/Masquer : Choisissez si le réseau sans fil est visible ou non par les autres appareils.
    • Connexions maximum : Choisissez le nombre d'appareils clients qui peuvent se connecter au réseau. Le paramètre de connexion maximum est 256.

Pour configurer la sécurité du réseau invité :

Choisissez l'un des éléments suivants dans le menu déroulant Paramètres de sécurité :

  • WEP : Un protocole de sécurité de base qui oblige les utilisateurs à entrer un mot de passe pour accéder au réseau sans fil, mais fournit une protection minimale.
  • WPA/WPA2 Personnel : Recommandé. Oblige les utilisateurs à entrer un mot de passe pour accéder au réseau sans fil, et est plus sécurisé que le WEP.

Pour configurer un calendrier de réseau invité :

Activer un calendrier de réseau invité vous permet de contrôler quand le réseau sans fil invité de votre DiskStation est actif et accessible.

  1. Cochez la case Activer le calendrier du réseau invité.
  2. Cliquez sur Définir le calendrier invité.
  3. Sur l'écran du calendrier, sélectionnez quand le réseau sans fil invité est actif.

Remarque :

Le réseau sans fil invité ne peut pas être actif pendant les périodes où le réseau sans fil hôte est désactivé.

Paramètre du filtre MAC*

Le filtre MAC (dans Panneau de configuration > Sans fil) spécifie quelles adresses MAC sont autorisées à accéder au réseau sans fil, vous permettant de bloquer ou autoriser l'accès à des appareils spécifiques. Cette fonction ne contrôle que l'accès au réseau, pas les autres paramètres de chiffrement.

Pour créer des règles de filtre MAC :

  1. Saisir la description et l'adresse MAC d'un appareil.
  2. Choisir une politique d'accès :
    • Refuser : Interdit aux adresses MAC spécifiées l'accès au réseau sans fil.
    • Autoriser : Autorise les adresses MAC spécifiées l'accès au réseau sans fil.

Contrôle parental*

La fonction de contrôle parental vous permet de contrôler le moment où chaque appareil client connecté au réseau local du DiskStation peut accéder à Internet. Par exemple, si les ordinateurs ou les appareils mobiles de vos enfants se connectent sur Internet via le DiskStation, vous devez configurer les paramètres de contrôle parental pour restreindre leur usage d'Internet en fonction des appareils et heures spécifiques.

Pour activer le Contrôle parental :

  1. Accédez à Panneau de configuration > Sans fil > Configuration du contrôle parental.
  2. Cliquez sur le bouton Ajouter pour ajouter un appareil client.
    • Nom du client : Choisissez un appareil qui est connecté au DiskStation ou saisissez un nom manuellement (par ex. le portable de Louisa).
    • Adresse MAC : Saisissez l'adresse MAC de l'appareil dont vous désirez réglementer l'usage d'Internet.
    • Définir le calendrier : Déterminez les moments où l'appareil en question sera autorisé à accéder à Internet via le DiskStation.

Liste de connexion*

La liste de connexion (dans Panneau de configuration > Sans fil) affiche des informations concernant les appareils connectés au DiskStation. Les appareils peuvent être directement ajoutés à la liste des filtres MAC à partir d'ici.

Pour ajouter des appareils à la liste de filtre MAC :

  1. Cochez la case à côté de l'appareil que vous souhaitez ajouter au filtre MAC.
  2. Cliquez sur Ajouter à la liste de filtre MAC.
