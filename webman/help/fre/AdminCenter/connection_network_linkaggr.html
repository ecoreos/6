Link Aggregation*

DiskStation pend en charge les réseaux locaux multiples, vous permettant de combiner ces interfaces en utilisant la technologie Link Aggregation. Link Aggregation augmente la bande passante de votre DiskStation en regroupant plusieurs interfaces réseau et offre un basculement du trafic pour maintenir la connexion réseau en cas de panne de celle-ci.

Une fois vos interfaces LAN combinées, vous pouvez voir une nouvelle interface nommée Bond dans Panneau de configuration > Réseau > Interface réseau, et qui peut être configurée comme toute autre interface (telle que LAN ou PPPoE).

Remarque :

  • Lorsque Link Aggregation est activé, le trafic réseau est automatiquement ajusté et équilibré pour chaque périphérique connecté.
  • Si les interfaces réseau sont connectées à plusieurs commutateurs, assurez-vous que les ports des commutateurs se trouvent sur le même LAN ou VLAN.

Pour combiner plusieurs réseaux locaux avec Link Aggregation :

  1. Accédez dans Panneau de configuration > Réseau > Interface réseau. Cliquez sur Créer > Créer Bond.
  2. Sélectionnez le mode souhaité.

    Descriptions du mode :

    • Adaptive Load Balancing : Ce mode permet d'optimiser le trafic réseau reçu et envoyé par votre DiskStation, que le commutateur prenne en charge Link Aggregation ou non. Pour éviter des erreurs inattendues, n'activez pas Link Aggregation sur le commutateur, même lorsqu'il est pris en charge.
    • Link Aggregation dynamique IEEE 802.3ad : Ce mode permet d'optimiser le trafic réseau reçu et envoyé par votre DiskStation, et nécessite l'activation de Link Aggregation (LACP, 802.1AX) (dynamique) IEEE 802.3ad sur le ou les commutateurs. Si vous utilisez plusieurs commutateurs, ceux-ci doivent être empilables et configurés de manière appropriée.
    • Balance XOR : Ce mode permet d'équilibrer le trafic réseau reçu et envoyé par votre DiskStation, et nécessite l'activation de Link Aggregation statique sur le ou les commutateurs. Si vous utilisez plusieurs commutateurs, ceux-ci doivent être empilables et configurés de manière appropriée. Si Link Aggregation n'est pas pris en charge par le commutateur, il est encore possible d'équilibrer le trafic envoyé par votre DiskStation. Grâce à ce mode, il est possible de détecter les liens perdus, mais pas la cause (par ex., lorsque que des câbles Ethernet sont débranchés ou que le commutateur n'est pas correctement configuré).
    • Actif/Veille : Ce mode offre une network fault tolerance à votre DiskStation. Cela signifie que lorsque l'interface réseau active tombe en panne, l'un des autres ports LAN assure la prise en charge afin de maintenir la connexion réseau. Vous pouvez sélectionner ce mode sans avoir à configurer le commutateur ou avec un commutateur qui prend en charge Link Aggregation IEEE 802.3ad.
  3. Sélectionnez les interfaces pour créer Link Aggregation.
  4. Configurez les paramètres de l'IP. Vous pouvez également activer VLAN, si nécessaire. Cliquez sur Appliquer pour achever l'installation.
  5. Lorsque vous avez terminé, vous pouvez voir la nouvelle interface nommée Bond 1 dans la liste Interface réseau.

Pour modifier les paramètres de Link Aggregation :

  1. Pour modifier le mode de Link Aggregation ou afficher les informations des périphériques physiques, accédez à Panneau de configuration > Réseau > Interface réseau.
  2. Sélectionnez l'interface Bond 1, et cliquez sur le bouton Modifier.
  3. Accédez aux onglets correspondant pour modifier les paramètres souhaités.

Pour supprimer Link Aggregation :

  1. Accédez dans Panneau de configuration > Réseau > Interface réseau.
  2. Sélectionnez l'interface Bond 1, et cliquez sur le bouton Supprimer.
  3. Lorsque vous avez terminé, vous pouvez voir que Bond 1 est maintenant remplacé par deux interfaces LAN.
