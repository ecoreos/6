Importer les utilisateurs

Vous pouvez créer des comptes utilisateurs sous forme de groupes en important une liste d'utilisateurs.

Pour créer la liste d'utilisateurs :

Méthode 1. Utilisez un éditeur de texte :

  1. Ouvrez un nouveau fichier avec un éditeur de texte.
  2. Entrez les informations de l'utilisateur dans l'ordre suivant en les séparant d'une tabulation (TAB) :
    • Nom d'utilisateur
    • Mot de passe
    • Description
    • Courrier électronique
    • Quota (Mo) du volume 1 (répétez cette étape si vous disposez de plusieurs volumes)

      Remarque :

      Les quotas d'utilisateurs ne sont pas pris en charge dans le système de fichiers Btrfs et sur les périphériques de stockage externes (par ex., le disque USB/eSATA).

  3. Allez à la ligne suivante et répétez l'étape 2 jusqu'à ce que tous les utilisateurs soient rentrés. Chaque ligne indique les informations d'un utilisateur.
  4. Enregistrez le fichier au format d'encodage UTF-8.

Méthode 2. Utilisez Microsoft Excel :

  1. Ouvrez un nouveau fichier Excel.
  2. Entrez les informations des utilisateurs dans l'ordre suivant en les séparant par des colonnes, sur une seule ligne :
    • Nom d'utilisateur
    • Mot de passe
    • Description
    • Courrier électronique
    • Quota (Mo) du volume 1 (répétez cette étape si vous disposez de plusieurs volumes)

      Remarque :

      Les quotas d'utilisateurs ne sont pas pris en charge dans le système de fichiers Btrfs et sur les périphériques de stockage externes (par ex., le disque USB/eSATA).

  3. Allez à la ligne suivante et répétez l'étape 2 jusqu'à ce que tous les utilisateurs soient rentrés. Chaque ligne indique les informations d'un utilisateur.
  4. Ouvrez un nouveau fichier avec un éditeur de texte. Copiez les entrées du tableur et copiez-les dans le fichier texte.
  5. Enregistrez le fichier au format d'encodage UTF-8.

Remarque :

  • Tous les champs sont optionnels sauf le nom d'utilisateur.
  • Le fichier texte en question pourrait ressembler à ceci :
    • rosy 1234 mère rosy@abc.com 80000
    chaddy 5678 frère chaddy@abc.com 80000
    danny 8901 cousin danny@abc.com 20000
    anny 2345 sœur anny@abc.com 20000

Pour importer la liste d'utilisateurs :

  1. Choisissez Créer > Importer les utilisateurs.
  2. Cochez Écraser les comptes en double si vous voulez remplacer les comptes en double existants par celui de la liste des utilisateurs.
  3. Cliquez sur Parcourir et sélectionnez le fichier de la liste des utilisateurs. Une fois le fichier sélectionné, la zone d'aperçu affichera la liste des utilisateurs importés. Les utilisateurs dont l'état est anormal ne seront pas importés.
  4. Cliquez sur Transférer pour créer les comptes utilisateur.

Remarque :

Le fichier de la liste des utilisateurs doit être au format UTF-8.
