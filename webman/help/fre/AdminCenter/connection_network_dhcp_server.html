DHCP Server

Votre DiskStation peut se comporter comme un serveur DHCP et attribuer des adresses IP dynamiques aux clients DHCP au sein de votre réseau local.

Pour activer un serveur DHCP :

  1. Accédez à Panneau de configuration > DHCP Server > Interface réseau.
  2. Sélectionnez une interface réseau (p. ex. LAN 1) pour affecter des adresses IP et cliquez sur Modifier.
  3. Dans la fenêtre contextuelle, cliquez sur l'onglet DHCP Server et sélectionnez Activer DHCP Server.
  4. Spécifiez les paramètres DHCP suivants :
    • Durée de bail de l'adresse (minutes)
    • DNS principal
    • DNS secondaire (optionnel)
    • Nom de domaine (optionnel)
    • Activer la découverte automatique du proxy web (optionnel)
  5. Configure au moins un sous-réseau. Pour la configuration du sous-réseau, cliquez sur Ajouter pour spécifier les informations suivantes :
    • Adresse IP de début
    • Adresse IP de fin
    • Masque de sous réseau
    • Passerelle
  6. Cliquez sur Créer pour afficher le sous-réseau créé dans la liste.
  7. Après avoir créé suffisamment de sous-réseaux, n'oubliez pas d'en activer au moins un.
  8. Cliquez sur OK.

Remarque :

  • Vous pouvez créer autant de sous-réseaux que vous le souhaitez.
  • S'il est coordonné avec un serveur relai DHCP, votre DiskStation, en tant que serveur DHCP, peut affecter des adresses IP à des périphériques sur un autre sous-réseau.

Clients DHCP

Vous pouvez trouver une liste des clients DHCP et leur configuration réseau (comme adresse MAC, adresse IP, nom d'hôte, temps restant avant l'expiration du bail DHCP et état de la réservation).

  1. Accédez à Panneau de configuration > DHCP Server > Interface réseau et sélectionnez l'interface souhaitée.
  2. Cliquez sur Modifier > Clients DHCP pour rechercher les clients et les détails.

Pour gérer des adresses IP pour des clients DHCP :

Vous pouvez faire en sorte qu'un client reçoive toujours la même adresse IP lors du renouvellement du bail DHCP et gérer ces clients.

  • Pour ajouter des clients DHCP : Cliquez sur Ajouter pour affecter l'adresse IP au périphérique client.
  • Pour réserver l'adresse IP : Cliquez sur l'icône du verrou pour réserver l'adresse IP pour le périphérique actuel.
  • Pour annuler la réservation : Cliquez sur l'icône du verrou pour autoriser l'utilisation de l'adresse IP actuelle par un autre périphérique une fois la durée du bail de l'adresse écoulée.

Remarque :

Si un périphérique possède des adresses IPv4 et IPv6, vous pouvez réserver une adresse IPv4 et une adresse IPv6 pour le périphérique.

Pour rechercher des clients DHCP :

  1. Cliquez sur l'icône de l'entonnoir, dans le coin supérieur droit.
  2. Sélectionnez un des critères de recherche ci-dessous :
    • Tous : Répertorier tous les clients DHCP et toutes les réservations DHCP.
    • Clients DHCP : Répertorier tous les clients DHCP avec des adresses IP réservées ou non.
    • Réservations DHCP : Répertorier toutes les adresses IP réservées Une adresse IP réservée peut ne pas encore être associée à un périphérique.

Options DHCP

Après avoir spécifié les options DHCP, votre DiskStation, en tant que serveur DHCP, est en mesure d'envoyer les informations connexes aux clients DHCP d'un sous-réseau particulier.

Pour spécifier les options DHCP d'un sous-réseau :

  1. Accédez à Panneau de configuration > DHCP Server > Interface réseau.
  2. Sélectionnez l'interface réseau souhaitée et cliquez sur Modifier.
  3. Accédez à DHCP Server, puis à la Liste de sous-réseau.
  4. Effectuez l'une des actions suivantes :
    • Pour spécifier les options DHCP d'un sous-réseau existant : Sélectionnez l'interface souhaitée et cliquez sur Modifier.
    • Pour spécifier un sous-réseau et ses options DHCP : Cliquez sur Ajouter et spécifiez les paramètres de sous-réseau.
  5. Sélectionnez les options DHCP souhaitées ; pour spécifier les valeurs correspondantes, cliquez sur les champs vierges à éditer.
  6. Cliquez sur OK.

Fournisseur

Les clients DHCP de différents fournisseurs peuvent nécessiter des informations DHCP supplémentaires. En configurant les options DHCP des fournisseurs, votre DiskStation est en mesure de fournir ces détails aux clients.

Contactez les fournisseurs en particulier pour connaître les options DHCP détaillées prises en charge par chaque fournisseur.

Pour ajouter des fournisseurs :

  1. Accédez à Panneau de configuration > DHCP Server > Fournisseur.
  2. Cliquez sur Créer pour spécifier le fournisseur et ses options DHCP :
    • Nom : Spécifiez le nom du fournisseur.
    • Code : Spécifiez le code numérique de l'option DHCP.
    • Valeur : Spécifiez la valeur de l'option.
    • + : ajouter une option DHCP.
    • - : supprimer une option DHCP.
  3. Cliquez sur Créer pour terminer la configuration.

Pour modifier les options DHCP des fournisseurs :

  1. Accédez à Panneau de configuration > DHCP Server > Fournisseur.
  2. Sélectionnez le fournisseur souhaité et cliquez sur Modifier pour gérer les options DHCP.
  3. Cliquez sur OK.

PXE

Votre DiskStation peut se comporter comme un serveur DHCP pour mettre en œuvre la configuration de PXE (Pre-Boot Execution Environment). PXE est un environnement réseau qui permet aux machines clientes de démarrer grâce à une interface réseau au lieu d'un périphérique de stockage de données (par ex. les disques durs).

Lors de l'implémentation de PXE, un chargeur de démarrage peut être stocké sur votre DiskStation ou un TFTP distant, et les appareils clients peuvent démarrer sur le réseau en utilisant le chargeur de démarrage.

Pour utiliser votre DiskStation comme un serveur de chargeur de démarrage :

  1. Assurez-vous d'avoir sélectionné Activer le service TFTP (dans Panneau de configuration > Services de fichiers > TFTP).
  2. Sélectionnez un dossier qui héberge le fichier du chargeur de démarrage en tant que dossier racine TFTP.
  3. Assurez-vous que le sous-réseau DHCP est activé (dans Panneau de configuration > DHCP Server > Interface réseau).
  4. Accédez à Panneau de configuration > DHCP Server > PXE.
  5. Sélectionnez Activer PXE (Pre-boot Execution Environment).
  6. Sélectionnez Serveur TFTP local et le fichier du chargeur de démarrage.
  7. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour utiliser un serveur TFTP comme serveur de démarrage :

En utilisant un serveur TFTP comme serveur de chargeur de démarrage, vous pouvez autoriser des clients à démarrer sans qu'ils aient à modifier l'infrastructure actuelle du réseau.

  1. Assurez-vous que le sous-réseau DHCP est activé (dans Panneau de configuration > DHCP Server > Interface réseau).
  2. Accédez à Panneau de configuration > DHCP Server > PXE.
  3. Sélectionnez Activer PXE (Pre-boot Execution Environment).
  4. Sélectionnez Serveur TFTP distant, et spécifiez l'adresse IP du serveur et le chemin d'accès du fichier du chargeur de démarrage.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.
