Planificateur de tâches

Le Planificateur de tâches vous permet de planifier et d'exécuter des services ou des scripts à des horaires prédéfinis.

Tâches prises en charge

Des tâches peuvent être créées pour faire ce qui suit :

  • Script défini par l'utilisateur : Exécuter des commandes ou des scripts définis par l'utilisateur.
  • Corbeille :Vider la corbeille de certains dossiers partagés spécifiques. De plus, les paramètres de la politique de conservation vous permettent de déterminer les fichiers qui doivent être définitivement supprimés lorsque vous videz la corbeille. Par exemple, les fichiers peuvent être supprimés ou conservés selon leur taille, leur temps de suppression ou le format de fichier.
  • Contrôle du bip : Fait biper le DiskStation.
  • Service : Démarre ou arrête les services réseau, tels que le Service de fichiers Windows, FTP, Blocage auto, etc.
  • Script défini par l'utilisateur : Exécuter des commandes ou des scripts définis par l'utilisateur.

  • Service : Démarre ou arrête les services réseau, tels que le Service de fichiers Windows, FTP, Blocage auto, etc.

Gestion des tâches

La page principale du Planificateur de tâches affiche toutes les tâches existantes et fournit des options pour créer, modifier ou exécuter des tâches.

Pour créer une tâche planifiée :

Les tâches planifiées sont des tâches qui s'exécutent à une date, une heure ou un intervalle spécifique.

  1. Accédez à Panneau de configuration > Planificateur de tâches, cliquez sur Créer et sélectionnez Tâche planifiée.
  2. Sélectionnez le type de tâche que vous souhaitez planifier : Script défini par l'utilisateur, Corbeille, Contrôle du bip ou Service.
  3. Quand la fenêtre Créer une tâche apparaît, cliquez sur Général et modifiez les paramètres suivants :
    • Paramètres généraux : Saisissez un nom de tâche, sélectionnez un utilisateur pour lequel la tâche s'exécutera et activez ou désactivez la tâche.
  4. Cliquez sur Planification et modifiez les paramètres suivants :
    • Date : Spécifiez le moment d'exécution de cette tâche. Les tâches peuvent être planifiées pour s'exécuter certains jours de la semaine (par ex. tous les jours, les weekends ou tous les dimanches, etc.) ou à des dates spécifiques (par exemple le 21/10/2015).
    • Nbre de fois : Spécifiez combien de fois la tâche s'exécute chaque jour, ainsi que la première et la dernière heure d'exécution.
  5. Cliquez sur Paramètres de la tâche pour organiser les paramètres propres à la tâche (uniquement pour certaines tâches).
    • Pour recevoir des détails sur l'exécution, sélectionnez Envoyer les détails d'exécution par e-mail.
    • Pour être informé uniquement lorsque des anomalies surviennent, sélectionnez Envoyer les détails d'exécution uniquement lorsque le script se termine de manière anormale.
    • Pour personnaliser le contenu de la tâche, entrez le scipt dans Script défini par l'utilisateur.
    • Si vous avez sélectionné Corbeille ou Service à l'étape 2, vous pouvez décider comment et quand les corbeilles sont vidées ou les services à démarrer/arrêter.
  6. Cliquez sur OK pour enregistrer les paramètres.

Pour créer une tâche déclenchée :

Les tâches déclenchées sont des tâches qui s'exécutent avant/après la réalisation d'événements spécifiques.

  1. Accédez à Panneau de configuration > Planificateur de tâches, cliquez sur Créer et sélectionnez Tâche déclenchée.
  2. Sélectionnez Script défini par l'utilisateur.
  3. Quand la fenêtre Créer une tâche apparaît, cliquez sur Général et modifiez les paramètres suivants :
    • Paramètres généraux : Saisissez un nom de tâche, sélectionnez un utilisateur pour lequel la tâche s'exécutera, sélectionnez un événement/une pré-tâche qui déclenchera la tâche et activez ou désactiver la tâche.
  4. Cliquez sur Paramètres de la tâche pour organiser les paramètres propres à la tâche (uniquement pour certaines tâches).
    • Pour recevoir des détails sur l'exécution, sélectionnez Envoyer les détails d'exécution par e-mail.
    • Pour être averti uniquement lorsque des anomalies surviennent, sélectionnez Envoyer les détails d'exécution uniquement lorsque le script se termine de manière anormale.
    • Pour personnaliser le contenu de la tâche, entrez le scipt dans Script défini par l'utilisateur.
  5. Cliquez sur OK pour enregistrer les paramètres.

Remarque :

  • Lors de la création d'une tâche déclenchée, il est possible que vous ayez à spécifier sa ou ses pré-tâches. La tâche créée ne s'exécute qu'une fois la pré-tâche effectuée.
  • Après que vous avez activé Envoyer les détails d'exécution uniquement lorsque le script se termine de manière anormale, le système ne fournit pas les détails si le script se termine par 0.

Pour modifier les paramètres de tâche :

  1. Sélectionnez une tâche existante.
  2. Cliquez sur Modifier.
  3. Une fenêtre apparaît dans laquelle vous pouvez modifier les paramètres de la tâche.

Pour supprimer une tâche :

  1. Sélectionnez une tâche existante.
  2. Cliquez sur Action > Supprimer.

Pour activer/désactiver une tâche :

  1. Sur la page principale du Planificateur de tâches, cochez ou décochez la case appropriée sous Activé pour activer ou désactiver une tâche.
  2. Cliquez sur Enregistrer.

Pour exécuter manuellement une tâche :

  1. Sélectionnez une ou plusieurs tâches existantes.
  2. Cliquez sur Exécuter.

Limites du Planificateur de tâches

  • Les noms de tâches peuvent comprendre 512 caractères affichables.
  • Nombre maximum de tâches : 1024.
  • Longueur maximale du chemin du script ou de la ligne de commande : 4095.
  • La durée du bip peut être définie de 1 à 600 secondes.
