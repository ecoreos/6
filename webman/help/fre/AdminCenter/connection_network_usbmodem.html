Mobile*

Lorsque l'appareil est équipé d'un dongle mobile à large bande, le DiskStation peut se connecter à Internet via un réseau sans fil 3G/LTE. Les paramètres figurant dans Panneau de configuration > Réseau > Interface réseau > Mobile vous aident à établir et à modifier une connexion Internet sans fil via votre dongle à large bande mobile.

Remarque :

Pour connaître la liste complète des dongles mobiles large bande pris en charge par les produits Synology, veuillez visiter www.synology.com.

Pour configurer un réseau mobile :

Lorsqu'un dongle mobile à large bande est connecté à l'un des ports USB sur votre DiskStation, le DiskStation essaie de se connecter automatiquement. S'il est impossible d'établir automatiquement une connexion ou si vous désirez modifier les paramètres de connexion, veuillez consulter ce qui suit :

  • Déverrouillage/paramétrages SIM : Si la carte SIM est protégée par un code PIN, elle doit être débloquée pour permettre au DiskStation de se connecter au réseau. Une fois débloquée, cette section vous fournit des options pour modifier les paramétrages PIN.
  • Modifier : Vous pouvez configurer manuellement la connexion ou la définir en fonction de l'emplacement ou de l'opérateur. La configuration manuelle de la connexion requiert la saisie de vos APN, numéro de téléphone, nom d'utilisateur et mot de passe. Si vous ne connaissez pas ces informations, veuillez contacter votre opérateur mobile pour un support.
  • Connexion/déconnexion : Ces boutons vous permettent de vous connecter ou de vous déconnecter du DiskStation depuis le réseau 3G/LTE.
  • Options : Cette section inclut une option pour définir l'interface du dongle mobile à large bande comme la passerelle par défaut pour votre DiskStation. Lorsque vous configurez le réseau mobile comme passerelle par défaut, le trafic Internet depuis votre DiskStation passe via le réseau mobile au lieu du réseau câblé.

Remarque :

  1. Lors d'une connexion à Internet, certains dongles mobiles à large bande n'affichent pas la force du signal s'ils ne prennent pas en charge cette fonction.
  2. Pour certains dongles mobiles à large bande, vous ne pouvez pas changer les paramètres SIM lorsqu'ils sont connectés à Internet.
