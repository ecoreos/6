Service Push

Le service Push permet au DiskStation d'envoyer des notifications vers votre appareil mobile ou navigateur Web quand des événements ou des erreurs spécifiques se produisent.

Ports :

Avant d'activer le service push, veuillez vous assurer que le DiskStation peut accéder à Internet en utilisant le port 8089.

Pour activer la notification par email fournie par Synology :

  1. Cochez Envoi de notifications concernant l'état du système via le serveur de courriels de Synology.
  2. Saisissez une ou plusieurs adresses e-mail dans le champ Destinataires. Les notifications système sont envoyées aux adresses saisies ici. Vous pouvez saisir deux adresses. Les comptes multiples peuvent être séparés avec un point-virgule (;).
  3. Cliquez sur Appliquer, et un email de vérification doit être envoyé aux adresses spécifiées dans le champ Destinataires. Veuillez suivre les étapes mentionnées dans le mail de vérification.
  4. De retour sur votre ordinateur, cliquez sur Envoyer un message test puis cliquez sur OK pour envoyer une notification test.

Pour activer les notifications des appareils mobiles :

  1. Accédez à Panneau de configuration > Notification > Service Push.
  2. Cochez la case Activer les notifications des appareils mobiles, puis cliquez sur Appliquer.
  3. Installez et lancez DS finder sur votre appareil mobile, puis connectez-vous à votre DiskStation dans la liste Favoris pour lier votre appareil avec le DiskStation.
  4. Cliquez sur Envoyer un message test puis cliquez sur OK pour envoyer une notification test. Ensuite lancez DS finder sur votre appareil mobile pour vérifier si le message test est reçu.

Pour gérer les appareils mobiles liés :

  1. Cliquez sur Gérer les appareils liés pour voir la liste de tous les appareils mobiles liés.
  2. Si vous voulez délier un des appareils, sélectionnez-le et cliquez sur Délier. (Appuyez sur Maj ou Ctrl et maintenez cette touche enfoncée tout en sélectionnant plusieurs éléments.) Aucune notification ne leur sera envoyée à moins que vous les liiez à nouveau.

Pour activer les notifications du navigateur :

  1. Cochez Activer les notifications des appareils mobiles, et ensuite cliquez sur Appliquer pour activer cette fonction.
  2. Cliquez sur Activer les notifications du navigateur pour lier votre DiskStation avec le navigateur.
  3. Effectuez l'une des actions suivantes qui apparaît dans la fenêtre :
    • Pour Chrome : Cliquez sur Installer maintenant pour installer l'extension Chrome et ensuite créer-vous un compte Google si vous n'en avez pas.
    • Pour Safari : Cliquez sur Autoriser pour autoriser le service notification.
  4. Lorsque vous êtes redirigé vers la page de notification du navigateur, cliquez sur l'onglet Appareil lié pour voir l'information des appareils liés ou pour délier votre DiskStation du navigateur, ou cliquez sur l'onglet Événement pour voir l'historique de notification de votre navigateur durant les sept derniers jours.

Remarque :

Le service de notification est pris en charge sous Chrome (35 ou ultérieur) et Safari (OS X v10.9 ou ultérieur).
