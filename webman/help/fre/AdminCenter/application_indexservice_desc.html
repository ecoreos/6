Service d'indexation

Indexation multimédia peut automatiquement analyser les fichiers multimédias, tels que les photos, la musique et les vidéos stockées sur votre DiskStation, et les compiler dans la bibliothèque multimédia pour être utilisés par les applications multimédia. Vous pouvez également paramétrer la qualité miniature pour les photos et les vidéos, gérer la conversion des paramétrages pour les vidéos et créer des dossiers indexés auxquels vous pouvez accéder par les appareils DMA et les applications multimédia DSM.

L'indexation de fichier vous permet d'indexer des dossiers et les fichiers qu'ils comportent sur votre DiskStation par nom de fichier, et vous pouvez approfondir l'indexation du contenu/des métadonnées des fichiers par type de fichier. Cette fonction d'indexation détaillée permet d'augmenter la vitesse de recherche pour obtenir des correspondances plus précises.
