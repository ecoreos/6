Bibliothèque multimédia

La bibliothèque média peut automatiquement analyser les fichiers multimédia, tels que les photos, la musique et les vidéos stockées sur votre DiskStation, et les compiler dans la bibliothèque multimédia pour être utilisés par les applications multimédia.

Avec la bibliothèque média, vous pouvez également paramétrer la qualité miniature pour les photos et les vidéos, gérer la conversion des paramétrages pour les vidéos et créer des dossiers indexés auxquels vous pouvez accéder par les appareils DMA et les applications multimédia DSM.
