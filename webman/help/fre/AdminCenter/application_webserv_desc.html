Services Web

Votre DiskStation est présenté avec des caractéristiques de contenu riche en web. Avec Web Station, vous pouvez facilement abriter et modifier votre propre site web avec la prise en charge VHost et les paramètres HTTP/HTTPS supplémentaires. De façon alternative, vous pouvez utiliser la prise en charge PHP intégrée et créer des sites web dynamiques, alimentés par une base de données pour votre usage personnel ou votre entreprise.
