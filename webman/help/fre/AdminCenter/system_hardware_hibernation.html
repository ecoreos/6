Hibernation du HDD

Vous pouvez spécifier la durée pendant laquelle les disques durs restent inactifs avant d'entrer en mode d'hibernation HDD. Pendant l'hibernation du HDD, les disques durs arrêtent de tourner, ce qui réduit la consommation d'énergie et prolonge la durée de vie des disques durs. Vous pouvez spécifier différentes durées pour les disques durs internes et les disques durs externes eSATA/USB.

Remarque :

  • L'hibernation HDD cesse lors de l'accès.
  • Seuls certains périphériques de stockage USB externes prennent en charge l'hibernation HDD.

Pourquoi mon DiskStation n'hiberne-t-il pas ?

Veuillez noter que les services suivants empêcheront le DiskStation d'entrer en mode d'hibernation. En outre, le système quittera le mode d'hibernation deux minutes avant toute tâche planifiée.

Paquets :

  • Download Station : Le DiskStation ne peut pas hiberner si Download Station exécute eMule, un téléchargement planifié RSS, ou des tâches actives.
  • Mail Station : Le DiskStation ne peut pas hiberner si Mail Station récupère des données depuis POP3.
  • Surveillance Station : Le DiskStation ne peut pas hiberner si Surveillance Station enregistre.
  • Serveur multimédia : Le DiskStation ne peut pas hiberner si le journal DMA est activé dans les paramètres de Serveur multimédia (allez à Menu principal > Serveur multimédia > Serveur multimédia > Diagnostiquer).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Autre logiciel tiers ou optware tel que SABnzbd

Processus :

  • Miniatures et indexation : le DiskStation ne peut pas hiberner pendant la génération des miniatures ou la ré-indexation des fichiers multimédia avant la mise à niveau. Pendant ce temps, le processus "synoindexd" affichera l'état "En cours d'exécution" (allez à Menu principal > Moniteur de ressources > Processus).
  • Local Master Browser : le DiskStation ne peut pas hiberner si Local Master Browser est activé (allez à Panneau de configuration > Win/Mac/NFS > Service de fichiers Windows > Activer Local Master Browser).
  • Lecteur Windows Media : Le DiskStation ne peut pas hiberner si le Service Partage réseau du Lecteur Windows Media est activé dans le réseau local.
  • Le DiskStation ne peut pas hiberner si le nombre de processus concurrents nécessite l'échange, où si la capacité de RAM a été dépassée et les disques durs sont utilisés temporairement pour la lecture/écriture.

Autres

  • Périphériques USB : Le DiskStation ne peut pas entrer en hibernation du système si des appareils USB sont connectés à l'unité.

Sommeil profond

Le sommeil profond les pour unités d'expansion peut être activé via l'interface du DiskStation associé. Vous pouvez configurer l'unité d'expansion et le DiskStation associé pour être inactifs pendant des durées différentes. Par conséquent, l'unité d'expansion peut être réglée pour entrer dans un sommeil profond pendant que le DiskStation associé reste en ligne.

En mode sommeil profond :

  • Tous les voyants de l'unité d'extension seront éteints.
  • L'alimentation des disques durs de l'unité d'extension sera coupée.
  • Le ventilateur de l'unité d'expansion s'arrêtera.

Remarque :

  • Le sommeil profond cesse lors de l'accès.
  • Pour assurer une opération normale lors de l'utilisation de la fonction sommeil profond sur l'unité d'expansion, vous devez déplacer le commutateur Auto/Manual sur le panneau arrière sur Auto.

Allumage/extinction auto

Vous pouvez spécifier la longueur de la période de temps pendant laquelle le disque interne reste en hibernation avant que votre DiskStation s'arrête automatiquement.

Remarque :

  • Vous devez d'abord activer l'hibernation du HDD.
  • Vous devez d'abord activer le service WOL afin que vous puissiez alimenter à distance votre DiskStation.
