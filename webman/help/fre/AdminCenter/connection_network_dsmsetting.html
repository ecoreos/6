Paramètres de DSM

Dans Panneau de configuration > Réseau > Paramètres de DSM, vous pouvez modifier le nombre de ports que les clients utilisent pour accéder à votre DiskStation.

Pour modifier le numéro de port par défaut :

Les connexions HTTP et HTTPS sont toujours activées dans DSM. Le port HTTP par défaut est 5000 et le port HTTPS par défaut est 5001.

  1. Saisissez le numéro de port dans le champ HTTP ou HTTPS.
  2. Cliquez sur Appliquer.

Pour rediriger les connexions HTTP vers HTTPS :

Vous pouvez rediriger des clients vers le port HTTPS lorsqu'ils essaient d'accéder à votre DiskStation via le port HTTP.

  1. Sélectionnez Rediriger automatiquement les connexions HTTP vers le HTTPS (exceptés Web Station et Photo Station).
  2. Cliquez sur Appliquer.

Ports utilisés pour la connexion HTTPS :

  • DSM : Le numéro de port par défaut est 5001, de sorte que vous pouvez accéder à DSM de façon sécurisée en saisissant l'adresse suivante : https://server-hostname:5001/
  • Web Station et Photo Station : Le numéro de port par défaut est 443, de sorte que vous pouvez accéder à Web Station and Photo Station de façon sécurisée en saisissant l'adresse suivante : https://server-hostname/

Pour découvrir le DiskStation sur le réseau :

Vous pouvez trouver votre DiskStation via la découverte de réseau de Windows ou la découverte de service Bonjour.

  1. Sélectionnez un service pour rechercher votre DiskStation sur le réseau :
    • Activer la découverte du réseau Windows
    • Activer la découverte du service Bonjour
  2. Cliquez sur Appliquer.

Domaine

Dans Domaine, vous pouvez spécifier un nom de domaine personnalisé pour accéder à DiskStation.

Pour activer un domaine personnalisé :

  1. Cochez Activer un domaine personnalisé.
  2. Spécifiez un nom de domaine appliqué pour votre DiskStation.
  3. Vous pouvez également cocher Activer HSTS pour forcer les connexions chiffrées HTTPS sur ce domaine.
  4. Cliquez sur Appliquer.

Remarque :

  • Chaque nom de domaine ne peut être utilisé que pour une application distincte sur votre DiskStation.
  • Si la longueur du nom de domaine n'excède pas 15 caractères et qu'elle est conforme aux conventions d'attribution de noms de NetBIOS, le système enregistre et diffuse automatiquement le nom du domaine (par ex., MyDSM). Les utilisateurs de Windows peuvent accéder au bureau DSM sur le même réseau local grâce à une URL similaire : http://MyDSM or https://MyDSM
  • Lorsque vous accédez à des applications via des URL qui utilisent des noms de domaine, vous passez par le port standard HTTP (80) ou le port standard HTTPS (443).
