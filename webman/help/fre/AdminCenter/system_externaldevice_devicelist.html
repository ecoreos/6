Périphériques externes

Accédez à Panneau de configuration > Périphériques externes > Périphériques externes pour afficher une liste des périphériques de stockage externes connectés à votre DiskStation.

Disques externes

En connectant un disque externe au système, vous pouvez partager la capacité du disque par le biais d’un dossier partagé créé par le système, appelé usbshare[numéro] (pour les disques USB). Le dossier partagé est supprimé automatiquement lorsque le disque externe est débranché du système.

En connectant un disque externe au système, vous pouvez partager la capacité du disque par le biais d’un dossier partagé créé par le système appelé usbshare[numéro] (pour les disques USB) ou satashare[numéro] (pour les disques eSATA). Le dossier partagé est supprimé automatiquement lorsque le disque externe est débranché du système.

DiskStation reconnaît les formats suivants : Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus et NTFS. Tout disque externe non reconnu doit être d'abord formaté avant d’être utilisé sur le système.

Remarque :

  1. Certains modèles prennent en charge HFS/HFS Plus en lecture seulement.
  2. Le journal n'est pas pris en charge sur HFS/HFS Plus.
  3. Veillez à éjecter le disque externe avant de le débrancher.

Pour formater le disque :

Cliquez sur Formater et suivez l'assistant. Vous pouvez formater vos disques externes aux formats ext4 et FAT32 dans votre DiskStation.

Remarque :

  1. Veillez à sauvegarder les données avant de formater le disque externe. Toutes les données seront supprimées pendant le processus, et elles ne pourront pas être restaurées.
  2. Pour formater le disque externe au format ext3 ou ext4, la taille du disque doit être supérieure à 1 Go.
  3. Lorsque le disque est formaté au format ext3 ou ext4, il n’est reconnu que par DiskStation. Si vous souhaitez lire les données directement depuis votre disque externe avec un PC et/ou un Mac, choisissez le format FAT32.

Pour planifier les tests S.M.A.R.T. :

Le test S.M.A.R.T. examine et génère un rapport sur l'état de vos disques durs, vous alertant à l'avance des défaillances imminentes des disques. Il est recommandé de changer rapidement votre disque si des erreurs sont détectées.
Pour effectuer le test S.M.A.R.T. Test, cliquez simplement sur le bouton Test S.M.A.R.T. et suivez l'assistant.
Pour vérifier les résultats du test, cliquez simplement sur Info S.M.A.R.T..

Remarque :

La durée du test varie en fonction de votre environnement système.

Limites

Le formatage du disque n'est pas disponible dans les situations suivantes :

  • Le disque externe est en cours d'initialisation (lorsque le périphérique externe vient juste d'être branché).
  • La capacité du disque externe est inférieure à 1 Go (format ext3 ou ext4).
  • Le disque externe est en cours de formatage.
  • Lorsque le système sauvegarde des fichiers sur le disque externe.
  • Le disque externe est endommagé.

Dépannage

  1. Échec de formatage du disque externe. Pourquoi ?
    • Vérifiez que le disque externe est inséré correctement.
    • Vérifiez que le disque externe n'est pas protégé en écriture. Le cas échéant, supprimez cette option avant de le formater.
  2. Le type de système de fichiers du disque externe est affiché comme Non reconnu, ou le système ne crée pas le dossier partagé usbshare[numéro] ou esatashare[numéro] pour le disque externe. Que faire ?
    • Le système actuel reconnaît les formats Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus et NTFS. Tout disque externe non formaté aux types ci-dessus n'est reconnu et son statut est noté comme Non reconnu.
  3. Je ne peux pas accéder au dossier partagé usbshare[numéro] ou satashare[numéro]. Que faire ?
    • Cela peut être causé par un problème dans le système de fichiers sur votre disque externe. Pour corriger ce problème, veuillez connecter le disque externe à un PC et effectuez une vérification du système de fichiers.
  4. Mon disque externe au format ext3 ou ext4 peut-il être connecté à mon PC et/ou Mac et accédé directement ?
    • Non, ext3 ou ext4 est uniquement reconnu par le DiskStation. Si vous souhaitez lire les données directement depuis votre disque externe avec un PC et/ou un Mac, choisissez le format FAT32.
  5. Pourquoi vois-je un dossier appelé « lost+found » dans mon disque externe ?
    • Il s'agit d'un dossier système par défaut, créé automatiquement lors de l'utilisation du système de fichier ext3/ext4. Il est normal que vous le voyiez dans votre disque externe. Bien que les administrateurs disposent de l'autorisation d'accéder et de gérer son contenu, nous vous recommandons fortement de ne pas toucher à ce dossier.

Pour paramétrer les autorisations par défaut pour tous les appareils externes :

Vous pouvez paramétrer les autorisations par défaut en cliquant sur Autorisations par défaut et spécifiez quels utilisateurs ou groupes peuvent accéder, voir ou accéder aux appareils externes et leurs contenus. Lorsqu'un appareil externe est inséré dans votre DiskStation, les autorisations que vous avez spécifiées sont assignées automatiquement à l'appareil externe.

Dans la fenêtre des autorisations, effectuez ce qui suit pour paramétrer les autorisations par défaut :

  1. Choisissez l'un des éléments suivants dans le menu déroulant :
    • Utilisateur du système interne : Attribue des privilèges d'accès aux utilisateurs système par défaut, tels que l'utilisateur anonyme FTP/WebDAV.
    • Utilisateurs locaux : Attribue des permissions aux utilisateurs locaux (y compris invité).
    • Groupes locaux : Attribue des permissions aux groupes locaux.
  2. Cochez ou décochez les cases appropriées pour chaque utilisateur ou groupe afin de personnaliser leurs privilèges d'accès pour les appareils externes :
    • Lecture/écriture : L'utilisateur ou le groupe peut accéder et apporter des modifications aux fichiers et dossiers sur les appareils externes.
    • Lecture seule : L'utilisateur ou le groupe peut accéder aux fichiers et dossiers du sur les appareils externes.
    • Pas d’accès : L'utilisateur ou le groupe ne peut pas accéder aux fichiers et dossiers du sur les appareils externes.
