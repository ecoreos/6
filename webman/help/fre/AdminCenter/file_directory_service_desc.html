Service d’annuaire

Votre DiskStation peut rejoindre un service d'annuaire existant en tant que domaine Windows ou client LDAP. Vous pouvez gérer les privilèges d'accès du domaine ou des utilisateurs LDAP pour partager les dossiers et les applications DSM, et activer le service d'accueil pour ceux-ci. Allez dans Panneau de configuration > Service d'annuaire pour joindre votre DiskStation à un service d'annuaire.
