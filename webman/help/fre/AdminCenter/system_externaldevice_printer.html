Imprimantes

DiskStation peut devenir un serveur d'impression, permettant aux ordinateurs clients ou appareils mobiles d'accéder à des imprimantes connectées à DiskStation. De plus, la prise en charge de l'impression sans fil Apple et de Google Cloud Print vous permet d'imprimer à distance en utilisant des appareils iOS ou des produits et services Google.

Rendez-vous sur www.synology.com pour connaître les imprimantes compatibles.

Configurer les imprimantes

Consultez les instructions ci-dessous pour configurer et connecter une imprimante USB ou réseau avec votre DiskStation.

Pour configurer une imprimante USB :

  1. Allumez et connectez l'imprimante à un port USB sur le DiskStation.
  2. accédez à Panneau de configuration > Périphériques externes > Imprimante.
  3. Sélectionnez l'imprimante que vous voulez configurer. Veuillez noter que le DiskStation ne peut pas reconnaître les imprimantes connectées via des concentrateurs USB.
  4. Cliquez sur Gestionnaire d'imprimante et sélectionnez Configurer l'imprimante.
  5. Dans la boîte de dialogue, sélectionnez l'un des modes suivants :
    • Network MFP : Permet aux clients sur le réseau local d'utiliser les fonctions imprimer, numériser et télécopie de votre imprimante multifonctions. Les fonctions numérisation et télécopie sont disponibles uniquement pour les ordinateurs Windows.
    • Imprimante réseau : Choisissez ce mode pour les imprimantes qui ne prennent en charge que l'impression ou si vous souhaitez autoriser les clients à utiliser la fonction d'impression uniquement.
  6. Configurez les paramètres d'Impression sans fil Apple ou de Google Cloud Print (optionnel).
  7. Cliquez sur Enregistrer.

Remarque :

Un MFP réseau ne peut être accédé que par un seul client à la fois.

Pour configurer une imprimante réseau :

  1. Allumez et connectez l'imprimante réseau au même réseau local que votre DiskStation.
  2. accédez à Panneau de configuration > Périphériques externes > Imprimante.
  3. Cliquez sur Ajouter une imprimante réseau et suivez les instructions de l'assistant de configuration.

Remarque :

  • Nom de l'imprimante : Lors de la configuration, vous devez saisir un nom d'imprimante qui peut comporter jusqu'à 32 caractères, comprenant des lettres, des chiffres et des traits de soulignement.
  • Protocole : En général, les imprimantes réseau HP utilisent Socket ; les imprimantes réseau Canon utilisent BJNP ; et d’autres utilisent le plus souvent LPR.
  • Nom de la file d'attente : Il est nécessaire d'entrer un nom de file d'attente pour les imprimantes réseau LPR et IPP, mais cela ne s'applique pas aux imprimantes Socket ou BJNP.
  • Pas de Network MFP : Les imprimantes réseau connectées à DiskStation ne peuvent actuellement pas utiliser le mode Network MFP.
  • Pas d'imprimante USB : Les imprimantes USB partagées par un produit Synology ne peuvent pas être définies comme imprimantes réseau.

Gestion des imprimantes

L'onglet des imprimantes comprend les options suivantes pour gérer les imprimantes USB ou réseau connectées au DiskStation.

  • Obtenir des informations : Affiche des informations relatives à l'imprimante sélectionnée, telles que le nom du périphérique, le fabricant, l'état, etc.
  • Supprimer : Supprime et déconnecte une imprimante du DiskStation.
  • Gestionnaire d'imprimante : Comprend les options suivantes :
    • Configurer l'imprimante  : Modifiez les paramètres de l’imprimante, telles que l'impression sans fil Apple ou Google Cloud Print (voir les instructions ci-dessous).
    • Nettoyer l'espace de la file d’attente : Annule toutes les tâches d'impression sur toutes les imprimantes.
    • Imprimer une page test : Imprime une page de test pour s'assurer que les configurations sont correctes.
    • Libérer l'imprimante : Déconnecte l'imprimante et le client (pour le mode Network MFP uniquement). Lorsqu'un client se connecte à une imprimante multifonctions pour recevoir des télécopies ou pour scanner des fichiers, la tâche peut prendre un certain temps. Si nécessaire, la connexion peut être arrêtée afin de permettre au client suivant dans la file d'attente d’accéder à l'imprimante.

Pour activer l'impression sans fil Apple :

  1. accédez à Panneau de configuration > Périphériques externes > Imprimante.
  2. Sélectionnez une imprimante dans la liste des périphériques.
  3. Cliquez sur Gestionnaire d'imprimante > Configurer l'imprimante.
  4. Cochez la case Activer l'impression sans fil Apple.
  5. Sélectionnez la marque et le pilote de votre imprimante.
  6. Cliquez sur Enregistrer.

Une fois la configuration terminée, vous pouvez imprimer une page de test en cliquant sur Gestionnaire d'imprimante > Imprimer une page test.

Remarque :

  • L'activation de l'impression sans fil Apple active également la fonction Diffusion imprimante Bonjour pour DSM.
  • L'impression sans fil Apple est disponible pour iOS 4.2 ou les versions ultérieures.

Pour activer Google Cloud Print :

  1. accédez à Panneau de configuration > Périphériques externes > Imprimante.
  2. Sélectionnez une imprimante dans la liste des périphériques.
  3. Cliquez sur Gestionnaire d'imprimante > Configurer l'imprimante.
  4. Cochez la case Activer Google Cloud Print.
  5. Saisissez les détails suivants :
    • Vos informations de compte Google.
    • La marque et le pilote de votre imprimante.
  6. Cliquez sur Enregistrer.

Une fois la configuration terminée, vous pouvez imprimer une page de test en cliquant sur Gestionnaire d'imprimante > Imprimer une page test.

Pour gérer les tâches Google Cloud Print, cliquez sur le lien Google Cloud Print.

Accès aux imprimantes

Voyez la section ci-dessous pour accéder aux imprimantes à l'aide d'un PC ou d'un Mac.

Pour accéder aux imprimantes à l'aide d'un PC Windows :

La meilleure façon d'accéder au service d'impression d'un périphérique DiskStation est de configurer la connexion à l’imprimante avec Synology Assistant. Consultez son Aide pour des instructions détaillées.

Si Synology Assistant n'est pas installé sur votre ordinateur, voir ci-dessous pour d'autres manières d'accéder au service d'impression du périphérique DiskStation.

Type 1. Imprimante réseau partagée

  1. Sur votre PC, ouvrez Réseau dans l'Explorateur de fichiers et double-cliquez sur le DiskStation.
  2. Double-cliquez sur usbprinter ou nom d'imprimante réseau ; le système vous invite à installer le bon pilote, puis cliquez sur Oui.
  3. Veuillez obtenir le pilote de l'imprimante sur le CD ou le site Web du fabricant. Sélectionnez le fabricant et le modèle de l'imprimante, ou cliquez sur Disque fourni pour sélectionner l'emplacement du pilote. Cliquez ensuite sur OK.
  4. Le système commence la configuration de l'imprimante réseau. Après la fin de la configuration, vous pouvez commencer à utiliser l'imprimante.

Type 2. Port LPR (Recommandé)

  1. Sur votre PC, cliquez sur Démarrer puis accédez à Périphériques et imprimantes.
  2. Cliquez sur Ajouter une imprimante.
  3. Choisissez Ajouter une imprimante réseau, sans fil ou Bluetooth.
  4. Choisissez Mon imprimante ne figure pas dans la liste.
  5. Sélectionnez Ajouter une imprimante à l'aide d'une adresse TCP/IP ou d'un nom d'hôte. Cliquez sur Suivant pour continuer.
  6. Choisissez le Type de périphérique Périphérique de port TCP/IP..
  7. Entrez l'adresse IP du DiskStation. Cliquez sur Suivant pour continuer.
  8. Sélectionnez Type de périphérique : Personnalisé > Paramètres > …. Cliquez sur Suivant pour continuer.
  9. Sélectionnez LPR en guise de protocole ; entrez usbprinter ou nom d'imprimante réseau pour le nom de file d'attente et cochez Comptage des octets LPR activé. Cliquez sur OK pour continuer.
  10. Veuillez obtenir le pilote de l'imprimante sur le CD ou le site Web du fabricant. Sélectionnez le fabricant et le modèle de l'imprimante, ou cliquez sur Disque fourni pour sélectionner l'emplacement du pilote. Cliquez ensuite sur OK.
  11. Le système commence la configuration de l'imprimante réseau. Après la fin de la configuration, vous pouvez commencer à utiliser l'imprimante.

Pour accéder aux imprimantes à l'aide d'un Mac :

Les utilisateurs de Mac peuvent accéder au service d'impression du DiskStation en activant la diffusion imprimante Bonjour de DSM et en ajoutant l'imprimante dans les Préférences du système.

Vérifiez bien que votre DiskStation utilise une imprimante Postscript avant de faire quoi que ce soit.

Etape 1. Pour activer la diffusion imprimante Bonjour de DSM :

  1. Connectez-vous à DSM en tant qu'admin ou utilisateur appartenant au groupe administrators.
  2. accédez à Panneau de configuration > Services de fichiers > Win/Mac/NFS.
  3. Cochez Activer la diffusion imprimante Bonjour.

Etape 2. Accès à l'imprimante à l'aide d'un Mac.

  1. Sur votre Mac, allez dans Préférences du système > Imprimer et numériser, puis cliquez sur +.
  2. Choisissez usbprinter[number] ou nom d'imprimante réseau dans le menu déroulant. Votre Mac configurera l'imprimante automatiquement.
