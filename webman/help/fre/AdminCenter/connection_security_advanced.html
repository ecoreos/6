Avancé

Vous pouvez modifier les paramètres suivants dans Panneau de configuration > Sécurité > Avancé pour les sessions de navigation DSM :

Compression HTTP

Vous pouvez activer Compression HTTP pour DiskStation. En réduisant la bande passante de trafic réseau, vous pouvez augmenter la vitesse de chargement de la page Web.

Pour activer/désactiver la compression HTTP：

  • Cochez/décochez Compression HTTP dans Panneau de configuration > Sécurité > Avancé.

Profil TLS / SSL

Vous pouvez choisir le niveau de sécurité des connexions chiffrées HTTPS. Il offre les 3 niveaux suivants :

  • Compatibilité moderne - Uniquement compatible avec les derniers navigateurs et périphériques mobiles.
  • Compatibilité intermédiaire - Niveau de sécurité compris entre le plus élevé et le plus bas. Il s'agit de la valeur par défaut du système.
  • Rétrocompatibilité ancienne - Non sécurisé, mais prend en charge la plupart des navigateurs et périphériques mobiles.
