Limites de connexion

Permettre une connexion SSL/TLS uniquement

Lorsque cette fonction est activée, le serveur n'accepte que les requêtes de connexion SSL ou TLS des clients FTP.

Limitez les connexions par IP

Lorsque cette fonction est activée, le serveur limitera les connexions par IP au nombre maximal. Activez cette fonction et la limite de bande passante FTP pour gérer efficacement la bande passante.

Activer la limite de bande passante FTP

Activez cette fonction pour limiter la bande passante d'une connexion FTP unique.
