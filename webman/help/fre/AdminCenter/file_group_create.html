Créer un groupe

  1. Allez à Panneau de configuration > Groupe.
  2. Cliquez sur Créer. Cette action lance l'Assistant de création d'utilisateur.
  3. Entrez le nom et la description du groupe. (Reportez-vous à la section ci-dessous pour plus d’informations). Cliquez sur Suivant.
  4. Suivez les étapes restantes de l'assistant pour terminer la création du groupe.
  5. Cliquez sur Appliquer pour terminer.

Conventions de dénomination :

  • Le nom du groupe est sensible à la casse et doit être composé de 1 à 32 caractères Unicode, sauf les symboles suivants :

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Le premier caractère ne peut pas être le signe moins ou un espace, et le dernier caractère ne peut pas être un espace.
  • La description du groupe est sensible à la casse et est limitée à 64 caractères Unicode affichables.
