Configuration du routeur

Si votre appareil DiskStation se trouve sur un réseau local, d'autres appareils sur Internet ne peuvent s'y connecter. Vous pouvez configurer les règles de transmission de port dans Panneau de configuration > Accès externe > Configuration du routeur pour rendre votre appareil DiskStation accessible sur Internet.

Configuration du routeur

Avant d'ajouter les règles de transmission de port, vous devez configurer une connexion entre votre appareil DiskStation et le routeur.

Remarque :

  1. Pour configurer les règles de transmission de ports et attribuer une adresse IP statique, vous devez disposer des privilèges administratifs sur le routeur.
  2. Avant de configurer la connexion du DiskStation à un routeur UPnP, assurez-vous que DSM a correctement récupéré l'adresse IP externe de votre DiskStation (dans Panneau de configuration > Accès externe > DDNS). Si ce n'est pas le cas, vérifiez les paramètres de réseau et de pare-feu de votre DiskStation et du routeur UPnP.

Pour configurer la connexion de l'appareil DiskStation à votre routeur :

  1. Cliquez sur Configuration du routeur pour lancer l'assistant de configuration. DSM commencera à tester les composants/paramètres du réseau suivant avant de passer à l'étape suivante.
    • Interface réseau
    • Connexion de la passerelle
    • Connexion Internet
    • Transmission de port
    • DNS
  2. Selon le type de votre routeur, effectuez l'une des actions suivantes :
    • Si votre routeur ne prend pas en charge l'UPnP, consultez l'étape 3.
    • Si votre routeur UPnP a été testé comme étant compatible avec DiskStation, cliquez sur Appliquer quand vous voyez un résumé des informations du routeur. La connexion de votre appareil DiskStation au routeur est configurée avec succès.
    • Si votre routeur UPnP n'a pas été testé comme étant compatible avec l'appareil DiskStation, effectuez l'une des actions suivantes :
      • Si vous voulez configurer automatiquement la connexion au routeur, choisissez Automatique, cliquez sur Suivant, pour que le système effectue un test de compatibilité :
        • Si le routeur a été testé comme étant compatible avec le service de configuration du routeur de l'appareil DiskStation, cochez la case pour envoyer le rapport de test pour aider Synology à mettre à jour la base de données de compatibilité des routeurs, puis cliquez sur Appliquer. La connexion de votre appareil DiskStation au routeur est configurée avec succès.
        • Si le routeur a été testé comme étant incompatible avec le service de configuration du routeur de l'appareil DiskStation, cochez la case pour envoyer le rapport de test pour aider Synology à mettre à jour la base de données de compatibilité des routeurs, puis cliquez sur Fermer pour quitter l'assistant de configuration.

          • Remarque :

          Si votre routeur a été testé comme étant incompatible, vous pouvez toujours configurer les règles de transmission de ports dans l'interface utilisateur du routeur. Rendez-vous ici pour plus d'informations.

      • Si vous voulez configurer manuellement la connexion au routeur, choisissez Manuel, cliquez sur Suivant, et reportez-vous à l'étape 3.
  3. Si vous devez configurer manuellement la connexion de l'appareil DiskStation au routeur, faites ce qui suit :
    1. Choisissez votre Routeur dans le menu déroulant Routeur.

      2. Remarque :

      Si vous ne parvenez pas à trouver le routeur ici, cliquez sur Actualiser la liste pour obtenir la dernière liste de Synology.

    2. Configurez le protocole et le port de votre routeur. Le protocole et le port varient selon le modèle de votre routeur. Si vous ne pouvez pas vous connecter via le Protocole par défaut (HTTP) et le Port (80), reportez-vous à la documentation livrée avec votre routeur pour connaître les paramètres corrects.
    3. Si votre routeur requiert un compte personnalisé, cochez Utiliser un compte personnalisé puis renseignez les champs Account et Mot de passe.
    4. Cliquez sur Appliquer pour terminer.

Gestion des règles de transmission de ports

Les règles de transmission de port peuvent déterminer les ports de destination de votre DiskStation qui recevra des paquets provenant de ports spécifiques de votre routeur.

Si votre DiskStation se connecte à Internet via un routeur, veuillez cliquer sur Créer l'assistant de configuration de Transmission de port.

Il y a deux manières d’ajouter des règles de transmission de port :

  • Si vous voulez créer des règles de transmission de port pour les services système (par exemple, FTP, Web Station, etc.), sélectionnez Application intégrée.
  • Si vous voulez créer des règles de transmission de ports pour d’autres services, sélectionnez Port personnalisé.

Vous pouvez utiliser la syntaxe suivante pour définir les numéros et plages de port :

  • Utilisez « - » pour indiquer la plage de port, par exemple : 6881-6890.
  • Utilisez « , » pour séparer les ports, par exemple : “21,22”, “21,55536-55663”.

Remarque :

  • Chaque port du routeur-passerelle ne peut être utilisé que pour une seule règle de transmission. L’utilisation d’un port de routeur pour deux règles ou plus provoque des conflits, et un message d’avertissement apparaît à l’écran. Vous devez définir un autre port dans la règle pour éviter les conflits.
  • Certains routeurs n'autorisent pas l'utilisation de ports personnalisés pour la transmission de services système.

Pour enregistrer des règles :

Cliquez sur Enregistreraprès avoir créé les règles. Autrement, les règles ne seront pas appliquées.

Après avoir enregistré les règles, il se peut que vous deviez attendre quelques minutes avant que les règles ne soient mises en fonction, selon les modèles de vos routeurs.

Pour vérifier si vos règles fonctionnent ou non :

Sélectionnez les règles que vous souhaitez tester et cliquez sur Test de connexion. Les résultats seront affichés dans la colonne Résultats du test de connexion.

Pour les configurations de routeur sans fil*

Lorsque votre DiskStation agit comme routeur sans fil, les règles de transmission de ports sont appliquées directement par le DiskStation.

Pour ajouter des règles de transmission de ports :

  1. Cliquez sur Créerpour ouvrir la fenêtre de configuration.

  2. Définissez le nom de Service, le Protocole (TCP ou UDP), l’Adresse IP privée, le Port privé et le Port public. Vous pouvez utiliser la syntaxe suivante pour définir les numéros et plages de port :
    • Utilisez « - » pour indiquer la plage de port, par exemple : 6881-6890.
    • Utilisez « , » pour séparer les ports, par exemple : “21,22”, “21,55536-55663”.
  3. Cliquez sur Créeren bas pour terminer le paramétrage.

Remarque :

La plupart des routeurs autorisent des ports machine et des ports routeur différents lors de la configuration des règles de transmission des ports. Si votre routeur prend en charge cette fonctionnalité :

  • Lors de l'enregistrement d'une règle, si le port d'un routeur est déjà utilisé par une règle existante, un nouveau port de routeur est affecté.
  • Lors de l'ajout ou de la modification des règles, vous pouvez indiquer directement le numéro de port de routeur.

DMZ*

Vous pouvez activer DMZ sur votre DiskStation quand la Configuration du mappage réseau est Routeur sans fil. Un hôte dans le DMZ peut se connecter directement au réseau externe, sans être restreint par les limites et la protection du pare-feu. Cela habituellement utilisé pour la configuration des serveurs.

Pour activer DMZ :

  1. Cliquez sur Paramètres DMZ pour ouvrir la fenêtre de configuration.
  2. Cochez la case Activer DMZ.
  3. Entrez l'adresse DMZ Host IP.
  4. Cliquez sur Appliquer.

Remarque :

Depuis le réseau externe, vous devrez avoir une adresse IP externe récupérée par le DiskStation pour vous connecter à un hôte dans le DMZ.

*Uniquement pris en charge sur certains modèles ou selon votre environnement.
