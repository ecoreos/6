Activer la corbeille

Lorsque vous activez la corbeille dans un dossier partagé, un sous-dossier sera créé pour stocker les fichiers supprimés. Vous pouvez récupérer les fichiers supprimés depuis la corbeille. Videz la Corbeille si vous souhaitez supprimer les fichiers définitivement.

Pour activer la corbeille :

  1. Sélectionnez un dossier partagé, puis cliquez sur Modifier.
  2. Cochez la case Activer la corbeille.
  3. Cochez Restreindre l'accès aux seuls administrateurs si vous voulez que la corbeille ne soit accessible qu'aux administrateurs.
  4. Cliquez sur OK pour enregistrer les paramètres.

Pour vider la corbeille :

Cliquez sur le bouton Vider la corbeille pour supprimer définitivement tous les fichiers supprimés dans la corbeille.

Pour vider toutes les corbeilles :

Choisissez Vider toutes les corbeilles dans le menu déroulant Action pour supprimer définitivement tous les fichiers supprimés dans toutes les corbeilles.

Remarque : Vous pouvez configurer les paramètres de privilège du dossier Corbeille dans File Station en cliquant-droit de le dossier #recycle. Sélectionnez Propriétés et cliquez sur l'onglet Permission.
