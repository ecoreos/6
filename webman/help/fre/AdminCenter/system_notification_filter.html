Paramètres Avancés

Vous pouvez sélectionner les types d'événements qui déclencheront le système pour l'envoi de messages de notification ainsi que le moyen (e-mail, SMS, mobile et programmes de messagerie instantanée) qui recevra les messages.

Chacun des événements a son propre message de notification par défaut, mais vous avez également l'option de personnaliser votre propre contenu de notification en modifiant le message de notification. Le message par défaut peut être restauré si vous décidez d'annuler vos paramètres personnalisés plus tard.

  • Toutes les notifications: Comme son nom l'indique, il comprend tous les types d'événements de notification.
  • Par priorité : Selon l'importance des événements de notification, ils sont filtrés en deux groupes : Critique et Informatif.
    • Critique: Les événements qui nécessitent de prendre immédiatement des mesures correctives appartiennent à ce groupe.
    • Informatif: Les événements qui ne nécessitent pas de mesures correctives et dont le but est de vous informer du changement d'état du système appartiennent à ce groupe.
  • Par catégorie: Selon la nature des événements de notification, ils sont filtrés en six groupes : Système, Stockage interne, Sauvegarde et restauration, Alimentation, USBCopy, CMS, et Stockage externe.
    • Système : Les événements relatifs aux erreurs ou changements du système appartiennent à ce groupe.
    • Stockage interne: Les événements relatifs au stockage interne appartiennent à ce groupe.
    • Sauvegarde et restauration: Les événements relatifs aux tâches de sauvegarde et restauration appartiennent à ce groupe.
    • Alimentation: Les événements relatifs à l'alimentation appartiennent à ce groupe.
    • USBCopy: Les événements relatifs à USBCopy appartiennent à ce groupe.
    • CMS: Les évènements relatifs à la connexion/déconnexion avec l'hôte CMS appartiennent à ce groupe.
    • Stockage externe: Les événements relatifs au stockage externe appartiennent à ce groupe.

Pour sélectionner les types et moyens des événements de notification :

  1. Sélectionnez un groupe d'événements dans le menu déroulant sous les onglets. Les événements du groupe sélectionné seront affichés.
  2. Cochez la ou les cases pour chaque événement individuel pour sélectionner le moyen de notification préféré.
  3. Cliquez sur Enregistrer pour appliquer les paramètres.

Pour modifier le message de notification :

  1. Sélectionnez un événement dont vous souhaitez modifier le message de notification dans la liste des événements, puis cliquez sur Modifier le message.
  2. Dans la fenêtre qui apparaît, saisissez la ligne de l'objet et le contenu du corps pour personnaliser le message de notification, puis cliquez sur Enregistrer pour appliquer les modifications.
  3. Si vous souhaitez restaurer le message par défaut d'une notification après que les modifications sont sauvegardées, procédez comme suit :
    1. Sélectionnez un événement dont vous souhaitez restaurer le message de notification aux valeurs par défaut dans la liste des événements, puis cliquez sur Modifier le message.
    2. Dans la fenêtre qui apparaît, cliquez sur Restaurer les paramètres par défaut puis sur Ouidans la boîte de dialogue de confirmation.

Remarque : Il est suggéré de conserver les segments contenant '%' dans la ligne de l'objet et le contenu du corps. Ils seront remplacés par les informations réelles du système quand une notification est envoyée.
