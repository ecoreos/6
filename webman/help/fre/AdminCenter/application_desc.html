Applications

Cette section vous permet de gérer les paramètres généraux relatifs aux aplications conçues par Synology telles que les privilèges d'accès et les paramètres de connexion des utilisateurs pour certaines applications, l'indexation média pour l'utilisation des applications multimédia. En plus, les outils et les options de service riches en web vous permettent d'abriter facilement un site web sur votre DiskStation.
