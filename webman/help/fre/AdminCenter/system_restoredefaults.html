Restaurer les paramètres par défaut

Vous pouvez effectuer une réinitialisation aux paramètres usine dans Panneau de configuration > Mise à jour et restauration > Restaurer les paramètres par défaut pour restaurer les paramètres originaux du fabricant sur votre DiskStation. Toutes les données stockées sur le disque dur seront supprimées et l'ensemble du système sera rétabli à sa configuration par défaut.

Remarque :

  • L'option Restaurer les paramètres par défaut est désactivée lorsque le DiskStation appartient à un cluster high-availability. Pour restaurer les paramètres par défaut d'un serveur appartenant à un cluster high-availability, vous devez d'abord retirer le cluster High Availability > Aperçu > Gérer
