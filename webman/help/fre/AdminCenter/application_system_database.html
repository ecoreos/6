Base de données du système

DiskStation utilise une base de données système pour garder des traces des données de paquets et des entrées du journaux stockés sur vos appareils de stockage externes, tels que usbshare1).

Important : Si vous désirez accéder aux données de paquets stockées sur les appareils de stockage externes attachés à votre DiskStation, ou si vous voulez voir vos enregistrements de journaux DSM dans Menu principal > Centre des journaux, la base de données système doit être activée en premier. Dans le cas contraire, tous les services correspondants seront désactivés.

Pour activer la base de données système :

  1. Cochez Activer la base de données système.
  2. Procédez de l'une des manières suivantes :
    • Sélectionnez Configuration automatique si vous voulez que DSM sélectionne automatiquement un dossier partagé pour stocker la base de données système. Si le périphérique externe de stockage qui contient le fichier partagé est supprimé, DSM sélectionnera automatiquement n'importe quel autre dossier partagé pour stocker la base de données.
    • Choisissez Déplacer la base de données actuelle vers le dossier partagé sélectionné, puis choisissez le menu déroulant pour déplacer la base de données actuelle vers un autre dossier partagé créé sur votre appareil externe.
    • Choisissez Choisir l'une des bases existantes, puis sélectionnez dans le menu déroulant une base de données existante s'il y a plus d'un appareil externe connecté sur votre DiskStation.
  3. Cliquez sur Appliquer.

Avertissement : Pour empêcher la base de données de se corrompre, vérifiez que vous avez éjecté l'appareil de stockage Menu principal > Panneau de configuration > Appareils externes > Appareils externes ou en cliquant sur l'icône des périphériques externes dans le coin supérieur droit avant de le retirer du DiskStation.
