Win/Mac/NFS

Les utilisateurs Windows, Mac et Linux peuvent accéder à votre DiskStation depuis le réseau une fois que vous avez activé les options dans Panneau de configuration > Services de fichiers > Win/Mac/NFS.

Les protocoles de partage de fichiers suivants sont pris en charge :

  • Pour Windows : SMB/CIFS (Favoris réseau), FTP, WebDAV
  • Pour Mac : SMB, FTP, AFP, WebDAV
  • Pour Linux : SMB, FTP, NFS, WebDAV
