Accueil utilisateur

Activez la fonction accueil utilisateur pour créer un dossier d' accueil personnel pour chaque utilisateur excepté guest. Tous les utilisateurs peuvent accéder à leurs propres dossiers personnalisés via CIFS, AFP, FTP ou File Station.

Admin peut accéder à tous les dossiers personnels du dossier partagé par défaut du système homes. Le nom du dossier d'accueil est le même que le compte utilisateur.

Pour activer le service d’accueil de l’utilisateur :

  1. Cliquez sur Accueil utilisateur.
  2. Cochez Activer le service d’accueil de l’utilisateur.
  3. S'il y a plus d'un volume sur Synology DiskStation, sélectionnez le volume où vous voulez stocker le dossier homes.
  4. Cliquez sur OK.

Remarque : une fois le service d'accueil de l'utilisateur local désactivé, le service d'accueil de l'utilisateur du domaine sera également désactivé parallèlement.

Pour désactiver le service d’accueil de l’utilisateur :

  1. Cliquez sur Accueil utilisateur.
  2. Décochez Activer le service d’accueil de l’utilisateur.

Après la désactivation du service d'accueil de l'utilisateur, l'accès au dossier partagé homes sera réservé à admin. Les utilisateurs pourront à nouveau accéder à leurs dossiers d'accueil quand le service d'accueil utilisateur sera réactivé.

    Remarque :
  1. Pour supprimer le dossier partagé homes, le service d'accueil utilisateur doit d'abord être désactivé.
  2. Le service d'accueil de l'utilisateur du domaine sera désactivé si le service d'accueil de l'utilisateur est désactivé.
