Sécurité

Vous pouvez modifier les paramètres de sécurité suivants dans Panneau de configuration > Sécurité > Sécurité pour les sessions de navigation DSM :

  • Délai de déconnexion (minutes) : Les utilisateurs seront automatiquement déconnectés de DSM s'ils sont inactifs pendant la période de temps spécifiée ici. Saisissez une valeur entre 1 et 65535.
  • Améliorez la compatibilité du navigateur en ignorant la vérification de l’IP : Si vous accédez au DiskStation via un proxy HTTP et que des déconnexions aléatoires surviennent, vous pouvez activer cette option pour ignorer la vérification de l'IP.
  • Améliorer la protection contre les falsifications de requêtes inter-site : Cette option améliore la protection du système contre les attaques de scripts intersite. Cette option prendra effet la prochaine fois que vous vous connecterez dans DSM.
  • Améliorer la sécurité grâce à l'en-tête HTTP Content Security Policy (CSP) : Cette option permet d'améliorer la sécurité du système vis-à-vis des attaques de scripts intersites (XSS) en autorisant uniquement les données provenant de sources de confiance et en restreignant l'exécution de scripts en ligne.
  • Ne pas permettre que DSM soit incorporé dans iFrame : Cette option permet d'empêcher d'autres sites web d'embarquer DSM dans d'autres pages web avec iFrame, empêchant ainsi certains types d'attaques émanant de sites mal intentionnés. L'on peut autoriser des sites web à embarquer DSM avec iFrame en cliquant sur le bouton Sites Web autorisés.
