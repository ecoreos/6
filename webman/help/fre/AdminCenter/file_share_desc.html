Dossier partagé

Les dossiers partagés sont les répertoires de base où vous pouvez stocker les fichiers et dossiers sur votre DiskStation. Avant de stocker des fichiers sur votre DiskStation, vous devez créer au moins un dossier partagé. Les données stockées sur des dossiers partagés peuvent être maintenues privées ou être partagées avec des utilisateurs spécifiques ou des groupes d'utilisateurs selon les autorisations d'accès personnalisées. En plus, les dossiers partagés peuvent être cryptés pendant la création, en ajoutant une couche de protection supplémentaire.

Dossiers partagés par défaut

Ci-dessous vous trouverez une liste de dossiers partagés qui sont créés automatiquement quand certaines applications, services ou paquets sont installés ou activés.

Nom Description
public Le dossier public est créé automatiquement après la première installation de votre DiskStation.*
web Le dossier web est l'emplacement de stockage par défaut pour les contenus d'un site web abrité sur le DiskStation. Il est automatiquement créé lorsque Web Station est activé.
photo Le dossier photo est utilisé pour stocker des photos et des vidéos que vous voulez partager avec Photo Station. Il est créé automatiquement lorsque Photo Station ou le DLNA/UPnP Media Server est activé.
music Le dossier music est utilisé pour stocker la musique que vous voulez écouter avec Audio Station. Il est automatiquement créé lorsque Audio Station est activé.
video Le dossier video est utilisé pour stocker des vidéos que vous voulez parcourir via les appareils DLNA/UPnP. Il est automatiquement créé lorsque le Serveur média est activé.
surveillance Le dossier surveillance est utilisé pour stocker les enregistrements de Surveillance Station. Il est automatiquement créé lorsque Surveillance Station est activé. Il est en lecture seule et seul l'administrateur système peut y accéder.
home Les dossiers home offre des espaces privés pour chaque utilisateur pour qu’il puisse stocker des données accessibles par lui uniquement. Il est automatiquement créé quand la fonction Accueil Utilisateur est activée.
homes Le dossier homes contient tous les dossiers accueil de tous les utilisateurs. Il est automatiquement créé quand la fonction Accueil Utilisateur est activée. Seul l'administrateur système peut avoir accès aux dossiers home de tous les utilisateurs et les gérer.
NetBackup Le dossier NetBackup est créé automatiquement lorsque la Sauvegarde réseau est activée.
usbshare Le dossier usbshare[number] est créé automatiquement quand vous vous connectez à un pilote USB externe vers le port USB du DiskStation.*
satashare Le dossier satashare est créé automatiquement quand vous vous connectez à un pilote eSATA externe vers le port eSATA du DiskStation.*

____
* Uniquement disponible sur certains modèles.
