TFTP

Le protocole TFTP (Trivial File Transfer Protocol) est un protocole de transfert de fichiers simple, généralement utilisé pour transférer des configurations ou des fichiers de démarrage lorsque l'authentification n'est pas requise. L'activation du service TFTP permet au DiskStation de transférer des chargeurs de démarrage aux appareils clients.

Pour configurer les paramètres TFTP :

  1. Accédez à Panneau de configuration > Services de fichiers > TFTP et sélectionnez Activer le service TFTP.
  2. Dans le champ Dossier root TFTP, spécifiez quel dossier sur le DiskStation peut être accédé par les clients TFTP.
  3. Appuyez sur Paramètres avancés pour configurer les paramètres suivants :
    • Activer le journal des transferts de fichiers TFTP : Gardez la trace des données transférées via TFTP.
    • Permission de client TFTP : Décidez des permissions du client TFTP :
      • Lecture seule : seuls les clients TFTP peuvent consulter le contenu du dossier racine.
      • Inscriptible : les clients TFTP peuvent modifier le contenu du dossier racine.
    • Clients autorisés : Choisissez quels clients sont autorisés à se connecter via TFTP :
      • Toutes les connexions : Tous les clients doivent se connecter via TFTP.
      • N'autoriser que la plage d'adresses IP suivante : Seuls les clients de la plage d'IP peuvent se connecter via TFTP.
    • Délai d'expiration : Si le serveur TFTP (c'est-à-dire votre DiskStation) ne répond pas pendant la période spécifiée, la connexion TFTP est interrompue.
  4. Cliquez sur Appliquer pour enregistrer les paramètres.
