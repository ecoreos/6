QuickConnect

QuickConnect permet aux applications clientes de se connecter à votre DiskStation via Internet sans l'inconvénient d'avoir à configurer les règles de transfert de port. QuickConnect peut aussi fonctionner avec les paquets développés par Synology, tels que Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Note Station, CMS, Cloud Station et des applications mobiles.

Pour activer QuickConnect :

  1. Accédez à Panneau de configuration > QuickConnect > Général.
  2. Vérifiez Activer QuickConnect.
  3. Saisissez les informations de votre compte Synology ou enregistrez un nouveau compte.
  4. Spécifiez un nouveau QuickConnect ID. Assurez-vous qu'il est facile à se rappeler, de sorte que vous pourrez accéder à votre DiskStation à tout moment.
  5. Cliquez sur Appliquer.

Remarque :

Un QuickConnect ID personnalisé peut inclure des lettres, chiffres et tirets (« - »), et doit commencer par une lettre.

Pour améliorer la connectivité de QuickConnect :

Votre DiskStation offre des options avancées pour rendre l'accès à QuickConnect plus adaptable aux différents environnements réseau.

  1. Accédez à Panneau de configuration > QuickConnect > Avancé.
  2. Sélectionnez les options ci-dessous les mieux adaptées à vos besoins :
    • Activer le service de relais de QuickConnect : Votre connexion QuickConnect sera relayée via Synology Relay Server lorsqu'un accès direct de QuickConnect au DiskStation n'est pas possible dans l'environnement réseau actuel.
    • Créer automatiquement des règles de transmission de ports : Si votre DiskStation de trouve sur un routeur où UPnP est activé, le routeur est informé de créer des règles de transmission de port pour QuickConnect.
  3. Cliquez sur Appliquer.

Remarque :

  • La connexion QuickConnect relayée peuvent être lente en raison d'une latence plus importante du réseau.
  • Si le routeur est autorisé à créer des règles de transmission de port pour QuickConnect, le DiskStation peut être exposé à un risque sécuritaire.

Pour activer/désactiver QuickConnect pour des applications/services spécifiques :

  1. Accédez à Panneau de configuration > QuickConnect > Avancé.
  2. Sélectionner les applications/services vous voulez pour activer QuickConnect.
  3. Cliquez sur Appliquer.

Remarque :

Pour une meilleure performance de QuickConnect, il est recommandé d'aller dans Panneau de configuration > Accès externe > Configuration du routeur pour configurer le transfert de port pour chaque service :

  • DSM : 5000 (HTTP), 5001 (HTTPS)
  • Photo Station : 80 (HTTP), 443 (HTTPS)
  • Cloud Station：6690

Dépannage

Si vous voyez le message Erreur de réseau, cela signifie que le service QuickConnect a été interrompu en raison d’erreurs de réseau. Veuillez vérifier ce qui suit :

  • Votre DiskStation est connecté à un réseau actif (dans Panneau de configuration > Réseau > Interface réseau).
  • Les paramètres de DNS server et de passerelle par défaut sont correctement configurés (dans Panneau de configuration > Réseau > Général).
  • Assurez-vous que le DiskStation peut accéder à Internet en utilisant les ports de destination suivants : 80, 443, 8888.

Quand les erreurs de réseau sont corrigées, le service QuickConnect devrait reprendre automatiquement après quelques minutes.
