Utilisateur

Grâce aux options de gestion de l'utilisateur flexibles, vous pouvez créer des comptes utilisateur pour les membres individuels de votre famille ou entreprise et définissez les permissions d'accès au dossier partagé pour chaque personne.

Pour créer un utilisateur :

  1. Allez à Panneau de configuration (Mode simple) > Utilisateur et cliquez sur l'icône + sur le panneau de gauche.
  2. Spécifiez un nom et un mot de passe pour l'utilisateur.
  3. Spécifiez les groupes pour lesquels appartiendra le nouvel utilisateur.
  4. Assignez les permissions de dossier partagé appropriées au nouvel utilisateur. Lorsque vous rencontrez des conflits de permissions, la priorité de la permission s'établit comme suit : Pas d'accès > Lecture/écriture > Lecture seule.
    • Aperçu : Cette colonne affiche les privilèges d'accès aux dossiers partagés de l'utilisateur, en fonction des paramètres actuels et des groupes auxquels l'utilisateur appartient.
    • Lecture/écriture : L'utilisateur peut accéder et effectuer des modifications sur les fichiers et sous-dossiers dans le dossier partagé correspondant.
    • Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers du dossier partagé correspondant.
    • Pas d’accès : L'utilisateur ne peut pas accéder aux fichiers ou sous-dossiers du dossier partagé correspondant.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.

    6. Restrictions concernant le nom d'utilisateur :

    • Le nom d'utilisateur n'est pas sensible à la casse et peut inclure de 1 à 64 caractères Unicode, mais ne peut pas contenir les symboles suivants :
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • Le premier caractère ne peut pas être le signe moins ou un espace, et le dernier caractère ne peut pas être un espace.
    • Les noms d'utilisateur suivants sont réservés pour le système :
      mailer-daemon, postmaster

    Restrictions concernant le mot de passe :

    • Le mot de passe est sensible à la casse et peut être composé de 127 caractères affichables au maximum, tels que des lettres, des chiffres, des symboles et des espaces.

Pour modifier un utilisateur :

  1. Accédez à Panneau de configuration (Mode simple) > Utilisateur.
  2. Sélectionnez l'utilisateur désiré sur le panneau de gauche.
  3. Modifiez l'information générale de l'utilisateur telle que le nom, le mot de passe, le groupe et les permissions du dossier partagé.
  4. Cochez Désactiver ce compte si vous voulez arrêter le fonctionnement du compte.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour supprimer un utilisateur :

  1. Accédez à Panneau de configuration (Mode simple) > Utilisateur.
  2. Sélectionnez l'utilisateur désiré sur le panneau de gauche.
  3. Cliquez sur l'icône Poubelle sur le panneau de gauche pour supprimer l'utilisateur sélectionné.
