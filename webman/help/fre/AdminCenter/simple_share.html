Dossier partagé

Les dossiers partagés sont les répertoires de base où vous pouvez stocker les fichiers et dossiers sur votre DiskStation. Avant de stocker des fichiers sur votre DiskStation, vous devez créer au moins un dossier partagé dans DSM. Les données stockées sur des dossiers partagés peuvent être maintenues privées ou être partagées avec des utilisateurs spécifiques ou des groupes d'utilisateurs selon les autorisations d'accès personnalisées.

Ci-dessous vous trouverez une liste de dossiers partagés par défaut qui sont créés automatiquement quand certaines applications, services ou paquets sont installés ou activés.

Nom Description
public Le dossier public est créé automatiquement après la première installation de votre DiskStation.*
web Le dossier web est l'emplacement de stockage par défaut pour les contenus d'un site web abrité sur le DiskStation. Il est automatiquement créé lorsque Web Station est activé.
photo Le dossier photo est utilisé pour stocker des photos et des vidéos que vous voulez partager avec Photo Station. Il est créé automatiquement lorsque Photo Station ou le Serveur multimédia DLNA/UPnP est activé.
music Le dossier Musique est utilisé pour stocker la musique que vous voulez écouter avec Audio Station. Il est créé automatiquement lorsque Audio Station ou le Serveur multimédia DLNA/UPnP est activé.
video Le dossier vidéo est utilisé pour stocker des vidéos que vous voulez parcourir via les appareils DLNA/UPnP. Il est créé automatiquement lorsque Video Station ou le Serveur multimédia DLNA/UPnP est activé.
surveillance Le dossier surveillance est utilisé pour stocker les enregistrements de Surveillance Station. Il est automatiquement créé lorsque Surveillance Station est activé. Il est en lecture seule et seul l'administrateur système peut y accéder.
home Les dossiers accueil offre des espaces privés pour chaque utilisateur pour qu’il puisse stocker des données accessibles par lui uniquement. Il est automatiquement créé quand la fonction Accueil Utilisateur est activée.
homes Le dossier homes contient tous les dossiers accueil de tous les utilisateurs. Il est automatiquement créé quand la fonction Accueil Utilisateur est activée. Seul l'administrateur système peut avoir accès aux dossiers accueil de tous les utilisateurs et les gérer.
NetBackup Le dossier NetBackup est créé automatiquement lorsque la Sauvegarde réseau est activée.
usbshare Le dossier usbshare[number] est créé automatiquement quand vous vous connectez à un pilote USB externe vers le port USB du DiskStation.*
satashare Le dossier satashare est créé automatiquement quand vous vous connectez à un pilote eSATA externe vers le port eSATA du DiskStation.*

Remarque :

  • Vous pouvez uniquement modifier les permissions de dossier partagé des utilisateurs dans le mode simple. Si vous voulez créez des groupes utilisateurs ou gérer des privilèges d'accès de groupe pour les dossiers partagés, veuillez aller à Panneau de configuration (Mode avancé) > Groupe.

Pour créer un dossier partagé :

  1. Allez à Panneau de configuration (Mode simple) > Dossier partagé et cliquez sur l'icône + sur le panneau de gauche.
  2. Spécifiez un nom et sélectionnez un volume sur lequel créer le dossier partagé.
  3. Cochez Activer la corbeille si nécessaire. Si cette option est activée, quand les fichiers dans le dossier partagé sont supprimés, ils seront déplacés vers un dossier nommé #recycle L'accès à la corbeille de recyclage peut être limité aux utilisateurs appartenant au groupe administrators.
  4. Cochez ou décochez les cases appropriées pour chaque utilisateur afin de personnaliser leurs privilèges d'accès au dossier partagé :
    • Lecture/écriture : L'utilisateur peut accéder et effectuer des modifications sur les fichiers et sous-dossiers du dossier partagé.
    • Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers du dossier partagé.
    • Pas d’accès : L'utilisateur ne peut pas accéder aux fichiers ou sous-dossiers du dossier partagé.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.

Remarque :

  • Les noms de dossiers partagés sont sensibles à la casse et peuvent être composés de 1 à 32 caractères. Les noms ne peuvent pas inclure les caractères suivants :
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Les noms suivants sont réservés pour le système :
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare.
  • Le premier caractère du nom d'un dossier partagé ne peut être ni le signe moins ni un espace tout comme le dernier caractère qui ne peut également pas être un espace.
  • Lorsque vous rencontrez des conflits de privilèges, la priorité des privilèges s'établit comme suit : Pas d'accès > Lecture/écriture > Lecture seule.
  • Lors de la création d'un nouveau dossier partagé, si les permissions pour les utilisateurs appartenant au groupe administrators sont définies sur Pas d'accès, ces utilisateurs ne pourront voir le dossier partagé que dans Panneau de configuration > Dossier partagé.
  • Vous pouvez configurer les paramètres de privilège du dossier Corbeille dans File Station en cliquant-droit de le dossier #recycle. Sélectionnez Propriétés et cliquez sur l'onglet Permissions.

Pour modifier un dossier partagé :

  1. Accédez à Panneau de configuration (Mode simple) > Dossier partagé.
  2. Sélectionnez le dossier partagé sur le panneau de gauche.
  3. Modifiez l'information générale d'un dossier partagé telle que le nom, l'emplacement et les permissions utilisateur.
  4. Si Activer la corbeille a déjà été coché, vous pouvez cliquer sur Vider la corbeille pour retirer tous les fichiers supprimés dans la corbeille de façon permanente.
  5. Cliquez sur Appliquer pour enregistrer les paramètres.

Pour supprimer un dossier partagé :

  1. Accédez à Panneau de configuration (Mode simple) > Dossier partagé.
  2. Sélectionnez le dossier partagé sur le panneau de gauche.
  3. Cliquez sur l'icône Poubelle sur le panneau de gauche pour supprimer le dossier partagé sélectionné.
