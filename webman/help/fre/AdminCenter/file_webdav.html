WebDAV

WebDAV est une extension du protocole HTTP conçue pour permettre aux utilisateurs de modifier et gérer les fichiers stockés sur des serveurs distants. Lorsque le service WebDAV est activé, les programmes clients qui prennent en charge WebDAV, comme certaines applications de Windows, Mac OS Finder, le navigateur de fichiers Linux, pourront accéder à distance au DiskStation exactement comme s'ils accédaient à un lecteur de réseau local.

Pour activer WebDAV :

  1. Cochez la case marquée Activer WebDAV ou Activer la connexion HTTPS de WebDAV si vous souhaitez activer les connexions WebDAV via HTTPS. Le port par défaut pour WebDAV est 5005, et 5006 pour WebDAV HTTPS.
  2. Cliquez sur Appliquer

Limite de vitesse

L'activation de la limite de vitesse vous permet de contrôler la bande passante maximale utilisable pour les utilisateurs ou groupes spécifiques lors des transferts des fichiers via WebDAV.

Pour activer la limite de vitesse :

  1. Cochez la case Activer la limite de vitesse, ou Activer la limite de vitesse selon un programme si vous désirez appliquer une limite de vitesse à certaines heures seulement. Ceci est un paramétrage à l'échelle du serveur qui remplace les paramétrages des utilisateurs individuels et groupes. L'option Pas de limite de vitesse désactive les paramétrages de limite de vitesse de tous les utilisateurs et groupes.
  2. Cliquez sur Paramètres de limite de vitesse pour modifier les paramètres des utilisateurs individuels ou groupes. Pour chaque utilisateur et groupe, vous pouvez sélectionner l'une des options suivantes :
    • Appliquer les paramètres de groupe : Les paramètres de limite de vitesse sont les mêmes que ceux du groupe auquel l'utilisateur appartient.
    • Aucune : Aucune limite de vitesse ne sera appliquée lorsque les membres du groupe transfèrent les fichiers avec le service.
    • Constante : Une limite de vitesse fixe, constante sera appliquée à l'utilisateur ou aux membres du groupe. Les vitesses de chargement et de téléchargement peuvent être spécifiées dans les champs sur la gauche.
    • Variable : Deux limites de vitesse différentes peuvent être spécifiées et appliquées à l'utilisateur ou aux membres du groupe en fonction d'un horaire. Cliquez sur le bouton Personnaliser pour modifier les paramètres de limite de vitesse et définir un planning.
  3. Cliquez sur Appliquer.

Remarque :

  • Pour plus de détails concernant la dynamique de limitation de la vitesse des groupes, veuillez voir ici.
  • Lorsqu'une limite de vitesse est définie pour un utilisateur, la section Résultat affiche la limite de vitesse réelle de l'utilisateur selon les paramètres individuels et de groupe. Par exemple, si un utilisateur « Achille » est affecté d'une limite de vitesse de téléchargement de 100 Ko/s, mais qu'il appartient à un groupe d'utilisateurs affecté d'une limite de téléchargement de 500 Ko/s, la section Résultats affiche alors « 0/500 ».

Paramètres avancés

Le bouton Paramètres avancés permet d'afficher et de modifier les paramètres suivants :

  • Activer le WebDAV anonyme : Permet aux utilisateurs d'accéder aux dossiers partagés en utilisant le nom d'utilisateur "anonymous" lorsqu'ils se connectent via WebDAV. Assurez-vous d'avoir attribué les privilèges d'accès corrects à l'utilisateur " Anonymous FTP/WebDAV" sur la page Dossier partagé.
  • Activer DavDepthInfinity : Lorsque vous utilisez un outil client, ceci permet aux utilisateurs d'accéder ou de supprimer des fichiers sans restrictions de longueur du chemin d'accès du fichier.
  • Activer le journal WebDAV : Ces options enregistrent les événements en rapport avec WebDAV. Cliquez sur Afficher les journaux, et vous verrez les journaux dans la fenêtre contextuelle.

Utilisation de WebDAV avec différents systèmes d'exploitation

Une fois que WebDAV est activé sur votre DiskStation, veuillez suivre les instructions ci-dessous pour commencer à accéder aux fichiers avec des machines Windows, Mac ou Linux ou certaines applis pour Smartphones de Synology.

Remarque :

  • Lors de la connexion avec les services WebDAV, nous suggérons d'utiliser un client WebDAV avec prise en charge d'UTF-8 afin d'éviter les erreurs de codage.

Pour Windows

Pour les utilisateurs de Windows qui voudraient utiliser le protocole WebDAV pour accéder aux fichiers sur des produits Synology, nous suggérons d'utiliser BitKinex, une application de mappage des lecteurs, qui est gratuite pour un usage non commercial. Vous pouvez également trouver de nombreuses autres applications clientes WebDAV disponibles sur Internet.

Remarque :

  • Si vous voulez utiliser la fonction Mapper un lecteur réseau de Windows pour accéder aux fichiers sur votre DiskStation via WebDAV, veuillez obtenir un certificat SSL valide et accédez à votre DiskStation via une connexion HTTPS. Lorsque vous utilisez le service WebDAV dans un environnement Windows, si vous souhaitez changer de compte utilisateur, il vous faut d'abord vous déconnecter du système Windows.

Pour Mac

Pour les utilisateurs Mac, vous pouvez vous connecter à votre DiskStation via WebDAV en utilisant simplement le Finder.

Pour utiliser WebDAV sur Mac :

  1. Sur votre ordinateur Mac, allez à Finder > Connexion au serveur.
  2. Dans le champ Adresse du serveur, entrez l'adresse IP de votre DiskStation ou le nom du domaine précédé de "http://" et suivi par ":5005" (ou tout autre numéro de port que vous avez spécifié en activant le service WebDAV). Par exemple : http://pm.synology.com:5005/.
  3. Entrez votre nom d'utilisateur et mot de passe de DSM.
  4. Cliquez sur Connexion.

Remarque :

  • WebDAV prend en charge la diffusion Bonjour.

Pour Linux

Pour les utilisateurs Ubuntu ou autres Linux, veuillez suivre les instructions ci-dessous pour accéder aux fichiers sur votre DiskStation via WebDAV.

Pour utiliser WebDAV sur Ubuntu :

  1. Cliquez sur Emplacements > Connexion au serveur.
  2. Sélectionnez WebDAV (HTTP) ou WebDAV (HTTPS) dans le menu déroulant.
  3. Entrez les informations requises, y compris l'emplacement du serveur, le port, le dossier, et le compte utilisateur. Cliquez sur Connecter pour continuer.
  4. Entrez votre mot de passe et cliquez sur Connecter.

Pour utiliser WebDAV via une ligne de commande Linux :

Si vous voulez utiliser une ligne de commande, vous pouvez utiliser cadaver, un logiciel client WebDAV à lignes de commandes pour configurer l'accès.

S'abonner aux calendriers à l'aide de clients CalDAV

Vous pouvez utiliser les clients CalDAV sur votre ordinateur ou sur votre appareil mobile pour vous abonner aux calendriers stockés sur DiskStation et ensuite gérer les événements, les remarques et les tâches à effectuer du calendrier. Les exemples de clients du calendrier comprennent Sunbird (Windows, Mac, Linux), Chandler (Windows, Mac, Linux), Evolution (Linux) ou iCal (Mac, appareil iOS).

Pour activer CaIDAV :

  1. Activez d'abord la connexion HTTPS de WebDAV ou WebDAV.
  2. Cochez Activer CalDAV.
  3. Cliquez sur Appliquer.

Pour gérer la liste des calendriers :

  1. Cliquez sur Afficher la liste des calendriers.
  2. Dans la fenêtre qui apparaît, procédez comme suit :
    • Si vous souhaitez créer un nouveau calendrier, cliquez sur Ajouter, nommez votre nouveau calendrier (par exemple, « cal1 ») et spécifiez la destination d'enregistrement du calendrier (par exemple, le dossier partagé « calendar ») puis cliquez sur OK.
    • Si vous souhaitez modifier le nom du calendrier et sa destination, sélectionnez le calendrier dans la liste des calendriers, cliquez sur Modifier pour apporter les modifications puis cliquez sur OK.
    • Si vous souhaitez supprimer le calendrier depuis votre DiskStation, sélectionnez-le dans la liste des calendriers puis cliquez sur Supprimer.
  3. Cliquez sur Fermer.

Remarque :

  • Les calendriers stockés sur votre DiskStation apparaîtront sous la forme d'un dossier sur votre DiskStation.

Pour vous abonner à votre calendrier à l'aide d'une application client (à l'aide de Sunbird par exemple) :

  1. Téléchargez et installez Sunbird sur votre ordinateur.
  2. Accédez à Fichier > S'abonner à Remote Calendar.
  3. Dans la fenêtre qui apparaît, sélectionnez Sur le réseauet cliquez sur Suivant.
  4. Sélectionnez CalDAV, saisissez http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name dans le champ Emplacement (par exemple, « http://demo.no-ip.info:5005/calendar/cal1 » puis cliquez sur Suivant.
  5. Affectez un nom à votre calendrier et cliquez sur Suivant.
  6. Saisissez votre nom d'utilisateur et votre mot de passe d'authentification DSM, cliquez sur OK puis cliquez sur Effectué.

Pour vous abonner à votre calendrier à l'aide de votre appareil mobile (à l'aide d'un appareil iOS par exemple) :

  1. Sur votre appareil iOS, allez dans Paramètres > Messagerie, Contacts, Calendriers.
  2. Dans Comptes, allez dans Ajouter un compte > Autre, et ensuite touchez Ajouter un compte CalDAV.
  3. Dans les champs, saisissez l'adresse IP de DiskStation (ou le nom de domaine), vos identifiants utilisateur et appuyez ensuite sur Suivant. Si un message apparaît, appuyez sur Continuer.
  4. Appuyez sur Paramètres avancés et procédez comme suit :
    1. Entrez 5005 (ou un autre numéro de port défini lors de l'activation de WebDAV) dans le champ Port.
    2. Modifiez l'URL dans le champ URL du compte par http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name (par exemple, « http://demo.no-ip.info:5005/calendar/cal1 »).
    3. Appuyez sur CalDAV.
  5. Appuyez sur Suivantpour vérifier les informations de votre compte puis appuyez sur Enregistrerpour terminer la configuration du compte.
  6. Revenez sur l'écran d'accueil de votre appareil iOS et ouvrez Calendrier pour gérer votre calendrier.

Remarque :

  • Pour plus d'informations sur la manière de gérer les événements du calendrier, les notes ou les tâches à effectuer, consultez les documents d'aide pour chaque application cliente.
