Base de données multimédia

DiskStation utilise une base de données multimédia pour garder des traces des fichiers multimédia et des entrées du journaux stockés sur la source média (dossiers partagés créés sur vos périphériques externes de stockage, tels que usbshare1).

Important : Si vous souhaitez que les services multimédia de DSM (tels que Serveur multimédia, iTunes, Audio Station et Download Station) permettent d'accéder aux fichiers multimédia stockés sur des périphériques de stockage externes connectés au port USB de votre DiskStation, ou si vous souhaitez activer les services du journal de DSM et afficher les enregistrements du journal dans Menu Principal > Informations système > Journal, la base de données multimédia doit être activée en premier. Dans le cas contraire, tous les services correspondants seront désactivés.

Pour activer la base de données multimédia :

  1. Cochez Activer la base de données multimédia.
  2. Employez l'une des méthodes suivantes :
    • Choisissez Configuration automatique si vous souhaitez que DSM sélectionne automatiquement un dossier partagé pour stocker la base de données. Si le périphérique externe de stockage qui contient le fichier partagé est supprimé, DSM sélectionnera automatiquement n'importe quel autre dossier partagé pour stocker la base de données.
    • Choisissez Sélectionnez un dossier partagé pour stocker votre base de données, puis sélectionnez manuellement dans le menu déroulant un dossier partagé pour stocker la base de données.
  3. Cliquez sur OK.

Avertissement : Pour éviter que la base de données ne soit corrompue, assurez-vous d'avoir déconnecté son périphérique de stockage dans Menu principal > Panneau de configuration > Périphériques externes avant de le supprimer de DiskStation.

Pour inclure ou exclure des dossiers partagés de la source multimédia :

  1. Cliquez sur Source multimédia.
  2. Cochez ou décochez les cases à côté du/des dossier(s) partagé(s) pour faire votre sélection.
  3. Cliquez sur OK.

Les fichiers multimédias et les enregistrements du journal dans les dossiers partagés sélectionnés seront indexés et les informations indexées seront ajoutées à la base de données multimédia.

Pour actualiser la base de données multimédia :

  1. Cliquez sur Actualiser la base de données. Toutes les données de la base de données multimédia seront remplacées par les informations réindexées sur la source multimédia actuelle.
  2. Cliquez sur OK.

Remarque : Si une grande portion des données présentes dans la base de données multimédia n'est plus utilisée (par exemple, vous déconnectez un disque USB qui ne sera plus jamais utilisé avec le DiskStation, ce qui provoquera une redondance des informations indexées dans la base de données), l'actualisation de la base de données multimédia peut vous permettre d'effacer toutes les données non utilisées ou de réduire la taille de la base de données. Toutefois, procéder ainsi implique des opérations de lecture et d'écriture répétées sur le disque, ce qui pourrait occuper des ressources système et réduire la durée de vie du disque. Utilisez cette fonction avec parcimonie.
