Modification d'un utilisateur

Recherchez le compte utilisateur (en entrant le nom de l'utilisateur dans le champ Rechercher), double-cliquez dessus ou cliquez sur Modifier, puis éditez les données du compte.

Remarque :
  1. Si vous désactivez un compte, il cesse de fonctionner.
  2. Quota doit être un entier supérieur ou égal à 0 et inférieur à la capacité du disque/volume. Une valeur de quota de 0 signifie que Quota effectif est déterminé par Limite du groupe.
  3. S'il existe plus d'un volume/shared folder sur le serveur, vous pouvez définir un quota pour l'utilisateur sur chaque volume/dossier partagé.
  4. Les informations de messagerie électronique sur la page « Notification » doivent être saisies correctement afin que le système puisse avertir l'utilisateur par e-mail une fois le compte créé ou modifié.
  5. Lors de la définition de la limite de vitesse pour un utilisateur, vous verrez les nombres sous Résultat si une limite de vitesse de groupe est déjà appliquée à l'utilisateur.
  6. La limite de vitesse affectée à un utilisateur n'est pas incluse dans la limite de vitesse du groupe auquel l'utilisateur appartient.
