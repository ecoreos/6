USBCopy

Vous pouvez sélectionner un dossier de destination pour la fonction USBCopy en accédant à Panneau de configuration > Périphériques externes > USBCopy/SDCopy.

Pour activer USBCopy :

  1. Cochez Activer USBCopy.
  2. Sélectionnez un dossier partagé de destination à partir du menu déroulant.

Lorsque vous raccordez un périphérique USB (comme un appareil photo numérique) au port USB sur le panneau avant du produit Synology, vous pouvez appuyer sur le bouton Copie sur le panneau avant pour copier les données sur le périphérique USB dans le dossier de destination partagé, défini sur celui-ci. Si le dossier de destination USBCopy a été supprimé, le système le remplacera automatiquement par le dossier partagé public.

Limites

  1. Seuls les périphériques USB au format FAT ou NTFS, les formats ext3 et ext4 sont pris en charge.
  2. Les périphériques USB doivent être branchés dans le port USB sur la façade du DiskStation.
  3. Seul l'admin ou les utilisateurs dans le groupe administrateurs disposent des autorisations d'accès appropriées pour réaliser les tâches de sauvegarde vers les périphériques USB.
  4. L'espace disponible du dossier partagé de destination doit être supérieur à l'espace utilisé sur le périphérique USB.

SDCopy*

Vous pouvez sélectionner un dossier de destination pour la fonction SDCopy en accédant à Panneau de configuration > Périphériques externes > USBCopy/SDCopy.

Pour activer SDCopy :

  1. Cochez Activer SDCopy.
  2. Sélectionnez un dossier partagé de destination à partir du menu déroulant.

Lorsque vous raccordez un périphérique SD dans la fente SD du panneau avant du produit Synology, vous pouvez appuyer sur le bouton Copie de celui-ci pour copier les données du périphérique SD vers le dossier de destination partagé, défini sur celui-ci. Si le dossier de destination SDCopy a été supprimé, le système le remplacera automatiquement par le dossier partagé public.

Limites

  1. Seuls les périphériques SD au format FAT ou NTFS, les formats ext3 et ext4 sont pris en charge.
  2. Les périphériques SD doivent être branchés dans la fente SD de la façade du produit Synology.
  3. Seul l'admin ou les utilisateurs dans le groupe administrateurs disposent des autorisations d'accès appropriées pour réaliser les tâches de sauvegarde vers les périphériques SD.
  4. L'espace disponible du dossier partagé de destination doit être supérieur à l'espace utilisé sur le périphérique SD.

____
* Uniquement disponible sur certains modèles.
