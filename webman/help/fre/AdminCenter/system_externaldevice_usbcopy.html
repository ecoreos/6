USBCopy

Vous pouvez sélectionner et définir un emplacement cible pour la fonction USBCopy.

Pour activer USBCopy :

  1. Cochez Activer USBCopy.
  2. Sélectionnez un dossier partagé de destination à partir du menu déroulant.
  3. Cliquez sur OK.

Lorsque vous raccordez un périphérique USB (comme un appareil photo numérique) au port USB sur le panneau avant de Synology DiskStation, vous pouvez appuyer sur le bouton Copie sur le panneau avant pour copier les données sur le périphérique USB dans le dossier de destination partagé, défini sur celui-ci. Si le dossier de destination USBCopy a été supprimé, le système le remplacera automatiquement par le dossier partagé public.

    Limites :
  1. Seuls les périphériques USB au format FAT ou NTFS sont pris en charge.
  2. Les périphériques USB doivent être branchés dans le port USB sur la façade du Synology DiskStation.
  3. L'espace disponible du dossier partagé de destination doit être supérieur à l'espace utilisé sur le périphérique USB.
