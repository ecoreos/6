Notification

Vous pouvez configurer le DiskStation pour qu'il vous envoie des messages de notification via e-mail, appareils mobiles, Windows Live Messenger ou Skype, lorsque l'état change ou que des erreurs se produisent dans le système, par exemple : perte connexion, changement de comportement du ventilateur, coupure de courant, volume de stockage plein, ou échec de sauvegarde et restauration, etc.
