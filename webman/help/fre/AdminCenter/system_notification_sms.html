SMS

Vous pouvez activer les notifications par SMS pour que votre DiskStation vous envoie des messages de notification via SMS quand des changements d’état ou des erreurs se produisent dans le système, comme une perte de connexion, un changement de comportement du ventilateur, une panne d’électricité, un manque d’espace libre sur un volume de stockage ou un échec de sauvegarde et restauration, etc.

Vous pouvez activer les notifications par SMS pour que votre DiskStation vous envoie des messages de notification via SMS quand des changements d’état ou des erreurs se produisent dans le système, comme une perte de connexion, une panne d’électricité, un manque d’espace libre sur un volume de stockage ou un échec de sauvegarde et restauration, etc.

Pour activer les notifications par SMS :

Cochez Activer les notifications par SMS et configurez les paramètres suivants :

  1. Sélectionnez un fournisseur SMS dans la liste déroulante Fournisseur de service SMS. Le fournisseur par défaut est « Clickatell ».
  2. Cliquez sur Ajouter un fournisseur de service SMS pour créer une nouvelle configuration SMS ou cliquez sur Supprimer pour supprimer la configuration de fournisseur SMS sélectionnée.
  3. Dans le champ Nom d'utilisateur, saisissez le nom d’utilisateur de votre compte SMS.
  4. Dans les champs Mot de passe et Confirmer le mot de passe, entrez et confirmez le mot de passe de votre compte SMS.
  5. Entrez votre Numéro de téléphone principal ou Numéro de téléphone secondaire. Si les deux numéros sont entrés, les messages SMS de notification seront envoyés aux deux numéros.
  6. Si vous ne voulez pas recevoir trop de messages, cochez la case Activer la limite d'intervalle SMS et limitez le nombre de messages de notification à un message par intervalle spécifié.
  7. Cliquez sur Envoyer un message SMS de test pour envoyer un message SMS de test au numéro de téléphone spécifié. Si vous ne recevez pas de message, la configuration est incorrecte.
  8. Cliquez sur Appliquerpour enregistrer la configuration ou sur Réinitialiserpour annuler le démarrage.

Remarque : La langue de notification par SMS est définie dans Langue de notification. Pour changer votre langue de notification par SMS, allez dans Menu principal > Panneau de configuration > Options régionales, cliquez sur l'onglet Langue, et choisissez votre langue préférée dans le menu déroulant de la section Langue de notification.

    Limites :
  1. Nom d’utilisateur : doit respecter la casse et être d'une longueur allant de 1 à 32 caractères Unicode.
  2. Mot de passe : doit respecter la casse et être d'une longueur allant de 0 à 128 caractères affichables, comme lettres, nombres, symboles et espaces.
  3. La limite d’intervalle SMS doit être comprise entre 1 et 10 000 minutes.

Pour créer une configuration de fournisseur SMS :

  1. Cliquez sur Ajouter un fournisseur de service SMS.
  2. Dans le champ Nom du fournisseur, saisissez le nom du fournisseur de service SMS dans le champ.
  3. Dans le champ URL SMS, entrez une URL SMS dont le message est « Hello world. » Vous pouvez obtenir cette URL SMS auprès de votre fournisseur de service SMS.
  4. Cliquez sur Suivant.
  5. Le système décomposera l’URL du SMS en différents paramètres. Définissez chaque paramètre en sélectionnant la catégorie correspondante. A l'exception d’Autre, les catégories de paramètre ne peuvent être utilisées qu'une fois.
    Limites :
  1. Nom du fournisseur : doit respecter la casse et être d'une longueur allant de 1 à 32 caractères Unicode.
  2. URL SMS : doit respecter la casse et être d'une longueur allant de 1 à 500 caractères Unicode. Le nombre de paramètres doit être entre 4 et 10. Cette URL SMS est uniquement destinée à la configuration. Aucun SMS ne sera envoyé une fois la configuration terminée. Le format de cette URL est fourni par le fournisseur de service SMS, et l'URL doit inclure les paramètres suivants : nom d'utilisateur, mot de passe, téléphone de destination et contenu du message. Avec Clickatell pour exemple, l'URL possible est : https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world. Notez que dans le « contenu du message », l’espace entre « Hello » et « world » sera remplacé par un caractère particulier défini par le fournisseur de service.
