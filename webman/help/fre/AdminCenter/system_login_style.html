Style de la connexion

Vous pouvez personnaliser l'apparence de l'écran de connexion de DSM en choisissant parmi tout un éventail de modèles, en changeant l'image d'arrière-plan ou en ajoutant un logo. Pour ce faire, allez dans Panneau de configuration > Style de connexion.

Pour personnaliser l'écran de connexion DSM :

Les options suivantes apparaissent sur la page Style de connexion :

  • Titre de la page de connexion : Le texte saisi ici apparaît sur l'écran de connexion.
  • Information sur la durée : Choisissez d'afficher ou de masquer les informations sur l'heure locale sur l'écran de connexion.
  • Modèle : Sélectionnez un thème de couleur et une image d'arrière-plan par défaut. Le modèle choisi prend effet la prochaine fois que vous vous connectez dans le DiskStation.
  • Arrière-plan : Pour modifier l'image d'arrière-plan qui apparaît sur l'écran de connexion, cliquez sur l'écran de connexion, cliquez sur l'image existante ou le signe plus (+) si aucune image est sélectionnée. Vous pouvez choisir parmi les papier peints par défaut, les images stockées sur le DiskStation ou les images précédemment utilisées. Seuls les formats de fichier image JPE, JPEG, JPG, BMP, GIF ou PNG sont pris en charge.
  • Logo : Pour ajouter un logo sur l'écran de connexion, cliquez sur le logo existant ou le signe plus (+) si aucune image est sélectionnée. Vous pouvez charger une image inférieure à 1 Mo, choisissez parmi les images stockées sur le DiskStation, ou choisissez une image précédemment utilisée.
