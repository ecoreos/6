Modifier l'information générale de dossier partagé

Vous pouvez modifier l'information générale d'un dossier partagé telle que le nom, la description et autres.

Pour modifier l'information générale d'un dossier partagé :

  1. Allez dans Panneau de configuration > Dossier partagé.
  2. Sélectionner un dossier partagé.
  3. Cliquez sur Modifier.
