Paramètres de sécurité

Changer la racine des utilisateurs sélectionnés pour l’accueil de l’utilisateur

Les utilisateurs sélectionnés ici ne peuvent accéder à leurs dossiers d'accueil que lorsqu'ils sont connectés Synology DiskStation via le FTP.

Activer le FTP anonyme

Activez cette fonction pour autoriser tout le monde à se connecter au service FTP du DiskStation (avec le compte anonyme ou ftp).

Pour configurer les privilèges de l'utilisateur anonyme :

Allez dans Menu principal > Panneau de configuration > Dossier partagé, puis cliquez sur Configuration des privilèges.

Changer la racine anonyme

Activez cette fonction pour changer le répertoire de connexion de l'utilisateur FTP anonyme. Sélectionnez le dossier partagé à partir du menu déroulant.
