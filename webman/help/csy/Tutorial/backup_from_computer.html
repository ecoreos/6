Zálohování dat na počítač PC nebo Mac pomocí služby Cloud Station

Přehled

Balíček Cloud Station Server není jen sofistikovaný meziplatformový synchronizační software: uživatelé ho mohou používat i jako chytřejší variantu pro zálohování svých osobních počítačů. Jako řešení zálohování v reálném čase funguje služba Cloud Station Backup okamžitě a šetří šířku pásma. Dále nabízí lepší ochranu díky až 32 historickým verzím jednoho souboru.

Nastavení balíčku Cloud Station Server na zařízení Synology NAS

  1. Přejděte k možnosti Centrum balíčků, vyhledejte položku Cloud Station Server a potom klikněte na možnost Instalovat.
  2. Spusťte službu Cloud Station Server.
  3. Přejděte k možnosti Přehled a v části Cloud Station Backup klikněte na položku Stáhnout. (Balíček Cloud Station Backup si můžete stáhnout také z webu Centrum pro stahování společnosti Synology.)
  4. Spusťte na počítači instalační program a postupujte podle pokynů pro instalaci balíčku Cloud Station Backup.
  5. Po instalaci na počítači balíček Cloud Station Backup spusťte. Klikněte na možnost Začít.
  6. Zadejte adresu zařízení Synology NAS se spuštěnou službou Cloud Station Server (nebo QuickConnect ID), uživatelské jméno a heslo. (Můžete také kliknout na ikonu vyhledávání vpravo a služba Cloud Station Backup automaticky v rámci vaší sítě LAN vyhledá ostatní zařízení Synology NAS.) Klikněte na možnost Další.
  7. Vyberte zdroj zálohování a zrušte zaškrtnutí všech podsložek, které synchronizovat nechcete.
  8. Kliknutím na možnost Vybrat vyberte složku na zařízení Synology NAS. Služba Cloud Station Backup vytvoří na vybraném cíli zálohování zálohovací složku s názvem vašeho počítače.
  9. Na možnost Pravidla zálohování klikněte, pokud si přejete nastavit filtry či pravidla synchronizace.
  10. Kliknutím na možnost Další ověřte nastavení zálohování a poté klikněte na možnost Hotovo.
  11. Postup zálohování se zobrazuje v hlavní aplikaci.

Stažení nebo obnovení předchozí verze synchronizovaného souboru ze služby Cloud Station Server

  1. Přejděte k možnosti Cloud Station Server > Historie verzí a najděte soubor či složku, které chcete obnovit.
  2. Vyberte soubor, který chcete načíst, a klikněte na možnost Akce. Klikněte na možnost Procházet předchozí verze. Jestliže byl soubor z počítače omylem odstraněn nebo odebrán, můžete ho obnovit kliknutím na možnost Obnovit.
  3. Vyberte verzi souboru, který chcete stáhnout, a klikněte na možnost Stáhnout. Nebo pokud si jste jistí, že chcete vybranou verzi obnovit, klikněte na možnost Obnovit – aktuální verze se takto přepíše.
