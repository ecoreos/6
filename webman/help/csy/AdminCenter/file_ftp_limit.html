Omezení připojení

Povolit pouze připojení SSL/TLS

Pokud je tato funkce aktivní, server bude od klientů FTP přijímat pouze požadavky přes připojení SSL nebo TLS.

Omezit připojení na IP adresu

Je-li aktivní tato funkce, server bude omezovat maximální počet připojení na IP adresu. Povolením této funkce a omezení šířky pásma pro FTP můžete efektivně spravovat šířku pásma.

Povolit omezení šířky pásma pro FTP

Pomocí této funkce můžete omezit šířku pásma pro jedno připojení k FTP.
