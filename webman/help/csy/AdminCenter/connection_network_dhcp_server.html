Server DHCP

Zařízení DiskStation může sloužit jako server DHCP a přidělovat klientům DHCP v místní síti dynamické IP adresy.

Povolení serveru DHCP:

  1. Přejděte do části Ovládací panel > Server DHCP > Síťové rozhraní.
  2. Vyberte síťové rozhraní (např. LAN 1), kterému se mají IP adresy přidělovat, a klikněte na možnost Upravit.
  3. V místním okně klikněte na kartu Server DHCP a vyberte možnost Povolit server DHCP.
  4. Určete následující nastavení DHCP:
    • Doba pronájmu adresy (v minutách)
    • Primární DNS
    • Sekundární DNS (volitelně)
    • Název domény (volitelně)
    • Povolit Web Proxy Automatic Discovery (volitelně)
  5. Nastavte alespoň jednu podsíť. Podsíť nastavíte kliknutím na možnost Přidat a stanovením následujících údajů:
    • Počáteční IP adresa
    • Koncová IP adresa
    • Maska sítě
    • Brána
  6. Po kliknutí na možnost Vytvořit se zobrazí vytvořená podsíť v seznamu.
  7. Po vytvoření dostatečného množství podsítí nezapomeňte alespoň jednu podsíť povolit.
  8. Klikněte na možnost OK.

Poznámka:

  • Podsítí je možné vytvořit, kolik si přejete.
  • Pokud se používá v koordinaci s přenosovým serverem DHCP, může zařízení DiskStation jako server DHCP přidělovat adresy zařízením v jiné podsíti.

Klienti DHCP

Zde se nachází seznam klientů DHCP a jejich síťové konfigurace (např. adresa MAC, IP adresa, název hostitele, doba před vypršením platnosti pronájmu DHCP a stav rezervace).

  1. Přejděte do části Ovládací panel > Server DHCP > Síťové rozhraní a vyberte jedno požadované rozhraní.
  2. Kliknutím na možnost Upravit > Klienti DHCP je možné vyhledat klienty a jejich údaje.

Správa IP adres pro klienty DHCP:

Během obnovení pronájmu DHCP a správy těchto klientů může být klientovi přidělována stále stejná IP adresa.

  • Přidání klientů DHCP: Kliknutím na možnost Přidat přidělíte klientskému zařízení IP adresu.
  • Rezervace IP adresy: Kliknutím na ikonu zámku zarezervujete IP adresu pro aktuální zařízení.
  • Zrušení rezervace: Kliknutím na ikonu zámku umožníte po uplynutí doby pronájmu využití aktuální IP adresy jiným zařízením.

Poznámka:

Jestliže má zařízení adresy IPv4 a rovněž IPv6, je možné pro zařízení rezervovat jednu adresu IPv4 a jednu adresu IPv6.

Hledání klientů DHCP:

  1. Klikněte na ikonu trychtýře v pravém horním rohu.
  2. Vyberte jedno z kritérií hledání níže:
    • Vše: Uvádí všechny klienty DHCP a rezervace DHCP.
    • Klienti DHCP: Uvádí všechny klienty DHCP s rezervovanými či nerezervovanými IP adresami.
    • Rezervace DHCP: Uvádí všechny rezervované IP adresy. Rezervovanou IP adresu není možné zatím spárovat s žádným zařízením.

Možnosti DHCP

Po stanovení možností DHCP může vaše zařízení DiskStation jako server DHCP odesílat související informace klientům DHCP v určité podsíti.

Zadání možností DHCP pro podsíť:

  1. Přejděte do části Ovládací panel > Server DHCP > Síťové rozhraní.
  2. Vyberte požadované síťové rozhraní a klikněte na možnost Upravit.
  3. Přejděte do části Server DHCP a přejděte do seznamu podsítí.
  4. Proveďte jeden z následujících úkonů:
    • Zadání možností DHCP pro stávající podsíť: Vyberte požadované rozhraní a klikněte na možnost Upravit.
    • Vytvoření podsítě a zadání jejích možností DHCP: Klikněte na možnost Přidat a zadejte nastavení podsítě.
  5. Vyberte požadované možnosti DHCP. Chcete-li zadat odpovídající hodnoty, klikněte na prázdná pole a upravte je.
  6. Klikněte na možnost OK.

Dodavatel

Klienti DHCP různých dodavatelů mohou vyžadovat další údaje DHCP. Nastavením možností DHCP pro dodavatele může zařízení DiskStation poskytovat tyto údaje klientům.

Informace o možnostech DHCP podporovaných jednotlivými dodavateli získáte od těchto dodavatelů.

Přidání dodavatelů:

  1. Přejděte do části Ovládací panel > Server DHCP > Dodavatel.
  2. Klikněte na možnost Vytvořit a zadejte dodavatele a jeho možnosti DHCP:
    • Název: Zadejte název dodavatele.
    • Kód: Zadejte číselný kód možnosti DHCP.
    • Hodnota: Zadejte hodnotu možnosti.
    • +: Přidejte možnost DHCP.
    • -: Odstraňte možnost DHCP.
  3. Nastavení dokončete kliknutím na možnost Vytvořit.

Úprava možností DHCP pro dodavatele:

  1. Přejděte do části Ovládací panel > Server DHCP > Dodavatel.
  2. Vyberte požadovaného dodavatele a po kliknutí na možnost Upravit upravte možnosti DHCP.
  3. Klikněte na možnost OK.

PXE

Zařízení DiskStation může sloužit jako server DHCP při implementaci nastavení PXE (Pre-Boot Execution Environment). PXE je síťové prostředí, které umožňuje klientským zařízením spouštění pomocí síťového rozhraní místo zařízení na ukládání dat (např. pevných disků).

Při implementaci prostředí PXE je možné mít na zařízení DiskStation nebo na vzdáleném serveru TFTP uložený zaváděcí program a klientská zařízení se mohou pomocí tohoto zaváděcího programu spouštět přes síť.

Použití zařízení DiskStation jako serveru spouštěcího zavaděče:

  1. Přesvědčte se, že jste vybrali možnost Povolit službu TFTP (v části Ovládací panel > Souborové služby > TFTP).
  2. Jako kořenovou složku TFTP vyberte složku, ve které se nachází soubor zaváděcího programu.
  3. Ověřte, zda je povolená alespoň jedna podsíť DHCP (v části Ovládací panel > Server DHCP > Síťové rozhraní).
  4. Přejděte do části Ovládací panel > Server DHCP > PXE.
  5. Vyberte možnost Povolit PXE (Pre-boot Execution Environment).
  6. Vyberte možnost Místní server TFTP a vyberte soubor zaváděcího programu.
  7. Nastavení uložte kliknutím na možnost Použít.

Použití vzdáleného serveru TFTP jako serveru spouštěcího zavaděče:

Pomocí vzdáleného serveru TFTP jako serveru spouštěcího zavaděče umožníte spouštění na klientech bez nutnosti měnit současnou síťovou infrastrukturu.

  1. Ověřte, zda je povolená alespoň jedna podsíť DHCP (v části Ovládací panel > Server DHCP > Síťové rozhraní).
  2. Přejděte do části Ovládací panel > Server DHCP > PXE.
  3. Vyberte možnost Povolit PXE (Pre-boot Execution Environment).
  4. Vyberte možnost Vzdálený server TFTP a zadejte IP adresu serveru a cestu souboru spouštěcího zavaděče.
  5. Nastavení uložte kliknutím na možnost Použít.
