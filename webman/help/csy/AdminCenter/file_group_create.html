Vytvoření skupiny

  1. Přejděte do části Ovládací panel > Skupina.
  2. Klikněte na možnost Vytvořit. Tato akce spustí Průvodce vytvořením uživatele.
  3. Zadejte název a popis skupiny. (Více informací najdete v části níže). Klikněte na možnost Další.
  4. Vytváření skupiny dokončíte podle zbývajících pokynů průvodce.
  5. Postup dokončíte kliknutím na možnost Použít.

Pravidla zadávání názvů:

  • V názvu skupiny se rozlišují velká a malá písmena. Název se musí skládat z 1 až 32 znaků Unicode a nesmí obsahovat následující symboly:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Prvním znakem nesmí být znaménko mínus ani mezera a posledním znakem nesmí být mezera.
  • V popisu skupiny se rozlišují velká a malá písmena a může obsahovat max. 64 zobrazitelných znaků Unicode.
