E-mail

Povolením upozornění e-mailem umožníte zařízení DiskStation posílat zprávy na váš e-mail, když nastanou specifické události, například chyby systému, ztráta připojení k síti, chování ventilátoru, výpadek napájení nebo selhání úlohy zálohování.

Povolením upozornění e-mailem umožníte zařízení DiskStation posílat zprávy na váš e-mail, když nastanou specifické události, například chyby systému, ztráta připojení k síti, výpadek napájení nebo selhání úlohy zálohování.

Povolení e-mailových upozornění:

  1. Přejděte do části Ovládací panel > Upozornění > E-mail.
  2. Zaškrtněte políčko Povolit e-mailová upozornění.
  3. Do pole Příjemci zadejte jednu či více e-mailových adres. Na zde zadané adresy budou posílána upozornění systému. Jsou povoleny dvě adresy. Více adres můžete oddělit středníkem (;).
  4. V případě potřeby můžete v poli Předpona předmětu zadat předponu předmětu. Tato předpona bude přidána do předmětu každé zprávy odeslané ze stanice DiskStation a umožní vám tak rozpoznat a vyfiltrovat zprávy.
  5. Chcete-li k přenášení zpráv používat e-mailový server třetí strany (např. Gmail, Outlook, Yahoo!) podívejte se níže do části „e-mailová služba třetí strany“. Chcete-li používat vlastní server, přejděte do části „vlastní server SMTP“.

Používání e-mailové služby třetí strany:

  1. V nabídce Poskytovatel služeb vyberte e-mailovou službu třetí strany, jako je například Gmail, Yahoo nebo Outlook.
  2. Zadejte své uživatelské jméno a heslo pro vybranou e-mailovou službu.
  3. Nastavení uložíte kliknutím na možnost Použít.
  4. Kliknutím na možnost Poslat zkušební e-mail můžete zkontrolovat správnost svých nastavení.

Používání vlastního serveru SMTP:

  1. V nabídce Poskytovatel služeb vyberte možnost Vlastní server SMTP.
  2. Do pole Server SMTP zadejte název nebo IP adresu serveru SMTP.
  3. Do pole Port SMTP zadejte číslo portu SMTP. (Pokud jste při nastavení pravidel předávání portů nastavili jiný port SMTP, použijte toto číslo.)
  4. Pokud váš server SMTP vyžaduje ověření, zaškrtněte možnost Je požadováno ověření a zadejte Uživatelské jméno a Heslo pro server SMTP.
  5. Pokud server SMTP vyžaduje připojení SSL/TLS, zaškrtněte možnost Je vyžadováno zabezpečené připojení (SSL/TLS).
  6. Chcete-li přizpůsobit jméno odesílatele uvedené v e-mailových zprávách odeslaných zařízením DiskStation, zadejte jméno do pole Jméno odesílatele. Chcete-li přizpůsobit e-mailovou adresu odesílatele, zadejte e-mailovou adresu do pole E-mail odesílatele. Pokud zůstane toto pole prázdné, použije se v poli Příjemci první e-mailová adresa.
  7. Nastavení uložíte kliknutím na možnost Použít.
  8. Kliknutím na možnost Poslat zkušební e-mail můžete zkontrolovat správnost svých nastavení.

Poznámka:

  • Chcete-li posílat uvítací zprávu každému nově vytvořenému uživatelskému účtu, zaškrtněte možnost Odeslat uvítací zprávu novým uživatelům a každý uživatel při vytvoření účtu obdrží uvítací zprávu.
  • Jazyk pro upozornění e-mailem lze nastavit v části Jazyk oznámení. Pokud chcete změnit jazyk upozornění e-mailem, přejděte do části Hlavní nabídka > Ovládací panel > Místní nastavení, klikněte na kartu Jazyk a vyberte preferovaný jazyk z rozevírací nabídky v části Jazyk oznámení.
  • Používáte-li server SMTP systému Gmail a povolili jste dvoufázové ověření, nezapomeňte použít „heslo specifické pro aplikaci“ poskytnuté společností Google pro ověření SMTP.

Název domény serveru SMTP:

Server SMTP lze zadat jako IP adresu nebo název domény. Pokud použijete název domény, zadejte na stránce Hlavní nabídka > Ovládací panel > Síť platnou IP adresu serveru DNS a zkontrolujte, zda je server připojen k Internetu.
