Uživatel

Díky flexibilním možnostem správy uživatelů můžete vytvářet uživatelské účty pro jednotlivé členy rodiny nebo pro vaši společnost. Vytváření uživatelů umožňuje definování a správu oprávnění pro jednotlivé osoby, jako jsou například přístupová oprávnění sdílených složek, kvóty kapacity úložiště/sdílené složkynebo omezení šířky pásma.
