Automatický blok

Automatický blok zabraňuje neoprávněnému přihlášení. Je-li tato služba povolena, IP adresy s příliš velkým množstvím neúspěšných pokusů o přihlášení budou blokovány. Lze také zařadit neúspěšné pokusy o přihlášení přes protokoly SSH, Telnet, rsync, Síťova záloha, Synchronizace sdílených složek, FTP, WebDAV, mobilní aplikace Synology, File Station nebo DSM. Můžete si také vytvořit Seznam povolených položek, kde budou zařazeny IP adresy, které nebudou nikdy blokovány.

Nastavení automatického bloku:

  1. Přejděte na kartu Obecné a zaškrtněte možnost Povolit automatický blok.
  2. Zadejte hodnoty pro položky Pokusy o přihlášení a Během (minuty). IP adresa bude blokována po nastaveném počtu neúspěšných pokusů o přihlášení během určitého počtu minut.
  3. Zaškrtněte možnost Povolit vypršení platnosti blokování a zadejte počet dní, po jejichž uplynutí bude blokování IP adresy zrušeno. Pokud není možnost Odblokovat po (dny) vybrána nebo je nastavena na hodnotu 0, zůstanou blokované IP adresy na tomto seznamu do doby, než je ručně neodstraníte
  4. Klepněte na možnost Použít.

Přidání blokované adresy:

Klepněte na kartu Seznam blokování a z rozevírací nabídky Vytvořit vyberte některou z následujících možností:

  • Přidat IP adresu

    • Zadejte IP adresu a nastavte dobu platnosti.

  • Importovat seznam IP adres

    • Importujte textový soubor se seznamem IP adres, který umožňuje zadat současně větší počet IP adres a dobu platnosti. Chcete-li přepsat stávající IP adresy, zaškrtněte příslušné políčko. Pokud není tato možnost vybrána, budou duplicitní IP adresy vynechány.

    Chcete-li importovat soubor, musí splňovat následující kritéria:

    • Soubor musí mít formát prostého textu.
    • Každý řádek souboru může obsahovat pouze jednu IP adresu.
    • Řádek komentáře začíná symbolem # (znak libry).

      • Pokud je vybraný soubor ve správném formátu, budou všechny IP adresy zobrazeny v oblasti náhledu. Import potvrďte klepnutím na možnost OK.

Odebrání blokovaných IP adres:

  1. Klepnutím na kartu Seznam blokování zobrazíte seznam zablokovaných IP adres.
  2. Vyberte IP adresy, které chcete odebrat ze seznamu, a klepněte na možnost Odebrat.
