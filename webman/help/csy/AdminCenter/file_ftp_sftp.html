SFTP

SFTP je protokol pro přenos souborů, který představuje rozšíření protokolu Secure Shell (SSH). Protokol SFTP vyžaduje pouze jediné číslo portu TCP. Kromě toho lze pomocí privátních a veřejných klíčů ověřit uživatele bez hesla. Rychlost přenosu souborů je však nižší a využití systémových prostředků je z důvodu šifrování intenzivnější.

Postup povolení služby SFTP:

  1. Klikněte na možnost Povolit službu SFTP.
  2. Klikněte na možnost Použít.

Číslo portu

Nastavte číslo portu služby SFTP.

Poznámka:

Výchozí číslo portu pro službu SFTP je 22. Můžete jej nastavit v rozmezí od 1 do 65535, ale nesmíte použít následující čísla:
20, 21, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002 a výchozí porty pro eMule: 4662 (TCP), 4672 (UDP)
