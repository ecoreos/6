Adresářová služba

Vaše zařízení DiskStation se může připojit k existující adresářové službě jako klient domény systému Windows nebo LDAP. Můžete provádět správu přístupových oprávnění uživatelů domény nebo LDAP ke sdíleným složkám a aplikacím systému DSM a povolit pro ně službu složky uživatele. Chcete-li připojit zařízení DiskStation k adresářové službě, přejděte do nabídky Ovládací panel > Adresářová služba.
