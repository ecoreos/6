TFTP

Trivial File Transfer Protocol (TFTP) je jednoduchý protokol pro přenos souborů, obecně používaný pro přenos konfigurací nebo zaváděcích souborů, když se nevyžaduje ověření. Povolení služby TFTP umožní zařízení DiskStation přenášet zaváděcí programy do klientských zařízení.

Konfigurace nastavení TFTP:

  1. Přejděte do části Ovládací panel > Souborové služby > TFTP a vyberte položku Povolit službu TFTP.
  2. V poli Kořenová složka TFTP určete, ke které složce na zařízení DiskStation mohou mít klienti TFTP přístup.
  3. Kliknutím na možnost Rozšířená nastavení nakonfigurujete následující nastavení.
    • Povolit protokol přenosu souborů TFTP: Sleduje data přenášená prostřednictvím protokolu TFTP.
    • Povolení klientského zařízení TFTP: Rozhodnutí o povoleních klientského zařízení TFTP:
      • Jen pro čtení: Klientům TFTP se zobrazuje pouze obsah kořenové složky.
      • Zapisovatelné: Klienti TFTP mohou upravovat pouze obsah kořenové složky.
    • Povolená klientská zařízení: Vyberte, kterým klientům je povoleno se připojit prostřednictvím protokolu TFTP:
      • Všechna připojení: Prostřednictvím protokolu TFTP se mohou připojit všichni klienti.
      • Povolit pouze následující rozsah IP adresy: Prostřednictvím protokolu TFTP se mohou připojit pouze klienti ve stanoveném rozsahu IP adres.
    • Časový limit: Jestliže server TFTP (tj. vaše zařízení DiskStation) nereaguje ve stanoveném časovém úseku, připojení TFTP se zastaví.
  4. Nastavení uložte kliknutím na možnost Použít.
