Obecné

Nastavení možností sítě, jako je výchozí brána, DNS, proxy a IPv6 tunneling, najdete v části Ovládací panel > Síť > Obecné.

Přejmenování zařízení DiskStation

Chcete-li přejmenovat zařízení DiskStation, zadejte nový název serveru a klikněte na možnost Použít.

V názvu serveru se rozlišují malá a velká písmena. Název může obsahovat 1 až 15 znaků, včetně písmen, čísel, podtržítek a symbolu mínus. Prvním znakem názvu musí být písmeno.

Zadání výchozí brány

Stačí pouze zadat IP adresu výchozí brány a nastavení uložit kliknutím na možnost Použít.

Zadání IP adresy serveru DNS

IP adresu serveru DNS můžete ručně nastavit. V takovém případě je nutné zadat IP adresu preferovaného serveru DNS. Lze zadat také alternativní IP adresu serveru DNS, a to pro případ vytvoření zálohy, pokud dojde k problémům se serverem DNS. Jakmile budou nastaveny podle potřeby IP adresy serveru DNS, uložte nastavení kliknutím na možnost Použít.

Poznámka:

Pole Server DNS a Výchozí brána budou vyplněna automaticky a nelze je změnit, pokud byla síťová rozhraní nakonfigurována automaticky nebo pokud používáte připojení PPPoE.

Připojení prostřednictvím serveru proxy

Chcete-li na zařízení DiskStation povolit připojení prostřednictvím serveru proxy, zaškrtněte položku Připojit prostřednictvím serveru proxy, zadejte hodnoty Adresa a Port a klikněte na možnost Použít.

Poznámka:

  • Kliknutím na položku Rozšířená nastavení provedete konfiguraci serveru proxy HTTP, proxy HTTPS a údajů pro ověření.
  • Ve výchozím nastavení jsou hodnoty serveru proxy HTTPS stejné jako u proxy HTTP.
  • Je-li zaškrtnuta možnost Připojit prostřednictvím serveru proxy, služby Aktualizace firmwaru DSM, Centrum balíčků a další balíčky se budou vždy připojovat k Internetu prostřednictvím serveru proxy.
  • Pokud je zaškrtnuta možnost Záložní server proxy pro místní adresy, ověřte, zda server DNS využívaný zařízením DiskStation dokáže překládat dotazy pro zařízení v místní síti.

Povolení funkce IPv6 Tunneling

Dříve než protokol IPv6 zcela nahradí protokol IPv4, je pro připojení hostitelů s adresou IPv6 ke službám IPv4 vyžadována funkce IPv6 tunneling. Tato technika nabízí způsob přenosu IPv6 v rámci stávajících sítí IPv4.
Pokud si nejste jistí, jakým způsobem nakonfigurovat protokol IPv6 nebo vyřešit problémy s připojením, obraťte se na poskytovatele internetových služeb.

Postup povolení tunelu IPv6:

  1. Chcete-li povolit podporu tunelu, zaškrtněte políčko Povolit tunel.
  2. Do pole Adresa serveru zadejte název hostitele nebo IP adresu poskytovatele tunelu, například: anonymous.freenet6.net.
  3. Vyberte jednu z možností Připojit anonymně nebo Připojit se stávajícím účtem.

Poznámka:

Vaše externí adresa IPv6 může být automaticky aktualizována poskytovatelem služeb. Zařízení DiskStation však neodstraní původní adresu IPv6 okamžitě.
