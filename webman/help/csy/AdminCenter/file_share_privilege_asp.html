Správa rozšířených oprávnění sdílené složky

Stránka uvedená v části Ovládací panel > Sdílená složka > Upravit > Upřesnit zahrnuje níže uvedené možnosti upřesnění přístupových oprávnění sdílené složky.

Rozšířená oprávnění sdílené složky

Rozšířená oprávnění sdílené složky nabízí dodatečnou vrstvu kontroly správy přístupových oprávnění sdílených složek. Je-li tato možnost povolena, uživatelé a skupiny mohou zobrazovat a upravovat obsah sdílené složky, pokud byla uživateli nebo skupině přidělena rozšířená oprávnění sdílené složky a zároveň oprávnění seznamu Windows ACL (v části Sdílená složka > Upravit > Oprávnění).

Postup povolení rozšířených oprávnění sdílené složky:

  1. Zaškrtněte políčko Povolit rozšířená oprávnění sdílené složky.
  2. Kliknutím na možnost Rozšířená oprávnění sdílené složky můžete upravit rozšířená oprávnění sdílené složky pro sdílenou složku.

Poznámka:

  • Rozšířená oprávnění sdílené složky budou použita tehdy, pokud uživatel aktivuje přístup ke sdílené složce přes následující souborové služby: Sdílení souborů Windows File Sharing, Sdílení souborů Apple File Sharing, File Station, FTP a WebDAV.

Rozšířená nastavení

V případě potřeby můžete nastavit další omezení uživatelům, kteří aktivují přístup ke sdílené složce přes File Station, FTP nebo WebDAV.

  • Zakázat procházení adresáře: Povolením této možnosti zakážete uživatelům procházení obsahu sdílené složky.
  • Zakázat úpravy stávajících souborů: Povolením této možnosti zakážete uživatelům přesouvat, odstraňovat nebo provádět úpravy sdílené složky. Pamatujte, že uživatelé budou moci i nadále zobrazovat, stahovat/odesílat, kopírovat nebo dekomprimovat obsah sdílené složky.
  • Zakázat stahování souborů: Povolením této možnosti zakážete uživatelům stahování obsahu sdílené složky.

Poznámka:

Výše uvedená omezení se nevztahují na uživatele, kteří patří do skupiny administrators.
