Hibernace

Můžete zadat dobu aktivní činnosti pevných disků HDD, po jejímž uplynutí disky přejdou do režimu spánku. Během spánku HDD bude zastaveno otáčení disků, čímž dojde ke snížení spotřeby energie a prodloužení životnosti pevných disků. Můžete zadat různá časová období pro interní pevné disky a externí pevné disky eSATA/USB.

Poznámka:

  • Režim spánku HDD je deaktivován v případě přístupu.
  • Funkci spánku HDD podporují pouze některá externí úložná zařízení USB.

Proč dochází ke spánku zařízení DiskStation?

Následující služby mohou na zařízení DiskStation deaktivovat režim spánku. Systém navíc ukončí režim hibernace 2 minuty před jakoukoli naplánovanou úlohou.

Balíčky:

  • Download Station: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud je v aplikaci Download Station naplánováno stahování eMule či RSS nebo probíhají aktivní úlohy.
  • Mail Station: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud v aplikaci Mail Station probíhá stahování dat ze serveru POP3.
  • Surveillance Station: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud aplikace Surveillance Station provádí záznam.
  • Media Server: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud je v aplikaci Media Server povolen protokol DMA (Přejděte do části Hlavní nabídka > Media Server > Media Server > Diagnostikovat).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Software třetích stran nebo optware, jako například SABnzbd

Procesy:

  • Miniatury a indexování: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud jsou vytvářeny miniatury nebo probíhá přeindexování mediálních souborů po dokončení upgradu. Během této doby bude v části „Spuštěno“ uveden proces „synoindexd“ (Přejděte do části Hlavní nabídka > Sledování zdrojů > Proces).
  • Prohlížeč Local Master Browser: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud je povolen prohlížeč Local Master Browser (Přejděte do části Ovládací panel > Win/Mac/NFS > Souborová služba Windows > Povolit prohlížeč Local Master Browser).
  • Program Windows Media Player: Na zařízení DiskStation nelze aktivovat funkci spánku, pokud je v síti LAN povolena služba Windows Media Player Network Sharing.
  • Na zařízení DiskStation nelze aktivovat funkci spánku, pokud počet současně spuštěných procesů vyžaduje přepnutí, při kterém došlo k překročení kapacity paměti RAM a pevné disky jsou dočasně využívány pro operace zápisu/čtení.

Ostatní

  • Zařízení USB: Na zařízení DiskStation nelze aktivovat funkci spánku systému, pokud jsou k jednotce připojena zařízení USB.

Hibernace systému

U zařízení DiskStation můžete nastavit aktivaci spánku systému jednu minutu po aktivaci spánku pevných disků. Funkce spánku systému snižuje spotřebu energie zařízení DiskStation.

V režimu spánku systému:

  1. Tlačítko napájení přejde do režimu pravidelných intervalů (pomalý cyklus zapnutí a vypnutí). Ostatní kontrolky LED budou vypnuty.
  2. Napájení pevných disků bude vypnuto.
  3. Činnost procesoru bude zastavena.
  4. Činnost externího ventilátoru bude zastavena.
  5. Kontrolky LED síťového portu LAN budou vypnuty.

Poznámka:

  • Funkci spánku systému může omezit jakýkoli vysílací paket SMB/CIFS, včetně souborových prohlížečů Windows v rámci sítě. Chcete-li při hibernaci systému ignorovat pakety pro vysílání CIFS, vyberte možnost Ignorovat pakety pro vysílání z prohlížeče Windows Explorer. Pokud vyberete tuto možnost, nelze pakety SMB/CIFS použít pro ukončení hibernace systému. Režim hibernace systému bude v případě serveru deaktivován také připojením jakýchkoli zařízení USB.

Hluboký spánek

Funkci hlubokého spánku rozšiřovacích jednotek lze povolit prostřednictvím rozhraní spárovaného zařízení DiskStation. U rozšiřovací jednotky a spárovaného zařízení DiskStation můžete provést konfiguraci různých časových období neaktivního provozu. Na rozšiřovací jednotce lze aktivovat režim hlubokého spánku i tehdy, je-li zařízení DiskStation stále v režimu online.

V režimu hlubokého spánku:

  • Všechny kontrolky LED rozšiřovací jednotky budou vypnuty.
  • Napájení pevných disků rozšiřovací jednotky nebude k dispozici.
  • Ventilátor rozšiřovací jednotky bude zastaven.

Poznámka:

  • Režim hlubokého spánku je deaktivován v případě přístupu.
  • Chcete-li při používání funkce hlubokého spánku zajistit běžný provoz, přepněte příslušný přepínač Automaticky/Ručně na zadním panelu do polohy Automaticky.

Automatické vypnutí

Můžete zadat dobu, po kterou bude interní disk ve stavu hibernace před vypnutím zařízení DiskStation.

Poznámka:

  • Nejdříve musíte povolit hibernaci pevných disků.
  • Musíte nejdřív povolit službu WOL, abyste mohli zařízení DiskStation vzdáleně zapnout.
