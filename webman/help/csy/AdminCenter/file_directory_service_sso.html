Klient SSO

Pokud se zařízení DiskStation připojilo ke stejné doméně nebo službě LDAP jako server SSO, můžete nastavit zařízení DiskStation jako klienta SSO. Po přihlášení k serveru SSO pomocí svých přihlašovacích údajů mohou uživatelé využívat služby poskytované zařízením DiskStation.

Než začnete, ověřte si, jestli se zařízení DiskStation připojilo do stejné domény nebo služby LDAP jako server SSO.

Nastavení zařízení DiskStation jako klienta SSO:

  1. Přejděte do nabídky Ovládací panel > Doména/LDAP > Klient SSO.
  2. Zaškrtněte políčko Povolit službu SSO.
  3. Do polí pod zaškrtávacím políčkem zadejte adresu URL serveru SSO a ID aplikace.

    Poznámka:

    ID aplikace získáte jedním z těchto dvou způsobů:

    • Metoda č. 1:
      1. Přihlaste se do systému DSM spuštěného jako server SSO.
      2. Přidejte zařízení DiskStation, které bude sloužit jako klientská aplikace, do seznamu aplikací na serveru SSO. Podrobné pokyny najdete v článku nápovědy pro Server SSO > Seznam aplikací.
      3. Po přidání zařízení DiskStation do serveru SSO zkopírujte a vložte ID aplikace přidělené serverem SSO.
    • Metoda č. 2:
      1. Po zadání adresy URL serveru SSO klikněte na položku Rychlá registrace vedle pole ID aplikace.
      2. V rámečku registrace zadejte do polí Správce serveru a Heslo přihlašovací údaje účtu správce serveru SSO.
      3. Do pole Název aplikace zadejte název své klientské aplikace.
      4. Přesvědčte se, jestli možnost Přesměrovat URI odkazuje přesně na přihlašovací stránku zařízení DiskStation.
      5. Klikněte na možnost OK. Zařízení DiskStation se přidá do serveru SSO a ID aplikace se automaticky vyplní.

    ______

    * Je potřeba, abyste při konfiguraci všech klientů SSO zadali totožnou adresu URL serveru SSO (název hostitele nebo IP).

  4. Nastavení uložte kliknutím na možnost Použít.

Poznámka:

  • Pokud server SSO používá protokol HTTPS, mohou prohlížeče přihlášení SSO zablokovat, protože schází důvěryhodný certifikát.
  • Pokud se k vašemu serveru SSO přistupuje prostřednictvím protokolu HTTP a pokud zařízení DiskStation (nastavené jako klient SSO) používá připojení HTTPS, musí uživatelé nakonfigurovat nastavení svých prohlížečů tak, aby mohli používat nezabezpečené připojení.
  • IP adresa serveru SSO ve formátu IPv6 nebude podporována, pokud jste v systému DSM v části Ovládací panel > Zabezpečení > Zabezpečení zaškrtli políčko Vylepšit zabezpečení pomocí záhlaví HTTP Content Security Policy (CSP).
