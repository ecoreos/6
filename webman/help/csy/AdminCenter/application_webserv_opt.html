HTTP služba

Vedle výchozího čísla portu 80 můžete pro použití stanic Photo Station a Web Station přidat další port.

Přidání dodatečného čísla portu pro přístup přes protokol HTTP:

  1. Zaškrtněte možnost Přidat dodatečný port HTTP.
  2. Zadejte do pole číslo portu. Pro přístup přes protokol HTTP můžete přidat dodatečné číslo portu z rozmezí od 1 do 65535.

Nelze použít následující porty:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, výchozí porty pro eMule: 4662 (TCP), 4672(UDP) a výchozí rozsah portů pro FTP: konkrétní rozsah se může u různých modelů lišit.

    Poznámka:
  1. Čísla portů pro přístup přes protokol HTTP nemohou být stejná jako čísla portů pro službu FTP/BT.
  2. Port HTTP a port HTTPS nesmí být v konfliktu.
