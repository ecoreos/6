Nastavení systému DSM

V části Ovládací panel > Síť > Nastavení DSM je možné změnit číslo portu, který klient používá k přístupu k zařízení DiskStation.

Změna výchozího čísla portu:

Připojení HTTP a HTTPS jsou v systému DSM povolená vždy. Výchozí port HTTP je 5000 a výchozí port HTTPS je 5001.

  1. Do pole HTTP nebo HTTPS zadejte číslo portu.
  2. Klikněte na možnost Použít.

Přesměrování připojení HTTP na připojení HTTPS:

Když se klienti pokoušejí přistupovat k zařízení DiskStation přes port HTTP, můžete je přesměrovat na port HTTPS.

  1. Vyberte možnost Automaticky přesměrovat připojení HTTP na HTTPS (kromě stanic Web Station a Photo Station).
  2. Klikněte na možnost Použít.

Porty používané pro připojení HTTPS:

  • DSM: Výchozím číslem portu je 5001, takže je možné se do systému DSM bezpečně připojit zadáním následující adresy: https://server-hostname:5001/
  • Web Station a Photo Station: Výchozím číslem portu je 443, takže ke službám Web Station a Photo Station je možné se bezpečně připojit zadáním následující adresy: https://server-hostname/

Nalezení zařízení DiskStation na síti:

Zařízení DiskStation je možné najít pomocí zjišťování sítě Windows nebo službou zjišťování Bonjour.

  1. Vyberte službu, pomocí které zařízení DiskStation na síti najít:
    • Povolit zjišťování sítě Windows
    • Povolit službu Bonjour service discovery
  2. Klikněte na možnost Použít.

Doména

V části Doména je možné zadat vlastní název domény pro přístup k zařízení DiskStation.

Povolení vlastní domény:

  1. Zaškrtněte políčko Povolit vlastní doménu.
  2. Zadejte použitý název domény pro své zařízení DiskStation.
  3. Zaškrtnutím políčka Povolit HSTS je možné v této doméně vynutit šifrovaná připojení HTTPS.
  4. Klikněte na možnost Použít.

Poznámka:

  • Každý název domény je možné použít pouze pro jednu odlišnou aplikaci na zařízení DiskStation.
  • Jestliže název domény nepřekročí délku 15 znaků a splňuje konvence pro názvy NetBIOS, systém název domény (např. MojeDSM) zaregistruje a bude vysílat automaticky. Uživatelé Windows se k ploše systému DSM ve stejné místní síti dostat pomocí podobné adresy URL: http://MojeDSM nebo https://MojeDSM
  • Když k aplikacím přistupujete prostřednictvím adres URL, které používají názvy domén, budete procházet přes standardní port HTTP(80) nebo standardní port HTTPS (443).
