Požadavky aplikace File Station

Chcete-li používat plně funkční aplikaci File Station, budete potřebovat nejnovější webový prohlížeč, aplikace Java a Flash Player.

  • Prohlížeč: Internet Explorer 8 nebo novější, Firefox 3.6 nebo novější, Safari 5.0 nebo novější, Chrome; s povolenou funkcí AJAX a JavaScript
  • Java (pro prohlížení složek na místním počítači): Java Runtime Environment (JRE) 5 nebo novější
  • Flash: Flash Player 9.0.28 nebo novější
