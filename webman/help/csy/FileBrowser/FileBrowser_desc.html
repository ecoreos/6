File Station

File Station je jednoduchý nástroj pro správu souborů, který uživatelům systému DSM umožňuje přístup ke složkám zařízení DiskStation prostřednictvím webového prohlížeče. Spuštěním dvou nebo více instancí File Station můžete provádět správu veškerých dat zařízení DiskStation jejich přetažením mezi jednotlivými instancemi File Station.
