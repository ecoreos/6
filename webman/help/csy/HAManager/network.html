Nastavení sítě

Tato stránka obsahuje tři karty, které zobrazují nastavení sítě pro cluster high-availability, aktivní a pasivní členské servery.

Karta clusteru HA

Při vytváření clusteru high-availability vytvoří systém obecnou IP adresu clusteru high-availability určenou pro příjem příchozích připojení pro tuto IP adresu. V případě potřeby lze vytvořit dodatečné IP adresy clusteru high-availability. IP adresy aktivovaných síťových rozhraní na aktivním serveru budou zobrazeny na této kartě (síťové rozhraní využívané připojením prezenčního signálu na této stránce zobrazeno nebude, protože zmíněné rozhraní bude použito pro synchronizaci dat mezi dvěma servery). Tato karta zobrazuje níže uvedené informace:

  • Síťové rozhraní: Aktivovaná síťová rozhraní na aktivním serveru.
  • Připojení: Zobrazuje, zda je toto rozhraní připojeno či nikoli.
  • IP adresa: Při vytváření clusteru high-availability vytvoří systém obecnou IP adresu clusteru high-availability určenou pro příjem příchozích připojení pro tuto IP adresu. Pomocí této IP adresy se můžete připojit ke clusteru high-availability, pokud chcete využívat služeb systému DSM bez ohledu na to, který server Synology plní funkci aktivního serveru. Díky tomu není nutné se po přepnutí připojovat k různým IP adresám.
  • Maska podsítě: Přidružená maska podsítě dané IP adresy clusteru high-availability.

Úprava položek IP adresa a Maska podsítě:

  1. Vyberte IP adresu nebo masku podsítě, kterou chcete upravit, a klepnutím na možnost Upravit nebo poklepáním na pole zahajte úpravy.
  2. Chcete-li povolit nebo zakázat jednotlivá rozhraní, zaškrtněte políčko Povolit. Můžete povolit více než jedno síťové rozhraní clusteru, původní síťové rozhraní nastavené při vytváření clusteru však nelze zakázat.

Změna názvu serveru clusteru high-availability:

  • Klepnutím na možnost Změnit síťová nastavení zobrazíte název clusteru high-availability a provedete změnu nastavení.
  • Klepnutím na možnost Použít uložíte aktuální nastavení názvu a IP adresy clusteru high-availability.
  • Aktuální změny zrušíte klepnutím na možnost Storno.

Karta aktivního a pasivního serveru (zobrazen jako název serveru)

Tyto karty zobrazují stav síťových rozhraní na aktivním a pasivním serveru. Z toto je připojení prezenčního signálu součástí jednoho síťového rozhraní jednotlivých aktivních a pasivních serverů. Tyto karty zobrazují následující informace:

  • Síťové rozhraní: Fyzické síťové rozhraní na serveru.
  • IP adresa: Přiřazená IP adresa daného síťového rozhraní.
  • Maska podsítě: Přiřazená maska podsítě dané IP adresy.
  • Popis: Zobrazuje informace o daném síťovém rozhraní a zpřístupnění přiřazené IP adresy. Toto pole zobrazí následující možné podmínky:
    1. Přístupná původní IP adresa: Původní IP adresa přidělená rozhraní na aktivním serveru je přístupná pro data a služby.
    2. Nepřístupná původní IP adresa: Původní IP adresa přidělená rozhraní na pasivním serveru není přístupná.
    3. IP adresa připojení prezenčního signálu, nepřístupná z Intranetu: Původní IP adresa automaticky přidělená připojení prezenčního signálu není přístupná pro data a služby.
