Sicherheit

Auf dieser Registerkarte können Sie die Netzwerksicherheitseinstellungen jeder Netzwerkschnittstelle ändern.

DoS Schutz

Der Denial-of-service (DoS) Schutz hilft bei der Vermeidung von bösartigen Angriffen über das Internet.

DoS Schutz aktivieren:

  1. Aktivieren Sie das Kästchen DoS Schutz aktivieren.
  2. Klicken Sie auf Übernehmen.

VPN Pass-through*

Wenn DiskStation als Router fungiert, können über VPN pass-through Clients im Netzwerk der Passthrough zu DiskStation NAT gestattet und ausgehende VPN-Verbindungen aufgebaut werden. Wenn Ihre interne Einrichtung einen VPN Server beinhaltet, gehen Sie bitte zu Systemsteuerung > Routerkonfiguration, um Portweiterleitungsregeln einzurichten und dem VPN Server den Aufbau von VPN Verbindungen zu gestatten.

VPN pass-through Einstellungen bearbeiten:

  1. Hier können die folgenden Einstellungen aktiviert oder deaktiviert werden:
    • PPTP Pass-through: Ermöglicht das Durchreichen von PPTP (Point-to-Point Tunneling Protocol) Verkehr und den Aufbau von Verbindungen.
    • IPSec Pass-through: Ermöglicht das Durchreichen von IPSec (IP Security Protocol) Verkehr und den Aufbau von Verbindungen.
    • L2TP Pass-through: Ermöglicht das Passthrough von L2TP (Layer 2 Tunneling Protocol) Verkehr und den Aufbau von Verbindungen.
  2. Klicken Sie nach dem Ändern der Einstellungen auf Übernehmen.

____
*Nur bei bestimmten Modellen verfügbar.
