Login-Stil

Sie können das Aussehen der DSM-Anmeldebildschirm durch eine Auswahl aus einer Vielzahl an Vorlagen, durch die Änderung des Hintergrundbildes oder das Hinzufügen eines Logos anpassen. Gehen Sie hierzu zu Systemsteuerung > Login-Stil.

So passen Sie den DSM-Anmeldebildschirm an:

Auf der Seite Login-Stil werden folgende Optionen angezeigt:

  • Titel der Anmeldeseite: Der hier eingegebene Text wird auf dem Anmeldebildschirm angezeigt.
  • Zeitinformationen: Entscheiden Sie, ob lokale Zeitinformationen auf dem Anmeldebildschirm angezeigt werden sollen.
  • Vorlage: Wählen Sie ein Farbschema und ein Standard-Hintergrundbild. Die gewählte Vorlage wird bei der nächsten Anmeldung bei DiskStation angewendet.
  • Hintergrund: Um das auf dem Anmeldebildschirm angezeigte Hintergrundbild zu ändern, klicken Sie auf das bestehende Bild oder auf das Pluszeichen (+), wenn kein Bild ausgewählt ist. Sie können aus den Standard-Hintergrundbildern, auf DiskStation gespeicherten Bildern oder bereits verwendeten Bildern auswählen. Nur die Bildformate JPE, JPEG, JPG, BMP, GIF oder PNG werden unterstützt.
  • Logo: Um dem Anmeldebildschirm ein Logo hinzuzufügen, klicken Sie auf das bestehende Logo oder auf das Pluszeichen (+), wenn kein Bild ausgewählt ist. Sie können ein Bild mit einer Größe von bis zu 1 MB hochladen, aus Bildern auswählen, die auf DiskStation gespeichert sind, oder ein bereits verwendetes Bild auswählen.
