Netzwerk-Routing

DSM bietet mehrere Netzwerk-Routing-Optionen für verschiedene Anforderungen an die Netzwerkkonfiguration.

So trennen Sie den Netzwerkdurchfluss der Netzwerk-Schnittstellen im selben LAN:

  1. Gehen Sie zu Systemsteuerung > Netzwerk > Erweitert.
  2. Markieren Sie unter Allgemein das Kontrollkästchen Antworten auf ARP-Anfragen, wenn die Ziel-IP-Adresse die lokale Adresse ist, die in der Eingangsschnittstelle konfiguriert wurde.
  3. Klicken Sie auf Anwenden, um Ihre Einstellungen zu speichern.

Dadurch wird die standardmäßige Routing-Regel für jede Netzwerk-Schnittstelle festgelegt, wodurch die Netzwerkpakete von den einzelnen Schnittstellen von ihren eigenen physischen Netzwerk-Schnittstellen gesendet werden.

Falls darüber hinaus mehr als zwei Netzwerk-Schnittstellen im gleichen LAN vorhanden sind, wird durch die Aktivierung dieser Option der Datenfluss im Netzwerk für jede Schnittstelle separat übertragen. Dann können auf jede Schnittstelle eigene Firewall-Regeln angewendet werden.

So wird jede Netzwerk-Schnittstelle über ihre jeweiligen Gateways weitergeleitet:

  1. Gehen Sie zu Systemsteuerung > Netzwerk.
  2. Markieren Sie unter Allgemein die Option Mehrere Gateways aktivieren.
  3. Klicken Sie auf Anwenden, um Ihre Einstellungen zu speichern.

Dadurch wird das Routing für jede Netzwerk-Schnittstelle in Ihrem Netzwerk über ihren entsprechenden Gateway geleitet.

So ändern Sie den Standard-Gateway:

  1. Gehen Sie zu Systemsteuerung > Netzwerk.
  2. Klicken Sie unter Netzwerk-Schnittstelle auf die Option Serviceauftrag.
  3. Ordnen Sie die Gateways um, indem Sie den Gateway auswählen und auf Auf oder Ab klicken.
  4. Klicken Sie auf OK.

Der Gateway an oberster Stelle fungiert als Standard-Gateway für Ihr LAN.

So richten Sie ein statisches Routing ein:

  1. Gehen Sie zu Systemsteuerung > Netzwerk.
  2. Klicken Sie unter Statische Route auf die Option Erstellen.
  3. Geben Sie unter Zielnetzwerk die IP-Adresse Ihres Ziels ein.
  4. Geben Sie unter Netzmaske die Netzmaske Ihres Ziels ein.
  5. Geben Sie unter Gateway die Gateway-Adresse des Ziels ein.
  6. Wählen Sie unter Schnittstelle die Schnittstelle des Ziels.
  7. Klicken Sie auf OK.

So wird die IP-Routingtabelle angezeigt:

  1. Gehen Sie zu Systemsteuerung > Netzwerk.
  2. Klicken Sie unter Statische Route auf die Option IP-Routingtabelle.
  3. Wählen Sie eine Netzwerk-Schnittstelle, um ihre IP-Routingtabelle mithilfe der Drop-down-Liste links oben anzuzeigen.
