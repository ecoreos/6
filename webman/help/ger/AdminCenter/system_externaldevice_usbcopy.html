USBCopy

Sie können einen Zielort für die USBCopy Funktion auswählen und festlegen.

Aktivieren von USBCopy:

  1. Markieren Sie USBCopy aktivieren.
  2. Wählen Sie einen gemeinsamen Ordner aus dem Drop-Down-Menü aus.
  3. Klicken Sie auf OK.

Wenn Sie über den USB Port an der Synology DiskStation Frontblende ein USB-Gerät angehängt haben (wie eine Digitalkamera), können Sie die Taste Kopieren an der Frontblende drücken, um Daten auf einem USB-Gerät zum freigegebenen Zielordner zu kopieren. Wenn der USBCopy Zielordner gelöscht wurde, ersetzt das System ihn automatisch mit einen öffentlich freigegebenen Ordner.

    Einschränkungen:
  1. Nur USB-Geräte im FAT- oder NTFS-Format werden unterstützt.
  2. USB-Geräte müssen am USB-Anschluss auf der Vorderseite der Synology DiskStation angeschlossen sein.
  3. Der verfügbare Speicherplatz des gemeinsamen Zielordners muss größer sein als der vom USB-Gerät verwendete Speicherplatz.
