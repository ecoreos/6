DSM-Aktualisierung

Synology veröffentlicht regelmäßig kostenlose DSM-Aktualisierungen. Die Aktualisierungen können neue Funktionen, Funktionsverbesserungen und Leistungserweiterungen beinhalten.

Wichtig: Normalerweise hat die DSM-Aktualisierung keine Auswirkung auf die Daten der DiskStation. Trotzdem empfehlen wir Ihnen, Ihre Daten zu sichern, bevor Sie mit der Aktualisierung beginnen.

Manuelle DSM-Aktualisierung durchführen

Sie können die „.pat-Aktualisierungsdatei“ im Synology Download-Zentrum herunterladen und eine manuelle DSM-Aktualisierung durchführen.

So aktualisieren Sie den DSM auf manuelle Weise:

  1. Besuchen Sie www.synology.com, um die neueste DSM-Aktualisierung herunterzuladen und Ihren Computer zu aktualisieren. Entpacken Sie die Datei, um eine .pat Aktualisierungsdatei zu erhalten.
  2. Klicken Sie auf Manuelle DSM-Aktualisierung, um zur Seite „Manuelle DSM-Aktualisierung“ zu wechseln
  3. Klicken Sie auf Durchsuchen, um die Aktualisierungsdatei auszuwählen.
  4. Klicken Sie auf Anwenden, um die Aktualisierung Ihres DSM zu starten. Das System startet automatisch neu, sobald die Aktualisierung abgeschlossen ist.

Fehlerbehebung:

Wenn Sie Ihren DSM nicht aktualisieren können, helfen Ihnen die folgenden Vorschläge möglicherweise, das Problem zu beheben:

  • Prüfen Sie, ob der Aktualisierungsprozess von einem Benutzer oder anderen unerwarteten Ereignissen, z. B. Stromausfällen, unterbrochen wurde.
  • Überprüfen Sie die DSM-Version. Sie können das System nur mit einer neueren DSM-Version aktualisieren.
  • Laden Sie die DSM-Aktualisierung erneut herunter, denn die Aktualisierungsdatei könnte beschädigt sein.

Besteht das Problem weiterhin, kontaktieren Sie bitte den Technischen Support von Synology.

Automatische DSM-Aktualisierung durchführen

Im Gegensatz zur manuellen Aktualisierung überprüft das System bei Synology, ob eine neue DSM-Version zur Verfügung steht. Gehen Sie zu DSM-Aktualisierung. Das Resultat wird mit grünem oder rotem Text angezeigt. Wenn das Ergebnis Ihnen anzeigt, dass der neuste DSM zum Download zur Verfügung steht, können Sie den neusten DSM herunterladen und die Aktualisierung durchführen.

So lassen Sie das System die neueste DSM-Version herunterladen und die Aktualisierung durchführen:

  1. Wenn die neueste DSM-Version zum Herunterladen bereitsteht, klicken Sie auf den Link Herunterladen. Das System wird mit dem Herunterladen der neuesten DSM-Version beginnen.
  2. Das System zeigt eine der folgenden Desktop-Benachrichtigungen, um Ihnen mitzuteilen, ob das Herunterladen erfolgreich war oder nicht: DSM-Download abgeschlossen oder DSM-Download fehlgeschlagen.
  3. Klicken Sie auf das Desktop-Benachrichtigungsfenster, um zur DSM-Aktualisierung zu gelangen.
  4. Führen Sie eine der folgenden Aktionen aus:
    • Klicken Sie nach dem Herunterladen der DSM-Version auf Jetzt aktualisieren, um mit der DSM-Aktualisierung sofort zu beginnen.
    • Sollte der DSM-Download fehlgeschlagen sein, klicken Sie auf Neuer Versuch, damit das System erneut die neueste DSM-Version von Synology herunterladen kann.

Regelmäßig nach Aktualisierungen suchen:

  1. Markieren Sie Automatisch nach DSM Aktualisierungen suchen.
  2. Wählen Sie einen Aktualisierungsplan, und legen Sie den Installationszeitplan fest.
  3. Klicken Sie auf OK.

Ein Desktop-Benachrichtigung wird angezeigt, wenn eine DSM-Aktualisierung heruntergeladen wurde und für die Installation bereit steht. Um die automatische DSM-Aktualisierung zu stoppen, können Sie sie in der Desktop-Benachrichtigung abbrechen oder indem Sie die Update-Einstellungen ändern. DSM-Aktualisierungen werden vor dem geplanten Herunterfahren installiert, wenn Sie Herunterfahren unter Energie-Zeitplan aktiviert haben.

Fehlerbehebung:

Falls während der Suche nach der neuesten DSM-Version oder während des Herunterladens Fehlermeldungen (mit rotem Text) angezeigt werden, versuchen Sie das Problem folgendermaßen zu lösen:

  • Stellen Sie sicher, dass die DiskStation mit dem Internet verbunden ist.
  • Möglicherweise ist der Download-Server von Synology derzeit besetzt. Versuchen Sie das Herunterladen zu einem späteren Zeitpunkt noch einmal.

Besteht das Problem weiterhin, kontaktieren Sie bitte den Technischen Support von Synology.
