Verzeichnisdienst

Ihr DiskStation kann einem bestehenden Verzeichnisdienst wie z. B. einer Windows-Domain oder einem LDAP-Client beitreten. Sie können die Zugriffsrechte von Domain- oder LDAP-Benutzern für freigegebene Ordner und DSM-Anwendungen verwalten sowie den Home-Dienst für sie aktivieren. Gehen Sie zu Systemsteuerung > Verzeichnisdienst, um mit Ihrem DiskStation einem Verzeichnisdienst beizutreten.
