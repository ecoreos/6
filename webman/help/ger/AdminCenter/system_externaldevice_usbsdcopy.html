USBCopy

Sie können einen Zielordner für die USBCopy-Funktion auswählen, indem Sie zu Systemsteuerung > Externe Geräte > USBCopy/SDCopy gehen.

USBCopy aktivieren:

  1. Aktivieren Sie USBCopy aktivieren.
  2. Wählen Sie einen freigegebenen Ordner aus dem Dropdown-Menü aus.

Wenn Sie an den USB-Port auf der Vorderseite des Synology-Produkts ein USB-Gerät anschließen (wie eine Digitalkamera), können Sie die Kopieren-Taste auf der Vorderseite drücken, um Daten auf dem USB-Gerät zu dem hier zugewiesenen freigegebenen Zielordner zu kopieren. Wenn der USBCopy Zielordner gelöscht wurde, ersetzt das System ihn automatisch durch einen öffentlichen freigegebenen Ordner.

Einschränkungen

  1. Nur USB-Geräte mit FAT-, NTFS-, ext3- und ext4-Format werden unterstützt.
  2. USB-Geräte müssen am USB-Anschluss auf der Vorderseite des/der DiskStation angeschlossen sein.
  3. Nur der Admin oder Benutzer der Gruppe administrators haben die entsprechenden Zugriffsrechte, um Sicherungsaufgaben auf USB-Geräten durchzuführen.
  4. Der verfügbare Speicherplatz des freigegebenen Zielordners muss größer sein als der vom USB-Gerät verwendete Speicherplatz.

SDCopy*

Sie können einen Zielordner für die SDCopy-Funktion auswählen, indem Sie zu Systemsteuerung > Externe Geräte > USBCopy/SDCopy gehen.

SDCopy aktivieren:

  1. Aktivieren Sie SDCopy aktivieren.
  2. Wählen Sie einen freigegebenen Ordner aus dem Dropdown-Menü aus.

Wenn Sie eine SD-Karte in den SD-Schlitz auf der Vorderseite des Synology-Produkts einstecken, können Sie die Kopieren-Taste auf der Vorderseite drücken, um Daten auf dem SD-Gerät zu dem hier zugewiesenen freigegebenen Zielordner zu kopieren. Wenn der SDCopy-Zielordner gelöscht wurde, ersetzt das System ihn automatisch durch einen öffentlichen freigegebenen Ordner.

Einschränkungen

  1. Nur SD-Geräte mit FAT-, NTFS-, ext3- und ext4-Format werden unterstützt.
  2. SD-Geräte müssen in den SD-Schlitz auf der Vorderseite des Synology-Produkts eingesteckt werden.
  3. Nur der Admin oder Benutzer der Gruppe administrators haben die entsprechenden Zugriffsrechte, um Sicherungsaufgaben auf SD-Geräten durchzuführen.
  4. Der verfügbare Speicherplatz des freigegebenen Zielordners muss größer sein als der vom SD-Gerät verwendete Speicherplatz.

* Nur bei bestimmten Modellen verfügbar.
