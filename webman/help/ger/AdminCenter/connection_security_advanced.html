Erweitert

Die folgenden erweiterten Einstellungen unter Systemsteuerung > Sicherheit > Erweitert können für DSM-Browser-Sitzungen geändert werden:

HTTP-Komprimierung

Sie können HTTP-Komprimierung für DiskStation aktivieren. Durch Verringerung der Bandbreite des Netzwerkverkehrs können Webseiten schneller geladen werden.

HTTP-Komprimierung aktivieren/deaktivieren:

  • Aktivieren bzw. deaktivieren Sie HTTP komprimieren unter Systemsteuerung > Sicherheit > Erweitert.

TLS- / SSL-Profil

Sie können die Sicherheitsstufe von HTTPS-verschlüsselte Verbindungen wählen. Es stehen die folgenden 3 Stufen zur Auswahl:

  • Moderne Kompatibilität - Ist nur mit den neuesten Browsern und mobilen Geräten kompatibel.
  • Zwischenkompatibilität - Die mittlere Sicherheitsstufe. Dies ist die Standardeinstellung.
  • Alte Abwärtskompatibilität - Unsicher, wird jedoch von den meisten Browsern und mobilen Geräten unterstützt.
