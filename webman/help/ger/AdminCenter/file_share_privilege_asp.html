Erweiterte Berechtigungen für gemeinsame Ordner verwalten

Die Seite unter Systemsteuerung > Gemeinsamer Ordner > Bearbeiten > Erweitert enthält die folgenden Optionen, um weitere Feinabstimmungen der Zugriffsberechtigungen eines gemeinsamen Ordners vorzunehmen.

Erweiterte Freigabeberechtigungen

Erweiterte Freigabeberechtigungen bieten eine zusätzliche Kontrollebene, um die Zugriffsberechtigungen für den gemeinsamen Ordner zu verwalten. Falls aktiviert, können Benutzer und Gruppen den Inhalt eines gemeinsamen Ordner nur anzeigen oder ändern, wenn dem Benutzer oder der Gruppe sowohl erweiterte Freigabeberechtigungen gewährt wurden, und Windows-ACL-Berechtigungen (unter Gemeinsamer Ordner > Bearbeiten > Berechtigungen).

So werden erweiterte Freigabeberechtigungen aktiviert:

  1. Markieren Sie das Kästchen Erweiterte Freigabeberechtigung aktivieren.
  2. Klicken Sie auf Erweiterte Freigabeberechtigungen, um die erweiterten Freigabeberechtigungen für den gemeinsamen Ordner zu ändern.

Hinweis:

  • Erweiterte Freigabeberechtigungen werden angewendet, wenn ein Benutzer über die folgenden Dateidienste auf einen gemeinsamen Ordner zugreift: Windows Dateifreigabe, Apple Dateifreigabe, File Station, FTP und WebDAV.

Erweiterte Einstellungen

Falls nötig, können Sie weitere Einschränkungen für Benutzer für den Zugriff auf den gemeinsamen Ordner über File Station, FTP oder WebDAV festlegen.

  • Durchsuchen von Verzeichnissen deaktivieren: Durch das Aktivieren dieser Option wird der Inhalt des gemeinsamen Ordners für Benutzer nicht mehr angezeigt.
  • Änderungen an bestehenden Dateien deaktivieren: Durch das Aktivieren dieser Option können Benutzer Dateien im gemeinsamen Ordner nicht mehr verschieben, löschen oder ändern. Bitte beachten Sie, dass die Benutzer immer noch in der Lage sind, den Inhalt des gemeinsamen Ordners anzuzeigen, herunter/hochzuladen, zu kopieren oder zu entpacken.
  • Herunterladen von Dateien deaktivieren: Durch das Aktivieren dieser Option können Benutzer den Inhalt des gemeinsamen Ordners nicht mehr herunterladen.

Hinweis:

Die oben genannten Beschränkungen gelten nicht für Benutzer, die zur Gruppe administrators gehören.
