Einen Benutzer bearbeiten

Suchen Sie das Benutzerkonto (durch Eingeben des Benutzernamens in das Suchfeld), doppelklicken darauf oder klicken Sie auf Bearbeiten, dann können Sie die Kontodaten ändern.

Hinweis:
  1. Wenn Sie ein Konto deaktivieren, kann es nicht mehr benutzt werden.
  2. Quote muss eine Ganzzahl größer als 0 und kleiner als die Festplatten-/Volume-Kapazität sein. Ein Quotenwert von 0 bedeutet, dass Effektive Quote durch Gruppengrenzwert bestimmt wird.
  3. Befinden sich mehr als ein Volume/gemeinsamer Ordner auf dem Server, können Sie die Quote für den Benutzer auf jedem Volume bzw. in jedem gemeinsamen Ordner einstellen.
  4. Die E-Mail-Informationen auf der Seite „Benachrichtigung“ müssen korrekt eingestellt werden, damit das System den Benutzer per E-Mail benachrichtigt, falls die Kontoinformationen erstellt oder verändert wurden.
  5. Wenn Sie die Geschwindigkeitseinschränkung für einen Benutzer einstellen, werden Zahlen unter Ergebnis angezeigt, wenn bereits eine Gruppengeschwindigkeitseinschränkung für diesen Benutzer angewendet wird.
  6. Die einem Benutzer zugewiesene Geschwindigkeitseinschränkung gilt nicht für die Geschwindigkeitseinschränkung der Gruppe, zu der der Benutzer gehört.
