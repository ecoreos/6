Gemeinsamer Ordner

Gemeinsame Ordner sind die Grundverzeichnisse, in denen Sie Dateien und Ordner auf Ihrem DiskStation speichern können. Sie müssen im DSM mindestens einen gemeinsamen Ordner erstellen, bevor Sie Daten auf dem DiskStation speichern können. In den gemeinsamen Ordnern gespeicherte Daten können privat bleiben oder mit bestimmten Benutzern oder Benutzergruppen geteilt werden, je nach den anpassbaren Zugriffsrechten.

Nachfolgend finden Sie eine Liste der voreingestellten gemeinsamen Ordner, die automatisch erstellt werden, wenn bestimmte Anwendungen, Dienste oder Pakete installiert oder aktiviert werden.

Name Beschreibung
public Der Ordner public wird bei der ersten Einrichtung des DiskStation automatisch erstellt.*
web Der Ordner web ist der Standard-Speicherort für die Inhalte einer Website, die auf DiskStation gehostet werden. Er wird automatisch erstellt, wenn Web Station aktiviert ist.
photo Der Ordner photo speichert Fotos und Videos, die Sie mit anderen auf der Photo Station teilen möchten. Er wird automatisch erstellt, wenn die Photo Station oder der DLNA/UPnP-Medienserver aktiviert ist.
music Der Ordner music enthält Musik, die Sie auf Audio Station hören möchten. Er wird automatisch erstellt, wenn die Audio Station oder der DLNA/UPnP-Medienserver aktiviert ist.
video Der Ordner video enthält Videos, die Sie über DLNA/UPnP-Geräte durchsuchen können. Er wird automatisch erstellt, wenn die Video Station oder der DLNA/UPnP-Medienserver aktiviert ist.
surveillance Der Ordner surveillance enthält die Aufzeichnungen der Surveillance Station. Er wird automatisch erstellt, wenn Surveillance Station aktiviert ist. Er hat nur Lesezugriff und kann nur vom System-Administrator aufgerufen werden.
home Der Ordner home bietet einen privaten Bereich für jeden Benutzer, in dem Daten gespeichert werden können, auf die nur dieser Benutzer zugreifen kann. Er wird automatisch erstellt, wenn die Funktion Benutzer-Home aktiviert ist.
homes Der Ordner homes enthält die home-Ordner aller Benutzer. Er wird automatisch erstellt, wenn die Funktion Benutzer-Home aktiviert ist. Nur der Systemadministrator hat Zugriff und kann alle Benutzer-Home-Ordner verwalten.
NetBackup Der Ordner NetBackup wird automatisch erstellt, wenn die Netzwerksicherung aktiviert ist.
usbshare Der Ordner usbshare [Nummer] wird automatisch erstellt, wenn Sie ein externes USB-Gerät an den USB-Port von DiskStation anschließen.*
satashare Der Ordner satashare wird automatisch erstellt, wenn Sie ein externes eSATA-Laufwerk an den eSATA-Port von DiskStation anschließen.*

Hinweis:

  • Sie können Berechtigungen für gemeinsame Ordner von Benutzern nur im Einfach-Modus bearbeiten. Wenn Sie Benutzergruppen erstellen oder Gruppen-Zugriffsrechte für gemeinsame Ordner verwalten möchten, gehen Sie zu Systemsteuerung (Erweiterter Modus) > Gruppe.

So erstellen Sie einen gemeinsamen Ordner:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Gemeinsamer Ordner, und klicken Sie auf der linken Seite auf das Symbol +.
  2. Geben Sie einen Namen an und wählen Sie ein Volume, auf dem der gemeinsame Ordner erstellt werden soll.
  3. Markieren Sie bei Bedarf Papierkorb aktivieren. Falls diese Option aktiviert ist und Dateien im gemeinsamen Ordner gelöscht werden, werden sie in einen Ordner namens #recycle verschoben. Der Zugriff auf den Papierkorb kann auf Benutzer der Gruppe administrators beschränkt werden.
  4. Aktivieren oder deaktivieren Sie die entsprechenden Felder für jeden Benutzer, um seine Berechtigungen für den gemeinsamen Ordner anzupassen:
    • Lesen/Schreiben: Der Benutzer hat Zugriff auf Dateien und Unterordner im gemeinsamen Ordner und kann Änderungen vornehmen.
    • Nur Lesen: Der Benutzer hat Zugriff auf die Dateien und Unterordner im gemeinsamen Ordner.
    • Kein Zugriff: Der Benutzer hat keinen Zugriff auf die Dateien und Unterordner im gemeinsamen Ordner.
  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Hinweis:

  • Bei Namen für gemeinsame Ordner wird die Groß-/Kleinschreibung nicht beachtet und sie können 1-32 Zeichen umfassen. Der Name darf die folgenden Zeichen nicht enthalten:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Die folgenden Namen sind für die Systemnutzung reserviert:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • Das erste Zeichen eines gemeinsamen Ordners darf kein Minus- oder Leerzeichen sein, und das letzte Zeichen darf ebenfalls kein Leerzeichen sein.
  • Beachten Sie bei Berechtigungskonflikten folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur Lesen.
  • Wenn beim Erstellen eines neuen gemeinsamen Ordners die Berechtigungen für die Benutzer der Gruppe administrators auf Kein Zugriff gesetzt sind, können diese Benutzer den gemeinsamen Ordner nur unter Systemsteuerung > Gemeinsame Ordner anzeigen.
  • Sie können die Berechtigungseinstellungen des Papierkorb-Ordners der File Station konfigurieren, indem Sie mit der rechten Maustaste auf den Ordner #recycle klicken. Wählen Sie Eigenschaften, und klicken Sie auf die Registerkarte Berechtigungen.

Bearbeiten eines gemeinsamen Ordners:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Gemeinsamer Ordner.
  2. Wählen Sie auf der linken Seite den gewünschten gemeinsamen Ordner.
  3. Bearbeiten Sie die allgemeine Informationen des gemeinsamen Ordners, z. B. Name, Speicherort und Benutzerrechte.
  4. Falls Papierkorb aktivieren bereits markiert wurde, können Sie auf Papierkorb leeren klicken, um alle gelöschten Dateien im Papierkorb endgültig zu entfernen.
  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Löschen eines gemeinsamen Ordners:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Gemeinsamer Ordner.
  2. Wählen Sie auf der linken Seite den gewünschten gemeinsamen Ordner.
  3. Klicken Sie auf der linken Seite auf das Symbol Papierkorb, um den ausgewählten gemeinsamen Ordner zu löschen.

* Nur bei bestimmten Modellen verfügbar.
