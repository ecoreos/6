Benutzer

Mit den flexiblen Benutzerverwaltungsoptionen können Sie Benutzerkonten für einzelne Familienmitglieder oder Mitarbeiter eines Unternehmens erstellen, und für jede Person Zugangsberechtigungen für gemeinsame Ordner festlegen.

So erstellen Sie einen Benutzer:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Benutzer, und klicken Sie auf der linken Seite auf das Symbol +.
  2. Geben Sie einen Namen und ein Passwort für den Benutzer ein.
  3. Legen Sie die Gruppen fest, denen der neue Benutzer angehören soll.
  4. Weisen Sie dem neuen Benutzer entsprechende Berechtigungen für gemeinsame Ordner zu. Beachten Sie bei Berechtigungskonflikten folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur Lesen.
    • Vorschau: Diese Spalte zeigt die aktuellen Zugriffsberechtigungen für gemeinsame Ordner gemäß den aktuellen Einstellungen und Gruppen an, zu welchen der Benutzer gehört.
    • Lesen/Schreiben: Der Benutzer hat Zugriff auf Dateien und Unterordner im entsprechenden, gemeinsamen Ordner und kann Änderungen vornehmen.
    • Nur Lesen: Der Benutzer hat Zugriff auf die Dateien und Unterordner im entsprechenden, gemeinsamen Ordner.
    • Kein Zugriff: Der Benutzer hat keinen Zugriff auf die Dateien und Unterordner im entsprechenden, gemeinsamen Ordner.
  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

    6. Beschränkungen für Benutzername:

    • Bei dem Benutzernamen wird die Groß-/Kleinschreibung nicht berücksichtigt, und er kann aus 1 bis 64 Unicode-Zeichen mit Ausnahme der folgenden Symbole bestehen:
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • Das erste Zeichen darf kein Minus- oder Leerzeichen sein und das letzte Zeichen darf kein Leerzeichen sein.
    • Die folgenden Benutzernamen sind für das System reserviert:
      mailer-daemon, postmaster

    Kennworteinschränkungen:

    • Beim Kennwort wird die Groß-/Kleinschreibung berücksichtigt und es ist beschränkt auf bis zu 127 anzeigbare Zeichen einschließlich Buchstaben, Zahlen, Symbolen und Leerzeichen.

Bearbeiten eines Benutzers:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Benutzer.
  2. Wählen Sie auf der linken Seite den gewünschten Benutzer.
  3. Bearbeiten Sie die allgemeinen Informationen des Benutzers, z. B. Name, Passwort, Gruppe und Berechtigungen für gemeinsame Ordner.
  4. Markieren Sie Dieses Konto deaktivieren, wenn das Konto nicht mehr benutzt werden soll.
  5. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Einen Benutzer entfernen:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > Benutzer.
  2. Wählen Sie auf der linken Seite den gewünschten Benutzer.
  3. Klicken Sie auf das Symbol Papierkorb auf der linken Seite, um den ausgewählten Benutzer zu löschen.
