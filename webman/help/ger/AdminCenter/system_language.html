Sprache

Sie können Ihre bevorzugte Sprache für die Programmoberfläche von DSM, Google Chrome Desktop-Benachrichtigungen, E-Mail, SMS, Sofortnachricht-Benachrichtigungen und die Applikations-Codeseite auswählen.

Anzeigesprache

Sie können Ihre bevorzugte Anzeigesprache für DSM und Google Chrome Desktop-Benachrichtigungen auswählen oder die Standardeinstellung Ihres Browsers beibehalten.

Benachrichtigungssprache

Sie können Ihre bevorzugte Sprache für E-Mail, SMS, Windows Live Messenger und Skype Benachrichtigungen einstellen. Sie wird auch für Benachrichtigungen übernommen, die Sie auf Windows Phone empfangen.

Die Benachrichtigungssprache auf einem iOS oder Android Gerät hängt dagegen von der Spracheinstellung des Mobilgerätes ab.

Codeseite

Für Anwendungen, die nicht Unicode Strings zu Unicode umwandeln, wie z.b. der FTP Service, UPnP Support, Music Metadata Indexing, müssen Sie die passende Codeseite für diese Anwendungen aus dem Pull-Down-Menü auswählen.
