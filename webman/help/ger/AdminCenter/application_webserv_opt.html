HTTP-Dienst

Zusätzlich zur Standardportnummer 80 können Sie noch einen weiteren Port zur Verwendung von Photo Station und Web Station hinzufügen.

Hinzufügen einer zusätzlichen Portnummer für den HTTP-Zugriff:

  1. Markieren Sie Zusätzlichen HTTP-Port hinzufügen.
  2. Geben Sie die Portnummer im Feld ein. Sie können eine zusätzliche Portnummer von 1 bis 65535 für den HTTP Zugriff hinzufügen.

Die folgenden Ports dürfen nicht verwendet werden:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule Standard-Ports: 4662 (TCP), 4672(UDP) und FTP Standard-Portbereich: die tatsächliche Reichweite kann je nach Routermodell variieren.

    Hinweis:
  1. Zusätzlich kann die Portnummer des HTTP Zugriffs nicht die gleiche Portnummer wie die für den FTP/BT-Service sein.
  2. HTTP-Port und HTTPS-Port dürfen nicht miteinander in Konflikt stehen.
