Router-Port

Falls Internet-Dienste (wie Google Docs oder Pixlr) auf Ihren DiskStation zugreifen dürfen, können Sie den öffentlichen, externen Hostnamen und die Portnummer unter Systemsteuerung > Externer Zugriff > Erweitert angeben.

Standardmäßig ist der Hostname Ihre öffentliche IP-Adresse (oder der unter Systemsteuerung > Externer Zugriff > DDNS festgelegte Hostname) und die Portnummer ist 5000 (oder 5001 für HTTPS-Verbindungen). Falls Sie allerdings einen anderen Hostnamen besitzen, der nicht auf der DDNS-Seite konfiguriert ist, oder Sie beim Einrichten der Portweiterleitungsregeln abweichende Portnummern auf Ihrem Router festgelegt haben, müssen Sie den Hostnamen oder die Router-Portnummern eingeben, um sicherzustellen, dass Dateien, die Sie über Internet-Dienste geteilt haben, wirklich zugänglich sind.

So aktivieren Sie die Router-Portkonfiguration:

  1. Geben Sie den Hostnamen oder die öffentliche IP-Adresse von DiskStation in das Feld Hostname oder statische IP ein.
  2. Geben Sie die Router-Portnummer, die Sie bei der Einrichtung der Portweiterleitungsregel für DSM in Ihrem Router festgelegt haben, im Feld DSM(HTTP) oder DSM (HTTPS) ein.
  3. Klicken Sie auf Übernehmen.
