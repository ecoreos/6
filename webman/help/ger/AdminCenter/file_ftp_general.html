Allgemein

Unter Systemsteuerung > Dateidienste > FTP können Sie im Bereich Allgemein die Geschwindigkeitseinschränkung und die Protokollierung der Dateiübertragung aktivieren sowie Sicherheitseinstellungen und andere erweiterte Einstellungen vornehmen.

FTP-Dateiübertragungsprotokollierung aktivieren

Klicken Sie auf Erweiterte Einstellungen, um diese Funktion zu aktivieren und die Protokolle aller herunter- oder hochgeladenen Dateien aufzuzeichnen.

Hinweis:

Die Aktivierung des FTP-Dateiübertragungsprotokolls kann die Systemleistung beeinträchtigen.

Benutzer-Root-Verzeichnisse ändern

Klicken Sie auf Erweiterte Einstellungen und aktivieren Sie Benutzer-Root-Verzeichnisse ändern. Auf diese Weise können die ausgewählten Benutzer nur auf bestimmte Ordner zugreifen, wenn sie sich über FTP bei DSM anmelden.

Root-Verzeichnis-Änderungsregel erstellen:

  1. Aktivieren Sie unter Erweiterte Einstellungen Benutzer auswählen.
  2. Klicken Sie auf Hinzufügen.
  3. Wählen Sie in Benutzer oder Gruppe aus, auf wen diese Regel angewendet werden soll.
  4. Wählen Sie aus, welches Root-Verzeichnis zur Verfügung stehen soll, wenn sich der Benutzer anmeldet. Sie können wählen zwischen:
    • Benutzer-Home: Die Benutzer gelangen nach der Anmeldung in ihr Home-Verzeichnis.
    • Anderes Verzeichnis: Wählen Sie als Root-Verzeichnis einen freigegebenen Ordner aus und klicken Sie auf Auswählen. Beachten Sie, dass zuerst die Ordnerberechtigungen für den Benutzer oder die Gruppe überprüft werden sollten. Ein Benutzer oder eine Gruppe mit unzureichenden Ordnerberechtigungen kann sich nicht anmelden.
  5. Klicken Sie auf OK.
  6. Nachdem alle Regeln hinzugefügt wurden, klicken Sie auf Übernehmen, um die Regeln zu speichern.

Hinweis:

  • Die Priorität der Regeln richtet sich nach ihrer Position in der Liste.
  • Mittels Drag-and-Drop können Sie die Regeln in der Liste neu sortieren.

Aktivieren von anonymem FTP

Klicken Sie auf Erweiterte Einstellungen und aktivieren Sie die Option, damit sich alle beim FTP-Dienst des DiskStation anmelden können (mit dem Konto anonymous oder ftp).

Anonymous root ändern

Klicken Sie auf Erweiterte Einstellungen und aktivieren Sie das Kontrollkästchen, um das Anmeldeverzeichnis des anonymen FTP-Benutzers zu ändern. Wählen Sie einen freigegebenen Ordner aus dem Dropdown-Menü aus.

Anwenden von Standard-UNIX-Berechtigungen

Klicken Sie auf Erweiterte Einstellungen und aktivieren Sie die Option, um die Standard-UNIX-Berechtigungen beim Hochladen oder Erstellen von Dateien und Ordnern anzuwenden. Angewandte Berechtigungen sind identisch mit den Berechtigungen, die vom UNIX-Befehl umask angewendet werden. Wenn diese Option aktiviert ist, sind die UNIX-Berechtigungen für Dateien 644 und für Ordner 755. Wenn diese Option deaktiviert ist, sind die UNIX-Berechtigungen für Dateien 666 und für Ordner 777. Der Standardwert für „umask“ ist 022.

Hinweis:

  • Führen Sie für gemeinsame Ordner, für die Windows ACL aktiviert ist (alle gemeinsamen Ordner, außer „photo“ und Freigaben auf externen Festplatten), den Befehl chmod auf Ihrem Linux- oder FTP-Client aus, um die Ordner- und Dateiberechtigungstypen von Windows ACL auf UNIX zu ändern.
  • Die Aktivierung dieser Option kann zu inkonsistenten Berechtigungen zwischen verschiedenen Protokollen führen. Um Inkonsistenzen zu vermeiden, empfehlen wir, diese Option deaktiviert zu lassen.

Aktivieren einer Geschwindigkeitseinschränkung

Aktivieren Sie eine Geschwindigkeitseinschränkung, um die maximal nutzbare Bandbreite für Benutzer und Gruppen zu regeln, die Daten über FTP übertragen.

Geschwindigkeitseinschränkung aktivieren:

  1. Aktivieren Sie das Kontrollkästchen Geschwindigkeitseinschränkung aktivieren oder Geschwindigkeitseinschränkung nach einem Zeitplan aktivieren, wenn Sie nur zu bestimmten Zeiten eine Geschwindigkeitseinschränkung festlegen möchten. Dies ist eine serverweite Einstellung, die die Einstellungen der einzelnen Benutzer und Gruppen überschreibt. Die Auswahl von Keine Geschwindigkeitseinschränkung deaktiviert die Einstellungen für die Geschwindigkeitseinschränkung aller Benutzer und Gruppen.
  2. Klicken Sie auf Einstellungen für Geschwindigkeitseinschränkung, um die Einstellungen für einzelne Benutzer oder Gruppen zu ändern. Für jeden Benutzer und jede Gruppe steht Folgendes zur Auswahl:
    • Gruppeneinstellungen anwenden: Die Einstellungen für Geschwindigkeitseinschränkung sind identisch mit der Gruppe, zu der der Benutzer gehört.
    • Ohne: Es wird keine Geschwindigkeitseinschränkung angewendet, wenn Gruppenmitglieder Dateien mit dem Dienst übertragen.
    • Konstant: Für alle Benutzer und Gruppenmitglieder gilt eine feste, konstante Geschwindigkeitseinschränkung. Upload- und Download-Geschwindigkeitseinschränkungen können in den Feldern links angegeben werden.
    • Angepasst: Für Benutzer oder Gruppenmitglieder können zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan festgelegt werden. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für die Geschwindigkeitseinschränkung und einen Zeitplan festzulegen.
  3. Klicken Sie auf Übernehmen.

Hinweis:

  • Weitere Informationen über Geschwindigkeitseinschränkungen für Gruppen finden Sie hier.
  • Beim Festlegen einer Geschwindigkeitseinschränkung für einen Benutzer zeigt der Bereich Ergebnis die tatsächliche Geschwindigkeitseinschränkung des Benutzers gemäß seiner individuellen und der Gruppeneinstellungen an. Wenn etwa der Benutzer „Achilles“ eine Download-Einschränkung von 100 KB/s hat, aber zu einer Benutzergruppe mit einer Download-Einschränkung von 500 KB/s gehört, wird im Bereich Ergebnisse „0/500“ angezeigt.
