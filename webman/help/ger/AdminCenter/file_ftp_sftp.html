SFTP

SFTP ist ein Dateiübertragungsprotokoll, das als Erweiterung im Secure Shell (SSH) Protokoll integriert ist. SFTP benötigt nur eine TCP Port Nummer. Zusätzlich können private und öffentliche Schlüssel verwendet werden, um Benutzer ohne Kennwörter zu authentifizieren. Die Übertragungsgeschwindigkeiten sind allerdings geringer und wegen der Verschlüsselung werden mehr Systemressourcen benötigt.

SFTP-Service aktivieren:

  1. Klicken Sie auf SFTP-Dienst aktivieren.
  2. Klicken Sie auf Übernehmen.

Portnummer

Richten Sie die Portnummer des SFTP Dienstes ein.

Hinweis:

Die Standard-Portnummer für den SFTP-Dienst ist 22. Sie können sie von 1 bis 65535 einstellen, ausgenommen die folgenden Nummern:
20, 21, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002 und eMule Standard-Ports: 4662 (TCP), 4672 (UDP)
