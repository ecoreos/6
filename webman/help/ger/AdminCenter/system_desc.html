System

In diesem Abschnitt können Sie die Systemeinstellungen für den DiskStation verwalten. Sie können den Anmeldebildschirm anpassen, die Zeitzone und Spracheinstellungen ändern, einen Zeitplan für das automatische Hoch-/Herunterfahren festlegen, externe Geräte verwalten, die DSM-Version aktualisieren oder das Gerät auf die Werkseinstellungen zurücksetzen.
