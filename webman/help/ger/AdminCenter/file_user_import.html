Benutzer importieren

Sie können Benutzerkonten in einem Paket erstellen, indem Sie Benutzerlisten importieren.

Benutzerliste erstellen:

Methode 1. Mit einem Texteditor:

  1. Öffnen Sie mit einem Texteditor eine neue Datei.
  2. Geben Sie die Informationen für einen Benutzer in der genannten Reihenfolge ein und trennen Sie diese durch einen TAB:
    • Benutzername
    • Kennwort
    • Beschreibung
    • E-Mail
    • Quote (MB) auf Volume 1 (wiederholen Sie diesen Schritt, wenn Sie mehr als ein Volume haben)

      Hinweis:

      Benutzerquote wird in Btrfs-Dateisystemen und auf externen Speichergeräten (z. B. USB-/eSATA-Laufwerke) nicht unterstützt.

  3. Gehen Sie zur nächsten Zeile und wiederholen Sie Schritt 2, bis Sie alle Benutzer eingegeben haben. Jede Zeile enthält die Informationen eines Benutzers.
  4. Speichern Sie die Datei als UTF-8 Kodierung.

Methode 2. Mit Microsoft Excel:

  1. Öffnen Sie in Excel eine neue Datei.
  2. Geben Sie die Informationen für einen Benutzer in der genannten Reihenfolge ein und trennen Sie diese durch Spalten in einer einzigen Zeile:
    • Benutzername
    • Kennwort
    • Beschreibung
    • E-Mail
    • Quote (MB) auf Volume 1 (wiederholen Sie diesen Schritt, wenn Sie mehr als ein Volume haben)

      Hinweis:

      Benutzerquote wird in Btrfs-Dateisystemen und auf externen Speichergeräten (z. B. USB-/eSATA-Laufwerke) nicht unterstützt.

  3. Gehen Sie zur nächsten Zeile und wiederholen Sie Schritt 2, bis Sie alle Benutzer eingegeben haben. Jede Zeile enthält die Informationen eines Benutzers.
  4. Öffnen Sie mit dem Texteditor eine neue Datei. Kopieren Sie die Einträge des Excel-Arbeitsblatts und fügen Sie diese in die Textdatei ein.
  5. Speichern Sie die Datei als UTF-8 Kodierung.

Hinweis:

  • Alle Felder mit Ausnahme des Benutzernamens sind optional.
  • Ein Beispiel einer Textdatei sehen Sie nachfolgend:
    • rosy 1234 Mutter rosy@abc.com 80000
    chaddy 5678 Bruder chaddy@abc.com 80000
    danny 8901 Cousin danny@abc.com 20000
    anny 2345 Schwester anny@abc.com 20000

Benutzerliste importieren:

  1. Wählen Sie Erstellen > Benutzer importieren aus.
  2. Aktivieren Sie Doppelte Konten überschreiben, wenn Sie die bestehenden doppelten Konten durch die der Benutzerliste ersetzen möchten.
  3. Klicken Sie auf Durchsuchen und wählen Sie die Datei aus der Benutzerliste aus. Nachdem Sie die Datei ausgewählt haben, zeigt der Vorschaubereich die Liste der importierten Benutzer an. Benutzer mit einem abweichenden Status werden nicht importiert.
  4. Klicken Sie auf Hochladen, um die Benutzerkonten zu erstellen.

Hinweis:

Die Datei der Benutzerliste muss das Format UTF-8 haben.
