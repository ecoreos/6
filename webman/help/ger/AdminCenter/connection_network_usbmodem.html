Mobil*

Mit einem mobilen Breitband-Dongle kann der DiskStation sich über ein drahtloses 3G/LTE-Netzwerk mit dem Internet verbinden. Über die Einstellungen unter Systemsteuerung > Netzwerk > Netzwerkschnittstelle > Mobil können Sie eine drahtlose Internet-Verbindung über Ihren mobilen Breitband-Dongle herstellen und anpassen.

Hinweis:

Die vollständige Liste der mobilen Breitband-Dongles, die von Synology Produkten unterstützt werden, finden Sie unter www.synology.com.

Mobiles Netzwerk einrichten:

Wenn ein mobiler Breitband-Dongle mit einem der USB-Ports auf Ihrem DiskStation verbunden ist, versucht DiskStation automatisch, die Verbindung herzustellen. Falls die Verbindung nicht automatisch hergestellt werden kann oder die Verbindungseinstellungen geändert werden sollen, müssen Sie Folgendes beachten:

  • SIM entsperren/Einstellungen: Wenn die SIM-Karte durch eine PIN geschützt ist, muss sie entsperrt werden, damit die DiskStation die Verbindung zum Netzwerk herstellen kann. Nach dem Entsperren enthält dieser Bereich Optionen zur Änderung der PIN-Einstellungen.
  • Bearbeiten: Sie können die Verbindung manuell konfigurieren oder nach Standort und Trägerfrequenz auswählen. Das manuelle Einrichten der Verbindung erfordert die Eingabe von APN, Telefonnummer, Benutzername und Passwort. Wenden Sie sich an Ihren Mobilfunkanbieter, falls Sie diese Informationen nicht besitzen.
  • Verbinden/Trennen: Mit diesen Schaltflächen können Sie die Verbindung von DiskStation zum 3G/LTE-Netzwerk herstellen oder trennen.
  • Optionen: Dieser Abschnitt enthält eine Option, um die mobile Breitband-Dongle-Schnittstelle als Standard-Gateway für Ihre DiskStation festzulegen. Wenn Sie das Mobilfunknetz als Standard-Gateway konfigurieren, wird der Internetverkehr von Ihrem DiskStation über das Mobilfunknetz anstelle über das Kabelnetz geleitet.

Hinweis:

  1. Einige mobile Breitband-Dongles zeigen die Signalstärke nicht an, wenn sie mit dem Internet verbunden sind, wenn diese Funktion nicht unterstützt wird.
  2. Bei einigen mobilen Breitband-Dongles können Sie die SIM-Einstellungen nicht ändern, wenn sie mit dem Internet verbunden sind.
