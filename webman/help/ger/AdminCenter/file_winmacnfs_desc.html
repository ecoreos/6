Win/Mac/NFS

Windows-, Mac- und Linux-Benutzer können über das Netzwerk auf DiskStation zugreifen, sobald die Optionen unter Systemsteuerung > Dateidienste > Win/Mac/NFS aktiviert wurden.

Die folgenden gemeinsamen Datei-Protokolle werden unterstützt:

  • Für Windows: SMB/CIFS (My Network Places), FTP, WebDAV
  • Für Mac: SMB, FTP, AFP, WebDAV
  • Für Linux: SMB, FTP, NFS, WebDAV
