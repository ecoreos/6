Benutzer

Mit den flexiblen Benutzerverwaltungsoptionen können Sie Benutzerkonten für Familienmitglieder oder Mitarbeiter eines Unternehmens erstellen. Durch das Erstellen von Benutzern können Berechtigungen für jeden festgelegt und verwaltet werden, beispielsweise Zugangsberechtigungen für gemeinsame Ordner, Quoten für Speicher und gemeinsame Ordner oder Bandbreitenbeschränkungen.
