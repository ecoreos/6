DSM-Einstellungen

Unter Systemsteuerung > Netzwerk > DSM-Einstellungen können die von den Clients verwendeten Portnummern für den Zugriff auf Ihre DiskStation geändert werden.

So ändern Sie die Standardportnummer:

Sowohl HTTP- wie auch HTTPS-Verbindungen sind in DSM immer aktiviert. Der Standard-HTTP-Port ist 5000 und der Standard-HTTPS-Port ist 5001.

  1. Geben Sie die Portnummer im Feld HTTP oder HTTPS ein.
  2. Klicken Sie auf Übernehmen.

HTTP-Verbindungen zu HTTPS umleiten:

Sie können Clients, die versuchen, über den HTTP-Port auf Ihre DiskStation zuzugreifen, zum HTTPS-Port umleiten.

  1. Wählen Sie HTTP-Verbindungen automatisch zu HTTPS umleiten (Web Station und Photo Station ausgeschlossen) aus.
  2. Klicken Sie auf Übernehmen.

Ports, die für die HTTPS-Verbindung verwendet werden:

  • DSM: Die Standardportnummer ist 5001, damit Sie über die folgende Adresse sicher auf den DSM zugreifen können: https://server-hostname:5001/
  • Web Station und Photo Station: Die Standardportnummer ist 443, sodass Sie über die folgende Adresse sicher auf Web Station und Photo Station zugreifen können: https://server-hostname/

Die DiskStation im Netzwerk finden:

Sie können Ihre DiskStation über die Windows-Netzwerkerkennung oder die Bonjour-Diensterkennung im Netzwerk finden.

  1. Wählen Sie einen Dienst aus, um Ihre DiskStation im Netzwerk zu finden:
    • Windows-Netzwerkerkennung aktivieren
    • Bonjour-Diensterkennung aktivieren
  2. Klicken Sie auf Übernehmen.

Domain

Unter Domain können Sie einen benutzerdefinierten Domainnamen für den Zugriff auf DiskStation angeben.

Benutzerdefinierte Domain aktivieren:

  1. Markieren Sie Benutzerdefinierte Domain aktivieren.
  2. Geben Sie einen Domainnamen an, der für DiskStation angewendet werden soll.
  3. Sie können auch HSTS aktivieren markieren, um auf dieser Domain HTTPS-verschlüsselte Verbindungen zu erzwingen.
  4. Klicken Sie auf Übernehmen.

Hinweis:

  • Jede Domain kann nur für jeweils eine bestimmte Anwendung auf Ihrer DiskStation verwendet werden.
  • Sofern der Domainname nicht länger als 15 Zeichen ist und den NetBIOS-Namenskonventionen entspricht, wird das Systen den Domainnamen (z. B. MyDSM) automatisch registrieren und übertragen. Windows-Benutzer können mit dieser URL auf den DSM-Desktop im selben lokalen Netzwerk zugreifen: http://MyDSM oder https://MyDSM
  • Der Zugriff auf Anwendungen über URLs, die Domainnamen verwenden, erfolgt über den Standard-HTTP-Port (80) oder den Standard-HTTPS-Port (443).
