Sicherheitseinstellungen

Ändern Sie das root-Konto der ausgewählten Benutzer auf home.

Die hier ausgewählten Benutzer können nur auf ihre Home-Ordner zugreifen, wenn die Anmeldung Synology DiskStation über FTP erfolgt.

Anonymous FTP aktivieren

Aktiviert diese Funktion, so dass sich jeder im FTP-Service der DiskStation (mit dem Anonymous oder dem FTP Konto) anmelden kann.

Das Privileg der anonymem Benutzer konfigurieren:

Gehen Sie auf Hauptmenü > Systemsteuerung > Gemeinsamer Ordner, und klicken Sie Privilegieneinstellung.

Anonymous root ändern

Aktivieren Sie diese Funktion, um das Anmeldeverzeichnis für die anonymen FTP-Benutzer zu ändern Wählen Sie den gemeinsamen Ordner aus dem Drop-Down-Menü aus.
