Benachrichtigungen

Sie können den DiskStation so einstellen, dass Sie Benachrichtigungen per E-Mail, Mobilgerät, Windows Live Messenger oder Skype erhalten, wenn es auf dem System zu Statusänderungen oder Fehlern kommt, z. B.: Unterbrochene Verbindung, verändertes Verhalten des Lüfters, Stromausfall, zur Neige gehender Speicherplatz auf Volume oder Fehler bei Sicherung und Wiederherstellung usw.

E-Mail

Über diese Option können Sie den E-Mail-Benachrichtigungsdienst verwalten.

Mobilgeräte, Windows Live Messenger oder Skype

Über diese Option können Sie den Benachrichtigungsdienst für Mobilgeräte, Windows Live Messenger oder Skype verwalten.
