Anwendungen

In diesem Abschnitt können Sie die allgemeinen Einstellungen für von Synology entwickelte Anwendungen, wie Zugriffsrechte der Benutzer und Verbindungseinstellungen für bestimmte Anwendungen, Medien-Indizierung für die Verwendung mit Multimedia-Anwendungen verwalten. Darüber hinaus erlauben umfangreiche Tools für Web-Dienste und Optionen das einfache Hosten einer Website auf Ihrer DiskStation.
