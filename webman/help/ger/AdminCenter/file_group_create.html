Eine Gruppe erstellen

  1. Gehen Sie zu Systemsteuerung > Gruppe.
  2. Klicken Sie auf Erstellen. Dies startet den Assistent Benutzererstellung.
  3. Geben Sie den Gruppennamen und die Beschreibung ein. (Weitere Informationen finden Sie im Abschnitt unten). Klicken Sie auf Weiter.
  4. Folgen Sie den restlichen Schritten des Assistenten, um das Erstellen der Gruppe abzuschließen.
  5. Klicken Sie auf Übernehmen, um den Vorgang zu beenden.

Namenskonventionen:

  • Bei dem Gruppennamen wird die Groß-/Kleinschreibung berücksichtigt und er kann aus bis zu 32 Unicode-Zeichen einschließlich der folgenden Symbole bestehen:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • Das erste Zeichen darf kein Minus- oder Leerzeichen sein und das letzte Zeichen darf kein Leerzeichen sein.
  • Bei der Gruppenbeschreibung wird die Groß-/Kleinschreibung berücksichtigt und sie ist auf bis zu 64 anzeigbare Unicode-Zeichen beschränkt.
