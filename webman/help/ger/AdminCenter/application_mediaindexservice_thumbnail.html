Konvertierungseinstellungen

Wenn Sie Fotos oder Videos zur Verwendung mit Medienserver und Photo Station hochladen, werden Miniaturansichten erstellt, die die Navigation erleichtern. Hier können Sie die Qualität der Miniaturansichten einstellen und die Konvertierung verfolgen.

Wenn Sie Videos im gemeinsamen Ordner photo der Photo Station auf Ihren mobilen Geräten anzeigen möchten, können Sie die Videokonvertierung aktivieren und den Fortschritt verfolgen.

So stellen Sie die Qualität der Miniaturansichten ein:

  1. Wählen Sie die gewünschte Qualität im Drop-down-Menü aus.
  2. Klicken Sie auf Übernehmen.
Hinweis:
  • Die Erstellung von hochwertigen Miniaturansichten ist zeitaufwändiger.

So aktivieren Sie die Videokonvertierung für Mobilgeräte:

  1. Markieren Sie Videokonvertierung für Mobilgeräte aktivieren.
  2. Klicken Sie auf Übernehmen und Sie werden in der Lage sein, Videos auf Ihren Mobilgeräten unterwegs anzuschauen.
Hinweis:
  • Sie können nur Videos, welche zum gemeinsamen Ordner photo auf der Photo Station hochgeladen wurden, in das Format konvertieren, das von Ihren Mobilgeräten unterstützt wird.
  • MP4 Dateien werden nicht konvertiert.

So zeigen Sie den Vorgang der Foto-/Videokonvertierung an:

  1. Wenn Fotos oder Videos auf DiskStation hochgeladen werden, klicken Sie auf das Symbol Konvertierungsvorgang oben rechts in der Taskleiste.
  2. Gehen Sie im angezeigten Fenster folgendermaßen vor:
    • Überwachen Sie unter Konvertierungsvorgang den Fortschritt und den Status der im Hintergrund erstellten Miniaturansichten.
    • Gehen Sie folgendermaßen vor, um den Konvertierungsvorgang anzuhalten:
      1. Wählen Sie im Drop-down-Menü Verzögerung um [Zahl] Stunde(n) und klicken Sie auf Pause, um den Vorgang anzuhalten. (Standardmäßig wird Verzögerung um 1 Stunde(n) vorgegeben.)
      2. Der Konvertierungsvorgang wird um die angegebene Verzögerungszeit verzögert. Um den Vorgang sofort fortzusetzen, können Sie auf Fortsetzen klicken.
      Hinweis:
      • Da der Konvertierungsvorgang die CPU stark auslastet, können durch eine Unterbrechung des Vorgangs Systemressourcen für andere laufende Aufgaben freigegeben werden.
  3. Wenn die Konvertierung abgeschlossen ist, wird das Symbol in der Taskleiste nicht mehr angezeigt.
Hinweis:
    Da Photo Station Uploader eine bessere Leistung bei der Erstellung von Miniaturansichten bietet, empfehlen wir die Verwendung von Photo Station Uploader, wenn Sie eine große Anzahl von Fotos/Videos in Ihre Photo Station-Bibliothek hochladen möchten.
