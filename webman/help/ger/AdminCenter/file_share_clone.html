Freigegebenen Ordner klonen*

Sie können einen freigegebenen Ordner auf Btrfs-Volumes klonen, um nahezu umgehend Kopien eines bestehenden freigegebenen Ordners zu erstellen. Das Klonen eines freigegebenen Ordners verbraucht zum Zeitpunkt der Erstellung auch nur einen Bruchteil des Speichers.

Freigegebenen Ordner klonen:

  1. Gehen Sie zu Systemsteuerung > Freigegebener Ordner.
  2. Wählen Sie den gewünschten freigegebenen Ordner im Btrfs-Dateisystem aus.
  3. Klicken Sie auf Erstellen > Klonen.
  4. Geben Sie einen Namen und eine Beschreibung für den neuen freigegebenen Ordner an und ändern Sie weitere Optionen, soweit erforderlich. (Nähere Informationen zu diesen Optionen finden Sie unter Freigegebenen Ordner erstellen.)

Hinweis:

  • Es können nur freigegebener Ordner auf Btrfs-Volumes geklont werden.
  • Der geklonte freigegebene Ordner wird auf demselben Volume erstellt wie der ursprüngliche freigegebene Ordner.
  • Die Einstellungen des erweiterten Datenintegritätsschutzes werden auf den geklonten freigegebenen Ordner angewendet und können nicht geändert werden.

____
*Nur bei bestimmten Modellen verfügbar.
