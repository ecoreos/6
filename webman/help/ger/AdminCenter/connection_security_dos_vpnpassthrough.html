Schutz

Systemsteuerung > Sicherheit > Schutz bietet Optionen zur Erweiterung der Netzwerksicherheit für die einzelnen Netzwerkschnittstellen.

DoS-Schutz

Der Denial-of-service (DoS)-Schutz hilft bei der Vermeidung von bösartigen Angriffen über das Internet.

DoS-Schutz aktivieren:

  1. Markieren Sie das Kästchen DoS-Schutz aktivieren.
  2. Klicken Sie auf Übernehmen.

VPN Pass-through*

Wenn DiskStation als Router fungiert, können Clients über VPN Pass-through im Netzwerk den DiskStation-NAT durchleiten sowie ein- oder ausgehende VPN-Verbindungen aufbauen. Wenn die interne Konfiguration einen VPN Server umfasst, gehen Sie zu Systemsteuerung > Externer Zugriff > Routerkonfiguration, um die Portweiterleitungsregeln einzurichten und zuzulassen, dass der VPN Server VPN-Verbindungen herstellt.

VPN Pass-through-Einstellungen bearbeiten:

  1. Hier können die folgenden Einstellungen aktiviert oder deaktiviert werden:
    • PPTP Pass-through: Ermöglicht den Pass-through von PPTP-Verkehr (Point-to-Point Tunneling Protocol) und den Aufbau von Verbindungen.
    • IPSec Pass-through: Ermöglicht den Pass-through von IPSec-Verkehr (IP Security Protocol) und den Aufbau von Verbindungen.
    • L2TP Pass-through: Ermöglicht den Pass-through von L2TP-Verkehr (Layer 2 Tunneling Protocol) und den Aufbau von Verbindungen.
  2. Klicken Sie nach dem Ändern der Einstellungen auf Übernehmen.

Hinweis:

Wenn Ihr DiskStation eine Verbindung mit dem Internet über PPPoE herstellt, müssen der damit verbundene DoS-Schutz oder die VPN-Pass-through-Regeln auf der entsprechenden PPPoE-Schnittstelle eingerichtet werden.

____
*Nur bei bestimmten Modellen verfügbar.
