Externe Geräte

Gehen Sie zu Systemsteuerung > Externe Geräte > Externe Geräte, um eine Liste von externen Speichergeräten anzuzeigen, die mit dem DiskStation verbunden sind.

Externe Festplatten

Durch Anschließen einer externen Festplatte an das System können Sie deren Kapazität über einen vom System erstellten Ordner mit dem Namen usbshare[Nummer] (bei USB-Laufwerken) gemeinsam nutzen. Der gemeinsame Ordner wird automatisch entfernt, wenn die externe Festplatte vom System getrennt wird.

Durch Anschließen einer externen Festplatte an das System können Sie deren Kapazität über einen vom System erstellten Ordner mit dem Namen usbshare[number] (für USB-Festplatten) oder satashare[number] (für eSATA-Festplatten) gemeinsam nutzen. Der gemeinsame Ordner wird automatisch entfernt, wenn die externe Festplatte vom System getrennt wird.

Der DiskStation erkennt die folgenden Formate: Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus und NTFS. Eine nicht erkannte externe Festplatte muss zuerst formatiert werden, bevor sie im System verwendet werden kann.

Hinweis:

  1. Einige Modelle unterstützen HFS/HFS Plus nur schreibgeschützt.
  2. Journal wird auf HFS/HFS Plus nicht unterstützt.
  3. Vergewissern Sie sich, dass Sie die externe Festplatte ausgeworfen haben, bevor Sie die den Stecker abziehen.

Festplatte formatieren:

Klicken Sie auf Formatieren und folgen Sie dem Assistenten. Sie können externe Festplatten für den DiskStation in den Formaten ext4 und FAT32 formatieren.

Hinweis:

  1. Vergewissern Sie sich, dass Sie Ihre Daten gesichert haben, bevor Sie mit dem Formatieren der externen Festplatte beginnen. Dabei werden alle Daten gelöscht und können nicht wiederhergestellt werden.
  2. Um die externe Festplatte im Format ext3 oder ext4 zu formatieren, muss sie größer als 1 GB sein.
  3. Wenn die Festplatte als ext3 oder ext4 formatiert ist, wird sie nur vom DiskStation erkannt. Wenn Sie Daten direkt von Ihrer externen Festplatte aus mit einem PC oder Mac lesen möchten, müssen Sie das FAT32-Format auswählen.

S.M.A.R.T.-Test durchführen:

Der S.M.A.R.T.-Test untersucht und meldet den Status Ihrer Festplatten und kann so auf eine frühzeitige Warnung vor drohendem Festplattenausfall hinweisen. Wir empfehlen den Austausch der Festplatte, wenn Fehler erkannt werden.
Um den S.M.A.R.T.-Test durchzuführen, klicken Sie einfach auf die Schaltfläche S.M.A.R.T.-Test und folgen dem Assistenten.
Um die Testergebnisse zu überprüfen, klicken Sie einfach auf S.M.A.R.T.-Info.

Hinweis:

Die Testdauer hängt von der Systemumgebung ab.

Einschränkungen

Das Formatieren der Festplatte ist in den folgenden Situationen nicht verfügbar:

  • Die externe Festplatte wird gerade initialisiert, d. h. das externe Gerät wurde gerade erst angeschlossen.
  • Die Kapazität der externen Festplatte beträgt weniger als 1 GB (Format ext3 oder ext4).
  • Die externe Festplatte wird formatiert.
  • Das System sichert gerade Dateien auf der externen Festplatte.
  • Die externe Festplatte ist beschädigt.

Fehlerbehebung

  1. Die externe Festplatte kann nicht formatiert werden. Warum?
    • Prüfen Sie, ob die externe Festplatte ordnungsgemäß eingesteckt ist.
    • Prüfen Sie, ob die externe Festplatte schreibgeschützt ist. Wenn ja, entfernen Sie den Schreibschutz vor der Formatierung.
  2. Das Dateisystem der externen Festplatte wird als Nicht unterstützt angezeigt oder das System erstellt die gemeinsamen Ordner usbshare[number] oder satashare[number] für die externe Festplatte nicht. Was soll ich tun?
    • Das aktuelle System erkennt die Formate Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus und NTFS. Externen Festplatten, die nicht mit den vorgenannten Formaten formatiert wurden, werden nicht erkannt und der Status Nicht unterstützt wird angezeigt.
  3. Ich kann nicht auf den gemeinsamen Ordner usbshare[number] oder satashare[number] zugreifen. Was soll ich tun?
    • Dies könnte durch ein Problem im Dateisystem auf der externen Festplatte verursacht werden. Um dies zu beheben, müssen Sie eine externe Festplatte an einen PC anschließen und eine Dateisystemprüfung durchführen.
  4. Kann meine externe Festplatte mit ext3- oder ext4-Format direkt mit meinem PC und Mac verbunden werden und kann direkt auf sie zugegriffen werden?
    • Nein, ext3 oder ext4 werden nur vom DiskStation erkannt. Wenn Sie Daten direkt von Ihrer externen Festplatte aus mit einem PC oder Mac lesen möchten, müssen Sie das FAT32-Format auswählen.
  5. Warum sehe ich auf meiner externen Festplatte einen Ordner mit dem Namen „lost+found“?
    • Dies ist ein Standard-Systemordner, der automatisch bei der Verwendung des ext3/ext4-Dateisystems erstellt wird. Es ist ganz normal, dass Sie den Ordner auf der externen Festplatte sehen. Auch wenn Administratoren die Berechtigung haben, auf den Inhalt zuzugreifen und ihn zu verwalten, wird dringend empfohlen, diesen Ordner unangetastet zu lassen.

Standardberechtigungen für alle externen Geräte einstellen:

Sie können Standardberechtigungen festlegen, indem Sie auf Standardberechtigungen klicken und angeben, welche Benutzer oder Gruppen externe Geräte und ihre Inhalte anzeigen, ändern oder darauf zugreifen können. Wenn ein externes Gerät in den DiskStation eingesetzt wird, werden die von Ihnen festgelegten Berechtigungen automatisch auch dem externen Gerät zugewiesen.

Gehen Sie im Fenster „Berechtigungen“ wie folgt vor, um Standardberechtigungen festzulegen:

  1. Wählen Sie eine der folgenden Optionen aus dem Dropdown-Menü aus:
    • Interner Systembenutzer: Weist Berechtigungen für Standard-Systembenutzer zu, z. B. dem anonymen FTP/WebDAV-Benutzer.
    • Lokale Benutzer: Weist Berechtigungen für lokale Benutzer (einschließlich guest) zu.
    • Lokale Gruppen:Weist Berechtigungen für lokale Gruppen zu.
  2. Aktivieren oder deaktivieren Sie die entsprechenden Felder für jeden Benutzer oder jede Gruppe, um ihre Berechtigungen für externe Geräte anzupassen:
    • Lesen/Schreiben: Der Benutzer oder die Gruppe hat Zugriff auf Dateien und Unterordner auf den externen Geräten und kann Änderungen an Dateien und Ordnern vornehmen.
    • Nur Lesen: Der Benutzer oder die Gruppe kann auf die Dateien und Ordner auf den externen Geräten zugreifen.
    • Kein Zugriff: Der Benutzer oder die Gruppe kann nicht auf die Dateien und Ordner auf den externen Geräten zugreifen.
