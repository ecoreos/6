Drucker

DiskStation kann als Druckserver fungieren, sodass Client-Computer oder Mobilgeräte auf Drucker zugreifen können, die an DiskStation angeschlossen sind. Zusätzlich ermöglicht die Unterstützung von Apple Wireless Printing und Google Cloud Print das Drucken per Fernzugriff mit iOS-Geräten oder Google-Produkten und -Diensten.

Kompatible Drucker finden Sie unter www.synology.com.

Einrichten von Druckern

Befolgen Sie die nachstehenden Anweisungen, um einen USB- oder Netzwerkdrucker für den DiskStation einzurichten und mit ihm zu verbinden.

USB-Drucker einrichten:

  1. Schalten Sie den Drucker ein und schließen Sie ihn an den USB-Port von DiskStation an.
  2. Gehen Sie zu Systemsteuerung > Externe Geräte > Drucker.
  3. Wählen Sie den Drucker aus, den Sie einrichten möchten. Beachten Sie, dass DiskStation keine Drucker erkennen kann, die über USB-Hubs verbunden sind.
  4. Klicken Sie auf Druckermanager und wählen Sie Drucker einrichten aus.
  5. Wählen Sie einen der folgenden Modi im Dialogfenster aus:
    • Netzwerk MFP: Ermöglicht Clients innerhalb des lokalen Netzwerks, die Druck-, Scan- und Faxfunktionen Ihres Multifunktionsdruckers zu verwenden. Die Scan- und Faxfunktionen sind nur für Windows-Computer verfügbar.
    • Netzwerkdrucker: Wählen Sie diesen Modus für Drucker aus, die nur das Drucken unterstützen, oder wenn Sie den Clients nur die Verwendung der Druckfunktion erlauben möchten.
  6. Konfigurieren Sie die Einstellungen von Apple Wireless Printing oder Google Cloud Print (optional).
  7. Klicken Sie auf Speichern.

Hinweis:

Auf ein Netzwerk-MFP-Gerät kann jeweils nur von einem Client aus gleichzeitig zugegriffen werden.

Netzwerkdrucker einrichten:

  1. Schalten Sie den Netzwerkdrucker ein und verbinden Sie ihn mit demselben Netzwerk wie DiskStation.
  2. Gehen Sie zu Systemsteuerung > Externe Geräte > Drucker.
  3. Klicken Sie auf Netzwerkdrucker hinzufügen und folgen Sie den Anweisungen des Einrichtungsassistenten.

Hinweis:

  • Druckername: Ein Druckername muss während der Einrichtung eingegeben werden und bis zu 32 Zeichen umfassen, einschließlich Buchstaben, Zahlen und Unterstriche.
  • Protokoll: Im Allgemeinen verwenden HP-Netzwerkdrucker Socket, Canon-Netzwerkdrucker verwenden BJNP und andere verwenden meist LPR.
  • Warteschlangenname: Die Eingabe eines Warteschlangennamens ist für LPR- und IPP-Netzwerkdrucker erforderlich, aber nicht für Socket- oder BJNP-Drucker.
  • Kein Netzwerk-MFP: An DiskStation angeschlossene Netzwerkdrucker können gegenwärtig den Modus "Netzwerk MFP" nicht nutzen.
  • Keine USB-Drucker: Von einem Synology-Produkt freigegebene USB-Drucker können nicht als Netzwerkdrucker eingerichtet werden.

Verwalten von Druckern

Die Registerkarte "Drucker" enthält die folgenden Optionen zur Verwaltung von USB- oder Netzwerkdruckern, die mit dem DiskStation verbunden sind.

  • Info abrufen: Zeigt Informationen zum ausgewählten Drucker an, wie Gerätename, Hersteller, Status, usw.
  • Entfernen: Entfernt und trennt einen Drucker von DiskStation.
  • Drucker-Manager: Enthält die folgenden Optionen:
    • Drucker einrichten: Geben Sie die Druckereinstellungen wie etwa Apple Wireless Printing oder Google Cloud Print ein (zur Anleitung siehe unten).
    • Spool-Platz bereinigen: Verwirft sämtliche Druckaufgaben auf allen Druckern.
    • Testseite drucken: Druckt eine Testseite, um sicherzustellen, dass die Einstellungen korrekt sind.
    • Drucker freigeben: Trennt den Drucker und den Client (nur für Modus "Netzwerk MFP"). Wenn sich ein Client mit einem Multifunktionsdrucker für den Empfang von Faxen oder zum Scannen von Dateien verbindet, könnte dieser Vorgang etwas dauern. Die Verbindung kann bei Bedarf gestoppt werden, damit der nächste Client in der Schlange auf den Drucker zugreifen kann.

Apple Wireless Printing aktivieren:

  1. Gehen Sie zu Systemsteuerung > Externe Geräte > Drucker.
  2. Wählen Sie einen Drucker aus der Geräteliste aus.
  3. Klicken Sie auf Druckermanager > Drucker einrichten.
  4. Aktivieren Sie das Kontrollkästchen Apple Wireless Printing aktivieren.
  5. Wählen Sie die Marke und den Treiber Ihres Druckers aus.
  6. Klicken Sie auf Speichern.

Nach Abschluss der Einrichtung können Sie eine Testseite drucken, indem Sie auf Druckermanager > Testseite drucken klicken.

Hinweis:

  • Die Aktivierung von Apple Wireless Printing aktiviert auch Bonjour Printer Broadcast für DSM.
  • Apple Wireless Printing ist für iOS 4.2 oder höher verfügbar.

Google Cloud Print aktivieren:

  1. Gehen Sie zu Systemsteuerung > Externe Geräte > Drucker.
  2. Wählen Sie einen Drucker aus der Geräteliste aus.
  3. Klicken Sie auf Druckermanager > Drucker einrichten.
  4. Markieren Sie das Kontrollkästchen Google Cloud Print aktivieren.
  5. Geben Sie die folgenden Details ein:
    • Ihre Google-Kontoinformationen
    • Die Marke und den Treiber Ihres Druckers
  6. Klicken Sie auf Speichern.

Nach Abschluss der Einrichtung können Sie eine Testseite drucken, indem Sie auf Druckermanager > Testseite drucken klicken.

Um Google Cloud Print-Aufgaben zu verwalten, klicken Sie auf den Link Google Cloud Print.

Zugreifen auf Drucker

Weitere Informationen zum Zugreifen auf Drucker von einem PC oder Mac aus finden Sie im nachfolgenden Abschnitt.

Mit einem Windows-PC auf Drucker zugreifen:

Die Einrichtung der Druckerverbindung mithilfe von Synology Assistant ist der beste Weg, um auf den Druckdienst des DiskStation-Geräts zuzugreifen. Ausführliche Anleitungen finden Sie in der entsprechenden Hilfe.

Falls Synology Assistant nicht auf Ihrem Computer installiert ist, haben Sie folgende Möglichkeiten, um auf den Druckdienst von DiskStation zuzugreifen.

Typ 1. Freigegebener Netzwerkdrucker

  1. Öffnen Sie von Ihrem PC aus in File Explorer den Link Netzwerk und doppelklicken Sie auf DiskStation.
  2. Doppelklicken Sie auf usbprinter oder Netzwerk-Druckername. Das System fordert Sie zur Installation des korrekten Treibers auf. Klicken Sie anschließend auf Ja.
  3. Laden Sie den Druckertreiber von der CD oder der Website des Herstellers. Wählen Sie den Hersteller und das Modell des Druckers aus oder klicken Sie auf Disk vorhanden, um den Speicherort des Treibers anzugeben. Klicken Sie anschließend auf OK.
  4. Das System beginnt, den Netzwerkdrucker einzurichten. Nach Abschluss der Einrichtung können Sie den Drucker verwenden.

Typ 2. LPR-Anschluss (empfohlen)

  1. Klicken Sie auf Ihrem PC auf Start und gehen Sie zu Geräte und Drucker.
  2. Klicken Sie auf Drucker hinzufügen.
  3. Wählen Sie Netzwerk, WLAN- oder Bluetooth-Drucker hinzufügen aus.
  4. Wählen Sie Der gewünschte Drucker ist nicht aufgelistet aus.
  5. Wählen Sie Einen Drucker über eine TCP/IP-Adresse oder Hostname hinzufügen aus. Klicken Sie auf Weiter, um fortzufahren.
  6. Wählen Sie als Gerätetyp TCP/IP-Port-Gerät aus.
  7. Geben Sie die IP-Adresse des DiskStation ein. Klicken Sie auf Weiter, um fortzufahren.
  8. Wählen Sie den Gerätetyp aus: Benutzerdefiniert > Einstellungen > …. Klicken Sie auf Weiter, um fortzufahren.
  9. Wählen Sie als Protokoll LPR aus. Geben Sie usbprinter oder Netzwerk-Druckername als Warteschlangennamen ein und aktivieren Sie das Kontrollkästchen LPR-Bytezählung aktiviert. Klicken Sie auf OK, um fortzufahren.
  10. Laden Sie den Druckertreiber von der CD oder der Website des Herstellers. Wählen Sie den Hersteller und das Modell des Druckers aus oder klicken Sie auf Disk vorhanden, um den Speicherort des Treibers anzugeben. Klicken Sie anschließend auf OK.
  11. Das System beginnt, den Netzwerkdrucker einzurichten. Nach Abschluss der Einrichtung können Sie den Drucker verwenden.

Mit einem Mac auf Drucker zugreifen:

Mac-Benutzer können auf den Druckdienst des DiskStation-Geräts zugreifen, indem Sie den Bonjour Printer Broadcast von DSM aktivieren und den Drucker unter „Systemeinstellungen“ hinzufügen.

Vergewissern Sie sich zuerst, dass Ihr DiskStation einen PostScript-Drucker verwendet, bevor Sie fortfahren.

Schritt 1. Bonjour Printer Broadcast von DSM aktivieren

  1. Melden Sie sich bei DSM als admin oder Benutzer der Gruppe administrators an.
  2. Gehen Sie zu Systemsteuerung > Dateidienste > Win/Mac/NFS.
  3. Aktivieren Sie Bonjour Printer Broadcast aktivieren.

Schritt 2. Auf den Drucker mit dem Mac zugreifen

  1. Gehen Sie auf Ihrem Mac zu Systemeinstellungen > Drucken und Scannen, und klicken Sie dann auf +.
  2. Wählen Sie usbprinter[Nummer] oder Netzwerk-Druckername im Dropdown-Menü aus. Der Mac richtet den Drucker automatisch ein.
