Benutzer-Home

Aktivieren Sie Benutzer-Home, um einen persönlichen Home -Ordner für jeden Benutzer zu erstellen, mit Ausnahme des Kontos guest. Jeder Benutzer kann über CIFS, AFP, FTP oder File Station auf seinen eigenen Home-Ordner zugreifen.

Der Admin kann auf alle personalisierten Ordner zugreifen, die sich im voreingestellten gemeinsamen Standardordner Homes befinden. Der Name des Home-Ordners ist derselbe wie das Benutzerkonto.

Aktivieren des Benutzer-Home-Dienstes:

  1. Klicken Sie auf Benutzer-Home.
  2. Markieren Sie Benutzer-Home-Dienst aktivieren.
  3. Wenn sich mehr als ein Volume auf Synology DiskStation befindet, wählen Sie aus, wo der Ordner Homes gespeichert werden soll.
  4. Klicken Sie auf OK.

Anmerkung: Ist der lokale Benutzer-Home-Dienst deaktiviert, wird der Domain-Benutzer-Home-Dienst zu gleichen Zeit deaktiviert.

Deaktivieren des Benutzer-Home-Dienstes:

  1. Klicken Sie auf Benutzer-Home.
  2. Entfernen Sie die Markierung Benutzer-Home-Dienst aktivieren.

Nachdem der Benutzer-Home-Dienst deaktiviert ist, stehen die gemeinsamen Ordner Homes nur noch dem Admin-Zugriff zur Verfügung. Die Benutzer können wieder auf Ihre Home-Ordner zugreifen, wenn der Benutzer-Home-Dienst wieder aktiviert ist.

    Anmerkung:
  1. Um den gemeinsamen Homes Ordner zu löschen, muss erst der Benutzer-Home-Dienst deaktiviert werden.
  2. Der Domain-Benutzer-Home-Dienst wäre dann deaktiviert, wenn der lokale Benutzer-Home-Dienst deaktiviert ist.
