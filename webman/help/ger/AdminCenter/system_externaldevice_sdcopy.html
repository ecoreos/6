SDCopy*

Sie können einen Zielort für die SDCopy Funktion auswählen und festlegen.

Aktivieren von SDCopy:

  1. Markieren Sie SDCopy aktivieren.
  2. Wählen Sie einen gemeinsamen Ordner aus dem Drop-Down-Menü aus.
  3. Klicken Sie auf OK.

Wenn Sie eine SD-Karte in den SD-Schlitz an der Synology DiskStation Frontblende eingesetzt haben, können Sie die Taste Kopieren an der Frontblende drücken, um Daten auf dem SD-Gerät zum hier zugewiesenen gemeinsamen Zielordner zu kopieren. Wenn der SDCopy Zielordner gelöscht wurde, ersetzt das System ihn automatisch mit einen öffentlichen gemeinsamen Ordner.

    Einschränkungen:
  1. Nur SD-Geräte im FAT- oder NTFS-Format werden unterstützt.
  2. SD-Geräte müssen in den SD-Schlitz auf der Vorderseite der Synology DiskStation eingesetzt werden.
  3. Der verfügbare Speicherplatz des gemeinsamen Zielordners muss größer sein als der vom SD-Gerät verwendete Speicherplatz.

____
* Nur bei bestimmten Modellen verfügbar.
