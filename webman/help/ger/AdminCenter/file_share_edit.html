Allgemeinen Informationen des gemeinsamen Ordners bearbeiten

Die allgemeinen Informationen eines gemeinsamen Ordners, z. B. Name, Beschreibung und weiteres können bearbeitet werden.

So bearbeiten Sie die allgemeinen Informationen des gemeinsamen Ordners:

  1. Gehen Sie zu Systemsteuerung > Gemeinsamer Ordner.
  2. Wählen Sie einen gemeinsamen Ordner aus.
  3. Klicken Sie auf Bearbeiten.
