Push-Dienst

Mit dem Push-Dienst kann der DiskStation Benachrichtigungen auf Ihr Mobilgerät oder an Webbrowser senden, wenn bestimmte Ereignisse oder Fehler auftreten.

Ports:

Überprüfen Sie vor der Aktivierung des Push-Dienstes, ob der DiskStation über Port 8089 auf das Internet zugreifen kann.

E-Mail-Benachrichtigung von Synology aktivieren:

  1. Aktivieren Sie Benachrichtigungen über den Systemstatus per E-Mail-Server von Synology senden.
  2. Geben Sie in das Feld Empfänger eine oder mehrere E-Mail-Adressen ein. Die Systembenachrichtigungen werden an die hier eingegebenen Adressen gesendet. Es sind zwei Adressen erlaubt. Trennen Sie die Adressen mit einem Semikolon (;) voneinander.
  3. Klicken Sie auf Übernehmen. Daraufhin wird eine Bestätigungs-E-Mail an die im Feld Empfänger angegebenen Adressen gesendet. Folgen Sie den in der Bestätigungs-E-Mail angegebenen Schritten.
  4. Klicken Sie nach der Verifizierung auf Testnachricht senden und anschließend auf OK, um eine Testbenachrichtigung zu senden.

Mobilgerätbenachrichtigungen aktivieren:

  1. Gehen Sie zu Systemsteuerung > Benachrichtigung > Push-Dienst.
  2. Aktivieren Sie das Kontrollkästchen Mobilgerätbenachrichtigungen aktivieren und klicken Sie auf Übernehmen.
  3. Installieren und starten Sie DS finder auf Ihrem Mobilgerät und fügen Sie anschließend Ihren DiskStation zur Liste Favoriten hinzu, um Ihr Mobilgerät mit DiskStation zu paaren.
  4. Klicken Sie auf Testnachricht senden und anschließend auf OK, um eine Testbenachrichtigung zu senden. Starten Sie anschließend DS finder auf Ihrem Mobilgerät, um zu prüfen, ob die Testnachricht empfangen wurde.

Gepaarte Mobilgeräte verwalten:

  1. Klicken Sie auf Gepaarte Geräte verwalten, um eine Liste mit allen gepaarten Mobilgeräten anzuzeigen.
  2. Wählen Sie, wenn Sie die Paarung von bestimmten Geräten aufheben möchten, die Geräte aus und klicken Sie auf Paarung aufheben. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.) Es werden erst wieder Benachrichtigungen an diese Geräte gesendet, wenn Sie sie wieder paaren.

Browser-Benachrichtigungen aktivieren:

  1. Aktivieren Sie Mobilgerät-Benachrichtigungen aktivieren und klicken Sie auf Übernehmen, um diese Funktion zu aktivieren.
  2. Klicken Sie auf Browser-Benachrichtigungen aktivieren, um Ihren DiskStation mit dem Browser zu verbinden.
  3. Führen Sie eine der folgenden Aktionen aus, die im Fenster erscheinen:
    • Für Chrome: Klicken Sie auf Jetzt installieren, um die Chrome-Erweiterung zu installieren, und melden Sie sich bei Ihrem Google-Konto an, sofern noch nicht geschehen.
    • Für Safari: Klicken Sie auf Zulassen, um den Benachrichtigungsdienst zu autorisieren.
  4. Wenn Sie zurück zur Browser-Benachrichtigungsseite weitergeleitet werden, klicken Sie auf die Registerkarte Gepaarte Geräte, um Informationen über die gepaarten Geräte anzuzeigen oder Ihren DiskStation vom Browser zu trennen, oder klicken Sie auf die Registerkarte Ereignis, um Ihren Browser-Benachrichtigungsverlauf der letzten sieben Tagen anzuzeigen.

Hinweis:

Der Browser-Benachrichtigungsdienst wird auf Chrome (35 oder höher) und Safari (OS X v10.9 oder höher) unterstützt.
