QuickConnect

QuickConnect ermöglicht Client-Anwendungen, über das Internet eine Verbindung mit Ihrem DiskStation herzustellen, ohne dass Sie Portweiterleitungsregeln einstellen müssen.

Hinweis:

QuickConnect kann mit DSM und einigen von Synology entwickelten Paketen wie Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Cloud Station und mobilen Anwendungen zusammenarbeiten.

QuickConnect aktivieren:

  1. Gehen Sie zu Systemsteuerung (Einfach-Modus) > QuickConnect.
  2. Markieren Sie das Kästchen QuickConnect aktivieren.
  3. Geben Sie Ihre bestehenden Synology-Kontoinformationen ein oder registrieren Sie ein neues Konto.
  4. Geben Sie eine neue QuickConnect-ID an. Stellen Sie sicher, dass sie leicht zu merken ist, sodass Sie jederzeit auf Ihren DiskStation zugreifen zu können.
  5. Klicken Sie auf Übernehmen.

Hinweis:

  • Bevor Sie QuickConnect für Client-Applikationen aktivieren, stellen Sie bitte sicher, das die nötigen DSM-Dienste bereits aktiviert sind. Zum Beispiel muss WebDAV auf der DiskStation aktiviert sein, bevor Sie QuickConnect für DS file aktivieren können.
  • Eine benutzerdefinierte QuickConnect ID kann Buchstaben, Zahlen und Bindestriche (-) enthalten und muss mit einem Buchstaben beginnen.
  • Für eine bessere QuickConnect-Leistung wird empfohlen, dass Sie unter Systemsteuerung (Erweiterter Modus) > Externer Zugriff > Routerkonfiguration die Portweiterleitung für jeden Dienst konfigurieren:
    • DSM: 5000 (HTTP); 5001 (HTTPS)
    • Photo Station: 80 (HTTP); 443 (HTTPS)
    • Cloud Station: 6690

Fehlerbehebung

Wenn die Meldung Netzwerkfehler aufgetreten angezeigt wird, bedeutet dies, dass der QuickConnect-Dienst aufgrund eines Netzwerkfehlers beendet wurde. Prüfen Sie Folgendes:

  • Ihr DiskStation ist mit einem aktiven Netzwerk verbunden (unter Systemsteuerung (Erweiterter Modus) > Netzwerk > Netzwerkschnittstelle).
  • Die Einstellungen für DNS-Server und Standard-Gateway sind korrekt konfiguriert (unter Systemsteuerung (Erweiterter Modus) > Netzwerk > Allgemein).
  • Vergewissern Sie sich, dass der DiskStation über die folgenden Zielports auf das Internet zugreifen kann: 22, 80, 443, 8888. 

Nachdem die Netzwerkfehler behoben wurden, sollte der QuickConnect-Dienst nach ein paar Minuten wieder zur Verfügung stehen.
