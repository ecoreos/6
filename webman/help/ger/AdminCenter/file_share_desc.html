Gemeinsamer Ordner

Gemeinsame Ordner sind die Grundverzeichnisse, in denen Sie Dateien und Ordner auf Ihrem DiskStation speichern können. Sie müssen mindestens einen gemeinsamen Ordner erstellen, bevor Sie Daten auf dem DiskStation speichern können. In den gemeinsamen Ordnern gespeicherte Daten können privat bleiben oder mit bestimmten Benutzern oder Benutzergruppen geteilt werden, je nach den anpassbaren Zugriffsrechten. Darüber hinaus können gemeinsame Ordner bei der Erstellung verschlüsselt werden, um eine zusätzliche Schutzschicht zu erreichen.

Voreingestellter gemeinsamer Ordner

Nachfolgend finden Sie eine Liste der gemeinsamen Ordner, die automatisch erstellt werden, wenn bestimmte Anwendungen, Dienste oder Pakete installiert oder aktiviert werden.

Name Beschreibung
public Der Ordner public wird bei der ersten Einrichtung des DiskStation automatisch erstellt.*
web Der Ordner web ist der Standard-Speicherort für die Inhalte einer Website, die auf DiskStation gehostet wird. Er wird automatisch erstellt, wenn Web Station aktiviert ist.
photo Der Ordner photo speichert Fotos und Videos, die Sie mit anderen auf der Photo Station teilen möchten. Er wird automatisch erstellt, wenn die Photo Station oder der DLNA/UPnP-Medienserver aktiviert ist.
music Der Ordner music enthält Musik, die Sie auf Audio Station hören möchten. Er wird automatisch erstellt, wenn Audio Station aktiviert ist.
video Der Ordner video enthält Videos, die Sie über DLNA/UPnP-Geräte durchsuchen können. Er wird automatisch erstellt, wenn Medienserver aktiviert ist.
surveillance Der Ordner surveillance enthält die Aufzeichnungen der Surveillance Station. Er wird automatisch erstellt, wenn Surveillance Station aktiviert ist. Er hat nur Lesezugriff und kann nur vom System-Administrator aufgerufen werden.
home Der Ordner home bietet einen privaten Bereich für jeden Benutzer, in dem Daten gespeichert werden können, auf die nur dieser Benutzer zugreifen kann. Er wird automatisch erstellt, wenn die Funktion Benutzer-Home aktiviert ist.
homes Der Ordner homes enthält die „home“-Ordner aller Benutzer. Er wird automatisch erstellt, wenn die Funktion Benutzer-Home aktiviert ist. Nur der Systemadministrator hat Zugriff und kann alle Benutzer-Home-Ordner verwalten.
NetBackup Der Ordner NetBackup wird automatisch erstellt, wenn die Netzwerksicherung aktiviert ist.
usbshare Der Ordner usbshare [Nummer] wird automatisch erstellt, wenn Sie ein externes USB-Gerät an den USB-Port von DiskStation anschließen.*
satashare Der Ordner satashare wird automatisch erstellt, wenn Sie ein externes eSATA-Laufwerk an den eSATA-Port von DiskStation anschließen.*

____
* Nur bei bestimmten Modellen verfügbar.
