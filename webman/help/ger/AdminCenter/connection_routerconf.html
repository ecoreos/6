Routerkonfiguration

Wenn Ihr DiskStation sich in einem lokalen Netzwerk befindet, können sich keine anderen Geräte im Internet damit verbinden. Unter Systemsteuerung > Externer Zugriff > Routerkonfiguration können Sie Portweiterleitungsregeln festlegen, damit DiskStation über das Internet zugänglich ist.

Einrichten des Routers

Bevor Sie Portweiterleitungsregeln hinzufügen, müssen Sie die Verbindung zwischen dem DiskStation und dem Router einrichten.

Hinweis:

  1. Um Regeln für die Portweiterleitung zu konfigurieren und statische IP-Adressen zuzuweisen, müssen Sie Administratorrechte für den Router besitzen.
  2. Bevor Sie die Verbindung vom DiskStation zu einem UPnP-Router einrichten, müssen Sie sicherstellen, dass DSM die externe IP-Adresse des DiskStation erfolgreich abgerufen hat (unter Systemsteuerung > Externer Zugriff > DDNS). Andernfalls müssen Sie die Netzwerk- und Firewall-Einstellungen Ihres DiskStation und des UPnP-Routers überprüfen.

Verbindung vom DiskStation zum Router einrichten:

  1. Klicken Sie auf Router einrichten, um den Einrichtungsassistenten zu starten. DSM beginnt mit dem Testen der folgenden Netzwerkkomponenten/Einstellungen, bevor der nächsten Schritt durchgeführt wird.
    • Netzwerkschnittstelle
    • Gateway-Verbindung
    • Internetverbindung
    • Portweiterleitung
    • DNS
  2. Führen Sie, abhängig vom Typ des Routers, einen der folgenden Schritte durch:
    • Wenn Ihr Router UPnP nicht unterstützt, fahren Sie mit Schritt 3 fort.
    • Klicken Sie, wenn der UPnP-Router auf Kompatibilität mit dem DiskStation-Gerät getestet wurde, auf Übernehmen, wenn die Zusammenfassung der Informationen des Routers angezeigt wird. Die Verbindung Ihres DiskStation mit dem Router ist dann erfolgreich eingerichtet.
    • Führen Sie einen der folgenden Schritte durch, wenn der UPnP-Router nicht auf Kompatibilität mit dem DiskStation getestet wurde:
      • Wenn Sie die Verbindung zum Router automatisch einrichten möchten, wählen Sie Automatisch aus, und klicken Sie auf Weiter, damit das System einen Kompatibilitätstest durchführt:
        • Wenn der Test ergibt, dass der Router mit dem Routerkonfigurationsdienst des DiskStation kompatibel ist, aktivieren Sie das Kontrollkästchen, um den Testbericht zu senden, damit Synology die Router-Kompatibilitätsdatenbank aktualisieren kann. Klicken Sie danach auf Übernehmen. Die Verbindung Ihres DiskStation mit dem Router ist dann erfolgreich eingerichtet.
        • Wenn der Test ergibt, dass der Router nicht mit dem Routerkonfigurationsdienst des DiskStation kompatibel ist, aktivieren Sie das Kontrollkästchen, um den Testbericht zu senden, damit Synology die Router-Kompatibilitätsdatenbank aktualisieren kann. Klicken Sie danach auf Schließen, um den Einrichtungsassistenten zu beenden.

          • Hinweis:

          Wenn der Test ergibt, dass der Router nicht kompatibel ist, können Sie dennoch Portweiterleitungsregeln auf der Benutzeroberfläche des Routers konfigurieren. Weitere Informationen finden Sie hier.

      • Wählen Sie, wenn Sie die Verbindung zum Router manuell einrichten möchten, Manuell aus, klicken Sie auf Weiter und fahren Sie mit Schritt 3 fort.
  3. Führen Sie die folgenden Schritte durch, wenn Sie aufgefordert werden, die Verbindung des DiskStation zum Router manuell einzurichten:
    1. Wählen Sie im Drop-down-Menü Router Ihren Router aus.

      2. Hinweis:

      Wenn Sie den Router nicht finden, klicken Sie auf Liste aktualisieren, um die neueste Liste von Synology abzurufen.

    2. Richten Sie Protokoll und Port des Routers ein. Protokoll und Port variieren je nach Router-Modell. Wenn keine Verbindung über das Standard-Protokoll (HTTP) und den Standard-Port (80) hergestellt werden kann, finden Sie in der Dokumentation Ihres Routers die korrekten Einstellungen.
    3. Wenn Ihr Router ein benutzerdefiniertes Konto benötigt, markieren Sie Benutzerdefiniertes Konto verwenden und füllen Sie die Felder Konto und Passwort aus.
    4. Klicken Sie auf Übernehmen, um den Vorgang zu beenden.

Verwalten von Portweiterleitungsregeln

Die Regeln zur Portweiterleitung können die Zielports Ihres DiskStation festlegen, die Pakete von speziellen Ports Ihres Routers empfangen.

Wenn Ihr DiskStation über einen Router mit dem Internet verbunden ist, klicken Sie auf Erstellen, um den Einrichtungsassistenten für die Portweiterleitung zu öffnen.

Sie haben drei Möglichkeiten, um Portweiterleitungsregeln hinzuzufügen:

  • Wenn Sie Portweiterleitungsregeln für Ihre Systemdienste (FTP, Web Station usw.) erstellen möchten, wählen Sie Integrierte Anwendung aus.
  • Wenn Sie Portweiterleitungsregeln für andere Dienste erstellen möchten, wählen Sie Benutzerdefinierter Port aus.

Sie können die folgende Syntax verwenden, um Portnummern und Portbereich festzulegen:

  • Verwenden Sie einen “-”, um den Portbereich anzugeben, zum Beispiel: 6881-6890.
  • Verwenden Sie ein Komma “,”, um die Ports zu trennen, zum Beispiel: “21,22”, “21,55536-55663”.

Hinweis:

  • Jeder Port des Gateway-Routers kann nur für eine Weiterleitungsregel verwendet werden. Wenn Sie einen Router-Port für zwei oder mehr Regeln verwenden, führt das zu Konflikten, und eine Warnmeldung erscheint auf dem Bildschirm. Sie müssen einen weiteren Port in der Regel festlegen, um Konflikte zu vermeiden.
  • Einige Router lassen nicht zu, dass die benutzerdefinierten Ports zur Weiterleitung für Systemdienste verwendet werden.

Regeln speichern:

Klicken Sie auf Speichern, nachdem Sie die Regeln erstellt haben. Ansonsten werden die Regeln nicht übernommen.

Nachdem Sie die Regeln gespeichert haben, müssen Sie je nach Router-Modell möglicherweise einige Minuten warten, ehe die Regeln funktionieren.

Regeln auf Funktion prüfen:

Wählen Sie die Regeln aus, die Sie testen möchten, und klicken Sie auf Verbindung testen. Das Ergebnis wird in der Spalte Verbindungstestergebnisse angezeigt.

Für WLAN-Routerkonfigurationen*

Wenn Ihr DiskStation als WLAN-Router fungiert, werden Portweiterleitungsregeln direkt vom DiskStation ausgeführt.

Portweiterleitungsregeln hinzufügen:

  1. Klicken Sie auf Erstellen, um das Einrichtungsfenster zu öffnen.

  2. Geben Sie Dienst-Namen, Protokoll (TCP oder UDP), Private IP-Adresse, Privater Port und Öffentlicher Port an. Sie können die folgende Syntax verwenden, um Portnummern und Portbereich festzulegen:
    • Verwenden Sie einen “-”, um den Portbereich anzugeben, zum Beispiel: 6881-6890.
    • Verwenden Sie ein Komma “,”, um die Ports zu trennen, zum Beispiel: “21,22”, “21,55536-55663”.
  3. Klicken Sie unten auf Erstellen, um die Einstellungen abzuschließen.

Hinweis:

Bei den meisten Routern brauchen die Computer-Ports und die Router-Ports beim Konfigurieren von Portweiterleitungsregeln nicht identisch zu sein. Wenn Ihr Router diese Funktion unterstützt, gilt Folgendes:

  • Wenn eine Regel gespeichert wird und ein Router-Port bereits von einer vorhandenen Regel verwendet wird, wird ein neuer Router-Port zugewiesen.
  • Beim Hinzufügen oder Bearbeiten von Regeln können Sie die Router-Portnummer direkt angeben.

DMZ*

Sie können DMZ auf Ihrem DiskStation aktivieren, wenn die Netzwerkabbildungskonfiguration „WLAN-Router“ ist. Ein Host in der DMZ kann sich direkt, ohne Einschränkungen und ohne den Schutz der Firewall mit dem externen Netzwerk verbinden. Sie wird für gewöhnlich für die Einrichtung von Servern verwendet.

DMZ aktivieren:

  1. Klicken Sie auf DMZ-Einstellungen, um das Einrichtungsfenster zu öffnen.
  2. Aktivieren Sie das Kontrollkästchen DMZ aktivieren.
  3. Geben Sie die IP-Adresse des DMZ-Hosts ein.
  4. Klicken Sie auf Übernehmen.

Hinweis:

Sie benötigen eine externe IP-Adresse von DiskStation aus dem externen Netzwerk, um sich mit einem Host in DMZ zu verbinden.
