Eine Gruppe bearbeiten

Sie können die Informationen sowie die Rechte von Gruppen für den Zugriff auf gemeinsame Ordner ändern.

Gruppeninformationen bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und bearbeiten Sie die Gruppeninformationen.
  3. Klicken Sie zum Beenden auf OK.

Gruppenzugriffsrechte für gemeinsame Order bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und Berechtigungen.
  3. Aktivieren oder deaktivieren Sie die folgenden Zugriffsrechte, um der Gruppe Berechtigungen zuzuweisen:
    1. Lesen/Schreiben: Die Gruppe hat Zugriff und kann Änderungen im gemeinsamen Ordner vornehmen.
    2. Nur Lesen: Die Gruppe hat Zugriff und kann jedoch keine Änderungen im gemeinsamen Ordner vornehmen.
    3. Kein Zugriff: Die Gruppe hat keinen Zugriff auf den gemeinsamen Ordner.
  4. Klicken Sie zum Beenden auf OK.

Hinweis:

Beachten Sie bei Berechtigungskonflikten folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur Lesen.

Kontingent einer Gruppe bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und gehen Sie zur Registerkarte Quote.
  3. Aktivieren Sie für jedes Volume bzw. jeden gemeinsamen Ordner das Kontrollkästchen unter Kontingent aktivieren. Geben Sie dann eine Quote in das Feld Quote ein (z. B. 10 GB).

Hinweis:

  • Jedes Gruppenmitglied übernimmt das angegebene Gruppenkontingent. Das Gruppenkontingent wird nicht auf die einzelnen Gruppenmitglieder aufgeteilt.
  • Das maximale für jede Gruppe einstellbare Speicherkontingent beträgt ca. 4 TB (4095 GB).

Gruppenzugriffsrechte auf Anwendungen bearbeiten:

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und gehen Sie zur Registerkarte Anwendungen.
  3. Für jeden Dienst gibt es die folgenden Optionen:
    • Zulassen: Die Gruppe kann auf die Anwendung zugreifen.
    • Verweigern: Die Gruppe kann nicht auf die Anwendung zugreifen.
    • Benutzerdefiniert: Verwaltet Zugriffsrechte nach IP-Adresse.
  4. Klicken Sie zum Beenden auf OK.

Hinweis:

  • Beachten Sie bei Berechtigungskonflikten folgende Priorität: Verweigern > Zulassen.
  • Nicht alle Pakete und Dienste unterstützen die Zugriffsrechte-Einstellungen Nach IP.

Geschwindigkeitseinschränkung für eine Gruppe bearbeiten:

Sie können für verschiedene Dienste (z. B. File Station, FTP usw.) eine Geschwindigkeitseinschränkung für eine Gruppe festlegen, um die von den Gruppenmitgliedern für die Datenübertragung über die Dienste mögliche Bandbreite zu beschränken. Geschwindigkeitseinschränkungen für Gruppen gelten für alle Gruppenmitglieder.

  1. Wählen Sie die Gruppe aus, die Sie bearbeiten möchten.
  2. Klicken Sie auf Bearbeiten und gehen Sie zur Registerkarte Geschwindigkeitseinschränkung.
  3. Für jeden Dienst gibt es die folgenden Optionen:
    • Ohne: Für die Nutzung des Dienstes gibt es für die Gruppenmitglieder keine Geschwindigkeitseinschränkung.
    • Konstant: Für alle Gruppenmitglieder gilt eine feste, konstante Geschwindigkeitseinschränkung. Geben Sie die Upload- und Download-Geschwindigkeitseinschränkungen in den Feldern links an.
    • Variabel: Für die Gruppenmitglieder gelten zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für Geschwindigkeitseinschränkung und den Zeitplan zu ändern.
  4. Klicken Sie zum Beenden auf OK.

Geschwindigkeitseinschränkung für Benutzer, die zu mehreren Gruppen gehören

Wenn ein Benutzer zu mehreren Gruppen gehört, hängt die Geschwindigkeitseinschränkung von der Gruppe mit der höchsten verfügbaren Bandbreite zum Zeitpunkt der Übertragung von Dateien ab. Da die Bandbreite gleichmäßig auf Benutzer, die Dateien übertragen, verteilt wird, hängt die verfügbare Bandbreite davon ab, wie viele Benutzer gleichzeitig die Bandbreite der Gruppe verwenden.

Der Benutzer „John“ gehört z. B. zur „Gruppe 1“ (Geschwindigkeitseinschränkung 500 kBit/s; insgesamt fünf Benutzer) und „Gruppe 2“ (Geschwindigkeitseinschränkung 400 kBit/s; insgesamt zwei Benutzer).

  • Wenn alle Benutzer in Gruppe 1 und Gruppe 2 gleichzeitig Dateien übertragen, beträgt die Geschwindigkeitseinschränkung für John 200 KB/s und sein Verbrauch wird auf die gemeinsame Bandbreite von Gruppe 2 angerechnet.
  • Wenn die anderen Benutzer in Gruppe 1 mit der Übertragung von Dateien fertig sind, wird die Geschwindigkeitseinschränkung von John sofort auf 500 KB/s umgestellt und sein Verbrauch auf die gemeinsame Bandbreite von Gruppe 1 angerechnet.
