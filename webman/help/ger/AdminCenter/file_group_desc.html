Gruppe

Kategorisieren Sie Benutzerkonten in benutzerdefinierte Benutzergruppen und bearbeiten Sie dann die Eigenschaften der Gruppe. So ersparen Sie sich die Bearbeitung einzelner Benutzer.

Gruppe erstellen

Über diese Option können Sie eine Gruppe erstellen und deren Beschreibung bearbeiten.

Eine Gruppe bearbeiten

Über diese Option können Sie die Informationen und Zugriffsrechte der Gruppe bezüglich der freigegebenen Ordner ändern.

Gruppenmitglieder hinzufügen/entfernen

Über diese Option können Sie Benutzer zu einer Gruppe hinzufügen oder aus ihr entfernen.

Eine Gruppe entfernen

Über diese Option können Sie eine Gruppe vom DiskStation entfernen.
