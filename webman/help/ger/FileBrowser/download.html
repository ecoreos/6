Dateien herunterladen

Gehen Sie folgendermaßen vor, um Dateien oder Ordner von der DiskStation zum Computer herunterzuladen.

Dateien zum lokalen Computer herunterladen:

  1. Wählen Sie die Dateien oder Ordner aus, die Sie herunterladen möchten. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  2. Klicken Sie auf Aktion > Herunterladen und wählen Sie je nach Ihrem Webbrowser den gewünschten Download-Modus aus:
    • Google Chrome: Klicken Sie auf Herunterladen, um einzelne Dateien (unkomprimiert) herunterzuladen. Klicken Sie auf Als .zip herunterladen, um mehrere Dateien in eine .zip-Datei herunterzuladen (unter Verwendung der unter Einstellungen > Allgemein angegebenen File Station-Codepage).
    • Andere Browser: Klicken Sie auf Herunterladen, um eine oder mehrere Dateien in eine .zip-Datei herunterzuladen (unter Verwendung der unter Einstellungen > Allgemein angegebenen File Station-Codepage).
    • Zur Warteschleife hinzufügen - Überschreiben oder Zur Warteschleife hinzufügen - Überspringen: Abhängig davon, ob Sie Dateien oder Ordner mit demselben Namen am Zielort überschreiben möchten, wählen Sie eine der Optionen zum Hinzufügen der Dateien zur Download-Warteschleife aus (die angezeigt wird, wenn Sie auf den Nach-unten-Pfeil oben rechts klicken).
    Hinweis:
  1. In der Windows-Umgebung werden die Zeichen \ * ? " < > | : in Dateinamen nach dem Herunterladen durch - ersetzt.
  2. Gehen Sie, um das automatische Herunterladen mehrerer Dateien mit Chrome zu deaktivieren, zu dessen Einstellungen und ändern Sie die Inhaltseinstellungen im Datenschutzbereich.
  3. Chrome unterstützt im Incognito-Modus nicht das Herunterladen einzelner unkomprimierter Dateien.

Download-Link für Datei/Ordner abrufen

Sie können den Download-Link für eine Datei oder einen Ordner abrufen und ihn für andere Benutzer freigeben.

Download-Link für eine Datei oder einen Ordner abrufen:

  1. Wählen Sie die Datei oder den Ordner aus, für die bzw. den Sie einen Download-Link abrufen möchten.
  2. Gehen Sie wie folgt vor, um das Fenster "Eigenschaften" zu öffnen:
    • Wählen Sie Eigenschaften im Menü Aktion aus.
    • Rechtsklicken Sie auf eine der Dateien oder einen der Ordner und wählen Sie Eigenschaften aus.
  3. Rechtsklicken Sie auf den Download-Link und wählen Sie die entsprechende Funktion im Kontextmenü aus, um den Download-Link zu kopieren. Der Name der Funktion kann je nach Webbrowser, den Sie verwenden, unterschiedlich sein.
  4. Senden Sie den Link per E-Mail oder SMS an andere Benutzer. Diese können dann die verknüpfte Datei oder den verknüpften Ordner herunterladen, indem sie ihren Benutzernamen und das Kennwort eingeben.

Hinweis: Um eine Datei oder einen Ordner über den Download-Link herunterzuladen, sind ein gültiger Benutzername und ein gültiges Kennwort erforderlich.
