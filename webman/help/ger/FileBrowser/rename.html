Datei oder Ordner umbenennen

Verwenden Sie eine der folgenden Methoden, um Dateien und Unterordner auf DiskStation umzubenennen.

Methode 1:

  1. Wählen Sie eine Datei oder einen Ordner aus, die/den Sie umbenennen möchten.
  2. Wählen Sie Umbenennen im Menü Aktion.
  3. Geben Sie im angezeigten Fenster einen neuen Namen ein und klicken Sie auf OK.

Methode 2:

  1. Klicken Sie mit der rechten Maustaste auf eine Datei oder einen Ordner und wählen Sie Umbenennen.
  2. Geben Sie im angezeigten Fenster einen neuen Namen ein und klicken Sie auf OK.
