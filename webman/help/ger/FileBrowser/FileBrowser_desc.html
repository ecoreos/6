File Station

Die File Station ist ein benutzerfreundliches Dateiverwaltungstool, mit dem DSM-Benutzer über das Internet mit ihren Web-Browsern auf DiskStation-Ordner zugreifen können. Indem Sie zwei oder mehrere File Stations öffnen, können Sie all Ihre DiskStation-Daten durch Ziehen und Ablegen in den verschiedenen File Stations verwalten.
