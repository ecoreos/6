Versenden von Dateien als E-Mail-Anlage

Sie können Dateien als E-Mail-Anlage direkt in der File Station versenden, indem Sie auf die ausgewählten Dateien rechtsklicken und die Option als E-Mail-Anlage versenden auswählen. Ein E-Mail-Fenster mit einigen nützlichen Eigenschaften wird geöffnet:

  • An: Falls Sie irgendwelche Kontakte für Ihr E-Mail-Konto synchronisiert haben, erscheint eine Kontaktliste, in der die mit Ihrem E-Mail-Konto verknüpften, synchronisierten Kontakte angezeigt werden.
  • Von: Sie können das Standard-E-Mail-Konto auswählen oder ein anderes E-Mail-Konto hinzufügen. Die Kontaktliste wird automatisch zum ausgewählten E-Mail-Konto wechseln.
  • Anlagen: Die ausgewählten Anlagen werden am unteren Rand der E-Mail angezeigt. Sie können mehrere Anlagen mit Hilfe der Anlagen-Taste hinzufügen.

Hinweis:

  • Nur Dateien können als Anlage hinzugefügt werden.
  • Sie können bis zu 10 Dateien an eine einzige E-Mail anhängen.
  • Die maximal zulässige Gesamtgröße aller Anhänge einer E-Mail ist von Ihrem E-Mail-Dienst abhängig. Wenn Sie nicht sicher sind, empfehlen wir die Gesamtdateigröße aller Anhänge auf 25 MB zu beschränken.
