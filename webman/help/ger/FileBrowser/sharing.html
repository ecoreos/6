Dateilinks freigeben

Dateien oder Ordner, die auf Ihrer DiskStation gespeichert sind, können schnell und einfach mit anderen geteilt werden. Rechtsklicken Sie einfach auf eine Datei oder einen Ordner und wählen Sie Dateilinks freigeben aus. Dadurch werden ein Link und ein QR-Code erstellt, die an Freunde oder andere Benutzer gesendet werden können. Sie können dem Link folgen, um die ausgewählte Datei oder den ausgewählten Ordner herunterzuladen, unabhängig davon, ob Sie ein DSM-Konto besitzen.

Dateilinks teilen:

  1. Wählen Sie die Dateien oder die Ordner aus, die Sie teilen möchten.
  2. Rechtsklicken Sie auf eine der ausgewählten Dateien oder Ordner oder klicken Sie auf das Menü Aktion. Wählen Sie dann Dateilinks teilen aus.
  3. Geteilte Links und QR-Codes werden im Fenster, das daraufhin erscheint, angezeigt.
  4. Die folgenden Optionen können ebenfalls geändert werden:
    • Gültigkeitszeitraum: Sie können gültige Zugriffszeiträume und Zugriffshäufigkeiten für freigegebene Links anpassen.
      • Gültigkeitszeitraum festlegen: Sie können festlegen, wie lange ein freigegebener Link nach der Erstellung aktiv bleiben soll. Der Link wird ungültig, sowie die angegebene Zeit überschritten ist.
      • Gültige Zugriffshäufigkeiten anpassen: Sie können festlegen, wie häufig auf einen freigegebenen Link zugegriffen werden darf. Der Link wird ungültig, sowie die angegebene Häufigkeit überschritten ist.
    • Für DSM-Benutzer freigeben: Sie können den Link für DSM-Benutzer oder -Gruppen freigeben, indem Sie sie aus der Dropdown-Liste auswählen. Beachten Sie, dass Sie den Link nicht für das Guest-Konto und Benutzergruppen freigeben können.
    • Für andere freigeben: Sie können kennwortgeschützte Links für andere Personen, einschließlich Nicht-DSM-Benutzer freigeben.

Gofile.me-Links aktivieren:

  1. Gehen Sie zu Systemsteuerung > QuickConnect und überprüfen Sie, ob QuickConnect aktivieren markiert ist. Falls Sie QuickConnect nicht eingerichtet haben, können Sie es auf dieser Seite tun.
  2. Klicken Sie dann auf Erweitert und stellen Sie sicher, dass Dateifreigabe aktiviert ist.
  3. Geteilte Dateien können heruntergeladen werden, ohne dass Sie ein DSM-Konto besitzen, indem Sie auf den Austausch-Link „gofile.me“ zugreifen.

Seite mit freigegebenen Links anpassen

File Station verfügt über mehrere Optionen zum Anpassen der Seite mit freigegebenen Links.

  1. Gehen Sie zu File Station > Einstellungen > Freigegebene Links > Benutzerdefiniert.
  2. Wählen Sie Seite mit freigegebenen Links anpassen aus und klicken Sie auf Benutzerdefinierte Einstellungen.
  3. Sie können die Seite auf folgende Weise anpassen:
    • Vorlage: Wählen Sie eine der Standardvorlagen aus.
    • Hintergrund: Klicken Sie auf die Miniaturansicht oder auf +, um ein Standard-Hintergrundbild, ein Bild auf Ihrem DiskStation oder ein zuvor angewendetes Bild auszuwählen.
    • Logo: Klicken Sie auf die Miniaturansicht oder auf +, um ein hochgeladenes Bild (kleiner als 1 MB), ein Bild auf Ihrem DiskStation oder ein zuvor angewendetes Bild auszuwählen. Das Logo kann in jede der vier Ecken positioniert werden.
  4. Klicken Sie auf Vorschau, um zu sehen, ob Anpassungen erforderlich sind.
  5. Klicken Sie auf OK, um die Einstellungen zu übernehmen.

Hinweis:

  1. Die benutzerdefinierten Einstellungen werden unabhängig von ihrer Erstellungszeit auf alle freigegebenen Links angewendet.
  2. Sie können nur Bilder der Formate .jpe, jpeg, .jpg, .bmp, .gif und .png als Hintergrundbild/Logo anwenden.

Austausch-Links verwalten

Bestehende Austausch-Links können über Extras > Verwaltung geteilter Links verwaltet werden. Im darauf angezeigten Fenster gibt es Optionen, um bestehende Austausch-Links zu bearbeiten, zu teilen oder zu entfernen.

Hinweis:

  1. Sie können bis zu 1000 freigegebene Links erstellen.
  2. Im Status-Bereich jedes Austausch-Links wird Folgendes angezeigt:
    • Gültig: Der freigegebene Link funktioniert einwandfrei.
    • Abgelaufen: Die Gültigkeitszeit des freigegebenen Links ist abgelaufen.
    • Ungültig: Die freigegebene Datei oder der freigegebene Ordner ist nicht mehr vorhanden oder Sie besitzen keine korrekte Berechtigung für den Zugriff auf die Datei oder den Ordner.
    • Noch nicht wirksam: Der freigegebene Link ist noch nicht aktiv.
