Dateien oder Ordner hochladen

Verwenden Sie eine der folgenden Methoden, um Dateien oder Ordner von Ihrem Computer auf DiskStation hochzuladen.

Um vollen Zugriff auf die unten stehenden Upload-Funktionen zu haben, installieren Sie zuerst Java Runtime Environment (JRE) und Flash Player.

Methode 1:

  1. Wählen Sie im oberen Teil des linken Feldes einen Zielordner auf DiskStation.
  2. Klicken Sie auf Hochladen, wählen Sie Hochladen - Überspringen oder Hochladen - Überschreiben und wählen Sie Dateien oder Ordner von Ihrem lokalen Computer.

Methode 2:

  1. Gehen Sie zu Einstellungen, markieren Sie Mit der File Station Dateien des lokalen Computers durchsuchen und aktualisieren Sie die Webseite, um Ordner Ihres lokalen Computers im unteren Bereich des linken Feldes anzuzeigen.

    2. Hinweis:

    Die Option Mit der File Station Dateien des lokalen Computers durchsuchen wird nur übernommen, wenn Sie JRE auf Ihrem Computer installiert und aktiviert haben.

  2. Wählen Sie im unteren Teil des linken Feldes einen Ordner von Ihrem lokalen Computer. Der Hauptbereich zeigt nun die Inhalte des ausgewählten Ordners an.
  3. Wählen Sie die gewünschten Dateien oder Ordner im Hauptbereich und klicken Sie auf Hochladen.
  4. Wählen Sie im Fenster, das dann erscheint, einen Zielordner auf DiskStation aus.
  5. Wählen Sie oben in der rechten Ecke des Fensters Überschreiben, wenn Sie die Dateien oder Ordner mit demselben Namen am Zielordner überschreiben möchten. Klicken Sie andernfalls auf Überspringen.
  6. Klicken Sie auf OK, um den Upload zu starten.

Methode 3:

  1. Gehen Sie zu Einstellungen, markieren Sie Mit der File Station Dateien des lokalen Computers durchsuchen und aktualisieren Sie die Webseite, um Ordner Ihres lokalen Computers im unteren Bereich des linken Feldes anzuzeigen.
  2. Ziehen Sie im linken Feld einen Ordner von Ihrem Computer (der untere Teil) in einen beliebigen Ordner auf DiskStation (oberer Teil).
  3. Wählen Sie im angezeigten Pop-Up Menü, ob Sie Dateien/Ordner mit identischem Namen im Zielordner überschreiben oder überspringen möchten.

Methode 4:

  1. Wählen Sie im oberen Teil des linken Feldes einen Zielordner auf DiskStation.
  2. Wählen Sie Dateien auf dem Desktop oder im Datei-Explorer Ihres Computers (z. B. Windows Explorer oder Mac Finder) und ziehen Sie diese in die File Station im Webbrowser.
  3. Wählen Sie im angezeigten Dialog, ob Sie Dateien mit identischem Namen im Zielordner überschreiben oder überspringen möchten.

    4. Hinweis:

    • Methode 4 wird nur von bestimmten Browsern unterstützt: Chrome, Firefox, Safari 5.1 und neuere Versionen. Wenn allerdings Mit der File Station Dateien des lokalen Computers durchsuchen markiert ist, wird Mac nicht unterstützt.
    • Sie können auch Ordner ziehen und ablegen, um Dateien zu leeren, aber nur wenn Mit der File Station Dateien des lokalen Computers durchsuchen für die oben stehenden Browser markiert ist. Anderenfalls können Sie mit Methode 4 keine Ordner hochladen und Dateien leeren.
    • Wenn Sie Chrome für Linux verwenden und über das HTTPS-Protokoll bei DSM angemeldet sind, müssen Sie ein vertrauenswürdiges Zertifikat importieren (unter Systemsteuerung > Sicherheit > Zertifikat), um mit Methode 4 Inhalte hochladen zu können.

Hinweis:

  1. Sie können bis zu 1000 Dateien zu Ihrer Upload-Warteschlange hinzufügen.
  2. Sie können Dateien mit einer Größe über 4GB hochladen, wenn Sie Google Chrome, Firefox, Safari 5.1 oder später verwenden oder wenn Mit der File Station Dateien des lokalen Computers durchsuchen markiert ist. Ansonsten können Sie nur bis zu 2 GB Dateien gleichzeitig hochladen.
  3. Bei gemeinsamen, nicht verschlüsselten Ordnern sollte der Datei-/Ordnername maximal 255 Zeichen lang sein (max. ca. 80 Zeichen bei nicht-lateinischen Sprachen); bei gemeinsamen, verschlüsselten Ordnern sollte der Datei-/Ordnername maximal 143 Zeichen lang sein (max. ca. 47 Zeichen bei nicht-lateinischen Sprachen).
  4. Datei-/Ordnernamen dürfen die folgenden Zeichen nicht enthalten: \ / : * ? " > |.
  5. Datei-/Ordnername darf nicht mit "._" beginnen.
  6. Sie Dateien 0 KB Dateien nur dann hochladen, wenn Sie JRE installiert haben.

Fehlerbehebung

Wenn Sie lokal auf Ihrem Mac gespeicherte Inhalte mit Safari nicht ordnungsgemäß durchsuchen können, gehen Sie folgendermaßen vor:

  1. Öffnen Sie auf Ihrem Mac Safari und wählen Sie Einstellungen im Menü.
  2. Klicken Sie in der Registerkarte Sicherheit auf Website-Einstellungen verwalten.
  3. Wählen Sie unter Java Ihre DSM-"Website".
  4. Wählen Sie Im unsicheren Modus ausführen.
