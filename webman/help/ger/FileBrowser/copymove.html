Dateien oder Ordner kopieren und verschieben

File Station bietet mehrere Möglichkeiten der Übertragung von Dateien/Ordnern zwischen zwei DiskStation, innerhalb einer DiskStation oder zu einem lokalen Computer.

Dateien/Ordner zwischen Browsern ziehen und ablegen

Ohne aufwändige Einrichtung von Dateiübertragungsprotokollen können Sie mit File Station Dateien/Ordner direkt von einer DiskStation zu einer anderen hochladen.

Ziegen und Ablegen zwischen Browsern aktivieren:

  1. Gehen Sie zu Einstellungen > Allgemein > File Station.
  2. Aktivieren Sie Ziegen und Ablegen zwischen Browsern aktivieren.
  3. Klicken Sie auf OK.
  4. Öffnen Sie zwei gleiche Webbrowser (z. B. Firefox) und stellen Sie mit ihnen die Verbindung zur Quell- bzw. zur Ziel- DiskStation her.
  5. Starten Sie File Station auf beiden DiskStation. Sie können jetzt über File Station Dateien/Ordner direkt zwischen Ihnen ziehen und ablegen.

Hinweis:

  • Sie können auch Dateien/Ordner zwischen zwei Registerkarten desselben Browsers ziehen und ablegen.
  • Von Internet Explorer wird Drag-and-Drop zwischen Browsern oder Tabs nicht unterstützt.
  • Werden mehrere Dateien/Ordner zum Ziehen und Ablegen ausgewählt, werden sie zum Hochladen in einem Archiv komprimiert.
  • Die maximale Dateigröße für das Ziehen und Ablegen unterscheidet sich je nach Webbrowser: Safari (1 GB).

Dateien/Ordner in einer DiskStation oder zu einem lokalen Computer kopieren oder verschieben

Gehen Sie folgendermaßen vor, um Dateien oder Ordner in einer DiskStation oder zu Ihrem Computer zu verschieben oder zu kopieren.

Methode 1:

  1. Wählen Sie die Dateien oder Ordner aus, die Sie verschieben oder kopieren möchten. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  2. Wählen Sie Ausschneiden oder Kopieren aus dem Dropdown-Menü Aktion aus.
  3. Wechseln Sie zum Zielordner auf der DiskStation oder Ihrem Computer.
  4. Wählen Sie Einfügen - Überschreiben oder Einfügen - Überspringen aus dem Dropdown-Menü Aktion aus, je nachdem, ob Sie Dateien oder Ordner mit dem gleichen Namen im Zielordner überschreiben möchten oder nicht.

Methode 2:

  1. Wählen Sie die Dateien oder Ordner aus, die Sie verschieben oder kopieren möchten. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  2. Ziehen Sie die Ordner oder Dateien vom Hauptbereich in einen anderen Ordner auf der linken Seite.
  3. Wählen Sie im angezeigten Popup-Menü Kopieren - Überschreiben, Kopieren - Überspringen, Verschieben - Überschreiben oder Verschieben - Überspringen aus, um Dateien oder Ordner mit demselben Namen im Zielordner zu überschreiben oder zu überspringen.

Methode 3

  1. Wählen Sie die Dateien oder Ordner aus, die Sie verschieben oder kopieren möchten. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  2. Drücken Sie Strg + X zum Ausschneiden oder Strg + C zum Kopieren der Dateien/Ordner. Mac-Benutzer drücken auf cmd + X und cmd + C.
  3. Wechseln Sie zum Zielordner auf der DiskStation oder Ihrem Computer.
  4. Drücken Sie Strg + V, um die Dateien/Ordner einzufügen (cmd + V für Mac).
  5. Wählen Sie im angezeigten Dialogfeld, ob Sie Dateien mit identischem Namen im Zielordner überschreiben oder überspringen möchten.

Tastenkombinationen:

Die folgenden Tastaturkombinationen können beim Verschieben/Kopieren von Dateien und Ordnern in File Station nützlich sein.

  • F2: Umbenennen
  • Entf: Löschen
  • Strg + A: Alle auswählen
  • Strg + X: Ausschneiden
  • Strg + C: Kopieren
  • Strg + V: Einfügen

Hinweis:

  • Beim Kopieren oder Verschieben von Dateien/Ordnern in ein Verzeichnis in einem FAT-Dateisystem werden die Zeichen \ *?">| im Dateinamen durch – ersetzt und die Metadaten von Mac-Dateien werden gelöscht.
  • Wenn Sie Dateien auf Ihrem lokalen Computer ausschneiden und in Ordner einer DiskStation einfügen, bleiben die Dateien auf dem lokalen Computer erhalten.
  • Beim Verschieben einer Datei oder eines Ordners bleibt der „Eigentümer“ gleich. Wenn eine Datei oder ein Ordner aber kopiert wird, wechselt der „Eigentümer“ zu dem Benutzer, der die Kopie angefertigt hat.
