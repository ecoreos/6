Dateianforderungen erstellen

Dateianforderungen sind Einladungen zum Hochladen von Dateien, die an Nicht-DSM-Benutzer geschickt werden. Mithilfe von Anforderungs-Links können Nicht-DSM-Benutzer mehrere Dateien direkt zu File Station auf Ihrem DiskStation hochladen, ohne dass sie entsprechende Benutzerkonten und Berechtigungen zum Hochladen von Dateien benötigen.

Links für Dateianforderung erstellen:

  1. Gehen Sie zu File Station und wählen Sie einen Ordner als Upload-Ziel aus.
  2. Klicken Sie auf Aktion Oder rechtsklicken Sie auf den Zielordner und wählen Sie Dateianforderung erstellen aus.
  3. Im Popup-Fenster können Sie die nachstehenden Informationen finden/ändern:
    • Dateipfad: Hier wird der Pfad des Zielverzeichnisses angezeigt.
    • Link: Hier wird der Anforderungs-Link angezeigt, der für bestimmte Benutzer freigegeben wird.
    • Ihr Name: Geben Sie einen Namen zur Identifizierung des Anforderers ein. Der voreingestellte Wert ist Ihr Benutzername.
    • Nachricht: Hier können Sie die Nachricht für die gewünschten Benutzer anpassen.
    • Kennwortschutz aktivieren: Wählen Sie diese Option aus, damit nur Benutzer mit dem Kennwort Dateien über den erstellten Link hochladen können. Geben Sie das Kennwort in das Feld unten ein.
    • Gültigkeitszeitraum: Klicken Sie auf diese Schaltfläche, um die Dauer dieses Links für Dateianforderung anzupassen:
      • Stoppzeit einrichten: Wählen Sie diese Option aus, um nach Datum und Zeit zu bestimmen, wann der Link zum Hochladen von Dateien nicht mehr funktioniert.
      • Startzeit einrichten: Wählen Sie diese Option aus, um nach Datum und Zeit zu bestimmen, ab wann der Link zum Hochladen von Dateien funktioniert.
      • Anzahl zulässiger Zugriffe: Wählen Sie diese Option aus, um festzulegen wie oft auf den Link zugegriffen werden kann.
  4. Klicken Sie auf Speichern, um die Einstellungen abzuschließen.

Links für Dateianforderung freigeben:

Gehen Sie, um nach der URL für Dateianforderungen zu suchen, zu File Station > Extras > Verwaltung freigegebener Links dnd doppelklicken Sie auf den gewünschten Link. Sie können den Link auf eine der nachstehenden Weisen freigeben:

  • Kopieren Sie einfach die URL für die betreffenden Benutzer.
  • Senden Sie den Link über die Standard-E-Mail (Ihr DSM-E-Mail-Konto) oder über E-Mail (eine E-Mail-Anwendung eines Drittanbieters). Erstbenutzer sollten der Anleitung des Assistenten folgen, um das zu verwendende E-Mail-Konto einzurichten.

Dateien hochladen:

Gehen Sie wie folgt vor, wenn Sie gebeten werden, Dateien über den Link hochzuladen:

  1. Öffnen Sie die angegebene URL auf dem lokalen Computer oder auf einem Mobilgerät (Android oder iOS). Geben Sie, soweit erforderlich, das Kennwort ein.
  2. Geben Sie auf der Upload-Seite einen Namen ein, der Sie identifiziert.
  3. Ziehen Sie einfach die hochzuladende(n) Datei(en) und legen Sie sie ab oder klicken Sie auf Dateien hinzufügen, um die Datei(en) auszuwählen.
  4. Klicken Sie auf Hochladen, um die Anforderung abzuschließen.

Hinweis:

  • Ordner können über einen Link für Dateianforderung nicht hochgeladen werden.
  • Ordner im freigegebenen Ordner "web" können nicht als Zielordner angegeben werden.
  • Die Upload-Funktion ist vom Browser abhängig:
    • Internet Explorer 8 und Internet Explorer 9 unterstützen nicht das Hochladen von Dateien per Drag-and-Drop.
    • Chrome auf iOS-Geräten unterstützt nur das Hochladen von Bildern.
    • Safari auf iOS-Geräten unterstützt nur das Hochladen von Bildern.
    • Für Browser auf Android-Geräten gelten keine Einschränkungen in Bezug auf den Dateityp.

Hochgeladene Dateien suchen:

Die von einem Benutzer hochgeladenen Dateien erscheinen im selben Ordner auf dem Ziel. Standardmäßig hat der Ordner den Namen Uploader_Name (Anforderer_Name).

Werden die Dateien beispielsweise von Watson zu Holmes hochgeladen, finden Sie die Dateien im Ordner mit dem Namen Watson (Holmes).

Verwalten von Links für Dateianforderung

Links für Dateianforderung ändern:

Nachdem Sie einen Link für Dateianforderung erstellt haben, können Sie die zugehörigen Einstellungen unter File Station > Extras > Verwaltung freigegebener Links ändern.

  1. Doppelklicken Sie auf den gewünschten Link oder wählen Sie den Link aus und klicken Sie auf Bearbeiten.
  2. Ändern Sie im Popup-Fenster die Einstellungen.
  3. Klicken Sie auf Speichern, um die neuen Einstellungen zu übernehmen.

Links für Dateianforderung löschen:

  1. Gehen Sie zu File Station > Extras > Verwaltung freigegebener Links.
  2. Wählen Sie den (die) gewünschten Link(s) aus und klicken Sie auf Löschen. Die entsprechende(n) URL(s) für Dateianforderung ist sofort ohne Funktion.

Berechtigungen für Dateianforderungen zuweisen:

Als admin oder als Benutzer der Gruppe administrators können Sie Benutzern Berechtigungen zuweisen, damit sie Links für Dateianforderung erstellen können.

  1. Gehen Sie zu File Station > Einstellungen > Freigegebene Links > Links für Dateianforderung.
  2. Wählen Sie die Benutzertypen aus, um ihnen Berechtigungen für Dateianforderungen zuzuweisen.
  3. Klicken Sie auf OK, um die Einstellungen zu speichern.
