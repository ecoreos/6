Eigene Favoriten oder Desktop-Verknüpfungen erstellen

Mit der File Station können Sie DiskStation Ordner zu Eigene Favoriten hinzufügen oder Desktop-Verknüpfungen für DiskStation Dateien oder Ordner erstellen.

Ordner zu Eigene Favoriten hinzufügen

Für häufig genutzte Unterordner auf Ihrer DiskStation können Sie unter Meine Favoriten oben auf der linken Seite Verknüpfungen hinzufügen, um direkt auf sie zugreifen zu können.

So fügen Sie einen DiskStation oder zu Meine Favoriten hinzu:

  1. Wählen Sie den Ordner aus und gehen Sie anschließend auf eine der folgenden Weisen vor:
    • Wählen Sie das Menü Aktion oder rechtsklicken Sie auf den Ordner. Wählen Sie anschließend Verknüpfung erstellen > Meine Favoriten.
    • Wenn Sie zuvor bereits einen anderen Ordner zu Eigene Favoriten hinzugefügt haben, wird Meine Favoriten oben links angezeigt. Ziehen Sie den Ordner dorthin, um eine Verknüpfung zu erstellen.
  2. Geben Sie im angezeigten Fenster einen Namen für die Verknüpfung ein und klicken Sie auf OK.

So ordnen Sie die Verknüpfungen unter Eigene Favoriten an:

Ziehen Sie eine Verknüpfung unter Meine Favoriten einfach an einen anderen Ort.

So geben Sie einer Verknüpfung einen anderen Namen:

  1. Klicken Sie mit der rechten Maustaste auf die Verknüpfung und wählen Sie Umbenennen.
  2. Geben Sie im angezeigten Fenster einen neuen Namen für den Favoriten ein und klicken Sie auf OK.

So löschen Sie eine Verknüpfung:

Rechtsklicken Sie auf die Verknüpfung und wählen Sie Aus Meine Favoriten entfernen. Klicken Sie anschließend auf Ja.

So löschen Sie alle Verknüpfungen zu nicht mehr existierenden Ordnern:

Rechtsklicken Sie auf Eigene Favoriten oben im linken Feld und wählen Sie anschließend Ungültige Favoriten löschen.

Desktop-Verknüpfungen für Dateien oder Ordner erstellen

Für häufig genutzte DiskStation Dateien oder Unterordner können Sie Desktop-Verknüpfungen erstellen, um schneller auf sie zugreifen zu können.

So erstellen Sie eine Desktop-Verknüpfung für eine DiskStation Datei oder einen Ordner:

Rechtsklicken Sie auf die Datei oder den Ordner im Hauptbereich. Gehen Sie anschließend auf eine der folgenden Weisen vor:

  • Wenn Sie eine Desktop-Verknüpfung für eine Datei erstellen, wählen Sie Desktopverknüpfung erstellen.
  • Wenn Sie eine Desktop-Verknüpfung für einen Ordner erstellen, wählen Sie Verknüpfung erstellen > DSM-Desktop.
