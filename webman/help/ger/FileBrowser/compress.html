Dateien oder Ordner komprimieren

  1. Wählen Sie im Hauptbereich Dateien oder Ordner von DiskStation aus.
  2. Wählen Sie Komprimieren zu “(Dateiname)”.zip im Dropdown-Menü Aktion, um die Komprimierung zu starten.
  3. Wenn Sie die Zip-Datei nicht sehen können, nachdem die Komprimierung abgeschlossen ist, wählen Sie Aktualisieren im Dropdown-Menü Aktion, damit die Datei angezeigt wird.
  4. Wenn Sie die Datei umbenennen, das Archiv verschlüsseln oder es mit einer anderen Komprimierungsstufe oder einem anderen Komprimierungsmodus archivieren möchten, wählen Sie stattdessen Zu Archiv hinzufügen….
    Hinweis:
  1. Sie können mit der File Station keine Dateien oder Ordner auf Ihrem lokalen Computer komprimieren.
  2. Unterstützte Archivformate: .zip, .7z.
  3. Wenn Sie unter Zu Archiv hinzufügen... .zip als Archivformat auswählen, können Sie die gewünschte Codepage für die .zip-Datei auswählen.
