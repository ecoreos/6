File Station-Einstellungen verwalten

Öffnen Sie File Station und klicken Sie auf die Schaltfläche Einstellungen, um die Einstellungen von File Station zu verwalten.

Allgemein

Hier können Sie das File Station Protokoll aktivieren, um die Aktivitäten aller Benutzer aufzuzeichnen. Die Benutzeraktivitäten umfassen Ordner erstellen, Hochladen, Herunterladen, Löschen, Umbenennen, Verschieben, Kopieren und Eigenschaft einstellen.

File Station Protokoll aktivieren:

  1. Wenn Sie die Aktivitäten aller Benutzer aufzeichnen möchten, aktivieren Sie File Station Protokoll aktivieren.
  2. Klicken Sie auf OK.

File Station Protokoll anzeigen:

  1. Klicken Sie auf Protokolle anzeigen, dann werden Sie zum Protokoll-Center weitergeleitet.
  2. Klicken Sie auf die Registerkarte Dateiübertragung.

Dateien automatisch über Upload per Ziehen und Ablegen oder Tastaturkürzel überschreiben:

Markieren Sie Immer Dateien überschreiben, wenn ich sie in einen Webbrowser ziehe und ablege oder mit Tastaturkürzeln einfüge, wenn Sie Dateien mit demselben Namen automatisch überschreiben möchten, wenn Sie sie vom Desktop Ihres Computers in einen Webbrowser ziehen und ablegen, oder wenn Sie Dateien mit Tastaturkürzeln innerhalb der File Station einfügen.

Intelligentes Ziehen und Ablegen aktivieren:

Wenn Sie Dateien innerhalb der File Station ziehen und ablegen, erkennt diese Funktion automatisch, ob Sie die Dateien kopieren oder verschieben.

  1. Wählen Sie Intelligentes Ziehen und Ablegen aktivieren.
  2. Wählen Sie eine der beiden folgenden Aktionen, die stets ausgeführt wird, wenn ein anderer Benutzer zur gleichen Zeit wie Sie Dateien mit demselben Namen an den gleichen Speicherort überträgt:
    • Überschreiben: Die von Ihnen übertragenen Dateien werden Dateien mit identischem Namen überschreiben.
    • Überspringen: Die von Ihnen übertragenen Dateien werden übersprungen und Dateien mit identischem Namen nicht überschreiben.
  3. Klicken Sie auf OK.

Hinweis:

Sie können die von intelligentes Ziehen und Ablegen bestimmte Aktion (kopieren oder verschieben) bei Bedarf mit den folgenden Hotkeys ändern:

  • Wechsel zu Kopieren: Drücken Sie während des Ziehens und Ablegens die Taste „Strg“.
  • Wechsel zu Verschieben: Drücken Sie während des Ziehens und Ablegens die Taste „Umsch“.

Mit File Station Dateien des lokalen Computers durchsuchen und verwalten:

  1. Aktivieren Sie Mit File Station Dateien des lokalen Computers durchsuchen.
  2. Klicken Sie auf OK.

DSM-Benutzerliste für Nicht-Admin-Benutzer freigeben:

Diese Option ermöglicht es Nicht-Admin-Benutzern, auf die vollständige DSM-Benutzerliste zuzugreifen, wenn Sie nach Dateien von bestimmten DSM-Benutzern/Gruppen suchen oder Dateilinks für andere DSM-Benutzer freigeben möchten.

  1. Wählen Sie DSM-Benutzerliste für Nicht-Admin-Benutzer freigeben.
  2. Klicken Sie auf OK.

Standard-UNIX-Befugnisse anwenden

Klicken Sie auf Erweiterte Einstellungen, und markieren Sie die Option, um die Standard UNIX-Befugnissse beim Hochladen oder Erstellen von Dateien und Ordnern anzuwenden. Angewandte Befugnisse sind identisch mit den Berechtigungen, die vom UNIX-Befehl umask angewendet werden. Wenn diese Option aktiviert ist, sind die UNIX-Berechtigungen für Dateien 644 und für Ordner 755. Wenn diese Option deaktiviert ist, sind die UNIX-Berechtigungen für Dateien 666 und für Ordner 777. Der Standardwert für „umask“ ist 022.

Hinweis:

  • Führen Sie für gemeinsame Ordner, für die Windows ACL aktiviert ist (alle gemeinsamen Ordner, außer "photo" und Freigaben auf externen Festplatten), den Befehl chmod auf Ihrem Linux- oder FTP-Client aus, um die Ordner- und Dateiberechtigungstypen von Windows ACL auf UNIX zu ändern.
  • Die Aktivierung dieser Option kann zu inkonsistenten Berechtigungen zwischen verschiedenen Protokollen führen. Um Inkonsistenzen zu vermeiden, empfehlen wir, diese Option deaktiviert zu lassen.

Anhängen/Verbindungen

Standardmäßig können alle Benutzer Remote-Ordner oder virtuelle Laufwerke anhängen. Hier können Sie einstellen, dass nur Administratoren auf die Verwaltung von Remote-Ordnern oder virtuellen Laufwerken zugreifen können.

Hinweis:

Benutzer, die keine Administratoren sind, können Remote-Ordner nur in ihrem Home-Ordner anhängen.

Anhängen von Remote-Ordnern oder virtuellen Laufwerken durch verschiedene Benutzer zulassen:

  1. Wählen Sie Alle Benutzer oder Administratoren unter Remote-Ordner oder Virtuelles Laufwerk aus.
  2. Klicken Sie auf OK.

Anhängen von Servern und Cloud-Diensten durch verschiedene Benutzer zulassen:

  1. Wählen Sie Alle Benutzer, Administratoren oder Bestimmter Benutzer aus.
  2. Durch die Auswahl Bestimmter Benutzer öffnet sich ein Fenster, in dem Sie für verschiedene Benutzer die Berechtigungen zum Anhängen konfigurieren können. Um den Zugriff für einen bestimmten Benutzer zu aktivieren, muss das Kontrollkästchen unter der Spalte Aktiviert aktiviert werden. Um den Zugriff für einen bestimmten Benutzer zu deaktivieren, muss das Kontrollkästchen unter der Spalte Aktiviert deaktiviert werden. Klicken Sie anschließend auf OK, um das Fenster zu schließen.

    3. Hinweis:

    • Benutzer, die zur Gruppe administrators gehören, können diese Funktion verwenden, aber keine Änderungen vornehmen. Benutzer, die sich als guest (Gast) angemeldet haben, können weder diese Funktion verwenden, noch Änderungen vornehmen. Berechtigungen für alle anderen Benutzer können hier festgelegt werden.
  3. Klicken Sie auf OK.

Freigegebene Links

Standardmäßig können alle Benutzer Dateilinks freigeben. Hier können Sie festlegen, dass nur Administratoren Zugriff auf die Verwaltung freigegebener Links haben.

Freigeben von Dateilinks durch verschiedene Benutzer zulassen:

  1. Wählen Sie Alle Benutzer, Administratoren oder Bestimmte Benutzer aus.
  2. Klicken Sie auf OK.

HTML-Dateien als Klartext öffnen:

  1. Aktivieren Sie das Kontrollkästchen HTML-Dateien immer im Klartext-Format öffnen. Wenn diese Option aktiviert ist, werden HTML-Dateien (d. h. HTM-, HTML-, JSP-, XHTML-Dateien) im Klartext-Format angezeigt, wenn die Benutzer sie über einen Dateifreigabe-Link öffnen. Diese Option verhindert, dass böswillige Dritte Benutzer-Anmeldesitzungen mithilfe von eingebetteten Skripts in HTML-Dateien übernehmen.
  2. Klicken Sie auf OK.

Geschwindigkeitseinschränkung

Sie können eine Geschwindigkeitseinschränkung aktivieren, um die maximal nutzbare Bandbreite für Benutzer und Gruppen zu regeln, die Daten über File Station übertragen.

Geschwindigkeitseinschränkung aktivieren:

  1. Aktivieren Sie das Kontrollkästchen Geschwindigkeitseinschränkung aktivieren oder Geschwindigkeitseinschränkung nach einem Zeitplan aktivieren, wenn Sie nur zu bestimmten Zeiten eine Geschwindigkeitseinschränkung festlegen möchten. Dies ist eine serverweite Einstellung, die die Einstellungen der einzelnen Benutzer und Gruppen überschreibt. Die Auswahl von Keine Geschwindigkeitseinschränkung deaktiviert die Einstellungen für die Geschwindigkeitseinschränkung aller Benutzer und Gruppen.
  2. Klicken Sie auf Einstellungen für Geschwindigkeitseinschränkung, um die Einstellungen für einzelne Benutzer oder Gruppen zu ändern. Für jeden Benutzer und jede Gruppe steht Folgendes zur Auswahl:
    • Gruppeneinstellungen anwenden: Die Geschwindigkeitseinschränkungseinstellungen sind identisch mit denen der Gruppe, zu der der Benutzer gehört.
    • Keine Geschwindigkeitseinschränkung: Es wird keine Geschwindigkeitseinschränkung angewendet, wenn Gruppenmitglieder Dateien mit dem Dienst übertragen.
    • Konstant: Für alle Benutzer und Gruppenmitglieder gilt eine feste, konstante Geschwindigkeitseinschränkung. Die Upload- und Download-Geschwindigkeitseinschränkungen können in den Feldern links angegeben werden.
    • Variabel: Für Benutzer oder Gruppenmitglieder können zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan festgelegt werden. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für Geschwindigkeitseinschränkung und einen Zeitplan festzulegen.
  3. Klicken Sie auf Übernehmen.

Hinweis:

  • Weitere Informationen über Geschwindigkeitseinschränkungen für Gruppen finden Sie hier.
  • Beim Festlegen einer Geschwindigkeitseinschränkung für einen Benutzer zeigt der Bereich Ergebnis die tatsächliche Geschwindigkeitseinschränkung des Benutzers gemäß seiner individuellen und der Gruppeneinstellungen an. Falls zum Beispiel ein Benutzer "Achilles" ein individuelles Download Limit von 100 kB/s hat, aber zu einer Benutzergruppe mit einem Downlod-Limit von 500 kB/s gehört, dann zeigt der Bereich Ergebnisse "0/500" an.

Sicherheit

Hier finden Sie die nachfolgend aufgeführten Sicherheitseinstellungen.

HTML-Dateien als Klartext öffnen:

  1. Aktivieren Sie das Kontrollkästchen HTML-Dateien immer im Klartext-Format öffnen. Wenn diese Option aktiviert ist, werden HTML-Dateien (d. h. HTM-, HTML-, JSP-, XHTML-Dateien) im Klartext-Format angezeigt, wenn sie in File Station geöffnet werden. Diese Option verhindert, dass böswillige Dritte Benutzer-Anmeldesitzungen mithilfe von eingebetteten Skripts in HTML-Dateien übernehmen.

    2. Hinweis:

    • Dies ist keine globale Einstellung. Jeder Benutzer kann gemäß seinen Anforderungen auswählen, ob diese Option aktiviert wird.
  2. Klicken Sie auf OK.
