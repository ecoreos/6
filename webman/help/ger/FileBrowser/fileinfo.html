Zusätzliche Attribute anzeigen

Sie können zusätzliche Attribute von Dateien oder Ordner anzeigen, um sie in neuen Spalten des Hauptbereichs anzuzeigen.

So zeigen Sie zusätzliche Attribute an:

  1. Fahren Sie mit Ihrer Maus auf eine Spaltenübersicht und klicken auf das nach unten zeigende Dreieck, das auf der rechten Seite angezeigt wird.
  2. Wählen Sie im angezeigten Dropdown-Menü das Untermenü Spalten.
  3. Markieren Sie das Kontrollkästchen neben den zusätzlichen Attributen, um diese in einer neuen Spalte anzuzeigen.
