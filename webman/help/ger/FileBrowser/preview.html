Dokumente, Fotos oder Videos anzeigen

Mit File Station können Sie Dokumente, Videos oder Fotos auf einfache Art und Weise anzeigen.

Anzeige der Elemente ändern:

Sie können zwischen einer Listenansicht und einer Symbolansicht wechseln. Klicken Sie auf die Schaltfläche für die Listenansicht oben rechts, um die Dateien in einer Liste anzuordnen oder auf die Schaltfläche für die Symbolansicht, um sie mit entsprechenden Symbolen und Text anzuzeigen.

Unter den folgenden Umständen werden Miniaturbilder von Fotos in der Symbolansicht angezeigt:

  • Die Fotos befinden sich in indizierten Ordnern unter Systemsteuerung > Indizierungsdienst > Medienindizierung > Indizierter Ordner und Photo Station oder Media Server ist auf Ihrem DiskStation bereits installiert.
  • Die in den EXIF-Einträgen Ihrer Digitalkamera gespeicherten Miniaturansichten sind verfügbar.

So zeigen Sie Dokumente in Google Docs an:

Klicken Sie einfach auf das Dokument, das Sie öffnen möchten, und wählen Sie dann In Google Docs anzeigen.

Unterstützte Formate:

Google docs-Viewer unterstützt die folgenden Formate: doc, docx, ppt, xlsx, pdf, tif

Hinweis:

Um Dokumente in Google Docs anzuzeigen, muss Ihr DiskStation uneingeschränkt über das Internet erreichbar sein. Wenn sich Ihr DiskStation hinter einem Router befindet, müssen Sie unter Systemsteuerung > Externer Zugriff > Routerkonfiguration die Portweiterleitungsregel für DSM angeben. Die Standard-Portnummern für DSM sind 5000 (HTTP) und 5001 (HTTPS). Wenn Sie immer noch Schwierigkeiten mit der Anzeige von Dokumenten in Google Docs haben, müssen Sie möglicherweise unter Systemsteuerung > Externer Zugriff > Erweitert den Router-Port für DSM angeben.

So zeigen Sie Dokumente in Office Online an:

Klicken Sie einfach auf das Dokument, das Sie öffnen möchten, und wählen Sie dann In Office Online anzeigen.

Falls Sie Office Viewer verwenden, um auf Ihrem DiskStation gehostete Microsoft Office-Dokumente anzuzeigen, können Sie auf diese Dokumente mit voll funktionsfähigen Office-Funktionen über einen Webbrowser zugreifen, ohne Microsoft Office auf Ihrem Computer installiert zu haben.

Unterstützte Formate:

Office Viewer unterstützt folgende Formate:

  • Word: docx, docm, dotm, dotx
  • Excel: xlsx, xlsb, xls, xlsm
  • PowerPoint: pptx, ppsx, ppt, pps, pptm, potm, ppam, potx, ppsm

Hinweis:

  • Um Dokumente in Office Online anzuzeigen, muss QuickConnect aktiviert sein. Falls dies nicht der Fall ist, melden Sie sich bei DSM als Benutzer der Gruppe administrators an und gehen Sie zu Systemsteuerung > QuickConnect.
  • Word- und PowerPoint-Dokumente müssen kleiner als 10 MB und Excel-Dokumente kleiner als 5 MB sein.
  • Der Dateiname des Dokuments darf nur Buchstaben und Zahlen enthalten.

Vorschau der Fotos anzeigen:

  1. Klicken Sie mit der rechten Maustaste auf ein Foto und wählen Sie Vorschau, um das Fenster „Photo Viewer“ anzuzeigen.
  2. Führen Sie den Mauszeiger unten über den Photo Viewer, um die Symbolleiste für eine schnelle Navigation aufzurufen.

Unterstützte Formate:

Photo Viewer unterstützt die folgenden Formate: jpg, jpeg, jpe, bmp, gif, png, tiff, tif, arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, 3fr, erf, mos, orf, rw2, raw, psd

Audiodateien (Vorschau) wiedergeben:

  • Audiodateien wiedergeben: Doppelklicken Sie auf eine oder mehrere Audiodateien, um sie mit Audio-Player zu öffnen. Sie können Audiodateien auch wiedergeben, indem Sie auf eine Audiodatei rechtsklicken und Wiedergeben auswählen.*
  • Audiodateien nach Ordner wiedergeben: Klicken Sie auf einen Ordner oder drücken Sie auf Ctrl (Windows-Benutzer)/Command (Mac-Benutzer) und klicken Sie auf mehrere Ordner. Rechtsklicken Sie dann auf den (die) ausgewählten Ordner und wählen Sie Wiedergeben aus, um sie mit Audio-Player zu öffnen.*

Unterstützte Formate:

Audio-Player unterstützt die folgenden Formate:

  • mp3, m4a, m4b, aac, ogg, wav, flac, ape, aiff, aif, wma.

Hinweis:

Sie können bis zu 1024 Audiodateien gleichzeitig wiedergeben.

Videos wiedergeben:

  1. Klicken Sie mit der rechten Maustaste auf das Video, und wählen Sie Wiedergeben, um das Fenster „Video Player“ zu öffnen.
  2. Das Video sollte sofort wiedergegeben werden. Ist dies nicht der Fall, stellen Sie bitte sicher, dass der VLC Player (Windows) auf Ihrem lokalen Computer installiert ist.

    3. Hinweis:

    • Stellen Sie bei der Installation von VLC Player sicher, dass das Mozilla-Plugin und das ActiveX-Plugin aktiviert sind.
    • VLC wird von folgenden Browsern unterstützt:
      Windows: Internet Explorer, Firefox, Google Chrome
    • Wenn VLC auf Ihrem Computer installiert ist und Sie sich über einen unterstützten Webbrowser mit DSM verbinden, werden folgende Formate von Video Player unterstützt: avi, dat, mkv, vob, asf, wmv, mov, qt, divx, flv, swf, rm, rmvb, mpg, mpeg, mpeg1, m1v, mpeg2, m2v, mp2, dvr-ms, m2ts, mts, m2t, ts, tp, trp, mpeg4, m4v, mp4, 3g2, 3gp, 3gp2, webm, mpe, xvid
    • Wenn VLC nicht auf Ihrem lokalen Computer installiert ist, können von HTML5 unterstützte Videos von Video Player wiedergegeben werden. Informationen über von verschiedenen Webbrowsern unterstützte HTML5-Videos finden Sie hier.
    • Wenn Sie sich mit DSM über das HTTPS-Protokoll verbinden, müssen Sie ein vertrauenswürdiges Zertifikat importieren (unter Systemsteuerung > Sicherheit > Zertifikat), um Videos mit File Station wiederzugeben.
    • VLC unterstützt kein HTTPS-Streaming.
  3. Wenn es eine Untertiteldatei mit demselben Namen und im selben Ordner wie das Video gibt, oder Ihr Video eingebettete Untertitel enthält, können Sie die Untertitel während der Videowiedergabe sehen. Wenn Sie eine andere Untertiteldatei verwenden möchten, wählen Sie sie im Dropdown-Menü aus.

* Bei bestimmten Modellen nicht verfügbar.
