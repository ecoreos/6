Support-Dienste

Die Support-Dienste umfassen Fernzugriff und Protokollerstellungs-Tools, sodass unsere technischen Support-Ingenieure schnell technische Probleme von einem Synology-Büro aus auf Ihrem DiskStation anzeigen, diagnostizieren und beheben können. Hierzu gehen Sie zu Support-Center > Support-Dienste.

So wird der Fernzugriff aktiviert:

Aktivieren Sie den Fernzugriff auf Ihrem DiskStation nur, wenn die Synology Support-Ingenieure darum bitten.

  1. Markieren Sie das Kontrollkästchen Fernzugriff aktivieren, und klicken Sie dann auf Übernehmen, damit die Änderungen wirksam werden.
  2. Geben Sie den Synology Support-Ingenieuren Ihr Admin-Passwort und den Support-Identifikationsschlüssel per E-Mail.

Hinweis:

  • Der Fernzugriff wird nach 14 Tagen automatisch deaktiviert. Er sollte nicht manuell deaktiviert werden, falls kein Synology Support-Techniker Sie dazu auffordert.
  • Bitte entfernen Sie alle Inhalte vom DiskStation, die für Sie als privat gelten.
  • Obwohl Synology nicht garantieren kann, dass der Fernzugriff das Problem beheben wird, unternimmt Synology angemessene Anstrengungen, um die Support-Dienste in einer professionellen Art und Weise durchzuführen.

So werden Protokolle erstellt:

Erstellen Sie Protokolle nur für und geben Sie sie nur an Synology Support-Ingenieure weiter.

  1. Klicken Sie auf Protokolle erstellen, dann warten Sie ein paar Minuten, bevor der Download abgeschlossen ist.
  2. Senden Sie die Protokolle per E-Mail an die Synology Support-Ingenieure.

Wichtiges über unsere Support-Dienste

  • Wir behandeln Ihren DiskStation, als ob es unser eigener wäre.
  • Ihr DiskStation wird nur für Systemdiagnosezwecke verwendet werden. Synology wird auf andere persönliche Daten auf Ihrem DiskStation nicht zugreifen und sie nicht offen legen.
