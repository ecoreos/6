Synology-Konto

Mit dem Synology-Konto können Sie mühelos auf die mit Ihrem DiskStation und Ihren Paketen verknüpften Einstellungen Ihrer Synology-Kontodienste und eNews zugreifen und diese verwalten.

  • Konto: Über dieser Option können Sie Ihre Synology-Kontoinformationen anzeigen und bearbeiten.
  • Kaufverlauf: Über diese Option können Sie Ihre Aktivitäten in Bezug auf Paketkäufe durchsuchen.
  • eNews: Wählen Sie eine oder mehrere Kategorien von eNews aus, die Sie von Synology erhalten möchten, und um den abonnierten Inhalt zu personalisieren.
  • Abmelden: Über diese Option beenden Sie alle mit ihrem DiskStation verknüpften Synology-Kontodienste.

Bei Synology-Konto anmelden:

  1. Gehen Sie zu Synology-Konto.
  2. Geben Sie im Popup-Fenster Bei Ihrem Synology-Konto anmelden die E-Mail-Adresse und das Kennwort ein, die Sie für die Registrierung eines Synology-Kontos verwendet haben.
    • Wenn Sie kein Synology-Konto besitzen, klicken Sie zur Registrierung auf Sie benötigen ein Synology-Konto?.
    • Klicken Sie, wenn Sie Ihr Kennwort vergessen haben, auf Kennwort vergessen?, um ein neues Kennwort per E-Mail zu erhalten, das von System erstellt wurde.
  3. Klicken Sie auf Anmelden, um auf die Dienste Ihres Synology-Kontos zuzugreifen.

Synology-Konto anzeigen und bearbeiten:

  1. Klicken Sie im linken Fenster auf Konto, um Ihre Kontoinformationen anzuzeigen.
  2. Klicken Sie in das Feld, um die Kontoinformationen zu ändern.
  3. Um die Einstellungen zu speichern, klicken Sie auf Übernehmen.

Kaufverlauf Ihres DiskStation anzeigen:

  1. Klicken Sie im linken Fenster auf Kaufverlauf.
  2. Wählen Sie eine der Möglichkeiten aus, um den Paketkaufverlauf Ihres DiskStation zu finden.
    • Für die Suche nach Schlüsselwörtern wählen Sie Suche nach Schlüsselwort.
      • Geben Sie das Schlüsselwort ein und wählen Sie den entsprechenden Filter aus.
    • Für die Suche nach Kaufdatum wählen Sie Suche nach Datum aus.
      • Klicken Sie auf die Kalendersymbole, um einen Zeitraum festzulegen, in dem Sie Ihren DiskStation oder die Pakete gekauft haben.
  3. Klicken Sie auf Suchen.
  4. Die abgefragten Paketkaufinformationen (z. B. Kaufdatum und -preis) werden angezeigt.

Synology eNews abonnieren:

  1. Klicken Sie im linken Fenster auf eNews.
  2. Aktivieren Sie unter eNews-Einstellungen eine oder mehrere eNews-Kategorien, die Sie von Synology erhalten möchten.
  3. Sie können in den Dropdown-Listen eine Feinabstimmung der eNews-Einstellungen vornehmen:
    • Neue Produkt- und Funktionsversionen: Wählen Sie eine Abonnementsprache aus.
    • Informationen aus erster Hand über Werbeveranstaltungen in Ihrer Nähe: Wählen Sie ein Land aus, um sich über die dort von Synology durchgeführten Veranstaltungen zu informieren.
    • Wichtige Updates für DSM: Wählen Sie eine oder mehrere DSM-Versionen für die entsprechenden Updates aus.
  4. Um die Einstellungen zu speichern, klicken Sie auf Übernehmen.

Von Synology-Konto abmelden:

  1. Klicken Sie unten links auf die Schaltfläche Abmelden.
  2. Nach dem Abmelden stehen alle Synology-Kontodienste (z. B. DDNS und QuickConnect) und -Pakete, die mit dem abgemeldeten Konto verknüpft sind, nicht mehr für Ihren DiskStation zur Verfügung.

Hinweis:

  1. Sobald Sie die E-Mail-Adresse ändern, wird eine Bestätigungsmeldung an die neue Adresse gesendet, um die Änderung zu überprüfen.
  2. Nach der Abmeldung bleiben die Pakete auf Ihrem DiskStation installiert, aber ihre Lizenzen werden ungültig.
