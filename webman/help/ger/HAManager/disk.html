Festplattenstatus

Diese Seite enthält Registerkarten, die den Status der Festplatten im High-Availability Cluster anzeigen. Jede Registerkarte eines Servers zeigt die folgenden Informationen.

  • Nummer: Zeigt an, in welchem Steckplatz sich die Festplatte befinden. Festplatten in den aktiven und passiven Servern, welche dieselben Steckplätze belegen, sollten gleich groß sein.
  • Plattengröße: Zeigt die Kapazität jeder Festplatte an.
  • Temperatur: Zeigt die Festplattentemperatur an.
  • S.M.A.R.T. Status: Zeigt den S.M.A.R.T. Status jeder Festplatte an.
  • Status: Zeigt Festplatteninformationen und Fehlerbenachrichtigungen an.

So tauschen Sie defekte Festplatten aus:

  1. Prüfen Sie die Festplattennummer.
  2. Tauschen Sie die defekte Festplatte durch eine neue aus. Für die Schritte hinsichtlich der Festplatteninstallation, lesen Sie bitte Schnellinstallationsanleitung Ihres Synology Produktes.
  3. Gehen Sie zu Speicherstatus.
  4. Wählen Sie das beschädigte Volume.
  5. Klicken Sie auf Reparieren, um das Volume zu reparieren.

Hinweis:

Die Kapazität von Ersatzfestplatten muss genau so groß oder größer als die der defekten Festplatte sein.

So fügen Sie neue Festplatten zu einem High-Availability Cluster hinzu:

Festplatten können zum High-availability Cluster hinzugefügt werden, um die Kapazität zu erweitern. Lesen Sie bitte Folgendes, bevor Sie Festplatten hinzufügen.

  • Wir empfehlen das Hinzufügen von Festplatten, wenn der aktive und der passive Server online sind.
  • Die Kapazität der Festplatten im passiven Server muss gleich groß oder größer sein als die Festplatten im aktiven Server.

Festplatten hinzufügen:

  1. Fügen Sie die Festplatten in die entsprechenden Steckplätze im aktiven und passiven Server ein. Wenn Sie zum Beispiel eine Festplatte in Steckplatz 1 des aktiven Servers einfügen, müssen Sie die entsprechende Festplatte in Steckplatz 1 des passiven Servers einfügen.
  2. Gehen Sie zu Speicher-Manager > Volume (oder Diskgruppe/iSCSI LUN/RAID/Gruppe je nach Ihrer Situation und Ihrem Modell). Wählen Sie das zu erweiternde Volume aus, klicken Sie auf die Schaltfläche Verwalten und befolgen Sie die Anweisungen, um Festplatten hinzuzufügen.
  3. Kehren Sie, nachdem Sie die Festplatten erfolgreich hinzugefügt haben, zu High Availability Manager > Speicherstatus zurück, um zu prüfen ob der Status Normal ist.
