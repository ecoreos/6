Speicherübersicht

Die Speicherübersicht zeigt den Status der installierten Festplatten, Kabelverbindungen und Hardware-Statusleuchten an.

Hardware-Statusleuchten

Die Hardware-Statusleuchten an der rechten Seite leuchten in den folgenden beiden Farben;

  • Grün: Der Hardware (z.B. Stromversorgung, Lüfter, Verbindungen zwischen Server und Erweiterungseinheiten, usw.) Status ist normal.
  • Rot: Ein Hardware-Fehler ist aufgetreten.

Wenn Sie auf eine Statusleuchte klicken, wird ein eine Liste mit dem detaillierten Status der Hardware angezeigt, einschließlich Stromversorgung, Lüfter und Verbindungen.

Festplattennummer anzeigen/ausblenden

Sie können die Anzahl der jeweils installierten Festplatten über Festplattennummer anzeigen/ausblenden anzeigen oder ausblenden.
