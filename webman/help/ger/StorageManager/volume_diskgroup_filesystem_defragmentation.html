Dateisystem-Defragmentierung*

Wenn Ihre Daten kontinuierlich geändert werden, nimmt die Anzahl der fragmentierten Dateien im Dateisystem zu, wodurch der Zugriff auf die Dateien auf den Festplatten verlangsamt wird. Mit der Dateisystem-Defragmentierung können Sie die Leistung verbessern.

Dateisystem-Defragmentierung durchführen:

  1. Öffnen Sie den Speicher-Manager.
  2. Gehen Sie zu Volume.
  3. Wählen Sie das Volume aus, auf dem die Dateisystem-Defragmentierung durchgeführt werden soll.
  4. Klicken Sie auf Verwalten.
  5. Wählen Sie Dateisystem-Defragmentierung aus.

Hinweis:

  • Die Dateisystem-Defragmentierung kann nur auf Volumes im Btrfs-Dateisystem durchgeführt werden.
  • Die Dateisystem-Defragmentierung kann nur durchgeführt werden, wenn das Volume den Status Normal hat.
  • Fahren Sie das System nicht herunter, bevor die Dateisystem-Defragmentierung abgeschlossen ist. Wenn das System vorher heruntergefahren wird, muss die Dateisystem-Defragmentierung wieder von neuem gestartet werden.
  • Die Volume-Auslastung könnte sich erhöhen, wenn das Volume Schnappschüsse von freigegebenen Ordnern enthält.

____
*Nur bei bestimmten Modellen verfügbar.
