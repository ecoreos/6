HDD/SSD

In diesem Abschnitt können Sie den Status der auf dem DiskStation installierten HDD/SSD durch Optionen wie die Verwaltung und Analyse der HDD/SSD-Leistung und deren Zustand überwachen. Gehen Sie zu Speicher-Manager > HDD/SSD, um diese Optionen zu verwalten.

So werden HDD/SSD-Infos überprüft:

  1. Wechseln Sie zur Registerkarte HDD/SSD.
  2. Wählen Sie ein Laufwerk aus der Liste aus und erweitern Sie es. Hier werden S.M.A.R.T.-Status, Volume-Info, Modellbezeichnung, Seriennummer, Firmware-Version und die Temperatur angezeigt.

So werden Informationen zum Integritätsstatus überprüft:

  1. Wechseln Sie zur Registerkarte HDD/SSD.
  2. Wählen Sie ein Laufwerk.
  3. Klicken Sie auf Integritätsstatus.
  4. Gehen Sie zur Registerkarte Überblick. Hier erfahren Sie, wie viele Stunden das Laufwerk eingeschaltet war, die aktuelle Temperatur, Anzahl der neuen Verbindungen/der erneuten Identifikationen, die fehlerhafte Sektorenanzahl und den gesamte S.M.A.R.T.-Status.
  5. Gehen Sie zur Registerkarte S.M.A.R.T.-Informationen. Die aufgelisteten Zahlen können Ihnen bei der Diagnose Ihres Laufwerksstatus helfen.

Hinweis:

SAS-Festplatten unterstützen keine S.M.A.R.T. Info.

S.M.A.R.T.-Test starten:

Mit dem S.M.A.R.T.-Test können Sie den Status Ihrer Festplatten untersuchen und ausgeben, sodass Sie auf mögliche Festplattenfehler hingewiesen werden. Wir empfehlen den umgehenden Austausch der Festplatten, wenn Fehler erkannt werden.

  1. Wechseln Sie zur Registerkarte HDD/SSD.
  2. Wählen Sie ein Laufwerk.
  3. Klicken Sie auf S.M.A.R.T.-Test.
  4. Wählen Sie einen Testtyp und klicken Sie auf Start, um mit dem Test zu beginnen.
  5. Die letzten Testergebnisse werden im Abschnitt Testergebnis angezeigt.

Hinweis:

Die Testdauer hängt von der Systemumgebung ab.

S.M.A.R.T.-Tests planen:

S.M.A.R.T. Tests können über das Erstellen von Aufgaben für eine automatische Ausführung geplant werden.

  1. Gehen Sie zur Registerkarte Test-Zeitplaner.
  2. Klicken Sie auf Erstellen, um eine neue Aufgabe einzurichten.
  3. Auf der Registerkarte Allgemein können Sie die folgenden Einstellungen ändern:
    • Aufgabe: Geben Sie einen Namen für die Aufgabe ein.
    • S.M.A.R.T. Test Typ: Wählen Sie aus, ob ein Schnelltest oder erweiterter Test durchgeführt werden soll.
    • S.M.A.R.T. Test Bereich: Wählen Sie aus, welche HDD/SSD bei jeder Ausführung des S.M.A.R.T.-Tests überprüft werden soll.
  4. Auf der Registerkarte Zeitplan können Sie die folgenden Einstellungen ändern:
    • Datum: Geben Sie an, wann die S.M.A.R.T. Tests durchgeführt werden sollen. S.M.A.R.T. Tests können an bestimmten Tagen der Woche (z.B. täglich, am Wochenende, jeden Sonntag usw.) oder an bestimmten Daten (z.B. 21.10.2015) ausgeführt werden.
    • Zeit: Geben Sie, wie oft der S.M.A.R.T. Test durchgeführt werden soll, sowie die das erste und letzte Mal für jeden Tag.
  5. Klicken Sie zum Beenden auf OK.

Secure Erase ausführen:

  1. Wechseln Sie zur Registerkarte HDD/SSD.
  2. Wählen Sie ein Laufwerk, und klicken Sie auf Secure Erase.

Hinweis:

  • "Secure Erase" kann auf Festplatten in Expansionseinheiten nicht durchgeführt werden.
  • "Secure Erase" wird alle Daten auf Ihrem Laufwerk komplett löschen. Stellen Sie bitte sicher, dass das gewählte Laufwerk nicht in Betrieb ist oder die Daten gesichert wurden, bevor Sie fortfahren.
  • Wenn das System "Secure Erase" durchführt, wird das gewählte Laufwerk vorübergehend gesperrt und dann automatisch entsperrt, wenn der Vorgang abgeschlossen ist.
  • Wenn "Secure Erase" während der Operation unsachgemäß angehalten wurde, können Sie "Secure Erase" direkt auf der Benutzeroberfläche erneut ausführen. Wenn die Laufwerke entfernt oder anderweitig verwendet werden, wenn diese noch gesperrt sind, dann geben Sie das Kennwort "Synology" ein, um die Laufwerke zu entsperren.

Schreibcache-Unterstützung für HDD aktivieren

Die Aktivierung der Schreibcache-Unterstützung verbessert die Leistung des DiskStation. Nur bestimmte HDD-Modelle unterstützen diese Funktion. Um den Datenschutz bei Verwendung der Schreibcache-Unterstützung zu gewährleisten, sollte unbedingt ein USV-Gerät verwendet werden. Wir empfehlen zudem, das System jedes Mal nach der Verwendung ordnungsgemäß herunterzufahren. Bei Deaktivierung des Schreibcaches ist das Risiko des Datenverlustes durch Stromausfall geringer, doch verringert sich auch die Systemleistung.

So wird die Schreibcache-Unterstützung aktiviert:

  1. Gehen Sie zur Registerkarte Cache-Management.
  2. Markieren Sie Schreibcache aktivieren. Diese Funktion ist durch die Standardeinstellung aktiviert.
  3. Klicken Sie auf Übernehmen.
