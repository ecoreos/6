Pakete installieren oder kaufen

Das Paketzentrum bietet eine reiche Vielfalt von Webanwendungen, die die Funktionalität Ihres DiskStation erweitern. Im Paketzentrum können Sie Pakete mit nur wenigen einfachen Klicks herunterladen, installieren und starten.

Im folgenden Artikel wird erläutert, wie Sie Pakete über das Paketzentrum auf Ihrem DiskStation installieren. Weitere Informationen zu den einzelnen Paketen finden Sie hier.

Pakete installieren oder kaufen:

  1. Öffnen Sie das Paketzentrum.
  2. Gehen Sie zur Seite Alle, um alle verfügbaren Pakete zu sehen.

    3. Hinweis:

    • Wenn Sie das gewünschte Paket nicht finden können, benötigt es möglicherweise eine spätere Version von DSM oder ist nur auf Modellen mit bestimmten Hardware-Anforderungen verfügbar. Weitere Informationen erhalten Sie vom Entwickler dieses Pakets.
  3. Gehen Sie zum Paket, das Sie installieren möchten, und klicken Sie auf Installieren. (Klicken Sie für Bezahlpakete auf Kaufen oder Ausprobieren, um sich mit dem Synology-Konto zu verbinden, oder verwenden Sie die Demoversion zum Testen.)
  4. Nach der erfolgreichen Installation des Pakets müsste es im Hauptmenü angezeigt werden.

Kaufen von Paketen:

Beachten Sie die nachstehenden Hinweise, bevor Sie Pakete kaufen.

  • Registrieren Sie ein Synology Synology-Konto, um Zugriff auf mehr Funktionen zu haben, wie etwa Vorabinformationen zur Abrechnung und Kaufverlauf. Die Angaben, die Sie hier machen, werden nur zur künftigen Verbesserung der Dienste verwendet.
  • Das Paketzentrum akzeptiert nur Onlinezahlungen über PayPal. Wenn Sie eine Rückerstattung wünschen, besuchen Sie bitte die offizielle PayPal-Webseite.
  • Die Pakete sind gegenwärtig in US-Dollar bepreist.

Haftungsausschluss bei Währungsumrechnung:

Für Transaktionen, die eine Währungsumrechnung umfassen, kann der Aussteller Ihrer Kredit-/Debitkarte zusätzliche Gebühren verlangen, die nicht im angezeigten Preis enthalten sind. Für Käufe könnten Gebühren für Fremdwährung oder grenzüberschreitende Zahlungen vom Kartenaussteller verlangt werden. Sämtliche anfallenden Gebühren werden von Ihrem Kartenaussteller bemessen; Synology kann dafür keine Rückzahlung leisten. Weitere Informationen erhalten Sie vom Aussteller Ihrer Kredit-/Debitkarte.

Pakete manuell installieren:

  1. Klicken Sie, wenn Sie ein Paket durch Hochladen der .spk-Datei (im Download-Zentrum von Synology oder auf der Website von Drittanbietern verfügbar) installieren möchten, auf Manuelle Installation und folgen Sie dem Assistenten, um das neue Paket zu installieren.

Nach einem Paket suchen:

  1. Geben Sie in die Suchleiste oben links ein Schlüsselwort ein und drücken Sie die Eingabetaste, um mit der Suche zu beginnen.

Sortierung von Paketen auswählen:

Sie können die Reihenfolge der anzuzeigenden Pakete im Paketzentrum ändern.  Wählen Sie eine Kategorie aus, klicken Sie auf das Listensymbol oben rechts und wählen Sie die Sortierung von Paketen nach einer der folgenden Optionen aus:

  • Sortieren nach Name: Klicken Sie auf diese Option, um die Pakete in alphabetischer Reihenfolge zu sortieren.
  • Sortieren nach aktueller Beliebtheit: Klicken Sie auf diese Option, um die Pakete nach der aktuellen Installationszahl zu sortieren, wobei die beliebtesten zuerst angezeigt werden.

Installierte Pakete ausblenden:

Wählen Sie eine Kategorie aus, klicken Sie oben rechts auf das Listensymbol und wählen Sie Installierte Pakete nicht anzeigen aus. Dadurch werden Pakete ausgeblendet, die bereits installiert wurden, und die angezeigt, die noch installiert, aktualisiert oder repariert werden sollen.
