Paket-Zentrum konfigurieren

So verwalten Sie, welche Pakete installiert werden dürfen:

  1. Wählen Sie unter der Registerkarte Allgemein die folgenden Optionen, um Ihre gewünschte Vertrauensebene festzulegen:
    • Synology Inc.: Nur die Installation von Paketen erlauben, die von Synology Inc. veröffentlicht wurden.
    • Synology Inc. und vertrauenswürdige Herausgeber: Nur die Installation von Paketen erlauben, die von Synology Inc. und vertrauenswürdigen Herausgebern veröffentlicht wurden. Vertrauenswürdige Herausgeber sind Herausgeber, die das Vertrauen von Drittanbietern über das Hinzufügen einer Paketquelle erhalten haben oder denen Benutzer Vertrauen entgegenbringen, indem sie das Zertifikat des Herausgebers importieren.
    • Jeder Herausgeber: Installation von jedem Paket, unabhängig vom Herausgeber und der Integrität, zulassen.
  2. Klicken Sie auf OK.

Ein vertrauenswürdiges Herausgeber-Zertifikat importieren:

  1. Klicken Sie unter der Registerkarte Zertifikat auf Importieren.
  2. Klicken Sie auf Durchsuchen, und wählen Sie das vertrauenswürdige Zertifikat des Herausgebers.
  3. Klicken Sie auf OK.

Synology Inc.-Zertifikat aktualisieren:

Wenn Sie mit dem Internet verbunden sind, wird das Zertifikat von Synology Inc. automatisch während der Paketinstallation aktualisiert. Manuell aktualisieren:

  1. Klicken Sie unter der Registerkarte Zertifikat auf Importieren.
  2. Klicken Sie auf Durchsuchen, und wählen Sie das Zertifikat von Synology Inc., das von der Synology-Websiteheruntergeladen werden kann.
  3. Klicken Sie auf OK.

Benachrichtigungen über Paketaktualisierungen aktivieren:

  1. Klicken Sie auf die Registerkarte Benachrichtigung und anschließend auf E-Mail-Benachrichtigung aktivieren und/oder Desktop-Benachrichtigung aktivieren. Die erste Option ermöglicht DSM, Ihnen E-Mails zuzusenden, wenn neue Updates vorhanden sind. Bei der zweiten Option zeigt DSM eine Benachrichtigung oben rechts auf dem DSM-Desktop an.
  2. Klicken Sie auf OK.

Hinweis:

Damit E-Mail-Benachrichtigungen wirksam werden, konfigurieren Sie zuerst die E-Mail-Benachrichtigungseinstellungen von DSM (unter Hauptmenü > Systemsteuerung > Benachrichtigung).

So fügen Sie eine Paket-Ressource eines Drittanbieters hinzu:

  1. Klicken Sie auf die Registerkarte Paketquellen, klicken Sie auf Hinzufügen und geben Sie anschließend den Namen und den Ort Ihrer Paketquellen ein.
    Der Speicherort kann eine URL aus dem Internet sein, die von einem Drittanbieter-Entwickler oder einer Community zur Verfügung gestellt wird.
  2. Klicken Sie auf OK. Nachdem Sie die Paketquelle erfolgreich hinzugefügt haben, wird eine Liste mit Paketen auf der Registerkarte Community angezeigt.
