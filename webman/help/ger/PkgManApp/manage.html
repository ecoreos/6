Pakete verwalten

Nach der manuellen oder Online-Installation von Paketen können Sie sie im Paket-Zentrum verwalten, damit sie immer aktualisiert sind. Die unten stehenden Anweisungen erklären die Ausführung, Aktualisierung, Deinstallation, das Starten und Stoppen von Paketen und die Fehlerbehebung bei Paketproblemen.

Gehen Sie auf eine der folgenden Weisen vor, um Pakete zu aktualisieren:

Halten Sie Ihre Pakete immer auf dem aktuellen Stand, um den guten Zustand von DiskStation zu bewahren.

  • Klicken Sie im linken Feld auf Aktualisieren, um verfügbare Aktualisierungen anzuzeigen. Klicken Sie auf die Schaltflächen Aktualisieren oder Alle aktualisieren, um Pakete zu aktualisieren.
  • Klicken Sie auf Manuelle Installation, um ein Paket durch Hochladen einer .spk-Datei (im Synology Download-Zentrum oder auf Websites von Drittanbietern verfügbar) zu installieren.
  • Die Paketaktualisierungen werden gemäß den Optionen unter Einstellungen > Automatische Updates angezeigt. Weitere Informationen finden Sie hier.

Hinweis: Wenn Sie eine E-Mail- oder Desktop-Benachrichtigung erhalten möchten, sobald neue Aktualisierungen verfügbar sind, klicken Sie auf Einstellungen > Allgemein > Benachrichtigung, markieren Sie die Kontrollkästchen und klicken Sie anschließend auf OK.

Gehen Sie auf eine der folgenden Weisen vor, um Pakete zu reparieren:

Wenn Fehler in einem Paket auf DiskStation aufgetreten sind, müssen Sie sie reparieren, um es wieder nutzen zu können. Manchmal treten diese auf, weil das Paket zu alt ist und nicht mehr mit der aktuellen Version von DSM kompatibel ist. Andere Ursachen könnten sein, dass die Programmdateien des Paketes fehlen oder beschädigt sind.  Pakete, die repariert werden müssen, werden unter der Kategorie Installiert angezeigt.

  • Klicken Sie auf die Schaltflächen Reparieren, um Pakete zu reparieren.
  • Klicken Sie auf Manuelle Installation, um ein Paket durch Hochladen einer .spk-Datei (im Synology Download-Zentrum oder auf Websites von Drittanbietern verfügbar) erneut zu installieren und zu reparieren.

Hinweis: Durch Anklicken von Reparieren wird die aktuelle Version dieses Paketes installiert, anstatt die Alte zu reparieren. Bitte kontaktieren Sie den Herausgeber oder Entwickler dieses Paketes, wenn Sie Probleme bei der Installation des Paketes haben.

So deinstallieren Sie vorhandene Pakete:

Klicken Sie auf ein installiertes Paket und wählen Sie Deinstallieren aus dem Drop-down-Menü Aktion, um das Paket zu deinstallieren.

So führen Sie Paketdienste aus oder halten sie an:

Klicken Sie auf ein installiertes Paket und wählen Sie Ausführen oder Anhalten aus dem Drop-down-Menü Aktion, um das Paket auszuführen oder anzuhalten.

Paketprobleme beheben:

Wenn Sie bei einem Paket auf ein Problem stoßen, versuchen Sie bitte Folgendes:

  • Aktualisieren Sie das Paket auf die neueste Version.
  • Entfernen Sie sowohl das Paket als auch seine Datenbank (falls vorhanden), und installieren Sie es neu.
  • Kontaktieren Sie den Herausgeber oder Entwickler, um Hilfe und Support zu erhalten.

So finden Sie die Kontaktinformationen eines Paketherausgebers:

Einige Pakete im Synology Paketzentrum werden von Drittentwicklern zusammengestellt und herausgegeben. Synology bietet daher keinen technischen Support für solche Anwendungen an. Bitte kontaktieren Sie die Herausgeber oder Entwickler der jeweiligen Anwendung, um neue Funktionen vorzuschlagen, wenn Sie Fragen haben oder ein Feedback geben möchten. Einige Pakete haben neben einem Entwickler noch einen Herausgeber. Der Herausgeber stellt das Paket zusammen und macht es für Synology-Produkte verfügbar, während die Grundfunktionen des Paketes mit Genehmigung des Entwicklers angeboten werden.  Wenn ein Herausgeber angegeben ist, wenden Sie sich für Support bitte an den Herausgeber.

  1. Gehen Sie zur Detailseite eines installierten Paketes.
  2. Links auf der Detailseite finden Sie die Kontaktinformationen des Herausgebers oder Entwicklers.
  3.  Wenn Sie für von Synology entwickelte Pakete Vorschläge zu weiteren Funktionen oder Fragen haben sowie Feedback geben möchten, besuchen Sie unsere Synology Website und kontaktieren Sie uns direkt.

Anwendungen ausführen

Nachdem Sie Pakete installiert und ausgeführt haben, können Sie die aktivierten Anwendungen auf unterschiedliche Weisen starten.

Gehen Sie folgendermaßen vor, um eine installierte und aktivierte Anwendung zu starten:

  • Gehen Sie zu Hauptmenü > [Anwendungsname].
  • Öffnen Sie das Hauptmenü und ziehen Sie die Miniaturansicht der Anwendung auf den DSM-Desktop. Klicken Sie anschließend auf [Anwendungsname] auf dem Desktop.
  • Gehen Sie zu Hauptmenü > [Anwendungsname]. Rechtsklicken Sie auf das Anwendungssymbol in der Taskleiste und wählen Zur Taskleiste legen. Wird die Anwendung geschlossen, verbleibt das Symbol in der Taskleiste und Sie können es anklicken, um die Anwendung wieder zu starten.
