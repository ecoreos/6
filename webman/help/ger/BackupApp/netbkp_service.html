Sicherungsdienste

Durch die Optionen unter Datensicherung & Replikation > Sicherungsdienste können andere Benutzer Daten ihrer DiskStation-Geräte oder rsync-kompatiblen Clients auf Ihrem DiskStation sichern.

Netzwerksicherungsdienst aktivieren

Wenn Sie Daten von einem DiskStation oder einem rsync-kompatiblen Server auf einen anderen DiskStation sichern möchten oder es einem anderen DiskStation-Gerät erlauben, Remote-Datensicherungsaufgaben durchzuführen und gesicherte Daten auf einem Volume Ihres DiskStation zu speichern, müssen Sie zunächst den Netzwerksicherungsdienst auf dem Zielserver aktivieren, bevor Sie eine Netzwerk-Datensicherungsaufgabe erstellen. Gehen Sie dabei folgendermaßen vor: Gehen Sie zu Systemsteuerung > Berechtigungen und gewähren Sie einem Benutzer die Netzwerksicherungsziel-Berechtigung. Sie können danach auf Ihrem DiskStation einen DSM-Benutzer mit geeigneten Berechtigungen für Anwendungen und freigegebene Ordner erstellen. Nach der Einrichtung können andere Personen unter Verwendung der Berechtigungen des Benutzers eine Netzwerksicherung von ihrem DiskStation zum freigegenenen Zielordner auf Ihrem DiskStation durchführen.

Sicherungsdienst für einzelne Versionen aktivieren:

  1. Gehen Sie zu Sicherungsdienste > Netzwerksicherungsziel und aktivieren Sie Netzwerksicherungsdienst aktivieren.
  2. Klicken Sie auf Übernehmen.
  3. Gehen Sie zu Systemsteuerung > Freigegebener Ordner, wählen Sie den gewünschten Ordner und klicken Sie auf Bearbeiten > Berechtigungen, und geben Sie einem Benutzer dann die Berechtigung für Lesen/Schreiben. Wiederholen Sie diesen Schritt, wenn Sie weitere freigegebene Ordner, die als Datensicherungsziele verwendet werden könnten, einschließen möchten.

Sicherungsdienst für mehrere Versionen aktivieren:

  1. Gehen Sie zu Sicherungsdienste > Netzwerkvolumen-Datensicherungsziel und aktivieren Sie das Kontrollkästchen neben Netzwerkvolumen-Datensicherungsziel aktivieren.
  2. Gehen Sie zu Datensicherungsziel und wählen Sie ein Datensicherungsziel aus, das mit einem lokalen Volume verbunden ist.
  3. Klicken Sie auf Bearbeiten > Freigabeliste > Hinzufügen, um den Benutzer oder die Gruppe zur Liste hinzuzufügen.

Hinweis:

  • Wenn andere Personen über einen anderen Client als den DiskStation eine Netzwerksicherung durchführen möchten, benötigen Sie dazu die Berechtigungen von admin oder rsync auf dem DiskStation. Wenn Sie Ihre admin-Anmeldedaten nicht an andere Personen weitergeben möchten, können Sie über Systemsteuerung > Benutzer einen Benutzer namens rsync erstellen, dessen Benutzerberechtigungen entsprechend anpassen und diese Anmeldedaten dann an andere Personen weitergeben.
  • Wenn Sie die Netzwerksicherung von einem DiskStation mit einer DSM-Version vor 3.0 aus durchführen möchten oder von einem Client aus, der nicht der DiskStation ist, und die Informationen des Besitzers und der Gruppe als Quelldaten beibehalten werden sollen, müssen Sie das Sicherungskonto zur Gruppe administrators hinzufügen und die Daten im freigegebenen Ordner NetBackup sichern.

Netzwerksicherungsdienst über benutzerdefinierte rsync-Konfiguration aktivieren

Sie können den Netzwerksicherungsdienst mit Hilfe der benutzerdefinierten rsync-Konfiguration aktivieren und Benutzerberechtigungen sowie das Datensicherungsziel auf Ihrem DiskStation festlegen. Nachdem die Einrichtung durchgeführt wurde, können andere Personen eine rsync-Sicherung von einem DiskStation-Gerät oder einem rsync-kompatiblen Client zu freigegebenen Zielordnern (oder „Datensicherungsmodulen“) auf dem DiskStation-Gerät durchführen.

Netzwerksicherungsdienst mit der benutzerdefinierten rsync-Konfiguration aktivieren:

  1. Gehen Sie zu Sicherungsdienste > Netzwerksicherungsziel.
  2. Aktivieren Sie Netzwerksicherungsdienst aktivieren.
  3. Aktivieren Sie Benutzerdefinierte rsync-Konfiguration verwenden.
  4. Klicken Sie auf Übernehmen.

Benutzerberechtigungen und Datensicherungsmodule auf dem DiskStation konfigurieren:

  1. Melden Sie sich beim DiskStation über Telnet oder SSH an.
  2. Geben Sie das Datensicherungsmodul und die autorisierten Benutzer in der rsync-Konfigurationsdatei "/etc/rsyncd.conf" an. Detaillierte Informationen finden Sie in der Anleitung von rsyncd.conf auf Ihrem rsync-Client oder hier.

Hinweis:

  • Wenn eine benutzerdefinierte rsync-Konfiguration verwendet wird, kann der DiskStation nicht mehr als Zielserver für Synchronisierung von freigegebenen Ordnern, Time Backup und LUN-Datensicherung fungieren.

Aktivieren einer Geschwindigkeitseinschränkung

Sie können eine Geschwindigkeitseinschränkung aktivieren, um die maximal nutzbare Bandbreite für Benutzer und Gruppen zu regeln, die Daten per Netzwerksicherung übertragen.

Geschwindigkeitseinschränkung aktivieren:

  1. Gehen Sie zu Sicherungsdienste > Netzwerksicherungsziel. Aktivieren Sie das Kontrollkästchen Geschwindigkeitseinschränkung aktivieren oder Geschwindigkeitseinschränkung nach einem Zeitplan aktivieren, wenn Sie nur zu bestimmten Zeiten eine Geschwindigkeitseinschränkung festlegen möchten. Dies ist eine serverweite Einstellung, die die Einstellungen der einzelnen Benutzer und Gruppen überschreibt. Die Auswahl von Keine Geschwindigkeitseinschränkung deaktiviert die Einstellungen für die Geschwindigkeitseinschränkung aller Benutzer und Gruppen.
  2. Klicken Sie auf Einstellungen für Geschwindigkeitseinschränkung, um die Einstellungen für einzelne Benutzer oder Gruppen zu ändern. Für jeden Benutzer und jede Gruppe steht Folgendes zur Auswahl:
    • Gruppeneinstellungen anwenden: Die Einstellungen für Geschwindigkeitseinschränkung sind identisch mit der Gruppe, zu der der Benutzer gehört.
    • Ohne: Es wird keine Geschwindigkeitseinschränkung angewendet, wenn Gruppenmitglieder Dateien mit dem Dienst übertragen.
    • Konstant: Für alle Benutzer und Gruppenmitglieder gilt eine feste, konstante Geschwindigkeitseinschränkung. Upload- und Download-Geschwindigkeitseinschränkungen können in den Feldern links angegeben werden.
    • Variabel: Für Benutzer oder Gruppenmitglieder können zwei unterschiedliche Geschwindigkeitseinschränkungen gemäß einem Zeitplan festgelegt werden. Klicken Sie auf die Schaltfläche Anpassen, um die Einstellungen für die Geschwindigkeitseinschränkung und einen Zeitplan festzulegen.
  3. Klicken Sie auf Übernehmen.

Hinweis:

Beim Festlegen einer Geschwindigkeitseinschränkung für einen Benutzer zeigt der Bereich Ergebnis die tatsächliche Geschwindigkeitseinschränkung des Benutzers gemäß seiner individuellen und der Gruppeneinstellungen an. Wenn etwa der Benutzer „Achilles“ eine Download-Einschränkung von 100 KB/s hat, aber zu einer Benutzergruppe mit einer Download-Einschränkung von 500 KB/s gehört, wird im Bereich Ergebnisse „0/500“ angezeigt.

Aktivieren des Time Backup-Dienstes

Mithilfe des Time Backup-Dienstes können mehrere Versionen von Ordnern per Fernzugriff von einem anderen DiskStation auf diesem DiskStation gesichert werden. Dazu muss sowohl der Netzwerksicherungs- als auch der Time Backup-Dienst aktiviert sein.

Netzwerksicherungs- und Time Backup-Dienst aktivieren:

Gehen Sie zu Sicherungsdienste > Time Backup. Aktivieren Sie das entsprechende Kontrollkästchen.

Hinweis:

Die Sicherungsverbindung für das Time Backup per Fernzugriff kann nur von einem Admin-Konto aus hergestellt werden.
