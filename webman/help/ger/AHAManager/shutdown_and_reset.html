Server herunterfahren und austauschen

Die nachstehende Anleitung beschreibt das Herunterfahren oder Austauschen eines Servers in einem High Availability-Cluster.

High Availability-Cluster mit High Availability Manager herunterfahren:

Führen Sie die nachstehenden Schritte durch, wenn Sie einen High Availability-Cluster herunterfahren möchten, um die Server an einen anderen Ort zu bringen oder um Wartungsarbeiten durchzuführen.

  1. Melden Sie sich bei DSM an.
  2. Gehen Sie zu High Availability Manager.
  3. Klicken Sie auf die Schaltfläche Verwalten.
  4. Wählen Sie High Availability-Cluster herunterfahren aus.
  5. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie zur Bestätigung auf OK.

High Availability-Cluster manuell herunterfahren:

Der High Availability-Cluster kann auch über die Ein/Aus-Taste der Server manuell heruntergefahren werden.

  1. Drücken Sie die Ein/Aus-Taste am passiven Server, bis er einen Signalton ausgibt, und warten Sie, bis der Server vollständig heruntergefahren ist.
  2. Drücken Sie die Ein/Aus-Taste am aktiven Server, bis er einen Signalton ausgibt, und warten Sie, bis der Server vollständig heruntergefahren ist.

Hinweis:

  • Server einschalten:
    Wenn Sie die Server wieder einschalten möchten, müssen der aktive und passive Server zur selben Zeit eingeschaltet werden (oder innerhalb von 5 Minuten). Wenn Sie nur einen der Server einschalten, versucht er, den anderen Server durch Senden der Meldung "Wake on LAN-Verbindung" einzuschalten.
  • Anormales Herunterfahren:
    Beim High Availability-Cluster ist die Option für den Neustart nach einem Stromausfall aktiviert. Wenn die Server aufgrund eines Stromausfalls nicht normal herunterfahren, werden sie automatisch neu gestartet, wenn die Stromversorgung wiederhergestellt ist.

Server in einem High Availability-Cluster austauschen:

Führen Sie die nachstehenden Schritte durch, um einen Server auszutauschen und einen anderen einzurichten.

  1. Gehen Sie zur Seite Überblick.
  2. Klicken Sie auf die Schaltfläche Verwalten.
  3. Wählen Sie Aktiven Server herunterfahren oder Passiven Server herunterfahren aus, um den Server herunterzufahren, der ausgetauscht werden soll.
  4. Überprüfen Sie die Verbindungen des neuen Servers. Nachdem der neue Server eingeschaltet wurde, wird er in High Availaibility Manager als "unbekannt" markiert .
  5. Gehen Sie erneut zur Seite Überblick.
  6. Klicken Sie auf die Schaltfläche Verwalten.
  7. Wählen Sie Unbekannten Server wiederherstellen aus, sodass der „unbekannte Server“ zum High Availability-Cluster hinzugefügt und als der passive Server erkannt wird.
