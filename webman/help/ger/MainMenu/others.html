Sonstiges

Sie können andere persönliche Konto-Optionen unter Persönlich > Sonstige anpassen.

So passen Sie sonstige persönliche Einstellungen benutzerspezifisch an:

  1. Wählen Sie eine der folgenden Optionen aus:
    • Bestätigungsdialog beim Verlassen von DSM nicht anzeigen: Überspringt den Bestätigungsdialog beim Schließen des Browserfensters.
    • DSM beim Anmelden mit den Fenstern laden, welche beim letzten Abmelden aktiv waren: Speichert offene Anwendungen oder Pakete und stellt sie bei der nächsten Anmeldung oder Aktualisierung des Browserfensters wieder her.
    • Sofortvorschau auf Taskleiste aktivieren: Zeigt eine schnelle Vorschau einer geöffneten Anwendung, wenn Sie den Mauszeiger über das Anwendungssymbol in der Taskleiste bewegen. Diese Funktion wird von früheren Internet-Explorer-Versionen unter Umständen nicht voll unterstützt.
    • Hilfe zu DSM nicht bei jeder Anmeldung automatisch starten: Wenn Sie sich bei DSM anmelden, wird nicht automatisch die DSM-Hilfeseite geöffnet.
    • Desktop-Tastaturkürzel aktivieren: Sie können Tastaturkürzel verwenden, um verschiedene Arten von Desktop- und DSM-Funktionen auszuführen (z. B. zwischen aktiven Fenstern wechseln, Optionen auswählen/Auswahl aufheben usw.). Drücken Sie Umsch + ?, um alle Tastaturkürzel anzuzeigen.
    • Erreichbarkeit deaktivieren (keine hörbaren Hinweisreize verfügbar): Deaktiviert die Barrierefreiheit-Funktion von DSM, sodass Sie keine akustischen Signale hören, wenn der Screenreader eingeschaltet ist.
    • Browser Desktop-Benachrichtigungen aktivieren: Zeigt Meldungen von Ihrem Webbrowser auf dem DSM-Desktop an. Browser Desktop-Benachrichtigungen werden nur von Google Chrome und Safari unterstützt. Je nach Version Ihres Webbrowsers kann die Einstellungsseite für Benachrichtigungen anders aussehen. Bitte rufen Sie die Einstellungsseite Ihres Browsers auf, um die Benachrichtigungseinstellungen zu verwalten.
    • Schaltfläche Pilot View in der Taskleiste aktivieren: Zeigt die Schaltfläche Pilot View an, mit der Sie alle offenen Anwendungen im Vorschaumodus anzeigen lassen können.
  2. Klicken Sie auf OK.
