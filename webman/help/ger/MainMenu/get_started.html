Desktop navigieren

DiskStation Manager (DSM) ist ein webbasiertes Betriebssystem, das speziell für Synology-Produkte konzipiert ist. Mit DSM können Sie schnell und einfach Folgendes auf dem DiskStation ausführen:

  • Systemeinstellungen ändern und personalisieren
  • Anwendungen und Pakete installieren und öffnen
  • RAID-Volumes erstellen und andere Speichertypen verwalten
  • ...und vieles mehr!

Erfahren Sie mehr über DSM

Weitere Informationen über alle verfügbaren Funktionen von DSM finden Sie unter www.synology.com.

Anmelden bei DSM

Nachdem Sie DSM auf Ihrem DiskStation installiert und eingerichtet haben, können Sie sich über einen Browser bei Ihrem Computer anmelden.

Installation von DSM

Die schrittweise Anleitung zum Installieren von DSM auf Ihrem DiskStation finden Sie in der Schnellinstallationsanleitung für Ihr Synology-Produkt unter www.synology.com.

Bei DSM anmelden:

  1. Überprüfen Sie, ob Ihr Computer und der DiskStation mit demselben lokalen Netzwerk verbunden sind.
  2. Öffnen Sie einen Browser auf Ihrem Computer und geben Sie eine der folgenden Adressen in das Adressfeld ein:
    • find.synology.com (nur wenn Ihr Computer und der DiskStation mit dem Internet verbunden sind)
    • Der Servername Ihres DiskStation, gefolgt von :5000, z. B. diskstation:5000 oder rackstation:5000 (Der genaue Name ist von den während der Ersteinrichtung vorgenommenen Einstellungen abhängig.)
    • Die IP-Adresse Ihres DiskStation, gefolgt von :5000, z. B. 192.168.48.14:5000 (Die IP-Adresse ist von den während der Ersteinrichtung vorgenommenen Einstellungen abhängig.)
  3. Geben Sie Ihren Benutzernamen und Ihr Kennwort ein und klicken Sie auf Anmelden.

Hinweis:

Der Name, die IP-Adresse und das admin-Kennwort für Ihren DiskStation müssen während der Erstinstallation festgelegt werden. Installieren Sie Synology Assistant oder gehen Sie zu find.synology.com, wenn Sie diese Informationen nicht finden können.

DSM-Desktop

Nachdem Sie sich angemeldet haben, sehen Sie den DSM-Desktop. Hier können Sie jetzt Einstellungen vornehmen, Pakete verwenden oder Benachrichtigungen anzeigen.

Taskleiste

Die Taskleiste befindet sich oben auf dem Bildschirm und enthält die folgenden Elemente:

  1. Desktop anzeigen: Minimiert alle offenen Anwendungs- und Paketfenster.
  2. Hauptmenü: Zeigt die auf dem DiskStation installierten Anwendungen und Pakete an und öffnet sie. Sie können auch klicken und ziehen, um Desktop-Verknüpfungen zu erstellen.
  3. Anwendungen öffnen: Zeigt die momentan geöffneten Anwendungen und Pakete an. Sie können mit einem Rechtsklick Pakete an die Taskleiste anheften, um zukünftig schneller darauf zugreifen zu können.
  4. Upload-Warteschleife: Wird angezeigt, wenn Dateien zum DiskStation hochgeladen werden. Klicken Sie hier, um mehr Details anzuzeigen, wie Fortschritt und Upload-Geschwindigkeit.
  5. Externe Geräte: Wird angezeigt, wenn ein externes Gerät (z. B. USB-Laufwerk) an Ihren DiskStation angeschlossen wird.
  6. Benachrichtigungen: Zeigt Benachrichtigungen wie Fehler, Statusaktualisierungen und Installationsbenachrichtigungen von Paketen an.
  7. Optionen:Optionen zum Herunterfahren, Neustarten, Abmelden oder Ändern des persönlichen Kontos.
  8. Optionen: Optionen für Anhalten, Neustart, Abmelden oder Ändern des persönlichen Kontos.
  9. Suchen: Über diese Option finden Sie schnell Anwendungen, Pakete oder Artikel der DSM-Hilfe.
  10. Widgets: Zeigt Widgets an oder blendet sie aus.
  11. Pilot View: Zeigt eine Vorschau aller offenen Anwendungs- und Paketfenster an.

Hauptmenü

Hier finden Sie eine Liste der auf der DiskStation installierten Anwendungen und Pakete.

Desktop-Verknüpfungen erstellen:

  1. Öffnen Sie das Hauptmenü.
  2. Klicken Sie auf eine Anwendung oder ein Paket und ziehen Sie sie bzw. es auf die Seite.
