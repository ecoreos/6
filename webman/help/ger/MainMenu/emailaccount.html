E-Mail-Konto

Sie können Ihre E-Mail-Konten unter PersÃ¶nlich > E-Mail-Konto hinzufügen, und Sie können Dateien, die in File Station gespeichert sind, als Anhänge von E-Mail-Adressen, die hier gespeichert wurden, hinzufügen.

Zu den unterstützten E-Mail-Serviceanbietern gehören Gmail, Outlook, Yahoo!, AOL und benutzerdefinierte SMTP-Server.

E-Mail-Konto hinzufügen:

  1. Klicken Sie auf Hinzufügen, und wählen Sie den Serviceanbieter für das E-Mail-Konto, das Sie einrichten möchten.
  2. Geben Sie Ihre E-Mail-Adresse, Passwort und den Absendername ein.
  3. Markieren Sie Dieses E-Mail-Konto als Standard verwenden, wenn Sie möchten, dass File Station dieses E-Mail-Konto standardmäßig nutzt, wenn Sie mehrere Konten eingerichtet haben.
  4. Klicken Sie auf Kontakte synchronisieren, um Ihre E-Mail-Kontakte mit DiskStation zu synchronisieren, wenn Ihr E-Mail-Serviceanbieter (Gmail, Outlook und Yahoo!) diese Funktion unterstützt. Beachten Sie, dass die Kontakte, die synchronisiert werden, eher aus dem E-Mail-Konto stammen, das Sie im Authentifizierungsfenster festgelegt haben, als aus demjenigen, dass Sie im vorherigen Schritt eingegeben haben. Wenn daher ein anderes oder unerwünschtes Konto im Authentifizierungsfenster angezeigt wird, wechseln Sie bitte manuell auf das gewünschte E-Mail-Konto oder auf das, das Sie zuvor eingegeben haben.
  5. Klicken Sie auf Anwenden, um die Einstellungen zu speichern.

E-Mail-Konto bearbeiten:

Wählen Sie das gewünschte E-Mail-Konto aus der Liste aus, und klicken Sie dann auf Bearbeiten.

E-Mail-Konto löschen:

Wählen Sie das gewünschte E-Mail-Konto aus der Liste aus, und klicken Sie dann auf Löschen.

Hinweis:

  • Das Standard-Konto kann nur gelöscht werden, wenn keine anderen Konten vorhanden sind. Möchten Sie ein E-Mail-Konto löschen, das derzeit als Standard festgelegt ist, dann legen Sie bitte ein anderes Konto als Ihr Standardkonto fest, bevor Sie fortfahren.

Das Standardkonto ändern:

Wählen Sie aus der Liste das E-Mail-Konto aus, dass Sie als Standard festlegen möchten, und klicken Sie dann auf Als Standard festlegen.

Dateien als Anhänge senden:

  1. Rechtsklicken Sie in File Station auf eine Dateien, und wählen Sie Als E-Mail-Anhänge senden.
  2. Wenn Sie mehr als ein E-Mail-Konto hinzugefügt haben, können Sie Ihre gewünschte Absender-E-Mail-Adresse in der Drop-down-Liste auswählen, um die Dateien an einen angegebenen Empfänger zu senden.
