Dateien von einem Windows PC innerhalb des lokalen Netzwerks auf dem Synology NAS speichern

Übersicht

Mit dem Synology NAS können Sie Dateien in Ihrem lokalen Netzwerk schnell und einfach speichern und teilen, um direkt auf gemeinsame Ordner und Dateien auf dem Synology NAS zugreifen zu können, ohne sich jedes Mal bei DSM anmelden zu müssen. Sie können zum Beispiel Dateien mit Windows Explorer wie auf jedem anderen Netzwerkgerät auf dem Synology NAS speichern.

Einbinden mit Windows Explorer

  1. Öffnen Sie ein Windows Explorer Fenster und gehen Sie zu Computer.
  2. Klicken Sie auf Netzlaufwerk einbinden. Durch diese Aktion wird das Netzlaufwerk verbinden Fenster angezeigt.
  3. Wählen Sie im Netzlaufwerk verbinden Fenster einen Laufwerksbuchstaben aus dem Dropdown-Menü Laufwerk aus.
  4. Geben Sie den Servernamen Ihres Synology NAS und den Namen des gemeinsamen Ordners getrennt durch einen Backslash in das Feld Ordner ein. Wenn der Servername Ihres Synology NAS z.B. "DiskStation" und der Name des gemeinsamen Ordners "Share1" ist, dann müssen Sie "\\DiskStation\Share1" eingeben. Klicken Sie anschließend auf Fertig stellen.
  5. Geben Sie Ihren DSM Benutzernamen und Ihr Kennwort ein.
    6. Kein Zugriff auf den gemeinsamen Ordner? Das hier eingegebene Benutzerkonto muss Zugriffberechtigungen für den gemeinsamen Ordner besitzen, auf den Sie zugreifen möchten.
  6. Abschließend wird ein Windows Explorer Fenster angezeigt. Der gemeinsame Ordner wird jetzt zugeordnet und ist unter Computer verfügbar.
