Überblick

Auf dieser Seite wird der Gesamtstatus Ihrer Schnappschussziele und von deren Schnappschüssen angezeigt.

Zusammenfassung

In diesem Abschnitt werden unter den Schnappschusszielen (einschließlich freigegebener Ordner und iSCSI LUNs) drei Status angezeigt. Außerdem finden Sie in diesem Abschnitt die Gesamtgröße der Schnappschussziele im jeweiligen Status.

  • Geplanter Schnappschuss: Die Erstellung von Schnappschüssen für die Schnappschussziele ist geplant.
  • Nicht geplant: Die Schnappschüsse für die Schnappschussziele wurden erstellt, jedoch nicht gemäß einem Plan.
  • Nicht geschützt: Für die Schnappschussziele wurde kein Schnappschuss erstellt.

Letzte Schnappschüsse

Dieser Abschnitt enthält eine Zeitachse kategorisierter Schnappschüsse, die die Wiederherstellung Ihrer Daten erleichtert.

Freigegebene Ordner oder iSCSI LUNs aus den zuletzt aufgenommenen Schnappschüssen wiederherstellen/klonen:

  1. Klicken Sie auf das Kalendersymbol und wählen Sie das Datum aus, für das die Daten wiederhergestellt werden sollen.
  2. Ziehen Sie die Zeitachse vor oder zurück und wählen Sie einen bestimmten Zeitpunkt aus, zu dem Schnappschüsse aufgenommen wurden.
  3. Wählen Sie einen freigegebenen Ordner oder einen iSCSI LUN aus, den Sie wiederherstellen/klonen möchten.
  4. Sie werden zur Seite Wiederherstellung weitergeleitet. Weitere Informationen zu den folgenden Schritten finden Sie unter Wiederherstellung.
