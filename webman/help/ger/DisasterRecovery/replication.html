Replikation

Auf dieser Seite können Systemadministratoren Replikationen verwalten. Sie können Replikationsaufgaben erstellen, bearbeiten und löschen. Außerdem können Sie die Synchronisierung starten und stoppen und Informationen zu Ihren Replikationsaufgaben anzeigen.

Freigegebener Ordner

Replikationsaufgabe für einen freigegebenen Ordner erstellen:

  1. Gehen Sie zur Registerkarte Freigegebener Ordner.
  2. Klicken Sie auf Erstellen.
  3. Der Assistent Replikationsaufgabe erstellen wird geöffnet. Befolgen Sie die Schritte des Assistenten, um eine Replikation eines freigegebenen Ordners zu erstellen.

Hinweis:

  • Die freigegebenen „homes“-Ordner können nicht repliziert werden.
  • Verschlüsselte freigegebene Ordner können nicht repliziert werden.
  • Der freigegebene Ordner muss sich auf einem Btrfs-Volume befinden.
  • Sie können Eine-bis-viele/erweiterte Replikationen für einen freigegebenen Ordner für verschiedene Zielserver erstellen. Jedoch können Sie für jeden freigegebenen Ordner nur eine Replikation für einen Zielserver erstellen.

Replikation eines freigegebenen Ordners löschen:

  1. Wählen Sie die zu löschende Replikationsaufgabe für einen freigegebenen Ordner aus.
  2. Klicken Sie auf Löschen.
  3. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Löschen, um das Löschen zu bestätigen.

Replikation eines freigegebenen Ordners bearbeiten:

  1. Wählen Sie die zu berarbeitende Replikationsaufgabe für einen freigegebenen Ordner aus.
  2. Klicken Sie auf Bearbeiten.
  3. Ein Fenster mit den folgenden Registerkarten wird geöffnet:
    • Partner-Server
    • Zeitplan
    • Aufbewahrung
  4. Auf der Registerkarte Partnerserver können Sie die Informationen zum Partnerserver, wie Hostname, Port, Benutzername und Kennwort, bearbeiten.
  5. Aktivieren Sie auf der Registerkarte Plan das Kontrollkästchen Schnappschuss-Zeitplan aktivieren und legen Sie die Zeit und die Häufigkeit für die Aufnahme von Schnappschüssen fest. Sie können das Kontrollkästchen deaktivieren, um die geplanten Schnappschüsse für den freigegebenen Ordner abzubrechen.
  6. Auf der Registerkarte Aufbewahrung können Sie folgende Optionen auswählen:
    • 256 letzten Schnappschüsse aufbewahren: Wenn die maximale Anzahl (256) von Schnappschüssen erreicht ist, werden die ältesten Schnappschüsse von neuen überschrieben.
    • Schnappschüsse immer aufbewahren: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, nimmt Die Funktion „Schnappschuss & Replikation“ keine neuen Schnappschüsse mehr auf, ohne die bereits aufgenommenen zu entfernen.
    • Schnappschüsse gemäß der folgenden Richtlinie aufbewahren: Sie können Aufbewahrungsregeln für bestimmte Zeitbereiche konfigurieren.
  7. Klicken Sie auf OK, um die neuen Einstellungen anzuwenden.

Hinweis:

  • Sie können den Zeitplan der Replikationsaufgabe auf dem Quellserver nur bearbeiten.
  • Die Änderung der Aufbewahrungseinstellungen beeinträchtigt die entsprechenden Einstellungen auf der Seite Schnappschuss.

Informationen zur Replikation eines freigegebenen Ordners anzeigen:

  1. Wählen Sie die anzuzeigende Replikationsaufgabe für einen freigegebenen Ordner aus.
  2. Klicken Sie auf Info.
  3. Ein Fenster mit den folgenden Informationen wird geöffnet:
    • Status dieser Replikation.
    • Statistik dieser Replikation.

Synchronisierung der Replikation eines freigegebenen Ordners durchführen:

  1. Wählen Sie die zu synchronisierende Replikationsaufgabe für einen freigegebenen Ordner aus.
  2. Klicken Sie auf Ausführen.
  3. Geben Sie die Beschreibung des aufzunehmenden Schnappschusses ein.
  4. Klicken Sie auf OK, um die Aktion zu bestätigen.

Synchronisierung der Replikation eines freigegebenen Ordners stoppen:

  1. Wählen Sie die zu stoppende Synchronisierung einer Replikationsaufgabe für einen freigegebenen Ordner aus.
  2. Klicken Sie auf Stopp.
  3. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Stopp, um die Aktion zu bestätigen.

iSCSI LUN

Replikationsaufgabe für eine iSCSI LUN erstellen:

  1. Gehen Sie zur Registerkarte iSCSI LUN.
  2. Klicken Sie auf Erstellen.
  3. Der Assistent Replikationsaufgabe erstellen wird geöffnet. Befolgen Sie die Schritte des Assistenten, um eine Replikation einer iSCSI LUN zu erstellen.

Hinweis:

  • Die iSCSI LUN muss als Erweiterte LUN erstellt worden sein.
  • Sie können Eine-bis-viele/erweiterte Replikationen für eine iSCSI LUN für verschiedene Zielserver erstellen. Jedoch können Sie für jede iSCSI LUN nur eine Replikation für einen Zielserver erstellen.

Replikation einer iSCSI LUN löschen:

  2. Klicken Sie auf Löschen.

    • Aufbewahrung
  4. Auf der Registerkarte Partnerserver können Sie die Informationen zum Partnerserver, wie Hostname, Port, Benutzername und Kennwort, bearbeiten.
  5. Aktivieren Sie auf der Registerkarte Plan das Kontrollkästchen Schnappschuss-Zeitplan aktivieren und legen Sie die Zeit und die Häufigkeit für die Aufnahme von Schnappschüssen fest. Sie können das Kontrollkästchen deaktivieren, um die geplanten Schnappschüsse für die iSCSI LUN abzubrechen.
  6. Auf der Registerkarte Aufbewahrung können Sie folgende Optionen auswählen:
    • 256 letzten Schnappschüsse aufbewahren: Wenn die maximale Anzahl (256) von Schnappschüssen erreicht ist, werden die ältesten Schnappschüsse von neuen überschrieben.
    • Schnappschüsse immer aufbewahren: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, nimmt Die Funktion „Schnappschuss & Replikation“ keine neuen Schnappschüsse mehr auf, ohne die bereits aufgenommenen zu entfernen.
    • Schnappschüsse gemäß der folgenden Richtlinie aufbewahren: Sie können Aufbewahrungsregeln für bestimmte Zeitbereiche konfigurieren.
  7. Aktivieren Sie auf der Registerkarte Anwendung die Option Anwendungskonsistente Schnappschüsse aktivieren, um anwendungskonsistente Schnappschüsse zu aktivieren. Deaktivieren Sie das Kontrollkästchen, um anwendungskonsistente Schnappschüsse abzubrechen.
  8. Klicken Sie auf OK, um die neuen Einstellungen anzuwenden.

Hinweis:

  • Sie können den Zeitplan bei der Replikationsquelle nur bearbeiten.
  • Die Änderung der Aufbewahrungs-Anwendungseinstellungen beeinträchtigt die entsprechenden Einstellungen auf der Seite Schnappschuss.

Eine-bis-viele/Erweiterte Replikation

Unter Verwendung des Schemas Eine-bis-viele können Sie mehrere Replikationen für einen freigegebenen Ordner bzw. für eine iSCSI LUN für mehrere Zielserver erstellen. Mithilfe des Schemas Erweitert können Sie außerdem für einen replizierten freigegebenen Ordner bzw. eine replizierte iSCSI LUN eine erweiterte Replikation erstellen.

Hinweis:

  • Unabhängig davon, ob das Schema „Eine-bis-viele“ oder „Erweitert“ verwendet wird, kann eine Replikationsaufgabe nur maximal drei Zielserver haben.

Aufbewahrungsrichtlinie

Diese Option bietet eine ideale Lösung zum Festlegen der maximalen Anzahl von Schnappschussversionen, um Volume-Speicherplatz zu sparen. Möglicherweise müssen Sie Ihre Schnappschüsse aber für längere Zeit aufbewahren.

Synology nutzt die GFS-Aufbewahrungsrichtlinie („Grandfather-Father-Son“). Sie können die maximale Anzahl von aufzubewahrenden Schnappschussversionen für die folgenden Zeiträume konfigurieren: stündlich, täglich, wöchentlich, monatlich und jährlich.

Der Datenschutz-Manager bewahrt für jeden Zeitraum die konfigurierte maximale Anzahl von Schnappschüssen auf. Existiert in einem Zeitraum mehr als eine Schnappschussversion, wird nur die früheste aufbewahrt. Legen Sie beispielsweise als Richtlinie 10 wöchentliche Schnappschüsse fest, bewahrt der Datenschutz-Manager für jede der letzten 10 Wochen den frühesten Schnappschuss auf (wenn pro Woche mehr als ein Schnappschuss aufgenommen wird).

Damit Sie die letzten Schnappschussversionen schnell finden und aus ihnen wiederherstellen können, werden alle Schnappschussversionen nach Aufnahme des neuesten Schnappschusses standardmäßig eine Stunde lang aufbewahrt. Um zu verhindern, dass ein Schnappschuss aufgrund der Aufbewahrungsregeln automatisch entfernt wird, können Sie ihn sperren.
