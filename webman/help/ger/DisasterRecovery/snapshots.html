Schnappschüsse

Auf dieser Seite können Sie Schnappschüsse aufnehmen, entfernen und bearbeiten sowie Einstellungen für das Planen und Aufbewahren von Schnappschüssen konfigurieren.

Für jeden gemeinsamen Ordner und jede iSCSI LUN können Sie bis zu 256 Schnappschüsse gleichzeitig kopieren. Da einzelne Blöcke anstatt ganze Dateien gespeichert werden, belegen Schnappschüsse nur einen geringen Speicherplatz. Zusätzlicher Speicherplatz kann belegt werden, wenn die Dateien in gemeinsamen Ordnern oder Blöcke in iSCSI LUNs aktualisiert oder entfernt werden.

Freigegebener Ordner

Schnappschuss für einen gemeinsamen Ordner aufnehmen:

  1. Wählen Sie den gemeinsamen Ordner aus, von dem Sie einen Schnappschuss aufnehmen möchten.
  2. Klicken Sie auf Einen Schnappschuss machen.
  3. Geben Sie in Beschreibung eine Beschreibung für den aufzunehmenden Schnappschuss ein.
  4. Aktivieren/deaktivieren Sie das Kontrollkästchen Sperren, um festzulegen, ob der Schnappschuss die Aufbewahrungseinstellungen ignorieren soll, wenn das automatische Entfernen aktiviert wird.
  5. Klicken Sie auf OK, um mit der Schnappschussaufnahme zu beginnen.

Hinweis:

  • Von den Einstellungen unter Systemsteuerung > Gemeinsamer Ordner > Bearbeiten wird nur die Windows ACL-Berechtigung in einem Schnappschuss erfasst.

Schnappschüsse von gemeinsamen Ordnern durchsuchen:

  1. Wählen Sie einen gemeinsamen Ordner aus, dessen Schnappschüsse Sie durchsuchen möchten.
  2. Klicken Sie auf Schnappschussliste.
  3. Wählen Sie einen Schnappschuss aus, den Sie durchsuchen möchten, und klicken Sie auf Durchsuchen.

Hinweis:

  • Das Durchsuchen von Schnappschüssen ist für verschlüsselte gemeinsame Ordner nicht verfügbar.
  • Gehen Sie, wenn Sie den Schnappschuss nicht suchen können, zu Systemsteuerung > Gemeinsamer Ordner > Bearbeiten, um Schnappschuss sichtbar machen zu aktivieren.

Schnappschüsse von gemeinsamen Ordnern entfernen:

  1. Wählen Sie einen gemeinsamen Ordner aus, dessen Schnappschüsse Sie entfernen möchten.
  2. Klicken Sie auf Schnappschussliste.
  3. Wählen Sie einen oder mehrere Schnappschüsse aus, die Sie entfernen möchten, und klicken Sie auf Entfernen. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  4. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Löschen, um das Entfernen zu bestätigen.

Schnappschüsse von gemeinsamen Ordnern bearbeiten:

  1. Wählen Sie einen gemeinsamen Ordner aus, dessen Schnappschüsse Sie bearbeiten möchten.
  2. Klicken Sie auf Schnappschussliste.
  3. Wählen Sie einen Schnappschuss aus, den Sie bearbeiten möchten, und klicken Sie auf Bearbeiten.
  4. In Beschreibung können Sie die Beschreibung des Schnappschusses ändern und Sie können das Kontrollkästchen Sperren aktivieren/deaktivieren, um festzulegen, ob der Schnappschuss die Aufbewahrungseinstellungen ignorieren soll, wenn das automatische Entfernen aktiviert wird.
  5. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.

Geplante Schnappschüsse konfigurieren:

Bei geplanten Schnappschüssen nimmt "Schnappschuss & Replikation" zur vorgegebenen Zeit und mit der festgelegten Häufigkeit automatisch Schnappschüsse auf, sodass der Aufwand der manuellen Aufnahme von Schnappschüssen entfällt.

  1. Wählen Sie einen gemeinsamen Ordner aus.
  2. Klicken Sie auf Zeitplan einstellen.
  3. Aktivieren Sie auf der Registerkarte Plan das Kontrollkästchen Schnappschuss-Zeitplan aktivieren und legen Sie die Zeit und die Häufigkeit für die Aufnahme von Schnappschüssen fest. Deaktivieren Sie das Kontrollkästchen, um geplante Schnappschüsse für diesen freigegebenen Ordner zu stornieren.
  4. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.
  5. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf OK, um die Konfiguration zu bestätigen.

Hinweis:

  • Die Konfiguration geplanter Schnappschüsse wird unter Systemsteuerung > Aktualisieren und Wiederherstellen > Sicherung der Systemkonfiguration nicht erfasst.

Aufbewahrungseinstellungen für die geplanten Schnappschüsse konfigurieren:

  1. Wählen Sie einen gemeinsamen Ordner aus.
  2. Klicken Sie auf Zeitplan einstellen.
  3. Wählen Sie die Registerkarte Aufbewahrung aus.
  4. Konfigurieren Sie Ihre Aufbewahrungseinstellungen. Wählen Sie eine der folgenden Optionen aus:
    • Nach Erreichen des Grenzwerts alte Schnappschüsse entfernen: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, werden die ältesten Schnappschüsse von neuen überschrieben.
    • Schnappschüsse immer aufbewahren: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, nimmt Die Funktion „Schnappschuss & Replikation“ keine neuen Schnappschüsse mehr auf, ohne die bereits aufgenommenen zu entfernen.
    • Schnappschüsse gemäß der folgenden Richtlinie aufbewahren: Sie können Aufbewahrungsregeln für bestimmte Zeitbereiche konfigurieren. Weitere Informationen finden Sie unter dem Hinweis Aufbewahrungsrichtlinie.
  5. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.

Hinweis:

  • Die Änderung der Aufbewahrungseinstellungen beeinträchtigt die entsprechenden Einstellungen auf der Seite Replikation.

iSCSI LUN

Schnappschuss für eine iSCSI LUN aufnehmen:

  1. Wählen Sie die iSCSI LUN aus, von der Sie einen Schnappschuss aufnehmen möchten.
  2. Klicken Sie auf Einen Schnappschuss machen.
  3. Geben Sie in Beschreibung eine Beschreibung für den aufzunehmenden Schnappschuss ein.
  4. Wählen Sie aus der Dropdown-Liste Sperren die Option Ja/Nein aus, um festzulegen, ob der Schnappschuss die Aufbewahrungseinstellungen ignorieren soll, wenn das automatische Entfernen aktiviert wird.
  5. Klicken Sie auf OK, um mit der Schnappschussaufnahme zu beginnen.

Schnappschuss einer iSCSI LUN entfernen:

  1. Wählen Sie eine iSCSI LUN aus, deren Schnappschüsse Sie entfernen möchten.
  2. Klicken Sie auf Schnappschussliste.
  3. Wählen Sie einen oder mehrere Schnappschüsse aus, die Sie entfernen möchten, und klicken Sie auf Entfernen. (Halten Sie die Umsch- oder Strg-Taste gedrückt, wenn Sie mehrere Dateien oder Ordner auswählen möchten.)
  4. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Ja, um das Entfernen zu bestätigen.

Schnappschuss einer iSCSI LUN bearbeiten:

  1. Wählen Sie eine iSCSI LUN aus, deren Schnappschüsse Sie bearbeiten möchten.
  2. Klicken Sie auf Schnappschussliste.
  3. Wählen Sie einen Schnappschuss aus, den Sie bearbeiten möchten, und klicken Sie auf Bearbeiten.
  4. Den Schnappschuss können Sie in folgender Weise bearbeiten:
    • Beschreibung des Schnappschusses in Beschreibung ändern
    • Wählen Sie Konsistenzstatus aus, wenn anwendungskonsistenter Schnappschuss aktiviert ist.
    • Wählen Sie Ja/Nein in der Dropdown-Liste Sperren aus, um festzulegen, ob der Schnappschuss die Aufbewahrungseinstellungen ignorieren soll, wenn das automatische Entfernen aktiviert wird
  5. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.

Bei geplanten Schnappschüssen nimmt "Schnappschuss & Replikation" zur vorgegebenen Zeit und mit der festgelegten Häufigkeit automatisch Schnappschüsse auf, sodass der Aufwand der manuellen Aufnahme von Schnappschüssen entfällt.

  1. Wählen Sie eine iSCSI LUN aus.
  2. Klicken Sie auf Zeitplan einstellen.
  3. Aktivieren Sie auf der Registerkarte Plan das Kontrollkästchen Schnappschuss-Zeitplan aktivieren und legen Sie die Zeit und die Häufigkeit für die Aufnahme von Schnappschüssen fest. Deaktivieren Sie das Kontrollkästchen, um geplante Schnappschüsse für diesen freigegebenen Ordner zu stornieren.
  4. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.

Aufbewahrungseinstellungen für die geplanten Schnappschüsse konfigurieren:

  1. Wählen Sie eine iSCSI LUN aus.
  2. Klicken Sie auf Zeitplan einstellen.
  3. Wählen Sie die Registerkarte Aufbewahrung aus.
  4. Konfigurieren Sie Ihre Aufbewahrungseinstellungen. Wählen Sie eine der folgenden Optionen aus:
    • Nach Erreichen des Grenzwerts alte Schnappschüsse entfernen: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, werden die ältesten Schnappschüsse von neuen überschrieben.
    • Schnappschüsse immer aufbewahren: Wenn die maximale Anzahl von Schnappschüssen erreicht ist, nimmt Die Funktion „Schnappschuss & Replikation“ keine neuen Schnappschüsse mehr auf, ohne die bereits aufgenommenen zu entfernen.
    • Schnappschüsse gemäß der folgenden Richtlinie aufbewahren: Sie können Aufbewahrungsregeln für bestimmte Zeitbereiche konfigurieren. Weitere Informationen finden Sie unter dem Hinweis Aufbewahrungsrichtlinie.
  5. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.
  6. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf erneut OK, um die Konfiguration zu bestätigen.

Konsistenzstatus Ihrer iSCSI LUN-Schnappschüsse konfigurieren:

  1. Wählen Sie eine iSCSI LUN aus.
  2. Klicken Sie auf Zeitplan einstellen.
  3. Wählen Sie die Registerkarte Anwendung aus.
  4. Aktivieren/deaktivieren Sie das Kontrollkästchen Anwendungskonsistente Schnappschüsse aktivieren, um zwischen den folgenden Konsistenzstatus zu wechseln:
    • Anwendungskonsistent: Schnappschüsse mit diesem Konsistenzstatus können nur aufgenommen werden, wenn Synology Snapshot Manager, je nach der verwendeten Benutzerumgebung für die Kommunikation mit DSM, auf Ihrem Windows-Server oder VMware vCenter-Server installiert ist. Wenn anwendungskonsistente Schnappschüsse in DSM ausgelöst werden, werden die Anwendungen entsprechend benachrichtigt. Sie übertragen daraufhin alle Daten aus dem Speicher zu den LUNs, von denen ein Schnappschuss aufgenommen werden soll, um die Konsistenz der Daten zu gewährleisten.

      Hinweis:

      • Ist Synology Snapshot Manager nicht in Ihrer Betriebsumgebung installiert, bleibt der anwendungskonsistente Schnappschuss auch dann deaktiviert, wenn das Kontrollkästchen aktiviert ist.
    • Absturzkonsistent: Wenn Synology Snapshot Manager auf keinem Ihrer Clients installiert ist, können Sie nur absturzkonsistente Schnappschüsse von LUNs aufnehmen. Wenn absturzkonsistente Schnappschüsse ausgelöst werden, werden die Anwendungen nicht angewiesen, die Daten vom Speicher zu den LUNs zu übertragen. Daher wird in diesem Fall keine Datenkonsistenz gewährleistet.
  5. Klicken Sie auf OK, um Ihre Einstellungen zu speichern.

Hinweis:

  • Die Änderung der Konsistenzstatuseinstellungen beeinträchtigt die entsprechenden Einstellungen auf der Seite Replikation.

Aufbewahrungsrichtlinie

Diese Option bietet eine ideale Lösung zum Festlegen der maximalen Anzahl von Schnappschussversionen, um Volume-Speicherplatz zu sparen. Möglicherweise müssen Sie Ihre Schnappschüsse aber für längere Zeit aufbewahren.

Synology nutzt die GFS-Aufbewahrungsrichtlinie („Grandfather-Father-Son“). Sie können die maximale Anzahl von aufzubewahrenden Schnappschussversionen für die folgenden Zeiträume konfigurieren: stündlich, täglich, wöchentlich, monatlich und jährlich.

"Schnappschuss & Replikation" bewahrt für jeden Zeitraum die konfigurierte maximale Anzahl von Schnappschüssen auf. Existiert in einem Zeitraum mehr als eine Schnappschussversion, wird nur die früheste aufbewahrt. Legen Sie beispielsweise als Richtlinie 10 wöchentliche Schnappschüsse fest, bewahrt "Schnappschuss & Replikation" für jede der letzten 10 Wochen den frühesten Schnappschuss auf (wenn pro Woche mehr als ein Schnappschuss aufgenommen wird).

Damit Sie die letzten Schnappschussversionen schnell finden und aus ihnen wiederherstellen können, werden alle Schnappschussversionen nach Aufnahme des neuesten Schnappschusses standardmäßig eine Stunde lang aufbewahrt. Um zu verhindern, dass ein Schnappschuss aufgrund der Aufbewahrungsregeln automatisch entfernt wird, können Sie ihn sperren.
