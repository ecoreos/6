安裝或購買套件

套件中心提供豐富多樣化的網頁應用程式，讓您可以盡情為 DiskStation 增添各種功能。透過套件中心，您只需點按幾下滑鼠即可輕鬆下載、安裝及開啟套件。

本文章將引導您如何使用套件中心來在 DiskStation 上安裝套件。如需更多各個套件的相關資訊，請參閱此處。

若要安裝或購買套件：

  1. 開啟套件中心。
  2. 前往全部頁面來檢視所有套件。

    3. 注意：

    • 若無法找到您所搜尋的套件，可能是因為該套件需要較新版本的 DSM，或該套件僅適用於滿足特定硬體需求的機種。請聯絡套件開發人員以取得更多資訊。
  3. 找到您想要安裝的套件，並按一下安裝套件。(針對付費套件，請按一下購買或試用來連至 Synology 帳戶，並使用 PayPal 購買或透過試用版進行評估。)
  4. 成功安裝套件後，該套件會出現在主選單中。

購買套件：

在購買任何套件之前，請先參閱下列說明。

  • 註冊一組 Synology Synology 帳戶來取得更多功能，例如先前帳單資訊及購買歷程記錄。您所分享的資訊僅會用來提供更完整的服務。
  • 套件中心僅接受 PayPal 線上付款。如您需要退貨，請造訪 PayPal 官方網站。
  • 套件目前以美金標價。

貨幣換算免責聲明：

對於涉及貨幣換算的交易，您的信用卡 / 金融卡發卡機構可能會加收不包含於顯示價格內的額外費用。所有訂購皆可能需再計算發卡機構收取的外匯或跨國處理費。所有應付費用均由您的發卡機構所核定，Synology 恕無法提供償款。請聯絡您的信用卡 / 金融卡發卡機構以獲得更多資訊。

若要手動安裝套件：

  1. 如果您想要上傳 .spk 檔案 (可於 Synology 下載中心或協力廠商網站取得) 來安裝套件，請按一下手動安裝並跟隨精靈指示。

若要搜尋套件：

  1. 在左上角的搜尋列中輸入關鍵字，並按下 Enter 鍵來開始搜尋。

若要選擇套件的排序方式：

在套件中心裡，您有數種方式可以改變套件的排列順序。選擇任一分類，按一下右上角的清單圖示，並選擇下列任一排序方式：

  • 依名稱排序：按一下讓套件依照英文字母順序排列。
  • 依近期熱門程度排序：按一下讓套件依照最近的下載次數排列，下載次數越多的出現在越前面。

若要隱藏已安裝的套件：

選擇任一分類，按一下右上角的清單圖示，並選擇不要顯示已安裝之套件。如此一來，已安裝的套件將會被隱藏，僅顯示您尚未安裝或需要升級、修復的套件。
