總覽

此頁面顯示 high-availability 叢集及成員伺服器的基本資訊。此外，本頁亦可讓您維護 high-availability 叢集及其成員伺服器。

High-availability 叢集狀態

此欄位顯示 high-availability 叢集的整體狀態，並在發生錯誤時提醒您。下列表格列出您可能會看到的情況：

狀態 描述
良好 High-availability 叢集總體狀態正常。
載入中 系統正在載入叢集狀態。在狀態讀取結束前系統將無法執行轉移。請參閱轉移。
設定中 系統正在設定中。在系統設定完成前將無法執行轉移。
警告 系統狀態異常，但服務仍可正常運作。在此情況下可能無法執行轉移。
危險 主伺服器與副伺服器皆出現異常，或儲存空間已損毀。High-availability 叢集無法正常運作。請檢查所有儲存空間 (例如 RAID Group、磁碟群組、儲存空間及 iSCSI LUN) 的狀態。在此情況下可能無法執行轉移。

警告訊息

下列表格顯示當系統發生錯誤時可能出現的警告訊息。出現多個警告訊息時，可使用向前 / 向後按鈕來瀏覽所有訊息。

警告 / 錯誤 描述
此 high-availability 叢集中沒有副伺服器 無法取得副伺服器資訊。請確定副伺服器已正確連接。此狀態下無法進行轉移。
無法偵測到副伺服器 由於副伺服器離線或主、副伺服器無法透過 Heartbeat 連線接通，而可能會出現此錯誤。此狀態下無法進行轉移。
無法執行轉移 主伺服器已觸發故障轉移，但並未成功執行。此狀態下，主伺服器將不會再次嘗試故障轉移。請依照下列步驟來排除錯誤：
  1. 嘗試解決 high-availability 叢集上的問題。
  2. 將副伺服器重新開機。
  3. 按一下管理 > 轉移來執行轉移。
監控的項目異常 一個或多個監控的項目 (包括系統狀態、服務或網路介面) 發生錯誤。
主 / 副伺服器語擴充裝置間的連線發生錯誤 擴充裝置沒有正確連接至主伺服器或副伺服器。請確定擴充裝置與主伺服器的連線正確，並與副伺服器建立相同的連線方式。此狀態下無法進行轉移。
儲存空間已降級 儲存空間 (例如 RAID Group、磁碟群組、儲存空間及 iSCSI LUN) 已降級。請查看儲存空間狀態頁面。
儲存空間已損毀 儲存空間 (例如 RAID Group、磁碟群組、儲存空間及 iSCSI LUN) 已損毀。請查看儲存空間狀態頁面。主伺服器無法存取硬碟時，若副伺服器可成功存取硬碟，將自動執行故障轉移。除此情況之外，將無法進行轉移。
Heartbeat 連線中斷 由於 Heartbeat 連線發生異常，而可能會出現此錯誤。此狀態下無法進行轉移。
硬體故障 由於主伺服器或副伺服器的硬體 (例如：電源或風扇) 發生故障，而可能會出現此錯誤。
實體記憶體大小不同 由於主伺服器和副伺服器的實體記憶體大小不一致，而可能會出現此錯誤。此狀態下無法進行轉移。
此 high-availability 叢集尚未配置固定 IP 由於 high-availability 叢集沒有固定 IP，而可能會出現此錯誤。此狀態下無法進行轉移。
監控中的網路介面尚未配置固定 IP 由於監控中的介面沒有固定 IP，而可能會出現此錯誤。此狀態下無法進行轉移。
修復副伺服器 正在修復副伺服器。此狀態下無法進行轉移。
副伺服器升級中 系統正在升級副伺服器。此狀態下無法進行轉移。

High-availability 叢集「管理」按鈕

此按鈕位於 high-availability 叢集資訊下方。此按鈕包含下列選項：

  • 轉移：進行從主伺服器到副伺服器的轉移。在此過程中，主伺服器會停止提供服務，並成為新的副伺服器。原本的副伺服器會變為新的主伺服器，開始提供 high-availability 叢集的所有服務。副伺服器狀態錯誤或未選擇副伺服器時。無法使用此功能。
  • 更新 DSM：開啟控制台中的系統更新頁面。您可在此處更新兩台伺服器。
  • 關閉 high-availability 叢集：將主、副伺服器關機。
  • 將主伺服器關機：關閉主伺服器。若副伺服器可正常運作，服務將進行轉移。
  • 將副伺服器關機：關閉副伺服器。
  • 將主伺服器重新開機：重新啟動主伺服器。若副伺服器可正常運作，服務將進行轉移。
  • 將副伺服器重新開機：重新啟動副伺服器。
  • 修復未知伺服器：若更換副伺服器，或系統讀取為未知的伺服器時，可修復副伺服器。
  • 關閉系統嗶聲：中斷主伺服器及副伺服器在錯誤發生時發出的嗶聲。若之後又發生其他錯誤時，系統仍會發出嗶聲。

注意：

  • 如果您將副伺服器關機或沒有副伺服器，當主伺服器發生錯誤時無法轉移服務。

伺服器圖示及資訊

伺服器圖示會顯示伺服器的狀態。圖示位置會對應到該伺服器所連接模組的位置。移動滑鼠到圖示上方來取得更多詳細資訊。

圖案說明包含下列情況：

  • 此 high-availability 叢集中沒有副伺服器：顯示空白圖框。
  • 未偵測到副伺服器：顯示模糊的伺服器圖案。
  • 驚嘆號：硬體可能故障，或 Heartbeat 連線異常導致無法取得伺服器資訊。

注意：

  • 若另一台伺服器顯示為「未知的伺服器」，則表示該伺服器不在 high-availability 叢集中。按一下管理並選擇修復未知伺服器來修正此問題。

伺服器資訊包含下列項目。未偵測到副伺服器時會顯示 N/A。

  • 產品序號：Synology 伺服器 S/N 號碼。
  • 風扇狀態：伺服器風扇狀態。風扇異常時會顯示警告訊息。若出現此類訊息，請連絡當地經銷商尋求協助。
  • 溫度：Synology 伺服器系統溫度。
  • 電源狀態：顯示伺服器的所有電源狀態是否正常。若伺服器產品型號具備備援電源，當其中一個電源供應器異常時會顯示警告訊息。
  • 實體記憶體：Synology 伺服器的實體記憶體大小。若兩台伺服器的實體記憶體大小不同將無法進行轉移。

兩台伺服器間的連線狀態

此部分說明主伺服器和副伺服器之間的圖案。圖案說明包含下列情況：

  • 沒有圖案：表示未偵測到副伺服器。
  • 靜止的圖案：表示 Heartbeat 連線不正確。
  • 驚嘆號：表示兩台伺服器間的通訊異常，或 Heartbeat 連線不正確。

網路介面狀態

此部分說明伺服器與資訊網路之間的圖案。圖案說明包含下列情況：

  • 沒有驚嘆號：表示未偵測到網路錯誤。
  • 驚嘆號：表示監控中的網路介面發生錯誤。

共用儲存空間圖示及資訊

共用儲存空間圖示會顯示共用儲存空間與伺服器之間的連接狀態。移動滑鼠到圖示上方來取得更多詳細資訊。

圖案說明包含下列情況：

  • 沒有驚嘆號：表示未偵測到錯誤。
  • 驚嘆號：表示擴充裝置的連接狀態異常。

共用儲存空間基本資訊包括以下項目。前往儲存空間管理員來取得更多詳細資訊。

  • 風扇狀態：顯示擴充裝置的風扇狀態。風扇異常時會顯示警告訊息。若出現此類訊息，請連絡當地經銷商尋求協助。
  • 電源狀態：顯示伺服器的所有電源狀態是否正常。當其中一個電源供應器異常時會顯示警告訊息。
