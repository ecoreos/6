使用限制及系統需求

Synology High Availability 功能需要使用兩台相同的 Synology 伺服器來分別擔任主伺服器與副伺服器的角色。在將兩台 Synology 伺服器組成 high-availability 叢集前，系統將檢查以下硬體及軟體限制，以確保 high-availability 叢集能夠順利組成。

系統需求

請注意下列硬體及軟體相關說明：

  • Synology 伺服器：主伺服器與副伺服器型號必須相同。
  • 擴充裝置：主伺服器及副伺服器都必須正確地連接到相同的擴充裝置。
  • 硬碟數量及類型：硬碟類型必須為 SAS (SSD/HDD)。安裝 DSM 前，您必須在擴充裝置中安裝至少 3 顆硬碟。
  • USB 外接裝置：僅支援 USB 儲存裝置。其他 USB 外接裝置 (例如：UPS、Wi-Fi 接收器、擴音器、印表機、藍芽接收器、數位電視棒等) 皆不支援。USB 儲存裝置僅可用於主伺服器，且無法轉移至副伺服器。

網路環境

在嘗試建立 high-availability 叢集時，請注意下列關於網路設定的指示。

  • 網路設定：兩台伺服器間的 Heartbeat 連線必須正確連接。兩台伺服器必須在相同的子網路之下，且安裝後需設定固定 IP 位址。
  • 網路孔數量：兩台伺服器必須擁有相同的網路孔數量。如果您的其中一台 DiskStation 配備額外的網路介面卡，另一台 DiskStation 也必須裝有相同的網路介面卡。
  • 連線限制：High-availability 叢集不支援 Wi-Fi 無線網路。

注意：

  • 主伺服器上的 NTP 伺服器將會自動啟動。

套件及軟體限制

請參閱下方的套件及軟體相關限制：

  • High-availability 叢集不支援以下套件及功能：
    • DVBLink 伺服器
    • CMS 中將 high-availability 叢集重新開機 / 關機的功能
    • Time Backup
  • 不支援以共用儲存空間的 high-availability 叢集來組成鏡像複製方式的 high-availability 叢集。

專用擴充裝置

主伺服器和副伺服器都必須連接到專用的擴充裝置。每一台擴充裝置都有兩個模組。

若要安裝 / 新增擴充裝置：

  • 一個模組包含兩個插孔；輸入插孔在左側，輸出插孔在右側。將模組的輸出插孔連接 SAS Cable 到下一台模組的輸入插孔。
  • 擴充裝置必須依序串連，一台伺服器使用同一邊的模組，另一台伺服器使用另一邊的模組。
  • 主伺服器和副伺服器都必須以相同順序連接到共用的擴充裝置。

注意：

欲取得擴充裝置安裝步驟的詳細資訊，請參閱伺服器及擴充裝置的快速安裝指南。
