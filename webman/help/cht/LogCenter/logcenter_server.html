日誌接收

您可以將 DiskStation 做為一台 syslog 伺服器使用，透過網路接收來自用戶端裝置的日誌。支援的接收日誌格式為 BSD (RFC 3164)、IETF (RFC 5424) 及自訂三種。所有接收到的日誌皆會儲存在日誌資料庫中。

在您開始進行之前：

前往日誌中心 > 封存設定，選擇一個存放及封存日誌的位置。

若要設定標準日誌接收：

  1. 前往日誌中心 > 日誌接收 > 標準。
  2. 您可以在需要的格式之下勾選接收來自其他裝置的日誌核取方塊，來啟動 BSD 或 IETF 格式日誌接收。
  3. 從傳輸協定中選擇 UDP 或 TCP。
  4. 指定一個連接埠編號以便接收連接埠的 syslog 訊息。預設的連接埠編號為 514。
  5. 如果您想要加強日誌傳輸的安全性，請勾選啟動安全連線 (SSL) 選項。只有在選擇 TCP 通訊協定時，才可啟動此選項。
  6. 按一下套用來儲存設定。

若要設定自訂日誌接收：

  1. 前往日誌中心 > 日誌接收 > 自訂。
  2. 按一下新增。
  3. 在名稱中輸入自訂的規則名稱。
  4. 在規則參數中輸入分析規則。參閱下方的注意事項，以取得關於建立模式分析器的詳細資訊。
  5. 從傳輸協定中選擇 UDP 或 TCP。
  6. 指定一個連接埠編號以便接收連接埠的 syslog 訊息。預設的連接埠編號為 514。
  7. 如果您想要加強日誌傳輸的安全性，請勾選啟動安全連線 (SSL) 選項。只有在選擇 TCP 通訊協定時，才可啟動此選項。
  8. 按一下確定來儲存設定。

注意：

關於模式分析器的詳細資訊，請參閱這裡。您也可以下在範例模式來參考。

若要管理自訂規則：

  1. 選擇一個已存在的自訂規則。
  2. 從下列選項中選擇：
    • 編輯：編輯選取的自訂規則。
    • 刪除：刪除選取的自訂規則。
    • 啟動：啟動選取的自訂規則。
    • 停用：停用選取的自訂規則。

匯出憑證

如果您勾選啟動安全連線 (SSL) 選項，您可以按一下匯出憑證按鈕來匯出憑證，以便授權其他裝置傳送日誌到 DiskStation。
