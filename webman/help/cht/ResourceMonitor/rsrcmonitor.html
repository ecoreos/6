資源監控

資源監控讓您可以監控 CPU 使用率、記憶體使用率、磁碟使用情形和網路流量。您可以選擇即時監控或檢視過去的資料。

資源監控讓您可以監控 CPU 使用率、記憶體使用率和網路流量。您可以選擇即時監控或檢視過去的資料。

注意：

  • 當您勾選設定頁籤中的啟動歷史記錄後，系統才會顯示過去的資料。
  • 在 Docker DSM 中，CPU 和記憶體為本機端的資訊。

效能

按一下效能頁籤來檢視資源使用情形。當滑鼠移上圖表時，會顯示詳細的日期、時間及資訊。

CPU

您可查看使用者、系統及 I/O 等待的整體 CPU 使用率。在剛開啟此畫面時，CPU 負載可能會很高，因為系統除了正在收集系統資源的統計資料外，還需負載顯示頁面。

使用者代表 CPU 用於處理使用者應用程式的比例。系統代表 CPU 用於系統核心、軟 / 硬體中斷 (IRQ) 的比例。I/O 等待則代表用於 I/O 等待的比例。

使用率為使用者及系統 CPU 使用率的總和。

您可看到 1、5、15 分鐘的平均負載，代表運行進程佇列 (執行中的程序數加上等待執行的程序數之總和)。

名詞解釋

  • I/O 等待：I/O 等待代表 CPU 處於閒置狀態且沒有被任何程式使用。

    • I/O 等待是正常的現象。CPU 速度通常遠大於 I/O，因此 CPU 經常需要等候磁碟或網路傳輸完畢。然而，如果 I/O 等待的比例過高，代表 CPU 耗費大部份的時間在 I/O 等待上，而沒有被妥善利用。您可以使用較有效率的磁碟群組配置 (例如 SHR 或 RAID 5) 或網路設定 (例如 Bonding) 來取得更佳的系統效能。

記憶體大小

畫面上會顯示整體的實體記憶體使用率及置換空間的記憶體使用率。記憶體使用率維持很高是因為系統將常用資料儲存於快取中，因此能快速地取得資料，不需存取硬碟。當整體記憶體不足時，系統會釋放快取記憶體。

如果置換空間使用率極高，代表系統記憶體不足，將會影響到系統效能。您可以從上方的下拉選單中選擇置換空間來檢視移入 / 移出率。

網路

網路流量圖會以每秒多少 KB 的方式顯示資料傳送與接收的流量。若有啟用 PPPoE 服務，亦會顯示 PPPoE 的網路流量。

磁碟

您可以按一下圖表上方的下拉選單，來選擇檢視整體的磁碟使用率、傳輸速率或讀寫次數。請按一下下方的檢視全部，勾選更多磁碟並將其新增至圖表。

儲存空間

此頁面顯示儲存空間整體傳輸的狀態。按一下圖表上方的下拉式選單來檢視傳輸用途、傳輸速度或傳輸效能 (IOPS)。按一下檢視全部按鈕來選擇個別的儲存空間，並將其新增至圖表。

iSCSI

此頁面顯示目前運作中的 iSCSI LUN 的整體狀態。按一下圖表上方的下拉式選單來選擇您要檢視哪一個 LUN 的狀態。預設會顯示三個圖表，分別為流量、讀寫次數以及延遲率。按一下選擇圖表按鈕來選擇要檢視的其他圖表以取得更詳細的資訊。

程序

按一下程序頁籤來檢視各個程序的 CPU 及記憶體使用率，系統會根據 CPU 負載由高至低排列。

程序狀態包含執行中、睡眠中及已停止。程序狀態對應至 Linux 則為 running、stopped/tracing、sleeping。

睡眠中可後接英文字母以表示詳細資訊，如「D」代表 disk sleep，「Z」代表 zombie，「X」代表 dead。

目前連線使用者

在此頁籤中，您可以檢視並管理目前連線至系統及存取資源的用戶端。此區塊顯示所有透過 HTTP、TELNET、SSH、SMB/CIFS、AFP、FTP、及 NFS 連線的用戶端。

若要管理目前連線，請執行下列任何操作：

  • 按一下儲存來將目前的清單匯出成 HTML 檔案。
  • 按一下重新整理來重新整理清單。
  • 按一下踢除連線來中止使用者對系統資源的存取動作。中斷連線的使用者必須經過重新授權才能存取系統資源。
  • 按一下停用使用者來停用使用者。若要再次啟動使用者，請前往主選單 > 控制台 > 使用者帳號。

注意：

  • 踢除連線的選項不適用於 NFS 用戶端。
  • 當 NFS 用戶端沒有正常卸載共用資料夾時，用戶端可能仍會顯示在目前連線使用者清單中。
  • 當 DiskStation 重新開機後，先前所連線的 NFS 用戶端將不會顯示在目前連線使用者清單中。

速度限制

在速度限制頁籤下方，您可以檢視並管理 DiskStation 上受到速度限制的檔案傳輸程序清單。

若要重新整理清單：

按一下重新整理來重新載入頁面，獲得最新的資料記錄。

若要中止檔案傳輸：

按一下踢除連線來中止檔案傳輸。您必須重新開始檔案傳輸才能完成此傳輸程序。

設定

按一下設定來設定下列選項：

  • 從下拉選單中選擇即時時間長度 (以分鐘為單位)。
  • 若您要檢視過去的資料，請勾選啟動歷史記錄。啟動後，您在檢視效能圖表時，可以從右上方的下拉選單中切換即時、1 天、1 週、1 個月、1 年。
