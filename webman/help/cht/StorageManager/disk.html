HDD/SSD

您可在此監控 DiskStation 上所安裝的 HDD/SSD 狀態，並透過此處的選項來管理、分析 HDD/SSD 效能及健康狀況。請前往儲存空間管理員 > HDD/SSD 來管理這些選項。

若要檢查 HDD/SSD 資訊：

  1. 前往HDD/SSD頁籤。
  2. 請從清單中選擇硬碟，並將其展開。您可在此查看 S.M.A.R.T. 狀態、儲存空間資訊、型號、序號、韌體版本及溫度。

若要檢查健康資訊：

  1. 前往HDD/SSD頁籤。
  2. 選擇硬碟。
  3. 按一下健康資訊。
  4. 前往總覽頁籤。您可在此得知硬碟運作時數、目前溫度、重新連線 / 重新辨識次數、壞軌數，及整體 S.M.A.R.T. 狀態。
  5. 前往 S.M.A.R.T. 資訊頁籤。此處列出的各項數值可幫助您進一步瞭解硬碟狀態。

注意：

SAS 硬碟不支援 S.M.A.R.T. 資訊。

若要執行 S.M.A.R.T. 檢測：

S.M.A.R.T. 檢測可檢查並回報硬碟狀態，在硬碟可能發生問題前提出警訊。若偵測出任何錯誤，建議您立刻更換硬碟。

  1. 前往HDD/SSD頁籤。
  2. 選擇硬碟。
  3. 按一下 S.M.A.R.T. 檢測。
  4. 選擇一種檢測類型並按一下開始來進行檢測。
  5. 最新的檢測結果會顯示在檢測結果區塊中。

注意：

檢測所需時間會因您的系統環境而有所不同。

若要安排 S.M.A.R.T. 檢測排程：

您可以建立任務來依排程自動執行 S.M.A.R.T. 檢測。

  1. 前往測試排程表頁籤。
  2. 按一下新增來建立任務。
  3. 在一般頁籤中修改下列設定：
    • 任務：輸入任務的名稱。
    • S.M.A.R.T.檢測類型：選擇要執行快速檢測或完整檢測。
    • S.M.A.R.T.檢測範圍：每次執行 S.M.A.R.T. 檢測時選擇要檢測哪些 HDD/SSD。
  4. 在排程頁籤，請修改下列設定：
    • 日期：指定何時要執行 S.M.A.R.T. 檢測。您可以排定在每週某些日子 (如每天、週末、每星期天等) 或指定的日期 (如 2015/10/21) 執行 S.M.A.R.T. 檢測。
    • 時間：指定要執行 S.M.A.R.T. 檢測的次數，以及當日開始執行與最後執行的時間。
  5. 按一下確定來完成。

若要執行 Secure Erase:

  1. 前往HDD/SSD頁籤。
  2. 選擇一個硬碟，然後按一下 Secure Erase。

注意：

  • Secure Erase 不可執行於擴充裝置的硬碟上。
  • Secure Erase 會永久移除硬碟上的所有資料。執行前，請先確認您已不使用該硬碟或已備份資料。
  • 系統在執行 Secure Erase 時，將會暫時鎖定所選的硬碟，並於操作完成後自動解鎖。
  • 操作過程中，若 Secure Erase 被不當停止，您可以直接從使用者介面上重新執行 Secure Erase。若硬碟在鎖定的狀態下被移除或是移至他處使用，您可以輸入密碼「synology」來解鎖硬碟。

啟動 HDD 寫入快取

啟動寫入快取可提升 DiskStation 效能。僅特定 HDD 型號支援此功能。為確保資料安全，強烈建議您使用 UPS 不斷電系統。同時，建議每次使用後正常關機。停用寫入快取能減少無預警斷電時的資料遺失，但相對的會降低系統效能。

若要啟動寫入快取功能：

  1. 前往硬碟快取管理頁籤。
  2. 勾選啟動硬碟寫入快取。依照預設，此功能是啟動的。
  3. 按一下套用。
