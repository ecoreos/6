還原

請前往備份 & 還原，按一下左下角的還原來還原備份資料。

若要還原資料 / LUN* 備份：

  1. 按一下左下角的還原，選擇資料或 LUN*，然後選擇要還原的備份任務。
  2. 畫面可能會提示您選擇系統設定檔、不同版本的備份資料等等。依想要還原的備份任務類型而定。
  3. 如果備份任務已加密，則需密碼 / 加密金鑰來成功還原。
  4. 跟隨精靈的指示來完成流程。

注意：

  • 還原解除安裝的套件時，系統會自動為您安裝同一版本或最新版本的套件。若已有安裝套件，且比當初備份的版本舊，系統會自動為您升級到當初備份的版本。
  • DSM 5.1 以上備份的應用程式，無法在 DSM 5.0 及 DSM 5.0 以下的系統中還原。

注意：

  • 還原完成後，段落分塊及檔案層級 LUN 皆會還成為檔案層級 LUN。
  • 還原套件時若沒有安裝該套件，系統會自動為您安裝同一版本或最新版本的套件。若已有安裝套件，且比當初備份的版本舊，系統會自動為您升級到當初備份的版本。
  • DSM 5.1 以上備份的應用程式，無法在 DSM 5.0 及 DSM 5.0 以下的系統中還原。

若要還原多重版本備份任務裡的單一檔案或資料夾：

  1. 在左側面板選擇備份任務，啟動備份瀏覽器 (按一下放大鏡圖示)。如果備份任務已加密，則需密碼 / 加密金鑰。
  2. 透過時間軸橫桿或是日曆圖示找到想要的備份版本。
  3. 選擇內部檔案或資料夾，按一下還原、複製到...或下載，來取得選擇的檔案 / 資料夾。如果用右鍵按一下檔案 / 資料夾，便會找到此三個選項。

若要搜尋特定版本：

若要加快檔案搜尋速度，可使用關鍵字來篩選想要的檔案 / 資料夾。 不符合的檔案 / 資料夾將被隱藏。

  1. 啟動備份瀏覽器，按一下右上角的漏斗圖示。
  2. 在下拉選單中，可以用以下條件來篩選檔案 / 資料夾：
    • 關鍵字：輸入任何檔案或資料夾名稱。您可以在關鍵字中使用萬用字元 [*] (表示零個以上的字元) 以及 [?] (表示一個隨機字元)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
    • 檔案類型：選擇任何、任何資料夾或任何檔案，進行更精準的搜尋。
    • 大小：選擇任何、等於、大於或小於並輸入數值。
    • 修改日期：從今天、昨天 、上週、上個月選擇您想要的日期區間，或選擇自訂並輸入開始和結束日期來搜尋特定日期區間裡的版本。
  3. 若要套用篩選設定，點一下篩選。
  4. 若要套用相同的篩選條件至其他備份版本，只須利用時間軸切換至想要的版本。
  5. 若您想重新設定您的搜尋條件，點一下重置。

注意：

  • 只有多版本備份任務可使用備份瀏覽器。
  • 如果備份目的地執行 DSM 5.1 或以下版本，則備份瀏覽器的功能可能受限。
  • 系統設定檔和應用程式僅能透過還原來還原。
  • 只有未加密備份任務能夠使用篩選功能。
  • 利用複製到... 或下載取得的檔案 / 資料夾將會遺失原先的權限設定。

若要從其它備份目的地還原備份任務：

  1. 按一下左下角的還原，然後選擇資料或 LUN*。
  2. 可以選擇、還原任一種類的備份任務：
    • 若要還原資料備份：按一下資料 > 從現存資料庫還原，選擇備份任務來還原資料。
    • 若要還原 LUN 備份：按一下 LUN*。
  3. 跟隨精靈的指示。如果要還原的任務已被加密，則需密碼 / 加密金鑰。
  4. 按一下套用來完成還原。

注意：

  • Embedded DataStation 機種僅能還原共用資料夾至 EXT3 或 EXT4 外接裝置上。
  • 系統不支援 Embedded DataStation 機種在沒有 EXT3 或 EXT4 外接裝置的情況下還原應用程式。

____
* 僅適用於特定機型。
