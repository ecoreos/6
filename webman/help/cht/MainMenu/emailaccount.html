電子郵件帳號

您可以在個人設定 > 電子郵件帳號中新增電子郵件帳號，並使用這些帳號來以附件寄送 File Station 上的檔案。

支援的電子郵件服務提供者包含 Gmail、Outlook、Yahoo!、AOL 及自訂 SMTP 伺服器。

若要新增電子郵件帳號：

  1. 按一下新增，然後選擇欲設定之電子郵件帳號的服務提供者。
  2. 輸入您的電子郵件位址、密碼及寄件人名稱。
  3. 若您希望即使建立了多個帳號，File Station 仍預設使用此電子郵件帳號，請勾選將此電子郵件設為預設帳號。
  4. 若您的電子郵件服務提供者 (Gmail、Outlook 及 Yahoo!) 支援此功能，您可以按一下同步聯絡人來將電子郵件的聯絡人同步至 DiskStation。請注意，所同步的聯絡人將來自您於顯示的認證視窗中所提供的電子郵件帳號，而非您於上個步驟所輸入的帳號。因此，若認證視窗顯示不同或非預期的帳號，請手動切換至您想要的或先前輸入的電子郵件帳號。
  5. 按一下套用來儲存設定。

若要編輯電子郵件帳號：

從清單中選擇欲編輯的電子郵件帳號，然後按一下編輯。

若要刪除電子郵件帳號：

從清單中選擇欲刪除的電子郵件帳號，然後按一下刪除。

注意：

  • 預設帳號無法被刪除 (除非已無其他現有帳號)。若您想刪除目前設為預設的帳號，請先變更您的預設帳號再進行刪除。

若要變更預設帳號：

從清單中選擇您欲設為預設的帳號，然後按一下設為預設。

若要以附件寄送檔案：

  1. 在 File Station 的任何檔案上按右鍵並選擇以電子郵件附件寄送。
  2. 若您設有多個電子郵件帳號，您可以於下拉式清單中選擇您想要的寄件人電子郵件位址，並寄送檔案給指定的收件人。
