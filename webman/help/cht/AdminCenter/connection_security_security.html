安全性

您可於控制台 > 安全性 > 安全性為 DSM 瀏覽工作階段修改下列安全性設定：

  • 登入逾時 (分鐘)：如果閒置時間到達指定的時間，使用者便會自動從 SRM 登出。 輸入 1 到 65535 之間的數值。
  • 忽略 IP 檢查來加強瀏覽器的相容性：如果您透過 HTTP 代理伺服器存取 DiskStation 時遭遇隨機登出的問題，便可啟動此選項來忽略 IP 檢查。
  • 強化抵禦跨網站要求偽造攻擊的能力：此選項可加強系統抵抗網頁惡意程式碼攻擊的能力。此選項會於您下次登入 DSM 時生效。
  • 啟動 HTTP 內容安全政策 (CSP) 標頭來提昇安全性：僅接受來自可信賴來源的資料並限制內嵌指令碼執行，藉此提昇系統安全性，加強抵禦網頁惡意程式碼攻擊的能力。
  • 禁止 SRM 被 iFrame 嵌入：您可以啟動此選項來限制其他網站使用 iFrame 將 SRM 嵌入其網頁，以避免 SRM 遭受外來網站的惡意攻擊。按一下允許網站清單可以指定允許使用 iFrame 將 DSM 嵌入的網站。
