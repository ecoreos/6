路由器流量控制

當您的 DiskStation 處於路由器模式時，路由器流量控制將會啟動。路由器流量控制 (位於 控制台 > 網路 > 路由器流量控制) 可管理 DiskStation 私人網路中上傳及下載的流量。在新增規則後，須設定該路由器的保證頻寬與最大頻寬來限制上傳及下載流量。

名詞解釋

  • 保證頻寬：當系統整體頻寬足夠時，該服務保證所能得到的對外流量。
  • 最大頻寬：當系統整體頻寬足夠且有系統剩餘頻寬時，該服務可以使用到的對外流量。建議您先行計算系統對外頻寬。然後，確保各服務所設定之保證頻寬總和不大於系統對外頻寬，否則設定可能無法正常運作。

設定公式

  • 系統對外頻寬 = 各服務保證頻寬之總和 + 系統剩餘頻寬
  • 各服務之保證頻寬 ≤ 各服務之最大頻寬

注意：

若對外封包不符合任何路由器流量控制規則，則預設是不限制流量。選擇所有連接埠，並設定保證頻寬與最大頻寬，您就可以在所有規則後新增您預設的路由器流量控制規則。

若要新增路由器流量控制規則：

  1. 從右上角的下拉式選單中選擇一項網路介面。
  2. 按一下新增。
  3. 在連接埠區塊中，請選擇下列其一：
    • 選擇所有來將規則套用到所有連接埠。
    • 選擇自訂，接著按一下 自訂。最多輸入 15 個連接埠 (以逗號區隔)，或是指定連接埠範圍以套用規則。
    • 您也可指定連接埠的通訊協定。
  4. 在來源 IP 區塊中，請選擇以下其一：
    • 選擇所有來將此規則套用到所有的 IP 位址。
    • 選擇 特定 IP，接著按一下選擇。您可以指定 IP 位址或 IP 範圍來套用此規則。

現在您可以在規則列表中看到新增的路由器流量控制規則。

若要刪除路由器流量控制規則，請執行下列任一動作：

  • 在流量控制規則清單中，取消勾選已啟動欄位的核取方塊。
  • 刪除路由器流量控制規則。

注意：

  • 您最多可以為一項網路介面新增 100 項規則。
  • 以拖拉的方式來調整規則順序。
  • 規則的順序越前面則優先權越高。
  • 指定連接埠指的是區域網路外網際網路服務所用的連接埠編號。
  • 來源 IP 指的是使用 Synology NAS 作為路由器的裝置 IP 位址。
