進階

您可於控制台 > 安全性 > 進階為 DSM 瀏覽工作階段修改下列進階設定。

HTTP 壓縮管理

您可以替 DiskStation 開啟 HTTP 壓縮，降低網路頻寬的使用量，進而加速網頁讀取速度。

若要啟動 / 停用 HTTP 壓縮管理：

  • 前往控制台 > 安全性 > 進階，勾選 / 反勾選 HTTP 壓縮管理。

TLS / SSL Profile

你可以依照需求自行選擇 HTTPS 加密連線的安全等級。有下列三個層級：

  • 現代相容性 - 只相容於較新版的瀏覽器與行動裝置。
  • 中級相容性 - 則是介於兩者中間，為系統預設值。
  • 回溯相容性 - 不安全，但支援絕大多數的瀏覽器與行動裝置。
