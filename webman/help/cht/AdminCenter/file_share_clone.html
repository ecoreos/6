複製共用資料夾*

您可以複製 Btrfs 儲存空間上的共用資料夾，來即時建立現有共用資料夾的備份。複製共用資料夾只會佔據一小部分的容量。

若要複製共用資料夾：

  1. 前往控制台 > 共用資料夾。
  2. 選擇 Btrfs 檔案系統中您想要複製的共用資料夾。
  3. 按一下新增 > 複製。
  4. 輸入新共用資料夾的名稱及描述，並視需求修改其他選項。(如需更多相關資訊，請參閱建立共用資料夾。)

注意：

  • 您僅能複製 Btrfs 儲存空間上的共用資料夾。
  • 共用資料夾會複製至與原共用資料夾相同的儲存空間。
  • 複製的共用資料夾會套用原共用資料夾的進階資料完整性保護設定。

____
* 僅適用於特定機種。
