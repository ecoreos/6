管理進階共用資料夾權限

控制台 > 共用資料夾 > 編輯 > 進階頁面包含下列選項，可讓您更精準地調整共用資料夾的存取權限。

進階共用權限

進階共用權限提供額外的控制功能，讓您管理共用資料夾的存取權限。啟動此功能後，使用者及使用者群組必須同時具備進階共用權限以及 Windows ACL 權限 (位於共用資料夾 > 編輯 > 權限)，才能檢視或修改共用資料夾內容。

若要啟動進階共用權限：

  1. 勾選啟動進階共用權限核取方塊。
  2. 按一下進階共用權限來為共用資料夾修改進階共用權限。

注意：

  • 進階共用權限會在使用者透過下列檔案服務存取共用資料夾時套用：Windows File Sharing、Apple File Sharing、File Station、FTP 及 WebDAV。

進階設定

您可以依需求來進一步限制使用者在透過 File Station、FTP 或 WebDAV 存取共用資料夾時的規則。

  • 禁止瀏覽目錄內容：此選項啟動之後，使用者將無法檢視共用資料夾內的內容。
  • 禁止修改檔案內容：此選項啟動之後，使用者將無法移動、刪除或修改共用資料夾中的檔案。請注意，使用者還是能夠檢視、下載 / 上傳、複製或解壓縮共用資料夾中的內容。
  • 禁止下載檔案：此選項啟動之後，使用者將無法下載共用資料夾內的內容。

注意：

上述限制不適用於屬於 administrators 群組的使用者。
