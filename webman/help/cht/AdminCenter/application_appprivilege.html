權限

您可以檢視並編輯各種服務及應用程式的使用者及群組權限。DSM 提供彈性的設定以控制應用程式權限，讓您容易得管理使用者及群組存取不同應用程式權限。

修改使用者及群組應用程式權限：

  1. 前往控制台 > 權限。
  2. 勾選您要修改的應用程式。按一下編輯。
  3. 按一下使用者帳號以檢視個人權限或使用者群組以檢視群組權限。
  4. 針對每個使用者或群組，將應用程式勾選允許以允許存取或拒絕以拒絕存取。根據應用程式，您也可勾選針對 IP 設定以根據 IP 位置來允許 / 拒絕存取。
  5. 按一下確定來完成 。

檢視使用者在不同網路 IP 位置的應用程式權限：

  1. 前往控制台 > 權限。
  2. 按一下權限檢視器。
  3. 從使用者或群組選擇您要查詢的使用者及在 IP 位置輸入 IP 位置以檢視權限。

設定所有使用者的預設權限：

  1. 前往控制台 > 權限。
  2. 勾選您要修改的應用程式。按一下編輯。
  3. 按一下預設權限及勾選授與權限給所有使用者。
  4. 按一下確定來完成 。

注意：

  • 您無法在此編輯 admin 及 guest 帳號的權限。
  • 遇到權限設定衝突時，權限設定的優先順序為：拒絕 > 授權。
  • 有些應用程式或服務不支援針對 IP 設定存取權限設定。
