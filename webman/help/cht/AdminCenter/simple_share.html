共用資料夾

共用資料夾是 DiskStation 基本的目錄結構，可讓您儲存檔案及資料夾。儲存檔案之前，您必須先在 DSM 上建立至少一個共用資料夾。存放在共用資料夾裡的資料可根據您自訂的存取權限，設定為不公開，或是和特定使用者或群組共享。

當您安裝或啟動特定應用程式、服務或套件時，會自動建立下列預設共用資料夾：

名稱 描述
public 初次設定 DiskStation 時，會自動建立 public 資料夾。*
web 在 DiskStation 上架設網站後，其內容的預設儲存地點為 web 資料夾。Web Station 啟動後，會自動建立此共用資料夾。
photo photo 資料夾用於存放您要在 Photo Station 上分享的相片及影片。Photo Station 或 DLNA/UPnP 媒體伺服器啟動後，會自動建立此共用資料夾。
music music 資料夾用於存放您要在 Audio Station 上聆聽的音樂。Audio Station 或 DLNA/UPnP 媒體伺服器啟動後，會自動建立此共用資料夾。
video video 資料夾用於存放您要透過 DLNA/UPnP 裝置瀏覽的影片。Video Station 或 DLNA/UPnP 媒體伺服器啟動後，會自動建立此共用資料夾。
surveillance surveillance 資料夾用於存放 Surveillance Station 的錄影檔。Surveillance Station 啟動後，會自動建立此共用資料夾。此共用資料夾的權限為唯讀，只有系統管理者可以存取。
home home 資料夾為每位使用者提供私密空間，讓他們放置只有自己可以存取的資料。使用者家目錄功能啟動後，會自動建立此共用資料夾。
homes homes 資料夾包含所有使用者的 home 資料夾。使用者家目錄功能啟動後，會自動建立此共用資料夾。只有系統管理者可以存取並管理所有使用者的 home 資料夾。
NetBackup 網路備份啟動後，會自動建立 NetBackup 資料夾。
usbshare 當您將 USB 硬碟接上 DiskStation 的 USB 埠時，會自動建立 usbshare[編號] 資料夾。*
satashare 當您將 eSATA 硬碟接上 DiskStation 的 eSATA 埠時，會自動建立 satashare 資料夾。*

注意：

  • 使用簡易模式時，您只能編輯使用者的共用資料夾權限。若您想要建立使用者群組或是管理群組對共用資料夾的存取權限，請前往控制台 (進階模式) > 使用者群組。

若要新增共用資料夾：

  1. 前往控制台 (簡易模式) > 共用資料夾並按一下左側面板上的 + 圖示。
  2. 輸入共用資料夾名稱，並選擇您想建立共用資料夾的儲存空間。
  3. 視需求勾選啟動資源回收筒。若啟動此選項，共用資料夾中的檔案被刪除後，會移至名為 #recycle 的資料夾。您可以將資源回收筒設為只有屬於 administrators 群組的使用者可存取。
  4. 為使用者勾選、取消勾選相對應的核取方塊來自訂該使用者的共用資料夾存取權限：
    • 可讀寫：該使用者帳號可以存取並變更共用資料夾中的檔案及子資料夾。
    • 唯讀：該使用者帳號可以存取共用資料夾中的檔案及子資料夾。
    • 禁止存取：該使用者帳號無法存取共用資料夾中的檔案及子資料夾。
  5. 按一下套用來儲存設定。

注意：

  • 共用資料夾名稱不分大小寫，可包含 1 至 32 個字元。名稱不可包含下列字元：
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • 下列名稱已被保留，僅供系統使用︰
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • 共用資料夾名稱的第一個字不可為減號及空白，最後一個字不可為空白。
  • 遇到權限設定衝突時，權限設定的優先順序為：禁止存取 > 可讀寫 > 唯讀。
  • 建立新的共用資料夾時，若屬於 administrators 群組的使用者之權限設定為禁止存取，則這些使用者只能在控制台 > 共用資料夾看到該資料夾。
  • 您可以在 File Station 中對著 #recycle 資料夾按右鍵，選擇內容，然後按一下權限頁籤來設定資源回收筒的權限。

若要編輯共用資料夾：

  1. 前往控制台 (簡易模式) > 共用資料夾。
  2. 在左側面板上選擇欲編輯的共用資料夾。
  3. 編輯該共用資料夾的一般資訊，例如名稱、位置及使用者權限。
  4. 若您先前已勾選啟動資源回收筒，您可以按一下清理資源回收筒來永久移除所有被刪除的檔案。
  5. 按一下套用來儲存設定。

若要刪除共用資料夾：

  1. 前往控制台 (簡易模式) > 共用資料夾。
  2. 在左側面板上選擇欲刪除的共用資料夾。
  3. 按一下左側面板上的垃圾筒圖示來刪除所選的共用資料夾。

____
* 僅適用於特定機型。
