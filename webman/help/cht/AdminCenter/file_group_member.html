新增/移除群組成員

您可以將 DiskStation 使用者加到您建立的群組中，或是將他們從群組中移除。

加入群組成員：

  1. 選擇您想要加入成員的群組，然後按一下編輯群組成員。
  2. 按一下新增。
  3. 在使用者清單中選擇要加入的使用者，然後按一下確定。（若您想要一次多選多個使用者，請按壓住 "Shift" 或 "Ctrl" 鍵，再點選使用者。
  4. 按一下完成按鈕套用設定。

    5. 注意：加入 Administrators 群組的使用者即擁有系統管理者權限，包括 DSM/CIFS/FTP/AFP/WebDAV 等應用程式。

移除群組成員：

  1. 選擇您想要移除成員的群組，然後按一下編輯群組成員。
  2. 在使用者清單中選擇要移除的使用者。
  3. 按一下是。
  4. 按一下完成按鈕套用設定。

    5. 注意：將使用者於 Administrators 群組移除，則此使用者將回復一般使用者權限。
