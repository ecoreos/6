新增使用者

請參見下方說明來在 DiskStation 新增使用者帳號。

若要新增使用者：

  1. 請前往控制台 > 使用者帳號。
  2. 按一下新增按鈕來新增使用者。此動作會開啟新增使用者精靈視窗。
  3. 在使用者帳號資訊頁面，填入下列項目：
    • 名稱：為使用者輸入名稱。此為必填欄位。
    • 描述：為使用者輸入簡短描述。
    • 電子郵件：為使用者輸入電子郵件地址。密碼重設訊息等系統通知信件會一律寄送至此地址。
    • 寄通知信給新使用者：啟動此選項，系統將在新帳號建立之後，寄送訊息知會該使用者。此選項須要先在主選單 > 控制台 > 通知設定 > 電子郵件啟動系統電子郵件通知。依照預設，使用者的密碼不會顯示在通知訊息中。但是，如果您希望通知訊息裡出現密碼，請勾選在通知郵件中顯示使用者密碼核取方塊。
    • 不允許此使用者修改密碼：啟動此選項，使用者將無法變更登入 DSM 使用的密碼。
  4. 在群組 > 新增 > 新增使用者精靈頁面，指定新使用者所屬的群組 (例如：users 或 administrators)。
  5. 在指派共用資料夾權限頁面，修改新使用者的存取權限來選擇其可存取的共用資料夾。權限優先順序為：禁止存取 > 可讀寫 > 唯讀。
    • 預覽：此欄位顯示使用者共用資料夾的實際存取權限，此權限乃根據目前設定及其所屬使用者群組而定。
    • 群組權限：此欄位顯示共用資料夾分配的存取權限，此權限乃根據使用者所屬群組而定。
  6. 在使用者空間配額設定頁面，您可以控制新使用者可使用的儲存空間大小。若要限制使用者可使用的空間大小，請為每一個儲存空間 /共用資料夾勾選設定使用空間配額核取方塊。然後在空間配額欄位 (例如：10 GB) 輸入可使用的最大容量。
  7. 在指派應用程式權限頁面，您可以控制使用者可存取的服務。
  8. 在使用者群組速度限制設定頁面，您可以為各種服務 (例如：File Station、FTP) 啟動速限，限制使用者傳輸檔案時可使用的最大頻寬。請針對每一項服務，選擇下列其中一個選項：
    • 無：速度限制設定會與使用者所屬的群組相同。
    • 固定：套用固定的速限。您可以在左側欄位指定上傳及下載的速限。
    • 變換：可指定兩種速限，速限將根據排程套用至使用者。按一下自訂按鈕來修改速限設定及排程。
  9. 確認設定後，按一下套用來結束。

若要新增使用者：

  1. 請前往控制台 > 使用者帳號。
  2. 按一下新增按鈕來新增使用者。此動作會開啟新增使用者精靈視窗。
  3. 在使用者帳號資訊頁面，填入下列項目：
    • 名稱：為使用者輸入名稱。此為必填欄位。
    • 描述：為使用者輸入簡短描述。
    • 電子郵件：為使用者輸入電子郵件地址。密碼重設訊息等系統通知信件會一律寄送至此地址。
    • 寄通知信給新使用者：啟動此選項，系統將在新帳號建立之後，寄送訊息知會該使用者。此選項須要先在主選單 > 控制台 > 通知設定 > 電子郵件啟動系統電子郵件通知。依照預設，使用者的密碼不會顯示在通知訊息中。但是，如果您希望通知訊息裡出現密碼，請勾選在通知郵件中顯示使用者密碼核取方塊。
    • 不允許此使用者修改密碼：啟動此選項，使用者將無法變更登入 DSM 使用的密碼。
  4. 在群組 > 新增 > 新增使用者精靈頁面，指定新使用者所屬的群組 (例如：users 或 administrators)。
  5. 在指派共用資料夾權限頁面，修改新使用者的存取權限來選擇其可存取的共用資料夾。權限優先順序為：禁止存取 > 可讀寫 > 唯讀。
    • 預覽：此欄位顯示使用者共用資料夾的實際存取權限，此權限乃根據目前設定及其所屬使用者群組而定。

  6. 在指派應用程式權限頁面，您可以控制使用者可存取的服務。
  7. 在使用者群組速度限制設定頁面，您可以為各種服務 (例如：File Station、FTP) 啟動速限，限制使用者傳輸檔案時可使用的最大頻寬。請針對每一項服務，選擇下列其中一個選項：
    • 無：速度限制設定會與使用者所屬的群組相同。
    • 固定：套用固定的速限。您可以在左側欄位指定上傳及下載的速限。
    • 變換：可指定兩種速限，速限將根據排程套用至使用者。按一下自訂按鈕來修改速限設定及排程。
  8. 確認設定後，按一下套用來結束。

使用者名稱限制：

  • 帳號名稱不分大小寫，可包含 1 到 64 個 Unicode 字元，但不可為下列字元︰
    !" # $ % & ' ( ) * + , / :< = >?@ [ ] \ ^ ` { } | ~
  • 第一個字元不可是減號 (-) 或空格，最後一個字元也不可為空格。
  • 下列使用者名稱保留給系統使用：mailer-daemon、postmaster。
  • 使用者描述最多可包含 64 個可顯示的 Unicode 字元。

密碼限制：

  • 密碼有區分大小寫，最多可輸入127 個可顯示的字元，包含英文字、數字、符號、空白鍵等等。

配額限制：

  • 您可為每位使用者輸入的配額上限約為 4 TB (4095 GB)。
