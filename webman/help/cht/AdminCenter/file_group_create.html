建立群組

  1. 前往控制台 > 使用者群組。
  2. 按一下新增。這個動作會開啟新增使用者群組精靈。
  3. 輸入使用者群組名稱及群組說明。請參閱以下區塊來瞭解更多資訊。
  4. 按照接下來的步驟來完成建立族群。
  5. 按一下確定完成設定。

命名規則：

  • 群組名稱不分大小寫，最多可以包含 32 個 Unicode 字元，但不可為下列字元︰

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • 此外，第一個字不可為減號及空白，最後一個字不可為空白。
  • 群組說明有大小寫的區分，最多可輸入64 個可顯示的Unicode 字元。
