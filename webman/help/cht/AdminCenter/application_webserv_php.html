PHP 設定

啟動 Web Station 後，您可以前往控制台 > 網頁服務 > PHP 設定來進行下述 PHP 相關設定。

若要啟動 PHP 快取

勾選啟動 PHP 快取來加強 PHP 執行效能。然而，在某些情況下，此設定並不會產生作用。

若要在網頁上顯示 PHP 錯誤訊息

如果您想要在網頁上顯示 PHP 錯誤訊息，請勾選啟動 display_errors 以顯示 PHP 錯誤訊息。

若要自訂 PHP open_basedir

  1. 勾選自訂 PHP open_basedir 來自訂 open_basedir。
  2. 在文字欄位中輸入自訂路徑，然後按一下套用。如果您想要使用系統預設路徑，請取消勾選自訂 PHP open_basedir 並按一下套用。

注意：

  • 使用分號 (：)來區隔路徑。例如：/etc:/tmp:/usr/bin/php
  • 為了確保安全性，homes 共用資料夾的路徑 (例如：/volume1/homes) 不可新增至 open_basedir。

若要選取 PHP 擴充模組

  1. 按一下選擇 PHP 擴充模組來自訂 PHP 擴充模組。
  2. 在顯示的 PHP 擴充模組清單中，請選擇您要啟動的擴充模組。
  3. 您也可以按一下最上方的系統預設來恢復預設值。
  4. 按一下確定來儲存變更。
