啟動資源回收筒

當您啟動共用資料夾的資源回收筒後，會新增一個子資料夾來存放刪除的檔案。您可以從資源回收筒中取回已刪除的檔案。若您要永久移除這些檔案，請清空資源回收筒。

若要啟動資源回收筒：

  1. 選擇一個共用資料夾，然後按一下編輯。
  2. 勾選啟動資源回收筒。
  3. 如果您想將資源回收筒設為只有管理者可存取，請勾選只允許管理者存取。
  4. 按一下確定來儲存設定。

若要清理資源回收筒：

按一下清理資源回收筒按鈕來永久移除所有被刪除的檔案。

若要清理所有資源回收筒：

從動作下拉式選單中選擇清理所有資源回收筒來永久移除所有資源回收筒中被刪除的檔案。

注意： 您可以在 File Station 中對著 #recycle 資料夾按右鍵，選擇內容，然後按一下權限頁籤來設定資源回收筒的權限。
