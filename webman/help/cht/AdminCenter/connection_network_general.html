一般

控制台 > 網路 > 一般提供一般性的網路選項，例如預設閘道器、DNS 伺服器、代理伺服器以及 IPv6 通道等。

重新命名 DiskStation 裝置

若要重新命名 DiskStation 裝置，請輸入新的伺服器名稱並按一下套用。

伺服器名稱無大小寫之分，可以包含 1 到 15 個字元，由英文字母、數字、底線和減號組成。名稱的第一個字元必須為字母。

設定預設閘道器

請輸入預設閘道器的 IP 位址並按一下套用來儲存設定。

設定 DNS 伺服器 IP 位址

您可以選擇手動設定 DNS 伺服器的 IP 位址。如果選擇此做法，您必須輸入慣用 DNS 伺服器的 IP 位址。您也可以同時輸入替代 DNS 伺服器的 IP 位址，如此一來，慣用 DNS 伺服器要是無法使用，替代伺服器便可提供備援服務。完成 DNS 伺服器的 IP 位址設定之後，請按一下套用來儲存設定。

注意：

當您使用 PPPoE 連線或是讓系統自動設定所有的網路介面時，系統會一併自動設定 DNS 伺服器和預設閘道器等欄位，您也無法變更這些欄位的數值。

透過代理伺服器連線

若要讓 DiskStation 裝置透過代理伺服器連線，請勾選透過代理伺服器連線，輸入主機位址、連接埠，然後按一下套用。

注意：

  • 按一下進階設定來設定 HTTP、HTTPS 代理伺服器以及憑證資訊。
  • 依照預設，HTTPS 代理伺服器的設定會和 HTTP 代理伺服器相同。
  • 勾選透過代理伺服器連線後，DSM 更新、套件中心與部分套件將會透過代理伺服器連線至網路。
  • 若您勾選了近端網址不使用代理伺服器，請確認 DiskStation 所使用的 DNS 伺服器能夠針對區域網路中的裝置解析查詢。

啟動 IPv6 通道

在 IPv6 完全取代 IPv4 前，您需透過 IPv6 通道來讓擁有 IPv6 位址的主機連到 IPv4 之服務。此技術能經由現有的 IPv4 網路傳送 IPv6 流量。
如果您對於設定 IPv6 有任何疑問或遇到連線上的問題，請聯絡您的網路服務供應商 (ISP)。

若要啟動 IPv6 Tunnel：

  1. 勾選啟用 Tunnel 核取方塊來啟動通道支援。
  2. 在伺服器位置中輸入 tunnel 服務供應商的主機名稱或 IP 位址，例如：annonymous.freenet6.net。
  3. 選擇是否要匿名連線或以現有的帳號連線。

注意：

服務供應商可能會主動更新您的外部 IPv6 位址。但 DiskStation 裝置不會立刻移除原有的 IPv6 位址。
