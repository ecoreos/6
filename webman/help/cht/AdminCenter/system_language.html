語言

您可以選擇您偏好的 DSM 使用介面、Google Chrome 桌面通知、電子郵件、簡訊、即時訊息通知及應用程式編碼的語言。

介面顯示語言

您可以設定您偏好的 DSM 使用介面及 Google Chrome 桌面通知語言，或是選擇讓兩者顯示的語言和瀏覽器的語言設定相同。

系統通知語言

您可以設定您偏好的電子郵件、簡訊、Windows Live Messenger 及 Skype 訊息通知的語言。所選的語言同時也會套用到 Windows Phone 接收的訊息通知上。

不過，iOS 及 Android 裝置顯示的系統通知語言則取決於行動裝置本身的語言設定。

語言編碼

針對會轉換非 Unicode 字串為 Unicode 的應用程式，例如 FTP 服務、UPnP 支援、建立音樂檔索引等，請從語言編碼下拉式選單中選擇適當的語言編碼。
