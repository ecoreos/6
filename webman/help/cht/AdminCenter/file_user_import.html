匯入清單

您可以使用匯入清單功能一次新增多數使用者帳號。

若要建立使用者清單：

    方法一、使用文件編譯器：

  1. 使用文件編譯器開啟一個新檔案。
  2. 依照下列順序輸入使用者資訊，並以 TAB 鍵分隔:
    • 使用者名稱
    • 密碼
    • 描述
    • 電子郵件
    • 儲存空間 1 的空間配額 (MB)(如果您有一個以上的儲存空間請重複這個步驟
      注意：

      Btrfs 檔案系統及外接儲存裝置 (例如 USB/eSATA 硬碟) 並不支援使用者的空間配額。

  3. 換至下一行，並重複步驟 2 輸入所有的使用者帳號。每一行僅能容納一位使用者的資訊。
  4. 以 UTF-8 編碼儲存檔案。

    方法二、使用 Microsoft Excel：

  1. 使用 Excel 開啟一個新檔案。
  2. 在同一行儲存格中，依照下列順序輸入使用者資訊：
    • 使用者名稱
    • 密碼
    • 描述
    • 電子郵件
    • 儲存空間 1 的空間配額(MB) (如果您有一個以上的儲存空間請重複這個步驟)

      注意：

      Btrfs 檔案系統及外接儲存裝置 (例如 USB/eSATA 硬碟) 並不支援使用者的空間配額。

  3. 換至下一行，並重複步驟 2 輸入所有的使用者帳號。每一行僅能容納一位使用者的資訊。
  4. 使用文件編譯器開啟一個新檔案。複製步驟 2, 3 輸入的使用者資訊至新增的文字檔案。
  5. 以 UTF-8 編碼儲存檔案。

注意：

  • 除使用者帳號之外，所有的欄位皆為選擇性欄位。
  • 以下列舉文字檔案的範例：
    • rosy 1234 mother rosy@abc.com 80000
    chaddy 5678 brother chaddy@abc.com 80000
    danny 8901 cousin danny@abc.com 20000
    anny 2345 sister anny@abc.com 20000

若要輸入使用者清單：

  1. 選擇新增 > 匯入清單。
  2. 如果您想使用清單中的使用者帳號來取代名稱重複的現有帳號，請勾選覆寫重複的使用者帳號。
  3. 按一下瀏覽並選擇使用者清單檔案。選好檔案之後，預覽區域將會顯示要輸入的所有使用者。狀態為異常的使用者將不會被匯入。
  4. 按一下上傳來建立使用者帳號。

注意：

使用者清單的檔案格式必須為 UTF-8。
