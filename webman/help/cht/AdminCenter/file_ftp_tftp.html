TFTP

TFTP (Trivial File Transfer Protocol) 是一種簡單的檔案傳輸通訊協定，通常用於在無須驗證的情況下來傳輸設定或開機檔案。啟動 TFTP 服務可讓 DiskStation 傳輸開機載入檔至用戶端裝置。

若要設定 TFTP 設定：

  1. 前往控制台 > 檔案服務 > TFTP，選擇啟動 TFTP 服務。
  2. 在 TFTP 根目錄欄位，指定哪些 DiskStation 的資料夾可讓 TFTP 用戶端進行存取。
  3. 按一下進階設定來指定下列設定：
    • 啟動 TFTP 檔案傳輸日誌：追蹤透過 TFTP 傳送的資料。
    • TFTP 用戶端存取權限：訂定 TFTP 用戶端的存取權限：
      • 唯讀：TFTP 用戶端僅可檢視根目錄內容。
      • 可讀寫：TFTP 用戶端可修改根目錄內容。
    • 允許用戶端：選擇哪些用戶端可透過 TFTP 進行連線。
      • 所有連線：所有用戶端皆可透過 TFTP 連線。
      • 只允許下列 IP 位址範圍：僅有位於指定 IP 範圍的用戶端可透過 TFTP 連線。
    • 逾時時間：如果 TFTP 伺服器 (即您的 DiskStation) 在指定的時間過後仍無回應，TFTP 連線便會中斷。
  4. 按一下套用來儲存設定。
