SFTP

SFTP 是一種檔案傳輸通訊協定，為 SSH (Secure Shell，「安全殼層」) 協定的延伸。SFTP 只需要一個 TCP 連接埠號碼。此外，無須密碼就可以直接使用私人或公開金鑰來驗證使用者。不過，傳輸速度較緩慢，且加密功能會導致系統消耗較多的資源。

若要啟動 SFTP 服務：

  1. 按一下啟動 SSH FTP 服務 (SFTP)。
  2. 按一下套用。

連接埠編號

為 SFTP 服務設定連接埠編號。

注意：

SFTP 服務的預設連接埠編號為 22。您可將其設定為 1 到 65535 的任何數字，但不可為下列編號：
20, 21, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002，以及預設的 eMule 埠：4662 (TCP), 4672 (UDP)
