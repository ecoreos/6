網路路由

DSM 提供數種網路路由選項以符合各種網路設定需求。

若要在同一個區域網路內分隔不同網路介面的網路流量：

  1. 前往控制台 > 網路 > 進階設定。
  2. 在 一般下方, 勾選當目標 IP 位址為設定在連入介面的本地位址時，回應 ARP 要求。
  3. 按一下套用來儲存設定。

此動作為每一個網路介面建立預設路由規則，使每一個介面的網路封包從各自的實體網路介面傳送出去。此外，若在同一個區域網路中您有兩個以上的網路介面，啟動此選項將分別傳送各個介面的網路流量。 您可分別套用不同的防火牆規則至各個網路介面。

若要經由各個網路介面的閘道器，對各介面進行路由傳送：

  1. 請前往控制台 > 網路。
  2. 在一般下方，勾選啟動多重閘道器。
  3. 按一下套用來儲存設定。

此動作將使路由連通您網路中的各個網路介面，以通過介面中的所有閘道器。

若要改變預設閘道器：

  1. 請前往控制台 > 網路。
  2. 在網路介面下方，按一下服務順序。
  3. 若要調整您的閘道器，請選擇閘道器並按一下上移或下移。
  4. 按一下確定。

最上方排序的閘道器將會成為您區域網路的預設閘道器。

若要設定靜態路由：

  1. 請前往控制台 > 網路。
  2. 在靜態路由下方，按一下建立。
  3. 在目的地網路中，輸入目的地 IP 位址。
  4. 在子網路遮罩中，輸入目的地子網路遮罩。
  5. 在預設閘道器中，輸入目的地預設閘道器位址。
  6. 在介面中，選擇目的地介面。
  7. 按一下確定。

若要檢視 IP 路由表：

  1. 請前往控制台 > 網路。
  2. 在靜態路由下方，按一下IP 路由表。
  3. 從左上方的下拉式選單中選擇網路介面來檢視其 IP 路由表。
