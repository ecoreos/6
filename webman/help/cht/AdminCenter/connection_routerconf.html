路由器設定

如果您的 DiskStation 裝置位於區域網路，則網際網路上的其他裝置無法進行連結。您可於控制台 > 外部存取 > 路由器配置設定連接埠轉送規則，讓您的 DiskStation 裝置可透過網際網路存取。

設定路由器

在新增連接埠轉送規則之前，您需要先設定 DiskStation 裝置與路由器之間的連線。

注意：

  1. 若要配置連接埠轉送規則並指派固定 IP 位址，您需要擁有此路由器的管理權限。
  2. 在設定 DiskStation 裝置與 UPnP 路由器之間的連線之前，請先確定 DSM 已成功擷取 DiskStation 的外部 IP 位址 (位於控制台 > 外部存取 > DDNS)。如果資訊不正確，請檢查 DiskStation 裝置及 UPnP 路由器的網路及防火牆設定。

若要設定 DiskStation 裝置與路由器之間的連線：

  1. 按一下設定路由器來開啟設定精靈。DSM 會先測試下列網路設定：
    • 網路介面
    • 閘道連線
    • 網路連線
    • 連接埠轉送
    • DNS
  2. 視您的路由器類型而定，請執行以下其一：
    • 如果您的路由器不支援 UPnP，請參考步驟 3。
    • 如果您的 UPnP 路由器經測試與 DiskStation 裝置相容，請在您看到路由器的總覽資訊時按一下套用。DiskStation 裝置與路由器之間的連線已成功設定。
    • 如果您的 UPnP 路由器尚未透過 DiskStation 裝置的相容性測驗，請執行以下其一：
      • 如果您想要自動連線至路由器，請選擇自動並按一下下一步來讓系統執行相容性測驗：
        • 如果路由器經測試與 DiskStation 裝置的路由器設定服務相容，請勾選核取方塊來傳送測驗報告以協助 Synology 更新路由器相容性資料庫，然後按一下套用。DiskStation 裝置與路由器之間的連線已成功設定。
        • 如果路由器經測試與 DiskStation 裝置的路由器設定服務不相容，請勾選核取方塊來傳送測驗報告以協助 Synology 更新路由器相容性資料庫，然後按一下關閉。

          • 注意：

          如果路由器經測試為不相容，您仍可在其管理介面中設定連接埠轉送規則。請參閱此處來瞭解更多資訊。

      • 如果您想要手動設定與路由器之間的連線，請選擇手動，按一下下一步，然後參閱步驟 3。
  3. 如果您需要手動設定 DiskStation 裝置與路由器之間的連線，請執行下列操作：
    1. 從路由器下拉式選單中選擇您的路由器。

      2. 注意：

      如果找不到您的路由器，請按一下更新清單來從 Synology 取得最新的清單。

    2. 設定路由器的通訊協定和連接埠。通訊協定和連接埠視路由器型號而定。如果您無法使用預設的通訊協定(HTTP) 和連接埠(80)，請參閱路由器隨附的說明文件來瞭解正確設定。
    3. 如果您需要自訂帳號才能存取路由器，請勾選使用自訂帳號並填寫帳號及密碼欄位。
    4. 按一下套用來完成。

管理連接埠轉送規則

連接埠轉送規則可以指定 DiskStation 的目標連接埠，其會接收來自路由器特定連接埠的封包。

若您的 DiskStation 透過路由器連接至網際網路，請按一下新增來打開連接埠轉送精靈。

您可以使用兩種方式來新增連接埠轉送規則：

  • 如果您要新增系統服務 (例如，FTP、Web Station 等) 的連接埠轉送規則，請選擇內建應用程式。
  • 如果您要新增其他服務的連接埠轉送規則，請選擇自訂連接埠。

您可以用下列語法來指定連結埠號碼和範圍：

  • 使用「-」來描述連接埠範圍，例如：6881-6890。
  • 使用「,」來區分連接埠，例如：「21,22」、「21,55536-55663」。

注意：

  • 每個閘道路由器的連接埠只能使用於一個轉送規則。將路由器的連接埠使用於兩個或更多規則會造成衝突，且畫面上會顯示一個警告訊息。您需要在規則中設定另一個連結埠才能避免衝突。
  • 某些路由器不允許在系統服務轉送方面使用自訂連接埠。

若要儲存規則：

新增規則後，請按一下儲存。否則，將不會套用規則。

儲存規則後，您可能會需要稍待數分鐘，規則才會開始生效，視路由器的型號而定。

若要測試規則是否正常運作：

選擇您要測試的規則，並按一下測試連線。結果會顯示在連線測試結果直欄中。

針對無線路由器設定*

當 DiskStation 設定為無線路由器時，連接埠轉送規則將由 DiskStation 直接實現。

若要增加連接埠轉送規則：

  1. 按一下新增來打開設定視窗。

  2. 請指定服務名稱、通訊協定 (TCP 或 UDP)、內部 IP 位址、內部連接埠及外部連接埠。您可以用下列語法來指定連結埠號碼和範圍：
    • 使用「-」來描述連接埠範圍，例如：6881-6890。
    • 使用「,」來區分連接埠，例如：「21,22」、「21,55536-55663」。
  3. 按一下底部的新增以完成設定。

注意：

大部分路由器皆允許設定連接埠轉送規則時，本地裝置可設定與路由器不同的連接埠。若您的路由器支援此功能：

  • 儲存規則時，若路由器連接埠已使用於存在的規則，將指派新的路由器連接埠。
  • 新增或編輯規則時，您可以直接指定路由器連接埠編號。

DMZ*

當 DiskStation 的網路地圖設定為無線路由器時，您可啟動 DMZ 功能。設置於 DMZ 的主機可直接與外部網路連接，並不受到防火牆的限制及保護。此功能通常用於架設伺服器。

若要啟動 DMZ：

  1. 按一下 DMZ 設定來打開設定視窗。
  2. 勾選啟動 DMZ。
  3. 輸入 DMZ Host IP 位址。
  4. 按一下套用。

注意：

外部網路需透過 DiskStation 取得的外部 IP 地址來連接位於 DMZ 的主機。
