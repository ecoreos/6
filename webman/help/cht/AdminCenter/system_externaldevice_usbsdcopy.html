USBCopy

您可於控制台 > 外接裝置 > USBCopy/SDCopy 為 USBCopy 功能選擇目的地資料夾。

若要啟動 USBCopy：

  1. 請勾選啟動 USBCopy。
  2. 請從下拉式選單中選擇目的地共用資料夾。

在您將一個 USB 裝置（例如數位相機）連接到 Synology 產品前方面板的 USB 連接埠後，您可以按壓前面板的 Copy 按鈕來將 USB 裝置上的資料複製到您指定的目的地共用資料夾。如果目的地共用資料夾被刪除，則系統將會自動變更 public 為 USBCopy 的目的地共用資料夾。

限制

  1. 僅支援格式為 FAT、NTFS、ext3 及 ext4 的 USB 裝置。
  2. USB 裝置必須插入 DiskStation 裝置前方面板的 USB 埠。
  3. 僅有 admin 或是 administrators 群組中的使用者擁有適當的存取權限來備份至 USB 裝置。
  4. 目的地共用資料夾的可用空間必須大於 USB 裝置的已使用空間。
