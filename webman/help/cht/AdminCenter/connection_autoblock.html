自動封鎖

設定自動封鎖可以防止未經允許的登入。啟用本服務後，登入失敗次數過多的 IP 位址都會遭到封鎖。舉凡透過 SSH、Telnet、rsync、網路備份、共用資料夾同步、FTP、WebDAV、Synology 行動應用程式、File Station、DSM 等的登入失敗次數都會累積。您亦可以建立允許清單，清單中的 IP 位址皆不會遭到封鎖。

若要設定自動封鎖：

  1. 前往一般頁籤，勾選啟用自動封鎖。
  2. 在嘗試登入次數和幾分鐘內後面輸入數字。系統將會封鎖在設定分鐘內登入失敗超過嘗試登入次數的 IP 位址。
  3. 勾選啟動封鎖過期並輸入數字來在指定天數後移除封鎖的 IP 位址。若不勾選此服務或封鎖天數設定為 0，被封鎖的 IP 位址將會永遠留在封鎖清單上，除非管理者從封鎖清單手動移除。
  4. 按一下套用。

若要新增封鎖 IP 位址：

按一下封鎖清單頁籤，並從新增下拉式選單中選擇下列其中一種方式：

  • 新增 IP 位址

    • 輸入一個 IP 位址並設定到期時間。

  • 匯入 IP 位址清單

    • 匯入一份列有 IP 位址的文字檔來一次增加多個 IP 位址並設定到期時間。若您欲覆寫現有之 IP 位址，請勾選對應的核取方塊。如果您未勾選此選項，則會跳過重複的 IP 位址。

    若要匯入檔案，該檔案必須符合下列條件：

    • 檔案必須為純文字檔。
    • 檔案中每一行僅能包含一個 IP 位址。
    • # (井字號) 開頭均為註解行.

      • 若您選擇的檔案格式正確，您會在預覽區域看到所有 IP 位址。按一下確定來匯入。

若要移除封鎖 IP 位址：

  1. 按一下封鎖清單頁籤來檢視已封鎖之 IP 位址列表。
  2. 選擇您要從清單移除的 IP 位址並按一下移除。

若要新增允許 IP 位址：

按一下允許清單頁籤，並從新增下拉式選單中選擇下列其中一種方式：

  • 新增 IP 位址

    • 輸入一個 IP 位址。

  • 匯入 IP 位址清單

    • 匯入一份列有 IP 位址的文字檔來一次增加多個 IP 位址。若您欲覆寫現有之 IP 位址，請勾選對應的核取方塊。如果您未勾選此選項，則會跳過重複的 IP 位址。

    若要匯入檔案，該檔案必須符合下列條件：

    • 檔案必須為純文字檔。
    • 檔案中每一行僅能包含一個 IP 位址。
    • # (井字號) 開頭均為註解行.

      • 若您選擇的檔案格式正確，您會在預覽區域看到所有 IP 位址。按一下確定來匯入。

若要移除允許 IP 位址：

  1. 按一下允許清單頁籤來檢視已允許之 IP 位址列表。
  2. 選擇您要從清單移除的 IP 位址並按一下移除。

注意：若您的DS是透過Reverse Proxy Server連接，建議將Reverse Proxy Server的IP加入允許名單中
