編輯群組

您可以修改群組的資訊及共用資料夾存取權限。

若要編輯群組資訊：

  1. 選擇您想編輯的群組。
  2. 按編輯按鈕，並編輯群組資料。
  3. 按一下確定來完成。

若要編輯群組對共用資料夾的存取權限：

  1. 選擇您想編輯的群組。
  2. 按一下編輯及權限。
  3. 勾選或取消勾選下列權限來為群組指派存取權限：
    1. 可讀寫：群組可以存取並變更此共用資料夾。
    2. 唯讀：群組可以存取共用資料夾，但無法對其做出變更。
    3. 禁止存取：群組無法存取此共用資料夾。
  4. 按一下確定來完成。

注意：

遇到權限設定衝突時，權限設定的優先順序為：禁止存取 > 可讀寫 > 唯讀。

若要編輯群組的空間配額限制：

  1. 選擇您想編輯的群組。
  2. 按一下編輯，然後前往空間配額頁籤。
  3. 請針對每一個儲存空間 /共用資料夾，勾選設定使用空間配額，核取方塊，然後在空間配額欄位輸入可使用的最大容量 (例如：10 GB)。

注意：

  • 此配額限制為每個群組成員分別計算，而非共享。
  • 您可為每個群組輸入的配額上限約為 4 TB (4095 GB)。

若要編輯群組的應用程式權限：

  1. 選擇您想編輯的群組。
  2. 按一下編輯然後前往應用程式頁籤。
  3. 請針對每一項應用程式選下列選項之一：
    • 允許：群組可存取應用程式。
    • 拒絕：群組不可存取應用程式。
    • 針對 IP 設定：以 IP 地址管理存取權限。
  4. 按一下確定來完成。

注意：

  • 遇到權限設定衝突時，權限設定的優先順序為：拒絕 > 允許。
  • 有些應用程式或服務不支援針對 IP 設定存取權限設定。

若要編輯群組的速度限制：

您可以為各種服務 (例如：File Station、FTP) 啟動群組速限，限制使用者群組成員透過每項服務傳輸檔案時可使用的最大頻寬。群組速限由全部群組成員分攤。

  1. 選擇您想編輯的群組。
  2. 按一下編輯然後前往速度限制頁籤。
  3. 請針對每一項服務，選擇下列其中一個選項：
    • 無：群組成員使用此服務時不會套用任何速限。
    • 固定：套用固定的速限至群組成員。您可以在左側欄位指定上傳及下載的速限。
    • 變換：可指定兩種速限，速限將根據排程套用至群組成員。按一下自訂按鈕來修改速限設定及排程。
  4. 按一下確定來完成。

隸屬多個群組使用者的速度限制

在使用者隸屬於不同群組的情況下，其速度限制會取決於檔案傳輸時擁有最多可用頻寬的群組。這是因為頻寬是平均分配給正在傳輸檔案的使用者，因此，可用的頻寬會隨著當下消耗群組共用頻寬的使用者數量而出現變化。

舉例來說，使用者「John」屬於「群組 1」(速度限制 500KB/s；共 5 位使用者) 及「群組 2」(速度限制 400KB/s；共 2 位使用者)。

  • 如果兩個群組的使用者同時都在傳送檔案，John 的速度限制會變成 200KB/s，可使用的頻寬則以群組 2 的共用配額來計算。
  • 當群組 1 的其他使用者傳送檔案完畢，John 的速度限制會立刻變成 500KB/s，可使用的頻寬則以群組 1 的共用配額來計算。
