WebDAV

WebDAV 是 HTTP 的延伸，讓使用者可以編輯並管理存放在遠端伺服器上的檔案。WebDAV 服務啟動之後，支援 WebDAV 的用戶端程式，例如特定的 Windows 應用程式、Mac OS Finder、Linux 檔案瀏覽器都可以像存取本地端網路硬碟般遠端存取 DiskStation 上的檔案。

若要啟動 WebDAV：

  1. 勾選啟動 WebDAV 核取方塊；如果您想透過 HTTPS 啟動 WebDAV 連線，則請勾選啟動 WebDAV HTTPS 安全連線。WebDAV 的預設連接埠為 5005，WebDAV HTTPS 則為 5006。
  2. 按一下套用。

速度限制

啟動速度限制可讓您控制特定使用者或群組經由 WebDAV 傳輸檔案時的最高速度。

若要啟動速度限制：

  1. 勾選啟動速度限制；或者，如果您希望速度限制僅在特定時間執行，勾選根據排程啟動速限。此設定會套用至全伺服器，且會覆寫個別使用者及群組的設定。選擇關閉速限將停用所有使用者及群組的速度限制。
  2. 按一下速度限制設定來修改個別使用者或群組的設定。針對每個使用者及群組，您可以選擇下列其一：
    • 套用群組設定：速度限制設定會與使用者所屬的群組相同。
    • 無：群組成員傳輸檔案時將不會套用任何速度限制。
    • 固定：固定的速度限制將會套用至使用者或群組成員。您可在左側欄位中指定上傳及下載之速度限制。
    • 變換：您可指定兩種不同速度限制，並根據排程套用至使用者或群組成員。按一下自訂按鈕來修改速度限制設定並設定排程。
  3. 按一下套用。

注意：

  • 如需更多群組速度限制設定的資訊，請參閱此處。
  • 為使用者設定速度限制時，根據使用者本身及群組設定所計算出的實際速度限制會顯示在結果區塊。例如，使用者「Achilles」本身的下載限制為 100 KB/s，但其所屬的群組下載限制為 500 KB/s，則結果區塊會顯示「0/500」。

進階設定

按一下進階設定按鈕可讓您修改下列選項：

  • 啟動匿名 WebDAV 帳號登入：允許用戶使用「anonymous」使用者名稱來透過 WebDAV 連線存取共用資料夾。請確認您已在共用資料夾頁面為「匿名的 FTP/WebDAV」帳號指定合適的存取權限。
  • 啟動 DavDepthInfinity：允許使用者使用用戶端工具來存取或刪除檔案時，不會受到檔案路徑長度的限制。
  • 啟動 WebDAV 日誌：此選項記錄和 WebDAV 相關的事件。按一下檢視日誌並在彈出視窗中檢閱日誌。

透過不同作業系統使用 WebDAV

DiskStation 的 WebDAV 啟動之後，請依照以下說明，使用 Windows、Mac、Linux 裝置或 Synology 特定的智慧型手機應用程式來存取檔案。

注意：

  • 連接至 WebDAV 服務時，建議您使用支援 UTF-8 編碼的 WebDAV 用戶端，避免出現編碼錯誤。

針對 Window

針對想要使用 WebDAV 通訊協定存取 Synology 產品上檔案的 Windows 使用者，建議您使用磁碟連線的應用程式「BitKinex」，此程式做為非營利目的使用時無須收費。或者您也可以從網際網路上取得其他 WebDAV 用戶端應用程式。

注意：

  • 如果您想使用 Windows 連線網路磁碟機透過 WebDAV 存取 DiskStation 上的檔案，請取得有效的 SSL 憑證，並透過 HTTPS 連線存取 DiskStation。在 Windows 環境下使用 WebDAV 服務時，如果您想要變更使用者帳號，您必須先登出再重新登入 Windows 系統。

針對 Mac

針對 Mac 使用者，您可以直接使用 Finder 透過 WebDAV 連接至 DiskStation。

在 Mac 上使用 WebDAV：

  1. 在 Mac 電腦上，前往 Finder > 連接伺服器。
  2. 在伺服器地址欄位中，輸入 DiskStation 的 IP 位址或網域名稱，開頭為「http://」，結尾為「:5005」(或是您在啟動 WebDAV 服務時所指定的其他埠編號)。例如：http://pm.synology.com:5005/。
  3. 輸入您的 DSM 使用者名稱和密碼。
  4. 按一下 Connect。

注意：

  • WebDAV 支援 Bonjour 廣播功能。

針對 Linux

針對 Ubuntu 或其他 Linux 使用者，請依照以下說明透過 WebDAV 存取 DiskStation 上的檔案。

在 Ubuntu 上使用 WebDAV：

  1. 按一下 位置 > 連接到伺服器。
  2. 從下拉式選單中選擇 WebDAV (HTTP) 或 WebDAV (HTTPS)。
  3. 輸入必要資訊，包含伺服器位址、埠、資料夾以及使用者帳號。按一下連線來進行下一步。
  4. 輸入您的密碼並按一下連線。

在 Linux 上透過命令行使用 WebDAV：

若您想使用命令列，您可以使用 cadaver 這套支援命令列的 WedDAV 用戶端程式來建立連線。

使用 CalDAV 用戶端程式訂閱行事曆

您可以使用電腦或行動裝置上的 CalDAV 用戶端程式來訂閱 DiskStation 上存放的行事曆，然後管理行事曆的事件、備忘錄或待辦事項。可用的行事曆用戶端程式包含 Sunbird (Windows、Mac、Linux)、Chandler (Windows、Mac、Linux)、Evolution (Linux) 或 iCal (Mac、iOS 裝置)。

若要啟動 CalDAV：

  1. 先啟動 WebDAV 或 WebDAV HTTPS 安全連線其中一個。
  2. 勾選啟動 CalDAV。
  3. 按一下套用。

若要管理行事曆清單：

  1. 按一下檢視行事曆清單。
  2. 在顯示的視窗中，請執行下列任何操作：
    • 如果您想要建立新的行事曆，請按一下新增，為行事曆命名 (例如，「cal1」) 並指定行事曆的儲存目的地 (例如，「calendar」共用資料夾)，然後按一下確定。
    • 如果您要變更行事曆名稱及目的地，請選擇行事曆清單中的行事曆，按一下編輯進行變更，然後按一下確定。
    • 如果您想要移除 DiskStation 中的行事曆，請選擇行事曆清單中的行事曆，然後按一下移除。
  3. 按一下關閉。

注意：

  • DiskStation 上存放的行事曆會在 DiskStation 上顯示為資料夾。

若要使用用戶端應用程式訂閱您的行事曆 (以 Sunbird 為例)：

  1. 下載並在電腦上安裝 Sunbird。
  2. 前往 File (檔案) > Subscribe to Remote Calendar (訂閱遠端行事曆)。
  3. 在顯示的視窗中，請選擇 On the Network (位於網路上) 並按一下 Next (下一步)。
  4. 選擇 CalDAV，在 Location (位置) 欄位中輸入 http://DiskStation_DDNS_主機名稱:WebDAV_連接埠編號/行事曆位置/行事曆名稱 (例如，「http://demo.no-ip.info:5005/calendar/cal1」)，然後按一下 Next (下一步)。
  5. 為您的行事曆命名，並按一下 Next (下一步)。
  6. 輸入您的 DSM 使用者名稱和密碼來進行驗證，按一下 OK (確定)，然後按一下 Done (完成)。

若要使用行動裝置訂閱您的行事曆 (以 iOS 裝置為例)：

  1. 在 iOS 裝置上，請前往設定 > 電子郵件 / 聯絡資訊 / 行事曆。
  2. 在帳號下方，請前往新增帳號 > 其他，然後點一下新增 CalDAV 帳號。
  3. 在各欄位中，請輸入 DiskStation 的 IP 位址 (或網域名稱) 以及您的使用者帳號憑證資訊，然後點一下下一步。如果畫面上顯示訊息，請點一下繼續。
  4. 點一下進階設定，並執行下列操作：
    1. 在傳輸埠欄位中輸入 5005 (或是您在啟動 WebDAV 時指定的其他任何埠編號)。
    2. 將帳號 URL 欄位中的網址修改成 http://DiskStation_DDNS_主機名稱:WebDAV_埠編號/行事曆位置/行事曆名稱 (例如，「http://demo.no-ip.info:5005/calendar/cal1」)。
    3. 點一下 CalDAV。
  5. 點一下下一步來驗證您的帳號資訊，然後點一下儲存完成帳號設定。
  6. 回到 iOS 裝置的主畫面螢幕，並打開行事曆來管理您的行事曆。

注意：

  • 如需更多管理行事曆事件、備忘錄或待辦事項的相關資訊，請參閱用戶端應用程式隨附的說明文件。
