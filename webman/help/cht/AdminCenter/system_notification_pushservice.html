推播服務

您可以啟動推播服務來讓 DiskStation 在特定事件發生時寄送通知訊息至您的行動裝置或網頁瀏覽器。

連接埠：

在啟動推播服務之前，請先確認 DiskStation 可以使用連結埠 8089 存取網際網路。

若要啟動 Synology 所提供的電子郵件通知功能：

  1. 勾選透過 Synology 電子郵件伺服器來寄送系統相關的通知訊息。
  2. 在收件人欄位中輸入一個或多個電子郵件位址。系統將會寄送通知訊息至您所輸入的位址。您最多只能輸入兩個電子郵件信箱，並請使用分號 (;) 來分隔。
  3. 按一下套用，系統將會寄送驗證郵件至收件人欄位中所指定的電子郵件位址。請按照驗證郵件中的步驟進行。
  4. 驗證完成後，您可以按一下發送測試通知，然後按一下確定來寄出測試訊息。

若要啟動行動裝置訊息通知：

  1. 前往控制台 > 通知設定 > 推播服務。
  2. 勾選啟動行動裝置訊息通知核取方塊，然後按一下套用。
  3. 在行動裝置上安裝並開啟 DS finder，然後把您的 DiskStation 加到我的最愛清單，完成行動裝置和 DiskStation 的配對。
  4. 按一下發送測試訊息，然後按一下確定來寄出測試訊息。接著在行動裝置上開啟 DS finder，確認是否已接收到測試訊息。

若要管理已配對的行動裝置：

  1. 按一下管理已配對的裝置來檢視所有已配對的行動裝置清單。
  2. 若您想要解除行動裝置的配對，點選該裝置然後按一下解除配對。(多選時請按住 Shift 或 Ctrl 鍵。)行動裝置解除配對之後便不會再收到系統通知，除非再配對一次。

若要啟動瀏覽器通知：

  1. 勾選啟動行動裝置訊息通知，然後按一下套用便可啟動功能。
  2. 按一下啟動瀏覽器訊息通知，便可將 DiskStation 與瀏覽器配對。
  3. 於顯示的視窗中，執行下列任一操作：
    • 針對 Chrome：按一下立即安裝來安裝 Chrome 擴充功能。若您尚未登入您的 Google 帳號，請先登入。
    • 針對 Safari：按一下 允許來授權給訊息通知服務。
  4. 系統將您重新導至瀏覽器訊息通知頁面後，按一下已配對裝置頁籤來檢視配對裝置的資訊，或是來解除 DiskStation 與瀏覽器的配對。也可以按一下事件頁籤來檢視最近七天內的瀏覽器通知紀錄。

注意：

瀏覽器訊息通知服務僅適用於 Chrome (35 或以上版本) 及 Safari (OS X v10.9 或以上版本)。
