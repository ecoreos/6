簡訊

您可以啟動簡訊通知，讓 DiskStation 在系統狀態改變或是錯誤出現的時候寄送訊息通知您，例如連線中斷、風扇狀態改變、斷電、儲存空間不足、備份或還原任務失敗等狀況 。

您可以啟動簡訊通知，讓 DiskStation 在系統狀態改變或是錯誤出現的時候寄送訊息通知您，例如連線中斷、斷電、儲存空間不足、備份或還原任務失敗等狀況 。

若要啟動簡訊通知：

勾選啟動簡訊通知並完成下列設定：

  1. 從簡訊服務供應商選擇您欲使用的簡訊服務供應商。預設的供應商為「clickatell」。
  2. 按一下新增來加入其它供應商的設定，或按下刪除來刪除所選定的供應商設定。
  3. 在使用者帳號欄位中，輸入您向簡訊供應商申請的使用者帳號。
  4. 在密碼及確認密碼欄位中，輸入您向簡訊供應商申請的使用者密碼，並再輸入一次進行確認。
  5. 輸入您的主要電話號碼或次要電話號碼。如果您已輸入兩組號碼，則通知簡訊亦會同時發送到這兩組號碼。
  6. 如果您不想收到太多簡訊，請勾選設定簡訊間隔限制，並限制在您指定的固定時間內只能收到一封通知簡訊。
  7. 點選寄送測試簡訊，系統將根據以上設定發出測試簡訊至您所設定的電話號碼。若沒有收到測試簡訊表示某些設定錯誤。
  8. 按一下套用來儲存設定，或是按一下重置來取消設定。

注意：簡訊通知的語言設定頁面在系統通知語言。若要變更簡訊通知語言，請前往主選單 > 控制台 > 區域選項，按一下語言頁籤，然後從系統通知語言區塊中的下拉式選單選擇您偏好的語言。

    限制：
  1. 使用者帳號：有大小寫之分，可設定 1 到32 個 Unicode 字元。
  2. 密碼：有大小寫之分，可設定 0 到 128 個 Unicode 字元，包含字母、數位、符號，和空格。
  3. 最小發送簡訊間隔可輸入 1 至 10000 分鐘。

若要新增簡訊服務供應商：

  1. 按一下新增。
  2. 在簡訊服務供應商名稱欄位中，輸入簡訊服務供應商名稱。
  3. 在測試網址輸入一組訊息內容為「Hello world」的簡訊網址。您可以從簡訊供應者處取得此網址。
  4. 按下一步。
  5. 系統會從測試網址中分析出數個參數。您需要為每個網址的參數設定一個相對應的類別。除了其他參數以外，各參數僅能使用一次。
    限制：
  1. 供應商名稱：有大小寫之分，可設定 1 到 32 個 Unicode 字元。
  2. 簡訊網址：有大小寫之分，可設定 1 到 500 個 Unicode 字元。網址列參數數目最多為十組。測試網址僅供設定之用。設定完成之後並不會傳送任何簡訊。網址的格式是由簡訊服務供應商提供，而網址必須包含下列參數：使用者帳號、密碼、目的端號碼及訊息內容。以 Clickatell 為例，網址可能如下：https://api.clickatell.com/http/sendmsg?user=TestUser&password=TestPassword&api_id=3148203&to=886123456789&text=Hello+world。請注意，「簡訊內容」參數中「Hello」與「world」之間的空格會以服務供應商定義的特定字元來取代。
