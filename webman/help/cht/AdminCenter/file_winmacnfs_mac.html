Mac 檔案服務

在控制台 > 檔案服務 > Win/Mac/NFS 啟動 Mac 檔案服務，便可讓 DiskStation 成為 Mac 檔案伺服器 (AFP 伺服器)。

若要啟動 Mac 檔案服務：

  1. 勾選啟動 Mac 檔案服務便可讓 DiskStation 成為 Mac 檔案伺服器 (AFP 伺服器)。
  2. 如有需要，可為 Time Machine 設定一個備份資料夾。(請參閱以下區塊來瞭解更多資訊。)
  3. 按一下套用。

注意：

  • 若要將資料從 Time Machine 備份至 DiskStation，請從下拉式選單中選擇一個共用資料夾。
  • 系統服務所使用的共用資料夾無法用在 Time Machine 上。

啟動傳輸日誌記錄

勾選此選項後便可紀錄透過 AFP (Apple Filing Protocol) 進行的檔案存取作業，包含建立、刪除、讀取、寫入、移動及重新命名檔案或資料夾。

進階設定

套用 UNIX 預設權限：

啟動此選項後，上傳或建立檔案及資料夾時將套用 UNIX 預設權限。所套用的權限和透過 UNIX 指令 umask 設定的權限完全相同。此選項啟動時，檔案的 UNIX 權限為 644，資料夾為 755。此選項停用時，檔案的 UNIX 權限為 666，資料夾為 777。預設的 umask 值為 022。

注意：

啟動此選項可能會導致不同通訊協定間出現權限不一致的情況。若要避免不一致的情況，建議您停用此選項。

斷線後立即釋放資源：

啟動此選項後，在 AFP 伺服器連線意外中斷時，將立即釋放資源。若您在使用 Time Machine 時收到「儲存使用中」的錯誤訊息，請啟動此選項。

若要啟動 Bonjour 印表機共用：

勾選啟動 Bonjour 印表機共享來允許 Mac 用戶端瀏覽並找到 DiskStation 上連接的印表機。

若要透過 Finder 上的 Spotlight 搜尋檔案內容：

當資料夾建立完整索引並掛載至 Mac 電腦上後，即可使用 Finder 上的 Spotlight 搜尋內部檔案及檔案內容。

  1. 請前往控制台 > 索引服務 > 檔案索引 > 索引資料夾清單 > 新增來索引特定資料夾，並可依檔案類型索引內部檔案。
  2. 按一下確定來進行索引。
  3. 掛載已建立索引的資料夾至 Mac 電腦上。
  4. 在 Mac 電腦上，按一下 Finder 左側面板上的掛載資料夾，然後使用 Finder 上的 Spotlight 來搜尋內部檔案。

注意：

  • 若要搜尋掛載資料夾內部檔案，請使用 Finder 上的 Spotlight。Mac 桌面上的 Spotlight 僅允許搜尋本機檔案。
  • 在 Mac OS X 10.9 及更新版本中，使用 Finder 的 Spotlight，便可支援依標籤名稱及種類來搜尋掛載資料夾內部檔案。
