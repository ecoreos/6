路由器連接埠

您可以在控制台 > 外部存取 > 進階設定指定公開的外部主機名稱及連接埠編號，讓網際網路服務 (如：Google Docs 或 Pixlr) 能夠存取 DiskStation。

依照預設，主機名稱為您的公用 IP 位址 (或是在控制台 > 外部存取 > DDNS 設定的主機名稱)，連接埠編號為 5000 (HTTPS 連線時為 5001)。但是，如果您擁有另一個未在 DDNS 頁面設定的主機名稱，或是當初設定連接埠轉送規則時指定了其它不同的連接埠編號，您則必須輸入主機名稱或是路由器連接埠編號，才能確保分享至網際網路服務的 DiskStation 檔案能夠被順利存取。

若要啟動路由器連接埠配置：

  1. 在主機名稱或固定 IP 欄位中輸入 DiskStation 的主機名稱或公用 IP 位址。
  2. 將您在路由器上設定 DSM 連接埠轉送規則時指定的路由器連接埠編號輸入至 DSM (HTTP) 或 DSM (HTTPS) 欄位中。
  3. 按一下套用。
