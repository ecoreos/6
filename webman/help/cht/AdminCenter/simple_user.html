使用者帳號

透過彈性的使用者管理功能，您可以為每位家中成員或企業員工建立使用者帳號，並定義每位使用者的共用資料夾存取權限。

若要新增使用者：

  1. 前往控制台 (簡易模式) > 使用者帳號並按一下左側面板上的 + 圖示。
  2. 為使用者輸入名稱及密碼。
  3. 指定使用者所屬的群組。
  4. 指派共用資料夾權限給新使用者。遇到權限設定衝突時，權限設定的優先順序為：禁止存取 > 可讀寫 > 唯讀。
    • 預覽：此欄位顯示使用者共用資料夾的實際存取權限，此權限乃根據目前設定及其所屬使用者群組而定。
    • 可讀寫：該使用者帳號可以存取並變更共用資料夾中的檔案及子資料夾。
    • 唯讀：該使用者帳號可以存取共用資料夾中的檔案及子資料夾。
    • 禁止存取：該使用者帳號無法存取共用資料夾中的檔案及子資料夾。
  5. 按一下套用來儲存設定。

    6. 使用者名稱限制：

    • 帳號名稱不分大小寫，可包含 1 到 64 個 Unicode 字元，但不可為下列字元︰
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • 第一個字不可為減號及空白，最後一個字不可為空白。
    • 下列使用者名稱保留給系統使用：
      mailer-daemon, postmaster

    密碼限制：

    • 密碼有區分大小寫，最多可輸入127 個可顯示的字元，包含英文字、數字、符號、空白鍵等等。

若要編輯使用者：

  1. 前往控制台 (簡易模式) > 使用者帳號。
  2. 在左側面板上選擇欲編輯的使用者帳號。
  3. 編輯該使用者的一般資訊，例如名稱、密碼、群組及共用資料夾權限。
  4. 若您想使該帳號停止運作，請勾選停用此使用者帳號。
  5. 按一下套用來儲存設定。

若要移除使用者：

  1. 前往控制台 (簡易模式) > 使用者帳號。
  2. 在左側面板上選擇欲刪除的使用者帳號。
  3. 按一下左側面板上的垃圾筒圖示來刪除所選的使用者。
