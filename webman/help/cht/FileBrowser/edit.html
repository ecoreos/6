編輯相片

在 File Station，可以使用 Aviary Editor (網頁介面的外部相片編輯器) 編輯相片。

若要使用 Aviary Editor 編輯相片：

  1. 瀏覽至相片所在的資料夾。
  2. 以滑鼠右鍵點一下相片，然後選擇使用 Aviary 編輯。
  3. Aviary Editor 便會在新的瀏覽器視窗中開啟。可使用畫面上提供的工具檢視相片並進行編輯。
  4. 完成相片編輯後，按一下上方工具列的儲存按鈕，便可將所有變更儲存至相片。

注意：

  • 使用 Aviary Editor 可編輯下列格式的相片：.jpg、.jpeg、.jpe、.png
  • Aviary Editor 允許的最大檔案大小為 5 MB。
  • 若要使用 Aviary Editor，DiskStation 須有適當的網路設定。檢查下列設定來進行疑難排解：
    • 確認 DiskStation 可透過 QuickConnect (位於控制台 > QuickConnect) 進行存取，或是可以取得外部 IP 位址 (位於控制台 > 外部存取 > DDNS，或位於控制台 > 網路 > 網路介面 > PPPoE)。
    • 如果 DiskStation 位於路由器後方，請確定您已指定適當的連接埠轉送規則 (位於控制台 > 外部存取 > 路由器配置)。
    • 如果使用 Aviary Editor 時仍遭遇困難，可能需要指定 DiskStation 的路由器連接埠 (位於控制台 > 外部存取 > 進階設定)。
