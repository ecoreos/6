File Station

File Station 是一套便於操作的檔案管理工具，可讓網際網路上的 DSM 使用者透過網頁瀏覽器存取 DiskStation 資料夾。只要開啟兩個或多個 File Station 視窗，您便可以在各個 File Station 視窗之間進行拖放來管理 DiskStation 上所有的資料。
