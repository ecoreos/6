建立檔案請求

檔案請求為寄送給非 DSM 使用者的檔案上傳邀請。透過檔案請求連結，非 DSM 使用者便能直接上傳數個檔案至 DiskStation 上的 File Station，毋須擁有相對的使用者帳號及檔案上傳權限。

若要建立檔案請求連結：

  1. 前往 File Station，選擇資料夾作為上傳目的地。
  2. 按一下操作或用右鍵按一下目的地資料夾，然後選擇建立檔案請求。
  3. 在彈出視窗中，便可找到 / 修改下列資訊：
    • 檔案路徑：列出目的地資料夾路徑。
    • 連結：列出與使用者分享的檔案請求連結。
    • 您的名字：輸入名稱來表示請求者的身份。預設值為您的使用者帳號。
    • 訊息：自訂顯示給使用者的訊息。
    • 啟動密碼保護：若要只讓擁有密碼的使用者能夠透過此連結上傳檔案，選擇此選項。請在下方空格指定密碼。
    • 有效期間：按一下此按鈕，便可自訂檔案請求連結的可用效期。
      • 設定結束時間：若要依日期、時間訂定何時讓連結停止檔案上傳功能，請選擇此選項。
      • 設定開始時間：若要依日期、時間訂定何時讓連結開始檔案上傳功能，請選擇此選項。
      • 可存取次數：若要訂定連結可被存取的次數，請選擇此選項。
  4. 按一下儲存來完成設定。

若要分享檔案請求連結：

若要找到檔案請求網址，請前往 File Station > 工具 > 分享連結管理員，然後針對想要分享的連結按兩下。便可透過下列任一方式分享連結：

  • 將網址複製給想要分享的使用者。
  • 透過預設電子郵件 (您的 DSM 電子郵件帳號) 或透過郵寄 (第三方電子郵件應用程式) 寄送連結。如果為首次使用，請依照精靈指示來設定電子郵件帳號。

若要上傳檔案：

如果您須透過連結上傳檔案，請依下列方式操作：

  1. 在本地電腦或行動裝置 (Android 或 iOS) 上開啟取得的網址。如果需要密碼，請輸入。
  2. 在上傳頁面，輸入能表示個人身份的名稱。
  3. 拖拉檔案便可進行上傳，或按一下加入檔案便可選擇檔案。
  4. 按一下上傳來完成請求。

注意：

  • 資料夾無法透過檔案請求連結上傳。
  • 「web」共用資料夾內的資料夾無法被指定為目的地資料夾。
  • 上傳功能依網頁瀏覽器類別而有所不同：
    • Internet Explorer 8 及 9 不支援以拖拉方式上傳檔案。
    • iOS 裝置上的 Chrome 僅支援上傳圖片。
    • iOS 裝置上的 Safari 僅支援上傳圖片及影片。
    • Android 裝置上的網頁瀏覽器無檔案類型限制。

若要找到已上傳的檔案：

同一位使用者上傳的檔案皆會出現在目的地的相同資料夾內。依照預設，資料夾名稱為上傳者名稱 (請求者名稱)。

例如，如果 Watson 上傳檔案給Holmes，可在名稱為 Watson (Holmes) 的資料夾內找到檔案。

管理檔案請求連結

若要修改檔案請求連結：

建立檔案請求連結後，可以修改設定 (位於 File Station > 工具 > 分享連結管理員)。

  1. 按兩下想要修改的連結，或是選擇連結並按一下編輯。
  2. 在彈出視窗中，修改相關設定。
  3. 按一下儲存來套用新設定。

若要刪除檔案請求連結：

  1. 請前往 File Station > 工具 > 分享連結管理員。
  2. 選擇想要刪除的連結，並按一下刪除。對應的檔案請求網址便會立即失效。

若要指派建立請求的權限：

admin 或屬於 administrators群組使用者可指派權限給使用者，使其能夠建立檔案請求連結。

  1. 請前往 File Station > 設定 > 分享連結 > 檔案請求連結。
  2. 選擇使用者類型，便可給予建立請求的權限。
  3. 按確定以儲存設定。
