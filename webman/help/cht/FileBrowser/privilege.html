檢視或更改檔案及資料夾權限

使用 File Station，您可以設定檔案及子資料夾的存取權限。當您想允許不同群組的使用者存取共用資料夾中的不同子資料夾時，此功能特別實用。

若要允許使用者帳號及使用者群組存取檔案及子資料夾：

  1. 選擇您要設定存取權限的檔案及資料夾。
  2. 以右鍵點按其中一個檔案或資料夾，或是選擇操作選單，然後選擇內容。
  3. 在擁有者 & 群組區塊中， 從擁有者下拉式選單中選擇所選檔案及資料夾的擁有者。
  4. 按一下權限頁籤，選擇擁有者或群組。點一下編輯來開啟權限編輯器。
  5. 進行下列任何操作來管理檔案或資料夾的 ACL 權限：
    • 新增：在權限編輯器視窗中輸入下列資訊來建立權限項目：
      1. 使用者或群組：指定您想自訂權限的使用者或群組。
      2. 繼承自：僅供檢視。檢視此處的資訊來查看權限是繼承自父資料夾或為明確權限 (顯示為無)。
      3. 類型：選擇允許或拒絕來授予或拒絕使用者或群組的權限。
      4. 套用至：若您正為資料夾建立權限項目，請勾選核取方塊來將權限套用至此資料夾、資料夾中的資料夾 (即子資料夾) 或檔案 (即子檔案)，或此資料夾中所有檔案與資料夾 (即所有子項目)。
      5. 管理：勾選讀取權限、變更權限或取得擁有權來指定使用者或群組的存取權限設定。
      6. 讀取或寫入：勾選這些區塊中的核取方塊來修改使用者或群組對於檔案或資料夾的權限設定。
    • 選擇進階選項 > 權限檢閱器並從使用者或群組下拉選單中進行選擇，以檢視使用者或群組對於檔案或資料夾的權限設定。
    • 若您正為資料夾管理權限，請勾選套用到這個資料夾、子資料夾及檔案來將清單上的權限項目套用至資料夾中所有檔案及資料夾。
  6. 按一下確定。

注意：您無法在此編輯 photo 共用資料夾中檔案及子資料夾的存取權限。
