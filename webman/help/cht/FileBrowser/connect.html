遠端連線

在 File Station 中，您可以連線至公有雲端服務 (Box、Dropbox、Google Drive、Microsoft OneDrive)，以及透過不同的通訊協定 (FTP、SFTP、WebDAV、WebDAV HTTPS) 連線至檔案伺服器。

若要連線至雲端服務：

  1. 前往工具 > 遠端連線 > 連線設定。
  2. 選擇您要連線的雲端服務並按一下圖示，再按一下下一步。
  3. 依據精靈的說明在彈出視窗中進行帳號認證。
  4. 設定完成後，您要連線的雲端服務便會出現在左側面板中。

雲端服務特定限制：

  • Box：
    • 與其他協作者 (collaborator) 共用的目錄，名稱之後會加上所有者的使用者帳號。
    • 若有多個與其他協作者共用的目錄採用相同名稱 (即目錄名稱後加上所有者的使用者帳號)，僅會顯示其中一個目錄。
    • 唯有透過 Box，才能開啟使用下列副檔名的檔案：.gdoc、.gsheet、.boxnote。
    • 檔案名稱不區分大小寫。
    • 若要正確設定防火牆，請參閱本篇文章。
  • Dropbox：
    • 若 Dropbox 目錄中含有超過 25000 個檔案，該目錄將無法顯示。
    • 若 Dropbox 目錄中含有超過 10000 個檔案，該目錄將無法重新命名。
    • 無法上傳超過 10 GB 的檔案。
    • 若檔案上傳耗時超過 24 小時，該項上傳任務則會失敗。
    • 檔案名稱不區分大小寫。
    • 若要正確設定防火牆，請參閱本篇文章。
  • Google Drive：
    • 若檔案 / 資料夾路徑長度大於 2048 個字元，將不會顯示該檔案 / 資料夾。
    • 若檔案名稱長度大於 255 個字元，將不會顯示該檔案。
    • 若檔案 / 目錄採用相同名稱，將在名稱之後個別加註建立時間。
    • 若透過 File Station 開啟 Google 專屬的檔案 (例如 Google Doc 和 Google Slide)，將會顯示網址來讓您存取檔案。
    • File Station 中的 My Drive 與 Starred 資料夾各自對應到 Google Drive 中的「我的雲端硬碟」及「已加星號」資料夾。
    • 若要正確設定防火牆，請參閱本篇文章。
  • Microsoft OneDrive：
    • OneNote 筆記本檔案將不會顯示。
    • 無法上傳超過 10 GB 的檔案。
    • 檔案 (如 .png、.jpg、.docx、.pdf) 可能顯示較大的檔案大小 (為原始檔案大小加上縮圖大小)。下載檔案時，原始檔案大小將不受影響。
    • 檔案名稱不區分大小寫。

若要瞭解更多限制，建議您前往該雲端服務所屬的官方網站了解詳情。

若要連線至檔案伺服器：

  1. 前往工具 > 遠端連線 > 連線設定。
  2. 選擇您要使用的伺服器通訊協定並按一下圖示，再按一下下一步。
  3. 提供下列資訊來完成設定：
    • 主機名稱或 IP：輸入您要連線的檔案伺服器的主機名稱或 IP 位址。
    • 連接埠：輸入檔案伺服器的連接埠編號。
    • 帳號名稱及密碼：若檔案伺服器要求帳號 / 密碼，請輸入登入資訊。
    • 語言編碼：選擇檔案伺服器的字元編碼，讓檔案名稱能正確顯示。
    • 設定檔名稱：輸入特定名稱以方便您辨認此檔案伺服器。
    • 使用 WebDAV HTTPS 連線：唯有當您在設定 WebDAV 通訊協定，而且檔案伺服器與用戶端皆支援 WebDAV HTTPS 時，才能勾選此選項。
    • 僅允許每位使用者建立單一連線：唯有當您在設定 FTP 或 SFTP 通訊協定時，才能勾選此選項。當伺服器限制連線數量時，勾選此選項可最佳化連線能力。
  4. 按一下套用來儲存設定。您要連線的檔案伺服器便會出現在左側面板中。

若要修改現有雲端服務或檔案伺服器的設定檔：

  1. 前往工具 > 遠端連線 > 連線清單。
  2. 按一下您要修改的雲端服務或檔案伺服器，來進行下列管理：
    • 編輯：按一下編輯，更改目前設定檔的內容，再按一下確定。
    • 中斷連線：按一下斷線來終止連線。
    • 重新連線：按一下連線來恢復連線。(此外，可在 File Station 左側面板中按一下該雲端服務或伺服器來重新連線。)
    • 刪除：按一下刪除，並確認捨棄該設定檔。(此外，可在 File Station 左側面板中按一下該雲端服務或伺服器，再按一下刪除便可刪除設定檔。)

注意：

  • 每位使用者最多可同時連線至 100 個雲端服務及檔案伺服器。
  • 請確認您是否已選擇正確的伺服器語言編碼，以避免錯誤發生。若不清楚伺服器的語言編碼，建議您選擇 Unicode。
  • 儲存於雲端服務或檔案伺服器中的檔案無法使用某些 File Station 的功能 (例如壓縮到、解壓縮、預覽等)。
  • 所有雲端服務或檔案伺服器連線後的傳輸速度皆受限於使用者設定的速度限制。
  • 在所有連線的雲端服務或檔案伺服器中，每位使用者最多可同時進行三項檔案傳輸。
  • 若要設定 WebDAV、WebDAV HTTPS、雲端服務的代理伺服器，請前往控制台 > 網路 > 一般 > 代理伺服器。若您在連線至雲端服務或檔案伺服器時變更代理伺服器設定，請先中斷連線後再重新連線。
  • 若已連線的雲端服務或檔案伺服器資訊被修改，連線狀態不會受到影響。
  • 若超過 30 分鐘未處理任何檔案，將自動中斷雲端服務 / 檔案伺服器連線，以釋放連線資源。請按一下該雲端服務 / 檔案伺服器來恢復連線。
  • 系統無法顯示採用下列命名格式的檔案 / 資料夾：
    • 開頭為 ._ 或 .SYNOPPSDB
    • 名稱內含 /
    • 名稱完全符合 .    ..     '    "    .DS_Store    @eaDir    @sharebin    @tmp    .SynologyWorkingDirectory
  • 遠端連線只在高權限模式支援。若需使用此功能，請至主機 Docker 套件選擇你想要編輯的 Docker DSM，點一下編輯並勾選使用更高的權限執行容器 選項。
