掛載遠端資料夾

您可以透過 File Station 掛載遠端伺服器 (例如網路上的電腦或其他 Synology NAS) 上的遠端資料夾，即可藉由 File Station 輕鬆瀏覽、管理掛載資料夾。

若要掛載遠端 CIFS 共用資料夾*：

  1. 按一下工具 > 掛載遠端資料夾 > CIFS 共用資料夾。
  2. 在資料夾欄位中，以 \\遠端伺服器\遠端資料夾格式來指定遠端資料夾路徑，例如 \\192.168.0.2\share 或 \\chad\cheeseburger\pictures。
  3. 如果遠端伺服器要求存取遠端資料夾的權限，請在帳號名稱及密碼欄位輸入遠端伺服器上的使用者帳號密碼。
  4. 按一下瀏覽，在 DiskStation 上選擇或新增空白的目的地資料夾，用來掛載遠端資料夾。
  5. 如果希望 DiskStation 在每次系統開機或重新開機時自動掛載此遠端資料夾，請勾選開機時自動掛載。
  6. 按一下掛載，即可掛載遠端資料夾至目的地資料夾。從 DiskStation 便可瀏覽、管理遠端資料夾。

若要掛載遠端 NFS 共用資料夾：

  1. 按一下工具 > 掛載遠端資料夾 > NFS 共用資料夾。
  2. 在資料夾欄位中，以 遠端伺服器:/遠端資料夾路徑格式來指定遠端資料夾路徑，例如 192.168.0.2:/volume1/share。
  3. 選擇想要的 NFS 版本及傳輸通訊協定。
  4. 按一下瀏覽，在 DiskStation 上選擇或新增空白的目的地資料夾，用來掛載遠端資料夾。
  5. 如果希望 DiskStation 在每次系統開機或重新開機時自動掛載此遠端資料夾，請勾選開機時自動掛載。
  6. 按一下掛載，即可掛載遠端資料夾至目的地資料夾。從 DiskStation 便可瀏覽、管理遠端資料夾。

注意：

NFSv4 僅支援 TCP 通訊協定。

若要卸載遠程資料夾：

  1. 選擇想要卸載的遠端資料夾，接著按一下工具 > 卸載。
  2. 另外，也可用右鍵按一下遠端資料夾，然後選擇卸載。

注意：

  • 同時間可掛載的最大遠端資料夾數量為 100。
  • 只有當遠端伺服器支援 CIFS 或 NFS 通訊協定時，才可以掛載它的遠端資料夾。
  • 遠端資料夾僅可掛載至共用資料夾下的子資料夾。
  • 僅有高權限模式支援掛載遠端資料夾。若要使用此功能，前往主機的 Docker 套件，選擇想在其中啟動加密共用資料夾的 Docker DSM，按一下編輯並勾選使用更高的權限執行容器。

____
* 僅適用於特定機型。
