複製或移動檔案或資料夾

File Station 提供多種方式，讓您能在兩臺 DiskStation 之間或單一 DiskStation 之中傳送檔案 / 資料夾，或是傳送至本機電腦上。

在瀏覽器之間拖拉檔案 / 資料夾

透過 File Station，您可以從 DiskStation 直接上傳檔案 / 資料夾至另一臺 DiskStation，毋須設定複雜的檔案傳輸通訊協定。

若要啟動跨瀏覽器拖拉功能：

  1. 請前往設定 > 一般 > File Station。
  2. 勾選啟動跨瀏覽器拖拉功能。
  3. 按一下確定。
  4. 開啟兩個相同種類的網頁瀏覽器 (例如 Firefox)，並個別連線至作為來源端及目的地的 DiskStation。
  5. 在兩臺 DiskStation 上開啟 File Station。您現在可以透過此兩臺上的 File Station 直接拖拉檔案 / 資料夾。

注意：

  • 您也可以在同一個瀏覽器的兩個頁籤之間拖拉檔案 / 資料夾。
  • Internet Explorer 不支援跨瀏覽器及跨頁籤的拖拉功能。
  • 若選取多個檔案 / 資料夾進行拖拉，所選的檔案 / 資料夾將會壓縮成一份壓縮檔後再進行上傳。
  • 拖拉檔案的大小限制依個別網頁瀏覽器有所不同：Safari (1 GB)。

在 DiskStation 中複製或移動檔案 / 資料夾，或是複製、移動至本機電腦

依照下列步驟的指示，在 DiskStation 之中複製、移動檔案 / 資料夾，或是複製、移動至您的本機電腦上。

方法 1：

  1. 選擇您想移動或複製的檔案或資料夾。(多選時請按住 Shift 或 Ctrl 鍵。)
  2. 從操作下拉式選單中選擇剪下或複製。
  3. 前往 DiskStation 或您電腦上的目的地資料夾。
  4. 從操作下拉式選單中選擇貼上 - 覆寫或貼上 - 略過，視您是否要覆寫目的地資料夾中檔名相同的檔案或資料夾而定。

方法 2：

  1. 選擇您想移動或複製的檔案或資料夾。(多選時請按住 Shift 或 Ctrl 鍵。)
  2. 將資料夾或檔案從主要區塊拖曳至左側面板上的另一個資料夾中。
  3. 在顯示的彈出式選單中，選擇複製 - 覆寫、複製 - 略過、移動 - 覆寫或移動 - 略過來覆寫或略過目的地資料夾中名稱相同的檔案或資料夾。

方法 3

  1. 選擇您想移動或複製的檔案或資料夾。(多選時請按住 Shift 或 Ctrl 鍵。)
  2. 請按 Ctrl + X 來剪下或 Ctrl + C 來複製檔案 / 資料夾。若為 Mac 使用者，請按 Command + X 或 Command + C。
  3. 前往 DiskStation 或您電腦上的目的地資料夾。
  4. 請按 Ctrl + V 來貼上檔案 / 資料夾。(Mac 請按 Command + V。)
  5. 在顯示的對話框中，選擇您是否要覆寫或略過目的地資料夾中相同名稱的檔案。

快速鍵：

在 File Station 中移動 / 複製檔案及資料夾時，下列快速鍵可能會派上用場：

  • F2：重新命名
  • Del：刪除
  • Ctrl + A：全選
  • Crtl + X：剪下
  • Crtl + C：複製
  • Crtl + V：貼上

注意：

  • 當檔案或資料夾複製或移動的目的地為 FAT 檔案系統的目錄時，檔名中的 \ *?">| 字元將會由 – 取代，且 Mac 檔案中的 Metadata (中繼資料) 會被刪除。
  • 如果您剪下本機電腦上的檔案並貼到 DiskStation 資料夾中，系統會保留本機電腦上的檔案。
  • 移動檔案或資料夾時，其「擁有者」將維持不變。然而，複製檔案或資料夾時，其「擁有者」將變更為複製該檔案或資料夾的使用者。
