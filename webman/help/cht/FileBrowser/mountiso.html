掛載虛擬裝置

您可以掛載虛擬裝置來使用 File Station 存取光碟映射檔的內容。

若要掛載虛擬裝置：

  1. 選擇光碟映像檔並選擇工具 > 掛載虛擬裝置。
  2. 按一下瀏覽來選擇或新增空的 DiskStation 資料夾作為掛載虛擬裝置的目的地位置。
  3. 如果您想要讓 DiskStation 在系統開機或重新開機時自動掛載此虛擬裝置，請勾選開機時自動掛載。按一下掛載來繼續。現在您可以前往已掛載虛擬裝置的目的地資料夾，並且存取光碟映射檔的內容。

注意：

虛擬裝置及 .iso 檔案僅可掛載至共用資料夾下的子資料夾。

若要卸載虛擬裝置：

  1. 以滑鼠右鍵點按已掛載虛擬裝置的目的地資料夾，然後選擇卸載。
    注意：
  1. 支援的光碟映射檔格式：檔案系統為 ISO 9660 及 UDF 的「.iso」檔案。
  2. 同時間可掛載的最大虛擬裝置數量為 16。
  3. 掛載虛擬裝置只在高權限模式支援。若要使用此功能，前往主機的 Docker 套件，選擇想在其中啟動加密共用資料夾的 Docker DSM，按一下編輯並勾選使用更高的權限執行容器。
