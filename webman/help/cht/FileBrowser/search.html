搜尋檔案或資料夾

File Station 的篩選功能提供了各種選項，可搜尋並顯示符合條件的檔案或資料夾。您可以在目前的資料夾中篩選檔案或資料夾，或執行包含所有子資料夾的進階搜尋來取得更精確的搜尋結果。

若要搜尋目前資料夾中的檔案或資料夾：

  1. 使用 File Station 來前往資料夾 (其中包含您要搜尋的內容)。
  2. 在右上角的篩選欄位中，請輸入關鍵字，並在視窗的主要區域中查看篩選結果。

若要執行進階搜尋：

  1. 使用 File Station 來前往資料夾 (其中包含您要搜尋的內容)。
  2. 在右上角的篩選欄位旁，按一下放大鏡來顯示進階搜尋選單。
  3. 指定下列的搜尋條件或從下拉式選單選擇：
    • 關鍵字：輸入關鍵字來搜尋名稱相符的檔案或資料夾。

      注意：

      • 您可以在關鍵字中使用萬用字元 [*] (表示零個以上的字元) 以及 [?](表示一個隨機字元)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
      • 您可以使用空白來區隔多個關鍵字。例如，使用關鍵字 [File Station] 將會列出包含 [File]、[Station] 或兩者的搜尋結果。
      • 您可以在關鍵字的開頭和結尾處加入雙引號 ["] 來執行精確、不限定大小寫的搜尋。例如，["Synology Inc."] 可以代表 [Synology Inc.]，但是不能代表 [Synology Incorporation]。
      • 下列運算元可讓搜尋結果更加精準：
        運算元 範例 用法
        && [X] && [Y]
      • 列出包含 [X] 及 [Y] 兩者的搜尋結果
        		•
        || [X] || [Y]
      • 列出包含 [X]、[Y] 或兩者的搜尋結果
      • [X] 及 [Y] 兩者皆包含的搜尋結果將置於清單頂端。
        		•
        + +[X]
      • 列出含 [X] 的搜尋結果
        		•
        - -[X]
      • 列出不含 [X] 的搜尋結果
        		•
        ! ![X]
      • 列出不含 [X] 的搜尋結果
        		•
        \ \[特殊字元]
      • 特殊字元：  +   -   &&    ||    !    (    )    {    }    [    ]    ^    "    ~    *    ?    :   \
      • 列出包含目標特殊字元的搜尋結果
        		•
    • 啟動檔案內容搜尋：針對內容或中繼資料，找出符合關鍵字的索引檔案。
    • 所在位置：從下拉式選單中進行選擇來設定進階搜尋的廣度及深度。您也可以選擇列出的任何共用資料夾來自訂搜尋範圍。
    • 檔案類型：選擇副檔名來將搜尋範圍限制為特定類型的檔案，或是選擇任何資料夾來將搜尋範圍限制為僅搜尋資料夾。
    • 日期：選擇修改日期、建立日期或最近存取時間，然後從日曆中選擇日期來搜尋日期相符的檔案。
    • 大小 (MB)：從等於、大於及小於中進行選擇，然後輸入大小來搜尋大小相符的檔案。(選擇等於時，搜尋結果會包含與指定大小接近的檔案。)
    • 擁有者/群組：選擇擁有者或群組，然後從下拉式選單中進行選擇，藉此搜尋所選使用者或群組擁有的檔案或資料夾。

      注意：

      • 若要執行進階搜尋，須指定所在位置及至少另外一項的搜尋條件。
      • 如果共用資料夾已建立索引，並在其中掛載 ISO 資料夾或遠端資料夾，便無法透過進階搜尋找到資料夾內檔案。
  4. 按一下搜尋來開始搜尋，或是按一下重置並再次指定搜尋條件。搜尋完成之後，條件相符的檔案會列在 File Station 的主要區域中。
