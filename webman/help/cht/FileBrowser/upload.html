上傳檔案或資料夾

使用下列任何方法將檔案或資料夾從電腦上傳至 DiskStation。

若要順利使用以下的上傳方法，請先安裝最新的 Java Runtime Environment (JRE) 和 Flash Player。

方法 1：

  1. 在左側面板的上半部中，請選擇 DiskStation 上的一個目的地資料夾。
  2. 按一下上傳，選擇上傳 - 略過或上傳 - 覆寫，並選擇您本機電腦上的檔案或資料夾。

方法 2：

  1. 前往設定，勾選使用 File Station 瀏覽本機電腦的檔案並重新整理網頁，以便在左側面板的下半部檢視本機電腦的資料夾。

    2. 注意：

    在您於電腦上安裝並啟動 JRE 之後，使用 File Station 瀏覽本機電腦的檔案選項才會生效。

  2. 在左側面板的下半部中，請選擇本機電腦上的一個資料夾。主要區塊會顯示所選資料夾的內容。
  3. 在主要區塊中選擇您要的檔案或資料夾，然後按一下上傳。
  4. 在顯示的視窗中，選擇 DiskStation 上的一個目的地資料夾。
  5. 如果您想要覆寫目的地資料夾中名稱相同的檔案或資料夾，請在視窗右上角選擇覆寫。否則，請選擇略過。
  6. 按一下確定來開始上傳。

方法 3：

  1. 前往設定，勾選使用 File Station 瀏覽本機電腦的檔案並重新整理網頁，以便在左側面板的下半部檢視本機電腦的資料夾。
  2. 在左側面板上，請將資料夾從您的電腦 (下半部) 拖曳至 DiskStation (上半部) 上的任何資料夾。
  3. 在顯示的彈出選單中，選擇您是否要覆寫或略過目的地資料夾中相同名稱的檔案 / 資料夾。

方法 4：

  1. 在左側面板的上半部中，請選擇 DiskStation 上的一個目的地資料夾。
  2. 選擇電腦桌面或檔案總管 (例如，Windows 檔案總管或 Mac Finder) 中的檔案，然後將它們拖放到瀏覽器中的 File Station 裡。
  3. 在顯示的對話框中，選擇您是否要覆寫或略過目的地資料夾中相同名稱的檔案。

    4. 注意：

    • 僅特定瀏覽器支援方法 4：Chrome, Firefox, Safari 5.1 或以上版本。然而，當您勾選使用 File Station 瀏覽本機電腦的檔案時，此方法無法於 Mac 上使用。
    • 只有在您於上述瀏覽器中勾選使用 File Station 瀏覽本機電腦的檔案時才可以拖放資料夾或空的檔案。否則，您無法使用方法 4 上傳資料夾及空的檔案。
    • 如果您是使用 Linux 版 Chrome 透過 HTTPS 通訊協定連線至 DSM，則需要在控制台 > 安全性 > 憑證中匯入信任的憑證，才能使用方法 4 的上傳方式。

注意：

  1. 您最多可以將 1000 個檔案加到上傳佇列中。
  2. 如果您是使用 Google Chrome、Firefox、Safari 5.1 或以上版本，或是已勾選使用 File Station 瀏覽本機電腦的檔案，便可以上傳總容量超過 4GB 的檔案。否則，您一次最多僅可以上傳 2GB 的檔案。
  3. 在非加密資料夾中，檔案或資料夾的名稱必須在 255 個字元以內 (非拉丁語系至多 80 個字元)；在加密資料夾中，檔案或資料夾的名稱必須在 143 個字元以內 (非拉丁語系至多 47 個字元)。
  4. 檔案或資料夾的名稱不可包含下列字元：\ / : * ? " > |.
  5. 檔案或資料夾的名稱不能以「._」開頭。
  6. 如果您已安裝 JRE，則可以上傳大小為 0KB 的檔案。

疑難排除

如果您無法使用 Safari 正常瀏覽 Mac 上的本機項目，請進行下列步驟：

  1. 在您的 Mac 上，開啟 Safari 並從 Safari 選單中選擇偏好設定。
  2. 在安全性頁籤中按一下管理網站設定。
  3. 在 Java 中，選擇您的 DSM「網站」。
  4. 選擇在非安全模式中執行。
