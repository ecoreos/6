協助工具功能

DiskStation Manager 內建協助工具功能，透過聽覺提示，引導視力障礙使用者瀏覽、管理 DSM 內的各項服務 / 設定。

設定 DSM 協助工具環境

若要讓協助工具功能發揮最大效果，建議您依照下列指示，設定操作環境：

  • 在本機電腦上安裝 NVDA 螢幕助讀程式 (僅供 Windows 環境使用)。
  • 安裝最新版本的 Mozilla Firefox 網頁瀏覽器。
  • 使用耳機或喇叭，並調整音量以符合個人需求。
  • 使用 NVDA 焦點模式 (按一下 NVDA 鍵 + 空白鍵來切換模式)。預設的 NVDA 鍵為 Insert。
  • 使用下列快速鍵來瀏覽 DSM：
    快速鍵 描述
    Tab 切換至使用者介面上的其他區塊
    方向鍵 切換至使用者介面區塊內的其他項目
    Enter 開啟應用程式 / 視窗
    Esc 關閉應用程式 / 視窗

協助工具功能目前在下列 DSM 區域中皆可使用：

  • DSM 登入頁面、桌面及工具列
  • 主選單
  • 控制台
  • File Station

注意：

若要瞭解 NVDA 操作細節，請參閱使用說明。

DSM 鍵盤快速鍵說明

DSM 不僅內建協助工具功能，更提供豐富的鍵盤快速鍵，讓操作、管理的過程更加簡單、流暢。按一下 Shift + ? 鍵，便可得知各類快速鍵：

  • 桌面鍵盤快速鍵：視窗操作及其他操作所需的快速鍵
  • 元件鍵盤快速鍵：控制使用者介面中各項元件 (例如選單、按鈕、搜尋欄位) 所需的快速鍵

若要暫時停用桌面相關的快速鍵，取消啟動桌面鍵盤快速鍵選項。
