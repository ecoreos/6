使用 Cloud Station 備份 Mac/PC 上的資料

總覽

Cloud Station Server 不僅是不僅是功能完整的跨平台同步軟體，亦可提供使用者更智慧的方式來備份個人電腦。  Cloud Station 能即時同步，相較於須消耗大量頻寬的傳統備份方式，還能在伺服器 (NAS) 端保留多達 32 個版本。

在 Synology NAS 上設定 Cloud Station Server

  1. 前往套件中心來尋找 Cloud Station Server 並點一下安裝。
  2. 啟動 Cloud Station Server。
  3. 前往總覽頁面並點一下 Cloud Station Backup 下方的立即下載按鈕。 (亦可前往 Synology 下載中心進行下載。)
  4. 在電腦上開啟安裝程式，並依照指示安裝 Cloud Station Backup。
  5. 安裝完成後，在電腦上執行 Cloud Station Backup。 按一下立即開始。
  6. 輸入您的電腦名稱、執行 Cloud Station Server 的 Synology NAS 位址 (或 QuickConnect ID)、使用者名稱及密碼。 （您可以點一下右邊的搜尋圖示讓 Cloud Station Backup自動尋找 LAN 裡其他的 Synology NAS）。 按一下下一步。
  7. 選擇您的備份來源並取消勾選您不要同步的子資料夾。
  8. 點一下選擇以選擇 Synology NAS 上的資料夾。 Cloud Station Backup 將會在您選擇的目的地建立一個同名的資料夾。
  9. 點一下備份規則來設定條件及規則。
  10. 點一下下一步，確認您的備份設定並點一下完成。
  11. 您可以在主應用程式裡檢視檔案的進度及狀態。

從 Cloud Station Server 下載或還原檔案的先前班本

  1. 前往 Cloud Station > 版本歷史並選擇您要還原的資料夾。
  2. 選擇您要還原的檔案並點一下動作。 點一下瀏覽先前的版本。 若您不小心刪除電腦上的檔案，您可以點一下回復 來還原檔案。
  3. 選擇您想要下載的檔案版本並點一下下載。 如果您確定想要還原您選擇的版本，點一下回復來覆寫現有的版本。

5. 進一步確保您的資料的安全性
