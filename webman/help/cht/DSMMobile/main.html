DSM mobile

DSM mobile 允許 DSM admin (或屬於 administrators 群組的使用者) 使用 iOS® (iPhone®、iPad® 或 iPod touch®) 或 Android® 裝置管理 DSM 設定並檢查 DSM 資訊。此 DSM mobile 說明為您提供下列主題及資源，協助您快速上手。

登入及登出

使用 iOS 或 Android 裝置的網頁瀏覽器連線至 DSM 來查看登入畫面，您可在其中輸入 DSM 帳號資訊。

若要登入 DSM mobile：

輸入您的 DSM 使用者憑證資訊並點一下登入。如果您要儲存登入資訊並在下次自動登入，請在登入之前開啟記住我的登入狀態開關。

若要登出 DSM mobile：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 的右上角點一下選項按鈕。
  2. 點一下設定，再點一下登出。

管理 DSM 使用者

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下使用者 & 群組來查看使用者清單。

若要新增使用者：

  1. 點一下 + 然後輸入新使用者的用戶名稱、密碼及電子郵件位址。
  2. 啟動寄送通知來在 DSM 使用者的帳號建立時寄送通知信至其電子郵件位址。
    注意：傳送通知之前，請確定您已完成 Synology NAS 的通知設定。請使用電腦的網頁瀏覽器登入桌面版 DSM，並前往主選單 > 控制台 > 通知 > 電子郵件來修改設定。
  3. 點一下完成來確認，或點一下左箭頭來取消。

若要快速尋找使用者：

在搜尋欄位中輸入使用者的名稱。

若要修改使用者的資訊：

  1. 點一下使用者名稱並執行下列任何操作：
    • 在名稱或密碼欄位中修改使用者的名稱或密碼。
    • 開啟停用來停用使用者。您可以選擇稍後啟動使用者，這樣做並不會遺失該使用者的設定。
    • 開啟系統管理者來將使用者加到 admin 群組。屬於 admin 群組的任何使用者皆擁有管理權限，可以使用 DSM mobile 或電腦的網頁瀏覽器修改 DSM 設定。
  2. 在權限管理下方，執行下列任何操作，然後點一下返回：
    • 點一下共用資料夾然後點一下共用資料夾的名稱來修改使用者的存取權限：可讀寫：使用者可以變更此共用資料夾中的檔案及子資料夾；唯讀：使用者可以存取共用資料夾中的檔案及子資料夾，但是無法變更它們；禁止存取：使用者無法存取此共用資料夾中的檔案及子資料夾。
      注意：若發生權限衝突時，權限的優先順序為：禁止存取 > 可讀寫 > 唯讀。
    • 點一下應用程式來修改使用者對 DSM 應用程式的存取權限。
  3. 點一下完成來確認，或點一下左箭頭來取消。

管理共用資料夾

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下共用資料夾來查看 Synology NAS 上的共用資料夾清單。

若要新增共用資料夾：

  1. 點一下 +，然後輸入新資料夾的名稱。
  2. 點一下完成來確認，或點一下左箭頭來取消。

若要修改共用資料夾：

點一下共用資料夾的名稱並執行下列任何操作：

  • 在名稱欄位中修改共用資料夾的名稱。
  • 點一下使用者權限然後點一下使用者來修改他的共用資料夾存取權限：可讀寫：使用者可以變更此共用資料夾中的檔案及子資料夾；唯讀：使用者可以存取共用資料夾中的檔案及子資料夾，但是無法變更它們；禁止存取：使用者無法存取此共用資料夾中的檔案及子資料夾。
    注意：若發生權限衝突時，權限的優先順序為：禁止存取 > 可讀寫 > 唯讀。
  • 您也可以點一下群組權限來設定群組的權限。
    注意：群組權限優先於使用者權限。

啟動服務

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下系統服務來啟動或停用 Synology NAS 服務。

若要管理 Synology NAS 的區域網路服務：

  • 啟動或停用下列任何項目：
    • Windows 檔案服務可讓 Synology NAS 成為區域網路中的 Windows (CIFS) 檔案伺服器。
    • Mac 檔案服務可讓 Synology NAS 成為區域網路中的 Mac (AFP) 檔案伺服器。
    • NFS 服務可讓區域網路中的 Linux 用戶端存取 Synology NAS 的資料。
    • Bonjour 印表機可讓 Mac 用戶端使用 Synology NAS 上連結的 USB 印表機進行列印。
    • 媒體伺服器可讓區域網路中的 DLNA/UPnP 家用裝置瀏覽並播放 Synology NAS 上的多媒體內容。
    • iTunes 可讓區域網路中的 iTunes 用戶端瀏覽並播放 Synology NAS 上的音樂或影片。

若要管理 Synology NAS 的網際網路服務：

  • 啟動或停用下列任何項目：
    • FTP 可讓您的 Synology NAS 成為 FTP 伺服器。
    • 網頁服務可讓您在 Synology NAS 上架設網站 (支援 PHP+MySQL)。
    • WebDAV 可讓用戶端程式透過 HTTP 通訊協定管理並編輯 Synology NAS 檔案。
    • WebDAV (HTTPS) 可讓用戶端程式透過 HTTPS 通訊協定管理並編輯 Synology NAS 檔案。
    • File Station 是網頁介面的應用程式，可讓您無須登入 DSM 即可透過自訂埠管理 Synology NAS 檔案。
    • File Station (HTTPS) 可讓您透過 HTTPS 通訊協定連線至 File Station。
    • Surveillance Station 是網頁介面的應用程式，其可管理網路攝影機來保衛家庭或辦公室環境的安全。
    • Audio Station 是網頁介面的應用程式，可讓您存取 Synology NAS 上的音樂資料庫。
    • Photo Station 是整合了網誌功能的相簿，存放在您的 Synology NAS 上。
    • Download Station (BT/HTTP/FTP/NZB/RSS) 是網頁介面的應用程式，可協助您將網際網路上的檔案下載到 Synology NAS 上。
    • eMule 可啟動 Download Station 的 eMule 支援功能。

注意：

如需更多上述服務的相關資訊，請參閱下列任一資料：

  • DSM 說明 (您可以使用電腦的網頁瀏覽器登入 DSM 的一般版，並前往主選單 > DSM 說明進行存取)
  • Synology DSM 使用手冊 (位於 Synology 下載中心)

監控系統資源狀態

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下資源監控來監控 Synology NAS 的 CPU 及記憶體用量。

檢視系統資訊

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下系統資訊來取得 Synology NAS 的狀態總覽。

若要檢視 Synology NAS 的系統資訊：

  1. 執行下列任何操作：
    • 點一下一般來檢視 Synology NAS 的基本資訊，包含序號、型號、實體記憶體大小、DSM 版本、時間資訊以及散熱狀態。
    • 點一下網路來檢視 Synology NAS 的網路設定以及各個網路介面的狀態。
    • 點一下儲存空間來檢視 Synology NAS 的儲存空間用量。
  2. 點一下右上角的重新整理按鈕來讓系統資訊保持最新狀態。

韌體

點一下韌體來檢視及更新您的 DSM 版本。

若要調整 DSM 升級設定：

  1. 點一下 DSM 升級。
  2. 請選擇下列其中一個選項：
    • 僅重大更新
    • 最新版 DSM 及所有更新
  3. 點一下完成。

若要下載可用的更新：

  1. 在狀態中檢查您的 DSM 版本。
  2. 若您的 DSM 不是最新版，點一下出現在狀態區塊下方的下載按鈕來升級 DSM。

管理 IP 封鎖

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下IP 封鎖來阻隔未授權的登入。

若要管理 IP 封鎖：

  1. 點一下右上角的設定按鈕。
  2. 開啟啟動自動封鎖來啟動 IP 封鎖。
  3. 在嘗試登入次數和幾分鐘內後面輸入數字。系統將會封鎖在設定分鐘內登入失敗超過指定的嘗試登入次數的 IP 位址。
  4. 您可以開啟啟動封鎖過期，並指定封鎖天數，在指定天數過後將解除封鎖 IP 位址。若您在此輸入「0」，IP 位址將被持續封鎖，直到您從封鎖清單中移除為止。
  5. 點一下完成。

若要新增 IP 位址至允許 / 封鎖清單：

  1. 若您要在清單中新增允許 / 封鎖 IP，點一下右上角的加入按鈕，再點一下允許清單或封鎖清單。
  2. 輸入 IP 位址後，點一下完成。

管理套件中心

Synology 已將協力廠商或是 Synology 自行設計的應用程式整合到套件中，您可以在 Synology NAS 上輕鬆地安裝並啟動這些程式。

若要管理套件中心：

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下套件中心來檢視套件資訊、啟動或停用套件，或將套件更新至最新版本。

若要啟動或停用安裝的套件：

點一下套件旁邊的切換按鈕來啟動或停用該套件。

若要檢視套件：

點一下套件來檢視其資訊。

若要更新套件：

  1. 點一下更新來檢視目前可供更新的版本。
  2. 點一下套件右方的按鈕來將套件更新至最新版本。

檢視日誌或當前的連線

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下日誌來監控當前連線或檢視 Synology NAS 服務的日誌。

若要監控當前連線：

  • 在總覽區塊中點一下最近 [幾筆] 日誌來檢視最近存取 Synology NAS 服務的使用者清單。

若要檢視特定服務的日誌記錄：

  • 在事件日誌或檔案傳輸日誌區塊中點按任一項目來觀看相關的日誌。

開啟行動應用程式

DSM mobile 為您提供一個便利的位置來開啟 iOS 或 Android 裝置上安裝的 Synology 行動應用程式。

若要使用 DSM mobile 開啟 Synology 行動應用程式：

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 的下半部點按下列任何應用程式：

  • DS photo (須先啟動 Photo Station 服務才可使用)
  • DS audio (須先啟動 Audio Station 服務才可使用)
  • DS cam (須先啟動 Surveillance Station 服務才可使用)
  • DS file (須先啟動 WebDAV 服務 服務才可使用)

注意：

  • 如需更多使用這些應用程式的相關資訊，請參閱應用程式隨附的說明檔案。

操控 Synology NAS
(重新啟動、關機或嗶聲控制)

DSM mobile 可讓您使用 iOS 或 Android 裝置為 Synology NAS 執行重新啟動或關機。此外，當您身邊有多部 Synology NAS 時，您也可以要求您的 Synology NAS 播放嗶聲以進行快速識別。

若要操控 Synology NAS：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 的右上角點一下選項按鈕。
  2. 執行下列任何操作：
    • 點一下重新開機然後點一下是來為 Synology NAS 重新開機。
    • 點一下關機然後點一下是來將 Synology NAS 關機。
    • 點一下搜尋我來要求 Synology NAS 發出嗶聲。點一下確定來停止嗶聲。

切換至 DSM 桌面模式

點一下設定按鈕，然後點一下桌面模式來切換至 DSM 桌面模式。

使用儲存空間管理員

儲存空間管理員可讓您快速檢視系統儲存空間與硬碟的使用及健康狀況。

若要檢視儲存空間資訊：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下儲存空間管理員。
  2. 點一下儲存空間頁籤。
  3. 從清單上點選一個儲存空間來檢視其詳細資訊。

若要檢視硬碟資訊：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下儲存空間管理員。
  2. 點一下 HDD/SDD 頁籤。
  3. 從清單上點選一個硬碟來檢視其建康狀況及 S.M.A.R.T. 資訊。

若要執行或停止 S.M.A.R.T. 檢測：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下儲存空間管理員。
  2. 點一下 HDD/SDD 頁籤。
  3. 從清單上點選一個硬碟來檢視其資訊頁面。
  4. 點一下 S.M.A.R.T. 檢測來執行 S.M.A.R.T. 檢測，或點一下停止來停止執行中的 S.M.A.R.T. 檢測。
