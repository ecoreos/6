快照

您可於此頁面拍攝、移除，和編輯快照，並可設定快照排程及保留設定。

您最多可以為每個共用資料夾或 iSCSI LUN 儲存 256 個快照版本。快照會以個別儲存區塊的方式保留資料，而非複製整個檔案，因此可使用最少的磁碟空間。當您更新或刪除共用資料夾的檔案或 iSCSI LUN 的儲存區塊時，可能會使用額外的空間。

共用資料夾

若要拍攝共用資料夾的快照：

  1. 選擇您要拍攝快照的共用資料夾。
  2. 按一下拍攝快照。
  3. 在描述欄位中輸入快照描述。
  4. 勾選 / 取消勾選鎖定核取方塊來決定該快照是否將在自動刪除觸發時忽略保留設定。
  5. 按一下確定以開始拍攝快照。

注意：

  • 在控制台 > 共用資料夾 > 編輯頁面上的設定中，只有 Windows ACL 權限將會記錄在快照中。

若要瀏覽共用資料夾的快照：

  1. 選擇您要瀏覽快照的共用資料夾。
  2. 按一下快照清單。
  3. 選擇您要瀏覽的快照，並按一下瀏覽。

注意：

  • 加密共用資料夾的快照將無法瀏覽。
  • 若您無法瀏覽快照，請前往控制台 > 共用資料夾 > 編輯來啟動開啟快照瀏覽功能。

若要移除共用資料夾的快照：

  1. 選擇您要移除快照的共用資料夾。
  2. 按一下快照清單。
  3. 選擇您要移除的快照，並按一下移除。(多選時請按住 Shift 或 Ctrl 鍵。)
  4. 您將會看到確認視窗。按一下刪除來確認移除。

若要編輯共用資料夾的快照：

  1. 選擇您要編輯快照的共用資料夾。
  2. 按一下快照清單。
  3. 選擇您要編輯的快照，並按一下編輯。
  4. 您可以在描述欄位中修改快照的描述，以及勾選 / 取消勾選鎖定核取方塊來決定該快照是否將在自動刪除觸發時忽略保留設定。
  5. 按一下確定來儲存設定。

若要設定排程快照：

排程快照可讓快照 & 複寫以事先設定的執行時間及重複間隔來自動拍攝快照，減輕手動快照的工作量。

  1. 選擇共用資料夾。
  2. 按一下設定排程。
  3. 在排程頁籤中，勾選啟動快照排程核取方塊，並設定拍攝快照的執行時間和重複間隔。若要取消該共用資料夾的排程快照，請取消勾選核取方塊。
  4. 按一下確定來儲存設定。
  5. 您將會看到確認視窗。按一下確定來確認設定。

注意：

  • 快照排程設定將不會包含在控制台 > 更新 & 還原 > 系統設定備份功能中。

若要為排程快照設定保留設定：

  1. 選擇共用資料夾。
  2. 按一下設定排程。
  3. 選擇保留頁籤。
  4. 設定您的保留設定。選擇下列其中一個選項：
    • 到達上限時移除舊版快照：達到快照數量上限時，最舊的快照將被新拍攝的快照覆寫。
    • 永久保留快照：達到快照數量上限時，快照 & 複寫將停止拍攝新的快照，但不會移除已拍攝的快照。
    • 根據以下規則保留快照：您可以為指定的時間範圍設定保留規則。詳細資訊請參閱保留規則備註。
  5. 按一下確定來儲存設定。

注意：

  • 變更保留設定將改變複寫頁面中相同的設定。

iSCSI LUN

若要拍攝 iSCSI LUN 的快照：

  1. 選擇您要拍攝快照的 iSCSI LUN。
  2. 按一下拍攝快照。
  3. 在描述欄位中輸入快照描述。
  4. 從鎖定下拉式選單中，選擇是 / 否來決定該快照是否將在自動刪除觸發時忽略保留設定。
  5. 按一下確定以開始拍攝快照。

若要移除 iSCSI LUN 的快照：

  1. 選擇您要移除快照的 iSCSI LUN。
  2. 按一下快照清單。
  3. 選擇您要移除的快照，並按一下移除。(多選時請按住 Shift 或 Ctrl 鍵。)
  4. 您將會看到確認視窗。按一下是來確認移除。

若要編輯 iSCSI LUN 的快照：

  1. 選擇您要編輯快照的 iSCSI LUN。
  2. 按一下快照清單。
  3. 選擇您要編輯的快照，並按一下編輯。
  4. 執行下列任何操作來編輯快照：
    • 在描述欄位中修改快照描述。
    • 應用程式一致性快照已啟動時，可選擇一致性方式。
    • 在鎖定下拉式選單中切換是 / 否來決定該快照是否將在自動刪除觸發時忽略保留設定。
  5. 按一下確定來儲存設定。

若要設定排程快照：

排程快照可讓快照 & 複寫以事先設定的執行時間及重複間隔來自動拍攝快照，減輕手動快照的工作量。

  1. 選擇一個 iSCSI LUN。
  2. 按一下設定排程。
  3. 在排程頁籤中，勾選啟動快照排程核取方塊，並設定拍攝快照的執行時間和重複間隔。若要取消該共用資料夾的排程快照，請取消勾選核取方塊。
  4. 按一下確定來儲存設定。

若要為排程快照設定保留設定：

  1. 選擇一個 iSCSI LUN。
  2. 按一下設定排程。
  3. 選擇保留頁籤。
  4. 設定您的保留設定。選擇下列其中一個選項：
    • 到達上限時移除舊版快照：達到快照數量上限時，最舊的快照將被新拍攝的快照覆寫。
    • 永久保留快照：達到快照數量上限時，快照 & 複寫將停止拍攝新的快照，但不會移除已拍攝的快照。
    • 根據以下規則保留快照：您可以為指定的時間範圍設定保留規則。詳細資訊請參閱保留規則備註。
  5. 按一下確定來儲存設定。
  6. 您將會看到確認視窗。再按一下確定來確認設定。

若要設定 iSCSI LUN 快照的一致性方式：

  1. 選擇一個 iSCSI LUN。
  2. 按一下設定排程。
  3. 選擇應用程式頁籤。
  4. 勾選 / 取消勾選啟動應用程式一致性快照核取方塊來切換以下兩種一致性方式：
    • 應用程式一致：只有在您的 Windows 伺服器或 VMware vCenter 伺服器 (根據您用來連接 DSM 的操作環境) 上已安裝 Synology Snapshot Manager 時可拍攝此一致性方式。當 DSM 上觸發應用程式一致性快照時，將會通知應用程式將記憶體的資料排清至要拍攝快照的 LUN 中，以確保資料一致。

      注意：

      • 若您的操作環境尚未安裝 Synology Snapshot Manager，即使勾選啟動應用程式一致性快照的核取方塊，該快照類型仍將維持停用狀態。
    • 損毀一致：若您的用戶端尚未安裝 Synology Snapshot Manager，您將只能拍攝此類型的 LUN 快照。觸發損毀一致性快照時，將不會通知應用程式將記憶體的資料排清至 LUN，而無法確保資料一致。
  5. 按一下確定來儲存設定。

注意：

  • 變更一致性方式設定將改變複寫頁面中相同的設定。

保留規則

此選項提供了理想的解決方案來幫助您指定快照版本數量上限以節省儲存空間，但您可能需要快照能保留更長的時間。

Synology 使用祖父子 (Grandfather-Father-Son，GFS) 三代輪換保留規則。您可以分別為下列的時間範圍設定保留快照版本的數量上限：小時、天、週、月、年等。

快照 & 複寫將為每種時間範圍保留您所設定上限數量的快照版本。若在一段時間範圍中有超過一個快照版本，將只保留該段時間中最先拍攝的快照。例如，若您設定的規則是 10 個每週快照，快照 & 複寫將保留最近 10 週裡每週最先拍攝的快照。

此外，資料保護管理員預設會保留最新拍攝快照時間點起算一小時之內的所有快照，讓您方便尋找並從最近快照版本中還原資料。您也可以鎖定快照以避免被您的保留規則自動刪除。
