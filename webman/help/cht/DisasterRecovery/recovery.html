還原

系統管理原可於此頁面進行還原。若要允許使用者存取共用資料夾快照，請啟動每個共用資料夾的開啟快照瀏覽功能，使用者即可透過 File Station 或 Windows 檔案總管來存取快照。

共用資料夾

若要還原共用資料夾中的個別檔案 / 資料夾：

  1. 選擇您要還原的共用資料夾。
  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下瀏覽，並複製您要還原的檔案 / 資料夾至目的地。

若要還原共用資料夾：

  1. 選擇您要還原的共用資料夾。
  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下動作，並從下拉式選單中選擇依原始狀態還原。
  5. 您將會看到確認視窗。按一下是來確認動作。

若要複製快照到新的共用資料夾：

  1. 選擇您要複製的共用資料夾。
  2. 按一下還原。
  3. 選擇您要複製到新共用資料夾的快照版本。
  4. 按一下動作，並從下拉式選單中選擇複製為新名稱。
  5. 您將被重新導向至控制台 > 共用資料夾 > 新增的複製頁面。輸入複製的共用資料夾的名稱，並按一下確定來完成。

注意：

  • 共用資料夾「homes」無法還原。
  • 複製的新共用資料夾無法加密。
  • 若在檔案傳輸仍在進行時拍攝快照，將無法確保資料一致性。
  • 加密共用資料夾的限制：
    • 掛載的加密共用資料夾將無法以任一方式還原。
    • 加密共用資料夾無法複製為新名稱。
    • 卸載的加密共用資料夾可使用依原始狀態還原方式還原為先前拍攝快照的狀態。

若要切換轉移共用資料夾的複寫任務：

  1. 選擇您要切換轉移的複寫共用資料夾。
  2. 按一下切換轉移。
  3. 若共用資料夾有多個複寫任務，請選擇新的目標伺服器。
  4. 按一下切換轉移來開始動作。

若要故障轉移共用資料夾的複寫任務：

  1. 選擇您要故障轉移的複寫共用資料夾。
  2. 按一下故障轉移 > 故障轉移。
  3. 選擇要還原的快照版本。
  4. 按一下故障轉移來開始動作。

注意：

  • 您僅能在來源伺服器故障時才能進行故障轉移。

若要重新保護共用資料夾的複寫任務：

故障轉移至目標伺服器時，複寫任務將停止兩台伺服器間的資料同步。您將需要在原先的來源伺服器復原後，重新保護複寫任務。

  1. 選擇已故障轉移且要重新保護的共用資料夾。
  2. 按一下故障轉移 > 重新保護。
  3. 選擇新的來源伺服器。
  4. 若共用資料夾有多個複寫任務，請選擇新的目標伺服器。
  5. 按一下重新保護來開始動作。

注意：

  • 新的目標伺服器上已存在的複寫共用資料夾資料將被新的來源伺服器覆蓋。
  • 若夥伴伺服器上不存在故障轉移時選取的快照版本，重新保護將可能耗時較久。

若要測試轉移複寫的共用資料夾：

  1. 選擇您要測試轉移的複寫共用資料夾。
  2. 按一下測試轉移 > 測試轉移。
  3. 若共用資料夾有多個複寫任務，請選擇測試伺服器。
  4. 輸入共用資料夾的測試轉移名稱，並選擇要測試的快照版本。
  5. 按一下測試轉移來開始動作。

注意：

  • 執行複寫任務的交換轉移 / 故障轉移時將清除測試轉移。
  • 您無法為測試轉移的共用資料夾新增複寫任務。

若要清除複寫共用資料夾的測試轉移：

  1. 選擇您要清除測試轉移的複寫共用資料夾。
  2. 按一下測試轉移 > 清除測試轉移。
  3. 勾選您要清除的測試轉移。
    注意：若您不要保留測試轉移建立的共用資料夾，請勾選刪除測試轉移資料。
  4. 按一下清除測試轉移來開始動作。

  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下動作，並從下拉式選單中選擇依原始狀態還原。

若要複製快照到新的 iSCSI LUN：

  1. 選擇您要複製的 iSCSI LUN。
