Visão geral

Essa página exibe o status geral dos alvos dos seus instantâneos e dos instantâneos propriamente ditos.

Resumo

Esta seção exibe três status listados abaixo dos alvos dos instantâneos (incluindo as pastas compartilhadas e iSCSI LUNs). Os tamanhos totais dos alvos dos instantâneos nos respectivos status são apresentados nesta seção também.

  • Instantâneo programado: A realização do instantâneo dos alvos de instantâneos está programada.
  • Não programado: Os instantâneos foram tirados para os alvos dos instantâneos, mas a realização do instantâneo não está programada.
  • Não protegido: Nenhum instantâneo foi tirado para os alvos de instantâneos.

Instantâneos recentes

Esta seção apresenta uma linha do tempo dos instantâneos categorizados para facilitar a recuperação de seus dados.

Para restaurar/clonar suas pastas compartilhadas ou iSCSI LUNs de instantâneos recentes:

  1. Clique no ícone do calendário e selecione a data para a qual deseja restaurar os dados.
  2. Arraste a linha do tempo para voltar ou avançar e selecione um ponto específico no tempo em que os instantâneos foram tirados.
  3. Selecione uma pasta compartilhada ou um iSCSI LUN que deseje restaurar/clonar.
  4. Você será direcionado para a página Recuperação. Para obter mais informações sobre as etapas seguintes, consulte Recuperação.
