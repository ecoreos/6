Recuperação

Essa página permite que os administradores do sistema realizem a recuperação. Para permitir que os usuários acessem os instantâneos da pasta compartilhada, habilite Tornar o instantâneo visível para cada pasta compartilhada para que os usuários possam acessar os instantâneos através do File Station ou Explorador de arquivos do Windows.

Pasta Compartilhada

Para restaurar arquivos/pastas individuais em uma pasta compartilhada:

  1. Selecione a pasta compartilhada que deseje restaurar.
  2. Clique em Recuperar.
  3. Selecione uma versão do instantâneo do qual deseje restaurar.
  4. Clique em Procurar e copie os arquivos/pastas que deseja restaurar para o destino.

Para restaurar uma pasta compartilhada:

  1. Selecione a pasta compartilhada que deseje restaurar.
  2. Clique em Recuperar.
  3. Selecione uma versão do instantâneo do qual deseje restaurar.
  4. Clique em Ação e selecione Restaurar no local da lista suspensa.
  5. Você verá uma mensagem de confirmação. Clique em Sim para confirmar a ação.

Para clonar um instantâneo para uma nova pasta compartilhada:

  1. Selecione a pasta compartilhada que deseje clonar.
  2. Clique em Recuperar.
  3. Selecione uma versão do instantâneo que deseja clonar para uma nova pasta compartilhada.
  4. Clique em Ação e selecione Clonar com um novo nome da lista suspensa.
  5. Você será redirecionado para a página Clonar em Painel de controle > Pasta compartilhada > Criar. Digite o nome da pasta compartilhada clonada e clique em OK para concluir.

Nota:

  • A pasta compartilhada "homes" não pode ser restaurada.
  • As pastas compartilhadas clonadas não podem ser criptografadas.
  • Se um instantâneo for tirado enquanto a transferência de qualquer arquivo ainda estiver em andamento, a consistência dos dados não será garantida.
  • Limitações sobre pastas compartilhadas criptografadas:
    • Nenhum dos tipos de recuperação está disponível para as pastas compartilhadas criptografadas montadas.
    • Clonar com um novo nome não está disponível para as pastas compartilhadas criptografadas.
    • As pastas compartilhadas criptografadas desmontadas podem ser restauradas de instantâneos tirados anteriormente usando-se Restaurar no local.

Para mudar a replicação de uma pasta compartilhada:

  1. Selecione a pasta compartilhada replicada que deseje mudar.
  2. Clique em Mudança.
  3. Selecione o novo servidor de destino se houver várias tarefas de replicação da pasta compartilhada.
  4. Clique em Mudança para iniciar a ação.

Para fazer failover em um replicação de pasta compartilhada:

  1. Selecione a pasta compartilhada replicada que deseje fazer failover.
  2. Clique em Failover > Failover.
  3. Selecione a versão do instantâneo para restaurar.
  4. Clique em Failover para iniciar a ação.

Nota:

  • Você pode iniciar um Failover somente após falha do servidor de origem.

Para proteger novamente a replicação de uma pasta compartilhada:

Quando houver failover no servidor de destino, a tarefa de replicação interromperá a sincronização de dados entre os dois servidores. Será necessário proteger novamente a tarefa de replicação após a recuperação do servidor de origem original.

  1. Selecione a pasta compartilhada com failover que deseje proteger novamente.
  2. Clique em Failover > Proteger novamente.
  3. Selecione o novo servidor de origem.
  4. Selecione o novo servidor de destino se houver várias tarefas de replicação da pasta compartilhada.
  5. Clique em Proteger novamente para iniciar a ação.

Nota:

  • Os dados da pasta compartilhada replicada existentes no novo servidor de destino serão sobrescritos pelos do novo servidor de origem.
  • Se a versão do instantâneo selecionado para o failover não existir no servidor parceiro, pode levar mais tempo para proteger novamente.

Para realizar um teste de failover para uma pasta compartilhada replicada:

  1. Selecione a pasta compartilhada replicada que deseje testar.
  2. Clique em Failover de teste > Failover de teste.
  3. Selecione o Servidor de teste se houver várias tarefas de replicação da pasta compartilhada.
  4. Digite o Nome do failover de teste da pasta compartilhada e selecione uma versão do instantâneo para testar.
  5. Clique em Failover de teste para iniciar a ação.

Nota:

  • A realização de uma mudança/failover de uma replicação limpará seu teste de failover.
  • Não é possível criar uma tarefa de replicação para uma pasta compartilhada de failover de teste.

Para limpar um failover de teste de uma pasta compartilhada replicada:

  1. Selecione a pasta compartilhada replicada da qual você deseja limpar o failover de teste.
  2. Clique em Failover de teste > Limpeza do failover de teste.
  3. Marque o failover de teste que deseja limpar.
    Nota: Marque Excluir dados do failover de teste se não quiser mais manter a pasta compartilhada criada para o failover de teste.
  4. Clique em Limpeza do failover de teste para iniciar a ação.

iSCSI LUN

Para restaurar um iSCSI LUN:

  1. Selecione o iSCSI LUN que deseja restaurar.
  2. Clique em Recuperar.
  3. Selecione uma versão do instantâneo do qual deseje restaurar.
  4. Clique em Ação e selecione Restaurar no local da lista suspensa.
  5. Você verá uma mensagem de confirmação. Clique em Sim para confirmar a ação.

Para clonar um instantâneo para um novo iSCSI LUN:

  1. Selecione o iSCSI LUN que deseja clonar.
  2. Clique em Recuperar.
  3. Selecione uma versão do instantâneo que deseja clonar para um novo iSCSI LUN.
  4. Clique em Ação e selecione Clonar com um novo nome da lista suspensa.
  5. Você será redirecionado para a página Clonar em Gerenciador de armazenamento > iSCSI LUN. Digite o nome do iSCSI LUN clonado e clique em OK para concluir.

Nota:

  • Se um instantâneo for tirado enquanto a transferência de qualquer arquivo ainda estiver em andamento, a consistência dos dados não será garantida.
