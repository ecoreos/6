Replicação

Essa página permite que os administradores do sistema gerenciem as replicações. Você pode criar, editar e excluir tarefas de replicação. Você também pode executar/interromper a sincronização e visualizar as informações de suas tarefas de replicação.

Pasta Compartilhada

Para criar uma tarefa de replicação para uma pasta compartilhada:

  1. Va para a guia Pasta compartilhada:
  2. Clique em Criar.
  3. Você verá o assistente Criar tarefa de replicação. Siga as etapas do assistente para criar uma replicação de pasta compartilhada.

Nota:

  • A pasta compartilhada "homes" não pode ser replicada.
  • A pasta compartilhada criptografada não pode ser replicada.
  • A pasta compartilhada deve ser localizada em um volume Btrfs.
  • Você pode criar replicações estendidas/um para muitos para uma pasta compartilhada para diferentes servidores de destino. Entretanto, você pode criar somente uma replicação para um servidor de destino para cada pasta compartilhada.

Para excluir uma replicação de pasta compartilhada:

  1. Selecione a tarefa de replicação da pasta compartilhada que deseja excluir.
  2. Clique em Excluir.
  3. Você verá uma mensagem de confirmação. Clique em Excluir para confirmar a exclusão.

Para editar uma replicação de pasta compartilhada:

  1. Selecione a tarefa de replicação da pasta compartilhada que deseja editar.
  2. Clique em Editar.
  3. Você verá uma janela contendo as seguintes guias:
    • Servidor parceiro
    • Programação
    • Retenção
  4. Na guia Servidor parceiro, você pode editar as informações do servidor parceiro, incluindo nome do host, porta, nome de usuário e senha.
  5. Na guia Programação marque a caixa de seleção Habilitar programação de instantâneo e ajuste o tempo e a frequência para tirar instantâneos. Você pode desmarcar a caixa de seleção para cancelar os instantâneos programados para a pasta compartilhada.
  6. Na guia Retenção, selecione qualquer uma das seguintes opções:
    • Manter 256 últimos instantâneos: Quando o número máximo de instantâneos (256) for atingido, os instantâneos mais antigos serão sobrescritos pelos mais novos.
    • Sempre manter instantâneos: Quando o número máximo de instantâneos for atingido, a função Instantâneo e replicação parará de tirar novos instantâneos sem remover os já tirados.
    • Manter os instantâneos com a política a seguir: Você pode configurar as regras de retenção para as faixas de tempo especificadas.
  7. Clique em OK para aplicar as novas configurações.

Nota:

  • Você pode editar apenas a programação da tarefa de replicação no servidor de origem.
  • A alteração das configurações de retenção afetará as configurações equivalentes na página Instantâneo.

Para visualizar as informações da replicação da pasta compartilhada:

  1. Selecione a tarefa de replicação da pasta compartilhada que deseja visualizar.
  2. Clique em Informações.
  3. Você verá uma janela contendo as seguintes informações:
    • Status dessa replicação.
    • Estatísticas dessa replicação.

Para executar a sincronização de uma replicação de pasta compartilhada:

  1. Selecione a replicação da pasta compartilhada que deseja sincronizar.
  2. Clique em Executar.
  3. Digite a descrição do instantâneo a ser tirado.
  4. Clique em OK para confirmar a ação.

Para interromper a sincronização de uma replicação de pasta compartilhada:

  1. Selecione a sincronização da replicação da pasta compartilhada que deseja interromper.
  2. Clique em Interromper.
  3. Você verá uma mensagem de confirmação. Clique em Interromper para confirmar a ação.

iSCSI LUN

Para criar uma tarefa de replicação para um iSCSI LUN:

  1. Vá para a guia iSCSI LUN.
  2. Clique em Criar.
  3. Você verá o assistente Criar tarefa de replicação. Siga as etapas do assistente para criar uma replicação de iSCSI LUN.

Nota:

  • O iSCSI LUN deve ter sido criado como LUN avançado.
  • Você pode criar replicações estendidas/um para muitos para um iSCSI LUN para diferentes servidores de destino. Entretanto, você pode criar somente uma replicação para um servidor de destino para cada iSCSI LUN.

Para excluir uma replicação de iSCSI LUN:

  1. Selecione a tarefa de replicação do iSCSI LUN que deseja excluir.
  2. Clique em Excluir.
  3. Você verá uma mensagem de confirmação. Clique em Excluir para confirmar a ação.

Para editar uma replicação do iSCSI LUN:

  1. Selecione a tarefa de replicação do iSCSI LUN que deseja editar.
  2. Clique em Editar.
  3. Você verá uma janela contendo as seguintes guias:
    • Servidor parceiro
    • Programação
    • Retenção
  4. Na guia Servidor parceiro, você pode editar as informações do servidor parceiro, incluindo nome do host, porta, nome de usuário e senha.
  5. Na guia Programação marque a caixa de seleção Habilitar programação de instantâneo e ajuste o tempo e a frequência para tirar instantâneos. Você pode desmarcar a caixa de seleção para cancelar os instantâneos programados para o iSCSI LUN.
  6. Na guia Retenção, selecione qualquer uma das seguintes opções:
    • Manter 256 últimos instantâneos: Quando o número máximo de instantâneos (256) for atingido, os instantâneos mais antigos serão sobrescritos pelos mais novos.
    • Sempre manter instantâneos: Quando o número máximo de instantâneos for atingido, a função Instantâneo e replicação parará de tirar novos instantâneos sem remover os já tirados.
    • Manter os instantâneos com a política a seguir: Você pode configurar as regras de retenção para as faixas de tempo especificadas.
  7. Na guia Aplicativo, marque Habilitar instantâneos consistentes com o aplicativo para habilitar os instantâneos consistentes com o aplicativo. Para cancelar os instantâneos consistentes com o aplicativo, desmarque a caixa de seleção.
  8. Clique em OK para aplicar as novas configurações.

Nota:

  • Você pode editar apenas a programação na origem da replicação.
  • A alteração das configurações de retenção/aplicativo afetará as configurações equivalentes na página Instantâneo.

Para visualizar informações de uma replicação do iSCSI LUN:

  1. Selecione a tarefa de replicação do iSCSI LUN que deseja visualizar.
  2. Clique em Informações.
  3. Você verá uma janela contendo as seguintes informações:
    • Status dessa replicação.
    • Estatísticas dessa replicação.

Para executar a sincronização de uma replicação do iSCSI LUN:

  1. Selecione a tarefa de replicação do iSCSI LUN que deseja sincronizar.
  2. Clique em Executar.
  3. Digite a descrição do instantâneo a ser tirado.
  4. Clique em OK para confirmar a ação.

Para interromper a sincronização de uma replicação do iSCSI LUN:

  1. Selecione a sincronização da tarefa de replicação do iSCSI LUN que deseja interromper.
  2. Clique em Interromper.
  3. Você verá uma mensagem de confirmação. Clique em Interromper para confirmar a ação.

Replicação estendida/Um para muitos

Você pode criar várias replicações para um pasta compartilhada/iSCSI LUN para vários servidores de destino formando um esquema um para muitos. Você também pode criar uma replicação estendida para uma pasta compartilhada replicada/iSCSI LUN formando um esquema estendido.

Nota:

  • Quer seja em um esquema um para muitos ou estendido, qualquer tarefa de replicação pode ter apenas três servidores de destino, no máximo.

Política de retenção

Essa opção fornece uma solução ideal para ajudá-lo a especificar a quantidade máxima de versões de instantâneos para economizar espaço de volume, mas pode ser necessário reter seus instantâneos por períodos mais prolongados.

A Synology emprega a GFS ou Política de retenção Avô-Pai-Filho. É possível configurar a quantidade máxima de versões de instantâneos a serem retidos para as seguintes faixas de tempo respectivamente: a cada hora, diariamente, semanalmente, mensalmente e anualmente.

O Gerenciador de proteção de dados manterá o número máximo configurado de instantâneos para cada faixa de tempo. Se houver mais de uma versão de um instantâneo dentro de uma faixa de tempo, apenas o mais recente será mantido. Por exemplo, se você definir uma política de 10 instantâneos semanalmente, o Gerenciador de proteção de dados manterá o instantâneo mais recente (se for tirado mais de um instantâneo em uma semana) para cada uma das últimas 10 semanas.

Além disso, todas as versões de instantâneos dentro de uma hora desde a realização do instantâneo mais recente serão mantidas por padrão para sua conveniência, para que você encontre e restaure as versões de instantâneos recentes. Você também pode bloquear um instantâneo para impedir que ele seja removido automaticamente pelas suas regras de retenção.
