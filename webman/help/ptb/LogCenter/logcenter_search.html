Registros

Na página Central de registros > Registros, é possível visualizar, pesquisar e exportar registros.

Para pesquisar logs:

  1. No menu suspenso de pesquisa, você pode especificar os seguintes critérios, dependendo da situação:
    • Palavra-chave: Digite uma palavra-chave para pesquisar as mensagens do log que contenham a palavra-chave correspondente.

      Nota:

      É possível usar expressões regulares como [*] (isto é, um número de caracteres aleatórios) ou [?] (isto é, 1 caractere aleatório) no campo da palavra-chave durante a pesquisa. Por exemplo, [a*n] pode representar [action], [attention], ou [an].

    • Intervalo de datas: pesquisar logs pelo intervalo de datas especificado.
    • Nível: pesquisar logs pelo nível especificado.
    • Nome do host: pesquisar logs pelo nome do host especificado.
    • Programa: pesquisar logs pelo nome do programa especificado.
    • Categoria: pesquisar logs pela categoria do programa especificado.
  2. Clique em Pesquisar para pesquisar logs correspondentes aos critérios especificados.

Para exportar logs:

  1. Clique na seta próxima ao botão Exportar.
  2. Escolha HTML ou CSV para baixar os logs utilizando o formato desejado.

Nota:

  • A codificação de caractere dos arquivos de log CSV esportados é UTF-8. Você precisará abrir o arquivo com um programa que suporte a codificação de caracteres UTF-8.
