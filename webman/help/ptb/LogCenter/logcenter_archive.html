Configurações de arquivamento

A página Configurações de arquivamento traz opções para configurar como e onde os registros recebidos pela Central de registros são arquivados. Use as opções abaixo para especificar o local de armazenamento e regras para criar automaticamente arquivos de registro.

Destino de armazenamento para arquivos

O destino refere-se à pasta em que os registros recebidos pelo Log Center são armazenados e arquivados. Clique em Selecionar local e escolha uma pasta.

Se desejar que os registros locais do DiskStation sejam arquivados no mesmo destino, clique na caixa marcada Arquive os registros locais para o local de armazenamento especificado acima. Em muitos casos, isso proporciona mais espaço de armazenamento para registros locais, permitindo que você mantenha registros do sistema mais completos.

Nota:

Registros locais (registros do sistema, de conexão e transferência de arquivo) são salvos em uma partição de sistema que tem capacidade de armazenamento limitada. Ao habilitar Arquivar registros locais no local de armazenamento especificado acima, todos os registros serão arquivados no destino especificado, de acordo com as Regras de arquivamento. Para procurar os registros locais arquivados, clique em Abrir arquivo de registro e selecione SYNOSYSLOGDB__LOCALARCH.DB.

Regras de arquivamento

Os registros são automaticamente arquivados de acordo com os critérios definidos na seção Regras de arquivamento. Os registros podem ser arquivados automaticamente quando o tamanho do banco de dados de registros atinge o tamanho especificado, o número de registros individuais atinge um número especificado ou os registros atingem o tempo especificado.

Formato de arquivo

Esta seção inclui as seguintes opções:

  • Arquivar os registros como formato de texto, além de formato padrão: Salve os arquivos como texto simples além dos arquivos SQLite padrão.
  • Compactar arquivos de registro: Salve arquivos de log como arquivos de formato ZIP compactado.
  • Arquivar os registros separadamente de acordo com o dispositivo: Salve arquivos separados para dispositivos cliente individuais que enviam logs para o DiskStation.
