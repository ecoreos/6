Desligamento e substituição do servidor

As instruções abaixo o guiarão através do procedimento de desligamento, remoção ou troca de um servidor em um grupo de high-availability.

Para desligar o grupo de high-availability usando o High Availability Manager:

Quando você quiser desligar um grupo de high-availability para transportar os servidores para um novo local físico ou para realizar manutenção, siga essas instruções.

  1. Faça login no DSM.
  2. Vá para High Availability Manager.
  3. Clique no botão marcado Gerenciar.
  4. Selecione Desligamento do grupo de high-availability.
  5. Uma mensagem de confirmação aparecerá.Clique em OK para confirmar.

Para desligar manualmente um conjunto de high-availability:

O conjunto de high-availability também pode ser desligado manualmente usando-se os botões liga/desliga em cada servidor.

  1. Pressione e segure o botão liga/desliga no servidor passivo até que ele emita um bipe e aguarde para que o servidor desligue completamente.
  2. Pressione e segure o botão liga/desliga no servidor ativo até que ele emita um bipe e aguarde para que o servidor desligue completamente.

Observação:

  • Ligação dos servidores:
    Quando desejar ligar os servidores novamente, os servidores ativo e passivo devem ser ligados ao mesmo tempo (ou em cinco minutos um em relação ao outro).Se você ligar apenas um dos servidores, ele tentará ligar o outro servidor transmitindo uma mensagem Wake on LAN.
  • Desligamento anormal:
    No grupo de high-availability, a opção de reiniciar após falha de energia está habilitada.Se os servidores desligarem anormalmente devido a uma falha de energia, eles tentarão ligar automaticamente quando a energia for restaurada.

Para substituir um servidor em um grupo de high-availability:

Quando você desejar remover um servidor e configurar outro, siga essas instruções.

  1. Vá para a página Visão geral.
  2. Clique no botão Gerenciar.
  3. Selecione Desligar servidor ativo ou Desligar servidor passivo para desligar o servidor que deseja substituir.
  4. Verifique as conexões do novo servidor.Após o novo servidor ser ligado, ele será marcado como "desconhecido" no gerenciador de High Availability.
  5. Vá para a página Visão geral novamente.
  6. Clique no botão Gerenciar.
  7. Selecione Recuperar servidor desconhecido para que o "servidor desconhecido" seja adicionado ao grupo de high-availability e reconhecido como o servidor passivo.
