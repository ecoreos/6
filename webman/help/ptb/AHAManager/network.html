Rede

Essa página contém o diagrama e a tabela de status das interfaces de rede.

Diagrama da interface

Exibe o item selecionado da tabela e cores diferentes são usadas para apresentar as interfaces de rede de 1Gbps e 10Gbps.

Tabela de status da interface

Exibe o status da interface nos dois servidores e se a interface é monitorada. Essa tabela mostra as seguintes informações:

  • Interface de rede: Exibe as interfaces de rede físicas nos servidores.
  • Servidor ativo: Exibe o status da interface e a velocidade de conexão no servidor ativo.
  • Servidor passivo/Servidor desconhecido: Exibe o status da interface e a velocidade de conexão no servidor passivo. Essa coluna mostrará o nome correspondente ao status do servidor.
  • Bond: Exibe as configurações de agregação de links.
  • Status: Exibe o status da rede se a interface estiver monitorada. Pode mostrar os seguintes status:
    • Aviso: Falha de conexão no servidor passivo ou a interface tem velocidades de conexão diferentes nos dois servidores;
    • Erro: Falha de conexão no servidor ativo;
    • Normal: Nenhuma falha.
  • Monitorar: Mostra se a interface é monitorada. Vá para Monitorar > Interface de rede para marcar/desmarcar a caixa de seleção Monitorar.
