Restaurar

Vá para Backup e restauração e clique em Restaurar no canto inferior esquerdo para restaurar os dados do backup.

Para restaurar o backup de dados/LUN*:

  1. Clique em Restaurar no canto inferior esquerdo, selecione Dados ou LUN* e selecione uma tarefa de backup para restaurar.
  2. O sistema pedirá que você selecione suas configurações, as diversas versões de dados de backup ou outras. Depende de que tipo de tarefa de backup você deseja restaurar.
  3. Se a tarefa de backup for criptografada, você precisará de uma senha/chave de criptografia para a restauração bem-sucedida.
  4. Siga o assistente para concluir a restauração.

Nota:

  • Ao restaurar um pacote não instalado, o sistema instalará automaticamente a mesma versão ou a versão mais recente do aplicativo. Se o aplicativo estiver instalado, mas for de uma versão mais antiga que a versão de backup, o sistema irá atualizá-lo automaticamente para a versão de backup original.
  • Os aplicativos de backup do DSM 5.1 ou superior não podem ser restaurados no DSM 5.0 ou inferior.

Nota:

  • Após a conclusão da restauração, os LUNs de nível de bloqueio e de nível de arquivo serão restaurados nos LUNs de nível de arquivo.*
  • Durante a restauração de um pacote, se o pacote não estiver instalado, o sistema instalará automaticamente a versão mais recente ou a mesma versão do aplicativo. Se o aplicativo estiver instalado, mas for de uma versão mais antiga que a versão de backup, o sistema irá atualizá-lo automaticamente para a versão de backup original.
  • Os aplicativos de backup do DSM 5.1 ou superior não podem ser restaurados no DSM 5.0 ou inferior.

Para restaurar um único arquivo ou pasta para tarefa de backup multiversão:

  1. Selecione a tarefa de backup desejada no painel esquerdo e abra o Explorador de backup (clicando no ícone da lupa). É necessária uma senha/chave de criptografia se a tarefa de backup estiver criptografada.
  2. Localize a versão do backup desejado com a barra da linha do tempo ou através do ícone calendário.
  3. Selecione um arquivo ou pasta contido e clique em Restaurar, Copiar para... ou Baixar para recuperar o arquivo/pasta selecionado. As três opções também estarão disponíveis se você clicar com o botão direito no arquivo/pasta.

Para pesquisar uma versão específica:

Para aumentar a velocidade de pesquisa do arquivo, você pode usar palavras-chave para filtrar o arquivo/pasta desejada. Os arquivos/pastas não correspondentes serão ocultados.

  1. Abra o Explorador de backup e clique no ícone do funil no canto superior direito.
  2. No menu suspenso, você pode filtrar os arquivos/pastas pelos critérios abaixo:
    • Palavra-chave: Digite o nome de qualquer arquivo ou pasta. Você pode usar curingas [*] (representando um ou mais caracteres) e [?] (representando um caractere aleatório) em suas palavras-chave. Por exemplo, [a*n] pode representar [action], [attention], ou [an].
    • Tipo de Arquivo: Selecione de Qualquer, Qualquer pasta ou Qualquer arquivo para refinar sua pesquisa.
    • Tamanho (KB): Selecione Qualquer ou selecione igual a, maior que, menor que e digite um valor.
    • Data de modificação: Selecione entre Hoje, Ontem, Semana passada ou Mês passado, ou Personalizar para digitar uma data de início e final a fim de localizar versões dentro de um intervalo específico.
  3. Para aplicar seus filtros, clique em Filtro.
  4. Para aplicar os mesmos filtros em outra versão de backup, basta alterar a versão desejada com a barra da linha do tempo.
  5. Para restabelecer sua pesquisa filtrada, clique no botão Redefinir.

Nota:

  • O explorador de backup está disponível apenas para tarefas de backup multiversão.
  • O explorador de backup tem funções limitadas quando o destino do backup é definido em DSM 5.1 ou inferior.
  • A configuração do sistema e dos aplicativos pode ser restaurada apenas através de Restaurar.
  • A função filtro está disponível apenas para tarefas de backup não criptografadas.
  • As pastas/arquivos recuperados usando Copiar para... ou Baixar perderão suas configurações de privilégio originais.

Para restaurar tarefas de backup de outro destino de backup:

  1. Clique em Restaurar no canto inferior esquerdo e selecione Dados ou LUN*.
  2. Você pode selecionar e restaurar qualquer um dos tipos de tarefa de backup:
    • Para restaurar o backup de dados: Clique em Dados > Restaurar de repositórios existentes e selecione o destino de backup desejado para restaurar os dados.
    • Para restaurar o backup do LUN: Clique em LUN*.
  3. Siga as instruções do assistente. É necessária uma senha/chave de criptografia se a tarefa a restaurar estiver criptografada.
  4. Clique em Aplicar para concluir a restauração.

Nota:

  • O Embedded DataStation só pode restaurar pastas compartilhadas para dispositivos externos ext3/ext4.
  • O Embedded DataStation não suporta a restauração de aplicativos sem um dispositivo externo ext3/ext4.

____
* Disponível apenas em modelos específicos.
