Backup do LUN

Na guia Backup de Backup e restauração, você pode usar Backup do LUN local para fazer backup dos iSCSI LUNs de seu DiskStation para pastas compartilhadas locais. Você também pode usar Backup do LUN de rede para fazer o backup para outro DiskStation.

Para fazer backup do iSCSI LUN:

  1. Selecione Criar > Tarefa de backup do LUN para abrir o Assistente de backup do LUN e clique em Avançar.
  2. Quando solicitado, selecione um tipo de destino, selecione uma das seguintes opções:
    • Backup do LUN local: selecione essa opção para fazer backup do LUN do DiskStation para uma pasta compartilhada local ou para um armazenamento externo conectado à porta USB ou eSATA de seu DiskStation.
    • Backup do LUN da rede: selecione essa opção para fazer backup do LUN de seu DiskStation para outro DiskStation.
  3. Quando solicitado, especifique o destino do backup e o LUN de origem e defina a programação de backup.
  4. Siga o assistente para completar a configuração.

Depois da criação da tarefa de backup, o DiskStation executará a tarefa de backup de acordo com a programação e armazenará o backup do LUN no destino. O backup do LUN é usado apenas para fins de backup. Para acessar o backup do LUN, consulte Para restaurar o backup do LUN para obter mais informações.

Observação:

  • Os LUNs normais de arquivo e de nível de bloco serão incluídos no backup como imagens de arquivo no destino. Se restaurados, os LUNs de Nível de Bloco serão alterados para LUNs de Arquivo.
  • Cada tarefa de backup pode incluir apenas um LUN como origem de backup.
  • O backup dos LUNs não pode ser feito em um disco externo com o formato FAT, FAT32, HFS, HFS Plus ou NTFS.
  • Antes de executar o Backup do LUN da rede, entre no DiskStation de destino para ter certeza de que o seu serviço de backup da rede está ativado em Backup e replicação > Serviços de backup.
  • Se o DiskStation se conecta a outra rede através de um roteador, certifique-se de ter especificado uma regra de encaminhamento de porta para o serviço iSCSI (porta TCP 3260) no roteador antes de realizar o backup do LUN da rede.

Sobre o backup completo e o backup incremental:

O DiskStation executará um backup completo do LUN selecionado pela primeira vez e posteriormente realizará backups incrementais. Entretanto, se o nome da pasta compartilhada ou o nome do diretório do destino do backup for alterado, o DiskStation realizará um backup completo da tarefa novamente.
