Usuários ou grupos de domínio/LDAP

Se o seu DiskStation tiver ingressado em um serviço de diretório como um domínio Windows ou cliente LDAP, você pode configurar e modificar a permissão de acesso de usuários ou grupos de domínio/LDAP nas pastas compartilhadas DiskStation e aplicativos DSM e habilitar o serviço de base para todos os usuários do domínio/LDAP.

Usuários de domínio/LDAP

Dependendo do tipo do serviço de diretório em que você está ingressando, clique na guia Usuários de domínio ou Usuários de LDAP para configurar as permissões da pasta compartilhada e do aplicativo e a cota da capacidade do disco de cada usuário.

Dependendo do tipo do serviço de diretório em que você está ingressando, clique na guia Usuários de domínio ou Usuários de LDAP para configurar as permissões da pasta compartilhada e do aplicativo de cada usuário.

As regras de configurações de privilégio incluem o seguinte:

  • Em caso de conflitos de privilégio, a prioridade de privilégio será: NA > RW > RO.
  • Visualizar privilégio: Os privilégios do usuário são mostrados aqui, e isso é definido baseado nas configurações de privilégio do usuário e dos grupos dos quais faz parte.
  • Privilégios de grupo: Atribua privilégios aos grupos de acesso para acessar pastas compartilhadas.

Base do usuário

Da mesma forma que os usuários locais, todos os usuários do domínio/LDAP podem acessar sua própria pasta home através de CIFS, AFP, FTP ou File Station.

Usuários com privilégios de administrador podem acessar todas as pastas personalizadas localizadas na pasta compartilhada homes. Todas as pastas base dos usuários estão localizadas em uma das sub pastas da pasta chamada "@DH-nome do domínio" (para usuários do domínio) ou "@LH-nome do FQDN" (para usuários do LDAP). O nome da pasta base do usuário é a conta de usuário mais um número exclusivo.

    Nota:
  1. Para excluir a pasta compartilhada homes, o serviço de base do usuário deve ser desabilitado primeiro.
  2. A habilitação do serviço base para usuários do domínio/LDAP também habilitará o serviço base para usuários locais se ainda não estiver habilitado.

Grupo de domínio/LDAP

Dependendo do tipo de serviço de diretório em que você está ingressando, clique na guia Grupo de domínio ou Grupo de LDAP para configurar os privilégios de acesso dos grupos de domínio/LDAP às pastas compartilhadas. A prioridade de privilégios é: NA > RW > RO.
