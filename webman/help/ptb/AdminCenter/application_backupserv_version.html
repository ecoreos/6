Serviço de backup

Serviço de backup permite que outros usuários façam backup dos dados de seus dispositivos Synology NAS através do pacote. Backup e restauração em seu Synology NAS.

Habilitação do Servidor de backup Synology

Para habilitar o serviço de backup:

Você pode transformar seu Synology NAS em um destino de backup de dados/LUN para outro Synology NAS remoto.

  1. Vá até Painel de controle > Serviço de backup.
  2. Clique em Habilite o destino de backup para os servidores remotos Synology.
  3. Clique em Aplicar para salvar as configurações.
  4. Vá para Painel de Controle > Pasta Compartilhada, selecione uma pasta compartilhada, clique em Editar > Permissões e conceda o privilégio Leitura/Gravação aos usuários desejados. Repita essa etapa para incluir mais pastas compartilhadas que podem ser usadas como destinos de backup.

Nota:

  • O Synology NAS remoto deve usar o pacote Backup e restauração para fazer backup de seus dados/LUN em seu Synology NAS.
  • Para permitir o backup LUN do Synology NAS remoto, vá para Painel de controle > Serviços de arquivos > rsync e selecione Habilitar serviço rsync no lado do destino.

Para atribuir privilégios de backup:

Atribua os privilégios de backup adequados aos usuários para que esses usuários possam fazer backup dos dados de um Synology NAS de origem para o Synology NAS de destino.

  1. Vá para Painel de controle > ﻿Privilégios, clique em Backup Synology > Editar.
  2. Atribuir Permitir privilégios para os usuários de destino ou especificar melhor os endereços IP permitidos para os usuários específicos.
