Roteamento de Rede

O DSM oferece várias opções de roteamento de rede para vários requisitos de configuração de rede.

Para separar o fluxo de rede das interfaces de rede na mesma LAN:

  1. Vá para Painel de controle > Rede > Avançado.
  2. Em Geral, marque Responda às solicitações ARP se o endereço IP de destino for o endereço local configurado na interface de entrada.
  3. Clique em Aplicar para salvar suas configurações.

Isso estabelece a regra de roteamento padrão para cada interface de rede, o que fará com que os pacotes de rede de cada interface sejam enviados de sua própria interface de rede física.

Além disso, se você tiver mais de duas interfaces de rede na mesma LAN, habilitar essa opção fará com que o fluxo de rede em cada interface seja transportado separadamente. Você pode aplicar regras de firewall separadas para cada interface.

Para encaminhar cada interface de rede por seus respectivos gateways:

  1. Vá até Painel de controle > ﻿Rede.
  2. Em Geral, marque Ativar Gateway Múltiplo.
  3. Clique em Aplicar para salvar suas configurações.

Isso irá gerar um roteamento para cada interface de rede em sua rede passando por seus respectivos gateways.

Para alterar o gateway padrão:

  1. Vá até Painel de controle > ﻿Rede.
  2. Em Interface de Rede, clique em Ordem de Serviço.
  3. Reorganize seus gateways selecionando o gateway e clicando em Para cima ou Para baixo.
  4. Clique em OK.

O gateway no topo da ordem servirá como gateway padrão para a rede local.

Para configurar o roteamento estático:

  1. Vá até Painel de controle > ﻿Rede.
  2. Em Roteamento estático, clique em Criar.
  3. Na Rede de destino, digite o endereço IP de seu destino.
  4. Em Máscara de rede, digite a máscara de rede de seu destino.
  5. Em Gateway, digite o endereço do gateway de seu destino.
  6. Em Interface, selecione a interface de seu destino.
  7. Clique em OK.

Para visualizar a tabela de roteamento de IP:

  1. Vá até Painel de controle > ﻿Rede.
  2. Em Roteamento estático, clique em Tabela de roteamento de IP.
  3. Selecione uma interface de rede para visualizar sua tabela de roteamento de IP a partir da lista suspensa superior esquerda.
