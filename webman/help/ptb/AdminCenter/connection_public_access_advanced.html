Porta do roteador

Se você deseja que os serviços da Internet (como Google Docs ou Pixlr) acessem o seu DiskStation, pode especificar o nome de host externo público e o número da porta em Painel de Controle > Acesso externo > Avançado.

Por padrão, o nome de host é o seu endereço público de IP (ou o nome de host configurado no Painel de Controle >Acesso externo > DDNS) e o número da porta é 5000 (ou 5001 para as conexões HTTPS). No entanto, se você tiver outro nome de host que não está configurado na página do DDNS, ou se especificou um número de porta diferente em seu roteador ao configurar as regras de encaminhamento de porta, será necessário inserir o nome de host ou os números da porta do roteador para garantir que os arquivos que você compartilhou com os serviços da Internet possam ser acessados com sucesso.

Para habilitar a configuração da porta do roteador:

  1. Digite o nome de host ou o endereço IP público de seu DiskStation no campo Nome de host ou IP estático.
  2. Digite o número de porta do roteador que especificou quando configurou a regra de encaminhamento de porta do DSM no roteador nos campos DSM (HTTP) ou DSM (HTTPS).
  3. Clique em Aplicar.
