Modo Simples

O modo simples do Painel de controle integra várias configurações básicas, porém essenciais, em uma interface de usuário mais centralizada, direta e otimizada, permitindo ter uma visão rápida do status do sistema, configurar o QuickConnect ID, além de criar pastas compartilhadas e usuários. Essa é a melhor opção para usuários domésticos que precisam gerenciar uma quantidade pequena ou moderada de usuários do DSM, bem como para aqueles que desejam configurar ou modificar poucas configurações essenciais e começar a usar o DSM rapidamente sem se preocupar com configurações avançadas.
