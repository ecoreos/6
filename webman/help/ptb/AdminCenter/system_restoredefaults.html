Restaurar padrões

Você pode executar uma restauração de fábrica em Painel de Controle > Atualização e Restauração > Restaurar Padrões para restaurar o seu DiskStation para as configurações originais do fabricante. Todos os dados de usuários armazenados nas unidades serão apagados e todo o sistema será restaurado para as configurações padrão.

Nota:

  • A opção Restaurar Padrões será desativada quando o DiskStation pertencer a um cluster high-availability. Para restaurar as configurações padrão de um servidor pertencente a um cluster high-availability, você deve primeiro remover o cluster high-availability em High Availability > Visão Geral > Gerenciar.
