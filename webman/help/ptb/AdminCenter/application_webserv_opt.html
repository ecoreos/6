Serviço HTTP

Além da porta padrão de número 80, é possível adicionar outra porta para uso do Photo Station e Web Station.

Para adicionar um número de porta para acesso HTTP:

  1. Marque Adicionar outra porta HTTP.
  2. Digite o número da porta no campo. Você pode adicionar números de porta de 1 a 65535 para acesso HTTP.

Os seguintes símbolos não são suportados:

20, 21, 22, 23, 25, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, portas padrão eMule: 4662 (TCP), 4672(UDP) e intervalo de portas padrão para FTP: a faixa atual pode variar dependendo dos modelos diferentes.

    Observação:
  1. Os números de porta para acesso HTTP não podem ser os mesmos que o número de porta para o serviço FTP/BT.
  2. As portas HTTP e HTTPS não podem conflitar entre sí.
