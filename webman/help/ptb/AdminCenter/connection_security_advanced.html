Avançado

Você pode modificar as seguintes configurações avançadas em Painel de controle > Segurança > Avançado para as sessões de navegação do DSM:

Compactação HTTP

Você pode habilitar Compactação HTTP para o DiskStation. Reduzir a largura de banda do tráfego de rede pode aumentar a velocidade de carregamento das páginas.

Para habilitar/desabilitar a compactação HTTP:

  • Marque/desmarque Compactação HTTP em Painel de controle > Segurança > Avançado.

Perfil TLS/SSL

Você pode escolher o nível de segurança das conexões HTTPS criptografadas. Há três níveis:

  • Compatibilidade moderna - Compatível somente com os navegadores e dispositivos móveis mais recentes.
  • Compatibilidade intermediária - Um intermédio entre os níveis de segurança mais alto e mais baixo. É o valor padrão do sistema.
  • Compatibilidade com versões anteriores - Não é seguro, mas suporta a maioria dos navegadores e dispositivos móveis.
