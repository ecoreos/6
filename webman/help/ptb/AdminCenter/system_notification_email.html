E-mail

A ativação de notificações de e-mail permite que o DiskStation envie mensagens para o seu e-mail quando ocorrerem eventos específicos, como erros de sistema, perda de conexão de rede, comportamento da ventoinha, falha de energia ou falha da tarefa de backup.

A ativação de notificações de e-mail permite que o DiskStation envie mensagens para o seu e-mail quando ocorrerem eventos específicos, como erros de sistema, perda de conexão de rede, falha de energia ou falha da tarefa de backup.

Para ativar a notificação por e-mail:

  1. Vá até Painel de Controle > Notificações > E-mail.
  2. Marque Ativar notificações por e-mail.
  3. Digite um ou mais endereços de e-mail no campo Destinatários. As notificações do sistema serão enviadas para os endereços inseridos aqui. São permitidos dois endereços. É possível separar vários endereços por um ponto e vírgula (;).
  4. Se necessário, adicione um prefixo de assunto no campo Prefixo do assunto. Esse prefixo será adicionado ao assunto de cada mensagem enviada pelo DiskStation, ajudando a identificar e filtrar mensagens.
  5. Se desejar usar um servidor de e-mail de terceiros (como Gmail, Outlook, Yahoo!) para encaminhar mensagens, consulte a seção "serviço de e-mail de terceiros" abaixo. Se desejar usar um servidor personalizado, consulte a seção "servidor SMTP personalizado".

Para usar um serviço de e-mail de terceiros:

  1. No menu Provedor de serviço escolha um serviço de e-mail de terceiros, como o Gmail, Yahoo ou Outlook.
  2. Digite seu nome de usuário e sua senha para o serviço de e-mail selecionado.
  3. Clique em Aplicar para salvar as configurações.
  4. Você pode clicar em Enviar um e-mail de teste para ver se as configurações estão corretas.

Para usar um servidor SMTP personalizado:

  1. No menu Provedor de serviço, selecione Servidor SMTP personalizado.
  2. Digite o nome do servidor SMTP ou o endereço IP no campo Servidor SMTP.
  3. Digite o número da porta SMTP no campo Porta SMTP. (Se você tiver especificado uma porta SMTP diferente ao configurar as regras de envio de porta, digite esse número.)
  4. Se seu servidor SMTP exigir autenticação, marque Autenticação obrigatória, digite o Nome de usuário e Senha do servidor de e-mail.
  5. Se seu servidor SMTP exigir conexão SSL/TLS, marque A conexão segura (SSL/TLS) é necessária.
  6. Se quiser personalizar o nome do remetente que aparece nos e-mails enviados pelo DiskStation, digite o nome no campo Nome do remetente. Se quiser personalizar o endereço de e-mail do remetente, digite o endereço de e-mail no campo E-mail do remetente. Se o campo estiver em branco, o primeiro endereço de e-mail no campo Destinatário será usado.
  7. Clique em Aplicar para salvar as configurações.
  8. Você pode clicar em Enviar um e-mail de teste para ver se as configurações estão corretas.

Nota:

  • Se desejar enviar uma mensagem de boas-vindas para cada conta de usuário recém-criada, marque Enviar uma mensagem de boas-vindas para novos usuários e todos os usuários receberão uma mensagem de boas-vindas ao criar uma conta.
  • O idioma da notificação de e-mail pode ser definida em Idioma de notificação. Para alterar o idioma das notificações de e-mail, vá para Menu Principal > Painel de Controle > Opções Regionais, clique na guia Idioma e selecione o idioma no menu suspenso na seção Idioma de Notificações.
  • Se você estiver usando o servidor SMTP do Gmail e tiver ativado a verificação em duas etapas, lembre-se de usar a "senha específica para aplicativo" fornecida pelo Google para autenticação SMTP.

Nome do domínio do servidor SMTP:

O servidor SMTP pode ser um endereço IP ou um nome de domínio. Caso seja um nome de domínio, verifique se o IP válido do DNS server foi digitado no Menu principal > Painel de controle > Rede e verifique se o servidor está na Internet.
