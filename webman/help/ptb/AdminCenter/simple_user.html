Usuário

Com as opções flexíveis de gerenciamento de usuários, você pode criar contas de usuário para cada membro da sua família ou empresa e definir permissões de acesso a pastas compartilhadas para cada pessoa.

Para criar um usuário:

  1. Vá para Painel de controle (modo simples) > Usuário e clique no ícone + no painel esquerdo.
  2. Especifique um nome e uma senha para o usuário.
  3. Especifique os grupos aos quais o novo usuário deverá pertencer.
  4. Atribua as permissões adequadas de pasta compartilhada ao novo usuário. Quando houver conflitos de permissão, a prioridade de permissões será a seguinte: Nenhum acesso > Leitura/Gravação > Somente leitura.
    • Visualizar: Essa coluna exibe os privilégios reais de acesso do usuário a pastas compartilhadas de acordo com as configurações atuais e com os grupos aos quais o usuário pertence.
    • Leitura/Gravação: O usuário pode acessar e fazer alterações nos arquivos e subpastas da pasta compartilhada correspondente.
    • Somente leitura: O usuário pode acessar os arquivos e as subpastas na pasta compartilhada correspondente.
    • Nenhum acesso: O usuário não pode acessar os arquivos ou subpastas na pasta compartilhada correspondente.
  5. Clique em Aplicar para salvar as configurações.

    6. Limitação do nome de usuário:

    • O nome de usuário não diferencia maiúsculas e minúsculas e deve ter de 1 a 64 caracteres Unicode, mas não pode conter os seguintes símbolos:
      ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
    • O primeiro caractere não pode ser um sinal de menos nem um espaço e o último caractere não pode ser um espaço.
    • Os seguintes nomes de usuário são reservados para uso do sistema:
      mailer-daemon, postmaster

    Limitação da senha:

    • A senha diferencia maiúsculas de minúsculas e é limitada até 127 caracteres visíveis, inclusive letras, números, sinais e espaços.

Para editar um usuário:

  1. Vá para Painel de controle (modo simples) > Usuário.
  2. Selecione o usuário desejado no painel esquerdo.
  3. Edite as informações gerais do usuário, como nome, senha, grupo e permissões de pasta compartilhada.
  4. Marque Desativar esta conta se desejar que a conta pare de funcionar.
  5. Clique em Aplicar para salvar as configurações.

Para remover um usuário:

  1. Vá para Painel de controle (modo simples) > Usuário.
  2. Selecione o usuário desejado no painel esquerdo.
  3. Clique no ícone Lixo no painel esquerdo para excluir o usuário selecionado.
