Configurações de segurança

Altere o root dos usuários selecionados para a base do usuário

Os usuários selecionados aqui podem acessar suas pastas base apenas quando fizerem logon em Synology DiskStation através do FTP.

Ativar FTP Anônimo

Ative esta função para permitir que qualquer pessoa faça logon no serviço FTP de DiskStation (com a conta anônimo ou ftp).

Para configurar o privilégio do usuário anônimo:

Vá até Menu principal > Painel de controle > Pasta compartilhada e clique em Configuração de privilégios.

Alterar root Anônimo

Ative essa função para alterar o diretório de logon do usuário anônimo de FTP. Selecione a pasta compartilhada no menu suspenso.
