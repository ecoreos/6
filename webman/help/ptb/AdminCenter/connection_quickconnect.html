QuickConnect

O QuickConnect permite que os aplicativos clientes se conectem ao seu DiskStation através da Internet sem o inconveniente da configuração das regras do encaminhamento de portas. O QuickConnect também pode trabalhar com pacotes desenvolvidos pela Synology, como o Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Note Station, CMS, Cloud Station e aplicativos móveis.

Para ativar o QuickConnect:

  1. Vá para Painel de controle > QuickConnect > Geral.
  2. Marque Ativar o QuickConnect.
  3. Digite as informações de sua conta Synology existente ou inscreva-se em uma nova conta.
  4. Especifique um novo QuickConnect ID. Certifique-se de que ele é fácil de lembrar, assim você será capaz de acessar seu DiskStation a qualquer momento.
  5. Clique em Aplicar.

Nota:

Um QuickConnect ID personalizado pode incluir letras, números e hífenes ("-") e deve começar com uma letra.

Para otimizar a conectividade do QuickConnect:

Seu DiskStation oferece opções avançadas para tornar o acesso do QuickConnect mais adaptável a diferentes ambientes de rede.

  1. Vá para Painel de controle > QuickConnect > Avançado.
  2. Selecione as opções a seguir que atendam às suas necessidades:
    • Habilitar o serviço de retransmissão do QuickConnect: Sua conexão do QuickConnect será retransmitida através do servidor de retransmissão da Synology quando o acesso direito do QuickConnect ao DiskStation não for possível no ambiente de rede atual.
    • Criar regras de encaminhamento de porta automaticamente: Quando seu DiskStation estiver conectado a um roteador com UPnP, o roteador será informado para criar regras de encaminhamento de porta para o QuickConnect.
  3. Clique em Aplicar.

Nota:

  • A conexão retransmitida do QuickConnect pode ser lenta devido à maior latência de rede.
  • Quando o roteador tiver permissão de criar regras de encaminhamento de porta para o QuickConnect, o DiskStation poderá estar exposto a riscos de segurança.

Para habilitar/desabilitar o QuickConnect para aplicativos/serviços específicos:

  1. Vá para Painel de controle > QuickConnect > Avançado.
  2. Selecione os aplicativos/serviços para os quais você deseja habilitar o QuickConnect.
  3. Clique em Aplicar.

Nota:

Para obter um desempenho melhor do QuickConnect, recomendamos que você vá até Painel de controle > Acesso externo > Configurações do roteador para configurar o envio de porta para cada serviço:

  • DSM: 5000 (HTTP); 5001 (HTTPS)
  • Photo Station: 80 (HTTP); 443 (HTTPS)
  • Cloud Station：6690

Solução de Problemas

Se a mensagem Ocorreu um erro de rede for exibida, significa que o serviço do QuickConnect foi interrompido devido a erros de rede. Verifique o seguinte:

  • Seu DiskStation está conectado a uma rede ativa (em Painel de controle > Rede > Interface de rede).
  • As configurações do servidor DNS e o gateway padrão estão configuradas corretamente (em Painel de controle > Rede > Geral).
  • Certifique-se de que o DiskStation possa acessar a Internet usando as seguintes portas de destino: 80, 443, 8888.

Quando os erros da rede forem reparados, o serviço QuickConnect deve retomar automaticamente em alguns minutos.
