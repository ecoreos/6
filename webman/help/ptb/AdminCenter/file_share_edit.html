Editar informações gerais de pasta compartilhada

Você pode editar informações gerais de uma pasta compartilhada, como nome, descrição e outras.

Para editar informações gerais de uma pasta compartilhada:

  1. Vá até Painel de Controle > Pasta Compartilhada.
  2. Selecione uma pasta compartilhada.
  3. Clique em Editar.
