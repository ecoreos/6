Atualização do DSM

Periodicamente a Synology libera atualizações gratuitas do DSM. As atualizações podem incluir novas características, melhoria de funcionalidades e de desempenho.

Importante: Normalmente, a atualização do DSM não afeta os dados no DiskStation. Entretanto, para garantir a segurança dos dados, é fortemente recomendado efetuar um backup de seus dados antes de iniciar.

Executar a atualização manual do DSM

Você pode fazer download do arquivo de atualização .pat no Centro de Downloads da Synology e executar a atualização manual do DSM.

Para executar a atualização manual do DSM:

  1. Visite www.synology.com para baixar a atualização mais recente do DSM para seu computador local. Descompacte o arquivo para obter um arquivo de atualização .pat.
  2. Clique em Atualização manual do DSM para entrar na página de Atualização manual do DSM
  3. Clique em Procurar para selecionar o arquivo de atualização.
  4. Clique em Aplicar para começar a atualizar seu DSM. O sistema será reiniciado automaticamente quando a atualização for completada.

Solução de problemas:

Se não puder atualizar o DSM, as sugestões a seguir podem ajudar a resolver seu problema:

  • Verifique se o processo de atualização for interrompido pelo usuário ou outros eventos inesperados, como uma pane na rede elétrica.
  • Verifique a versão do DSM. Você pode atualizar o sistema apenas com uma versão mais nova do DSM.
  • Baixe a atualização do DSM novamente porque o arquivo de atualização pode estar corrompido.

Se o problema persistir, contate o Suporte Técnico da Synology.

Executar a atualização automática do DSM

Diferente da atualização manual, todas as vezes que você for à Atualização do DSM, o sistema verificará com a Synology se há um novo DSM disponível para download e exibirá o resultado em texto verde ou vermelho. Se o resultado disser que o DSM mais recente está pronto para download, o sistema fará o download do DSM para você e executará a atualização.

Para que o sistema faça a atualização do DSM mais recente e execute a atualização:

  1. Se o DSM mais recente estiver pronto para download, clique no link Baixar para que o sistema baixe o DSM mais recente para você.
  2. O sistema mostrará uma das notificações de área de trabalho a seguir para lhe avisar se o download foi ou não bem sucedido: Download do DSM concluído ou Erro no Download do DSM.
  3. Clique na janela da notificação de área de trabalho para ir à Atualização do DSM.
  4. Execute um dos seguintes procedimentos:
    • Se o download do DSM foi concluído, clique em Atualizar agora para iniciar imediatamente a atualização do DSM.
    • Se houve erro no download do DSM, clique em Tentar novamente para que o sistema baixe novamente o DSM mais recente da Synology.

Para verificar regularmente as atualizações:

  1. Marque Verificar atualizações do DSM automaticamente.
  2. Escolha um plano de atualização e configure a Programação de instalação.
  3. Clique em OK.

Uma notificação será exibida na área de trabalho quando uma atualização do DSM for baixada e estiver pronta para ser instalada. Para interromper a atualização automática do DSM, você pode cancelá-la na notificação da área de trabalho ou alterando as Configurações de atualização. As atualizações do DSM serão instaladas antes do desligamento programado, se você tiver habilitado o Desligamento no Agendamento de energia.

Solução de problemas:

Se você vir mensagens de erro (em texto vermelho) durante a verificação ou download do DSM mais recente, as sugestões a seguir podem ajudar a resolver seu problema:

  • Certifique-se de que o DiskStation está conectado à Internet.
  • O servidor de download da Synology pode estar ocupado no momento. Tente fazer o download de novo mais tarde; isso pode resolver o problema.

Se o problema persistir, contate o Suporte Técnico da Synology.
