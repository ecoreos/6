Configuração do roteador

Se seu dispositivo DiskStation estiver em uma rede de área local, outros dispositivos na Internet não poderão conectar a ele.É possível regras de encaminhamento de portas em Painel de Controle > Acesso Externo > Configuração de Roteador para deixar seu dispositivo DiskStation acessível na Internet.

Configurar roteador

Antes de adicionar regras de encaminhamento de portas, é necessário configurar uma conexão entre o dispositivo DiskStation e o roteador.

Nota:

  1. Para configurar as regras de encaminhamento de porta e atribuir endereços IP estáticos, você precisa ter privilégios administrativos para o roteador.
  2. Antes de configurar a conexão do dispositivo DiskStation com um roteador UPnP, certifique-se de que o DSM tenha recuperado o endereço IP externo com êxito do DiskStation (em Painel de Controle > Acesso Externo > DDNS).Caso contrário, verifique as configurações da rede e do firewall de seu dispositivo DiskStation e do roteador UPnP.

Para configurar a conexão do dispositivo DiskStation com seu roteador:

  1. Clique em Configurar roteador para abrir o assistente de configuração.O DSM começará testando os seguintes componentes/configurações de rede antes de avançar para a próxima etapa.
    • Interface de rede
    • Conexão do gateway
    • Conexão com a Internet
    • Encaminhamento de porta
    • DNS
  2. Dependendo do tipo do seu roteador, realize o seguinte:
    • Se o seu roteador não suportar UPnP, consulte a etapa 3.
    • Se o seu roteador UPnP foi testado e for compatível com o dispositivo DiskStation, clique em Aplicar após observar o resumo das informações do roteador.A conexão do dispositivo DiskStation para o roteador está configurada com êxito.
    • Se você tiver um roteador UPnP sem comprovação de compatibilidade com o dispositivo DiskStation, realize uma das seguintes ações:
      • Se você quiser configurar a conexão automaticamente para o roteador, selecione Automático e clique em Avançar para que o sistema execute o teste de compatibilidade:
        • Se o roteador for compatível com o serviço de configuração do roteador do dispositivo DiskStation, marque a caixa de seleção para enviar o relatório de teste para ajudar a Synology a atualizar o banco de dados de compatibilidade do roteador e depois clique em Aplicar.A conexão do dispositivo DiskStation para o roteador está configurada com êxito.
        • Se o roteador não for compatível com o serviço de configuração do roteador do dispositivo DiskStation, marque a caixa de seleção para enviar o relatório de teste para ajudar a Synology a atualizar o banco de dados de compatibilidade do roteador e depois clique em Fechar para sair do assistente de configuração.

          • Nota:

          Se o roteador não for compatível, você ainda poderá configurar as regras de encaminhamento de portas na interface de usuário do roteador.Consulte aqui para obter mais informações.

      • Se você quiser configurar manualmente a conexão para o roteador, selecione Manual, clique em Avançar e consulte a etapa 3.
  3. Se for exigido que você configure manualmente a conexão do dispositivo DiskStation com o roteador, faça o seguinte:
    1. Escolha seu roteador no menu suspenso Roteador.

      2. Nota:

      Se você não conseguir encontrar seu roteador aqui, clique em Atualizar Lista para obter a lista mais recente da Synology.

    2. Configure o protocolo e a porta do seu roteador.O protocolo e a porta variam conforme o modelo do roteador.Se não conseguir conectar usando o Protocolo (HTTP) e Porta (80), consulte a documentação fornecida com o roteador para as configurações corretas.
    3. Se seu roteador exigir uma conta personalizada, marque Usar conta personalizada e preencha os campos Conta e Senha.
    4. Clique em Aplicar para concluir.

Gerenciamento de regras de encaminhamento de porta

A regra de encaminhamento de portas pode especificar as portas de destino do seu DiskStation que receberão pacotes de portas específicas do roteador.

Se seu DiskStation se conectar à Internet através de um roteador, clique em Criar para abrir o assistente de configuração encaminhamento de portas.

Há duas maneiras de adicionar regras de encaminhamento de porta:

  • Se quiser criar regras de encaminhamento de porta para serviços de sistema (por exemplo, FTP, Web Station, etc), selecione Aplicativo integrado.
  • Se desejar criar regras de encaminhamento de portas para outros serviços, selecione Porta personalizada.

Você pode usar a sintaxe a seguir para especificar os números e o intervalo da porta:

  • Use "-" para descrever o intervalo da porta, por exemplo: 6881-6890.
  • Use "," para separar as portas, por exemplo: “21,22”, “21,55536-55663”.

Nota:

  • Cada porta do gateway do roteador pode ser usada apenas para uma regra de envio.Se usar uma porta do roteador para duas ou mais regras isso causará conflitos, e uma mensagem de aviso aparecerá na tela.Você precisará configurar outra porta na regra para evitar conflitos.
  • Alguns roteadores não permitem o uso de portas personalizadas para o envio de serviços de sistema.

Para salvar as regras:

Clique em Salvar após criar as regras.Caso contrário, as regras não serão aplicadas.

Após salvar as regras, você terá de esperar alguns minutos antes que as regras comecem a funcionar, dependendo dos modelos dos roteadores.

Par testar se as regras estão funcionando ou não:

Selecione as regras que deseja testar, e clique em Testar Conexão.O resultado será exibido na coluna Resultado do Teste de Conexão.

Para configuração do roteador sem fio*

Se seu DiskStation estiver funcionando como um roteador sem fio, as regras de encaminhamento de porta são executadas diretamente pelo DiskStation.

Para adicionar regras de encaminhamento de porta:

  1. Clique em Criar para abrir a janela de configuração.

  2. Especifique o nome do Serviço, Protocolo (TCP ou UDP), Endereço IP Privado, Porta Privada e Porta Pública.Você pode usar a sintaxe a seguir para especificar os números e o intervalo da porta:
    • Use "-" para descrever o intervalo da porta, por exemplo: 6881-6890.
    • Use "," para separar as portas, por exemplo: “21,22”, “21,55536-55663”.
  3. Clique em Criar na parte inferior para concluir as configurações.

Nota:

A maioria dos roteadores permite que as portas da máquina local e portas do roteador sejam diferentes ao se configurarem as regras de encaminhamento de porta.Se seu roteador suportar esse recurso:

  • Ao salvar uma regra, se a porta do roteador já estiver em uso por uma regra existente, uma nova porta do roteador será atribuída.
  • Ao adicionar ou editar regras, você pode especificar diretamente o número da porta do roteador.

DMZ*

Você pode habilitar o DMZ em seu DiskStation quando a Configuração do Mapa da Rede for o Roteador sem fio.Um host no DMZ pode se conectar diretamente à rede externa, livre da limitação e proteção do firewall.Ele geralmente é usado para a configuração de servidores.

Para habilitar a DMZ:

  1. Clique em Configurações de DMZ para abrir a janela de configuração.
  2. Marque a caixa de verificação Habilitar DMZ.
  3. Digite o endereço do DMZ Host IP.
  4. Clique em Aplicar.

Nota:

Da rede externa, você precisará de um endereço IP externo recuperado pelo DiskStation para se conectar a um host no DMZ.

____
* Disponível apenas em certos modelos ou de acordo com seu ambiente.
