Clonar uma pasta compartilhada*

Você pode clonar uma pasta compartilhada nos volumes Btrfs para criar cópias quase instantâneas de uma pasta compartilhada existente. Clonar uma pasta compartilhada consumirá apenas uma fração do armazenamento no momento da criação.

Para clonar uma pasta compartilhada:

  1. Vá até Painel de Controle > Pasta Compartilhada.
  2. Selecione a pasta compartilhada desejada em sistema de arquivo Btrfs.
  3. Clique em Criar > Clonar.
  4. Especifique um nome e descrição da nova pasta compartilhada e modifique outras opções se necessário. (Para obter informações detalhadas sobre essas opiniões, consulte Criação de uma pasta compartilhada.)

Nota:

  • Apenas pastas compartilhadas em volumes Btrfs podem ser clonadas.
  • A pasta compartilhada clonada será criada no mesmo volume da pasta compartilhada original.
  • As configurações da proteção de integridade avançada de dados serão aplicadas na pasta compartilhada clonada e não podem ser alteradas.

____
*Disponível apenas em certos modelos.
