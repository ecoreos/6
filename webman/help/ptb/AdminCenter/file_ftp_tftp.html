TFTP

O protocolo de transferência de arquivo trivial (TFTP) é um protocolo de transferência de arquivo simples, usado geralmente para transferir arquivos de configuração ou reinicialização quando a autenticação não é necessária. A habilitação do serviço TFTP permite que o DiskStation transfira carregadores de inicialização para os dispositivos clientes.

Para ajustar as configurações de TFTP:

  1. Vá para Painel de controle > Serviços de arquivos > TFTPe selecione Habilitar serviço TFTP.
  2. No campo pasta TFTP raiz, especifique qual pasta do DiskStation pode ser acessada pelos clientes TFTP.
  3. Clique em Configurações avançadas para ajustar as seguintes configurações:
    • Habilitar registro de transferência de arquivos por TFTP: Acompanhe os dados transferidos por TFTP.
    • Permissão de cliente TFTP: Decidir permissões de cliente TFTP:
      • Somente leitura: Clientes TFTP podem visualizar apenas os conteúdos da pasta raiz.
      • Gravável: Clientes TFTP podem modificar os conteúdos da pasta raiz.
    • Clientes permitidos: Escolha quais clientes têm permissão para se conectar via TFTP:
      • Todas as conexões: Todos os clientes podem se conectar através do TFTP.
      • Permitir apenas a seguinte faixa de endereço IP: Apenas clientes na faixa de IP especificada podem se conectar através do TFTP.
    • Tempo limite: Se o servidor TFTP (i.e. seu DiskStation) permanecer sem resposta no período especificado, a conexão TFTP será interrompida.
  4. Clique em Aplicar para salvar as configurações.
