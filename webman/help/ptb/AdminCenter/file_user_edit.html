Editar usuário

Localize a conta de usuário (informando o nome de usuário no campo pesquisar), dê um clique duplo nela ou clique em Editar e você poderá editar os dados de sua conta.

Nota:
  1. Se você desabilitar uma conta, ela irá parar de funcionar.
  2. A Cota deve ser um inteiro maior ou igual a 0 e menor que a capacidade do disco/volume. Um valor de cota de 0 significa que a Cota efetiva será determinada pelo Limite do grupo.
  3. Se houver mais de um volume/pasta compartilhada no servidor, você pode definir uma cota para o usuário em cada volume/pasta compartilhada.
  4. As informações sobre e-mail na página "Notificação" devem ser definidas corretamente para que o sistema notifique o usuário por e-mail depois que a conta tenha sido criada ou modificada.
  5. Ao configurar o limite de velocidade para um usuário, você verá números em Resultado se já houver um limite de velocidade de grupo aplicado ao usuário.
  6. O limite de velocidade atribuído a um usuário não será incluído no limite de velocidade do grupo ao qual o usuário pertence.
