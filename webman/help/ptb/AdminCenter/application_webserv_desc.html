Serviços Web

Seu DiskStation vem com recursos avançados de hospedagem na Web. Usando o Web Station, você pode hospedar e publicar facilmente seu próprio site com o suporte a Host Virtual e configurações adicionais de HTTP/HTTPS. Alternativamente, você pode implementar o suporte integrado a PHP e criar sites dinâmicos e orientados a bancos de dados para seu uso pessoal ou de negócios.
