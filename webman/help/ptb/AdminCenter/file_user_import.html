Importar Usuários

Você pode criar contas de usuário em modo lote importando uma lista de usuários.

Para criar a lista de usuários:

Método 1. Use um editor de texto:

  1. Abra um novo arquivo com um editor de texto.
  2. Informe as informações de usuário na ordem a seguir e separe-as com TAB:
    • Nome de usuário
    • Senha
    • Descrição
    • E-mail
    • Cota (em MB) no volume 1 (repita esta passo se tiver mais de um volume)

      Nota:

      A cota do usuário não é suportada no sistema de arquivos Brtfs e dispositivos de armazenamento externos (ex.: unidades USB/eSATA).

  3. Vá para a próxima linha e repita o passo 2 até que tenha informado todos os usuários. Cada linha indica a informação de um usuário.
  4. Salve o arquivo com codificação UTF-8.

Método 2. Use o Microsoft Excel:

  1. Abra um novo arquivo no Excel.
  2. Entre as informações de usuários na ordem a seguir separadas por colunas em uma única linha:
    • Nome de usuário
    • Senha
    • Descrição
    • E-mail
    • Cota (em MB) no volume 1 (repita esta passo se tiver mais de um volume)
    • Cota (MB) no volume 1 (repita esta passo se tiver mais de um volume)

      Nota:

      A cota do usuário não é suportada no sistema de arquivos Brtfs e dispositivos de armazenamento externos (ex.: unidades USB/eSATA).

  3. Vá para a próxima linha e repita o passo 2 até que tenha informado todos os usuários. Cada linha indica a informação de um usuário.
  4. Abra um novo arquivo com um editor de texto. Copie as entradas da planilha e cole-as no arquivo texto.
  5. Salve o arquivo com codificação UTF-8.

Nota:

  • Todos os campos são opcionais exceto o nome de usuário.
  • Um exemplo do arquivo de texto é mostrado a seguir:
    • rosy 1234 mãe rosy@abc.com 80000
    chaddy 5678 irmão chaddy@abc.com 80000
    danny 8901 primo danny@abc.com 20000
    anny 2345 irmã anny@abc.com 20000

Para importar a lista de usuários:

  1. Selecione Criar > Importar usuários.
  2. Marque Sobrescrever contas duplicadas se quiser substituir as contas duplicadas por aquela da lista de usuários.
  3. Clique em Procurar e selecione o arquivo da lista de usuários. Após selecionar o arquivo, a região de visualização irá mostrar a lista de usuários importados. Usuários com status anormal não serão importados.
  4. Clique em Carregar para criar as contas de usuário.

Nota:

O arquivo da lista de usuários deve estar no formato UTF-8.
