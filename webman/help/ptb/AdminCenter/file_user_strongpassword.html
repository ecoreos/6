Configurações da senha

Clique no botão ï»¿Configurações da senha para ver e modificar as configurações descritas abaixo.

Redefinir senha esquecida

Esta seção inclui uma opção marcada Permitir que usuários que não sejam o administrador possam redefinir suas senhas por e-mail. Quando essa opção está ativada, o link marcado Esqueceu sua senha? irá aparecer na página de login do DSM. Se um usuário esquecer a senha, ele poderá clicar neste link e inserir seu nome de usuário. Nesse caso, o sistema enviará uma mensagem ao usuário contendo um link para redefinir a senha esquecida.

Observação:

  • Antes de ativar essa opção, você já deve ter ativado as notificações por e-mail do SMTP. Para fazer isso, vá até Painel de controle > Notificação > Geral. Se as notificações não estiverem ativadas, o sistema não poderá enviar mensagens aos usuários que desejam redefinir suas senhas.
  • Para receber mensagens do sistema, os usuários devem fornecer um endereço de e-mail na seção de informações do usuário de suas contas.
  • Os usuários que pertençam ao grupo administrators não podem redefinir senhas com esta opção.
  • Os usuários de domínio não podem redefinir senhas com esta opção.
  • Os usuários LDAP podem redefinir senhas com esta opção, desde que o DiskStation atue como o LDAP diretory server.

Força da Senha

Você pode configurar as regras da força da senha para os usuários no seu DiskStation.

Observação:

As regras da força da senha se aplicam apenas às senhas novas. Por exemplo, as regras de senhas novas são aplicadas apenas ao criar uma nova conta de usuário, ou quando um usuário existente altera sua senha. As senhas existentes e aquelas que pertencem às contas de usuário importadas são excluídas das regras de novas senhas.

Para ativar regras de força da senha:

  1. Marque a caixa de seleção Aplicar regras de força da senha para habilitar as restrições de senha.
  2. Selecione mais de uma das restrições por senha das regras abaixo:
    • Excluir nome e descrição do usuário da senha: a senha não deve conter o nome ou a descrição do usuário. Entretanto, os caracteres codificados com UTF-8 são excluídos.
    • Permitir letras maiúsculas e minúsculas: São permitidas maiúsculas e minúsculas na senha.
    • Incluir caractere numérico: a senha deve conter pelo menos um caractere numérico (0~9).
    • Incluir caractere especial: A senha deve conter pelo menos um caractere especial ASCII (i.e., ~, `, !, @, #, $, %, ^, &, *, (, ), -, _, =, +, [, {, ], }, \, |, ;, :, ', ", <, >, /, ?).
    • Comprimento mínimo da senha: A senha deve ter pelo menos o valor especificado no campo de texto. O comprimento deve ser um número entre 6 e 127.
  3. Clique em Aplicar para salvar as configurações.
