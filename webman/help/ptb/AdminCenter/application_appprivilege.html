Privilégios

É possível visualizar e editar os privilégios de usuários e grupos para serviços e aplicativos individuais. O DSM oferece configurações flexíveis para controlar os privilégios de aplicativos, tornando mais fácil gerenciar o acesso de usuários e grupos a diferentes aplicativos.

Para modificar os privilégios de aplicativos de usuários e grupos:

  1. Vá até Painel de controle > ﻿Privilégios.
  2. Selecione o aplicativo que você deseja modificar. Clique em Editar.
  3. Clique em Usuário para visualizar privilégios individuais ou em Grupo para visualizar privilégios de grupo.
  4. Para cada usuário ou grupo, marque Permitir para permitir acesso ou Negar para negar acesso ao aplicativo selecionado. Dependendo do aplicativo, você também pode selecionar Por IP para permitir/negar acesso com base no endereço IP.
  5. Clique em OK para salvar suas alterações.

Para visualizar os privilégios de aplicativos de usuários que possuem diferentes endereços IP de rede:

  1. Vá até Painel de controle > ﻿Privilégios.
  2. Clique em Visualizador de permissão.
  3. Selecione um usuário em Nome de usuário e digite o endereço IP cujas permissões você deseja visualizar em Endereço IP.

Para definir os privilégios de aplicativos padrão para todos os usuários:

  1. Vá até Painel de controle > ﻿Privilégios.
  2. Selecione o aplicativo que você deseja modificar. Clique em Editar.
  3. Clique em Privilégios padrão e marque Conceder este privilégio a todos.
  4. Clique em OK para salvar suas alterações.

Observação:

  • Os privilégios das contas admin e guest não podem ser modificados aqui.
  • Quando encontrar conflitos de privilégios, a prioridade de privilégios é a seguinte: Negar > Permitir.
  • Nem todos os pacotes e serviços são compatíveis com configurações de privilégio de acesso Por IP.
