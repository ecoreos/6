Gerenciar permissões avançadas de pasta compartilhada

A página em Painel de controle > Pasta compartilhada > Editar > Avançado inclui as opções abaixo para ajuste fino das permissões de acesso de uma pasta compartilhada.

Permissões de compartilhamento avançadas

As permissões de compartilhamento avançadas oferecem uma camada adicional de controle para gerenciar as permissões de acesso de pastas compartilhadas. Quando ativada, os usuários e grupos podem visualizar ou modificar o conteúdo de uma pasta compartilhada somente se o usuário ou grupo tiver permissões de compartilhamento avançadas e do Windows ACL (localizadas em Pasta compartilhada > Editar > Permissões).

Para ativar as permissões de compartilhamento avançadas:

  1. Marque a caixa Ativar permissões de compartilhamento avançadas.
  2. Clique em Permissões de compartilhamento avançadas para modificar essas permissões para a pasta compartilhada.

Observação:

  • As permissões de compartilhamento avançadas são aplicadas quando um usuário acessa uma pasta compartilhada através dos seguintes serviços de arquivo: Windows File Sharing, Apple File Sharing, File Station, FTP e WebDAV.

Configurações avançadas

Se necessário, você pode colocar mais restrições em usuários quando eles acessam uma pasta compartilhada através do File Station, FTP ou WebDAV.

  • Desabilitar pesquisa em diretórios: ativar essa opção restringe aos usuários a visualização do conteúdo da pasta compartilhada.
  • Desabilitar a modificação de arquivos existentes: ativar essa opção restringe aos usuários a movimentação, exclusão ou modificação de arquivos da pasta compartilhada. Observe que os usuários ainda poderão ver, baixar/carregar, copiar ou descompactar o conteúdo da pasta compartilhada.
  • Desabilitar download de arquivos: ativar essa opção restringe aos usuários fazer download do conteúdo da pasta compartilhada.

Observação:

Observe que essas restrições não se aplicam aos usuários que pertencem ao grupo administrators.
