Habilitar lixeira

Quando você habilita a Lixeira em uma pasta compartilhada, é criada uma subpasta para armazenar os arquivos excluídos. Você pode recuperar os arquivos excluídos da Lixeira. Esvazie a Lixeira se quiser remover os arquivos permanentemente.

Para habilitar a Lixeira:

  1. Selecione uma pasta compartilhada e depois clique em Editar.
  2. Marque a caixa de seleção Habilitar Lixeira.
  3. Marque Restringir acesso apenas a administradores se quiser que a Lixeira fique acessível apenas para administradores.
  4. Clique em OK para salvar as configurações.

Para esvaziar a Lixeira:

Clique no botão Esvaziar Lixeira para remover todos os arquivos excluídos da lixeira permanentemente.

Para esvaziar todas as Lixeiras:

Selecione Esvaziar todas as lixeiras do menu suspenso Ação para remover os arquivos excluídos de todas as lixeiras permanentemente.

Nota: É possível ajustar as configurações de privilégio da pasta Lixeira no File Station clicando-se com o botão direito na pasta #recycle. Selecione Propriedades e clique na guia Permissão.
