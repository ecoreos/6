Editar um grupo

Você pode editar as informações do grupo e seus privilégios de acesso às pastas compartilhadas.

Para editar as informações do grupo:

  1. Selecione o grupo que você deseja editar.
  2. Clique em Editar e edite as informações do grupo.
  3. Clique em OK para concluir.

Para editar os privilégios de acesso do grupo às pastas compartilhadas:

  1. Selecione o grupo que você deseja editar.
  2. Clique em Editar e Permissões.
  3. Marque ou desmarque os privilégios a seguir para atribuir privilégios de acesso ao grupo:
    1. Leitura/Gravação: O grupo pode acessar e fazer alterações à pasta compartilhada.
    2. Somente leitura: O grupo pode acessar a pasta compartilhada, mas não pode fazer alterações a ela.
    3. Nenhum acesso: O grupo não pode acessar a pasta compartilhada.
  4. Clique em OK para concluir.

Nota:

Quando encontrar conflitos de privilégios, a prioridade de privilégios é a seguinte: Nenhum acesso > Leitura/Gravação > Somente leitura.

Para editar a cota de uso do grupo:

  1. Selecione o grupo que você deseja editar.
  2. Clique em Editar e vá para a guia Cota.
  3. Para cada volume/pasta compartilhada, marque a caixa em Habilitar cota. Em seguida digite uma cota de uso no campo Cota (ex.: 10 GB).

Nota:

  • Cada membro do grupo herda a cota de uso de grupo específica. A cota de uso do grupo não é compartilhada entre cada membro do grupo.
  • A cota de uso de armazenamento máxima que pode ser inserida para cada grupo é de aproximadamente 4 TB (4095 GB).

Para editar os privilégios de acesso de um grupo aos aplicativos:

  1. Selecione o grupo que você deseja editar.
  2. Clique em Editar e vá para a guia Aplicativos.
  3. Para cada serviço, você pode selecionar uma das seguintes opções:
    • Permitir: O grupo poderá acessar o aplicativo.
    • Negar: O grupo não poderá acessar o aplicativo.
    • Personalizado: Gerencie os privilégios de acesso por endereço IP.
  4. Clique em OK para concluir.

Nota:

  • Quando encontrar conflitos de privilégios, a prioridade de privilégios é a seguinte: Negar > Permitir.
  • Nem todos os pacotes e serviços são compatíveis com configurações de privilégio de acesso Por IP.

Para editar o limite de velocidade do grupo:

Você pode ativar um limite de velocidade de grupo para diferentes serviços (por exemplo, File Station, FTP etc.) a fim de restringir a largura de banda consumida pelos membros do grupo que transferem arquivos através de cada serviço. Os limites de velocidade de um grupo são compartilhados por todos os seus membros.

  1. Selecione o grupo que você deseja editar.
  2. Clique em Editar e vá para a guia Limite de velocidade.
  3. Para cada serviço, você pode selecionar uma das seguintes opções:
    • Nenhum: Nenhum limite de velocidade será aplicado quando os membros do grupo usarem o serviço.
    • Constante: Um limite de velocidade fixo e constante será aplicado aos membros de grupos. Especifique os limites de velocidade de upload e download nos campos à esquerda.
    • Variável: Dois limites de velocidade diferentes podem ser especificados e aplicados aos membros de grupo, de acordo com uma programação. Clique no botão Personalizar para modificar as configurações de limite de velocidade e a programação.
  4. Clique em OK para concluir.

Limite de velocidade para usuários que pertencem a vários grupos

Quando um usuário pertence a vários grupos, seu limite de velocidade é decidido com base no grupo com a largura de banda disponível mais alta no momento da transferência dos arquivos. Como a largura de banda é dividida uniformemente entre os usuários que estão transferindo arquivos atualmente, a largura de banda disponível muda com base em quantos usuários estão consumindo a largura de banda compartilhada atualmente.

Por exemplo, um usuário "John" pertence ao "Grupo 1" (limite de velocidade de 500KB/s; total de cinco usuários) e o "Grupo 2" (limite de velocidade de 400KB/s; total de dois usuários).

  • Se os usuários do Grupo 1 e Grupo 2 estiverem todos transferindo arquivos ao mesmo tempo, então o limite de velocidade de John será de 200KB/s e seu consumo contará na cota de largura de banda compartilhada do Grupo 2.
  • Quando os outros usuários do Grupo 1 tiverem terminado de transferir os arquivos, o limite de velocidade de John mudará imediatamente para 500KB/s e seu consumo contará na cota de largura de banda compartilhada do Grupo 1.
