USBCopy

Você pode selecionar e definir um local alvo para a função USBCopy.

Para ativar USBCopy:

  1. Marque Ativar USBCopy.
  2. Selecione a pasta compartilhada de destino no menu suspenso.
  3. Clique em OK.

Ao conectar um dispositivo USB (tal como uma câmera digital) à porta USB no painel frontal do Synology DiskStation, você pode pressionar o botão Copiar no painel frontal para copiar os dados no dispositivo USB para a pasta compartilhada de destino atribuída aqui. Se a pasta de destino do USBCopy for deletada, o sistema irá substituí-la pela pasta compartilhada pública automaticamente.

    Limitações:
  1. Somente são suportados dispositivos USB no formato FAT ou NTFS.
  2. Os dispositivos USB devem estar conectados à porta USB no painel frontal do Synology DiskStation.
  3. O espaço disponível da pasta compartilhada de destino deve ser maior que o espaço usado no dispositivo USB.
