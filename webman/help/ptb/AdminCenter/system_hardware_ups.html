UPS

Um no-break (Fonte de alimentação ininterrupta) é um dispositivo de energia de segurança que permite que o DiskStation continue a funcionar por um curto período se ocorrer uma falha de energia. Essa função em Painel de Controle > Hardware e Alimentação > UPS ajuda a evitar perda de dados dando ao DiskStation tempo suficiente para salvar os dados e desmontar volumes antes de perder a energia e desligar.

Essa função em Painel de Controle > Hardware e Alimentação > UPS ajuda a evitar perda de dados dando ao DiskStation tempo suficiente para salvar os dados e desmontar volumes antes de perder a energia e parar.

Modo Seguro

Quando o DiskStation entra no modo seguro, ele para todos os serviços e desmonta volumes para evitar perda de dados e desliga com segurança quando o dispositivo no-break fica sem energia. Por padrão o sistema entra no Modo seguro quando o no-break começa a funcionar com pouca energia. Você também pode especificar quanto tempo antes o DiskStation entrará no Modo seguro quando ocorrer falha de energia. Entretanto, se o dispositivo no-break ficar com a bateria fraca antes do tempo especificado, o sistema entrará no Modo seguro imediatamente.

Por padrão o sistema entra no Modo seguro quando o no-break começa a funcionar com pouca energia. Você também pode especificar quanto tempo antes o DiskStation entrará no Modo seguro quando ocorrer falha de energia. Entretanto, se o dispositivo no-break ficar com a bateria fraca antes do tempo especificado, o sistema entrará no Modo seguro imediatamente.

Nas situações em que o DiskStation desliga durante o modo de segurança, ele liga automaticamente quando a energia for restaurada, se você ativar a opção Reiniciar automaticamente após uma falha de energia (localizada no Painel de controle > Hardware e Alimentação > Geral).

Suporte ao no-break local

Para obter detalhes sobre a instalação de hardware, consulte o manual do usuário do dispositivo no-break. Quando a instalação estiver concluída e o DiskStation tiver sido conectado ao dispositivo no-break, você pode seguir as etapas abaixo para habilitar o suporte ao no-break.

Para habilitar o suporte ao no-break local:

  1. Conecte o dispositivo no-break a uma porta USB do DiskStation.
  2. Na guia UPS, marque a caixa de seleção Habilitar suporte a UPS.
  3. Selecione quando o DiskStation deve entrar no Modo seguro após ocorrer falha de energia.
  4. Selecione se será ou não enviado um sinal de desligamento ao no-break quando o sistema entrar no Modo Seguro. A maioria dos dispositivos no-break podem ligar automaticamente quando a energia é restaurada.
  5. Clique em Aplicar.

Suporte ao no-break SNMP

Seu DiskStation também suporta conexão a dispositivos no-break SNMP (Protocolo de gerenciamento de rede simples). Para obter detalhes sobre a instalação de hardware, consulte o manual do usuário do dispositivo no-break SNMP. Quando o DiskStation tiver sido conectado a um dispositivo e conectado à mesma rede, você pode seguir as etapas abaixo para habilitar o suporte ao no-break.

Para habilitar o suporte ao no-break SNMP:

  1. Na guia UPS, marque a caixa de seleção Habilitar suporte a UPS.
  2. Selecione UPS SNMP no menu suspenso Tipo de UPS de Rede.
  3. No campoEndereço IP do UPS SNMP, digite o endereço IP do dispositivo UPS SNMP.
  4. Selecione quando o DiskStation deve entrar no Modo seguro após ocorrer falha de energia.
  5. Digite o segmento de comunidade do dispositivo UPS SNMP no campo Comunidade SNMP.
  6. Clique em Aplicar.

Nota:

  • É altamente recomendado usar SNMP UPS para implementações do Synology High Availability (SHA).
  • O DiskStation suporta dispositivos UPS SNMPv1 e SNMPv2c.

Suporte ao no-break de rede

Se houver dois produtos Synology usando um dispositivo no-break com porta USB única. então apenas um DiskStation pode obter informações sobre o status do dispositivo no-break. Nesse caso, um DiskStation pode funcionar como um "servidor de no-break de rede" e conectar o dispositivo no-break através de USB ou rede (dependendo do dispositivo). O servidor de no-break de rede obtém informações do no-break e as retransmite para outro "cliente" DiskStation.

Para habilitar o suporte a UPS de rede:

  1. Faça login no DiskStation que está conectado ao dispositivo no-break (via USB ou rede). Se o suporte ao no-break local ainda não tiver sido habilitado nesse DiskStation, consulte a seção abaixo.
  2. Na guia UPS marque a caixa Habilitar servidor UPS de rede para permitir que o DiskStation retransmita as informações do UPS para o cliente DiskStation.
  3. Clique no botão Dispositivos DiskStation permitidos.
  4. Na janela que aparece, digite os endereços IP do outro DiskStation que está usando o dispositivo no-break, mas não está conectado via USB ou rede. Clique em OK.
  5. Clique em Aplicar para salvar as configurações.

Agora o servidor de no-break de rede pode obter informações do no-break e retransmiti-las para o outro DiskStation que está usando o dispositivo no-break. Consulte as instruções abaixo para habilitar o suporte do no-break no outro DiskStation.

Para habilitar o suporte ao no-break no cliente DiskStation:

  1. Faça login no outro DiskStation que está usando o dispositivo no-break, mas não está conectado via USB ou rede.
  2. Na guia UPS, marque a caixa Habilitar suporte a UPS.
  3. Selecione Servidor UPS Synology no menu suspenso Tipo de UPS de Rede.
  4. Digite o endereço IP do servidor UPS da rede (i.e. o DiskStation conectado ao dispositivo UPS) no campo IP do servidor UPS de rede.
  5. Selecione quando o DiskStation deve entrar no Modo seguro após ocorrer falha de energia. Para simplificar as configurações de Modo Seguro, você pode selecionar Mesma do servidor.
  6. Clique em Aplicar para salvar as configurações.

Requisitos do no-break de rede:

  • Para retransmitir as informações do dispositivo no-break, cada DiskStation deve estar conectado à mesma rede.
  • O dispositivo de rede (ex.: switch ou hub) ao qual cada DiskStation está conectado deve ser acoplado ao mesmo dispositivo no-break. Caso contrário os DiskStations não poderão retransmitir as informações do dispositivo no-break quando ocorrer falha de energia.
