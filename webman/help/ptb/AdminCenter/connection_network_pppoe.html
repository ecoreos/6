PPPoE

Se você usa um modem DSL ou a cabo para se conectar à Internet e seu Provedor de Serviços de Internet (ISP) usa PPPoE (Protocolo Ponto a Ponto pela Ethernet), você pode ativar PPPoE no Painel de controle > Rede > Interface de Rede e digitar as informações da conta para se conectar diretamente à Internet sem o roteador.

Para obter mais informações sobre PPPoE, entre em contato com seu ISP ou administrador de rede.

Configuração de IPv6

Os dispositivos DiskStation suportam os esquemas de pilha dupla IPv4/IPv6. Isto significa que tanto IPv4 quanto IPv6 podem trabalhar em seu DiskStation ao mesmo tempo. A configuração padrão para IPv6 é Auto. Selecione uma interface de rede existente, clique no botão Editar e vá para a guia IPv6 para alterar as configurações.

Até o IPv6 suplantar completamente o IPv4, os mecanismos de transição de IPv6 são necessários para permitir que os hosts com endereços IPv4 se conectem a serviços IPv6. Escolha entre os diferentes mecanismos de transição de IPv6:

  • 6in4: Conecte-se à rede IPv6 por meio de um agente de túnel. Você precisará preencher Endereço IPv6, Comprimento do Prefixo, Prefixo e Endereço IPv4 do Agente de Túnel. Isso requer um endereço IPv4 público.
  • 6to4: Conecte-se à rede IPv6 por meio de um agente de túnel utilizando Anycast. Isso requer um endereço IPv4 público.
  • DHCP-PD: Obtenha um prefixo IPv6 do DHCP Server por meio da delegação de prefixo com a LAN. Isso requer suporte a ISP e funciona somente em modo de roteador.
