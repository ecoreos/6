Dispositivos Externos

Vá para Painel de Controle > Dispositivos Externos > Dispositivos Externos você pode ver a lista de dispositivos de armazenamento externo conectados ao seu DiskStation.

Discos Externos

Ao conectar o disco externo ao sistema, você poderá compartilhar a capacidade do disco através de uma pasta compartilhada criada pelo sistema, chamada usbshare[número] (para discos USB). A pasta compartilhada será removida automaticamente quando o disco externo for desconectado do sistema.

Ao conectar um disco externo ao sistema, você poderá compartilhar a capacidade do disco através de uma pasta compartilhada criada pelo sistema com o nome usbshare[número] (para discos USB) ou satashare[número] (para discos eSATA). A pasta compartilhada será removida automaticamente quando o disco externo for desconectado do sistema.

DiskStation reconhece os formatos a seguir: Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus e NTFS. Qualquer disco externo não reconhecido deve ser formatado antes de ser usado pelo sistema.

Observação:

  1. Alguns modelos oferecem suporte a HFS/HFS Plus com somente leitura.
  2. O diário não é suportado em HFS/HFS Plus.
  3. Verifique se o disco externo foi ejetado antes de desconectá-lo.

Para formatar o disco:

Clique em Formatar e siga o assistente. Você pode formatar discos externos nos formatos ext4 e FAT32 com o DiskStation.

Observação:

  1. Lembre-se de fazer backup dos dados antes de formatar o disco externo. Todos os dados serão apagados durante o procedimento e não poderão ser restaurados.
  2. Para formatar o disco externo em formato ext3 ou ext4, o tamanho do disco deve ser maior que 1 GB.
  3. Quando o disco estiver formatado no formato ext3 ou ext4, somente será reconhecido pelo DiskStation. Para ler os dados diretamente de seu disco externo com seu PC ou Mac, escolha o formato FAT32.

Para realizar o teste S.M.A.R.T.:

O teste S.M.A.R.T. do disco examina e relata o status de seus discos rígidos, fornecendo um alerta antecipado sobre falhas iminentes no disco. Recomendamos a troca imediata do disco em caso de detecção de algum erro.
Para realizar o teste S.M.A.R.T., basta clicar no botão Teste S.M.A.R.T. e seguir o assistente.
Para verificar os resultados do teste, basta clicar em Informações S.M.A.R.T.

Observação:

O tempo de teste varia de acordo com o ambiente do sistema.

Limitações

A formatação de disco não está disponível nas seguintes situações:

  • O disco externo está sendo iniciado, ou seja, quando o dispositivo externo foi conectado recentemente.
  • A capacidade do disco externo é menor que 1 GB (formatos ext3 ou ext4).
  • O disco externo está sendo formatado.
  • Quando o sistema estiver executando backup de arquivos para o disco externo.
  • O disco externo está danificado.

Solução de problemas

  1. Falha ao formatar o disco externo. Por quê?
    • Verifique se o disco externo está devidamente inserido.
    • Verifique se o disco externo está protegido contra gravação. Caso positivo, remova-o antes de formatar.
  2. O sistema de arquivos do disco externo é exibido como Não suportado ou o sistema não cria a pasta compartilhada usbshare[número] ou satashare[número] do disco externo. O que devo fazer?
    • O sistema atualmente reconhece os formatos Btrfs, ext3, ext4, FAT, FAT32, HFS, HFS Plus e NTFS. Qualquer disco externo não formatado como os tipos acima não será reconhecido e seu status será mostrado como Não Suportado.
  3. Não consigo acessar a pasta compartilhada usbshare[número] ou satashare[número]. O que devo fazer?
    • Isso pode ser resultado de um problema no sistema de arquivos do disco externo. Para solucionar esse problema, conecte o disco externo ao PC e realize uma verificação do sistema de arquivos
  4. O meu disco externo com o formato ext3 e ext4 pode ser conectado ao meu PC ou Mac e acessado ​​diretamente?
    • Não, o ext3 ou ext4 é reconhecido apenas pelo DiskStation. Para ler os dados diretamente de seu disco externo com seu PC ou Mac, escolha o formato FAT32.
  5. Por que vejo uma pasta chamada "lost+found" em meu disco externo?
    • Esta é uma pasta padrão do sistema que é criada automaticamente quando o sistema de arquivos ext3/ext4 é usado. É normal ver essa pasta no disco externo. Apesar de os administradores terem o direito de acessar e gerenciar o conteúdo interno, recomendamos que você deixe a pasta intacta.

Para definir as permissões padrão de todos os dispositivos externos:

Você pode definir as permissões padrão clicando em Permissões Padrão e especificando quais usuários ou grupos podem acessar, visualizar ou modificar os dispositivos externos e seus conteúdos. Quando um dispositivo externo é inserido no DiskStation, as permissões que você especificou serão atribuídas automaticamente ao dispositivo externo.

Na janela de permissões, faça o seguinte para definir as permissões padrão:

  1. Selecione uma opção no menu suspenso:
    • Usuário interno do sistema: Atribui permissões para os usuários padrão do sistema, como o usuário Anônimo de FTP/WebDAV.
    • Usuários locais: Atribuir permissões para usuários locais (incluindo guest).
    • Grupos locais: Atribui permissões para grupos locais.
  2. Marque ou desmarque as caixas apropriadas para cada usuário ou grupo, para personalizar suas permissões de acesso a dispositivos externos:
    • Leitura/Gravação: O usuário ou grupo pode acessar e fazer alterações aos arquivos e pastas nos dispositivos externos.
    • Somente leitura: O usuário ou grupo pode acessar os arquivos e pastas nos dispositivos externos.
    • Nenhum acesso: O usuário ou grupo não pode acessar os arquivos e pastas nos dispositivos externos.
