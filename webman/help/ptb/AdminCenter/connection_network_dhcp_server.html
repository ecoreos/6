DHCP Server

O DiskStation pode funcionar como um servidor DHCP e atribuir endereços IP dinâmicos para clientes DHCP dentro de sua rede local.

Para habilitar o servidor DHCP:

  1. Vá para Painel de controle > DHCP Server > Interface de rede.
  2. Selecione uma interface de rede (ex.: LAN 1) para atribuir endereços IP e clique em Editar.
  3. Na janela pop-up, clique na guia DHCP Server e selecione Habilitar DHCP server.
  4. Especifique as seguintes configurações DHCP:
    • Tempo de concessão do endereço (minutos)
    • DNS primário
    • DNS secundário (opcional)
    • Nome do domínio (opcional)
    • Habilitar Descoberta Automática de Proxy Web (opcional)
  5. Configure pelo menos uma sub-rede. Para a configuração da sub-rede, clique em Adicionar para especificar as seguintes informações:
    • Endereço IP inicial
    • Endereço IP final
    • Másc. rede
    • Gateway
  6. Clique em Criar para mostrar a sub-rede criada na lista.
  7. Após criar sub-redes suficientes, lembre-se de habilitar pelo menos uma sub-rede.
  8. Clique em OK.

Nota:

  • Você pode criar quantas sub-redes desejar.
  • Se estiver coordenado com um servidor de retransmissão DHCP, seu DiskStation como DHCP server pode atribuir endereços IP para dispositivos em outra sub-rede.

Clientes DHCP

Você pode encontrar uma lista de clientes DHCP e suas configurações de rede (por exemplo, endereço MAC, endereço IP, nome de host e o tempo restante antes que a concessão do DHCP expire e o status de reserva).

  1. Vá para Painel de controle > DHCP Server > Interface de rede e selecione a interface desejada.
  2. Clique em Editar > Clientes DHCP para encontrar os clientes e os detalhes.

Para gerenciar endereços IP para clientes DHCP:

Você pode fazer com que um cliente sempre receba o mesmo endereço IP durante a renovação da concessão do DHCP e gerenciar esses clientes.

  • Para adicionar clientes DHCP: Clique em Adicionar para atribuir o endereço IP ao dispositivo cliente.
  • Para reservar o endereço IP: Clique no ícone do cadeado para reservar o endereço IP para o dispositivo atual.
  • Para cancelar a reserva: Clique no ícone do cadeado para permitir que o endereço IP atual seja usado por outro dispositivo após a passagem do tempo de concessão do endereço.

Nota:

Se um dispositivo puder ter endereços IPv4 e IPv6, você pode reservar um endereço IPv4 e um IPv6 para ele.

Para pesquisar clientes DHCP:

  1. Clique no ícone do funil no canto superior direito.
  2. Selecione um dos critérios de pesquisa abaixo:
    • Todos: Lista todos os clientes DHCP e reservas DHCP.
    • Clientes DHCP: Lista todos os clientes DHCP com endereços IP reservados ou não reservados.
    • Reservas de DHCP: Lista todos os endereços IP reservados. Um endereço IP reservado ainda não pode ser emparelhado com nenhum outro dispositivo.

Opções DHCP

Depois que você especificar as opções DHCP, seu DiskStation como servidor DHCP poderá enviar informações relacionadas para os clientes DHCP em determinada sub-rede.

Para especificar as opções DHCP para uma sub-rede:

  1. Vá para Painel de controle > DHCP Server > Interface de rede.
  2. Selecione a interface de rede desejada e clique em Editar.
  3. Vá para DHCP Server e depois Lista de sub-redes.
  4. Execute uma das ações abaixo:
    • Para especificar as opções DHCP para uma sub-rede existente: Selecione a interface desejada e clique em Editar.
    • Para criar uma sub-rede e especificar suas opções DHCP: Clique em Adicionar e especifique as configurações de sub-rede.
  5. Selecione as opções DHCP desejadas. Para especificar os valores correspondentes, clique nos campos em branco para edição.
  6. Clique em OK.

Fornecedor

Os clientes DHCP de outros fornecedores podem exigir informações DHCP adicionais. Ao configurar as opções DHCP para os fornecedores, seu DiskStation pode fornecer esses detalhes para os clientes.

Para saber as opções DHCP oferecidas por cada fornecedor, entre em contato com o fornecedor específico para obter detalhes.

Para adicionar fornecedores:

  1. Vá até Painel de controle > ﻿DHCP Server > Fornecedor.
  2. Clique em Criar para especificar o fornecedor e suas opções DHCP:
    • Nome: Especifique o nome do fornecedor.
    • Código: Especifique o código numérico da opção DHCP.
    • Valor: Especifique o valor da opção.
    • +: Adicione uma opção DHCP.
    • -: Exclua uma opção DHCP.
  3. Clique em Criar para concluir a configuração.

Para editar as opções DHCP de fornecedores:

  1. Vá até Painel de controle > ﻿DHCP Server > Fornecedor.
  2. Selecione o fornecedor desejado e clique em Editar para gerenciar suas opções DHCP.
  3. Clique em OK.

PXE

Seu DiskStation pode funcionar como um DHCP server para implementar a configuração do PXE (ambiente de execução pré-inicialização). O PXW é um ambiente de rede que permite que os dispositivos clientes sejam inicializados usando uma interface de rede em vez de um dispositivo de armazenamento de dados (por exemplo, discos rígidos).

Ao implementar o PXE, um carregador de inicialização pode ser armazenado no seu DiskStation ou em um servidor TFTP remoto e os dispositivos clientes podem ser inicializados pela rede através do carregador de inicialização.

Para usar seu DiskStation como um servidor do carregador de inicialização:

  1. Certifique-se de ter selecionado Habilitar serviço TFTP (em Painel de controle > Serviços de arquivo > TFTP).
  2. Selecione uma pasta que hospede o arquivo carregador de inicialização como pasta raiz TFTP.
  3. Verifique se pelo menos uma sub-rede DHCP está habilitada (em Painel de controle > DHCP Server > Interface de rede).
  4. Vá até Painel de controle > ﻿DHCP Server > PXE.
  5. Selecione Habilitar PXE (Ambiente de Execução Pré-inicialização).
  6. Selecione Servidor TFTP local e selecione o arquivo do carregador de inicialização.
  7. Clique em Aplicar para salvar as configurações.

Para usar o servidor TFTP remoto como um servidor de carregador de inicialização:

Usando um servidor TFTP remoto como servidor de carregador de inicialização, você pode permitir que os clientes sejam inicializados sem a necessidade de modificar a infraestrutura de rede atual.

  1. Verifique se pelo menos uma sub-rede DHCP está habilitada (em Painel de controle > DHCP Server > Interface de rede).
  2. Vá até Painel de controle > ﻿DHCP Server > PXE.
  3. Selecione Habilitar PXE (Ambiente de Execução Pré-inicialização).
  4. Selecione Servidor TFTP remoto e especifique o endereço IP do servidor e o caminho do arquivo do carregador de inicialização.
  5. Clique em Aplicar para salvar as configurações.
