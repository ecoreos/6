Restrição na conexão

Permitir somente conexão SSL/TLS

Quando esta função estiver ativada, o servidor só aceitará solicitações de conexão SSL ou TLS de clientes FTP.

Limitar conexões por IP

Quando esta função estiver ativada, o servidor limitará o número máximo de conexões por IP. Ative esta função e a restrição da largura de banda de FTP para um gerenciamento eficiente da largura de banda.

Ativar restrição de largura de banda de FTP

Ative esta função para restringir a largura de banda de uma única conexão FTP.
