Link Aggregation*

O DiskStation suporta multi-LAN, o que permite a combinação das interfaces de LAN usando a tecnologia Link Aggregation. Link Aggregation aumenta a largura de banda de seu DiskStation agregando várias interfaces de rede e fornece failover de tráfego para manter a conexão de rede em caso de queda da conexão.

Depois de combinar as interfaces de LAN, você verá uma nova interface chamada Bond em Painel de controle > Rede > Interface de rede, que pode ser configurada do mesmo modo que as outras interfaces (como LAN ou PPPoE).

Observação:

  • Com o Link Aggregation habilitado, o tráfego de rede se ajustará e se balanceará automaticamente para cada dispositivo conectado.
  • Se as interfaces de rede estiverem conectadas a vários switches, certifique-se de que as portas dos switches estejam na mesma LAN ou VLAN.

Para combinar várias LANs com o Link Aggregation:

  1. Vá até Painel de Controle > Rede > Interface de Rede. Clique em Criar > Criar Bond.
  2. Selecione o modo que deseja.

    Descrições de modo:

    • Adaptive Load Balancing: Esse moto otimiza o tráfego de rede recebido e enviado por seu DiskStation se o switch suportar o Link Aggregation. Para evitar erros inesperados, não habilite o Link Aggregation no switch mesmo quando ele for suportado.
    • Link Aggregation Dinâmico IEEE 802.3ad: Esse modo otimiza o tráfego de rede recebido e enviado pelo seu DiskStation e requer que o IEEE 802.3ad (Dynamic) Link Aggregation (LACP, 802.1AX) seja habilitado no(s) switch(es). Se vários switches forem usados, eles precisam ser empilháveis e configurados adequadamente.
    • Balance XOR: Esse modo balanceia o tráfego de rede recebido e enviado pelo seu DiskStation e requer que o Link Aggregation estático seja habilitado no(s) switch(es). Se vários switches forem usados, eles precisam ser empilháveis e configurados adequadamente. Se o Link Aggregation não for suportado pelo switch, o tráfego enviado pelo seu DiskStation ainda pode ser balanceado. Com esse modo, os links perdidos podem ser detectados mas não a causa (ex.: quando os cabos Ethernet forem desconectados ou o switch for desconfigurado).
    • Ativo/Em espera: Esse modo fornece ao seu DiskStation tolerância de falha de rede. Isto é, quando a interface de rede ativa cair, uma das outras portas LAN assumirá o controle para manter a conexão de rede. Você pode escolher esse modo sem configurar o switch ou com um switch que suporte o IEEE 802.3ad Link Aggregation.
  3. Selecione as interfaces para criar o Link Aggregation.
  4. Ajuste as configurações de IP. Você também pode habilitar o VLAN se necessário. Clique em Aplicar para completar a configuração.
  5. Depois de terminar você pode ver a nova interface nomeada Bond 1 na lista de Interfaces de rede.

Para editar as configurações do Link Aggregation:

  1. Para alterar o modo Link Aggregation ou visualizar as informações dos dispositivos físicos, vá para Painel de controle > Rede > Interface de rede.
  2. Selecione a interface Bond 1 e clique no botão Editar.
  3. Vá para as guias correspondentes para modificar as configurações desejadas.

Para remover o Link Aggregation:

  1. Vá até Painel de Controle > Rede > Interface de Rede.
  2. Selecione a interface Bond 1 e clique no botão Excluir.
  3. Quando concluir você verá que Bond 1 agora foi substituído por duas interfaces LAN.
