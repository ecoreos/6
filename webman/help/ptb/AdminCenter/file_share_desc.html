Pasta compartilhada

As pastas compartilhadas são os diretórios básicos onde você pode armazenar arquivos e pastas em seu DiskStation. Antes de armazenar quaisquer arquivos de seu DiskStation, será necessário criar pelo menos uma pasta compartilhada. Os dados armazenados em pastas compartilhadas podem ser mantidos privados ou compartilhados com usuários ou grupos específicos, de acordo com as permissões de acesso personalizáveis. Além disso, as pastas compartilhadas podem ser criptografadas durante a criação, incluindo uma camada adicional de proteção.

Pastas compartilhadas padrão

Abaixo está uma lista de pastas compartilhadas que são criadas automaticamente quando certos aplicativos, serviços ou pacotes são instalados e ativados.

Nome Descrição
public A pasta public é criada automaticamente depois que seu DiskStation é configurado pela primeira vez.*
web A pasta web é o local de armazenamento padrão para conteúdo de um site hospedado no DiskStation. Ela é criada automaticamente quando Web Station é ativado.
photo A pasta photo é usada para armazenar fotos e vídeos que você pode compartilhar no Photo Station. Ela é criada automaticamente quando Photo Station ou DLNA/UPnP Media Server são ativados.
music A pasta music é usada para armazenar músicas que você deseja ouvir com o Audio Station. Ela é criada automaticamente quando Audio Station é ativado.
video A pasta video é usada para armazenar arquivos de vídeo que você quer pesquisar através de dispositivos DLNA/UPnP. Ela é criada automaticamente quando Media Server é ativado.
surveillance A pasta surveillance é usada para armazenar gravações do Surveillance Station. Ela é criada automaticamente quando Surveillance Station é ativado. Ela é uma pasta somente leitura e só pode ser acessada pelo administrador do sistema.
home As pastas home fornecem espaços particulares para que cada usuário armazene dados onde os outros usuários não terão acesso. Ela é criada automaticamente quando Base do usuário é ativado.
homes A pasta homes contém as pastas de páginas iniciais de todos os usuários. Ela é criada automaticamente quando Base do usuário é ativado. Somente o administrador do sistema pode acessar e gerenciar as pastas base de todos os usuários.
NetBackup A pasta NetBackup é criada automaticamente quando o Backup de rede é ativado.
usbshare A pasta usbshare[number] é criada automaticamente quando você conecta um dispositivo de unidade externa USB à porta USB do DiskStation.*
satashare A pasta satashare é criada automaticamente quando você conecta um dispositivo de unidade externa eSATA à porta eSATA do DiskStation.*

____
* Disponível apenas em determinados modelos.
