WebDAV

O WebDAV é uma extensão do HTTP projetada para permitir que os usuários editem e gerenciem arquivos armazenados em servidores remotos. Quando o serviço WebDAV estiver habilitado, os programas do cliente que suportam WebDAV, tais como certos aplicativos Windows, Mac OS Finder, Navegador de arquivos Linux, poderão acessar remotamente o DiskStation como se estivessem acessando uma unidade de rede local.

Para ativar WebDAV:

  1. Marque a caixa Habilitar WebDAV ou Habilitar conexão WebDAV HTTPS se desejar habilitar as conexões WebDAV através do HTTPS. A porta padrão para WebDAV é 5005 e WebDAV HTTPS é 5006.
  2. Clique em Aplicar

Limite de velocidade

A habilitação do limite de velocidade permite que você controle a largura de banda máxima utilizável para grupos ou usuários específicos durante a transferência de arquivos através do WebDAV.

Para ativar o limite de velocidade:

  1. Marque a caixa Ativar limite de velocidade ou Ativar limite de velocidade de acordo com uma programação se deseja implementar um limite de velocidade apenas em momentos específicos. Esta é uma configuração para todo o servidor e anulará as configurações de usuários e grupos individuais. Escolha Nenhum limite de velocidade para desativar as configurações de limite de velocidade de todos os usuários e grupos.
  2. Clique em Configurações do limite de velocidade para modificar as configurações para usuários ou grupos individuais. Para cada usuário e grupo, você pode selecionar uma das seguintes opções:
    • Aplicar configurações de grupo: As configurações do limite de velocidade serão iguais às do grupo ao qual o usuário pertence.
    • Nenhum: Nenhum limite de velocidade será aplicado quando os membros do grupo transferirem arquivos com o serviço.
    • Constante: Um limite de velocidade fixo e constante será aplicado ao usuário ou membros do grupo. Os limites de velocidade de upload e download podem ser especificados nos campos à esquerda.
    • Variável: Dois limites de velocidade diferentes podem ser especificados e aplicados ao usuário ou aos membros de grupo, de acordo com uma programação. Clique no botão Personalizar para modificar as configurações de limite de velocidade e também uma programação.
  3. Clique em Aplicar.

Nota:

  • Para obter mais detalhes sobre a dinâmica do limite de velocidade do grupo, clique aqui.
  • Ao configurar o limite de velocidade para um usuário, a seção Resultado exibe o limite de velocidade real do usuário de acordo com as configurações individuais dele e as do grupo. Por exemplo, se o usuário "Achilles" tiver um limite de download individual de 100 KB/s, mas pertence a um grupo com um limite de 500 KB/s, a seção Resultados mostrará "0/500".

Configurações avançadas

Clicar no botão Configurações avançadas permite a modificação das seguintes opções:

  • Habilitar Anonymous WebDAV: Permite que os clientes acessem as pastas compartilhadas usando o nome de usuário "anonymous" ao se conectar através do WebDAV. Certifique-se de que o usuário "Anonymous FTP/WebDAV" tenha recebido os privilégios de acesso adequados na página Pasta compartilhada.
  • Habilitar DavDepthInfinity: Ao usar a ferramenta do cliente, isso permite que os usuários acessem ou excluam arquivos sem restrições de tamanho de caminho do arquivo.
  • Habilitar registro de WebDAV: Essas opções registram os eventos relacionados ao WebDAV. Clique em Exibir registros e você verá os registros na janela pop-up.

Uso do WebDAV com sistemas operacionais diferentes

Quando o WebDAV tiver sido habilitado em seu DiskStation, siga as instruções abaixo para começar a acessar arquivos com dispositivos Windows, Mac, Linux ou alguns aplicativos para smarphones da Synology.

Nota:

  • Ao se conectar com os serviços WebDAV, sugerimos o uso de um cliente WebDAV com suporte UTF-8 para evitar erros de codificação.

Para Windows

Para usuários Windows que gostariam de usar o protocolo WebDAV para acessar arquivos em produtos Synology, sugerimos o uso do BitKinex, um aplicativo de mapeamento de unidades grátis para uso não comercial. Alternativamente, existem vários outros aplicativos de clientes WebDAV disponíveis na Internet.

Nota:

  • Se você desejar usar Mapear unidade de rede do Windows para acessar arquivos em seu DiskStation através do WebDAV, obtenha um certificado SSL válido e acesse seu DiskStation através de uma conexão HTTPS. Ao usar o serviço WebDAV em ambiente Windows, se quiser mudar para outra conta de usuário, precisará antes fazer logout do sistema Windows.

Para Mac

Para usuários Mac, você pode conectar-se ao seu DiskStation através do WebDAV simplesmente usando o Finder.

Para usar WebDAV no Mac:

  1. Em seu computador Mac, vá para Finder > Conectar ao servidor.
  2. No campo Endereço do Servidor, digite o endereço IP do seu DiskStation ou o nome do domínio precedido por "http://" e seguido por ":5005" (ou qualquer número de porta que você especifique durante a habilitação do serviço WebDAV). Por exemplo: http://pm.synology.com:5005/.
  3. Digite o seu nome de usuário e a senha DSM.
  4. Clique em Conectar.

Nota:

  • WebDAV suporta Bonjour broadcast.

Para Linux

Para usuários Ubuntu ou outros usuários Linux, siga as instruções abaixo para acessar os arquivos em seu DiskStation através do WebDAV.

Para usar WebDAV no Ubuntu:

  1. Clique em Locais > Conectar ao servidor.
  2. Selecione WebDAV (HTTP) ou WebDAV (HTTPS) no menu suspenso.
  3. Digite as informações necessárias, incluindo localização do servidor, porta, pasta e conta do usuário. Clique em Conectar para continuar.
  4. Digite sua senha e clique em Conectar.

Para usar WebDAV pela linha de comando Linux:

Se desejar usar a linha de comando, é possível usar o cadaver, um software cliente do WebDAV com linha de comando para configurar o acesso.

Assinatura de calendários com Clientes CalDAV

Você pode usar clientes CalDAV no computador ou dispositivo móvel para assinar calendários armazenados no DiskStation e gerenciar eventos, notas ou tarefas a fazer de calendários. Exemplos de clientes de calendário incluem Sunbird (Windows, Mac, Linux), Chandler (Windows, Mac, Linux), Evolution (Linux), ou iCal (dispositivo Mac, iOS).

Para ativar CalDAV:

  1. Habilite a conexão WebDAV ou WebDAV HTTPS primeiro.
  2. Marque Ativar CalDAV.
  3. Clique em Aplicar.

Para gerenciar a lista de calendários:

  1. Clique em Ver lista de calendário.
  2. Na janela que aparece, faça uma das seguintes opções:
    • Se quiser criar um novo calendário, clique em Adicionar, nomeie o novo calendário (por exemplo, "cal1") e especifique o destino onde o calendário será salvo (por exemplo, a pasta compartilhada "calendar") e clique em OK.
    • Se desejar alterar o nome e o destino do calendário, selecione-o na lista de calendários, clique em Editar para fazer as alterações e clique em OK.
    • Se desejar excluir o calendário do DiskStation, selecione-o na lista de calendários e clique em Excluir.
  3. Clique em Fechar.

Nota:

  • Os calendários armazenados no DiskStation serão exibidos como uma pasta no DiskStation.

Para assinar seu calendário com um aplicativo cliente (usando Sunbird como exemplo):

  1. Faça Download e instale o Sunbird no computador.
  2. Vá para Arquivo > Assinar calendário remoto.
  3. Na janela que aparece, escolha Na Rede e clique em Avançar.
  4. Escolha CalDAV, digite http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name no campo Localização (por exemplo: "http://demo.no-ip.info:5005/calendar/cal1") e clique em Avançar.
  5. Dê um nome ao calendário e clique em Avançar.
  6. Digite seu nome de usuário e senha do DSM para autenticação, clique em OK e clique em Concluído.

Para assinar seu calendário com um dispositivo móvel (usando o dispositivo iOS como exemplo):

  1. No seu dispositivo iOS, vá para Configurações > Mail, Contatos, Calendários.
  2. Em Contas, vá para Adicionar Conta > Outros e toque em Adicionar Conta CalDAV.
  3. Nos campos, digite o endereço IP (ou nome de domínio) de DiskStation e suas credenciais de usuário e toque em Avançar. Se surgir uma mensagem, toque em Continuar.
  4. Toque em Configurações Avançadas e faça o seguinte:
    1. Digite 5005 (ou qualquer outro número de porta que tenha especificado ao habilitar WebDAV) no campo Porta.
    2. Modifique a URL no campo URL da Conta para http://DiskStation_DDNS_Hostname:WebDAV_Port_Number/Calendar_Location/Calendar_Name (por exemplo: "http://demo.no-ip.info:5005/calendar/cal1").
    3. Toque em CalDAV.
  5. Toque em Avançar para verificar as informações da sua conta e toque em Salvar para concluir a configuração da conta.
  6. Volte à tela inicial do seu dispositivo iOS e abra Calendário para gerenciar seu calendário.

Nota:

  • Para obter mais informações sobre como gerenciar eventos, notas ou tarefas a fazer de calendários, consulte os documentos da Ajuda de cada aplicativo cliente.
