Idioma

Você pode selecionar seu idioma preferido para a interface de usuário do DSM, Notificações da área de trabalho do Google Chrome, e-mail, SMS, notificações de mensagens instantâneas e página de código do aplicativo.

Idioma de exibição

É possível selecionar seu idioma preferido para exibição para DSM e notificações da área de trabalho do Google Chrome ou usar o mesmo que a configuração padrão do navegador.

Idioma de notificação

Você pode definir seu idioma preferido para notificações de e-mail, SMS, Windows Live Messenger e Skype. Ele também será aplicado às notificações recebidas no Windows Phone.

Entretanto, o idioma de notificação em um dispositivo iOS ou Android depende da configuração de idioma de seu dispositivo móvel.

Página de código

Para aplicativos que convertam strings não Unicode para Unicode, como o serviço FTP, suporte a UPnP ou indexação de Metadata de músicas, você terá de escolher a página de código apropriada no menu suspenso Página de código.
