Agendador de tarefas

O agendador de tarefas permite que você programa e execute serviços ou scripts em horários predefinidos.

Tarefas suportadas

As tarefas podem ser criadas para fazer o seguinte:

  • Script definido pelo usuário: Execute os scripts ou comandos definidos pelo usuário.
  • Lixeira: Esvazie a lixeira das pastas compartilhadas especificadas. Além disso as configurações da Política de retenção permitem controlar quais arquivos são excluídos permanentemente ao esvaziar a lixeira. Por exemplo, eles podem ser excluídos ou mantidos de acordo com tamanho, tempo de exclusão ou formato de arquivo.
  • Controle do bipe: Faça o DiskStation soar um bipe.
  • Serviço: Inicie ou interrompa serviços de rede como Serviço de arquivo do Windows, FTP, Bloqueio automático, etc.
  • Script definido pelo usuário: Execute os scripts ou comandos definidos pelo usuário.
  • Lixeira: Esvazie a lixeira das pastas compartilhadas especificadas. Além disso as configurações da Política de retenção permitem controlar quais arquivos são excluídos permanentemente ao esvaziar a lixeira. Por exemplo, eles podem ser excluídos ou mantidos de acordo com tamanho, tempo de exclusão ou formato de arquivo.
  • Serviço: Inicie ou interrompa serviços de rede como Serviço de arquivo do Windows, FTP, Bloqueio automático, etc.

Gerenciamento de tarefas

A página principal do Agendador de tarefas exibe todas as tarefas existentes e fornece opções para criar, editar ou executar tarefas.

Para criar uma tarefa programada:

As tarefas programadas são tarefas executadas em uma data, horário ou intervalo especificado.

  1. Vá para Painel de controle > Agendador de tarefas, clique em Criar e selecione Tarefa agendada.
  2. Selecione o tipo desejado de tarefas a serem agendadas: Script definido pelo usuário, Lixeira, Controle de bipes ou Serviço.
  3. Quando a janela Criar tarefa aparecer, clique em Geral e modifique as configurações a seguir:
    • Configurações gerais: Digite o nome de uma tarefa, selecione um usuário para o qual a tarefa será executada e habilite ou desabilite a tarefa.
  4. Clique em Agendamento e modifique as seguintes configurações:
    • Data: Especifique quando executar a tarefa. A execução das tarefas pode ser agendada em certos dias da semana (ex.: diariamente, finais de semana ou todo domingo, etc.) ou em datas específicas (ex.: 21/10/2015).
    • Vezes: Especifique em quantas horas as tarefas serão executadas a cada dia, bem como a primeira e a última hora de execução.
  5. Clique em Configurações da tarefa para organizar as configurações específicas da tarefa (apenas para algumas tarefas).
    • Para receber detalhes de execução da tarefa, selecione Enviar detalhes de execução por e-mail.
    • Para ser notificado apenas quando ocorrerem anormalidades, selecione Enviar detalhes de execução apenas quando o script terminar de modo anormal.
    • Para personalizar o conteúdo da tarefa, digite o script em Script definido pelo usuário.
    • Se você selecionou Lixeira ou Serviço na Etapa 2, poderá decidir como e quando as lixeiras serão limpas ou qual serviço iniciar/parar.
  6. Clique em OK para salvar as configurações.

Para criar uma tarefa disparada:

Tarefas disparadas são tarefas executadas antes/após a conclusão de eventos específicos.

  1. Vá para Painel de controle > Agendador de tarefas, clique em Criar e selecione Tarefa disparada.
  2. Script Script definido pelo usuário.
  3. Quando a janela Criar tarefa aparecer, clique em Geral e modifique as configurações a seguir:
    • Configurações gerais: Digite o nome de uma tarefa, selecione um usuário para o qual a tarefa será executada, selecione um evento/pré-tarefa que dispara a tarefa e habilite ou desabilite a tarefa.
  4. Clique em Configurações da tarefa para organizar as configurações específicas da tarefa (apenas para algumas tarefas).
    • Para receber detalhes de execução da tarefa, selecione Enviar detalhes de execução por e-mail.
    • Para ser notificado apenas quando ocorrerem anormalidades, selecione Enviar detalhes de execução apenas quando o script terminar de modo anormal.
    • Para personalizar o conteúdo da tarefa, digite o script em Script definido pelo usuário.
  5. Clique em OK para salvar as configurações.

Nota:

  • Ao criar uma tarefa disparada, pode ser necessário especificar suas pré-tarefas. A tarefa criada será executada apenas após a execução da pré-tarefa.
  • Após habilitar Enviar detalhes de execução apenas quando o script terminar de modo anormal, o sistema não entregará os detalhes se o script terminar com 0.

Para editar as configurações da tarefa:

  1. Selecione uma tarefa existente.
  2. Clique em Editar.
  3. Aparece uma janela em que você pode modificar as configurações da tarefa.

Para excluir uma tarefa:

  1. Selecione uma tarefa existente.
  2. Clique em Ação > Excluir.

Para habilitar ou desabilitar uma tarefa:

  1. Na página principal Agendador de tarefas, marque ou desmarque a caixa de seleção adequada em Habilitado para habilitar ou desabilitar uma tarefa.
  2. Clique em Salvar.

Para executar manualmente uma tarefa:

  1. Selecione uma ou mais tarefas existentes.
  2. Clique em Executar.

Limitações do agendador de tarefas:

  • Os nomes das tarefas podem ter 512 caracteres exibíveis.
  • Número máximo de tarefas: 1024.
  • Comprimento máximo do caminho do script ou linha de comando: 4095.
  • A duração do bipe pode ser definida de 1 a 600 segundos.
