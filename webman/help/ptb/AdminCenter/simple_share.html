Pasta compartilhada

As pastas compartilhadas são os diretórios básicos onde você pode armazenar arquivos e pastas em seu DiskStation. Antes de armazenar arquivos em seu DiskStation, é preciso criar pelo menos uma pasta compartilhada no DSM. É possível manter os dados armazenados nas pastas compartilhadas privados ou compartilhados com usuários específicos ou grupos de usuários de acordo com permissões de acesso personalizáveis.

Veja a seguir uma lista das pastas compartilhadas padrão que são criadas automaticamente quando determinados aplicativos, serviços ou pacotes são instalados ou ativados.

Nome Descrição
public A pasta public é criada automaticamente depois que o DiskStation é configurado pela primeira vez.*
web A pasta web é o local de armazenamento padrão para o conteúdo de um site hospedado no DiskStation. Ela é criada automaticamente quando o Web Station é ativado.
photo A pasta photo é usada para armazenar fotos e vídeos que você deseja compartilhar no Photo Station. Ela é criada automaticamente quando o Photo Station ou o Servidor de Mídia DLNA/UPnP é ativado.
music A pasta music é usada para armazenar músicas que você deseja ouvir no Audio Station. Ela é criada automaticamente quando o Audio Station ou o Servidor de Mídia DLNA/UPnP é ativado.
video A pasta video é usada para armazenar arquivos de vídeo pelos quais você deseja navegar em dispositivos DLNA/UPnP. Ela é criada automaticamente quando o Video Station ou o Servidor de Mídia DLNA/UPnP é ativado.
surveillance A pasta surveillance é usada para armazenar gravações do Surveillance Station. Ela é criada automaticamente quando o Surveillance Station é ativado. Ela é uma pasta somente leitura e só pode ser acessada pelo administrador do sistema.
home A pasta home oferece espaços particulares para cada usuário armazenar dados que somente ele pode acessar. Ela é criada automaticamente quando a Base do usuário é ativada.
homes A pasta homes contém as pastas de páginas iniciais de todos os usuários. Ela é criada automaticamente quando a Base do usuário é ativada. Somente o administrador do sistema pode acessar e gerenciar as pastas base de todos os usuários.
NetBackup A pasta NetBackup é criada automaticamente quando o Backup de Rede é ativado.
usbshare A pasta usbshare[número] é criada automaticamente quando você conecta um dispositivo de unidade externa USB à porta USB do DiskStation.*
satashare A pasta satashare é criada automaticamente quando você conecta um dispositivo de unidade externa eSATA à porta eSATA do DiskStation.*

Observação:

  • Você só pode editar as permissões de pasta compartilhada dos usuários no modo simples. Se você desejar criar grupos de usuários ou gerenciar os privilégios de acesso de grupos às pastas compartilhadas, vá para Painel de controle (modo avançado) > Grupo.

Para criar uma pasta compartilhada:

  1. Vá para Painel de controle (modo simples) > Pasta compartilhada e clique no ícone + no painel esquerdo.
  2. Especifique um nome e selecione o volume no qual a pasta compartilhada será criada.
  3. Marque Habilitar lixeira se necessário. Se essa opção for ativada, quando os arquivos na pasta compartilhada forem excluídos, eles serão movidos para uma pasta chamada #recycle. O acesso à lixeira pode ser limitado a usuários que pertencem ao grupo administrators.
  4. Marque ou desmarque as caixas correspondentes a cada usuário para personalizar suas permissões de acesso à pasta compartilhada:
    • Leitura/Gravação: O usuário pode acessar e fazer alterações aos arquivos e sub pastas na pasta compartilhada.
    • Somente leitura: O usuário pode acessar os arquivos e as subpastas na pasta compartilhada.
    • Nenhum acesso: O usuário não pode acessar os arquivos ou subpastas na pasta compartilhada.
  5. Clique em Aplicar para salvar as configurações.

Observação:

  • Os nomes das pastas compartilhadas não diferenciam maiúsculas de minúsculas e podem ter de 1 a 32 caracteres. Os nomes não podem ter os seguintes caracteres:
    ! " # $ % & ' ( ) * + , / : ; <= > ? @ [ ] \ ^ ` { } | ~
  • Os seguintes nomes são reservados para uso do sistema:
    ., .., global, home, homes, printers, public, satashare, surveillance, usbbackup, usbshare
  • O primeiro caractere do nome de uma pasta compartilhada não pode ser um sinal de menos ou um espaço, e o último caractere não pode ser um espaço.
  • Quando houver conflitos de permissão, a prioridade de permissões será a seguinte: Nenhum acesso > Leitura/Gravação > Somente leitura.
  • Ao criar uma nova pasta compartilhada, se as permissões dos usuários pertencentes ao grupo administrators estiverem definidas como Nenhum acesso, esses usuários só poderão ver as pastas compartilhadas em Painel de controle > Pasta compartilhada.
  • É possível ajustar as configurações de privilégio da pasta Lixeira no File Station clicando com o botão direito do mouse na pasta #recycle. Selecione Propriedades e clique na guia Permissões.

Para editar uma pasta compartilhada:

  1. Vá para Painel de controle > Pasta compartilhada.
  2. Selecione a pasta compartilhada desejada no painel esquerdo.
  3. Edite as informações gerais da pasta compartilhada, como nome, local e permissões de usuário.
  4. Se a opção Habilitar lixeira já estiver marcada, você pode clicar em Esvaziar lixeira para remover todos os arquivos excluídos da lixeira permanentemente.
  5. Clique em Aplicar para salvar as configurações.

Para excluir uma pasta compartilhada:

  1. Vá para Painel de controle > Pasta compartilhada.
  2. Selecione a pasta compartilhada desejada no painel esquerdo.
  3. Clique no ícone Lixo no painel esquerdo para excluir a pasta compartilhada selecionada.

* Disponível apenas em certos modelos.
