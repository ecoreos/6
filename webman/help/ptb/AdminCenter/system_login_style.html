Estilo de logon

É possível personalizar a aparência da tela de login do DSM escolhendo a partir de uma variedade de modelos, alterando a imagem de fundo ou adicionando um logotipo. Para fazer isso, vá para Painel de controle > Estilo de logon.

Para personalizar a tela de logon do DSM:

As opções a seguir aparecem na página Estilo de logon:

  • Título da página de logon: O texto inserido aqui aparece na tela de logon.
  • Informações de tempo: Selecione para mostrar ou ocultar informações meteorológicas locais na tela de logon.
  • Modelo: Selecione uma imagem de fundo e esquema de cores padrão. O modelo escolhido terá efeito na próxima vez que você entrar no DiskStation.
  • Fundo: Para alterar a imagem de fundo que aparece na tela de logon, clique na imagem existente ou no sinal de mais (+) se nenhuma imagem estiver selecionada. Você pode escolher entre papéis de parede padrão, imagens armazenadas no DiskStation ou imagens utilizadas anteriormente. Apenas os formatos de arquivo de imagem JPE, JPEG, JPG, BMP, GIF ou PNG são suportados.
  • Logotipo: Para adicionar um logotipo na tela de logon, clique no logotipo existe ou no sinal de mais (+) se nenhuma imagem estiver selecionada. Você pode carregar uma imagem menor do que 1 MB, escolher entre as imagens armazenadas no DiskStation ou escolher uma imagem usada anteriormente.
