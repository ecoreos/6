Serviço de arquivos Mac

Em Painel de Controle > Serviços de arquivos > Win/Mac/NFS, você pode ativar o serviço de arquivos Mac para tornar DiskStation um servidor de arquivos Mac (servidor AFP).

Para ativar o serviço de arquivos Mac:

  1. Marque Ativar serviço de arquivos Mac para tornar o DiskStation um servidor de arquivos Mac (servidor AFP).
  2. Configure a pasta de backup Time Machine se necessário. (Consulte a seção abaixo para mais informações.)
  3. Clique em Aplicar.

Nota:

  • Para fazer o backup de dados do Time Machine para o DiskStation, selecione uma pasta compartilhada no menu suspenso.
  • As pastas compartilhadas que estão sendo usadas por outros serviços não estarão disponíveis para o Time Machine.

Ativação do registro de transferência

Marcar essa opção habilita o registro das operações de acesso a arquivos, incluindo criação, exclusão, leitura, gravação, transferência e renomeação de arquivos ou pastas através do AFP, também conhecido como protocolo de arquivamento Apple.

Configurações avançadas

Aplique as permissões UNIX padrão:

Ativar essa opção aplica as permissões UNIX padrão ao carregar ou criar arquivos e pastas. As permissões aplicadas são as mesmas das permissões aplicadas pelo comando UNIX umask. Quando essa opção está habilitada, as permissões UNIX são 644 para arquivos e 755 para pastas. Quando essa opção está desabilitada, as permissões UNIX são 666 para arquivos e 777 para pastas. O valor umask padrão é 022.

Nota:

Habilitar essa opção pode causar problemas de permissão inconsistente entre os diferentes protocolos. Para evitar inconsistências, sugerimos deixar essa opção desabilitada.

Liberar recursos imediatamente após a desconexão:

A habilitação dessa opção libera recursos imediatamente quando a conexão do servidor AFP é desconectada inesperadamente. Se você receber mensagens de erro "Volume em uso" ao usar o Time Machine, habilite essa opção.

Para ativar o Bonjour Printer Broadcast:

Marque Ativar Bonjour Printer Broadcast para que os clientes Mac pesquisem e localizem a impressora conectada ao DiskStation.

Para pesquisar conteúdos de arquivo através do Spotlight no Finder:

Após as pastas serem totalmente indexadas e montadas em seu computador Mac, você pode usar o Spotlight para pesquisar os arquivos dentro dela e seu conteúdo.

  1. Vá para Painel de controle > Serviço de indexação > Indexação de arquivo > Lista de pastas indexadas > Criar para indexar pastas específicas e os arquivos dentro delas por tipo de arquivo.
  2. Clique em OK para executar a indexação.
  3. Monte as pastas indexadas no seu computador Mac.
  4. No seu computador Mac, clique na pasta montada desejada no painel esquerdo do Finder e use o Spotlight no Finder para pesquisar os arquivos dentro dela.

Nota:

  • Para pesquisar nas pastas montadas, use o Spotlight no Finder. O Spotlight na mesa do Mac só permite a pesquisa de arquivos locais.
  • No Mac OS X 10.9 e versões posteriores, o Spotlight no Finder suporta a pesquisa em pastas montadas por nome da marca e categoria.
