SDCopy*

Você pode selecionar e definir um local alvo para a função SDCopy.

Para ativar SDCopy:

  1. Marque Ativar SDCopy.
  2. Selecione a pasta compartilhada de destino no menu suspenso.
  3. Clique em OK.

Ao conectar um dispositivo SD na ranhura SD no painel frontal do Synology DiskStation, você pode pressionar o botão Copiar no painel frontal para copiar os dados no dispositivo SD para a pasta compartilhada de destino atribuída aqui. Se a pasta de destino do SDCopy for deletada, o sistema irá substituí-la pela pasta compartilhada pública automaticamente.

    Limitações:
  1. Somente são suportados dispositivos SD no formato FAT ou NTFS.
  2. Os dispositivos SD devem estar conectados à ranhura SD no painel frontal do Synology DiskStation.
  3. O espaço disponível da pasta compartilhada de destino deve ser maior que o espaço usado no dispositivo SD.

____
* Disponível apenas em modelos específicos.
