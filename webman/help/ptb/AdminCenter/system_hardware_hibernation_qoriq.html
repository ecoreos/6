Hibernação

Você pode especificar o tempo em que os discos rígidos permanecem ociosos antes de entrar no modo de hibernação do HDD. Durante a hibernação do HDD, os discos rígidos param de girar, reduzindo o consumo de energia e estendendo a vida útil dos discos rígidos. É possível especificar períodos diferentes para discos rígidos internos e discos rígidos externos eSATA/USB.

Observação:

  • A hibernação do HDD é interrompida no acesso.
  • Apenas alguns dispositivos de armazenamento USB externo suportam a hibernação do HDD.

Por que meu DiskStation não hiberna?

Observe que os serviços a seguir impedirão que o DiskStation entre no modo de hibernação. Além disso, o sistema sairá do modo de hibernação dois minutos antes de qualquer tarefa agendada.

Pacotes:

  • Download Station: o DiskStation não pode hibernar se o Download Station estiver executando downloads agendados de RSS, eMule ou tarefas ativas.
  • Mail Station: O DiskStation não poderá hibernar se o Mail Station estiver buscando do POP3.
  • Surveillance Station: O DiskStation não pode hibernar se o Surveillance Station estiver gravando.
  • Servidor de mídia: O DiskStation não pode hibernar se o registro DMA estiver habilitado em configurações do Servidor de mídia (Vá para Menu principal > Servidor de mídia > Servidor de mídia > Diagnóstico).
  • Directory Server
  • Webalizer
  • Logitech® Media Server
  • Plex Media Server
  • Outro software de terceiro ou optware como o SABnzbd

Processos:

  • Miniaturas e indexação: O DiskStation não pode hibernar ao gerar miniaturas ou reindexar arquivos multimídia após a atualização. Durante esse tempo, o processo “synoindexd” será indicado como “Em execução” (Vá para Menu principal > Monitor de recursos > Processo).
  • Local Master Browser: O DiskStation não pode hibernar se o Local Master Browser estiver habilitado (Vá para Painel de Controle > Win/Mac/NFS > Serviço de Arquivos do Windows > Habilitar Local Master Browser).
  • Windows Media Player: O DiskStation não pode hibernar se o serviço de compartilhamento de rede do Windows Media Player estiver habilitado na LAN.
  • O DiskStation não pode hibernar se o número de processos concomitantes exigir troca, em que a capacidade de memória RAM tenha sido ultrapassada e os HDDs sejam usados temporariamente para operações de leitura/gravação.

Outros

  • Dispositivos USB: O DiskStation não pode entrar na Hibernação do sistema se algum dispositivo USB estiver conectado à unidade.

Hibernação do sistema

Você pode definir o DiskStation para entrar em hibernação do sistema um minuto após os discos rígidos iniciarem a hibernação. A hibernação do sistema reduz o consumo de energia do DiskStation.

Durante o modo de hibernação do sistema:

  1. O botão Liga/Desliga entra no modo intermitente (ciclo liga/desliga lento). Os outros LEDs apagam.
  2. A energia dos discos rígidos é desligada.
  3. A CPU para.
  4. A ventoinha externa para.
  5. Os LEDs das portas LAN desligam.

Observação:

  • Qualquer pacote de transmissão SMB/CIFS pode impedir que o servidor entre na hibernação do sistema, incluindo qualquer navegador de arquivos do Windows funcionando na rede. Caso você deseje ignorar os pacotes de transmissão do CIFS ao entrar na hibernação do sistema, selecione Ignorar os pacotes de transmissão do Windows Explorer. Esteja ciente de que quando essa opção for selecionada, os pacotes SMB/CIFS não poderão ser usados para sair da hibernação do sistema. A conexão de qualquer dispositivo USB também impedirá que o servidor entre na hibernação do sistema.

Sono profundo

O sono profundo para unidades de expansão pode ser habilitado através da interface emparelhada do DiskStation. Você pode configurar a unidade de expansão e o DiskStation emparelhado para ficar ocioso por períodos diferentes. Portanto, as unidades de expansão podem ser configuradas para entrar em sono profundo enquanto o DiskStation emparelhado permanece on-line.

Durante o modo sono profundo:

  • Todos os LEDs da unidade de expansão serão apagados.
  • A energia dos discos rígidos da unidade expansão será cortada.
  • A ventoinha da unidade de expansão parará.

Observação:

  • O sono profundo é interrompido no acesso.
  • Para garantir a operação normal ao usar a função sono profundo na unidade de expansão, deslize o interruptor Auto/Manual do painel traseiro para Auto.

Desligamento automático

Você pode especificar o período de tempo durante o qual o disco interno irá permanece em estado de hibernação antes de seu DiskStation desligar automaticamente.

Observação:

  • Você deve habilitar a hibernação do HDD interno primeiro.
  • Você deve habilitar o serviço WOL primeiro para que você possa acionar remotamente o seu DiskStation.
