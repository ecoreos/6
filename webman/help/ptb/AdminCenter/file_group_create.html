Criar grupo

  1. Vá até Painel de controle > ï»¿Grupo.
  2. Clique em Criar. Essa ação abre o Assistente de criação de usuário.
  3. Digite o nome e uma descrição para o grupo. (Consulte a seção abaixo para mais informações). Clique em Avançar.
  4. Siga as demais etapas do assistente para finalizar a criação do grupo.
  5. Clique em Aplicar para concluir.

Convenções de nomes:

  • O nome de grupo diferencia maiúsculas e minúsculas e pode conter de 1 a 32 caracteres Unicode, com exceção dos seguintes símbolos:

    • ! " $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~

  • O primeiro caractere não pode ser um sinal de menos nem um espaço e o último caractere não pode ser um espaço.
  • A descrição do grupo diferencia maiúsculas e minúsculas e pode ter até 64 caracteres Unicode visíveis.
