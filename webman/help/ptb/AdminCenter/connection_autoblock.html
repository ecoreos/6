Bloqueio automático

O bloqueio automático permite evitar o logon não autorizado. Após ativar o serviço, se um endereço IP tiver muitas falhas em tentativas de logon ele será bloqueado. Falhas de tentativa de login via SSH, Telnet, rsync, Backup de rede, Sincronização de pasta compartilhada, FTP, WebDAV, aplicativos móveis Synology, File Station ou DSM irão se somar. Você também pode ter uma Lista permitida que inclua os endereços IP que nunca serão bloqueados.

Para configurar o bloqueio automático:

  1. Vá para a guia Geral e marque Habilitar bloqueio automático.
  2. Digite um número para Tentativas de logon e Dentro de (minutos) para bloquear um endereço IP depois do número definido de falhas em tentativas de logon dentro do número de minutos especificado.
  3. Marque Ativar expiração do bloqueio e digite um número para remover um endereço IP bloqueado depois do número de dias especificado. Endereços IP bloqueados permanecerão indefinidamente na lista de bloqueio até que seja removido manualmente da lista, caso esta opção não seja selecionada ou se Desbloquear depois de (dias) for definido como 0.
  4. Clique em Aplicar.

Para adicionar endereços IP bloqueados:

Clique na guia Lista de bloqueio e selecione um dos seguintes no menu suspenso Criar:

  • Adicionar endereço IP

    • Digite um endereço IP e defina o tempo de expiração.

  • Importar lista de endereço IP

    • Importe um arquivo texto que liste os endereços IP para adicionar um número de endereços IP e definir o tempo de expiração ao mesmo tempo. Marque a caixa de seleção adequada se desejar sobrescrever os endereços IP existentes. Se essa opção não estiver selecionada, os endereços IP duplicados serão ignorados.

    Para importar um arquivo, ele deve atender os seguintes critérios:

    • O arquivo deve ser um arquivo de texto sem formatação.
    • Cada linha do arquivo pode conter apenas um endereço IP.
    • As linhas de comentário começam com o símbolo #.

      • Se o arquivo selecionado tiver um formato correto, você verá todos os endereços IP na área de visualização. Clique em OK para importar.

Para remover endereços IP bloqueados:

  1. Clique na guia Lista de bloqueio para ver a lista de endereços IP bloqueados.
  2. Selecione os endereços IP que deseja remover da lista e clique em Remover.

Para adicionar os endereços IP permitidos:

Clique na guia Lista permitida e selecione um dos seguintes no menu suspenso Criar:

  • Adicionar endereço IP

    • Digitar um endereço de IP.

  • Importar lista de endereço IP

    • Importe um arquivo texto que liste os endereços IP para adicionar um número de endereços IP ao mesmo tempo. Marque a caixa de seleção adequada se desejar sobrescrever os endereços IP existentes. Se essa opção não estiver selecionada, os endereços IP duplicados serão ignorados.
