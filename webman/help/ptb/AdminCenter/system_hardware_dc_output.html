Saída CC*

Em Painel de controle > Hardware e Alimentação > Saída CC, você pode configurar o sistema para ligar ou desligar a saída CC automaticamente em um horário especificado.

Para adicionar um novo agendamento de saída ativada/desativada:

  1. Clique no botão Criar para abrir a caixa de diálogo Adicionar Agendamento.
  2. Escolha Saída ativada ou Saída desativada.
  3. Configure a Data e a Hora para o agendamento de saída ativada/desativada.
  4. Clique em OK.
  5. Clique em Salvar.

O novo agendamento será adicionado à lista de agendamentos.

Para ver o resumo do agendamento:

Clique em Resumo para procurar o agendamento semanal do agendamento de saída ativada/desativada.

Observação:

  • Você pode especificar até 100 agendamentos para um DiskStation.
  • Nos seguintes casos, o sistema pode não conseguir desligar a saída CC automaticamente:
    • O Gerenciador de armazenamento está executando tarefas.
    • O sistema está criptografando as pastas compartilhadas.
    • O sistema está executando o Secure Erase nos discos rígidos.
    • O DSM está sendo atualizado.
    • O Centro de pacotes está executando tarefas.
    • O sistema está atualizando banco de dados.
    • O sistema está executando tarefa de backup de dados.

____
* Disponível apenas em modelos específicos.
