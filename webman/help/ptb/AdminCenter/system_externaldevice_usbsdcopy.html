USBCopy

Você pode selecionar uma pasta de destino para a função USBCopy indo para Painel de Controle > Dispositivos Externos > USBCopy/SDCopy.

Para ativar o USBCopy:

  1. Marque Habilitar USBCopy.
  2. Selecione a pasta compartilhada de destino no menu suspenso.

Ao conectar um dispositivo USB (tal como uma câmera digital) à porta USB no painel frontal do produto Synology, você pode pressionar o botão Copiar no painel frontal para copiar os dados no dispositivo USB para a pasta compartilhada de destino atribuída aqui. Se a pasta de destino do USBCopy for excluída, o sistema irá substituí-la pela pasta compartilhada pública automaticamente.

Limitações

  1. São suportados apenas dispositivos USB nos formatos FAT, NTFS, ext3 e ext4.
  2. Os dispositivos USB devem estar conectados à porta USB no painel frontal do dispositivo DiskStation.
  3. Somente o administrador ou usuários no grupo administrators têm privilégios de acesso apropriados para a realização de tarefas de backup para dispositivos USB.
  4. O espaço disponível da pasta compartilhada de destino deve ser maior que o espaço usado no dispositivo USB.

SDCopy*

Você pode selecionar uma pasta de destino para a função SDCopy indo para Painel de Controle > Dispositivos Externos > USBCopy/SDCopy.

Para ativar SDCopy:

  1. Marque Habilitar SDCopy.
  2. Selecione a pasta compartilhada de destino no menu suspenso.

Ao conectar um dispositivo SD na ranhura SD no painel frontal do produto Synology, você pode pressionar o botão Copiar no painel frontal para copiar os dados no dispositivo SD para a pasta compartilhada de destino atribuída aqui. Se a pasta de destino do SDCopy for deletada, o sistema irá substituí-la pela pasta compartilhada pública automaticamente.

Limitações

  1. São suportados apenas dispositivos SD no formato FAT, NTFS, ext3 e ext4.
  2. Os dispositivos SD devem estar conectados à ranhura SD no painel frontal do produto Synology.
  3. Somente o administrador ou usuários no grupo administrators têm privilégios de acesso apropriados para a realização de tarefas de backup para dispositivos SD.
  4. O espaço disponível da pasta compartilhada de destino deve ser maior que o espaço usado no dispositivo SD.

* Disponível apenas em modelos específicos.
