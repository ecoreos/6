Segurança

Você pode modificar as seguintes configurações de segurança em Painel de controle > Segurança > Segurança para as sessões de navegação do DSM:

  • Temporizador de logout (minutos): Os usuários serão automaticamente desconectados do DSM se ficarem inativos pelo período especificado aqui. Digite qualquer valor entre 1 e 65535.
  • Melhorar a compatibilidade do navegador ignorando a verificação de IP: Se você acessar o DiskStation através de um proxy HTTP e encontrar encerramentos aleatórios de sessão, você pode ativar esta opção para ignorar a verificação de IP.
  • Aprimorar a proteção contra ataques de solicitações entre sites forjadas: Essa opção aumenta a proteção do sistema contra os ataques de scripts de entre sites. Ela surtirá efeito no seu próximo login no DSM.
  • Aprimore a segurança com o cabeçalho da Política de segurança de conteúdo (CSP) HTTP: Esta opção aumenta a segurança do sistema contra ataques de scripts entre sites (XSS) permitindo somente dados de fontes confiáveis e restringindo a execução de script embutido.
  • Não permitir que o DSM seja inserido com o iFrame: Você pode ativar essa opção para restringir outros sites de inserir o DSM em outras páginas da web com o iFrame, impedindo assim certos tipos de ataques de sites mal-intencionados. Sites específicos podem receber a permissão para inserir o DSM com o iFrame, clicando no botão Sites permitidos.
