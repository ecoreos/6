Permitir / Negar

O firewall integrado pode impedir logins não autorizados e acesso aos serviços de controle. Além disso, você pode escolher entre permitir ou negar acesso a certas portas de rede de endereços IP específicos.

Para criar uma regra de firewall:

  1. Vá para Painel de controle > Firewall e selecione a interface de rede da coluna da esquerda de acordo com seu tipo de conexão de rede.
  2. Clique em Criar.
  3. Na seção Portas, selecione uma das seguintes opções para a regra:
    • Escolha Todos para aplicar as regras a todas as portas.
    • Escolha Selecionar em uma lista de aplicativos integrados.
    • Escolha Personalizado e clique em Personalizado e digite até 15 portas separadas por vírgulas, ou especifique um intervalo de porta.
    • Você também pode especificar os protocolos.
  4. Na seção IP de origem, selecione uma das seguintes opções para a regra:
    • Escolha Todos para permitir ou negar a todos os endereços IP.
    • Digite um Endereço IP de Host único.
    • Digite uma Subrede.
  5. Na seção Ação, selecione uma das seguintes opções para a regra:
    • Selecione Permitir para permitir o acesso de acordo com as configurações de IP de origem e portas acima.
    • Selecione Negar para negar o acesso de acordo com as configurações de IP de origem e portas acima.

Você poderá ver a regra de firewall criada na lista de regras.

Para excluir as regras do firewall, faça o seguinte:

  • desmarque a caixa de seleção da regra na coluna Ativado da lista
  • Exclua a regra do firewall

Você também pode escolher Permitir acesso ou Negar acesso na parte inferior da lista de regras para permitir ou negar o acesso que não corresponda a nenhuma regra.

    Nota:
  1. Você pode criar até 100 regras para uma interface de rede e pode reordenar as regras da lista arrastando e soltando.
  2. As regras são priorizadas de acordo com as suas posições da lista.
  3. Se o sistema tem várias portas de rede conectadas à mesma sub rede, as regras de firewall podem não funcionar corretamente.*
  4. Quando se combinam múltiplas portas LAN com link aggregation, o firewall aplicará as regras da primeira interface de rede e reservará as regras da segunda interface de rede.

____
* Aplicável apenas para modelos específicos.
