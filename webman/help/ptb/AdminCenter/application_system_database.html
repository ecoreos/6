Banco de Dados do Sistema

DiskStation utiliza um banco de dados do sistema para acompanhar os dados do pacote e entradas de registro armazenadas em pastas compartilhadas criadas em seus dispositivos externos de armazenamento, como usbshare1.

Importante: Para acessar os dados do pacote armazenados em dispositivos de armazenamento externos conectado ao DiskStation, ou para visualizar registros de log do DSM em Menu Principal > Log Center, o banco de dados do sistema tem que ser ativado primeiro. Em caso contrário, todos os serviços relacionados serão inacessíveis.

Para habilitar o banco de dados do sistema:

  1. Marque Habilitar banco de dados do sistema.
  2. Execute um dos seguintes procedimentos:
    • Selecione Configuração automática se quiser que o DSM selecione automaticamente uma pasta compartilhada para o armazenamento do banco de dados do sistema. Se o dispositivo de armazenamento externo que contém a pasta compartilhada for removido, o DSM selecionará automaticamente outra pasta compartilhada para o armazenamento do banco de dados.
    • Selecione Mova o banco de dados atual para a pasta compartilhada selecionada e selecione no menu suspenso para mover o banco de dados atual para outra pasta compartilhada criado no dispositivo externo.
    • Selecione Selecione um dos bancos de dados existentes e selecione no menu suspenso um banco de dados em existência se houver mais de um dispositivo externo conectado ao DiskStation.
  3. Clique em Aplicar.

Aviso: Para evitar a corrupção do banco de dados, certifique-se de que o dispositivo de armazenamento foi ejetado em Menu Principal > Painel de Controle > Dispositivos Externos > Dispositivos Externos ou clicando no ícone de dispositivos externos no canto superior direito antes de removê-lo do DiskStation.
