Photo Station

Photo Station Pessoal é um álbum de fotos online com blog, propriedade de e gerenciado por um usuário do DSM. O DiskStation fornece a pasta home/photo onde você pode armazenar fotos e vídeos que queira compartilhar. O sistema criará miniaturas de índice das fotos e vídeos automaticamente, e as pessoas poderão ver os álbuns de fotos através de um navegador da web.

Para habilitar o Photo Station Pessoal:

Clique em Habilitar o Photo Station Pessoal - [nome de usuário] e clique em OK.

Para editar as configurações do Photo Station Pessoal:

  1. Marque Definir novas pastas como álbuns públicos se desejar que os álbuns de fotos carregados estejam disponíveis para visitantes.
  2. Clique em Reindexar para reconstruir a estrutura de indexação dos arquivos de imagem, se houver imagens faltando no Photo Station.
  3. Clique em OK.

Para iniciar o Photo Station, faça uma das seguintes opções:

  • Vá até Menu principal e clique em Photo Station.
  • Digite o endereço web a seguir na barra de endereço do navegador: http://Synology_Server_IP_Address/photo/

Para iniciar o Photo Station pessoal, faça uma das seguintes opções:

  • Clique em Iniciar Photo Station.
  • Vá até Menu Principal e clique em Photo Station - [nome do usuário].
  • Digite o endereço web a seguir na barra de endereço do navegador: http://Synology_Server_IP_Address/~username/photo/

Suporte e limitações:

  • Formatos de imagem suportados: BMP, JPEG, GIF, RAW (arw, srf, sr2, dcr, k25, kdc, cr2, crw, nef, mrw, ptx, pef, raf, raw, 3fr, erf, mef, mos, orf, rw2, dng, x3f), TIFF, PNG
  • Formatos de vídeo suportados: 3G2, 3GP, ASF, AVI, DAT, DivX, FLV, M4V, MOV, MP4, MPEG, MPG, MTS, M2TS, M2T QT, WMV, XviD, RM(1), RMVB(1), VOB(1)
  • Se você fizer upload de conteúdo com o Photo Station Pessoal, só poderá enviar 2GB de conteúdo de cada vez.

    • Nota (1): Os usuários podem baixar arquivos dos formatos especificados, mas não poderão reproduzi-los no Photo Station como um arquivo Flash compactado.

Como usar o Photo Station Pessoal?

Você pode enviar arquivos de imagens ou vídeos para a pasta home/photo e criar um blog. Para fazer isso:

  1. Organize em pastas as fotos e vídeos que deseja compartilhar. Essas pastas tornar-se-ão álbuns em Photo Station Pessoal.
  2. Faça upload das pastas de álbuns para a pasta compartilhada home/photo.
  3. Aponte o navegador para http://servername/~username/photo/ para visualizar seus álbuns.
  4. Para criar seu blog, clique em Blog no canto superior direito e registre-se para começar a postar artigos.

Como posso publicar meus álbuns e blog em Photo Station Pessoal?

Para publicá-los no Photo Station Pessoal e torná-los disponíveis na Internet, você terá de configurar seu DiskStation para usar um IP externo.

  1. Faça login no DiskStation como admin ou como um usuário que pertença ao grupo administrators.
  2. Para conectar ao DiskStation na internet você pode aplicar:
    • Um IP externo. Vá até Menu principal > Painel de controle > Rede para editar as configurações de rede do servidor, caso tenha recebido um endereço específico de IP externo de seu provedor de serviços de rede.
    • As configurações PPPoE. Vá até PPPoE no Menu principal > Painel de controle > Rede para ativar a conexão PPPoE.
  3. Se deseja usar uma conta DDNS registrada, vá para Menu principal > Painel de Controle > Acesso Externo > DDNS para habilitar o serviço DDNS.
  4. Concluído!

Agora as pessoas na Internet podem explorar seus álbuns e seu blog apontando seus navegadores para http://serverIP/~username/photo/ ou http://DDNS_hostname/~username/photo/.
