Depuração de Dados*

A depuração de dados é um recurso de manutenção de dados que inspeciones grupos de disco ou volumes. Há dois tipos de depuração de dados:

  • Depuração do RAID: Essa função pode ser usada com volumes ou grupos de discos que utilizem o SHR (composto de três ou mais discos), RAID 5 ou RAID 6. Pode reparar qualquer inconsistência de dados detectada.
  • Depuração do sistema de arquivos: Essa função pode ser usada com volumes no sistema de arquivos Btrfs. Se qualquer inconsistência de dados com a soma de verificação for detectada, o caminho do arquivo será gravado no Log Center. Recomendamos enfaticamente que você use um backup para reparar os dados.

Recomendamos a execução regular da depuração de dados para garantir a consistência dos dados e evitar perda de dados em caso de falha no disco rígido.

Para executar a depuração de dados:

  1. Abra o Gerenciador de armazenamento.
  2. Vá para Volume ou Grupo de Discos.
  3. Selecione o volume ou grupo de discos no qual executar a depuração de dados.
  4. Clique em Gerenciar.
  5. Selecione Depuração de dados.
  6. Selecione o tipo de depuração de dados desejado. (O sistema ignorará automaticamente essa etapa se houver apenas um tipo disponível.)

Nota:

  • A depuração de dados pode ser executada apenas quando o status do volume ou grupo de discos for Normal.
  • Se o sistema for desligado antes da conclusão, será necessário reiniciar manualmente a depuração de dados desde o começo.
