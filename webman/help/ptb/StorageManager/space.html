Volume

Os volumes são espaços de armazenamento básicos nos quais o usuário pode criar pastas compartilhadas, iSCSI LUN (arquivos regulares), salvar dados ou instalar pacotes. Antes de criar qualquer volume, certifique-se de já ter criado um RAID Group. Para obter mais informações sobre RAID Groups, consulte RAID Group.

Para criar um volume:

  1. Certifique-se de que seu DiskStation tenha um RAID Group existente com espaço alocável.
  2. Clique no botão Criar.
  3. Selecione um RAID Group no qual deseja criar o volume.
  4. Modifique as configurações desejadas.
  5. Defina o tamanho da alocação para o volume. (A definição do tamanho da alocação não está disponível para volumes criados em RAID Groups de volumes únicos ou iSCSI LUN)

Nota:

  • O tamanho máximo de volume que pode ser alocado é de 200 TB.
  • O tamanho mínimo alocado para um volume Btrfs deve ser de pelo menos 10 GB.
  • Os volumes Btrfs reservam 4% do espaço para metadados.

Para remover um volume:

  1. Selecione o volume que deseja remover.
  2. Clique no botão Remover e siga o assistente para concluir o processo.

Nota:

  • A remoção de um volume apagará todos os dados existentes nele.

Para forçar a montagem de um volume:

Se estiver faltando cache em um volume , o sistema não montará o volume durante a inicialização. Se o cache não for recuperável e você quiser acessar o volume, você pode clicar em Forçar montagem.

Nota:

  • Essa função está disponível apenas em modelos específicos.

Para reparar um volume:

Se o status do volume indicar Degradado, vá para a guia RAID Group e encontre um RAID Group adequado. Em seguida siga as instruções sugeridas para repará-lo.

Para expandir o tamanho de um volume

  • Para volumes criados em um RAID Group para volumes múltiplos ou iSCSI LUNs (nível de bloqueio).
    1. Selecione o volume que deseja expandir.
    2. Clique em Editar.
    3. Digite no novo tamanho e clique em OK para expandir.
  • Para volumes criados em RAID Groups de volumes único ou iSCSI LUN (nível de bloqueio), consulte a seção Para expandir o tamanho de um RAID Group de RAID Group, o tamanho do volume será expandido após a expansão do RAID Group básico.

SSD TRIM

Se o volume consistir totalmente em SSDs (Unidades de Estado Sólido), recomendamos ativar o SSD TRIM. Esse recurso melhora o desempenho de gravação e leitura dos volumes criados nos SSDs, aumentando a eficiência além de prolongar a vida útil de seus SSDs.

Para ativar o SSD TRIM:

  1. Abra o Gerenciador de armazenamento e vá para Volume.
  2. Selecione um volume que consista inteiramente de SSDs e clique no botão Configurar.
  3. Na janela que aparece, vá para a aba SSD TRIM, e marque a caixa Ativar TRIM.
  4. Clique no botão Definir programação para escolher quando o sistema realizará o SSD TRIM.
  5. Clique em OK.

Nota:

  • O SSD TRIM está disponível apenas em certos modelos de DiskStation e tipos de RAID.
  • Alguns modelos de SSD não conseguem fazer o SSD TRIM em volumes RAID 5 e RAID 6. Consulte a lista de compatibilidade em www.synology.com.
  • O SSD TRIM não pode ser ativado nas seguintes situações:
    • O volume é montado com cache SSD.
    • O pacote Time Backup está instalado.
    • Um cluster de high-availability foi criado com o Gerenciador de High Availability.
  • O SSD TRIM não pode ser ativado em iSCSI LUNs de nível de bloqueio.
  • O desempenho do sistema será afetado ao executar o SSD TRIM. O tempo necessário para terminar a execução do TRIM varia de acordo com o tamanho do volume e o modelo do SSD.
  • Quando o TRIM estiver habilitado em um volume, ele também poderá ser ativado em outros criados no mesmo Grupo de discos ou RAID Group.

Depuração do sistema de arquivos*

A depuração do sistema de arquivos é um recurso de manutenção de dados que inspeciona volumes. Se qualquer inconsistência de dados com a soma de verificação for detectada, o caminho do arquivo será gravado no Log Center e recomendamos que você use um backup para reparar os dados.

Recomendamos executar regularmente a depuração do sistema de arquivos para garantir a consistência dos dados.

Para executar a depuração do sistema de arquivos:

  1. Abra o Gerenciador de armazenamento e vá para Volume.
  2. Selecione o volume no qual realizará a depuração do sistema de arquivos.
  3. Clique em Gerenciar.
  4. Selecione Depuração do sistema de arquivos.

Nota:

  • A depuração do sistema de arquivos pode ser realizada apenas em volumes no sistema de arquivos Btrfs.
  • A depuração do sistema de arquivos pode ser realizada apenas quando o status do volume for Normal.
  • Não desligue o sistema até que o processo de depuração seja concluído. Se o sistema for desligado antes da conclusão, será necessário reiniciar manualmente a depuração do sistema de arquivos desde o começo.

Desfragmentação do sistema de arquivos*

Se seus dados forem modificados constantemente, a quantidade de arquivos fragmentados aumentará no sistema de arquivos, o que resultará em baixo desempenho de acesso dos arquivos nos discos rígidos. A realização da desfragmentação do sistema de arquivos pode melhorar o desempenho.

Para realizar a desfragmentação do sistema de arquivos:

  1. Abra o Gerenciador de armazenamento e vá para Volume.
  2. Selecione o volume em que deseja realizar a desfragmentação do sistema de arquivos.
  3. Clique em Gerenciar.
  4. Selecione Desfragmentação do sistema de arquivos.

Nota:

  • A desfragmentação do sistema de arquivos pode ser realizada apenas em volumes no sistema de arquivos Btrfs.
  • A desfragmentação do sistema de arquivos pode ser realizada apenas quando o status do volume for Normal.
  • Não desligue o sistema até que o processo de desfragmentação seja concluído. Se o sistema for desligado antes da conclusão, será necessário reiniciar manualmente a desfragmentação do sistema de arquivos desde o começo.
  • O uso do volume pode aumentar se ele contiver instantâneos de pastas compartilhadas.
