HDD/SSD

Essa seção permite que você monitore o status do HDD/SSD instalado no seu DiskStation, fornecendo opções para gerenciar e analisar a integridade e o desempenho dos HDD/SSD. Vá para Gerenciador de Armazenamento > HDD / SSD para gerenciar essas opções.

Para verificar as informações do HDD/SSD:

  1. Vá para a aba HDD/SSD.
  2. Selecione uma unidade na lista e a expanda. Aqui você verá o status do S.M.A.R.T., informações do volume, nome do modelo, número de série, versão do firmware e temperatura.

Para verificar as informações de integridade:

  1. Vá para a aba HDD/SSD.
  2. Selecione uma unidade.
  3. Clique em Informações de Integridade.
  4. Vá para a guia Visão geral. Aqui você vai descobrir quantas horas a unidade ficou ligada, a temperatura atual, a contagem de reconexão/reidentificação, contagem de setores ruins e o estado S.M.A.R.T. geral.
  5. Vá para a guia Informações S.M.A.R.T.. Os números listados podem ajudá-lo a diagnosticar melhor o status de sua unidade.

Nota:

As unidades SAS não suportam informações S.M.A.R.T.

Para iniciar um teste S.M.A.R.T.:

O teste S.M.A.R.T. irá ajudar a examinar e relatar o status das unidades, alertando-o sobre possíveis falhas na unidade. Recomendamos a troca imediata das unidades em caso de detecção de algum erro.

  1. Vá para a aba HDD/SSD.
  2. Selecione uma unidade.
  3. Clique em Teste S.M.A.R.T.
  4. Selecione o tipo de teste e clique em Iniciar para começar o teste.
  5. Os resultados de teste mais recentes são exibidos na seção Resultado do Teste.

Nota:

O tempo de teste varia de acordo com o ambiente do sistema.

Para agendar testes S.M.A.R.T.:

Os testes S.M.A.R.T. podem ser agendados para serem executados automaticamente criando tarefas.

  1. Vá para a aba Programador de Testes.
  2. Clique em Criar para configurar uma nova tarefa.
  3. Modifique as seguintes configurações da guia Geral:
    • Tarefa: Digite um nome para a tarefa.
    • Tipo de teste S.M.A.R.T.: Selecione para realizar um teste rápido ou estendido.
    • Faixa do teste S.M.A.R.T.: Selecione quais HDD/SSD testar cada vez que o teste S.M.A.R.T. for realizado.
  4. Na guia Agendamento, modifique as seguintes configurações:
    • Data: Especifique quando realizar o teste S.M.A.R.T. Os testes S.M.A.R.T. podem ser agendados para serem executados em certos dias da semana (ex.: diariamente, finais de semana ou todo domingo, etc.) ou em datas específicas (ex.: 21/10/2015).
    • Vezes: Especifique quantas vezes o teste S.M.A.R.T. é realizado bem como a primeira e a última vez a cada dia.
  5. Clique em OK para concluir.

Para executar Secure Erase:

  1. Vá para a aba HDD/SSD.
  2. Selecione uma unidade e clique em Secure Erase.

Nota:

  • O Secure Erase não pode ser executado em discos em unidades de expansão.
  • O Secure Erase irá excluir permanentemente todos os dados armazenados em seu disco. Verifique se a unidade escolhida não está em uso ou se foi feito backup dos dados antes de fazer isso.
  • Quando o sistema estiver executando o Secure Erase, a unidade escolhida será temporariamente bloqueada e depois desbloqueada automaticamente quando a operação for concluída.
  • Se o Secure Erase for interrompido indevidamente durante a operação, você poderá executar novamente o Secure Erase diretamente na interface do usuário. Se as unidades forem removidas ou utilizadas em outros lugares quando elas ainda estiverem sendo bloqueadas, digite a senha "Synology" para desbloquear as unidades.

Habilitação do Suporte à Gravação de Cache para HDD

Habilitar o suporte do cache de gravação aprimora o desempenho do seu DiskStation. Apenas alguns modelos de HDD suportam esse recurso. Para garantir a proteção de dados durante o uso do suporte do cache de gravação, recomendamos enfaticamente o uso de um dispositivo no-break. Também sugerimos que o sistema seja sempre desligado adequadamente após cada uso. A desabilitação do cache de gravação reduzirá a possibilidade de perda de dados causada pelos cortes anormais de energia, mas o desempenho do sistema diminuirá.

Para habilitar o suporte à gravação de cache:

  1. Vá para a guia Gerenciamento de Cache.
  2. Marque Habilitar cache de gravação. Por padrão, essa função é habilitada.
  3. Clique em Aplicar.
