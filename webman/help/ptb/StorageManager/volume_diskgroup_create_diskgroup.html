Criar um grupo de discos

Este artigo explica como criar grupos de discos em seu DiskStation.

Antes de Começar:

  • Verifique se não há dados importantes nos discos rígidos. Todos os dados existentes serão apagados ao criar um novo grupo de discos.
  • Vá para a página HDD/SSD e verifique se o status de cada disco rígido é Normal ou Não inicializado.

Para criar um grupo de discos:

  1. Abra o Gerenciador de armazenamento.
  2. Vá para Grupo de Discos e clique em Criar.
  3. Quando o assistente de criação de grupo de discos aparecer, escolha quais discos rígidos serão usados para criar o grupo de discos. Em seguida, siga as etapas no assistente para finalizar.
