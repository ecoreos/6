Acessibilidade

Com o recurso de acessibilidade integrado, o DiskStation Manager fornece pistas de auditoria para guiar os usuários com deficiências visuais durante a navegação e o gerenciamento dos serviços/configurações do DSM.

Configuração de acessibilidade do DSM

Para maximizar a acessibilidade, sugerimos que você configure o ambiente de operação conforme a seguir:

  • Instale o leitor de tela NVDA (disponível apenas para ambientes Windows) em seu computador local.
  • Instale o navegador de Internet Mozilla Firefox mais recente.
  • Use fones de ouvido ou alto-falantes e ajuste o volume conforme suas necessidades.
  • Use o NVDA no modo de foco (Pressione a tecla NVDA + Espaço para o modo switch). A tecla NVDA padrão é Insert.
  • Navegue no DSM com as seguintes teclas de atalho:
    Tecla de atalho Descrição
    Tab Mude para outras seções de interface do usuário
    Teclas de seta Mude para itens dentro da mesma seção de interface de usuário
    Enter Abrir aplicativos/janelas
    Esc Feche aplicativos/janelas

O recurso de acessibilidade agora está funcionando nas seguintes áreas do DSM:

  • Página de login do DSM, área de trabalho e barra de tarefas
  • Menu principal
  • Painel de controle
  • File Station

Nota:

Para obter detalhes sobre a operação NVDA, consulte seu guia do usuário.

Guias de atalho de teclado do DSM

Juntamente com o recurso de acessibilidade, o DSM oferece um conjunto rico de atalhos de teclado para implementar um gerenciamento simplificado e uniforme. Pressione Shift + ? para encontrar os atalhos:

  • Atalhos do teclado da área de trabalho: Teclas de atalho para operações de janela e mais
  • Atalhos do teclado do componente: Teclas de atalho para controlar componentes da interface do usuário (ex.: listas, botões e campos de pesquisa)

Para suspender os atalhos relacionados à área de trabalho, cancele a opção Habilitar atalhos do teclado da área de trabalho.
