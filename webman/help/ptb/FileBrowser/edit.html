Editar fotos

No File Station, você pode editar fotos com o Aviary Editor, um editor de fotos externo on-line.

Para editar uma foto com o Aviary Editor:

  1. Navegue até a pasta onde a foto está localizada.
  2. Clique com o botão direito na foto e selecione Editar no Aviary.
  3. O Aviary Editor será aberto em uma nova janela do navegador. Você poderá visualizar sua foto e editá-la com as ferramentas disponíveis.
  4. Após terminar de editar a foto, clique no botão "salvar" na barra de ferramentas superior para salvar todas as alterações feitas na foto.

Nota:

  • Com o Aviary Editor, você pode editar fotos nos seguintes formatos: .jpg, .jpeg, .jpe, .png
  • O tamanho de arquivo máximo permitido pelo Aviary Editor é de 5 MB.
  • O Aviary Editor exige que as configurações de rede do seu DiskStation estejam definidas corretamente. Verifique as configurações a seguir para solucionar possíveis problemas:
    • Verifique se o seu DiskStation pode ser acessado através do QuickConnect (em Painel de controle > QuickConnect) ou se pode recuperar um endereço IP externo (em Painel de controle > Acesso externo > DDNS ou em Painel de controle > Rede > Interface de rede > PPPoE).
    • Se o DiskStation estiver localizado atrás de um roteador, verifique se foram especificadas regras corretas de encaminhamento de portas (em Painel de controle > Acesso externo > Configuração do roteador.
    • Caso ainda tenha problemas para abrir o Aviary Editor, você talvez precise especificar a porta do roteador para o DiskStation (em Painel de controle > Acesso Externo > Avançado).
