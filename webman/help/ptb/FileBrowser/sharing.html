Compartilhar links do arquivo

Os arquivos ou pastas armazenados no seu DiskStation podem ser compartilhados rápida e facilmente. Basta clicar com o botão direito no arquivo ou pasta e selecionar Compartilhar links do arquivo. Isso gera um link e um código QR, que podem ser enviados para amigos ou outros usuários. Eles podem seguir o link para baixar o arquivo ou pasta selecionada, independentemente de terem ou não uma conta DSM.

Para compartilhar links do arquivo:

  1. Selecione os arquivos ou pastas para compartilhar.
  2. Clique com o botão direito em um arquivo ou pasta selecionada, ou clique no menu Ação. Em seguida, selecione Compartilhar links do arquivo.
  3. Os links compartilhados e códigos QR são exibidos na janela que é aberta.
  4. As seguintes opções também podem ser modificadas:
    • Período de validade: Você pode personalizar os períodos de acesso válidos e tempos de acesso para links compartilhados.
      • Personalizar período de validade: Você pode determinar quanto tempo um link compartilhado permanecerá ativo após a criação. O link se tornará inválido quando passar o período especificado.
      • Personalize tempos de acesso válidos: Você pode determinar quantas vezes um link compartilhado pode ser acessado. O link se tornará inválido quando passar as vezes especificadas.
    • Compartilhar com os usuários do DSM: Você pode compartilhar o link com usuários ou grupos do DSM selecionando-os na lista suspensa. Observe que não é possível compartilhar com a conta de convidado e grupos de usuários.
    • Compartilhar com outros: Você pode compartilhar links protegidos por senhas com outras pessoas, incluindo usuários não DSM.

Para habilitar os links de gofile.me:

  1. Vá para Painel de Controle > QuickConnect e verifique se Ativar QuickConnect está marcado. Se você não configurou o QuickConnect, poderá fazê-lo nessa página.
  2. Em seguida, clique em Avançado e verifique se Compartilhamento de arquivos está marcado.
  3. Os arquivos compartilhados podem ser baixados sem ter uma conta DSM acessando o link de compartilhamento de gofile.me.

Personalizar página de links compartilhados

O File Station oferece a você várias opções para estilizar a página do link compartilhado.

  1. Vá para File Station > Configurações > Links compartilhados > Personalizar.
  2. Selecione Personalizar página de links compartilhados e clique em Configurações personalizadas.
  3. Você pode personalizar a página das maneiras a seguir:
    • Modelo: Selecione qualquer modelo padrão.
    • Fundo: Clique na miniatura ou em + para escolher um papel de parede, imagem padrão em seu DiskStation ou uma imagem aplicada anteriormente.
    • Logotipo: Clique na miniatura ou em + para escolher uma imagem carregada (menor que 1 MB), em seu DiskStation ou uma imagem aplicada anteriormente. O logotipo pode ser posicionado em qualquer um dos quatro cantos.
  4. Clique em Visualização para ver se são necessários ajustes.
  5. Clique em OK para aplicar as configurações.

Nota:

  1. As configurações personalizadas são aplicadas a todos os links compartilhados, independentemente da sua hora de criação.
  2. Você pode aplicar imagens apenas nos seguintes formatos como papel de parede/logotipo: .jpe, jpeg, .jpg, .bmp, .gif, e .png

Gerenciando links compartilhados

Os links compartilhados existentes podem ser gerenciados indo para Ferramentas > Gerenciadores de links compartilhados. Na janela que aparece, existem opções para editar, compartilhar ou remover os links compartilhados existentes.

Nota:

  1. Você pode criar até 1000 links compartilhados.
  2. A seção Status de cada link compartilhado exibe uma das seguintes opções:
    • Válido: O link compartilhado está funcionando adequadamente.
    • Expirado: O período de validade do link compartilhado expirou.
    • Inválido: O arquivo ou pasta compartilhada não existe mais ou você não tem privilégio suficiente para acessá-lo(a).
    • Ainda não em vigor: O link compartilhado ainda não se tornou ativo.
