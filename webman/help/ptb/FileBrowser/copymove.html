Copiar ou mover arquivos ou pastas

O File Station fornece a você vários métodos de transferência de arquivos/pastas entre dois DiskStation, dentro de um DiskStation ou para um computador local.

Arraste e solte arquivos/pastas entre navegadores

Sem a configuração completa dos protocolos de transferência de arquivos, você pode carregar arquivos/pastas diretamente através do File Station de um DiskStation para outro.

Para habilitar o recurso de arrastar e soltar entre navegadores:

  1. Vá para Configurações > Geral > File Station.
  2. Marque Habilitar o recurso arrastar e soltar entre navegadores.
  3. Clique em OK.
  4. Abra dois navegadores de Internet do mesmo tipo (ex.: Firefox) e conecte-os respectivamente ao DiskStation origem e alvo.
  5. Abra o File Station nos dois DiskStation. Agora você pode arrastar e soltar arquivos/pastas diretamente entre eles através do File Station.

Observação:

  • Você também pode arrastar e soltar arquivos/pastas entre duas guias do mesmo navegador.
  • O Internet Explorer não suporta arrastar e soltar entre navegadores ou entre guias.
  • Se vários arquivos/pastas forem selecionados para arrastar e soltar, os arquivos/pastas serão compactados em um arquivo para carregamento.
  • O limite de tamanho do arquivo para arrastar e soltar varia de acordo com os navegadores de Internet: Safari (1 GB).

Copie ou mova arquivos/pastas dentro do DiskStation ou para o computador local

Siga as etapas abaixo para mover ou copiar arquivos ou pastas dentro do DiskStation ou para seu computador local.

Método 1:

  1. Selecione os arquivos ou pastas que deseja mover ou copiar. (Pressione e segure a tecla Shift ou Ctrl para seleção múltipla.)
  2. Selecione Recortar ou Copiar no menu suspenso Ação.
  3. Vá para a pasta de destino em DiskStation ou seu computador.
  4. Selecione Colar - Sobrescrever ou Colar - Ignorar do menu suspenso Ação, dependendo se deseja sobrescrever os arquivos ou pastas com o mesmo nome na pasta de destino.

Método 2:

  1. Selecione os arquivos ou pastas que deseja mover ou copiar. (Pressione e segure a tecla Shift ou Ctrl para seleção múltipla.)
  2. Arraste as pastas ou arquivos da seção principal para outra pasta no painel esquerdo.
  3. No menu pop-up que aparece, selecione Copiar - Sobrescrever, Copiar - Ignorar, Mover - Sobrescrever ou Mover - Ignorar para sobrescrever ou ignorar os arquivos ou pastas com o mesmo nome na pasta de destino.

Método 3

  1. Selecione os arquivos ou pastas que deseja mover ou copiar. (Pressione e segure a tecla Shift ou Ctrl para seleção múltipla.)
  2. Pressione Ctrl + X para cortar ou Ctrl + C para copiar os arquivos/pastas. Para usuários de Mac, pressione Command + X e Command + C.
  3. Vá para a pasta de destino em DiskStation ou seu computador.
  4. Pressione Ctrl + V para colar os arquivos/pastas. (Command + V para Mac.)
  5. Na caixa de diálogo que aparece, selecione se deseja sobrescrever ou ignorar arquivos com o mesmo nome na pasta de destino.

Atalhos de Teclado:

Os seguintes atalhos do teclado podem ser úteis ao mover/copiar arquivos e pastas no File Station:

  • F2: Renomear
  • Del: Excluir
  • Ctrl + A: Selecionar todos
  • Crtl + X: Recortar
  • Crtl + C: Copiar
  • Crtl + V: Colar

Observação:

  • Ao copiar ou mover arquivos/pastas para um diretório em um sistema de arquivos FAT, os caracteres \ *?">| no nome do arquivo serão substituídos por – e os metadados de arquivos Mac serão excluídos.
  • Se você recortar arquivos de seu computador local e depois colá-los em uma pasta DiskStation, os arquivos ainda serão mantidos em seu computador local.
  • Ao mover arquivos ou pastas, o "proprietário" permanece o mesmo. No entanto, ao copiar um arquivo ou pasta, o "proprietário" é alterado para o usuário que fez a cópia.
