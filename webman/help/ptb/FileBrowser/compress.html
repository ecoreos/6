Compactar arquivos ou pastas

  1. Escolha os arquivos ou pastas no DiskStation na área principal.
  2. Escolha Compactar para "(nome de arquivo)".zip no menu suspenso Ação para iniciar a compactação.
  3. Se não vir o arquivo zip quando a compactação estiver concluída, selecione Atualizar no menu suspenso Ação para ver o arquivo.
  4. Se quiser renomeá-lo, criptografar o arquivo ou arquivá-lo usando diferentes níveis ou modos de compactação, escolha a opção Adicionar ao arquivo....
    Observação:
  1. Não é possível compactar arquivos ou pastas no computador local usando o File Station.
  2. Formatos de arquivo suportados: .zip, .7z.
  3. Quando você escolhe .zip como Formato de arquivo em Adicionar ao arquivo..., você pode selecionar uma página de código desejada para o arquivo .zip.
