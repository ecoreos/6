Download de arquivos

Siga as etapas abaixo para baixar arquivos ou pastas do DiskStation para seu computador.

Para fazer o download de arquivos para seu computador:

  1. Selecione os arquivos ou pastas para o download. (Pressione e segure a tecla Shift ou Ctrl para seleção múltipla.)
  2. Clique em Ação > Download e selecione o modo de download desejado de acordo com seu navegador:
    • Google Chrome: Clique em Download para baixar arquivos individuais (não compactados). Clique em Baixar como .zip para baixar vários arquivos em um arquivo .zip (usando a página de código do File Station especificada em Configurações > Geral).
    • Outros navegadores: Clique em Download para baixar um ou vários arquivos em um arquivo .zip (usando a página de código do File Station especificada em Configurações > Geral).
    • Adicionar à fila - Sobrescrever ou Adicionar à fila - Ignorar: Dependendo de desejar ou não sobrescrever os arquivos ou pastas com o mesmo nome no destino, selecione uma das opções para adicionar os arquivos à fila de download (que aparece quando se clica na seta para baixo no canto superior direito).
    Nota:
  1. No ambiente Windows, os caracteres \ * ? " < > | : contidos no nome do arquivo serão substituídos por - após o download.
  2. Para impedir que o Chrome baixe automaticamente vários arquivos, vá para as configurações desse navegador e altere as configurações de conteúdo na seção de privacidade.
  3. O Chrome no modo anônimo não suporta o download de arquivos individuais como arquivos descompactados.

Obter o link de download do arquivo/pasta

Você pode obter o link de download de um arquivo ou pasta e compartilhar o link com outros usuários.

Para obter o link de download de um arquivo ou pasta:

  1. Selecione o arquivo ou pasta cujo link de download deseja obter.
  2. Faça uma das seguintes ações para abrir a janela Propriedades:
    • Selecione Propriedades no menu Ação.
    • Clique com o botão direito em um dos arquivos ou pastas e selecione Propriedades.
  3. Clique com o botão direito no link de download e selecione a função pertinente no menu de atalho para copiar o link de download. O nome da função pode variar, dependendo do navegador da web que estiver usando.
  4. Envie o link para outros usuários através de e-mail ou mensagens instantâneas. Eles podem fazer o download do arquivo ou pasta vinculado digitando seu nome de usuário e senha.

Nota: Para baixar o arquivo ou pasta com o link de download, um nome de usuário e senha válidos são necessários.
