Montar pastas remotas

Você pode montar pastas remotas em servidores remotos (como computadores na rede ou outros dispositivos Synology NAS) para navegar e gerenciar suas pastas montadas pelo File Station.

Para montar uma pasta compartilhada CIFS remota*:

  1. Clique em Ferramentas > Montar pasta remota > Pasta compartilhada CIFS.
  2. No campo Pasta, especifique o caminho da pasta remota no formato \\Remote_Server\Remote_Folder, como \\192.168.0.2\share ou \\chad\cheeseburger\pictures.
  3. Se o servidor remoto exigir privilégios de acesso para a pasta remota, digite suas credenciais de usuário do servidor remoto nos campos Nome da conta e Senha.
  4. Clique em Procurar para selecionar ou criar uma pasta de destino vazia em seu DiskStation para montar a pasta remota.
  5. Marque Montar automaticamente ao iniciar se desejar que o DiskStation monte essa pasta remota a cada inicialização ou reinicialização do sistema.
  6. Clique em Montar para que a pasta remota seja montada na pasta de destino. Agora você poderá navegar e gerenciar a pasta remota no seu DiskStation.

Para montar uma pasta compartilhada NFS remota:

  1. Clique em Ferramentas > Montar pasta remota > Pasta compartilhada NFS.
  2. No campo Pasta, especifique o caminho da pasta remota no formato Remote_Server:/Remote_Folder_Path, como 192.168.0.2:/volume1/share.
  3. Selecione a versão do NFS e o protocolo de transporte desejados.
  4. Clique em Procurar para selecionar ou criar uma pasta de destino vazia em seu DiskStation para montar a pasta remota.
  5. Marque Montar automaticamente ao iniciar se você desejar que o seu DiskStation monte essa pasta remota a cada inicialização ou reinicialização do sistema.
  6. Clique em Montar para que a pasta remota seja montada na pasta de destino. Agora você poderá navegar e gerenciar a pasta remota no seu DiskStation.

Nota:

NFSv4 só suporta o protocolo TCP.

Para desmontar uma pasta remota:

  1. Selecione a pasta remota que você deseja desmontar e clique em Ferramentas > Desmontar.
  2. Como alternativa, basta clicar com o botão direito na pasta remota e selecionar Desmontar.

Nota:

  • O número máximo de pastas remotas concomitantes é de 100.
  • Você só tem permissão para montar pastas remotas em um servidor remoto que suporte os protocolos CIFS ou NFS.
  • Pastas remotas só podem ser montadas em subpastas dentro de pastas compartilhadas.
  • A montagem de pastas remotas é suportada apenas no modo de alto privilégio. Para usar esse recurso, vá para o pacote Docker em seu DSM host e selecione o Docker DSM com o qual deseja habilitar as pastas compartilhadas criptografadas, clique em Editar e marque a caixa ao lado de Executar o contêiner usando altos privilégios.

____
* Disponível apenas em modelos específicos.
