Requisitos do File Station

Para usar todos os recursos do File Station, você precisará do navegador web, Java e Flash Player mais recentes.

  • Navegador: Internet Explorer 8 ou mais recente, Firefox 3.6 ou mais recente, Safari 5.0 ou mais recente, Chrome; com AJAX e JavaScript habilitado
  • Java (para pesquisar pastas no computador local): Java Runtime Environment (JRE) 5 ou mais recente
  • Flash: Flash Player 9.0.28 ou mais recente
