File Station

O File Station é a ferramenta de gerenciamento de arquivo fácil de usar e permite que os usuários do DSM através da Internet tenham acesso às pastas do DiskStation com seus navegadores. Ao abrir dois ou mais File Stations, você pode gerenciar todos os dados no DiskStation arrastando-os e soltando-os entre File Stations diferentes.
