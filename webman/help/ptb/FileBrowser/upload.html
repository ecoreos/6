Upload de arquivos ou pastas

Use um dos métodos a seguir para fazer o carregamento de arquivos ou pastas do computador para o DiskStation.

Para ter acesso completo às funções de carregamento exibidas abaixo, instale o Java Runtime Environment (JRE) e o Flash Player primeiro.

Método 1:

  1. Na parte superior do painel esquerdo, selecione uma pasta de destino no DiskStation.
  2. Clique em Carregar, selecione Carregar - Ignorar ou Carregar - Sobrescrever e selecione os arquivos do seu computador local.

Método 2:

  1. Vá para Configurações, marque Procurar arquivos do computador local com o File Station e atualize a página da Internet para visualizar as pastas de seu computador local na parte inferior do painel esquerdo.

    2. Observação:

    A opção Procurar arquivos do computador local com o File Station funciona apenas após você ter instalado e habilitado o JRE em seu computador.

  2. Na parte inferior do painel esquerdo, escolha uma pasta no computador local. A área principal agora exibirá o conteúdo da pasta selecionada.
  3. Escolha os arquivos ou pastas que desejar na área principal, e clique em Carregar.
  4. Selecione uma pasta de destino no DiskStation na janela que irá aparecer.
  5. No canto superior direito da janela, selecione Sobrescrever se quiser sobrescrever arquivos ou pastas com o mesmo nome na pasta de destino. Caso contrário, selecione Ignorar.
  6. Clique em OK para iniciar o carregamento.

Método 3:

  1. Vá para Configurações, marque Procurar arquivos do computador local com o File Station e atualize a página da Internet para visualizar as pastas de seu computador local na parte inferior do painel esquerdo.
  2. No painel esquerdo, arraste uma pasta do computador (a parte inferior) para qualquer pasta no DiskStation (a parte superior).
  3. No menu pop-up que aparece, selecione se deseja sobrescrever ou ignorar arquivos/pastas com o mesmo nome na pasta de destino.

Método 4:

  1. Na parte superior do painel esquerdo, selecione uma pasta de destino no DiskStation.
  2. Selecione os arquivos na área de trabalho ou explorador de arquivos do computador (por exemplo, o Windows Explorer ou o Mac Finder) e arraste e solte-os no File Station no navegador de Internet.
  3. Na caixa de diálogo que aparece, selecione se deseja sobrescrever ou ignorar arquivos com o mesmo nome na pasta de destino.

    4. Observação:

    • O Método 4 é suportado apenas em navegadores específicos: Chrome, Firefox, Safari 5.1 ou mais recentes. Entretanto, ele não é suportado para Mac quando Procurar arquivos do computador local com o File Station estiver marcado.
    • Você também pode arrastar e soltar pastas ou arquivos vazios, mas apenas quando Procurar arquivos do computador local com o File Station estiver marcado para navegadores indicados acima. Caso contrário, você não poderá carregar pastas e arquivos vazios usando o Método 4.
    • Se estiver usando o Chrome para Linux e tiver se conectado ao DSM através do protocolo HTTPS, você deverá importar um certificado confiável (no Painel de controle > Segurança > Certificado) para poder fazer upload usando o Método 4.

Observação:

  1. Você pode adicionar até 1000 arquivos à sua fila de carregamento.
  2. Você pode carregar arquivos com tamanho acima de 4 GB se estiver usando o Google Chrome, Firefox, Safari 5.1 ou posterior ou quando Procurar arquivos do computador local com o File Station estiver marcado. Caso contrário, somente poderá carregar até 2 GB de arquivos de uma só vez.
  3. Na pasta compartilhada não criptografada, o nome do arquivo/pasta deve ter até 255 caracteres (até cerca de 80 caracteres para idiomas não latinos); na pasta compartilhada criptografada, ele deve ter até 143 caracteres (até cerca de 47 caracteres para idiomas não latinos).
  4. O nome do arquivo/pasta não pode ter os seguintes caracteres: \ / : * ? " > |.
  5. O nome do arquivo/pasta não deve iniciar com "._".
  6. Você pode carregar arquivos que tenham 0 KB de tamanho somente quando tiver o JRE instalado.

Resolução de problemas

Se não for possível procurar itens locais armazenados no Mac usando o Safari corretamente, faça o seguinte:

  1. No Mac, abra o Safari e selecione Preferências no menu Safari
  2. Clique em Gerenciar configurações de site na guia Segurança.
  3. Em Java, selecione seu "site" DSM.
  4. Selecione Executar no modo inseguro.
