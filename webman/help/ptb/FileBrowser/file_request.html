Criar solicitação de arquivo

As solicitações de arquivo são convites de carregamento de arquivo enviados para usuários não DSM. Com a solicitação de links de arquivos, os usuários não DSM podem carregar vários arquivos diretamente no File Station em seu DiskStation, sem a necessidade de possuir contas correspondentes de usuário e privilégios de carregamento de arquivo.

Para criar links de solicitação de arquivos:

  1. Vá para File Station e selecione uma pasta como destino do carregamento.
  2. Clique em Ação ou clique com o botão direito na pasta de destino e selecione Criar solicitação de arquivo.
  3. Na janela pop-up, você pode encontrar/modificar as informações abaixo:
    • Caminho do arquivo: Liste o caminho do diretório de destino.
    • Link: Liste o link de solicitação de arquivo para compartilhar com usuários alvo.
    • Seu nome: Digite um nome para indicar a identidade do solicitante. O valor padrão é seu nome de usuário.
    • Mensagem: Personalize a mensagem para usuários alvo.
    • Habilitar proteção por senha: Selecione essa opção para que apenas usuários com a senha possam carregar arquivos através do link criado. Especifique a senha na caixa abaixo.
    • Período de validade: Clique nesse botão para personalizar a duração desse link de solicitação de arquivo:
      • Configurar hora de parada: Selecione essa opção para determinar por data e hora quando o link para de funcionar para o carregamento do arquivo.
      • Configurar hora de início: Selecione essa opção para determinar por data e hora quando o link começa de funcionar para o carregamento do arquivo.
      • Número de acessos permitidos: Selecione essa opção para determinar quantas vezes o link pode ser acessado.
  4. Clique em Salvar para concluir as configurações.

Para compartilhar links de solicitação de arquivo:

Para localizar o URL de solicitação do arquivo, vá para File Station > Ferramentas > Gerenciador de links compartilhados e clique duas vezes no link desejado. Você pode compartilhar o link de uma das maneiras abaixo:

  • Basta copiar o URL para os usuários alvo.
  • Envie o link através do E-mail padrão (sua conta de e-mail do DSM) ou através do Mail (um aplicativo de e-mail de terceiros). Para usuários de primeira vez, siga as instruções do assistente para configurar uma conta de e-mail para usar.

Para carregar arquivos:

Se for necessário carregar arquivos através do link, faça o seguinte:

  1. Abra o URL dado no computador local ou em um dispositivo móvel (Android ou iOS). Digite a senha se ela for exigida.
  2. Na página de carregamento digite um nome que indique sua identidade.
  3. Basta arrastar e soltar os arquivos para carregar ou clicar em Adicionar arquivos para selecionar arquivos.
  4. Clique em Carregar para concluir a solicitação.

Nota:

  • As pastas não podem ser carregadas através de um link de solicitação de arquivo.
  • As pastas dentro da pasta compartilhada "web" não podem ser especificadas como pasta de destino.
  • A função de carregamento varia com os navegadores de Internet:
    • Internet Explorer 8 e Internet Explorer 9 não suporta o carregamento de arquivo através do recurso arrastar e soltar.
    • O Chrome em dispositivos iOS suporta apenas carregamento de imagem.
    • O Safari em dispositivos iOS suporta apenas carregamento de imagem e vídeo.
    • Os navegadores de Internet e dispositivos Android não estão sujeitos a nenhum tipo de limitação de arquivo.

Para localizar arquivos carregados:

Os arquivos carregados pelo mesmo usuário aparecerão em uma pasta idêntica dentro do destino. Por padrão, o nome da pasta será Carregador_Nome (Solicitante_Nome).

Por exemplo se os arquivos forem carregados por Watson para Holmes, você pode localizar os arquivos na pasta nomeada Watson (Holmes).

Gerenciamento de links de solicitação de arquivo

Para modificar links de solicitação de arquivos:

Após criar um link de solicitação de arquivo, você pode modificar as configurações em File Station > Ferramentas > Gerenciar links compartilhados.

  1. Clique duas vezes no link desejado ou selecione o link e clique em Editar.
  2. Na janela pop-up, modifique as configurações relevantes.
  3. Clique em Salvar para aplicar as novas configurações.

Para excluir links de solicitação de arquivos:

  1. Vá para File Station > Ferramentas > Gerenciador de links compartilhados.
  2. Selecione os links desejados indesejado e clique em Excluir. O URL de solicitação de arquivo correspondente parará de funcionar imediatamente.

Para atribuir privilégios de arquivamento de solicitação:

Como admin ou um usuário que pertença ao grupo administrators, você pode atribuir privilégios para usuários para que eles possam criar links de solicitação de arquivo.

  1. Vá para File Station > Configurações > Links compartilhados > Arquivar links de solicitação.
  2. Selecione o tipo de usuário para permitir privilégios de arquivamento de solicitação.
  3. Clique em OK para salvar as configurações.
