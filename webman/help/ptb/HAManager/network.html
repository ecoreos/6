Configurações da rede

Essa página contém três guias que exibem as configurações de rede do conjunto de high-availability, o servidor do membro ativo e o servidor do membro passivo.

Guia do conjunto HA

Ao criar um conjunto de high-availability, o sistema gera um endereço IP comum para que o IP do conjunto de high-availability receba conexões de entrada. Podem ser criados endereços IP adicionais para o conjunto de high-availability, se necessário. Os endereços IP das interfaces de rede ativadas no servidor ativo serão exibidos nessa guia (a interface de rede usada pela conexão de Pulsação não será exibida nessa página, pois tal interface será usada para a replicação de dados entre os dois servidores). Essa guia mostra as informações abaixo:

  • Interface de rede: A interface de rede ativada no servidor ativo.
  • Conexão: Exibe se a interface está conectada ou não.
  • Endereço IP: Ao criar um conjunto de high-availability, o sistema gera um endereço IP comum para que o IP do conjunto de high-availability receba conexões de entrada. Você pode se conectar ao conjunto de high-availability usando esse endereço IP quando quiser entrar no DiskStation Manager (DSM), independente de qual servidor Synology for o servidor ativo. Isso evita a necessidade de se conectar a endereços IP diferentes quando ocorre uma mudança.
  • Máscara de Sub-rede: A máscara de sub-rede associada com o endereço IP dado do conjunto de high-availability.

Para editar o Endereço IP e a Máscara sub-rede:

  1. Selecione o endereço IP ou a máscara de sub-rede que deseja editar e clique em Editar ou clique duas vezes no campo para iniciar a edição.
  2. Marque as caixas de seleção Habilitar para habilitar ou desabilitar cada interface. Você pode habilitar mais de uma interface de rede de conjunto, mas a interface de rede original que foi configurada durante a criação do conjunto não pode ser desabilitada.

Para alterar o nome do servidor do conjunto de high-availability:

  1. Clique em Alterar configurações de rede para visualizar o nome do conjunto de high-availability e alterar suas configurações.
  2. Clique em Aplicar para salvar o nome do conjunto de high-availability atual e as configurações do endereço IP.
  3. Para descartar as configurações atuais clique em Cancelar.

Guias do servidor ativo e passivo (exibidas como nome de servidor)

Essas guias mostram o status das interfaces de rede do servidor ativo ou passivo. Desses, a Conexão de pulsação ocupará uma interface de rede em cada um dos servidores ativo e passivo. Essas guias exibem as seguintes informações:

  • Interface de rede: A interface de rede física no servidor.
  • Endereço IP: O endereço IP atribuído para dada interface de rede.
  • Máscara de Sub-rede: A máscara de sub-rede associada com o endereço IP dado.
  • Descrição: Exibe informações de dada interface de rede e acessibilidade de seu endereço IP atribuído. Esse campo mostrará as seguintes condições possíveis:
    1. Endereço IP original acessível: O endereço IP original atribuído à interface do servidor ativo é acessível para dados e serviços.
    2. Endereço IP original não acessível: O endereço IP original atribuído à interface no servidor passivo não está acessível.
    3. IP da conexão de pulsação, não acessível da Intranet: O endereço IP original atribuído automaticamente à conexão de pulsação não é acessível para dados e serviços.
