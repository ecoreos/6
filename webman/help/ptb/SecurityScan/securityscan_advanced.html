Avançadas

Você pode visualizar e alterar as configurações avançadas do Consultor de Segurança.

Linha de Base de Segurança

Você pode escolher quais regras de segurança o Consultor de Segurança verificará selecionando uma Linha de base de segurança. O número e os tipos das regras variam para cada Linha de base de segurança. Faça sua escolha conforme suas necessidades. Você está usando o DiskStation para trabalho e negócios ou em casa e para uso pessoal? Você pode selecionar entre as seguintes opções:

  • Para uso doméstico e pessoal: O Consultor de segurança executará verificações essenciais para proteger seu DiskStation contra as ameaças cibernéticas mais comuns.
  • Para trabalho e negócios: O Consultor de segurança executará verificações completas para atender aos requisitos de conformidade de segurança de empresas.
  • Personalizado: Selecione quais regras de segurança você deseja que o Consultor de Segurança verifique.

Agendamento de Verificação

Você pode definir um agendamento para verificações regulares clicando em Habilitar agendamento de verificação regular e selecionando o horário em que deseja que o Consultor de Segurança execute uma verificação. O Consultor de Segurança verificará as regras de acordo com a Linha de Base de Segurança que você selecionou.
