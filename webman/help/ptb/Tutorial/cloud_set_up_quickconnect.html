Acesse seu Synology NAS pela Internet e compartilhe arquivos

Visão geral

Com o QuickConnect você pode facilmente se conectar ao seu Synology NAS pela Internet sem precisar criar regras de encaminhamento de portas ou outras configurações de rede complicadas. O QuickConnect permite que você se conecte por um endereço personalizável ​​simples, como Quickconnect.to/exemplo.

1. Configure o QuickConnect

  1. Vá para Painel de controle > QuickConnect.
  2. Marque a caixa Habilitar QuickConnect.
  3. Se você não tiver uma conta Synology, clique em Faça login ou cadastre-se para ter uma conta Synology. Insira as informações necessárias e clique em OK. Se você já tiver uma conta Synology, digite as informações de sua conta.
  4. Crie o seu próprio QuickConnect ID no campo QuickConnect ID. Em seguida, clique em Aplicar.
  5. As informações de seu link QuickConnect serão exibidas. Use esses links para acessar o Synology NAS. Nesse exemplo, o endereço do DSM é http://Quickconnect.to/YourID.
  6. Se você não vir o link do DSM QuickConnect, clique em Avançado e verifique se o DSM está habilitado.
  7. Agora você pode usar o link do DSM QuickConnect para acessar seu Synology NAS!

2. Compartilhe arquivos em seus NAS usando o QuickConnect

Com o QuickConnect habilitado, você pode facilmente compartilhar arquivos armazenados em seu Synology NAS com qualquer pessoa, sem se preocupar com o encaminhamento de portas.

  1. Primeiro ative a conexão HTTPS para compartilhar seus arquivos com segurança. Vá para Painel de controle > Rede.
  2. Marque a caixa Habilitar conexão HTTPS e clique em Aplicar.
  3. Em seguida, habilite o compartilhamento de arquivos para o QuickConnect. Vá para Painel de controle > QuickConnect.
  4. Clique em Avançado e verifique se a caixa de compartilhamento de arquivos está marcada.

    5. Nota:

    Ao configurar as configurações avançadas do QuickConnect, recomendamos que você faça login no DSM com o endereço IP local ou nome de domínio do Synology NAS.

  5. Você está pronto para começar a compartilhar arquivos usando o QuickConnect. Abra o File Station e clique com o botão direito do mouse no arquivo ou pasta que você deseja compartilhar. Em seguida, clique em Compartilhar links do arquivo no menu de contexto.
  6. Um link de compartilhamento de arquivos será gerado. Basta compartilhar esse link com a família e amigos para fazer o download do arquivo/pasta compartilhada.
