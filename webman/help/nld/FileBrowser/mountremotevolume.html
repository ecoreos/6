Externe mappen koppelen

U kunt externe mappen van externe servers koppelen (zoals netwerkcomputers of andere Synology NAS-apparaten) om gekoppelde mappen via File Station te doorzoeken en te beheren.

Een externe CIFS-gedeelde map koppelen*:

  1. Klik op Hulpmiddelen > Externe map koppelen > CIFS-gedeelde map.
  2. Specificeer in het veld Map het externe mappad in de indeling \\Remote_Server\Remote_Folder, zoals \\192.168.0.2\share of \\chad\cheeseburger\pictures.
  3. Vereist de externe server toegangsmachtigingen voor de externe map, voer dan uw gebruikersgegevens voor de externe server in de velden Accountnaam en Wachtwoord in.
  4. Klik op Bladeren om een lege bestemmingsmap op het DiskStation te selecteren of te maken om de externe map te koppelen.
  5. Tik op Automatisch koppelen bij start als u wilt dat DiskStation deze externe map na elke (her)start opnieuw koppelt.
  6. Klik op Koppelen om de externe map aan de doelmap te koppelen. Nu kunt u de externe map vanaf uw DiskStation doorbladeren en beheren.

Een externe NFS-gedeelde map koppelen:

  1. Klik op Hulpmiddelen > Externe map koppelen > NFS-gedeelde map.
  2. Specificeer in het veld Map het externe mappad in de indeling Remote_Server:/Remote_Folder_Path, zoals 192.168.0.2:/volume1/share.
  3. Selecteer de gewenste NFS-versie en het transportprotocol.
  4. Klik op Bladeren om een lege bestemmingsmap op het DiskStation te selecteren of te maken om de externe map te koppelen.
  5. Tik op Automatisch koppelen bij start als u wilt dat uw DiskStation deze externe map na elke (her)start opnieuw koppelt.
  6. Klik op Koppelen om de externe map te koppelen aan de doel-map. Nu kunt u de externe map vanaf uw DiskStation doorbladeren en beheren.

Opmerking:

NFSv4 ondersteunt uitsluitend het TCP-protocol.

Om een externe map te ontkoppelen:

  1. selecteer de externe map die u wilt ontkoppelen en klik op Hulpmiddelen > Ontkoppelen.
  2. Dit is ook mogelijk met een rechtsklik op de externe map en het selecteren van Ontkoppelen.

Opmerking:

  • het maximum aantal gelijktijdige externe mappen is 100.
  • U mag uitsluitend externe mappen van een externe server die het CIFS- of NFS-protocol ondersteunt koppelen.
  • Externe mappen kunnen alleen aan submappen onder gedeelde mappen worden gekoppeld.
  • Het koppelen van externe mappen wordt alleen ondersteund in de modus hoge machtiging. Om deze functie te gebruiken, ga naar het Docker-pakket op uw host-DSM, selecteer de Docker DSM waarvoor u gecodeerde gedeelde mappen wilt inschakelen, klik op Bewerken en schakel het selectievakje naast Container uitvoeren met hoge machtigingen in.

* Uitsluitend beschikbaar op specifieke modellen.
