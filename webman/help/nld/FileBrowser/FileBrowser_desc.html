File Station

File Station is een gebruiksvriendelijk hulpprogramma voor bestandsbeheer waarmee DSM-gebruikers via het internet DiskStation-mappen in hun webbrowser kunnen openen. Door twee of meer File Stations op te starten, kunt u al uw DiskStation-gegevens beheren met slepen-en-neerzetten tussen verschillende File Stations.
