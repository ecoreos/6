File Station vereisten

Om de volledige versie van File Station te gebruiken moet u de meest recente browser, Javaversie en Flash Player gebruiken.

  • Browser: Internet Explorer 8 of hoger, Firefox 3.6 of hoger, Safari 5.0 of hoger, Chrome; met AJAX en JavaScript ingeschakeld
  • Java (voor het bladeren in mappen op de lokale computer): Java Runtime Environment (JRE) 5 of hoger
  • Flash: Flash Player 9.0.28 of hoger
