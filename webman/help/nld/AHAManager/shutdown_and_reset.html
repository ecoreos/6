Uitschakeling en serververvanging

De onderstaande instructies beschrijven de uitschakelings- of serververvangingsprocedure in een high-availability cluster.

Om een high-availability cluster uit te schakelen met High Availability Manager:

Volg de onderstaande instructies als u een high-availability cluster wilt uitschakelen om de servers naar een nieuwe fysieke locatie te verplaatsen of onderhoudswerkzaamheden wilt uitvoeren.

  1. Meld u aan bij DSM.
  2. Ga naar High Availability Manager.
  3. Klik op de knop Beheren.
  4. Selecteer High-availability cluster uitschakelen.
  5. Er verschijnt een bevestigingsmelding. Om te bevestigen klik op OK.

Om een high-availability cluster handmatig uit te schakelen:

High-availability cluster kan ook handmatig worden uitgeschakeld via de voedingsknop van elke server.

  1. Druk en houd de aan/uit-knop van de passieve server ingedrukt tot u een geluidsignaal hoort en wacht tot de server volledig uitschakelt.
  2. Druk en houd de aan/uit-knop van de actieve server ingedrukt tot u een geluidsignaal hoort en wacht tot de server volledig uitschakelt.

Opmerking:

  • Servers opstarten:
    als u de servers opnieuw wilt inschakelen, moeten de actieve en passieve servers tegelijk worden ingeschakeld (of met een interval van minder dan vijf minuten). Als u slechts een van de servers inschakelt, zal deze server met een Wake-on-LAN-melding proberen om de andere server op te starten.
  • Abnormale uitschakeling:
    in high-availability cluster is het opnieuw opstarten na een stroomstoring ingeschakeld. Als de servers door een stroomstoring worden uitgeschakeld, zullen ze automatisch bij herstel van de stroomvoorziening worden opgestart.

Om een server in een high-availability cluster te vervangen:

volg deze instructies om een server te verwijderen en een andere server in te stellen.

  1. Ga naar pagina Overzicht.
  2. Klik op de knop Beheren.
  3. Selecteer Actieve server uitschakelen of Passieve server uitschakelen om de server die u wilt vervangen uit te schakelen.
  4. Controleer de verbindingen van de nieuwe server. Na het opstarten van de nieuwe server wordt deze in de High Availability Manager gemarkeerd als “onbekend”.
  5. Ga opnieuw naar pagina Overzicht.
  6. Klik op de knop Beheren.
  7. Selecteer De onbekende server herstellen zodat de "onbekende server" wordt toegevoegd aan de high-availability cluster en wordt erkend als passieve server.
