Overzicht

Deze pagina toont basisinformatie over de high-availability cluster en servers die er deel van uitmaken. Deze pagina biedt daarnaast hulpmiddelen om onderhoud uit te voeren op high-availability clusters en lidservers.

High-Availability Cluster-status

Dit veld toont de globale status van uw high-availability cluster en wijst u op mogelijke fouten. Hieronder volgt een tabel van mogelijke condities die in dit veld worden weergegeven:

Status Beschrijving
Gezond De algemene conditie van uw high-availability cluster is goed.
Bezig met laden Het systeem is bezig met het laden van de clusterstatus. Overschakelen kan niet worden uitgevoerd tot de status laden is voltooid. Zie Overschakelen
Bezig met configureren Het systeem wordt geconfigureerd. Overschakelen kan niet worden uitgevoerd tot de configuratie is voltooid.
Waarschuwing De status van HA-cluster wijkt af, maar de services zijn nog steeds operationeel. In deze situatie kan Overschakelen niet worden uitgevoerd.
Gevaar Zowel de actieve als de passieve servers hebben een storing of de opslag is vastgelopen. De high-availability cluster kan niet normaal werken. Controleer de status van alle opslagruimtes (bijv. RAID-groepen, schijfgroepen, volumes en iSCSI LUN's). In deze situatie kan Overschakelen niet worden uitgevoerd.

Waarschuwingsmeldingen

De volgende tabel toont een overzicht van waarschuwingsmeldingen die systeemfouten kunnen worden weergegeven. Wanneer er meer dan een waarschuwingsmelding verschijnt, kunt u met de knoppen voorwaarts/achterwaarts doorheen de meldingen bladeren.

Waarschuwing/fout Beschrijving
Er is geen passieve server in deze high-availability cluster. De informatie van de passieve server is niet beschikbaar. Controleer of de passieve server correct is verbonden. In dit geval is overschakelen niet beschikbaar.
De detectie van de passieve server is mislukt Deze fout kan optreden omdat de passieve server offline is of omdat de actieve en passieve servers niet kunnen communiceren via de Heartbeatverbinding. In dit geval is overschakelen niet beschikbaar.
Uitvoering van failover is mislukt Er is een failover geactiveerd op de actieve server maar de uitvoering is mislukt. In dit geval zal de actieve server geen tweede failover uitvoeren. Volg de onderstaande stappen om deze fout de verwijderen:
  1. Probeer de problemen in de high availability-cluster op te lossen.
  2. Schakel de passieve server opnieuw in.
  3. Klik op Beheren > Overschakelen om een overschakeling uit te voeren.
Bewaakte doelfout Er is een fout opgetreden in een of meer gecontroleerde targets, inclusief systeemstatus, service of netwerkinterface.
Er is een fout (zijn fouten) opgetreden in de verbinding tussen de uitbreidingseenheden en de actieve/passieve server. De uitbreidingseenheden zijn niet correct verbonden met de actieve of passieve server. Controleer de correcte verbinding met de actieve server en stel dezelfde verbinding in met de passieve server. In dit geval is overschakelen niet beschikbaar.
Een volume is beschadigd Een opslagruimte (bijv. RAID-groep, schijfgroep, volume, iSCSI LUN en SSD-cache) is beschadigd. Controleer de pagina Opslagstatus.
Een volume is vastgelopen Een opslagruimte (bijv. RAID-groep, schijfgroep, volume, iSCSI LUN en SSD-cache) is vastgelopen. Controleer de pagina Opslagstatus. Failover wordt automatisch uitgevoerd wanneer de passieve server schijven herkent waartoe de actieve server geen toegang heeft. Anders zal overschakelen niet beschikbaar zijn.
De heartbeat-verbinding is verloren gegaan Deze fout kan optreden als gevolg van een storing van de Heartbeat-verbinding. In dit geval is overschakelen niet beschikbaar.
Hardwarestoring Deze fout kan optreden bij een hardwarestoring (bijv. voedingseenheid of systeemventilator) op de actieve of passieve server.
Verschillende fysieke RAM-grootte Deze fout kan optreden wanneer de fysieke RAM-grootte van de actieve en passieve servers niet overeenkomen. In dit geval is overschakelen niet beschikbaar.
Er is geen statisch IP toegewezen aan deze high-availability-cluster Deze fout kan optreden bij het ontbreken van een statisch IP voor de high-availaility cluster. In dit geval is overschakelen niet beschikbaar.
Er is geen statisch IP op de gecontroleerde interface Deze fout kan optreden bij het ontbreken van een statisch IP voor de gecontroleerde interface. In dit geval is overschakelen niet beschikbaar.
Bezig met herstel van passieve server De herstelprocedure van de passieve server wordt uitgevoerd. In dit geval is overschakelen niet beschikbaar.
Bezig met upgraden van de passieve server Het systeem voert een upgrade uit van de passieve server. In dit geval is overschakelen niet beschikbaar.

High-Availability Cluster knop "Beheren"

Deze knop bevindt zich onder de informatie van de high-availability cluster. Het bevat de volgende acties:

  • Overschakelen: initieert overschakelen van de actieve server naar de passieve server. Tijdens dit proces stopt de actieve server met het verlenen van services en wordt hij de nieuwe passieve server. De oude passieve server verandert vervolgens in de nieuwe actieve server en begint met de verlening van alle services van de high-availability cluster. Deze optie is niet beschikbaar als de passieve server de status fout heeft of niet is geselecteerd.
  • DSM updaten: opent de pagina DSM updaten van het Configuratiescherm. Hier kunt u beide servers bijwerken.
  • High-availability-cluster uitschakelen: schakelt zowel de actieve als de passieve servers uit.
  • Actieve server uitschakelen: schakel de actieve server uit. De services worden overgeschakeld wanneer de passieve server gezond is.
  • Passieve server uitschakelen: schakelt de passieve server uit.
  • Actieve server opnieuw opstarten: voor het opnieuw opstarten van de actieve server. De services worden overgeschakeld wanneer de passieve server gezond is.
  • Passieve server opnieuw opstarten: start de passieve server opnieuw op.
  • De onbekende server herstellen: herstelt de passieve server wanneer de passieve server wordt vervangen of nog steeds op de status onbekend staat.
  • Systeempieptoon uit: stopt de pieptoon van de actieve en passieve server bij een fout. Het systeem blijft een pieptoon weergeven indien een andere fout op de actieve of passieve server optreedt.

Opmerking:

  • schakelt u de passieve server uit of is er geen passieve server dan kunnen de services niet worden overgeschakeld wanneer er een fout op de actieve server optreedt.

Servergrafiek en informatie

De servergrafiek toont de serverstatus. De locatie komt overeen met de kant van de modules waarop de server is verbonden. Plaats de muiscursor over de locatie voor gedetailleerde informatie.

Het bevat de volgende situaties:

  • Er is geen passieve server in deze high-availability cluster: toont een leeg kader.
  • De detectie van de passieve server is mislukt: toont een onscherp beeld.
  • Uitroepteken: hardwarestoring of serverinformatie is niet beschikbaar door onderbroken Heartbeat-verbinding.

Opmerking:

  • wordt "onbekend" aangegeven op de andere server dan betekent dit dat de server niet tot de high-availability cluster behoort. Klik op Beheren en selecteer De onbekende server herstellen om dit probleem op te lossen.

De serverinformatie bevat de volgende velden. Het toont N/A wanneer er geen passieve server wordt gedetecteerd.

  • Serienummer: serienummer van de Synologyserver.
  • Ventilatorstatus: status van de ventilatoren van de server. Bij een storing van een van de ventilatoren verschijnt een melding. Neem in dat geval contact op met uw plaatselijke verdeler voor ondersteuning.
  • Temperatuur: temperatuur van de Synologyserver.
  • Vermogenstatus: toont of het vermogen van de server normaal is. AIs het servermodel is uitgerust met redundante stroombronnen, verschijnt er een waarschuwing als bij een van beide een storing optreedt.
  • Fysiek geheugen: de fysieke RAM-grootte van de Synology-server. Overschakelen kan niet worden uitgevoerd wanneer de twee servers niet dezelfde fysieke RAM-grootte hebben.

Connectiviteitstatus tussen servers

Dit gedeelte bevat afbeeldingen tussen de actieve en passieve servers. Het bevat de volgende situaties:

  • Geeb afbeelding: er is geen passieve server gedetecteerd.
  • Stilstaande afbeelding: er is een onjuiste Heartbeat-verbinding.
  • Uitroepteken: wijst op een abnormale communicatie tussen de twee servers of incorrecte Heartbeat-verbinding.

Netwerkinterfacestatus

Dit gedeelte bevat afbeeldingen tussen servers en gegevensnetwerk. Het bevat de volgende situaties:

  • Geen uitroepteken: geeft aan dat er geen netwerkstoring is gedetecteerd.
  • Uitroepteken: geeft aan dat er een storing is opgetreden in de gecontroleerde netwerkinterface.

Gedeelde-opslaggrafiek en -informatie

De gedeelde-opslaggrafiek toont de connectiviteitstatus tussen servers en gedeelde opslag. Plaats de muiscursor over de locatie voor informatie.

Het bevat de volgende situaties:

  • Geen uitroepteken: geeft aan dat er een fout is gedetecteerd.
  • Uitroepteken: geeft de abnormale connectiviteitstatus van de uitbreidingseenheden aan.

De basisinformatie van de gedeelde opslag bevat de volgende velden. Ga naar Opslagbeheer voor gedetailleerde informatie.

  • Ventilatorstatus: toont de status van de ventilatoren van uw uitbreidingseenheden. Als een ventilator slecht werkt, verschijnt er een waarschuwing. Neem in dat geval contact op met uw plaatselijke verdeler voor ondersteuning.
  • Vermogenstatus: toont of het vermogen van de server normaal is. Een waarschuwing verschijnt bij een storing van een van de voedingseenheden.
