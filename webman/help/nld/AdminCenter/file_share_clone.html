Een gedeelde map klonen*

U kunt een gedeelde map op Btrfs-volumes klonen om directe kopieën van een bestaande gedeelde map te maken. Het klonen van een gedeelde map zal slechts een fractie van de opslagruimte in gebruik nemen. 

Een gedeelde map klonen:

  1. ga naar Configuratiescherm > Gedeelde map.
  2. Selecteer de gewenste gedeelde map in het Btrfs-bestandssysteem.
  3. Klik op Maken > Klonen.
  4. Specificeer een naam en beschrijving voor de nieuwe gedeelde map en wijzig indien nodig andere opties. (Gedetailleerde informatie over deze opties vindt u in Een gedeelde map maken.)

Opmerking:

  • alleen gedeelde mappen op Btrfs-volumes kunnen worden gekloond.
  • De gekloonde gedeelde map wordt op hetzelfde volume van de originele gedeelde map gemaakt.
  • De instellingen van geavanceerde gegevensintegriteitsbescherming worden toegepast op de gekloonde gedeelde map en kan niet worden gewijzigd.
