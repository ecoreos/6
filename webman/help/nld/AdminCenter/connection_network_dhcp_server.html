DHCP Server

Uw DiskStation neemt de rol aan van DHCP Server en wijst dynamische IP-adressen toe aan DHCP-clients binnen uw lokaal netwerk.

Om DHCP Server in te schakelen:

  1. ga naar Configuratiescherm > DHCP Server > Netwerkinterface.
  2. Selecteer een netwerkinterface (bijv. LAN 1) om IP-adressen toe te wijzen en klik op Bewerken.
  3. Klik in het pop-upvenster op het tabblad DHCP Server en selecteer DHCP Server inschakelen.
  4. Geef de volgende DHCP-instellingen op:
    • Leasetijd adres (minuten)
    • Primaire DNS
    • Secundaire DNS (optioneel)
    • Domeinnaam (optioneel)
    • Web Proxy Automatic Discovery inschakelen (optioneel)
  5. Stel minstens één subnet in. Klik voor de installatie van subnet op Toevoegen om de volgende informatie op te geven:
    • Start IP-adres
    • Einde IP-adres
    • Netmasker
    • Gateway
  6. Klik op Maken om het aangemaakte subnet in de lijst te tonen.
  7. Nadat u voldoende subnetten hebt aangemaakt, moet er een minstens een activeren.
  8. Klik op OK.

Opmerking:

  • u kunt het aantal subnetten maken dat u wilt.
  • Indien gecoördineerd met een DHCP relay-server, kan uw DiskStation als DHCP Server IP-adressen toewijzen aan apparaten op een ander subnet.

DHCP-clients

Hier ziet u een lijst van DHCP-clients en hun netwerkconfiguraties (bv. MAC-adres, IP-adres, hostnaam, geldigheidsduur van de DHCP-licentie en de reserveringsstatus).

  1. Ga naar Configuratiescherm > DHCP Server > Netwerkinterface en selecteer de gewenste interface.
  2. Klik op Bewerken > DHCP-clients om clients en details te zoeken.

Om IP-adressen voor DHCP-clients te beheren:

tijdens verlenging van de DHCP-lease kunt u clients altijd hetzelfde IP-adres toewijzen en clients beheren.

  • Om DHCP-clients toe te voegen: Klik op Toevoegen om het IP-adres toe te wijzen aan het clientapparaat.
  • Om het IP-adres te reserveren: Klik op het slotpictogram om het IP-adres voor het huidige apparaat te reserveren.
  • Om de reservering te annuleren: klik op het slotpictogram om toe te staan dat het huidige apparaat door een ander apparaat wordt gebruikt na verstrijken van de leasetermijn van het adres.

Opmerking:

als een apparaat zowel IPv4 als IPv6-adressen kan hebben, kunt u een IPv4-adres en een IPv6-adres voor het apparaat reserveren.

Om te zoeken naar DHCP-clients:

  1. Klik op het filterpictogram in de rechterbovenhoek.
  2. Selecteer een van de onderstaande zoekcriteria:
    • Alle: lijst van alle DHCP-clients en DHCP-reserveringen.
    • DHCP-clients: lijst van alle DHCP-clients met gereserveerde of niet-gereserveerde IP-adressen.
    • DHCP-reserveringen: lijst van alle gereserveerde IP-adressen Een gereserveerd IP-adres kan nog niet aan om het even welk apparaat worden gekoppeld.

DHCP-opties

Na het specificeren van de DHCP-opties kan de als DHCP Server fungerende DiskStation gerelateerde informatie naar DHCP-clients in een bepaald subnet verzenden.

DHCP-opties voor een subnet specificeren:

  1. ga naar Configuratiescherm > DHCP Server > Netwerkinterface.
  2. Selecteer de gewenste netwerkinterface en klik op Bewerken.
  3. Ga naar DHCP Server en Subnetlijst.
  4. Voer een van de onderstaande handelingen uit:
    • DHCP-opties voor een bestaand subnet specificeren: selecteer de gewenste interface en klik op Bewerken.
    • Een subnet maken en bijbehorende DHCP-opties specificeren: klik op Toevoegen en specificeer de subnetinstellingen.
  5. Selecteer de gewenste DHCP-opties om waarden te specificeren en klik op de lege velden om de invoer te bewerken.
  6. Klik op OK.

Leverancier

DHCP-clients of andere leveranciers kunnen aanvullende DHCP-informatie vereisen. Door DHCP-opties voor leveranciers in te stellen, levert uw DiskStation dergelijke informatie aan clients.

Om de door een leverancier ondersteunde DHCP-opties te weten, moet u contact opnemen met de desbetreffende leverancier.

Leveranciers toevoegen:

  1. ga naar Configuratiescherm > DHCP Server > Leverancier.
  2. Klik op Maken om de leverancier en DHCP-opties te specificeren:
    • Naam: specificeer de naam van de leverancier.
    • Code: specificeer de numerieke code van de DHCP-optie.
    • Waarde: specificeer de optiewaarde.
    • +: een DHCP-optie toevoegen.
    • -: een DHCP-optie verwijderen.
  3. Klik op Maken om de installatie te voltooien.

DHCP-opties voor leveranciers bewerken:

  1. ga naar Configuratiescherm > DHCP Server > Leverancier.
  2. Selecteer de gewenste leverancier en klik op Bewerken om de bijbehorende DHCP-opties te beheren.
  3. Klik op OK.

PXE

Uw DiskStation kan fungeren als DHCP Server voor de implementatie van de setup van PXE (Pre-Boot Execution Environment). PXE is een netwerkomgeving waarmee clientapparaten via een netwerkinterface kunnen worden opgestart in plaats van gegevensopslagapparaten (bv. harde schijven).

Door PXE te implementeren, kan een opstartlader op uw DiskStation of een externe TFTP-server worden opgeslagen en kunnen clientapparaten met de opstartlader via het netwerk worden opgestart.

Om uw DiskStation als een boot loader-server te gebruiken:

  1. Controleer of u TFTP-service inschakelen hebt geselecteerd (onder Configuratiescherm > Bestandsservices > TFTP).
  2. Selecteer een map die het boot loaderbestand host als de TFTP rootmap.
  3. Vergewis u ervan dat minstens een DHCP-subnet is ingeschakeld (bij Configuratiescherm > DHCP Server > Netwerkinterface).
  4. Ga naar Configuratiescherm > DHCP Server > PXE.
  5. Selecteer PXE inschakelen (Pre-boot Execution Environment).
  6. Selecteer Lokale TFTP-server en selecteer het boot loader-bestand.
  7. Klik op Toepassen om de instellingen op te slaan.

Om de externe TFTP-server als een boot loader-server te gebruiken:

door een externe TFTP-server als boot loader-server te gebruiken, kunnen uw clients zonder het wijzigen van de huidige netwerkinfrastructuur een boot-up uitvoeren.

  1. Vergewis u ervan dat minstens een DHCP-subnet is ingeschakeld (bij Configuratiescherm > DHCP Server > Netwerkinterface).
  2. Ga naar Configuratiescherm > DHCP Server > PXE.
  3. Selecteer PXE inschakelen (Pre-boot Execution Environment).
  4. Selecteer Externe TFTP-server en specificeer het server-IP-adres en het bestandspad van het boot loader-bestand.
  5. Klik op Toepassen om de instellingen op te slaan.
