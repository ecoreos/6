Geavanceerde gedeelde mapmachtigingen beheren

De pagina in Configuratiescherm > Gedeelde map > Bewerken > Geavanceerd bevat de onderstaande opties om de toegangsmachtigingen van een gedeelde map verder te verfijnen.

Geavanceerde sharemachtigingen

Geavanceerde sharemachtigingen bieden een extra controlelaag voor het beheer van toegangsmachtigingen van gedeelde mappen. Waneer ingeschakeld, kunnen gebruikers en groepen alleen de inhoud van een gedeelde map wijzigen als de gebruiker of groep over geavanceerde sharemachtigingen en Windows ACL-machtigingen beschikt (onder Gedeelde map > Bewerken > Machtigingen).

Geavanceerde sharemachtigingen inschakelen:

  1. Schakel het selectievakje Geavanceerde sharemachtigingen inschakelen in.
  2. Klik op Geavanceerde sharemachtigingen om de geavanceerde sharemachtigingen van de gedeelde map te wijzigen.

Opmerking:

  • geavanceerde sharemachtigingen zijn van toepassing wanneer een gebruiker toegang maakt tot een gedeelde map via de volgende services: Windows File Sharing, Apple File Sharing, File Station, FTP en WebDAV.

Geavanceerde instellingen

U kunt indien gewenst verdere beperkingen instellen voor gebruikers die toegang maken tot een gedeelde map via File Station of WebDAV.

  • Bladeren in map uitschakelen: het inschakelen van deze optie beperkt voor gebruikers de weergave van de inhoud van de gedeelde map.
  • Wijziging van bestaande bestanden uitschakelen: het inschakelen van deze optie beperkt het verplaatsen, verwijderen of wijzigen van bestanden in de gedeelde map door gebruikers. Denk eraan dat de gebruikers nog steeds de inhoud van de gedeelde map kunnen bekijken, downloaden/uploaden, kopiëren of uitpakken.
  • Bestand downloaden uitschakelen: het inschakelen van deze optie beperkt het downloaden van de inhoud van de gedeelde map voor gebruikers.

Opmerking:

de bovenstaande beperkingen zijn niet van toepassing voor gebruikers van de groep administrators.
