Gedeelde map

Gedeelde mappen zijn basismappen waar u bestanden en mappen op uw DiskStation kunt opslaan. Voor u bestanden op uw DiskStation opslaat, moet u minstens één gedeelde map maken. Afhankelijk van de aan te passen toegangsrechten kunt u opgeslagen gegevens privé houden of met specifieke gebruikers of gebruikersgroepen delen. Daarnaast kunnen gedeelde mappen worden gecodeerd op het moment dat ze worden aangemaakt, waardoor een extra beveiligingslaag wordt gegarandeerd.

Standaard gedeelde mappen

Hieronder vindt u een lijst van gedeeldmappen die automatisch worden aangemaakt wanneer bepaalde toepassingen, services of pakketten worden geïnstalleerd of ingeschakeld.

Naam Beschrijving
public De map public wordt automatisch gegenereerd bij de eerste installatie van uw DiskStation.*
web De map web is de standaardopslaglocatie voor de inhoud van een website gehost door DiskStation. Deze map wordt automatisch aangemaakt wanneer Web Station wordt ingeschakeld.
foto De map photo wordt gebruikt voor de opslag van foto’s en video’s die u wilt delen op Photo Station. Deze map wordt automatisch aangemaakt wanneer Photo Station of DLNA/UPnP Media Server worden ingeschakeld.
muziek De map music wordt gebruikt voor de opslag van muziek die u wilt beluisteren met Audio Station. Deze map wordt automatisch aangemaakt wanneer Audio Station wordt ingeschakeld.
video De map video wordt gebruikt voor de opslag van video's waarin u kunt bladeren met DLNA/UPnP-apparaten. Deze map wordt automatisch aangemaakt wanneer Media Server wordt ingeschakeld.
surveillance De map surveillance wordt gebruikt voor de opslag van opnames van Surveillance Station. Deze map wordt automatisch aangemaakt wanneer Surveillance Station wordt ingeschakeld. Het is een alleen-lezenmap waar alleen de systeembeheerder toegang toe heeft.
home De map home biedt iedere gebruiker een persoonlijke ruimte waarin hij gegevens kan opslaan en waartoe alleen de gebruiker toegang heeft. Deze map wordt automatisch aangemaakt wanneer Gebruiker basismap wordt ingeschakeld.
homes De map homes bevat de basismappen van alle gebruikers. Deze map wordt automatisch aangemaakt wanneer Gebruiker basismap wordt ingeschakeld. Alleen de systeembeheerder heeft toegang tot de basismappen van alle gebruikers en kan deze beheren.
NetBackup De map NetBackup wordt automatisch aangemaakt wanneer Netwerkback-up is ingeschakeld.
usbshare De map usbshare[aantal] wordt automatisch aangemaakt wanneer u een externe USB-schijf aansluit op de USB-poort van DiskStation.*
satashare De map satashare wordt automatisch aangemaakt wanneer u een externe eSATA-schijf aansluit op de eSATA-poort van DiskStation.*

____
* Uitsluitend beschikbaar op specifieke modellen.
