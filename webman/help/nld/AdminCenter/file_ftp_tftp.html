TFTP

TFTP (Trivial File Transfer Protocol) is een eenvoudig protocol voor bestandsoverdracht dat doorgaans wordt gebruikt voor de overdracht van configuraties of het opstarten van bestanden wanneer geen verificatie wordt vereist. Door de TFTP-service in te schakelen, kan DiskStation opstartladers overdragen aan clientapparaten.

TFTP-instellingen configureren:

  1. ga naar Configuratiescherm > Bestandsservices > TFTP en selecteer TFTP-service inschakelen.
  2. Geef in het veld TFTP-hoofdmap op welke map van DiskStation toegankelijk is voor TFTP-clients.
  3. Klik op Geavanceerde instellingen om de volgende instellingen te configureren:
    • Logboek van TFTP-bestandsoverdracht inschakelen: via TFTP overgedragen gegevens bijhouden.
    • TFTP-clientmachtiging: TFTP-clientmachtigingen bepalen:
      • Alleen-lezen: TFTP-clients kunnen uitsluitend de inhoud van de basismap bekijken.
      • Beschrijfbaar: TFTP-clients kunnen de inhoud in de basismap wijzigen.
    • Geautoriseerde clients: kies welke clients een verbinding mogen maken via TFTP:
      • Alle verbindingen: alle clients kunnen via TFTP verbinding maken.
      • Alleen adressen van het volgende IP-adresbereik toestaan: alleen clients van het gespecificeerde IP-bereik kunnen een verbinding maken via TFTP.
    • Time-out: blijft uw TFTP-server (ofwel uw DiskStation) gedurende een ingestelde tijdsduur onbereikbaar dan wordt de TFTP-verbinding gestopt.
  4. Klik op Toepassen om de instellingen op te slaan.
