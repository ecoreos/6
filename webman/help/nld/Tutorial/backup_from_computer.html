Hoe maak ik met Cloud Station een back-up van de gegevens op mijn pc/Mac

Overzicht

Cloud Station Server is meer dan alleen gesofisticeerde platformonafhankelijke synchronisatiesoftware. Gebruikers kunnen hem ook gebruiken als ze op een meer doeltreffende wijze een back-up van hun persoonlijke computer willen maken Cloud Station Backup is een real-time back-upoplossing die minder bandbreedte gebruikt en een betere bescherming biedt met maximaal 32 historische versies van een enkel bestand.

Cloud Station Server instellen op Synology NAS

  1. Ga naar Package Center, zoek Cloud Station Server en klik vervolgens opÂ Installeren.
  2. Start Cloud Station Server op.
  3. Ga naar Overzicht en klik op Nu downloaden onder Cloud Station Backup. (U kunt Cloud Station Backup ook downloaden vanaf  Synology Download Center.)
  4. Start het installatieprogramma op uw computer en volg de instructies om Cloud Station Backup te installeren.
  5. Voer na de installatie Cloud Station Backup uit op uw computer. Klik op Nu starten.
  6. Voer het adres in voor het Synology NAS waarop Cloud Station Server (of QuickConnect ID) wordt uitgevoerd, samen met uw gebruikersnaam en wachtwoord. (U kunt ook rechtsklikken op het zoekpictogram en Cloud Station Backup zoekt automatisch naar andere Synology NAS in uw LAN). Klik op Volgende.
  7. Selecteer uw back-upbron en schakel de selectievakjes van submappen uit die u niet wilt synchroniseren.
  8. Selecteer een map op uw Synology NAS door op Selecteren te klikken. Cloud Station Backup maakt een back-upmap met uw computernaam in het geselecteerde doel.
  9. Klik op Back-upregels wanneer u synchronisatiefilters of regels wilt instellen.
  10. Klik op Volgende om uw back-upinstellingen te bekijken en klik vervolgens op  Voltooid.
  11. In de hoofdtoepassing ziet u de back-upvoortgang.

Een vorige versie van een op Cloud Station Server gesynchroniseerd bestand downloaden of terug te zetten

  1. Ga naar Cloud Station Server > Versiegeschiedenis en lokaliseer het bestand of de map die u wilt terugzetten.
  2. Selecteer het bestand dat u wilt ophalen en klik op Actie. Klik op Vorige versies doorbladeren. Werd een bestand per ongeluk verwijderd of van uw computer verwijderd, dan kunt u het terugzetten met een klik op Terugzetten.
  3. Selecteer de versie van het bestand dat u wilt downloaden en klik op Downloaden. Als u zeker weet dat u de geselecteerde versie wilt terugzetten, klikt u op Terugzetten waarna uw huidige versie overschreven zal worden.
