Comment puis-je passer au mode bureau de DSM ?

Appuyez sur le bouton d'option, dans le coin supérieur droit du panneau de gauche, puis sur le mode de bureau.

Comment puis-je passer au mode mobile depuis le mode de bureau ?

Appuyez sur le bouton « DSM mobile », en bas du bureau.

Je suis redirigé vers le mode bureau et je ne peux pas à passer au mode mobile

Le mode mobile n'est disponible que pour les utilisateurs du groupe d'administration. Les autres utilisateurs ne peuvent utiliser que le mode bureau.

Quels périphériques DSM prend-il en charge ?

Périphériques mobiles : iPhone, iPad, et iPod sous iOS 7 ou une version ultérieure ; les périphériques sous Android 4.0 ou une version ultérieure
Navigateurs : Chrome, Safari
