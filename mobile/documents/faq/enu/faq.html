How do I switch to the desktop mode of DSM

Please tap the option button on the upper right corner in left panel, and tap the desktop mode.

How do I switch to the mobile mode from desktop mode

Please tap the "DSM mobile" button at the bottom of the desktop.

I am redirected to desktop mode and cannot switch to mobile mode

Mobile mode is only available for users in administration group. Other users can use only desktop mode.

What devices does DSM mobile support

Mobile devices: iPhone, iPad, and iPod running iOS 7 or later; devices running Android 4.0 or later
Browsers: Chrome, Safari
