Como mudo para o modo área de trabalho do DSM?

Toque no botão de opção no canto superior direito no painel esquerdo e toque no modo área de trabalho.

Como mudo para o modo móvel do modo área de trabalho

Toque no botão "DSM mobile" na parte inferior da área de trabalho.

Sou redirecionado para o modo área de trabalho e não consigo mudar para o modo móvel

O modo móvel está disponível apenas para usuários do grupo administration. Outros usuários podem usar apenas o modo área de trabalho.

Quais dispositivos o DSM mobile suporta

Dispositivos móveis: iPhone, iPad, e iPod com iOS 7 ou posterior, dispositivos com Android 4.0 ou posterior
Navegadores: Chrome, Safari
